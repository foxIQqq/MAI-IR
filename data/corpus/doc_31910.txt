Релиз дистрибутива для создания сетевых хранилищ FreeNAS 8.0.1

Представлен релиз дистрибутива для быстрого развёртывания сетевого хранилища (NAS, Network-Attached Storage) - FreeNAS 8.0.1. Дистрибутив основан на FreeBSD 8 и полностью настраивается через веб-интерфейс, поддерживает FTP, NFS, Samba, AFP, rsync и iSCSI протоколы, программный RAID (0,1,5). Для авторизации может быть использован LDAP/Active Directory. Размер iso-образа составляет 102 Мб. Основные улучшения: В БД отделены внешние разделы от устройств FreeBSD; Добавлена поддержка экспортирования ZFS zvols (виртуальные устройства) через iSCSI; В интерфейс добавлена система уведомления о наступлении событий; Переработан бэкенд электронной почты; Многочисленные изменения в web-интерфейсе, добавлены дополнительные проверки корректности заполнения полей ввода; Добавлена поддержка контроля за системой бесперебойного питания; Добавлена поддержка клиента и сервера rsync; В web-интерфейс добавлена возможность планирования работ через cron; Значительно переработан код системы репликации снапшотов; Фреймворк django, на базе которого построен управляющий web-интерфес, обновлен до версии 1.3.1; Устранены уязвимости, на днях обнаруженные в ветке FreeBSD 8; В интерфейс добавлена кнопка для просмотра статуса zpool; Улучшена поддержка iSCSI; Вместо текстового поля для выбора метода шифрования для LDAP теперь отображается выпадающий список с перечнем возможных элементов; Устранены проблемы с удалением снапшотов; Устранена ошибка, мешающая импортированию zpool-ов, созданных в Solaris; Реализована возможность добавления публичных ключей для SSH. Развитие проекта FreeNAS курирует компания iXsystems, развивающая операционную систему PC-BSD и способствующая улучшению поддержки ZFS во FreeBSD. Одновременно в рамках проекта CoreNAS (OpenMediaVault) ведется работа по созданию варианта FreeNAS, основанного на пакетной базе Debian GNU/Linux. Оба варианта развиваются параллельно, версия на базе FreeBSD ориентирована в основном на использование возможностей ZFS, в то время как версия на базе Linux развивается в направлении расширенной поддержки встраиваемых устройств и гибкой системы установки дополнений. Кроме того, основателем дистрибутива FreeNAS создан проект BSD Router Project, нацеленный на предоставление инструментов для создания программных маршрутизаторов для сетей предприятий и небольших интернет-провайдеров (поддержка протоколов маршрутизации OSPFv2, OSPFv3, RIP, RIPng и BGP основана на базе пакетов Quagga и BIRD, управление производится через CLI-интерфейс в стиле Cisco).
