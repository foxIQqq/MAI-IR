Выпуск SpaceVim 2.0, дистрибутива редактора Vim

Представлен выпуск проекта SpaceVim 2.0, развивающего дистрибутив текстового редактора Vim с подборкой плагинов для поддержки различных языков программирования и возможностей, свойственных интегрированным средам разработки. Плагины сгруппированы в наборы с реализацией тех или иных возможностей. Например, набор для разработчиков на языке Python включает плагины deoplete.nvim, neomake и jedi-vim для автодополнения кода, проверки синтаксиса и интерактивного обращения к документации. Таким образом пользователю достаточно лишь выбрать необходимую функциональность без необходимости раздельного подбора плагинов. В новой версии предложены новые наборы для разработчиков, использующих cmake, jr, jsonnet, octave, yang, haxe, postscript, teal, verilog и django. В набор с чатом добавлена поддержка Gitter и IRC. Добавлены новые клавиатурные комбинации. Реализован плагин для автоматического сохранения. Для vim8 добавлена поддержка буфера обмена и реализована полоса прокрутки.
