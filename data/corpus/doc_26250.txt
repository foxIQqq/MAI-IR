Релиз свободной системы отслеживания ошибок Bugzilla 3.6

Выпущен релиз Bugzilla 3.6, системы для ведения базы данных ошибок, контроля за их исправлением и общего координирования процесса разработки. Наиболее значительным новшеством версии 3.6 является реализация поддержки подключения дополнений, представляющих собой обособленные модули на языке Perl, расширяющие функциональность Bugzilla без модификации основной кодовой базы. Из других изменений можно отметить: Общее улучшение юзабилити пользовательского интерфейса; Усовершенствование функции быстрого поиска (Quicksearch); Добавление упрощенного интерфейса для просмотра нерешенных проблем; Поддержку Apache-механизма SUExec, позволяющего выполнять Bugzilla под отдельным идентификатором пользователя; Возможность отправки писем-уведомлений с приложениями; Реализация JSON-RPC API для интеграции с другими web-службами; Улучшение средств для миграции с других систем отслеживания ошибок. Одновременно с выходом Bugzilla 3.6 объявлено о прекращении поддержки ветки Bugzilla 3.0.x, последним релизом которой останется 3.0.11.
