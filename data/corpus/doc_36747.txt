Вышел Calculate Linux 13.4

Доступен релиз отечественного дистрибутива Calculate Linux 13.4, построенного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в 7 редакциях: Calculate Directory Server (CDS), Calculate Linux Desktop с рабочим столом KDE (CLD), GNOME (CLDG) и XFCE (CLDX), Calculate Media Center (CMC), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS). Основные изменения: Добавлена поддержка загрузки с использованием UEFI. Для перехода в спящий режим теперь используются патчи TuxOnIce. Добавлена поддержка планировщика ввода/вывода BFQ . Добавлена опциональная поддержка полного удаления ядра при удалении пакета calculate-sources. В состав утилит Calculate добавлена совместимая с Gentoo поддержка бэкапа файлов настроек, а также модуль применения изменений, аналогичный dispatch-conf, поддерживающий работу как в консоли, так и в графическом клиенте. В установщике расширены опции автоматической разметки. Выполнен переход на новые имена сетевых интерфейсов, используемый в последних версиях udev. Ускорена загрузка сетевых Linux клиентов при отсутствии сети. Добавлена синхронизация профилей доменных пользователей при перезагрузке или выключении питания. В CLDG и CLDX IRC клиент XChat заменён на HexChat. В CLD в качестве графического редактора дисковых разделов теперь используется GParted. Поддержка GNOME 3 прекращена. В состав CLDG возвращён GNOME 2.32. В следующей версии разработчики планируют полностью отказаться от поддержки GNOME. Особенности Calculate Linux: 100% совместимость с Gentoo с поддержкой бинарных репозиториев обновлений. Поставка специально разработанных в рамках проекта Calculate утилит для настройки, сборки и установки системы. Готовое клиент-серверное решение. Быстрое развёртывание на предприятии. Полноценная работа в гетерогенных сетях. Удобство администрирования. Rolling-release модель обновлений. Поддерживается интерактивная сборка системы - подготовка ISO-образа системы под свои задачи. Состав версий: CLD (KDE desktop, i686 - 1.97 G, x86_64 - 2.18 G): KDE 4.10.1, Amarok 2.7.0, digiKam 3.0, Chromium 26.0.1410.43, ядро Linux 3.8.4, LibreOffice 3.6.4.3; CLDG (GNOME desktop, i686 - 1.51 G, x86_64 - 1.72 G): GNOME 2.32, Gimp 2.8.4, Chromium 26.0.1410.43, ядро Linux 3.8.4, LibreOffice 3.6.4.3; CLDX (XFCE desktop, i686 - 1.40 G, x86_64 - 1.60 G): Xfce 4.10, GIMP 2.8.4, Chromium 26.0.1410.43,ядро Linux 3.8.4, LibreOffice 3.6.4.3; CMC (Media Center, i686 - 933 M, x86_64 - 1.09 G): XBMC 12.1, ядро Linux 3.8.4, Xorg-server 1.13.1; CLS (Linux Scratch, i686 - 648 M, x86_64 - 669 M): Openbox 3.5.0, ядро Linux 3.8.4, Xorg-server 1.13.1 , минимальный набор драйверов и утилит для сборки пользовательских Live-систем под свои задачи; CDS (Directory Server, i686 - 637 M, x86_64 - 661 M): Bind 9.9.2_p2, ядро Linux 3.8.4, OpenLDAP 2.4.30, Postfix 2.10, ProFTPD 1.3.4a, Samba 3.6.12; CSS (Scratch Server, i686 - 370 M, x86_64 - 386 M): ядро Linux 3.8.4 и минимальный набор компонентов для сборки серверных Live-систем под свои задачи.
