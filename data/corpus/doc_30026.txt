Релиз полностью свободного Linux дистрибутива Trisquel 4.5

Представлен релиз полностью свободного Linux-дистрибутива Trisquel 4.5, основанного на пакетной базе Ubuntu Linux 10.10 и ориентированного для небольших предприятий, домашних пользователей и образовательных учреждений. В дистрибутиве проведена чистка всех несвободных компонентов, таких как бинарные драйверы, firmware и элементы графического оформления, распространяемые под несвободной лицензией или использующие зарегистрированные торговые марки. Trisquel лично одобрен Ричардом Столманом, официально признан Фондом Свободного ПО в качестве полностью свободного и помещен в список рекомендованных фондом дистрибутивов. Размер установочного iso-образа составляет 683 Мб. Из изменений можно отметить: Обновление версий программ (совершен переход с пакетной базы Ubuntu 10.04 на Ubuntu 10.10). Из новых версий можно отметить: Linux-ядро 2.6.35-libre, Xorg 7.5, GNOME 2.32, браузер на основе Firefox 3.6.15, OpenOffice.org 3.2; Переработан процесс разработки дистрибутива, переписаны скрипты для пересборки пакетов, автоматизировано использование изолированных окружений для сборки, упрощен процесс внесения изменений в оригинальные пакеты; Добавлен новый менеджер загрузки Live-системы, обновлен инсталлятор; В комплект включены новые приложения, такие как клиент для организации удаленного доступа к рабочему столу Remmina, клиент для интеграции с социальными сетями Gwibber и инструмент для резервного копирования Deja-dup; Изменены базовые настройки web-браузера, включен режим "http pipelining", добавлены дополнения для усиления приватности и блокирования отслеживания перемещений пользователя через cookie. В качестве поисковой системы по умолчанию используется мета-поисковик DuckDuckGo, отличающийся блокированием рекламных сайтов и трепетным отношением к приватности. Добавлен плагин FlashVideoReplacer, позволяющий смотреть видеоролики в YouTube, Vimeo и других сервисах без Adobe Flash; По умолчанию задействован свободный видеодрайвер Nouveau; Несмотря на полный отказ от проприетарных компонентов, Trisquel совместим с Java (OpenJDK) и Adobe Flash (Gnash), поддерживает большинство аудио и видео форматов, включая работу с защищенными DVD, задействуя при этом только полностью свободные реализации данных технологий.
