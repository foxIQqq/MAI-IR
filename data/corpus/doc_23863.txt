Вышел первый кандидат в релизы openSUSE 11.2

Вышел первый кандидат в релизы openSUSE 11.2, означающий завершение разработки и переход на финальную фазу тестирования перед релизом, намеченным на 12 ноября 2009 года. По сравнению с восьмым тестовым выпуском в дистрибутив включены новые версии программ: Linux ядро 2.6.31.3, GNOME 2.28, PulseAudio 0.9.19, Evolution 2.28, Qt 4.5.3, SeaMonkey 2.0 RC 1, Bluez 4.55, X.Org драйвер radeonhd 1.3, Blender 2.49a. Базовые новшества openSUSE 11.2: Добавлена функция совершения обновления в Live-режиме с минимальным временем простоя, позволяющая целиком обновить работающую систему с версии 11.1 до 11.2 методом напоминающим dist-upgrade в Debian, с фоновой загрузкой пакетов перед непосредственным началом проведения обновления; В комплект включены приложения для работы с десктопа в социальных сетях подобных Facebook, Twitter и identi.ca; В качестве файловой системы по умолчанию задействована Ext4 Образ LiveCD теперь может быть использован и на USB Flash. В состав интегрированы средства для подготовки собственных LiveCD образов. Для сжатия данных на LiveCD отныне используется алгоритм сжатия LZMA; В качестве десктоп-окружения по умолчанию, при установке с DVD, будет использоваться KDE 4.3; Время поддержки дистрибутива (выпуска обновлений) будет сокращено с 24 до 18 месяцев; Разработан web-интерфейс к YaST для осуществления удаленного управления дистрибутивом Пакеты дистрибутива собраны GCC 4.4.1. Пакеты сформированы с использование RPM версии 4.7.1 В состав LiveCD с GNOME добавлен Midnight Commander, Bittorrent-клиент Transmission и Gimp. В состав LiveCD с KDE добавлены новые программы: Choqok, KNode, Kompare, KSystemlog, Okteta, YaKuake. По умолчанию LiveCD содержит переводы для немецкого, французского, итальянского, польского и русского языков; Новое визуальное оформление инсталлятора YaST2 и новая визуальная тема для KDE; Для YaST создан новый центр управления конфигурацией (YaST Control Center) с интерфейсом на базе Qt4; В Qt версии конфигуратора YaST появилась поддержка настраиваемого внешнего вида. В YaST добавлены средства работы с кешем пакетов; В zypper добавлена опция "--download-only", позволяющая осуществить только загрузку компонент, без выполнения действи. Опцию можно использовать совместно с командами install, update, patch, dup и verify. Улучшены средства работы с зеркалами репозиториев пакетов; В интерфейсе для настройки параметров дисплея gnome-display-properties добавлена поддержка автоматического сброса настроек в случае проблем. Улучшение поддержки субноутбуков.
