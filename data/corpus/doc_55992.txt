Выпуск OpenTTD 12.0, свободного симулятора транспортной компании

Доступен релиз OpenTTD 12.0, свободной стратегической игры, симулирующей работу транспортной компании в реальном времени. Начиная с предложенного выпуска изменена нумерация версий - разработчики отбросили потерявшую смысл первую цифру в версии и вместо 1.12 сформировали релиз 12.0. Код проекта написан на языке С++ и распространяется под лицензией GPLv2. Установочные пакеты подготовлены для Linux, Windows и macOS. Изначально OpenTTD развивался как аналог коммерческой игры Transport Tycoon Deluxe, но позднее превратился в самодостаточный проект, значительно обогнавший по возможностям эталонный вариант игры. В частности, в рамках проекта создан альтернативный набор игровых данных, новое звуковое и графическое оформление, существенно расширены возможности игрового движка, увеличены размеры карт, реализован сетевой режим игры, добавлено много новых игровых элементов и моделей. В новой версии значительно улучшена поддержка многопользовательской игры. Для совместной игры теперь достаточно запустить сервер, который может быть настроен как для входа по приглашениям, так и для неограниченного доступа. Благодаря добавлению поддержки механизмов STUN и TURN при организации подключения к сети за транслятором адресов сервер сразу будет доступен без лишних усложнений, таких как настройка проброса сетевых портов. Из других изменений отмечается отображение индикатора потери транспортного средства, автоперемещение камеры по демонстрации на заглавном экране, отключение по умолчанию межблоковых сигналов в GUI и повышение лимита на число файлов NewGRF (Graphics Resource File) до 255.
