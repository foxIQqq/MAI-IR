Выпуск VirtualBox 5.1.16

Компания Oracle выпустила корректирующий релиз системы виртуализации VirtualBox 5.1.16, в котором отмечено 28 исправлений. Среди изменений: В компонентах для гостевых систем и хостов обеспечена совместимость с ядром Linux 4.11; В установщике для Linux исключена пересборка неиспользуемых модулей ядра; В .desktop-файл, входящий в состав хост-компонентов, добавлено действие для открытия окна менеджера виртуальных машин; В дополнениях для гостевых систем на базе Linux решены проблемы с компиляцией модуля ядра для совместного доступа к каталогам в окружении с ядром Linux 4.10; В дополнениях для гостевых систем на базе Linux налажена установка правил загрузки модулей ядра в системах без /etc/depmod.d; Устранён крах, проявляющийся в свежих сборках Windows 10 при отключении поддержки 3D; Решена проблема с выводом ошибки VERR_SSM_LOAD_CPUID_MISMATCH на системах без установленного бита CPUID/HTT, проявляющаяся при сохранении или восстановлении состояния гостевых систем, работающих в режиме SMP; Устранены единичные зависания гостевых систем при определённом стечении обстоятельств; В GUI устранены проблемы с работой полноэкранного режима на системах с несколькими мониторами; Улучшена обработка ошибок в настройках сервера DHCP; Обеспечено скрытие мини-панели из панели задач на некоторых системах X11; Решены проблемы с инициализацией режима Host-only Network, мешающие установке VirtualBox на хостах с Windows 10; Устранены две проблемы в реализации сервера DHCP, которые потенциально могли приводить к крахам; В реализации ICH9 решены проблемы с некорректной инициализацией основной шины для мостов PCI; Решены проблемы с подключением некоторых USB-устройств, содержащих некорректные символы в идентификаторе; Внесены исправления, связанные с поддержкой USB/IP; Устранён крах при завершении работы VBoxSVC и переполнение стека, проявляющееся только в отладочных сборках Windows; Реализована возможность импорта OVF-окружений, использующих более 10 сетевых адаптеров. Решены проблемы с импортом окружений, содержащих большое число дисков.
