Компания Yahoo выпустила кеширующий Traffic Server под лицензией Apache

В понедельник компания Yahoo выпустила под лицензией Apache высокопроизводительный сервер приложений для распределенных систем (cloud computing) - Traffic Server. Данное ПО представляет собой систему динамической обработки HTTP и использовалось Yahoo последние 8 лет для доставки web контента конечным пользователям. В своем сообщении для прессы Yahoo подчеркивает следующие характеристики сервера: Обработка 35 тыс. запросов в секунду, что составляет в день более 3 млрд. сформированных и отправленных пользователям web страниц. Суммарный дневной объем трафика превышает 400 Тб. Одной из сфер применения Трафик Сервера может стать кеширование статичных объектов, таких как файлы, новостные статьи или картинки, что значительно ускорит работу сайта. Также сервер включает в себя набор сервисов для продуктивной работы в качестве распределенной «cloud»-системы: средства конфигурирования, управления сессиями, балансировки, авторизации и маршрутизации запросов. По словам представителей Yahoo Трафик Сервер предлагает «решение проблемы, которую трудно решить», а именно, надежную и расширяемою платформу, способную справиться со все возрастающими объемами Интернет трафика. Выпуская открытую версию Трафик Сервера Yahoo надеется обзавестись сообществом по примеру Hadoop. «Мы можем, буквально, нанять разработчиков и начать сотрудничать с любыми исследовательскими институтами, знакомыми с технологиями, которые мы используем» - сказал вице-президент компании Шелтон Шугар (Shelton Shugar). Проект Traffic Server выпущен под свободной лицензией Apache и доступен через Apache Incubator.
