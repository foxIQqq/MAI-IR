Вышла 3-я версия OLAP сервера Palo

Вышел новый релиз Palo, OLAP сервер с открытым исходным кодом, написанный на C++. Palo предназначен для построения инфраструктуры бизнес-анализа, например, планирования, анализа затрат и генерации бизнес-отчетов. Основная информация о Palo: хранит все данные в оперативной памяти позволяет пользователю задавать свои функции агрегации с помощью правил, в том числе возможна прямая запись в ячейки, содержащие агрегированные данные (splash) разделение прав пользователей (права на БД, кубы, измерения) возможность выделения подмножеств из элементов измерения в соответствии с определенными условиями SDK содержит API для языков C, C++, Java (jpalo.com), PHP. для просмотра срезов многомерного куба могут использоваться plug-in для OpenOffice (palooca) или клиентское приложение на основе Eclipse. Основным изменением в 3-ей версии является увеличение скорости работы при использовании многоядерных процессоров.
