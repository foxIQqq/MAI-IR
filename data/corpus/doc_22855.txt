Вышел релиз X Server 1.6.3

Вышел корректирующий релиз xorg-server 1.6.3, содержащий исправления ошибок, накопившихся в ветке 1.6.x. Следующий релиз X.Org 7.5 / X Server 1.7 ожидается в сентябре или октябре. В X.Org 7.5 / X Server 1.7 появится поддержка технологии Multi-Pointer X и переработанной подсистемы ввода X Input 2, что позволит организовать работу нескольких независимых устройств ввода, например, несколько управляемых разными мышами курсоров на экране или ввод в разные окна с разных клавиатур.
