Доступен Solaris 11.4 SRU24

Опубликовано обновление операционной системы Solaris 11.4 SRU 24 (Support Repository Update), в котором предложена серия очередных исправлений и улучшений для ветки Solaris 11.4. Для установки предложенных в обновлении исправлений достаточно выполнить команду 'pkg update'. В новом выпуске: До версии 20.2 обновлён Oracle Explorer, инструментарий для построения детального профиля конфигурации и состояния системы; Добавлена поддержка накопителей SAMSUNG 7.68TB (MS9AC2DD2SUN7.6T) и 200GB (MS9AC2DD6SUN200G); В состав включён движок SpiderMonkey 60; Добавлены Python-пакеты pyrsistent, chardet и bcrypt, а также пакеты iso8601, netaddr и pycups для Python 3.7; Обновлены версии 87 программ, включая GNOME 3.34, Erlang 22.2.8, GCC 9.3, GNU screen 4.8.0, LLVM/Clang 10.0.0, Node.js 12.16.3, Samba 4.11.9, cups 2.3.1 и xorg-server 1.20.8. Обновлены версии с устранением уязвимостей: Ant 1.10.8 Apache Tomcat 8.5.55 Apache Web Server 2.4.43 BIND 9.11.20 Firefox 68.9.0esr GDB 9.2 GnuPG 2.2.20 Mesa 20.0.2 MySQL 5.6 5.6.48 MySQL 5.7 5.7.30 NSS 3.51 OpenLDAP 2.4.50 PHP 7.4.5 PyYAML version 5.3.1 RabbitMQ 3.8.3 Thunderbird 68.9.0 Wireshark 3.2.4 dnsmasq 2.81 git 2.19.5 glib 2.62.5 libexif 0.6.22 mailman 2.1.33 ntp 4.2.8p14 python 2.7 2.7.18 rsyslog 8.2002.0 ruby 2.5 2.5.8 ruby 2.6 2.6.6 sqlite 3.32.1 squid 4.11 sudo 1.8.31p1 vim 8.2.0444 webkitgtk 2.28.2 zsh 5.8
