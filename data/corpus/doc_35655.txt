Сняты ограничения на тестирование Steam для Linux. Представлены Humble Indie Bundle 7 и OpenMW 0.20

Несколько связанных с играми событий: Компания Valve сняла ограничения на распространение бета-версии клиента Steam для Linux, ранее доступного только лицам, получившим специальные приглашения. Отныне протестировать Steam для Linux может любой желающий, имеющий зарегистрированный аккаунт в сервисе Steam. Следует отметить, что за месяц тестирования, число доступных в каталоге Steam игр для Linux увеличилось с 26 до 36. Пользователи Ubuntu могут установить тестовую версию Steam из специально подготовленного репозитория, обладатели иных дистрибутивов могут попытаться вручную установить deb-пакет. Анонсирована седьмая акция "Humble Indie Bundle", в рамках которой пользователь может получить комплект из 7 многоплатформенных игр в обмен на пожертвование проекту любой суммы. В пакет входят следующие игры: Legend of Grimrock, The Binding of Isaac, Dungeon Defenders, Closure, Shank 2, Snapshot, The Movie. Распространяемые игры могут работать на платформах Mac OS X, Windows и Linux. По договорённостью с компанией Canonical обеспечена возможность установки указанных игр в Ubuntu через штатный интерфейс Software Center. На момент написания новости собрано уже около 950 тыс. долларов. Средний размер суммы взноса составил $6.38. Средний размер взноса пользователей Linux систем составил $10.15, Mac OS X - $7.42, Windows - $5.92. Часть собранных средств (каждый пользователь индивидуально определяет пропорции распределения денег) передается в качестве пожертвований некоммерческой правозащитной организации Electronic Frontier Foundation и фонду Child's Play Charity, осуществляющему снабжение детей, лежащих в больницах, игрушками. Примечательно, что одна из представленных в рамках Humble Indie Bundle 7 игр - Dungeon Defenders, стала первой доступной для Linux игрой на базе движка Unreal Engine 3, что позволяет рассчитывать на скорую адаптацию для Linux и других игр на основе данного движка. Движок был портирован для Linux при участии Райана Гордона (Ryan C. Gordon), который с прошлого года отвечает за принятие патчей и выпуск новых версий библиотек SDL. Доступна новая версия OpenMW 0.20.0, свободной реализации движка для ролевой игры в жанре фэнтези The Elder Scrolls 3: Morrowind. Для Ubuntu доступен PPA-репозиторий, для ROSA 2012.1 пакет можно установить из официального репозитория Contrib, а для других платформ пакеты можно найти здесь. В новом выпуске почти полностью реализована поддержка системы диалогов, представлена начальная поддержка классов системы искусственного интеллекта, подготовлен второй уровень глобальной карты.
