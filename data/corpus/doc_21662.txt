Портирование Android SDK на FreeBSD

В рамках проекта BSDroid ведется работа по портированию во FreeBSD комплекта разработки приложений для мобильной платформы Google Android. Задача проекта - создание полноценного окружения для создания Android приложений на машинах с FreeBSD, без задействования Linux версии Android SDK. В настоящий момент для FreeBSD 8.0-CURRENT уже подготовлены сборки с набором утилит и эмулятором из состава Android SDK.
