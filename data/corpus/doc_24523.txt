Вышел Calculate Linux 10.0

Доступны для загрузки все варианты поставки Calculate Linux 10.0: два легковесных дистрибутива, на базе которых можно собрать систему (CLS/CLSG), два десктопа, оптимизированных под рабочий стол KDE и XFCE (CLD/CLDX), и сервер (CDS). Особенности: 100% совместимость с Gentoo. Поддержка интерактивной сборки системы - внесения изменений в состав дистрибутива. Поддерживаемые языки: английский, испанский, немецкий, португальский, итальянский, русский, украинский, польский и французский. Поддерживается установка на HDD, USB-HDD с файловой системой: ext4, ext3, ext2, reiserfs, xfs и jfs, или на USB-Flash с FAT32 Основные изменения Перенос профилей дистрибутивов в Calculate overlay для управления зависимостями. Сбор дистрибутивов через мета-пакеты с поддержкой перехода между версиями. В браузер Firefox дистрибутивов CLD/CLDX включено расширение Adblock Plus. В CDS добавлена поддержка создания DHCP сетей с различными масками. В CLD добавлен P2P клиент Eiskaltdc, Kbluetooth, в Kopete включена поддержка Facebook. Добавлена настройка OpenGL при загрузке системы с livecd или usb-hdd. Вместо TWM в CLS теперь используется менеджер окон Openbox. Исправлены ошибки, в т.ч. установка системы на 3ware raid массив. Состав версий Calculate Directory Server 10.0: Calculate-server 2.1.6, Bind 9.6.1-P2, Kernel 2.6.31.6, OpenLDAP 2.4.19, Postfix 2.6.5, ProFTPD 1.3.2b, Samba 3.4.3. Calculate Linux Desktop KDE 10.0: X.Org 7.4, Kernel 2.6.31.6, Firefox 3.5.4, KDE 4.3.3, OpenOffice 3.1.1, Calculate-client 2.1.8 Calculate Linux Desktop XFce 10.0: X.Org 7.4, Kernel 2.6.31.6, Firefox 3.5.4, XFCE 4.6.1, OpenOffice 3.1.1, Calculate-client 2.1.8 Calculate Linux Scratch 10.0: X.Org 7.4, Kernel 2.6.31.6, Firefox 3.5.4 Calculate Linux Scratch Gnome 10.0: X.Org 7.4, Kernel 2.6.31.6, Firefox 3.5.4, Gnome 2.26.3
