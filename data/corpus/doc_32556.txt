Релиз Univention Corporate Server 3.0

Доступен релиз Univention Corporate Server 3.0, основанного на Debian GNU/Linux 6 (Squeeze) промышленного дистрибутива, ориентированного на обеспечение поддержки корпоративной IT-инфраструктуры. Для установки задействован собственный инсталлятор, оптимизированный для быстрого развёртывания готовых сервисов. Для локального и централизованного администрирования используется специальный web-интерфейс. Для управления конфигурацией подготовлен инструментарий Univention Configuration Registry, позволяющий через наглядный интерфейс генерировать файлы конфигурации для разных сервисов и программ. В состав дистрибутива входит набор компонентов, позволяющих решать такие задачи, как ведение централизованного LDAP-каталога учетных записей пользователей, обеспечение единой системы аутентификации и авторизации, интеграция с MS Active Directory, автоматизация развёртывания серверов и рабочих станций, обеспечение работы тонких клиентов, мониторинг, создание серверов печати, резервное копирование, управление сетевыми службами (DHCP, DNS, прокси, почтовый сервер), обеспечение совместной работы сотрудников предприятия (groupware), управление работой виртуальных окружений на базе Xen и KVM. Ключевые новшества Univention Corporate Server 3.0: Осуществлён переход на пакетную базу Debian 6.0 (Squeeze) и обеспечена полная совместимость со стандартными репозиториями Debian (ранее использовались только свои репозитории); Интеграция тестовой версии Samba 4 и обеспечение работы в качестве контроллера домена для Windows-машин; Полная переработка управляющего web-интерфейса, в котором теперь активно используется Ajax. В новом интерфейсе объединены функции ранее разрозненных интерфейсов Univention Management Console и Univention Directory Manager; Добавлен модуль для упрощения создания дубликатов программной начинки физических или виртуальных серверов, позволяя при клонировании быстро откорректировать настройки, которые будут отличаться на новой системе; Расширена поддержка IPv6; Улучшена реализация средств для управления виртуальными окружениями на базе Xen и KVM.
