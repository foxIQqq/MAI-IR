Релиз Linux-дистрибутива openSUSE 11.3

Официально представлен релиз Linux дистрибутива openSUSE 11.3, доступного в сборках для 32- и 64-разрядных систем и отличающегося улучшенной поддержкой нетбуков, интеграцией Linux-ядра 2.6.34 и обеспечением поддержки файловой системы Btrfs в инсталляторе. В качестве десктоп-окружений пользователям предложены свежие версии KDE, GNOME, Xfce и LXDE. Следующий релиз openSUSE 11.4 запланирован на март 2011 года. Основные улучшения в openSUSE 11.3: Программы Пользователю: KDE 4.4.4, GNOME 2.30.1, Xfce 4.6.1, LXDE 0.5.5, OpenOffice.org 3.2.1, Firefox 3.6.6, X.Org 7.5, xorg-server 1.8.0, seamonkey 2.0.4, Amarok 2.3, k3b 2.0; Средства разработки: GCC 4.5, Perl 5.12, Mono 2.6.4, KDevelop 4.0, NetBeans 6.8, Qt-Creator 1.3.1, Monodevelop 2.2.2; Системные программы: rpm 4.8.0, glibc 2.11.1, zypper 1.4.1, parted 2.2 с поддержкой дисков с 4K-секторами; Системы виртуализации: Xen 4.0, KVM 0.12.3, VirtualBox-ose 3.2.6; В поставку включен Grub2, который не планируется использовать по умолчанию; В качестве программы для просмотра PDF в GNOME LiveCD использован epdfview; В качестве клиента для мгновенного обмена сообщениями в GNOME теперь по умолчанию используется Empathy; В систему добавлены разработанные в рамках проекта RusXMMS библиотеки librcd и librcc, обеспечивающие автоматическое распознавание русскоязычных кодировок по отрывку текста. В openSUSE библиотеки используется для корректной распаковки zip-архивов с не английскими именами файлов; В состав дистрибутива интегрирован визуальный анализатор дискового пространства xdiskusage; В состав дистрибутив включены пакеты с СУБД MariaDB и MySQL Cluster; Улучшения для пользователей На установочный DVD, наряду с GNOME, KDE и Xfce, интегрировано десктоп-окружение LXDE; Произведена интеграция в дистрибутив нового пользовательского интерфейса MeeGo для нетбуков, а также адаптированного для нетбуков окружения на базе KDE - Plasma Netbook; Существенно улучшена поддержка синхронизации, обмена данными и использования в роли модема смартфонов Android, iPhone и BlackBerry. Поддержка iPhone и iPod интегрирована в файловый менеджер nautilus и музыкальный проигрыватель Rhythmbox; Реализован механизм установки модулей и программ KDE в момент необходимости в них; Обновлен набор конфигураторов и плагинов для настройки сети, в частности NetworkManager 0.8 и ModemManager 0.3; Поддержка оборудования В систему интегрирован видеодрайвер nouveau. Для видеокарт Intel, ATI и Nvidia (драйвер nouveau) в систему интегрировать KMS-модуль для переключения видеорежимов на уровне ядра. Вместо драйвера xf86-video-radeonhd для видеокарт ATI/AMD теперь используется по умолчанию xf86-video-ati с поддержкой KMS/DRI. Предусмотрена возможность быстрого выключения KMS; Добавлен инструмент Imagewriter для копирования ISO-образов на USB Flash.; Добавлен GUI-конфигуратор для настройки тачпадов Synaptics; Установка и управление пакетами Поддержка в инсталляторе файловой системы Btrfs; Улучшение пакетного менеджера zypper: добавлена возможность прерывания работы PackageKit через DBus (блокировка zypper во время проведения обновлений и дополнительных фоновых установок сильно раздражала пользователей); улучшена обработка ошибок при чтении данных с разнообразных носителей; улучшена обработка зависимостей пакетов, если пакет удаляется при помощи опции --clean-deps можно удалить все связанные только с этим пакетов зависимости; улучшены средства для выбора пакетов для установки/обновления/удаления ( [теперь поддерживается синтаксис +/-]repo:type:nameOPversion.arch). Для организации прямой загрузки в команды "in", "up", "dup", "ve" и "patch" добавлена опция "download", поддерживающая режимы "only", "in-advance", "in-heaps" и "as-needed". Утилита для фоновой загрузки пакетов aria2 обновлена до версии 1.8.0, в которой реализовано большое число новых XML-RPC методов и добавлена возможность записи меда-данных в виде torrent-файлов. В LiveCD добавлены средства (пакет guest-additions) для акселерации работы под управлением системы виртуализации virtualbox; Системные улучшения Для резервного копирования, синхронизации данных между машинами и организации совместного доступа к файлам задействован сервис SpiderOak (cloud-хранилище размером 2 Гб предоставляется бесплатно); Добавление опциональной поддержки системы инициализации upstart, разработанной изначально для дистрибутива Ubuntu, но также используемую и в проекте Fedora. Upstart базируется на парадигме обработки событий, а для определения последовательности запуска сервисов и оценки возможности их выполнения в параллельном режиме, применяется метод учета зависимостей. Осуществлен переход на использование новой реализации системы Cron - cronie, разработанной в недрах проекта Fedora. Добавлены userspace-утилиты из комплекта conntrack-tools, позволяющие системному администратору взаимодействовать с модулем ядра conntrack, выполняющим отслеживание состояний соединений; Для индексации локальных документов используется поисковый механизм Tracker, вместо ранее используемого Beagle.
