Компания Farata Systems открыла исходные тексты фреймворка Clear Toolkit

Компания Farata Systems открыла исходные тексты фреймворка Clear Toolkit, предназначенного для быстрой разработки интерактивных приложений на Adobe Flex и Java. Пакет включает в себя библиотеку Flex компонентов и набор плагинов для среды Eclipse: Clear Data Builder - позволяет генерировать CRUD приложения на основе SQL выражений или объектов данных Java; DTO2Fx - утилита для генерации ActionScript классов на основе прототипов на Java; Log4Fx - плагин для автоматизации ведения логов, построенный поверх Flex API для ведения логов; Fx2Ant - генератор оптимизированных скртиптов сборки ANT для проектов, использующих Flex Builder.
