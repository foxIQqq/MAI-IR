Online хранилище iFolder доступно для пользователей OpenSUSE 11.1

Разработчики проекта openSUSE сообщили о добавлении в репозиторий обновлений openSUSE 11.1 пакетов с iFolder клиентом. iFolder представляет собой финансируемый компанией Novell свободный продукт, направленный на создание безопасного, простого в использовании и доступного из любой точки сети сетевого хранилища для синхронизации файлов между машинами и обмена данными между группами пользователей. После установки iFolder, сохраненные в определенные локальные директории данные будут автоматически загружены на сервер хранения и доставлены на другие машины пользователя. Клиентская часть iFolder доступна для Linux, Mac OS X и Windows, поддерживается управление сохраненными файлами через web-браузер. В отличие от online-хранилищ Dropbox, Mandriva Click’n Backup и Ubuntu One, серверная часть iFolder не привязана к инфраструктуре производителя и может быть установлена на стороне пользователя. Для упрощения развертывания серверной части iFolder подготовлен специальный образ виртуальной машины, который может быть запущен в окружениях vmWare или VirtualBox, кроме того пользователи openSUSE Build Service могут загрузить готовый пакет ifolder3-enterprise. Управление сервером производится через web-интерфейс, поддерживается задание индивидуальных дисковых квот, управление пользователями, масштабирование хранилища на несколько серверов, создание областей только для чтения, шифрование пользовательских файлов, имеется поддержка LDAP. Проект iFolder развивается в тесной интеграции с другим продуктом - Kablink Teaming, нацеленным на создание web-ориентированной платформы для организации взаимодействия и общения групп пользователей. Пакет включает в себя средства для совместного управления документами, версионного контроля изменений планирования, поведения экспертизы, ведения блогов, форумов, wiki, базы знаний и других элементов социальных сетей. Релиз Teaming 2.0.0 вышел всего несколько дней назад.
