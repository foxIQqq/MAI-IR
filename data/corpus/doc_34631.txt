Релиз системы виртуализации VirtualBox 4.1.20

Компания Oracle выпустила корректирующий релиз системы виртуализации VirtualBox 4.1.20, в котором отмечено 19 исправлений. В новой версии: Решена проблема с установкой в Gentoo Linux; Исправлены проблемы с работой мыши в гостевых окружениях с Fedora 17; Налажен процесс сборки набора дополнений для Linux (Linux Additions) в RHEL/CentOS 6.3 и при использовании ядра Linux 3.5-rc1; Устранён крах, проявляющийся для виртуальных машин запущенных без использования механизмов аппаратной виртуализации; Устранены крахи при восстановлении снапшотов, использовании NAT и VGA; В Virtio-net решена проблема с потерей пакетов при обмене данными между хост-системой и гостевым окружением с Windows XP; Для новых VM по умолчанию отключено использование буфера обмена; Проведена адаптация хост-системы для работы на Mac OS X Mountain Lion; Устранён ряд проблем при работе в режиме хоста на платформе (например, крах при освобождении памяти).
