Новый выпуск Raspbian, дистрибутива для Raspberry Pi

Разработчики проекта Raspberry Pi опубликовали обновление дистрибутива Raspbian. Для загрузки подготовлены две сборки - сокращённая (261 Мб) для серверных систем и полная (1.2 Гб), поставляемая с пользовательским окружением PIXEL (ответвление от LXDE). Для установки из репозиториев доступно около 35 тысяч пакетов. В новом выпуске добавлен мастер начальной настройки, автоматически запускаемый при первой загрузке дистрибутива и предлагающий базовый набор настроек. На первом экране предлагается выбрать язык, часовой пояс и страну. На остальных экранах начального конфигуратора предлагается сменить пароль доступа, активировать сетевое соединение и установить актуальные обновления. В дальнейшем для вызова мастера можно использовать команду "sudo piwiz". Добавлен интерфейс с обзором рекомендованных приложений, которые интересны только отдельным категориям пользователей или предлагаются бесплатно для пользователей Raspberry Pi по соглашению с производителями. Вместо Xpdf в состав включен новый просмотрщик документов qpdfView, предоставляющий более современный интерфейс, демонстрирующий более высокую производительность отрисовки и поддерживающий упреждающую загрузку и кэширование страниц. В новом выпуске также отмечается обновление версий программ (например, Chromium 65). Изменение громкости спецклавишами на клавиатуре теперь производится с шагом 5% вместо 10% и затрагивает текущее активное устройство вывода звука. Обеспечено отображение пароля при повторном подключении к беспроводной сети. Добавлена настройка для включения/выключения консоли для последовательного порта. В конфигураторе раскладки клавиатуры теперь применяются настройки не только для рабочего стола, но и для консоли.
