Увидел свет Linux-дистрибутив ROSA Desktop 2012

Компания Роса представила релиз дистрибутива ROSA Desktop.Fresh 2012, изначально развивавшегося под именем ROSA Desktop 2012, но в последний момент дополненного словом "Fresh", чтобы подчеркнуть ориентацию дистрибутива на людей, желающих получить подборку наиболее свежих версий приложений. Для свободной загрузки подготовлена DVD-сборка, размером 1.5 Гб, оформленная в вариантах для платформ i586 и x86_64. Дистрибутив позиционируется для энтузиастов и опытных пользователей и отличается предоставлением возможности использования свежих версий программ и широкими возможностями по настройке, в зависимости от личных предпочтений. Несмотря на использование свежих версий приложений и системных компонентов удалось добиться высокой стабильности работы всех аспектов работы дистрибутива, в основном благодаря переходу на новые методы разработки, сборки, тестирования и проверки качества. Релиз посвящён Евгению Додонову, трагически погибшему в результате дорожно-транспортного происшествия. Особенности дистрибутива: Новые версии программ: Системные компоненты: ядро Linux 3.6.10, Systemd 194, rpm 5.4.10, Grub2, Glibc 2.15, GCC 4.7.3 c патчами от проекта Linaro, Perl 5.14.2, PHP 5.4.9, Ruby 1.9.3, python 2.7.3, python3 3.2.3, NetworkManager 0.9.6.4, KnetworkManager 0.9.0.5, VPNPPTP 0.3.4; Графический стек: X-Server 13, Mesa 9.1, свежие версии драйверов для видеокарт AMD, NVidia и Intel; Десктоп-окружения: KDE 4.9.4, GNOME 3.6.2, Mate 1.4.0, с E17 0.17 gamma, Xfce 4.10; Пользовательские приложения: Firefox 17.0, Thunderbird 17.0, LibreOffice 3.6.2, Amarok 2.6.0; Поддержка загрузки на системах с EFI/UEFI; В инсталлятор добавлен режим "Install in basic mode" для гарантированного запуска установщика на большинстве оборудования; Для загрузки задействован GRUB2 с автоматическим определением разрешения, поддерживаемого системой. Обеспечено автоматическое обновление меню загрузчика GRUB2 при установке/удалении ядер. Добавлены новые темы оформления для GRUB2 и Plymouth; Произведена оптимизация сервисов и значительно сокращено (на 30-40%) время загрузки системы; Для графических эффектов по умолчанию используется OpenGL, что позволяет на большинстве машин задействовать дополнительные возможности композитного менеджера, такие как "размытость" полупрозрачного окна и куб рабочих столов. В случае если ускорение OpenGL не поддерживается адаптером, переходит автоматический откат на программную реализацию (XRender). Существенно обновлена тема оформления рабочего стола. Наглядно выделены области за которые можно передвигать окна приложений. В движок тем QgtkStyleAlt добавлена возможность перетаскивать окна KDE-приложений за меню и панель не нажимая дополнительных клавиш модификаторов; Расширен фирменный набор пиктограмм, добавлено более 30 новых иконок; Интерфейс для запуска приложений SimpleWelcome полностью переписан на QML, переработан дизайн, добавлены возможности группировки и перемещения иконок, организован вывод эскизов для недавних документов. Добавлена возможность закрепления последних приложений на вкладке "Приветствие", поддержка очистки списка последних документов, налажена работа с системным интерфейсом Drag and Drop; Интерфейс для отслеживания активности пользователя в системе TimeFrame полностью переписан на QML. Добавлена возможность просматривать локальные документы, изображения и видео на одной непрерывной горизонтальной ленте событий. Из TimeFrame теперь можно просматривать свою социальную ленту Facebook и Вконтакте, а также отправлять сообщения в социальные сети. Реализована возможность фильтровать файлы по типу контента (картинка, документ, видео) и события в социальной ленте по типу аккаунта; Проведена оптимизация внутренней структуры панели для быстрого запуска RocketBar, которая стала потреблять существенно меньше ресурсов, особенно при операциях изменения размера и перемещения. Графический интерфейс апплета Rosa SysTray переписан на QML, переработан дизайн всплывающего окна, теперь его элементы отображаются в виде "тучки"; Графический интерфейс KDM переписан на QML, реализован список пользователей в виде ленты (карусели); Апплет для организации быстрого доступа к наиболее используемым каталогам и файлам StackFolder переделан в полноценный вплывающий апплет, что позволило избавиться от проблем при взаимодействии с другими программами. Организовано более тесное взаимодействие с KLook, теперь работают встроенные элементы управления контентом (прокрутки, смена страниц, и т.д.); В интерфейсе для быстрого просмотра группы файлов KLook значительно улучшено качество рендеринга, добавлена возможность работать с удаленными файловыми системами по протоколам SMB, WebDav, FTP, и т.п.; Обновлено оформление медиапроигрывателя ROSA Media Player, унифицированы диалоговые окна и пиктограммы, добавлена поддержка midi-файлов и возможность работы с тегами в формате ID3v2.4.
