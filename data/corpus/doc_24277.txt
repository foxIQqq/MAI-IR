Планы подготовки релиза Apache Tomcat 7

Опубликовано интервью Марком Томасом (Mark Thomas), разработчиком Apache Tomcat, с раскрытием планов относительно релиза Tomcat 7 с реализацией полной поддержки спецификации Java Servlet 3.0. В частности, появится: поддержка стандартного API для асинхронной обработки запросов; расширенные средства для трекинга сессий; более широкие возможности по обнаружению ошибок в коде на этапе сборки; расширенные средства контроля доступа; механизмы для динамического управления конфигурацией. Первый альфа релиз Tomcat 7 намечен на декабрь 2009 г. или январь 2010 г. Также приводится интересная статистика, показывающая, что Tomcat используется на 75% сайтах, базирующихся на Java.
