Доступна для тестирования вторая бета-версия Ubuntu 12.04

Представлена финальная вторая бета-версия Ubuntu 12.04. Релиз намечен на 28 апреля. Релиз 12.04 будет относиться к категории выпусков с длительным сроком поддержки, при этом время поддержки десктоп-редакции для LTS-выпусков Ubuntu будет увеличено с трёх до пяти лет. Загрузочные iso-образы с тестовой версией Ubuntu 12.04 можно получить на данной странице. Одновременно выпущены тестовые версии смежных проектов: Kubuntu (KDE 4.8), Xubuntu (XFCE 4.8), Lubuntu (LXDE), Mythbuntu, Edubuntu (с подборкой обучающего ПО) и Ubuntu Studio (для обработки мультимедиа информации). По сравнению с первой бета-версией, в Ubuntu 12.04-beta2 отмечены следующие улучшения: Ядро Linux обновлено до версии 3.2.12 (3.2.0-20.33). Возвращена отключенная в прошлых тестовых выпусках поддержка AUFS, не смотря на это разработчики не отказываются от планов по прекращению поддержки AUFS в будущем, поэтому пользователям данной ФС рекомендуется перейти на использование похожей по возможностям OverlayFS. Обновлены патчи с поддержкой технологии мандатного контроля доступа AppArmor. Добавлена поддержка seccomp filter, интересного механизма ограничения доступа к системным вызовам для приложений. Особенностью seccomp является определение правил доступа к системным вызовам в BPF-представлении (Berkley Packet Filter), активно используемом для фильтрации сетевых пакетов. BPF позволяет реализовывать достаточно сложные правила доступа, например, учитывающие передаваемые и возвращаемые аргументы. В простейшем случае в код программы добавляется структура с перечнем допустимых системных вызовов (ALLOW_SYSCALL) и реакции в случае несовпадения (например, KILL_PROCESS). Для определения используемых системных вызовов следует использовать правило EXAMINE_SYSCALL. Программа сама определяет какие системные вызовы её необходимы, все остальные системные вызовы блокируются, что позволяет ограничить возможности атакующего в случае эксплуатации уязвимости в защищённом при помощи seccomp приложении. Кроме Ubuntu, система seccomp скоро будет интегрирована в Chrome OS; Расширены возможности облачного сервиса Ubuntu One. В дополнение к реализованной в прошлом тестовом выпуске поддержке добавления и удаления произвольных директорий для синхронизации, добавлена полнофункциональная поддержка прокси, удалена возможность синхронизации списка контактов, удалена кнопка синхронизации из Nautilus. Реализована новая панель управления Ubuntu One, которая поддерживает операции по установке, настройке, добавлению и удалению директорий для синхронизации; Улучшена работа добавленного в первой бета-версии режима быстрого ввода команд 'Head-Up Display' (HUD), при котором пользователю выводится поисковая строка, начало набора названия желаемых действия в которой приводит к автоматическому подбору и выводу наиболее вероятных названия программ из списка установленных приложений, а также элементов меню, представленных в штатном меню текущей активной программы или апплета. Добавлен вызов HUD при нажатии клавиши Alt и последующем начале набора на клавиатуре; LibreOffice обновлён до версии 3.5.1; В состав серверной редакции включен кандидат в релизы облачной платформы OpenStack Essex. В репозиторий Universe добавлены пакеты с Zentyal, добавлен OpenMPI 1.5 с поддержкой платформы ARM. По умолчанию включен acpid; Обеспечена поддержка системы виртуализации KVM 1.0 для архитектуры x86, в том числе с поддержкой создания вложенных виртуальных окружений; В Ubuntu Core подготовлен образ rootfs для архитектуры ARM hard float (armhf); Проект Kubuntu подготовил сборку "Kubuntu Active" с окружением Plasma Active, ориентированную на использование на планшетных ПК; В Ubuntu Studio по умолчанию задействовано ядро Linux с патчами для увеличения отзывчивости. Завершена миграция на окружение Xfce. Улучшена интеграция Pulse Audio и JACK; В состав Mythbuntu включена тестовая версия медиацентра MythTV 0.25; В Xubuntu для архитектуры i386 задействовано ядро Linux без поддержки PAE, с целью обеспечения работы на как можно большем числе машин; О других новшествах Ubuntu 12.04 можно прочитать в анонсе первой и второй альфа-версии, анонсе первой бета-версии, заметке про цели развития нового релиза и в сводной новости с обзором решений, принятых на саммите разработчиков Ubuntu.
