Выпуск Tizen IVI 3.0 для автомобильных информационно-развлекательных систем

Представлен релиз редакции мобильной платформы Tizen IVI 3.0, рассчитанной на использование на автомобильных информационно-развлекательных системах (IVI). Платформа Tizen развивается под покровительством организации Linux Foundation, при участии LiMo Foundation, Intel и Samsung. Проект продолжает развитие систем MeeGo и LiMO, но отличается предоставлением возможности использования Web API и web-технологий (HTML5/JavaScript/CSS) для создания мобильных приложений. Ветка Tizen 3.0 примечательна использованием нового композитного менеджера, базирующегося на Wayland/Weston 1.6, и Ozone-Wayland (вариант браузерного движка Chromium для Wayland), а также переходом по умолчанию на systemd. В новой версии также реализована поддержка многопользовательского режима, добавлены новые фреймворки для Bluetooth и WiFi, осуществлена интеграция Web runtime на базе Crosswalk (набор компонентов для создания универсальных мобильных приложений на основе технологий HTML5). Представлен новый фреймворк для разработки визуально насыщенных интерфейсов, отличающийся высокой производительностью и движком рендеринга, способным создавать реалистичную анимацию и манипулировать 2D- и 3D-объектами. Tizen IVI 3.0 полностью соответствует спецификации GENIVI 7.0, определяющей требования по обеспечению совместимости программных продуктов с аппаратными компонентами IVI-систем и возможности по взаимозаменяемости программных платформ. IVI-система может применяться для выполнения различного рода автомобильных приложений, используемых для оснащения автомобильных информационно развлекательных систем (IVI), интерактивных приборных панелей, встроенных информационных систем и других автомобильных устройств. Из поддерживаемых в платформе аппаратных устройств отмечены платы Nexcom VTC 1010-IVI и MinnowBoard Max. Для разработки специфичных для IVI-систем приложений предлагается использовать специальный Tizen IVI SDK, предоставляющий набор компонентов интерфейса пользователя и расширенный Web API для доступа к автомобильным подсистемам (Vehicle API), организации голосового управления (Speech API) и создания мультимедийных систем (MediaServer API). В состав также входит интегрированная среда разработки на базе Eclipse, сценарии сборки на реальных аппаратных устройствах, web-симулятор платформы и эмулятор аппаратного обеспечения. В качестве альтернативного решения для сборки решений на базе Tizen IVI обеспечена экспериментальная поддержка инструментария Yocto 1.7. Из поставляемых примеров приложений можно отметить домашний экран, интерфейс для управления установкой приложений, климат-контроль, мультимедийный проигрыватель с поддержкой DLNA, приборную панель, навигационную систему (используется Google Maps), интерфейс для настройки звуковой подсистемы, программу для чтения новостей. Платформой также предлагаются компоненты для голосового управления, поиска информации, взаимодействия со смартфоном по Bluetooth и подключения к CAN-сети для доступа к датчикам и передачи данных между узлами автомобиля.
