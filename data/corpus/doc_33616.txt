Представлена панель Cairo-Dock 3

Увидел свет релиз Cairo-Dock 3, визуально насыщенной панели для обеспечения запуска программ, использующей для организации вывода библиотеку Cairo. Готовые бинарные пакеты подготовлены для Ubuntu Linux, для других дистрибутивов пакеты будут подготовлены позднее. Cairo-Dock может работать на любом оборудовании, с любым оконным менеджером, интегрируясь с большинством десктоп-окружний и используя уже имеющиеся настройки. На современных системах используется OpenGL и современные визуальные эффекты, а на устаревших системах панель работает в упрощённом режиме, потребляя минимум ресурсов. Благодаря гибким средствам расширения функциональности Cairo-Dock не ограничивается запуском программ, позволяя устанавливать плагины с реализацией новых визуальных эффектов и апплеты с реализацией дополнительных действий, таких как отображение состояния почтового ящика, приём и отправка мгновенных сообщений, контроль за работой медиаплеера, слежение за RSS-лентами, просмотр состояния загрузки торрентов, отображение погоды, вывод заданий из календаря-планировщика и т.п. Особенности Cairo-Dock 3: Значительно улучшенная панель задач, поддерживающая группировку и изменение порядка отображения открытых окон на панели, быстрый запуск новых экземпляров приложений, режим переключения между окнами одного приложения колесом мыши и т.п.; Меню вызова приложений, содержащее ссылки на все доступные приложения, сгруппированные по тематическим категориям. Программы можно перетаскивать из меню на панель через интерфейс drag&drop Переработанный апплет для организации выхода из системы, отныне позволяющий переключаться между несколькими пользовательскими аккаунтами. Апплет также позволяет завершать выполнение, блокировать экран, переходить в спящий или ждущий режим; Расширены средства по управлению действиями через клавиатурные комбинации. Для настройки горячих клавиш подготовлен единый конфигуратор; Интеграция с апплетами-индикаторами Ubuntu; В апплет для управления звуком интегрирована поддержка звукового меню Ubuntu; Простой апплет для управления календарём-планировщиком; Апплет для управления доступными накопителями и быстрого перехода в избранные директории; Добавлен апплет для быстрой отправки сообщений в Twitter; Добавлен апплет для быстрого вызова хранителя экрана; Апплет для просмотра лога событий, например, последних открытых документов, просмотренных видеороликов, прослушанных музыкальных композиций или открытых web-страниц; Осуществлён переход на Gtk3+ для улучшения интеграции с рабочим столом GNOME; Расширена поддержка управления через DBus API; Новая визуальная тема; Мощный интерфейс для переключения между окнами и виртуальными рабочими столами, наглядно отображающий какие окна доступны на виртуальных рабочих столах; Множество других возможностей, включая автоматическое скрытие, анимированные 3D-эффекты, размещение в любой части экрана, просмотр истории работы с буфером обмена, стековое отображение файлов, апплет для мониторинга за работой системы и т.д.
