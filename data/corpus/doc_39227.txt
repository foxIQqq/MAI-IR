Доступен серверный Linux-дистрибутив SME Server 8.1

После двух лет разработки представлен релиз серверного Linux-дистрибутива для малого и среднего бизнеса - SME Server 8.1, построенного на основе пакетной базы CentOS 5.10 (на базе CentOS 6 будет выпущен релиз SME Server 9, который пока находится на стадии бета-тестирования). Особенностью дистрибутива является наличие в составе преднастроенных типовых компонентов, полностью готовых к работе и настраиваемых через web-интерфейс. Среди таких компонентов можно отметить почтовый сервер с фильтрацией спама, web-сервер, сервер печати, файловый архив, служба каталогов, межсетевой экран и т.п. Размер ISO-образа 736 Мб. В новой выпуске: Обновлён инструментарий для резервного копирования рабочих станций, в том числе задействован новый выпуск утилиты Dar 2.4.11, обеспечена возможность задания дней недели для выполнения полного бэкапа, добавлена поддержка задания уровня сжатия для режимов Desktop и Console, учтены многочисленные нюансы создания резервных копий рабочих станций. В реализации файлового сервера добавлена поддержка входа в домен Windows 8, внесены изменения для увеличения производительности, добавлены новые опции. В реализации почтового сервера до ClamAV 0.98 обновлены компоненты антивирусной защиты, улучшена поддержка SMTP Auth. В установщике по умолчанию теперь предлагается опция nodmraid, из-за многочисленных сообщениях о проблемах при использовании dmraid. Внесены незначительные улучшения в управляющий интерфейс, webmail, LDAP-сервер, web-сервер.
