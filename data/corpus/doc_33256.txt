Доступен дистрибутив Oracle Linux 5 Update 8

Компания Oracle выпустила восьмое обновление дистрибутива Oracle Linux, основанного на пакетной базе Red Hat Enterprise Linux 5.8. Дистрибутив доступен для бесплатной загрузки в сборках для архитектур x86 и x86_64 (размер iso-образа 3.1 Гб). По сравнению с RHEL 5.8 дистрибутив Oracle Linux включает дополнительные исправления ошибок и оптимизации, в основном нацеленные на улучшение работы продуктов Oracle и использование дистрибутива для работы гипервизора Xen. В частности, добавлена поддержка xen pv netconsole, максимальный размер памяти для паравиртуальных гостевых систем Xen расширен до 256 Гб, обновлено 14 драйверов для сетевых устройств и 12 драйверов для систем хранения. В базовую поставку входит три ядра Linux: стандартное ядро из состава RHEL 5, расширенное ядро RHEL 5 с исправлениями от Oracle и ядро Unbreakable Enterprise Kernel, развиваемое силами Oracle на основе Linux-ядра 2.6.32. По умолчанию предлагается ядро Unbreakable Enterprise Kernel. Готовые бинарные сборки ядра от Oracle для совместимых c RHEL 4 и 5 дистрибутивов и src.rpm-пакеты представлены на сайте public-yum.oracle.com, там же можно загрузить и другие присутствующие в дистрибутиве пакеты. Oracle не накладывает никаких ограничений на поставку подготовленного в Oracle ядра в сторонних продуктах.
