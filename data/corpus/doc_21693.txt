Вышла LiveCD сборка дистрибутива Zenwalk Linux 6.0

Вышла LiveCD сборка дистрибутива Zenwalk Linux 6.0, основанного на Slackware Linux и графическом окружении XFCE 4.6. В дистрибутиве используется Linux ядро 2.6.28.7 с дополнительными патчами для поддержки Aufs, Squashfs и LZMA. В Zenwalk Live 6.0 представлены следующие новшества: Переработано внешнее оформление; Утилиты LiloFix и LiveClone переписаны с использованием Python и графической библиотеки GTK+. В LiveClone возможность генерации LiveCD с начинкой по желанию пользователя, расширена поддержкой LiveUSB; Увеличена скорость загрузки, переписаны скрипты инициализации; Добавлена новая панель управления с GUI для управления дисками, беспроводной и проводной сетью; Для выполнения пользователем привилегированных действий добавлена поддержка Gksu и PAM аутентификации; В пакетном менеджере Netpkg появилась поддержка offline-операций и удаления пакетами с учетом более не используемых в системе зависимостей; Добавлен потоковый медиаплеер Exaile; Программа для просмотра фотографий Gthumb заменена на Gqview и Gtkam; По умолчанию установлены Lightning расширение для управления персональной информацией в Icedove (Thunderbird);
