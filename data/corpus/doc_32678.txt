Вышел медиа-центр Boxee 1.5, заключительный релиз для ПК

Представлен релиз кроссплатформенного медиа-центра Boxee 1.5 для платформ Linux, Mac OS X и Windows. К сожалению, это последняя версия Boxee для обычных ПК, начиная с января 2012 года разработка будет сосредоточена только на развитии прошивки для телеприставок, таких как D-Link Boxee Box. Пользователям настольных систем рекомендуется использовать медиа-центр XBMC, разработчики которого несколько дней назад начали тестирование первой бета-версии XBMC 11.0. В ближайшее время планируется выпустить обновление прошивки Boxee 1.5 для телеприставки Boxee Box, которая по заявлению разработчиков значительно превосходит по функциональности версию для настольных систем. Например, из-за ограничений, связанных с DRM и сертификацией, в доступной для свободной загрузки версии отсутствует поддержка расширенных компонентов, таких как модули для работы с сервисами Netflix, Vudu, и Pandora. Напомним, что в Boxee Box блокируется возможность модификации прошивки. Применение тивоизации разработчики объясняют дополнительными требованиями к обеспечению защиты платного контента. Код Boxee основан на открытом проекте XBMC, расширенном в направлении создания дополнительных проприетарных компонентов, обеспечения возможности интеграции с социальными сетями и online-сервисами, расширении средств для упрощения интеграции и разработки внешних дополнений. Программное обеспечение поддерживает все популярные мультимедиа форматы и сервисы потокового видео и аудио вещания. Например в Boxee интегрирована поддержка таких ресурсов, как Netflix, MLB.TV, Comedy Central, Pandora, Last.fm, YouTube, Hulu и Flickr. Кроме того в платформу интегрированы средства для работы с социальными сетями, такими как Twitter или Facebook. Для загрузки файлов из сети в медиа центр интегрирован BitTorrent-клиент. Разработка платформы ведётся на языке C++ с использованием Python плагинов. Часть исходных текстов Boxee распространяется под лицензией GPL, но некоторые новые подсистемы, например, библиотека libboxee, доступны только в бинарном виде. Из новшеств Boxee 1.5 можно отметить: Улучшенный домашний экран, на который добавлены ссылки для запуска браузера и использования Live TV (требуется подключения специального USB-адаптера); Переработано меню, которое теперь всплывает над остальным контентом и упрощает поиск и навигацию между секциями; В интерфейсе для управления просмотром фильмов и телешоу упрощена навигация по жанру и источнику, обеспечен быстрый доступ к просмотру трейлеров; Добавлен новый экран для просмотра детальной информации о фильме, такой как более подробное описание, список ролей и данные о создателях фильма; В режиме просмотра телешоу упрощён переход между сезонами сериалов и пометка уже просмотренных серий; В интерфейсе для отложенного просмотра расширены эскизы секций и добавлена возможность отсеивания клипов, телешоу и фильмов. Из новинок, представленных в XBMC 11.0-beta1, можно отметить: Возможность отката на прошлую версию дополнения после установки более новой версии дополнения (Addon Rollbacks); Улучшение используемой по умолчанию визуальной темы Confluence и общая доработка интерфейса; Увеличение производительности благодаря изменениям в коде рендеринга и интеграции нового декодировщика JPEG; Улучшение поддержки сети в обеспечение поддержки дополнительных протоколов; Улучшение работы с незашифрованным BluRay-контентом; Поддержка AirPlay; Обновлённый апплет для отображения погоды; Расширение поддержки оборудования, в том числе Apple TV2, iPad, и других устройств на базе iOS; Расширение JSON-RPC с целью упрощения организации удалённого управления; Поддержка задействования средств для аппаратного ускорения декодирования видео с использованием VAAPI на видеокарта от компании AMD; Значительно переработан развиваемый в рамках проекта Live-дистрибутив на базе XBMC и Ubuntu;
