Выпуск дистрибутива для создания сетевых хранилищ Rockstor 3.9

Состоялся релиз специализированного Linux-дистрибутива Rockstor 3.9, предназначенного для развёртывания сетевых (NAS) и облачных хранилищ. Дистрибутив построен на пакетной базе CentOS и использует для локального хранения файловую систему Btrfs, а для внешнего доступа протоколы Samba/CIFS, NFS, SFTP и AFP. Размер установочного iso-образа 824 Мб. Особенностью Rockstor является поддержка запуска дополнительных приложений-обработчиков, упакованных в контейнеры Docker. Например, для быстрой установки доступны docker-образы с ownCloud, Syncthing, OpenVPN и Plex. Управление работой дистрибутива осуществляется через специально развиваемый проектом web-интерфейс, возможна интеграция с другими системами через REST API. Из расширенных функций упоминается поддержка работы со снапшотами, хранение данных в сжатом виде, простое расширение хранилища (достаточно добавить диски), организация резервного копирования клиентских систем через Rsync, Apple Time Machine и Windows Backup, поддержка переноса данных на запасной сервер с использованием асинхронной репликации, возможность контроля целостности по цифровым подписям, встроенная система мониторинга ресурсов. Наработки дистрибутива распространяются под лицензией GPLv2, а сборки релизов распространяются без ограничений, но доступ к стабильному репозиторию с обновлениями пакетов, осуществляется только по платной подписке. При этом бесплатно доступен репозиторий с тестовыми обновлениями, в котором проводится первичная проверка обновлений перед их переносом в стабильный репозиторий.
