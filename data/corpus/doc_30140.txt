Canonical закрывает сервис бесплатной рассылки дисков с Ubuntu

Компания Canonical объявила о закрытии сервиса ShipIt, в рамках которого любой желающий мог заказать бесплатно доставку компакт диска со свежей версией Ubuntu. Доставка осуществлялась по всему миру, что позволяло использовать данный сервис пользователям развивающихся стран, не имеющим возможность загрузить диск по сети. Возможность платного заказа дисков c Ubuntu 11.04 через интернет-магазин Canonical будет сохранена, бесплатно заказать диск с Ubuntu 11.04 станет невозможно. Вместо прямой рассылки дисков Canonical станет более активно сотрудничать с локальными группами пользователей Ubuntu (группы LoCo), которым будут отправляться большие партии дисков для распространения среди пользователей в регионе. Кроме того, Canonical начнет продвигать программу по распространению дисков среди пользователей, стимулирующую самостоятельное тиражирование и распространение между собой записанных собственными силами копий Ubuntu. Кроме того, планируется создать новый online-сервис, который позволит начать знакомство с дистрибутивом без его установки, используя web-браузер. При использовании нового сервиса приложения будут выполняться в специальной cloud-инфраструктуре, а содержимое экрана будет транслироваться на удаленную машину пользователя. Ожидается, что сервис позволит в первую очередь заинтересовать Windows-пользователей, дав им возможность собственноручно поработать в Ubuntu, не требуя затрачивать каких-либо усилий, кроме открытия страницы в web-браузере. Экспериментальный сервис WebLive уже доступен для ознакомления с работой Edubuntu.
