Вышел PelicanHPC 4.1, Linux-дистрибутив для развертывания кластеров

Представлен релиз Linux-дистрибутива для быстрого развёртывания кластеров - PelicanHPC 4.1. Дистрибутив построен на пакетной базе Debian Jessie и позволяет превратить обычный офисный компьютерный парк в вычислительный кластер, загрузив один управляющий компьютер с LiveCD/LiveUSB, а остальные машины использовав в качестве загружаемых через PXE узлов. В качестве рабочего стола предлагается Xfce или GNOME. Размер установочных iso-образов 1.6 Гб для Xfce и 1.9 Гб для GNOME. Дистрибутив доступен только в 64-разрядной версии. В состав входят такие кластерные приложения, как пакет Open MPI, MPI-биндинги для GNU Octave, интерфейс мониторинга работы кластера Ganglia, набор для тестирования производительности кластеров Linpack. Дистрибутив поддерживает средства централизованного управления конфигурацией загружаемых по сети узлов, что позволяет использовать дистрибутив для обеспечения функционирования постоянно работающих кластеров, а не только для проведения одноразовых экспериментов. В новой версии произведена синхронизация пакетной базы с Debian 8.6 (Jessie), для сборки задействован live-build 4.x. Кроме варианта с Xfce добавлена сборка с GNOME.
