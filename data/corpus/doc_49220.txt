Релиз браузерного движка WebKitGTK+ 2.22.0

Состоялся выпуск новой стабильной ветки WebKitGTK+ 2.22.0, порта браузерного движка WebKit для платформы GTK+. WebKitGTK+ позволяет использовать все возможности WebKit через GNOME-ориентированный программный интерфейс на основе GObject и может применяться для интеграции средств обработки web-контента в любые приложения, от использования в специализированных HTML/CSS-парсерах, до создания полнофункциональных web-браузеров. Из известных проектов, использующих WebKitGTK+, можно отметить Midori и штатный браузер GNOME (Epiphany). Ключевые изменения: Новый JavaScriptCore GLib API; В мультимедийный бэкенд GStreamer добавлена поддержка API playbin3; В API WebDriver добавлены команды для расширенного взаимодействия с пользователем; Предлагаемая по умолчанию реализация меню переведена на использование GtkTreeView.
