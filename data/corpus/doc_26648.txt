Консорциум OSADL, развивающий realtime Linux, присоединился к организации Linux Foundation

Консорциум OSADL (Open Source Automation Development Lab), развивающий решения на базе Linux и открытого ПО для промышленной встраиваемой техники, объявил о вступлении в ряды членов некоммерческой организации Linux Foundation. Из известных проектов OSADL можно отметить Safety Critical Linux, real-time Ethernet, Industrial I/O Framework, разработку драйверов для встраиваемых устройств, а также флагманский проект Realtime Linux и "Realtime-Preempt" патчи для Linux ядра (-rt ветка), которые используется в real-time редакциях таких промышленных Linux дистрибутивов, как MontaVista, Red Hat и Novell. В составе Linux Foundation консорциум OSADL намерен усилить взаимодействие с разработчиками ядра по интеграции в основное ядро новых разработок, связанных с работой в реальном режиме времени, обеспечением отказоустойчивых технологий и решений для промышленных встраиваемых систем. Linux Foundation занимается координацией широкого спектра работ, связанных с развитием Linux, защищает интересы Linux и решает юридические проблемы, развивает систему стандартов и обеспечивает финансовую поддержку некоторых разработчиков Linux ядра. Из недавно присоединившихся к Linux Foundation компаний можно отметить: ServInt, Planet, LG, Collabora, Meyer Sound, CUBRID, Parallels, Ricoh, Linpus Technologies.
