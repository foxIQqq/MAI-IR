Microsoft уверила пользователей в безопасности использования Mono на мобильных платформах

Пользуясь случаем, связанным с судебным разбирательством между Oracle и Google в отношении Java-технологий, компания Microsoft поспешила уверить пользователей в том, что компания не намерена выдвигать претензии против проекта Mono и основанного на его базе продукта MonoDroid c реализацией технологий C# и .Net для платформы Android. Суть претензий Oracle сводится к тому, что реализация Java от Google для платформы Android не соблюдает все требования спецификации Java и существенно отличается в принципах организации выполнения байткода. Mono для Android также не является полностью совместимым с эталонными спецификациями .Net и предоставляет разработчикам измененный API, требующий переработки программ для организации интерфейса и задействования мобильных функций.
