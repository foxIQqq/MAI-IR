Выпуск браузера Pale Moon 27.0

После восьми месяцев разработки состоялся релиз независимого браузерного проекта Pale Moon 27.0, построенного на ответвлении от кодовой базы Firefox, модифицированного для обеспечения более высокой эффективности работы, cохранения классического интерфейса, минимизации потребления памяти и предоставления дополнительных возможностей по настройке. На уровне поддержки базовых web-технологий и тем оформления браузер совместим с Firefox. Сборки Pale Moon формируются для Windows и Linux (x86 и x86_64). После выхода Pale Moon 27.0 поддержка прошлой ветки 26.x прекращена. Проект придерживается классической организации интерфейса, без перехода к интегрированному в Firefox 29 интерфейсу Australis. Из удалённых компонентов можно отметить DRM, Social API, WebRTC, PDF-просмотрщик, Сrash Reporter, код для сбора статистики, интерфейс группировки вкладок Panorama, средства для родительского контроля и людей с ограниченными возможностями. Из улучшений выделяется поддержка инструкций SSE2, оптимизации для процессоров Atom, дополнительные варианты восстановления сеанса после краха и возможность отключить загрузку изображений. Новые возможности: Развиваемый проектом браузерный движок Goanna обновлён до версии 3.0, в которую перенесены изменения из свежих выпусков движка Gecko. Необходимость поддержания собственного движка обусловлена требованиями торговой марки Mozilla, запрещающей распространение модифицированного варианта движка с сохранением названия Gecko; Обновлён JavaScript-движок, в который перенесена поддержка многих возможностей ECMAScript6, в том числе механизма Promise и генераторов, что позволило улучшить совместимость с современными web-приложениями, использующими новые возможности JavaScript для формирования интерфейса; Проведена работа по увеличению эффективности кэширования контента; Полная поддержка протокола HTTP/2; Поддержка «Агат»-аннотации (Ruby-аннотаций) при отображении символов (уточняющие символы, выводимые над иероглифами); Поддержка API Media Source Extensions, позволяющего формировать мультимедийные потоки из JavaScript-приложений (используется во многих online-видеопроигрывателях). Включение Media Source Extensions производится в настройках, но разработчики пока не рекомендует включать данный API для WebM, так как ещё не решена проблема с потерей звука в YouTube после смены позиции в потоке; Поддержка загрузки и воспроизведения фрагментированных файлов MP4, решающая проблемы с работой некоторых online-плееров; Поддержка использования SSL/TLS при соединении с прокси-серверами; Поддержка загрузки шрифтов в формате WOFF2; На платформе Windows обеспечена поддержка DirectX 11 и Direct2d 1.1; Из Firefox портированы все исправления уязвимостей, включая те, что исправлены в Firefox 50. Обновлены версии библиотек, поставляемых в комплекте. Удалённые возможности: Прекращена поддержка Windows XP; Удалён встроенный просмотрщик PDF; Прекращено включение в сборку средств для web-разработчиков. Для пользователей которым необходима данная функциональность реализована возможность установки инструментов в форме дополнения; Удалён активный фильтр XSS, так как он требует переработки для сохранения совместимости с новой платформой Pale Moon; Оставлен только англоязычный интерфейс, пакеты локализации пока не готовы, но будут опубликованы позднее. Из-за существенных изменений в новой версии, использовать старые языковые наборы не получится; Удалена поддержка расширений Add-on SDK (расширений JetPack), что привело к разрыву совместимости с дополнениями на базе Mozilla/Gecko SDK. Таким образом выпуск Pale Moon 27.0 теперь несовместим с большим числом дополнений к FireFox. В полном объёме продолжена поддержка плагинов NPAPI, XUL-расширений с бинарными компонентами и XPCOM, тем оформления, неподписанных и подписанных дополнений, шифра Camellia, отрисовки шрифтов через Graphite, API Sync 1.1 и средств для полной настройки интерфейса пользователя.
