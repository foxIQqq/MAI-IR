Компания Adobe выпустила свободный моноширинный шрифт Source Code Pro

Следом за публикацией свободного семейства шрифтов Source Sans Pro, компания Adobe представила Source Code Pro - новый свободный шрифт с фиксированной шириной (моноширинный), предназначенный для использования в таких программах как эмуляторы терминалов и редакторы кода. Шрифт представлен в шести градациях толщины от экстра тонких до жирных. К сожалению, отсутствует изначальная поддержка кириллицы, как и в первом релизе Source Sans Pro. Шрифт распространяется в форматах OpenType и TrueType под свободной лицензией Open Font License 1.1 (OFL 1.1), допускающей модификацию, создание производных шрифтов и неограниченное распространение. Шрифт можно загрузить как в форме готового архива, так и через web-сервисы Adobe Edge Web Fonts, Typekit, WebINK и Google Web Fonts. Дополнительно на GitHub опубликован набор вспомогательных исходных материалов, макетов и файлов сборки для Adobe Font Development Kit, предназначенных для упрощения доработки и расширения шрифта. Одновременно объявлено о введении в строй Adobe Edge Web Fonts - нового бесплатного сервиса для распространения шрифтов для web-приложений. Для использования на своём сайте шрифта из коллекции Adobe (по договорённости с Google доступны также шрифты с сервиса Google Web Fonts) достаточно подключить соответствующий шрифту JavaScript-код (например, "http://use.edgefonts.net/league-gothic.js") и активировать использование выбранного семейства шрифтов через CSS (например, "font-family: league-gothic, serif;").
