Новая версия менеджера фотографий Shotwell 0.14

Увидел свет релиз программы для управления коллекцией фотографий Shotwell 0.14.0. Shotwell предоставляет удобные возможности каталогизации и навигации по коллекции, поддерживает группировку по времени и тегам, предоставляет инструменты для импорта и конвертации новых фотографий, поддерживает выполнение типовых операций по обработке изображений (вращение, устранение эффекта красных глаз, корректировка экспозиции, оптимизация цветности и т.п.), содержит средства для публикации в социальных сетях, таких как Facebook, Flickr и Picasa. Готовые бинарные сборки доступны для Linux-дистрибутивов Ubuntu и Fedora. Основные улучшения Shotwell 0.14: Существенно улучшенный и переработанный режим слайдшоу. Добавлены новые эффекты для отображения перехода между изображениями в режиме слайдшоу. Например, доступно замещение одного изображения другим в виде надвигающихся полос, квадратов или шахматной доски; Расширенная поддержка RAW-изображений. При импорте с цифровых камер налажена парная обработка RAW-изображений и связанных с ними JPEG-файлов, добавлена поддержка отдельного экспорта JPEG-файлов из связки RAW+JPEG; Переход на использование мультимедийного фреймворка GStreamer 1.0 для обработки видео; Режим древовидного отображения директорий в боковой панели; При сбое в процессе импорта файлов теперь выводится более понятный отчёт о сути проблемы и возможна запись информации о скопированных файлах в отдельный лог; В модуле интеграции с Facebook добавлена поддержка Graph API и возможность отображения папок с изображениями в боковой панели; Исправлено 176 ошибок.
