LiveCD, генератор PHP кода и ERP на базе PostgreSQL

Представлено несколько новых продуктов, базирующихся на использовании СУБД PostgreSQL: PostgreSQL PHP Generator 9.12 - бесплатно распространяемый GUI-интерфейс для генерации PHP скриптов для выполнения типовых операций с таблицами, представлениями и запросами данных в web-приложениях, выполняющих функции системы управления контентом. Например, при помощи системы можно быстро подготовить форму для добавления, редактирования и удаления записей определенной структуры, создать необходимые отчеты, организовать загрузку файлов. Пример создаваемых скриптов можно посмотреть в online-демонстрации; PostERP 2.6 - система планирования ресурсов предприятия (ERP) тесно интегрированная с PostgreSQL. К сожалению клиентский GUI интерфейс PostERP в настоящий момент доступен только для Windows, но официально поддерживается запуск системы под управлением Wine. PostgreSQL Live CD for 8.4.2 - специализированного LiveCD на базе Fedora 12 c набором инструментов для проведения экспериментов c PostgreSQL 8.4.2 и тестирования программ на совместимость. В комплект входят pgAdmin III, Apache, PHP, GNOME, Pidgin, Firefox и т.д.
