Вышли MySQL Community Server 5.0.82 и MySQL SDBC драйвер для OpenOffice.org

Вышел релиз MySQL Community Server 5.0.82, включающий исправления 20 ошибок, накопившихся за последние 3 недели. Например, устранен крах при использовании оператора HANDLER совместно с INFORMATION_SCHEMA. Кроме того, компания Sun Microsystems представила первую бета-версию MySQL SDBC драйвера, известного также под именами "Sun Connector for MySQL" и "MySQL Native Driver". Драйвер позволяет осуществить подключение OpenOffice 3.1 к СУБД MySQL версии 5.1.x. В отличие от аналогов, драйвер не требует установки Java и дополнительных ODBC/JDBC драйверов, максимально прост в установке, позволяет одновременно работать с несколькими MySQL базами, поддерживает платформы Windows, Linux, Solaris и Mac OS X.
