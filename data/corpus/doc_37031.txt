Предварительный выпуск браузера Opera Next 15 вышел без поддержки Linux

Представлен Opera Next 15, первый предварительный выпуск нового браузера Opera для настольных систем, перешедшего с собственного движка Presto на движки, развиваемые в рамках проекта Chromium (браузерный движок Blink (форк WebKit) и JavaScript-движок V8). К сожалению тестирование нового браузера пока доступно только для пользователей Windows и Mac OS X. Версия для Linux пока не готова, но по заверению разработчиков, компания Opera Software заинтересована в поддержке платформы Linux и работает над созданием соответствующей редакции. Новой версии чтобы избежать путаницы присвоен номер 15, так как выпуск построен на более новой кодовой базе Chromium, чем уже поставляемый Opera 14 для платформы Android. Кроме миграции на новый движок и изменений интерфейса, новый выпуск примечателен исключением из состава браузера встроенного почтового клиента, который будет поставляться в виде отдельного обособленного приложения - Opera Mail. Тестовая версия Opera Mail также доступна только для Windows и Mac OS X, но в будущем выйдет и в версии для Linux. Некоторые особенности Opera Next 15: Переработанная экспресс-панель, которая тесно интегрирована с системой ведения закладок и комбинирует при выводе как ссылки для быстрого запуска, так и разделы системы закладок. Поддерживаются средства группировки, переименования и перемещения содержимого экспресс-панели, а также объединение ссылок по разделам; Адресная строка комбинирует в себе поле для ввода адресов и форму для отправки поисковых запросов; Новая панель Discover, предназначенная для обнаружения нового контента в Сети и слежения за новостными лентами по интересам. При этом браузер сам формирует подборку источников в зависимости от выбранной тематики и производит фильтрацию популярного и востребованного контента; Функция Stash для удобного сравнения нескольких страниц (например, для сравнения цен на разных сайтах); Для ускорения работы на низкоскоростных каналах связи или при необходимости экономить трафик, предусмотрен режим Off-Road, при котором активируется аналог ранее используемой технологии Opera Turbo, переведённой на использование протокола SPDY. При включении режима Off-Road обработка контента производится на стороне серверной инфраструктуры Opera Software с передачей данных клиенту в компактном виде, что позволяет снизить трафик и обеспечить комфортную работу при роуминге или при работе в 2G-сетях;
