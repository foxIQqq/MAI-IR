Доступно десктоп-окружение LXQt 0.10

После девяти месяцев разработки состоялся релиз пользовательского окружения LXQt 0.10 (Qt Lightweight Desktop Environment), развиваемого объединённой командой разработчиков проектов LXDE, Razor-qt и Maui/Hawaii. Интерфейс LXQt продолжает следовать идеям классической организации рабочего стола, привнося современное оформление и приёмы, увеличивающие удобство работы. LXQt позиционируется как легковесное, модульное, быстрое и удобное продолжение развития рабочих столов Razor-qt и LXDE, вобравшее лучшие черты обеих оболочек. Код размещён на GitHub и поставляется под лицензиями GPL 2.0+ и LGPL 2.1+. Появление готовых сборок ожидается для Ubuntu, Arch Linux, Siduction, Fedora, openSUSE, Mageia, FreeBSD и ALT Linux. Основные новшества: Задействованы два штатных фреймворка KDE: Solid - заменил собой liblxqt-mount и код управления питанием; libkscreen - пришёл на смену xrandr и обеспечил совместимость с Wayland; Новый модуль lxqt-sudo, предназначенный для организации выполнения команд с повышенными привилегиями. Для запуска команд представлены утилиты lxsu и lxsudo; В панель добавлена поддержка автоматического скрытия, перетаскивания мышью эскизов открытых окон и группировки эскизов окон, связанных с одним приложением; Для панели представлен плагин с реализацией совместимой с Wayland спецификации Status Notifier Item, определяющей способ размещения элементов в системном лотке; В интерфейсе запуска Runner обеспечено скрытие дубликатов для бинарных файлов и приложений; Реализован механизм отслеживания udev для обновления настроек клавиатуры в случае подключения к системе новых клавиатур; В модуль настройки добавлена поддержка смены набора папок по умолчанию (Desktop, Downloads, Music); В файловом менеджере реализованы функции поиска, улучшена поддержка корзины и расширены средства навигации мышью; Полностью переделан интерфейс настройки параметров экрана, который теперь является надстройкой над libkscreen; Плагин монтирования разделов и модуль управления питанием переведены на фреймворк Solid; Повышены требования к версии сборочного инструментария - для сборки LXQt теперь требуется CMake 3.0.2+; Все упоминания "LxQt" заменены на "LXQt".
