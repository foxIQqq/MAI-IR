Релиз youtube-dl 2021.12.17

Спустя шесть месяцев после предыдущего выпуска опубликован выпуск утилиты youtube-dl 2021.12.17, предоставляющей интерфейс командной строки для загрузки звука и видео из YouTube и многих других сайтов и online-сервисов, включая VK, YandexVideo, RUTV, Rutube, PeerTube, Vimeo, Instagram, Twitter и Steam. Код проекта написан на языке Python и распространяется в форме общественного достояния. До этого активности в проекте не было с 1 июля. Сразу после релиза разработчик youtube-dl Сергей М. перенёс своё имя в список неактивных разработчиков. Из изменений можно отметить: Обновлены шаблоны для извлечения сигнатур со страниц YouTube, что позволило решить проблемы, возникшие после изменения JavaScript-кода YouTube. Повышена надёжность обработки вызовов get_video_info processing и добавлен обход запросов get_video_info, что решило проблемы с загрузкой видео с выставленными возрастными ограничениями. Обновлён список альтернативных серверов для доступа к YouTube, использующих web-фронтэнд Invidious. При ошибках обеспечен вывод отладочных данных, возвращаемых пакетом FFmpeg. Улучшена поддержка извлечения контента из серверов PeerTube. Для сайта Pornhub обеспечена возможность извлечения видео с сервера pornhubthbh7ap3u.onion в анонимной сети Tor, реализовано определение ограничений доступа в зависимости от географического местоположения пользователя, решены проблемы с пропусканием видео наивысшего качества. Решены проблемы при извлечении контента из сервисов appleconnect, periscope, bilibili, umg.de, egghead, tvthek и nrk. Прекращена поддержка сервиса liveleak.
