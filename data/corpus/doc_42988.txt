Доступен Turnkey Linux 14.0, набор мини-дистрибутивов для быстрого развертывания приложений

Спустя почти два года с момента прошлого выпуска доступен релиз набора Turnkey Linux 14.0, в рамках которого сформирована коллекция из 105 минималистичных сборок Debian, пригодных для использования в системах виртуализации и облачных окружениях. Средний размер каждой сборки - 250 Мб. Сразу после установки пользователь имеет возможность получить полностью работоспособные из коробки рабочие окружения с LAMP (Linux, Apache, MySQL, PHP/Python/Perl), Ruby on Rails, Joomla, MediaWiki, Wordpress, Drupal, Apache Tomcat, LAPP, Django, MySQL, PostgreSQL, Node.js, Jenkins, Typo3, Plone, SugarCRM, punBB, OS Commerce, ownCloud, MongoDB, OpenLDAP, gitlab, CouchDB и т.д. Управление программным обеспечением производится через специально подготовленный web-интерфейс. Сборки снабжены системой автоматического резервного копирования и средством для автоматической установки обновлений. Новая версия сформирована только для архитектуры x86_64 и основана на пакетной базе Debian GNU/Linux 8.0. В качестве web-интерфейса для настройки задействован Adminer, который заменил собой PHPMyAdmin и PHPPgAdmin. Ужесточены требования к параметрам TLS в наборах для развёртывания http-серверов на базе Apache, LigHTTPd и Nginx. Конфигураторы Webmin и Webshell теперь доступны только через HTTPS (выполнен проброс через stunnel). В системе автоматической установки обновлений реализована отправка уведомлений администратору по электронной почте, с указанием информации об установленных обновлениях. В состав образов включена простая система мониторинга на базе Monit и настроена отправка уведомлений о превышении потребления памяти, CPU или недостаточном дисковом пространстве. По сравнению с прошлым выпуском в коллекцию добавлены сборки с системой управления конфигурацией Ansible, легковесной системой автоматизации взаимоотношений с клиентами EspoCRM, платформы для создания сервиса совместных закупок (Food Coop) Foodsoft, PHP-фреймворком Laravel, SuiteCRM, форком SugarCRM, новыми ветками систем управления контентом Drupal8 (beta) и Joomla3.
