Релиз Linux дистрибутива ZevenOS Neptune 1.9 с интерфейсом в стиле BeOS

Вышел релиз Linux дистрибутива ZevenOS 1.9, построенного на базе Debian Squeeze и снабженного не требовательным к ресурсам BeOS-подобным интерфейсом пользователя. В качестве десктоп-окружения используется стилизованный под BeOS релиз KDE 4.4.5, для маломощных систем предусмотрена возможность использования связки fluxbox, lxpanel и rox-filer. Для настройки приложений в рамках проекта создано ПО MAGi 2, в качестве панели для запуска программ задействован Zeebar, меню реализовано при помощи Deskbar 2. При подготовке новой версии основное внимание было уделено обеспечению работы с USB-накопителей в Live-режиме, с сохранением между сессиями всех изменений и пользовательских данных. В новой версии осуществлен переход на Linux-ядро 2.6.35.5, в качестве web-браузер используется Chromium (в поставку входит Adobe Flash), для работы с фотографиями задействован Gwenview, для работы с мультимедиа в поставку входит Amarok 2.3.1. Размер загрузочного iso-образа составляет 1.4 Гб.
