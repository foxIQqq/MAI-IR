Компания Microsoft сделала шаг навстречу проекту Eclipse

Компания Microsoft объявила о запуске нескольких проектов, направленных на улучшение взаимодействия между технологиями Windows 7 и Windows Azure с открытыми платформами Eclipse и Java. Microsoft намерена разработать серию дополнений к интегрированной среде разработки Eclipse, оптимизирующих выполнение IDE в Windows 7 и сборки приложений для Windows 7 с задействованием новых элементов интерфейса. Созданные улучшения планируется опубликовать под лицензией Eclipse Public License в первом квартале следующего года. Кроме того, анонсирован новый открытый плагин для Eclipse - Windows Azure Tools, позволяющий организовать разработку web-приложений на языке PHP для cloud-окружений Windows Azure.
