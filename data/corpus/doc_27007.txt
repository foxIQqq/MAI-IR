Новая версия MonoDevelop 2.4, среды для разработки .NET приложений в Linux

Доступен релиз MonoDevelop 2.4, интегрированной среды для разработки разработки Mono и .Net приложений на платформах Linux, MacOS X и Windows. Из поддерживаемых в MonoDevelop технологий создания приложений отмечены Gtk#, ASP.NET MVC, Silverlight, MonoMac и MonoTouch. Основные улучшения новой версии (скриншот-обзор): Проведена работа по улучшению пользовательского интерфейса: изменено цветовое оформление, добавлены тени, тулбары интегрированы в док, подготовлено 5 уровней изменения компактности GUI, добавлены новые диалоги; Добавлена поддержка автоскрытия таких блоков, как список ошибок, лог работы в системе контроля версий, результаты поиска. Расширены возможности редактора исходных текстов, в интерактивной справке появилась поддержка руководств WCF (Windows Communication Foundation). Обновлен порт для платформы Mac OS X; Реализована поддержка обработки T4-макросов. В будущем релизе обещают добавить поддержку Git через задействование GitSharp, обеспечить возможность написания Mono-программ для платформы Android, реализовать поддержку MVC2 и шаблонов.
