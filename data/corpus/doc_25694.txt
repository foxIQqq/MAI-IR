Компания Microsoft выпустила новый пакет для обеспечения работы Linux в Hyper-V

Компания Microsoft выпустила новую версию набора Linux Integration Components, в который включен набор паравиртуальных драйверов для ускорения работы Linux в гостевых окружениях в системе виртуализации Windows Server 2008 Hyper-V. Помимо ранее публиковавшихся готовых драйверов для SUSE Linux Enterprise Server, в новой версии представлены пакеты и для Red Hat Enterprise Linux 5.2, 5.3 и 5.4. Драйверы основаны на коде, открытом летом прошлого года под лицензией GPL. Пока драйверы не включены в состав RHEL на пользователей действуют правила ограниченной поддержки Red Hat, при использовании в системе модулей ядра от сторонних производителей. Из других ограничений отмечена возможность работы в гостевом окружении только с одним виртуальным CPU.
