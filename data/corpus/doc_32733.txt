Компания Microsoft обеспечит возможность запуска Linux в Windows Azure

Компания Microsoft планирует весной обеспечить возможность выполнения виртуальных машин с Linux в облачной платформе Windows Azure. Изначально платформа Azure развивается в соответствии с концепцией PaaS (Platform as a service), т.е. ориентирована на выполнение специально подготовленных приложений. Но по многочисленным пожеланиям пользователей в скором времени будет обеспечена поддержка выполнения обособленных виртуальных машин с Linux, состояние выполнения которых будет сохраняться между запусками (persistent VM).
