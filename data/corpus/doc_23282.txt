Вышел релиз web-браузера Arora 0.9, базирующегося на движке WebKit и библиотеке Qt

Представлена новая версия web-браузера Arora (0.9.0), в которой исправлена 21 ошибка, внесено 12 улучшений и добавлено несколько значительных новшеств: Поддержка задания ключевых слов для поиска прямо в адресной строке и сопоставления ключей для обращения к заданным поисковым системам; Поддержка отправки POST-запросов для поисковых плагинов OpenSearch; Использования HTTP pipelining для всех сетевых запросов (только для Qt 4.6); Прекращена поддержка Qt 4.4, отныне для работы браузера требуется Qt версии 4.5 и выше. Основные возможности: Многоплатформенность, браузер доступен для Linux, FreeBSD, Windows, Mac OS X, различных мобильных ОС. Несмотря на использование Qt, прекрасно вписывается в интерфейс GNOME и XFCE; Браузер упрощенный, содержит только функции первой необходимости. Изначально развивался как демонстрация использования API QtWebKit; Поддерживается: интерфейс на базе табов; режим приватной навигации без оставления следов; менеджер Cookies; возможность хранения закладок и просмотра истории посещений; менеджер загрузок; поисковые плагины OpenSearch; Netscape-совместимые плагины, например, можно подключить Adobe Flash; встроенный набор инструментов для web-разработчиков (WebInspector); возможность управления сессиями; функция ClickToFlash, для проигрывания Flash-роликов только по запросу пользователя (нужно кликнуть на картинке); В будущих версиях планируется реализовать автозаполнение форм, интегрировать AdBlock-подобный механизм для блокирования рекламы, добиться включения в состав Kubuntu 9.10, улучшить интеграцию с KDE, добавить поддержку расширений, парсер RSS и модуль проверки орфографии в формах.
