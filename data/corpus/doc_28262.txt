Linspotting - подборка для организации процесса редактирования видео

В рамках проекта Linspotting подготовлен RPM-репозиторий с подборкой программ для организации рабочего окружения для редактирования видео. В поставку включен мультитрековый редактор нелинейного видеомонтажа Kdenlive, подборка мультимедиа кодеков (mp3, mpeg2, mpeg4, DnxHD, Theora), программы для преобразования видео для web-сервисов (Vimeo, Youtube), наборы видеоэффектов, программы для оформления субтитров, инструменты для захвата видео, извлечения содержимого DVD, перекодировния H264 и т.д. Из использованных открытых проектов в состав Linspotting входят kdenlive, xine, kaffeine, mplayer, vlc, ffmpeg, mlt. Все приложения оформлены в виде единого законченного решения, а не просто набора разрозненных программ. Одна из задач проекта - создание готового открытого продукта для редактирования видео, своеобразного джентльменского набора, который можно было бы использовать для проведения сертификации. Процесс редактирования видео с использованием Linspotting наглядно описан в подробном руководстве пользователя.
