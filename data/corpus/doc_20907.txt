Представлена первая рабочая реализация X Input 2

"First XI2 server implementation" - представлена первая рабочая реализация X Input 2, переработанной подсистемы ввода для X сервера. Внедрение X Input 2 даст возможность задействовать в X-сервере такую функцию, как Multi-Pointer X, что позволит организовать работу нескольких независимых устройств ввода, например, несколько управляемых разными мышами курсоров на экране или ввод в разные окна с разных клавиатур. Ранее технологии Multi-Pointer X и X Input 2 планировали включить в состав X Server 1.6.0, но из-за их неготовности было решено отложить их до следующего релиза X Server 1.7.
