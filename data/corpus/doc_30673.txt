Вышел дистрибутив Linux Mint 11

Представлен релиз дистрибутива Linux Mint 11, основанного на кодовой базе Ubuntu 11.04, но дополненного рядом оригинальных приложений, упрощающих настройку и работу в системе начинающих пользователей. Например, в Linux Mint представлена переработанная система меню, собственный менеджер установки и обновления приложений, оригинальный интерфейс для настройки системы mintConfig, "мастер" для настройки различных параметров системы для начинающих пользователей в стиле вопрос-ответ, интерфейс для настройки параметров десктопа, средство для выполнения резервного копирования mintBackup и т.п. Одним из ключевых отличий нового выпуска, является использование классического GNOME-десктопа, вместо графического окружения Unity, на которое перешел дистрибутив Ubuntu, и оболочки GNOME-Shell, используемой в GNOME 3 (в Linux Mint 11 задействован GNOME 2.32). В качестве Linux-ядра используется версия 2.6.38. Для загрузки доступны LiveCD (665 Мб), LiveDVD (867 Мб) и сборка для OEM-производителей (664 Мб). Из новшеств, не свойственных Ubuntu 11.04 и Linux Mint 10, можно отметить: В состав LiveCD не включены мультимедиа кодеки и проприетарные программы, такие как Adobe Flash и Java, тем не менее установка дополнительных несвободных компонентов в Linux Mint 11 значительно упрощена: на страницу приглашения входа в систему добавлены две ссылки: "Upgrade to the DVD Edition" и "Install Multimedia Codecs", позволяющие двумя кликами довести состав установленных пакетов до состояния LiveDVD и установить мультимедиа кодеки. Дополнительно, данные функции можно найти в стандартном меню: "Menu/Administration/Upgrade to the DVD Edition" и "Menu/Sound and Video/Install Multimedia Codecs". Из дополнительных компонентов, присутствующих на LiveDVD можно отметить: VLC, Gimp, Giver, Tomboy, LibreOffice, Java, Samba, дополнительные шрифты, фоновые рисунки, визуальные темы и наборы пиктограмм; Проведена работа по улучшению интерфейса менеджера установки приложений. На основном экране увеличен размер пиктограмм, отождествленных с категориями приложений. Добавлена новая категория с шрифтами; Переработано оформление и содержимое экрана с информацией о приложении. До установки приложений пользователь имеет возможность оценить какие зависимые пакеты дополнительно будут установлены, сколько место будет израсходовано на диске с учетом зависимостей, сколько данных потребуется загрузить по сети; Пиктограммы приложений в списке формируются не только на основе содержимого пакета mintinstall-icon, но и на основе текущей визуальной темы, что позволяет расширить число доступных пиктограмм и отображать их даже когда пакет mintinstall-icons не установлен; По умолчанию при выполнении операции поиска пакетов, учитываются не только названия программ, но и сопутствующее краткое описание. Вывод производится с учетом релевантности. Изменить данное поведение можно в настройках ("Edit/Preferences/Search in packages summary"); Менеджер установки обновлений Проведена работа по увеличению скорости проверки наличия обновлений, за счет встраивания в Update Manager набора правил с определением уровня безопасности для каждого пакета, который больше нет необходимости загружать из вне. Теперь при выходе новой версии набора правил, одновременно выпускается новая версия менеджера обновлений, который обновляется вне очереди штатным способом. Одним из главных новшеств менеджера обновлений является поддержка обработки зависимостей - в списке теперь отображаются только базовые обновления, без упоминания связанных с ними зависимостей. Например, при выполнении значительного обновления пакета, требующего установки или удаления других пакетов, информация об этих дополнительных пакетах теперь выводится на всплывающем диалоге перед началом обновления; Переработка интерфейса пользователя: все диалоги теперь модальные; Update Manager молча скрывает себя после успешного выполнения обновления; вкладки с предупреждениями и дополнительной информацией более не показываются для пакетов установка которых прошла гладко и для пакетов с которыми не связаны дополнительные данные; загрузка списка изменений производится в фоне; Инструмент настройки параметров рабочего стола, изначально разработанный для GNOME, теперь не зависит от типа графической оболочки, автоматически определяет её и выводит только связанный с ней набор настроек. В будущем, данный инструмент будет интергрирован в сборки Linux Mint на базе KDE, Xfce, LXDE и Fluxbox; Добавлены новые элементы художественного оформления (основная визуальная тема и набор пиктограмм оставлены прежними) и новые фоновые рисунки; По умолчанию активированы всплывающие полосы прокрутки, которые появляются только при наведении на них мыши; Внесены улучшения в визуальную тему Mint-X, например, улучшена интеграция с Deluge, Synaptic, Gimp и Banshee; Firefox 4, Chromium и Opera поставляются с дополнением mint-search-addon, упрощающим процесс поиска информации на популярных ресурсах, таких как Wikipedia, Youtube, eBay, Amazon, IMDB, flickr, Yahoo, Answers.com, Google; В пакетный менеджер APT добавлена поддержка команды "apt download" , которая выполняет функцию загрузки указанного пакета и всех связанных с ним зависимостей; Для 32-разрядных систем Adobe Flash теперь ставится в двух вариантах: стабильная версия Flash 10.2 и бета-версия Flash 10.3, для переключения между ними можно использовать команду "sudo update-alternatives --config libflashplayer.so"; Изменен список поставляемых по умолчанию программ. Из базового комплекта исключен Gwibber, вместо F-Spot для просмотра фотографий используется gThumb, вместо музыкального плеера Rhythmbox поставляется Banshee, вместо офисного пакета OpenOffice.org используется LibreOffice. Из поставки удалены padevchooser, paman, paprefs, pavumeter и pavucontrol.
