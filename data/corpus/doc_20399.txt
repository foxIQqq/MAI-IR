Компания ALT Linux выпустила новый серверный дистрибутив для школ

Набор продуктов, разработанных компанией ALT Linux в рамках школьного проекта, пополнился новым дистрибутивом, выпущенным под названием "Школьный Сервер" 4.1 и основанным на кодовой базе серверного дистрибутива для малого и среднего бизнеса - ALT Linux Office Server, адаптированного для образовательных учреждений. Установочный образ имеет размер 670 Мб и распространяется свободно. Как и в Office Server, в дистрибутиве "Школьный Сервер" управление и настройка всех компонент осуществляется через web-интерфейс. Хранение учетных записей для все базовых сервисов организовано через LDAP. Особенности дистрибутива: Предустановленные web-приложения для организации учебного процесса: база знаний на основе Mediawiki 1.13.0 и система дистанционного и интерактивного обучения с примерами курсов на основе ПО Moodle 1.9.2 ; Серверные приложения (настройка производится через web-интерфейс): почтовый сервер с поддержкой средств борьбы с вирусами и спамом (Postfix 2.4.9, SpamAssassin 3.2.4); web-сервер (Apache 2.2.8); прокси-сервер ( Squid 2.6.STABLE18 + средства для отображения статистики работы); DHCP-сервер; DNS-сервер (Bind 9.3.5); сервер точного времени (OpenNTPd 3.9p1); интерфейс для настройки межсетевого экрана; Samba-сервер ( Samba 3.0.30); FTP-сервер (ProFTPd 1.3.0); СУБД MySQL 5.0.51; сервер печати (CUPS 1.3.9). Средства обслуживания сервера: система резервного копирования на базе Synbak 1.2.2; управление учётными записями (с возможностью импорта учётных записей из системы 1С:Хронограф); интерфейс для обновления системы;
