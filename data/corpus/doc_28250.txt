Релиз DVD-сборки Ubuntu - Super OS 10.10

Для загрузки доступен релиз Linux-дистрибутива Super OS 10.10, который представляет собой DVD-сборку Ubuntu 10.10, построенную по принципу "все в комплекте". Например, в состав включены дополнительные мультимедиа приложения (VLC), полезные открытые программы (KDE-библиотеки, Ubuntu Tweak, GParted, cd2usb), кодеки (MP3, QuickTime, Real video, WMV, DivX, FLV, Xvid) и популярные проприетарные программы (Google Chrome, Opera, Skype, Adobe Flash, aMSN, Java). В комплект Super OS также включены дополнительные средства для упрощения установки и запуска программ: RUNZ/SuperDeb (самодостаточные пакеты, в которые интегрированы несколько deb-пакетов, для установки отдельных программ со всеми зависимостями) и App Runner (апплет для универсального запуска программ с привязкой к расширению). Размер установочного iso-образа - 1.1 Гб. В новой версии упрощен процесс установки проприетарных драйверов (Nvidia, ATI, Broadcom), которые теперь доступны в комплекте и не требуют загрузки дополнительных пакетов из сети. Вместо cd2usb для создания Live-установок на USB-накопителях задействована система usb-creator. Вместо OpenJDK в комплект интегрирована оригинальная реализация языка Java SE от компании Oracle (некоторые финансовые сервисы оказались неработоспособными при использовании OpenJDK).
