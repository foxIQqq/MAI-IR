Релиз "Пятой платформы" ALT Linux

Компания "Альт Линукс" представила финальную версию набора Linux дистрибутивов, развиваемых под кодовым именем «Пятая платформа». В рамках данной платформы на основе одной пакетной базы выпущены продукты Альт Линукс 5.0 Ковчег, для создания комплексных интегрированных решений, и Альт Линукс 5.0 Школьный, для использования в общеобразовательных учреджениях. Комплект «Альт Линукс 5.0 Ковчег» представлен двумя сборками: Сервер (ALT Linux 5.0 Ark Server). Особенности: Использование Linux ядра 2.6.27, GCC 4.4, GLIBC 2.10.1; Интеграция систем виртуализации KVM и OpenVZ; Web-интерфейс для настройки основных компонентов и подсистем; В комплект включены средства для организация сети предприятия, обеспечения централизованной авторизация пользователей и сервисов, организации взаимодействия территориально распределённых офисов (VPN), централизованного обновления серверов и рабочих станций, установки рабочих станций по сети, создания единого файлового хранилища c поддержкой квот. Рабочая станция (ALT Linux 5.0 Ark Desktop). Особенности: Использование Linux ядра 2.6.30, GNOME 2.26.3, Xorg 1.6.3, OpenOffice.org 3.1.1, GIMP 2.6.7, Scribus 1.3.5.1, Inkscape 0.47; Поддержка технологии Zeroconf; Упрощённое взаимодействие с ALT Linux сервером и виртуальными контейнерами Готовность к работе с терминальными приложениями; Самодостаточный комплект ПО для офисной работы; Централизованная аутентификация на сервере. Интеграция с файловым хранилищем на Сервере. В свою очередь набор дистрибутивов «Альт Линукс 5.0 Школьный» включает: Альт Линукс 5.0 Школьный Сервер (от ALT Linux 5.0 Ark Server отличается учетом специфики работы образовательных учреждений); Альт Линукс 5.0 Школьный Лёгкий (десктоп-окружение на базе XFCE 4.6.1) Альт Линукс 5.0 Школьный Юниор (десктоп-окружение на базе GNOME 2.26.3) Альт Линукс 5.0 Школьный Мастер (десктоп-окружение на базе KDE 4.3.2)
