Концептуальная реализация вируса на языке Python для десктоп среды Linux

"How to write a Linux virus in 5 easy steps" - концептуальная реализация вируса на языке Python для десктоп среды Linux. Например, чтобы позволить пользователю запустить файл, сохраненный из приложения к письму, не имея прав для запуска, для KDE или GNOME создается файл с описанием параметров запуска (launcher), имитирующий odt документ. Для активации после перезагрузки используются стандартные средства автозапуска. Дополнение: вышло продолжение статьи. Во второй части резюмированы некоторые идеи, прозвучавшие в комментариях и отзывах.
