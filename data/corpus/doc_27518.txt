В рамках проекта Ubuntu подготовлен новый индикатор для управления звуком в GNOME

Марк Шаттлворт представил в своем блоге новый универсальный индикатор для панели GNOME, предоставляющий единый механизм для управления звуком. В текущем состоянии апплет позволяет изменять уровень громкости и одновременно из панели управлять проигрыванием музыкальных композиций в плеере Rhythmbox. Новый индикатор будет интегрирован в состав осеннего релиза Ubuntu 10.10. Кроме Rhythmbox разработчики обещают добавить поддержку и других плееров. Апплет предоставляет другим мультимедиа приложениям dbus API для интеграции собственных блоков управления в рамках единого звукового индикатора.
