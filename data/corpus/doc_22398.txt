Вышел релиз Linux дистрибутива Yellow Dog Linux 6.2

Вышел релиз оптимизированного для процессоров Cell и Power Linux дистрибутива Yellow Dog Linux 6.2, поддерживающего установку на приставки Sony PlayStation 3, рабочие станции Apple G4/G5, YDL PowerStation и IBM System P. Yellow Dog Linux базируется на CentOS с некоторыми компонентами из Fedora Linux. Основное пользовательское окружение дистрибутива построено на основе Enlightenment 17. Наиболее интересные улучшения Yellow Dog Linux 6.2: В стандартном репозитории пакетов обновлены версии более 600 программ, среди которых Linux ядро 2.6.29 для архитектуры ppc64 (для ppc32 используется 2.6.27), OpenOffice.org 3.0, Firefox 3.0.x и IBM Cell SDK 3.1.0.1; Дополнительный репозиторий пакетов EPEL Extra (Extra Packages for Enterprise Linux), включает в себя более 2500 пакетов доступных для установки при помощи пакетного менеджера yum; Поддержка оболочки XFCE 4 в качестве альтернативного десктоп-окружения; Добавлена поддержка установки и загрузки с использованием USB Flash; В состав новой версии включена свободная библиотека libfreevec, заменяющая некоторые стандартные функции системной библиоткеи GLIBC, такие как memcpy(), strlen() и т.п. Функции заменяются на переработанные и оптимизированные аналоги, использующие возможности векторного движка AltiVec процессоров G4/G4+ PowerPC, что дает примерно 25% выигрыш в производительности; Для проигрывания Adobe Flash роликов используется Gnash v0.8.4; Значительное ускорение работы технологии ps3vram (на 50% быстрее), используемой для размещения временного хранилища файлов в видеопамяти на приставке PS3. Начиная с версии 6.2 ps3vram используется по умолчанию для хранения раздела подкачки; Для настройки беспроводного доступа используется GUI-интерфейс wicd. В настоящее время дистрибутив могут загрузить только коммерческие подписчики сервиса YDL.net, для всех остальных пользователей свободная загрузка будет открыта в конце июля.
