В проприетарном драйвере NVIDIA 325.08 прекращена поддержка ядер Linux 2.4

Компания NVIDIA начала бета-тестирование новой ветки своих проприетарных драйверов NVIDIA 325.08. Драйвер доступен для Linux, FreeBSD и Solaris. Для пользователей Linux новая версия примечательна прекращением поддержки ядер Linux серии 2.4, для работы драйвера теперь требуется ядро Linux 2.6.9 или более новая версия. В прошлых LTS-сериях 319.x, 173.14.xx и 304.xx поддержка ядер 2.4.x будет сохранена. Из других изменений, не связанных с исправлением ошибок, можно отметить добавление в nvidia-settings настройки поворота экрана для каждого дисплея в отдельности. В разделе "X Server Display Configuration" конфигуратора nvidia-settings добавлена поддержка настройки режимов SLI Mosaic и Base Mosaic. Увеличена производительность сложных градиентов.
