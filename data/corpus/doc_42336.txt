Релиз системы для математических расчётов GNU Octave 4.0.0

Выпущена новая значительная ветка системы для выполнения математических расчётов GNU Octave 4.0.0, предоставляющей интерпретируемый язык, во многом совместимый с Matlab. GNU Octave может использоваться для решения линейных задач, нелинейных и дифференциальных уравнений, вычислений с использованием комплексных чисел и матриц, визуализации данных, проведения математических экспериментов. Ключевым новшеством GNU Octave 4.0.0 является задействование по умолчанию появившегося в прошлой ветке нового графического интерфейса пользователя, построенного с использованием библиотеки Qt и использующего для отрисовки OpenGL (при недоступности OpenGL и Qt возможна отрисовка при помощи gnuplot). Для запуска предлагаемого ранее интерфейса командной строки и визуализации графиков через FLTK, следует использовать опции "--no-gui" и "--no-gui-libs". Отрисовка через Qt и FLTK теперь может выполняться в режиме offscreen, при котором вывод может осуществляться при отключении визуализации графика. Вторым важным новшеством является поддержка синтаксиса classdef для объектно-ориентированного программирования. Кроме того, в новом выпуске добавлена порция новых функций, в том числе добавлена большая порция функций для воспроизведения, записи и обработки звука. Улучшена совместимость с Matlab и добавлен опциональный режим для более полной совместимости с Matlab, при котором отключаются некоторые специфичные для Octave оптимизации.
