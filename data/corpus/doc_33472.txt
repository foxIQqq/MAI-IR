Релиз GNOME 3.4. Обзор новшеств

После шести месяцев разработки официально представлен релиз десктоп-окружения GNOME 3.4. В новой версии полностью переработан интерфейс web-браузера Epiphany, добавлено приложение для управления виртуальными машинами и организации удалённого доступа Boxes, интегрированы функции поиска документов, улучшена поддержка графических планшетов, реализовано глобальное меню для приложений, обеспечена поддержка плавной прокрутки, расширены возможности программ для работы с документами и адресной книгой, доработаны многие подсистемы, устранены известные ошибки. В подготовке новой версии приняло участие 1275 разработчиков, по сравнению с прошлым выпуском внесено 41000 изменений. К сожалению, разработчики не успели подготовить демонстрационный Live-дистрибутив для быстрого ознакомления с новой версией GNOME, iso-образы пока недоступны, но ожидаются в ближайшее время. Ключевые улучшения: Поддержка Application Menus - глобального меню с управлением через D-Bus (GApplication). Реструктуризация меню некоторых приложений и обеспечение возможности переноса меню текущего активного приложения в верхнюю панель GNOME. Новый подход к размещению меню подразумевает выделение большего места для контента приложения за счёт ухода от использования классической строки меню. Меню может формироваться динамически, в зависимости от текущего состояния программы (активных режимов работы) и типа контента. В настоящее время перенос меню в верхнюю панель выполнен для единичных приложений, таких как Boxes, Contacts, Documents, Iagno, Mines, Tetravex, Web (Epiphany). В скором времени ожидается реализация нового меню для Totem и Empathy; Новый пользовательский интерфейс web-браузера Epiphany, нацеленный на предоставление максимального экранного пространства для контента. Из особенностей нового интерфейса можно отметить переработанную адресную строку, вынос в глобальное меню не связанных с текущим контентом операций, реализация выпадающего меню в адресной строке; Расширены возможности и полностью переработан интерфейс Documents, приложения для упрощения работы с документами, организации поиска документов и обеспечения их просмотра. Переработано оформление, убрана боковая панель, расширены возможности поиска, реализован режим выбора группы документов. Добавлена поддержка организации коллекций документов и вывода документов на печать; В обзорный режим Gnome Shell (Activities Overview) добавлен модуль для поиска документов в дополнение к ранее реализованным функциям поиска приложений, контактов и настроек. Поиск документов осуществляется непосредственно с использованием приложения Documents, т.е. имеется возможность поиска не только среди коллекции локальных документов, но и документов, размещённых в online-сервисах, таких как Google Docs. В следующей версии ожидается поддержка поиска музыки и видео; В состав включено приложение Boxes, которое сочетает средства для удаленного соединения с рабочими столами с механизмами управления и запуска виртуальных окружений на локальных и внешних компьютерах. Для конечного пользователя Boxes позволяет имея на руках установочный или загрузочный образ системы, с минимальными усилиями подготовить рабочую виртуальную машину, не задумываясь о настройке и используемом гипервизоре (по сути нужно лишь выбрать iso-образ, все остальное будет сконфигурировано автоматически). Находясь дома или в дороге, пользователь также может воспользоваться Boxes для соединения с рабочей машиной, без выполнения сложных настроек на обеих сторонах.Для организации эффективного доступа к графическим приложениям на внешней виртуальной машине поддерживается протокол SPICE. Для запуска локальных виртуальных окружений используется KVM и Qemu; Переработан интерфейс приложения для управления адресной книгой. Добавлена возможность быстрого просмотра детальной информации для заданного адресата, создана новая форма загрузки аватара, реализована возможность редактирования данных на странице без перехода к форме редактирования, обеспечено автоматическое определение связей между записями в адресной книге (например, email и IM-контакт одного пользователя); В клиенте для мгновенного обмена сообщениями Empathy реализован новый интерфейс для голосового общения и видеочата. Добавлена поддержка протоколов Windows Live Messaging и чата Facebook. Переработан диалог настройки параметров аккаунта. Улучшена интеграция с адресной книгой GNOME; Поддержка плавной прокрутки. Ранее прокрутка осуществлялась немного резко и скачками, отныне содержимое сдвигается гладко и изящно. Операция прокрутки теперь позволяет остановить смещение контента на нужной позиции, делая операцию более осязаемой и ощущаемой физически. Оптимизировано оформление полос прокрутки, убраны "стрелочки" по краям для пошагового перемещения, полосы стали более тонкими и занимающими меньше места. Плавный скроллинг является первым шагом на пути к реализации новой системы ввода для сенсорных экранов; Обновлены компоненты интерфейса. Переработан диалог выбора цвета, в котором значительно упрощёны как операции выбора из готовой палитры, так и определения параметров своих цветов. При раскрытии окон для некоторых приложений теперь автоматически скрывается верхняя полоса с заголовком, что позволяет увеличить доступное вертикальное пространство экрана; Поддержка динамических фоновых рисунков, например, по умолчанию используется фоновое изображение, которое становится днём ярче, а вечером затеняется, а ночью становится тёмным; Улучшение в конфигураторе. Переработано оформление панели управления режимами энергосбережения. Полностью изменено оформление панели настройки параметров сети, портирована поддержка настройки VPN, реализованы диалоги задания пароля. Расширены средства для настройки графических планшетов Wacom - добавлены функции калибровки планшета, настройки одновременно нескольких планшетов, привязки кнопок. Улучшена панель центра управления для управления аккаунтами пользователей - добавлена поддержка идентификации через Kerberos. Добавлен новый диалог для определения параметров масштабирования в Gnome Shell. Улучшена поддержка оборудования. Расширены средства калибровки цветов, например, добавлена возможность сохранения и привязки к устройству цветовых профилей. Улучшена поддержка док-станций и внешних мониторов (например, если к ноутбуку подключен внешний монитор, то он не перейдёт в спящий режим после закрытия крышки). Добавлена поддержка кнопок управления громкостью для USB-гарнитур и колонок. Поддержка средств для настройки Multiseat, когда несколько человек могут работать за одним компьютером, к которому подключается несколько мониторов, клавиатур и мышей; Значительно переработана утилита для управления дисками, которая теперь поставляется под именем Диски (Disks); Программа для создания скринкастов позволяет создавать более плавные видеоролики, потребляя при этом меньше ресурсов. Для начала записи достаточно нажать Control+Shift+Alt+R; Улучшение системы вывода уведомлений, которые теперь избегают всплывания при близком нахождении курсора. При неактивности в течение определённого времени всплывает суммарная информация о непрочитанных уведомлениях, что позволяет не пропустить новые сообщения; В файловый менеджер Nautilus добавлена поддержка отката изменений (Undo); В Cheese, приложении для работы с web-камерой, по умолчанию теперь используется формат WebM вместо Theora; В программе для просмотра изображений добавлена боковая панель для быстрого просмотра метаданных и свойств изображений; В Evolution добавлена возможность соединения с серверами организации совместной работы Kolab, в том числе с использованием нескольких аккаунтов одновременно. Обеспечена полная поддержка работы в offline-режиме и реализована система выявления и разрешения конфликтов при синхронизации. Упрощён процесс настройки аккаунтов для типовых email-сервисов.
