Обновление системы автоматизации для малого бизнеса NORD POS 3.0.3CE

Вышел очередной релиз кросс-платформенного приложения NORD POS, являющегося форком проекта Openbravo POS, разработка которого была заморожена в 2010 году. В обновление вошли следующие изменения: Улучшена панель управления складом; Добавлена возможность назначать для пользователя склад по-умолчанию; Обновлены интерфейсы для создания и подключения сторонних библиотек этикет принтеров и устройств ввода-вывода(терминалы сбора данных, весы с печатью этикеток и т.п.) Новый отчёт для списка товаров; Другие мелкие исправления в интерфейсе и шаблонах. Стоит также отметить, что ключевыми отличиями NORD POS от Openbravo POS являются: Применения системы процентных и суммарных скидок при продажах; Генерация штрих-кодов EAN-13 для товаров; Создание QR-кодов для отчётов и чеков; Механизм синхронизации с внешними система через схемы Pentaho Data Integration; Обновлённая система подключения оборудования через Java сервисы; Встроенные серверы Apache Derby 10.11 для баз данных и Jetty 9 для веб-приложений; Перевод исходного кода на Java 7; Полное обновление всех библиотек до актуальных версий. Также, в отличие от других форков Openbravo POS, в NORD POS реализован механизм обновления базы данных последней версии Openbravo POS 2.30.2 (в том числе и русской версии). Система NORD POS, хоть и является полноценным инструментом автоматизации кассового и складского учёта, развивается разработчиком в первую очередь как платформы для мобильных веб-приложений nordpos.mobi. В рамках платформы продолжается разработка мобильных веб-приложений для ведения списка товаров и обслуживания в кафе, к которым одновременно с выходом NORD POS 3.0.3CE был добавлен Java сервлет для создания интернет магазина интегрированного с настольной версий приложения.
