Microsoft и Red Hat объявили о партнёрстве в области облачных систем и платформы .NET

Компании Microsoft и Red Hat заявили о налаживании партнёрских отношений в области облачных технологий и продвижения платформы .NET. В рамках партнёрства будет обеспечена полноценная поддержка Red Hat Enterprise Linux, JBoss и других продуктов Red Hat для клиентов Microsoft Azure, а также подготовлены инструменты для обеспечения запуска приложений .NET в окружениях OpenShift и Red Hat Enterprise Linux. В рамках соглашения дистрибутив Red Hat Enterprise Linux станет первичной платформой для разработки и эталонной операционной системой для внедрения фреймворка .NET Core в Linux. Microsoft Azure получит статус сертифицированного для продуктов Red Hat облачного окружения (Red Hat Certified Cloud), а образы виртуальных машин с продуктами Red Hat можно будет загрузить через каталог Azure Marketplace. До сих пор, в числе официально поддерживаемых в Microsoft Azure Linux-дистрибутивов упоминались только Ubuntu, CentOS, Oracle Linux, SUSE Linux Enterprise и openSUSE. Кроме того, компании займутся развитием средств для построения гибридных облачных систем, включающих решения на базе продуктов Red Hat и Microsoft, и предоставят средства для унифицированного управления подобными смешанными системами. В том числе, в Red Hat CloudForms появятся инструменты для управления системами в Microsoft Azure, а в Microsoft System Center Virtual Machine Manager будут добавлены возможности по управлению работой Red Hat Enterprise Linux в окружениях Microsoft Azure и Hyper-V. Для поддержки клиентов гибридных систем будет создана специальная группа поддержки, в которую войдут специалисты с опытом работы с продуктами Azure и Red Hat.
