Релиз Wine 1.3.16

Вышла новая версия реализации Win32 API - Wine 1.3.16. С момента выпуска версии 1.3.15 было закрыто 52 отчёта об ошибках. Основные изменения в новой версии: Встроенный HTML-движок обновлён и теперь базируется на коде Mozilla Firefox 4. Улучшена поддержка отладочной информации формата dwarf. Улучшена поддержка анимированных курсоров. На Mac OS X для сборки и работы Wine больше не требуется библиотека GnuTLS. Произведены улучшения в поддержке GDI-инфраструктуры. Исправлены некоторые ошибки в поддержке MSI-инсталлятора на основе .Net. Улучшена работа GDI+ библиотеки. Обновлены переводы. Исправлено множество ошибок.
