Выпуск Chrome OS 67

Компания Google представила релиз операционной системы Chrome OS 67, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 67. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 67 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 67: Обеспечена возможность запуска приложений, работающих в режиме Progressive Web Apps (PWA), позволяющем организовать работу с web-приложением, как с обособленной программой (размещение в отдельном окне, размещение собственной пиктограммы в панели задач и списке запущенных программ, интеграция с меню, работа при отсутствии сетевого соединения, установка как отдельной программы и т.п.). Добавлена возможность извлечения информации о цветовых пространствах, в которых может работать монитор, и улучшено качество цветопередачи на мониторах с поддержкой цветового пространства Wide-gamut; В панели изменён интерфейс для отправки ответов на уведомления, поступившие от Android-приложений, выполняемых при помощи ARC++ (App Runtime for Chrome, прослойка для запуска Android-приложений). Например, получив уведомление о новом сообщении можно сразу набрать ответ без открытия дополнительных окон; При длительном удержании кнопки отключения питания на Chromebooks с сенсорными экранами по аналогии со смартфонами теперь выводится специальное меню с выбором дальнейших действий (выключение или завершения сеанса); Размер виртуальной клавиатуры теперь может адаптивно выбираться в зависимости от размера экрана - для больших экранов вместо растягивая от края до края теперь используется компактное окно, которое можно переместить в любое место экрана. Включение нового поведения осуществляется выбором режима "Floating mode" в настройках "chrome://flags/#enable-floating-virtual-keyboard"; Обеспечено сохранение позиции окон приложений в случае отключения и повторного подключения внешнего монитора, т.е. если было открыто окно на внешнем мониторе, то после отключения монитора это окно переместится на встроенный экрана, а при повторном подключении монитора вернётся на прежнее место; Добавлен новый интерфейс для выбора обоев рабочего стола, оптимизированный для управления с сенсорного экрана. Кроме доступа к типовой коллекции обоев для установки также предложена секция "My Photos", в которой можно выбрать в качестве обоев произвольную фотографию. Для включения можно использовать опцию "chrome://flags/#enable-new-wallpaper-picker"; На странице chrome://system обеспечено отображение служебной информации о всех мониторах, полученной через EDID (Extended Display Identification Data); Изменён интерфейс с информацией о доступных горячих клавишах, добавлена встроенная функция для поиска необходимой клавиатурной комбинации; Для устройств с сенсорным экраном реализован новый вариант оформления интерфейса Chrome в стиле Material Design (Touchable material 2.0), отличающийся увеличенными отступами, скруглённым полем адресной строки, новой кнопкой "+" для открытия вкладок, изменением расцветки панелей и полей ввода; Для устройств с сенсорными экранами включен режим разделения экрана - при нажатии пиктограммы "[]||" в нижнем правом углу экрана показываются все активные окна, которые можно прикрепить в нужных местах; В файловом менеджере добавлена поддержка работы с zip-архивами в Google Drive; В режиме разработчика добавлена поддержка отладочного интерфейса ADB (Android Debug Bridge) при подключении через USB; Средства масштабирования страниц теперь применимы и для приложений из Google Play; Обеспечено визуальное сопровождение процесса миграции накопителей с ecryptfs на файловую систему ext4 с включенным шифрованием; На странице входа в систему добавлена возможность отправки отзыва о работе устройства; Переработан список Bluetooth-устройств; В контекстное меню для форм ввода текста добавлена кнопка для отображения интерфейса вставки Emoji (активируется через chrome://flags/#enable-emoji-context-menu); Включена по умолчанию функция чтения вслух текста в выделенном блоке (select-to-speak). Для голосового чтения достаточно нажать и удерживать клавишу Search, после чего выделить блок для чтения; В интерфейс для запуска приложений добавлена возможность создания и изменения каталогов в режиме inline через сенсорный экран; Для устройств со съёмной базовой частью (Detachable Chromebook, например, HP Chromebook X2), у которых экран может отсоединяться для использования в виде планшета, добавлено определение подключения экранного блока к базовому блоку с клавиатурой от другого устройства с выводом соответствующего предупреждения пользователю; Для устройств на базе архитектуры ARM в ядре Linux 4.4 включена защита от атаки Spectre 2.
