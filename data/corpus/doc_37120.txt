Вышел Turnkey Linux 12.1, набор мини-дистрибутивов для быстрого развертывания приложений

Доступна новая версия проекта Turnkey Linux 12.1, в рамках которого сформирована коллекция из более ста минималистичных сборок Debian, пригодных для использования в качестве гостевых ОС в системах виртуализации VMware, Xen, OpenVZ, KVM, VirtualBox или для быстрого развертывания в Cloud-окружениях Amazon EC2. Средний размер каждой сборки - 200 Мб. Сразу после установки пользователь имеет возможность получить полностью работоспособные из коробки рабочие окружения с LAMP (Linux, Apache, MySQL, PHP/Python/Perl), Ruby on Rails, Joomla, MediaWiki, Wordpress, Drupal, Apache Tomcat, LAPP, Django, MySQL, PostgreSQL, Node.js, Jenkins, Typo3, Plone, SugarCRM, punBB, OS Commerce, ownCloud, MongoDB, OpenLDAP, gitlab, CouchDB и т.д. Управление программным обеспечением производится через специально подготовленный web-интерфейс. Сборки снабжены системой автоматического резервного копирования и средством для автоматической установки обновлений. Выпуск примечателен переходом к формированию 64-разрядных сборок и задействованием новой системы сборки tkldev, которую планируется в ближайшее время оформить в виде самодостаточного открытого продукта. Открытие кода tkldev позволит любому желающему создавать и пересобирать собственные варианты сборок, используя наработки проекта Turnkey. Выпуск Turnkey Linux 12.1 построен на пакетной базе Debian Squeeze 6.0.7 и включает свежие версии обеспечивающих работу сервисов приложений, бэкпортированные для Debian Squeeze. Параллельно тестируется версия TurnKey 13, основанная на Debian Wheezy.
