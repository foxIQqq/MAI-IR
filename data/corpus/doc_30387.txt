Обновление Java SE 6 Update 25

Вышел двадцать пятый корректирующий релиз Java SE Runtime (JRE) 6 и Java SE Development Kit (JDK) 6, в котором отмечено 193 исправления, направленных на увеличение стабильности и производительности. Исправленные ошибки не связаны с проблемами безопасности, последним корректирующим релизом, направленным на устранение узявимостей, был выпуск Java SE 6 Update 24. Из изменений можно отметить: Обновление виртуальной машины Java Hotspot VM 20, в которую были внесены улучшения, связанные с повышением стабильности и надежности. В новой версии Java Hotspot VM расширены возможности по диагностике проблем, связанных с исчерпанием памяти и нехватки места в "куче", увеличена детализация hs_err-файлов, улучшенна обработка неожиданных исключений в коде приложений. В Server VM добавлен режим "tiered"-компиляции (опция -XX:+TieredCompilation), позволяющей ускорить запуск. Продолжена работа по совершенствованию сборщика мусора Garbage First (G1), который все еще помечен как экспериментальный; Улучшение совместимости с браузерами Internet Explorer 9, Firefox 4 и Chrome 10. Улучшение реализации класса BigDecimal, в результате чего удалось добиться повышения производительности на 30%; Увеличение эффективности работы класса java.util.logging.LogRecord; Обеспечение поддержки систем: Oracle Linux 6, Oracle Solaris 11 Express 2010.11, Windows 7 SP1, Windows 2008 R2 SP1 и VirtualBox 4.
