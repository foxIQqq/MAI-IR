Выпуск Chrome OS 87

Состоялся релиз операционной системы Chrome OS 87, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 87. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 87 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 87: Функциональность размещения данных в хранилище Google Drive расширена возможностью переименования файлов и сохранения файлов в выбранный каталог Google Drive. Включена функция быстрого поиска вкладок, которая позволяет просмотреть список всех открытых вкладок и быстро отфильтровать нужную вкладку, независимо от того в текущем или другом окне она находится. Для присоединённых Bluetooth-устройств в настройках обеспечен показ уровня заряда аккумулятора. Для пользователей с нарушениями моторики рук, не позволяющими работать с традиционными клавиатурами и мышами, реализована поддержка работы с использованием специализированных Switch-устройств. Улучшен интерфейс для настройки языковых параметров для пользователей, использующих несколько языков. Предложен более интерактивный интерфейс для переключения между открытыми окнами (Alt+Tab). В выводимом списке появилась возможность выбора окна мышью, стилусом или касанием сенсорного экрана. Модернизирован компонент для переименования виртуальных рабочих столов и каталогов в Launcher.
