Началось бета-тестирование Steam для Chrome OS

Компании Google и Valve перевели реализацию сервиса доставки игр Steam для платформы Chrome OS на стадию бета-тестирования. Бета-выпуск Steam уже предложен в тестовых сборках Chrome OS 108.0.5359.24 (включается через chrome://flags#enable-borealis). Возможность использования Steam и предлагаемых в нём игровых приложений доступна для устройств Chromebook производства Acer, ASUS, HP, Framework, IdeaPad и Lenovo, оснащённых как минимум CPU Intel Core i3 или AMD Ryzen 3 и 8GB ОЗУ (рекомендуется i5 / Ryzen 5 и 16GB ОЗУ). По сравнению с ранее предлагавшимися альфа-выпусками в бета-версии в три раза увеличено число поддерживаемых Chromebook, упрощён интерфейс, повышена производительность и улучшена совместимость с играми (в список проверенных игр добавлено 50 новых игр). Поддерживается запуск игр, поставляемых как для Linux и так и Windows. При выборе игры в первую очередь предпринимается попытка запуска Linux-сборки игры, но если Linux-версия отсутствует можно установить и Windows-версию, для запуска которой будет использована прослойка Proton, основанная на Wine, DXVK и vkd3d. Игры запускаются в отдельной виртуальной машине с Linux-окружением, реализация которой основана на подсистеме CrosVM, использующей гипервизор KVM. Внутри базовой виртуальной машины запускаются отдельные контейнеры с программами (используется LXC), которые можно устанавливать как обычные приложения для Chrome OS. Установленные Linux-приложения запускаются по аналогии с Android-приложениями в Chrome OS c отображением пиктограмм в панели приложений. Для функционирования графических приложений CrosVM предоставляет встроенную поддержку Wayland-клиентов (virtio-wayland) с выполнением на стороне основного хоста композитного сервера Sommelier. Поддерживается как запуск приложений на базе Wayland, так и обычных X-программ (используется прослойка XWayland). Основные изменения в бета-выпуске: Добавлена поддержка устройств на базе CPU AMD Ryzen 5000 C и 12 поколения Intel Gen Core, минимальные требования снижены до CPU Intel Core i3 и AMD Ryzen 3. Добавлены специфичные для разных игр оптимизации и настройки. Снижено потребление дискового пространства за счёт изменения формата кэша шейдеров. Обеспечено сохранение кэша шейдеров между сеансами. Добавлена начальная поддержка DirectX 12. Добавлена поддержка Vulkan 1.3. Снижена нагрузка на процессор в играх, использующих графические API DirectX и Vulkan. Переработано управление хранилищем для размещения игр, загружаемых из Steam. Ранее место выделялось в зависимости от размера установочных файлов и не учитывало содержимое, которое могло загружаться напрямую из игры. Теперь дополнительное место может выделяться динамически. Проведена оптимизация операций масштабирования и отрисовки с использованием GPU. Повышена производительность при масштабировании вывода для экранов с очень высоким разрешением (QHD и UHD). Улучшено переключение в полноэкранный режим. Улучшена работа с клавиатурой. Улучшено управление окнами. Добавлено уведомление о низком заряде аккумулятора, выводимое в том числе в полноэкранных играх. Предложен новый инсталлятор. Повышена производительность при использовании OpenGL и Vulkan.
