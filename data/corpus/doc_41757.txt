Релиз платформы для организации групповой работы Kolab 3.4

Доступен релиз Kolab 3.4, платформы для организации совместной коллективной работы, базирующейся на технологиях KDE. Основу платформы составляет серверное ПО Kolab Server, позволяющее организовать работу серверной инфраструктуры, реализующей средства групповой работы, единой адресной книги, календаря-планировщика и почтового сервиса. Бинарные сборки Kolab 3.4 подготовлены для RHEL, SUSE, Debian, Fedora, Ubuntu и openSUSE. Новый выпуск примечателен использованием нового оформления web-клиента и проведением большой работы по стабилизации кодовой базы. В качестве клиентских приложений могут выступать как стандартные программы из состава KDE PIM (Kontact, KMail, KOrganizer и KAddressbook), так и сторонние системы, в том числе доступен набор коннекторов для Microsoft Outlook (Bynari Connector, плагин Toltec Connector 2 и MAPI-хранилище KONSEC Konnektor). В качестве базового web-клиента в составе платформы поставляется Roundcube Webmail, в полной мере поддерживающий вторую и третью версии формата хранилища Kolab. Для интеграции с Thunderbird и SeaMonkey проектом развивается специальный плагин SyncKolab. Модуль интеграции также доступен для GNOME Evolution. Для организации работы с мобильными системами предоставляется ряд решений по синхронизации данных, таких как CalDAV/CardDAV и ActiveSync-совместимые решения Kolab ActiveSync Server и Z-Push. Для организации работы в Kolab Server задействованы стандартные открытые компоненты, такие как сервер директорий 389, Apache httpd, Postfix, Cyrus IMAP, SASL, OpenSSL. Для управления системой подготовлен специальный административный web-интерфейс. Поддерживается ведение единой адресной книги, интегрируемой с почтовым сервисом с поддержкой POP3 и IMAP4. Для шифрования почты предлагается S/MIME-шифрование на стороне клиента.
