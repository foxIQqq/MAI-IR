Компания Qt Software представила проект Qt Solutions

Компания Qt Software представила пакет Qt Solutions, объединяющий в едином каталоге дополнительные программные компоненты для Qt, распространяемые в рамках лицензии LGPL, т.е. которые можно использовать в закрытых проектах без открытия исходных текстов. Qt Solutions представляет собой единую платформу и набор компонент и утилит, призванных обеспечить интеграцию тулкита с продуктами сторонних производителей, а также включающую экспериментальные возможности, которые в будущем планируется добавить в состав фреймворка Qt. В первый выпуск Qt Solutions включены: MML Widget - компонент для прорисовки математических формул, определенных в формате MathML 2.0; Pie Menu - модуль для формирования круговых контекстных меню; SOAP - компонент для взаимодействия с web-сервисами по протоколу SOAP 1.1; State Machine Framework - API для обработки событий методом иерархических конечных автоматов; Animation Framework - экспериментальный фреймворк для создания и управления анимированными объектами на экране; Window Menu - реализация всплывающего меню со списком открытых окон в пространстве QWorkspace; Locked File - класс QtLockedFile расширяющий возможности QFile по совместному блокированию файлов, при необходимости организации записи из нескольких процессов в один файл; Thumb Wheel - виджет с реализацией элемента прокрутки в виде горизонтального или вертикального вращающегося колеса, например, для выполнения операций масштабирования или прокрутки;
