Доступен релиз Mozilla SeaMonkey 2.0

Проект Mozilla представил релиз SeaMonkey 2.0, объединяющий в одном продукте такие приложения, как web-браузер на основе Firefox 3.5, программу для чтения почты на основе Thunderbird, WYSIWYG редактор html страниц, IRC клиент ChatZilla, набор средств для разработчиков, включающих DOM inspector и JavaScript отладчик. Основные новшества: Используется JavaScript движок TraceMonkey и базовая начинка, основанная на коде Firefox 3.5.4; Отмечено значительное улучшение пользовательского интерфейса; Добавлена возможность архивирования почты. Добавлена поддержка вкладок в почтовом клиенте (например, в табах можно открыть папки, отдельные сообщения, и даже календарь, доступный через установку расширения Lightning); Добавлен новый менеджер тем и дополнений, напоминающий по возможностям менеджер дополнений Firefox; Полностью переработан менеджер загрузок, который теперь поддерживает возобновление загрузки после закрытия сессии. Улучшена работа менеджера сохраненных паролей. В системе просмотра cookie появилась возможность поиска по имени cookie и сайту; Для web-форм теперь автоматически запоминаются введенные значения, которые предлагаются при последующем вводе через выпадающее меню; Добавлена возможность отката операции закрытия окна браузера; Панель инструментов в браузере, почтовом клиенте теперь полностью настраиваема пользователем, включая поддержку изменения размера пиктограмм; В программе для чтения почты и news-конференций появилась поддержка RSS и Atom лент; Изменен метод хранения истории посещений, добавлен интеллектуальный инструмент выбора релевантных web-страниц при формировании запроса в строке ввода адреса (как в Firefox); Добавлена поддержка упрощенного интерфейса для разработчиков расширений (SMILE); Изменился путь хранения профиля. Старые профили, за исключением дополнений, тем и плагинов, подвергнутся миграции, их данные не будут затронуты; Прекращена поддержка Win9x и MacOS 10.3, из-за перехода на GTK 2.10 и Cairo.
