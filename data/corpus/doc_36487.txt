Вышел OpenELEC 3.0, дистрибутив для создания домашних кинотеатров

Увидел свет релиз дистрибутива OpenELEC 3.0 (Open Embedded Linux Entertainment Center), предназначенного для создания домашних кинотеатров (HTPC - Home Theatre PC). В качестве основы использован медиацентр XBMC, обладающий богатой функциональностью и поддерживающий расширение возможностей через подключение дополнений. Размер системного образа, пригодного для загрузки с USB-накопителя или SD-карты, составляет 163 Мб. Дополнительно сформированы специализированные сборки для Raspberry Pi, NVIDIA ION/ION2, AMD Fusion, HTPC на базе CPU Intel. OpenELEC не является ответвлением от существующих дистрибутивов и основывается на собственных разработках. Кроме штатных возможностей XBMC, OpenELEC поддерживает ряд дополнительных функций, нацеленных на максимальное упрощение работы. Например, подготовлено специальное конфигурационное дополнение, позволяющее настроить параметры сетевого подключения, управлять параметрами LCD-экранов, разрешить или запретить автоматическую установку обновлений. При этом все манипуляции можно провести без подключения клавиатуры, только при наличии пульта дистанционного управления. Из коробки дистрибутив поддерживает такие возможности, как использование пульта дистанционного управления (возможно управление как через инфракрасный порт, так и через Bluetooth), организация совместного доступа к файлам (встроен сервер Samba), встроенный BitTorrent-клиент Transmission, автоматический поиск и подключение локальных и внешних накопителей. При помощи OpenELEC можно превратить любой компьютер в медиацентр, работать с которым не сложнее, чем с DVD-проигрывателем или телеприставкой. Основной принцип дистрибутива "всё просто работает", для получения полностью готового к работе окружения достаточно просто загрузить OpenELEC с Flash-накопителя. Пользователю нет необходимости заботиться о поддержании системы в актуальном состоянии - в дистрибутиве используется система автоматической загрузки и установки обновлений, активируемая при подключении к глобальной сети. Предусмотрена возможность расширения функциональности дистрибутива через систему дополнений, которые устанавливаются из отдельного репозитория, развиваемого разработчиками проекта. Новшества OpenELEC 3.0: Переход на новую версию медиацентра XBMC Frodo 12.1. В прошлом выпуске использовался XBMC 11, по сравнению с которым задействована новая звуковая подсистема XBMC AudioEngine, добавлена поддержка работы в роли DVR/PVR (просмотр Live TV, прослушивание интернет-радио, работа с электронным телегидом и организация видеозаписи по расписанию), улучшенна поддержка AirPlay, расширенны функции выбора контента в медиабиблиотеке, добавлена поддержка тегов для группировки фильмов по категориям, упрощена организация совместного доступа к библиотеке контента через UPnP; Переход на движок AudioEngine позволил улучшить определение звукового оборудования, но привёл к изменению наименования звуковых устройств и способа обработки звука, что может негативно отразиться на работоспособности после обновления с прошлого выпуска. В случае проблем следует обновить конфигурацию в интерфейсе настройки XBMC и удалить файл /storage/.config/asound.conf; Добавлена встроенная поддержка 4TheRecord, DVB Viewer, Mediaportal, MythTV, NextPVR, TVHeadend, VDR, Njoy N7 и VU+; Подготовлена официальная сборка для Raspberry Pi; В сотрудничестве с компанией Arctic налажена предустановка OpenELEC на устройства Arctic MC001; Интегрированы дополнительные патчи для улучшения поддержки AppleTV; Стабилизирована поддержка акселерации декодирования видео (H.264 и VC-1) с использованием API XVBA (X-Video Bitstream Acceleration) на оборудовании AMD. Особенности дистрибутива: Наличие открытых и проприетарных драйверов для карт AMD/ATI, NVIDIA и Intel. В также состав входит большая коллекция драйверов для TV-тюнеров, проводных и беспроводных сетевых карт; Поддержка средств акселерации декодирования видео: Intel VAAPI, NVIDIA VDPAU, AMD XVBA (через VAAPI) и Broadcom CrystalHD; Дополнительные оптимизации для различных типов процессоров и чипсетов, используемых в неттопах, включая Intel Atom, AMD Fusion, NVIDIA ION и Raspberry Pi; Высокая скорость загрузки - дистрибутив полностью готов к работе уже через 10 секунд; Поддержка сетевых ФС для Windows (SMB/CIFS), MacOS (AFP), Linux (NFS/SSH/SFTP); Наличие встроенного NTP-сервиса для установки точного времени; Встроенный samba-сервер для организации доступа к локальному хранилищу; Пподдержка проигрывания потоков с DVB-устройств и просмотра контента через IP TV; Возможность расширенной настройки через SSH; Поддержка локальных файловых систем: EXT2, EXT3, EXT4, FAT32, NTFS, HFS, HFS+, JFS, XFS и ReiserFS; Поддержка пультов дистанционного управления: HID (Boxee, Nyxboard), Windows MCE, ATI X10, IMON, IRTRANS, Nuvoton, Fintek, ENE eHome, ITE8x12, RedRat3, StreamZap и т.п.
