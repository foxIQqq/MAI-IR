Выпуск дистрибутива для резервного копирования Rescuezilla 2.2

Доступен выпуск дистрибутива Rescuezilla 2.2, предназначенного для резервного копирования, восстановления систем после сбоев и диагностики различных аппаратных проблем. Дистрибутив построен на пакетной базе Ubuntu и продолжает развитие проекта "Redo Backup & Rescue", разработка которого была прекращена в 2012 году. Для загрузки предлагаются live-сборки для 64-разрядных систем x86 (805MB). Rescuezilla поддерживает резервное копирование и восстановление случайно удалённых файлов в разделах Linux, macOS и Windows. Выполняется автоматический поиск и подключение сетевых разделов, которые можно использовать для размещения резервных копий. Графический интерфейс основан на оболочке LXDE. Формат создаваемых резервных копий полностью совместим с дистрибутивом Clonezilla. При восстановлении поддерживается работа с образами Clonezilla, Redo Rescue, Foxclone и FSArchiver. Основные изменения: Осуществлён переход на пакетную базу Ubuntu 21.04. Помимо основной сборки также сформирован альтернативный вариант на основе LTS-выпуска Ubuntu 20.04, который более стабильно загружается на некоторых устройствах. Добавлена функция клонирования, позволяющая напрямую переносить разделы с одного устройства на другое. Добавлена возможность восстановления и анализа образов виртуальных машин. Поддерживаются все форматы с которыми работает qemu-nbd (VirtualBox VDI, VMWare VMDK, Qemu QCOW2, HyperV VHDx, raw .dd/.img и т.д.). Добавлена поддержка восстановления и изучения начинки образов с резервными копиями, созданными в продуктах Redo Rescue, Foxclone, FSArchiver, Redo v0.9.2 и частично FOG Project. Предоставлена поддержка настройки формата и уровня сжатия (gzip, zstandard, без сжатия). Реализована недостающая логика восстановления образов Clonezilla, учитывающая такие особенности как расширение размера ФС для записи в раздел большего размера, стадия проверки корректности восстановления, очистка dirty-флага для NTFS, удаление записей о MAC-адресе в udev, переустановка загрузчиков (syslinux, GRUB), обновление initramfs, восстановление метаданных LVM VG и обновление EFI NVRAM для загрузочного устройства. Добавлен режим для выключения компьютера или перезагрузки после завершения операции. Добавлена поддержка подсоединения к внешним хранилищам по NFS и SSH (SFTP). Добавлена возможность прикрепления описания к резервным копиям. Добавлена поддержка создания скриншотов через нажатие Print Screen В состав включены файлы с переводом на русский язык.
