Релиз Chrome OS 70

Компания Google представила релиз операционной системы Chrome OS 70, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 70. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 70 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 70: Добавлена возможность монтирования сетевых хранилищ Windows и Samba с использованием протокола SMB. Поддерживаются механизмы аутентификации Kerberos, Active Directory и NTLMv2. Функция подключения удалённого раздела встроена в штатный файловый менеджер Chrome OS. Настройка параметров монтирования осуществляется в конфигураторе (в программу Settings добавлен новый раздел "Network File Shares"); Добавлена возможность отдельного включения/выключения функции автозаполнения форм ввода для адресов и платёжных реквизитов; Реализована возможность выборочной активации дополнений только для определённых сайтов, с запретом применения дополнений для сайтов, не включённых в белый список. Также доступен режим индивидуальной активации дополнения на каждой странице, при котором дополнение включается только после явного клика на значке в панели; В настройки синтезатора речи в разделе для людей с ограниченными возможностями добавлена поддержка полей для поиска; В состав сборок для архитектуры x86 включён декодировщик для видеокодека AV1, разработанного альянсом Open Media (AOMedia), в котором представлены такие компании, как Mozilla, Google, Microsoft, Intel, ARM, NVIDIA, IBM, Cisco, Amazon, Netflix, AMD, VideoLAN, CCN и Realtek. AV1 позиционируется как общедоступный и не требующий оплаты отчислений свободный формат кодирования видео, который заметно опережает H.264 и VP9 по уровню сжатия; Обновлён интерфейс приложения для работы с камерой. Снятые в приложении фотографии и видеозаписи теперь сохраняются в каталог Downloads в файловом менеджере; В настройки системы ввода добавлена возможность переопределения привязки клавиш Search, Command и Windows для подключаемых внешних клавиатур; Для устройств с сенсорным экраном по аналогии со смартфонами реализована возможность использования всплывающей виртуальной клавиатуры для набора текста одним пальцем.
