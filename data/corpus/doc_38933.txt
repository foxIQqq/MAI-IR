Представлено новое десктоп-окружение Ome, использующее браузерные технологии Mozilla

Началось развитие нового кросс-платформенного десктоп-окружения Ome (Open minded environment), основанного на наработках проекта Firefox OS и работающего поверх браузерного движка Mozilla Gecko. Компоненты окружения и приложения развиваются с использованием Asm.js, низкоуровневого подмножества языка JavaScript со строгой типизацией. Проект находится на начальной стадии развития, готовый для экспериментов код пока не размещён в публичном репозитории. Тем не менее, планы по развитию Ome достаточно обширны. Например, для распространения приложений планируется использовать собственный формат пакетов OPK, созданных по аналогии с пакетами APK от платформы Android и распространяемых через web-каталог приложений. Приложения могут группироваться в "комнаты", в которых программы объединены по области их использования (например, в одной комнате могут быть объединён доступ к программам для работы с графикой). В систему будут встроены средства для автоматизации операций с приложениями, т.е. можно определить сценарий для автоматического выполнения тех или иных действий в интерфейсе программы по аналогии с Sikuli и Selenium. Для хранения данных предлагается использовать систему ячеек, которые могут объединяться в коллекции типизированных данных (например, в ячейке может размещаться MP3-файл и такие ячейки могут образовать коллекцию с музыкой). Ячейки могут храниться как локально, так и удалённо (например, часть музыкальной коллекции может быть размещена на локальной системе, а часть на другом компьютере). В Ome также ожидается наличие обширных средств для интеграции с социальными сетями. Запланирована возможность организации совместного доступа к "комнатам" с приложениями, при котором несколько пользователей могут совместно работать с определённым набором программ (участники будут видеть общее содержимое экрана, для каждого пользователя будет доступен свой курсор).
