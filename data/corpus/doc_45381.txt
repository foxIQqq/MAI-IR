Релиз операционной системы Chrome OS 54

Компания Google опубликовала выпуск операционной системы Chrome OS 54, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 54. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 54 доступна для всех актуальных моделей Chromebook, за исключением HP Chromebook 13 G1, Acer Chromebook 14 for Work и Thinkpad 13 Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 54: Новый интерфейс переключения между окнами, активируемый сочетанием клавиш Alt-Tab; Оформление обзорного режима переведено на элементы Material Design. Для просмотра всех открытых окон можно использовать горячую клавишу F5 (клавиша со значком "[]||"). Поддерживается поиск по имени окна и возможность перехода между эскизами окон клавишами управления курсором; В файловом менеджере реализована функция быстрого предпросмотра содержимого файлов. Для предпросмотра видео, звука или изображений достаточно выбрать файл и нажать пробел; В настройки добавлена опция для постоянного отображения на панели меню управления раскладкой клавиатуры; К внешним USB- и SD-накопителям по умолчанию теперь предоставляется доступ только на чтение; Улучшен предоставляемый на экране входа в систему интерфейс удаления пользователя; Добавлен экспериментальный режим (chrome://flags/#quick-unlock-pin) разблокировки экрана по отдельному цифровому коду, отличающемуся от пароля учётной записи. Пароль задаётся в новой секции настроек "Screen Lock", для блокировки экрана можно использовать комбинацию клавиш Ctrl + L. Возможность активации (chrome://flags/#secondary-ui-md) альтернативного набора пиктограмм в стиле Material Design для панели запуска программ; Добавлены новые опции для людей, имеющих проблемы со зрением, позволяющие дополнительно выделить красной рамкой курсор и активные области ввода, подсветить изменение фокуса и движение указателя мыши;
