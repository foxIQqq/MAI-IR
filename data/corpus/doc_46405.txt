Представлен почтовый клиент Nylas Mail 2.0

Компания Nylas представила стабильный выпуск открытого почтового клиента Nylas Mail 2.0 (бывший N1). Исходные тексты проекта распространяются под лицензией AGPLv3. Приложение построено с использованием платформы Electron, позволяющей создавать самодостаточные пользовательские приложения при помощи технологий Chromium и Node.js. При построении интерфейса применяются развиваемые компанией Facebook компоненты React и Flux. Готовые сборки подготовлены для Linux, Windows и macOS. Новый выпуск примечателен обеспечением полной поддержки платформ Linux и Windows. В основной состав включены компоненты, которые ранее поставлялись в платной Pro-редакции почтового клиента, такие как система напоминаний о необходимости отправить сообщение, отправка по расписанию и возможность отложить сообщения (snooze) для чтения после того, когда будут выполнены первоочередные задачи. Проведена работа по увеличению производительности. В Pro-версии остаются такие возможности, как работа с Microsoft Exchange через ActiveSync, обеспечение совместного доступа к сообщениям, назначение встреч, слияние почты, фильтрация сообщений, интеграция с Salesforce и плагин для PGP. Особенности Nylas Mail: Гибкие средства расширения функциональности через плагины, что может быть использовано для создания производных решений, оптимизированных для нетипичных методов работы с электронной почтой; Большая подборка плагинов, включая средства для автоматизированной фильтрации и классификации почты, загрузки информации о пользователе из профиля на GitHub, привязки персональных пиктограмм к пользователям, выявления фишинга, перевода сообщений с одного языка на другой, быстрой отправки ответа, ввода Emoji и т.д. Совместимость с большинством почтовых сервисов, включая Gmail, Yahoo, iCloud и Microsoft Exchange; Помимо IMAP для хранение почты можно использовать специальный движок Nylas Sync Engine, предназначенный для организации работы со своей перепиской из разных мест и с разных устройств. Движок может запускаться как на локальной системе пользователя вместе с Nylas Mail, так и отдельно в виде облачного сервиса. Поддержка тем оформления; Использования единого Inbox для всех учётных записей; При работе через облако Nylas доступна функция отзыва отправленного сообщения; Шаблоны для быстрой отправки типовых писем; Возможность вывода дополнительной информации об отправителе, полученной из сторонних источников (например, из профиля на GitHub и из социальных сетей); Система отслеживания открытия отправленных писем и перехода по указанным в письме ссылкам.
