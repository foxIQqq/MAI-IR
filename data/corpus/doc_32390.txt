Новая версия программных продуктов Dr.Web для Unix

Компания «Доктор Веб» сообщила о выпуске новой версии (6.0.2) набора проприетарных программных продуктов Dr.Web для почтовых серверов, интернет-шлюзов и файловых серверов Unix (Linux, FreeBSD и Solaris 10). Некоторые изменения: Интеграция с Центром управления Dr.Web Enterprise Security Suite продуктов Dr.Web для интернет-шлюзов, почтовых и файловых серверов Unix (в том числе Novell Storage Services). Улучшена интеграция Dr.Web для рабочих станций Linux с Центром управления: к примеру, администратор ЦУ теперь может задать расписание для автоматического сканирования пользовательских станций; В продуктах Dr.Web для файловых серверов, интернет-шлюзов и почтовых серверов Unix существенно сократилось время перезапуска антивирусного модуля Dr.Web Daemon при обновлении вирусных баз: вместо полной перезагрузки вирусных баз теперь выполняется подгрузка обновлений; Антивирусный демон получил функцию оповещения пользователей об устаревании вирусных баз. Реализован механизм резервного копирования обновлений: в случае если последующее обновление пройдет некорректно, возможен откат антивируса до предыдущего состояния. Существенно повышена стабильность работы Dr.Web для почтовых серверов Unix в условиях высокой нагрузки; Возобновлена поддержка почтового сервера Sendmail 8.13; В Dr.Web для файловых серверов Unix добавлена поддержка Samba 3.6; В чёрный список Dr.Web для интернет-шлюзов добавлена новая категория Social Engineering, содержащая ресурсы, созданные с использованием технологий социальной инженерии. Антивирусный демон продукта Dr.Web для рабочих станций Linux теперь постоянно загружен в память: это нововведение устраняет этап загрузки вирусных баз при каждой новой проверке и, соответственно, ускоряет обработку пользовательского запроса на сканирование; Сканирование в Dr.Web для рабочих станций Linux теперь осуществляется в несколько потоков, при этом список проверяемых областей расширен загрузочными секторами жесткого диска; Файловый монитор в составе Dr.Web для рабочих станций Linux теперь контролирует обращения к файлам во всей системе, независимо от того, какая учетная запись активна. Дополнительно компанией Dr.Web поставляются бесплатные универсальные диски LiveCD и LiveUSB, основанные на Linux и предназначенные для проверки дисков антивирусным пакетом Dr.Web, а также для восстановления операционной системы, после краха в результате вирусной атаки.
