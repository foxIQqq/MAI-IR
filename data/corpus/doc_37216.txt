Началось альфа-тестирование OpenMandriva 2013.0

Объявлено о доступности альфа-версии дистрибутива OpenMandriva 2013.0. Для загрузки предлагаются DVD-сборки, размером 1.1 Гб. Находящийся в тестировании выпуск Mandriva 2012 будет пропущен. Из особенностей OpenMandriva 2013.0 отмечается переход на ядро Linux 3.8.12 и обновление приложений KDE 4.10.4, LibreOffice 4.0.3.3, Firefox 21, Clementine 1.1.1, ROSA Media Player 1.6-1. Среди ключевых новшеств можно отметить: Подсистема HAL полностью исключена из состава дистрибутива; Переход на использование системной библиотеки eglibc; Использование для пересборки пакетов GCC 4.7.x. Интеграция в GCC патчей от проекта Linaro для улучшения поддержки архитектуры ARM; Возвращение загрузчика Aurora или портироваение возможностей Aurora в plymouth; Задействование в качестве клиента для синхронизации точного времени Chrony NTP, который отличается низким потреблением памяти (около 1 Мб), отсутствием активации фонового процесса без необходимости и соответственно более высокой экономией энергии. Кроме того, Chrony NTP обладает лучшими характеристиками по поддержанию точного времени для систем без постоянного соединения с Интернетом, не работающих 24 часа в сутки или содержащих низкокачественные/нестабильные часы (например, в виртуальных машинах); Включение в поставку поддержки msec (Multi-Security Tool) и msec-gui, интерфейса для управления настройками, связанными с безопасностью. Msec также может использоваться для формирования отчетов на основе выполнения периодических проверок, например для контроля за правами доступа к файлам; Одновременная поддержка urpmi и smart для управления пакетами; Улучшение поддержки настройки соединения к беспроводным сетям в drakx-net; Включение в репозиторий пакетов с пользовательским окружением Razor-Qt; Интеграция в дистрибутив пакета Bumblebee с поддержкой технологии Nvidia Optimus; Задействование по умолчанию технологии prelink, которая позволяет ускорить загрузку программ, связанных с большим количеством библиотек; Собран расширенный набор пакетов для встраиваемых окружений на базе uClibc; Начата работа по портированию репозитория main на архитектуру armv8 (aarch64); Активно разивается порт armv7l (softfloat); Подготовлен bootstap-сборщик armv7hl (hardfloat); Процесс кросскомпиляции под новые платформы значительно упрощен средствами rpm5.
