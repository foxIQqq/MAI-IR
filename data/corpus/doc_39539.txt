Выпуск языка программирования Erlang 17.0

Доступен релиз функционального языка программирования Erlang 17.0, ориентированного для разработки распределённых отказоустойчивых приложений, обеспечивающих параллельную обработку запросов в режиме реального времени. Язык получил распространение в таких областях, как телекоммуникации, банковские системы, электронная коммерция, компьютерная телефония и организация мгновенного обмена сообщениями. Одновременно выпущен релиз OTP 17.0 (Open Telecom Platform) - сопутствующего набора библиотек и компонентов для разработки распределённых систем на языке Erlang. Код проекта распространяется под модифицированной свободной лицензией MPL. Основные новшества Erlang/OTP 17.0: Выполнено портирование Erlang/OTP для операционной системы режима реального времени OSE; Новый словарный тип данных Maps; Средства для более естественного маппинга ASN.1 OCTET STRING и BIT STRING в типы Erlang; Для TCP, UDP и SCTP добавлена поддержка опции сетевых сокетов {active, N}; Новый опциональный механизм балансировки нагрузки на планировщик; Во всех внутренних распределителях памяти ERTS-приложений (Erlang Run-Time System Application) включена по умолчанию миграция областей размещения блоков в памяти (carriers); Увеличена интенсивность работы сборщика мусора; Экспериментальная поддержка "грязных планировщиков" (dirty schedulers); Улучшение поддержки Unicode; Новая схема наименования версий.
