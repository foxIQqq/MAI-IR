Вышел релиз системы сетевого мониторинга Zenoss 2.4

Вышел релиз системы сетевого мониторинга Zenoss Core 2.4, распространяемая под лицензией GPL и позиционируемая в качестве открытой альтернативы продуктам IBM Tivoli, HP OpenView, BMC Patrol. Программа написана на языке Python с использованием Zope, в качестве СУБД используется MySQL, в web-интерфейсе присутствуют элементы Ajax. Поддерживается подключение модулей мониторинга от системы Nagios. Из новшеств релиза 2.4 можно отметить: добавления мастера установки (Set-Up Wizard для быстрой инсталляции и автоматического нахождения объектов мониторинга в сети), возможность использования SSH в качестве транспорта для проверки удаленных хостов, расширение возможностей по мониторингу Linux серверов без установки на них специальных программ-агентов, улучшения системы формирования отчетов (например, поддержка нормализации метрик мониторинга), подготовлено расширенное руководство по организации работы системы мониторинга.
