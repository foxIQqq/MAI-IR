Новая версия Modem Manager GUI 0.0.16

Следом за релизом системного сервиса Modem Manager 1.0 увидел свет новый выпуск Modem Manager GUI 0.0.16, фронтэнда для управления Modem Manager в графическом режиме. Программа написана с использованием библиотеки GTK3+ и распространяется под лицензией GPL v3. Особенности программы: Отправка и приём SMS; Отправка запросов USSD; Отображение различной информации о модеме и SIM-карте (оператор, IMEI, уровень сигнала и др.); Сканирование доступных сетей; Управление процессом передачи данных и ограничения трафика и времени. Новые функции в версии 0.0.16: Поддержка новой версии бэкэнда modemmanager 0.7 и Wader API (поддержка modemmanager 0.6 сохранена); Работа с контактами: поддержка импорта контактов из адресной книги GNOME или из модема/телефона при помощи Wader (поддержка KDE PIM появится в следующих выпусках); Сворачивание в трей; Интеграция с глобальным меню GNOME; Поступающие SMS могут выводиться в верхней части списка; Возможно назначение звукового файла для проигрывания при приходе нового сообщения; Реализован интерфейс для просмотра истории потребления трафика; Возможность работы без NetworkManager с использованием базы соединений pppd; Файлы конфигурации теперь сохраняются в иерархии директорий ~/.config, в соответствии со стандартом XDG.
