Доступен дистрибутив Scientific Linux 5.6

Анонсирован релиз дистрибутива Scientific Linux 5.6, построенного на пакетной базе Red Hat Enterprise Linux 5.6 и дополненного средствами, ориентированными на использование в научных учреждениях. Дистрибутив доступен в DVD и CD сборках для архитектур i386 и x86_64. Релиз Scientific Linux 6.0, основанный на RHEL 6.0, был выпущен в марте. Следующим шагом станет подготовка релиза Scientific Linux 6.1 на базе RHEL 6.1. Выпуск CentOS 6.0 ожидается через одну или две недели. Из поддерживаемых разработчиками Scientific Linux улучшений, относительно пакетной базы RHEL, можно отметить: В состав дистрибутива включена научная библиотека SciPy, которую можно использовать для научных, инженерных и математических рассчетов, в также для построения графиков и диаграмм, на основе набора статистических данных. Кроме того, состав были включены: пакет визуализации Graphviz; библиотека Numpy, для организации научных расчетов; математическая библиотека FFTW (Fastest Fourier Transform); Добавлена библиотека cfitsio с реализацией поддержки формата FITS (Flexible Image Transport System), широко используемого в астрономических исследованиях, для языков Си и FORTRAN; Интеграция технологии DKMS (Dynamic Kernel Module Support), позволяющей автоматически перестраивать текущие модули ядра с драйверами устройств после обновления версии ядра. Внедрение DKMS позволит автоматически устанавливать драйверы, которые необходимы для работы определенных аппаратных компонент текущей системы, но отсутствуют в стандартном Linux ядре. Для разработчиков, DKMS предоставляет средства создания драйверов без оглядки на особенности различных версий Linux ядра. В состав дистрибутива для драйвера iwlwifi включены дополнительные прошивки (firmware для беспроводных карт Intel серий 5150, 3945, 4945, 5000 и, начиная с текущей версии, 6000. Также добавлены драйверы для беспроводных карт Ralink; В состав включена утилита 915resolution для модификации видео BIOS для графических карт на базе GPU Intel 845G, 855G, 865G, 915G, 915GM и 945G, что позволяет добиться установки дополнительных разрешений экрана; OpenJDK заменен на пакет Sun Java/JDK 1.6.0p20; Добавлен оконный менеджер IceWM 1.2.37; Добавлена поддержка языков Lua и GNU R (язык для статистических расчетов); Поддержка файловой системы XFS для архитектуры x86_64. В программе установки пока нельзя создать XFS раздел, но установить Scientific Linux на ранее созданный XFS раздел можно; В состав включен набор обучающих приложений для KDE - KDEEDU; В поставку включен код AUFS, альтернативной реализации UnionFS; В состав включено несколько дополнительных FUSE-модулей: fuse-smb и fuse-sshfs, для монтирования SMB и SSH Вместо консольного почтового клиента Pine используется Alpine 2.0; В составе дистрибутива поставляются дополнительные Perl-модули: DBD-XBase, MailTools, Parse-RecDescent, SQL-Statement, TermReadKey, Text-CSV_XS, Text-Template, Tk; Добавлены дополнительные мультимедиа кодеки (gstreamer-plugins-extras из SL4 и k3b-extras из репозитория livna); Добавлен пакет NdisWrapper, позволяющий использовать в Linux NDIS драйверы из Windows; OpenAFS 1.4.14 - открытая реализация распределенной ФС Andrew File System; Yum обновлен до версии 3.2.22.
