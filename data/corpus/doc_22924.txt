Формирование вопросов для интервью с вице-президентом группы разработки Solaris в Sun

Журнал Linux Format приглашает всех желающих принять участие в составлении вопросов для интервью с Джеффом Джексоном, cтаршим вице-президентом группы разработки Solaris в компании Sun Microsystems. Джефф Джексон осуществляет руководство разработкой операционной системы Solaris, программного обеспечения для кластеризации, баз данных и систем хранения. Джефф принимал непосредственное участие в программе по открытию исходных текстов OpenSolaris и Java Desktop, а также в области других технологий Solaris. До руководства разработкой Solaris, Джефф руководил в компании Sun платформами Java Standard Edition и Java Enterprise, программами поддержки разработчиков и группами средств разработки. Являясь одним из основателей проекта Java EE, Джефф имеет уникальный взгляд на будущее технологии Java. Он направлял усилия в этой области от замысла до воплощения. Джефф также выполнял двойную функцию в проекте Java EE, руководя как разработкой, так и маркетингом. Оставить свой вопрос можно в специально созданной теме на Линуксфоруме или в комментарии к данной новости. Вопросы принимаются до 22:00 MSK 10 августа.
