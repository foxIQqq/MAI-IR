Доступна операционная система Chrome OS 40

Компания Google представила релиз операционной системы Chrome OS 40, основанной на ядре Linux, открытых компонентах и web-браузере Chrome. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 40 доступна для всех актуальных моделей Chromebook. Основные изменения: Модернизировано оформление меню выбора приложений. Представлена возможность быстрого доступа к коллекции смайликов через экранную клавиатуру; Добавлена возможность синхронизации установленных пользователем обоев рабочего стола и настроек беспроводной сети между разными устройствами; По умолчанию предлагаются обои рабочего стола, оформленные в стиле концепции Material Design; Экспериментальная поддержка режима умной разблокировки экрана, при котором можно связать ноутбук со смартфоном на базе платформы Android 5, после чего экран будет автоматически разблокироваться, когда смартфон находится поблизости и становится достижим по Bluetooth. Добавлен API chrome.fileSystemProvider, позволяющий создавать расширения для доступа из файлового менеджера к удалённым файловым системам, по аналогии с доступом к Google Drive; Аудиопроигрыватель преобразован в отдельное приложение. Кроме того, можно отметить реализацию в бета-выпуске Chrome OS 41 упрощённого интерфейса для включения отладочных возможностей при загрузке в режиме разработчика, позволяющих получить доступ к дополнительным возможностям Chrome OS, таким как модификация системных файлов, обеспечение доступа к системе по SSH и загрузка произвольной сторонней операционной системы с USB-накопителя. Для простой установки Ubuntu Linux в качестве второй операционной системы на устройствах Chromebook можно воспользоваться установщиком ubuntu-chromebook-installer. Для создания chroot-окружения с Ubuntu или Debian и работы с ним из ChromeOS можно использовать crouton и дополнение crouton-integration (возможен запуск графических приложений и типичных для Linux рабочих столов, не выходя из Chrome OS, в том числе полноценный рабочий стол Linux может отображаться в отдельном окне внутри сеанса Chrome OS).
