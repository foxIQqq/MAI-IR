Релиз дистрибутива Nitrux 1.3.9 с рабочим столом NX Desktop

Опубликован выпуск дистрибутива Nitrux 1.3.9, построенного на пакетной базе Debian, технологиях KDE и системе инициализации OpenRC. Дистрибутив развивает собственный рабочий стол NX Desktop, который представляет собой надстройку над пользовательским окружением KDE Plasma. Для установки дополнительных приложений продвигается система самодостаточных пакетов AppImages и собственный центр установки приложений NX Software Center. Размер загрузочных образов составляет 4.6 ГБ и 1.4 ГБ. Наработки проекта распространяются под свободными лицензиями. Рабочий стол NX Desktop предлагает иное стилевое оформление, собственную реализацию системного лотка, центра вывода уведомлений и различных плазмоидов, таких как конфигуратор сетевых соединений и мультимедийный апплет для регулирования громкости и управления воспроизведением мультимедийного контента. Из развиваемых проектом приложений также выделяется интерфейс для настройки межсетевого экрана NX Firewall, позволяющий управлять сетевым доступом на уровне отдельных приложений. Среди входящих в базовую поставку приложений: файловый менеджер Index (можно использовать и Dolphin), текстовый редактор Kate, архиватор Ark, эмулятор терминала Konsole, браузер Chromium, музыкальный проигрыватель VVave, видеопроигрыватель VLC, офисный пакет LibreOffice и просмотрщик изображений Pix. В новом выпуске: Дистрибутив перешёл с пакетной базы Ubuntu (с переносом некоторых пакетов из Devuan) в пользу Debian GNU/Linux. Для установки на выбор предложены пакеты с ядром Linux 5.4.108, 5.10.26, 5.11.10, Linux Libre 5.10.26 и Linux Libre 5.11.10, а также ядро 5.11 с патчами от проектов Liquorix и Xanmod. Компоненты рабочего стола обновлены до версий KDE Plasma 5.21.2, KDE Frameworksn 5.79.0 и KDE Gear (KDE Applications) 20.12.3. Обновлены приложения, в том числе Kdenlive 20.12.3, LibreOffice 7.1.1, Firefox 87.0. На основе темы оформления Lightly предложен новый стиль приложений KStyle, который заменил собой прошлую тему Kvantum и предлагает несколько вариантов декорирования окон. По умолчанию кнопки управления окном перенесены в левый верхний угол. Переработан интерфейс для предпросмотра тем оформления. Добавлены новые модули KCM (KConfig Module): Online Accounts и Software Updates. До версии 1.2.1 обновлён набор приложений на основе фреймворка Maui, предназначенного для разработки кроссплатформенных графических приложений. Добавлены новые приложения Shelf и Clip. Добавлено дополнение KIO Fuse, позволяющее из любых приложений обращаться к файлам на внешних хостах (SSH, SAMBA/Windows, FTP, TAR/GZip/BZip2, WebDav). KIO Fuse использует механизм FUSE для отражений внешних файлов в локальной ФС, что позволяет работать с удалёнными хранилищами не только из программ на базе KDE frameworks, но и из приложений на базе других фреймвроков, например, из LibreOffice, Firefox и приложений на основе GTK. Из поставки удалены mpv и qpdfviewer. На той же пакетной базе, что и основной выпуск, сформирована урезанная сборка (minimal ISO), размером 1.4 ГБ.
