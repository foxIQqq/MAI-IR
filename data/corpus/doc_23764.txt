Увидел свет релиз открытой СУБД Ingres 9.3

Анонсирован выход релиза свободной СУБД Ingres 9.3, имеющей почти 40-летнюю историю развития и распространяемой начиная с 2006 года в исходных текстах под лицензией GPL. Ingres является прародителем таких СУБД как MSSQL, Sybase и PostgreSQL (Post Ingres). В СУБД Ingres 9.3 представлены следующие новшества: Набор инструментов для миграции с MySQL и проприетарных систем, таких как Oracle, MS SQL Server и Sybase; Поддержка PAM (Pluggable Authentication Modules); Улучшенная поддержка средств для интеграции с языком Java (JDBC); Поддержка определенных в стандарте SQL 2003 идентифицирующих столбцов (identity columns) для автоматической генерации уникальных последовательностей (не обязательно через увеличение счетчика, значения могут генерировать и в неупорядоченном виде); Хранимые процедуры, возвращающие наборы значений (SET), теперь можно вызывать в SELECT запросах через поле FROM; В ODBC драйвере появилась поддержка перемещаемых вперед и назад курсоров (scrollable cursors), как для чтения, так и для записи данных; Выборку по маске с использованием конструкции LIKE теперь можно использовать и для столбцов, хранящих большие объемы данных (Long Objects).
