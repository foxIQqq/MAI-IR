Canonical прекращает поддержку облачного хранилища Ubuntu One и планирует открытие его кода

Компания Canonical приняла решение по закрытию облачного хранилища Ubuntu One и связанных с ним сервисов (синхронизация файлов, потоковое вещание музыки и т.п.). Работа Ubuntu One будет прекращена 1 июня 2014 года, после чего для зарегистрированных пользователей сервиса будет выделено ещё два месяца, в течение которых будет предоставлена возможность загрузки сохранённых данных. 1 августа 2014 года все данные будут удалены. Возможность покупки дополнительного дискового пространства в Ubuntu One прекращается с сегодняшнего дня. Остатки оплаченных в рамках готовой подписки средств будут возвращены пользователям. Поддержка Ubuntu One не войдёт в состав выпуска Ubuntu 14.04 и будет удалена из ещё поддерживаемых прошлых веток. В качестве причины закрытия Ubuntu One называется необходимость значительных инвестиций для поддержания сервиса на достойном уровне и невозможность конкурировать с другими бесплатными облачными хранилищами, которые предлагают 25-50 Гб бесплатного дискового пространства. Компания Canonical намерена не разрываться во все стороны, а сосредоточить имеющиеся ресурсы на основном стратегическом направлении - создании лучшей операционной системы для смартфонов, десктопов, планшетов и других устройств. Вместо попыток конкурировать в непрофильной области, развивая собственный сервис хранения, Canonical намерена заключить партнёрские соглашения по обеспечению доступа пользователей Ubuntu к лучшим сторонним облачным хранилищам. В настоящее время разработчики из Canonical работают над подготовкой инструментов для упрощения переноса контента из Ubuntu One на другие облачные хранилища. Более того, планируется открыть исходные тексты всех связанных с Ubuntu One клиентских и серверных компонентов, в том числе служб для организации синхронизации данных и обеспечения потокового вещания. Таким образом, у представителей сообщества появится возможность развернуть на своих мощностях аналог Ubuntu One.
