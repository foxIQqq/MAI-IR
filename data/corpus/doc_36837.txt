Релиз web-браузера Rekonq 2.3, развиваемого проектом KDE

Анонсирован релиз веб-браузера Rekonq 2.3, базирующегося на движке WebKit (QtWebKit) и технологиях KDE. Браузер Rekonq развивается в направлении построения полнофункционального web-браузера для KDE, который мог бы конкурировать с Konqueror и со временем занять место веб-браузера по умолчанию в KDE. Во многих дистрибутивах Linux, использующих окружение KDE, включая Kubuntu и Chakra, Rekonq уже используется по умолчанию. Среди добавленных в Rekonq 2.3 возможностей: В полноценном виде возвращена поддержка панелей для работы с закладками и историей работы; Добавление интерфейса для сохранения состояния текущего сеанса, загрузки сохранённых сеансов и их переименования. Добавлена опция для вывода диалога работы с сессиями при каждом новом запуске браузера; Изменено оформление адресной строки. Добавлена пиктограмма для отображения статуса SSL-соединения. Переработка строки ввода URL пока не завершена, например, отсутствует запланированная возможность подсветки в процессе ввода адреса; В стиле Opera реализован доступ к списку часто используемых сайтов (Favorites). Для помещения сайта в любимые ссылки и в закладки теперь используется единая пиктограмма с изображением звёздочки; Добавлена поддержка appmenu-qt, позволяющая организовать доступ к меню браузера через кнопку в заголовке окна; Доработана реализация контекстных меню в режиме Web View; Поддержка "комнат" (Activities) KDE - при открытии окон теперь учитывается привязка в комнатам; В приватном режиме запрещёно сохранение сессий и совместный доступ к cookie из разных вкладок с одним открытым сайтом; Поддержка открытия URL в режиме обособленного Web-приложения (не отображаются управляющие элементы интерфейса браузера). Базовые возможности Rekonq: Основанный на вкладках (табах) интерфейс; Реализация домашней страницы (аналог speeddial из Opera), на которой отображаются кнопки для открытия избранных сайтов, последние закрытые табы, история работы и закладки (см. скриншоты); Поддержка пользовательских сессий и восстановления открытых табов в случае аварийного закрытия браузера; Использование для загрузки файлов подсистемы загрузки из KDE; Возможность совместного использование закладок с Konqueror; Полная интеграция с KDE-подсистемой KIO, которая используется для получения настроек работы через прокси, хранения cookie; Режим анонимной навигации по сети; Поддержка инспектирования содержимого web-страниц. Поддержка чтения подписок в формате RSS; Возможность восстанавливать случайно закрытые вкладки; Легковесное оформление интерфейса при котором скрыта строка меню, панель вкладок перемещена в самый верх окна и упрощена адресная строка; Встроенная поддержка проверки правописания; Поддержка прикрепления вкладок (pinning tabs), как это сделано в Mozilla Firefox и Google Chrome; Режим WebApp, предназначенный для выполнения веб приложений, открываемых в отдельных окнах без каких-либо дополнительных элементов управления; Поддержка KWallet, позволяющая использовать сохраненные пароли совместно с Konqueror; Механизм AdBlock для блокирования назойливой рекламы; Режим управляемой загрузки контента, обрабатываемого плагинами (например, для просмотра Flash-ролика появляется кнопка "clickToFlash"); Поддержка синхронизации между несколькими машинами паролей, закладок и истории посещений; Поддержка выполнения web-приложений, каждое из которых выполняется в отдельном процессе с минималистичным интерфейсом без элементов управления (без панели управления, адресной строки и т.п.) в виде, приближенном к окнам локальных программ.
