Выпуск музыкального проигрывателя Exaile 3.4.0

Спустя почти два года с момента выпуска 3.3.0 опубликован релиз музыкального проигрывателя Exaile 3.4.0, истоки которого начинались с попытки создания аналога Amarok 1.4, написанного на языке Python с использованием GTK+ и GStreamer. Начиная с данного выпуска разработка, в том числе приём сообщений об ошибках и координация внесения изменений, перенесена с Launchpad на GitHub, вместо системы управления версиями Bazaar отныне будет использоваться Git. Код проекта распространяется под лицензией GPLv2. Бинарные сборки подготовлены для Linux (для Ubuntu доступен неофициальный PPA), Windows и OS X. Exaile может использоваться для управления музыкальной коллекцией, предоставляет удобный интерфейс на основе вкладок (разные списки воспроизведения могут быть открыты в разных вкладках), обеспечивает гибкую систему плагинов, предоставляет возможность прямой работы с портативными медиа-плеерами и поддерживает такие функции, как караоке, эквалайзер, нормализация громкости, прослушивание потокового сетевого радио, и автоматическое получение текстов песен и обложек альбомов через online-сервисы, в том числе Last.fm, Amazon и Wikipedia. Особенности нового выпуска: Добавлены "умные" списки воспроизведения, которые можно создавать на основе других списков воспроизведения с задействованием фильтров по разным типам тегов; Поддержка подсистем Disks/UDisks2 для работы с внешними накопителями в Linux; В интерфейс пользователя добавлена возможность отображения списков воспроизведения в нескольких панелях; Возможность свернуть левую панель со вкладками; Расширены возможности менеджера обложек, добавлена поддержка сохранения обложки на диск; Возможность переключения устройства вывода звука не прерывания воспроизведение композиции; Улучшение поддержки Unicode; Опция для управления наложением при попытке воспроизведения до окончания проигрывания другой композиции (полезно для DJ); Изменения, связанные с плагинами: Реализован упрощённый API для создания плагинов (поддержка старого API сохранена); Улучшена интеграция плагина BPM Counter (ручная установка BPM для композиции) в интерфейс; Потенциально проблемные плагины теперь не показываются в интерфейсе установки плагинов; Добавлены новые плагины: console, Icecast Radio, lyricsmania, mono output, playlist analyzer, Soma.FM; Удалены устаревшие плагины: lyricsfly, shoutcast; Официальная поддержка платформы OS X; Улучшена поддержка платформы Windows: сокращено время запуска, в инсталлятор добавлена поддержка 64-разрядных сборок Python, решены проблемы с перезапуском, улучшен плагин для поддержки мультимедийных клавиш.
