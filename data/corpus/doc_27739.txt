Вышел Python 2.6.6, последний корректирующий релиз в серии 2.6.x

Вышел релиз Python 2.6.6, последний корректирующий релиз ветки 2.6 в котором проведена работа по исправлению ошибок. Поддержка ветки Python 2.6 продлится до октября 2013 года, но отныне будут исправляться только уязвимости, обновлений с исправлением проблем не связанных с безопасностью для ветки Python 2.6 больше не будет. Пользователям рекомендуется постепенно спланировать переход на Python 2.7. В Python 2.6.6 исправлено около 140 проблем, из которых можно отметить избавление от нескольких утечек памяти и решение проблем с Unicode. Кроме того в коде исправлено 2 незначительные проблемы безопасности: проблема с расчетом длины строки в модуле audioop и возможность установки sys.argv без соответствующей правки sys.path в функции PySys_SetArgvEx.
