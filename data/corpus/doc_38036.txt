Зафиксирована атака на форум свободной игры 0 A.D., пользователям рекомендовано сменить пароли

Разработчики свободной игры 0 A.D., которая представляет собой стратегию реального времени с игровым процессом во многом похожим на игры серии Age of Empires, сообщили о попытках взлома форума проекта и захвата личной информации пользователей. Для предотвращения взлома и утечки информации в будущем выполнено обновление программного обеспечения (на сайте использовалась устаревшая версия IP.Board, потенциально содержащая уязвимости), проведены меры по усилению безопасности и задействовано шифрование для хранения персональных данных. Пользователям форума 0 A.D. настоятельно рекомендует изменить пароль, а если данный пароль используется на других сайтах, следует срочно изменить его на всех используемых ресурсах. Для сброса пароля можно использовать данную форму. Степень успешности проведённой злоумышленниками атаки не сообщается, но судя по опубликованным рекомендациям им удалось получить доступ к базе паролей пользователей форума.
