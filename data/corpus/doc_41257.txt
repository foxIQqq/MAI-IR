Выпуск Wine 1.7.33

Вышел очередной экспериментальный выпуск открытой реализации Win32 API - Wine 1.7.33. С момента выпуска версии 1.7.32 были закрыто 29 отчётов об ошибках. Наиболее важные изменения: В MSHTML добавлена поддержка помещения и вставки команд из буфера обмена; В DirectWrite добавлена поддержка дополнительных шрифтов; Улучшена обработка потери фокуса для окон Direct3D; Добавлены дополнительные DLL для API Sets; Закрыты отчёты об ошибках, связанные с работой игр и приложений: Starcraft II, OriginPro 8, Visual C++ 2010, Patrol Falcon, Ableton Live 9.1, Gears of War, Age of Wushu, Sketchup 2015, Propellerhead Reason 8, K-Lite.
