Релиз rdesktop 1.7

После трёх лет разработки увидел свет релиз клиента для удаленной работы по протоколу RDP на терминальных серверах Windows - rdesktop 1.7. Начиная с данного выпуска лицензия на код проекта изменена на GPLv3. Список изменений: Устранена уязвимость, позволяющая получить доступ к закрытым областям файловой системы через указание "/.." в пути; Поддержка Brush-кэша (кэширование битовых карт 8x8, используемых для заполнения областей экрана); Убран жестко прописанный в коде лимит на размер имени пользователя; Размер имени домена увеличен до 255 символов; Улучшена совместимость с PulseAudio/padsp; Добавлена поддержка новых указателей мыши; Устранены ошибки, мешающие работать на 64-разрядных системах; Добавлена поддержка RandR: возможность автоматически менять размер сессии, если используется относительный размер экрана; Улучшена поддержка Windows 2008 Session Broker Устранены проблемы с проигрыванием звука, рекурсивным удалением и перенаправлением дисков при использовании Windows 2008 R2 / 7.
