Вышел релиз web-фреймворка Yahoo! User Interface 3.1.0

Вышел релиз JavaScript-библиотеки Yahoo! User Interface (YUI) 3.1, предназначенной для создания пользовательского интерфейса в веб-приложениях. Пакет распространяется под лицензией BSD и состоит из базовой инфраструктуры (YUI, Node и Event) и набора утилит (Animation, IO и Drag&Drop). В новой версии произошли следующие основные изменения: Подготовлена компонентная инфраструктура, необходимая для реализации виджетов Internationalization utility - модуль для интернационализации приложений, позволяющий отделить языковые ресурсы от программного кода Обновлен виджет TabView, предназначенный для создания вкладок Новый модуль сортировки позволяет организовать сортируемые списки Загрузчик YUI 3 loader позволяет загружать любые сторонние модули, разработанные независимыми разработчиками
