Вышел релиз распределенной системы управления версиями Mercurial 1.4

Вышел релиз распределенной системы управления версиями Mercurial 1.4. Основные новшества: Добавлена команда summary для отображения суммарных данных о текущем состоянии репозитория; Реализован кэш тэгов, позволяющий значительно ускорить операцию с тэгами; Для команды diff добавлены опции "--stat" (отображение diffstat) и "--reverse"; В команде clone появилась опция -u/--updaterev; Устранены проблемы, приводящие к ошибке "broken pipe" при передаче больших объемов данных в момент выполнения команд push/pull; Исправлены ошибки в web-интерфейсе hgweb и добавлена возможность определения через файл конфигурации порядка вывода элементов директорий. Добавлена опция web.descend для отключения отображения всех связанных репозиториев; В комплект включен скрипт contrib/shrink-revlog.py для чистки некоторых внутренних структур БД, что приводит к заметному уменьшению размера репозитория; В расширение extdiff добавлена возможность отображения изменений между тремя различными слияниями кода; Добавлено расширение relink для установки хардлинков для идентичных файлов в клонированных репозиториях.
