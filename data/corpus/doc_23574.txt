Представлена новая открытая прошивка для mp3-плееров - Rockbox 3.4

Представлен релиз Rockbox 3.4, альтернативной прошивки для разнообразных моделей mp3-плееров производства Archos, iRiver, Apple (iPod), Cowon, Olympus, SanDisk, Toshiba, Samsung, Philips. Rockbox распространяется под лицензией GPL и может быть загружен с карты памяти, без удаления основной прошивки плеера. Основные новшества релиза Rockbox 3.4: Добавлен Pitch Detector, плагин для выделения основного тона, что может быть использовано для точной настройки инструментов; На LCD дисплее теперь можно размещать viewport-ы произвольного размера; Поддержка проигрывания RealMedia файлов, содержащих аудио-данные в форматах ATRAC3, AC3, AAC и Cook. Общий список поддерживаемых кодеков представлен на данной странице; Добавлен плагин для отображения PNG картинок на цветных LCD-дисплеях; Добавлен плагин с игрой Clix (клон игры Same Game); Увеличена стабильность внутренней базы данных; Добавлена функция калибровки сенсорных экранов; Возможность раздельного регулирования скорости воспроизведения и тональности; Добавлено средство для удаленного управления музыкальным проигрывателем на ПК через подлюченный к USB-порту плеер; По умолчанию задействован альтернативный алгоритм сортировки файлов, учитывающий цифры в именах;
