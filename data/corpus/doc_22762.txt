Вышел четвертый тестовый выпуск openSUSE 11.2

Вышел четвертый тестовый выпуск openSUSE 11.2. Релиз openSUSE 11.2 намечен на ноябрь. По сравнению с прошлым тестовым выпуском отмечены следующие изменения: Образ LiveCD теперь может быть использован и на USB Flash; В состав LiveCD добавлен Midnight Commander, а в KDE-вариант LivCD программа YaKuake; Для YaST создан новый центр управления конфигурацией (YaST Control Center) с интерфейсом на базе Qt4; Пакеты пересобраны с использованием gcc 4.4.1; Обновление версий программ: Linux kernel 2.6.30.2 CUPS 1.3.10 NetworkManager 0.7.1 Samba 3.4.0 Firefox 3.5.1 Bluez 4.45 KDE 4.3 RC3 Qt 4.5.2 Qt Creator 1.2.1 Seamonkey 2.0b1 VirtualBox 3.0.2 Базовые новшества openSUSE 11.2: Пакеты дистрибутива собраны GCC 4.4.x В качестве файловой системы по умолчанию задействована Ext4 В состав Live CD включен Gimp; Для сжатия данных на LiveCD отныне используется алгоритм сжатия LZMA; По умолчанию LiveCD содержит переводы для немецкого, французского, итальянского, польского и русского языков; В Qt версии конфигуратора YaST появилась поддержка настраиваемого внешнего вида. В YaST добавлены средства работы с кешем пакетов; В Zypp улучшены средства работы с зеркалами репозиториев пакетов; В интерфейсе для настройки параметров дисплея gnome-display-properties добавлена поддержка автоматического сброса настроек в случае проблем. Ведется разработка web-интерфейса к YaST для осуществления удаленного управления дистрибутивом Появление установочных образов для USB Flash и утилита для их подготовки Улучшение поддержки субноутбуков
