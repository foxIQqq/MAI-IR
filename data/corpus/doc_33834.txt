Для Linux подготовлен открытый драйвер для сенсорных панелей Microsoft Surface 2.0

Один из энтузиастов разработал открытый драйвер для сенсорных панелей Microsoft Surface 2.0. Драйвер функционирует на пользовательском уровне и использует для взаимодействия с панелью libusb. Код драйвера и демонстрационное приложение опубликованы на GitHub. Для работы требуется загрузка Linux на панели Surface 2.0, который можно загрузить c USB Flash не затрагивая установленную на панели Windows 7. С точки зрения аппаратного обеспечения панель представляет собой обычный ПК, на котором успешно был использован Ubuntu 11.10. При загрузке Ubuntu всё оборудование успешно поддерживалось из коробки, за исключением сенсорного экрана, для которого и был написан представленный драйвер. В текущем состоянии драйвер позволяет только отображать информацию, но не поддерживает передачу событий в систему через задействование фреймворка utouch. Поддержка ввода пока реализована только в специальном демонстрационном приложении. Тем не менее, на следующей неделе планируется написать специальную прослойку для uinput, которая обещает задействовать в системе возможность ввода и мультитач.
