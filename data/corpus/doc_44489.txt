Ubuntu MATE переходит на GTK3+

Разработчики дистрибутива Ubuntu MATE сообщили о решении отказаться от сборок с GTK2+ и поставлять в составе Ubuntu MATE 16.10 только GTK3+. Привязанные к GTK2+ устаревшие библиотеки будут удалены из состава дистрибутива. Параллельно уже около месяца ведётся работа по переводу на GTK3+ пакетов c MATE для Debian. Большая часть компонентов уже адаптирована для работы с GTK3+, из пока нерешённых проблем упоминается несовместимость MATE Menu с GTK3+, а также проблемы при использовании апплетов MATE Indicator, MATE Dock и Topmenu в окружении на базе GTK3+. Отдельно отмечается заброшенность GNOME Main Menu, который уже не сопровождается и может работать только с GTK2+. GNOME Main Menu скоро будет удалён из репозиториев Debian и Ubuntu, так как в проекте MATE не нашлось разработчиков, заинтересованных в его портировании на GTK3+.
