Выпуск свободного математического пакета OpenOpt Suite 0.51

Вышел новый ежеквартальный релиз свободного математического кросс-платформенного пакета OpenOpt Suite 0.51, написанного на языке Python. Главные изменения: Улучшение автоматического дифференцирования в FuncDesigner, в том числе для моделирования квадратичных задач оптимизации (MI)(QC)QP; Подключены новые солверы: knitro (NLP), Octave QP solver, MATLAB linprog (LP), quadprog (QP), lsqlin (LLSP), bintprog (MILP); Добавлено несколько элементов интервального анализа второго порядка (квадратичными функциями), некоторые другие улучшения для интервального солвера interalg (для решения задач с требуемой точностью); Новые классы задач: knapsack problem (KSP), bin packing problem (BPP), dominating set problem (DSP); FuncDesigner теперь может моделировать задачи SOCP; SpaceFuncs (геометрические расчеты) приведён в соответствие к последним версиям Python и NumPy;
