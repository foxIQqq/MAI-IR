Новая версия аудиопрогрывателя mpg123 1.10.0

Доступна новая версия консольного аудиопроигрывателя mpg123 1.10.0. Из изменений можно отметить: В интерфейсе управления реализована команда "pitch" для регулирования высоты тона; Устранены проблемы сборки. Исправлены проблемы, мешающие работе на платформе Windows (проблема с именами файлов в UTF-8, поддержка опции "--realtime", добавлены файлы сборки для Visual Studio 2010 Beta 2). Возобновлена возможность запуска в OS/2; В комплект включен mpg123clr, враппер для .NET; Добавлено API для покадрового управления звуковым потоком;
