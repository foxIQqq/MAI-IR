Доступна для тестирования финальная бета-версия Ubuntu 13.04

Вышел финальный бета-выпуск Ubuntu 13.04 (Raring Ringtail). Релиз намечен на 25 апреля. Представленная бета-версия ознаменовала собой подготовку тестовых установочных iso-образов Ubuntu и Ubuntu Server, которые формировались только для ежедневных сборок, промежуточные альфа- и бета-сборки были отменены. Кроме того, сформирована первая тестовая сборка нового официального ответвления - Ubuntu GNOME, поставляемого со штатным десктоп-окружением GNOME. Готовые тестовые образы также созданы для Edubuntu, Kubuntu, Lubuntu, UbuntuKylin (версия для пользователей из Китая), Ubuntu Cloud, Ubuntu Studio и Xubuntu. Среди основных новшеств Ubuntu 13.04 можно отметить: Задействовано ядро Linux 3.8; Система инициализации Upstart обновлена до версии 1.8. В качестве экспериментальной возможности добавлена поддержка управления сеансами пользователей (режим Session Init), появившаяся в Upstart 1.7 и позволяющая использовать Upstart для инициализации графических сеансов. По умолчанию данная возможность будет отключена (для включения нужно раскомментировать строку "ubuntu" в файле /etc/upstart-xsessions); В базовом составе дистрибутива решено поставлять только Python 3.3 (Python 2 остаётся в репозиториях); Пользовательская оболочка Unity обновлена до версии 6.12, отличающейся заметно возросшей производительностью. Из изменений в интерфейсе можно отметить новую реализацию меню для синхронизации данных с использованием сервиса Ubuntu One, новые диалоги завершения работы, новые визуальные эффекты (например, в процессе перемещения окна за пределы экрана в призрачной форме указывается на область, которую приложение займёт при отпускании кнопки мыши), новые пиктограммы панели и линз, упрощённый режим перебора из панели нескольких открытых окон приложения (для перебора окон можно использовать скроллинг мышью на пиктограмме приложения), новые настройки панели (например, скрытие иконки с виртуальными экранами, добавление кнопки сворачивания всех окон). В Kubuntu отмечается обновление компонентов KDE до версии 4.10.2, поставка центра управления приложениями Muon Suite 2, обновление браузера Rekonq до версии 2.2.1, интеграция полноэкранного меню приложений Homerun (см. скриншот ниже), новая система настройки многомониторных конфигураций, поставка шрифтов Oxygen Sans, переработка интерфейса инсталлятора, поддержка доступа к мобильным устройствам по протоколу MTP (используется в Android 3+ вместо эмуляции USB-накопителя), поддержка загрузки в режиме UEFI Secure Boot, поставка OpenJDK7 в качестве реализации Java по умолчанию; В Xubuntu произведено увеличение размера iso-образа до 850 Мб, в поставку возвращены Gnumeric и GIMP, произведено обновление визуальной темы; В Ubuntu Studio обновлено окружение Xfce 4.10 и задействован графический инсталлятор Ubiquity. Добавлены новые программы для работы со звуком supercollider, faust, lmms; программы для работы с видео: denlive и recordmydesktop; графические пакеты: krita и create-resources; В Ubuntu GNOME задействовано десктоп-окружение GNOME 3.6. Выпущенный недавно релиз GNOME 3.8 доступен через PPA-репозиторий, но его решено не включать в релиз, так как данный выпуск недостаточно протестирован и приводит к возникновению ряда проблем в процессе использования. По сравнению с Ubuntu GNOME Remix 12.10 в Ubuntu GNOME по умолчанию задействован Firefox вместо Epiphany, Ubuntu Software Center и Update Manager вместо конфигураторов GNOME, LibreOffice вместо Abiword и Gnumeric. Дополнительно можно упомянуть, что не все возможности, запланированные для Ubuntu 13.04, попадут в релиз. Например, решено отложить до осеннего выпуска оболочку Unity 7 c реализацией умных областей (Smart Scopes) и средств для управления приватной информацией. Smart Scopes представляют собой серверные обработчики, позволяющие представлять в Dash определённые виды локальной или внешней информации с определением порядка вывода областей с учётом их релевантности и востребованности пользователем. Cредства для управления приватной информацией позволят более точно контролировать какие сервисы использовать, например, определить фильтры для локального и глобального поиска (таким образом, в Ubuntu 13.04 как и в прошлом выпуске будет можно либо разрешить, либо запретить online-поиск целиком). Также решено прекратить поставку WUBI, программы для установки Ubuntu из Windows, так как в ней не реализована поддержка Windows 8 и наблюдается ряд серьёзных недоработок.
