Лучшие открытые проекты по мнению читателей журнала Linux Journal

Издание Linux Journal опубликовало результаты опроса предпочтений читателей. В опросе приняло участие около 5 тыс. респондентов, которые выбрали лучшие по их мнению продукты и сервисы для 36 тематических категорий. В результате были определены следующие лучшие продукты: Дистрибутив: Ubuntu; Десктоп окружение: GNOME; Web-браузер: Firefox; Программа для чтения электронной почты: Mozilla Thunderbird; Офисный пакет: OpenOffice.org; Программы для обработки звука: Audacity; Аудиоплеер: Amarok; Медиа плеер: VLC (в прошлом году лидером был MPlayer); Средство для общения: Pidgin; Графический пакет: GIMP; ПО для работы с фотографиями: Picasa (лидер прошлого года - digiKam); Текстовый редактор: vi; Система контроля версий: Subversion; СУБД: MySQL; Система мониторинга: Nagios; Язык для создания скриптов: Bash; Язык программирования: Python (в прошлом году победил Си); GUI удаленного доступа или средство работы в сети: SSH и X; Интегрированная среда разработки: Eclipse; Платформа для создания интерактивных web-приложений (RIA): Adobe Air; Игровое приложение: Frozen Bubble; Система виртуализации: VirtualBox (лидер прошлого года - VMware); Системы резервного копирования: Amanda (лидер прошлого года - Simple Linux Backup); Утилиты резервного копирования: rsync (в прошлом году победил tar); Управление пакетами программ: Apt; Система управления контентом: WordPress; HTTP-сервер: Apache; Ноутбук на базе Linux: ASUS Eee PC; Гаджет на базе Linux: ASUS Eee PC (в прошлом году - Nokia N8x0); Производитель десктоп машин с Linux: Dell; Производитель Linux серверов: Dell; Энергосберегающее средство: системы виртуализации; Продукт года: T-Mobile G1 на базе платформы Android (в прошлом году - ASUS Eee PC);
