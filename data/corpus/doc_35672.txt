Доступна редакция дистрибутива Linux Mint 14 с десктоп-окружением Xfce

Вышла Xfce-редакция дистрибутива Linux Mint 14.0. Дистрибутив основан на пакетной базе Xubuntu 12.10 и поставляется с десктоп-окружением Xfce 4.10, дополненном серией оригинальных приложений, упрощающих настройку и работу в системе. В частности, в Linux Mint используется переработанная система меню, собственный менеджер установки и обновления приложений, оригинальный интерфейс для настройки системы mintConfig, средство для выполнения резервного копирования mintBackup и т.п. Размер загрузочного iso-образа составляет 872 Мб. Дистрибутив может работать на системах с 384 Мб ОЗУ, но для комфортной работы рекомендуется 1 Гб оперативной памяти. Ключевые особенности Linux Mint 14 "Xfce": В базовую поставку входят свежие приложений Firefox, Thunderbird, LibreOffice, Gimp, Banshee, Pidgin, Xchat. В качестве файлового менеджера используется Thunar, для записи CD/DVD - Xfburn, для просмотра изображений - Ristretto, для управления Bluetooth-устройствами - Blueman. Для вывода уведомлений задействован демон xfce4-notifyd; В качестве дисплейного менеджера задействована свежая версия MDM (форк GDM 2.20, развиваемый в рамках проекта MATE), в которой добавлена поддержка классических тем оформления GDM 2, в поставку включена подборка из 30 подобных визуальных тем. Улучшена организация вывода списка пользователей и ассоциированных с ними картинок. Отныне вместо ввода имени можно сразу выбрать пользователя по его имени или изображению. Улучшен интерфейс переключения между пользователями, поддержка которого интегрирована в хранитель экрана (в режиме блокировки выводится кнопка "Switch user..."); Переработаны некоторые внутренние подсистемы менеджера установки приложений. Вместо процесса aptdaemon, из-за которого наблюдались крахи и зависания, теперь используется собственный APT-клиент. Добавлена полноценная поддержка debconf, благодаря которой отпала необходимость использования Synaptic для установки использующих debconf пакетов с диалогами по настройке, таких как пакеты с шрифтами Microsoft или Wine. Вместо требования ввода пароля в перед началом установки или обновления менеджер установки приложения теперь сразу запускается под пользователем root (пароль вводится один раз). Добавлены настройки режима поиска по мере ввода; Серия системных изменений: Вместо USB-ImageWriter задействован MintStick, отличающийся более удобным пользовательским интерфейсом. Редактор Gedit 2.30 заменён на Gedit 3, который поддерживает больше возможностей и более стабилен; В MintSystem добавлены дополнительные команды: dns-fix для задания собственной привязки имени хоста в OpenDNS, xchat-systray для добавления постоянной иконки Xchat на системный лоток.
