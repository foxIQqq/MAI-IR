Компания Oracle выпустила средство для синхронизации своей СУБД с SQLite

Компания Oracle представила новый выпуск коммерческого продукта Oracle Database Lite, предназначенного для организации работы базы данных на ограниченных по ресурсам встраиваемых или мобильных устройствах, синхронизированной с сервером на основе промышленной СУБД Oracle. В качестве одного из основных новшеств называется возможность двунаправленной синхронизации между БД SQLite и централизованной СУБД Oracle, поддерживаемая наравне с собственной реализацией минималистичной SQL-совместимой БД.
