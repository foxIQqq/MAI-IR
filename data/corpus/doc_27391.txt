Представлена открытая платформа для размещения видео в сети - Kaltura 2.0

Увидел свет релиз открытой платформы Kaltura CE 2.0, предназначенной для организации работы сервера для потокового видеовещания в сети, поддерживая при этом все возможные конфигурации web-браузеров на стороне клиента (HTML5 video, Flash, Silverlight). Код платформы распространяется в рамках свободной лицензии AGPLv3. Преимущества Kaltura: Гибкие возможности по изменению оформления через CSS и специальные скины; Поддержка подключения расширяющих функциональность JavaScript-модулей (например, есть модули для редактирования видео и для локального перекодирования); Доступность готовых плагинов для популярных систем управления контентом (Drupal, Joomla, Moodle, Sakai, Blackboard, Wordpress) и wiki-движка MediaWiki; 100% свободность разрабатываемых в рамках проекта технологий. Набор инструментов для управления видеоколлекцией, публикации материала, мониторинга, синдикации контента и монетизации видеосервиса. Ключевые улучшения версии 2.0: Средства видео-аналитики; Поддержка контроля доступа и планировки; Инструменты для синдикации данных с внешних ресурсов и SEO-оптимизации; Профили с адаптивной пропускной способностью и транскодированием потоков; В пакет интегрирован oline-реактор видео, модуль для организации дерева категорий контента и Kaltura Player; Новый модуль для пакетной обработки ( batch processing); В административную консоль добавлена поддержка работы нескольких администраторов и издателей контента, возможность контроля пакетной обработки и средства мониторинга; Из новых возможностей, требующих дополнительных компонентов, использования сетей доставки контента (CDN) или дополнительного лицензирования, отмечены: доставка RTMP трафика, адаптивное изменение битрейта, средства аналитики по утилизации пропускной способности CDN и географии посетителей, поддержка записи с web-камер, ограничение доступа в зависимости от местоположения, движок для показа рекламы. Из проектов, в которых на базе Kaltura реализовано видеовещание, можно отметить Wikipedia и html5video.org. Видео в данных проектах оформлено в формате Theora, но благодаря использованию Kaltura трансляция работает во всех современных web-браузерах. Возможность работы в Internet Explorer и Safari достигается через технику эмуляции HTML5-тега video и прозрачного перекодирования Theora-потока в поддерживаемый браузером тип видео (H.264, MOV, FLV), используя на стороне клиента специальное Flash-приложение или Java-апплет.
