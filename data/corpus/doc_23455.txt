Вышел Calculate Linux Scratch 9.9

Вышла новая версия Calculate Linux Scratch (CLS) - специализированного livecd, предназначенного для сборки системы под любые задачи, с возможностью самостоятельного выбора включаемых пакетов. Версия Calculate Linux Scratch GNOME (CLSG) включает урезанную версию графической среды Gnome. Основные изменения: Переход на файловую систему Aufs2. Ядро и загрузочный initrd образ запакованы в lzma формат. Переход на ядро gentoo-sources. Новая утилита cl-unmask для демаскировки пакетов. В CLS по умолчанию включена параллельная загрузка сервисов. В CLSG исправлена ошибка при установке из GNOME приводящая к ошибке "Resource is busy".
