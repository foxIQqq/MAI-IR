Доступен Ora2Pg 12.0, инструмент для переноса БД из Oracle в PostgreSQL

Представлен новый выпуск программы Ora2Pg, предназначенной для автоматизации миграции баз данных из СУБД Oracle в PostgreSQL. Утилита подсоединяется к Oracle, автоматически извлекает все данные, связанные с выбранной БД, на лету преобразует структуру и объекты БД (таблицы, представления, перечисляемые последовательности, триггеры, индексы, хранимые процедуры, привязки к внешним таблицам, блобы, группы, функции назначения привилегий и т.п.), и генерирует SQL-файл, пригодный для создания аналогичной базы в PostgreSQL. Преобразование кода PL/SQL в PL/PGSQL требует выполнения ручной работы, но Ora2Pg предлагает специальный помощник для упрощения конвертации. В новой версии добавлена поддержка экспорта материализованных представлений с использованием аналогичной функциональности из PostgreSQL 9.3, обеспечен экспорт таблиц и индексов с учётом табличных пространств (tablespace), реализована автоматическая перегруппировка столбцов для минимизации занимаемого места на диске. Код Ora2Pg распространяется под лицензией GPLv3.
