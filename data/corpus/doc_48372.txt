Intel и Linux Foundation представили Akraino Edge Stack

Организация Linux Foundation представила новый совместный проект Akraino Edge Stack, в рамках которого будет развиваться открытый программный стек с реализацией технологии Edge Сomputing. В качестве основы новой платформы будут использоваться продукты Wind River Titanium Cloud Software и Intel Network Edge Virtualization SDK, которые были переданы под покровительство Linux Foundation компанией Intel, а также наработки компании AT&T по созданию edge-приложений для виртуальных машин. К рабочей группе по развитию нового стека уже подключились компании Altiostar, China Electronics Standardization Institute (CESI), China Mobile, China Telecom, China Unicom, Docker, AT&T, Huawei, iFlyTek, New H3C Group, Tencent, ZTE и 99Cloud. Edge Сomputing предлагает новый метод построения облачных вычислений в распределённых инфраструктурах, имеющих большое число территориально разнесённых объектов, генерирующих большой объём данных. В отличие от традиционных облачных систем, вычислительные ресурсы в которых сосредотачиваются в одном месте и узлы находятся в одной локальной сети, в Edge Сomputing серверы, на которых производится запуск виртуальных машин с приложениями, выносятся на периферию как можно ближе к источникам данных и могут обеспечить их обработку в режиме реального времени без задержек на передачу запросов в централизованное облако. Таким образом, вместо отправки данных в облако, вся мощность которого сосредоточена в одном ЦОД, узлы этого облака разнесены рядом с источникам данных или потребителями услуг. Подобный подход востребован при организации обработки данных от датчиков и IoT-устройств, для максимально быстрой обработки критических данных, при реализации концепции виртуализации компонентов сетей (NFV, Network Functions Virtualization), а также при развёртывании сервисов в сотовых сетях нового поколения (Mobile Edge Computing - обработчики предлагается запускать не централизовано, а поблизости от базовых станций, чтобы исключить задержки от передачи данных в ЦОД). В качестве основы Akraino Edge Stack выступила платформа Wind River Titanium Cloud Software, в которой уже применяются типовые открытые компоненты: управление облаком построено на базе OpenStack, для хранения данных задействовано распределённое хранилище Ceph, а на вычислительных узлах применяется Linux с гипервизором KVM и виртуальным коммутатором Open vSwitch. Средства для разработки приложений, работающих в виртуальных машинах и контейнерах в облаках Akraino Edge, планируется построить на базе наработок, ранее открытых компанией AT&T и переданных в Linux Foundation. Проекту также передан Intel Network Edge Virtualization SDK, который предоставляет эталонный набор библиотек и API для создания различных сетевых решений, работающих в облаке Wind River Titanium.
