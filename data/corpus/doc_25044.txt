Вышел QEMU 0.12.2 и представлен план развития QEMU 0.13

Вышел очередной корректирующий релиз эмулятора QEMU 0.12.2, в котором исправлено 32 ошибки. Дополнительно опубликован обзор новшеств будущего релиза QEMU 0.13, выход которого ожидается 1 июня: Поддержка gPXE в virtio-blk; Улучшенная система настройки параметров сети; Поддержка вывода статистики в virtio-balloon драйвере; Полная поддержка QMP (QEMU Monitor Protocol); Поддержка протокола для обеспечения Live-миграции для сторонних ответвлений; Увеличение стабильности драйвера eepro100; Создание режима интерпретатора для tcg (Tiny Code Generator); Подготовка инсталлятора для платформы Windows; Обеспечение полного соответствия требованиям PCI при пробросе устройств в виртуальное окружение через virtio; Поддержка технологии VEPA (Virtual Ethernet Port Aggregator); Принятие Xen qemu-dm патчей; kvm irqchip; virtio-console; vhost-net; p35 + pcie.
