Релиз десктоп-окружения Cinnamon 2.2, продолжающего традиции GNOME 2

После пяти месяцев разработки увидел свет релиз пользовательского окружения Cinnamon 2.2, в рамках которого сообществом разработчиков дистрибутива Linux Mint развивается форк оболочки GNOME Shell, файлового менеджера Nautilus и оконного менеджера Mutter, нацеленный на предоставление окружения в классическом стиле GNOME 2 c поддержкой удачных элементов взаимодействия из GNOME Shell. Cinnamon основывается на компонентах GNOME, но эти компоненты поставляются как периодически синхронизируемый форк, не связанный внешними зависимостями с GNOME. Установочные пакеты подготовлены для Linux Mint, Ubuntu, Debian, Fedora, Arch Linux, Gentoo, openSUSE и ALT Linux. Cinnamon 2.2 будет поставляться в составе выпуска дистрибутива Linux Mint 17, запланированного на конец мая. Основные новшества Cinnamon 2.2: Обновлён графический интерфейс системного конфигуратора. Осуществлена оптимизация разделения настроек по категориям и выполнено выделение отдельных субсекций, что позволило обеспечить постоянную доступность всех настроек, избавившись от разделения нормального и расширенного режимов настройки. Полностью переделана организация настройки управления питанием и хранителем экрана, которые теперь явно разделены для избежания путаницы. Улучшены средства для настройки яркости экрана; Удалён модуль для осуществления региональных настроек, оставлен только интерфейс для конфигурации раскладки клавиатуры, который выделен в отдельный модуль. Для установки параметров локали рекомендуется использовать штатные универсальные инструменты дистрибутивов; Возвращён оригинальный модуль для настройки даты и времени. Добавлен режим отображения 24-часового формата вывода времени в апплетах, календаре и хранителе экрана; Интерфейс для прикрепления постоянно видимого (HUD) окна к краю экрана стал менее навязчивым и теперь появляется только при перемещении границ окна непосредственно к краю экрана; Добавлен конфигуратор для настройки действий при попадании курсора в каждый из углов экрана. Возможна раздельная привязка действий как к клику в горячем углу, так и к наведению курсора на угол; Реализована возможность привязки к апплетам ролей, позволяющих передать данные о функциональности апплета, что используется для автоматического скрытия дублирующих пиктограмм системного лотка, функции которых пересекаются с реализованными в апплете. Например, при удалении штатного апплета настройки сети будет показана пиктограмма Network Manager, а при возвращении штатного апплета настройки сети, пиктограмма Network Manager будет скрыта. Ранее список перекрываемых Cinnamon-апплетами приложений, таких как Network Manager и Banshee, был жестко вшит в код; В апплет с реализацией меню приложений добавлены две возможности из mintMenu: возможность инициировать удаление программы (Uninstall) через контекстное меню, показываемое при клике правой клавишей мыши, и подсветка в меню недавно установленных программ; В апплет управления звуком добавлена поддержка взаимодействия со сторонними медиапроигрывателями, поддерживающими стандарт MPRIS. MPRIS-проигрывателем также можно управлять через мультимедийные клавиши. В звуковой апплет добавлена возможность настройки скрытия пиктограммы из системного лотка для тех или иных мультимедиа проигрывателей; Поддержка экранов сверхвысокого разрешения (HiDPI/Retina); Возвращён плагин Wacom с поддержкой графических планшетов; Расширены возможности файлового менеджера Nemo, в рамках которого развивается форк Nautilus 3.4 в котором сохранены такие возможности, как панель инструментов и меню, двухпанельный режим, меню со ссылками для быстрого перехода, компактная форма отображения списка файлов и боковая панель. В новой версии добавлена возможность отображения в боковой панели недавно открытых позиций в файловой системе. Для переключения между вкладками теперь можно использовать комбинации ctrl+tab и ctrl+shift+tab. В панель добавлена кнопка для создания новых директорий; Обеспечен режим интерактивного масштабирования, позволяющий увеличивать области экрана через нажатие Alt и перемещение колеса прокрутки мыши. В оконном менеджера реализована функция управления степенью затенения или прозрачности окна через перемещение колеса прокрутки мыши при нахождении курсора в заголовке окна. Опция позволяет быстро оценить что творится в перекрытой окном области без переключения окон. Функция включается в модуле "Windows"; Полная поддержка декорирования окон на стороне клиента; Улучшена интеграция с установленным на той же машине окружением GNOME (в Cinnamon больше не отображается GNOME Control Center, а в GNOME - Cinnamon Settings); Улучшена поддержка systemd-logind.
