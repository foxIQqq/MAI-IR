Релиз Linux-платформы MeeGo 1.0

Представлен первый релиз Linux-платформы MeeGo, появившейся в результате объединения проектов Maemo и Moblin. В отличие от первого тестового выпуска, содержащего только базовый костяк платформы без графического интерфейса, в MeeGo 1.0 представлена полнофункциональная графическая оболочка, прекрасно адаптируемая для смартфонов, нетбуков, интернет-планшетов и стационарных ПК. В текущем виде MeeGo 1.0 позиционируется как стабильная основа для разработки конечных приложений и совершенствования интерфейса пользователя. В настоящий момент для загрузки представлена только версия для нетбуков и базовая сборка для смартфонов Nokia N900 (без графического интерфейса). Работа над вариантом для карманных устройств начнётся в июне. В будущем новые релизы будут выпускаться в рамках фиксированного 6-месячного цикла разработки, например, версия MeeGo 1.1 выйдет в октябре и будет включать в себя поддержку устройств, для управления которыми используется только сенсорный экран, таких как наладонники, планшеты и автомобильные информационно-развлекательные системы. Одновременно с базовой платформой представлен комплект MeeGo SDK и программный интерфейс MeeGo API, основанные на использовании Qt 4.6 и предназначенные для сторонних разработчиков приложений. В текущем виде MeeGo SDK пока ограничивается средствами для разработки программ для нетбуков, но в следующей версии, первый тестовый выпуск которой выйдет в июне, SDK будет расширен поддержкой карманный устройств и планшетов. Особенности сборки для нетбуков: Удобный и визуально насыщенный графический интерфейс пользователя, основанный на последних открытых технологиях; Мгновенный доступ на домашнем экране к синхронизированному календарю, списку задач, напоминаниям, последним используемым файлам, информации из социальных сетей; Средства для агрегирования контента социальных сетей, позволяют на одном экране наблюдать за активностью в социальных сетях, общаться с друзьями, устанавливать статусы. В качестве web-браузера предустановлен Google Chrome. В версии для карманных устройств (MeeGo Handset) будет использован Mozilla Fennec (Firefox Mobile); Подборка простых в использовании программ для работы с электронной почтой, календарем и мультимедиа контентом; Оптимизация для обеспечения максимальной производительности и минимального энергопотребления; Наличие русской локализации; Поддержка нетбуков: Asus EeePC 901/1000H/1005HA/1008HA/1005PE, Eeetop ET1602, Dell mini10v, Inspiron Mini 1012, Acer Aspire One D250, AO532-21S, Revo GN40, Aspire 5740-6025, Lenovo S10, MSI U130, AE1900, HP mini 210-1044 и Toshiba NB302. Особенности базовой каркасной сборки (Core Software Platform): Использование Linux-ядра 2.6.33 и Qt 4.6; Поддержка платформ: нетбуки и автомобильные устройства на базе CPU Intel Atom, Nokia N900; Организация взаимодействия с оборудованием через DeviceKit и udev; Современный графический 2D/3D стек, поддерживающий переключение видеорежимов на уровне ядра (KMS) и не требующий для работы X-сервера привилегий суперпользователя; Для обеспечения функций голосовой связи и передачи данных задействованы телефонный стек oFono и система конфигурирования ConnMan; В платформе используется: Bluetooth-стек BlueZ, управляющая шина D-BUS, мультимедийный фреймворк GStreamer и звуковой сервер PulseAudio; Поддержка Universal Plug and Play (gUPnP); В качестве файловой системы по умолчанию используется Btrfs.
