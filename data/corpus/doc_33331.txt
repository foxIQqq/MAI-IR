Релиз Midori 0.4.4, легковесного веб-браузера на базе движка WebKit

Представлен релиз web-браузера Midori 0.4.4, разрабатываемого участниками проекта Xfce. Midori построен на базе движка WebKit и библиотеки GTK+. В браузере реализованы следующие возможности: поддержка табов, закладки, управление сессиями, панель быстрого доступа к поисковым системам, средства для создания пользовательских меню, возможность использования пользовательских скриптов и стилей, средства для фильтрации рекламы (Adblock). Также поддерживаются внешние расширения, написанные на языках Си, Vala и Lua. Из изменений можно отметить: Улучшена поддержка GTK+ 3. К сожалению пока остаётся несколько нерешенных проблем, что мешает привести поддержку GTK+ 3 к уровню GTK+2. Для сборки с GTK+ 3 следует использовать опцию "--enable-gtk3"; Улучшена поддержка меню пользовательской оболочки Unity, в том числе решены ранее наблюдаемые проблемы из-за которых браузер был прописан в чёрном списке menuproxy; В качестве поисковой системы по умолчанию задействован поисковый сервис Duck Duck Go, отличающийся блокированием рекламы и вырезанием всех лишних данных, которые могут использоваться для отслеживания предпочтений и перемещений пользователя. По отношению к другим поисковым системам Midori перешёл к практике заключения соглашений о выплате отчислений от показа рекламы пользователям, использующим поисковую форму браузера; Поддержка экспериментального FullScreen API для отображения видео в полноэкранном режиме; Решены проблемы с удалением и экспортом закладок; Реализована функция открытия картинки во внешней программе для просмотра изображений (меню 'Open in Image Viewer'); Улушена поддержка клавиши "=" при использовании украинской раскладки клавиатуры; На экране быстрого перехода "Speed Dial" добавлена поддержка изменения порядка элементов через перетаскиванием мышью в режиме "drag and drop"; При использовании режима "инкогнито" теперь выводится дополнительный пояснительный текст, предупреждающий что не стоит слепо полагать, что пользователь остаётся анонимным; При работе на системах с ОЗУ менее 353 Мб, автоматически отключается кэширвоание страниц в ОЗУ для экономии памяти; Поддержка ведения белого и чёрного списков Cookie (пока только базовая поддержка без пользовательского интерфейса) и задаваемых сайтами политик в отношении хранения данных на стороне пользовтеля; Расширен диалог очистки персональных данных (Clear Private Data); В диалог загрузки добавлено отображение имени файла. Для работы Midori среди обязательных зависимостей присутствуют: WebKitGTK+ 1.1.17 и Vala 0.10, GTK+ 2.10, libXML2, libsoup 2.25.2 и sqlite 3.0; среди опциональных зависимостей: Unique 0.9, libidn, libnotify. Установить новую версию Midori в Ubuntu можно воспользовавшись специальным PPA-репозиторием.
