Компания Microsoft объявила о покупке GitHub за 7.5 миллиардов долларов

Компания Microsoft официально объявила о заключении соглашения по покупке сервиса совместной разработки GitHub за 7.5 миллиардов долларов, что больше на 2.5 милларда долларов, чем прогнозировали аналитики. Сделку планируется завершить до конца года, после согласования с антимонопольными службами США и Евросоюза. Потенциально согласование может вызвать проблемы, так как Microsoft предлагает систему совместной разработки Visual Studio Team Services, конкурирующую с GitHub. Всего три года назад GitHub оценивался в 2 миллиарда долларов, но тогда создатели сервиса намеревались продолжить независимое существование, что позволило превратить GitHub в нейтральную платформу разработки открытого кода, привлекательную для различных поставщиков. Репозитории GitHub использовались для разработки открытых проектов многих крупных компаний, включая Google, Facebook, Apple и Microsoft. В настоящее время на GitHub размещено более 85 млн репозиториев и зарегистрировано 28 млн разработчиков. После перехода GitHub в руки Microsoft нейтральность площадки будет утеряна, независимо от политики поддержания и развития сервиса. Представители Microsoft заявили, что после поглощения GitHub сохранит за собой статус общедоступной платформы, бесплатной для открытых проектов, независимой от языков программирования, лицензий, операционной системы и инструментов разработки. Утверждается, что Microsoft осознаёт ответственность перед сообществом, которую компания берёт на себя в результате сделки, и она приложит все усилия чтобы предоставить каждому разработчику возможность творить, выдумывать новое и решать насущные проблемы. GitHub продолжит работу в качестве независимого подразделения, а Microsoft также начнёт форсировать продвижение коммерческих сервисов GitHub для предприятий, используя свою сеть продаж и партнёрские каналы для увеличения клиентской базы. Из планов также отмечается использование GitHub как площадки для продвижения инструментов для разработки и сервисов Microsoft среди новой аудитории. Должность руководителя GitHub займёт Нэт Фридмэн (Nat Friedman), который сейчас занимает пост вице-президента в Microsoft. В 1999 году Нэт Фридмэн и Мигель де Икаса (Miguel De Icaza) основали компанию Ximian, занимавшуюся разработкой и поддержкой решений на базе GNOME. После продажи Ximian компании Novell, Нэт занял пост руководителя Novell по разработке Open Source стратегии и технологий. После продажи Novell он совместно с Мигелем де Икаса основал компанию Xamarin, в которой была продолжена разработка платформы Mono. В 2016 году компания Xamarin была куплена Microsoft. Как руководитель GitHub Фридмэн будет в подчинении у Скота Гатри (Scott Guthrie), вице-президента Microsoft, курирующего Microsoft Cloud и группу по развитию технологий искусственного интеллекта. Крис Ванстрас (Chris Wanstrath), нынешний руководитель и сооснователь GitHub, получит статус почётного технического деятеля Microsoft (Microsoft Technical Fellow).
