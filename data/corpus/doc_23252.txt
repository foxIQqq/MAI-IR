Релиз программы виртуализации Parallels Desktop 4 для Linux и Windows

После двух лет затишья основанная отечественными разработчиками компания Parallels, поддерживающая разработку свободного ПО OpenVZ, в дополнение к своему флагманскому продукту для платформы Mac OS X, выпустила проприетарную программу виртуализации Parallels Desktop 4 для Linux и Windows, пропустив при этом версию 3. Новые возможности включают: Полная поддержка аппаратных технологий виртуализации от Intel и AMD. Ускорение некоторых операций виртуализации до 8 (восьми) раз. Поддержка практически всех USB устройств на рынке. Поддержка 64 битных гостевых ОС. Поддержка новых гостевых ОС. Утилита Parallels Transporter для конвертации существующих физических инсталляций ОС в виртуальные машины. Утилита Parallels Image Tool для лёгкого управления размерами виртуальных хранилищ. Seamless режим для приложений гостевых ОС. Поддержка вплоть до 8 процессоров для гостевых ОС. Экспериментальная поддержка Windows 7 в качестве гостя, полная поддержка Windows Vista и всех современных Linux дистрибутивов, включая Fedora 11, Ubuntu 9.04 и OpenSuse 11.1.
