Вышли сборки MySQL Workbench OSS 5.0.30 и 5.1.7

Вышел стабильный релиз комьюнити сборки системы для визуального проектирования структуры БД MySQL Workbench 5.0.30 и тестовая версия MySQL Workbench 5.1.7. Ветка 5.0.x доступна только в сборках для Windows, 5.1.x - для Linux, MacOS X и Windows. Открытая ("Community (OSS) Edition") редакция продукта распространяется под лицензией GPL. Коммерческая редакция ("Standard Edition"), отличающается наличием дополнительных модулей для оптимизации работы, синхронизации БД и обратного инжиниринга.
