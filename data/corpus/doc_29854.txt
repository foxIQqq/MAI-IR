Canonical объединяет редакции Ubuntu для нетбуков и настольных систем

Джери Карр (Gerry Carr), директор по маркетингу платформы в компании Canonical, объявил об изменении подхода к подготовке базовых редакций Ubuntu Linux. Начиная с выпуска 11.04 редакции Ubuntu Netbook Edition и Ubuntu Desktop Edition будут поставляться в виде единой сборки, развиваемой под упрощенным единым брендом Ubuntu. В отличие от прошлых выпусков, Ubuntu 11.04 будет поставляться с новым графическим окружением Unity, адаптированным как для работы с небольшими экранными разрешениями нетбуков, так и с большими экранами настольных ПК и ноутбуков. Подобная унификация интерфейса делает бессмысленным развитие отдельной ветки Ubuntu Netbook Edition, поставляемой с адаптированной для небольших экранов пользовательской оболочкой. Отдельно отмечается, что прекращение сборки Ubuntu Netbook Edition не следует воспринимать как уход Ubuntu с рынка нетбков. Более того, статистика загрузки дистрибутива показывает значительный рост интереса пользователей нетбуков, поэтому компания Canonical рассматривает оптимизацию работы Ubuntu для устройств с небольшими экранами как одну из основных задач, внимание которой будет уделяться ничуть не меньше, чем вопросам работы на ПК и ноутбуках. Ожидается, что использование единой оболочки на всех типах устройств увеличит привлекательность дистрибутива. Что касается редакции Ubuntu Server Edition, то поддержка серверного варианта продолжит осуществляться в виде отдельного продукта, который будет распространяться под именем Ubuntu Server 11.04. Напомним, что вариант Ubuntu для пользователей получил официальное название Ubuntu Desktop Edition в 2005 году, после начала развития серверной системы Ubuntu Server Edition. В конечном итоге подобный подход привел к путанице среди пользователей, некоторые из которых считали, что дистрибутив "Ubuntu Desktop Edition" нацелен только на настольные ПК и пытались найти несуществующую редакцию для ноутбуков "Ubuntu Laptop Edition".
