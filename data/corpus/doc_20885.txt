Вышел релиз Linux дистрибутива Frugalware 1.0

"Frugalware 1.0 (Anacreon) released" - вышел релиз Linux дистрибутива Frugalware 1.0. В новой версии добавлена поддержка субноутбуков ASUS Eee PC, доступна сборка для архитектуры PPC, добавлена возможность создания 32-разрядных chroot-окружения на x86_64 системах, для управления запуском сервисов в комплект включен интерфейс gService, добавлена поддержка установки на ext4 разделы. Из поставляемых в комплекте программ можно отметить: Linux ядро 2.6.28.7, KDE 3.5.10, GNOME 2.24.3, OpenOffice.org 3.0.1, Firefox 3.0.6, x86_64 Java plugin. В репозитории Frugalware 1.0 насчитывается 4701 пакет. Дистрибутив Frugalware разработан для опытных пользователей не боящихся командной строки, основное направление развития дистрибутива - максимальное упрощение. Концепция построения дистрибутива похожа на Slackware, в качестве пакетного менеджера задействован pacman из Arch Linux. При создании пакетов, разработчики стараются публиковать программы как есть, избегая внесения дополнительных патчей.
