Анонсирован Linux From Scratch 6.8

Представлено руководство Linux From Scratch 6.8, в котором приведены инструкции по созданию базовой Linux-системы с нуля, используя лишь исходные тексты необходимого программного обеспечения. В новой версии произведен переход на использование Linux ядра 2.6.37, набора компиляторов GCC 4.5.2, системной библиотеки Glibc-2.13. Также были произведены редакторские работы в пояснительных материалах по всей книге, благодаря чему повысилась точность и ясность текста. При построении базовой системы в LFS 6.8 используются новые версии следующих пакетов: Autoconf 2.68, Bash 4.2, Binutils 2.21, Bzip2 1.0.6, Coreutils 8.10, E2fsprogs 1.41.14, Expect 5.45, File 5.05, GCC 4.5.2, Glibc 2.13, Grep 2.7, Groff 1.21, IPRoute2 2.6.37, Libtool 2.4, Linux 2.6.37, M4 1.4.15, Man-DB 2.5.9, Man-pages 3.32, Perl 5.12.3, Psmisc 22.13, Readline 6.2, Shadow 4.1.4.3, Tar 1.25, TCL 8.5.9, Udev 166, util-linux-2.19. Добавлен пакет xz-5.0.1. Кроме руководства по созданию базовой системы в рамках проекта выпускается несколько дополнительных книг: "Beyond Linux From Scratch" - руководство по конфигурированию и сборке более 500 программных пакетов в окружении LFS. Текущий релиз: svn-20110219, перечень изменений; "Automated Linux From Scratch" - фреймворк для автоматизации сборки LFS-системы и управлению пакетами; "Cross Linux From Scratch" - описание кроссплатформенной сборки LFS-системы, поддерживаются архитектуры: x86, x86_64, sparc, mips, PowerPC, alpha, hppa, arm. Последние версии: CLFS GIT-20110213, Embedded CLFS GIT-0.0.1-20110224; "Hardened Linux From Scratch" - инструкции по повышению безопасности LFS, применению дополнительных патчей и ограничений; "LFS Hints" - подборка дополнительных советов с описанием альтернативных решений для описанных в LFS и BLFS шагов. Добавлена инструкция по сборке LFS для PowerPC (32bit); "LFS LiveCD" - проект по подготовке LiveCD. На данный момент, не развивается. Любые комментарии о текущем выпуске, разработчики просят отправлять в рассылку lfs-dev at linuxfromscratch.org (требуется регистрация).
