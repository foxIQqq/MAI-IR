Обновление прошивки Ubuntu Touch OTA-13

Компания Canoniсal опубликовала обновление OTA-13 (over-the-air) для смартфонов и планшетов на базе платформы Ubuntu Touch. Обновление автоматически будет доставлено пользователям смартфонов Meizu MX4/PRO 5, Nexus 4, Nexus 7 и Bq Aquaris E5/E4.5/M10, укомплектованных прошивкой на базе Ubuntu. Основные новшества: Для классических настольных приложений, запускаемых в окружении Libertine, обеспечена возможность копирования и вставки из буфера обмена. Таким образом теперь можно совместно использовать буфер обмена между обычными (LibreOffice, Firefox, GIMP) программами и мобильными приложениями; Добавлена новая панель вывода уведомлений, позволяющая отдельно настраивать звук, оформление и вибросигнал для каждого приложения; Переработана панель проверки обновлений, которая теперь работает значительно стабильнее. В интерфейсе обновления приложений организован вывод краткой информации об изменениях в новых версиях и добавлен список недавно установленных обновлений; Налажена работа настройки расширенных параметров вибросигнала ("Other vibrations"); Реализован индикатор клавиатуры; В календарь-планировщик добавлена поддержка iCal и CalDav. Обеспечена возможность синхронизации нескольких календарей-планировщиков и возможность синхронизации с облачным хранилищем Owncloud или через аккаунт в Google Calendar; Проведена оптимизация времени запуска различных приложений (календарь, калькулятор, камера, интерфейс набора номера); Добавлена новая реализация фонового процесса для управления питанием (repowerd); Улучшены средства управления экраном и логика работы с датчиком приближения; Представлена начальная реализация фреймворка для работы с накопителями; Добавлена новая C++ библиотека UbuntuMetrics, которая может использоваться для мониторинга различных метрик работы приложений с интерфейсом на базе QtQuick 2; Обновлены версии сторонних компонентов: Oxide 1.16.8, Gstreamer 1.8.2; Добавлен упрощённый интерфейс для ввода Emoji.
