Ubuntu Budgie получил статус официальной редакции Ubuntu

Сборка Budgie Remix принята в число официальных редакций Ubuntu и отныне будет поставляться под именем Ubuntu Budgie, наряду с Kubuntu, Xubuntu, Ubuntu GNOME, Lubuntu, Ubuntu MATE, Ubuntu Studio и UbuntuKylin. Ubuntu Budgie предлагает пользовательское окружение на основе рабочего стола Budgie, который развивается проектом Solus. Budgie работает поверх штатных низкоуровневых компонентов и библиотек GNOME, но не является форком GNOME. Рабочий стол построен на основе собственной реализации оболочки GNOME Shell, панели, апплетов и системы вывода уведомлений. Код проекта написан на языках Си и Vala и распространяется под лицензией GPLv2. Сообщество разработчиков продемонстрировало свою состоятельность, выпустив два неофициальных релиза Budgie Remix, а также выполнило все требования к официальным сборкам. На состоявшемся на днях совещании комитет по техническому развитию Ubuntu (Ubuntu Tech Board) утвердил новую редакцию. Начиная с выпуска Ubuntu 17.04, сборка c рабочим столом Budgie будет интегрирована в основную сборочную инфраструктуру Ubuntu, станет распространяться с официальных зеркал, придерживаться типового цикла разработки, использовать службы тестирования и формирования ежедневных сборок. Для взаимодействия разработчиков будет использоваться Launchpad, а для поддержки пользователей сайты Ask Ubuntu, Ubuntu Forums и wiki.ubuntu.com. Основу Budgie составляет панель, близкая по организации работы к классическим панелям рабочего стола. Все элементы панели являются апплетами, что позволяет гибко настраивать состав, менять размещение и заменять реализации основных элементов панели на свой вкус. Среди доступных апплетов можно отметить классическое меню приложений, систему переключения задач, область со списком открытых окон, просмотр виртуальных рабочих столов, индикатор управления питанием, апплет управления уровнем громкости, индикатор состояния системы и часы. Для управления окнами используется оконный менеджер Budgie Window Manager (BWM), являющийся расширенной модификацией базового плагина Mutter.
