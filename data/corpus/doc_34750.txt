Релиз Linux-дистрибутива openSUSE 12.2

Увидел свет релиз Linux-дистрибутива openSUSE 12.2. Дистрибутив доступен для 32- и 64-разрядных x86-систем в четырёх сборках: универсальный установочный DVD (4.1 Гб), LiveCD с GNOME 3.4, LiveCD с KDE 4.8 и сокращённая версия для установки по сети. В отличие от прошлых выпусков на подготовку openSUSE 12.2 было потрачено 10 месяцев вместо 8, что было обусловлено необходимостью дополнительной стабилизацией пакетной базы и проведением реорганизации процесса разработки. Особенности openSUSE 12.2: Пользовательское окружение GNOME обновлено до версии 3.4.2, а KDE - до версии 4.8.4 (в прошлом выпуске поставлялись GNOME 3.2 и KDE 4.7). Дополнительно обеспечена возможность установки KDE 4.9.1 из отдельного репозитория KDE:Release:49. Воспользовавшись репозиторием KDE:Active, также можно установить оболочку Plasma Active, подготовленную для устройств с сенсорными экранами; В состав дистрибутива включено десктоп-окружение Razor-Qt. Дополнительно, в качестве легковесных десктоп-окружений поставляются Xfce 4.10 и LXDE; Из официального репозитория можно установить окружение KDE 3, в которое добавлена поддержка Udisks2 и Upower; Обеспечен бесшовный процесс отображения заставки в процессе загрузки, реализованный с использованием системы Plymouth, вместо ранее используемого bootsplash; Осуществлён переход на загрузчик GRUB 2.0, с поддержкой UEFI, LVM и загрузки с разделов Btrfs; Обновление ядра Linux 3.4.6, набора компиляторов GCC 4.7, системной библиотеки glibc 2.15, системного менеджера systemd 44; Задействование для ведения логов systemd-подсистемы Journal; Обновлены компоненты графического стека: X.Org Server 1.12.3, Mesa 8, Qt 4.8.1, видеодрайвер Intel 2.20.0 с бэкендами SNA и GLAMOR. В состав включен Gallium3D-драйвер для обеспечения поддержки 3D при запуске системы внутри окружения виртуализации VMware; В поставку добавлен драйвер llvmpipe, что позволило обеспечить работу GNOME Shell без 3D-драйверов, за счёт осуществления программного рендеринга; Обновлены средства для разработчиков: Python 3 3.2.3, Qt Creator 2.5, OpenJDK 7, Perl 5.16.0, Go 1.0.2; Серверное ПО: BIND 9.9, OpenSSH 6.0, openssl 1.0.1c, QEMU 1.1.0, Tomcat 7, Firebird 2.5, Новые версии приложений для пользователей: LibreOffice 3.5, Calligra 2.4.2, Gnumeric 1.11.3, GIMP 2.8, Inkscape 0.48.3, Amarok 2.5, Scribus 1.4.1, wine 1.5.6, rekonq 0.9.2, Firefox 14, Chromium 21; Перенос всех исполняемых файлов и библиотек из корня в директорию /usr. В частности, содержимое /bin перенесено в /usr/bin, /sbin в /usr/sbin, /lib в /usr/lib и /lib64 в /usr/lib64. Проведена чистка структуры файловой системы, убраны устаревшие символические ссылки на устройства извлекаемых дисков, которые были оставлены для обеспечения совместимости с ранними выпусками SUSE Linux; Упрощён интерфейс настройки подключения к беспроводным сетям и управления питанием. Расширены возможности NetworkManager по проверке сертификатов при настройке подключения к корпоративным сетям; Правила Polkit модифицированы для предоставления возможности установки online-обновлений любым пользователем системы; Увеличена скорость выполнения операций в пакетном менеджере zypper; Включены дополнительные наработки по интеграции Firefox и LibreOffice с пользовательской оболочкой Plasma в KDE. Улучшена работа файлового менеджера Dolphin, в котором использован новый движок, переписанный с целью обеспечения более высокой производительности, масштабируемости и наглядности отображения. Увеличена скорость работы с директориями с большим числом файлов. Упрощена навигация при просмотре больших фотографий в Gwenview; В состав дистрибутива включена поддержка протокола SPICE, предназначенного для организации удаленной работы с рабочим столом, функционирующим в виртуальном окружении. SPICE позволяет значительно повысить эффективность вывода мультимедиа информации из локально запущенных виртуальных окружений, с привлечением средств акселерации вывода видео, звука, а также 2D и 3D графики.
