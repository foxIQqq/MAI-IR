Релиз дистрибутива Nitrux 1.6.0 с рабочим столом NX Desktop

Опубликован выпуск дистрибутива Nitrux 1.6.0, построенного на пакетной базе Debian, технологиях KDE и системе инициализации OpenRC. Дистрибутив развивает собственный рабочий стол NX Desktop, который представляет собой надстройку над пользовательским окружением KDE Plasma. Для установки дополнительных приложений продвигается система самодостаточных пакетов AppImages. Размер загрузочных образов составляет 3.1 ГБ и 1.5 ГБ. Наработки проекта распространяются под свободными лицензиями. Рабочий стол NX Desktop предлагает иное стилевое оформление, собственную реализацию системного лотка, центра вывода уведомлений и различных плазмоидов, таких как конфигуратор сетевых соединений и мультимедийный апплет для регулирования громкости и управления воспроизведением мультимедийного контента. В поставку также входят приложения из набора MauiKit, среди которых файловый менеджер Index (можно использовать и Dolphin), текстовый редактор Note, эмулятор терминала Station, музыкальный проигрыватель Clip, видеопроигрыватель VVave и просмотрщик изображений Pix. В новом выпуске: Компоненты рабочего стола обновлены до версий KDE Plasma 5.22.4, KDE Frameworksn 5.85.0 и KDE Gear (KDE Applications) 21.08. До ветки 2.0 обновлён развиваемый проектом фреймворк MauiKit и построенные на его основе приложения Index, Nota, Station, VVave, Buho, Pix, Communicator, Shelf и Clip, которые можно использовать как на настольных системах, так и на мобильных устройствах. Обновлены приложения, в том числе Firefox 91.0.2, Heroic Games Launcher 1.9.2, LibreOffice 7.2.0.4. Предложен новый центр управления приложениями NX Software Center 1.0.0, предлагающий для установки пакеты в формате AppImage, которые после установки полностью интегрируются с рабочим столом. Доступно три режима работы: просмотр доступных для установки приложений с поддержкой поиска, навигации по категориям и рекомендаций наиболее популярных программ; просмотр загруженных пакетов; оценка состояния загрузки новых приложений. По умолчанию включена поддержка управления жестами при помощи тачпада. Предложена по умолчанию новая тема оформления командной оболочки ZSH - Powerlevel10k. В минимальных сборках продолжено использование темы agnoster. Для KWin добавлены скрипты: MACsimize для перемещения раскрытого на весь экран окна на другой виртуальных рабочий стол и возвращения к исходному рабочему столу после закрытия окна; ForceBlur для применение эффекта размытия к пользовательским окнам. Из базовой поставки удалены приложения Plasma Discover и LMMS. Для установки на выбор предложены пакеты с ядром Linux 5.4.143, 5.10.61 и 5.14.0, Linux Libre 5.10.61 и Linux Libre 5.13.12, а также ядра 5.13 с патчами от проектов Liquorix и Xanmod.
