Выпуск KDE Applications 15.04, набора приложений KDE

Состоялся релиз KDE Applications 15.04, включающий в себя подборку пользовательских приложений, адаптированных для работы с KDE Frameworks 5. Набор KDE Applications пришёл на смену приложениям из состава KDE SC, которые теперь развиваются в рамках отдельного цикла разработки, не привязанного к веткам KDE, и выпускаются по новой схеме нумерации версий. Оценить работу нового выпуска без установки можно через Live-сборку от проекта Kubuntu. Статус подготовки бинарных пакетов для различных дистрибутивов можно оценить на данной странице. Общее число портированных на KDE Frameworks 5 и Qt5 приложений достигло 72 из 150 программ, поставляемых для KDE 4. По сравнению с декабрьским выпуском в состав KDE Applications 15.04 включены такие компоненты, как система нелинейного видеомонтажа Kdenlive, набор приложений для мгновенного обмена сообщениями KDE Telepathy (по сравнению с версией для KDE4 пока не поддерживается видео и голосовые звонки), система обучения теории графов Rocs, математический пакет Cantor, обучающая игра KHangMan, графический интерфейс для сравнения файлов Kompare и некоторые игры KDE. Из уже входящих в состав KDE Applications приложений, в которых отмечаются наиболее значительные улучшения, выделяется система UML-моделирования Umbrello, в которой проведена работа по увеличению стабильности и улучшению удобства интерфейса, функция поиска расширена возможностью привязки выборки к категориям.
