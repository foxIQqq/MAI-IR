Компания ALT Linux представила релиз дистрибутива Альт Линукс 6.0 Кентавр

Доступен релиз Альт Линукс 6.0 Кентавр, многофункционального дистрибутива для серверов и рабочих станций, основанного на «Шестой платформе» ALT Linux. Дистрибутив прежде всего ориентирован на использование в корпоративных сетях, что подчёркивает пятилетний срок поддержки. Установочный образ сформирован в гибридном виде, который можно использовать как для записи на DVD, так и на USB Flash. Основные особенности дистрибутива: Поставка десктоп-окружения GNOME 2.32.1 для использования на рабочих станциях. Дополнительно, в качестве минимальной среды для администрирования сервера поставляется FVWM; Возможность конфигурирования серверных компонентов через web-интерфейс; Реализация компонента "ALT-домен", включающего преднастроенный комплект из взаимосвязанных серверов LDAP, Kerberos, DNS, Samba, DHCP, Postfix, Dovecot, сервера сетевой загрузки и сервера обновлений; Универсальное серверное решение, позволяющее создавать домены с единой аутентификацией (SSO); Установка серверных решений и типовых комплектов приложений для конечных пользователей с одного диска. Поддержка древовидного выбора групп пакетов в инсталляторе; Возможность развернуть типовые серверные и клиентские решения, например, web-сервер с базовым набором приложений (Apache, MySQL, PHP), на этапе инсталляции или через установку одного мета-пакета; Средства для создания Kerberos/LDAP домена с серверной стороны, а также компоненты для организации доступа к ресурсам этого домена как с рабочих станций, так и с бездисковых клиентов; Включение в дистрибутив системы резервного копирования Bacula, в том числе интеграция в интерфейс управления; Интеграция в дистрибутив SELinux; Использование GRUB2 в качестве загрузчика; Работа инсталлятора через VNC; Возможность установки "минимальной системы" без альтератора, виртуализации, bind и т.п.; Реализация режима автоматизации процесса установки (kickstart); Простой режим конфигурирования бездисковых клиентов с загрузкой по сети; Некоторые версии входящих в состав программ: samba 3.5.10, postfix 2.5.14, exim 4.76, dovecot 2.0.12, openldap 2.4.24, bind 9.3.6, mysql 5.1.56, postgresql 9.0.4, perl 5.12.3, php 5.3.8, GCC 4.5.1, LibreOffice 3.4.3.
