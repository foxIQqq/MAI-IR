Выпуск Chrome OS 89, приуроченный к 10-летию проекта Сhromebook

Состоялся релиз операционной системы Chrome OS 89, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 89. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 89 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Выпуск приурочен к десятилетию проекта, поэтому содержит много значительных новшеств. Основные изменения в Chrome OS 89: Добавлен Phone Hub, центр управления смартфоном, позволяющий с устройства Chromebook выполнять типовые действия со смартфоном на базе платформы Android, такие как просмотр поступающих сообщений и уведомлений, контроль за уровнем заряда аккумулятора, доступ к настройкам точки доступа (hotspot), определение местоположения смартфона. Кроме того, Phone Hub позволяет в браузере на Chromebook просмотреть содержимое вкладок, недавно открытых на смартфоне. Активация устройств для Phone Hub осуществляется в настройках "Settings > Connected devices", после чего в блоке быстрых настроек на панели появляется специальная пиктограмма с дополнительными функциями. Расширен спектр устройств, которые можно использовать с функцией Wi-Fi Sync, предназначенной для синхронизации настроек беспроводной сети между несколькими устройствами. Например, пароль к Wi-Fi запоминается в профиле пользователя и автоматически применяется при входе данного пользователя с других устройств, без необходимости повторного ручного набора пароля подключения к Wi-Fi на новом устройстве. Настройки Wi-Fi теперь могут совместно использоваться между разными устройствами на базе Chrome OS и Android, привязанными к одной учётной записи в Google. В буфере обмена реализована функция MultiPaste, позволяющая сохранять историю с пятью последними операциями копирования. Имеется возможность вставить разом несколько сохранённых элементов или выбрать любой из них через интерфейс, выводимый при нажатии комбинации "Launcher + V" Например, теперь можно не переключаясь между окнами скопировать в буфер несколько фрагментов, а потом разом вставить их в нужную форму. Предложен новый интерфейс для создания скриншотов и скринкастов, который можно вызвать через меню с быстрыми настройками. Интерфейс также вызывается при нажатии комбинации "Ctrl + Windows". Сразу после создания скриншота внизу появляется меню, позволяющее откорректировать созданный снимок или начать запись видео с действиями на экране. В панели рядом с меню быстрого вызова настроек появилась новая пиктограмма "Tote", позволяющая в один клик получить доступ к недавно сохранённым скриншотам, закреплённым файлам или загрузкам. В приложение для работы с камерой встроена функция для сканирования QR-кодов. Для сканирования QR кода следует нажать пиктограмму с изображением QR-кода в правой части окна. Упрощено управление воспроизведением мультимедийных файлов - кнопки для управления воспроизведением из браузера или Android-приложений теперь отображаются на панели в меню быстрого вызова настроек. Пользователь может быстро остановить/возобновить воспроизведение или переключиться на следующую композицию, а также закрепить медиапроигрыватель в панели. Добавлены новые опции для обмена файлами, изображениями и ссылками. Для приложений и сайтов теперь показывается кнопка Share, позволяющая напрямую передать файл, изображение или ссылку в другое приложение. Например, при помощи кнопки Share можно быстро передать картинку из приложения Files в текстовый редактор. В одном из следующих выпусков ожидается включение функции Nearby Share, позволяющей быстро и безопасно передавать файлы между находящимися поблизости устройствами Chrome OS и Android разных пользователей. Расширены возможности, связанные с виртуальными рабочими столами. Разрешено создание до 8 виртуальных рабочих столов и перегруппировка их в произвольном порядке при помощи механизма drag&drop. В меню, показываемое при клике правой кнопкой мыши, добавлены кнопки для закрепления окна на определённом виртуальном рабочем столе или для отображения окна на всех рабочих столах. Реализована возможность использования комбинации Alt+Tab для просмотра окон в привязке к текущему виртуальному рабочему столу или всех окон без разделения на рабочие столы. В контекстное меню, отображаемое при клике правой кнопкой мыши на выделенном слове или фразе, добавлена функция Quick Answers, позволяющая получить дополнительные сведения, например, отобразить данные из словаря, выполнить перевод или осуществить конвертацию величин. Добавлены дополнительные настройки функции чтения вслух текста в выделенном блоке (select-to-speak). Например, появилась возможность на лету изменить темп, приостановить чтение и переключиться на чтение других отрывков. Настройка режима родительского контроля "Family Link" интегрирована в мастер начальной настройки нового Chromebook и позволяет родителям сразу привязать школьную учётную запись своего ребёнка и организовать контроль за его работой на устройстве. В подсистему печати добавлена поддержка функций сканирования, предоставляемых в многофункциональных устройствах, сочетающих принтер и сканер. Сканирование осуществляется через новое приложение Scan. Обновлены пиктограммы встроенных приложений.
