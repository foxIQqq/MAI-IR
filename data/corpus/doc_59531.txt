Выпуск дистрибутива MX Linux 23

Опубликован релиз легковесного дистрибутива MX Linux 23, созданного в результате совместной работы сообществ, образовавшихся вокруг проектов antiX и MEPIS. Выпуск основан на пакетной базе Debian с улучшениями от проекта antiX и пакетами из собственного репозитория. В дистрибутиве используется система инициализации sysVinit и собственные инструменты для настройки и развёртывания системы. Для загрузки доступны 32- и 64-разрядные сборки (1.8 ГБ, x86_64, i386) с рабочим столом Xfce (2.2 ГБ), а также 64-разрядные сборки с рабочим столом KDE (2.6 ГБ) и сборки (1.7 ГБ) с оконным менеджером Fluxbox. В новом выпуске: Осуществлена синхронизация с пакетной базой Debian 12. Обновлены версии приложений. Как и в прошлых выпусках по умолчанию продолжает использоваться система инициализации sysVinit, а systemd можно установить в качестве опции. Пользовательские окружения обновлены до Xfce 4.18, Fluxbox 1.3.7 и KDE Plasma 5.27. Для работы со звуком вместо звукового сервера pulseaudio задействован мультимедийный сервер Pipewire и менеджер звуковых сеансов WirePlumber. В загрузочное меню Live-сборок добавлена опция для проверки целостности носителя. Для людей с проблемами со зрением в состав включён экранный ридер orca и утилита для увеличения выбранных областей экрана. Обновлены развиваемые проектом утилиты MX Tools. Для запуска с предоставлением дополнительных привилегий вместо mx-pkexec задействован PolicyKit с отдельными настройками для каждого приложения. Добавлено новое приложение "User Installed Packages" для создания файла со списком приложений, установленных пользователем, который можно использовать для автоматизации установки тех же приложений на другой системе или после обновления дистрибутива до новой значительной версии. В утилиту MX Snapshot добавлены настройки загрузки снапшотов и скрыты настройки, которые могут помешать загрузке созданной live-системы на другом компьютере. В утилите MX-Tweak добавлены настройки тем оформления для Xfce и Fluxbox. В MX-Updater предоставлена возможность использования nala в качестве бэкенда установки пакетов вместо apt. Для фильтрации пакетов по умолчанию задействован межсетевой экран UFW. Добавлена порция новых настроек для пользовательского окружения на основе Fluxbox.
