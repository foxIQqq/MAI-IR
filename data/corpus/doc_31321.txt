Ошибка в Launchpad привела к внеплановой синхронизации пакетов Ubuntu с Debian

Разработчики Ubuntu получили уведомление о внеплановой синхронизации с пакетами из архива Debian GNU/Linux около 900 пакетов из состава репозиториев, используемых для разработки новой версии Ubuntu. Синхронизация была вызвана ошибкой, всплывшей в результате финального тестирования новой версии системы Launchpad. Ошибка была связана с тем, что реализация новой функции синхронизации c Debian отдельных пакетов через web-интерфейс при определённых обстоятельствах приводила к синхронизации всех пакетов вместо одного. В настоящее время ведётся работа по откату произведённых изменений. Все обновлённые в результате инцидента пакеты в настоящее время ожидают завершения стадии пересборки, до окончания которой могут наблюдаться проблемы со стабильностью. Разработчики Launchpad уже определили причину ошибки и протестировали исправление на экспериментальном сервере. В ближайшее время планируется возобновить тестирование на рабочем сервере с целью выявления побочных эффектов, связанных с исправленной ошибкой. Из синхронизированных по ошибке пакетов 78 относились к репозиторию "main", а 813 к репозиториям "universe" и "multiverse". Всего в репозитории "main" насчитывается 3261 пакетов, "universe" - 14736, "multiverse" - 504. В процессе разбора полётов была выявлена ещё одна недоработка, в результате которой синхронизации подверглись некоторые пакеты, в которые были внесены оригинальные изменения Ubuntu (ошибка затронула 7 пакетов). В остальном, в процессе инцидента были обновлены только пакеты, скопированные в Ubuntu без внесения изменений, что значительно упрощает процесс возвращения репозиториев в состояние до возникновения инцидента. Основная масса пакетов Ubuntu импортируется из Debian. В примерно 17% импортированных пакетов разработчики Ubuntu вносят дополнительные изменения. Специально для Ubuntu создано около 7% пакетов из общего состава репозиториев данного дистрибутива. Большинство из пакетов, изменённых силами проекта Ubuntu, относится к репозиторию "main"; репозиторий "universe" более близок по своему составу к официальным пакетам Debian.
