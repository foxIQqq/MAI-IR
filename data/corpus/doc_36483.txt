Анонсирован порт развиваемого проектом KDE офисного пакета Calligra для Android

Представлен первый альфа-выпуск редакции свободного офисного пакета Calligra для платформы Android. Разработка ведётся в рамках недавно инициированного проекта Coffice, нацеленного на создание версии Calligra для мобильных платформ, таких как Android, Blackberry 10, Jolla SailfishOS и Ubuntu Phone. Установить готовые сборки мобильной редакции Calligra можно из каталога Google Play. Размер сборки составляет 2.5 Мб (без учёта Qt). Для установки и обеспечения работы Qt-библиотек дополнительно требуется установка программы Ministro. На первом этапе своего развития Coffice напоминает развиваемый компанией Nokia просмотрщик документов для смартфонов N9, и пока включает только интерфейс для просмотра ODT-документов, созданный на базе Calligra Words. На втором этапе планируется интегрировать поддержку работы с форматами doc/docx и обеспечить возможность редактирование документов. В дальнейшем ожидается портирование табличного процессора Calligra Sheets и программы для работы с презентациями Calligra Stage. Для обеспечения работы Calligra в окружении платформы Android используется порт тулкита Qt для Android, который запланирован для включения в ожидаемый в ближайшие недели выпуск Qt 5.1. Для сборки вместо cmake задействован qmake, который более оптимально подходит для сборки Qt для мобильных платформ. Вместо используемого в мобильном просмотрщике от компании Nokia урезанного варианта библиотек KDE, в Coffice предполагается использование активно развиваемого KDE Frameworks, который отличается модульной структурой, оформленной в виде взаимодействующих друг с другом независимых фреймвоков. В настоящее время Coffice завязан только на Qt и для трансляции необходимых вызовов kdelibs использует специальную библиотеку, транслирующую некоторые элементы API kdelibs в API Qt, без поддержки таких элементов, как D-Bus и сервисные демоны. Из достижений проекта также отмечается обеспечение сборки большей части библиотек и некоторых плагинов Calligra, а также возможность сборки приложения Calligra Words. Интерфейс пока реализован в виде простейшего приложения на QML.
