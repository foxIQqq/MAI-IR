Релиз системы виртуализации VirtualBox 4.0.6

Компания Oracle выпустила корректирующий релиз системы виртуализации VirtualBox 4.0.6, в котором отмечено 56 исправлений. Из изменений можно выделить: Поддержка X.Org Server 1.10 и Linux-ядра 2.6.39-rc1; Решение проблем с мышью при запуске гостевой системы с SUSE 11 SP 1; Обеспечена поддержка тестовых версий Ubuntu 11.04, в VirtualBox теперь можно оценить работу оболочки Unity; Устранены проблемы при восстановлении виртуальной машины из замороженного состояния или снапшота; Устранён крах при запуске на хост-системе с более чем 64 процессорными ядрами. Добавлена поддержка работы на системах, имеющих до 256 процессорных ядер; Решены проблемы с запуском гостевых систем в режиме PAE на хост системах не поддерживающих PAE; Устранены проблемы, приводящие к замедлению работы гостевых систем с Linux в raw-режиме; Добавлена возможность сохранения снапшотов в директории, где хранится образ виртуальной машины; Устранена утечка памяти; Решены проблемы с работой iSCSI targets через внутреннюю сеть; Решены проблемы с чтением данных с диска, при наличии более чем одного снапшота для образов VHD и VMDK и при выключенном кэше; Устранены зависания при заморозке виртуальной машины; Решены проблемы, приводившие к неработоспособности сети после возвращения окружения из заморозки; Устранены ошибки, приводящие к краху при подключении некоторых USB-устройств; На платформе Linux устранены недоработки, мешающие установке VirtualBox внутри виртуальной машины с активными драйверами гостевой системы (Guest Additions); Добавлен обходной путь, для предотвращения крахов при использовании Glibc версии меньше 1.11.
