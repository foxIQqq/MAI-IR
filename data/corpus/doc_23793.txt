Вышла новая тестовая версия MySQL Server 5.4.3

Вышел MySQL 5.4.3, третий бета-релиз ветки MySQL 5.4, обзор новшеств которой можно прочитать здесь. В новой версии исправлено 48 ошибок, из которых 14 приводило к краху. Из новшеств можно отметить несколько изменений параметров по умолчанию для хранилища InnoDB, нарушающих совместимость с логами транзакций созданными в прошлых версиях MySQL, для Linux добавлена возможность использования альтернативного высокопроизводительного менеджера распределения памяти Google tcmalloc.
