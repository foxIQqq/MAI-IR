Выпуск дистрибутива Nitrux 1.8.0 с рабочим столом NX Desktop

Опубликован выпуск дистрибутива Nitrux 1.8.0, построенного на пакетной базе Debian, технологиях KDE и системе инициализации OpenRC. Дистрибутив развивает собственный рабочий стол NX Desktop, который представляет собой надстройку над пользовательским окружением KDE Plasma, а также фреймворк построения интерфейсов пользователя MauiKit, на основе которого развивается набор типовых пользовательских приложений, которые можно использовать как на настольных системах, так и на мобильных устройствах. Для установки дополнительных приложений продвигается система самодостаточных пакетов AppImages. Размер загрузочного образа составляет 3.2 ГБ. Наработки проекта распространяются под свободными лицензиями. Рабочий стол NX Desktop предлагает иное стилевое оформление, собственную реализацию системного лотка, центра вывода уведомлений и различных плазмоидов, таких как конфигуратор сетевых соединений и мультимедийный апплет для регулирования громкости и управления воспроизведением мультимедийного контента. Из приложений, созданных при помощи фреймворка MauiKit, можно отметить файловый менеджер Index (можно использовать и Dolphin), текстовый редактор Note, эмулятор терминала Station, музыкальный проигрыватель Clip, видеопроигрыватель VVave, центр управления приложениями NX Software Center и просмотрщик изображений Pix. В новом выпуске: В качестве опции добавлена начальная реализация пользовательского окружения Maui Shell. Предложено два варианта запуска Maui Shell: со своим композитным сервером Zpace, использующим Wayland, и через запуск отдельно оболочки Cask внутри сеанса на основе X-сервера. Компоненты основного рабочего стола обновлены до версий KDE Plasma 5.23.4 (в прошлом выпуске использовался KDE 5.22), KDE Frameworksn 5.89.0 и KDE Gear (KDE Applications) 21.12.0. Обновлены версии программ, в том числе Firefox 95, Kdenlive 21.12.0, Pacstall 1.7, Ditto menu 1.0. В предлагаемую по умолчанию панель Latte Dock добавлены виджеты pager и trashcan. Обеспечено автоскрытие верхней панели через 3 секунды при наличии развёрнутых на весь экран окон. В приложениях из набора Maui Apps по умолчанию включено декорирование окон на стороне клиента (CSD), изменить данное поведение можно отредактировав файл ~/.config/org.kde.maui/mauiproject.conf. Для установки на выбор предложены пакеты с ядром Linux 5.15.11 (по умолчанию), 5.14.21, 5.4.168, Linux Libre 5.15.11 и 5.14.20, а также ядер 5.15.0-11.1, 5.15.11 и 5.14.15-cacule, с патчами от проектов Liquorix и Xanmod. Инсталлятор Calamares переведён на использование файловой системы XFS для установки дистрибутива. В поставку включены 113 профилей AppArmor. В системный монитор добавлены две настраиваемые страницы для отслеживания интенсивности ввода/вывода, доступного места в хранилище и статистики по GPU (использование видеопамяти, нагрузка на GPU, частота и температура). Из-за наличия нерешённых проблем отключён сеанс KDE Plasma на базе Wayland.
