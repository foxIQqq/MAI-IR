Представлен QmlWeb, позволяющий использовать QML для разработки web-приложений

Представлен новый проект сообщества KDE - QmlWeb, в рамках которого развивается JavaScript-библиотека, осуществляющая разбор и выполнения конструкций на языке QML (Qt Meta-Object Language), позволяя использовать данный язык вместо JavaScript для создания динамических сайтов. QML является центральным звеном среды Qt Quick, ориентированной на разработку интерфейса пользователя. По мнению авторов QmlWeb, если QML прекрасно справляется с созданием графических интерфейсов, то им также будет удобно пользоваться и для создания интерактивных web-интерфейсов. Текущая реализация не поддерживает все возможности QML, присутствующие в Qt. Тем не менее, поддерживается базовый синтаксис и элементы Item, Rectangle, Text, Image, BorderImage, MouseArea, Repeater, ListModel, NumberAnimation, SequentialAnimation. Кроме того, добавлена поддержка специфичных для web-элементов, таких как Button, TextInput и TextArea.
