Adobe Flash ждут изменения, направленные на усиление приватности пользователей

Компания Adobe прислушалась к мнению борцов, отстаивающих право пользователей на соблюдение тайны частной жизни, и планирует реализовать в Adobe Flash функцию легкого удаления генерируемых средствами Flash Cookie-идентификаторов, которые могут быть использованы для отслеживания перемещения пользователя в сети. В отличие от традиционных браузерных Cookie, формируемые средствами Flash идентификаторы невозможно удалить традиционными средствами браузера. Для решения проблем с оседанием неконтролируемых данных, локально сохраняемых плагинами, разработчики Mozilla подготовили специальный API ClearSiteData, поддержку которого планируют интегрировать во Flash-плеер. Кроме Firefox данный API будет поддерживаться в следующем выпуске браузера Chrome. Для не поддерживающих новый API браузеров, Adobe добавит в состав Flash интерфейс для локальных настроек "Settings Manager", который ранее был доступен только в виде внешнего сервиса.
