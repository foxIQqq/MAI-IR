Релиз открытой системы композитинга Natron 1.2.0

Доступен релиз Natron 1.2.0, открытой кроссплатформенной системы композитинга на основе нодовой архитектуры, позволяющей создавать качественные визуальные эффекты на уровне инструментов постпроцессинга, используемых при съёмке современных фильмов. Система предоставляет средства для совмещения нескольких слоёв видеоматериалов, наложения эффектов и финальной сборки отснятого видео. Например, Natron можно использовать для совмещения слоя с действиями актёров на зелёном фоне со сценой, сформированной в системе 3D-моделирования. Код Natron распространяется под лицензией Mozilla Public License 2.0. Бинарные сборки подготовлены для различных дистрибутивов Linux, FreeBSD, OS X и Windows. Системой предоставляются полноценные средства привязки нод к ключевым точкам (предлагается несколько алгоритмов) и средства ротоскопирования по ключевым кадрам. Интерфейс построен в стиле пакета Nuke и использует те же принципы работы, в том числе доступен режим интерактивного предпросмотра. Обеспечена поддержка существующих OpenFX-плагинов c реализацией различных визуальных эффектов (поддержка стандарта OpenFX v1.3). Возможна работа с изображениями EXR и DPX. Доступны механизмы привлечения GPU для ускорения выполняемых операций и средства многопоточного рендеринга. Поддерживается интеграция с системой 3D-моделирования Blender. В новом выпуске в интерфейсе просмотра обеспечена поддержка применения всех эффектов трансформации для областей перекрытия (overlay), добавлена возможность настройки используемых в интерфейсе цветов, улучшены средства для работы с использованием планшетов Wacom, добавлены новые ноды ImageStatistics и CheckerBoard, продолжено развитие инструментов для написания скриптов на языке Python.
