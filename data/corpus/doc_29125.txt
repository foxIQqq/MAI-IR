Вышел Finnix 101, Live-дистрибутив для системных администраторов

Доступен релиз минималистичного Live-дистрибутива для системных администраторов - Finnix 101, который содержит обновление ПО, соответствующее состоянию Debian testing от 20 декабря. Дистрибутив поддерживает только работу в консоли, но содержит хорошую подборку консольных утилит для нужд администратора. Размер iso-образа - 127 Мб, при этом без учета сжатия в комплект входит около 350 Мб утилит, например, в состав включены LVM2, cryptsetup, cdpr, iftop, irssi, OTP calculator, VLAN tools и т.д. В новой версии: Осуществлено обновление Linux ядра до версии 2.6.36; Возобновлена поддержка и сборка пакетов для архитектуры PowerPC; Переработаны связанные с формированием сборок компоненты и реорганизована внутренняя структура проекта. Например, изменено размещение файлов на CD, подготовлено окружение для создания собственных переработанных сборок, подготовлена специфичная для Finnix SysV-совместимая система инициализации со специально оптимизированными RC-скриптами. В загрузочное меню добавлена утилита Hardware Detection Tool (HDT), позволяющая на раннем этапе загрузки (до начала загрузки ОС) сформировать полное описание состава текущего аппаратного обеспечения.
