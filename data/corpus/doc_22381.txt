Релиз интегрированной среды разработки NetBeans IDE 6.7

Вышел релиз NetBeans IDE 6.7, интегрированной среды разработки приложений на языках Java, PHP, Ruby, JavaScript, Python, Groovy и C/C++, распространяемой в исходных текстах под лицензиями GPLv2 и CDDL. Основные новшества: Интеграция поддержки SourceForge-подобной платформы для совместной разработки и хостинга открытых проектов Kenai.com, созданной Sun Microsystems, поддерживающей системы контроля версий SVN, Mercurial и Git. Теперь управлять проектами и создавать проекты Kenai можно через встроенные инструменты NetBeans; Поддержка работы с ПО Apache Maven для управления проектами и организации процесса сборки: поддерживается построение дерева зависимостей для библиотек, доступны средства для создания Java EE программ и Web-сервисов; Полная интеграция с Bugzilla; Поддержка JavaScript 1.7; Возможность удаленной отладки Ruby On Rails приложений; Поддержка Grails 1.1, реализация режима автодополнения кода для Grails; В модуле разработки для Java ME появилась полная поддержка работы с SVG компонентами в режиме Visual Mobile Designer ; Улучшение интеграции GlassFish Поддержка разработки C/C++ проектов с использованием библиотеки Qt; Система автоматического профилирования кода "Profile Me Now!" с возможностью экспорта отчетов в форматах CSV, HTML и XML ; Для разработчиков на языке PHP добавлена поддержка фреймворка Selenium, в PHP-редакторе появилась возможность автодополнения кода в SQL-вставках, улучшена поддержка PHPUnit; Реализация поддержки сервиса автоматической сборки Java-проектов через интеграцию с ПО Hudson; Поддержка Zembly, единого репозитория популярных Web API. NetBeans позволяет создать каркас Java и JavaFX приложения для работы с заданным API через "Drag and Drop" интерфейс.
