Выпуск дистрибутива KaOS 2022.02

Представлен выпуск KaOS 2022.02, дистрибутива с непрерывной моделью обновления, нацеленного на предоставление рабочего стола на основе свежих выпусков KDE и приложений, использующих Qt. Из специфичных для дистрибутива особенностей оформления можно отметить размещение вертикальной панели в правой стороне экрана. Дистрибутив развивается с оглядкой на Arch Linux, но поддерживает собственный независимый репозиторий, насчитывающий более 1500 пакетов, а также предлагает ряд собственных графических утилит. В качестве файловой системы по умолчанию применяется XFS. Сборки публикуются для систем x86_64 (3 ГБ). В новом выпуске: По умолчанию задействован сеанс KDE на основе протокола Wayland. Компоненты рабочего стола обновлены до KDE Plasma 5.24, KDE Frameworks 5.91.0, KDE Gear 21.12.2 и Qt 6.2.3 (также доступен вариант Qt 5.15.3, сопровождаемый проектом KDE). Предложен новый интерфейс для настройки мониторов, упрощён процесс перемещения панелей в разные части экрана и включён новый обзорный режим для просмотра содержимого виртуальных рабочих столов и оценки результатов поиска в KRunner. Из-за проблем с поддержкой Wayland медиапроигрыватель SMplayer заменён на Haruna, который также является надстройкой над MPV. Из дополнительных возможностей отмечается интеграция с yt-dlp для загрузки видео с YouTube. В качестве офисного пакета вместо Calligra по умолчанию предложен LibreOffice с бэкендом отрисовки на базе Qt5/kf5. В инсталляторе Calamares реализован вывод предупреждений при выявлении конфликтов при разбивке дисковых разделов. В состав включён новый календарь-планировщик Kalendar, предоставляющий средства для управления задачами и событиями, и поддерживающий интеграцию с внешними календарями на базе Nextcloud, Google Calendar, Outlook и Caldav. Обновлены версии программ, в том числе Glibc 2.33, GCC 11.2, Perl 5.34.0, PHP 8.1.2, GStreamer 1.20.0, ядро Linux 5.15.23, Systemd 250.3, Curl 7.81.0, Mesa 21.3.6, Wayland 1.20.0, Sudo 1.9.9 и Openldap 2.6.1.
