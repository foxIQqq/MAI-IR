Вышло обновление 0.9.0 openSUSE-сборки Hackeurs Sans Frontieres

Доступен четвёртый публичный выпуск Live-сборки Hackeurs Sans Frontières, основанной на пакетной базе openSUSE и включающей подборку готовых решений для обхода цензуры и избежания слежки за жителями государств, в которых применяются средства контроля трафика. В состав также входит большая подборка программ общего назначения и инструментов для диагностики и восстановления после сбоев, а также поддержка звукового сервера JACK из коробки (с сохранением совместимости с Pulse, ALSA и OSS). Размер iso-образа 2.8 Гб. Из интересных изменений: Переход на пакетную базу openSUSE Tumbleweed/Factory (обновления пакетов от 2016-03-23) и ядро Linux 4.5.0 (своя сборка, заменяющая брошенный openSUSE "desktop flavour" и имеющая практически все модули ядра). Теперь официальный репозиторий считается более приоритетным, чем большинство репозиториев для разработчиков. Включена поддержка записи для носителей, не ограниченных режимом только для чтения. Для этого создаётся R/W ext4-раздел из "лишнего" пространства в добавок к основному, R/O squashfs-разделу. Возможно повреждение R/W раздела при перезагрузках из-за ошибки systemd при работе с overlayfs, поддержку записи можно выключить параметром загрузки 'kiwi_hybridpersistent=false'. Также можно попробовать лишить этот раздел метки "hybrid", чтобы загрузочный скрипт не смог его использовать. Теперь минимальным размером целевого устройства считается 4Гб, в образ включается вся основная документация, базовые пакеты для сборки и полезные 32-битные пакеты, такие как wine (с поддержкой Gallium D3D9) и установщик Steam. Изменён основной набор шрифтов: в глобальном приоритете семейство Google Noto, Terminus использован в ядерном и графическом терминалах, Hack - в текстовых редакторах, а DejaVu - в интерфейсе. Вид оформления в целом приведён к тёмным тонам с яркими детальными иконками. В сборке Mesa включена поддержка OpenMAX IL, VAAPI & VDPAU, но по-умолчанию не используется из-за несовместимости с продвинутой обработкой изображения mpv и серьёзными проблемами открытых драйверов. Включена открытая поддержка OpenCL (AMD & Intel)/ Обеспечено высококачественное проигрывание midi через fluidsynth/JACK. Добавлена поддержка UPnP для PulseAudio посредством pulseaudio-dlna. Bomi заменён на связку Xt7/mpv, а в mpv включена поддержка интерполяции на основе движения (генерация "пропущенных" кадров для достижения 60 fps, работает только на самых мощных процессорах) и масштабирования с помощью алгоритма nnedi3. Добавлены новые модули YaST, включая поддержку Docker. Добавлена коллекция эмуляторов практически для всех консолей: mednafen - мультисистемный эмулятор с поддержкой JACK DOSBox для IBM PC/DOS OpenMSX для MSX PPSSPP для Sony PSP PCSX-R для Sony PlayStation PCSX 2 для Sony PlayStation 2 Dolphin-Emu для Nintendo Gamecube & Wii Desmume для Nintendo DS Mupen64Plus для Nintendo N64 Yabause для Sega Saturn Добавлен rEFInd (графический загрузчик для EFI-систем). Добавлена программа записи компакт-дисков K3B. Добавлен просмотрщик карт Marble. Добавлен Skype (только для совместимости, не рекомендуется к использованию) и просмотрщик его данных. Добавлена поддержка ZFS (только для совместимости, не рекомендуется к использованию, может быть удалена при проблемах со сборкой модулей ядра). Добавлена программа для исследования остаточных данных оперативной памяти volatility. Возвращён PeaZip до тех времён, когда родное GUI p7zip решит начать компилироваться. Заменены Bumblebee/Primus/VirtualGL/bbswitch в пользу PRIME/DRI3. Заменены LMT/pm-profiler/pm-utils в пользу tuned/systemtap/powertop. Заменён tc-play в пользу veracrypt. Заменён uTox в пользу qTox. Удалён toxic, консольный клиент Tox. Удалён conky из-за устаревания и неработоспособности с актуальной версией Lua. Удалён скрипт live-{fat-stick,usb-gui}. Начато использование нескольких DNS-тоннелей dnscrypt одновременно для повышения отказоустойчивости. Запрещены расточительные автоматические coredump'ы. Firefox и Thunderbird собираются без официальных тем, чтобы избежать удалённого блокирования дополнений компанией Mozilla, удалены патчи поддержки KDE4. Общие исправления пакетов и файлов конфигурации. Из интересных изменений от версии 0.8.0 до 0.8.1: Миграция LXQt на Qt5. Исправление отсутствия графического интерфейса в YaST2-firstboot. Добавление IPTV-плеера Tano на основе VLC. Добавление efitools and UEFITool для манипуляций U/EFI. Удаление поддержки VAAPI в GStreamer из-за её автоматического включения, способного привести к краху графических драйверов. Заменён кеширующий proxy-сервер polipo на privoxy (для работы Tor). Заменён unetbootin на imagewriter и live-{fat-stick,usb-gui}. Старый NPAPI flash-плагин заменён на PPAPI от Хрома (через freshplayerplugin, есть проблемы с flash-плеерами видео). Замена gnote на keepnote из-за удаления в GNOME поддержки системного лотка. Замена n/oping на fping из-за падений. Замена guvcview на qv4l2 по аналогичной причине. Замена PeaZip на xarchiver. Замена Calibre на CoolReader3-Qt5. Удаление XScreenSaver из-за проблем с его подавлением плеерами и браузерами. Удаление SIP Witch. Активирована поддержка mDNS по-умолчанию. Firefox принуждён к использованию OMTC и выключению чёрного списка видеодрайверов (иначе HTML5-видео работает отвратительно). Изменения только для версии 0.8.1 (но не новее): Поддержка записи отключена для совместимости с медленными и невместительными (до 2Гб) Flash-накопителями. Удалены базовые сборочные пакеты.
