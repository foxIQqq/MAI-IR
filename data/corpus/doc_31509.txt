Для CentOS 5.6 доступны обновления, выпущенные для RHEL 5.7

Разработчики проекта CentOS объявили о создании непрерывно обновляемого репозитория (Continuous Release) для ветки CentOS 5.6, который позволит установить связанные с безопасностью исправления, выпускаемые для уже вышедшего релиза Red Hat Enterprise Linux 5.7, на базе которого будет сформирован выпуск CentOS 5.7. Релиз CentOS 5.7 ожидается через 7-10 дней, тем не менее всем пользователям CentOS 5.6 настоятельно рекомендуется не дожидаясь релиза обновить свои системы из представленного CR-репозитория. В связи с задержкой выхода CentOS 5.7 обновления для CentOS 5.x не выпускаются уже почти месяц. В начале года отставание релиза CentOS 5.6 от RHEL 5.6 составило 82 дня, все это время пользователи оставались без обновлений с исправлением проблем безопасности. На этот раз, желая не повторить печальный опыт, было решено провести перепаковку для CentOS 5.6 обновлений, выпускаемых для RHEL 5.7. После релиза CentOS 5.7 временный CR-репозиторий будет удалён. Для активации CR-репозитория нужно установить специальные пакеты - i386 или x86_64. Аналогичный CR-репозиторий планируется создать и для версии CentOS 6.0. Релиз RHEL 6.1 был выпущен 19 мая, поэтому пользователи CentOS 6.0 в настоящее время лишены возможности установки пакетов с исправлением проблем безопасности. Для сравнения, проект Scientific Linux изначально выпускает обновления в непрерывном цикле, не завися от последней ветки RHEL.
