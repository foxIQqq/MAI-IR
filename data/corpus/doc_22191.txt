Вышел финальный релиз Jython 2.5.0

"Jython 2.5.0 Final is out!" - вышел финальный релиз Jython 2.5.0, реализации языка Python 2.5 на Java. В отличие от стандартного CPython, Jython транслирует исходные тексты Python скриптов в Java байткод , выполняющийся внутри виртуальной машины Java и не отличающийся от байткода Java программ, что позволяет использовать готовые Java библиотеки в Python программах и наоборот.
