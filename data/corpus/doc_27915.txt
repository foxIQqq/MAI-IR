Вышел релиз LaTeX-дистрибутива TeX Live 2010

Вышел релиз популярного LaTeX-дистрибутива TeX Live 2010, созданного в 1996 году на базе проекта teTeX. TeX Live является наиболее простым способом для развертывания инфраструктуры для подготовки научной документации, независимо от используемой операционной системы. Полная DVD-сборка TeX Live 2010 занимает около 1.9 Гб и содержит рабочее Live-окружение, полный набор установочных файлов для различных операционных систем, копию репозитория CTAN (Comprehensive TeX Archive Network), подборку документации на разных языках (включая русский). Размер iso-образа вырос по сравнению с прошлым годом на 700 Мб, поэтому разработчики дистрибутива рекомендуют использовать метод сетевой установки, при котором копируются только необходимые для заданной системы пакеты.
