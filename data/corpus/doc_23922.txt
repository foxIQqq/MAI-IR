Вышел второй кандидат в релизы SeaMonkey 2.0

Вместо ожидаемого сегодня финального выпуска представлен второй, дополнительный, кандидат в релизы SeaMonkey 2.0. Если не возникнет непредвиденных проблем, SeaMonkey 2.0-rc2 через несколько дней переродится в финальный релиз, который ожидается 26 октября. SeaMonkey объединяет в одном продукте популярные программы, разрабатываемые в рамках проекта Mozilla: web-браузер на основе Firefox 3.5, программа для чтения почты на основе Thunderbird, WYSIWYG редактор html страниц, IRC клиент ChatZilla, набор средств для разработчиков, включающих DOM inspector и JavaScript отладчик. Основные отличия новой версии: Используется JavaScript движок TraceMonkey и базовая начинка, основанная на коде Firefox 3.5.4; Отмечено значительное улучшение пользовательского интерфейса; Добавлена возможность архивирования почты. Добавлена поддержка вкладок в почтовом клиенте (например, в табах можно открыть папки, отдельные сообщения, и даже календарь, доступный через установку расширения Lightning); Добавлен новый менеджер тем и дополнений, напоминающий по возможностям менеджер дополнений Firefox; Полностью переработан менеджер загрузок, который теперь поддерживает возобновление загрузки после закрытия сессии. Улучшена работа менеджера сохраненных паролей. В системе просмотра cookie появилась возможность поиска по имени cookie и сайту; Для web-форм теперь автоматически запоминаются введенные значения, которые предлагаются при последующем вводе через выпадающее меню; Прекращена поддержка Win9x и MacOS 10.3, из-за перехода на GTK 2.10 и Cairo. Добавлена возможность отката операции закрытия окна браузера; Добавлена поддержка упрощенного интерфейса для разработчиков расширений (SMILE); Изменился путь хранения профиля. Старые профили, за исключением дополнений, тем и плагинов, подвергнутся миграции, их данные не будут затронуты; Изменен метод хранения истории посещений, добавлен интеллектуальный инструмент выбора релевантных web-страниц при формировании запроса в строке ввода адреса (как в Firefox); Панель инструментов в браузере, почтовом клиенте теперь полностью настраиваема пользователем, включая поддержку изменения размера пиктограмм; В программе для чтения почты и news-конференций появилась поддержка RSS и Atom лент;
