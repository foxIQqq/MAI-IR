Названы лауреаты премии, отметившей лучшее открытое ПО. Nginx в числе победителей

Объявлены призеры ежегодного конкурса издания InfoWorld, нацеленного на выявление лучшего открытого ПО - Best of Open Source Software Awards 2010. В качестве критерия отбора программ использовалась оценка полезности приложения как такового для определенной аудитории (корпоративных пользователей, разработчиков, сетевых администраторов). В роли экспертов, выбравших лучшие продукты из всего многообразия открытых решений, выступали редакторы и обозреватели издания InfoWorld. Примечательно, что среди победителей отмечены два отечественных проекта - http-сервер nginx и изолированные контейнеры OpenVZ. Победители выбраны в 4 категориях: Лучшие открытые программы (для предприятий). В нынешнем году был отмечен качественный скачок в развитии ERP, CRM, систем управления документами и бизнес-приложений, что отразилось на составе лауреатов. Система планирования ресурсов предприятия Openbravo ERP; CRM-система SugarCRM, предназначенная для организации взаимодействия с клиентами на предприятии, обеспечения работы службы продаж, ведения партнерской сети, предоставления сервисов и планирования работы сотрудников; Система бизнес-аналитики Pentaho BI Suite; Среда управления контентом предприятия Alfresco ECM; Система управления web-контентом Drupal; Движок для создания блогов WordPress; Система управления документами LogicalDoc; Специализированный wiki-движок с элементами для организации совместной работы Tiki Wiki CMS Groupware; Платформа для создания поискового сервера масштаба предприятия Apache Solr; Лучшее открытое ПО для разработки приложений Язык программирования Go; Реализация языка Ruby для виртуальной машины Java - JRuby; PHP-фреймворк для разработки web-приложений Zend Framework; JavaScript-библиотека JQuery; JQuery-плагин для создания web-приложений для мобильных устройств с сенсорным экраном - jQTouch;
