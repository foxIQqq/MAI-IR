Релиз Linux дистрибутива Salix 13.1

Вышел релиз Linux-дистрибутива Salix 13.1, развиваемого создателем Zenwalk Linux, покинувшим проект в мае прошлого года в результате конфликта с другими разработчиками, отстаивавшими политику максимальной похожести на Slackware. Дистрибутив Salix 13.1 на 100% совместим со Slackware Linux 13.1, исповедует подход "одно приложение на одну задачу" и поддерживает три режима установки: Полная - десктоп окружение на основе Xfce, Firefox, почтовый клиент Claws-mail, полная версия офисного пакета OpenOffice.org, Java Runtime Environment, мультимедиа плеер Totem, ПО для ведения архива музыки Exaile, пакетный менеджер gslapt; Основная - Xfce, Firefox и пакетный менеджер gslapt; Базовая - только минимальный набор консольных утилит, подходит для создания серверов. В отличие от прошлых версий установочный iso-образ подготовлен не только для 32-разрядных систем, но и в виде 64-разрядной сборки (размер 560 Мб).
