Вышел релиз LiveCD дистрибутива Parted Magic 4.2

Вышел релиз LiveCD дистрибутива Parted Magic 4.2, предназначенный для восстановления системы после сбоя. Дистрибутив построен на базе Linux From Scratch (LFS) и графического окружения Xfce. ISO-образ занимает 91 Мб и подходит для использования на USB Flash, CD или для загрузки по сети (PXE). Основные новшества Parted Magic 4.2: В дистрибутив интегрирован пакет Clonezilla, предназначенный для быстрого клонирования дисков (копируются только используемые блоки) и выступающий в роли свободного аналога Norton Ghost. Clonezilla можно вызывать прямо из рабочего окружения Parted Magic, без использования двойной загрузки (изначально Clonezilla поставляется в виде LiveCD); Исправлена ошибка, мешающая сохранению сессий при работе с LiveUSB; Обновлены версии входящих в комплект программ: linux ядро 2.6.30, lftp-3.7.14, rsync-3.0.6, util-linux-ng-2.15.1, fsarchiver-0.5.7, gpicview-0.1.99, gsmartcontrol-0.8.4, ntfs-3g-2009.4.4AR.10, TrueCrypt-6.2, reiser4progs-1.0.7, progsreiserfs_0.3.0.5, safecopy-1.3, conky-1.7.1.1, ext3grep-0.10.1, sshfs-fuse-2.2, testdisk-6.11-3, galculator-1.3.4, syslinux-3.82, grsync-0.9.0, firefox-3.5b99; В комплект включены программы: curl-7.19.5, partclone_0.1.1-10, drbl-1.9.4-33, awk-3.1.6, sed-4.1.5, grep-2.5.3, tightvnc-1.3.9 (vncviewer), mkswap-uuid-0.1.0-2. Из программ в комплекте можно отметить: GParted, TestDisk, PhotoRec, Partition Image, Xarchive, Xfburn, ISO Master, Leafpad, Firefox, Grsync. Из утилит командной строки присутствуют dd, ddrescue , cfdisk, fdisk, tcpdump. Имеется поддержка восстановления файловых систем ext2, ext3, ext4, btrfs, FAT16, FAT32, HFS, HFS+, JFS, NTFS, ReiserFS, Reiser4, XFS.
