Вышел релиз Linux дистрибутива openSUSE 11.2

Официально представлен релиз Linux дистрибутива openSUSE 11.2, первый выпуск в рамках нового восьмимесячного цикла подготовки релизов. По сравнению с прошлыми версиями время поддержки дистрибутива (выпуска обновлений) openSUSE 11.2 сокращено с 24 до 18 месяцев. Обзор основных улучшений: Пользовательские улучшения В качестве десктоп-окружения по умолчанию, при установке с DVD, используется KDE 4.3.1. В настоящий момент разработчики KDE выпустили версию 4.3.3, но это не должно смущать пользователей openSUSE, так как в дистрибутив импортированы все важные исправления, доступные для ветки KDE 4.3. В качестве второго десткоп-окружения доступен GNOME 2.28. Для GNOME и KDE обновлены визуальные темы, причем для KDE проведена большая работа по обеспечению бесшовной интеграции в рабочее окружение таких приложений, как Firefox и OpenOffice.org; В интерфейсе для настройки параметров дисплея gnome-display-properties добавлена поддержка автоматического сброса настроек в случае проблем. Улучшение поддержки субноутбуков. Обновление входящих в комплект программ Пользовательские приложения: SeaMonkey 2.0, Firefox 3.5.5, xorg-x11-server 1.6.5, Banshee 1.5.1, K3b 1.68 Alpha 3, Evolution 2.28, X.Org драйвер radeonhd 1.3, Blender 2.49a, OpenOffice.org 3.1.1, Gnumeric 1.9.12, KTorrent 3.2.4, Pidgin 2.6.2, Thunderbird 3.0 Beta 4, VirtualBox 3.0.6, Wine 1.1.28, GIMP 2.6.7; Серверные и системные приложения: Linux ядро 2.6.31.5, Samba 3.4.2, Bluez 4.56, Alsa 1.0.21, PulseAudio 0.9.19, GCC 4.4.1, RPM 4.7.1; Программы для разработчиков: Subversion 1.6.6, Qt 4.5.3, PostgreSQL 8.4.1, php 5.3.0, Apache 2.2.13, Python 3.1; В состав LiveCD с GNOME добавлен Midnight Commander, Bittorrent-клиент Transmission и Gimp. В состав LiveCD с KDE добавлены новые программы: Choqok, KNode, Kompare, KSystemlog, Okteta, YaKuake. В комплект включены приложения для работы с десктопа в социальных сетях, подобных Facebook, Twitter и identi.ca. Например: choqok (Twitter и identi.ca клиент для KDE), gwibber (Twitter, Facebook и identi.ca клиент для GNOME), поддержка Facebook в kopete, дополнительные KDE4 плазмойды для общения в Twitter и identi.ca; Установка и управление пакетами При установке в качестве файловой системы по умолчанию задействована Ext4. Более того, в YaST добавлена возможность полного шифрования всего диска. В менеджер дисковых разделов YaST добавлена поддержка файловой системы Btrfs и переработан интерфейс; Новое визуальное оформление инсталлятора YaST2; Образ LiveCD теперь может быть использован и на USB Flash. В состав интегрированы средства для подготовки собственных LiveCD образов. Для сжатия данных на LiveCD отныне используется алгоритм сжатия LZMA; По умолчанию LiveCD содержит переводы для немецкого, французского, итальянского, польского и русского языков; Настройка и управление пакетами Добавлена функция совершения обновления в Live-режиме с минимальным временем простоя, позволяющая целиком обновить работающую систему с версии 11.1 до 11.2 методом напоминающим dist-upgrade в Debian, с фоновой загрузкой пакетов перед непосредственным началом проведения обновления; Разработан web-интерфейс к YaST для осуществления удаленного управления дистрибутивом; Для YaST создан новый центр управления конфигурацией (YaST Control Center) с интерфейсом на базе Qt4; В Qt версии конфигуратора YaST появилась поддержка настраиваемого внешнего вида. В YaST добавлены средства работы с кешем пакетов; Пакеты дистрибутива собраны GCC 4.4.1. Пакеты сформированы с использование RPM версии 4.7.1; В zypper добавлена опция "--download-only", позволяющая осуществить только загрузку компонент, без выполнения действи. Опцию можно использовать совместно с командами install, update, patch, dup и verify. Улучшены средства работы с зеркалами репозиториев пакетов. Улучшена производительность операций индексации и поиска в zypper.;
