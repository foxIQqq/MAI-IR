Вышел релиз Arora 0.10, легковесного web-браузера на базе движка WebKit

Увидел свет релиз web-браузера Arora 0.10.0, основанного на QtWebKit (WebKit, включенный в комплект Qt 4.5) и распространяемого в рамках лицензии GPLv2. В новой версии исправлено около 50 ошибок и реализовано несколько интересных новшеств: В состав браузера интегрирована система AdBlock для блокирования рекламы на web-страницах. В качестве эталона необходимой функциональности при разработке системы использован Firefox плагин Adblock Plus (поддерживаются все основные возможности, включая загрузку внешних списков блокировки); Добавлена возможность запоминания введенных в формах паролей и автозаполнения форм; Реализован режим отображения содержимого директорий локальной файловой системы; Проведена работа над ошибками, мешающими использованию web-сервисов Google. Основные возможности: Многоплатформенность, браузер доступен для Linux, FreeBSD, Windows, Mac OS X, различных мобильных ОС. Несмотря на использование Qt, прекрасно вписывается в интерфейс GNOME и XFCE; Браузер упрощенный, содержит только функции первой необходимости. Изначально развивался как демонстрация использования API QtWebKit. В настоящий момент для полноценной работы не хватает только проверки орфографии в формах; Поддерживается: интерфейс на базе табов; режим приватной навигации без оставления следов; менеджер Cookies; возможность хранения закладок и просмотра истории посещений; менеджер загрузок; поисковые плагины OpenSearch; Netscape-совместимые плагины, например, можно подключить Adobe Flash; встроенный набор инструментов для web-разработчиков (WebInspector); возможность управления сессиями; задание ключевых слов для поиска прямо в адресной строке и сопоставления ключей для обращения к заданным поисковым системам; функция ClickToFlash, для проигрывания Flash-роликов только по запросу пользователя (нужно кликнуть на картинке);
