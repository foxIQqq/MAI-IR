Выпуск дистрибутива Redcore Linux 2101

После года разработки опубликован релиз дистрибутива Redcore Linux 2101, который пытается совместить функциональные возможности Gentoo с удобством для обычных пользователей. Дистрибутив предоставляет простой инсталлятор, позволяющий быстро развернуть рабочую систему, не требуя пересборки компонентов из исходных текстов. Пользователям предоставляется репозиторий с готовыми бинарными пакетами, сопровождаемый с использованием непрерывного цикла обновлений (rolling-модель). Для управления пакетами задействован собственный пакетный менеджер sisyphus. Для установки предлагается iso-образ с рабочим столом KDE, размером 3.6 ГБ (x86_64). В новой версии: Выполнена синхронизация с деревом Gentoo testing по состоянию на 31 мая. Для установки на выбор предложены пакеты с ядром Linux 5.11.22, 5.10.40 LTS и 5.4.122 LTS. Обновлены версии glibc 2.32, gcc 10.2.0, binutils 2.35, llvm 12, mesa 21.1.1, libdrm 2.4.106, xorg-server 1.20.11, alsa 1.2.5, pulseaudio 13.0, gstreamer 1.16.3 KDE Plasma 5.21.5, KDE apps 21.04.1. Из браузеров предложены firefox 89.0, chrome/chromium 91, opera 76, vivaldi 3.8, microsoft-edge 91 и falkon 3.1.0-r1. Добавлена поддержка самодостаточных пакетов в формате flatpak. Убрана необходимость ввода пароля при загрузке в Live-режиме. В состав включены пакеты open-vm-tools (работа с виртуальными машинами vmWare) и spice-vdagent (агент с поддержкой протокола удалённого доступа SPICE для QEMU/KVM). В пакетном менеджере sisyphus обновлена документация. При переходе между ветками master (stable) и next (testing) реализовано запоминание заданных пользователем USE-флагов, ключевых слов и масок. Изменена логика обновлений - sisyphus теперь не пытается обновить систему до ветки "testing", а поддерживает пакеты в актуальном виде на базе ветки stable.
