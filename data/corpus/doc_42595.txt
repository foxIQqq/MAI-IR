Выпуск KDE Frameworks 5.12.0

Представлен выпуск платформы KDE Frameworks 5.12.0, в рамках которой развивается реструктуризованный и портированный на Qt 5 базовый набор библиотек и runtime-компонентов, лежащих в основе KDE. Фреймворк включает в себя 63 библиотеки, часть которых может работать в качестве самодостаточных надстроек над Qt, а часть формируют программный стек KDE. Подробнее с особенностями KDE Frameworks 5 можно познакомиться в анонсе первого выпуска. Статус подготовки бинарных сборок KDE Frameworks 5.12 можно оценить на данной странице. Протестировать без установки можно через Live-сборку от проекта Kubuntu. Из изменений можно отметить: В KActivities добавлена опция для отключения сохранения лога активности пользователя для выбранных комнат (activities). В KAuth реализована возможность применения kauth-policy-gen из различных исходных текстов. В KConfig добавлена новая опция генерации кода TranslationDomain и возможность применения kconfig_compiler из различных исходных текстов. В KCoreAddons добавлена возможность фильтрации плагинов и приложений по формфактору. KDeclarative представлен QQC-клон кнопок KColorButton, обеспечено прикрепление при возможности QmlObject для каждого экземпляра kdeclarative, осуществлено вливание кода из репозитория mart/singleQmlEngineExperiment. В KGlobalAccel добавлен внутренний runtime API, необходимый для работы в KWin плагина с поддержкой Wayland. В KItemModels добавлена новая модель прокси KRearrangeColumnsProxyModel, поддерживающая перегруппировку и скрытие столбцов в исходной модели; В KNotification реализован способ определения действия по его имени; В KPeople реализованы элементы для фильтрации по PersonsModel В KTextEditor для QScrollBar добавлена поддержка обработки событий от колеса мыши. В KWallet Framework расширены возможности утилиты kwallet-query, добавлена поддержка перезаписи записей в таблице сопоставления; В Plasma Framework представлен новый класс ContainmentView, переработана тёмная тема оформления, реализован отдельный кэш для отмасштабированных и исходных элементов, улучшена поддержка HiDPI. В Sonnet добавлена подсветка ошибок проверки орфографии для форм ввода, реализованных при использовании QPainTextEdit.
