Вышел пакет для организации коллективной работы Open-Xchange 6.14

Представлен релиз Open-Xchange 6.14, открытой системы для организации коллективной работы. Работа в пакете производится через web-интерфейс, поддерживающей функции системы документооборота, чтения почты, форума, календаря-планировщика, базы знаний и единой адресной книги. С обзором базовых возможностей пакета можно познакомиться в тексте анонса прошлой версии. Из новшеств добавленных в версии 6.14 можно отметить: Добавлен инструмент для упрощения настройки аккаунтов для существующих и новых пользователей. Реализована поддержка настройки получения почты из внешних сервисов, таких как Gmail, GMX, Web.de, а также для загрузки сообщений из социальных сетей подобных Facebook, LinkedIn и Xing. Кроме того, значительно упрощен процесс синхронизации данных c мобильных телефонов; В календаре-планировщике реализован новый режим для просмотра групповых планов; Проведена работа по улучшению юзабилити пользовательского интерфейса, основанного на технологии AJAX; Улучшена работа c Microsoft Outlook модуля OXtender (плагин для почтового клиента Mozilla Thunderbird).
