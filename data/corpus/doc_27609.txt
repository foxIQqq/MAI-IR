Релиз открытой микроядерной ОС Genode 10.08

Вышел релиз открытой микроядерной операционной системы Genode OS Framework 10.08. Genode предоставляет разработчикам унифицированную инфраструктуру для создания пользовательских приложений, работающих поверх собственного микро-ядра или ядер Linux (32 и 64 бит), OKL4v2 (x86 и ARMv4), L4/Fiasco и L4ka::Pistachio. Входящее в состав паравиртуализированное Linux ядро OKLinux позволяет выполнять в Genode обычные Linux программы. Ядро OKLinux не работает с оборудованием напрямую, а использует сервисы Genode через набор виртуальных драйверов. Интегрированный движок Qt4/WebKit дает возможность разработчикам создавать Genode программы, выполненные в виде web-приложений. В новой версии представлено несколько значительных улучшений, таких как новая инфраструктура для создания драйверов блочных устройств, новый драйвер PATA/SATA, портирование из Linux драйвера MadWifi, переход на использование Qt 4.6.3 и интеграция графической подсистемы Gallium3D для упрощения написания видеодрайверов с поддержкой 3D-акселерации (в настоящий момент обеспечена только поддержка карт Intel GMA ).
