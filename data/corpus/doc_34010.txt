Началось бета-тестирование KDE 4.9 и альфа-тестирование Calligra 2.5

Представлена бета-версия десктоп-окружения KDE 4.9. Сообщается, что кодовая база KDE 4.9 переведена на стадию заморозки, при которой вся запланированная функциональность уже реализована и усилия разработчиков переключены на тестирование, исправление ошибок и оттачивание новых возможностей. Релиз запланирован на 1 августа. Среди наиболее интересных улучшений KDE 4.9 можно отметить: Продолжена работа по переводу компонентов рабочего стола Plasma на использование технологии декларативного описания интерфейса Qt Quick. Компоненты на базе Qt Quick позволяют добиться большой гибкости в изменении внешнего оформления, интерфейс полностью отделён от кода и легко подстраивается под различные классы устройств. Особенности работы созданных с использованием Qt Quick виджетов ничем не отличаются от виджетов, созданных с использованием классического API Plasma, но виджеты на Qt Quick имеют более привлекательное визуальное оформление и лучше поддерживают работу на устройствах с сенсорными экранами. В новой версии представлен ряд модулей с реализацией для Qt Quick разнообразных новых возможностей API Plasma. В состав включён новый хранитель экрана, переписанный с использованием QML, переписан плазмоид Microblog, добавлен новый диалог завершения работы; В файловом менеджере Dolphin улучшены средства для отображения, сортировки и поиска на основе метаданных. Например, для элементов в различных режимах отображения теперь показываются такие метаданные, как рейтинг, теги, комментарии, разрешение изображений, имена музыкантов и т.п. Файлы теперь можно переименовывать на месте, без открытия дополнительных диалогов; Обеспечена более глубокая интеграция комнат (Activity) с файлами, окнами и другими ресурсами: пользователю теперь значительно проще привязать любые файлы и окна к заданной комнате. В окне Folderview теперь можно отобразить файлы, связанные с указанной комнатой, что упрощает группировку файлов по контексту; В kdelibs переработан код настройки цветов и реализована поддержка создания приватных комнат (Activity) с зашифрованным контентом (для данного режима пока не создан пользовательский интерфейс); Обеспечена поддержка D-Bus интерфейса MPRIS2 (Media Player Remote Interfacing Specification), предназначенного для организации удалённого управления медиаплеерами. На MPRIS2 теперь базируется новый плазмоид Now Playing, переписанный с использованием QML. Поддержка MPRIS2 также добавлена в плееры JuK и Dragon; В Gwenview представлен режим полноэкранной навигации; В Okular обеспечена поддержка интегрированных в PDF-документы аннотаций, реализован более агрессивный режим предзагрузки страниц, при поиске теперь игнорируется дефис, добавлена возможность создания нескольких закладок на одной странице, по умолчанию поиск теперь начинается с текущей страницы, а не с начала документа; В Kopete добавлена поддержка переименования имён в списке контактов. В окне чата обеспечен показ смены статуса. Добавлена опция для группировки всех неактивных контактов в блок "Offline Users"; Проведена интеграция нескольких разработок отечественной компании Роса: апплет с реализацией режима StackFolder, предназначенного для организации быстрого доступа к наиболее используемым каталогам и файлам; приложение KLook для быстрого просмотра группы файлов; Многочисленные улучшения производительности и исправления ошибок. Дополнительно можно упомянуть выход первой альфа-версии развиваемого под эгидой проекта KDE офисного пакета Calligra 2.5, продолжающего развитие KOffice. Начиная с выпуска 2.5 проект Calligra перешел на новую модель разработки, в рамках которой новые возможности будут помещаться в основное дерево исходных текстов только после полной готовности и стабилизации, что позволит сократить цикл подготовки релиза. Например, версия Calligra 2.5 ожидается уже в июле, в то время как на разработку прошлой версии ушло больше года. До момента готовности новшества будут развиваться в отдельных ветках. Основные новшества Calligra 2.5-alpha: В Words улучшена поддержка редактирования таблиц, обеспечено компактное обтекание текста вокруг изображений, расширены библиографические инструменты, добавлена возможность манипулирования границами таблиц, реализована функция перетаскивания блоков текста мышью. В Stage, программе для подготовки презентаций, проведена работа по улучшению юзабилити; Во Flow, программе для построения диаграмм, добавлена поддержка новых шаблонов для создания определённых пользователем очертаний элементов; В программе для рисования Krita добавлена новая панель для формирования композиций, полезная при выполнении задач раскадровки видео. Добавлена поддержка нового режима рисования с использованием текстур. Улучшены средства вращения содержимого холста. Расширены инструменты для управления автоматически сохранёнными файлами. Проведены оптимизации производительности; Добавлены новые фильтры импорта для файлов Visio, MS Works и xfig. Улучшена работа ранее добавленных фильтров для форматов MS Office, особенно улучшена поддержка формата OOXML; Представлен новый набор фильтров для обработки изображений, таких как преобразование в серые тона или чёрно-белый вид. Обеспечено зеркалирование изображений по произвольной оcи; Одновременно с десктоп-версией пакета выпущен вариант Calligra Active для планшетов с сенсорными экранами и смартфоров. Принято решение о развитии Calligra Active как основного варианта для мобильных устройств (параллельно развивался упрощённый Calligra Mobile для Nokia N900). Готовые пакеты с альфа-версией Calligra 2.5 подготовлены для Ubuntu, Fedora, Arch Linux, openSUSE, FreeBSD и Windows.
