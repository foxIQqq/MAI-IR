Релиз системы управления контентом Joomla 3.0

Представлен релиз Joomla 3.0, универсальной системы управления web-контентом, которая подходит для создания больших и маленьких корпоративных сайтов, интранет порталов, online-магазинов, сайтов сообществ и персональных страниц. Из особенностей Joomla можно отметить: гибкие инструменты по управлению пользователями, интерфейс для управления медиа-файлами, поддержка создания многоязычных вариантов страниц, система управления рекламными кампаниями, адресная книга пользователей, голосования, встроенный поиск, функции категоризации ссылок и учета кликов, WYSIWYG-редактор, система шаблонов, поддержка меню, управление новостными потоками, XML-RPC API для интеграции с другими системами, поддержка кэширования страниц и большой набор готовых дополнений. Joomla 3.0 является выпуском со стандартным временем поддержки (6 меcяцев). Для тех кому не столь важны нововведения продолжается выпуск корректирующих обновлений для ветки 2.5, отнесённой к категории версий с длительным сроком поддержки (до весны 2014 г.). Поддержка старой LTS-ветки Joomla 1.5 будет прекращена в ближайшее время. Среди добавленных в Joomla 3.0 новшеств: Интеграция JavaScript-фреймворка Bootstrap в пакет jui, разработанного компанией Twitter; Новые шаблоны для административного интерфейса, адаптированные для использования на мобильных устройствах; Новый фронтэнд Protostar, созданный с использованием JavaScript-фреймворка Bootstrap; Обновление шаблона Beez3 для людей с ограниченными возможностями; Новый, упрощённых до трёх шагов, процесс установки; Драйвер для обеспечения поддержки работы сайтов на базе Joomla 3.0 с использованием СУБД PostgreSQL; Драйвер для работы с Memcached; Задействование JFeed вместо SimplePie для управления новостными лентами; Поддержка установки языковых пакетов непосредственно из менеджера дополнений; Включение по умолчанию группы для гостевого входа; Новый модуль для показа статистики в административном интерфейсе; Обновление TinyMCE до версии 3.5.6 Выполнение работы по чистке устаревшего неиспользуемого кода, файлов и полей в БД. Стандартизация стиля написания кода; Улучшение реализации "умного поиска"; Использование unit-тестирования CMS; Поддержка в модуле menu кастомизированных активных элементов меню.
