Выпуск SpaceVim 1.1, дистрибутива редактора Vim

Представлен выпуск проекта SpaceVim 1.1, развивающего дистрибутив текстового редактора Vim с подборкой плагинов для поддержки различных языков программирования и возможностей, свойственных интегрированным средам разработки. Плагины сгруппированы в подборки с реализацией тех или иных возможностей. Например, подборка для разработчиков на языке Python включает плагины deoplete.nvim, neomake и jedi-vim для автодополнения кода, проверки синтаксиса и интерактивного обращения к документации. Таким образом пользователю достаточно лишь выбрать необходимую функциональность без необходимости раздельного подбора плагинов. В новом выпуске добавлена поддержка всплывающих окон (например, для показа документации и результатов поиска через flygrep), реализовано меню для поискового плагина fzf, добавлен набор для разработчиков на языке Rust, добавлена обвязка над командой "git log", добавлен файловый менеджер defx.
