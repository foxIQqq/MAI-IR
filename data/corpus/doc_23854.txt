Выпущен релиз дистрибутива Caos Linux NSA 1.0.25

Спустя 8 месяцев с момента прошлого релиза вышла новая версия дистрибутива Caos Linux NSA 1.0.25. В новой версии представлено новое оформление инсталляционного диска (i386, x86_64, размер 1.8 Гб), который теперь выполнен в виде LiveDVD, позволяющего не только установить систему на жесткий диск, но и оценить возможности дистрибутива, быстро поднять узел кластера или организовать сервер для быстрой установки хостов по сети в kickstart-режиме (подготовлено несколько типовых профилей устанавливаемых наборов программ). Из других новшеств можно отметить обновление версий программ в пакетах, улучшение конфигурационных утилит, улучшение средств для создания приватных и публичных cloud-окружений, расширение поддерживаемого оборудования, улучшение совместимости с серверными платформами Intel, SuperMicro и HP. Дистрибутив Caos Linux предназначен для развертывания специализированных окружений и узлов кластеров. При создании Caos NSA первоочередной задачей является обеспечение высокой стабильности, приемлемой для выполнения критически важных сервисов. Дополнительные критерии развития - безопасность, высокое быстродействие, эффективность и легковесность. Многие разработчики Caos одновременно занимаются развитием дистрибутива CentOS. Разработки проекта активно используются в кластерах, входящих в список 500 самых мощных суперкомпьютеров в мире. Дистрибутив сертифицирован по программе ICR (Intel Cluster Ready) и гарантирует совместимость с приложениями от независимых поставщиков ПО для выполнения вычислительных задач, например, Abaqus, Anima, Fluent, Petrel и т.д. Для повышения безопасности в Caos NSA интегрированы наработки проекта GRSecurity, в состав дистрибутива включен PHP с "hardened" патчем. Для управления пакетами используются RPM 5 и Smart Package Manager. При желании использования Caos Linux на рабочей станции предусмотрена возможность установки графического окружения на базе XFCE. В рамках проекта были созданы следующие открытые приложения: Warewulf - средство для централизованного управления инфраструктурой кластера, технология базируется на формировании загрузочного образа ОС в виде ram-диска и распространения его по узлам кластера Perceus - тулкит для развертывания, обеспечения работы и управления группой серверов. Abstractual - система для управления cloud-окружениями; Equilibria - балансировщик нагрузки.
