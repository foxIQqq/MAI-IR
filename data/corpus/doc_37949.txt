Выпуск интегрированного набора интернет-приложений SeaMonkey 2.21

Представлен релиз пакета SeaMonkey 2.21, объединяющего в рамках одного продукта набор приложений для работы в сети, разрабатываемых под эгидой проекта Mozilla: web-браузер, почтовый клиент, систему агрегации новостных лент (RSS/Atom), WYSIWYG-редактор html-страниц Composer, IRC-клиент ChatZilla, набор средств для web-разработчиков, включающих DOM inspector и JavaScript-отладчик. Пакет доступен в сборках для платформ Linux, Windows и OS X. Кроме синхронизации с кодовой базой Firefox 24 с заимствованием свойственных для данного продукта новшеств, в SeaMonkey 2.21 реализованы следующие улучшения: Реализована опция для сортировки дискуссий по дате; По умолчанию разрешено удалять новостные публикации; В панель добавлена опция предпросмотра вкладок; Добавлена поддержка вывода уведомлений на рабочий стол, после подтверждения полномочий для каждого сайта; В поиск с заданием приоритета добавлен оператор исключения.
