Вышел релиз Linux дистрибутива Salix 13.0 от создателя Zenwalk Linux

Вышел релиз Linux дистрибутива Salix 13.0, созданного бывшим лидером сообщества разработчиков Zenwalk Linux, покинувшим проект в мае в результате конфликта с другими разработчиками, отстаивавшими политику максимальной похожести на Slackware. Дистрибутив Salix остался на 100% совместим со Slackware Linux 13.0 и поддерживает три режима установки: Полная - десктоп окружение на основе Xfce, Firefox, почтовый клиент Claws-mail, полная версия офисного пакета OpenOffice.org, Java Runtime Environment, мультимедиа плеер Totem, ПО для ведения архива музыки Exaile, пакетный менеджер gslapt; Основная - Xfce, Firefox и пакетный менеджер gslapt; Базовая - только минимальный набор консольных утилит, подходит для создания серверов.
