Компания РОСА выпустила серверный дистрибутив на базе Red Hat Enterprise Linux 6

Представлен новый серверный дистрибутив ROSA Enterprise Linux Server "Helium" 2012, принципиально отличающийся от ранее выпускаемых продуктов РОСА использованием пакетной базы Red Hat Enterprise Linux 6.x, в то время как прошлые дистрибутивы были основаны на наработках проекта Mandriva. 32- и 64-разрядные сборки нового дистрибутива доступны для свободной загрузки. Также публично доступны репозитории с исходными и бинарными пакетами и обновлениями с устранением проблем безопасности. Для корпоративных пользователей предоставляются услуги технической поддержки, с бесплатным 30 дневным пробным периодом. Кроме импортированных из RHEL пакетов в состав дистрибутива входит ряд компонентов, собранных на основании кода upstream-проектов или включающих собственные разработки компании РОСА, например, добавлены пакеты с LXDE, средства интеграции с корпоративными сетями, web-интерфейс для наглядной настройки и управления сервером, средства для централизованного управления учетными записями пользователей, компоненты для автоматизации поддержания в актуальном состоянии данных DNS. При этом использование в качестве основы пакетной базы RHEL позволило обеспечить бинарную совместимость с популярными коммерческими приложениями уровня предприятия. Процесс пересборки и формирования дистрибутива организован с использованием сборочной среды ABF, которая используется для формирования ранее выпускаемых дистрибутивов ROSA и на которую переходит проект Mandriva. В будущем, на базе ROSA Enterprise Linux планируется сформировать ещё один продукт ROSA Cloud Platform, нацеленный на построение облачных инфраструктур класса «инфраструктура как сервис» (IAAS) и «платформа как сервис» (PAAS). Среди оригинальных разработок компании РОСА, вошедших в состав ROSA Enterprise Linux Server "Helium" 2012: ROSA Server Setup — веб-интерфейс для установки и настройки основных серверных компонентов дистрибутива. Позволяет быстро развернуть такие службы, как сервер каталогов, стек LAMP (Linux/Apache/MySQL/PHP), СУБД (MySQL и PostgreSQL), файловые серверы SAMBA и NFS, сервер печати, сетевые сервисы DHCP и DNS, почтовый сервер (Postfix) с защитой от спама и вирусов, сервер резервного копирования (Bacula); ROSA Management Server — система управления настройками установленных серверных приложений; ROSA Directory Server — компонент для администрирования учетных записей пользователей и управления доступом к ресурсам локальной сети организации. ROSA Directory Server основан на коде Mandriva Directory Server, расширенном поддержкой Kerberos и возможностями по администрированию нескольких подсетей. Поддерживаются функции централизованного управления учетными записями; создание групповых политик, контролирующих доступ пользователей к сетевым ресурсам; управление общими сетевыми ресурсами; управление сетевыми сервисами (DNS+DHCP, Postfix, сервер печати и т.п.); Dnsmod - представляет средства для редактирования данных DNS непосредственно через LDAP, автоматически отслеживая изменения данных и инициируя операции обновления DNS-зон; LdapBindFS - FUSE-модуль для монтирования хранимых в LDAP-каталоге данных в форме виртуальной файловой системы, в которой содержимое DNS-параметров из LDAP транслируется в форму файлов конфигурации зон для DNS-сервера Bind. Таким образом Bind работает с обычными файлами зон, которые на самом деле прозрачно создаются на основании данных в LDAP; Интеграция пакетов с рабочим столом на базе окружения LXDE. Из легковесных приложений предлагается использовать PDF-просмотрщик epdfview, графический интерфейс для управления пакетами, плагины VPN для NetworkManager, менеджер буфера обмена Parcellite и пр. Для установки также доступны десктоп-окружения GNOME 2.28 и KDE 4.3.5. Базовые возможности ROSA Enterprise Linux Server "Helium" 2012: Web-службы: размещение web-сайта организации и других web-приложений; Proxy-сервер: контроль и статистика доступа в интернет; Cлужбы каталога, инвентаризации, политики управления: управление учетными записями пользователей и компьютеров, на которых установлен как Linux, так и Windows; Службы виртуализации, терминальные службы: создание и размещение виртуальных серверов и рабочих мест и доступ к ним; VPN-сервер: безопасный доступ пользователей "извне" к внутренним ресурсам предприятия; Службы доступа к файлам: хранилище данных и документов; Антивирусные службы и службы резервного копирования: защита и резервное копирование данных; Групповая работа: электронная почта, календарь, задачи, мгновенные сообщения, аудио- и видеоконференции, совместная работа над документами; Набор сервисов и служб для построения приватных и публичных облачных инфраструктур (IaaS), базирующиеся на компонентах платформы OpenStack. В состав также входит сервис для организации распределенного хранения данных и удобный управляющий web-интерфейс. Основные особенности продукта: Специально адаптированный русскоязычный пользовательский интерфейс; Повышенный уровень безопасности данных и служб за счет строгой авторизации пользователей с функцией единого входа ("Single SignOn"); Простая интеграция в существующие корпоративные сети; Подробная иллюстрированная документация от разработчика по всем вопросам использования; Дружественные графические средства администрирования системы и приложений с веб-интерфейсом; Поддержка платформ Windows и Linux в качестве клиентов.
