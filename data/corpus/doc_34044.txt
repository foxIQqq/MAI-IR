Началось тестирование первой альфа-версии Ubuntu 12.10

Вышла первая альфа-версия Ubuntu 12.10 "Quantal Quetzal", в которой проведена синхронизация пакетной базы с Debian Unstable. Загрузочные iso-образы с тестовой версией Ubuntu 12.10 можно получить на данной странице. Релиз запланирован на 18 октября, после выхода еще двух альфа и двух бета версий. Некоторые изменения: В первой альфа-версии изменения в основном касаются обновления версий программ: осуществлён переход на ядро Linux 3.4, GCC 4.7, binutils 2.22, eglibc 2.15, firefox 13, thunderbird 13; По умолчанию в составе десктоп-сборки 12.10 будет поставляться только Python 3.0, но на время тестирования Python 2 пока остаётся в поставке до момента завершения портирования всех приложений. По умолчанию включена система отправки уведомлений о крахах. Осуществлена интеграция в репозитории компонентов GNOME 3.4, в том числе новые версии приложений Disks, Evolution, Seahorse/GNOME Keyring и Totem. Дополнительно уже интегрированы некоторые составляющие GNOME 3.5.1/3.5.2. В Ubuntu 12.10 будет использован GNOME 3.6. Пакеты с KDE обновлены до версии 4.8.3; Проведена интеграция подсистемы UDisks2, которая теперь используется в качестве бэкенда по умолчанию для Unity и GNOME; Переработано оформление менеджера установки обновлений; Из инсталлятора убран ассистент для миграции документов и настроек из других ОС из-за достаточно большого числа проблем; CD и DVD образы десктоп-редакции Ubuntu 12.10 объединены в единый образ размером 800 Мб, пригодный для установки на USB Flash и DVD. Установочный образ серверной версии по прежнему будет вмещаться на CD (703MB), но в будущих тестовых выпусках ситуация может измениться; Для серверной редакции прекращено формирование CD-образов для архитектуры i386, в качестве первичной платформы рассматривается X86_64 (AMD64). Для установки на i386 будет продолжена поддержка mini-образа и образа для установки по сети.
