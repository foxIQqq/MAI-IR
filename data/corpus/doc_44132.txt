Canonical и Microsoft работают над возможностью запуска Ubuntu в Windows 10

Издание ZDNet опубликовало полученную от компании Canonical информацию о совместной с Microsoft разработке механизма для бесшовного запуска Ubuntu в окружении Windows 10. Утверждается, что запуск приложений Ubuntu будет производиться без использования виртуализации с предоставлением интеграции со штатным окружением Windows. Подробности будут раскрыты в официальном анонсе проекта, который ожидается через несколько часов на конференции Microsoft Build. Из уже существующих похожих проектов можно отметить andLinux, speedLinux, Portable Ubuntu Remix и Ulteo Virtual Desktop, которые построены на базе ядра coLinux и X-сервера Xming. Запускаемые в andLinux и Ulteo Virtual Desktop процессы работают как отдельно запущенные windows-приложения, имеющие доступ к файловой системе Windows и интегрируемые на общий рабочий стол. В текущем виде coLinux не поддерживает Windows 10, последний стабильный выпуск был сформирован пять лет назад, а самые свежие изменения в экспериментальной 64-разрядной ветке внесены в 2014 году. Дополнение: Опубликованы подробности реализации в Windows 10 поддержки запуска исполняемых файлов Linux.
