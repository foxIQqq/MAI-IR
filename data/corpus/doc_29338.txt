Разработчики Mozilla поместили Skype Toolbar в чёрный список

Разработчики Mozilla заблокировали работу дополнения Skype Toolbar, автоматически интегрируемого в браузер при установке или обновлении Skype. Дополнение обеспечивает преобразование телефонных номеров на web-страницах в ссылки, кликнув на которые можно сразу осуществить звонок. Первой причиной блокировки стало крайне низкое качество данного дополнения, занимающего первое место в рейтинге наиболее нестабильных дополнений. Например, только за прошлую неделю разработчикам Firefox поступила информация о 40 тысячах крахов Firefox 3.6.13 из-за ошибок в Skype Toolbar. В качестве второй причины называется неоптимальная организация работы, существенно замедляющая отображение страниц из-за дополнительной обработки контента. Причём степень замедления для некоторых страниц доходит до трёхсот раз. В настоящее время представители Mozilla связались с разработчиками из компании Skype, которые приступили к исправлению наблюдаемых проблем. Блокировка будет снята после того, как все мешающие работе ошибки будут устранены. До снятия блокировки пользователи, желающие воспользоваться данным дополнением, могут вручную активировать его в менеджере дополнений Firefox. Полный список заблокированных Firefox-дополнений можно найти на данной странице. Из присутствующих там дополнений можно отметить Microsoft Bing Bar, Microsoft Default Manager, Search Helper Extension, Java Deployment Toolkit, Yahoo Application State, Apple QuickTime Plugin и AVG SafeSearch.
