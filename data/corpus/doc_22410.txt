Компания Syntext открыла исходные тексты XML редактора Serna Free

Компания Syntext открыла под лицензией GPLv3 исходные тексты многоплатформенного WYSIWYG редактора XML файлов Serna Free. Редактор Serna поддерживает подключение дополнений на языке Python, что позволяет рассматривать его как платформу для создания приложений на базе XML. Serna Free базируется на стандартных стилях XSLT и XSL-FO, что позволяет отображать документы в представлении максимально близком к печатному, создавать профилированные виды XML документов, а также целостно представлять XML документы, составленные из множества частей. Ключевые возможности XML редактора Syntext Serna Free: Встроенная поддержка всех популярных XML форматов - DITA, Docbook, XHTML, TEI P4, NITF и других; Контекстная помощь по элементам; Проверка корректности XML документа в реальном времени с использованием XML Schema; Редактирование содержимого XML Entity и XInclude прямо в месте их вставки; Перетаскивание контента (drag-and-drop) с мгновенной валидацией; Поддержка UNICODE и многоязычная проверка орфографии.
