Релиз дистрибутива openSUSE Leap 42.1, включающего наработки SUSE Linux Enterprise

Проект openSUSE анонсировал релиз openSUSE Leap 42.1, в рамках которого выполнен переход на новую архитектуру построения дистрибутива, совмещающую вклад сообщества с разработками для корпоративных систем. В отличие от ранее формируемых независимых выпусков openSUSE, основой openSUSE Leap выступают наработки промышленного дистрибутива SUSE Linux Enterprise, поверх которых поставляются свежие выпуски рабочего стола и новых версий пользовательских приложений. Новый гибридный подход к построению дистрибутива позволяет совместить потребности в стабильном системном окружении и свежих версиях конечных приложений для пользователей, администраторов и разработчиков. При включении в репозиторий версий программ разработчики попытались найти оптимальный баланс между новизной и зрелостью пакетов, для самых свежих выпусков рекомендуется использовать репозиторий "Tumbleweed". С точки зрения разработки дистрибутива значительно упростится сопровождение базовой системы, для которой будут формироваться единые с SUSE Linux Enterprise обновления. Для пользователей коммерческих дистрибутивов SUSE появится возможность установки свежих версий программ из поддерживаемых сообществом репозиториев. Содержимое базовой системы и весь стек YaST, AutoYaST и Linuxrc в openSUSE Leap 42.1 заменены на компоненты, развиваемые для SUSE Linux Enterprise 12 SP 1. Репозиторий openSUSE Leap 42.1 насчитывает более 7000 пакетов, из которых 1500 импортированы из SUSE Linux Enterprise. Для загрузки доступна универсальная DVD-сборка, размером 4.4 Гб. Кроме сборок для систем x86_64, доступны редакции дистрибутива для платформ Power 8 (ppc64le), ARMv8 (AArch64), ARMv7 и ARMv6. Формирование сборок для 32-разрядных систем x86 прекращено. В дистрибутиве применена новая нумерация выпусков "Leap 42.x". Слово "Leap" идентифицирует дистрибутив по аналогии с выпусками "Tumbleweed". Номер "42" выбран по кодовому имени проекта "openSUSE 42" в Open Build Service (42 = 12 (релиз SLE) + 30), который уже получил известность в сообществе. Число "x" в номере "42.x" соответствует номеру сервисного выпуска SUSE Linux Enterprise на основе которого скомпонована базовая часть релиза. Например, текущий релиз 42.1 основан на SLE 12 SP1 и послужит тестовой площадкой для подготовки данного обновления. При увеличении номера релиза SLE будет увеличен и номер openSUSE, например, SLE 13 SP2 будет соответствовать openSUSE Leap 43.2. Пользователям предлагается несколько рабочих столов: KDE Plasma 5.4, GNOME 3.16, MATE 1.10, Xfce 4.12.1 и Enlightenment 19, а также подборка свежий версий пользовательских, системных и серверных приложений, в том числе LibreOffice 5, Qt 5.5, ядро Linux 4.1, GCC 4.8.5 (как опция GCC 5.2), QEMU 2.3.1, VirtualBox 5.0.6, Docker 1.8.2. По умолчанию предлагается файловая система Btrfs, для которой в новом выпуске задействована возможность загрузки выбранного снапшота, автоматически создаваемого инструментарием Snapper ежечасно или перед каждым обновлением. Представлен новый инструментарий командной строки Machinery для создания системных сборок и оценки их состояния. В состав включена подборка новых модулей к YaST, включая модули настройки шрифтов, контейнеров Docker и анализа логов.
