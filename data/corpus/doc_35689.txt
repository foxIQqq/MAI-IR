GitHub опубликовал статистику за 2012 год

Хостинг свободных проектов GitHub.com, базирующийся на системе управления исходными текстами Git и позволяющий разработчикам общаться в формате единой социальной сети, опубликовал статистику развития сервиса в 2012 году. По сравнению с началом года активность пользователей в GitHub практически удвоилась. В среднем за рабочий день 10 тысяч пользователей заводят аккаунт на GitHub, загружается около 140 Гб новых данных, создаётся 25 тысяч репозиториев, отправляется 7 тысяч pull-запросов, добавляется новый код в 125 тысяч репозиториев, 10 тысяч пользователей заводят свой первый репозиторий. Общее число пользователей оценивается в 2.8 миллиона, прирост пользовательской базы за год составил 133%, а число репозиториев выросло на 171% и достигло отметки 4.6 млн. Из репозиториев, которые пользователи сочли наиболее интересными, отмечены проекты (при оценке учитывалось число голосований через нажатие звёздочки): Font-Awesome (9394) - символьный шрифт (iconic), созданный для использования с Twitter Bootstrap; textmate (7854) - текстовый редактор для Mac OS X; meteor (6614) - JavaScript-фреймворк; hw3_rottenpotatoes (3983) - платформа для создания SaaS-сервисов на базе Ruby on Rails; select2 (4341) - JavaScript библиотека на базе jQuery для создания интерактивных web-форм; httpie (4026) - упрощённый аналог утилиты curl; ratchet (3311) - прототип мобильного приложения для iPhone; bower (3307) - созданный компанией Twitter пакетный менеджер для web; mailcheck (3136) - Javascript-библиотека для выявления ошибок в имени домена в формах ввода email; Prince-of-Persia-Apple-II (2987) - оригинальный код игры Prince of Persia. По числу вовлечённых в разработку участников лидируют проекты: homebrew - пакетный менеджер для Mac OS X; Ruby on Rails - MVC-фреймворк для Ruby; cyanogenmod - альтернативная сборка платформы Android; CocoaPods - менеджер учета зависимостей для библиотек на Objective-C; Symfony - PHP-фреймфорк; Zend Framework - PHP-фреймворк; OpenStack - облачная платформа; salt - система централизованного управления конфигурацией; TrinityCore - открытый MMORPG-фреймворк; hubot-scripts - дополнительные скрипты для бота hubot.
