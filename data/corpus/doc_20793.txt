Релиз digiKam 0.10.0 с поддержкой KDE4

Вышел релиз программы для управления коллекцией фотографий - digiKam 0.10.0. Главные новшества: Поддержка KDE4 и Qt4, для взаимодействия с оборудованием теперь используется KDE4 API Solid; Переход на систему сборки Cmake. Убрана зависимость от X11 библиотек, что позволило сформировать сборки для MacOS X и Windows; Значительная переработка внешнего вида, изменен дизайн интерфейса для работы с камерой, улучшена работа с альбомами фотографий ; Появилась возможность предварительного просмотра видео и прослушивания аудио файлов через KDE4 интерфейс Phonon; Ускорение формирования эскизов (thumnails), через их генерацию в многопоточном режиме; Добавлена поддержка мета-данных в формате XMP (требует наличия библиотеки Exiv2 версии 0.16) и возможность редактирования мета-данных в RAW форматах DNG, NEF и PEF; БД с описанием коллекции теперь может хранится в произвольном месте и позволяет подключать информацию от альбомах с нескольких источников, например с удаленных машин. Код для работы с БД полностью переписан и теперь базируется на Qt4 SQL плагине; Реализована возможность определения геопозиции на основе мета-данных из фотографий и ее отображение в боковой панели через виджет KDE4 Marble; В переработанном интерфейсе для управления камерой появилось новое средство для захвата изображения; В GUI для управления альбомами фотографий упрощена навигация между изображениями, интегрирован инструмент для поиска по текстовым маскам и полностью переработана инфраструктура поиска.
