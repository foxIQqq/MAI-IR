Обзор изменений в Debian Jessie, связанных с мультимедиа

Команда сопровождающих пакеты с мультимедийными приложениями в Debian опубликовала отчёт об улучшениях и изменениях, реализуемых для Debian 8.0 "Jessie". Наиболее интересные моменты: Прекращена поставка mplayer (и mencoder), так как не нашлось заинтересованного в его поддержке сопровождающего. Вместо mplayer рекомендовано использовать mplayer2 или mpv, которые совместимы с ним на уровне опций командной строки. Из дистрибутива также исключены пакеты beast, flumotion, jack-rack, jokosher, lv2fil, phat, plotmm, specimen, zynjacku, для которых наблюдаются проблемы с разработкой или сопровождением. Поддержка из коробки многих новых мультимедиа форматов и кодеков. Библиотека libavcodec, которая используется для воспроизвделения контента в таких приложениях, как vlc, xine, mpv и totem (gstreamer1.0-libav), обновлена до свежего выпуска Libav 11, что позволило обеспечить поддержку таких форматов, как HEVC, Opus, Matroska 3D, Apple ProRes. В состав включена библиотека libebur128 с реализацией спецификаций EBU R128 (European Broadcasting Union Loudness Recommendation), выступающих в роли альтернативы технологии нормализации громкости ReplayGain. Добавлена библиотека libltc для кодирования и декодирования LTC (Linear Timecode). Libva и драйвер для GPU Intel обновлены до выпуска 1.4.0, в libva добавлена поддержка Wayland. Добавлены библиотеки fftease, lyonpotpourrie, iemlib, pdstring и pd-lua. Прекращена поддержка ASH Audio Session Handler в пользу новой системы ladish (LADI Session Handler), которая позволяет одновременно запускать несколько приложений JACK и легко сохранять/восстанавливать их конфигурацию при помощи графического интерфейса gladish. Реализация стандарта разработки звуковых плагинов LV2 обновлена до версии 1.10.0. Добавлен тулкит LV2 Toolkit, включающий библиотеки для использования LV2 в программах на C/C++. Добавлен инструментарий Vee One Suite, включающий несколько вариантов синтезаторов и cэмплеров, поставляемых как в виде LV2-плагинов, так и в форме отдельных приложений. Добавлены пакеты x42-plugins и zam-plugins с подборкой плагинов для высококачественной обработки звука. Добавлен Fomp, LV2-порт плагинов MCP, VCO, FIL и WAH. Сформировано несколько наборов пакетов (task) с подборкой компонентов для конкретных мультимедийных применений; Обновлены версии программ и библиотек, в том числе: Audacious 3.5.2, Audacity 2.0.6, Blender 2.72b, VLC 2.2.x. Добавлены новые приложения: XBMC - популярный медиацентр, ранее не поставляемый в Debian из-за несоответствия лицензионным требованиям. В Debian Jessie будет поставляться вариант "XBMC from Debian", по аналогии с Firefox отличающийся проведением ребрендинга; mpv - аудио/видео проигрыватель, основанный на кодовых базах MPlayer и mplayer2; Ardour3 - мощный звуковой редактор, предназначенный для многоканальной записи, обработки и микширования звука; Advene (Annotate Digital Video, Exchange on the NEt) - приложение для создания аннотаций для видео; Cantata - GUI для MPD, написанный с использованием Qt4; Csound 6 и IDE CsoundQT, язык программирования для обработки звука и связанная с ним среда разработки; din - синтезатор звука с поддержкой JACK, MIDI, OSC; dvd-slideshow - набор утилит для создания слайдшоу из набора изображений; dvdwizard - инструментарий для автоматизации создания DVD-Video с поддержкой меню, разделов и многоязычных представлений; flowblade - простой редактор видео, основанный на движке MLT; forked-daapd - вариант сервера потокого вещания daapd (DAAP/DACP), переведённый с Grand Central Dispatch на libevent; harvid - HTTP Ardour Video Daemon, декодирует изображения из видео и транслирует их по HTTP; Groove Basin - серверный музыкальный проигрыватель с web-интерфейсом и поддержкой протокола MPD; HandBrake - система перекодирования видео, поддерживающая большое число форматов и кодеков; laborejo - библиотека обработки нот; radium-compressor - JACK-приложение с реализацией системы упаковки звука Radium; smtube - проигрыватель и система навигации по видео, размещённого в таких сервисах, как YouTube (в том числе позволяет сохранять ролики с YouTube для локального просмотра); sonic-visualiser - наглядный анализатор содержимого музыкальных файлов; SoundScapeRenderer - движок рендеринга для пространственного звука; Videotrans - набор скриптов для преобразования набора видеофайлов в формат VOB, используемый на DVD; zita-bls1 - преобразователь стереосигналов; zita-mu1 - система мониторинга для jackd; zita-njbridge - Jack-клиенты для передачи многоканального звука через локальноую сеть по IP;
