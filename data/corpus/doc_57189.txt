Выпуск дистрибутива для создания сетевых хранилищ TrueNAS 13.0

После полутора лет разработки компания iXsystems представила выпуск TrueNAS CORE 13, дистрибутива для быстрого развёртывания сетевого хранилища (NAS, Network-Attached Storage). TrueNAS CORE 13 основан на кодовой базе FreeBSD 13, отличается интегрированной поддержкой ZFS и возможностью управления через веб-интерфейс, построенный с использованием Python-фреймворка Django. Для организации доступа к хранилищу поддерживается FTP, NFS, Samba, AFP, rsync и iSCSI, для повышения надежности хранения может применяться программный RAID (0,1,5), для авторизации клиентов реализована поддержка LDAP/Active Directory. Размер iso-образа 900МБ (x86_64). Параллельно развивается дистрибутив TrueNAS SCALE, использующий Linux вместо FreeBSD. Основные новшества в TrueNAS CORE 13.0: Реализация файловой системы ZFS обновлена до OpenZFS 2.1, а содержимое базового окружения синхронизировано с FreeBSD 13.1. Отмечается, что переход на ветку FreeBSD 13 и добавленные оптимизации позволили добиться повышения производительности крупных NAS до 20%. Существенно сокращено время импорта пулов ZFS за счёт распараллеливания операций. Время перезапуска и восстановления после сбоев на крупных системах сокращено более чем на 80%. Реализация сетевого SMB-хранилища переведена на использование Samba 4.15. Повышена производительность iSCSI Target и повышена эффективность ввода/вывода. Для NFS реализована поддержка режима nconnect, позволяющего распределить нагрузку через несколько установленных с сервером соединений. В высокоскоростных сетях распараллеливание потоков позволяет повысить производительность до 4 раз. В интерфейсе пользователя предоставлена возможность просмотра логов виртуальных машин. В интерфейсе пользователя добавлена поддержка группировки секций с учётными записями, хранилищами, сетевыми параметрами, приложениями, настройками, отчётами и многими другими разделами. Обновлены плагины iconik и Asigra. Дополнительно можно отметить обновление дистрибутива TrueNAS SCALE 22.02.1, отличающегося от TrueNAS CORE использованием ядра Linux и пакетной базы Debian. Решения на основе FreeBSD и Linux сосуществуют и дополняют друг друга, используя общую кодовую базу инструментария и типовой web-интерфейс. Предоставление дополнительной редакции на базе ядра Linux объясняется желанием реализовать некоторые идеи, недостижимые при использовании FreeBSD. Например, в TrueNAS SCALE реализована поддержка Kubernetes Apps, гипервизора KVM, REST API и Glusterfs. В новой версии TrueNAS SCALE осуществлён переход на OpenZFS 2.1 и Samba 4.15, добавлена поддержка NFS nconnect, в состав включено приложение мониторинга Netdata, добавлена поддержка самошифруемых дисков, улучшен интерфейс управления пулами, улучшена поддержка источников бесперебойного питания, расширены API Gluster и кластерного SMB.
