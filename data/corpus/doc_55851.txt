Выпуск пользовательского окружения GNOME 41

После шести месяцев разработки представлен выпуск десктоп-окружения GNOME 41. Для быстрой оценки возможностей GNOME 41 предложены специализированные Live-сборки на основе openSUSE и установочной образ, подготовленный в рамках инициативы GNOME OS. GNOME 41 также уже включён в состав экспериментальной сборки сборок Fedora 35. В новом выпуске: Расширены возможности по настройке энергопотребления. Предоставлена возможность быстрой смены режима энергопотребления ("экономия энергии", "высокая производительность" и "сбалансированные настройки") через меню управления состоянием системы (System Status). Приложениям, предоставлена возможность запроса определённого режима энергопотребления - например, чувствительные к быстродействию игры могут запросить активацию режима высокой производительности. Добавлены опции для настройки режима экономии энергии (Power Saver), позволяющие управлять снижением яркости экрана, выключением экрана после определённого периода неактивности пользователя и автоматическим выключением при низком заряде аккумулятора. Переработан интерфейс управления установкой приложений, в котором упрощена навигация и поиск интересующих программ. Списки приложений оформлены в виде более наглядных карт с кратким описанием. Предложен новый набор категорий для разделения приложений по тематике. Переработана страница с детальной информацией о приложении, в которой увеличен размер скриншотов и повышена информативность сведений о каждом приложении. Также переработано оформление настроек и списков уже установленных программ и программ для которых имеются обновления. В конфигуратор (GNOME Control Center) добавлена новая панель Multitasking для настройки управления окнами и рабочими столами. В частности, в секции Multitasking предоставлены опции для отключения вызова обзорного режима через касание к верхнему левому углу экрана, изменения размера окна при его перетаскивании к краю экрана, выбора числа виртуальных рабочих столов, показа рабочих столов на дополнительно подключённых мониторах и переключения между приложениями только текущего рабочий стола при нажатии комбинации Super+Tab. Добавлена новая панель Mobile Network для управления подключением через сотовых операторов, выбора типа сети, ограничения трафика при роуминге, настройки работы модемов для сетей 2G, 3G, 4G и GSM/LTE, и переключения между сетями для модемов, поддерживающих вставку нескольких SIM-карт. Панель показывается только при подключения модема, поддерживаемого в системе. В состав включено новое приложение Connections с реализацией клиента для удалённого подключения к рабочему столу с использованием протоколов VNC и RDP. Приложение заменило собой функциональность для удалённого доступа к рабочим столам, ранее предлагавшуюся в программе Boxes. Изменено оформление интерфейса GNOME Music, в котором увеличен размер графических элементов, скруглены углы, добавлен показ фотографий музыкантов, переделана панель управления воспроизведением и предложен новый экран просмотра информации об альбоме с кнопкой перехода к воспроизведению. В состав включён интерфейс для совершения вызовов GNOME Calls, в который помимо совершения звонков через сотовых операторов добавлена поддержка протокола SIP и совершения звонков через VoIP. Проведена оптимизация производительности и отзывчивости интерфейса. В сеансе на базе Wayland повышена скорость обновления информации на экране, сокращено время реакции при нажатии клавиш и движении курсора. В GTK 4 задействован новый движок отрисовки на базе OpenGL, который позволил сократить потребление энергии и ускорить отрисовку. Выполнена чистка кодовой базы оконного менеджера Mutter, которая позволила повысить его эффективность и упростить сопровождение. Повышена надёжность и предсказуемость обработки мультитач жестов. В файловом менеджере Nautilus переделан диалог для управления сжатием, добавлена возможность создания архивов ZIP, защищённых паролем. В календаре-планировщике реализована поддержка импорта событий и открытия файлов ICS. Предложена новая всплывающая подсказка с информацией о событии. В браузере Epiphany обновлён встроенный PDF-просмотрщик PDF.js и добавлен блокировщик рекламы в YouTube, реализованный на основе скрипта AdGuard. Кроме того, расширена поддержка тёмного оформления, улучшена обработка зависаний при открытии сайтов и ускорена операция масштабирования щипком. Полностью переделан интерфейс калькулятора, который теперь автоматически адаптируется к размеру экрана на мобильных устройствах. В системе уведомлений добавлена поддержка категорий. В дисплейном менеджере GDM появилась возможность запуска сеансов на базе Wayland, даже если экран входа запущен на базе X.Org. Разрешено использование сеансов Wayland для систем с GPU NVIDIA. В gnome-disk для шифрования задействован LUKS2. Добавлен диалог для настройки владельца ФС. В мастер начальной настройки возвращён диалог для подключения сторонних репозиториев. В GNOME Shell обеспечена поддержка запуска X11-программ при помощи Xwayland на системах, не использующих systemd для управления сеансом. В GNOME Boxes добавлена поддержка воспроизведения звука из окружений, для подключения к которым использован VNC.
