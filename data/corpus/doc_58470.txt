Выпуск дистрибутива Альт Образование 10.1

Опубликовано обновление продукта Альт Образование 10.1 на основе Деcятой платформы ALT (p10 Aronia). Сборки подготовлены для платформ x86_64, AArch64, i586. Дистрибутив предназначен для образовательных учреждений и внесён в Единый реестр российских программ и баз данных. Продукт поставляется в рамках Лицензионного договора, который предоставляет возможность свободного использования физическими лицами, но юридическим лицам допускается только тестирование, а для использования требуется приобрести коммерческую лицензию или заключить лицензионный договор в письменной форме. Новшества Альт Образование 10.1: Добавлена поддержка российских удостоверяющих центров (УЦ) (пакет ca-certificates-digital.gov.ru). Система программирования PascalABC.NET дополнена графической средой. Пользователям 32-битной версии предложен веб-браузер Mozilla Firefox, в остальные версии включён Chromium. Добавлена поддержка установки системы при помощи утилиты Ventoy. Появилась возможность создавать подразделы файловой системы BTRFS при установке. Файловая система постоянно использует метод «copy-on-write» для всех файлов, что влияет на отказоустойчивость и простоту администрирования. Плюсом к безопасности в случае использования BTRFS является создание точки восстановления перед обновлением системы. LiveCD представлен как отдельный образ для ознакомления, но не поддерживает установку. Обновления в проекте групповых политик: Добавлена поддержка настройки общих каталогов (network shares) для компьютеров. В новом выпуске инструмента gpupdate поддерживаются применения для компьютера всех политик web-браузеров Mozilla Firefox и Chromium. Добавлены механизмы применения политик скриптов - logon/logoff/startup/shutdown; механизмы применения параметров системных настроек (preferences): операции с файлами (Files), каталогами (Folders), файлами конфигураций (Ini-files). Добавлено новое действие для обновления статуса сервисов в gpupdate-setup - ключ update запускает все необходимые службы при обновлении задействованного gpupdate. Периодичность запуска gpupdate теперь настроена через таймер: в релизе 0.9.11.2 для Systemd появился системный таймер gpupdate.timer и пользовательский таймер gpupdate-user.timer для мониторинга и контроля времени выполнения службы gpupdate.service. В файловом менеджере Xfce — Thunar, — появился вызов поиска файлов. Исправлена проблема запуска графических приложений после смены имени компьютера (пакет xauthority-hostname-updater). Система устанавливается с двумя ядрами (std-def — 5.10 и un-def — 5.15). Для нового оборудования при установке на EFI есть возможность выбрать ядро. Wine 7.19 поддерживает одновременный запуск как 32-битных, так и 64-битных исполняемых файлов Windows из одного профиля. Добавлены пакеты: alterator-zram-swap (для настройки подкачки в оперативной памяти), eiciel (графическое средство управления ACL файлов и каталогов), packagekit (универсальный способ управления пакетами). Пользователи дистрибутивов «Альт», построенных на Десятой платформе, не обнаружат проблем совместимости при переходе на 10.1. Для новых корпоративных пользователей предусмотрена возможность получения тестовых версий, а частным пользователям традиционно предлагается бесплатно скачать нужную версию ОС «Альт» с сайта «Базальт СПО» или с сайта загрузки getalt.ru.
