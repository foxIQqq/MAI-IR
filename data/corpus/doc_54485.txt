Выпуск Chrome OS 88

Состоялся релиз операционной системы Chrome OS 88, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 88. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 88 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 88: Добавлена возможность быстрого входа на сайты при помощи PIN-кода или отпечатка пальца, установленных для разблокировки текущего устройства. PIN или отпечаток пальца может применяться вместо привязанного к сайту пароля или как дополнительный признак при двухфакторной аутентификации на сайтах. Функциональность работает на сайтах, поддерживающих аутентификацию при помощи API WebAuthn (Web Authentication), позволяющем организовать подключение с использованием дополнительных методов аутентификации, таких как аппаратные токены или сенсоры отпечатков пальцев. При обнаружении поддержки сайтом WebAuthn браузер выдаст специальной уведомление о возможности применения PIN или отпечатка пальца. Например, WebAuthn поддерживается для входа на сайты Dropbox, GitHub и Okta. Реализована возможность персонализации хранителя экрана. Через настройку "Personalization > Screen saver" можно организовать показ своих изображений во время блокировки экрана. Изображения могут показываться из альбомов в Google Photos или коллекций изображений. Во время блокировки экрана также можно включить вывод прогноза погоды и виджета управления воспроизведением музыки. В приложении для работы с камерой изменено местоположение для сохранения фотографий и видео. Фотографии теперь сохраняются в подкаталоге "Camera", доступном в файловом менеджере в секции "My files". Ранее сохранённые фотографии и видео продолжат находиться в каталоге Downloads. Улучшены возможности автокорректировки правописания. Интерфейс автокорректировки теперь наглядно подсвечивает внесённые исправления и даёт возможность откатить изменение. После исправления ошибки, слово теперь остаётся подчёркнутым. Управление включением автокорректировкой производится в секции настроек "Device > Keyboard > Change language and input settings > Input method > New > Auto-correction". Улучшено переключение между виртуальными рабочими столами. Для быстрой смены рабочего стола теперь можно два или три раза коснуться ярлыка "Switch Desk" на сенсорном экране или нажать на клавиатуре "Search + [" или "Search + ]" для перехода к левому или правому виртуальному рабочему столу. В лупе, используемой для увеличения отдельных областей на экране, добавлена возможность изменения фокуса при помощи клавиатуры (Ctrl + Alt + клавиши управления курсором). Жесты управления с тачпада теперь согласуются с настойками обратной прокрутки (Reverse Scrolling).
