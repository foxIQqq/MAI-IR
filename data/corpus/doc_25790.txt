Доступен для свободной загрузки мартовский номер журнала BSD Magazine

Доступен для свободной загрузки (PDF, 5Мб) мартовский номер журнала "BSD Magazine", посвященного BSD-системам. В номере: Настройка сервера для централизованного обновления парка FreeBSD-машин в сети; Использование OpenBSD и PF для создания виртуального межсетевого экрана для Windows; Обзор методов для поддержания версий установленных во FreeBSD приложений в актуальном состоянии; Настройка на базе OpenBSD сервера для фильтрации спама; Выбор и установка оконных менеджеров во FreeBSD; Размышления о том, могут ли BSD-системы конкурировать в области решений для рабочего стола.
