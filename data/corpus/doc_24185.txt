Microsoft добавит в Visual Studio 2010 набор средств для интеграции с Eclipse

После заявления о разработке средств, расширяющих возможности платформы Eclipse в ОС Windows, компания Microsoft сообщила о заключении соглашения о приобретении подразделения компании SourceGear, занимающегося созданием средств разработки Teamprise, нацеленных на расширение возможностей платформы Eclipse инструментами для сборки приложений в Microsoft Visual Studio Team Foundation Server. По заявлению Microsoft наработки Teamprise войдут в состав релиза Visual Studio 2010. Под эгидой Teamprise выпускается три кроссплатформенных продукта, выпускаемых для Linux, Windows и MacOS X: Плагин для Eclipse, позволяющий управлять исходными текстами, вести трекинг ошибок, генерировать отчеты и использовать другие возможности MS Team Foundation Server; Teamprise Explorer - обособленная среда разработки на базе Eclipse. Клиент для работы в режиме командной строки, позволяющий разрабатывать сценарии для автоматизации действий с Team Foundation Server.
