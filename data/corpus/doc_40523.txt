Вышел дистрибутив Calculate Linux 14

Доступен релиз отечественного дистрибутива Calculate Linux 14, построенного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в шести редакциях, собранных для архитектур i686 и x86_64: Calculate Directory Server (CDS), Calculate Linux Desktop с рабочим столом KDE (CLD) и XFCE (CLDX), Calculate Media Center (CMC), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS). Все версии дистрибутива распространяются в виде загрузочного Live-образа с возможностью установки на жесткий диcк или USB-накопитель. Основные изменения: Удобное средство обновления системы, включающее автопроверку обновлений, вывод уведомлений в трее и в консоли. Графическая и консольная (cl-update) утилиты обновления с поддержкой удалённого обновления, локализованным описанием пакетов, кэшированием результатов просчёта зависимостей и полным циклом обновления, начиная от синхронизации репозиториев и заканчивая обновлением зависимых библиотек и настройкой конфигурационных файлов. Переработанная система профилей дистрибутивов, упрощающая создание и хранение модифицированных профилей, которая включает: Разделение оверлея системы на два - базового оверлея Calculate и оверлея дистрибутива Distros. Утилиту смены профиля cl-update-profile с поддержкой загрузки наследуемых оверлеев, поддержкой коротких указателей на популярные репозитории GitHub и Bitbucket. Поддержку наследования профиля из нескольких оверлеев при сборке дистрибутива. Улучшен интерфейс графического клиента для вызова утилит Calculate. Переход на использование Dracut для формирования образов начальной загрузки initramfs, включая создание обычного SafeMode и оптимизированного образа; Использование графического оформления Plymouth на экране загрузки системы. Перенос настроек портежей /etc/make.conf в директорию /etc/portage/make.conf.
