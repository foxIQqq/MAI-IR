Доступен дистрибутив Oracle Linux 5 Update 7

Компания Oracle выпустила седьмое обновление дистрибутива Oracle Linux, основанного на пакетной базе Red Hat Enterprise Linux 5.7. Дистрибутив доступен для бесплатной загрузки в сборках для архитектур x86 и x86_64, после регистрации на сайте edelivery.oracle.com. По сравнению с RHEL 5.7 дистрибутив Oracle Linux включает дополнительные исправления ошибок и оптимизации, в основном нацеленные на улучшение работы продуктов Oracle и использование дистрибутива для работы гипервизора Xen. В частности, добавлена поддержка xen pv netconsole, обновлено 19 драйверов для сетевых устройств и 12 драйверов для систем хранения. В базовую поставку входит стандартное ядро из состава RHEL 5, но по умолчанию используется подготовленный компанией Oracle пакет на основе Linux-ядра 2.6.32. Кроме того, из репозитория можно установить ещё один вариант ядра из RHEL 5, в котором силами Oracle проведено дополнительное исправление ошибок.
