Релиз Linux-дистрибутива Mandriva Enterprise Server 5.2

Компания Mandriva объявила о выходе коммерческого серверного дистрибутива Mandriva Enterprise Server 5.2. Одним из основных направлений развития дистрибутива является организация централизованного управления над инфраструктурой серверов и рабочих станций предприятия (поддерживаются разные Linux дистрибутивы, Unix и Windows). Для организации управления используется платформа Pulse2, поддерживающая такие функции, как поддержание инвентаризационной базы данных компьютеров, установка и обновление приложений на группе машин, диагностика служб, удаленное управление. В дистрибутиве используется Linux-ядро 2.6.33.7, поддерживаются системы виртализации Xen 3.3.1 и KVM, настройка производится через специальных упрощенный мастер конфигурирования, для организации резервного копирования в комплект входят продукты Bacula и Arkeia. В состав дистрибутива также входит сервер каталогов Mandriva Directory Server, позволяющий организовать централизованную базу пользователей с интеграцией с такими серверными приложениями, как Samba, Postfix, Squid и CUPS. Для управления LDAP-сервером подготовлен специальный web-интерфейс. Mandriva Enterprise Server 5.2 прошел сертификацию на предмет совместимости со стандартом LSB 4.0. Список входящих в комплект приложений и перечень поддерживаемого оборудования можно посмотреть здесь. Оценить возможности Mandriva Enterprise Server можно загрузив образ для запуска дистрибутива в виртуальных окружениях. Стоимость дистрибутива равна 299 евро на одну машину с годовым контрактом технической поддержки. Время жизни ветки 5.x составляет 5 лет, поддержка продлится до 2014 года.
