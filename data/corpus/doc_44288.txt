Релиз Linux-дистрибутива Ubuntu 16.04

Состоялся релиз дистрибутива Ubuntu 16.04 "Xenial Xerus", который отнесён к категории выпусков с длительным сроком поддержки (LTS), обновления для которых формируются в течение 5 лет. Готовые сборки созданы для Ubuntu, Ubuntu Server, Ubuntu Core, Kubuntu, Ubuntu GNOME, Xubuntu, Ubuntu Studio, Lubuntu, Ubuntu MATE, Ubuntu Cloud, Ubuntu Studio и UbuntuKylin (редакция для Китая). Основные новшества: Рабочий стол Добавлена возможность горизонтального размещения панели Unity (положение панели можно изменить в Dconf Editor через настройку "com > canonical > unity > launcher > launcher-position") ; Улучшена интеграция Launcher с файловым менеджером и интерфейсом подключения новых устройств. Серия пиктограмм в Launcher теперь связана с определёнными окнами файлового менеджера, например, клик на корзине приведёт к открытию окна Nautilus с содержимым корзины, а клик на USB-устройстве - покажет содержимое устройства; В меню быстрого вызова (quicklist) добавлена поддержка форматирования внешних накопителей; Улучшена поддержка GTK-приложений, в которых панель инструментов и меню перенесены в заголовок окна (HeaderBars); В центр управления Unity добавлена опция для включения постоянного показа меню в приложениях (System Settings/Appearance/Behavior tab); Поддержка масштабирования курсора на системах с экранами HiDPI; В панели обеспечен показ работающих пиктограмм приложений, независимо от того, как было запущено приложение; В Dash улучшена поддержка тем оформления, добавлена возможность вызова системных действий (завершение работы, перезагрузка) и реализованы новые перекрывающиеся полосы прокрутки (как в GTK+); В Dash по умолчанию отключена функция online-поиска, т.е. отныне без специальной настройки система не будет обращаться к внешним ресурсам при выполнении поиска с использованием умных областей (Smart Scope) в Unity 7, в том числе будет отключен показ контекстной рекламы Amazon. Изменение будет применено только для новых установок и не затронет прошлые выпуски и системы, перешедшие на Ubuntu 16.04 в результате обновления; Вместо Ubuntu Software Center задействован штатный интерфейс установки программ GNOME Software, который интегрирован с Unity Launcher. Поддержка специфичных для Ubuntu Software Center возможностей, таких как рейтинг, обзоры и платные приложения, реализована через плагины. В базовую поставку включена программа GNOME Calendar. Из базового состава исключены программа для записи CD/DVD Brasero и клиент для мгновенного обмена сообщениями Empathy. Подобный шаг объясняется потерей актуальности дисков CD/DVD и классических систем обмена сообщениями, которые вытеснили Flash-накопители и мобильные/web-ориентированные чат-сервисы. В апплете управления звуком обеспечен показ ползунка управления чувствительностью микрофона, если устройство имеет встроенный или подключенный микрофон (ранее ползунок микрофона показывался при использовании микрофона приложением); Все поставляемые в базовой поставке приложения и библиотеки, работающие с web-контентом, переведены на движок WebKit 2; Из базовой поставки в репозиторий Universe перемещены модули умных областей Audacious, Clementine, gmusicbrowser, Gourmet, Guayadeque и Musique, а также полностью прекращена поддержка сервиса Music Store. Прекращена поставка проприетарного драйвера AMD Catalyst (fglrx), который не вошёл в состав выпуска. Пользователям рекомендуется перейти на использование открытых драйверов AMDGPU/Radeon; Улучшена работа заставки на экранах высокого разрешения (HiDPI); Расширена поддержка языков и пакетов локализации в базовой поставке; Обеспечена возможность запуска X-сервера без прав root (при наличии kms-драйверов режим можно активировать удалив пакет "xserver-xorg-legacy"); В поставляемом в качестве опции дисплейном сервере Mir появилась поддержка графического API Vulkan. Для оценки работы Mir и Unity 8 без изменения состава основной системы можно воспользоваться специально подготовленным контейнером; Общесистемные и инфраструктурные изменения Обновлены версии пакетов, в том числе задействованы ядро Linux 4.4, LibreOffice 5.1, Python 3.5, Golang 1.6, Chromium 48, Firefox 45, OpenSSH 7.2p2, PHP 7.0, MySQL 5.7, Nginx 1.9.15, Docker 1.10, glibc 2.23 release, binutils 2.26, GCC 5.3.0; Добавлена возможность использования самодостаточных пакетов snap, не связанных зависимостями с остальными компонентами дистрибутива. Поддержка установки snap-пакетов даёт возможность поставлять для Ubuntu новые выпуски программ, не заботясь об обеспечении привязки к поставляемым в дистрибутиве библиотекам, поставляя актуальные приложения даже спустя несколько лет после выпуска релиза дистрибутива. О намерении приступить к формированию snap-пакетов уже объявили разработчики Firefox; Пакетный менеджер Apt обновлён до версии 1.2. Задействован новый механизм разделения привилегий при котором сетевые операции и парсинг результатов сетевых запросов выполняются с использованием отдельного непривилегированного пользователя "_apt"; Обеспечена поддержка менеджера логических томов и файловой системы ZFS в составе дистрибутива. Реализация основана на использовании проекта ZFSonLinux, работающего на уровне ядра Linux и демонстрирующего неплохую производительность и стабильность. Модуль zfs.ko включен в штатную поставку пакета с ядром, а инструментарий zfsutils-linux доступен из репозитория "main", с предоставлением обновлений и технической поддержкой от компании Canonical. В качестве основного назначения ZFS в Ubuntu называется использование в качестве хранилища изолированных контейнеров; Улучшения для серверных систем Представлен порт серверной и облачных редакций Ubuntu для 64-разрядной архитектуры мэйнфреймов IBM LinuxONE (z/Architecture). Пользователям доступно около 95% пакетов, в том числе системы OpenStack, Juju, MAAS и LXD; Для построения облачных систем в состав включён выпуск платформы OpenStack Mitaka, который оснащён драйвером nova-compute-lxd для поддержки инструментария LXD для ораганизации инфраструктуры управления изолированными контейнерами вместо виртуальных машин; Поставка гипервизора библиотеки Libvirt 1.3.1, инструментариев управления изолированными контейнерами LXD 2.0 и Docker 1.10, обновление QEMU 2.5, поставка виртуального коммутатора Open vSwitch 2.5; Обновлён Juju 2.0, интерфейс для быстрого развёртывания преднастроенных сервисов. Для использования с Juju подготовлено более 300 сервисов (Juju Charms), готовых к работе сразу после их установки. Среди таких сервисов различные СУБД (MySQL, PostgreSQL, Redis, Cassandra, Mongodb, Membase), web-приложения (wordpress, drupal, Node.js, Django, Ruby on Rails), серверные системы (lamp, tomcat, vsftpd), системы мониторинга, облачные платформы и т.д. В Juju 2.0 добавлена поддержка платформ AWS, Microsoft Azure, Google Cloud Engine, Rackspace, Joyent, LXD, MAAS и ручное развёртывание. Kubuntu: Компоненты рабочего стола обновлены до выпусков KDE Plasma 5.5.4 и KDE Applications 15.12, задействована тема оформления Breeze; Ubuntu GNOME: большинство компонентов рабочего стола, в том числе GNOME Shell, обновлены до GNOME 3.18. В репозиторий добавлены пакеты со средой разработки GNOME Builder. В базовую поставку включены программы GNOME Logs и Photos. Продолжается тестирование экспериментального сеанса на базе Wayland, для использования которого следует установить пакет gnome-session-wayland и выбрать при входе "GNOME on wayland" (работает только при использовании открытых видеодрайверов). Ubuntu MATE: В Ubuntu MATE Welcome добавлен интерфейс Software Boutique с обзором более 120 дополнительных программ (можно сразу установить выбранное приложение, отфильтровать вывод по категориям, посмотреть детали и скриншоты). Компоненты рабочего стола обновлены до MATE 1.12.1, в котором заметно снижена нагрузка на CPU. Добавлены MATE Dock Applet 0.67 и Topmenu Applet 0.2.1, на базе которых реализована новая раскладка панелей Mutiny (включается в настройках). Обеспечена возможность использования ланчера Synapse (отключен по умолчанию, активация через MATE Tweak), напоминающего по своим функциям Dash. Улучшена интеграция с композитным менеджером Compiz. Lubuntu: так как разработка окружения LXQt пока не завершена, переход Lubuntu на LXQt в очередной раз откладывается и в Lubuntu 16.04 по-прежнему будет использован LXDE. В новой версии добавлена поддержка платформы PowerPC и обновлены версии приложений; Xubuntu: исправления ошибок и обновление версий приложений. Добавлен новый апплет управления питанием. Улучшено оформление экрана входа в систему;
