Состояние подготовки релиза Debian GNU/Linux 6.0 (Squeeze)

Разработчики проекта Debian опубликовали отчет о состоянии подготовки релиза Debian GNU/Linux 6.0 (Squeeze). В настоящий момент неисправленными остаются 400 влияющих на выпуск стабильного релиза ошибок, из которых 300 еще не имеют исправлений в ветке "unstable". Общее число критических ошибок составляет 720, для заморозки пакетной базы желательно снижение числа таких ошибок примерно до 300. Из утвержденных на включение в Debian GNU/Linux 6.0 версий программ названы: Linux ядро 2.6.32, Python 2.6, X.Org 7.5, Gnome 2.30, Qt 4.6 и KDE 4.4. Вместо стандартной системной библиотеки GNU C Library (glibc) будет использована библиотека eglibc 2.11. Для реализации графического режима установки в инсталляторе вместо DirectFB будет задействован стандартный X.Org. Подробный обзор улучшений в программе установки можно посмотреть в тексте анонса альфа-версии инсталлятора, а список изменений в общей организации работы дистрибутива в анонсе предстоящей заморозки пакетной базы.
