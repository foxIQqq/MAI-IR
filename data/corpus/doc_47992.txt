Red Hat поглощает компанию CoreOS

Компания Red Hat объявила о покупке компании CoreOS, занимающейся развитием средств для сопровождения и организации работы систем контейнерной изоляции. Сумма сделки составляет 250 млн долларов. Среди развивавшихся в CoreOS проектов, которые конкурировали с решениями Docker и Red Hat Atomic, можно отметить атомарно обновляемое Linux-окружение Container Linux/CoreOS (содержит только минимальный набор компонентов, необходимых для запуска контейнеров), реестр образов контейнеров Quay, инструментарий управления контейнерами rkt, распределенную систему хранения конфигурации etcd, платформу оркестровки контейнеров Tectonic (на базе Kubernetes), систему хранения Torus, инструментарий для начальной конфигурации системы Ignition, конфигуратор сети Flannel, сервис идентификации dex, систему для анализа уязвимостей в начинке контейнеров Clair. Большинство продуктов CoreOS развивалось как свободное ПО, недостающие исходные тексты будут открыты в ближайшие месяцы. После завершения сделки компания Red Hat намерена интегрировать технологии CoreOS в свои продукты OpenShift и Red Hat Atomic, а также предложить пользователям ряд новых продуктов на основе разработок CoreOS. План создания комбинированных продуктов и миграции на них существующих клиентов CoreOS, среди которых eBay и Salesforce, будет опубликован позднее. Проект Tectonic, который пересекается с собственными разработками Red Hat, связанными с платформами OpenShift и Kubernetes, планируется использовать для расширения функциональности OpenShift в области автоматизации выполнения операций и управления. Реестр Quay планируется продолжить использоваться в качестве обособленного компонента, а также использовать связанные с ним наработки для улучшения Red Hat Container Catalog и интегрированных в OpenShift средств работы с реестром. Container Linux, а также инструментарии rkt и etcd, пересекаются с продуктом Red Hat Enterprise Linux Atomic Host, интегрированными в RHEL инструментами управления платформой и runtime для контейнеров. Компания Red Hat намерена создать объединённую команду разработчиков для проведения работы по рецензированию, интеграции и совмещению технологий Container Linux с проектом Atomic. Проекты Container Linux, etcd и rkt переданы организации CNCF (Cloud Native Computing Foundation), развивающей стандарт Open Container Initiative (OCI), платформу Kubernetes и runtime containerd.
