Представлена открытая база акустических данных AcousticBrainz

Проект MusicBrainz, поддерживающий открытую базу метаданных о музыкальных произведениях, представил новую инициативу AcousticBrainz, в рамках которой начато формирование коллекции открытых акустических данных, включающих низкоуровневую спектральную информацию (например, данные о темпе) в привязке к таким высокоуровневым характеристикам, как жанр, тональность, ритм, тембр, стиль, ключ, лад (мажор/минор) и т.п. В отличие от проекта AcoustID, в рамках которого накапливаются акустические шаблоны, AcousticBrainz нацелен на сбор более всесторонних и глубоких данных, не ограничивающихся задачей идентификации музыкальных композиций. Информацию из БД AcousticBrainz можно использовать для определения характеристик конкретного музыкального произведения, например, можно автоматически определить к какому жанру оно относится. С практический точки зрения, кроме средств автоматической классификации, БД AcousticBrainz может применяться для разработки новых музыкальных технологий, в том числе для анализа структуры популярной музыки, создания генераторов списков воспроизведения и разработки движков формирования рекомендаций (например, пользователь может указать какие произведения ему нравятся и система предложит близкие по духу композиции). Коллекция формируется за счёт совместной работы сообщества заинтересованных энтузиастов. Для выполнения акустического анализа предлагается использовать свободный инструментарий Essentia. Для отправки данных подготовлено специальное клиентское ПО (GUI и утилиты командной строки) для Linux, Windows и OS X. Первый выпуск AcousticBrainz включает в себя данные, полученные в результате анализа 650 тысяч музыкальных треков. Конечной целью проекта является формирование коллекции акустических данных для всех музыкальных произведений в мире. Вся информация распространятся под лицензией CC0 (общественное достояние).
