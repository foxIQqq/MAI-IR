GitHub добавил поддержку отслеживания уязвимостей в проектах на языке Dart

Компания GitHub объявила о добавлении поддержки языка Dart в сервисы для отслеживания уязвимостей в пакетах, содержащих код на языке Dart. Поддержка Dart и фреймворка Flutter в том числе добавлена в каталог GitHub Advisory Database, в котором публикуются сведения об уязвимостях, затрагивающих проекты, размещённые на GitHub, а также выполняется отслеживание проблем в пакетах, связанных зависимостями с уязвимым кодом. В каталог добавлена новая секция, позволяющая отслеживать появление уязвимостей в коде на языке Dart (в настоящее время предоставлены сведения о 3 уязвимостях). Ранее в каталоге была предоставлена поддержка репозиториев, в которых развиваются пакеты на базе Composer (PHP), Go, Maven (Java), npm (JavaScript), NuGet (С#), pip (Python), Rust и RubyGems (Ruby).
