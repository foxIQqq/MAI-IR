Доступен кандидат в релизы openSUSE 11.3. Обсуждение стратегии развития openSUSE

Вышел кандидат в релизы openSUSE 11.3, последний тестовый выпуск перед финальным релизом, намеченным на 15 июля. По сравнению с openSUSE 11.3 Milestone 7 в новом выпуске отмечены только изменения, связанные с исправлением ошибок. Кроме того, разработчики openSUSE выставили для публичного обсуждения черновые варианты серии документов, определяющих стратегию развития дистрибутива в таких направлениях, как взаимодействие с сообществом, упрощение создания производных дистрибутивов на пакетной базе openSUSE, поддержание надлежащих инструментов для разработчиков, улучшение поддержки мобильных и cloud-систем. После публичного обсуждения и учета всех замечаний и предложений, на базе данных документов будет сформирован план по дальнейшему улучшению дистрибутива. Основные улучшения в openSUSE 11.3 Обновление программ: GCC 4.5, KDE 4.4.3, GNOME 2.30.1, Xfce 4.6.1, LXDE 0.5.5, OpenOffice.org 3.2.0, Firefox 3.6.4-dev, rpm 4.8.0, zypper 1.4.1, Perl 5.12, parted 2.2 с поддержкой дисков с 4K-секторами, NetBeans 6.8, X.Org 7.5, xorg-server 1.8.0, seamonkey 2.0.4, Amarok 2.3, k3b 2.0 RC2, glibc 2.11.1; Поддержка адаптированного для нетбуков окружения на базе Moblin и KDE; Поддержка в инсталляторе файловой системы Btrfs; Улучшение пакетного менеджера zypper: добавлена возможность прерывания работы PackageKit через DBus (блокировка zypper во время проведения обновлений и дополнительных фоновых установок сильно раздражала пользователей); улучшена обработка ошибок при чтении данных с разнообразных носителей; улучшена обработка зависимостей пакетов, если пакет удаляется при помощи опции --clean-deps можно удалить все связанные только с этим пакетов зависимости; улучшены средства для выбора пакетов для установки/обновления/удаления ( [теперь поддерживается синтаксис +/-]repo:type:nameOPversion.arch). Для организации прямой загрузки в команды "in", "up", "dup", "ve" и "patch" добавлена опция "download", поддерживающая режимы "only", "in-advance", "in-heaps" и "as-needed". Утилита для фоновой загрузки пакетов aria2 обновлена до версии 1.8.0, в которой реализовано большое число новых XML-RPC методов и добавлена возможность записи меда-данных в виде torrent-файлов. В систему интегрирован видеодрайвер nouveau. Для видеокарт Intel, ATI и Nvidia (драйвер nouveau) в систему интегрировать KMS-модуль для переключения видеорежимов на уровне ядра. Вместо драйвера xf86-video-radeonhd для видеокарт ATI/AMD теперь используется по умолчанию xf86-video-ati с поддержкой KMS/DRI. Предусмотрена возможность быстрого выключения KMS; Для резервного копирования, синхронизации данных между машинами и организации совместного доступа к файлам задействована система SpiderOak; На установочный DVD, наряду с GNOME, KDE и Xfce, интегрировано десктоп-окружение LXDE; Реализован механизм установки модулей и программ KDE в момент необходимости в них; Добавлен инструмент Imagewriter для копирования ISO-образов на USB Flash.; В систему добавлены разработанные в рамках проекта RusXMMS библиотеки librcd и librcc для перекодирования тегов на лету; В состав дистрибутива интегрирован визуальный анализатор дискового пространства xdiskusage; Осуществлен переход на использование новой реализации системы Cron - cronie, разработанной в недрах проекта Fedora. Переход на систему инициализации upstart, разработанную изначально для дистрибутива Ubuntu, но также используемую и в проекте Fedora. Upstart базируется на парадигме обработки событий, а для определения последовательности запуска сервисов и оценки возможности их выполнения в параллельном режиме, применяется метод учета зависимостей. По умолчанию активирован модуль conntrack, позволяющий выполнять операции фильтрации сетевых соединений с учетом TCP-сессий; Произведена интеграция в дистрибутив нового пользовательского интерфейса Moblin для нетбуков; В LiveCD добавлены средства (пакет guest-additions) для акселерации работы под управлением системы виртуализации virtualbox; В поставку включен Grub2, который не планируется использовать по умолчанию; В качестве программы для просмотра PDF в GNOME LiveCD использован epdfview; В инсталляционном DVD диске появилась возможность выбора установки рабочего стола на базе десктоп-окружения LXDE; Обновлен набор конфигураторов и плагинов для настройки сети, в частности NetworkManager 0.8 и ModemManager 0.3;
