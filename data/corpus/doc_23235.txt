Новый Apache-проект для создания сервисов на основе REST

Увидел свет новый проект, целью которого является разработка среды для создания web-сервисов, использующих технологию REST, выпустил первый релиз. Apache Wink на данный момент находится в Инкубаторе, где "набирает силы" перед тем, как стать полноправным проектом Apache. Исходный код Wink был внесен в инкубатор компаниями HP и IBM, которые и в дальнейшем собираются принимать участие в его судьбе. Версия 0.1 среды разработки содержит модули сервера и клиента, обрабатывающие данные согласно спецификации сервисов на основе REST. Текущая их реализация опирается на JAX-RS 1.0 (Java API для RESTful Web Services; JSR 311), к которой в дальнейшем будут добавлены дополнительные функции, облегчающие построение web-служб REST. REST или передача состояния представления (Representational State Transfer) — это новая популярная тенденция в разработке web приложений. С ее помощью каждый web ресурс сети представляется в виде объекта, доступ к которому может быть организован через стандартные методы протокола HTTP, такие, как GET, POST, PUT и DELETE.
