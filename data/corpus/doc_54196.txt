Декабрьское обновление дистрибутива Raspberry Pi OS

Разработчики проекта Raspberry Pi опубликовали декабрьское обновление дистрибутива Raspberry Pi OS (Raspbian), основанного на пакетной базе Debian 10 "Buster". Для загрузки подготовлены три сборки - сокращённая (438 МБ) для серверных систем, с рабочим столом (1.1 ГБ) и полная с дополнительным набором приложений (2.9 ГБ). Дистрибутив поставляется с пользовательским окружением PIXEL (ответвление от LXDE). Для установки из репозиториев доступно около 35 тысяч пакетов. В новом выпуске: Осуществлён переход на использование звукового сервера PulseAudio, вместо вывода через низкоуровневый интерфейс звуковой подсистемы ALSA. Использование PulseAudio позволило снять ограничение на одновременное использование только одного устройства входа и выхода звука, а также реализовать возможность обработки передачи звука через Bluetooth и смешивания звуков нескольких приложений. Отображаемый в панели интерфейс для управления громкостью и звуковыми устройствами остался почти тем же. Изменение в интерфейсе затрагивает лишь средства выбора устройств входа и выхода звука, в котором теперь используются звуковые профили, позволяющие использовать разные каналы вывода в одном устройстве, например, для Bluetooth-устройств можно выбрать профиль HSP для режима гарнитуры или профиль A2DP для высококачественного вывода звука. Браузер Chromium обновлён до выпуска 84, в котором появилась поддержка доступных в Raspberry Pi механизмов аппаратного ускорения видео. Изменение позволило заметно повысить качество воспроизведения видео, как при просмотре роликов на сайтах, подобных YouTube, так и при использование сервисов проведения видеоконференций, таких как Google Meet, Microsoft Teams и Zoom. В базовый состав включены сервер печати CUPS и интерфейс настройки вывода на печать system-config-printer, который предложен в качестве конфигуратора по умолчанию. Ранее для поддержки вывода на печать нужно было отдельно устанавливать CUPS из репозитория. Улучшена поддержка инструментов для людей с проблемами со зрением. В экранном ридере Orca улучшено управление конфигуратором Raspberry Pi. Для быстрой установки Orca добавлена клавиатурная комбинация ctrl+alt+пробел. В конфигураторе для плат с одним светодиодным индикатором (Raspberry Pi Zero или Raspberry Pi 400) предоставлена настройка для выбора режима использования индикатора - для показа включения питания или для отражения активности диска. Во вкладке управления производительностью добавлена настройка подключения кулера при использовании Raspberry Pi Case Fan - пользователь может выбрать к каким выводам GPIO подключён кулер и задать температуру, при которой он будет включаться.
