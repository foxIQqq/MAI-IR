Релиз дистрибутива Debian 7.4

Анонсирован выпуск дистрибутива Debian 7.4, в который включены все доступные обновления пакетов и устранены недоработки в инсталляторе. Напомним, что начиная с Debian Wheezy изменена схема нумерации релизов - первая цифра в номере версии теперь связана со значительными выпусками, а вторая с корректирующими обновлениями. Установочные сборки доступны для загрузки. Системы, установленные ранее и поддерживаемые в актуальном состоянии, получают обновления, присутствующие в Debian 7.4, через штатную систему установки обновлений. Включенные в Debian 7.4 исправления проблем безопасности доступны пользователям по мере выхода обновлений через сервис security.debian.org. Debian 7.4 включает 37 обновлений с устранением серьезных проблем со стабильностью и 37 обновлений с устранением уязвимостей. Из наиболее важных изменений можно отметить: Из состава дистрибутива удалён пакет Iceape (Seamonkey) в связи с прекращением работы по портированию исправлений с устранением уязвимостей. Ядро Linux обновлено до версии 3.2.54, из версии 3.4.76 портированы модули drm и agp. Обновлён до последней версии инструментарий для управления изолированными контейнерами LXC. В пакете openssl включены ассемблерные вставки для ARM-архитектур. Включена поддержка ec_nistp_64_gcc_128 для *-amd64. Из пакета ruby-gsl удалена документация под несвободной лицензией, а из пакета ruby-opengl набор примеров с неопределённой лицензией. В пакет wget добавлена поддержка SNI (Server Name Indication, позволяет обеспечить доступ по HTTPS к виртуальным хостам на одном IP). В Gnash устранены проблемы с просмотром видео через Flash-плеер Youtube, используя медиаобработчик ffmpeg.
