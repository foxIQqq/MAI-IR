Вышел Zenwalk Core Linux 6.2

Вышел релиз минималистичной "Core" сборки (iso 170 Мб) совместимого с пакетной базой Slackware пользовательского дистрибутива Zenwalk Linux 6.2. Core сборка не содержит графических приложений и является базовой платформой для построения основанных на Zenwalk пользовательских дистрибутивов. Основные улучшения: Переход на использование LZMA сжатия в пакетах программ; В качестве файловой системы по умолчанию используется EXT4; Увеличение скорости загрузки и переработка скриптов инициализации; Автоопределение аппаратной конфигурации в инсталляторе, позволяющее выбрать вариант Linux ядра оптимального для текущей системы (архитектура i386/x86_64, sata, pata, scsi); Переход на Linux ядро 2.6.30.5;
