Новые версии свободных игр Wargus 2.2.6 и Alien Arena 7.53

Представлен релиз Wargus 2.2.6, игры в жанре стратегии в реальном времени на базе коллекции игровых данных Warcraft II и движка Stratagus. Поддерживается как сетевая игра, так и сражения с компьютером. Код игры распространяется в рамках лицензии GPLv2, бинарные сборки доступны для Windows и широкого спектра Linux-дистрибутивов, включая Maemo 5. Неофициально развиваются порты для мобильных платформ Symbian, Android и Windows Mobile 6. Для запуска Wargus требуется наличие графических и звуковых файлов, скопированных с оригинального CD c Warcraft II. Одновременно представлен релиз открытого 3D-шутера Alien Arena 7.53, построенного на базе движка CRX, основанного в свою очередь на движке id Tech 2. Исходные тексты игры распространяются в рамках лицензии GPL, бинарные сборки доступны для Linux и Windows. В новой версии значительно увеличена производительность (например, задействованы VBO - OpenGL Vertex Buffer Objects); добавлено два новых игровых уровня (ctf/tca); улучшена работа искусственного интеллекта ботов в плане навигации, сражений и поведения персонажей; мета-данные с описанием игры выделены в отдельные файлы, что позволяет создавать несколько разных типов игр и конфигураций на базе одной карты; добавлена реализация легковесного DeathMatch-сервера; улучшены визуальные эффекты.
