Начат сбор средств для превращения Pitivi в профессиональный инструмент видеомонтажа

Разработчики системы нелинейного видеомонтажа Pitivi объявили о проведении через организацию GNOME Foundation краудфандинг-кампании по сбору средств для перевода проекта на новый уровень. Собранные средства позволят разработчикам уделить всё своё время развитию проекта и превратить Pitivi в простой и пригодный для профессионального применения видеоредактор. Для реализации всех планов необходимо собрать €100 000, для выпуска релиза Pitivi 1.0 будет достаточно €35 000. В отличие от других кампаний по сбору средств, пользователю, отправляющему пожертвование, предоставляется возможность распределения средств на реализацию тех или иных новых функций. В качестве первой задачи отмечается контроль качества, стабилизация кодовой базы и написание документации для выпуска Pitivi 1.0. На втором этапе планируется переработать интерфейс для применения видеоэффектов, реализовать функцию копирования и вставки через буфер обмена, создать API для разработки плагинов и расширения интерфейса пользователя, обеспечить интеграцию с приложением для редактирования субтитров, подготовить плагины для быстрой публикации результатов работы в Web, добавить поддержку тегов и привязку метаданных, переработать шкалу времени, добавить поддержку форматов AVCHD, EDL и fcpx, расширить средства рендеринга, подготовить плагин для распознавания лиц, добавить средства для корректировки цветов, выпустить сборки для Windows и OS X, реализовать средства для совместного редактирования и т.д.
