Выпуск Chrome OS 99

Доступен релиз операционной системы Chrome OS 99, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 99. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 99 доступна для большинства актуальных моделей Chromebook. Исходные тексты распространяются под свободной лицензией Apache 2.0. Кроме того, продолжается тестирование Chrome OS Flex, редакции для Chrome OS использования на обычных компьютерах. Энтузиастами также формируются неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Основные изменения в Chrome OS 99: В функции Nearby Share, позволяющей быстро и безопасно передавать файлы находящимися поблизости устройствам, на которых запущен браузер Chrome, реализована поддержка фонового сканирования устройств. Фоновое сканирование даёт возможность выявлять устройства, готовые к передаче данных, и уведомлять пользователя об их появлении, что позволяет начать передачу без перехода в режим поиска устройств. Добавлена возможность возвращения полноэкранного режима открытия приложений после разблокировки устройства. Раньше при возвращения из спящего режима полноэкранные приложения возвращались в оконный режим, что мешало нормальной работе с виртуализированными рабочими столами. Файловый менеджер (Files) теперь поставляется в форме SWA-приложения (System Web App), а не Chrome App. Функциональность осталась без изменений. Проведена оптимизация управления с сенсорных экранов и улучшена обработка мультитач жестов. В обзорном режиме (Overview) предоставлена возможность перемещения окон мышью на новый виртуальный рабочий стол, который создаётся автоматически. В приложении для работы с камерой добавлена возможность записи видео в форме анимированных GIF-изображений. Размер подобных видео не может превышать 5 секунд. Устранены уязвимости: проблемы с аутентификацией в VPN-клиенте, обращение к уже освобождённой памяти в оконном менеджере, Nearby Share, ChromeVox и интерфейсе вывода на печать.
