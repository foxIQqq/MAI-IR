Началось бета-тестирование FreeBSD 10.2

Доступен для тестирование первый бета-выпуск FreeBSD 10.2. Релиз FreeBSD 10.2запланирован на 31 августа. Выпуск FreeBSD 10.2-BETA1 доступен для архитектур amd64, i386, ia64, powerpc, powerpc64, sparc64 и armv6. Дополнительно подготовлены образы для систем виртуализации (QCOW2, VMDK, raw) и облачных окружений Amazon EC2 и Microsoft Azure. Среди наиболее заметных изменений: Добавлена новая утилита uefisign для формирования цифровых подписей для файлов EFI, используемых для загрузки в конфигурациях с UEFI Secure Boot; В базовую систему включена новая утилита dpv, наглядно отображающая прогресс передачи данных для одного или нескольких потоков; Добавлена утилита fstyp для определения типа файловой системы на указанном разделе; Добавлена библиотека figpar, предоставляющая API для разбора файлов конфигурации; В сетевой драйвер em добавлена поддержка многоуровневых очередей (multiqueue). Включение производится через опцию ядра EM_MULTIQUEUE; В ядре GENERIC по умолчанию включена поддержка RACCT и RCTL, подсистем для аккаунтинга и лимитирования ресурсов; Код drm-драйверов для графических карт синхронизирован с ядром Linux 3.8.13; Для ZFS добавлена настройка vfs.zfs.spa_slop_shift, позволяющая изменить зарезервированное по умолчанию дисковое пространство; В загрузчик добавлена поддержка ввода пароля для GELI-шифрования на этапе до загрузки ядра; Обновлён гипервизор bhyve и драйверы для Hyper-V; Обновлены версии сторонних программ, в том числе resolvconf 3.7.0, acpi 20150515, file 5.23, unbound 1.5.3, sendmail 8.15.2, OpenSSL 1.0.1p.
