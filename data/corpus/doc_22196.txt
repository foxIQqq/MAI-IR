Релиз системы высокой готовности для Carrier Grade Linux - OpenSAF 3.0

Проект OpenSAF анонсировал новую версию открытой системы высокой готовности для сетевой подсистемы дистрибутивов, отвечающих требованиям спецификаций Carrier Grade Linux (CGL). В OpenSAF 3.0 добавлено множество инструментов управления платформой, улучшены ее пользовательские качества, а также включена поддержка программных интерфейсов языка Java. Наиболее важные нововведения OpenSAF 3.0: Более полная поддержка текущей спецификации SAF для систем высокой готовности, включающая специальную программную среду управления доступностью (Availability Management Framework, AMF). Новый сервис управления информационными моделями (Information Model Management, IMM) позволяет стандартизировать как работу с приложениями, так и саму OpenSAF. Новая система оповещения (NTF) стандартизирует процесс обработки системных сообщений. Множественные улучшения эксплуатационных свойств системы, в том числе простая в использовании система сборки модулей. Поддержка Java API. OpenSAF 3.0 распространяется свободно под лицензией LGPL 2.1 и может быть загружен с официального сайта проекта. В то, чтобы новый релиз состоялся внесли свой вклад более 200 участвующих организаций и индивидуальных контрибьюторов. Beta-версия всесторонне тестировалась начиная с августа прошлого года. Новый релиз более полно отвечает спецификациям SAF и SCOPE.
