Фонд СПО признал проект GNU PDF состоявшимся и исключил его из списка приоритетных разработок

Фонд свободного ПО объявил о признании проекта GNU PDF состоявшимся и исключении его из списка приоритетных проектов, в котором отражены наиболее важные области для которых не созданы достойные открытые альтернативы проприетарным решениями. Целью проекта GNU PDF было создание и распространение свободного набора высококачественных и многоплатформенных компонентов, позволяющих работать с форматом PDF в полном соответствии со спецификацией PDF 1.7 и стандартом ISO 32000-1. Изначально в рамках проекта развивалась библиотека GNU PDF Library, на базе которой разрабатывается программа для просмотра и редактирования PDF - GNU Juggler. Проект создан в начале 2007 года, при этом основным камнем преткновения, мешающим достижению цели, было отсутствие поддержки некоторых расширений, определенных в стандарте ISO 32000-1. В настоящее время все трудности остались позади и фонд СПО с гордостью заявляет, что в рамках свободной библиотеки libpoppler, развиваемой на кодовой базе xpdf 3.0, доведена до рабочего состояния поддержка подобных расширений, в частности, возможности использования аннотаций и определения заполняемых форм. Используя данную библиотеку в любое свободное приложение можно легко интегрировать полнофункциональную поддержку формата PDF. Среди проектов, которые уже используют libpoppler, можно упомянуть: Evince, Gimp, Cups, TexLive, Inkscape, Karbon и плагин pdfimport к OpenOffice.org/LibreOffice. Кроме того, в рамках проекта pdfreaders.org сформирована большая подборка ссылок на открытые программы для просмотра PDF, доступные для разных операционных систем. Среди таких программ: BePDF, ePDFView, Evince, Gv, KPDF, MuPDF, Okular, Sumatra PDF, Vindaloo, Xpdf, Zathura, Ghostscript. В настоящее время в список приоритетных проектов Фонда свободного ПО входят следующие проекты: Gnash - развитие свободного проигрывателя Flash-контента; Coreboot (бывший LinuxBIOS) - разработка свободной альтернативы проприетарным BIOS; Разработка свободной альтернативы Skype. В качестве примеров свободных проектов, развивающихся в этой области отмечены: Ekiga, Twinkle, Coccinella, QuteCom и SIP-Communicator. Также упоминается проекты Mingle (реализация одновременных вызовов для Jabber) и openMSRP (реализацию протокола MSRP); Свободное ПО для редактирования видео (упоминаются такие проекты, как Kino, Cinelerra, Avidemux, Kdenlive, LiVES, Lumiera, PiTiVi, Blender и Open Movie Editor. ) Свободная альтернатива геоинформационной системы Google Earth (как вариант рассматривается проект Marble и сервис Open Street Maps); Развитие полностью свободных Linux-дистрибутивов (например, gNewSense, Trisquel и Dragora); Разработка свободной альтернативы математическому пакету Matlab (например, GNU Octave); LibreDWG - разработка аналога пакету OpenDWG, с реализацией поддержки популярных форматов CAD-систем; Разработка функций обратимой отладки для GDB (возможность вернуться на определенное число шагов назад); Создание свободных прошивок и драйверов для сетевых маршрутизаторов (в пример приводится проект OrangeMesh). Разработка свободной замены Oracle Forms, работающей со свободными СУБД; Создание свободного ПО по автоматическому преобразованию речи в текст (в качестве примера приводится сервис автоматизации создания субтитров в YouTube.) Из свободных проектов, работающих в этом направлении можно упомянуть CMU Sphinx, Simon и VoxForge; Создание полнофункциональных свободных драйверов для графического движка PowerVR, используемого в телефонах и нетбуках. Среди завершённых проектов, которые как и GNU PDF признаны состоявшимися и убраны из списка, упоминаются TheUnarchiver, позволяющий обеспечить распаковку архивов в формате RARv3, и CiviCRM - пакет для организации взаимодействия некоммерческих организаций с клиентами, спонсорами, членами сообществ, волонтёрами и соискателями на получение грантов. Из более ранних успешных проектов, признанных завершёнными, можно вспомнить: создание полноценного компилятора GCJ (GNU Compiler for Java); развитие проекта GNU Classpath; разработку открытых 3D-драйверов для видеокарт ATI; адаптацию OpenOffice.org для работы с GCJ.
