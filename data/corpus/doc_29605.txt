Для Linux продолжено портирование игрового движка Source и игры Postal III

Руководитель компании Running With Scissors подтвердил информацию о продолжении разработки Linux-клиента для игры Postal III, основанной на игровом движке Source. Если все пойдет по плану, то первый публичный выпуск Postal III для Linux увидит свет в мае. В скором времени ожидается завершение подготовки первой бета-версии Postal III для Linux, разработка которой ведется в Москве. Портирование Source Engine позволит в будущем адаптировать для Linux и другие игры на базе данного игрового движка, такие как Half-Life 2, Counter-Strike: Source и Team Fortress 2.
