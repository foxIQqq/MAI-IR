Первая альфа-версия Ubuntu 15.04

Объявлено о начале альфа-тестирования дистрибутива Ubuntu 15.04 "Vivid Vervet". В соответствии с планом разработки, уже опробованном при подготовке прошлых выпусков, отдельные альфа-сборки не будут сформированы, вместо них для тестирования предлагается использовать ежедневные экспериментальные сборки. Готовые тестовые образы созданы только для Kubuntu, Lubuntu, Ubuntu GNOME и Ubuntu Cloud. Второй альфа-выпуск Ubuntu 15.04 запланирован на 22 января, бета-выпуск ожидается 26 февраля. Релиз намечен на 23 апреля. В альфа-выпуске осуществлена синхронизация новых версий пакетов из репозиториев Debian Unstable. Ядро Linux обновлено до версии 3.18. В версии 15.04 по-прежнему по умолчанию предлагается окружение Unity 7, а сборка с Unity 8 доступна в качестве опции. В Kubuntu осуществлён переход на окружение рабочего стола KDE Plasma 5, которое заменило собой KDE 4. В состав включены приложения из свежего выпуска KDE Applications 14.12.0. В Ubuntu GNOME пока поставляется Gnome-shell 3.12, но в следующем альфа-выпуске ожидается поставка 3.14. В настоящее время из GNOME 3.14 в дистрибутив уже перенесено несколько приложений. В gnome-terminal возвращена поддержка прозрачного фона. Кнопка закрытия окна теперь размещается на штатной для GNOME позиции, для её перемещения в левый угол следует использовать gnome-tweak-tool. Дополнительно можно отметить доведение до рабочего состояния средств для бесшовного запуска X11-приложений в окружении Unity 8 c использованием прослойки Xmir, в том числе с обеспечением недоступной ранее поддержки OpenGL и архитектуры 2D-ускорения Glamor.
