В Испании стартуют продажи первых потребительских смартфонов на базе Firefox OS

Компании Telefónica, ZTE и Mozilla объявили о начале продаж в Испании ZTE Open, первого телефона на базе Firefox OS, ориентированного на конечных потребителей. Одновременно введён в строй новый вариант сайта проекта Firefox OS. Напомним, что с апреля для заказа доступны модели телефонов Keon и Peak, но данные устройства позиционировались исключительно для разработчиков. По своим характеристикам ZTE Open уступает модели Keon, позиционируемой как прототип простого и недорогого устройства на базе Firefox OS. ZTE Open поставляется всего с 256MB ОЗУ и 512MB Flash, но в комплекте доступна SD-карта размером 4 Гб. Как и в Keon устройство снабжено 3.5-дюймовым сенсорным экраном (HVGA, 480 x 320) с поддержкой мультитач, CPU 1 GHz Qualcomm Snapdragon MSM7225A и 3.2-мегапиксельной камерой. Из приложений отмечается браузер Firefox с предустановленным доступом к online-картам Nokia HERE и сервисами интеграции с социальными сетями Facebook и Twitter. Начиная с завтрашнего дня модель ZTE Open можно будет купить в Испании через сеть магазинов Movistar по цене 69 евро. Через несколько недель продажи начнутся в Колумбии и Венесуэле. В пресс-релизе отмечается, что ZTE Open не просто первое устройство на базе Firefox OS, но и первый действительно открытый телефон, несущий открытость Web в сферу мобильных устройств. По мнению Telefónica уже назрела потребность в более открытых смартфонах и платформа Firefox OS, построенной вокруг открытых web-технологий, как нельзя лучше подходит для достижения данной цели. Дополнительно упоминается о подготовке к началу поставки на европейский рынок телефонов на базе Firefox OS компаниями Deutsche Telekom и Alcatel, которые совместно готовят к выпуску модель Alcatel One Touch Fire, по характеристикам похожую на ZTE Open. В качестве главных преимуществ новых устройств называется низкая цена и поддержка установки приложений из произвольных web-каталогов, без привязки к магазинам отдельных поставщиков. Экосистема приложений для Firefox OS ещё только формируется, но уже отмечено появление мобильных приложений для Firefox OS от известных контент-провайдеров и социальных сетей. Производители телефонов на базе Firefox не привязаны к базовому каталогу-магазину Firefox Marketplace и могут создавать собственные подконтрольные каталоги распространения приложений. Приложения для Firefox OS создаются с использованием открытых web-технологий, могут распространяться без посредников и не требуют создания отдельных сборок для устройств различных типов и с различной архитектурой. Используемые в Firefox OS спецификации WebAPI по возможности продвигаются в качестве web-стандартов. Грани между приложением для Firefox OS и обычным web-приложением размыты и условны: если разработчик уже создаёт приложения для Web, то уже может создавать приложения и для Firefox OS. Из особенностей Firefox OS отмечается также новая для смартфонов концепция адаптивного поиска приложений, которые можно использовать для удовлетворения текущих потребностей без необходимости локальной установки. Подобные приложения по способу работы напоминают специально адаптированные для смартфонов сайты, то есть программу не нужно предварительно загружать и устанавливать, а достаточно открыть в браузере. При желании приложения для Firefox OS можно установить на телефон для постоянного использования или работы в offline, но если они требуются только один раз или время от времени, можно мгновенно запустить их из Web без какой-либо предварительной установки.
