Вышел Calculate Linux Desktop 9.4

Вышел релиз десктоп-ориентированного дистрибутива Calculate Linux Desktop 9.4, представляющего собой скомпилированный stage4-образ Gentoo Linux, снабженный программной для быстрой установки. Основные изменения: Добавлен индикатор процесса загрузки профиля при входе в домен CDS. Оптимизирована скорость загрузки профиля с удаленного сервера. Добавлен параметр calculate --set-composite=on|off для включения композитного режима. По умолчанию параметр выключен. При обновлении системы эта настройка переносится. Переработанный набор иконок Calculate icons pack теперь интегрирован в KDE. Добавлена поддержка запуска LiveCD в виртуальных машинах с эмулятором SCSI устройств. Добавлен пакет виртуализации VirtualBox, поддерживающий запуск операционных систем Microsoft Windows, GNU/Linux, Mac OS X и SUN Solaris/OpenSolaris. Программа записи CD и DVD дисков Graveman заменена на K3B. Пакет gtk-kde4 заменен на QtCurve - набор стилей виджетов для придания однообразного вида GTK2, Qt3, Qt4 приложениям. Добавлена программа Porthole - графический интерфейс для Portage. Состав: KDE 4.2.1, Kernel 2.6.27.21, OpenOffice 3.0.1, X.Org 7.4.
