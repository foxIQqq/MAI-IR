Тестовый выпуск Firefox OS 1.3 для разработчиков

Проект Mozilla выпустил предварительную версию мобильной платформы Firefox OS 1.3, нацеленную на первичное ознакомление разработчиков приложений. Выпуск основан на движке Gecko 28, также используемом в Firefox 28. Тестовая версия Firefox OS 1.3 доступна как в виде образа прошивки для смартфонов Geeksphone Keon и Peak, так и виде симулятора, реализованного в форме дополнения к настольной версии Firefox. Основные новшеcтва: Обеспечена возможность использования на устройствах с несколькими SIM-картами (Multi-SIM). Добавлена поддержка кодека Opus в WebM, декодировщика для формата VP9 и поддержка VTT (текстовые субтитры для видео). Для сертифицированных приложений включен по умолчанию DataStore API, позволяющий приложению предоставить доступ к своим данным для других приложений или задействовать в нескольких приложениях общее хранилище. Добавлен SpeakerManager API. Добавлен API для NFС, интерфейс для связывания устройств по NFC и поддержка чтения меток по NFC. В почтовый клиент включена поддержка POP3; Расширена поддержка MMS/SMS, обеспечена возможность отправки MMS на email и указания заголовка в MMS; Для приложений и web-контента реализована поддержка асинхронного панорамирования и масштабирования; Поддержка управления воспроизведением музыки из хранителя экрана и области уведомлений; В программу для работы с камерой добавлена поддержка непрерывного автофокуса и вспышки; Поддержка группировки изображений в менеджере фотографий; Увеличена производительность прокрутки и сокращены задержки в процессе работы интерфейса; Реализация фреймворка для потокового вещания по rtsp, поддержка RTP/RTCP через UDP/IP и RTSP через TCP/IP; Добавлен инженерный режим работы; Обновлена виртуальная клавиатура; Завершена первая фаза модернизации интерфейса пользователя.
