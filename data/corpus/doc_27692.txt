Релиз менеджера фотографий Shotwell 0.7

Вышел релиз программы для управления коллекцией фотографий Shotwell 0.7, которая будет включена в базовую поставку дистрибутива Ubuntu 10.10, заменив собой основанный на Mono проект F-Spot. Программа сочетает в себе: инструменты для импорта и конвертации новых фотографий; удобные возможности каталогизации и навигации по имеющейся коллекции, поддержка группировки по времени и тегам; поддержку выполнения типовых операций по обработке изображений (вращение, устранение эффекта красных глаз, корректировка экспозиции, оптимизация цветности и т.п.); средства для публикации в социальных сетях, таких как Facebook, Flickr и Picasa. В новой версии представлены следующие новшества: Добавлены функции для упрощения миграции с F-Spot: Shotwell теперь может напрямую импортировать фотографии из библиотеки F-Spot, сохраняя все установленные теги и параметры рейтинга; Реализована возможность оценки фотографий по пятибальной шкале и пометки изображений для последующего игнорирования по умолчанию. При просмотре фотографий фильтр можно настроить для показа только изображений с рейтингом не ниже определенного порога; В боковой панели появился блок со списком последних импортированных фотографий, позволяющий иметь под рукой наиболее свежие фото; Улучшено оформление элементов управления и боковой панели, добавлены новые пиктограммы и реализована функций переименования на лету. Из планов на будущее отмечаются такие возможности, как автоматическое отслеживание появление новых фотографий в файловой системе, добавление видеоэффектов и фоновой музыки при показе слайдшоу, инструменты для автоматической группировки фотографий по данным о местоположении и размещении на диске, базовая поддержка показа видео, механизмы для синхронизации фотографий между несколькими компьютерами. Пользователи Ubuntu могут установить новую версию, воспользовавшись специально подготовленным PPA-репозиторем. Готовые бинарные сборки также доступны для Fedora Linux и Windows.
