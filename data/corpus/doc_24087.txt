Платформа Java SE 5 достигла стадии полного прекращения поддержки

30 октября, после пяти с половиной лет существования, платформа Java SE 5 достигла стадии прекращения поддержки. Отныне для Java 5 не будут выпускаться исправления ошибок, но как показывает практика, исправления особо критических уязвимостей продолжают выходить, в порядке исключения, и после прекращения времени поддержки. Всем пользователям рекомендуется перейти на платформу Java 6 или приобрести услугу бизнес поддержки, в рамках которой обновления для Java 5 будут предоставляться до 2019 года.
