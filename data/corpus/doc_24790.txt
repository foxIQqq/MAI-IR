Вышла бета-версия проприетарного видеодрайвера NVIDIA 195.30

Выпущена новая бета-версия проприетарных драйверов Nvidia 195.30. Кроме Linux, данная версия доступна одновременно для Solaris и FreeBSD. В версии 195.30 отмечены следующие изменения: Устранены регрессивные изменения, приводящие к потере производительности при выполнении операций с текстом в KDE4 без использования сглаживания; Задействована аппаратная акселерация для многих операций композитного рендеринга (например, смешивание альфа-компонентов), которые ранее были реализованы программным путем; Добавлена экспериментальная поддержка аппаратной акселерации для выполнения операции рендеринга "CompositeTrapezoids". Для включения необходимо выполнить команду "nvidia-settings -a AccelerateTrapezoids=1". Данный режим работает только с картами GeForce GTX; Исправлена ошибка в VDPAU, приводящая при определенных обстоятельствах к излишней нагрузке на CPU; В командной строке теперь можно выполнить команду "nvidia-settings --query all" для вывода отчета по состоянию всех доступных атрибутов.
