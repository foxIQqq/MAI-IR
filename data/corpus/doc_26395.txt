Первый стабильный релиз системы управления контентом MjNCMS

Вышел первый стабильный релиз новой системы управления web-контентом MjNCMS, написанной на языке Perl и распространяемой в рамках свободной лицензии Artistic. Особенности новой CMS: Единая авторизация с форумом (smf, расширяема). Модель разграничения доступа делит пользователей на рабочие места и роли на этих местах; Пользователь может менять место:роль, выбирая комбинацию из разрешённых; Пользователь может подменять других пользователей (прозрачная замена); Весь цикл регистрации на сайте: регистрация, подтверждение, профиль, напоминание пароля; Редактор меню, категорий, страниц, их переводов, коротких url'ов; Система расстановки разрешений - для рабочего места, для роли Поддержка кластерной конфигурации mysql (запись на один сервер, чтение с других), разные группы кластеров для разных целей; Поддержка групп memcached серверов; Многоязычность, поддержка русского и английского языков реализована из коробки; В интерфейсе администратора присутствует поддержка выполнения переводов текста; Пользовательские сессии реализован по аналогии с PHP (т.е. хеш данных сохраняемый в БД) В качестве шаблонизатора по умолчанию используется Template-Toolkit.
