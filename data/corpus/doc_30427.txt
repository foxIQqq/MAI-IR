Релиз дистрибутива Superb Mini Server 1.6.0

Представлен релиз Linux-дистрибутива Superb Mini Server 1.6.0, основанного на Slackware Linux 13.37 и предназначенного для быстрого развертывания серверов для обслуживания предприятий среднего и малого бизнеса. Дистрибутив поддерживает развертывание web-сервера, межсетевого экрана, почтового сервера, сервера печати и других сетевых служб. Настройка параметров сервисов осуществляется через web-интерфейс, реализованный на базе проекта Webmin. Дистрибутив работает в Live-режиме, размер загрузочного iso-образа составляет 397 Мб. Новшества Superb Mini Server 1.6.0: Использование Linux-ядра 2.6.37.6; Поддержка файловой системы Btrfs и дисковых разделов GPT в инсталляторе; Добавление PXE-сервера для установки дистрибутива по сети; В поставку включены новые пакеты: btrfs-progs, lrzip, gdisk, rrdtools, Midnight Commander, openmpi, enchant, idnkit, dovecot-pigeonhole, iwlwifi-100-ucode, iwlwifi-6xxx-ucode и iptraf-ng. В расширенный комплект SMS.Native.CD-Extra.iso включены пакеты: nut, powerman, clusterglue, clusterresourceagents, corosync, libesmtp, libnet, openais, pacemaker; Live CD теперь использует squashfs с xz-сжатием, все модули теперь имеют расширение *.xzm (поддержка модулей *.lzm прекращена). В состав Live CD добавлены модули mpi.xzm с openmpi, gcc, gcc-g++, gcc-gfortran, kernel-headers и binutils для запуска mpi-приложений; Включение в комплект PXE-сервера и кластерного ПО, позволяет использовать дистрибутив для быстрого развертывания кластера с загрузкой узлов по сети;
