Релиз Wine 1.1.40

Традиционно и в срок вышел очередной экспериментальный релиз реализации Win32 API для Linux - Wine 1.1.40. Основные изменения этой версии: Добавлена поддержка перенаправления ветви реестра Wow64 для 64-разрядных приложений. Увеличено покрытие т.н. "действий" MSI. Улучшена работа редактора Wordpad. Добавлено несколько исправлений библиотеки shell32 для лучшей поддержки работы Internet Explorer. Улучшена поддержка данных формата typelib. Улучшен механизм обнаружения OpenGL-видеокарт. Исправлено 62 ошибки.
