Бета-тестирование Flash Player 10.2

Компания Adobe объявила о выходе первой бета-версии Flash Player 10.2 для платформ Linux, Windows и Mac OS X. Основные нововведения: Представлен Stage Video API с реализацией нового метода аппаратной акселерации всех стадий воспроизведения видео, позволяющего значительно разгрузить центральный процессор, снизить потребление памяти и повысить качество вывода. В частности, средства GPU привлекаются не только для ускорения декодирования потока H.264, но и для таких стадий, как вывод на экран, преобразование цветов, масштабирование и перемещение блоков данных из ОЗУ в видеопамять (блиттинг); Поддержка использования определенных пользователем курсоров мыши; Поддержка вывода в полноэкранном режиме для многомониторных конфигураций, что позволяет отображать контент на полном экране второго монитора не мешая работе на другом экране; Реализация нового метода субпиксельного рендеринга текста, позволяющего улучшить читаемость текста; Задействование аппаратной акселерации при формировании вывода в браузере Internet Explorer 9. Дополнение: для задействования аппаратной акселерации в Linux в новой версии Flash используется API VDPAU, который пока поддерживается только проприетарными драйверами NVIDIA и недоступен для видеоркат Intel и ATI/AMD. Ресурс Phoronix провел тестирование производительности Flash Player 10.2-beta, которые подтвердили заметное понижение нагрузки на CPU.
