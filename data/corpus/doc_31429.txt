Релиз открытой биллинговой системы ABillS 0.53

Представлен релиз свободной биллинговой системы ABillS 0.53. Одновременно объявлено о доступности сертифицированной в Российской Федерации коммерческой версии ABillS 0.6. Основные изменения ABillS 0.53 : Добавлена поддержка FreeRadius DHCP. Раздача динамических IP исходя из параметров опции 82 (порт, коммутатор, vlan), гостевой аккаунт, шейпер. Улучшена работа с Cisco ISG. Более быстрая работа с CoA интерфейсом Ashield Обновлен к версии протокола 6.0. Все действия с аккаунтом пользователя теперь производятся через веб интерфейс биллинга Новый модуль складского учёта Storage В модуль карт Maps добавлена поддержка Google Maps, возможность рисовать маршруты прокладки кабелей Добавлена поддержка платёжной системы PayPal Добавлена утилита snmp_control.pl. Управление скоростью на портах коммутаторов D-Link, Edge core, а также управление VLAN. Добавлена возможность переключения скорости в зависимости от временных интервалом для mpd шейпера с radius-парами. Улучшен интерфейс кабинета пользователяABillS 0.53 , доработан интерфейс администратора: быстродействие, эргономика Добавлена плотность подключения в журнал адресов Много изменений коснулось Voip: добавлена возможность ведения цен телефонии в условных единицах; добавлена программа получения курса НБУ и пересчёта цен исходя из курса условной единицы; добавлены группы расширенно тарификации; добавлен новый тип сервера cisco_voip; улучшена производительность авторизации и аккаунтинга модуля Voip; Много изменений коснулось модуля Abon: добавлена индивидуальная скидка для каждой услуги; добавлена опция учета скидок; добавлена возможность установки даты с которой будет производиться снятие денег модулем Abon; добавлен внешний обработчик для модуля Abon, который разрешает запускать внешние программы при активации или блокировании услуги; создание счёта при активации услуги модулем Abon (периодические платежи); периодическое оповещение о завершении услуги и отправка счёта на оплату для модуля Abon В список систем миграции добавлено MikBill, Nodeny, Stargezer, Trafpro Информационные сообщения по работе billd заносятся в журнал ошибок Добавлен механизм группирования и ротации записей журнал ошибок Добавлен экспорт в XML журналов платежей и начислений**7 Добавлена поддержка DvCrypt для модуля Iptv В журнале пользователей добавлен фильтр по кредиторам
