Выпущен релиз Linux дистрибутива для субноутбуков - Easy Peasy Linux 1.5

Выпущен релиз ориентированного на субноутбуки Linux дистрибутива Easy Peasy Linux 1.5. Дистрибутив основан на Ubuntu Netbook Remix 9.04 и Linux ядре 2.6.30, оптимизированном для субноутбуков. В новой версии: Расширен список поддерживаемых из коробки устройств. Главное отличие Easy Peasy Linux от Ubuntu Netbook Remix в более широком диапазоне поддерживаемых устройств, которые не ограничиваются моделями на базе CPU Intel Atom; Подготовлены гибридные установочные образы, поддерживающие загрузку с CD/DVD и USB-накопителей; Обновлены версии программ (Picasa, OpenOffice 3.1 и т.д.); Уменьшен размер устанавливаемой на диск базовой системы; По умолчанию задействована файловая система ext4; Для видеокарт Intel теперь используется архитектура акселерации UXA; Вместо Songbird по умолчанию интегрирован музыкальный проигрыватель Banshee; Задействованы видеоэффекты композитного оконного менеджера (десктоп-эффекты для интерфейса, оптимизированного для нетбуков).
