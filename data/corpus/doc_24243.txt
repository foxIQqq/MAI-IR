Вышел альфа-релиз InfiniDB, нового хранилища для MySQL

Вышел альфа-релиз InfiniDB, нового хранилища для MySQL, ориентированного на выполнения аналитических запросов, охватывающих большой объем данных. В отличие от других хранилищ, InfiniDB хранит данные не построчно, а с разбивкой по столбцам, что позволяет оптимизировать выполнение группировки по столбцам из БД большого размера. Особенно эффективен подобный подход, когда данные в столбцах повторяются. InfiniDB подходит для создания хранилища для средств бизнес-аналитики и программ с интенсивным чтением данных из БД (производительность добавления данных - слабое место InfiniDB). Примеры использования InfiniD можно найти в статье "MySQL Database Analytics with InfiniDB from Calpont" (часть 1, часть 2).
