Рассказ об использовании открытого ПО в online-бирже BullionVault

"BullionVault.com - Gold with an open source soul" - рассказ об online-бирже для продажи золота BullionVault.com (в сделках участвует около 15 тонн золота), работающей исключительно на базе open source решений: в качестве операционной системы используется CentOS Linux. код интерфейса для обмена золота построен на Java с использованием фреймворка SpringMVC, новостной сайт построен на базе Drupal, с организацией поиска на основе Apache Lucene кеширование контента - ehcache, трекинг ошибок - Bugzilla, тестирование - jUnit. мониторинг - Nagios, организация синхронизации данных на серверах горячей замены - Drbd, для балансировки нагрузки - Heartbeat; планирование работ - Quartz, управление версиями контента сайта - Subversion. в офисе используется OpenOffice; офисная телефония построена на базе дистрибутива TrixBox (Asterisk, Apache, MySQL, PHP)
