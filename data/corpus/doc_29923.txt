Третий номер приложения к PDF-журналу UserAndLINUX для системных администраторов

Представлен третий выпуск приложения к журналу UserAndLINUX - "Больше чем USER" (PDF, 6.6 Мб, 46 стр.). Приложение ориентировано на системных администраторов, программистов и просто продвинутых пользователей. На страницах журнала представлена подборка из ранее опубликованных в сети материалов и переводов. В выпуске: Настройка PPPoE с шейпингом трафика для небольшой сети (Debian Linux); Инструкция по серверу Ejabberd (Ubuntu Server 10.04 LTS); Lenovo – звук в наушниках (настройка ноутбука Lenovo g650 в Linux Ubuntu 10.04); Управление несколькими компьютерами одной клавиатурой и мышью; Laptop Mode Tools - утилита для уменьшения энергопотребления ноутбука; Vim: необходимый минимум знаний; Основы линукс. Часть 1 (введение в bash); Полезные команды терминала GRUB2; Access Control List - списки контроля доступа; Communigate: Часть 3. Настройка почтового домена, учетных записей пользователей; urpmi: краткое руководство пользователя;
