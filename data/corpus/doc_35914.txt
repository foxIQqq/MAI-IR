Релиз CrossOver 12.1 для Linux и Mac OS X

Компания CodeWeavers выпустила релиз пакета Crossover 12.1, основанного на коде Wine и предназначенного для выполнения программ и игр, написанных для платформы Windows. CodeWeavers входит в число ключевых участников проекта Wine, спонсирует его разработку и возвращает в проект около 98% всех новшеств, реализованных для своих коммерческих продуктов. В новой версии прекращена поставка по умолчанию компонента Wine-Mono с альтернативной реализацией .Net API, так как он значительно увеличивает размер пакета. Wine-Mono теперь следует устанавливать отдельно. Увеличена стабильность игры Quicken, улучшена поддержка Microsoft Office, исправлена проблема, мешающая добавлению оглавления в Word, решены проблемы с серверными соединениями в Outlook. Для платформы Linux устранён крах при запуске игры Guild Wars 2 в Ubuntu с оболочкой Unity.
