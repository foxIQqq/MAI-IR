Доступна облачная платформа Apache CloudStack 4.6

Состоялся релиз облачной платформы Apache CloudStack 4.6, которая позволяет автоматизировать развертывание, настройку и поддержание приватной, гибридной или публичной cloud-инфраструктуры (IaaS, инфраструктура как сервис). Платформа CloudStack передана Фонду Apache компанией Citrix, которая получила проект после поглощения компании Cloud.com. Установочные пакеты подготовлены для RHEL/CentOS и Ubuntu. CloudStack не зависит от типа гипервизора и позволяет использовать в одной cloud-инфраструктуре одновременно Xen (XenServer и Xen Cloud Platform), KVM, Oracle VM (VirtualBox) и VMware. Для управления пользовательской базой, хранилищем, вычислительными и сетевыми ресурсами предлагается интуитивно понятный web-интерфейс и специальный API. В простейшем случае облачная инфраструктура на базе CloudStack состоит из одного управляющего сервера и набора вычислительных узлов, на которых организуется выполнение гостевых ОС в режиме виртуализации. В более сложных системах поддерживается использование кластера из нескольких управляющих серверов и дополнительных балансировщиков нагрузки. При этом, инфраструктура может быть разбита на сегменты, каждый из которых функционирует в отдельном дата-центре. Из новшеств Apache CloudStack 4.6 можно отметить: Переход на новый процесс подготовки релизов, подразумевающий поддержание находящейся в разработке ветки на уровне качества, постоянно готовым к релизу (непрерывная интеграция и тестирование вместо разделения этапов наращивания функциональности и исправления ошибок); Поддержка использования SSH для отправки команд на виртуальный маршрутизатор. Возможность импорта, удаления и получения SSH-ключей через web-интерфейс на уровне отдельных учётных записей; Поддержка отправки статистики, накопленной через StatsCollector, на сервер Graphite для последующей визуализации; Возможность создания отказоустойчивых виртуальных маршрутизаторов для VPC, работающих в режиме active-pasive (перехват управления запасным узлом после сбоя первичного); Возможность загрузки в Cloudstack шаблонов или разделов через браузер; Подготовка Docker-образов для запуска модулей Apache CloudStack, которые могут применяться для тестирования, демонстрации или быстрого запуска рабочих конфигураций; Плагин для платформы виртуальных сетей NuageVsp (Nuage Networks Virtualized Services Platform); Поддержка Globo DNSAPI; Плагин для протокола SAML 2.0 (Security Assertion Markup Language); Управляемое хранилище для KVM; Улучшенный плагин для CloudByte Storage; Расширенные сетевые возможности при управлении невиртуализированным (baremetal) хостом Настройки для управления параметрами генерируемых паролей; Поддерживаемые гипервизоры и операционные системы: LXC-контейнеры в RHEL, Hyper-V 2012 R2, KVM (CentOS 6.2+, RHEL 6.2+), XenServer 6.1, 6.2 SP1 и 6.5, VMware 5.0 Update 3a, 5.1 Update 2a и 5.5 Update 2, Oracle VM 3.0+.
