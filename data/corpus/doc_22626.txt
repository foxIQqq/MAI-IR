Представлен Open Atrium, пакет для создания корпоративного интранет портала

Вышла первая публичная тестовая версия проекта Open Atrium, представляющего собой пакет для быстрого развертывания корпоративного интранет портала. В качестве основы используется система управления контентом Drupal. Пакет поддерживает ведение корпоративных блогов, микро-блоги для обмена информации между сотрудниками, календарь-планировщик, список заданий, wiki для совместной работы над документами, средства для групповой работы, сводную страницу с web-виджетами в стиле iGoogle.
