Доступны дополнительные редакции Sabayon Linux 5.5: E17, XFCE, LXDE, ServerBase, OpenVZ

Доступны пять легковесных редакций дистрибутива Sabayon Linux 5.5, основанного на Gentoo: E17 Spin (613 Мб) - сборка с десктоп-окружением на базе Enlightenment 17. XFCE Spin (658 Мб) - сборка с десктоп-окружением на базе XFCE 4.8. LXDE Spin (600 Мб) - сборка с десктоп-окружением на базе LXDE 0.5; SpinBase/OpenVZ (350 Мб) - сборка для быстрого развертывания гостевых систем, запущенных внутри контейнера OpenVZ. ServerBase (420 Мб) - сборка для создания серверных систем, включающая оптимизированное для серверных задач Linux-ядро 2.6.37; Пакетная база дистрибутива синхронизирована с коллекцией портажей Gentoo и полностью совместима с Portage 2.2 (EAPI3), пользователь имеет возможность установки программ как из исходных текстов, так и используя готовые бинарные пакеты, собранные для платформ x86 и x86-64. Сборки основаны на Linux-ядре 2.6.37 c патчами Tuxonice и AUFS, для настройки сети задействован NetworkManager 0.8, в состав входят NTFS-3G, web-браузер Midori 0.3.2, VirtualBox.
