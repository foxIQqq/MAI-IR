Выпуск Wine 1.5.27

Вышел очередной экспериментальный выпуск открытой реализации Win32 API - Wine 1.5.27. С момента версии 1.5.26 было закрыто 34 отчёта об ошибках. Из наиболее важных изменений отмечается: В драйвер для платформы Mac OS X добавлена поддержка системного лотка; Серия улучшений в C runtime; Добавлены увеличенные варианты пиктограмм для встроенных приложений; Устранены проблемы при запуске игр: SimCity 3000 Unlimited, SpongeBob SquarePants: Diner Dash 2, PopCap, Serious Sam HD TSE, Age of Empires II, Metacell, Call of Cthulhu: DCoTE, игры RPG Maker, Nimbus, Naver LINE, Icebreakers, Torchlight 2; Налажена работа приложений: SlingPlayer, QQ International 1.1, Sony PC Companion 2.10, Visual Studio 2008/2010 Express Edition, DVDFab HD Decrypter, Excel 2007, Brothers Keeper.
