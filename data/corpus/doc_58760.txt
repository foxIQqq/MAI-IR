Выпуск музыкального проигрывателя Audacious 4.3

Представлен релиз легковесного музыкального проигрывателя Audacious 4.3, ответвившегося в своё время от проекта Beep Media Player (BMP), являющегося форком классического плеера XMMS. Выпуск поставляется с двумя интерфейсами пользователя: на основе GTK и Qt. Сборки подготовлены для различных дистрибутивов Linux и для Windows. Основные новшества Audacious 4.3: Добавлена поддержка GTK3, которая пока не применяется по умолчанию (в GTK-сборках продолжает использоваться GTK2). Стабилизирована поддержка Qt 6, но по умолчанию продолжает применяться Qt 5. Добавлен плагин для вывода через мультимедийный сервер PipeWire. Добавлен плагин для декодирования формата Opus. Доведена до готовности поддержка сборочной системы Meson, которая протестирована на всех основных платформах (поддержка Autotools пока сохранена). В диалоге с информацией о композиции предоставлена возможность копирования в буфер обмена пути к файлу. Добавлена поддержка аудиопотоков FLAC в контейнере Ogg. Добавлена поддержка чтения встроенных в файл тегов с текстами песен. В интерфейсе поиска обеспечен учёт исполнителя альбома. В плагине SID добавлена поддержка нового формата БД c информацией о продолжительности песен. Добавлена поддержка тегов с данными об издателе и каталожном номере. В диалог экспортирования списка воспроизведения добавлен фильтр файлов.
