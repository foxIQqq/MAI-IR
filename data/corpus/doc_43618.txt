Выпуск клиента для мгновенного обмена сообщениями Pidgin 2.10.12

Спустя более года с момента прошлого выпуска представлено очередное корректирующее обновление клиента для мгновенного обмена сообщениями Pidgin 2.10.12. По сравнению с прошлым выпуском в списке изменений отражено всего несколько записей: Обработчик purple-url-handler адаптирован для работы с Python 3; Устранена ошибка, приводившая к удалению установленных статусов при запуске; Смайлик "крик" приведён в соответствие c используемой по умолчанию темой оформления; Обновлены включенные в сборку для Windows внешние компоненты. Удалена поддержка работы Tcl-плагинов в Windows-сборках; Библиотека libgadu, используемая для обеспечения работы протокола Gadu-Gadu, обновлена до версии 1.12.1. При этом в репозитории разработка ведётся достаточно активно - по сравнению с прошлым выпуском внесено несколько сотен исправлений, большая порция из которых связана с поддержкой чата Facebook.
