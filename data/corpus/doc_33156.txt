TryStack - проект по предоставлению бесплатных окружений для тестирования OpenStack

Участвующие в разработке облачной платформы OpenStack компании совместными усилиями ввели в строй сервис TryStack, в рамках которого любой желающий может получить доступ к уже настроенному изолированному облачному окружению на базе OpenStack. Сервис нацелен на тестирование работы приложений, поддерживающих управляющий API OpenStack. После регистрации по заявке в специальной группе Facebook, разработчик получает доступ в интерфейс TryStack Dashboard и возможность использования Nova API для автоматизации развёртывания своих окружений. Окружение предоставляются бесплатно, но каждое созданное окружение функционирует только 24 часа, после чего оно автоматически удаляется и нужно загружать новое окружение. Кроме того, для тестирования финансовых операций в системе фигурирует виртуальная валюта Stack Dollars, которая используется для эмуляции оплаты за использование различных услуг (после заведения аккаунта начисляется определённая сумма, при этом со временем баланс Stack Dollars регенерируется). В настоящее время обеспечивающая работу сервиса TryStack инфраструктура включает в сумме 156 процессорных ядер, 1 Тб ОЗУ и 60 Тб дискового пространства. Инфраструктура создана благодаря кооперации компаний Dell, NTT, Rackspace, Cisco, Equinix и HP. В частности, компания Dell предоставила проекту серверы, Equinix обеспечила их размещение в своём датацентре, HP планирует организовать инфраструктуру для тестирования облачных систем, включающих несколько территориально разделённых датацентров.
