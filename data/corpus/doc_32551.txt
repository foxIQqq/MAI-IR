Вышел DRAKON Editor 1.5 с генерацией кода

Вышла новая версия DRAKON Editor 1.5, свободного кросс-платформенного редактора диаграмм для визуального языка ДРАКОН, разработанного в рамках космической программы «Буран» и оперирующего созданием наглядных блок-схем. ДРАКОН позволяет переложить работу по созданию технологичных программ на плечи инженеров, которые не обладая должными навыками программирования, досконально владеют материалом и прекрасно разбираются в сущности процессов в прикладной области для которой создаётся программа. Код DRAKON Editor распространяется как общественное достояние (Public Domain), поддерживается работа в Linux, Mac OS X и Windows. В новой версии добавлена возможность генерации кода на языках C, C++, Python и Tcl. Процесс редактирования сделан более удобным: при перемещении линий теперь перемещаются все связанные с ними объекты. Добавлено множество улучшений пользовательского интерфейса, например, переход к диаграмме по имени, поиск всех ссылок, переход на заданное определение.
