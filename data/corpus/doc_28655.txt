Релиз Linux-дистрибутива ZevenOS 3.0 с интерфейсом в стиле BeOS

Вышел релиз Linux-дистрибутива ZevenOS 3.0, снабженного не требовательным к ресурсам BeOS-подобным интерфейсом. Новая версия отличается переходом на пакетную базу Ubuntu 10.10 (прошлые версии формировались на основе Debian). В качестве десктоп-окружения используется стилизованный под BeOS оконный менеджер Sawfish 1.7.1 с использованием компонентов Xfce 4.6.2. Для настройки приложений в рамках проекта создано ПО MAGi 2, в качестве панели для запуска программ задействован Zeebar, меню реализовано при помощи Deskbar 2. Размер iso-образа 700 Мб. В новой версии использовано Linux-ядро 2.6.35, вместо драйвера nv для видеокарт Nvidia теперь используется открытый драйвер Nouveau, значительно расширена поддержка беспроводных адаптеров. В Deskbar реализована система контекстных меню (например, через меню можно менять время и громкость) и обеспечена совместимость системы меню со спецификациями freedesktop.org, что позволяет редактировать меню внешними стандартными инструментами. Вместо видеоплеера Totem в дистрибутиве теперь используется Parole, для редактирования видео вместо avidemux в состав включен OpenShot, для прослушивания музыки вместо Rhythmbox интегрирован Deadbeef, для просмотра изображений вместо Gthumb задействован Shotwell. Как и раньше, для управления файлами используется Thunar, для работы в Web - Firefox, для просмотра почты - Claws-Mail, для выполнения офисных задач - Abiword и Gnumeric. Интересным новшеством новой версии также является возможность быстрой установки окружения ZevenOS в любом Ubuntu-совместимом дистрибутиве, через подключение специального репозитория и установку пакета zevenos-desktop.
