Обновление программ, разработанных в рамках проекта Openwall

Представлено несколько последних достижений проекта Openwall: Вышла адаптированная для Linux ядра 2.4.37.7 версия "-ow" (Openwall) патчей, повышающих безопасность системы; Обновлены установочные ISO-образы усовершенствованного в плане безопасности серверного Linux дистрибутива Owl. Судя по всему эту будет последняя сборка, основанная на Linux ядре серии 2.4.x, в настоящий момент ведется активная работа по переводу дистрибутива на ветку 2.6.x; Вышел релиз passwdqc 1.1.4, инструментария для проверки стойкости паролей; В рамки общественного достояния (public domain) переведено несколько полезных библиотек, среди которых: эффективная реализация кода для расчета MD5 и MD4 хэшей основанная на использовании алгоритма Blowfish версия функции crypt, подходящая для использования в многопоточных программах функции для объединения строк функции для парсинга MIME-сообщений.
