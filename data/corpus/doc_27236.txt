Вышла LXDE-редакция Linux дистрибутива Salix 13.1

Представлена дополнительная редакция Linux-дистрибутива Salix 13.1, базирующаяся на десктоп-окружении LXDE. Дистрибутив на 100% совместим со Slackware Linux 13.1 и исповедует подход "одно приложение на одну задачу". Salix развивается создателем дистрибутива Zenwalk Linux, покинувшим проект в мае прошлого года в результате конфликта с другими разработчиками, отстаивавшими политику максимальной похожести на Slackware. Размер загрузочного iso-образа составляет 464 Мб. Поддерживается три режима установки: Полная - десктоп окружение на основе LXDE, web-браузер Midori, почтовый клиент Claws-mail, клиент для мгновенного обмена сообщениями Pidgin, BitTorrent-клиент Transmission, текстовый редактор Abiword, табличный процессор Gnumeric, медиаплеер Whaawmp, ПО для ведения архива музыки Exaile, программа для просмотра фотографий Viewnior, ПО для записи CD/DVD Brasero, пакетный менеджер gslapt. Основная - LXDE, файловый менеджер PCManFM, web-браузер Midori и пакетный менеджер gslapt; Базовая - только минимальный набор консольных утилит, подходит для создания серверов.
