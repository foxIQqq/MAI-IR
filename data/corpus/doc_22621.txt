Компания Google выпустила Google Web Toolkit 1.7 и набор инструментов для отладки

Для пакета Google Web Toolkit выпущен дополнительный набор инструментов для отладки приложений. Google Web Toolkit представляет собой набор средств для разработки динамических web-приложений, которые изначально могут быть написаны на языке Java, а затем при помощи специального транслятора преобразованы в представление на базе JavaScript и HTML. Дополнение: Вышел релиз Google Web Toolkit 1.7, в котором проведена работа по исправлению ошибок, добавлен код для обеспечения полной совместимости с брауерами IE 8, Firefox 3.5 и Safari 4.
