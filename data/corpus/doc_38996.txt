Вышел Google Web Toolkit 2.6.0

Объявлено о выходе Google Web Toolkit 2.6 (GWT), набора средств для разработки динамических web-приложений, которые изначально могут быть написаны на языке Java, а затем при помощи специального транслятора преобразованы в представление на базе JavaScript и HTML. Новый выпуск примечателен обеспечением поддержки Java 7 и использованием данной ветки по умолчанию. Кроме того, в GWT 2.6 прекращена поддержка Internet Explorer 6/7 и режима отладки WT Development Mode.
