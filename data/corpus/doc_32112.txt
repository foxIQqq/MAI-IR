Релиз Linux дистрибутива ZevenOS Neptune 2.0 с интерфейсом в стиле BeOS

Представлен релиз Linux дистрибутива ZevenOS 2.0, построенного на пакетной базе Debian и снабженного интерфейсом пользователя, стилизованным под BeOS. В качестве десктоп-окружения используется релиз KDE 4.7.1, для маломощных систем предусмотрена возможность использования LXDE. Для настройки приложений в рамках проекта создано ПО MAGi 2, меню и панель реализованы при помощи Deskbar, для кодирования видео подготовлено специальное приложение Encode. Размер загрузочного iso-образа для полной версии составляет 1.3 Гб, сокращённой - 660 Мб. Дистрибутив базируется на ядрe Linux 3.0.4. В состав сокращённой версии входят такие приложения, как файловый менеджер PCManFM 0.9.8, аудиоплеер DeaDBeeF 0.5.1, медиаплеер Gnome-Mplayer 1.0.4 на базе Mplayer2, текстовый процессор Abiword 2.8.6, браузер Iceweasel 7 (Firefox) и почтовый клиент Iceweasel 5 (Thunderbird). Полная редакция включает KDE 4.7.1, Chromium 14, Iceweasel 5, VLC 1.1.11, Amarok 2.4.3, LibreOffice 3.4.3, Eclipse, Wireshark, Aircrack-NG, Kismon, Audacity, Kdenlive. Для управления коллекцией фотографий используется Gwenview. Обе редакции поставляются с полным набором мультимедиа кодеков. Для быстрой установки проприетарных драйверов, таких как видеодрайверы ATI и NVIDIA, подготовлено специальное приложение ZevenOS-Hardwaremanager. Отдельно отмечается наличие специального инструментария remaster-kit для изменения базовой начинки и быстрого создания производных Live-систем, оптимизированных для установки на USB-накопители. В качестве примеров подготовлено несколько дополнительных сборок с графическими окружениями на базе XFCE 4.8, Enlightenment E17 и GNOME 3. Созданная конфигурация может быть экспортирована в виде файла и использована другими пользователями в качестве шаблона для создания своих систем.
