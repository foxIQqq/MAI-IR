Доступна новая версия клиента Steam для Linux

Компания Valve выпустила обновление клиента Steam для платформы Linux. Обновление выпущено одновременно с версией для Windows и содержит 15 исправлений не зависящих от используемой платформы и 4 изменения, специфичные для Linux. В числе общих изменений улучшение системы навигации, добавление новой страницы для оценки наиболее популярного контента, улучшение поддержки расширенных наборов символов, расширение логики выявления работы через прокси, возможность автоматического восстановления для наиболее важных файлов из состава Steam без необходимости выхода в Сеть. Из специфичных для Linux изменений отмечается поддержка SDL 2.0 ABI, решение проблемы с потерей фокуса для окон и показом без необходимости диалога настройки нового драйвера в режиме Big Picture Mode, устранение подвисания при отправке отчёта об ошибке.
