Zend, Microsoft и IBM ведут совместную работу по созданию Cloud API для PHP

В кооперации с IBM, Microsoft, Nirvanix, Rackspace и GoGrid компания Zend Technologies начинает открытый проект, цель которого - предоставить разработчикам интерфейс программирования для реализации "облачных" служб, запущенных на различных платформах. Простой API для служб "облачных" вычислений (Simple Cloud API) - это способ запустить обычные приложения в "облачном" окружении и разрабатывать "облачные" приложения без необходимости адаптации кода для получения доступа в подобные окружения. По имеющимся сведениям, приложения будут запускаться на большинстве облачных платформ, это, к примеру, IBM Cloud Computing, Azure от Microsoft, Nirvanix Storage Delivery Network, Rackspace и GoGrid. Производители этих платформ числятся в соучредителях проекта. Также в число поддерживаемых платформ входит Amazon Web Services (AWS), хотя этот лидер рынка не принял участие в разработке. Поначалу API будет поддерживать только PHP, но проводимые переговоры с членами сообществ других языков, например, Java, Perl и Python, являются признаком будущего сотрудничества. Первые начальные реализации предлагают поддерживаемым платформам интерфейс хранения данных, документную базу данных и простые службы очередей. В будущем в API будет добавлена поддержка системы проверки подлинности. PHP-реализации будут доступны в составе будущих версий свободного фреймворка Zend под именем "Zend Cloud".
