Релиз VMWare Workstation 7.1 и VMWare Player 3.1

Вышла версия 7.1 программы виртуализации для десктопа VMware Workstation. Основные изменения этой версии: Добавлена официальная поддержка хостовых и гостевых ОС: Ubuntu 8.04.4, Ubuntu 10.04, OpenSUSE 11.2, Red Hat Enterprise Linux 5.5. Добавлена официальная поддержка гостевых ОС: Fedora 12, Debian 5.0.4, Mandriva 2009.1. Для гостевых ОС Windows 7 и Windows Vista теперь поддерживается ускорение OpenGL 2.1. Значительно улучшена производительность графической подсистемы для всех гостевых ОС, особенно заметная в плавности проигрывания видео и лёгкости работы Aero в Windows Vista/7. Также была значительно ускорена работа DirectX в гостевых ОС. VMWare Workstation теперь может автоматически себя обновлять до более новой версии. Прямой запуск - теперь можно запускать приложение из гостевой ОС, используя ярлык в хостовой ОС, даже в том случае, если гость выключен (в этом случае VMWare автоматически запустит гостевую ОС). Автологин - поддерживается автоматический ввод имени пользователя и пароля для гостевой ОС. Теперь полностью поддерживается формат виртуальных машин OVF 1.1. Гостевые ОС могут использовать до 8 виртуальных процессорных ядер. Увеличен максимальный размер диска для гостевой ОС с 950GB до 2TB. Для шифрования гостевой ОС теперь используются новые инструкции (AESNI) процессоров нового поколения Intel (Arrendale и Clarkdale), если они поддерживаются. Улучшено графическое представление объёма RAM, выделяемого для гостевой ОС. Добавлена возможность участия в добровольной программе по улучшению работы и обмену данными о работе гостевых ОС и самой виртуальной машины. После регистрации на сайте, VMWare Workstation можно бесплатно протестировать в течение 30 дней. VMWare Player поддерживает создание виртуальных машин и является совершенно бесплатным продуктом.
