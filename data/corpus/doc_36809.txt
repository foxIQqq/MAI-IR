Новая версия системы управления контентом Joomla 3.1

Представлен новый стабильный выпуск Joomla 3.1, универсальной системы управления web-контентом, которая подходит для создания больших и маленьких корпоративных сайтов, интранет порталов, online-магазинов, сайтов сообществ и персональных страниц. Из особенностей Joomla можно отметить: гибкие инструменты по управлению пользователями, интерфейс для управления медиа-файлами, поддержка создания многоязычных вариантов страниц, система управления рекламными кампаниями, адресная книга пользователей, голосования, встроенный поиск, функции категоризации ссылок и учета кликов, WYSIWYG-редактор, система шаблонов, поддержка меню, управление новостными потоками, XML-RPC API для интеграции с другими системами, поддержка кэширования страниц и большой набор готовых дополнений. Ключевым новшеством Joomla 3.1 является поддержка привязки тегов к любому контенту во всех базовых компонентах CMS, а также в некоторых плагинах. Пометка тегами может быть использована для организации привязки контента одновременно к нескольким категориям и организации гибких выборок информации. Кроме того, отмечается включение в штатную поставку плагинов для интеграции с MediaWiki и OpenStreetMap, а также удаление плагина GeSHi для интеграции с одноимённой библиотеки для подсветки синтаксиса (плагин оказался востребован лишь небольшим числом пользователей).
