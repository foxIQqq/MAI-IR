Выпуск CyanogenMod 13.0, независимой сборки мобильной платформы Android

Представлен релиз мобильной платформы CyanogenMod 13.0r1, в котором осуществлён переход на использование платформы Android 6.0. Сборки CyanogenMod 13.0 получили статус релиза пока не для всех устройств, а только для тех, работа новой версии на которых не вызывает нареканий. Основные улучшения, специфичные для CyanogenMod (список улучшений в Android 6.0 можно посмотреть здесь): Исключение из поставки системы WhisperPush, позволявшей принимать и отправлять сообщения SMS и MMS, защищённые при помощи протокола TextSecure. Пользователям рекомендуется перейти на основную реализацию TextSecure, воспользовавшись открытым приложением Signal. Из-за несовместимости с Android Gatekeeper прекращена поддержка функции быстрой разблокировки; Переход на использование штатного приложения Android для обмена сообщениями SMS/MMS; Новое приложение "Snap" для работы с камерой, позволяющее использовать расширенные возможности устройств на базе чипов Qualcomm Snapdragon. Для устройств с обычными камерами продолжена поставка программы Camera2; В системе защищённых приложений появилась возможность подтверждения доступа к контенту по отпечатку пальца, добавлен новый интерфейс для просмотра защищённого контента, улучшены средства защиты (защищённое приложение не может запускаться через Play Store) Отказ от режима расширенных настроек (About phone/Advanced mode), все настройки теперь показываются в обычном режиме; Из конфигуратора удалён блок быстрых настроек (Notification Drawer/Quick Settings), управлять раскладкой элементов теперь можно прямо из панели быстрой настройки; Добавлены дополнительные средства для управлением пиктограммами элементов строки состояния; Добавлен демонстрационный режим, который можно активировать через три касания в области About phone/Build date; В настойки памяти добавлен новый экран для оценки потребления памяти за указанный период времени; Работа тихого режима и функция пробуждения после двойного касания переведены на штатные реализации платформы Android; Задействование в Android новой модели ограничения доступа потребовало переработки Privacy guard. По умолчанию при выборе нормального уровня безопасности для пользователей не подключенных к сервисам Google используется проверка полномочий через пакетный менеджер от CyanogenMod, а для пользователей Google - через интерфейс управления пакетами Google. Вернуться к ранее предлагаемому режиму тонкой настройки полномочий можно через меню Privacy/Privacy Guard; Пакет дополнительных приложений Cyanogen OS пока не доступен, но будет подготовлен в течение недели;
