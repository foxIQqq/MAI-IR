Финальная версия Javascript библиотеки Yahoo! User Interface 2.8

Вышла финальная версия библиотеки Yahoo! User Interface (YUI) 2.8, предназначенное для создания пользовательского интерфейса в веб-приложениях. В новой версии добавлены следующие компоненты: Storage Utility - сохранение данных на клиентском компьютере. Поддерживаются три механизма хранения: HTML 5, Google Gears, Flash. ProgressBar Control - индикатор выполнения SWFStore Utility - хранение пар ключ/значение с помощью Flash, сжатие на лету данных ключ/значение (основа для компонента Storage Utility). SWF Utility - компонент для встраивания Flash-приложений в веб-страницы. В дальнейшем планируется, что этот компонент объединит в себе все остальные гибридные (Flash/Javascript) компоненты библиотеки YUI, такие как Charts и SWFStore. Среди изменений стоит также отметить: Поддержка междоменных (cross-domain) запросов в компоненте Connection Manager на основе технологии Flash. Поддержка многоосных графиков Кроме того, недавно появился серверный вариант утилиты для определения зависимостей между компонентами, их агрегирования и загрузки - YUI PHP Loader Utility. Утилита предназначена для языка PHP и является аналогом клиентского компонента YUI Loader Utility.
