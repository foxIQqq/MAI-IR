Выпуск графического редактора GIMP 2.10.30

Опубликован выпуск графического редактора GIMP 2.10.30. Для установки доступны пакеты в формате flatpak (пакет snap пока не готов). Выпуск в основном включает исправления ошибок. Все усилия по наращиванию функциональности сосредоточены на подготовке ветки GIMP 3, которая находится на стадии тестирования предварительных выпусков. Из изменений в GIMP 2.10.30 можно отметить: Улучшена поддержка форматов файлов AVIF, HEIF, PSD, DDS, RGBE и PBM. Например, для экспорта AVIF задействован кодировщик от проекта AOM, а в формате PSD добавлена поддержка дополнительных вариантов содержимого (маски слоёв неверной размерности, CMYK без прозрачности или без слоёв, объединённые изображения с 16-битами на цветовой канал RGBA, имеющие непрозрачный альфа-канал). Для Linux и систем, в которых для доступа к ресурсам вне контейнера применяются порталы Freedesktop, обеспечена работа инструмента определения цвета на экране (color picker) через обращение к API Freedesktop. Кроме того, инструмент для создания скриншотов теперь рассматривает API Freedesktop как приоритетный и при его наличии использует вместо API, специфичных для KDE и GNOME (в KDE 5.20 и GNOME Shell 41 данные API были ограничены из соображений безопасности). Из ветки 2.99.8 перенесено изменение для корректного отображения границы выделенной области в выпусках macOS начиная с "Big Sur", в которых ранее не показывались контуры на холсте. На платформе Windows осуществлён переход на использование API WcsGetDefaultColorProfile() вместо функции GetICMProfile(), корректная работа которой была нарушена в Windows 11 (наблюдается сбой при попытке получения профиля монитора). В основной состав и плагины внесены улучшения, связанные с поддержкой метаданных. В инструменте работы с текстом прекращено применение системных настроек субпиксельного рендеринга шрифтов, так как данный вид отрисовки шрифтов предназначен для улучшения отображения GUI на жидкокристаллических мониторах и не рассчитан на использование в изображениях, которые могут масштабироваться, выводиться на печать и показываться на разных типах экранов.
