Релиз phpPgAdmin 5.0, web-интерфейса для управления PostgreSQL

Представлен релиз phpPgAdmin 5.0, написанного на языке PHP web-интерфейса для организации управления СУБД PostgreSQL. Особенности новой версии: Обеспечена поддержка PostgreSQL 8.4 и 9.0; Для PostgreSQL 8.4 и более новых версий теперь можно использовать разные локали для разных баз одного сервера; Поддержка экспорта на уровне схемы данных; Добавлена функция изменения владельца схемы данных, переименования ролей и пользователей, установки нового пароля; Возможность добавления комментариев для функций; Улучшен пользовательский интерфейс; Значительно улучшении возможности полнотекстового поиска; Добавлен график для оценки оптимизации приложений; Поддержка задания привилегий на уровне столбцов; Реализован интерфейс по администрированию отдельных таблиц; Переработан код автодополнения SQL-запросов; Добавлен режим просмотра внешних ключей.
