Релиз графического редактора Krita 2.9

После восьми месяцев разработки увидел свет релиз растрового графического редактора для художников и иллюстраторов Krita 2.9.0, который поддерживает многослойную обработку изображений, предоставляет средства для работы с различными цветовыми моделями и обладает большим набором средств для цифровой живописи, создания скетчей и формирования текстур. Программа распространяется как в форме самодостаточного приложения, так и в составе развиваемого проектом KDE офисного пакета Calligra. Проект курируют фонд Krita Foundation и организация KO GmbH (Krita Studio). Благодаря успешно проведённой на Kickstarter кампании совместного финансирования разработки, над новым выпуском удалось обеспечить работу двух разработчиков в режиме полного рабочего дня. В итоге, по числу новшеств и улучшений Krita 2.9 побил все рекорды и признан самым значительным выпуском в истории проекта. Основные новшества: Реализация начальной версии для платформы OS X; Возможность работы сразу с несколькими изображениями. По умолчанию для переключения между открытыми изображениями предлагается система вкладок. Дополнительно предлагается альтернативный режим Subwindow, основанный на мозаичной или каскадной группировке редактируемых изображений. Кроме того, реализована возможность одновременного показа нескольких представлений одного изображения, например, можно одновременно открыть варианты с разными уровнями масштабирования; Улучшение инструментов трансформации и деформации изображения. Представлены новые режимы и маски трансформации, существенно повышающие гибкость редактирования изображений. Добавлен новый инструмент для изменения перспективы слоя (Perspective); Добавлен инструмент для пластичной деформации по обрамляющей объект клетке (Cage); Переработан интерфейс инструмента трансформации, увеличено качество преобразований. Обеспечена возможность недеструктивного применения масок трансформации; Улучшены средства рисования HDR-изображений. Добавлена возможность быстрого повышения или снижения экспозиции, реализована панель для переключения между различными типами отображения цветов. Улучшен интерфейс расширенного выбора цветов, в котором обеспечена возможность работы в цветовых моделях HSI и HSY, упрощающих сохранение уровня яркости при изменении тона и насыщенности. Добавлена составная экспериментальная панель Color Slider, через которую можно добиться произвольной комбинации ползунков HSV/HSL/HSI/HSY; Улучшения, связанные с рисованием. Увеличено качество работы смешанной кисти (mix brush), при работе которого теперь не отмечается артефактов. Для кистей добавлена возможность создания сверхгладких тонких мазков, хорошо подходящих для прорисовки деталей. Улучшен вывод всплывающей палитры кистей, для быстрого перехода к искомым кистям можно использовать теги. Добавлена новая обзорная панель (Overview Docker), позволяющая быстро перемещаться по холсту и выполнить операции масштабирования. Появилась возможность определения иного размера ластика, отличающегося от размера текущей кисти. Обеспечена возможность изменения радиуса пипетки для получения более широкого диапазона цветов из изображения. Возможность определения временных преднастроек кистей, позволяющих сохранить небольшие изменения для текущего сеанса без сохранения данных изменений в оригинальный набор преднастроек. Поддержка закрепления определённых настроек кисти - после установки специального флажка, применение новой преднастройки не повлияет на помеченную флажком настройку, значение которой сохранится; Расширенные возможности по выделению отдельных областей изображения. Поддержка отрисовки или трансформации непосредственно по глобальной маске. Возможность выделения прозрачных каналов в отдельные маски прозрачности. Поддержка сохранения или чтения подготовленных масок. Добавлены новые помощники для рисования параллельных, бесконечных и сходящихся линий, упрощающие формирование перспективы. Внесены улучшения в инструменты заливки, построения сетки, вывода и стабилизации линий; Расширены возможность G’MIC фильтров раскраски, добавлена поддержка предпросмотра. Через инструмент Split Layer реализована возможность выделения областей разного цвета в отдельные слои. Добавлены новые фильтры Posterize для сокращения количества цветов для каждого канала, Index Colors для упрощения попиксельного рисования; Увеличено разрешение векторных объектов - векторные пиксели теперь фактически являются обычными пикселями, без привязки к 72 DPI; Добавлены новые возможности для создания 3D-текстур и изображений. Добавлена поддержка задания случайного смещения при наложении текстуры. Реализована поддержка сохранения сведённых изображений. Улучшена совместимость с Photoshop, в том числе добавлена возможность загрузки некоторых файлов CS6 PSD. Поддерживаются все, кроме четырёх, режимов смешивания PSD, возможна работа с группами слоёв, улучшена поддержка 16-разрядных многослойных файлов. Возможен экспорт в PSD векторных слоёв. В фильтр OpenEXR добавлена поддержка чтения и записи одноканальных (в оттенках серого) файлов EXR. Добавлена возможность импорта RAW-файлов; Добавлена поддержка записи 16-разрядных изображений в файлы tiff, jpeg и ppm.
