Релиз web-браузера Opera 11

Вышел релиз web-браузера Opera 11, доступный для платформ Windows, MacOS X, Linux и FreeBSD. Из ключевых улучшений новой версии можно отметить улучшенный JavaScript-движок, поддержку дополнений и возможность группировки вкладок на экране. Основные новшества: Поддержка расширений. Для написания расширений используются стандартные web-технологии: HTML5, CSS, JavaScript и W3C Widget API. Расширения можно загрузить и установить из специально подготовленного каталога. К сожалению, расширения для Opera 11 построены на базе технологии ранее доступных пользовательских JavaScript-виджетов, которые имеют гораздо меньше возможностей, чем, например, расширения на основе XUL в Mozilla Firefox. С другой стороны, задействование стандартных web-технологий позволяет свести к минимум усилия при портировании уже существующих расширений, разработанных для Google Chrome. Из уже появившихся для Opera расширений можно отметить: NoAds - расширение для блокирования рекламы; LastPass - менеджер паролей и система для организации автоматического заполнения форм FastestTube - дополнение для загрузки на локальный диск роликов YouTube; Image preview - отображает всплывающие эскизы по ссылке на изображение Translate - система перевода содержимого web-страниц на другой язык Улучшение производительности JavaScript-движка. Обновленная версия JavaScript-движка демонстрирует заметный рост производительности. Кроме того, в три раза сокращен размер инсталляционного файла; Группировка вкладок. Появилась возможность группировки вкладок (tabs) путём переноса одной вкладки на другую. Демонстрационный видеоролик можно посмотреть здесь. Безопасная адресная строка. Opera 11 последовала примеру Google Chrome и отказалась от привычного отображения протокола доступа к веб сайтам - теперь вместо "http:/"/ и "https://" Opera показывает выразительные картинки. Управление через жесты мышью. Значительно упрощена работа с жестами мышью, благодаря добавлению удобного и понятного интерфейса для создания ассоциаций условных движений мышью с командами. Улучшена поддержка новейших веб-технологий. В Opera 11 добавлена поддержка серверных событий и веб-сокетов, определенных в стандарте HTML 5. К сожалению по умолчанию из-за вскрывшихся проблем безопасности поддержка протокола WebSockets заблокирована. В тесте html5test.com новый веб-браузер набирает 179 баллов, уступая по этому параметру Google Chrome для Linux, который набирает 231 балл. Улучшен механизм автообновления. Теперь веб-браузер более дотошно следит, чтобы расширения и приложения Opera Unite обновлялись своевременно. Загрузка плагинов по требованию. Появилась возможность загрузки плагинов, таких как Adobe Flash, по требованию. Чтобы использовать эту возможность, её нужно включить в опциях веб браузера в Preferences / Advanced / "Enable plug-ins only on demand" (загружать плагины по требованию). По заявлению разработчиков, поддержка загрузки плагинов на лету, позволяет до 30% увеличить скорость запуска браузера; Улучшена работа встроенного почтового клиента. Добавлена новая почтовая панель, которая позволяет контролировать порядок отображения учётных записей и писем. Появилась возможность создавать папки для организации почты и новостных списков подписки (RSS). Переработана панель для избранного.
