Релиз DNS сервера Unbound 1.3.

Компания NLnet Labs выпустила очередной релиз кеширующего DNS сервера Unbound. Unbound распространяется под лицензией BSD, имеет модульную структуру и поддерживает работу резолвера в рекурсивном и кэширующем режиме, также имеется возможность проверки валидности DNSSEC сигнатур, поддержка инфраструктуры DLV, асинхронных запросов и библиотеки для интеграции кода резолвера в пользовательские приложения (stub-resolvers). Формат файла конфигурации не похож на Bind, но прост и понятен для восприятия. Основные изменения и новшества релиза: Официальная поддержка платформы Windows (XP/Vista). Интеграция libunbound с python - позволяет вызывать функции unbound из кода программ написанных на python, а так же писать на python дополнительные плагины для unbound. Экспериментальная поддержка алгоритмов rsasha256 и rsasha512. Новая опция log-time-ascii для изменения формата записи времени в лог. Новая функциональность для unbound-control - позволяет на лету менять адреса для форвардинга запросов. Множество мелких изменений и исправлений ошибок.
