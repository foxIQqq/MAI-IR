Qt Software прекращает развитие проекта Qt Jambi

Подразделение Qt Software компании Nokia объявило, что релиз Qt Jambi 4.5 будет последним, после чего разработка пакета силами компании будет прекращена, а поддержка пользователей будет осуществляться до марта 2010 года. Высвободившиеся ресурсы будут направлены на развитии средств для разработки кроссплатформенных приложений в тулките Qt и на совершенствование фреймворка для построения интерфейса пользователя. Qt Jambi представляет собой Java интерфейс ко всем функциям тулкита Qt, который позволяет разработчикам напрямую использовать в своих Java программах API библиотеки Qt, включая поддержку WebKit и Phonon. Код Qt Jambi будет выпущен под лицензией LGPL и полностью отдан в руки независимого сообщества разработчиков, т.е. будущее проекта целиком будет зависеть от комьюнити.
