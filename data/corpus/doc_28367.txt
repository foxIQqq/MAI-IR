Вышла первая альфа-версия web-браузера Opera 11

Анонсирован выход первой альфа версии веб-браузера Opera 11, который содержит следующие заметные нововведения: Появилась поддержка расширений. Для написания расширений не нужно изучать достаточно сложный фреймворк XUL, вместо этого разработчики Opera, по примеру Goolge Chrome, реализовали расширения с помощью стандартных web-технологий: HTML5, CSS и JavaScript. Расширения можно загрузить и установить из специально подготовленного каталога. Улучшена производительность. Разработчики Opera в очередной раз представили новую версию собственного JavaScript-движка со значительно возросшей производительностью. Кроме этого, в Opera 11 появилась возможность загрузки плагинов, таких как Adobe Flash, по требованию. Чтобы использовать эту возможность, её нужно включить в опциях веб браузера в Preferences → Advanced → “Enable plug-ins only on demand” (загружать плагины по требованию); Улучшена поддержка новейших веб технологий. В Opera 11 добавлена поддержка серверных событий и веб-сокетов стандарта HTML 5. В тесте html5test.com новый веб-браузер набирает 179 баллов, уступая по этому параметру Google Chrome, который набирает 231 балл; Расширены дополнительные возможности: новая почтовая панель позволяет контролировать порядок отображения учётных записей и писем. Появилась возможность создавать папки для организации почты и новостных списков подписки (RSS). Переработана панель для избранного.
