Выпуск проприетарного драйвера AMD Radeon Software Crimson Edition Linux 15.12

Компания AMD опубликовала новую версию проприетарного драйвера AMD Catalyst, который выпущен под новым именем AMD Radeon Software Crimson Edition Linux 15.12 (177 Мб). Драйвер поддерживает X-Server 1.17, ядро Linux 3.19 и glibc 2.3. Из официально поддерживаемых дистрибутивов названы Red Hat Enterprise Linux 7.2-6.5, Ubuntu 15.10, 12.04.4, 14.04.2, 15.04, SUSE Linux Enterprise 11 SP3, 12 и openSUSE 13.1. Из изменений отмечается только решение проблем со сборкой пакетов для Ubuntu 15.10.
