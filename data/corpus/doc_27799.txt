Релиз тестового комплекта Phoronix Test Suite 2.8

Вышел релиз пакета для измерения производительности системы - Phoronix Test Suite 2.8. Пакет поддерживает работу в Linux, OpenSolaris, системах семейства BSD, Windows и Mac OS, распространяется под лицензией GPLv3 и призван обеспечить потребность простых пользователей в удобном инструменте для измерения производительности системы. По сравнению с прошлой версией в пакет внесено около 70 изменений. В состав версии 2.8 входит 134 тестовых профиля и 57 тестовых наборов. В новой версии добавлена поддержка проведения тестов на платформе Windows x64, расширены возможности OpenCL-тестов, обновлены многие тестовые профили, добавлен код для управления внешними зависимостями. Спецификация проведения каждого теста оформлена в виде XML файла, в качестве тестов используются внешние приложения, например, измеряется FPS в различных играх, оценивается время сборки Linux ядра, скорость кодирования mencoder, время сжатия bzip2 и т.д. Во время проведения тестов осуществляется контроль за изменением состояния оборудования (скорость вращения кулера, температура, напряжение). Итоги тестирования можно опубликовать в каталоге PTS Global, представляющем собой централизованную базу данных с результатами, полученными от различных пользователей.
