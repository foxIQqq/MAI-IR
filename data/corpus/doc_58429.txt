Выпуск дистрибутива Nitrux 2.6 с рабочим столом NX Desktop

Опубликован выпуск дистрибутива Nitrux 2.6.0, построенного на пакетной базе Debian, технологиях KDE и системе инициализации OpenRC. Проект предлагает собственный рабочий стол NX Desktop, который представляет собой надстройку над пользовательским окружением KDE Plasma. На основе библиотеки Maui для дистрибутива развивается набор типовых пользовательских приложений, которые можно использовать как на настольных системах, так и на мобильных устройствах. Для установки дополнительных приложений продвигается система самодостаточных пакетов AppImages. Размер полного загрузочного образа составляет 3 ГБ. Наработки проекта распространяются под свободными лицензиями. Рабочий стол NX Desktop предлагает иное стилевое оформление, собственную реализацию системного лотка, центра вывода уведомлений и различных плазмоидов, таких как конфигуратор сетевых соединений и мультимедийный апплет для регулирования громкости и управления воспроизведением мультимедийного контента. Из приложений, созданных при помощи фреймворка MauiKit, можно отметить файловый менеджер Index (можно использовать и Dolphin), текстовый редактор Note, эмулятор терминала Station, музыкальный проигрыватель VVave, видеопроигрыватель Clip, центр управления приложениями NX Software Center и просмотрщик изображений Pix. Основные новшества Nitrux 2.6: По умолчанию добавлена поддержка мультимедийного сервера PipeWire и протокола Wayland. В менеджер входа SDDM в качестве опции добавлена возможность запуска сеанса KDE на базе Wayland. Из основного iso-образа удалены dpkg, APT и PackageKit (но оставлены в live-сборках). В качестве приоритетных методов установки программ заявлены форматы AppImage и Flatpak. При этом при помощи инструментария Distrobox предоставлена возможность использования любых пакетных менеджеров и дистрибутивов внутри Nitrux. Distrobox позволяет установить и запустить в контейнере любой дистрибутив Linux и обеспечить его интеграцию с основной системой. Компоненты рабочего стола NX Desktop обновлены до версий KDE Plasma 5.26.4, KDE Frameworks 5.101.0 и KDE Gear (KDE Applications) 22.12. Обновлены версии программ, включая Mesa 23.0-git и Firefox 108.0.1. По умолчанию задействовано ядро Linux 6.1 с патчами Liquorix. По умолчанию включена поддержка каталога пакетов Flathub. В состав добавлен пакет Samba. Корневой раздел защищён от изменений при помощи overlayroot для исключения конфликтов во время обновления и проблем при установке пакетов из сторонних репозиториев. Изменены настройки для инсталлятора Calamares, изменено оформление режима очистки диска и отображения информации о разделах. Добавлен эквалайзер Pulseaudio Equalizer. Обновлён обзорный экран в Plasma System Monitor, добавлен вывод списка процессов вместо списка приложений. Добавлены пакеты с отладочными символами для пакетов KWin и Latte Dock.
