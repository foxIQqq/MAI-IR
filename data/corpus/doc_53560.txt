Релиз видеоредактора Kdenlive 20.08

Разработчики проекта KDE опубликовали релиз видеоредактора Kdenlive 20.08, который позиционируется для полупрофессионального использования, поддерживает работу с видеозаписями в форматах DV, HDV и AVCHD, и предоставляет все базовые операции по редактированию видео, например, позволяет используя шкалу времени произвольно смешивать видео, звук и изображения, а также применять многочисленные эффекты. При работе программы используются такие внешние компоненты, как FFmpeg, фреймворк MLT и система оформления эффектов Frei0r. Для установки подготовлен самодостаточный пакет в формате AppImage. В новом выпуске: Предложено несколько рабочих пространств с разными вариантами компоновки элементов интерфейса для каждой стадии производства видео: Журналирование (Logging) - для оценки отснятого контента и добавления меток для фрагментов; Редактирование (Editing) - для компоновки ролика с использованием временной шкалы. Звук (Audio) - для микширования и корректировки звука. Эффекты (Effects) - для добавления эффектов. Цвет (Color) - для подгонки и коррекции цветов. Представлена начальная реализация нового рабочего процесса для обработки звука. В текущей версии добавлена поддержка одновременной работы с несколькими звуковыми потоками. В следующих версиях ожидается появление средств для маршрутизации звуковых потоков и маппинга звуковых каналов. Модернизирован интерфейс микширования звуков. В панели наложения эффектов и в интерфейсе отслеживания клипа реализованы масштабируемые полосы управления (zoom bar), упрощающие корректировку ключевых кадров и навигацию по клипу. В настройках представлен новый интерфейс для управления кэшированием, позволяющий контролировать размер файлов с кэшем и проксируемыми данными, а также файлов с резервными копиями. Имеется возможность задания времени жизни элементов для автоматической очистки старых данных в кэше. Добавлена возможность назначения меток, привязанных к определённой позиции в клипе. Добавлена настройка для размещения панели мониторинга звука под видео, не перекрывая его. Добавлена кнопка для сохранения копии проекта. В диалог выбора скорости добавлена настройка для корректировки размера клипа. Добавлена опция для сохранения титров и их добавления в проект одним действием. Добавлена возможность изменения цвета эскизов звуковой волны. Значительно переработан файл проекта, решены проблемы с конфликтом десятичного разделителя (запятая или точка), который был причиной многих крахов. Ценой изменения стало нарушение обратной совместимости файлов проекта Kdenlive 20.08 (.kdenlive) с прошлыми выпусками. Повышена производительность генерации миниатюр для звуковых файлов и воспроизведения серий JPEG-изображений.
