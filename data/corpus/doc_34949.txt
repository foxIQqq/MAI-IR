Вышла новая версия Bittorrent клиента и сервера Vuze 4.7.2

Доступен релиз многоплатформенного торрент клиента/сервера Vuze 4.7.2, разработка которого ранее велась под именем Azureus. Код программы написан на языке Java и распространяется под лицензией GPL. Графический интерфейс построен с использованием тулкита SWT, базирующегося на Gtk+ в Linux. Vuze является одним из наиболее богатых по возможностям торрент-приложений, поддерживает удаленное управление, снабжен интегрированными средствами поиска, обладает гибкими возможностями по управлению пропускной способностью, поддерживает работу через анонимные сети I2P, Tor и Nodezilla. Для загрузки доступны сборки для Linux, Windows и Mac OS X, а также Jar-файл и исходный код. Основные улучшения: Опция случайного выбора слушающего порта TCP/UDP при запуске Возможность отключить отдельные клиентские протоколы трекера: HTTP, HTTPS. Ранее это было возможно только для UDP. Сейчас эта возможность остаётся недоступной только для DHT Настройки DNS Tracker, позволяющие заблокировать трафик, которого не должно быть, в случае если в торрент-файле указан несуществующий URL-адрес анонса Добавлен явный хеш, используемый для запуска операций Расширена функциональность окна создания torrent-файлов: теперь в torrent-файл можно добавить несколько каталогов или несколько файлов. Добавлен Drag'n'drop файлов и каталогов в это окно Новые места для поиска: архив интерета, в котором доступно более миллиона фильмов, книг и песен, ClearBits, ресурс, распространяющий лицензированные открытыми лицензиями мультимедийные файлы, и Legit Torrents, ресурс, распространяющий только легальный контент Возможность пресечь уход компьютера в спящий режим при условиях загрузки торрентов или раздачи торрента с высшим приоритетом (только для windows). Возможность отображения оставшегося для загрузки времени в режиме абсолютного времени, а также возможность изменения пользователем формата даты в колонках Колонки типа файла и хеша MD5 и CRC-32 файла Значок статуса активности Vuze Remote Control. Для функции move-on-compete, перемещающей файлы в другой каталог после окончания загрузки, добавлена информация в панели информации. Кроме того, реализована асинхронность выполнения этой функции В меню информации о файле добавлена информация об индексе торрент-файла и о расположении этого файла в файловой системе Добавлены сочетания клавиш для остановки и возобновления загрузки/раздачи торрента Опция, позволяющая не отображать уведомление о завершении резервного копирования Отображение на панели количества непрочитанных сообщений в блоге Возможность регулироания высоты строки названия колонок Возможность использования переменных для поисковых запросов (пока не реализовано в графическом интерфейсе) Программа установки (не Linux) теперь спрашивает, классический ли вам нужен интерфейс, или интерфейс Vuze Плагин "Запись DVD" теперь содержит контекстное меню "Библиотека", используемое для создания нового DVD, а также добавления контента в уже начатый проект Переименование отображаемого названия торрента или файлов в нём Конфигурирование команд, запускаемых для открытия известных типов файлов
