Вышел TeX Live 2013

Доступен релиз популярного LaTeX-дистрибутива TeX Live 2013, созданного в 1996 году на базе проекта teTeX. TeX Live является наиболее простым способом для развертывания инфраструктуры для подготовки научной документации, независимо от используемой операционной системы. Для загрузки доступна DVD-сборка (2 Гб) TeX Live 2013, которая содержит рабочее Live-окружение, полный набор установочных файлов для различных операционных систем, копию репозитория CTAN (Comprehensive TeX Archive Network), подборку документации на разных языках (включая русский). Из новшеств можно отметить: Изменена структура дистрибутива, содержимое директории texmf/ перемещено в каталог texmf-dist/, на который теперь указывают обе переменные окружения TEXMFMAIN и TEXMFDIST; Для упрощения установки объединены многие небольшие коллекции поддержки языков; В MetaPost добавлена встроенная поддержка вывода в формате PNG и преобразований с плавающей запятой (IEEE double); LuaTEX переведён на использование Lua 5.2 и включает новую библиотнеку pdfscanner для обработки содержимого внешнего контента в формате PDF; Улучшения XeTEX: Для вывода шрифтов задействована библиотека HarfBuzz вместо ICU (использование ICU продолжено для работы с кодировками для ввода, двунаправленного текста и некоторых других функций); Для раскладки Graphite вместо SilGraphite задействованы Graphite2 и HarfBuzz; На платформе Mac OS X вместо ATSU теперь используется Core Text; При наличии TrueType/OpenType шрифтов с теми же именами, что и шрифты Type1, они используются в первую очередь; Рендеринг в xdvi переведён на использование FreeType вместо t1lib; В microtype.sty добавлена частичная поддержка XeTEX и LuaTEX; В tlmgr упрощена настройка работы с несколькими репозиториями; Добавлена или обновлена поддержка платформ armhf-linux, mips-irix, i386-netbsd и amd64-netbsd. Прекращена поддержка powerpc-aix. Компоненты для поддержки некоторых других платформ исключены из состава DVD, но доступны для загрузки по сети.
