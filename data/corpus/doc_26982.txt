Релиз интегрированной среды разработки приложений NetBeans IDE 6.9

Увидел свет релиз интегрированной среды разработки приложений NetBeans IDE 6.9, поддерживающей разработку на языках Java, JavaFX, PHP, Ruby, JavaScript, Python, Groovy и C/C++. NetBeans распространяется в исходных текстах под лицензиями GPLv2 и CDDL. NetBeans 6.9 является первым релизом, выпущенным под патронажем компании Oracle. Особенности новой версии: Добавлен инструментарий JavaFX Composer, предназначенный для визуального проектирования интерфейса JavaFX-приложений. По своей сути JavaFX Composer напоминает ранее реализованный Swing GUI builder для программ на языке Java. JavaFX Composer даёт возможность быстрого создания, изменения GUI-интерфейса и отладки интерактивных интернет-приложений (RIA) в визуальном режиме, при этом в состав также входят средства для связывания с внешними источниками данных, такими как web-сервисы; Реализована совместимость со стандартом OSGi в средствах для разработки приложений на базе платформы NetBeans. Возможность генерации OSGi-модулей также добавлена в основанные на Maven инструменты управления проектами и организации процесса сборки. Дополнительно доступны инструменты для преобразования NetBeans-модулей в OSGi-пакеты, с возможностью последующего запуска из OSGi-контейнера. С поддержкой стандартов OSGi и Swing платформа NetBeans теперь предоставляет уникальную комбинацию возможностей по стандартному построению графического интерфейса пользователя, сочетающегося со стандартной системой модулей. В настоящий момент поддерживаются следующие реализации стандарта OSGi - Apache Felix 2.0.3 и Eclipse Equinox. Полная поддержка JavaFX SDK 1.3, Spring Framework 3.0, Java Card 3, PHP Zend framework и Ruby on Rails 3.0. Переработана поддержка апплетов и Web Start (JNLP). Также заявлено о реализации поддержки спецификации JSR-299 (Java EE Contexts and Dependency Injection - CDI); Поддержка в основанных на платформе NetBeans приложениях генерации инсталляторов для наиболее популярных операционных систем; Улучшения редактора кода на языке Java и отладчика Java-приложений. В редакторе добавлена поддержка проверки орфографии, реализована возможность подключения обработчиков аннотирования, улучшены средства формирования кода для различных языков программирования, расширены инструменты для редактирования CSS и HTML. В отладчике добавлена поддержка группировки точек останова, прикрепления параметров из истории отладки; Для разработчиков на языках C/C++ добавлена возможность интеграции в проект Unit-тестов, улучшена поддержка разработки удаленных проектов (включая загрузку внешних файлов и просмотр содержимого удаленных каталогов), расширены возможности по гипертекстовой навигации между переопределенными методами и классами.
