Oracle оставит Java открытым продуктом и продолжит развитие MySQL

На совещании руководства Oracle с сотрудниками Sun Microsystems, президент Oracle заявил, что Java останется открытым продуктом, а развитие MySQL будет продолжаться в полном объеме, без каких-либо изменений. Компания Oracle также не намерена выделять какие-либо из продуктов Sun в отдельные компании, что подтверждается прошлым опытом поглощений (ранее Oracle уже приобрела 55 компаний). Представителями Oracle ничего не было сказано по поводу дальнейшей судьбы OpenOffice.org, но, как считает глава Sun Microsystems Джонатан Шварц, проекту ничего не угрожает.
