Доступен Solaris 11.4 SRU27

Опубликовано обновление операционной системы Solaris 11.4 SRU 27 (Support Repository Update), в котором предложена серия очередных исправлений и улучшений для ветки Solaris 11.4. Для установки предложенных в обновлении исправлений достаточно выполнить команду 'pkg update'. В новом выпуске: Компоненты рабочего стола GNOME обновлены до выпуска 3.34. Во vsan (Virtual Storage Area Network) и vdc (Virtual Disk Client) добавлена поддержка klog. В состав включён bandit, статический анализатор для языка Python. Добавлен поисковый движок GNOME tracker-miners. Обновлены многие пакеты, в том числе FreeType 2.10.2, GCC 10.2, Rust 1.44.1, Samba 4.11.11, dbus 1.10.32, gdm 3.34.1, sqlite 3.32.3, xterm 358. Обновлён микрокод для процессоров Intel с устранением уязвимостей. Обновлены версии программ с устранением уязвимостей: Django 2.2.16, Firefox 78.3.0esr, GnuTLS 3.6.14, Thunderbird 78.3.0, Wireshark 3.2.7, lua 5.4.0, mutt 1.14.5, ntp 4.2.8p15, python 3.7.9 и т.д.
