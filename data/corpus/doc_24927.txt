LispWorks 6.0 - коммерческая реализация и IDE для Common Lisp

Вышла новая версия интегрированной среды разработки для Common Lisp - LispWorks 6.0. Среди новшеств стоит упомянуть: поддержка SMP (Symmetric Multiprocessing) на платформах Windows, Mac OS X, Linux, FreeBSD, Solaris поддержка GTK+ сохранение сессии среды разработки функция hcl:do-profiling для профилирования многопоточных приложений интеграция http://common-lisp.net/project/asdf/ - средства для описания организации файлов с исходными кодами строка состояния редактора теперь отображает диапазон номеров строк редактор выделяет скобки цветами в зависимости от уровня вложенности объединение руководства пользователя и справочника в один документ. Подробно обо всех изменениях можно прочесть здесь. Бесплатная редакция Personal Edition версии 6.0 будет выпущена позднее.
