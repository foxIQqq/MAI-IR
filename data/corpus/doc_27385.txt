Вышел дистрибутив Yellow Dog Linux for CUDA 6.2.1

Представлен релиз Yellow Dog Linux for CUDA 6.2.1, основанного на пакетной базе CentOS 5.5 дистрибутива, отличающегося интеграцией в систему компонентов для задействования технологий CUDA (Compute Unified Device Architecture) и OpenCL для обеспечения акселерации за счет привлечения GPU для решения вычислительных задач. Для свободной загрузки доступен установочный iso-образ (4.3 Гб) и загрузочный LiveDVD-вариант (1.5 Гб) дистрибутива. Особенности дистрибутива: Интеграция поддержки CUDA позволила увеличить производительность некоторых стандартных приложений до 9%. В дистрибутив интегрирован полный комплекс инструментов для организации выполнения и разработки программ с использованием разработанного компанией NVidia интерфейса CUDA; В состав входит несколько разных версий CUDA SDK (2.2, 2.3, 3.0 и вышедший недавно релиз 3.1) и поддерживается простой механизм для их переключения (через установку переменной окружения); При обновлении дистрибутива гарантируется сохранность функциональности системы, так как все пакеты для CUDA-редакции Yellow Dog Linux, включая Linux-ядро и драйверы устройств, проходят основательное тестирование, направленное на выявление проблем и регрессий при использовании GPU-вычислений; Для упрощения разработки, в комплект включена основанная на Eclipse специально созданная интегрированная среда для разработки приложений с использованием CUDA, базирующаяся на специально подготовленном плагине к Eclipse. В состав входят дополнительные продукты, такие как Visual Profiler, CUDA-пакет для языка Fortran и PGI accelerator.
