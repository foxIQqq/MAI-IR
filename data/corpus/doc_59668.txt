Выпуск KDE Gear 23.08, набора приложений от проекта KDE

Представлено августовское сводное обновление приложений (23.08), развиваемых проектом KDE. Напомним, что сводный набор приложений KDE c апреля 2021 года публикуется под именем KDE Gear, вместо KDE Apps и KDE Applications. Всего в рамках обновления опубликованы выпуски 544 программ, библиотек и плагинов. Информацию о наличии Live-сборок с новыми выпусками приложений можно получить на данной странице. Наиболее заметные новшества: В файловом менеджере Dolphin по умолчанию скрыты временные файлы и файлы с резервными копиями. Добавлен индикатор прогресса расчёта размера элементов. В информационной панели обеспечено отображение сведений о выделенном файле, а не файле на котором находится курсор. Реализована поддержка быстрого дублирования вкладки через двойной щелчок мышью на вкладке. В двухпанельном режиме (split view) добавлены новые элементы контекстного меню и клавиатурные комбинации для быстрого перемещения и копирования из одной панели в другую. В просмотрщике документов Okular расширены возможности, связанные с заверением PDF-документов цифровой подписью. Подписывающие документ теперь могут добавлять к подписи различные метаданные, такие как пояснение причины и сведения о местоположении, а также размещать за подписью фоновое изображение. Кнопка для вызова интерфейса для создания подписи перемещена в верхний уровень меню "гамбургер". В Okular также добавлены возможности для установки режима масштабирования, применяемого по умолчанию при выводе на печать. В контекстное меню добавлен пункт для копирования в буфер обмена текста аннотации. Календарь-планировщик Kalendar переименован в Merkuro так как старое название перестало соответствовать нынешней функциональности приложения, которая расширена средствами для работы с адресной книгой. В планах интеграция в Merkuro почтового клиента. Переработан интерфейс приложения, добавлена возможность обособленной работы в режимах календаря-планировщика и адресной книги. Расширены возможности ассистента в путешествиях KItinerary. Добавлена возможности импорта железнодорожных билетов по номеру бронирования и имени пассажира, а также поддержка загрузки деталей бронирования с сайта ЖД-компании (пока поддерживаются только Deutsche Bahn и SNCF). Расширены возможности извлечения данных из документов транспортных компаний Aegean Air, Air Asia, ATPI, B&B Hotels, Best Western, České dráhy, Deutsche Bahn, European Sleeper, Eventbrite, FlixBus, Gepard Express, Grimaldi Lines, MÁV, ÖBB, Ouigo, Qatar Airways, SNCB, SNCF, Sunny Cars, Vueling и Westbahn. Изменено оформление страниц для запроса маршрутов. В приложении для сканирования документов Skanpage предоставлена возможность перегруппировки отсканированных страниц через их перемещение в режиме drag-and-drop. Добавлены настройки для корректировки яркости, контраста и цветового баланса. В режиме предпросмотра появилась поддержка выделения нескольких частей изображения для сканирования и автоматичекого разделения области на две страницы. В текстовом редакторе Kate продолжено развитие возможностей для использования в роли интегрированной среды разработки. Во встроенный LSP-клиент (Language Server Protocol) добавлена поддержка языка шейдеров GLSL и сценариев игрового движка Godot. В панели инструментов плагина отладки появились кнопки для навигации в интерактивном режиме GDB, реализовано окно с настройками. Переработано оформление программы для обмена сообщениями Neochat, использующей протокол Matrix. Добавлена поддержка отображения на карте местоположения пользователей и событий. Переработано оформление приложения Tokodon с реализацией клиента к децентрализованной микроблогинговой платформе Mastodon. Добавлены инструменты для модерирования и управления собственным сервером Mastodon, а также для управления пользователями и для настройки подсоединения к другим серверам. Кроме того, добавлена поддержка отправки сообщений через другие приложения, сохранения токена доступа при помощи KWallet, просмотра всех заблокированных или замороженных учётных записей, привязки статуса к профилю, просмотра набирающих популярность тегов. В музыкальном проигрывателе Elise улучшен поиск файлов с обложками альбомов. Добавлен визуальный эффект для плавной замены обложек. Предоставлена возможность управления настройками повторного воспроизведения и воспроизведения в случайном порядке через другие приложения, поддерживающие протокол MPRIS, такие как виджет Media Player или запускаемое на смартфоне приложение KDE Connect. Переработано и переписано с использованием QML обучающее приложение KWordQuiz. В просмотрщике изображений переделан визуальный эффект, применяемый при переключении от одного изображения к другому. Модернизирована панель масштабирования. Улучшены панели инструментов и настройки. В Spectacle, программе для создания скриншотов, упрощена работа с пометками. В браузере Konqueror обеспечено восстановления ранее открытых окон и вкладок после перезапуска. Добавлена опция для изменения User Agent. В эмуляторе терминала Konsole реализован вывод уведомления после завершения выполнения длительных процессов.
