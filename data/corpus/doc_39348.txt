Новая версия менеджера фотографий Shotwell 0.18

Представлен релиз программы для управления коллекцией фотографий Shotwell 0.18. Shotwell предоставляет удобные возможности каталогизации и навигации по коллекции, поддерживает группировку по времени и тегам, предоставляет инструменты для импорта и конвертации новых фотографий, поддерживает выполнение типовых операций по обработке изображений (вращение, устранение эффекта красных глаз, корректировка экспозиции, оптимизация цветности и т.п.), содержит средства для публикации в социальных сетях, таких как Facebook, Flickr и Picasa (для MediaGoblin существует экспериментальный плагин). Готовые бинарные сборки доступны для Linux-дистрибутивов Ubuntu и Fedora. Основные изменения в Shotwell 0.18: Улучшен поиск по комментариям к событиям, обеспечена корректная обработка диакритических знаков; Решены проблемы с публикацией контента в Tumblr; Устранена утечка ресурсов при импорте видео; Обновлены файлы с переводами, переведены разделы с помощью по установке и сборке;
