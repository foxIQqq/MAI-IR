Релиз Wine 1.5.13

Вышел очередной экспериментальный релиз открытой реализации Win32 API - Wine 1.5.13. С момента выпуска версии 1.5.12 было закрыто 104 отчёта об ошибках. Основные изменения в новой версии: Реализован рендеринг клиентской части окон приложений на основе движка DIB; Добавлена поддержка ввода данных напрямую из мыши и клавиатуры, минуя X сервер; Добавлена поддержка полупрозрачных окон; Стала правильно поддерживаться C++ RTTI на платформе x86_64; С этого релиза Wine возвращает информацию о всех найденных процессорах (включая HyperThreading); Добавлена поддержка локализованных имён шрифтов; Обновлены переводы; Исправлено множество ошибок.
