Проект Debian анонсировал сервисы Debian Social

Разработчики Debian представили набор сервисов Debian Social, которые будут размещаться на сайте debian.social и нацелены на упрощение общения и обмена контентом между участниками проекта. Конечной целью является создание безопасного пространства для разработчиков и сторонников проекта, в котором они могут делится информацией о проводимой работе, демонстрировать результаты, общаться с коллегами и делиться знаниями. В настоящее время в тестовом режиме запущены следующие сервисы: pleroma.debian.social (используется ПО Pleroma) - децентрализованная платформа для ведения микроблогов, напоминающая Mastodon, Gnu Social и Statusnet; pixelfed.debian.social (используется ПО Pixelfed) - сервис обмена фотографиями, который может применяться, например, для размещения фотоотчётов; peertube.debian.social (используется ПО PeerTube) - децентрализованная платформа для организации видеохостинга и видеовещания, которую можно использовать для размещения видеоруководств, интервью, подкастов и записей докладов с конференций и встреч разработчиков. Например, в Peertube будут загружены все видео с конференций Debconf; jitsi.debian.social (используется ПО Jitsi) - система для проведения видеоконференций через Web; wordpress.debian.social (используется ПО WordPress) - платформа для ведения блогов разработчиков; writefreely (используется ПО WriteFreely) - децентрализованная система для ведения блогов и заметок. Также проводятся эксперименты с развёртыванием децентрализованной системы блогов на основе платформы Plume; В отдалённом будущем также не исключается возможность создания сервиса обмена сообщениями на базе Mattermost, коммуникационной площадки на базе Matrix и сервиса для обмена звуковыми файлами на основе Funkwhale. Большинство из сервисов децентрализованы и поддерживают объединение в федеративную сеть для взаимодействия с другими серверами. Например, используя учётную запись в сервисе Pleroma можно отслеживать новые видео в Peertube или изображения в Pixelfed, а также оставлять комментарии в децентрализованных сетях Fediverse и взаимодействовать с другими сервисами, поддерживающими протокол ActivityPub. Для создания учётной записи в сервисах предлагается создать заявку в salsa.debian.org (требуется наличие учётной записи в salsa.debian.org). В будущем планируется организовать аутентификацию напрямую через salsa.debian.org при помощи протокола OAuth.
