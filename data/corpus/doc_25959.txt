Вышел корректирующий релиз VirtualBox 3.1.6

Компания Oracle выпустила корректирующий релиз системы виртуализации VirtualBox 3.1.6, в котором исправлена 41 ошибка. Из изменений можно отметить: Устранены проблемы с асинхронным таймером при запуске хост-системы под управлением Linux ядра начиная с версии 2.6.31. Для хост-систем под управлением Linux ядра начиная с версии 2.6.30 налажена работа при переходе в спящий режим; Исправлена ошибка, которая при определенных обстоятельствах могла привести к повреждению образа виртуального окружения после выполнения операции слияния со снапшотом; Улучшена стабильность работы гостевых окружений, работающих в SMP-режиме; Исправлено ряд проблем с работой OpenGL в гостевых окружениях; Устранена возможность повреждения памяти при пропуске через NAT определенного вида ICMP-трафика; Добавлена возможность переключения сетевого интерфейса хост-системы или внутренней сети в процессе работы VM; При работе с VHD-образами исправлена проблема, проявляющаяся при использовании размера блока отличного от 2 Мб; Устранено несколько ошибок при работе с USB-устройствами, например, крах в драйвере OHCI или зависание при определенном стечении обстоятельств в момент подсоединения USB-устройства; В драйверах для гостевых систем (Linux Additions) устранено зависание при запуске X-сервера в Fedora 12 при условии использования Linux ядра 2.6.33; добавлена поддержка уровня инициализации speedboot в Mandriva; добавлен контекст безопасности для запуска mount.vboxsf под управлением SELinux; добавлена поддержка Ubuntu 10.04.
