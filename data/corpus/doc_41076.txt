Релиз десктоп-ориентированной ОС PC-BSD 10.1

Доступен релиз десктоп-ориентированной операционной системы PC-BSD 10.1, основанной на кодовой базе FreeBSD 10.1. В качестве пользовательского окружения по умолчанию предлагается KDE 4.14.2, но в комплекте также доступны окружения Lumina, GNOME3, Mate (форк GNOME 2), Cinnamon, LXDE и Xfce. Развитие PC-BSD курирует компания iXsystems, которая активно участвует в развитии FreeBSD, например, способствует улучшению поддержки ZFS и спонсирует многие инициативы FreeBSD Foundation. Для загрузки подготовлен гибридный iso-образ (3.3 GB, только для amd64), подходящий для использования на DVD и USB-накопителях. Отдельно распространяется iso-образ TrueOS (615 Мб), серверной редакции PC-BSD, работающей без графического интерфейса, предлагающей текстовый вариант инсталлятора. TrueOS поддерживает установку с использованием ZFS на корневом разделе, установку с шифрованием всего диска (с зашифрованным разделом /boot), поддержку окружений загрузки (boot environment), поставку работающих в режиме командной строки версии утилит PC-BSD, в том числе менеджера jail-окружений Warden и системы резервного копирования Life-Preserver. Кроме изменений, свойственных FreeBSD 10.1, в новой версии PC-BSD реализованы следующие улучшения: Поставка нового выпуска развиваемого проектом легковесного окружения рабочего стола Lumina 0.7.1, написанного с использованием библиотеки Qt (без применения QML); Обновление выпусков предлагаемых для установки десктоп-окружений: KDE 4.14.2, GNOME 3.12.2, Cinnamon 2.2.16 Обновление версий программ: Chromium 38.0.2125.104_1, Firefox 33.1. NVIDIA Driver 340.24, Pkg 1.3.8_3; Новый web-интерфейс pc-softweb, комбинирующий возможности AppCafe и Warden, и позволяющий организовать удалённое управление установленным ПО и jail-окружениями с внешнего компьютера или смартфона; В AppCafe добавлена кнопка "Install in Jail" для создания jail-окружения и установки в него выбранной программы. В утилиту pbi_add для выполнения установки PBI-пакета в отдельный jail добавлена опция "-J"; В качестве базовой системы для Linux-эмулятора Linuxulator задействован дистрибутив CentOS 6.6. Ранее в качестве базовой системы для Linuxulator использовался устаревший Fedora 10, который не позволял использовать Skype 4, Sublime Text 2 и различные современные игры, для которых недоставало необходимых библиотек. В графический интерфейс для настройки беспроводной стети добавлена поддержка режима HostAP для создания точек доступа; Поддержка загрузки с использованием UEFI, как для загрузчика системы, так и для установочного диска; Автоматическая настройка оптимальных параметров потребления памяти для ZFS во время установки; В графический и текстовый инсталлятор возвращена поддержка включения шифрования диска целиком (GELI), без оставления незашифрованного раздела /boot (в том числе для mirror/raidz); Новые установочные обазы для развёртывания серверов (TrueOS), снабжённые текстовым инсталлятором; Формирование образов в форматах .vmdk, .vdi и.raw для запуска десктоп и серверных сборок PC-BSD в системах виртуализации VirtualBox и VMware; В состав установочного образа добавлены новые компоненты: QGit, LibreOffice, Pipelight, HexChat, IcedTea, OpenJDK 7 и 8, KRDC, Pidgin, OpenVPN и SaltStack;
