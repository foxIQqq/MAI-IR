Релиз открытой биллинговой системы Ubilling 0.5.9

Представлен новый выпуск открытой биллинговой системы Ubilling, выступающий надстройкой над Stargazer, свободной системы учёта и авторизации в локальных, домашних и офисных сетях. Код Ubilling написан на PHP и распространяется под лицензией GPLv2. Основные возможности Ubilling: Возможность организации работы в нескольких населенных пунктах Множественные хозяйственные контрагенты Разделение понятий сетей и сервисов Тегирование пользователей Учет и планирование работ Встроенная система мгновенных сообщений для общения между сотрудниками Мониторинг активного сетевого оборудования и его опрос, при помощи SNMP Возможность расстановки сетевого оборудования и подключенных домов по карте Мониторинг окружения по произвольным параметрам с SMS информированием о возникших внештатных ситуациях Возможность контроля времени, затраченного на восстановление работы оборудования после сбоя Ассоциация тегов с виртуальными сервисами и начисление по ним абонентской платы Генерация графиков загрузки канала по каждому пользователю на NAS с FreeBSD/Linux/MikroTik Поддержка различных типов авторизации пользователей (IP+MAC, авторизатор, PPTP, PPPoE, RADIUS, Option 82) Поддержка управления серверами доступа на MikroTik Возможность получения статистики сигнала с БС на базе MikroTik Гибкая и полностью автоматизированная работа с DHCP Удобное управление NAS-серверами различных типов Миграция пользователей между сервисами в 1 клик Поддержка различных типов платежей Шаблонизируемая возможность печати документов для пользователей. Шаблонизируемый экспорт оплат Возможность использования карточек пополнения счета Мастер для построения отчетов Встроенный хелпдеск для оперативного общения с клиентами Сервис приема заявок на подключение Функциональный и настраиваемый кабинет пользователя Учет КТВ Встроенные сервисы SMS-информирования абонентов Изменения в новом выпуске: Модуль «Реестр запрещенных сайтов»: готов к использованию. Модуль «Заявки на подключение»: возвращен на место забытый контроль отсылки SMS при создании задач. Модуль «Текущий кеш FDB»: теперь принудительно сохраняет свое состояние. Модуль «Регистрация пользователей»: добавлены уведомления о слишком коротких логинах. Модуль «Регистрация пользователей»: улучшено ведение лога. Модуль «Хелпдеск»: появилась возможность быстрой вставки из заготовок типичных ответов. Модуль «Популярность тарифов»: добавлены отдельные счетчики активных/неактивных пользователей. -**УКВ: кнопка сохранения в форме редактирования пользователя больше не должна самовольно мигрировать. Модуль «Отчет по подключениям»: исправлена сортировка популярности тарифов. Модуль «Скорость тарифов»: добавлено фильтрование лишних пробелов, по кругу от скорости. Модуль «Отчет по услуге кредитования»: исправлена проблема с отображением относительных графичков. Модуль «Заявки на подключение»: теперь умеет показывать список заявок в виде календарика. Сервис приема заявок на подключение: переписан с нуля. Сохранена обратная совместимость со старым по формату заявок. jQuery DataTables обновлен до последнего стабильного релиза 1.10.2
