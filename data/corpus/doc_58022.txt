Выпуск свободной гоночной игры SuperTuxKart 1.4

После года разработки Опубликован релиз Supertuxkart 1.4, свободной гоночной игры с большим количеством картов, трасс и возможностей. Код игры распространяется под лицензией GPLv3. Бинарные сборки доступны для Linux, Android, Windows и macOS. В новом выпуске: Сбалансированы стартовые позиции и переработано размещение элементов при гонках на футбольных полях для того чтобы сделать состязание удобным, независимо от числа участников (вплоть до гонок "один против одного"). Для улучшения планирования стратегии прохождения маршрута на поле добавлена разметка. Добавлен режим прохождения пробного круга. Реализована возможность формирования Windows-сборок для архитектуры ARMv7. Восстановлена поддержка платформы macOS с версии 10.9 по 10.14 включительно. Добавлена новая анимации для элементов и звёзд. Реализована поддержка экранов с высокой плотностью пикселей (HiDPI). Добавлен экспериментальный движок отрисовки, использующий графический API Vulkan. Для включения доступна опция "--render-driver=vulkan" и команда "/vulkan". На экран с сетевыми операциями добавлена функция для поиска треков. Обеспечена возможность настройки ограничения числа игроков. Добавлен новый карт Godette. Обновлён карт Konqi. Обновлены трассы Battle Island и Cave X. Устранены проблемы с трассой Antediluvian Abyss. Добавлены новые текстуры для трассы Shifting Sands
