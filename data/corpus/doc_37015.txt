Выпуск Wine 1.5.31

Доступен очередной экспериментальный выпуск открытой реализации Win32 API - Wine 1.5.31. С момента версии 1.5.30 было закрыто 54 отчёта об ошибках. Из наиболее важных изменений отмечается: Переход интегрированного браузера на новую версию движка Gecko, используемую в Firefox 21; Для Mac OS X добавлена поддержка обратной связи с джойстиком; В драйвере для Mac OS X устранены проблемы с управлением окнами; Закрытие отчётов об ошибках, связанных с работой игр и приложений: Autodesk Revit Architecture 2008, ProntoEdit NG 2.1.8, Chessmaster 10, Starting Everquest 2, PDF Factory Pro, Everquest 2, Saira, pcad2002, LabVIEW 2010, Battlezone, Adobe Photoshop CS4, Wallace & Gromit, Batman: Arkham Asylum, Warlords III, Microsoft SQL Server Management Studio Express 2008, Windows Download Integrator 3.x.
