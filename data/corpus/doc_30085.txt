Релиз VMWare Workstation 7.1.4 и VMWare Player 3.1.4 с исправлением ошибки в безопасности

Компания VMWare выпустила обновления для настольных программ виртуализации VMWare Workstation 7.1.4 и VMWare Player 3.1.4 (бесплатный для некоммерческого использования). Список изменений включает: Исправлена опасная ошибка в безопасности приложения vmrun CVE-2011-1126, которая позволяла атакующему выполнить код в системе с правами запустившего утилиту пользователя (обычно vmrun запускает root), посредством создания специальных файлов в файловой системе (организация загрузки разделяемых библиотек из нестандартных директорий). Проблема проявляется только на платформе Linux; Добавлена совместимость с ядрами Linux по версию 2.6.38 включительно; Исправлена регрессия в выделении памяти для гостевых ОС Windows, которая приводила к ухудшению производительности; Улучшена поддержка USB: теперь в виртуальную машину можно подключить до 16 устройств, вместо доступных ранее 10; Улучшена функция записи видео виртуальной машины: ранее было невозможно записывать ролики больше 1GB в размере, теперь это ограничение убрано; Исправлена невозможность запуска графической версии vmware-modconfig в среде KDE ОС SUSE Linux Enterprise Desktop (SLED) 11; Ранее пользователи, не администраторы в гостевой ОС Windows, могли обновлять VMWare Tools, теперь эта возможность закрыта; Добавлена совместимость с ОС Windows 7 SP1 и Ubuntu 10.10; Для гостевой ОС Fedora 14 и Red Hat Linux 6 исправлена ошибка невозможности лёгкой установки; Добавлена опция для отмены синхронизации времени гостевой ОС после цикла засыпания-просыпания ОС хоста. Для этого в конфигурации гостя добавьте опцию time.synchronize.resume.host = FALSE; Улучшена стабильность и исправлены ошибки в работе VMWare Workstation/Player;
