Обновление специализированного антивирусного Live-дистрибутива ViAvRe

Вышло очередное обновление live-дистрибутива Virtual Antivirus Recheck ViAvRe 09.10, который может быть использован для комплексной проверки компьютера несколькими антивирусными пакетами. Сентябрьский релиз содержит в себе сборку оболочки рабочего стола LXDE, собранную на базе openSUSE 11.3, что значительно увеличивает количество поддерживаемого оборудования, файловых хранилищ и RAID. В дистрибутив включены антивирусы: avg, avast, f-prot, bitdefender и cureit. Для обновления всех антивирусных баз после загрузки необходимо выполнить команду viavre-update. Состав дистрибутива: Linux ядро 2.6.34 Firefox с Flash-плагином pppoe VirtualBox с guest additions (позволяет запустить виртуальную машину с поддержкой сети) Midnight Commander rdesktop (управление терминальными сессиями windows) x11vnc vncview (управление компьютером windows/linux) wine (для запуска других windows приложений) unrar samba client Скрипт для инсталляции Live-образа на жесткий диск Так же доступна версия на основе KDE 4.4.4, которая включает в себя VLC-плеер и программу для управления дисковыми разделами Gparted. Размер загрузочного образа на базе LXDE составляет 577 Мб, а на базе KDE - 787 Мб.
