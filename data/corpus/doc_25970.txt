Готовится к открытию код новой среды разработки Code Bubbles

В рамках прокат Code Bubbles создан прототип интегрированной среды разработки, предлагающей принципиально новый способ организации интерфейса пользователя. Рабочее пространство в Code Bubbles представляет собой неограниченное экраном множество связанных с собой блоков (пузырей), которые представляют собой визуально связанные блоки с кодом и данными, которые можно увеличивать или разворачивать при работе. Одновременно на экране также представлена и косвенно связанная с проектом информация, например, подсказки или заметки с планом работы. Изначально проект написан в качестве Ph.D-диссертации с использованием технологии .NET, но в настоящее время ведется работа по его переработке на языке Java. После готовности кода для бета тестирования исходные тексты проекта будут открыты под свободной лицензией EPL, сама же IDE Code Bubbles будет выполнена в виде плагина для среды Eclipse.
