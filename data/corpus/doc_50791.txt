Выпуск дистрибутива Nitrux 1.1.7 с рабочим столом Nomad

Доступен выпуск дистрибутива Nitrux 1.1.7, построенного на пакетной базе Ubuntu и технологиях KDE. Дистрибутив развивает собственный рабочий стол Nomad, который представляет собой надстройку над пользовательским окружением KDE Plasma. Для установки дополнительных приложений продвигается система самодостаточных пакетов AppImages и собственный центр установки приложений NX Software Center. Размер загрузочного образа составляет 1.5 Гб. Наработки проекта распространяются под свободными лицензиями. Рабочий стол Nomad предлагает иное стилевое оформление, собственную реализацию системного лотка, центра вывода уведомлений и различных плазмоидов, таких как конфигуратор сетевых соединений и мультимедийный апплет, сочетающий средства для регулирования громкости с интерфейсом управления воспроизведением мультимедийного контента. Из развиваемых проектом приложений также выделяется интерфейс для настройки межсетевого экрана Nomad Firewall, позволяющий управлять сетевым доступом на уровне отдельных приложений. Среди входящих в базовую поставку приложений: файловый менеджер Index (можно использовать и Dolphin), текстовый редактор Kate, архиватор Ark, эмулятор терминала Konsole, браузер Chromium, музыкальный проигрыватель VVave, видеопроигрыватель VLC, офисный пакет LibreOffice и просмотрщик изображений Pix. В новом выпуске обновлены компоненты KDE Plasma 5.15.5, KDE Frameworks 5.58.0 и KDE Applications 19.04.1. Включены новые версии ядра Linux 4.19.47, Waterfox 56.2.10, LibreOffice 6.2.4, драйверов NVIDIA 430.14. В znx, инструментарии для управления операционной системой и организации атомарных обновлений системы, обеспечена возможность развёртывания систем в различном порядке (каждая система имеет собственный каталог с данными), добавлен режим restore-esp для восстановления системного раздела EFI, реализована команда reset для удаления всех пользовательских данных из указанного образа и добавлена команда stats для вывода статистики.
