Релиз FFmpeg 0.6.3 и 0.7-rc1

Представлен корректирующий релиз мультимедиа пакета FFmpeg 0.6.3, представляющего собой набор приложений и коллекцию библиотек для манипулирования различными мультимедиа форматами (запись, преобразование и декодирование звуковых и видео форматов). Пакет распространяется под лицензиями LGPL и GPL, разработка FFmpeg ведется смежно с проектом MPlayer. Одновременно выпущен кандидат в релизы ветки FFmpeg 0.7, отличающейся добавлением нескольких существенных изменений: Интегрированы наработки проекта FFmpeg-mt по обеспечению многопоточного декодирования кадров, что позволит значительно ускорить работу FFmpeg на многоядерных CPU; Поддержка видео фильтров из библиотеки libmpcodecs; Проведена оптимизация для процессоров Sandy Bridge; Поддержка формата WebM в муксере и демуксере (упаковщики и распаковщики медиа-контейнеров) Matroska; Понижена нагрузка на систему при упаковке медиа-контейнеров Ogg; Добавлена поддержка MMS-TCP и MMS-HTTP; Реализована поддержка кодирования и декодирования VP8 через библиотеку libvpx; Добавлены декодеры: Mobotix MxPEG, Pictor/PC Paint, G.722 ADPCM, R10k, a64, LATM/LOAS, AMR-WB, Chronomaster DFA и HE-AAC v2; Поддержка декодирования расширений DTS-ES (XCh); Поддержка муксеров и демуксеров для форматов: SAP (Session Announcement Protocol, RFC 2974), Lego Mindstorms RSO, IEC 61937, Mobotix .mxg, Windows Televison (WTV), FFmpeg metadata, Playstation Portable PMP, Leitch/Harris VR (LXF), xWMA; Добавлен собственный декодер для кодека VP8; Поддержка создания RTSP-туннелей поверх HTTP; В ffplay добавлены опции "-exitonkeydown" и "-exitonmousedown". В ffprobe добавлена опция "-show_packets". В ffmpeg добавлены опции "-force_key_frames" и "-copytb"; Реализован собственный декодер для кодеков GSM / GSM MS; В поставку включена библиотека libavcore; Добавлен муксер и демуксер для форматов субтитров MicroDVD и SubRip, а также кодировщик и декодировщик для субтитров SubRip и ASS; Поддержка RTP-пакетизации для кодеков Theora, Vorbis и VP8. Поддержка RTP-депакетизации для кодеков MP4A-LATM, QCELP, X-QT QuickTime и VP8; Добавлен демуксер для потокового вещания Apple HTTP Live Streaming; Новые фильтры: ocv_smooth, frei0r, yadif, blackframe, transpose, cropdetect, setpts, overlay, hqdn3d, fieldorder, gradfun; Реализован демуксер для приема raw-потоков rtp:// через указание URL без SDP-описания; Добавлены ассемблерные оптимизации для платформы Win64; Добавлен фильтр для преобразования битовых потоков MJPEG/AVI1 в JPEG/JFIF; Поддержка инкапсуляции IEC 61937 для E-AC3, TrueHD и DTS-HD; Добавлен парсер для формата FLAC; Добавлен использующий операции с плавающей точкой кодировщик AC-3; Поддержка устройства ввода framebuffer для Linux; Поддержка кодирования AAC через библиотеку libvo-aacenc и кодирования AMR-WB через libvo-amrwbenc;
