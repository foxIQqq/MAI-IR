Релиз Linux-дистрибутивов Lucid Puppy 5.0 и Quirky 1.1

Вышел релиз легковесного Linux-дистрибутива Lucid Puppy Linux 5.0 (Lupu 5.0). Установочный LiveCD-образ занимает 128 Мб (на 23 Мб больше, чем в прошлом выпуске), но при этом в дистрибутиве представлен полный спектр современных возможностей, функционирующих при достаточно скромных требованиях к аппаратному обеспечению. Графическое окружение пользователя базируется на оконном менеджере JWM, собственном наборе GUI-конфигураторов (Puppy Control Panel) и виджетов (Pwidgets - часы, календарь, RSS, состояние соединения и т.п.). Приставка "Lucid" в названии дистрибутива не случайна, в отличие от прошлых выпусков версия Lucid Puppy 5.0 построена на базе бинарных пакетов из Ubuntu 10.04, что позволило значительно сократить время подготовки и тестирования релиза, и одновременно обеспечить пакетную совместимость с репозиториями Ubuntu. Другой особенностью Lucid Puppy 5.0 является то, что это первая версия подготовленная силами сообщества без прямого участия основателя проекта Barry Kauler, область интересов которого сместилась в развитие проекта Quirky. Параллельно с Lucid Puppy 5.0 развивается еще три ответвления дистрибутива Puppy, использующие систему пакетов T2: классический Puppy 4.4 (+ вариант Puppeee для нетбуков), инновационный Quirky и оптимизированный для 64-разрядных систем FatDog64 (+ Wary с подборкой новых программ). Из представленных в Lucid Puppy 5.0 новшеств можно отметить: Quickpet - возможность быстрой установки в один клик предконфигурированных и оттестированных наборов программ, для решения определенного круга задач. Например, для редактирования графики представлен Gimp, для для создания сайтов - Kompozer, для управление музыкальной коллекцией - Songbird, для навигации в web - браузеры Firefox, Seamonkey, Chromium и Opera, при этом браузеры можно установить все разом или выборочно. После загрузки Lucid Puppy пользователю сразу доступен рабочий стол (без диалога аутентификации) и набор средств для его настройки под собственные нужды. Не составляет труда выбрать дополнительный язык и локаль, предоставляются новые упрощенные GUI-конфигураторы для установки обновлений, смены базовых приложений, настройки сети и других параметров системы. Дополнение: вслед за Lucid Puppy 5.0 вышел релиз Quirky 1.1, варианта Puppy Linux, развиваемого основателем проекта с целью обкатки новых идей, которые позднее могут быть перенесены в другие ветки Puppy Linux. Релиз в основном носит корректирующий характер, с особенностями Quirky можно познакомиться в анонсе первого выпуска. Кроме исправления ошибок в версии 1.1 обновлены некоторые программы (например, произведена миграция SeaMonkey с ветки 1.x на 2.x), в качестве оконного менеджера и панели по умолчанию задействован JWM (ранее использовалась связка Openbox/Fbpanel, но по заявлению разработчиков JWM работает заметно быстрее).
