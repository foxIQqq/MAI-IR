Syntext Serna 4.1 - WYSIWYG редактор XML документов

Вышла очередная версия кроссплатформенного визуального XML редактора - Syntext Serna 4.1. Особенности: Коммерческая и бесплатная версии; Визуализация документов при помощи XSLT+XSL-FO в процессе редактирования; Целостное представление документов, составленных из отдельных фрагментов с помощью XML entities, XInclude-фрагментов и функции XSLT document() (особенно актуально для DITA); Публикация в PDF и HTML - интегрирован с FOP, XEP, Antenna House XSL Formatter; Поддержка DITA и ее специализаций, Docbook и других популярных XML форматов; Кроссплатформенность - работает на Linux, Mac OS X, Sparc Solaris 8, MS Windows (2000/XP/Vista), написан с использованием Qt; Можно расширять функционал при помощи Python & C++ API;
