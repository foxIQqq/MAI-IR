В Windows 10 продемонстрирован запуск Wine

В свежей эксперментальной сборке Windows 10 (Windows 10 Insider Preview Build 15025) проведена очередная модернизация прослойки WSL ("Windows Subsystem for Linux"), обеспечивающей трансляцию системных вызовов Linux в системные вызовы Windows и позволяющей запускать приложения Linux в Windows. Наиболее интересным следствием такой модернизации стала возможность запустить немодифицированную 64-разрядную сборку Wine (пакет wine64-development) в Windows и использования свободной реализации Windows API без применения ограниченной по функциональности сборки WineOnWindows. Wine на лету транслирует Windows API в системные вызовы POSIX, которые затем прослойкой WSL транслируются обратно в вызовы Windows API.
