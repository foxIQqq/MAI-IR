Выпуск VirtualBox 7.0.8

Компания Oracle опубликовала корректирующий релиз системы виртуализации VirtualBox 7.0.8, в котором отмечено 21 исправление. Одновременно сформировано обновление прошлой ветки VirtualBox 6.1.44 с 4 изменениями, в числе которых улучшение определения использования systemd, поддержка ядра Linux 6.3 и решение проблем со сборкой vboxvide с ядрами из RHEL 8.7, 9.1 и 9.2. Основные изменения в VirtualBox 7.0.8: Предоставлена возможность отключения верификации модулей ядра Linux по цифровой подписи через указание параметра VBOX_BYPASS_MODULES_SIGNATURE_CHECK="1" в файле /etc/vbox/vbox.cfg для хост-окружений и в файле /etc/virtualbox-guest-additions.conf для гостевых систем. Добавлена начальная поддержка ядра Linux 6.3. В дополнениях для гостевых систем на базе Linux добавлена экспериментальная поддержка перезагрузки модулей ядра и пользовательских сервисов после окончания установки VirtualBox, что позволяет обойтись без перезагрузки всей системы после обновления набора Guest Additions. В VBoxManage добавлена команда "modifynvram enrollmok" для добавления ключа MOK (Machine Owner Key) в NVRAM, который может использоваться для верификации модулей ядра гостевой системы с Linux. Добавлен API для добавления цифровых подписей в список MOK (Machine Owner Key). В дополнениях для гостевых систем на базе Linux улучшено определение использования systemd для инициализации системы. Решены проблемы со сборкой модуля vboxvideo при использовании ядер из состава RHEL 8.7, 9.1 и 9.2. В менеджере виртуальных машин улучшена поддержка вложенного запуска виртуальных машин. Устранены проблемы, проявлявшиеся при использовании гипервизора Hyper-V. Налажен запуск гостевых систем с UEFI при использовании macOS 13.3+. В GUI в диалоге закрытия виртуальной машины возвращён флаг для восстановления текущего снапшота, решены проблемы с записью значения порта в редакторе USB-фильтров, налажено редактирование имени VM и типа ОС в панели с детальной информацией о виртуальной машине. В Oracle VM VirtualBox Extension Pack решены проблемы с поставкой криптографического модуля для обеспечения полного шифрования виртуальной машины. Для драйвера E1000 упрощён процесс смены привязки к сети. В virtio-net добавлены изменения для улучшения поддержки FreeBSD 12.3 и pfSense 2.6.0. Решены проблемы с графикой, проявлявшиеся при использовании гостевых систем с Windows 7.
