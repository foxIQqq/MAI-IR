Релиз операционной системы Chrome OS 47

Компания Google представила релиз операционной системы Chrome OS 47, основанной на ядре Linux, открытых компонентах и web-браузере Chrome 47. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 47 доступна для всех актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache. Основные изменения в Chrome OS 47: Снижена нагрузка на CPU при воспроизведении видео с YouTube на системах с новым поколением процессоров Intel. Прирост производительности наиболее заметен при показе видео с частотой 60 кадров в секунду; В экранной клавиатуре появилась поддержка набора жестами (набор через непрерывное движение между образующими слово буквами) на устройствах сенсорными экранами; На устройствах с сенсорными экранами реализованы новые жесты для удаления и выделения текста; Добавлен баннер для добавления текущего приложения в закладки; Созданы новые горячие клавиши для замены комбинаций, конфликтующих с web-страницами. Более контрастное выделение окна в режиме инкогнито с оформлением в стиле "material design"; Включены по умолчанию уведомления об окончании загрузки.
