Релиз TeX Live 2011

Доступен релиз популярного LaTeX-дистрибутива TeX Live 2011, созданного в 1996 году на базе проекта teTeX. TeX Live является наиболее простым способом для развертывания инфраструктуры для подготовки научной документации, независимо от используемой операционной системы. Для загрузки доступна DVD-сборка (2 Гб) TeX Live 2011, которая содержит рабочее Live-окружение, полный набор установочных файлов для различных операционных систем, копию репозитория CTAN (Comprehensive TeX Archive Network), подборку документации на разных языках (включая русский). Из изменений можно отметить прекращение поддержки Mac OS X Panther и Tiger (оставлена поддержка Leopard и более новых версий); в комплект включена программа для библиографической обработки biber; программа MetaPost (mpost) больше не создает и не использует .mem-файлы; для всех платформ в Perl задействована переработанная реализация updmap, отличающаяся более высокой производительностью.
