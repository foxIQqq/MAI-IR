Релиз программы для мгновенного обмена сообщениями Pidgin 2.7.7

Вышел корректирующий релиз клиента для мгновенного обмена сообщениями Pidgin 2.7.6 в котором отмечено 21 изменение. Особенности выпуска: Добавлена поддержка протокола MSNp16, включая поддержку нескольких одновременных входов (Multiple Points of Presence - MPOP). Переработана архитектура, связанная с передачей файлов в MSN-сети; В поставку включены дополнительные сертификаты Microsoft Internet Authority 2010 и Microsoft Secure Server Authority 2010; Для ICQ/AIM опция настройки SSL-шифрования заменена на меню из трех элементов: "Не использовать шифрование", "Использовать шифрование, если возможно" и "Обязательно использовать шифрование"; Исправлены проблемы с передачей URL через ICQ/AIM; Для XMPP добавлена возможность реализации методов аутентификации через плагины; Дополнение: вышел релиз Pidgin 2.7.7 в котором исправлены проблемы с установкой SSL-соединения и крахом после потери соединения в ICQ/AIM. Устранена ошибка обработки сертификатов, проявляющаяся при работе в сети MSN.
