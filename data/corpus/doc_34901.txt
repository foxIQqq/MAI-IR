Компания Canonical намерена монетизировать Ubuntu через продажу контента в Unity

Оливер Райс (Oliver Ries), директор по технологиям компании Canonical, представил новые средства монетизации дистрибутива Ubuntu, которые будут внедрены начиная с выпуска Ubuntu 12.10. Кроме продажи приложений и контента через Центр управления приложениями и сервис Ubuntu One в новой версии дистрибутива появятся средства вывода контекстных предложений о покупке музыки и видео, интегрированные в оболочку Unity. Подобные коммерческие предложения будут выводиться в специальной секции "Дополнительные предложения", выводимой при выполнении операции глобального поиска в Dash - пользователь получит возможность не только найти локально хранящийся контент, но и купить контент из online-архивов музыки и видео. Кроме того, по умолчанию на панель будут добавлены ярлыки приложений для покупки в магазинах Amazon и Ubuntu One. Дополнение 1: Опробовать новую возможность можно установив представленный вчера выпуск Unity 6.6, который войдёт в состав второй бета-версии Ubuntu 12.10. Дополнение 2: Прекратить показ контекстной рекламы можно удалив пакет unity-lens-shopping ("sudo apt-get remove unity-lens-shopping"). Данный пакет будет установлен в Ubuntu 12.04 по умолчанию, но не связан обязательной зависимостью с Unity. Также можно ограничить работу локальным поиском вызывая панель поиска по комбинации Super+A для приложений и Super+F для файлов. Дополнение 3: Марк Шаттлворт опубликовал в своём блоге сообщение с ответами на некоторые критические замечания по поводу новой возможности. Например, при передаче поисковых запросов в Amazon данные будут проходить через транзитный сервер Canonical и отправляться в обезличенной форме, что позволит не допустить отслеживания предпочтений конкретных пользователей.
