Выпуск дистрибутива LinuxConsole 2.0 для любителей компьютерных игр

Анонсирован релиз Live-дистрибутива LinuxConsole 2.0, нацеленного на создание простой десктоп-системы, сформированной по аналогии с игровыми консолями. Размер загрузочного iso-образа 630 Мб. Дистрибутив может быть установлен не выходя из Windows и работать без создания дополнительного дискового раздела. LinuxConsole является независимым дистрибутивом, построенным с использованием разработанного собственными силами сборочного инструментария dibab. Графическое окружение основано на LXDE. Версия LinuxConsole 2.0 примечательна полной переработкой кодовой базы, упрощением системы сборки, реализацией поддержки установки пакетов в формате deb вместо модулей собственного формата, обеспечением совместимости с Debian. В состав входят следующие версии базовых компонентов: Busybox 1.21.1, ядро Linux 3.12.5, Mesa 9.1.6, Xorg 1.14.3, Network Manager 0.9.8.6, Firefox 26.0. В поставку включены игры: openttd, tuxpaint, 2H4U, bzflag, cultivation, etr, neverball, xmoto, frozen-bubble, dangerdeep, dossizola, foobillard, freecraft, supertuxkart, teeworlds.
