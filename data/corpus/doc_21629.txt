Представлен релиз XenServer 5.5

Компания Citrix Systems выпустила новую версию виртуализационной платформы XenServer 5.5, доступную для бесплатного использования. В версии 5.5 добавлены средства для подключения внешних модулей резервного копирования, интегрирована поддержка Active Directory для управления хостами, расширены возможности утилит для конвертации образов виртуальных машин из одного формата в другой (VMDK, VHD, OVF, OVA и XVA), улучшена поддержка выполнения в качестве гостевой ОС дистрибутивов SUSE Linux Enterprise Server 11, Debian 5.0 и RedHat/CentOS 5.3.
