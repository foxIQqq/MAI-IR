Релиз редактора научных документов Lyx 1.6.6

Вышла версия 1.6.6 текстового редактора Lyx, построенного по принципу WYSIWYM (What You See Is What You Mean). Lyx - это графический редактор на основе LaTeX и предназначен в первую очередь для создания научных документов. Структура документа частично определяется выбранной разметкой (макетом). Разметка соответствует классам LaTeX, однако Lyx может также работать и с разметкой DocBook. Экспорт осуществляется в два этапа: преобразование в формат LaTeX, а затем в выходной формат (DVI, PDF). В новой версии произошли следующие изменения: Добавлена поддержка формата SVG через librsvg или Inkscape не ниже версии 0.47; В модуль импорта документов LaTeX (tex2lyx) добавлено распознавание типа кавычек, в зависимости от языка документа; Добавлена поддержка Aspell для поддержки орфографии, при использовании библиотеки "aspell" проверка может производиться в параллельном режиме. Для проверки орфографии отныне также может использоваться библиотека enchant; В графической оболочке реализована возможность перетаскивания документов мышью для их открытия; Обновлено руководство и переводы на различные языки, включая русский; Устранены многочисленные ошибки, приводящие к падению программы. Например, крах при выполнении операции Undo при сборке с Lyx с Qt 4.6. Основные возможности программы: WYSIWYG редактор таблиц и формул (возможен также ввод с помощью команд LaTeX), WYSIWYG для масштабирования и поворота изображений Поддержка BibTeX Автонумерация заголовков и параграфов Проверка орфографии с помощью GNU Aspell, автодополнение текста Базовая поддержка математических пакетов Maple, Maxima, Octave, Mathematica Поддержка множества графических форматов (включая XFig, Dia и др.) Интерфейс на различных языках, включая русский, возможность создания документов, содержащих различные языки Полная поддержка LaTeX с возможностью вставки LaTeX кода в любое место документа Поддержка различных форматов импорта и экспорта документа (LaTeX, PDF, Postscript, DVI, ASCII, HTML, OpenDocument, RTF, MS Word) Поддержка литературного программирования (literate programming) Поддержка систем контроля версий Возможность привязки желтых клейких заметок Поддержка Linux, Mac, Windows (через Cygwin) Ознакомиться со свежими планами разработчиков на версию 2.0, которая должна выйти в августе 2010 года, можно на данной странице. Среди ожидаемых новшеств: Расширенные возможности поиска (поиск формул и других отформатированных блоков); Проверка орфографии на лету (слова с ошибками будут подчеркиваться с возможностью выбора в меню оптимального варианта замены); Функция сравнения нескольких документов (различия выделяются разными цветами); Использование многоязычного тезауруса, совместимого с OpenOffice thesauri; Поддержка привязки к документу нескольких индексов; Бэкенд с поддержкой альтернативного LaTeX-препроцессора XeTeX; Возможность настройки форматов для вывода данных, которые будут использоваться по умолчанию; Поддержка LaTeX-команд LaTeX-commands \phantom, \hphantom, \vphantom, \sout, \uuline, \uwave; Улучшение интерфейса для управления таблицами.
