GitHub получил 100 млн долларов инвестиций и выпустил клиентское приложение для платформы Android

Популярный хостинг открытых проектов GitHub объявил о заключении партнёрского соглашения с финансовой компанией Andreessen Horowitz, в рамках которого GitHub получил инвестиции в размере 100 млн долларов. Примечательно, что это первые внешние инвестиции в проект, до этого GitHub развивался на собственные доходы от оказания платных сервисов и не зависел от внешних инвесторов. В настоящее время GitHub обслуживает более 3 млн репозиториев, на сайте зарегистрировано более 1.7 млн разработчиков и пользователей, ежедневно около 7 тысяч пользователей создают новые репозитории, а в около 80 тысячах репозиториев вносятся изменения. Полученные средства планируется направить на расширение персонала компании, усовершенствование подсистем и расширение поддержки дополнительных платформ, таких как мобильные устройства. Из новшеств, упоминается создание специализированных web-приложений для различных операционных систем. В конечном счёте, планируется обеспечить удобные средства для работы в GitHub для различных категорий пользователей, от разработчиков-одиночек до больших корпоративных коллективов, от программистов до дизайнеров и авторов контента. GitHub станет проще для начинающих и мощнее для экспертов. Будут расширены средства для организации совместной работы. Что касается расширения персонала, то GitHub уже объявил о приёме на работу шести новых разработчиков. К GitHub присоединились: Jason Dixon (будет заниматься средствами виртуализации), Derek Greentree (специалист по Map/Reduces и организации хранения больших объёмов данных), Drew Miller (будет заниматься развитием клиента для Windows), Luke Hefson (будет заниматься развитием клиента для Mac OS X), Zachary Kaplan (будет реализовывать новые идеи) и Austin Spires (будет развивать средства для корпоративных пользователей). Одновременно объявлено о выходе первой версии приложения-клиента для работы в GitHub с устройств на базе платформы Android. Программа позволяет обеспечить быстрый доступ с телефона ко многим функциям GitHub, например, можно работать со своими репозиториями и Gist-ами, общаться с разработчиками, отслеживать решение проблем.
