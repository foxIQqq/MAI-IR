Выпуск GNOME 3.10, обзор новшеств

После шести месяцев разработки официально представлен выпуск десктоп-окружения GNOME 3.10. Для быстрого ознакомления с возможностями нового выпуска в ближайшее время будет подготовлен специальный демонстрационный Live-дистрибутив на основе openSUSE. Из наиболее значительных изменений в новой версии можно отметить экспериментальную поддержку работы под управлением Wayland; единое системное статусное меню; заголовочная строка, объединяющая заголовки и панели инструментов; новый интерфейс для настройки параметров экрана; новое приложение для работы с картами; новая программа для прослушивания музыки; интерфейс установки программ; доведение до готовности приложения для ведения заметок; поддержка аутентификации с использованием смарткарт. Подробнее об улучшениях: Возможность запуска оболочки GNOME Shell в качестве композитного сервера Wayland, для чего обеспечена поддержка базовых интерфейсов, идентичных с эталонным композитным сервером Weston. Реализация работы поверх Wayland в GNOME 3.10 рекомендована только для начального ознакомления, довести порт до полной готовности планируется в выпуске GNOME 3.12, который выйдет в марте 2014 года. Для запуска пользовательского сеанса на базе Wayland следует использовать команду "gnome-session --session=gnome-wayland", для запуска отдельных приложений с использованием Wayland перед запуском следует установить переменную окружения "GDK_BACKEND=wayland"; Поддержка входа в систему и разблокирования рабочего стола с использованием смарткарт. В том числе возможно использование смарткарт вместо парольной аутентификации и настройка системы для блокирования экрана или завершения сеанса при отсоединении смарткарты; Реализация объединённого системного статусного меню, комбинирующего в единой области информацию из всех системных индикаторов. Вывод системной информации в едином блоке позволяет не тратить время на определение с какой пиктограммой связаны те или иные настройки, а получить доступ ко всей информации сразу (заряд батареи, яркость экрана, управление громкостью, параметры сетевых соединений, блокировка экрана и завершение работы, вызов настроек, параметры текущего сеанса и т.п.). По умолчанию для каждого типа параметров показывается однострочная базовая информация, а более подробные настройки вынесены в субменю (например, показано имя текущего WiFi-соединения, при нажатии на которое появляется блок с возможностью выбора другой сети, отключения соединения и детальных настроек); Представлен новый элемент интерфейса - заголовочная строка (header bar), объединяющая в общей заголовочной области заголовок окна и панель инструментов приложения, что позволяет высвободить больше экранного пространства для основного содержимого окна. В отличие от ранее практикуемого метода скрытия заголовка для окон, раскрытых на полный экран, "header bar" позволяет решить проблемы с неявным способом закрытия развёрнутого окна и тратой экранного места на отображение заголовка окна в ситуациях, когда окно не развёрнуто на весь экран. Кроме того, указанное изменение является подготовкой для обеспечения поддержки рендеринга на стороне клиента при использовании Wayland; Новое приложение Software, унифицирующее в едином интерфейсе все операции, связанные с управлением установкой приложений. Новый интерфейс позволяет находить нужные сторонние приложения, устанавливать или удалять программы, следить за появлением обновлений. Наиболее востребованные приложения представлены в специальной отдельной секции рекомендованных программ. В будущем ожидается появление системы рейтинга, отправки комментариев и публикации скриншотов; В обзорном режиме список доступных для запуска приложений теперь выводится с разбивкой на страницы (ранее использовалась прокрутка общего списка). Проведена работа по увеличению удобства определения искомого приложения; В Центр управления добавлена новая универсальная панель для настройки параметров для людей с ограниченными возможностями; Реализован новый интерфейс для настройки параметров экрана и многомониторных конфигурций; В панель смены обоев рабочего стола добавлена поддержка изменения фоновых изображений для заставки, показываемой во время блокирования экрана; Добавлена новая панель для настройки даты и времени; В менеджер online-аккаунтов добавлена поддержка входа в сервисы Google с использованием двухэтапной проверки. Интегрирована работа с аккаунтами чата и поддержка Flickr; В GNOME интегрирован фреймворк для определения текущего местоположения на основании различных источников, в том числе GPS и параметров интернет-соединения. Фреймворк задействован для автоматизации выбора часового пояса и для отображения актуального времени с учётом текущего местоположения. Кроме того, подготовлено новое базовом приложение для работы с картами GNOME Maps. В настоящее время программа пока ограничивается возможностью просмотра карт, поиска искомого места и указания текущего местоположения. В качестве источника картографических данных используется база OpenStreetMap; Новая программа для прослушивания музыки и навигации по музыкальной коллекции - GNOME Music. Интерфейс программы достаточно прост и позволяет выполнять навигацию по музыкальной коллекции с детализацией по исполнителям, альбомам или композициям. На будущее запланирована поддержка воспроизведения потокового звука от медиасерверов через UPNP и загрузка музыки из online-сервисов; Доведённое до полнофункционального состояния приложение для ведения заметок Bijiben, в том числе с поддержкой обмена данными с Tomboy, синхронизации через online-аккаунт GNOME и хранение заметок в облачном хранилище ownCloud; Доведено до полноценного вида приложение Photo, позволяющее просматривать, искать и группировать фотографии, размещённые в локальных и облачных хранилищах, организовывать совместный доступ к фотографиям, выводить на печать. В новой версии добавлена поддержка сервиса Flickr. Программа Photo может быть использована для отправки фотографий на другие устройства, такие как телевизоры с поддержкой протокола DLNA или другие компьютеры с GNOME; Добавлен режим точной прокрутки, позволяющий при прокрутке с удерживанием клавиши Shift (или кликнув и удерживая курсор на движке полосы прокрутки) сдвигать содержимое очень мелкими шагами; Новое оформление утилиты GNOME tweak, используемой для доступа к расширенным настройкам, выходящим за рамки Центра управления; Поддержка корректного автоматического масштабирования элементов интерфейса на экранах сверхвысокого разрешения, например, используемых в Chromebook Pixel; Web-браузер GNOME переведён на использование по умолчанию поисковой системы DuckDuckGo вместо Google. Заголовок и панель инструментов объединены в одну заголовочную строку. Произведена интеграция со службой системного поиска, что позволяет искать данные в истории посещений и определять открытые страницы прямо из обзорного режима; Значительно обновлён базовый шрифт Cantarell, который расширен поддержкой дополнительных языков и оптимизирован для улучшения читаемости; В программу Boxes, которая сочетает средства для удаленного соединения с рабочими столами с механизмами управления и запуска виртуальных окружений на локальных и внешних компьютерах, добавлена поддержка импорта образов виртуальных машин в форматах QEMU (.cow, .qcow и .qcow2), raw (.img), VirtualBox 1.1 (.vdi), VMWare 3/4 (.vmdk), VirtualPC (.vpc) и cloop (Linux Compressed Loop). Также добавлена поддержка drag&drop интерфейса для перемещения файлов между текущей системой и виртуальной машиной; В программу для работы с документами (Documents) добавлена поддержка навигации и просмотра документов, размещённых в облачном хранилище ownCloud; В просмотрщике документов появилась поддержка формата DjVu, возможность caret-навигации для PDF, поддержка перемещения по документу с использованием курсора и новая панель для вывода результатов поиска; Добавлена опция (EnablePrimaryPaste=false) для включения нового интерфейса для работы с буфером обмена. Вместо вставки данных из буфера обмена по средней кнопке мыши, при включении опции нажатие на среднюю кнопку инициирует начало выделения блока текста и вызов контекстного меню для выбора действия с выделенным текстом. Новая схема нацелена на решение проблем с управлением буфером обмена на устройствах без манипулятора мышь, например, на планшетах с сенсорным экраном.
