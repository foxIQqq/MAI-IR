Создатель Puppy Linux представил релиз проекта Quirky 1.0

Создатель проекта Puppy Linux представил первый релиз нового Live-дистрибутива Quirky 1.0, продолжающего развитие Puppy и нацеленного на апробацию некоторые необычных идей, которые в будущем могут появиться в Puppy. Для подготовки дистрибутива используется единая с Puppy Linux система сборки Woof. По сравнению с Puppy, проект Quirky имеет более узкую специализацию и нацелен главным образом на эксперименты. Изначально Quirky создавался как проект для себя, без расчета на предпочтения других пользователей. Загрузочный образ Quirky занимает 106 Мб и содержит полный комплект программ, доступных в стандартном Puppy Linux 4.3.1. Из наиболее заметных отличий от Puppy Linux 4.3.1 можно отметить: Добавлен GUI-конфигуратор для настройки параметров тачпада; Кроме оконного менеджера JWM, добавлена поддержка Openbox и Fbpanel; Обновлены такие приложения как Abiword и Pmusic; Осуществлен переход на Linux ядро 2.6.33.2 и обновленный набор прошивок; Добавлены новые апплеты для системного трея: Traytemp, Powerapplet, Freememapplet, Retrovol. В виде апплета также оформлено приложение для мониторинга сети; Добавлено новое приложение для управления громкостью и микширования звуков; В качестве медиа-плееров задействованы Gecko-mediaplayer и Gnome-mplayer. В комплект также входят консольные Mplayer и Ffmpeg; Использованы альтернативные Xorg-драйверы intel_drv.so и i810_drv.so. В комплект включен набор проприетарных драйверов Nvidia; Для доступа к web-камере задействована программа Gtkam; Для монтирования и навигации по SMB-разделам в комплект включено приложение lameSMBxplorer; Для управления системой печати используется GUI GtkLP; Для проигрывания потокового контента используются программы pupRadio и pupTelly, для загрузки видеороликов с YouTube - You2Pup, для записи звука - Precord, для преобразования мультимедиа файлов - FFconvert; В состав включено GUI-приложение для управления частотой процессора в зависимости от текущих задач. Некоторые идеи, по развитию Quirky в будущем: Встраивание дистрибутива в Linux-ядро: последние версии ядра позволяют встраивать в ядро образ initramfs, содержащий полноценную ФС с дистрибутивом. В итоге дистрибутив будет выглядеть как одно большое Linux-ядро. Значительное сокращение размера дистрибутива от версии к версии. Первые версии Puppy занимали 20 Мб, последний стабильный релиз занимает 105 Мб, а бета-версия будущего релиза уже 150 Мб. В Quirky все будет наоборот; Вместо switch_root, переключения на корневой раздел, во время загрузки планируется использовать изолированное chroot-окружение.
