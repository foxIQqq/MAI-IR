На разработку открытого новостного движка Armstrong выделено 975 тысяч долларов

Фонд поддержки журналистской деятельности Knight Foundation объявил о предоставлении гранта в размере 975 тысяч долларов на развитие открытой платформы для организации публикации новостного контента Armstrong. Грант получили некоммерческие новостные ресурсы The Texas Tribune и The Bay Citizen, c 2009 года разрабатывающие платформу Armstrong для своих сайтов. Цель финансирования - формирование легко устанавливаемой и готовой к использованию открытой платформы, пригодной для создания новостных изданий. По мнению представителей фонда, появление подобной платформы будет способствовать развитию региональной журналистики и созданию местных новостных изданий, публикующих локальные новости и освещающих события из жизни отдельных регионов. Armstrong отличается ориентацией на новостные порталы и поддерживает полный цикл приема, редактирования и помещения новостей. Интерфейс оптимизирован для активного использования в новостях мультимедиа контента и поддерживает инструменты для обслуживания локального архива изображений, видео и звуковых файлов. В системе присутствуют элементы для глубокой интеграции с социальными сетями, такими как Facebook и Twitter. Поддерживаются: формирование версии сайта для различных мобильных устройств; механизмы организации личного кабинета посетителя; формирование подписных листов; интеграция с открытыми CRM-системами SugarCRM и Salesforce; средства аналитики и CEO-оптимизации. В настоящий момент код, написанный на языке Python с использованием фреймворка Django, можно загрузить на странице проекта в GitHub. В качестве СУБД используется PostgreSQL, имеется модуль для интеграции CouchDB. Для кэширования задействован memcached. Наработки проекта распространяются в рамках лицензии BSD. Первый релиз планируется выпустить в июне.
