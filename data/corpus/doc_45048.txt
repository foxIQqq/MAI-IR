Выпуск свободного видеоредактора OpenShot 2.1

Состоялся выпуск свободной системы нелинейного видеомонтажа OpenShot 2.1. Код проекта поставляется под лицензией GPLv3: интерфейс написан на Python и PyQt5, ядро обработки видео (libopenshot) написано на C++ и использует возможности пакета FFmpeg, интерактивная шкала времени написана с использованием HTML5, JavaScript и AngularJS. Для пользователей Ubuntu пакеты с последним выпуском OpenShot доступны через специально подготовленный PPA-репозиторий, для остальных дистрибутивов сформирована самодостаточная сборка в формате AppImage. Имеются сборки для Windows и macOS. Редактор отличается удобным и интуитивно понятным пользовательским интерфейсом, позволяющим редактировать видео даже начинающим пользователям. Программа поддерживает несколько десятков визуальных эффектов, дает возможность работы с многотрековыми монтажными шкалами с возможностью перемещения мышью элементов между ними, позволяет масштабировать, кадрировать, осуществлять слияние блоков видео, обеспечивать плавное перетекание из одного ролика в другой, производить наложение полупрозрачных областей и т.п. Имеется возможность перекодирования видео с предварительным просмотром изменений на лету. Благодаря задействованию библиотек проекта FFmpeg, OpenShot поддерживает огромное количество всевозможных форматов видео, звука и изображений (включая полную поддержку SVG). Основные новшества: Расширены средства работы со звуком. Добавлены возможности отображения графиков формы звуковых волн на шкале времени, появилась возможность замены звуковых каналов и выделения звука в другой клип, что позволяет выполнять такие операции, как настройка громкости отдельных звуковых каналов, удаление определённых звуковых каналов и простое устранение опережения или отставания звука; Новый интерфейс изменения свойств, который позволяет корректировать значения путём простого перемещения ползунка мышью и автоматически привязывать их к ключевому кадру. Клик правой кнопкой мыши на любом значении позволяет изменить режим интерполяции или удалить ключевой кадр. При перемещении по клипу панель с параметрами также меняется, отражая параметры для текущего момента на клипе; Новый выпадающий блок в редакторе свойств наглядно показывает выбранный элемент (клип, преобразование, эффект) и позволяет быстро переключаться между ними. Для каждого элемента отображается эскиз и список вложенных эффектов; Возможность изменения настроек горячих клавиш. Пользователь теперь может посмотреть все доступные горячие клавиши и при желании переопределить их, в том числе назначить свои клавиатурные комбинации для таких операций как "выделить всё" и "поместить в буфер обмена"; Улучшение шкалы времени. Добавлена поддержка перетаскивания файлов и преобразований на шкалу времени. Реализована возможность закрепления трека с запретом его перемещения или изменения размера. Обеспечена остановка воспроизведения на последнем клипе шкалы времени, вместо показа черного кадра. Добавлены новые контекстные меню для выравнивания, разбиения и других операций. Обновлено оформление; Добавлено интерактивное руководство для ознакомления новичков, которое можно отключить в один клик; Операция разделения теперь может применяться сразу для всех треков, что можно использовать для разделения всех клипов и преобразований на текущей позиции с сохранением левой стороны, правой стороны или обеих сторон; Проведена работа по оптимизации наиболее медленных частей libopenshot, что позволило значительно повысить производительность эффектов, в частности, основанных на применении фильтров яркости и насыщенности; Поддержка определения собственных профилей; Улучшена поддержка проектов, созданных в OpenShot 1.x; Расширены средства отладки - в настройках теперь можно включить отладочный режим, создающий подробный лог действий в директории /.openshot_qt; Сформированы сборки для 64-разрядных систем Windows.
