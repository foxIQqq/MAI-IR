Ubuntu Studio переходит с GNOME на Xfce

Разработчики официально развиваемого в недрах проекта Ubuntu дистрибутива Ubuntu Studio сообщили об утверждении решения по уходу от использования GNOME. Вместо GNOME в будущих выпусках Ubuntu Studio будет использовано десктоп-окружение на основе Xfce и панели Avant Window Navigator. По мнению разработчиков проекта, оболочки на базе Unity и GNOME-Shell плохо сочетаются с реализуемыми дистрибутивом задачами и негативно воспринимаются пользовательской аудиторией Ubuntu Studio. Дистрибутив Ubuntu Studio предназначен для обработки и создания мультимедиа контента. В состав входит подборка профессиональных и любительских инструментов для работы со звуком (Ardour, Beast, Fluidsynth), видео (Blender, Kino, Stopmotion, OpenMovieEditor) и графическими данными (GIMP, Inkscape, wacom-tools).
