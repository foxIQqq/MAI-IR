Для интеграции в X.Org представлены патчи с поддержкой мультитач

В списке рассылки разработчиков X.Org представлен набор патчей с реализацией расширения X Input 2.2, в котором определена поддержка мультитач для X-сервера и сопутствующих компонентов, таких как драйвера xf86-input-evdev и библиотеки libXi. Интеграция поддержки мультитач и X Input 2.2 ожидается в X.Org Server 1.12, который планируется выпустить в марте. Разработка включает в себя реализацию протокола Multitouch, описывающего методы взаимодействия между X Window System, мультитач-устройствами и пользовательскими приложениями. Кроме того, добавлена раздельная обработка событий, связанных с одновременными касаниями к экрану, реализована эмуляция нескольких указателей для избранных событий и механизмы для перехвата и переотправки управляющих последовательностей, связанных с касаниями к экрану.
