Выпуск пакетного менеджера DNF 4.6

Доступен релиз пакетного менеджера DNF 4.6, который используется по умолчанию в дистрибутивах Fedora Linux и RHEL 8. DNF является ответвлением от Yum 3.4, адаптированным для работы с Python 3 и использующим библиотеку hawkey в качестве бэкенда для разрешения зависимостей. По сравнению с Yum, DNF обладает заметно более высокой скоростью работы, низким потреблением памяти и более качественным управлением зависимостями. В новой версии в операциях отката истории изменений (redo, rollback и undo) добавлена поддержка "comps", файла с метаданными для разбивки пакетов на функциональные группы. В директиву filter_modules добавлена опция для отсеивания устаревших версий на основе параметра module_obsoletes. В логе dnf.log обеспечено отражение пакетов, установленных или удалённых через DNF API. В API добавлены функции для загрузки кэша репозитория.
