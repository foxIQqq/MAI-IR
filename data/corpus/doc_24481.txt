Объявлены победители конкурса лучших дополнений к Firefox 3.5

Подведены итоги конкурса "Extend Firefox 3.5", проводимого среди разработчиков дополнений. В качестве основных критериев отбора были названы: использование новых возможностей Firefox 3.5, инновационный подход, использование открытых стандартов, дружелюбность интерфейса и полезность для пользователя. На конкурс принимались недавно появившиеся расширения или расширения в новых версиях которых представлены значительные улучшения или возможности, основанные на новшествах Firefox 3.5. В результате рассмотрения присланных на конкурс работ, жюри, состоящее из 8 известных экспертов, определило 8 победителей в 4 номинациях: Лучшее новое расширение: FireFound - интерфейс для организации поиска потерянных или украденных ноутбуков или телефонов. Используя встроенную в Firefox 3.5 реализацию Geolocation API дополнение осуществляет мониторинг местоположения устройства и периодически отправляет данные на специальный сервер. Если на потерянном ноутбуке кто-то запустит Firefox, местоположение устройства будет зафиксировано; Multifox - позволяет организовать управление идентификационной информацией. Например, полезно когда пользователь использует несколько разных логинов для одного сервиса; Voyage - визуальный навигатор по истории посещений, расширение пытается определить контекстную связь и представить данные в виде графа. Лучшее обновленное расширение: Smart Find - расширяет стандартные возможности поиска слов на странице средствами нечеткой логики, позволяя искать слова не зная их точного написания, даже если ключевое слово задано с ошибкой; Book Burro - при покупке книг в online, осуществляет запрос в популярные сервисы и выводит лучшие доступные для данной книги ценовые предложения; Speed Dial - реализация одноименной функции из web-браузера Opera, позволяющей вместо пустой страницы выводить скриншоты для быстрого перехода на 9 выбранных пользователем сайтов; Расширение для упрощения совершения покупок: Grocery List Generator - интерфейс для составления списка покупок, интегрированный с сайтами, на которых публикуются кулинарные рецепты. Расширение для игр и развлечений: Destroy the Web - аркадная игра, оперирующая элементами любой открытой web-страницы. В качестве награды победителям были вручены призы: ноутбуки MacBook Pro, лицензионные версии TextMate IDE и ExpanDrive, электронные книги издательства Manning Publication, видеозаписи и книги от InformIT, комплекты атрибутики Firefox. Кроме того определены 6 поощрительных призов, которые были вручены создателям новых расширений, не занявших основные призовые места: Boomtango - реализация принципиально нового способа просмотра истории посещений; Tab Flick - упрощает переброс табов между окнами; History Tree - отображает историю посещений в наглядном древовидном виде; WikiSubs - позволяет организовать совместную работу по написанию субтитров к видео; Customize Your Web - WYSIWYG редактор для изменения web-страниц; QuickFox Notes by Amin E - система хранения заметок в стиле похожем на закладки.
