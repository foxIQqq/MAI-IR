Четвёртый выпуск Replicant, полностью свободной прошивки на основе Android

Фонд свободного ПО анонсировал кампанию по привлечению средств для разработки проекта Replicant, включённого в список высокоприоритетных проектов Фонда СПО. В рамках проекта развивается полностью свободный вариант платформы Android, включающий в себя только свободный код, без использования проприетарных компонентов и закрытых драйверов. Для расширения спектра поддерживаемых в Replicant смартфонов, Фонд СПО организовал сбор пожертвований, которые позволят поддержать инфраструктуру проекта и приобрести серию новых устройств для портирования и тестирования свободной прошивки. В заявлении Фонда СПО указывается, что несмотря на то, что платформа Android является свободным ПО, производители оборудования комплектуют прошивки для своих устройств проприетарными компонентами, которые обычно используются для обеспечения поддержки аппаратного обеспечения, например, для взаимодействия с GPU, WiFi и Bluetooth чипами. Кроме того, поставляемые по умолчанию прошивки, как правило, укомплектованы большим числом проприетарных приложений. Задачей проекта Replicant является предоставление пользователям функциональности традиционных прошивок, подготовленной с использованием только свободных комопонентов. Для установки приложений в Replicant по умолчанию предлагается каталог F-Droid, в котором размещены только свободные приложения и который Фонд СПО рекомендует для использования на любых Android-устройствах вместо Google Play. Одновременно представлен четвёртый выпуск прошивки Replicant на базе исходных текстов Android 4.0. По сравнению с оригинальной прошивкой, в Replicant произведена замена достаточно большой порции проприетарных компонентов, включая видеодрайверы, бинарные прошивки для WiFi, библиотеки для работы с GPS, компасом, web-камерой, радио-интерфейсом и модемом. Новый выпуск примечателен увеличением числа формируемых сборок с 5 до 9. Кроме ранее поддерживаемых устройств Galaxy Tab 2 (7.0), Nexus S, Galaxy S, Galaxy S2 и Galaxy Nexus, образы прошивки теперь доступны для Galaxy S3 и нескольких вариантов Galaxy Tab 2 (10.1). В настоящее время в Replicant для Galaxy S3 и Galaxy Tab 2 работает 2D, звук, телефония, 3G-модем, WiFi, Bluetooth и сенсоры. Пока отсутствует поддержка ускорения графики, работы с камерой, GPS, NFC и FM-радио. Несмотря на заметный прогресс в развитии прошивки для свободного телефона Goldelico GTA04, продолжающего развитие проекта OpenMoko, установочные образы ещё не готовы к публикации из-за неготовности ядра Android для платформы DM37x. Из устройств, для которых поддержка Replicant находится в развитии, также можно отметить HTC Dream/HTC Magic и Nexus One.
