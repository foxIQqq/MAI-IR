Вышел кластерный дистрибутив Rocks 5.5 и 6.0

Одновременно выпущены релизы Rocks 5.5 и 6.0, основанные на пакетных базах CentOS 5.8 и 6.2. В рамках проекта Rocks развивается дистрибутив для создания вычислительных кластеров, grid-сетей и инфраструктур для обеспечения работы виртуальных окружений на базе Xen и KVM. Размер универсального загрузочного iso-образа 2.6 Гб. Дополнительно подготовлено около десятка специализированных roll-образов, позволяющих упростить изменение конфигурации различных приложений (изменённые приложения ставятся в /opt) при развёртывании узлов на большом числе машин (например, созданы образы c пакетами для web-сервера, bio, condor, ganglia, perl, python, hpc, java, kvm, sge). Дистрибутив может быть использован для быстрого развертывания кластеров в локальной сети с загрузкой основного управляющего узла с LiveCD/LiveUSB. После загрузки управляющего узла, дополнительные узлы могут быть загружены по сети через PXE и автоматически настроены. В комплект входит набор специально разработанных средств для централизованного управления кластером и мониторинга его работы. Основные изменения: Параллельно сформированы две сборки на базе CentOS 5.8 и 6.2, отличающиеся компонентами CentOS - все развиваемые проектом Rock дополнения в обоих сборках идентичны и развиваются в рамках одной кодовой базы; Для CentOS 6 (64-bit) подготовлено roll-дополнение с поддержкой виртуализации на базе KVM. В настройках Yum оставлены изначально поставляемые дистрибутивом репозитории пакетов, но по умолчанию они отключены; Возможность произвольного выбора на машине-фронтэнде сетевых интерфейсов для приватной и публичной сети (ранее eth0 использовался для внутренней, а eth1 для внешней сети); Поддержка именованных через BIOS Ethernet-устройств (для того чтобы всегда использовать только стандартные имена следует загружать ядро на узлах с опцией biosdevname=0); Поддержка создания машин-фронтэндов с одним физическим интерфейсом (если в системе только одна сетевая карта настраивается виртуальный сетевой интерфейс eth0:0); Поддержка Upstart-скриптов, созданных для CentOS 6; Новая команда "rocks report post" для преобразования post-секции XML-файла в формат исполняемого shell-скрипта; Новый пакет foundation-sqlite c SQLite3.
