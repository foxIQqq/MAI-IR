Наиболее важные события 2014 года

Небольшая итоговая подборка наиболее важных и заметных событий уходящего года: Значительные выпуски Red Hat Enterprise Linux 7.0, SUSE Linux Enterprise 12, FreeBSD 10, MariaDB 10.0, Musl 1.0.0, Java 8, GCC 4.9, Solaris 11.2, Android 5, OpenWRT 14.07, CUPS 2.0. Новые открытые продукты: кластерная СУБД MariaDB Enterprise 1.0, VPN-сервер SoftEther, HTTP-сервер H2O, WebScaleSQL (высоко масштабируемый вариант MySQL), СУБД Postgres-XL, SSH-сервер TinySSH, микроядро seL4, Bitrig (форк OpenBSD) Критические уязвимости в OpenSSL (Heartbleed), GnuTLS, Bash (ShellShock), NTPD. Компрометация протокола SSLv3. Бэкдор в генераторе псевдослучайных чисел Dual_EC_DRBG. Атака BadUSB. Рекордные DDoS-атаки, использующие NTP в качестве усилителя трафика. Форки OpenSSL: LibreSSL от проекта OpenBSD и BoringSSL от компании Google. Новый NTP-сервер Ntimed. Эпопея c утверждением перехода Debian по умолчанию на systemd. Решение по переходу Ubuntu на systemd. Усиление конфронтации сторонников и противников systemd. Создание Devuan, форка Debian без systemd, и uselessd, минималистичного варианта systemd. Форк ConsoleKit от проекта Xfce. Разработка для BSD-систем аналогов компонентов systemd, востребованных в GNOME. Введение в практику расширенной поддержки (LTS) для прошлых стабильных веток Debian (oldstable). Возвращение Debian GNU/Linux с Eglibc на GLibc; Создание фонда для финансирования разработки ключевых открытых проектов. Выделение openSUSE Factory в независимый дистрибутив с rolling-обновлениями. Закрытие проекта TrueCrypt. Начало превращения дистрибутива Fedora Linux в многослойный продукт с выделением Fedora Base, Fedora Workstation, Fedora Server, Fedora Cloud и Fedora Atomic. Переход проекта CentOS под крыло компании Red Hat. Волна новых продуктов, использующих каркас браузера Chromium в качестве платформы для разработки. Например, развиваемый в GitHub редактор кода Atom. Передача развития DNS-сервера BIND10 в руки независимого сообщества. Разработка DHCP-сервера из состава BIND 10 вынесена в отдельный субпроект Kea. Samsung разорвал контракт по выплате отчислений Microsoft за платформу Android. Китайские власти раскрыли список патентов, используемых Microsoft для сбора отчислений за платформу Android. Создание мобильной платформы LuneOS, продолжившей развитие webOS. Выпуск свободных систем для обновления ядра Linux без перезагрузки: kGraft от SUSE, kpatch от Red Hat, livepatch универсальный базовый компонент на основе kpatch и kGraft. Развитие средств контейнерной виртуализации: активное продвижение Docker в дистрибутивы, выпуск инструментария LXC 1.0, первый стабильный выпуск серверной Linux-системы CoreOS. Red Hat начал развитие Atomic, модульной ОС на базе изолированных контейнеров. Первый выпуск гипервизора Jailhouse, развиваемого компанией Siemens. Объединение продуктов OpenVZ и Parallels Cloud Server в единый открытый проект Virtuozzo Core. Создание Rocket, конкурирующего с Docker инструментария управления контейнерами. Расширение возможностей ядра Linux: классификатор трафика на основе BPF, пакетный фильтр Nftables, класс планирования задач Deadline, блочное устройство zRAM, механизм защиты KASLR (рандомизация памяти ядра), единая унифицированная иерархия cgroup, подсистема для создания туннелей поверх UDP, слой блочных устройств blk-mq, ФС OverlayFS. Открыт код Fastsocket, новой высокомасштабируемой реализации сетевой подсистемы ядра Linux. Начало работы над OpenGL NG. Переход компании AMD к открытой стратегии разработки драйверов для Linux. Разработка в Китае новой операционной системы COS (China Operating System), основанной на Linux. Выход стандарта HTML5 и обновлённой спецификации протокола HTTP/1.1. Публикация международного стандарта C++14. Первые стабильные релизы KDE Frameworks 5, KDE Plasma 5 и KDE Applications 14.12. Уход от поставки KDE Software Compilation в пользу развития компонентов KDE в рамках отдельных веток и циклов разработки. Инициативы по адаптации Firefox OS для телевизоров, планшетов и персональных компьютеров. Начало массовых продаж смартфонов на основе Firefox OS. Представлены первые производители смартфонов на базе Ubuntu, подписаны контракты, но выпуск смартфонов отложен на 2015 год. Развитие решений на базе платформы Android для автомобильных информационно-развлекательных систем; Серия блокировок проектов на GitHub под прикрытием действующего в США Закона об авторском праве в цифровую эпоху (DMCA). Длительная блокировка GitHub в России. Первый релиз профессионального видеоредактора Lightworks для Linux. Первый стабильный выпуск открытой системы композитинга Natron. Работа над полнометражным свободным фильмом Gooseberry. XMPP перешёл на обязательное шифрование передачи данных; Facebook открыл код Hack, варианта языка PHP со статической типизацией. Dropbox опубликовал Pyston, реализацию языка Python, базирующуюся на JIT. Начало развития PHPNG. Новые десктоп-окружения Moonlight (на основе Qt5) и Lumina (от проекта PC-BSD). Первый выпуск десктоп-окружения LXQt, созданного в результате объединения LXDE и Razor-Qt. Пользовательские оболочки Orbital и Quantum, построенные на основе Wayland. Новые выпуски GNOME 3.12 и GNOME 3.14. GTK+ перешел на новую тему оформления. XWayland интегрирован в основную кодовую базу X.Org. Новый свободный игровой движок Godot. Игровой движок CryEngine портирован для Linux. Игровой движок Unreal Engine 4 вышел с поддержкой Linux. Появление версий многих современных игр для Linux. Распределение последних блоков адресов IPv4. Проект Mozilla выпустил новый высококачественный кодировщик JPEG-изображений mozjpeg. Фабрис Беллар предложил новый формат изображений BPG. Инициатива по созданию контролируемого сообществом удостоверяющего центра. Появление новых web-браузеров Otter Browser, Breach, Fifth и Gngr. Firefox Developer Edition пришёл на смену Aurora-сборкам Firefox. Внедрение нового интерфейса Australis в Firefox. В Firefox добавлен видеочат на базе WebRTC, механизмы DRM, кодек H.264. Экспериментальный интерфейс Firefox.html. Возрождение инициативы по переводу Firefox на многопроцессную архитектуру. Возобновление поддержки платформы Linux в новой редакции браузера Opera. Chrome для Linux перешёл с GTK+ на собственный графический стек Aura. Дополнение по мотивам комментариев к новости: Значительно усилилась активность разработки свободной операционной системы ReactOS. Для развития пригодного для широкого использования выпуска создан проект ReactOS Community Edition. Компания Microsoft учредила некоммерческую организацию .NET Foundation и начала движение по переводу .NET на открытую модель разработки. Открыт код компилятора Roslyn для C# и Visual Basic, ASP.NET vNext, Open XML SDK, а также серверной платформы .Net Core (.NET Framework Libraries, .NET Core Framework Libraries и RyuJit VM). Компоненты .Net для построения пользовательских приложений открывать не планируется. Прекращена разработка Series 40 и Android-смартфонов Nokia X. Появление отечественного движка для создания браузерных 3D-приложений Blend4Web. Компания NVIDIA начала работу над поддержкой Wayland и Mir в проприетарном драйвере для Linux, связанные с EGL улучшения уже доступны в ветке 346.хх. Проект FreeBSD вывел из эксплуатации инфраструктуру, обеспечивающую работу сервиса CVSup. Для FreeBSD задействован новый графический стек. OpenBSD перешёл на использование OpenSMTPD по умолчанию. Из состава OpenBSD удалены apache httpd, sendmail, Bind, nginx и ALTQ, началась реализация собственного http-сервера.
