Релиз дистрибутива Clonezilla Live 1.2.10-14

Представлен релиз Linux дистрибутива Clonezilla Live 1.2.10-14, предназначенного для быстрого клонирования дисков (копируются только используемые блоки). Задачи выполняемые дистрибутивом сходны с проприетарным продуктом Norton Ghost. Размер iso-образа дистрибутива - 99 Мб (прошлая версия занимала 133 Мб). Дистрибутив основан на Debian GNU/Linux, в своей работе использует код проектов DRBL, Partition Image, ntfsclone, partclone, udpcast. Возможна загрузка с CD, USB Flash и по сети (PXE). Поддерживаются ФС: ext2, ext3, ext4, reiserfs, reiser4, xfs, jfs, FAT, NTFS, UFS, HFS+ (Mac OS X). Поддерживается режим массового клонирования в multicast режиме, например, при тестировании, за 10 минут удалось провести клонирование исходного 5.6 Гб диска на 41 клиентскую машину. Основные новшества: Проведена синхронизация с пакетной базой Debian Sid по состоянию от 22 сентября; Обновлены версии программ: ядро Linux 3.0.0-3, Partclon 0.2.29, iPXE 20110910; Gdisk обновлен до версии 0.8.0, которая скомпилирована без поддержки Unicode, что позволило исключить из состава дистрибутива libicu44 и уменьшить размер iso-образа на 6 Мб; После завершения восстановления системы из директории /etc/udev/rules.d/ теперь удаляются автоматически создаваемые udev-правила 70-persistent-cd.rules и 70-persistent-net.rules. Для отмены данного поведения в экспертном режиме следует использовать опцию "-irhr"; Подготовлен архив с исходными текстами всех используемых в дистрибутиве компонентов; Добавлен параметр загрузки nicif, при помощи которого можно назначить другой MAC-адрес для сетевой карты (например, nicif=aa:bb:cc:dd:ee:ff).
