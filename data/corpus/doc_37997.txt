Релиз медиаплеера VLC 2.1.0

После полутора лет разработки увидела свет новая значительная ветка медиапроигрывателя VLC 2.1.0. В процессе подготовки новой версии внесено более 7 тысяч коммитов от 140 разработчиков, исправлено несколько сотен ошибок. Кроме расширения функциональности новая версия примечательна переводом большинства модулей с лицензии GPLv2+ на LGPLv2.1+, что допускает связывание модулей с проприетарными продуктами и открытыми проектами, распространяемыми под лицензиями, не совместимыми с GPL. Ключевые улучшения: Полностью переписано ядро для работы со звуком и звуковые модули, что позволило добиться улучшения регулирования громкости и управления звуковыми устройствами. Обеспечена корректная поддержка многоканальных раскладок для форматов 5.1, 6.1 и 7.1. Добавлены новые модули вывода для Windows Vista, Android, iOS, OpenBSD и OSSv4. Добавлено несколько новых звуковых фильтров, в том числе новый канальный фильтр, фильтр для улучшения стереозвучания, плагин для визуализации VSXu и фильтр для управления громкостью с применением инструкций ARM NEON. Встроенная поддержка акустических отпечатков, позволяющих идентифицировать звук по форме сигнала, что даёт возможность выявлять идентичные звуковые файлы в коллекции, независимо от различия метаданных и названия; Возможность использования VDPAU для ускорения декодирования видео в Linux. Поддержка средств аппаратного ускорения декодирования видео, доступных на платформах OS X (VDADecoder), Android (MediaCodec) и Windows (Intel QuickSyncVideo); Обеспечена начальная поддержка Qt 5.0; Поддержка воспроизведения видео на удалённом рабочем столе при помощи протоколов VNC/rfb и RDP/Remote Desktop; Новый модуль вывода через OpenGL, работающий с использованием шейдеров GLSL для преобразования YUV в RGB. Портирование модуля вывода через OpenGL на использование OpenGL ES и включение по умолчанию использования OpenGL ES1/ES2. Новый анаглифический фильтр для вывода 3D-изображений бок о бок; Поддержка звукового кодека Opus (используется libopus), а также кодеков CDXL, Ut Video, VBLE, Dxtory, Ulead DV, TechSmith Screen Codec 2, Microsoft Expression Encoder Screen, Microsoft Application Screen (MSS1, MSS2), Indeo Audio Coder, RealAudio Lossless; Поддержка видеокодеков G2M4, MSS1, MSS2, TSCC2, CDXL, Ut, VBLE; Полноценная поддержка субтитров SCTE-27 и EIA-608; Поддержка фрагментированных файлов в форматах MP4 и Wave/RF64; Расширена поддержка метаданных для медиаконтейнеров Ogg, AVI, MP4 и MKV. Поддержка звука FLAC, Atrac, ADPCM, DV Type 1, 12bits DV внутри контейнеров AVI. Расширенная поддержка потоков AVI, MKV и MJPEG. Улучшена поддержка записи в AVI и MKV; Переписан модуль ввода через V4L2. Улучшены модули ввода для Blu-Ray, Dash и HTTP; Новый модуль ввода для RTMP, работающий с использованием libavformat; Порт для платформы Android, поддерживающий версии от 2.1 до 4.3, и работающий на архитектурах ARMv6, ARMv7, x86 и MIPS; Начальный порт для WinRT, поддерживающий Windows 8, 8.1 и WP8; Новый порт VLC для iOS, поддерживающий версии iOS от 5 до 7, и работающий на всех моделях iPads и iPhones, выпущенных после 3GS; Для web-плагинов реализован безоконный режим, позволяющий обеспечить более гладкую интеграцию с HTML-элементами; В дополнение к ранее переведённым с лицензии GPLv2+ на LGPLv2.1+ базовых библиотек libVLC, libVLCcore, libdvbpsi, libaacs и libbluray, отмечено завершение процесса смены лицензии для почти всех модулей. Лицензия LGPL теперь применяется к модулям разграничения доступа, кодекам, упаковщикам и распаковщикам медиа-контейнеров, модулям для фильтрации, вывода и смешивания звука, модулям вывода и корректировки видео, модулям вывода текста, парсеру XML и блокам оптимизации с использованием инструкций ARM NEON и SSE2. Таким образом, почти весь код VLC теперь доступен под лицензией LGPL.
