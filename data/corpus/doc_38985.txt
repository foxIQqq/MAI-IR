Началось альфа-тестирование OpenMandriva Lx 2014.0

Объявлено о доступности альфа-версии дистрибутива OpenMandriva Lx 2014.0, продолжившего развитие Mandriva Linux после передачи компанией Mandriva S.A. управления проектом в руки некоммерческой организации "OpenMandriva Association". Для загрузки предлагаются (torrent) Live-сборки размером 1.6 Гб (i586, x86_64). Основные новшества: Обновление ядра Linux до выпуска 3.11 с патчами nrjQL и поддержкой многопоточной распаковки SquashFS; Поддержка сборки пакетов для архитектуры armv7hl c поставкой для данной платформы пользовательского окружения RazorQt; Обновление системного менеджера Systemd до версии 208; Обновление графического стека: X.org server 1.15 и Mesa 10.x; Поддержка аутентификации пользователей при помощи realmd; Интеграция пакетов shorewall и drakfirewall для конфигурирования межсетевого экрана; Поставка свежего Java-стека, основанного на пакетах из Fedora Linux; Обновление пользовательского окружения KDE до версии 4.12.1. Включение в поставку медиацентра Plasma Mediacenter; Поддержка работы с мобильными устройствами на базе iOS 7; Поддержка протокола MTP для доступа к данным на устройствах с Android 4; Добавление libdvdcss в репозиторий; Новые версии GIMP 2.8.10, Scribus 1.5, LibreOffice 4.2, hplip 3.13.13, cups 1.7.1; Доведение до работающего состояния поддержки загрузки на системах с UEFI; Замена MySQL на MariaDB. Обеспечение поддержки кластерной БД Galera.
