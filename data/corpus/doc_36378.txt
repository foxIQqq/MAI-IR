Новые выпуски дистрибутивов для создания интернет-киосков: Webconverger 18.0 и Porteus 2.0 Kiosk Edition

Представлены новые выпуски двух дистрибутивов, предназначенных для создания интернет-киосков, предоставляющих пользователю средства для выхода в Web в общественных местах, таких как интернет кафе. Дистрибутивы позволяют быстро сформировать окружение для организации ограниченного доступа пользователей в сеть, которым доступен только интерфейс web-браузера, без возможности выполнения иных программ. Webconverger 18.0. Дистрибутив основан на свежих версиях пакетов из репозиториев Debian GNU/Linux, дополнительно стабилизированных в рамках проекта Progress Linux. В качестве web-браузера используется Firefox 19 (в комплекте с Adobe Flash). Дистрибутив может быть установлен на жёсткий диск, загружаться по сети, с CD/DVD или с USB-диска. Размер загрузочного iso-образа 372 Мб. Из добавленных в новом выпуске улучшений можно отметить переход на ядро Linux 3.8.2, включение в поставку плагина Google talk для совершения голосовых и видео вызовов, новый набор шрифтов с поддержкой кириллицы и, самое главное, официальная поддержка русского языка в базовой поставке (достаточно активировать язык при загрузке и затем без лишних настроек можно переключать раскладку по Alt+shift). Для обеспечения работы в режиме web-киоска предусмотрены расширения: webconverger (главная страница с подборкой ссылок на web-сервисы), webcnoaddressbar (запрещение ручного ввода url), webcfullscreen (полноэкранный режим без элементов интерфейса браузера). Дистрибутив поддерживает задание таймаута неактивности, после истечения которого будет осуществлён перезапуск браузера. Предусмотрена возможность включения или выключения обращения к локальному диску или USB Flash-накопителю клиента. Имеется средство для быстрой настройки доступа на уровне пакетного фильтра: можно разрешить доступ только к заданным IP или запретить хосты по чёрному списку. Интересной особенностью дистрибутива является реализация системы обновления дистрибутива, базирующаяся на установке обновлений напрямую из Git-репозитория, размещённого на GitHub. Компоненты дистрибутива работают поверх Git-репозитория, в котором сохранено содержимое иерархии директорий рабочей системы, для монтирования которой задействован FUSE-модуль git-fs. В процессе загрузки по умолчанию используется последняя стабильная ветка из локального Git-репозитрия, периодически синхронизируемого с внешним Git-репозиторием. Porteus 2.0 Kiosk Edition - модульный дистрибутив, построенный на пакетной базе Slackware 14 Linux и отличающийся поставкой только минимального набора компонентов, необходимых для запуска браузера Firefox 19.0. Удаление лишних составных частей дистрибутива позволило свести размер загрузочного образа всего до 37 Мб и повысить общую безопасность. Дистрибутив поставляется в преднастроенном виде, но поддерживает изменение настроек и кастомизацию загрузочного образа при помощи специально подготовленного приложения "Kiosk Wizard". Например, можно установить страницу по умолчанию, задать пароль для гостевого входа, поменять фоновое изображение, настроить оформление Firefox, добавить дополнительные плагины, включить поддержку беспроводной сети, настроить переключение раскладки клавиатуры и т.д. Итоговый образ работает только в режиме чтения (используется ФС iso9660) и недоступен для модификации пользователем интернет-киоска. Все временные данные, накопленные в рамках текущего сеанса пользователя, очищаются при каждом перезапуске Firefox. Благодаря небольшому размеру, по умолчанию дистрибутив загружается целиком в ОЗУ, что позволяет заметно увеличить скорость работы.
