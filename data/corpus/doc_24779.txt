Выпущена новая версия WINE-LXP 1.4

Компания ТрастВерс сообщила о выходе основанного на Wine проприетарного продукта WINE-LXP 1.4, конкурирующего с Wine@Etersoft и предназначенного для запуска в Linux наиболее популярных российских бизнес-приложений для платформы Windows. Использование WINE-LXP позволяет решить проблему перехода с Windows на Linux, избавляя от необходимости использования Windows в виртуальных машинах или как вторую систему лишь для запуска одного-двух приложений, не предназначенных изначально для использования в Linux. По сравнению с предыдущей версией можно отметить следующие улучшения: Добавлена поддержка 1С:ХроноГраф Школа 2.5; Добавлена поддержка MetaTrader4; Добавлен скрипт автоматической настройки wine для использования 1С: Предприятие 7.7; Добавлен скрипт автоматического скачивания и установки библиотек, необходимых для работы отдельных программ. Некоторые из поддерживаемых приложений: Бухгалтерия, автоматизация производства - 1С Предприятие 8.1, 1С Предприятие 7.7 (локальные и сетевые версии) Программы подготовки налоговой отчетности - Налогоплательщик ЮЛ Правовые системы - Гарант F1, КонсультантПлюс (c ограничениями) Офисные приложения- Microsoft Office (2003, 2007) (кроме Outlook) Работа в сети Интернет - Internet Explorer 6.0 Обработка изображений - Adobe Photoshop CS2
