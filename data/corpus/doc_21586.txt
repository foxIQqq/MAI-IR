Возрождение электронного журнала для пользователей и разработчиков GNOME

После более двух лет неактивности представлен новый выпуск электронного журнала для пользователей и разработчиков GNOME. В апрельском номере представлены следующие статьи: GNOME Asia Summit 2008 Report - отчет с конференции азиатских разработчиков GNOME; GConf Configuration System Concepts for Developers - основы работы с системой конфигурации GConf при разработке GNOME-приложений; An Introduction to the Message Indicator - встраивание возможностей indicator-applet в приложения на Си и Python; Cooking With Gourmet Recipe Manager - обзор приложения Gourmet Recipe Manager для хранения кулинарных рецептов; Behind the Scenes: Stormy Peters - интервью с руководителем организации GNOME Foundation; Letter From the Editor: GNOME 3.0 - заметка о планах по поводу GNOME 3.0.
