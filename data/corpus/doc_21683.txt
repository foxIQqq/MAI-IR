Открыт публичный Git репозиторий Qt. Qt Jambi отправлен в свободное плавание

Начиная с версии 4.5 исходные тексты Qt распространяются под лицензией LGPL, что дает возможность производителям коммерческого ПО бесплатно использовать библиотеку без необходимости открытия исходных текстов своего продукта под лицензией GPL. Сегодня компания Nokia сделала еще один важный шаг - открыла публично доступный Git репозиторий, доступ к которому могут получить не только работники подразделения Qt Software, но и независимые энтузиасты, желающие принять участие в развитии проекта. Кроме библиотеки Qt в репозитории можно найти и сопутствующие проекты, такие как интегрированная среда разработки Qt Creator, Java интерфейс Qt Jambi и код развиваемый в рамках инициативы Qt Labs. Процесс принятия кода от сторонних разработчиков максимально упрощен, достаточно зарегистрировать аккаунт, согласиться с соглашением Qt Contribution Agreement, выполнить клонирование Git репозитория на локальную машину разработчика, произвести необходимые правки и выполнить слияние изменений. Далее код будет проверен ответственными разработчиками Qt и либо отвергнут с объяснением причины, либо принят и отправлен на утверждение в юридический департамент, после чего, если код не нарушает чужих авторских интересов, он будет принят в основной репозиторий. К сожалению среднее время проверки составляет около 2 недель. В будущем планируется интеграция с репозиторием системы трекинга ошибок, добавление системы автоматического тестирования отправляемого кода, оптимизация процесса утверждения изменений. Для заинтересованных независимых разработчиков подразделение Qt Software опубликовало специальный план развития проекта Qt, в котором освещены текущие задачи и основные направления, требующие доработки. Дополнительно сообщается о выпуске Qt Jambi 4.5.0_01 - Java интерфейса ко всем функциям тулкита Qt, который позволяет разработчикам напрямую использовать в своих Java программах API библиотеки Qt, включая поддержку WebKit и Phonon. Главные новшества новой версии - изменение лицензии на LGPL и передача проекта в руки независимого сообщества разработчиков, Qt Software своими силами больше развивать данный продукт не будет. В связи с этим ищутся добровольцы, готовые принять участие в проекте.
