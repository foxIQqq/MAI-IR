Релиз музыкального плеера DeaDBeeF 0.5.0

Представлен релиз DeaDBeeF 0.5.0, лёгкого музыкального плеера с удобным функциональным интерфейсом и поддержкой широкого спектра аудиоформатов и метаданных. Плеер написан на языке Си и может работать с минимальным набором зависимостей. Базовый интерфейс построен с использованием библиотеки GTK+. В новой версии существенно доработаны многие подсистемы плеера и добавлены следующие ключевые новшества: Поддержка вывода многоканального звука (5.1); Поддержка вывода звука с дискретизацией 8, 24, 32 и float32 бит; Новый API для создания DSP-плагинов (например, для преобразования форматов) и плагинов для обработки списков воспроизведения. Расширено API для создания VFS-плагинов, позволяющих напрямую манипулировать содержимым архивов; Добавлен новый плагин для проигрывания композиций напрямую из ZIP-архивов, без предварительной распаковки; Добавлен новый плагин для импорта и экспорта плейлистов в форматах m3u и pls; В меню добавлен пункт Edit->Sort с реализацией гибких возможностей сортировки; Улучшенные возможности редактирования метаданных, включая поддержку изменения значений нестандартных пользовательских полей в тегах и одновременное редактирование тегов в нескольких треках; Увеличена стабильность и производительность mp3-плагина; Новая опция для автоматического переименования списков воспроизведения при добавлении каталогов; Новый плагин Converter с поддержкой перекодирования треков из одного формата в другой, используя внешний кодировщик; Новый плагин Soundtouch для проведения изменения частоты дискретизации и других параметров звука; Поддержка работы с файлами больше 2 Гб; В OSD-нотификациях теперь поддерживается отображение обложки альбома; Исходные тексты проекта распространяются в рамках лицензии GPLv2. Пользователи Ubuntu могут установить плеер из специально подготовленного PPA-репозитория. Для пользователей остальных дистрибутивов подготовлен статически скомпонованный бинарный файл для 32-разрядных систем.
