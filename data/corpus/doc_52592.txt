Выпуск музыкального проигрывателя Audacious 4.0

Представлен релиз легковесного музыкального проигрывателя Audacious 4.0, ответвившегося в свое время от проекта Beep Media Player (BMP), являющегося форком классического плеера XMMS. Выпуск поставляется с двумя интерфейсами пользователя: на основе GTK+ и на основе Qt. Сборки подготовлены для различных дистрибутивов Linux и для Windows. Основные новшества Audacious 4.0: Осуществлён переход по умолчанию на интерфейс на основе Qt 5. Интерфейс на базе GTK2 больше не развивается, но оставлен в качестве опции, которую можно активировать во время сборки. В целом оба варианта близки по организации работы, но в Qt-интерфейсе реализованы некоторые дополнительные возможности, такие как более простой для навигации и сортировки режим просмотра списка воспроизведения. В WinAmp-подобном интерфейсе на базе Qt пока не вся функциональность готова, поэтому пользователи подобного варианта интерфейса, возможно, захотят продолжить использовать интерфейс на основе GTK2. Добавлена поддержка сортировки списка воспроизведения при клике на заголовки столбцов; Добавлена возможность перегруппировки столбцов списка воспроизведения через их перетаскивание мышью; Добавлены общие для всего приложения настройки громкости и размера шага; Реализована опция для скрытия вкладок со списком воспроизведения; Добавлен режим сортировки списка воспроизведения с показом каталогов после файлов; Реализованы дополнительные вызовы MPRIS для совместимости в KDE 5.16+; Добавлен плагин с трекером на базе OpenMPT; Добавлен новый плагин визуализации VU Meter; Добавлена опция для выхода в Сеть через прокси SOCKS; Добавлены команды для переключения на следующий и предыдущий альбомы; В редакторе тегов появилась возможность редактирования одновременно нескольких файлов; Добавлено окно с преднастройками эквалайзера; В плагин Lyrics добавлена возможность сохранения и загрузки текстов песен с локального накопителя; Для Qt портированы плагины MIDI, Blur Scope и Spectrum Analyzer; Расширены возможности плагина вывода через звуковую систему JACK; Добавлена опция для зацикливания файлов PSF.
