Вышла новая версия PHP фреймворка Zend Framework 1.9

Вышла новая версия PHP фреймворка Zend Framework 1.9. По сравнению с прошлым релизом отмечено более 700 изменений, среди которых реализация поддержки PHP 5.3 и включение новых средств для разработки "enterprise" приложений: добавлены компоненты Zend_Queue (API для сервисов управления очередями, подобных Apache ActiveMQ, MemcacheQ и Zend Job Queue) и Zend_Test_PHPUnit_Db (интегрируемый с Zend_Db модуль поддержки DBUnit для Zend_Test), значительно улучшены возможности Zend_Ldap, Zend_Rest_Route и Zend_Db_Adapter_Sqlsrv. Zend Framework сочетает в себе средства MVC (Model View Controller) разработки, прослойку для работы с базами данных, построенный на базе Lucene поисковый механизм, компоненты интернационализации (I18N), API для аутентификации и фильтрации поступающих от пользователя данных, классы для создания интерактивных web-приложений и т.д.
