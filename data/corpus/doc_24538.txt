Web-браузер Chromium портирован для FreeBSD

Энтузиасты портировали Web-браузер Chromium (Google Chrome) для ОС FreeBSD, в качестве основы была использована текущая тестовая версия браузера для платформы Linux. Готовый порт можно загрузить с данной страницы, бинарные сборки для FreeBSD 7.x доступны на сайте chromium.jaggeri.com. В настоящий момент порт содержит более 280 патчей. Напомню, что для Linux тестовую сборку Chromium в виде deb-пакета можно загрузить с сайта Google, пакеты для Fedora 10, 11 и 12 регулярно публикуются на сайте spot.fedorapeople.org, а пакеты для Ubuntu можно установить через специально поддерживаемый PPA-репозиторий.
