Поддержка старых дополнений в Firefox 3.5, советы по улучшению работы с табами

"Firefox add-ons: Which work in 3.5?" - анализ проблем с поддержкой старых дополнений в Firefox 3.5. Например, наблюдаются проблемы с Google Gears, Better Gmail 2 Drag & DropZones gDocsBar Google Gears (read above) Headup Minimize To Tray Enhancer Open in Google Chrome Prism Searchery Tab Mix Plus Tab Minus Update Notifier Работают нормально: Adblock Plus All-In-One Sidebar AVG Safe Search ColorfulTabs CoolIris (requires manual redownload or reinstallation) Customize Google FireFTP Flagfox FlashGot Forecastfox Enhanced FoxyTunes Greasemonkey Googlepedia Google Preview Google Toolbar IE Tab iMacros Java QuickStarter McAfee Site Advisor NoScript Personal Menu Read It Later Roboform Select-n-Go by Cleeki Session Manager Shareaholic Site Launcher SmarterFox SmoothWheel Ubiquity XMarks Более подробный статус поддержки различных расширений можно посмотреть на странице Add-on Compatibility Report. Следует заметить, что для некоторых неработающих с Firefox 3.5 расширений достаточно в install.rdf файле увеличить значение поля maxVersion. В другой заметке, "Disable tab tearing and six other Firefox 3.5 tab tweaks", представлены советы по улучшению работы с табами в Firefox 3.5: bug489729 - расширение для перетаскивания табов на рабочий стол; Для скрытия мешающего "+" в строке табов можно в userChrome.css указать ".tabs-newtab-button {display: none;}"; Для скрытия строки табов при открытии только одной страницы в меню Options/Tab нужно снять отметку на путнкте "Always show the tab bar"; Чтобы не закрывать окно после закрытия последнего таба в about:config можно указать browser.tabs.closeWindowWithLastTab = false; Для увеличения числа окон и табов подлежащих восстановлению при случайном закрытии в about:config следует увеличить значение полей browser.sessionstore.max_windows_undo и browser.sessionstore.max_tabs_undo; Для увеличение числа одновременно отображаемых табов в about:config нужно уменьшить значение browser.tabs.tabMinWidth.
