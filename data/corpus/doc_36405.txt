Доступна для тестирования первая бета-версия Ubuntu 13.04

Представлен первый бета-выпуск Ubuntu 13.04 (Raring Ringtail). Готовые тестовые образы созданы только для Edubuntu, Kubuntu, Lubuntu, UbuntuKylin (версия для пользователей из Китая), Ubuntu Cloud, Ubuntu Studio и Xubuntu. Готовые сборки Ubuntu и Ubuntu GNOME будут выпущены 28 марта, при формировании второй бета-версии. В конце месяца также ожидается подготовка обзора новшеств Ubuntu 13.04, который пока ограничивается списком планов. Из новшеств Kubuntu 13.04-beta отмечается обновление компонентов KDE до версии 4.10.1, поставка центра управления приложениями Muon Suite 2 RC, обновление браузера Rekonq до версии 2.2.1, интеграция полноэкранного меню приложений Homerun (см. скриншот ниже), новая система настройки многомониторных конфигураций, поставка шрифтов Oxygen Sans, переработка интерфейса инсталлятора, поддержка доступа к мобильным устройствам по протоколу MTP (используется в Android 3+ вместо эмуляции USB-накопителя), поддержка загрузки в режиме UEFI Secure Boot, поставка OpenJDK7 в качестве реализации Java по умолчанию, переход на ядро Linux 3.8. Из новшеств Xubuntu 13.04-beta отмечается увеличение размера iso-образа до 850 Мб, возвращение в поставку Gnumeric и GIMP, включение новой версии Parole 0.5.0 и обновление визуальной темы. Несмотря на заморозку Ubuntu 13.04 от внесения улучшений и переходе на стадию исправлений ошибок, отмечается, что до выпуска второй бета версии будет сделано одно исключение - в дистрибутив будет включена новая версия оболочки Unity 7. Из новшеств Unity 7 можно выделить включение в поставку новых устанавливаемых по умолчанию клиентских дополнений (Scopes), которые планировалось довести до 100, но будет чуть меньше. Будет добавлена поддержка умных областей (Smart Scopes) - серверных обработчиков, позволяющих представлять в Dash определённые виды локальной или внешней информации с определением порядка вывода областей с учётом их релевантности и востребованности пользователем. Например, при вводе в поиске "Битлз" в первую очередь будут выведены области с музыкой и видео, в которых будет отображён локальный и внешний контент, как из персональных облачных хранилищ пользователя, так и из сервисов подобных YouTube, Last.fm и Amazon. Кроме того, будут расширены средства для управления приватной информацией, которые позволят более точно контролировать какие сервисы использовать, например, определить фильтры для локального и глобального поиска (в Ubuntu 12.10 можно либо разрешить, либо запретить online-поиск целиком). Тем не менее, по умолчанию режим поиска во внешних сервисах, вызвавший волну критики, по прежнему будет включен.
