Ubuntu 17.10 перейдёт на Wayland по умолчанию и вернёт кнопки в правый угол окна

Разработчики Ubuntu рассказали на конференции GUADEC 2017 о ближайших планах по переводу дистрибутива на GNOME Shell. Кроме возвращения дистрибутива на использование рабочего стола GNOME, подтверждены ранее озвученные планы по задействованию по умолчанию в Ubuntu 17.10 системы Wayland вместо традиционного X-сервера. Для запуска X-приложений в окружении на базе Wayland будет применяться DDX-компонент XWayland. Полноценный сеанс на базе Xorg останется доступен в качестве опции. По мнению разработчиков переход на Wayland в Ubuntu 17.10 позволит на практике оценить возможность применения Wayland в грядущем LTS-выпуске Ubuntu 18.04, решение о переходе на Wayland для которого ещё не принято. Также подтверждены намерения вернуть кнопки сворачивания и закрытия окна в традиционный правый верхний угол окна после переноса в Ubuntu 10.04 кнопок в левый угол. Пользователи Unity уже привыкшие к кнопкам в левом углу смогут поменять их позицию при помощи GNOME Tweak Tool. Решение принято после изучения результатов опроса пользователей Ubuntu: 53.8% опрошенных предпочитают кнопки в правом углу. Привычный для пользователей Unity рабочий стол планируется построить на базе расширений GNOME, включая dash-to-dock. Но данное окружение решено максимально приблизить к поведению GNOME, например, не будет поставляться всплывающий поиск HUD, глобальное меню, специфичное для Ubuntu поведение Alt-Tab, меню обмена сообщениями, уведомления об изменении громкости, средства интеграции с ланчером, умные области (lenses & scopes) и другие свойственные для Unity решения. При желании пользователи смогут подобрать для себя похожую функциональность в форме расширений к GNOME Shell. Кроме специфичного для Ubuntu сеанса пользователям также будет предложен и классический сеанс GNOME Shell без каких-либо предустановленных расширений к GNOME Shell. Окружение рабочего стола, включающее намеченные для Ubuntu 17.10 изменения, планируют предложить для тестирования в ближайшие несколько недель.
