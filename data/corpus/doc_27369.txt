Экспериментальный выпуск MySQL Community Server 5.5.5-m3

Вышел MySQL Community Server 5.5.5-m3, пятый тестовый выпуск новой ветки MySQL 5.5, находящейся на третьей стадии (milestone 3) формирования нового релиза и имеющей качество бета-версии. Ветка MySQL 5.5 базируется на не выпущенной серии MySQL 5.4 и содержит ряд значительных улучшений, связанных с повышением масштабируемости и производительности. Ключевым изменением в версии 5.5.5-m3 является переход по умолчанию на использование хранилища InnoDB, вместо MyISAM. С обзором улучшений, добавленных в ветке MySQL 5.5 можно познакомиться в тексте прошлого анонса, об улучшениях добавленных в MySQL 5.4 можно прочитать здесь.
