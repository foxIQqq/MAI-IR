Доступен Calculate Linux 13.6

К 6-летию проекта выпущен релиз дистрибутива Calculate Linux 13.6. Доступны для загрузки следующие редакции дистрибутива: Calculate Directory Server (CDS), Calculate Linux Desktop с рабочим столом KDE (CLD) и XFCE (CLDX), Calculate Media Center (CMC), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS). Основные изменения: В несколько раз ускорена загрузка десктопов за счёт оптимизации планировки загрузки сервисов и включения параллельной загрузки. Добавлена поддержка шифрования домашних директорий пользователей. В CLDX добавлена поддержка одновременной работы в нескольких графических пользовательских сеансах. В качестве менеджера входа в сеанс CLDX теперь используется LightDM. Архиватор Xarchiver заменён на File Roller. Улучшена поддержка мультимедиа клавиатур. Уменьшен размер образа Calculate Media Center. Обновление ПО: Ядро Linux 3.9.6, KDE 4.10.4, LibreOffice 4.0.4.2, Chromium 28.0.1500.45 Особенности Calculate Linux: 100% совместимость с Gentoo с поддержкой бинарных репозиториев обновлений. Поставка специально разработанных в рамках проекта Calculate утилит для настройки, сборки и установки системы. Готовое клиент-серверное решение. Быстрое развёртывание на предприятии. Полноценная работа в гетерогенных сетях. Удобство администрирования. Rolling-release модель обновлений. Поддерживается интерактивная сборка системы - подготовка ISO-образа системы под свои задачи.
