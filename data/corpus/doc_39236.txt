Chromium адаптирован для дисплейного сервера Mir

Сотрудники компании Canonical выполнили портирование web-браузера Chromium для работы в окружении дисплейного сервера Mir. Портирование выполнено с использованием наработок проекта Ozone-wayland, в рамках которого разработчики из компании Intel адаптировали Chromium для работы с Wayland, заменив прослойку Ozone, используемую в Chromium для абстрагирования ввода и вывода. По аналогии с Ozone-wayland был подготовлен Ozone-mir, вариант прослойки Ozone для Mir.
