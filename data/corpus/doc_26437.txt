Opera прекращает выпуск браузера для платформы Solaris

Компания Opera приняла решение прекратить развитие сборки своего браузера для платформы Solaris, из Unix-подобных систем Opera будет выходить только для Linux и FreeBSD (не считая Mac OS X), которые по заявлению разработчиков остаются приоритетными для браузера платформами. В анонсе сообщается, что прекращение поддержки Solaris даст возможность ускорить стабилизацию ветки 10.5x для платформ Linux/FreeBSD и приблизит выход релиза для данных ОС. Кроме того, опубликован очередной тестовый снапшот Opera 10.53 для Linux/FreeBSD, в анонсе которого сказано о планах в ближайшее время выпустить первую бета-версию. Из до сих пор нерешенных проблем называется неработоспособность Java-плагина и проблемы с отображением truetype-шрифтов.
