Facebook Messenger неофициально доступен для Linux

Jack O’Conner из компании Facebook подготовил вариант неофициального десктоп-клиента Facebook Messenger для платформы Linux. Программа написана на языке Python с использованием модуля PyQt4. Код открыт под лицензией BSD. Программа пока заметно отстаёт по своим возможностям от официального клиента Facebook Messenger для платформы Windows. В настоящее время поддерживается ведение чата с друзьями из Facebook с разделением отдельных разговоров по вкладкам, выводятся уведомления и запросы друзей, обновляется статус друзей. Пользователи Ubuntu могут загрузить готовую сборку программы из PPA-репозитория: "sudo add-apt-repository ppa:nilarimogard/webupd8; sudo apt-get update; sudo apt-get install fbmessenger". Пользователи Arch Linux могут найти Facebook Messenger в AUR.
