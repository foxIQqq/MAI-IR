Новые версии дистрибутивов SystemRescueCd 1.5.2 и Parted Magic 4.10

Доступны обновленные версии двух популярных Live-дистрибутивов, предназначенных для восстановления системы после сбоя: SystemRescueCd 1.5.2 - специализированного LiveCD дистрибутива на основе Gentoo Linux, предназначенного для восстановления системы после сбоя. В качестве графического окружения используется Xfce. Размер iso-образа - 257 Мб. Из улучшений можно отметить использование LZMA-сжатия для образа RAM-диска initramfs.igz, обновление версии Linux ядра до версии 2.6.33.2, а squashfs до версии 4.0. Parted Magic 4.10 - небольшой LiveCD/LiveUSB дистрибутив (79 Мб) для восстановления системы после сбоя. Parted Magic построен на базе Linux From Scratch (LFS) и графического окружения Xfce. В минимальном режиме дистрибутив может работать на машинах с 48 Мб ОЗУ. В новой версии улучшена работа со SCSI-устройствами при загрузке, FAT32-разделы теперь по умолчанию монтируются в режиме UTF-8, обновлены версии программ grep 2.6.3, BusyBox 1.16.1, SimpleBurn 1.5.0, sshfs-fuse 2.2, Linux ядро 2.6.32.11, в комплект включены программы EncFS 1.5.2, GEncFS 1.0.0, GSSHFS 1.0.0, RLog 1.4, UNetbootin 429 и emelFM 0.6.0. Из базовых программ можно отметить: web-браузер Chromium, GParted, TestDisk, PhotoRec, Partition Image, Xarchive, Xfburn, ISO Master, Leafpad, Grsync. Из утилит командной строки присутствуют dd, ddrescue , cfdisk, fdisk, tcpdump. Имеется поддержка восстановления файловых систем ext2, ext3, ext4, btrfs, FAT16, FAT32, HFS, HFS+, JFS, NTFS, ReiserFS, Reiser4, XFS.
