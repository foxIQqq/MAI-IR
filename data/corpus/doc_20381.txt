Вышла вторая альфа-версия LiveCD Chakra

Доступна вторая альфа версия The Chakra Project, Live CD дистрибутива, основанного на Arch Linux с поставляемым по умолчанию графическим окружением KDE 4. Была проделана большая работа по устранению найденных ошибок и дальнейшего совершенствованию проекта. Среди изменений разработчики выделяют улучшенную поддержку оборудования, обновленную страницу локализации в инсталляторе Tribe, LZMA-сжатие и USB образ для установки с флешки. Создатели Chakra обращаются к сообществу с просьбой протестировать этот релиз максимально широко, по возможности, на реальном оборудовании. Регистрировать ошибки ( патчи - приветствуются) и делиться идеями по дальнейшему развитию проекта можно здесь. Предварительно желательно свериться с информацией в Wiki. Изменения относительно Alpha 2 включают: Новые функции KDE 4.2.0 плюс дополнительные приложения и компоненты Koffice 2 beta6 Amarok 2.0.1 Kdenlive 0.7.2.1 Изменения в инсталляторе Tribe В имени пользователя и машины допустимы только цифры, буквы в нижнем регистре и дефисы На странице настройки загрузчика дополнительный функционал по умолчанию скрыт Уже смонтированные носители перед установкой размонтируются (bugs #38, #74) Исправлено падение, если доступных для установки разделов не было найдено (bug #39) Использование swap-раздела опционально Все обнаруженные модули добавляются в список модулей в rc.conf Вместо просто имени пакета показывается название проприетарной лицензии целиком Grub принудительно больше не устанавливается (bug #108) Добавляются только уникальные разделы и точки монтирования Поправлены страницы некоторых этапов установки (bugs #13, #23) Новая страница выбора локализации (основана на карте Marble) Генерируются только локали, указанные в /etc/locale.gen Изменения в системе LiveCD Улучшения в установщике X.org Настройка устройств ввода на основе правил HAL Свободные драйвера теперь используются по умолчанию Запасной вариант настройки не-DDC/неисправных/нераспознанных дисплеев Более продуманный выбор видеокарты для несвободных драйверов Усовершенствованное конфигурирование оборудования (хотя впереди ещё долгий путь) Более надёжное определение размера памяти и диска Определение целевой платформы теперь тоже должно работать LZMA-сжатие файловой системы livecd Grub был заменён на isolinux/syslinux для лучшей совместимости Splashy удалён, теперь вместо него используется стандартная для Arch процедура загрузки Скриншоты можно посмотреть здесь.
