Тестирование неофициальной сборки дистрибутива ALT Linux Lite

Выпущена первая публичная бета-версия неофициальной сборки дистрибутива ALT Linux Lite (на базе пакетов из 5.0/branch). При создании сборки поставлены цели по созданию дистрибутива дружественного к начинающему пользователю, содержащего сбалансированный и обособленный набор приложений, дающего возможность использования дистрибутива без загрузки дополнений из сети, 99% уровень локализации. Некоторые приложения, входящие в комплект: XFce 4.6.1, набор полезных дополнений для файлового менеджера thunar, OpenOffice.org, Brasero для записи дисков, словари StarDict, Evince для просмотра PDF, Firefox с плагинами для Adobe Flash, Sun Java и totem-gstreamer, Thunderbird, Pidgin, NetworkManager для управления сетью, видео-проигрыватель Totem, аудио-проигрыватель Sonata, GIMP, Inkscape, Gqview.
