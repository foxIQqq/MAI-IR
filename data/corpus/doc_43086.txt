Доступен дистрибутив OpenIndiana 2015.10, продолжающий развитие OpenSolaris

Состоялся выпуск проекта OpenIndiana 2015.10, в рамках которого независимым сообществом развивается полностью свободный дистрибутив, пришедший на смену бинарному дистрибутиву OpenSolaris, развитие которого было прекращено компанией Oracle. Для загрузки сформировано два вида iso-образов - редакция с консольными приложениями (646 Мб) и сборка с графическим окружением GNOME (1.2 Гб). OpenIndiana предоставляет пользователю рабочее окружение, построенное на базе свежего среза кодовой базы проекта Illumos. Непосредственно разработка технологий OpenSolaris продолжается проектом Illumos, в котором развивается ядро, сетевой стек, файловые системы, драйверы, а также базовый набор пользовательских системных утилит и библиотек. Основные изменения в OpenIndiana 2015.10: Код системы управления пакетами IPS синхронизирован с веткой компании Everycity, в которую включены обновления и исправления, полученные от Oracle. Из добавленных в IPS новшеств отмечается поддержка Python 2.7 и учёт изолированных зон при работе. Скрипты сборки дистрибутива переведены на использование Python 2.7; Проведена подготовка к поддержке загрузчиков, отличных от Grub; В текстовом инсталляторе обеспечено создание отдельной ФС для /var. Текстовый инсталлятор включен в сборки с GUI из-за запущенного состояния с графическим инсталлятором, оставшимся без сопровождающего (GUI-инсталлятор написан на Си, а текстовый на Python и более прост для сопровождения); Версия интерпретатора Perl в базовой системе обновлена для выпуска 5.22 (поддержка Perl 5.16 пока сохранена, а 5.10 прекращена); Sun JDK/JRE заменён на OpenJDK. В состав системы входит OpenJDK 1.7.76; Обновлены версии компонентов графического стека: X.org-Server 1.14.7 (обновлению до более новой версии мешает привязка драйвера radeon к KMS), freetype 2.5.5, драйверы NVIDIA 340.93, Qt 4.8.7, Mesa 10.5.9, Compiz 0.8.10; В состав включен звуковой сервер Pulseaudio 1.1. Удалён ранее используемый апплет управления громкостью gnome-volume-applet; Обновлены версии пользовательских программ, в том числе Thunderbird 31.8.0, GIMP 2.8.14, Inkscape 0.48.5, Ntfsprogs 2015.3.14, Synergy 1.7.4, bash 4.2.53, Midnight Commander 4.8.14. Интегрирована библиотека управления цветом Lcms2; Обновлены средства для разработчиков: GCC 4.8.5, Python 2.7.10, Ruby 2.2.3, PHP 5.5.29, Lua 5.2.4, Cmake 2.8.12.2, Nasm 2.11.8, Yasm 1.3.0, GNU binutils 2.25.1, Node.js 0.12.7, Git 1.9.4. Mercurial 3.4.2, Subversion 1.7.20, GNU Patch 2.7.5, Boost 1.58.0; Обновлены серверное ПО: Tomcat 8.0.24, Apache 2.4.16 (удалён Apache 1.3), mod_jk 1.2.41, mod_perl 2.0.9, Nginx 1.8.0, Lighttpd 1.4.36, Mariadb 5.5.44 (удалён MySQL 5.1), percona-server 5.6.16.64.2, OpenLDAP 2.4.42, Samba 4.1.19 , ISC Bind 9.9.7-P3, ISC DHCPD 4.2.8, KVM 20150903 Joyent, OpenVPN 2.3.8; Подготовлен SFE-репозиторий для OpenIndiana.
