Первый выпуск рабочего стола MaXX, адаптации IRIX Interactive Desktop для Linux

После десяти лет разработки представлен выпуск рабочего стола MaXX 1.0.0, в рамках которого предпринята попытка воссоздать пользовательскую оболочку IRIX Interactive Desktop (SGI Indigo Magic Desktop) с использованием технологий Linux. Изначально IRIX Interactive Desktop поставлялся на графических рабочих станциях производства SGI, оснащённых операционной системой IRIX, пик популярности которых пришёлся на конец 1990-х годов и производство которых велось вплоть до 2006 года. Оболочка работает поверх оконного менеджера 5dwm 2.2 и библиотек SGI-Motif 2.3.1. В качестве файлового менеджера применяется ROX-Filer. Работа графического интерфейса реализована с задействованием OpenGL для аппаратного ускорения и визуальных эффектов. Кроме того, для ускорения работы и снижения нагрузки на CPU организована многопоточная обработка операций и вынос вычислительных задач на плечи GPU. GPU-оптимизации пока доступны только для видеокарт NVIDIA. Рабочий стол не зависит от экранного разрешения и использует векторные пиктограммы. Поддерживается расширение рабочего стола на несколько мониторов. Сборки пока подготовлены только для Fedora 25 (x86_64, ia64), но ожидаются и для Debian 8, Ubuntu 16.04, RHEL 7 и Linux Mint. В дальнейшем запланирован ежеквартальный выпуск новых значительных версий рабочего стола c ежемесячными корректирующими выпусками. Разработка ведётся по специальному соглашению с компанией SGI, разрешающему полное воссоздание всех функций IRIX Interactive Desktop для платформы Linux на архитектурах x86_64 и ia64. MaXX Desktop поставляется в двух редакциях: общедоступной Community Edition (CE) и коммерческой Professional Edition (PE). Коммерческая версия отличается применением дополнительных оптимизаций и расширенных возможностей рабочего стола. Коммерческая редакция в основном ориентирована на клиентов, продолжающих использование рабочих станций SGI на базе IRIX и желающих развернуть похожие окружения в системах на базе Linux. Исходные тексты доступны по специальному запросу и представляет собой смесь из проприетарного кода (в соответствии с требованиями соглашения с SGI) и кода под различными открытыми лицензиями.
