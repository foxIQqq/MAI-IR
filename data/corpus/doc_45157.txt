JavaScript, Font-Awesome и Microsoft лидируют в статистике GitHub

GutHub опубликовал отчёт с анализом статистики и тенденций за последний год. В настоящее время в GitHub представлены 5.8 млн активных пользователей, 331 тысяча организаций и 19.4 млн репозиториев. Рейтинг, отражающий популярность применения в репозиториях языков программирования, как и прошлые три года возглавляет язык JavaScript для которого за год зафиксировано 1.6 млн pull-запросов, что на 97% больше, чем в прошлом году. На втором месте Java (763 тысяч), на третьем Python (740 тысяч), на четвёртом Ruby (740 тысяч), на пятом PHP (478 тысяч), на шестом C++ (330 тысяч). Лидером рейтинга огранизаций, в проектах которых больше всего участников, возглавила компания Microsoft, в открытых проектах которой зафиксировано 16419 участников. На втором месте Facebook - 15682 участников, на третьем Docker - 14059, на четвёртом Angular - 12841, на пятом Google - 12140. Рейтинг репозиториев с наибольшим числом участников возглавляет Font-Awesome - 10654 участника, на втором месте Docker - 8253, на третьем npm - 7041, на четвёртом patchwork - 6806, на пятом Facebook react-native - 6250, на шестом Microsoft vscode - 5855. Интересно, что число непосредственных разработчиков для лидирующих проектов относительно невелико (для Font-Awesome - 75, vcode - 163, для сравнения react-native - 970, docker - 1496), но к формирующим рейтинг участникам также причислены авторы сообщений об ошибках (issue), комментариев и пользователи, принимающие участие рецензировании. Т.е. высокий рейтинг некоторых проектов указывает не столько на активность разработки, сколько на интенсивность обсуждений.
