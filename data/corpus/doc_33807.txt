Вышла редакция Linux дистрибутива Salix 13.37 с рабочим столом MATE

Представлена редакция Linux-дистрибутива Salix 13.37, базирующаяся на десктоп-окружении MATE 1.2 (форк GNOME 2). Дистрибутив на 100% совместим со Slackware Linux 13.37 и исповедует подход "одно приложение на одну задачу". Salix развивается создателем дистрибутива Zenwalk Linux, покинувшим проект в результате конфликта с другими разработчиками, отстаивавшими политику максимальной похожести на Slackware. Размер загрузочного iso-образа составляет 610 Мб. Для управления пакетами задействован пакетный менеджер gslapt, являющийся эквивалентом slapt-get, в качестве графического интерфейса для установки программ из SlackBuilds в дополнение к gslapt поставляется программа Sourcery, являющаяся специально разработанным в рамках проекта Salix фронтэндом к slapt-src. В настоящий момент в SlackBuilds репозитории доступно более 2600 пакетов. Стандартные инструменты для управления пакетами Slackware модифицированы для обеспечения поддержки Spkg, что позволяет использовать внешние приложения, подобные sbopkg, без нарушения совместимости со Slackware. Как и другие редакции Salix версия с окружением MATE снабжена штатным инсталлятором, поддерживающим три режима установки: Полная - все приложения для десктоп окружения MATE, LibreOffice, Claws-mail, Pidgin, GIMP; Основная - минимальный состав MATE, файловый менеджер Caja, web-браузер Firefox, пакетный менеджер gslapt, SlackBuild-менеджер Sourcery и системные программы Salix; Базовая - только минимальный набор консольных утилит, подходит для создания серверов.
