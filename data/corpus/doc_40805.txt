Релиз дистрибутива Lunar Linux 1.7.0, развивающего пакетный менеджер lin

После четырёх лет разработки представлен выпуск дистрибутива Lunar Linux 1.7.0, отличающегося использованием специфичного пакетного менеджера lin, выполняющего сборку программы из исходных текстов для каждого устанавливаемого пакета. Пакеты выполнены в виде модулей, представленных в иерархическом каталоге Moonbase, напоминающей систему портов FreeBSD. Установочный образ занимает 361 Мб и содержит базовое окружение, пригодное только для сборки дополнительных пакетов из Moonbase. Сборка программ не представляет сложности, например, для установки пользовательского окружения на базе KDE4 достаточно выполнить "lin kde4", GNOME - "lin gnome3", Xfce - "lin xfce4". В новом выпуске осуществлён переход с системы инициализации sysvinit на systemd, добавлена поддержка файловой системы Btrfs, задействованы новые версии системных компонентов (ядро Linux 3.16.3, gcc 4.9.1, glibc 2.19), добавлена поддержка загрузчика GRUB2 и образа начальной загрузки initrd, переработан инсталлятор, обновлены модули с предлагаемыми для установки программами.
