Выпуск программы для управления фотографиями digiKam 6.2

После 4 месяцев разработки опубликован релиз программы для управления коллекцией фотографий digiKam 6.2.0. В новом выпуске закрыто 302 отчёта об ошибках. Установочные пакеты подготовлены для Linux (AppImage), Windows и macOS. Основные новшества: Добавлена поддержка форматов RAW-изображений, предоставляемых камерами Canon Powershot A560, FujiFilm X-T30, Nikon Coolpix A1000, Z6, Z7, Olympus E-M1X и Sony ILCE-6400. Для обработки RAW-изображений задействована библиотека libraw 0.19.3, предоставляющая поддержку более 1000 вариантов форматов RAW; Добавлена поддержка библиотеки Exiv2 0.27.2, используемой для работы с метаданными в файлах с изображениями; Встроенный проигрыватель видео обновлён для поддержки фреймворка QtAv 1.13.0; Добавлена поддержка отрисовки пиктограмм содержимого альбома для HiDPI экранов с разрешением 4K; Подготовлены 32- и 64-разрядные переносимые сборки для Windows, для запуска digiKam при использовании которых достаточно распаковать содержимое архива в домашний каталог и запустить приложение digiKam или Showfoto, не прибегая к традиционным установщикам.
