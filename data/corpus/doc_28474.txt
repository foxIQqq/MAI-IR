Организация Apache Software Foundation опровергла слухи о наличии спорного кода в Apache Harmony

После публикации компанией Oracle дополнительных материалов, касающихся нарушения связанной с Java-технологиями интеллектуальной собственности в платформе Android, в сети появилась информация о том, что фигурирующие в деле части библиотеки классов были взяты компанией Google из проекта Apache Harmony. Организация Apache Software Foundation опубликовала официальное заявление, в котором опровергла связанные с этим спекуляции. По данным Apache Software Foundation ни один из показанных компанией Oracle файлов с исходными текстами не был заимствован из Apache Harmony, несмотря на ссылки на проект Apache в заголовках. Данное упоминание связано исключительно с лицензией Apache, которую используют многие проекты, в том числе и продукты Google. Проверить данное заявление достаточно легко, весь код проекта Apache доступен в SVN-репозитории и любой может убедится, что, например, класс PolicyNodeImpl.java не является классом Apache Harmony, несмотря на то, что он распространяется под лицензией Apache.
