Вышло обновление MeeGo 1.1.1, решающее проблему с русскоязычной раскладкой клавиатуры

Представлен первый сервис-выпуск для представленной в октябре мобильной платформы MeeGo 1.1. Обновление доступно для базового компонента Core Software Platform, редакции для нетбуков Netbook UX и версии для автомобильных информационно-развлекательных систем - In-Vehicle Infotainment (IVI) UX. Кроме исправления ошибок, в новом выпуске можно отметить следующие улучшения: Добавлена возможность использования русскоязычной раскладки клавиатуры. Несмотря на наличие опции для включения русскоязычной раскладки, переключение раскладок не работало; В Linux-ядре улучшена поддержка оборудования автомобильных информационно-развлекательных систем (IVI-систем), например, платы Russellville IA IVI; Налажена работа с протоколом MSN для мгновенного обмена сообщениями; Устранены проблемы с отображением статуса на панели некоторых нетбуков; Исправлены ошибки в приложении для осуществления звонков; Обеспечена возможность работы браузера в полноэкранном режиме; Отключена по умолчанию функция автозагрузки медиа-носителей в Nautilus.
