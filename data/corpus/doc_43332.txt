Релиз дистрибутива ClearOS 7.1, основанный на пакетной базе RHEL

Представлен Linux-дистрибутив ClearOS 7.1, построенный на пакетной базе Red Hat Enterprise Linux 7.1 с элементами из проектов Fedora и CentOS. ClearOS ориентирован для применения в качестве серверной ОС в небольших организациях, в том числе для связывания удаленных офисов в единую сетевую инфраструктуру. Для загрузки доступны установочные образы размером 797 Мб и 373 Мб, собранные для архитектуры x86_64. Среди особенностей дистрибутива выделяется полностью открытая архитектура, исключающая зависимость от одного проприетарного вендора, низкая стоимость обслуживания, возможность наращивания компонентов и расширенная поддержка. В состав ClearOS входят средства для защиты локальной сети, отслеживания внешних угроз, фильтрации web-контента и спама, организации обмена сообщениями и файлами, развёртывания сервера централизованной авторизации и аутентификации на базе LDAP, использования в качестве контроллера домена для Windows ПК. При использовании в качестве сетевого шлюза поддерживаются сервисы DNS, NAT, прокси, OpenVPN, PPTP, управления пропускной способностью и организации доступа в интернет через нескольких провайдеров. Настройка всех аспектов работы дистрибутива и управление пакетами производится через специально созданный web-интерфейс. В новой версии: Переход на пакетную базу RHEL 7.x, Поставка модуля с реализацией замены Microsoft Active Directory на базе технологий Samba 4; Обновление компонента Microsoft Active Directory Connector; Поддержка смены тем оформления web-интерфейса с возможностью адаптации для мобильных устройств; Динамическая панель мониторинга за состоянием системы; Обновление движков блокирования спама и вирусов, а также систем обнаружения и предотвращения атак; Реализация фреймворка для вывода уведомлений; Возможность локализации интерфейса. Предоставление русскоязычного перевода элементов интерфейса; Поддержка файловых систем XFS и Btrfs; Расширение средств виртуатизации и улучшение совместимости с гипервизорами KVM и Hyper-V.
