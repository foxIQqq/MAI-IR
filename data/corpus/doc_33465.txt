Apache Rave принят в число первичных проектов Apache

Организация Apache Software Foundation объявила о придании проекту Apache Rave статуса первичного проекта Apache, после успешной проверки в "инкубаторе". В рамках проекта Apache Rave развивается Java-платформа для размещения, обслуживания и агрегации социальных виджетов и сервисов. Rave может использоваться на интернет и интранет порталах в роли движка для взаимодействия с социальными сетями, а также для добавления на сайты контекстно-зависимых блоков и средств для организации общения, используя универсальный набор инструментов, не привязанных к конкретному социальному сервису. Среди поддерживаемых стандартов отмечаются Activity Streams, OpenSocial и W3C Widgets. Перевод Apache Rave в разряд первичных означает, что продукт и развивающее его сообщество подтвердили способность следования принципам разработки Apache и теперь готовы для самостоятельного существования, не требующего дополнительного надзора. Для управления разработкой и координации подготовки релизов Apache Rave будет выбран комитет, состоящий из активных участников проекта.
