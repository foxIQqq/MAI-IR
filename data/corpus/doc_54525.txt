Выпуск музыкального проигрывателя Audacious 4.1

Представлен релиз легковесного музыкального проигрывателя Audacious 4.1, ответвившегося в свое время от проекта Beep Media Player (BMP), являющегося форком классического плеера XMMS. Выпуск поставляется с двумя интерфейсами пользователя: на основе GTK и Qt. Сборки подготовлены для различных дистрибутивов Linux и для Windows. Основные новшества Audacious 4.1: По умолчанию включена сборка двух вариантов интерфейса, на базе Qt и GTK. Переключение между интерфейсами может быть осуществлено на экране с настройками, без необходимости редактирования файла .desktop. В прошлой версии Audacious перешёл по умолчанию на интерфейс на основе Qt 5, но в ответ на пожелания разработчиков дистрибутивов Debian и Fedora включена сборка обоих интерфейсов. Оба варианта близки по организации работы, но в Qt-интерфейсе реализованы некоторые дополнительные возможности, такие как более простой для навигации и сортировки режим просмотра списка воспроизведения. На платформе Windows по умолчанию оставлен только интерфейс на основе Qt. Обеспечена полноценная поддержка сборочной системы Meson. Добавлена начальная поддержка Qt 6. Для интерфейса на базе Qt реализован новый плагин для настройки горячих клавиш. Добавлена возможность использования колеса мыши для перемещения по треку в интерфейсе на базе Qt. Вместо ModPlug задействован новый плагин с трекером на базе OpenMPT. Предоставлена возможность скрытия показа обложки альбома на информационной панели. Обеспечено выделение жирным шрифтом воспроизводимого трека. В списке воспроизведения сделан более заметным индикатор drag&drop-операций. В окне с информацией о композиции показано число звуковых каналов. В Channel Mixer реализована поддержка преобразования из 2 каналов в 4. Добавлена возможность назначения произвольного таймаута для скрытия уведомлений.
