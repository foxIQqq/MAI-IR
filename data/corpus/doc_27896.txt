Релиз DVD-сборки Ubuntu - Super OS 10.04

Для загрузки доступен релиз Linux-дистрибутива Super OS 10.04, который представляет собой DVD-сборку Ubuntu 10.04, построенную по принципу "все в комплекте". Например, в состав включены дополнительные мультимедиа приложения (VLC), полезные открытые программы (Firefox, KDE-библиотеки, Ubuntu Tweak, GParted, cd2usb), кодеки (MP3, QuickTime, Real video, WMV, DivX, FLV, Xvid) и популярные проприетарные программы (Google Chrome, Opera, Skype, Adobe Flash, aMSN, Java). В комплект Super OS также включены дополнительные средства для упрощения установки и запуска программ: RUNZ (самодостаточные пакеты со всеми зависимыми библиотеками) и App Runner (апплет для универсального запуска программ с привязкой к расширению). Размер установочного iso-образа - 1 Гб. Особенности нового выпуска: Поддержка сборки для 64-разрядных систем, в комплект которой включены библиотеки Ia32-libs, позволяющие организовать запуск 32-разрядных программ в 64-разрядном окружении; В поставку включен инсталлятор Wubi, позволяющий устанавливать дистрибутив не выходя из Windows; Для записи на USB Flash в состав включена GUI-утилита cd2usb; Пакет Oracle Java заменен на OpenJDK; Обновление версий программ: Firefox 3.6.8, Google Chrome 6, Opera 10.61 и т.д.; С целью экономии места из поставки удалены некоторые библиотеки, связанные с KDE и Qt; Из поставки удален пакет Moonlight.
