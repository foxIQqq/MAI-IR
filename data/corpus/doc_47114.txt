Релиз дистрибутива для образовательных учреждений Ubuntu EducationPack 16.04

Проект UALinux выпустил сборку Ubuntu EducationPack 16.04, ориентированную на использование в учебных учреждениях. Дистрибутив основан на пакетной базе Ubuntu 16.04 и предоставляет из коробки полностью настроенное решение для образовательного учреждения, готовое для использования сразу после установки. Сборки поставляются для 32-х и 64-разрядных систем (4.1 Гб). Установочный диск содержит более 140 предустановленных программ для образования, науки, программирования и организации учебного процесса. Ещё более 80 программ доступно на дополнительном диске. Интерфейс основан на окружении на GNOME Flashback. В комплект входит LTSP-сервер, позволяющий использовать старые компьютеры в качестве тонких клиентов или бездисковых рабочих станций, загружающихся из сети без установки.
