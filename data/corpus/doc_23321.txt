Открыт код GT portalBase, платформы для разработки web-приложений на базе PostgreSQL

Компания GT webMarque объявила о переводе своей проприетарной платформы GTpB (GT portalBase) в разряд свободных продуктов и опубликовала исходные тексты под лицензией GPL. GTpB представляет собой платформу для быстрой разработки промышленных web-приложений (интранет порталов), направленных на решение задач анализа данных и поддержание информационной инфраструктуры коммерческих компаний. Скелет платформы составляют: PostgreSQL 8.4, Java 5+, Hibernate и Apache Tomcat. Из базовых возможностей GTpB можно отметить: Визуальная среда для построения прототипов приложений и web-форм; GUI-интерфейс для управления СУБД с поддержкой создания таблиц, представлений, полного доступа к содержимому БД; Простая система для расширенной фильтрации выводимых данных; Генератор суммарных отчетов; Средства импорта и экспорта; Возможность настройки прав доступа и ролей; Средства для рефакторинга БД. Расширенные функции: Поддержка отображения интерфейса во всех популярных web-браузерах. Режимы просмотра для iPhone и телефонов на базе Google Android; Средства для интеграции с сервисом youTube; Автоматический учет истории изменений для загруженных в БД документов; Поддержка генерации облака тэгов; Возможность группировки представлений и модулей в тематические секции (План/Сделать/Проверить/Реализовать, Маркетинг/Продажи/Финансы); Поддержка дополнительных API для интеграции с google maps и выполнения поиска по телефонному номеру; Автоматическое задействование операций объединения таблиц (JOIN) и создания полных представлений (full view); Функции контроля безопасности и проведения аудита.
