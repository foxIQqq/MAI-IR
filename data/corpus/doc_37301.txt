Проект Ubuntu объявил о доступности альфа-версии 13.10

Разработчики Ubuntu Linux объявили о начале альфа-тестирования выпуска 13.10 "Saucy Salamander". В соответствии с новым планом разработки, уже опробованном при подготовке Ubuntu 13.04, отдельные альфа-сборки не будут сформированы, вместо них для тестирования предлагается использовать ежедневные экспериментальные сборки. Готовые тестовые образы созданы только для Lubuntu, Ubuntu GNOME и Kubuntu. Бета-версия Ubuntu 13.10, запланированная на 26 сентября, будет сформирована в виде полноценной сборки. Релиз намечен на 17 октября. Из доступных на настоящем этапе разработки изменений отмечается обновление версий программ, использование ядра Linux 3.9.7. В Kubuntu отмечен переход на KDE 4.11 Beta, задействование нового приложения Muon Discover для поиска и установки программ. В Lubuntu осуществлена замена браузера Chromium на Firefox, удалён хранитель экрана XScreensaver и обеспечена возможность использования технологии ZRam (размещённое в ОЗУ блочное устройство на которое производится своппинг со сжатием). В Ubuntu GNOME задействовано десктоп окружение GNOME 3.8 (в 13.04 поставлялся GNOME 3.6), предоставлена возможность выбора сеанса с реализацией классического рабочего стола в стиле GNOME 2. Из общих планов развития Ubuntu 13.10 можно отметить задействование по умолчанию дисплейного сервера Mir, поставка пользовательского окружения Unity 7 c реализацией умных областей (Smart Scopes) и средств для управления приватной информацией. Smart Scopes представляют собой серверные обработчики, позволяющие представлять в Dash определённые виды локальной или внешней информации с определением порядка вывода областей с учётом их релевантности и востребованности пользователем. Cредства для управления приватной информацией позволят более точно контролировать какие сервисы использовать, например, определить фильтры для локального и глобального поиска. Кроме того, не исключена миграция с Firefox на Chromium в базовой поставке.
