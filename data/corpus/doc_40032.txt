Вышел TeX Live 2014

Доступен релиз популярного LaTeX-дистрибутива TeX Live 2014, созданного в 1996 году на базе проекта teTeX. TeX Live является наиболее простым способом для развертывания инфраструктуры для подготовки научной документации, независимо от используемой операционной системы. Для загрузки доступна DVD-сборка (2,6 Гб) TeX Live 2014, которая содержит рабочее Live-окружение, полный набор установочных файлов для различных операционных систем, копию репозитория CTAN (Comprehensive TeX Archive Network), подборку документации на разных языках (включая русский). Из новшеств можно отметить: Пакет TeX обновлён до версии 3.14159265 (прошлое обновление было в 2008 году), в которой возвращено отображение сведений о предзагруженном формате и появилась корректная обработка нулевых управляющих последовательностей (\\csname\\endcsname). В pdfTeX добавлен параметр \\pdfsuppresswarningpagegroup для подавления вывода предупреждений, реализованы новые примитивы \\pdfinterwordspaceon, \\pdfinterwordspaceoff и \\pdffakespace для управления расстановкой межсловных пробелов; В LuaTEX внесены исправления, связанные с загрузкой шрифтов и слогоделением. Добавлен новый вариант движка luajittex, использующий JIT-компилятор Lua; В XeTEX обеспечена поддержка единого набора форматов изображений на всех платформах; В MetaPost добавлена новая десятичная численная система (numbersystem decimal), которая теперь поддерживается вместе с внутренней системой numberprecision. Решены проблемы с выводом в форматах SVG и PNG; Будет прекращена поддержка команды pstopdf ConTEXt как отдельной утилиты из-за пересечения со сторонним приложением pstopdf. Для вызова pstopdf следует использовать "mtxrun --script pstopdf"; Существенно переработан пакет psutils, отдельные утилиты (fix*, getafm, psmerge, showchar) теперь доступны только из директории scripts/. Добавлен новый скрипт psjoin; Добавлена поддержка платформы x86_64-cygwin. Прекращена поддержка mips-irix. Не гарантируется сохранение работоспособности в Windows XP; Компоненты для поддержки некоторых других платформ исключены из состава DVD, но доступны для загрузки по сети.
