Релиз операционной системы Chrome OS 53

Компания Google опубликовала выпуск операционной системы Chrome OS 53, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 53. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 53 доступна для всех актуальных моделей Chromebook, за исключением Chromebook Pixel (2015), Acer R11, ASUS Flip C100PA, HP CB1 и Acer C7. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache. Основные изменения в Chrome OS 53: Оформление обзорного режима переведено на стиль Material design. Индикатор активности приложений теперь выполнен в виде точки под пиктограммой; Интерфейс для просмотра истории (chrome://history) переведён на Material design; Добавлен Storage Manager, позволяющий оценить свободное место на диске, а также объём загрузок и Offline-файлов Google Drive (для включения chrome://flags/#enable-storage-manager); Для web-приложений Chrome Apps добавлена поддержка сервиса аутентификации SAML SSO (Security Assertion Markup Language Single Sign-On); Добавлен API для включения отслеживания активности в приложениях, работающих в режиме интернет-киоска; Для доступа к истории индикатор уведомлений теперь всегда отображается в строке статуса, даже когда нет непрочитанных уведомлений; Упрощён способ перехода на экспериментальные выпуски Chrome OS. В терминале crosh, вызываемом по Ctrl+Alt+T, теперь достаточно набрать команду "live_in_a_coal_mine" и ответить "y"; Добавлена горячая клавиша для быстрого перехода в спящий режим - Shift+Search+L (полезно для устройств Chromebox, Chromebase и Chromebit, в которых нельзя просто захлопнуть крышку ноутбука); Для людей с проблемами со слухом добавлен режим вывода звука сразу на все доступные громкоговорители (превращение стерео в моно); Расширены возможности индикатора раскладки клавиатуры, который позволяет быстро переключиться между языками и запустить виртуальную клавиатуру для ввода emoji, рукописного текста или включения голосового ввода.
