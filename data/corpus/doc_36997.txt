Вышел Chrome 27 для платформы Android

Спустя день с момента выпуска версии браузера Chrome 27 для стационарных систем анонсирована мобильная версия Chrome 27 для платформы Android. Среди добавленных в Chrome 27 для Android изменений: Поддержка полноэкранного режима в версии для телефонов. Панель с адресной строкой теперь автоматически скрывается при прокрутке страницы вниз (ранее основные нарекания вызывало постоянное наличие адресной строки на экрана без возможности её скрытия); Упрощение поиска - при вводе поискового запроса в адресной строке, ключевые слова остаются видимы и доступны для редактирования, а не замещаются URL поисковой системы; Поддержка подключения собственных SSL-сертификатов; При длительном удержании кнопки "назад" в интерфейсе для планшетов теперь выводится история работы с вкладкой.
