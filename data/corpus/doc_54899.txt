Выпуск TeX-дистрибутива TeX Live 2021

Подготовлен релиз дистрибутива TeX Live 2021, созданного в 1996 году на базе проекта teTeX. TeX Live является наиболее простым способом развертывания инфраструктуры для подготовки научной документации, независимо от используемой операционной системы. Для загрузки сформирована DVD-сборка (4.4 ГБ) TeX Live 2021, которая содержит рабочее Live-окружение, полный набор установочных файлов для различных операционных систем, копию репозитория CTAN (Comprehensive TeX Archive Network), подборку документации на разных языках (включая русский). Из новшеств можно отметить: В TeX и Metafont включены изменения, предложенные Дональдом Кнутом и нацеленные на решение некоторых малоизвестных проблем, связанных с обработкой '\tracinglostchars' и '\tracingmacros'. В LuaTeX интерпретатор Lua обновлён до версии 5.3.6. В интерпретаторе языка логического описания графических объектов MetaPost добавлена переменная окружения SOURCE_DATE_EPOCH для обеспечения повторяемых сборок. В pdfTeX реализованы новые примитивы "\pdfrunninglinkoff" и "\pdfrunninglinkon" для отключения генерации ссылок и нижних колонтитулов. Прекращена поддержка библиотеки poppler - в pdfTeX теперь всегда используется libs/xpdf. В Dvipdfmx по умолчанию включён режим безопасного (dvipdfmx-unsafe.cfg) запуска Ghostscript
