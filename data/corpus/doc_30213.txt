Релиз web-браузера Opera 11.10

Представлен релиз web-браузера Opera 11.10, доступный для платформ Windows, MacOS X, Linux и FreeBSD. Несмотря на непримечательный номер версии, в Opera 11.10 реализовано несколько существенных новшеств, таких как возможность работы через Socks-прокси, обновление браузерного движка Presto 2.8, поддержка изображений WebP, API фильтрации URL в расширениях, поддержка W3C File API, формата шрифтов WOFF и дополнительных возможностей HTML5. Из изменений можно отметить: Расширение возможностей области Speed Dial: На кнопках для быстрого доступа к наиболее часто используемым сайтам теперь может не только отображаться статический эскиз сайта, но и размещаться динамически обновляемый контент, подготовленный при помощи CSS-режима "view-mode: minimized", или логотип сайта; Снято ограничение на число добавляемых элементов; Добавлены анимационные эффекты и поддержка автоматического масштабирования кнопок; Расширены настройки по группировке кнопок и их оформлению; Осуществлен переход на новую версию браузерного движка Presto 2.8.131, в котором проведена дополнительная оптимизация производительности и обеспечена поддержка множества новых web-технологий; Оптимизирована работа ускорителя интернета Opera Turbo: Использован новый метод упаковки изображений, позволяющий уменьшить размер картинок и обеспечить при этом более высокое качество, чем в прошлой реализации Opera Turbo. Добавлена возможность просмотра отчета об эффективности работы Opera Turbo и показателях экономии трафика; Для разработчиков дополнений представлен URL Filter API, позволяющий дополнениям взаимодействовать со встроенной системой блокирования контента. Созданные правила блокировки ассоциируются с создавшим их дополнением, т.е. при удалении или отключении дополнения правила блокировки отменяются; Обеспечена поддержка предложенного компанией Google откытого формата для хранения изображений - WebP; В состав включена бета-версия Dragonfly, инструмента для интерактивной отладки web-приложений. Поддерживаются такие функции, как отладка JavaScript, инспектирование CSS и DOM, выявление ошибок в скриптах, отслеживание Cookie, мониторинг сетевой активности. Особенностью Dragonfly является поддержка отладки web-страниц, открытых в браузере Opera на другом компьютере (например, на мобильном телефоне или КПК). Во встроенном почтовом клиенте добавлен поиск по фразам, переработана панель. Улучшена поддержка работы с аккаунтами Gmail, например, добавлено распознавание и сопоставление IMAP-папок "Sent items", "Spam" и "Trash", а также выявление дублирующихся в разных папках сообщений; Возможность работы через прокси-сервер, поддерживающий протокол SOCKS 5. Настройка осуществляется через "opera:config". К сожалению, для преобразования имен используется локальный DNS-преобразователь, а не возможность отправки DNS-запросов через Socks, что делает поддержку Socks в Opera в большом числе случаев бесполезной; На страницы с ошибками добавлено поле для быстрого обращения к поисковым системам; Улучшения для web-разработчиков: Поддержка JavaScript-событий BeforeCSS и AfterCSS; Поддержка средств CSS3 по управлению градиентами; Поддержка W3C CSS Multi-column Layout Module; Поддержка W3C CSS Device Adaptation (viewport); Поддержка CSS-свойств "zoom-in" и "zoom-out" для параметра "cursor"; Поддержка глобальных HTML5-атрибутов "data-*"; Поддержка HTML5 API по работе с историей пользовательских сессий и навигации (интерфейс "history"); Поддержка блочных HTML5-элементов section, article, nav и т.п. Поддержка W3C File API, который позволяет разработчиком организовать обработку локальных файлов через JavaScript после их выбора пользователем в web-форме, но без загрузки на сервер. Например, можно локально отредактировать набор изображений перед их отправкой на сервер; Поддержка формата Web Open Font Format (WOFF) для распространения шрифтов OpenType, Open Font Format или TrueType в сжатом виде. Формат WOFF позволяет добавлять при компоновке файла с шрифтом дополнительные метаданные, например, информацию об авторах или определение допустимых рамок использования шрифтов.
