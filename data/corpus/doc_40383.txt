Выпуск SUSE Cloud 4

Компания SUSE представила продукт SUSE Cloud 4, представляющего собой вариант SUSE Linux Enterprise Server, рассчитанный на быстрое развёртывание инфраструктуры для обслуживания приватных облачных сервисов на базе платформы OpenStack. Дополнительно подготовлен репозиторий, который можно использовать для установки пакетов с компонентами OpenStack в системах на базе SLES 11 SP3 и openSUSE 13.1. Версия SUSE Cloud 4 примечательна обеспечением поддержки распределённой файловой системы Ceph, переходом на новую версию облачной платформы OpenStack 2014.1 "Icehouse" и расширением средств для интеграции существующих инфраструктур на базе VMware vSphere с OpenStack, в том числе добавлена поддержка VMware Virtual SAN. Кроме того, в новом выпуске представлены реализации в виде сервисов БД, балансировщика нагрузки и межсетевого экрана (database-as-a-service, load balancing-as-a-service, firewall-as-a-service). При помощи SUSE Cloud компании могут развернуть локальную облачную инфраструктуру, подобную Amazon EC2, но выполняемую исключительно на собственных серверах, полностью подконтрольных компании. Дистрибутив поддерживает установку сервера Cloud Administration Server для централизованного управления облачной инфраструктурой, а также узлов Cloud Control Nodes, Compute Nodes и Storage Nodes на которых осуществляется управление пользователями, хранение данных и выполнение виртуальных машин. В комплект также входят средства интеграции с конструктором образов виртуальных окружений SUSE Studio и системой для управления инфраструктурой Linux-серверов SUSE Manager.
