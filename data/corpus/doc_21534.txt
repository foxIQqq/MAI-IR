GNote - попытка переписать программу для ведения заметок TomBoy на C++

Rahul Sundaram из компании Red Hat, участвующий в проектах AbiWord, GNOME и Fedora, выпустил третью начальную версию GNote - реализации популярной программы для создания заметок в wiki-стиле TomBoy, переписанной на языке C++. В настоящее время TomBoy является одним из редких приложений на базе C# и Mono, для которого не разработано достойной замены на других языках программирования. Интересно, что началом работы над проектом GNote послужила первоапрельская шутка, которая обернулась разработкой реального прототипа программы. Программу Gnote можно загрузить в исходных текстах с FTP или в виде пакета для Fedora Linux из репозитория Rawhide. Программа находится пока на начальном этапе развития, но уже поддерживается установка дополнений (реализованы плагины Time Stamp, Bugzilla), создан апплет для панели задач GNOME, присутствует поддержка печати. Близкие по тематике программы для ведения заметок: Zim - десктоп wiki для ведения заметок на Gtk2-Perl (скриншоты); wixi - десктоп wiki на wxPython (скриншоты); WikidPad - десктоп wiki для ведения todo заметок на wxPython (скриншоты); Tagore Desktop Wiki - десктоп wiki на Ruby (скриншоты); Newton - wiki апплет для GNOME на Python/PyGTK2 (скриншот).
