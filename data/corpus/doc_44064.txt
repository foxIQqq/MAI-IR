Открыты исходные тексты игрового движка Atomic

Студия Thunderbeast объявила об открытии исходных текстов игрового движка Atomic, включающего также мощную среду для проектирования и разрабокти игровых приложений. Движок поддерживает создание 2D- и 3D-игр для платформ Windows, OSX, Android, iOS, Linux и Web (WebGL, WebRTC) на языках JavaScript, TypeScript и C++. Код написан на языке C++ и опубликован на GitHub под лицензией MIT, аналогичной лицензии на проект Urho3D, от которого в 2014 году ответвился движок Atomic. В поставку входят runtime-компоненты игрового движка, игровой редактор Atomic Editor, CLI-интерфейс для сборки и управления проектами, большая подборка разнообразных примеров, включая 2D-платформеры, 3D-бродилки, расчёт физики, спрайтовая анимация, отрисовка смены дневных и ночных циклов, реалистичные облака и т.д. Atomic Editor реализован поверх Chromium WebView и включает в себя полноценную среду разработки на языке JavaScript с подсветкой синтаксиса, автодополнением кода и другими возможностями, присущими IDE. Для разработчиков 2D-игр предлагаются интегрированный редактор уровней (форк Tiled), модули для отрисовки спрайтов, симуляции физических процессов (на базе Box2D), расширенных моделей освещения с тенями, поддержки популярных форматов (Spriter, TexturePacker). Для 3D-игр предлагаются редактор 3D-сцен с интерактивным предпросмотром результата, модули для отрисовки через OpenGL и Direct3D, формирования ландшафтов и воды, пост-обработки рендеринга, симуляции физических процессов (на базе Bullet), спрайтовой анимации и выбора оптимального маршрута через навигационные меши.
