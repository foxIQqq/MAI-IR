Два новых релиза дистрибутивов на основе Slackware: Slax 6.1.0 и Kwort Linux 2.4.1

Выпущены новые релизы двух дистрибутивов, основанных на Slackware Linux 12.2: Slax 6.1.0 - легкий и быстрый livecd-дистрибутив Linux с модульным строением, приятным дизайном и отличной поддержкой русского языка. Дистрибутив отличается возможностью быстрого формирования загрузочного образа, скомпонованного из заданного набора модулей (в настоящее время представлен 741 модуль). Базовая поставка занимает около 190 Мб. В качестве графической оболочки используется KDE 3.5.10. Это последний выпуск Slax 6, далее будет выпущена версия на основе KDE 4. Kwort Linux 2.4.1 - Linux дистрибутив на базе XFCE 4.6, имеет собственный пакетный менеджер kpkg, переписанную систему инициализации и набор GUI приложений для конфигурирования (Kwort user manager для управления пользователями, Kwort network manager для настройки сети). Из программ можно отметить Linux ядро 2.6.28.7, firefox 3.0.7, gimp 2.6.4, mplayer, OpenOffice.org 3.0, pidgin 2.5.5, Thunderbird Размер установочного iso-образа составляет 490Мб.
