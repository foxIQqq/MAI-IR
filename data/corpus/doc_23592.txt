Вышел релиз Wget 1.12

Вышел релиз Wget 1.12, консольной программы для автоматического получения файлов по сети. В новой версии представлены следующие улучшения: Добавлена поддержка парсинга CSS (например, для выделение ссылок из CSS блоков); Добавлена возможность прозрачной трансляции в кодированный вид интернационализированных URI (IRI - Internationalized Resource Identifiers) и содержащих не-ASCI символы доменов (тест.ru = xn--e1aybc.ru); При запросе страницы с авторизацией, пользователю теперь выводится интерактивный запрос для ввода пароля, при указании опции "--ask-password"; Опция "--input-file" теперь позволяет выявить ссылки для загрузки из заданного файла; Исправлена уязвимость, дающая возможность принятия поддельных сертификатов, используя особенности интерпретации имен хостов, содержащих в себе символ с нулевым кодом: CA сервер воспримет часть после \0, а большинство браузеров - до \0.
