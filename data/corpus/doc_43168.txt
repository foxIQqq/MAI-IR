Представлен первый релиз свободного синтезатора речи Gnuspeech

Проект GNU анонсировал первый выпуск свободной системы синтеза речи Gnuspeech. Отмечается, что при создании Gnuspeech использованы новейшие разработки в области синтеза речи и данный продукт может как применяться для высококачественной компьютерной генерации речи, так и рассматриваться в качестве инструмента для продолжения исследований и проведения психофизических экспериментов. Пакет включает в себя полноценную артикуляционную модель, базу данных произношения более 70 тысяч слов и набор правил для английского языка. Для отсутствующих в словаре слов предлагается модуль для побуквенного произношения. Модели ритма и интонации пока доступны только для английского языка. Поддерживается работа в OS X и GNU/Linux. Система состоит из двух частей: Для тех кому необходимо синтезировать речь на основе текста и интегрировать функции синтезатора речи в свои приложения предлагается пакет gnuspeechsa, включающий минимальный набор необходимых библиотек и многоплатформенную утилиту для работы из командной строки. Для исследователей и разработчиков, которые желают получить интерактивную систему с инструментами для контроля работы алгоритмов, утилитами для модификации базы произношения, средствами для тонкой настройки вывода и инструментарием для добавления поддержки новых языков предлагается пакет gnuspeech. В состав также включены некоторые сторонние вспомогательные приложения, такие как интерактивный интерфейс пополнения БД Monet (My Own Nifty Editing Tool) и GUI построения звуковых моделей TRAcT (the Tube Resonance Access Tool).
