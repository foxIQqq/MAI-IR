Релиз рабочего стола KDE Plasma 5.19

Доступен релиз пользовательской оболочки KDE Plasma 5.19, построенной с использованием платформы KDE Frameworks 5 и библиотеки Qt 5 с применением OpenGL/OpenGL ES для ускорения отрисовки. Оценить работу новой версии можно через Live-сборку от проекта openSUSE и сборки от проекта KDE Neon User Edition. Пакеты для различных дистрибутивов можно найти на данной странице. Ключевые улучшения: Обновлён внешний вид апплета для управления воспроизведением мультимедийных файлов, размещённого в системном лотке. В менеджере задач обновлено оформление всплывающих информационных подсказок. Проведена работа по унификации дизайна и заголовков апплетов в системном лотке, а также уведомлений, отображаемых на рабочем столе. В панели улучшена расстановка отступов и обеспечена возможность автоматического центрирования виджетов. Полностью переписаны виджеты мониторинга параметров системы. Предложен новый набор фотографических аватаров, доступных в интерфейсе настройки параметров пользователя. По умолчанию предложены новые обои рабочего стола Flow. В интерфейсе выбора обоев для рабочего стола предоставлена возможность просмотра информации об авторе изображения. Проведена работа по повышению удобства работы с липкими заметками. Добавлены дополнительные опции управления видимостью экранного индикатора изменения громкости. Реализована возможность мгновенного применения новой цветовой схемы к приложениям на базе GTK3. Решены проблемы с отображением корректных цветов в приложениях на базе GTK2. Для повышения читаемости текста размер моноширинных шрифтов по умолчанию увеличен с 9 до 10. В виджете управления звуком предоставлена страница настроек с удобным интерфейсом для переключения между доступными звуковыми устройствами, приближенная по оформлению к другим виджетам. В конфигураторе "System Settings" переработаны секции для управления приложениями по умолчанию, учётными записями online-служб, глобальными клавиатурными комбинациями, сценариями KWin и фоновыми сервисами. При вызове модулей конфигуратора из KRunner или меню запуска приложений обеспечен запуск полноценного приложения "System Settings" с открытием искомой секции настроек. В настройках параметров экрана реализован показ коэффициента соотношения сторон для каждого предлагаемого разрешения экрана. Добавлена возможность более точной настройки скорости анимации эффектов для рабочего стола. Добавлена возможность настройки индексации файлов для отдельных каталогов. Реализована опция для отключения индексации скрытых файлов. При использовании Wayland. предоставлена возможность настройки скорости прокрутки для мыши и тачпада. Внесено множество мелких исправлений и улучшений в интерфейс настройки шрифтов. Переработан интерфейс приложения для просмотра информации о системе (Info Center), который приближен к интерфейсу конфигуратора. Добавлена возможность просмотра сведений о графическим оборудовании. В оконном менеджере KWin реализована новая техника подрезки границ субповерхностей (subsurface clipping), решающая проблему с мерцанием во многих приложениях. При работе под управлением Wayland также реализована поддержка поворота экрана на планшетах и ноутбуках-трансформерах. Обеспечена корректировка цветов пиктограмм в заголовках для их приведения к соответствию активной цветовой схеме. В Центре установки приложений и дополнений (Discover) проведена унификация оформления с другими компонентами Plasma. Упрощено удаление репозиториев Flatpak. Обеспечен показ версии приложения, например, для выбора нужного варианта пакета при наличии нескольких версий приложения в разных репозиториях. В KSysGuard добавлена поддержка систем с более чем 12 ядрами CPU.
