Новая версия сетевого анализатора Wireshark 1.10.0

После года разработки представлен релиз новой стабильной ветки сетевого анализатора Wireshark - 1.10.0. Напомним, что изначально проект развивался под именем Ethereal, но в 2006 году из-за конфликта с владельцем торговой марки Ethereal, разработчики были вынуждены переименовать проект в Wireshark. Ключевые новшества Wireshark 1.10.0: Поддержка сборки с использованием GTK+ 3; Добавлена поддержка всех профилей и протоколов Bluetooth; Обновлено содержимое панели инструментов для беспроводных сетей; В версии для Linux налажено определение добавления и удаления сетевых интерфейсов; Обеспечена поддержка сравнения двух полей при задании правил фильтрации вывода на экран, например, можно указать udp.srcport != udp.dstport; Улучшено определение типа и имени продукта для USB-устройств; В 32- и 64-разрядные сборки для Windows добавлена функция автоматической установки обновлений. В состав сборки для Windows включена библиотека WinPcap 4.1.3 с поддержкой Windows 8; Поддержка просмотра списка отрезолвленных имён хостов в формате файла "hosts"; Добавлена поддержка учёта времени между отправкой HTTP-запроса и поступлением HTTP-ответа; Capinfos теперь выводит статистику в читаемом виде; Поддержка открытия связанного пакета (например, ответа на запрос) в новом окне; Tshark по умолчанию теперь вводит только данные пакета в формате hex/ascii без детализации и суммарных данных, для вывода которых теперь следует использовать опцию "-Px"; Переработан код многопроходного анализа и фильтрации в Tshark для корректного учета последовательности пакетов и расчёта кадров в процессе транзитной пересборки пакетов; Обновлены пиктограммы приложений и панелей; Wireshark теперь поставляется с двумя глобальными файлами конфигурации: Bluetooth с новыми правилами подсветки для Bluetooth и Classic с правилами подсветки в старом стиле.
