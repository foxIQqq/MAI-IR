Увидели свет Red Hat Software Collections 1.2 и и Red Hat Developer Toolset 3

Компания Red Hat анонсировала выпуск продукта Red Hat Software Collections 1.2, позволяющего использовать в Red Hat Enterprise Linux новые технологии разработки и компоненты web-стека, не ожидая появления следующей ветки RHEL и не устанавливая сторонние пакеты из неофициальных внешних источников. Время поддержки Red Hat Software Collections составляет три года, в течение которых пользователи будут снабжаться всеми необходимыми обновлениями с устранением проблем безопасности и критических ошибок. Red Hat Software Collections 1.2 включает в себя пакеты с более свежими версиями программ, чем входящие в состав штатного RHEL. Например, для установки предлагаются Maven 3.0, Git 1.9.4, Nginx 1.6, а также свежие выпуски Ruby with Rails, Python, PHP, Perl, MongoDB, MariaDB, MySQL и PostgreSQL. Для популярных коллекций программ, востребованных разработчиками, подготовлены шаблоны для создания контейнеров Docker. В состав Red Hat Software Collections также включен ранее поставляемый отдельно набор Red Hat Developer Toolset 3, предлагающий пакеты с более новыми выпусками средств для разработчиков приложений, в том числе доступны новые выпуски Eclipse, ltrace 0.7.91 и GCC 4.9.
