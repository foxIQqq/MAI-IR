Доступна третья бета-версия Firefox 4 для платформ Android и Maemo

Началось тестирование третьей бета-версии Firefox 4 для мобильных устройств на базе платформ Android и Maemo. Установить Firefox для платформы Android теперь можно через Android Market. Пакеты для Maemo можно найти на данной странице. По сравнению с прошлой бета-версией можно отметить следующие улучшения: Изменения, свойственные сборке для платформы Android: Улучшена поддержка экранных и аппаратных клавиатур Android; Меню переделано в типичном для Android стиле; Исправлена проблема с открытием ссылок из внешних приложений; Добавлена поддержка загрузки сайтов из локальной системы на сервер; В контекстном меню сайта добавлена команда "Save as PDF" для записи страницы в формате PDF; Проведена работа по упрощению интерфейса для настройки Firefox Sync. Добавлена возможность указания собственных серверов синхронизации для Firefox Sync; Поддержка просмотра видео в полноэкранном режиме; Улучшение производительности кода, ответственного за вывод звука; В интерфейсе пользователя добавлена поддержка работы с буфером обмена (Copy/paste) в адресной строке. Поисковые системы теперь появляются на экране в процессе набора текста. В менеджер дополнений добавлена поддержка вывода рекомендуемых дополнений; Для разработчиков добавлена поддержка API для работы с локальным хранилищем данных (localStorage API). На платформе Android теперь можно устанавливаться обработчики событий, связанных с изменением ориентации экрана. Список улучшений, реализованных в прошлых тестовых версиях можно посмотреть в первом анонсе.
