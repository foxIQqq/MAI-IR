Внеплановый корректирующий релиз проприетарного видеодрайвера AMD Catalyst 11.4b

Компания AMD выпустила внеплановую "hotfix" версию проприетарного видеодрайвера для видеокарт ATI - AMD Catalyst 11.4b. В новой версии обеспечена поддержка поступивших в продажу видеокарт AMD Radeon HD 6790. Поддержка X.Org Server 1.10 по прежнему отсутствует, для использования драйвера совместно с последней версией X-сервера необходимо использовать вариант драйвера (пакет fglrx-installer 8.840-0ubuntu1), подготовленный для включения в состав Ubuntu 11.04 по договоренности с компанией Canonical.
