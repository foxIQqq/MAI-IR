Релиз PyQt 5.6 с полной поддержкой Qt 5.6

Вслед за Qt 5.6 LTS вышел соответствующий релиз библиотеки PyQt для разработки графических приложений на языке Python. Особенности выпуска: Добавленная полная поддержка Qt 5.6.0. Поддержка Python 3.5 доступна для 64-битного Linux, 64-битного OS X и 32-и 64-битных Windows. (После этого выпуска для Windows больше не будут поставляться инсталляторы в формате .exe) Добавлен модуль QtWebEngineCore. Восстановлен qt_set_sequence_auto_mnemonic() для QtGui. Восстановлен MouseLock к QWebEnginePage.Feature. Восстановлен WA_DontShowOnScreen. Файлы поддержки PEP 484 ставятся по умолчанию. Добавлен аргумент командной строки --import-from для pyuic5. Добавлены опции "--no-stubs" и "--stubsdir" для configure.py. Добавлена опция "--qtconf-prefix" для configure.py.
