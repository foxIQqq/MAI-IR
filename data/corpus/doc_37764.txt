Доступен ROSA Enterprise Linux Server 6.4

Компания Роса представила второй выпуск своего серверного дистрибутива - ROSA Enterprise Linux Server 6.4 (RELS). Новая версия построена на пакетной базе Red Hat Enterprise Linux 6.4, дополненной оригинальными разработками компании Роса. 32- и 64-разрядные сборки дистрибутива доступны для свободной загрузки. Также публично доступен репозиторий с пакетами и обновлениями с устранением проблем безопасности. Кроме штатных пакетов из состава RHEL в состав ROSA Enterprise Linux Server включены дополненные инструменты для развёртывания частных облаков, интеграции с корпоративными сетями, средства централизованного управления учетными записями пользователей, сетевыми ресурсами и системами хранения данных, web-интерфейс для наглядной настройки и управления сервером. Из улучшений в оригинальных компонентах дистрибутива отмечаются: В ROSA Server Setup, веб-интерфейс для установки и настройки основных серверных компонентов дистрибутива, добавлены средства для быстрой установки и настройки прокси-сервера Squid; Поддержка репликации; Улучшенные средства настройки сервера сетевой аутентификации Kerberos 5; В ROSA Directory Server, компоненте для администрирования учетных записей пользователей и управления доступом к ресурсам локальной сети организации, учтены пожелания пользователей в форуме или при обращении в службу поддержки; В инсталлятор добавлено подробное описание классов установки, упрощена установка с USB-накопителей.
