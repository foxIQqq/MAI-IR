Компания Zend представила пакет для создания web-серверов с поддержкой PHP

Компания Zend Technologies, курирующая разработку языка программирования PHP, представила новый продукт - Zend Server, представляющий собой полнофункциональный web-стек, поставляемый в одном пакете. Zend Server для Linux доступен в виде бинарного пакета для всех популярных Linux дистрибутивов в 32-х и 64-х разрядных сборках (например, поддерживаются Fedora, RHEL, CentOS, Ubuntu, Debian). Пакет представляет собой полностью готовое к эксплуатации решение, работоспособное сразу после установки. Включает в себя http-сервер Apache, PHP, Zend Framework, набор модулей для поддержки БД (Oracle, DB2, MySQL, PostgreSQL), коннектор к Java, специально разработанный web-интерфейс для управления конфигурацией, отладчик, оптимизатор байткода и API для организации кеширования. Панель управления включает как оригинальные разработки для управления web-сервером, так и типовые, например, phpmyadmin. Продукт распространяется только в бинарном виде, исходные тексты доступны только для базовых компонентов, дополнения разработанные Zend распространяются только в закрытом виде. Поддерживаются две версии Zend Server - бесплатная Community редакция и коммерческая версия, отличающаяся возможностью работы на высоконагруженных системах, доступностью механизма промежуточного кеширования страниц, системы мониторинга работоспособности web-приложений, программы для диагностики проблем и выявления узких мест, приложения для организации сервера загрузок (Zend Download Server), предоставлением технической поддержки и возможностью инкрементальной установки обновлений. Стоимость коммерческой версии составляет 1195 долларов в год, для ознакомления предоставляется 30-дневный пробный период.
