Компания Valve начала отбор добровольцев для тестирования Linux-версии Steam

В рамках процесса обеспечения поддержки платформы Linux в своих продуктах, компания Valve начала набор добровольцев, желающих принять участие в тестировании первого пробного выпуска Linux-клиента для сервиса доставки игр Steam. Через проведение опроса для зарегистрированных клиентов Steam компания намерена отобрать 1000 опытных пользователей Linux, которые смогут адекватно оценить работу приложения в Linux. Через несколько дней из общей массы пользователей, принявших участие в опросе, будут отобраны участники тестирования, которым будут отправлены индивидуальные приглашения и ссылки для загрузки. Для освещения событий, связанных с процессом прохождения бета-тестирования запущен отдельный раздел на сайте steamcommunity.com. Бета-версия будет оформлена в виде пакетов для Ubuntu 12.04 и 12.10, в состав войдёт клиент Steam для Linux и одна из игр Valve.
