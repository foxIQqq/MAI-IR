Компания Microsoft присоединилась к организации Eclipse Foundation

Компания Microsoft вошла в число участников некоммерческой организации Eclipse Foundation, курирующей развитие платформы для создания интегрированных сред разработки. Компания Microsoft не вошла в число крупных спонсоров и стратегических партнёров, а лишь присоединилась к многочисленной группе Solutions Member, предоставляющих решения на базе платформы Eclipse и никак не влияющих на развитие платформы, но имеющих возможность более тесно взаимодействовать с сообществом. Для подтверждения своих благих намерений Microsoft также открыла исходные тексты плагина Team Explorer Everywhere, интегрировала поддержку Codenvy в Visual Studio Team Services, подготовила образы с Codenvy VM для Azure, опубликовала улучшенный инструментарий Azure для Eclipse, добавила поддержку Azure IoT во фреймворк Eclipse Kura и реализовала в Azure ряд улучшений для разработчиков на языке Java.
