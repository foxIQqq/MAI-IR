Компания Oracle выпустила обновление Solaris 11.3 SRU 33

Сформировано обновление операционной системы Solaris 11.3 SRU 33, в котором предложена очередная серия исправлений и улучшений для ветки Solaris 11.3, изначально опубликованной в 2015 году. Параллельно развивается новая значительная ветка Solaris 11.4, которая пока находится на стадии бета-тестирования. Основные изменения в Solaris 11.3 SRU 33: Обновлены версии программ (в основном для устранения выявленных уязвимостей): HMP 2.4.4.0, curl 7.59.0, GNU binutils 2.30, GNU patch 2.6.7, Firefox 52.8.0esr, Thunderbird 52.8.0, Wireshark 2.6.1, FreeType 2.9.0, PHP 5.6.36 & 7.1.17, MySQL 5.5.60 & 5.6.40; AI Server (Automated Install) переведён на использование Apache httpd 2.4; В ZFS улучшена отправка данных в сжатом виде; В драйверы для сетевых контроллеров Intel добавлено накопление дополнительной статистики о сбоях; Для IR2/IR3 реализовано отображение свойств SAS-контроллеров в ILOM (Integrated Lights Out Manager).
