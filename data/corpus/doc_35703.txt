Релиз открытой платформы финансовой аналитики OpenGamma 1.2.0

Представлен релиз открытой платформы финансовой аналитики OpenGamma 1.2.0, ориентированной на обеспечение функций фронт-офиса для взаимодействия с клиентами, торговой аналитики и анализа рисков для фирм, специализирующихся на оказании финансовых услуг. OpenGamma комбинирует в едином программном продукте движок расчётов, учитывающий зависимости между различными финансовыми показателями, средства бизнес-аналитики, прослойку для управления данными, систему агрегации внешней финансовой информации и модуль для хранения и управления историческими наборами данных для расчёта рисков. Код OpenGamma написан на языке Java и распространяется в рамках лицензии Apache. В версии 1.2.0 представлен новый web-интерфейс с поддержкой многопанельного режима просмотра аналитики, возможностью произвольного перетаскивания данных мышью и копирования содержимого областей, выводом вспомогательных информационных панелей, различными режимами оценки, просмотра финансовых данных и статистики. Добавлены новые аналитические библиотеки для оценки кредитных дефолтных свопов, расчёта изменения цен в будущем, учёта товарных фьючерсов и коррекции дивидендов. В модуль управления данными добавлена начальная реализация новой системы разделения привилегий при доступе к данным, добавлен инструмент для автоматического обновления схемы БД. Расширены возможности компонента для интеграции с Excel. Улучшена работа инсталлятора и добавлена поддержка сборки с использованием Maven.
