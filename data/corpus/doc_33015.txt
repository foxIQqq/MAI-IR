Adobe обеспечит дополнительный уровень изоляции Flash-плагина для Firefox

Разработчики из компании Adobe сообщили о выпуске экспериментальной реализации дополнительного уровня изоляции для Flash-плагина, распространяемого для Firefox. Кроме уже реализованного в Firefox выноса плагина в отдельный процесс, который позволяет защитить браузер от краха Flash, в новом плагине задействован дополнительный sandbox, созданный по аналогии с техникой Flash Player Protected Mode, ранее реализованной для браузера Google Chrome. В настоящее время бета-версия Flash-плагина 11.2.300.130 с дополнительным уровнем защиты уже доступна для тестирования. Реализация не требует модификации кода Firefox и работает с любыми выпусками данного браузера начиная с версии 4.0. К сожалению, тестовая версия плагина пока не доступна для Linux, поддерживается только Windows 7 и Vista (32-битные редакции). Отключить дополнительный уровень изоляции можно через директиву ProtectedMode=0 в файле конфигурации mms.cfg.
