Третий предварительный выпуск графического редактора GIMP 3.0

Доступен для тестирования выпуск графического редактора GIMP 2.99.6, продолжающий развитие функциональности будущей стабильной ветки GIMP 3.0, в которой выполнен переход на GTK3, добавлена штатная поддержка Wayland и HiDPI, проведена значительная чистка кодовой базы, предложен новый API для разработки плагинов, реализовано кэширование отрисовки, добавлена поддержка выделения нескольких слоёв (Multi-layer selection) и обеспечено редактирование в исходном цветовом пространстве. Для установки доступен пакет в формате flatpak (org.gimp.GIMP в репозитории flathub-beta) и сборки для Windows. По сравнению с прошлым тестовым выпуском добавлены следующие изменения: Продолжено развитие средств для редактирования за пределами холста - реализована возможность размещения направляющих за границей холста, что может оказаться полезным в ситуациях, когда изначально выбранного размера холста не хватает. Что касается ранее предоставляемой возможности удаления направляющей путём её перемещения за границу холста, то данное поведение немного изменилось и вместо границ хоста для удаления теперь нужно переместить направляющую за пределы видимой области. В диалоге установки размера холста добавлена возможность выбора предопределённых шаблонов, описывающих типовые размеры, соответствующие распространённым форматам страниц (A1, A2, A3 и т.п.) Размер рассчитывается на основании фактического размера с учётом выбранного DPI. Если при изменении размера холста DPI шаблона и текущего изображения отличаются, на выбор предлагается изменить DPI для изображения или масштабировать шаблон для его приведения к DPI изображения. Добавлена поддержка масштабирования холста через жест "щипок" на тачпадах и сенсорных экранах. Масштабирование щипком пока работает только в окружениях на базе Wayland, в сборках для X11 указанная возможность появится в ближайшие месяцы после принятия в X Server патча с необходимой функциональностью. Улучшена работа экспериментального инструмента Paint Select, позволяющего постепенно выделить область, используя приблизительные мазки кистью. Инструмент основан на применении алгоритма выборочного сегментирования (graphcut) для выделения только интересующей области. Выделение теперь учитывает видимую область, что позволяет заметно ускорить выполнение операции при масштабировании. Добавлен плагин для генерации цветового ICC-профиля на основе встроенных в PNG-изображение метаданных gAMA и cHRM, описывающих параметры гамма-коррекции и цветности. Указанная возможность позволяет корректно отображать и редактировать в GIMP PNG-изображения, поставляемые с gAMA и cHRM. Предложено несколько реализаций плагина для создания скриншотов. В частности, добавлен вариант, использующий порталы Freedesktop для создания снимков экрана в окружениях на базе Wayland и для работы из flatpak-пакетов, использующих изоляцию приложения. В данном плагине логика создания скриншота вынесена на сторону портала, который сам формирует диалог о параметрах захватываемого содержимого, без показа старого диалога GIMP. В плагине для экспорта в формате TIFF обеспечено сохранение цветового профиля и комментариев для каждого слоя изображения. Продолжена переработка API для разработки плагинов. Для генерации GTK-диалогов теперь достаточно нескольких строк кода. По умолчанию обеспечена передача массива областей отрисовки (drawable), так как GIMP теперь поддерживает многослойное выделение. Проведена работа по унификации наименования функций. Предоставлена возможность сохранения и доступа к дополнительным данным, прикреплённым к изображению, слою или экземпляру GIMP, что позволяет плагину сохранять произвольные бинарные данные между перезапусками.
