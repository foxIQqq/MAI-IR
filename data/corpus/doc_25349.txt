Релиз Apache Ant 1.8

Спустя три года с момента выпуска прошлой версии представлен релиз свободной системы сборки Java проектов Apache Ant 1.8. Из изменений можно отметить: Возможность ограничения области видимости локальных свойств, например, при создании макроса внутренние переменные теперь могут быть видны только в рамках блока "macrodef"; Директива "import" позволяет подключать ресурсы из внешних файлов (например, правила сборки из JAR-файла) или URL; Улучшена работа кода сканирования содержимого директорий, значительно увеличена производительность (большие директории теперь сканируются в десятки раз быстрее), добавлена проверка на зацикливание символических ссылок; Переработана система создания дополнений, расширяющих возможности Ant. К сожалению это привело не только к упрощению и расширению функциональности, но и к нарушению совместимости API; Добавлена поддержка нового элемента "extension-point", упрощающего использования типовых файлов сборки; В качестве минимально поддерживаемой версии Java теперь требуется версия 1.4.
