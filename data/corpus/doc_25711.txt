Вышел релиз Linux дистрибутива Frugalware 1.2

Вышел релиз Linux дистрибутива общего назначения Frugalware 1.2, разработанного для опытных пользователей не боящихся командной строки, основное направление развития дистрибутива - максимальное упрощение. Концепция построения дистрибутива похожа на Slackware, в качестве пакетного менеджера задействован pacman из Arch Linux. При создании пакетов, разработчики стараются публиковать программы как есть, избегая внесения дополнительных патчей. Основные улучшения Frugalware 1.2: Обновлены версии программ: Linux ядро 2.6.32.8, Glibc 2.11.1, Xorg 7.5, GNOME 2.28, KDE 4.3.5; По умолчанию для переключения видеорежимов для видеокарт Intel и ATI Radeon используются KMS-модули ядра; В базовую систему добавлена поддержка расширяемых модулей аутентификации PAM (Pluggable Authentication Modules); Содержимое директории /dev теперь формируется при помощи devtmpfs; Из установочных образов удалено несколько занимающих много дискового пространства игр; Вариант десктоп-окружения на базе KDE переведен на использование ветки 4.x; Осуществлен переход на Bluetooth-стек bluez 4.x, что позволило задействовать встроенные в GNOME и KDE инструменты для работы с устройствами через интерфейс Bluetooth.
