Релиз интегрированного набора интернет-приложений SeaMonkey 2.9

С небольшим отставанием от Firеfox 12 опубликован релиз пакета SeaMonkey 2.9, объединяющего в рамках одного продукта набор приложений для работы в сети, разрабатываемых под эгидой проекта Mozilla: web-браузер, почтовый клиент, систему агрегации новостных лент (RSS/Atom), WYSIWYG-редактор html-страниц Composer, IRC-клиент ChatZilla, набор средств для web-разработчиков, включающих DOM inspector и JavaScript-отладчик. Пакет доступен в сборках для платформ Windows, Mac OS X и Linux. Кроме синхронизации с кодовой базой Firefox 12 и Thunderbird 12 с заимствованием свойственных для данных продуктов новшеств, отмечается добавление возможности произвольного изменения размера диалогов работы с файлами и управления закладками. Обеспечена поддержка автоматического начала загрузки после вставки URL из буфера обмена в окне менеджера управления загрузками. Синхронизация установленных дополнений теперь может быть настроена через штатный интерфейс встроенного конфигуратора. Плагины теперь могут быть отключены для всего пакета, а не только для почтового интерфейса. Отключено автоматическое начало проигрывание блоков с HTML5-видео, для начала воспроизведения теперь необходимо нажать специально отображаемую кнопку.
