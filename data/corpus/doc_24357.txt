Microsoft собирается открыть исходные тексты компиляторов Visual Basic и C#

На конференции PDC09 представители компании Microsoft заявили о планах по открытию исходных текстов компиляторов Visual Basic и C#. Сдвиг отношения компании Microsoft по отношению к движению open source также отметил Tim O'Reilly, известный сторонник свободного и открытого ПО, в своем докладе на конференции Web 2.0 Expo. В качестве признаков называется заметный переход от навязывания своих технологий на попытки пойти на сближение с другими компаниями, работающими в области Web-технологий. Также упоминается, что представленная облачная технология Windows Azure изначально рассчитана на работу с открытыми технологиями и поддерживает запуск таких приложений, как PHP, MySQL, Apache и WordPress.
