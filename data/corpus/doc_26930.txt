Linux-производитель Ulteo заключил партнерское соглашение с Microsoft

Компания Ulteo, основанная Жилем Дювалем (Gaël Duval), создателем дистрибутива Mandrake Linux, и развивающая продукт Ulteo Open Virtual Desktop, объявила о заключении с компанией Microsoft партнерского соглашения, в рамках которого будет проведена работа по укреплению интеграции окружения Ulteo Open Virtual Desktop с платформой Windows. В рамках соглашения компании Ulteo будет обеспечена техническая поддержка по интеграции с такими продуктами, как Sharepoint, Microsoft Remote Desktop Protocol, Microsoft HyperV, App-V и Azure. Open Virtual Desktop позволяет организовать процесс работы с удаленными десктоп-окружениями или отдельными приложениями, которые могут выполняться на внешних серверах приложений Linux и Windows (для пользователя формируется смешанное окружение, в котором представлены как Linux, так и Windows программы). Все компоненты Ulteo Open Virtual Desktop доступны пользователю независимо от его места нахождения, работа производится через web-браузер со специально установленным Java-плагином.
