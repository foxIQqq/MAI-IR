Вышла бета-версия дистрибутива SUSE Linux Enterprise 12

Компания SUSE анонсировала доступность бета-версии промышленного дистрибутива SUSE Linux Enterprise 12. Финальные выпуски продуктов на базе платформы SUSE Linux Enterprise, такие как SUSE Linux Enterprise Server 12, SUSE Linux Enterprise High Availability Extension 12 и SUSE Linux Enterprise Desktop 12, планируется выпустить в течение 2014 года. Для участия в программе бета-тестирования следует отправить запрос по адресу SLE12beta@suse.com. Особенности SUSE Linux Enterprise 12: Переход на использование ветки ядра Linux 3.12; Задействование технологии для обновления ядра Linux на лету, без перезагрузки и остановки работы; Переход на новый инсталлятор, который позволит избавиться от второй фазы установки, на которой осуществляется начальная настройка системы. Если сейчас после завершения установки пакетов требуется перезагрузка и запуск второй фазы инсталлятора, осуществляющего проведение настройки, то с использованием нового инсталлятора система будет сразу полностью готова к работе без дополнительных промежуточных перезагрузок; Переход на фреймворк Wicked для настройки сетевой конфигурации. Wicked заменяет собой скрипты, подобные /sbin/ifup, и оперирует файлами конфигурации в формате XML; Использование варианта конфигуратора YaST, переписанного на языке Ruby. Ранее YaST развивался с использованием специфичного языка программирования YCP (YaST Control Programming Language), что мешало вовлечению в проект новых разработчиков. Кроме использования более привычного для разработчиков языка, новый YaST отличается более высокой производительностью и эффективностью работы. Для решения специфичных задач пользователи смогут развивать собственные модули на языке Ruby; Полноценная поддержка файловых систем Btrfs и Ceph; Поддержка функции создания снапшотов состояния системы и отката изменений в случае выявления проблем. Поставка реализации контроллера домена и сервиса Active Directory на базе Samba 4. Прекращение поддержки создания разделов с файловой системой ReiserFS в YaST и установщике; Прекращение поддержки графических драйверов: xf86-video-ark xf86-video-chips xf86-video-geode xf86-video-glint xf86-video-i128 xf86-video-neomagic xf86-video-newport xf86-video-r128 xf86-video-savage xf86-video-siliconmotion xf86-video-tdfx xf86-video-tga xf86-video-trident xf86-video-voodoo xf86-video-sis xf86-video-sisusb xf86-video-openchrome xf86-video-unichrome xf86-video-mach64.
