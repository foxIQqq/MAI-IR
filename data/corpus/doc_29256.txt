Доступен кандидат в релизы инсталлятора Debian GNU/Linux Squeeze

Представлен первый кандидат в релизы инсталлятора Debian GNU/Linux 6.0 (Squeeze). Что касается пакетной базы дистрибутива, то настоящее время неисправленными остаются 65 блокирующих релиз ошибок. По сравнению с прошлой тестовой-версией инсталлятора добавлены следующие улучшения: Пакет с Linux-ядром обновлен до версии 2.6.32-29; Обновлено художественное оформление визуальной темы для Squeeze; Устранена проблема с юзабилити, возникающая из-за вывода слишком длинных вопросов с просьбой подтвердить согласие с лицензией при попытке установки проприетарных прошивок; Устранен крах mklibs при работе со статическими объектами; Предотвращение загрузки модулей для уже сконфигурированных сетевых интерфейсов; Поддержка нового набора "volatile"; Устранена проблема с ручным вводом данных для resolv.conf, в случае если параметры DNS-серверов не были переданы по DHCP; Обеспечен перевод сообщений инсталлятора на 70 языков, при этом для 57 языков перевод выполнен полностью; Live Installer: устранены недоработки с загрузкой модулей для файловых систем; при обычной установке реализовано удаление пакетов, обеспечивающих работу live-boot и live-config; улучшена совместимость с вручную собранными Linux-ядрами; Изменения специфичные для GNU/kFreeBSD: отключение partman-zfs для kfreebsd-i386; устранение проблем с формированием связанных с CD-ROM записей в fstab; отключение некоторых модулей из-за проблем с прошивками; Изменения специфичные для GNU/Linux: В состав группы nic-modules добавлен модуль qlcnic; В ata-modules обязательно включается модуль ata_generic; Для платформы armel добавлена поддержка систем QNAP TS-119P+ и TS-219P+; Для платформы powerpc добавлена поддержка YDL PowerStation, CHRP-машин с платой IBM Bimini и SLOF-прошивкой; обеспечена поддержка всех моделей G5 PowerMac; улучшена поддержка iMac iSight; обеспечена загрузка надлежащих модулей для Ethernet-карт IBM Power 6 и более новых плат System P; исправлено множество ошибок.
