Компания Hewlett-Packard выпустила пятое обновление ОС HP-UX 11i v3

Компания Hewlett-Packard выпустила пятое обновление операционной системы ОС HP-UX 11i v3, поддерживающей работу на оборудовании на базе архитектуры PA-RISC (HP 9000) и Intel Itanium. По сравнению с прошлой версий: Проведена работа по улучшению ОС в плане повышения безопасности при обеспечении работы критически-важных сервисов, в комплект включен инструментарий Bastille, позволяющий автоматически повысить безопасность системы за счет блокирования сетевых портов и других изменений, направленных на предотвращение неавторизированного доступа; В составе комплекта Virtual Server Environment теперь поставляется файловая система VxFS (Veritas file system) версии 5.0, которая обеспечивает почти двукратный прирост производительности по сравнению с базовой файловой системой; Увеличена производительность выполнения многопоточных программ; Расширены кластерные возможности, улучшена технология балансировки памяти для NUMA-кластеров; Улучшены средства для управления системой, программа для проведения обновления теперь может провести одновременное обновление до 100 систем.
