Dell и HP будут поставлять серверы с Solaris, Oracle Enterprise Linux и Oracle VM

Компания Oracle анонсировала переход на новый уровень сотрудничества с компаниями Dell и Hewlett-Packard, которые отныне будут сертифицировать и продавать серверы на базе архитектуры x86 с предустановленными операционными системами Oracle Solaris, Oracle Enterprise Linux (основан на пакетах Red Hat Enterprise Linux 5) и Oracle VM (дистрибутив для развертывания решений виртуализации на базе Oracle Enterprise Linux и Xen). Пользователям серверов, поставляемых с операционными системами Oracle, будет предоставлен полный доступ к системе обновлений и возможность подписки на первичный сервис технической поддержки Oracle. В анонсе сделан особый акцент на продолжения полноценного развития ОС Solaris, которая будет развиваться и предлагаться клиентам наравне с Linux-решениями компании. Отдельно подчеркивается, что начало предустановки операционных систем Oracle не является навязыванием потребителям новых систем, возможность поставки серверов Dell и HP ProLiant с предустановкой операционных систем от других производителей, таких как Red Hat и Novell, будет сохранена в полном объеме. Дополнительно можно отметить, что несмотря на то, что Oracle Enterprise Linux основан на пакетной базе Red Hat Enterprise Linux и полностью бинарно совместим с данным дистрибутивом, по утверждению представителей Oracle это не форк - компания вносит множество изменений, которые возвращаются в родительские проекты (upstream), активно участвует в разработке Linux ядра и системы виртуализации Xen, развивает Btrfs и кластерную файловую систему OCFS (Oracle Cluster File System). В состав дистрибутива Unbreakable Linux входит кластерное ПО Oracle Clusterware, которое раньше поставлялось только для клиентов Real Application Clusters (RAC). Red Hat в системах виртуализации и кластерных технологиях придерживается другой позиции, отдавая предпочтение KVM и файловой системе GFS.
