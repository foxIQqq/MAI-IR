Корректирующий выпуск системы виртуализации VirtualBox 4.3.12

Компания Oracle выпустила корректирующий релиз системы виртуализации VirtualBox 4.3.12, в котором отмечено около 27 исправлений, направленных на увеличение стабильности и устранение регрессивных изменений. Из наиболее заметных улучшений можно отметить: В код поддержки 3D внесена порция исправлений, направленных на улучшение работы в Ubuntu 14.04; Устранены проблемы с зависимостями загрузочных скриптов для старых выпусков Debian; Для гостевых окружений с Linux поправлена символическая ссылка на хелпер для организации совместной работы с каталогами; В дополнениях для гостевых окружений с Linux устранено несколько проблем, приводящих к краху процессов и сервиса VBoxService при запуске гостевого окружения; Устранён ряд проблем в менеджере виртуальных машин, в том числе налажена эмуляция MSR на некоторых видах оборудования; В GUI решены редко проявляющиеся проблемы, приводящие к краху при переходе в полноэкранный режим; Решена проблема с прекращением работы совместного буфера обмена после создания снапшота; В реализации NAT устранён потенциальный крах при работе DNS-прокси, решены проблемы с прекращением работы перенаправления портов и повреждением содержимого кэша ARP; В VRDP устранён потенциальный крах при отсоединении клиента; В инсталляторе для Windows реализована поддержка msm-модуля.
