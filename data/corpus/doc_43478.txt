Релиз CrossOver 15.0 для Linux и OS X

Компания CodeWeavers выпустила релиз пакета Crossover 15.0, основанного на коде Wine 1.8-rc и предназначенного для выполнения программ и игр, написанных для платформы Windows. CodeWeavers входит в число ключевых участников проекта Wine, спонсирует его разработку и возвращает в проект все новшества, реализованные для своих коммерческих продуктов. Исходные тексты открытых компонентов CrossOver 15.0 можно загрузить на данной странице. В новой версии представлен полностью новый пользовательский интерфейс для Linux и переработанный установщик для Linux и OS X. Улучшено качество вывода звука на платформе Linux, за счёт задействования звукового сервера PulseAudio. Добавлена поддержка запуска новых приложений, в том числе Quicken 2016. Улучшена поддержка Microsoft Office.
