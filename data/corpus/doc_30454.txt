Релиз Alpine Linux 2.2.0, дистрибутива для сетевых шлюзов

Вышел релиз Alpine Linux 2.2.0, специализированного дистрибутив для построения сетевых шлюзов, межсетевых экранов, VPN-серверов и VoIP-систем. Дистрибутив сформирован с оглядкой на повышенную безопасность и собран с патчами PaX и SSP (Stack Smashing Protection). Загрузочный iso-образ занимает 213 Мб, в качестве системой библиотеки используется uClibc, а в качестве стандартных утилит - BusyBox. Из добавленных в новой версии улучшений можно отметить: Переход на использование Linux-ядра 2.6.38; Сформирована сборка для архитектуры x86_64; Использование по умолчанию SHA512-хэшей в /etc/shadow для хранения паролей; Предварительная поддержка контроля доступа на основе ролей (Role Based Access Control), реализованная проектом grsecurity В утилиту для разбивки диска и установки (setup-disk) добавлена поддержка LVM-разделов, программного RAID и использования файловой системы EXT4 по умолчанию. Улучшена работа пакетного менеджера apk, добавлена поддержка работы с несколькими архитектурами, расширены возможности интерактивного режима и пользовательского интерфейса; Добавлена поддержка использования загрузочного раздела, доступного только для чтения, что позволило обеспечить возможность развертывания гибридных установок, при которых система работает на немодифицируемом разделе, а изменённые данные и конфигурация сохраняются на внешний носитель или в отдельный раздел; Добавлена предварительная поддержка Xen dom0, доступная пока только в сборке для архитектуры x86_64; Добавлена поддержка установки рабочего стола GNOME; Обновление версий программ: Xfce 4.8, Firefox 4, MySQL 5.5, OpenRC 0.8.2, Busybox 1.18.4.
