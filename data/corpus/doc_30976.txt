Вышел десктоп-ориентированный дистрибутив Sabayon Linux 6

Вышел релиз дистрибутива Sabayon Linux 6, сочетающегося в себе удобство для конечного пользователя с гибкостью Gentoo. Дистрибутив доступен в сборках с GNOME 2.32.2 (iso 1.5 Гб) и KDE 4.6.4 (iso 1.9 Гб). Sabayon Linux оформлен в виде Live-дистрибутива, снабженного возможностью легкой установки на жесткий диск. В комплекте поставляется полный набор всех необходимых мультимедиа кодеков и проприетарных драйверов для видеокарт, беспроводных адаптеров и софт-модемов. Пакетная база дистрибутива синхронизирована с коллекцией портажей Gentoo и полностью совместима с Portage 2.2 (EAPI2), пользователь имеет возможность установки программ как из исходных текстов, так и используя готовые бинарные пакеты, собранные для платформ x86 и x86-64. Для управления пакетами подготовлен GUI интерфейс "Sulfur" и консольная программа "equo". Особенности выпуска Sabayon Linux 6: Дистрибутив поставляется с Linux-ядром 2.6.39.1, оптимизированным для выполнения десктоп-задач. Дополнительно поставляются пакеты с Linux-ядром, оптимизированным для использования на сервере или для развертывания изолированных контейнеров на базе OpenVZ или Vserver; Официальная поддержка файловой системы Btrfs; Переработанное графическое оформление, улучшение визуальных тем для широкоформатных экранов 16:9 и 16:10; Упрощенная система трансформации дистрибутива в медиацентр на базе XBMC; В состав входит новая версия пакетного менеджера Entropy 1.0_rc10. Увеличена скорость работы Entropy Store (Sulfur). Добавлена поддержка загрузки только измененных частей пакетов (delta-обновления), параллельной загрузки нескольких пакетов, обновления репозитория через HTTPS; Улучшение инсталлятора, который базируется на исходных текстах новой ветки Anaconda, используемой для установки Fedora Linux. В инсталляторе присутствует возможность шифрования дисковых разделов, разбивки программных RAID, работы с LVM, установки языковых пакетов; Добавлен интерфейс для управления пакетным фильтром и фронтэнды для GNOME и KDE; X.Org Server обновлен до версии 1.10; По умолчанию активирован композитный режим в Sane Desktop; Вместо проприетарного Java VM теперь используется IcedTea6; Осуществлен переход на библиотеку jpeg-turbo, в которой значительно увеличена скорость обработки JPEG-изображений; Осуществлен переход с OpenOffice.org на LibreOffice 3.3.3; В качестве браузера по умолчанию используется Chromium; Пакеты с проприетарными драйверами NVIDIA и ATI разделены на две части - пакет с модулями ядра и пакет с компонентами, работающими на пользовательском уровне; GRUB обновлен до версии 1.99; Представлено приложение "kernel-switcher", предназначенное для быстрого переключения между доступными в репозиториях Linux-ядрами; Python обновлен до версии 2.7, а GCC до версии 4.5.2; Готовые для внедрения конфигуратор оборудования Dracut и система бесшовной графической загрузки Plymouth (ожидаются в Sabayon 7).
