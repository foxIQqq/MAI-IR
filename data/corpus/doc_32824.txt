Релиз Nide 0.2, интегрированной среды разработки для Node.js

Представлен релиз проекта Nide, в рамках которого развивается интегрированная среда разработки приложений для платформы Node.js, позволяющей организовать обособленное выполнение высокопроизводительных сетевых приложений на языке JavaScript. IDE Nide построена с использованием web-технологий и может быть запущена как локальное приложение или как удалённый сервис, работающий через web-браузер. В комплект также входит утилита nide, позволяющая выполнять типовые действия над проектами и NPM-пакетами из командной строки. Код проекта распространяется под лицензией MIT. Nide может быть запущен на любых платформах, на которых работает Node.js, сборка с обособленным приложением пока доступна только для Mac OS X.
