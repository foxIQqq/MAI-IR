Девятая версия патчей для ядра Linux с поддержкой языка Rust

Для ядра Linux предложена девятая версия патчей с компонентами для разработки драйверов устройств на языке Rust. Новая версия представляет собой урезанный вариант восьмого выпуска, опубликованного несколько дней назад. Набор отличается существенным сокращением размера и оставлением только самого необходимого минимума, достаточного для сборки модуля ядра, написанного на языке Rust. Предполагается, что минимальный патч упростит принятие поддержки Rust в основной состав ядра. После обеспечения минимальной поддержи планируется постепенно наращивать имеющуюся функциональность, перенося и другие изменения из ветки Rust-for-Linux. По сравнению с восьмой версией размер патча сокращён с 40 до 13 тысяч строк кода. Например, девятая версия включает лишь 3% кода (500 строк) crate-пакета "kernel" и 60% библиотеки alloc, позволяющие создавать простейшие модули ядра, использующие тип Vec<i32> и выводящие информацию в лог ядра при помощи макроса "pr_info!".
