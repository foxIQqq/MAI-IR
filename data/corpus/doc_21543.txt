Доступно седьмое обновление Solaris 10 (05/09)

Вышел релиз Solaris 10 05/09 (Update 7) в котором представлены следующие новшества: Управление ресурсами системы и новшества в области администририрования Поддержка использования ZFS клонов при клонировании изолированных зон; В zoneadm появилась опция "-b" для отката примененных обновлений; Добавлены SMF сервисы для IPsec, через которые отныне производится управление IPsec; Для Intel-процессоров добавлена поддержка T-State; Добавлены средства локализации для казахского и украинского языков; Для архитектуры SPARC в комплект включен Fp-scrubber демон Безопасность NAT-Traversal - добавлены средства для обеспечения работы IKM (IPsec Key Management) из сети с NAT; Для IPsec добавлена поддержка более надежных алгоритмов: 2048-bit, 3072-bit, 4096-bit в Diffie-Hellman; sha256, sha384 и sha512; SHA-2 на базе HMAC; SunSSH с поддержкой OpenSSL PKCS#11 движка. Средства для разработчиков: SunVTS 7.0 Patch Set 5 Для отладки и профилирования на системах с архитектурой x86 возможно использование CPU Performance Counter для процессоров Intel; Добавлен набор утилит SUNWpatchutils. Драйверы Новый драйвер hermon; Обновлен код реализации iSCSI Target - улучшена производительность, стабильность и масштабируемость; Добавлен драйвер NetXen 10-GigE; В E1000g драйвере добавлена поддержка Intel ICH10 и Hartwell NIC; В драйвере xge добавлена поддержка MRR (Multiple Receive Rings) и MSI-X. Выпуск Solaris 11, основанный на кодовой базе OpenSolaris, ожидается в середине 2010 года.
