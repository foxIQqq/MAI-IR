Sun Microsystems выпустила пакет виртуализации Virtual Desktop Infrastructure 3

Компания Sun Microsystems выпустила Virtual Desktop Infrastructure (VDI) 3, новую версию пакета для создания виртуальных десктопов (на одном сервере, множество виртуальных десктопов, к которым подключены тонкие клиенты) с гостевыми системами Windows XP, Windows 2000, Windows Vista, OpenSolaris 2008.11 и Ubuntu8.10. Для виртуализации Windows систем используется VMware, для запуска Ubuntu 8.10 и OpenSolaris 2008.11 необходима установка дополнения Sun xVM VirtualBox for Sun VDI 3.0. Система проприетарная, распространяется по подписке из расчета от 40 до 140 долларов в год на один виртуальный десктоп.
