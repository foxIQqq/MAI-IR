Выпуск текстового редактора GNU Emacs 28.2

Проект GNU опубликовал релиз текстового редактора GNU Emacs 28.2. Вплоть до выпуска GNU Emacs 24.5 проект развивался под личным руководством Ричарда Столлмана. В новой версии изменён способ переопределения каталога для установки исполняемого файла. При установке в нестандартный каталог при сборке теперь нужно запускать скрипт 'configure' с опцией '--bindir=' (использование 'bindir=DIRECTORY' в "make install" недостаточно, так как информация, на основе которой вычисляется путь к скомпилированным файлам "*.eln", прописывается в исполняемый файл при сборке). Команда 'kdb-macro-redisplay' переименована в 'kmacro-redisplay'. В остальном в GNU Emacs 28.2 отмечается только исправление ошибок.
