Mozilla разрабатывает версию Firefox OS для Raspberry Pi

Представители Mozilla продемонстрировали на проходящем в Великобритании мероприятии MozFest 2014 проект по портированию мобильной платформы Firefox OS для одноплатных компьютеров Raspberry Pi. В настоящий момент проект находится в состоянии готовности демонстрационного прототипа Foxberry Pi, который можно запустить внутри предлагаемой по умолчанию операционной системы Raspbian. В дальнейшем планируется подготовить легко устанавливаемый обособленный выпуск Firefox OS для Raspberry Pi, который будет снабжён новым интерфейсом, оптимизированным для использования в роли медиапроигрывателя, а также средствами для использования в робототехнике. Из основные планов по развитию проекта отмечается: Довести Foxberry Pi до уровня Raspbian/RPi как окружения для энтузиастов робототехники. Подготовить средства для использования Foxberry Pi в качестве платформы для роботов, дронов и домашней автоматизации, в том числе обеспечить доступ web-приложений к данным сенсоров, управлению двигателями и взаимодействия с разнообразной периферией; Обеспечить работу Foxberry Pi в роли медиапроигрывателя, способного конкурировать с уже существующими медиацентрами для Raspberry Pi. Разработать специализированный пользовательский интерфейс для использования Firefox OS в роли медиацентра; Реализовать возможность использования Foxberry Pi в роли интегрированной среды разработки, рассчитанной на начинающих программистов. Разработать web-ориентированную обучающую среду разработки, похожую на IDLE и Scratch; Предоставить Web API и средства DOM/CSS для разработки роботов, базирующиеся на построении декларативной модели системы с обратной связью. В том числе планируется подготовить Web API для GPIO, SPI, i2c, RS-232, а также CSS-классы для микроэлектронных систем и их симуляции.
