Релиз Debian 6.0.3

Представлена третья корректирующая сборка Debian Squeeze - 6.0.3, в которую включены все выпущенные обновления и устранены недоработки в инсталляторе. Сборка Debian 6.0.3 включает 83 обновления, устраняющих серьезные проблемы со стабильностью, и 52 обновления с устранением уязвимостей. Использовать представленную сборку имеет смысл только для новых установок, системы установленные ранее могут получить все присутствующие в Debian 6.0.3 изменения через штатную систему установки обновлений. Включенные в Debian 6.0.3 исправления проблем безопасности были доступны по мере выхода обновлений через security.debian.org и уже присутствуют на машинах пользователей. Из изменений в инсталляторе Debian 6.0.3 можно отметить: Устранены проблемы с сетевой установкой на платформах IP22 / IP32 (mips); Возможность использования при необходимости старого загрузчика Grub вместо Grub 2; Обновлён пакет с Linux-ядром, в котором отражены важные исправления, устранены проблемы с безопасностью и добавлена поддержка дополнительного оборудования. Например, обновлены сетевые драйверы e1000e, igb, igbvf, r8169, tg3 и broadcom; Обновлено ядро FreeBSD, используемое в инсталляторе GNU/kFreeBSD, в котором обновлен драйвер if_msk для гигабитных Ethernet-карт. Из обновленных пакетов можно выделить: ClamAV - поставка новой версии антивирусного пакета; firmware-nonfree - добавлены прошивки для VIA VT6656, Realtek RTL8105E-1 и RTL8168E-1/2/3; Apache2 - обновление скриптов инициализации и обновление документации; akonadi - поддержка примонтированного по сети $HOME; clive - добавлена совместимой с новой версией плеера с youtube.com; freebsd-utils - добавлены файлы конфигурации и скрипты инициализации для devd, в ifconfig включена поддержка беспроводных сетей; kfreebsd-8 - добавлен бэкпортированный из 8-STABLE драйвер if_msk; включены сборки некоторых дополнительных модулей; kfreebsd-kernel-di-amd64 и kfreebsd-kernel-di-i386 - ядро пересобрано из пакета kfreebsd-8 8.1+dfsg-8+squeeze1; git - различные исправления ошибок; heimdal - возможность использования DES с NFS; grub2 - поддержка разбитых на разделы дисковых образов Xen; дополнительные проверки корректности номеров RAID-массивов; устранение проблемы с определением ATA-устройств, используя драйвер "ata" в kFreeBSD 9; ia32-libs и ia32-libs-gtk - обновление пакетов из веток stable и security; ipmitool - устранены ошибки, приводящие к краху; krb5 - разрешена опция gss_set_allowable_enctypes, что позволяет новым NFS-клиентам использовать NFS-сервер на базе Debian Squeeze без понижения безопасности для не связанных с NFS приложений; libpcap - устранена проблема с повреждением размера снапшота при непрерывном перехвате трафика и проблема с распознаванием устройств при использовании агрегации интерфейсов (bonding); linux-kernel-di-* - ядро пересобрано из пакета linux-2.6 2.6.32-38; mod-gnutls - устранены крахи; mesa - в GLX решена проблема с выводом BadRequest при использовании DRI2Connect для нелокальных клиентов; openssh - отключен вывод лишних данных в лог, когда в несколько органичений "from=" заданы в разных строках файла authorized_keys для одного и того же ключа; postgresql-8.4 - устранено регрессивное изменение в реализации plpgsql, приводящие к пропаданию столбцов rowtype; system-tools-backends - налажена корректная обработка переименования файлов конфигурации; typo3-src - устранены проблемы с кэшированием; tzdata - обновлены данные о переходе на зимнее/летнее время; update-inetd - устранена несовместимость с дополнительными inetd-пакетами, не используемыми по умолчанию; usbutils - обновлен список идентификаторов USB-устройств.
