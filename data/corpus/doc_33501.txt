Релиз Gentoo 12.1 LiveDVD

Анонсирован релиз официальной Live-сборки проекта Gentoo - Gentoo LiveDVD 12.1. LiveDVD подготовлен для платформ x86 и x86_64. В дистрибутиве используется файловая система AUFS с поддержкой записи поверх основы, доступной только на чтение, что позволяет устанавливать дополнительные пакеты. Также предусмотрена возможность перемещения домашнего каталога ($HOME) на дополнительный раздел диска или Flash-накопителя. Для установки системы с LiveDVD подготовлен специальный инсталлятор (Gentoo Install Wizard). ISO-образ имеет размер 3.5 Гб (почти на гигабайт больше чем прошлый выпуск) и поддерживает установку на DVD и USB Flash. Версия для 32-разрядных систем является гибридной и включает дополнительные 64-разрядные сборки библиотек и ядра Linux (gentoo64), что позволяет использовать один LiveDVD как на 32-х, так и на 64-разрядных системах. В гибридной сборке по умолчанию программы собраны для 32-разрядной архитектуры, 64-разрядные сборки программ представлены в отдельном iso-образе (3.69 Гб) livedvd-amd64-multilib, поддерживающем работу только на архитектуре x86_64. Состав LiveDVD достаточно разнородный и содержит приложения, подходящие для различных категорий пользователей: Системные пакеты: Linux ядро 3.3.0 (с дополнительными патчами от разработчиков gentoo), Bash 4.2, Glibc 2.14.1, GCC 4.5.3, Binutils 2.22, Python 2.7.2 и 3.2.2, Perl 5.12.4; Десктоп окружения и оконные менеджеры: KDE 4.8.1, GNOME 3.2.1, Xfce 4.8, Enlightenment 1.0.10, Openbox 3.5.0, Fluxbox 1.3.2, XBMC 11.0, Awesome 3.4.11 и LXDE 0.5.5; Офисные и графические пакеты: LibreOffice 3.5.1.2, XEmacs 21.5.31, GVim 7.3.487, Scribus 1.4.0, GIMP 2.6.12, Inkscape 0.48.3, Blender 2.60, XSane 0.998; Web-браузеры: Mozilla Firefox 11.0, Opera 11.62, Epiphany 3.2.1, Seamonkey 2.8 (из поставки исключены Chromium и Aurora); Приложения для общения: Pidgin 2.10.1, Quassel 0.8.9, Mozilla Thunderbird 11.0, Claws Mail 3.8.0, QTwitter 0.10.0, Irssi 0.8.15; Средства разработки: KDevelop 4.3.0, KDESvn 1.5.5, Bluefish 2.2.2; Мультимедийные приложения: Amarok 2.5, MPlayer 1.0_rc4, VLC 2.0.1, MPlayer2 2.0, DVDAuthor 0.7.0, LAME 3.99.5, FFmpeg 0.10.2, GNOME-MPlayer 1.0.5, SMPlayer 0.7.1; Для людей имеющих проблемы со зрением в состав интегрирована система голосового управления Speakup, позволяющая задействовать синтезатор речи. В поставку включены компоненты для полноценной поддержки ZFS на уровне ядра Linux, созданные силами проекта ZFSOnLinux.
