Linux Mint предоставит возможность использования GNOME 3

В прошлом году, разработчики Linux Mint, пользующегося популярностью дистрибутива на пакетной базе Ubuntu (второе место в рейтиге distrowatch.com), заявили о намерении использовать классическое окружение GNOME 2.x и не использовать GNOME Shell или Unity. Начиная с версии 11.10 дистрибутив Ubuntu перешёл на использование компонентов GNOME 3, таким образом разработчики Linux Mint были поставлены перед выбором: либо поддерживать официально не поддерживаемый GNOME 2.32 своими силами, т.е. существенно отклониться от Ubuntu и по сути развивать форк GNOME 2, либо перейти на использование GNOME 3. В итоге, разработчики Linux Mint объявили, что приняли решение начать внедрение GNOME 3 в будущих выпусках. Присмотревшись к GNOME 3.2 разработчики Linux Mint сделали вывод, что по сравнению с GNOME 3.0 оболочка стала более зрелой и готовой для создания окружения, выглядящего и работающего лучше, чем GNOME 2. Релиз Linux Mint 12, который выйдет в ноябре, по прежнему будет по умолчанию основан на GNOME 2.32. Но дополнительно планируется приступить к интеграции GNOME 3 и выпустить отдельную сборку на основе GNOME 3.2. По возможности поддержка GNOME 2 будет сохранена, для этого разработчики намерены скооперироваться с командой проекта MATE Desktop Environment, развивающей форк GNOME 2.
