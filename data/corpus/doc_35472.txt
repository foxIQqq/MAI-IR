Релиз Chrome OS 23 и обновление браузера Chrome для настольных и мобильных систем

Компания Google выпустила релиз операционной системы Chrome OS 23, основанной на ядре Linux, open source компонентах и web-браузере Chrome 23. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, с недавних пор Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 23 доступна для нетбуков Acer AC700, Samsung Chromebook Series 5 550 и Samsung Series 5, а также для миниатюрного ПК Samsung Chromebox Series 3. Оценить возможности ChromeOS 23 также можно воспользовавшись неофициальными сборками ChromiumOS, развиваемыми энтузиастами и работающими на широком спектре оборудования, - Lime (в состав входят дополнительные драйверы для беспроводных чипов и видеокарт) и Vanilla (сборка, максимально приближенная к оригинальной версии ChromiumOS). Среди добавленных в новой версии улучшений: Включение по умолчанию работы в offline-режиме для ключевых web-приложений; Обновлены поставляемые по умолчанию приложения; Расширены возможности оконного менеджера; Добавлена новая область для вывода уведомлений; Улучшено удобство работы в пользовательском интерфейсе; Переработан интерфейс подключения к Google-аккаунту; Добавлена функция сброса состояния устройства и возвращения всех настроек к заводским установкам; Новая система управления обоями рабочего стола, основанная на использовании облачного сервиса. Одновременно представлены обновления версий браузера Chrome: Chrome 23.0.1271.95 для Windows, Mac OS X, Linux и ChromeFrame c устранением двух опасных, но не критических, уязвимостей. Первая уязвимость связана с возможностью обхода ограничений в коде обработки файловых путей. Вторая уязвимость, за которую выплачено вознаграждение $7331, вызвана обращением к уже освобождённой области памяти в коде работы с источниками медиа потоков; Корректирующее обновление выпуска Chrome for Android 18 для устройств на базе платформ ARM и x86. В новой версии устранена порция ошибок, влияющих на стабильность. Chrome for iOS 23 для iPhone и iPad. В новой версии представлена серия улучшений, среди которых поддержка открытия PDF из других приложений и автоматическое определение кодировки.
