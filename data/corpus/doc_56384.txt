Выпуск дистрибутива Elementary OS 6.1

Представлен выпуск дистрибутива Elementary OS 6.1, позиционируемого в качестве быстрой, открытой и уважающей конфиденциальность альтернативы Windows и macOS. Основное внимание в проекте уделяется качественному дизайну, нацеленному на создание простой в использовании системы, потребляющей минимальные ресурсы и обеспечивающей высокую скорость запуска. Пользователям предлагается собственное окружение рабочего стола Pantheon. Для загрузки подготовлены загрузочные iso-образы (2.47 ГБ), доступные для архитектуры amd64 (для бесплатной загрузки с сайта проекта в поле с суммой пожертвования необходимо ввести 0). При разработке оригинальных компонентов Еlementary OS используется GTK3, язык Vala и собственный фреймворк Granite. В качестве основы дистрибутива используются наработки проекта Ubuntu. На уровне пакетов и поддержки репозиториев Еlementary OS 6.x совместим с Ubuntu 20.04. Графическое окружение основано на собственной оболочке Pantheon, которая объединяет собой такие компоненты, как оконный менеджер Gala (на базе LibMutter), верхнюю панель WingPanel, лаунчер Slingshot, панель управления Switchboard, нижнюю панель задач Plank (переписанный на Vala аналог панели Docky) и менеджер сессий Pantheon Greeter (на основе LightDM). В состав окружения входит набор тесно интегрированных в единое окружение приложений, необходимых для решения задач пользователей. Среди приложений большую часть составляют собственные разработки проекта, такие как эмулятор терминала Pantheon Terminal, файловый менеджер Pantheon Files, текстовый редактор Code и музыкальный проигрыватель Music (Noise). Проектом также развиваются менеджер фотографий Pantheon Photos (ответвление от Shotwell) и почтовый клиент Mail (ответвление от Evolution). Ключевые новшества: Продолжена модернизация центра установки приложений, в котором, в дополнение к пакетам из штатных репозиториев, предлагается более 90 отдельно поддерживаемых программ, распространяемых в формате Flatpak. В новой версии значительно переработано оформление главной страницы, в верхней части которой размещены сменяющие друг-друга баннеры с информацией о недавних выпущенных программах, развиваемых при поддержке проекта. Под баннерами предложена новая секция с 12 недавно обновлёнными приложениями, для которых сразу показана кнопка для быстрой установки. Информация о показываемых приложениях теперь извлекается из локальной БД, а не с внешнего сервера, что позволило до трёх раз ускорить время формирования главной страницы. Переделаны страницы, на которых показывается содержимое выбранных категорий приложений (например, программы для работы со звуком или системные приложения). Информация о программах в категориях теперь показывается в форме сетки с явным разделением бесплатных, платных и не связанных с проектом приложений (загружаемых из внешних репозиториев, таких как Flathub). Добавлена новая категория "Конфиденциальность и безопасность". Категория с развлекательными приложениями перемещена в раздел с играми. Доработано оформление страниц с детальной информацией о выбранном приложении. Вместо вывода предупреждения о содержимом (например, отправке телеметрии, наличии жестоких сцен или обнажённых тел в играх) во время установки, подобные предупреждения теперь постоянно отображаются под верхним баннером с названием приложения. Улучшен выбор цвета верхнего баннера. В нижней части предложена новая секция с подборкой приложений от того же автора. Улучшено отображение AppCenter на небольших экранах. Добавлен новый индикатор прогресса установки, обновления или удаления пакетов, который совмещён с кнопкой "Отмена" и показывается в верхнем правом углу окна. Улучшен интерфейс для покупки платных приложений. В разных частях AppCenter задействован единый виджет для проведения оплаты, в который для приложений, продаваемых по схеме "оплати сколько хочешь", встроена форма для изменения цены. Улучшена работа с программами из внешних репозиториев, таких как Flathub, например, программы из внешних репозиториев сразу отображаются после активации данных репозиториев, без необходимости перезапуска. В случае неудачного поиска в AppCenter пользователю выводится напоминание о возможности установки внешних программ через интерфейс Sideload. При установке драйверов, завязанных на DKMS, AppCenter теперь автоматически установит необходимые заголовочные файлы ядра Linux. Обновлена web-версия AppCenter. Реализована сборка всех распространяемых через AppCenter приложений для платформы ARM64, что позволяет использовать их на таких устройствах, как Pinebook Pro и Raspberry Pi 4. Переделан интерфейс для быстрого переключения между окнами. Вместо индикации текущего окна непосредственно в нижней панели с выводом самого окна на передний план, переключение между окнами теперь управляется при помощи отдельного виджета, выводимого в центре экрана и показывающего пиктограммы и заголовки открытых окон. Добавлена новая анимация вывода диалоговых окон, которые теперь появляются из верхней части родительского окна. При выводе модальных диалогов родительское окно теперь затемняется для того, что бы дать понять о невозможности работы с ним до закрытия диалога. Значительно доработан диалог выбора файлов, в котором появилась возможность создания новых каталогов, добавлена опция для показа файлов только определённого типа и реализован параметр для ограничения доступа к выбранному файлу режимом только для чтения (при доступе к ресурсам вне flatpak-контейнера через систему порталов). Для программ в формате Flatpak предложен новый диалог AppChooser, позволяющий открыть файл в другом приложении. Улучшена реализация тёмного стиля оформления и связанных с ним настроек. Обеспечено следование выбранному стилю приложений из GNOME, а в будущем и приложений из KDE. В меню приложений предоставлена возможность запуска программ с привязкой к определённому GPU на системах с гибридной графикой (например, NVIDIA Optimus). Улучшено определение только что установленных программ. Функции поиска в меню приложений теперь охватывают сетевые разделы, каталоги, добавленные в закладки, а также типовые каталоги, такие как Downloads и Pictures. Улучшен поиск настроек в конфигураторе. Поиск теперь не привязан к выбранному файловому менеджеру. В инсталляторе и интерфейсе начальной настройки упрощён процесс смены имени хоста и решены проблемы с назначением произвольного имени, удобного пользователю (например, пользователь может дать системе имя "Cassidy’s StarBook", которое будет отображаться в конфигураторе Elementary OS, но для системных приложений будет сохранено в /etc/hostname как Cassidys-StarBook). В инсталляторе решены проблемы с неиспользуемыми дисковыми разделами в режиме ручной установки и добавлен вывод предупреждения об особенностях работы при установке в виртуальной машине. В программе для высвобождения свободного дискового пространства (Housekeeping) помимо настройки автоматического удаления временных файлов и очистки корзины, предоставлена возможность удаления загрузок. Интерфейс Housekeeping приближен к дизайну конфигуратора и приложения Welcome. В конфигураторе улучшен раздел настройки масштабирования и параметров экрана, решены проблемы с выбором экрана в многомониторных конфигурациях. В отдельное подменю вынесены разрешения экранов с другим коэффициентом соотношения сторон. На отдельную страницу вынесены настройки масштабирования текста. Предоставлена возможность выставления промежуточных значений масштабирования. В индикаторе управления звуком заменены пиктограммы, улучшена прокрутка ползунка громкости и скрыты некорректные аналоговые устройства вывода. На странице настройки параметров звука в конфигураторе улучшена раскладка настроек и задействованы цветные пиктограммы. В интерфейсе настройки параметров клавиатуры улучшены инструменты для определения собственных горячих клавиш, а также редактирования и удаления горячих клавиш. Во вкладку с настройками раскладки клавиатуры добавлен экранный переключатель раскладок. В индикаторе энергопотребления реализована настройка для включения или выключения показа процентов заряда аккумулятора в панели. На экране с системной информацией выведены детали о графических чипсетах, включая Intel Xe Graphics. Улучшено запоминание состояния Bluetooth при перезапуске и выходе из спящего режима. Упрощено портирование рабочего стола из Elementary OS в другие дистрибутивы, такие как Fedora и NixOS. Расширены возможности службы Online Accounts, в которой повышена надёжность определения методов аутентификации для IMAP и предоставлена возможность редактирования учётных записей CalDAV и IMAP. В почтовом клиенте Mail, реализованном на основе Evolution Data Server (EDS), продолжено доведение возможностей до паритета со старым клиентом на базе движка Geary. Добавлены уведомления о поступлении новых сообщений, показываемые даже при закрытом окне. Улучшена группировка уведомлений. Добавлено контекстное меню, показываемое при клике правой кнопкой мыши на заголовки сообщений в списке. Обеспечено автоматическое выделение следующей нити сообщений после удаления или архивирования текущей. Добавлены новые кнопки для ручного получения новых сообщений и отдельно активируемые фильтры для непрочитанных и помеченных звёздочкой сообщений. В приложении Tasks, помогающем вести списки задач и заметок, реализован диалог подтверждения операций удаления. Повышена производительность обработки списков задач. В календаре-планировщике повышена надёжность отправки и показа уведомлений. Улучшена поддержка часовых поясов. Браузер Web из синхронизирован с GNOME Web 41 (Epiphany 41). Изменено оформление панели инструментов, внесены оптимизации производительности и реализовано скругление углов окна. В настройки добавлены функции поиска. Полностью переделано оформление интерфейсов управления поисковыми движками и паролями. Добавлена настройка для блокирования доступа к локальному хранилищу. По умолчанию отключены поисковые рекомендации Google. В настройках внешнего вида предоставлена возможность добавления своего JavaScript-кода. Добавлена плавная анимация открытия, закрытия и перегруппировки вкладок. В файловом менеджере улучшено перемещение закладок на боковую панель в режиме drag&drop. Решены проблемы с переименованием закладок и добавлена возможность открытия закладок в новой вкладке. Улучшено выделение групп файлов мышью (уже выполненное выделение теперь не пропадает при клике на окружающие области). Обеспечено восстановление открытых вкладок после перезапуска. Улучшено отображение прав доступа для файлов и каталогов на сетевых разделах. Добавлен портал Send Mail для отправки файлов на email. В программе для работы с камерой обеспечено сохранение видео на диск в полном разрешении, несмотря на то, что для повышения производительности картинка в окне продолжает выводиться в низком разрешении. Решены проблемы, возникавшие при подключении нескольких камер.
