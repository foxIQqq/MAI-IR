Выпуск дистрибутива Parrot 4.10 с подборкой программ для проверки безопасности

Доступен релиз дистрибутива Parrot 4.10, основанный на пакетной базе Debian Testing и включающий подборку инструментов для проверки защищённости систем, проведения криминалистического анализа и обратного инжиниринга. Для загрузки предложены несколько iso-образов с окружением MATE (полный 4.2 ГБ и сокращённый 1.8 ГБ), с рабочим столом KDE (2 ГБ) и с рабочим столом Xfce (1.7 ГБ). Дистрибутив Parrot позиционируется как переносная лаборатория с окружением для экспертов по безопасности и криминалистов, основное внимание в которой уделяется средствам для проверки облачных систем и устройств интернета-вещей. В состав также включены криптографические инструменты и программы обеспечения защищённого выхода в сеть, в том числе предлагаются TOR, I2P, anonsurf, gpg, tccf, zulucrypt, veracrypt, truecrypt и luks. В новом выпуске: Осуществлена синхронизация с пакетной базой Debian Testing по состоянию на август 2020 года. Ядро Linux обновлено до версии 5.7. В базовую поставку включены Python 3.8, go 1.14, gcc 10.1 и 9.3. Предложена третья редакция анонимного режима работы Anonsurf, который разделён на три независимых модуля: GUI, демон и инструментарий. GUI, который написан на языке NIM и использует Gintro GTK для формирования интерфейса, предоставляет средства для управления поведением Anonsurf (например, включение активации на стадии загрузки дистрибутива) и мониторинга за состоянием Tor и трафиком. Демон отвечает за запуск и остановку Anonsurf. Утилиты включают CLI с набором консольных команд и dnstool для управления настройками DNS в системе. Добавлена новая ветка платформы для анализа уязвимостей Metasploit 6.0, в которой появилась поддержка оконечного шифрования в meterpreter и предложена новая реализация клиента SMBv3. Подготовлена редакция с рабочим столом Xfce. Добавлены новые пакеты с Security Manager 11 и OpenVAS 7.
