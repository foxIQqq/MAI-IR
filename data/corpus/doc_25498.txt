Вышел MySQL Community Server 5.1.44

Вышел релиз комьюнити сборки MySQL 5.1.44, содержащей исправление 19 ошибок, 4 из которых приводили к краху рабочего процесса (при выполнении "ALTER TABLE ... REORGANIZE PARTITION", при попытке сортировки содержимого полей CHAR(0), при выполнении определенным образом оформленных сложных вложенных запросов и операций полнотекстового поиска). В составе 5.1.44 поставляется версия плагина InnoDB Plugin 1.0.6, который имеет статус кандидата в релизы. Для сервера добавлена опция "--binlog-direct-non-transactional-updates", позволяющая для не поддерживающих транзакции таблиц при использовании репликации организовать прямую запись в бинарный лог, без промежуточного помещения данных в кэш транзакций.
