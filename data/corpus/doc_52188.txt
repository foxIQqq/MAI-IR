Canonical планирует поменять тему оформления в Ubuntu 20.04

Команды разработчиков и дизайнеров из компании Canonical, отвечающие за визуальный стиль и рабочий стол в Ubuntu, планируют задействовать в Ubuntu 20.04 по умолчанию новую тему оформления, которая продолжит развитие текущей темы Yaru, предлагаемой, начиная с Ubuntu 18.10. Если в текущем варианте Yaru доступно два варианта оформления - тёмный (тёмные заголовки, тёмный фон и тёмные элементы управления) и светлый (темные заголовки, светлый фон и светлые элементы управления), то в новой теме появится третий, полностью светлый вариант. Из цветовых изменений также отмечается намерение заменить зелёный фон элементов переключателей на баклажановый цвет. Также ведутся эксперименты по задействованию новых пиктограмм каталогов, которые можно будет ассоциировать с Ubuntu и которые будут обладать должным контрастом при отображении на светлом и тёмном фоне. Кроме того, пользователям будет предложен обновлённый интерфейс для смены вариантов темы оформления. В дальнейшем этот интерфейс планируется расширить возможностью выборочной смены темы для отдельных элементов, например, можно будет сменить только оформление верхней панели или всплывающих уведомлений. Для смены темы на лету, без завершения сеанса, для GNOME Shell планируют реализовать необходимые изменения. При подготовке новой темы ставится задача сохранить узнаваемость бренда, но при этом упростить проверку корректности отрисовки сторонних приложений с данной темой оформления. Для организации проверки без запуска Ubuntu в отдельной виртуальной машине, тема Yaru уже предлагается в формате Flatpak для тестирования в Fedora и в AUR-репозитории Arch Linux. В новой теме планируется продолжить работу по сближению Yaru со штатной темой GNOME (Adwaita). Для отслеживания расхождений на базе GitHub Actions реализован обработчик, который автоматически транслирует все изменения в Adwaita в форме pull-запросов, отправляемых в репозиторий Yaru. Тем временем, разработчики GNOME опубликовали демонстрацию прототипа обновлённой темы оформления GNOME Shell, которую планируется предложить в выпуске GNOME 3.36. Кроме общей полировки темы, визуальные изменения наиболее заметны в календаре/области уведомлений (появились тени) и поиске (Search Overview, изменён фон и группировка результатов). Ускорена отрисовка пиктограмм и исключены лишние перерисовки.
