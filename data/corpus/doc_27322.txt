Компания Vodafone открыла исходные тексты навигационного ПО Wayfinder

Компания Vodafone Group объявила об открытии под лицензией BSD исходных текстов навигационной платформы Wayfinder, а также других разработок компании Wayfinder Systems, которая была куплена Vodafone в 2008 году за 23 миллиона евро. После начала распространения бесплатного навигационного ПО компаниями Google и Nokia, подразделение было признано нерентабельным и работа его была свернута, а все наработки решено перевести в разряд свободного ПО. В число свободных проектов переведены не только навигационные клиенты для мобильных платформ Android, iPhone, BlackBerry, Symbian S60 и телефонов поддерживающих Java ME, а также серверное ПО, инструментарий для создания привязанных к местоположению пользователя сервисов и набор утилит для преобразования картографических форматов и организации управления распределенной инфраструктурой серверов. Исходные тексты проектов уже доступны для загрузки из репозиториев GitHub.com. Код сервера MC2 (Map Central, version 2), выполняющего такие функции как прокладывание маршрутов, геокодирование, управление пользователями, обработка карт, привязка к местоположению и поиск по заданному адресу, написан на языке C++ (вспомогательные скрипты на Python). В качестве первичной операционной системы для сервера в Vodafone использовался Linux-дистрибутив CentOS. Сервер отлично масштабируется и поддерживает создание распределенной сети для обработки данных, состоящей из кластера серверов. В системе активно используются открытые проекты, например, для кеширования задействован Memcached, для хранения данных о пользователях MySQL (карты хранятся в собственной БД, работающей в режиме только для чтения), для шифрования - OpenSSL, для работы с шрифтами - FreeType, для обработки графики - ImageMagick, librsvg2, Cairo и Gd. К сожалению, данные с картами остаются закрытыми, в комплекте имеется только небольшой набор демонстрационных карт. Частично данный недостаток компенсируется тем, что в сервере уже реализованы средства для импорта данных из свободного сервиса OpenStreetMap, а также обеспечена поддержка карт Tele Atlas и Navteq.
