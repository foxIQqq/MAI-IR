Началось бета-тестирование инсталлятора Debian 7.0

Спустя два с половиной месяца после начала альфа-тестирования увидела свет первая бета-версия программы установки Debian GNU/Linux для грядущего релиза "Wheezy" (7.0). Улучшения, добавленные в бета-версию инсталлятора Debian 7.0 (обзор возможностей, запланированных для Debian 7.0, можно посмотреть здесь, а список новшеств альфа-версии инсталлятора - здесь): Изменение оформления экрана загрузки и инсталлятора на визуальную тему Wheezy Joy; ABI Linux обновлён до версии 3.2.0-3, ядро Linux обновлено до версии 3.2.21; Улучшена работа через прокси-серверы; Добавлены проверочные ключи для репозиториев Wheezу; Добавлен модуль virtio-modules-udeb для использования в qemu и модули hyperv-modules для Hyper-V; Улучшена работа режима установки на системах с небольшим размером ОЗУ; Существенно сокращён размер ядра за счёт сжатия образа initrd с использованием метода xz; Доработан интерфейс настройки параметров подключения к сети. Добавлена поддержка аутентификации по публичным ключам; Прекращено формирование сокращённых образов Businesscard, вместо них следует использовать образ netinst; Для Debian/kFreeBSD по умолчанию активирована опция kfreebsd-9 для использования на этапе установки ядра FreeBSD 9.0, если оно присутствует на носителе; Расширена поддержка оборудования: Обновлена база идентификаторов PCI-устройств; Поддержка устройств хранения LaCie Kirkwood NAS; Поддержка чипсетов rt2800usb и Ralink RT5392/RF5372; Улучшенная поддержка архитектур armhf и armel; Поддержка карт Skyhawk; Удалены модули nvidiafb и rivafb; Добавлен драйвер для USB-устройств Synaptics. Из известных проблем отмечается невозможность установки полного десктоп-окружения при наличии образа CD1 без загрузки дополнительных пакетов по сети; проблемы с сетевым соединением при установке десктопа при использовании беспроводного канала связи; проблемы с запуском grub-install для kfreebsd.
