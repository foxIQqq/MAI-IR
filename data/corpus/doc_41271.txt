PaaS-платформа Cloud Foundry переведена под покровительство Linux Foundation

Организация Linux Foundation, курирующая широкий спектр работ, связанных с развитием Linux, объявила о принятии под своё покровительство проекта Cloud Foundry, в рамках которого развивается открытая платформа PaaS (Platform as a Service, платформа как сервис), предназначенная для разработки, развертывания и обеспечения работы приложений в приватном или публичном облачном окружении. Изначально код платформы был открыт в 2011 году компанией VMware. Перевод процесса управления проектом под покровительство некоммерческой организации Linux Foundation позволит создать нейтральную от отдельных вендоров площадку для совместного развития и управления проектом. Ключевыми учредителями проекта выступили компании EMC, HP, IBM, Intel, Pivotal, SAP и VMware. CloudFoundry является полностью открытой PaaS-платформой, которая позволяет сформировать инфраструктуру для выполнения в облачных окружениях конечных приложений на Java (Spring), Go, PHP, Python, Grails, Ruby (Rails, Sinatra), JavaScript (Node.js), Scala и различных языках, работающих поверх JVM. Имеется возможность работы совместно с контейнерной виртуализацией на базе Docker и в облачных системах, работающих с использованием OpenStack. PaaS-платформа, в отличие от IaaS, работает на более высоком уровне, чем выполнение готовых образов операционных систем, избавляя потребителя от необходимости обслуживания ОС и системных компонентов, таких как СУБД, языки программирования, программные фреймворки и т.п. В PaaS от пользователя требуется только загрузка приложения, которое будет запущено в готовом окружении, предоставляемом платформой. Например, Cloud Foundry позволяет программам работать с СУБД MySQL, PostgreSQL, Redis и MongoDB, беря управление данными СУБД на себя.
