Компания Dell выпустила сборку ChromiumOS для нетбука Mini 10v

Разработчики из компании Dell объявили об адаптации выпущенной чуть больше недели назад операционной системы ChromiumOS для работы на нетбукe Dell Inspiron Mini 10v, поставляемом в комплектации с Ubuntu Moblin Remix. Для загрузки доступен дисковый образ (7.5 Гб) для установки на USB Flash. Сборка пока носит неофициальный характер. Из отличий от базовой сборки ChromiumOS отмечается добавление поддержки беспроводных адаптеров Broadcom, которая правда еще находится в экспериментальном состоянии. Например, для обнаружения точки доступа стандартным менеджером соединений ChromiumOS может потребоваться 5-10 минут. Для входа в сборку добавлен дополнительный аккаунт "dell" (пароль тоже "dell"), не требующий наличия сетевого соединения и работающий без аутентификации в Gmail. По умолчанию корневой раздел монтируется в стандартном для ChromiumOS режиме "только для чтения", для продвинутых пользователей в состав сборки доставлен скрипт /etc/mount_rw.sh, позволяющий перемонтировать корневой раздел с возможностью записи (через Ctrl+Alt+T вызывается терминал, а затем запускается sudo /etc/mount_rw.sh). Кроме того, можно отметить, что на прошлой неделе независимыми энтузиастами был сформирован похожий универсальный образ ChromiumOS для работы напрямую с USB Flash (в сжатом виде образ занимает 300 Мб, в распакованном - 2.8 Гб, логин/пароль для входа - facepunch/facepunch)
