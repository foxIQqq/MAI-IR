Доступен Nokia Qt SDK 1.0.1

Анонсирован выход обновления пакета Nokia Qt SDK 1.0.1, ориентированного на упрощение разработки, сборки, отладки и тестирования кросс-платформенных мобильных приложений, основанных на Qt. Nokia Qt SDK базируются на фреймворке Qt 4.6.3, интегрированной среде разработки Qt Creator 2.0.1 и наборе библиотек Qt Mobility 1.0.2. Подготовленное с использованием Nokia Qt SDK приложение без дополнительных модификаций можно скомпилировать для устройств на базе платформ Symbian/S60 3.1/3.2/5.0, Maemo 5 и MeeGo 1.0. Кроме исправлений ошибок в новой версии незначительно расширены возможности симулятора Qt Simulator, обновлен инструментарий Madde для разработки для смартфона N900, добавлена поддержка удаленной компиляции для Mac OS X, обновлен комплект примеров и демонстраций. Дополнительно сообщается о стабилизации кодовой базы Qt 4.7 - решены все критические проблемы, блокирующие выпуск релиза. Таким образом выход Qt 4.7.0 можно ждать уже в ближайшее время. Одновременно уже началась работа над выпуском Qt 4.7.1, в котором будут устранены некоторых проблемы при работе в MacOS X и добавлена поддержка OpenGL для платформы Symbian.
