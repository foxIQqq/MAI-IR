Обновление открытой прошивки wl500g для SOHO-маршрутизаторов Asus

Доступна новая версия базирующейся на Linux прошивки для маршрутизаторов Asus от проекта wl500g, берущей начало от "прошивок Олега" и известной под именем "прошивка от энтузиастов". В проекте поддерживается большое количество SOHO-маршрутизаторов Asus. Код прошивки распространяется в рамках лицензии GPL. Это обновление будет последним перед переходом на Broadcom SDK 5.60. Среди поддерживаемых проектом маршрутизаторов: RT-N10, RT-N12, RT-N16, WL500W, WL500gp, WL500gpv2, WL700g. Основные изменения: Ядро: Из upstream бэкпортированы изменения для arp, блочных устройств, сетевой подсистемы vfs, netfilter, futex, файловых систем; В netfilter-модуле h323 увеличено максимальное количество портов RTP с 4 до 8; Поддержка ext4 бэкпортирована из 2.6.30.9; bcmnat: увеличена скорость исходящего потока; 3g/cdma Добавлена конфигурация для модема ZTE MF-192; Исправления в usb_modeswitch; База устройств usb-modeswitch-data от 2011-10-12; Добавлена поддержка модема Qisda H21 3G, а также патч для usb_modeswitch; Устранено случайное определение USB-устройств в качестве модемов, если модем не выбран основным каналом WAN. Скрипты инициализации и конфигурации (rc): Добавлена поддержка zeroconfig для не dhcp сетей; Предварительная поддержка аутентификации провайдера Convex; Отрегулированы максимальные размеры области /tmp, syslog.log для устройств с ОЗУ <= 16MB; Исправление загрузки syslog. Новые версии программ: dnsmasq 2.59 ntfs-3g 2011.4.12AR.7 quagga 0.99.20 radvd 1.8.2 busybox 1.19.2 Исправления ошибок: Web-интерфейс; E2fsprogs fixes from upstream; Udpxy Patch-1 build 20; Miniupnpd: устранён бесконечный цикл в SendResp_upnphttp(); Исправлены проблемы загрузки на устройствах с MIPS32r1-процессорами; Igmpproxy: исправления, заглушки для igmpv3; Утилита "pwrled" переименована в "gpio".
