Вышел пятый тестовый выпуск openSUSE 11.2

Вышел пятый тестовый выпуск openSUSE 11.2. Релиз openSUSE 11.2 намечен на ноябрь. По сравнению с прошлой тестовой версией можно отметить следующие изменения: Включение в комплект KDE 4.3, первой бета версии OpenOffice.org 3.1.1, Xen 3.4.1 RC10, CUPS 1.3.11, GNOME 2.27.5, PackageKit 0.5, Apache 2.2.12, Choqok 0.6.1 (Twitter/identi.ca клиент), Arora 0.8 (web-браузер на базе движка WebKit), VirtualBox 3.0.2, Firefox 3.5.1, SeaMonkey 2.0b1; Сборка Linux ядра 2.6.31-rc4 со специальной оптимизацией для десктоп-систем. Базовые новшества openSUSE 11.2: Пакеты дистрибутива собраны GCC 4.4.1 В качестве файловой системы по умолчанию задействована Ext4 В состав Live CD включен Gimp; Для сжатия данных на LiveCD отныне используется алгоритм сжатия LZMA; По умолчанию LiveCD содержит переводы для немецкого, французского, итальянского, польского и русского языков; В Qt версии конфигуратора YaST появилась поддержка настраиваемого внешнего вида. В YaST добавлены средства работы с кешем пакетов; В Zypp улучшены средства работы с зеркалами репозиториев пакетов; В интерфейсе для настройки параметров дисплея gnome-display-properties добавлена поддержка автоматического сброса настроек в случае проблем. Ведется разработка web-интерфейса к YaST для осуществления удаленного управления дистрибутивом Появление установочных образов для USB Flash и утилита для их подготовки Улучшение поддержки субноутбуков Образ LiveCD теперь может быть использован и на USB Flash; В состав LiveCD добавлен Midnight Commander, а в KDE-вариант LiveCD программа YaKuake; Для YaST создан новый центр управления конфигурацией (YaST Control Center) с интерфейсом на базе Qt4;
