SUSE Cloud - решение на базе OpenStack для поддержания облачных инфраструктур

Подразделение SUSE представило новое решение для развёртывания и поддержания работы облачных инфраструктур, в котором используется сочетание компонентов платформы OpenStack с конструктором образов виртуальных окружений SUSE Studio и системой для управления инфраструктурой Linux-серверов SUSE Manager. Из компонентов OpenStack в продукте задействованы: инструментарий Nova для управления работой больших групп VPS-серверов, сервис доставки и хранения образов виртуальных машин Glance, система идентификации Keystone и управляющий web-интерфейс Dashboard. В настоящее время для тестирования доступна экспериментальная сборка новой платформы, построенная на базе SUSE Linux Enterprise Server 11 SP1. Кроме того, подготовлен репозиторий, который можно использовать для установки пакетов с компонентами OpenStack в системах на базе SLES 11 SP1 и openSUSE 11.4.
