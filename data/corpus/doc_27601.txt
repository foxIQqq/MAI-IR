Доступна третья бета-версия Firefox 4

Доступна для тестирования третья бета-версия web-браузера Firefox 4. Подробный обзор новшеств Firefox 4 можно прочитать в тексте анонса первой бета-версии. Из внесенных с момента прошлой тестовой версии улучшений можно отметить: Для платформы Windows 7 добавлена поддержка режима multi-touch, позволяющего управлять работой браузера используя интуитивно понятные жесты (видеодемонстрация), например, передвигать содержимое через удержание касания к сенсорному экрану, масштабировать содержимое двумя пальцами, вращать объекты в web-приложениях, используя специальный API. Использование C++-представления для организации работы с переменными JavaScript, позволило повысить эффективность выполнения скриптов, требующих интенсивных численных вычислений, например, для обработки графики. Для всех типов переменных теперь используется единая 64-разрядная переменная jsval, в первых 32-битах которой закодирован тип, позволяющий отличить указатели на строки, объекты, числа с плавающей точкой и целочисленных переменных; В поставку входит разработанное в рамках проекта Mozilla Labs Test Pilot дополнение, предназначенное для приема комментариев от пользователей и для организации анонимного анализа поведения пользователя с целью накопления статистики для последующей оптимизации интерфейса (дополнение активируется только после согласия пользователя).
