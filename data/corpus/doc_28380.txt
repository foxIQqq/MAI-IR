Компания SkySQL представила альтернативный вариант MySQL Enterprise

Компания SkySQL, основанная уволившимися из Oracle разработчиками MySQL, финансируемая несколькими бывшими инвесторами MySQL AB и возглавляемая Ульфом Санбергом (Ulf Sandberg), бывшим вице-президентом по предоставлению глобальных сервисов MySQL, анонсировала выпуск альтернативной промышленной сборки MySQL - SkySQL Enterprise. Продукт распространяется по подписке и снабжен полноценной технической поддержкой (помощь в решении проблем, консультации по настройке, устранение последствий сбоев) и консалтинговым сервисом (планирование внедрения и проведение оптимизации). В пакет включены: MySQL Server и MariaDB Server; SkySQL Enterprise Monitor (клон MySQL Enterprise Monitor), позволяющий выявлять узкие места, производить тюнинг производительности, осуществлять мониторинг, выполнять операции по обслуживанию базы; SkySQL Visual Editor (клон MySQL Query Analyzer) - GUI-интерфейс для выполнения запросов, просмотра результатов и проведения анализа их выполнения.
