Intel открыл код реализации OpenCL, выполняемой на CPU

Компания Intel открыла исходные тексты OpenCL CPU RT (OpenCL CPU RunTime), реализации стандарта OpenCL, предназначенной для выполнения OpenCL-ядер на центральном процессоре. Стандарт OpenCL, определяет API и расширения языка С для организации кросс-платформенных параллельных вычислений. Реализация насчитывает 718996 строк кода, распределённого по 2750 файлам. Код адаптирован для интеграции с LLVM и будет предложен для включения в основной состав LLVM. Исходные тексты открыты под лицензией Apache 2.0. Из альтернативных проектов, развивающих открытые реализации OpenCL можно отметить PoCL (Portable Computing Language OpenCL), Rusticl и Mesa Clover. Реализация от Intel оценивается как демонстрирующая более высокую производительность и обеспечивающая более широкую функциональность.
