Вышел релиз LiveCD дистрибутива Parted Magic 4.4

Вышел релиз LiveCD дистрибутива Parted Magic 4.4. Дистрибутив предназначен для восстановления системы после сбоя, имеет размер 90 Мб, построен на базе Linux From Scratch (LFS) и графического окружения Xfce. В новой версии: Добавлена возможность выхода в сеть через модем посредством конфигуратора 'Start Network'; Добавлены модули ALSA для обеспечения работы звуковой подсистемы; Задействована файловая система Unionfs для организации записи в директории в хранилище, доступном только для чтения; Понижено потребление памяти, дистрибутив может работать целиком загрузившись в ram-диск размером 256 Мб. При загрузке теперь запускается SSH сервер и генерируются все необходимые для удаленного доступа ключи; Обновлены версии программ: pcmanfm-svn878, fsarchiver-0.5.8, firefox-3.5.2, clonezilla-2.3.3-68, linux-2.6.30.4, drbl-1.9.4-47, e2fsprogs-1.41.8, usbutils-0.84, gtk+-2.16.5, hdparm-9.16, lftp-3.7.15, parted-1.9.0, gpicview-0.2.1, lxpanel-0.5.0, pigz-2.1.5, libburn-0.6.8.pl00, lxtask-0.1.1, safecopy-1.5, libisofs-0.6.20, mount-gtk-1.0.4, xfburn-0.4.2, gdisk-0.3.1, g4l30n. Из базовых программ можно отметить: GParted, TestDisk, PhotoRec, Partition Image, Xarchive, Xfburn, ISO Master, Leafpad, Firefox, Grsync. Из утилит командной строки присутствуют dd, ddrescue , cfdisk, fdisk, tcpdump. Имеется поддержка восстановления файловых систем ext2, ext3, ext4, btrfs, FAT16, FAT32, HFS, HFS+, JFS, NTFS, ReiserFS, Reiser4, XFS.
