Новая версия менеджера фотографий Shotwell 0.13

Представлен релиз программы для управления коллекцией фотографий Shotwell 0.13.0. Shotwell предоставляет удобные возможности каталогизации и навигации по коллекции, поддерживает группировку по времени и тегам, предоставляет инструменты для импорта и конвертации новых фотографий, поддерживает выполнение типовых операций по обработке изображений (вращение, устранение эффекта красных глаз, корректировка экспозиции, оптимизация цветности и т.п.), содержит средства для публикации в социальных сетях, таких как Facebook, Flickr и Picasa. Основные улучшения Shotwell 0.13: Поддержка публикации контента в Tumblr; Налажена работа с датами до 1969 года; Опция для удаления конфиденциальной информации из метаданных перед публикацией фотографий в социальных сетях; Переход на использование класса GtkApplication; Поддержка автоматической перегенерации миниатюр фотографий, что значительно упростило перенос коллекции между компьютерами (все недостающие миниатюры создаются при запуске); Модуль для публикации в сервисе Picasa переведён на использование механизма аутентификации OAuth. Готовые бинарные сборки доступны для Linux-дистрибутивов Ubuntu и Fedora.
