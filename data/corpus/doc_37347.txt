Один из основателей Apache выбран президентом учреждённого Microsoft фонда Outercurve

Организация Outercurve Foundation (бывшая CodePlex Foundation), учреждённая компанией Microsoft для поддержки открытых проектов для платформы Windows, объявила о назначении на должность президента Джима Ягельски (Jim Jagielski), одного из создателей Apache Software Foundation. Джим является членом управляющих советов организаций OSI и Apache Software Foundation (также занимал должность президента Apache Software Foundation), входит в число основных разработчиков http-сервера Apache и сервера приложений Tomcat, принимает участие в разработке таких проектов, как Sendmail, BIND, PHP, Perl и FreeBSD. Интересно, что в начале года Росс Гардлер (Ross Gardler), вице-президент Apache Software Foundation, также перешёл на работу в Microsoft. Outercurve Foundation позиционирует себя в роли посредника между корпорациями и сообществом независимых разработчиков. Цель организации - сблизить разработчиков открытого и коммерческого ПО, дать им возможность работать совместно и привлечь авторов открытых проектов к платформе Windows. Поддержка хостинга CodePlex осуществляется компанией Microsoft и не контролируется фондом.
