Кандидат в релизы ROSA Desktop 2012

Доступен для тестирования кандидат в релизы дистрибутива ROSA Desktop 2012. Для загрузки подготовлена DVD-сборка, размером 1.5 Гб. Финальную версию ROSA Desktop 2012 планируется выпустить через три недели. Особенности нового выпуска: На 30-40% сокращено время запуска рабочего стола; Добавлены новые темы оформления для GRUB2 и Plymouth; Существенно обновлена тема оформления рабочего стола. Наглядно выделены области за которые можно передвигать окна приложений. В движок тем QgtkStyleAlt добавлена возможность перетаскивать окна KDE-приложений за меню и панель не нажимая дополнительных клавиш модификаторов; Добавлено 30 новых иконок, общий набор иконок был доработан для улучшенного отображения в GNOME 3. Новые иконки удостоились рецензии в разделе Бизнес-линч на сайте студии Артемия Лебедева; В интерфейс для отслеживания активности пользователя в системе TimeFrame добавлена возможность отправки сообщений в социальные сети; В интерфейсе для запуска приложений SimpleWelcome упрощена работа с группами, добавлена возможность закрепления последних приложений на вкладке "Приветствие", поддержка очистки списка последних документов, налажена работа с системным интерфейсом Drag and Drop; В KLook значительно улучшено качество рендеринга, добавлена возможность работать с удаленными файловыми системами по протоколам SMB, WebDav, FTP, и т.п.; Задействовано ядро Linux 3.6.7 с модификациями nrj для улучшения работы в Desktop режиме. В репозитории доступны также обычные ядра, PAE и RT, а также ядра с поддержкой CK1 +BFS +BFQ +TOI. Обновлён графический стек и видеодрайверы (X-server 1.13, Mesa 9). Обновлены версии программ: Systemd 194, Dracut 024, Grub 2, Qt4 4.8.4, KDE 4.9.4, Firefox 17, Thunderbird 17, Libreoffice 3.6.2, Chromium 23.0.1271.95, ffmpeg 1.0.1, RPM 5.4.10, gcc 4.7, glibc 2.15; В загрузочные образы интегрирована поддержка EFI/UEFI, в релизе ожидается поставка заверенного загрузчика для работы на системах с активным UEFI Secure Boot; Исправлены проблемы с обнаружением оборудования, например, тачпадов Synaptic, принтеров и беспроводных адаптеров.
