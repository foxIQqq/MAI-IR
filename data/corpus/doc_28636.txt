Apple присоединяется к OpenJDK с целью развития Java для MacOS X как открытого проекта

Компания Apple анонсировала своё присоединение к работе над проектом OpenJDK с целью открытия Java-технологий для Mac OS X. Компания Apple готова предоставить сообществу ключевые компоненты, инструменты и технологии, необходимые для реализации поддержки платформы MacOS X в Java SE 7, включая 32- и 64-битный вариант виртуальной машины HotSpot, библиотеки классов, код сетевого стека и основу для создания нового графического клиента. Утверждается, что Apple продолжит поставку собственного варианта Java SE 6 для Mac OS X Snow Leopard и грядущего релиза Mac OS X Lion. Но создание Java SE 7 и более новых версии Java для Mac OS X будет делегировано компании Oracle. По мнению Apple наилучшим вариантом для пользователей будет вариант, когда они смогут получить наиболее новые и безопасные версии Java непосредственно от Oracle. При этом компания готова оказать всяческое содействие Oracle в направлении обеспечения поддержки платформы MacOS X.
