Релиз дистрибутива Superb Mini Server 1.5.4

Представлен релиз Linux-дистрибутива Superb Mini Server 1.5.4, основанного на Slackware Linux и предназначенного для быстрого развертывания серверов для обслуживания предприятий среднего и малого бизнеса. Дистрибутив поддерживает развертывание web-сервера, межсетевого экрана, почтового сервера, сервера печати и других сетевых служб. Настройка параметров сервисов осуществляется через web-интерфейс, реализованный на базе проекта Webmin. Дистрибутив работает в Live-режиме, размер загрузочного iso-образа составляет 368 Мб. Из входящих в комплект программ можно отметить: Linux-ядро 2.6.35.7, glibc 2.12.1, gcc 4.5.1, clamav 0.96.5, dovecot 1.2.16, apache 2.2.17, hylafax 5.4.2, iptables 1.4.10, mysql-5.1.53, postfix 2.7.2, proftpd-1.3.3c, samba 3.5.6, squid 3.1.9, vsftpd-2.3.2, OpenbravoERP 2.50MP23, asterisk 1.6.2.13, openldap 2.4.23 а также WebERP и PhpLDAPAdmin. Дополнительно в состав входит TorrentFlux, написанный на языке php BitTorrent-клиент, управляемый через web.
