Выпуск Raspbian 2020-02-05, дистрибутива для Raspberry Pi. Новая плата HardROCK64 от проекта Pine64

Разработчики проекта Raspberry Pi опубликовали обновление дистрибутива Raspbian, основанного на пакетной базе Debian 10 "Buster". Для загрузки подготовлены две сборки - сокращённая (433 Мб) для серверных систем и полная (1.1 Гб), поставляемая с пользовательским окружением PIXEL (ответвление от LXDE). Для установки из репозиториев доступно около 35 тысяч пакетов. В новом выпуске: В файловый менеджер, основанный на PCmanFM, в верхнюю часть боковой панели добавлена секция "Places", предоставляющая быстрый доступ к примонтированным накопителям и часто используемым файловым путям. В панель инструментов добавлена кнопка для создания нового каталога. В режиме просмотра каталогов в боковой панели налажено отображение индикатора наличия подкаталогов; В список рекомендованных приложений добавлен экранный ридер Orca, адаптированный для управления рабочим столом PIXEL и приложениями на базе GTK и Qt (c приложениями на базе других тулкитов, такими как Thonny, Sonic Pi и Scratch, а также с Chromium, экранный ридер пока не совместим (рекомендуется использовать Firefox или свежий выпуск Chromium 80, в котором проблемы решены)); Обучающая среда визуального программирования Scratch 3 обновлена до версии 1.0.4, в которой добавлена возможность загрузки файлов из командной строки, а в расширение Sense HAT добавлены новые блоки ‘display stage’ и ‘display sprite’; До версии 3.2.6 обновлена интегрированная среда разработки Thonny, которая может использоваться для обучения новичков написанию приложений на языке Python. Thonny сочетает в одном приложении простой для освоения начинающими интерфейс и продвинутые возможности для опытных программистов, такие как пошаговое выполнение кода и инспектирование переменных. В новом выпуске проведена работа по увеличению производительности при отладке проектов; В список рекомендованных приложений добавлен набор игр Code the Classics, написанных на языке Python; Функции настройки внешних звуковых устройств, перенесены из отдельного приложения Audio Device Preferences в панельный апплет управления звуком; В конфигураторе выделена отдельная секция с настройками дисплея, в которой появились новые пункты для управления отключением экрана и дублирования пикселей при масштабировании; Изменено значение комбинации Ctrl-Alt-Delete, которая вместо вызова менеджера задач теперь вызывает диалог завершения работы (для вызова менеджера задач следует использовать Ctrl-Shift-Escape); Пакет Mesa обновлён до версии 19.3.2 с поддержкой OpenGL ES 3.1; Улучшена поддержка многомониторных конфигураций; В OpenSSL активированы оптимизации на базе инструкций NEON. Дополнительно можно отметить анонсирование проектом Pine64 конкурирующей с Raspberry Pi 4 платы HardROCK64, которая оснащена 6-ядерным CPU (SoC Rockchip RK3399 с CPU ARM Cortex A53 и GPU Mali T860 MP4, такой же как в Pinebook Pro и ROCKPro64). Плата совместима с Linux-образами, подготовленными для ROCKPro64. Устройство оснащено поддержкой 802.11ac Wi-Fi, Bluetooth 5, Gigabit Ethernet, microSD, 2xUSB 3.0, 2xUSB 2.0, GPIO, SPI flash, eMMC, CSI, DSI и IR-приёмником. Отличия от платы ROCKPro64 сводятся к отсутствию портов USB-C и PCIe. Стоимость платы с 1 ГБ ОЗУ составляет $35, с 2 ГБ - $45, 4ГБ - $55 (для сравнения плата ROCKPro64 с 2ГБ ОЗУ продаётся за $60). В продажу плата HardROCK64 поступит ориентировочно в апреле.
