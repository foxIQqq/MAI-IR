Вышел Icaros Desktop 1.2.6, дистрибутив операционной системы AROS

Вышел релиз Live-дистрибутива свободной AmigaOS-совместимой десктоп-ориентированной операционной системы AROS - Icaros Desktop 1.2.6. Icaros Desktop позволяет на обычном ПК запустить Amiga-подобное десктоп окружение, для которого в комплект включено множество развлекательных приложений и игр. Для загрузки доступны два загрузочных образа: сокращенный (190 Мб) и полный (1 Гб). Особенности нового выпуска: Улучшенная поддержка 3D-графики, в состав интегрирован пакет Mesa 7.1 и обновлен порт технологии Gallium3D, позволяющий использовать созданные в рамках X.Org наработки для поддержки аппаратной 3D-акселерации для современных видеокарт. Добавлен новый драйвер для видеокарт Intel GMA, обновлен драйвер Nouveau для видеокарт NVIDIA. В состав включены библиотеки для поддержки EGL и OpenVG; В поставку включен эмулятор Bochs, для которого подготовлены образы для запуска DLX Linux и FreeDos. Обновлены эмулятор игровых приставок MAME и эмулятор платформы ZX Spectrum - Fuse. Добавлены новые эмуляторы: эмулятор платформ Sega/Game Gear - Dega и эмулятор Atari ST - Hatari; Реализована возможность использования родного графического режима AROS, обеспечивающего свойственный платформе AmigaOS метод работы с драйверами; Осуществлена синхронизация с кодом проекта AROS по состоянию на 26 января; Расширена поддержка оборудования, добавлены дополнительные драйверы для беспроводных карт; Обновлена версия системы AmiBridge, позволяющей выполнять игры и демонстрации для платформы Amiga в формате ADF.
