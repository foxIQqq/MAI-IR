Релиз дистрибутива SystemRescueCd 2.0.0

Представлен релиз SystemRescueCd 2.0.0, специализированного LiveCD-дистрибутива на основе Gentoo Linux, предназначенного для восстановления системы после сбоя. В качестве графического окружения используется Xfce. Размер iso-образа - 280 Мб. Из изменений можно отметить: Стандартное Linux-ядро обновлено до версии 2.6.35.10, в роли альтернативного ядра используется 2.6.36.2; Для сжатия образа файловой системы SquashFS вместо LZMA теперь используется формат XZ; Обновлен поставляемый в составе дистрибутива комплект прошивок; Для видеокарт Nvidia теперь по умолчанию задействован видеодрайвер xf86-video-nouveau; Обновлены версии программ: xorg-server 1.9.3, GNU Parted 2.3 (с патчами из Ubuntu), GParted 0.7.1, fsarchiver 0.6.12, aufs 2.1; Обновлена начинка образа загрузочного RAM-диска initramfs; Для настройки параметров сети задействован Network-Manager 0.8.x; Прекращена поддержка альтернативного графического сервера Xfbdev; В комплект включена 64-разрядная версия программы для тестирования работы памяти memtester (/usr/sbin/memtester-64bit); Добавлен консольный оконный менеджер tmux (аналог утилиты screen) В опции "dodhcp" реализована возможность указания нескольких сетевых интерфейсов ("dodhcp=eth0,eth2").
