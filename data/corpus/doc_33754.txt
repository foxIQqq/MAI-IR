Корректирующий релиз LibreOffice 3.5.3

Организация Document Foundation анонсировала корректирующий выпуск офисного пакета LibreOffice 3.5.3, в котором представлены только исправления ошибок и обновления файлов с переводами. Готовые установочные пакеты подготовлены для платформ GNU/Linux, Mac OS X и Windows. Финальный релиз 3.5.3 полностью аналогичен ранее представленному второму кандидату в релизы. В анонсе отмечается, что в течение апреля к проекту присоединилось 34 новых разработчика, при этом от 8 из них поступило более одной порции изменений. Дополнительно сообщается об одобрении 10 заявок студентов для работы в рамках программы Google Summer of Code 2012. Среди задач, которые попытаются решить студенты: оптимизация производительности Calc, улучшение Lightproof, разработка системы совместного редактирования таблиц с использованием Telepathy (анонс прототипа), фильтр импорта для Microsoft Publisher, поддержка экспорта подписанных PDF-документов, разработка средств для управления со смартфона, новый пользовательский интерфейс для выбора шаблонов, написанный на Java интерфейс для просмотра документов на платформе Android, улучшенный фильтр экспорта в формате SVG, расширение тестового набора.
