Релиз композитного менеджера cairo-compmgr 0.3.0

Вышел релиз cairo-compmgr 0.3.0 (Cairo Composite Manager), универсального композитного менеджера, использующего для рендеринга библиотеку Cairo. Cairo Composite Manager легко расширяем при помощи плагинов, которые могут быть использованы для создания интересных визуальных эффектов на рабочем столе. Изменения в версии 0.3: Прекращено использование бэкендов Glitz и OpenVG, оставлена поддержка XRender и программного ускорения. Ведется разработка нового бэкенда на базе OpenGL, который позволит использовать в cairo-compmgr новые 3D-эффекты; Приведены к более стандартному виду окна настройки и создания снапшотов; Язык Vala включен в список обязательных зависимостей, необходимых для сборки cairo-compmgr; Добавлен плагин "Мозаика", включаемый нажатием сочетания "Управляющая кнопка" + Tab; Добавлен экспериментальный плагин для клонирования окон, позволяющий внешнему приложению получить контент из любого видимого окна; Добавлена опция "--configure" для запуска процесса настройки из командной строки; Устранены накопившиеся ошибки и решены проблемы с производительностью.
