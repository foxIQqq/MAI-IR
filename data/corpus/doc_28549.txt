Доступны дополнительные редакции Sabayon Linux 5.4: E17, XFCE, LXDE, SpinBase/OpenVZ и ServerBase

Представлены пять экспериментальных редакций основанного на Gentoo дистрибутива Sabayon Linux 5.4: E17 Spin (686 Мб) - сборка с десктоп-окружением на базе Enlightenment 17. XFCE Spin (681 Мб) - сборка с десктоп-окружением на базе XFCE 4.6.2. LXDE Spin (655 Мб) - сборка с десктоп-окружением на базе LXDE 0.5; SpinBase/OpenVZ (400 Мб) - сборка для быстрого развертывания гостевых систем, запущенных внутри контейнера OpenVZ. ServerBase (400 Мб) - сборка для создания серверных систем, включающая оптимизированное для серверных задач Linux-ядро 2.6.35; Сборки основаны на Linux-ядре 2.6.35 c патчами Tuxonice и AUFS, для настройки сети задействован NetworkManager 0.8, в состав входят NTFS-3G, Firefox 3.6.11, VirtualBox.
