Opera 10.6 alpha 1. Версия 10.5x для Unix выпущена не будет

Выпущена первая альфа-версия веб-браузера Opera 10.6, которая включает в себя следующие изменения: Улучшена производительность JavaScript, DOM и CSS. Сделан более логичным внешний вид веб браузера: появилось слово "Menu" в кнопке, вызывающей меню; встроенные в веб-браузер закладки (такие как, например, "История" посещений) получили красивые иконки, вместо отображения предпросмотров (preview); внешний вид всплывающего предпросмотра веб-страниц также был улучшен; На платформе MacOS продолжается замена устаревшего кода API Carbon на API Cocoa. Было заявлено, что стабильная версия ветки 10.5x для Unix будет пропущена, вместо неё будет выпущена версия 10.6, с кодом синхронизированным для всех поддерживаемых платформ.
