Началось альфа-тестирование инсталлятора Debian 11 "Bullseye"

Началось тестирование первой альфа-версии инсталлятора следующего значительного релиза Debian - "Bullseye". Релиз ожидается примерно через полтора-два года. Ключевые изменения в инсталляторе: Упоминания CD и CD-ROM заменены на "установочные носители"; В apt-setup переработана генерации строк в файлах sources.list для обновлений, связанных с устранением проблем с безопасностью. Строки {dist}-updates переименованы в {dist}-security. В sources.list разрешено отделение блоков "[]" несколькими пробелами; Прекращена установка промежуточного пакета apt-transport-https; Прекращено создания файла my-at-spi-dbus-bus.desktop в профиле пользователя (at-spi2-core теперь всегда запускает шину at-spi); В качестве хоста по умолчанию для собственных зеркал задействован "deb.debian.org"; Обеспечено выставление параметра gfxpayload=keep в подменю загрузчика, что решает проблемы с нечитаемыми шрифтами на экранах HiDPI при загрузке образов для установки по сети через EFI; Документация преобразована в формат DocBook XML 4.5 В grub2 добавлено определение модулей для подписанных образов для UEFI; Обеспечена установка пакета cryptsetup-initramfs вместо cryptsetup; Прекращено формирование образов для плат QNAP TS-11x/TS-21x/HS-21x, QNAP TS-41x/TS-42x и HP Media Vault mv2120; Добавлена поддержка ARM-платы Olimex A20-OLinuXino-Lime2-eMMC; В mini.iso обеспечена работа режима загрузки по сети в EFI для платформы ARM; Обеспечена установка пакетов для поддержки систем виртуализации, если обнаружен запуск в окружениях под их управлением; В образ ядра Linux добавлен модуль thermal_sys; Добавлен пакет virtio-gpu для графического вывода в виртуальных машинах; Для Rasperry Pi Compute Module 3 бэкпортирована поддержка DTB (Device Tree).
