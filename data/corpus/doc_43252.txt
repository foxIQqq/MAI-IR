Увидел свет Firefox 42

Анонсирован релиз web-браузера Firefox 42, а также мобильной версии Firefox 42 для платформы Android. В ближайшие часы ожидается обновление ветки с длительным сроком поддержки Firefox 38.4.0, а также почтового клиента Thunderbird 38.0.4 и пакета SeaMonkey 2.39. В скором времени на стадию бета-тестирования перейдёт ветка Firefox 43 и будет отделён Firefox Developer Edition 44. В соответствии с шестинедельным циклом разработки релиз Firefox 43 намечен на 15 декабря, а Firefox 44 на 26 января. Основные новшества: В режиме соблюдения конфиденциальности (Private window) включена блокировка элементов страниц, которые могут использоваться для отслеживания поведения пользователя и перемещений между сайтами. Для борьбы с отслеживанием применяется метод блокирования внешних JavaScript-скриптов, изображений и iframe-страниц с сайтов, занесённых в чёрный список disconnect.me. На кнопки вкладок добавлен индикатор, сигнализирующий о воспроизведении звука в открытом во вкладке сайте и позволяющим в один клик приглушить его; Внесены улучшения в менеджер учётных записей: расширена эвристика при сохранении логинов и паролей, добавлена возможность редактирования и просмотра всех логинов, поддержка копирования/вставки логинов/паролей через контекстное меню, появилась функция для миграции базы сохранённых паролей из Chrome и IE; В интерфейсе настройки переработано оформление разделов управления безопасностью и режимом конфиденциальности; Изменено оформление индикаторов уровня защищённости доступа к просматриваемой странице. Наиболее существенным изменением является то, что сайты, использующие сертификаты, верифицированные по привязке к домену (DV), теперь помечаются как защищённые. Т.е. верификация по домену приравнена к верификации по владельцу домена (EV).Кроме того, доступ по HTTPS к страницам, содержащим загружаемые по HTTP ресурсы, теперь более явно сигнализирует о проблемах с безопасностью. Улучшения WebRTC: поддержка IPV6, настройки для управления генерацией ICE и видимостью IP, для расширений добавлены хуки для разрешения/запрета инициирования вызова или ответа, расширены средства мониторинга и управления тем, какие устройства используются в getUserMedia; Увеличена производительность интерактивных web-сайтов, работа с которыми приводит к большому числу изменений стилей; API Media Source Extension для видео HTML5 теперь доступно для всех сайтов, а не только для YouTube; Реализация встроенного объекта Reflect, определённого в спецификации ECMAScript 2015 и предоставляющего серию методов для перехвата JavaScript-операций; Поддержка вызовов ImageBitmap и createImageBitmap(), позволяющий получить картинку из содержимого тегов canvas, img и video, в том числе можно определить сохранить соответствующий заданным координатам блок; Поддержка Push API, позволяющего устанавливать обработчики получаемых со стороны сервера сообщений, срабатывающие даже в ситуации, когда web-приложение не активно. Обработчики задаются через Service Workers и не зависят от времени жизни приложения, что позволяет создавать новые классы web-приложений, способные обрабатывать запросы даже после закрытия страницы с приложением; Просмотр кода страницы теперь производится не в отдельном окне, а во вкладке; Добавлен режим удалённой отладки сайта через WiFi, позволяющий отлаживать открытый в Firefox для Android сайт без соединения устройства через USB и без запуска ADB; В средствах для web-разработчиков реализована возможность отслеживания выполнения асинхронных вызовов, установленных через setTimeout, обработчики событий DOM и обработчики Promise; В WebIDE добавлена возможность настройки симулятора Firefox OS для симулирования интерфейса для смартфонов, планшетов и TV. В режиме инспектирования добавлен набор предустановленных CSS-фильтров и возможность сохранить собственный фильтр для дальнейшего использования. Осуществлённый в бета-версии перевод сборок для платформы GNU/Linux на GTK3+ не вошёл в релиз. В версии для платформы Android: Добавлена страница "about:logins", отображающая список всех сохранённых логинов и предоставляющая возможность редактирования их параметров; Возможность одновременного открытия нескольких ссылок (ссылки вначале помечаются на странице, а потом инициируется их открытие в разных вкладках); Возможность открытия в фоне внешних URL, переданных Android-приложениями; Режим семейной навигации для урезанных профилей Android; Возможность прямого голосового ввода из адресной строки; Реализация навигации в панелях через прокрутку вкладок. Как обычно следует отметить, что кроме новшеств и исправления ошибок в Firefox 42 устранена серия уязвимостей, из которых несколько помечены как критические, т.е. могут привести к выполнению кода злоумышленника при открытии специально оформленных страниц. В настоящее время информация с подробностями об исправленных проблемах безопасности недоступна, ожидается, что перечень уязвимостей будет опубликован в течение нескольких часов. Тем временем, в Firefox Developer Edition 44 представлено заслуживающее внимания нововведение для разработчиков - визуальный редактор анимации и фильтров CSS, позволяющий наглядно анализировать, приостанавливать и изменять любой анимированный элемент на странице, а также добавлять, удалять и корректировать CSS-фильтры на лету, сразу наблюдая результат.
