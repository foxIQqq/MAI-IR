Анонсирован специализированный вариант библиотеки jQuery для мобильных устройств

Разработчики популярной свободной JavaScript-библиотеки jQuery представили проект jQuery Mobile, в рамках которого развивается легковесный вариант библиотеки, оптимизированный для использования на мобильных устройствах. Код библиотеки распространяется в рамках лицензий MIT и GPLv2. jQuery Mobile позволяет создавать интерактивные web-приложения с унифицированным интерфейсом пользователя, работающим в большинстве мобильных браузеров, доступных для платформ Apple iOS, Symbian S60/UIQ/Platform, Blackbery, Android, Windows Mobile, webOS, bada, Maemo, MeeGo. Из особенностей библиотеки отмечается оптимизация управления через сенсорные экраны, обеспечение единого внешнего вида для разных типов браузеров и набор специализированных виджетов для построения интерфейса пользователя.
