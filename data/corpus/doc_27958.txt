Релиз дистрибутива Superb Mini Server 1.5.3

Представлен Linux-дистрибутив Superb Mini Server 1.5.3, основанный на Slackware Linux и предназначенный для быстрого развертывания серверов для обслуживания предприятий среднего и малого бизнеса. Дистрибутив поддерживает развертывание web-сервера, межсетевого экрана, почтового сервера, сервера печати и других сетевых служб. Настройка параметров сервисов осуществляется через web-интерфейс, реализованный на базе проекта Webmin. Дистрибутив работает в Live-режиме, размер загрузочного образа составляет 348 Мб. Из входящих в комплект программ можно отметить: Linux-ядро 2.6.33.5, dovecot-2.0.2, bind 9.7.1_P2, clamav 0.96.2, apache 2.2.16, mysql 5.1.50, openldap 2.4.23, php 5.3.3, phpmyadmin 3.3.7, proftpd 1.3.3, samba 3.5.4, squid 3.1.8, postfix 2.7.1, openbravo 2.50MP18, а также WebERP и PhpLDAPAdmin. Дополнительно в состав входит TorrentFlux, написанный на языке php BitTorrent-клиент, управляемый через web.
