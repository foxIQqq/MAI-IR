Новая версия свободной системы 3D-моделирования Blender 2.77

После пяти месяцев разработки представлен выпуск свободного пакета 3D-моделирования Blender 2.77. Основные новшества: Рендеринг Cycles: Улучшено подповерхностное рассеяние, ускорена работа на GPU. Поддержка задействования GPU для отрисовки огня, дыма и объёмных текстур; Настраиваемое положение размытия движения, поддержка роллинг-шаттера. Пользовательские проходы запекания. Оптимизирована производительность сложных нодовых шейдеров. Улучшены настройки по умолчанию. Пользовательский интерфейс Полностью переработано оформление индикатора процесса выполнения операции; Поддержка показа многострочного текста во сплывающих подсказках, штампах рендеринга, нодовых кадрах и метаданных. В рендере OpenGL улучшен антиалиасинг; Моделирование Модификатор Decimate теперь поддерживает симметрию. Новые булевы операции для использования в режиме правки, удобны для быстрого редактирования. Улучшенные инструменты выделения: шаг грани, шахматное выделение, следующий/предыдущий, отражение по нескольким осям. Возможность получить UV-развёртку по умолчанию для создаваемого геометрического примитива. Мелкие улучшения: триангуляция и преобразование граней в вееры, заполнение сеткой для неоднородных сеток, поддержка отверстий инструментом «Пересечение», инструмент «Разделение по рёбрам» Редактор видеоряда Модификаторы ленты: добавлены модификаторы баланса белого и тонального отображения, а также возможность добавлять модификаторы ко всем выбранным лентам. Эффекты: увеличена скорость работы размывания Гаусса Ленты: При импорте новой ленты теперь используется частота кадров загружаемого клипа. Маски теперь могут быть как относительными для ленты, так и абсолютными для времени сцены. Поддержка лент со вложенными сценами и перенос слов в текстах редактора. Предварительный просмотр ленты сцены с использованием OpenGL теперь соблюдает режим прозрачности ленты. Скульптинг и рисование Улучшен инструмент «Подвес» (Snake Hook), используемый для вытягивания длинных сегментов Режим рисования веса: Улучшены автоматическая нормализация и мультирисование. Режим рисования по вершинам: поддержка вращения последнего штриха Анимация Ограничители: добавлен новый метод интерполяции, основанный на полярной декомпозиции Редактор графов: параметр «Cursor X» теперь допускает дробные значения при работе с драйверами, добавлена возможность вставки ключевых кадров на позиции курсора. Экспозиционный лист: улучшенный диапазон кадров для новых экземпляров редактора. Скелеты: выделение теперь поддерживает выделение потомков, непосредственных потомков и родственников. Пути движения: функция очищения путей теперь применяется ко всем объектам/костям, кнопка «Обновить пути костей» теперь также присутствует в тулбаре. Эскизный карандаш Скульптинг штрихов. Полноценный режим редактирования во вьюпорте. Обновлённый ластик с чувствительностью к нажатию. Возможность ограничить редактирование только активным слоем («Изолировать слой»). Поддержка манипулятора трансформации. Улучшенное редактирование анимации, в том числе копирование и вставка, а также более продвинутый фильтр каналов. Управление блоками данных и библиотеками Если при загрузке не удаётся найти ту или иную библиотеку, будет добавлен пустой блок данных - местозаполнитель, который может использоваться для восстановления или перелинковки. Отсутствующие библиотеки и блок данных теперь помечены пиктограммой «broken lib» в окне структуры проекта. Появилась возможность полного удаления библиотеки и всех связанных с ней блоков данных. Игровой движок В настройки физики персонажа добавлен параметр «Максимальное количество прыжков». Новый актуатор для снятия скриншотов. Снятие скриншотов выполняется в отдельном потоке. Дополнения Новое дополнение: Blend File Utils, поддерживает упаковку blend-файлов (включая все библиотеки и зависимости) в единый zip-архив. В дополнении OBJ IO добавлена ограниченная поддержка маппинга текстур MTL; В дополнении X3D/VRML IO расширена поддержка подмножеств стандарта; Python обновлён до версии 3.5.1. Все функции Python API load() снабжены аргументом check_existing, по умолчанию имеющим значение «ложно». Добавлен модуль gpu.offscreen для закадрового рисования. в связи с переработкой инструмента «Эскизный карандаш» удалён GreasePencil.draw_mode; Другие особенности Кэширование OpenVDB: симуляции дыма и объёмные текстуры могут быть сохранены с использованием OpenVDB. Улучшена поддержка многопоточности. Редактор изображений теперь может раздельно визуализировать цветовые каналы. Прекращена поддержка Windows XP. Удалена система сборки SCons. Удалена библиотека redcode, вместо неё используется ffmpeg, которая теперь поддерживает redcode. Отдельно стоит упомянуть изменения, сделанные командой Blend4Web: Добавлена поддержка ноды Vector Transform в Viewport и Blender Render; Добавлена поддержка кубических карт в Viewport; Добавлена поддержка "polygon stipple" в базовом шейдере; Исправлена некорректная работа опции Real Sky; Исправлена ошибка с отображением имени FCurve после переименования.
