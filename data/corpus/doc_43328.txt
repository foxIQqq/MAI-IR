В SourceForge добавлены средства для распространения программ для платформы Android

Хостинг свободных проектов SourceForge объявил о реализации средств для организации распространения сборок приложений для платформы Android (поддерживаются расширения .apk, .odex и .dex). Пользователи, загружающие проект на Android-устройство, получают возможность бесшовной установки и автоматического обновления до свежих выпусков уже установленных программ. Из проектов, которые уже воспользовались опцией для распространения приложений для платформы Android, отмечаются Videoder (поиск по видео), LinuxonAndroid (образы разных дистрибутивов для запуска в chroot) и YouTube Downloader (загрузчик контента с Youtube).
