Выпуск SciPy 1.0, библиотеки для научных и инженерных расчётов

После 16 лет развития проекта представлен знаковый релиз библиотеки для научных, математических и инженерных расчётов SciPy 1.0. SciPy предоставляет большую коллекцию модулей для таких задач, как вычисление интегралов, решение дифференциальных уравнений, обработка изображений, статистический анализ, интерполяция, применение преобразований Фурье, поиск экстремума функции, векторные операции, преобразование аналоговых сигналов, работа с разреженными матрицами и т.д. Код проекта распространяется под лицензией BSD и использует высокопроизводительную реализацию многомерных массивов от проекта NumPy. Релиз 1.0 выпущен после достижения проектом ключевых первичных целей и позиционируется как индикатор зрелости и стабильности, подтверждённый успешными промышленным применением. Среди задач, решение которых дало зелёный свет для релиза 1.0 упомянуты создание бинарных сборок в формате Windows Wheels, внедрение системы непрерывной интеграции и формирование сообщества со своей структурой управления, планированием работ и принятыми правилами поведения. Выпуск SciPy 1.0 гарантирует сохранение обратной совместимости API. Из новшеств отмечается добавление нового набора функций для решения обыкновенных дифференциальных уравнений и унификация связанного с ними программного интерфейса, обеспечение охвата системой непрерывной интеграции сборок для macOS и Windows в дополнение к Linux, включение в поставку двух новых алгоритмов оптимизации доверительной области (trust-region), порция новых функций линейной алгебры BLAS (Basic Linear Algebra Subprograms) и LAPACK (Linear Algebra PACKage).
