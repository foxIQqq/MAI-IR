Вышел Calculate Linux 18

Представлен релиз дистрибутива Calculate Linux 18, который развивается русскоязычным сообществом, построен на основе Gentoo Linux, поддерживает непрерывный цикл выпуска обновлений и оптимизирован для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в нескольких редакциях, собранных для архитектур i686 и x86_64: Calculate Linux Desktop с рабочим столом KDE (CLD), MATE (CLDM) и Xfce (CLDX), Calculate Directory Server (CDS), Calculate Linux Scratch (CLS), Calculate Scratch Server (CSS) и редакция для контейнеров Calculate Container Scratch (CCS). Все версии дистрибутива распространяются в виде загрузочного Live-образа с возможностью установки на жесткий диcк или USB-накопитель. Calculate Linux совместим с портежами Gentoo, использует систему инициализации OpenRC и применяет Rolling-модель обновлений. Репозиторий насчитывает 12363 бинарных пакета. Live USB включает как открытые, так и проприетарные видеодрайверы. Поддерживается мультизагрузка и модификация загрузочного образа при помощи утилит Calculate. Система поддерживает работу с доменом Calculate Directory Server с централизованной авторизацией в LDAP и хранением профилей пользователей на сервере. В состав входит специально разработанная для проекта Calculate подборка утилит для настройки, сборки и установки системы. Предоставляются средства для создания специализированных ISO-образов, адаптированных под задачи пользователя. Основные изменения: Утилиты и графический установщик Calculate портированы на библиотеку Qt5. Улучшен стиль Qt5 приложений в Calculate Linux Desktop Xfce, MATE и Cinnamon; Изменена модель настройки сети. По умолчанию во время установки системы настройка сети отныне не выполняется, вместо этого переносятся настройки из текущей системы. Отключено обновление сетевых настроек при установке пакетов отвечающих за работу сети. Для настройки сети в утилиту cl-setup-system добавлена опция '--network'; Добавлена проверка индекса бинарных пакетов по цифровой подписи; В установщике добавлена возможность выбора раскладки клавиатуры и выбора разметки диска с корневым разделом на весь диск. Упрощена установка системы на VPS/VDS; В консольной утилите установки добавлено определение использования авторазметки в зависимости от переданных параметров; Оптимизирована загрузка LiveUSB. При загрузке с LiveUSB добавлен вывод лога установки видеодрайвера NVIDIA на консоль tty12; В Calculate Linux Desktop Xfce добавлена отдельная тема для GTK3-приложений, оформленная в одном стиле с основной темой. Обновлена тема рабочего стола в Calculate Linux Desktop MATE. Вместо uvesafb задействованы Vesa Framebuffer и Efi Framebuffer; Для обработки звука по умолчанию используется звуковая система ALSA. В CLDC добавлена поддержка регулировки настроек звука при использовании ALSA. Настройка звуковой карты выполняется через идентификатор устройства, а не его номер. При загрузке с LiveUSB на системах с UEFI добавлена возможность выбора звуковой системы Обновлены версии программ, в том числе задействованы GCC 7.3 и Python 3.6; В браузере Firefox отключён брендинг и расширение Screenshot; Устранены проблемы при установке системы на раздел с XFS; В конфигурацию ядра добавлена поддержка GRE-туннелей; Прекращено использование zswap.
