Новые выпуски дистрибутива Raspbian и установщика NOOBS для Raspberry Pi

Доступен для загрузки новый выпуск дистрибутива Raspbian 2014-09-09, официально рекомендуемого для использования на Raspberry Pi. Дистрибутив основан на пакетной базе Debian Wheezy и специально оптимизирован для Raspberry Pi (сборка для ARMv6 с расширениями "hard float"). Новый выпуск примечателен задействованием по умолчанию web-бразуера собственной разработки, который заменил собой ранее поставляемый Midori. Кроме того, в новом выпуске обновлены прошивки и ядро Linux, задействован JDK 8, включена в поставку игра Minecraft Pi, доступны новые выпуски Mathematica 10 и Sonic Pi 2, добавлен драйвер для беспроводных карт на основе чипа BCM43143 (802.11n). Одновременно сообщается о выходе NOOBS 1.3.10, пользовательского интерфейса для быстрой установки различных операционных систем на SD-карту для Raspberry Pi. Особенностью приложения является его установка на SD карту, после загрузки с данной карты NOOBS предоставляет интерфейс для установки и запуска выбранной системы. Поддерживается установка Raspbian, Arch Linux, Pidora, RISC OS и двух вариантов медиацентра XBMC (OpenELEC и RaspBMC). Обособленная установка с SD-карты теперь возможна только для Raspbian, для установки остальных систем требуется наличие сетевого соединения.
