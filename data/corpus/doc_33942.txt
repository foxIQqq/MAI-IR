Релиз Wine 1.5.5

Вышел очередной экспериментальный релиз открытой реализации Win32 API - Wine 1.5.5. С момента выпуска версии 1.5.4 было закрыто 84 отчёта об ошибках. Основные изменения в новой версии: Добавлена поддержка установки Mono (Open Source аналог .Net) в качестве дополнительного пакета; Расширены возможности DIB движка; Добавлена поддержка установки .Net 4.0; Для D3DX9 реализована поддержка файлов текстур поверхностей (расширение DDS); Обновлены переводы; Исправлено множество ошибок.
