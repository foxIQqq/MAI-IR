Релиз свободной прошивки для маршрутизаторов на базе RTL8186

Вышел релиз wive-ng-0.2.14, свободной прошивки для маршрутизаторов на базе SoC RTL8186. Кроме исправлений ошибок, представлены следующие новшества: Вместо userlevel pptp теперь используется ядерная реализация ACCEL-PPTP, тем самым решена проблема низкой производительности pptp туннелей Вместо userlevel l2tp теперь используется ядерная реализация openl2tp Добавлены раздельные опции для управления MTU/MRU для PPPOE/PPTP, также в interfaces перенесены настройки логина/пароля для конкретного типа соединения Решена проблема с большими пакетами при использовании pptp и l2tp без шифрования, для pppoe проблема актуальна Добавлен выбор userspace/kernelspace режима работы pppoe (последний пока не работает) Добавлена поддержка persist в accel-pptp гарантирующая что соединение будет переустановлено как только сервер вновь будет доступен Обновлена реализация sha1 благодаря чему mppe теперь работает с любыми pptp серверами включая mpd После решения проблем с kernel level pppoe будет последний релиз 0.2.х ветки, в 0.3.х ветке будут добавлены новые возможности. Отдельно пользователям предлагается протестировать скорость работы новых pptp и l2tp клиентов в реальных условиях.
