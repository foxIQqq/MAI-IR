Вышла программа оцифровки графиков Engauge Digitizer 7.0

Доступен релиз программы Engauge Digitizer 7.0, предоставляющей средства для извлечения данных на основе оцифровки файлов с изображениями графиков. По своему назначению программа выполняет функции, противоположные системам отрисовки графиков, пытаясь восстановить исходные данные на основе которых эти графики были построены. Кроме исправления ошибок в новой версии реализованы следующие возможности: Реализован экспертный режим, позволяющий работать с несколькими координатами одновременно и выполнять оцифровку осей с одной известной координатой (плавающие оси); Изображения для оцифровки теперь можно вставлять из буфера обмена; При импорте документов, сохраненных в предыдущих версиях программы, реализована проверка корректности данных; Улучшена работа с локалями для языков, отличных от английского;
