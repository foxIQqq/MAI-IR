Опубликован перевод руководства по Ubuntu Server 12.04

Русскоязычное сообщество Ubuntu перевело на русский язык "Ubuntu Server Guide", официальное руководство по серверным возможностям Ubuntu. Перевод доступен в форматах PDF и ePub. При обнаружении в тексте стилистических, орфографических, пунктуационных, смысловых и других ошибок, исправления рекомендуется отправлять на Launchpad или писать в теме Перевод Ubuntu Server Guide. Отдельно выражается благодарность пользователю ALT Linux Михаилу Радюку, который помог с отображением русских шрифтов при компиляции проекта Ubuntu Server Guide из Apache FOP. Также спасибо неравнодушным с форума the-ebook.org, которые направили в помогли советом по создании ePub версии для электронных читалок.
