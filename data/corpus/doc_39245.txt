Релиз графического редактора Krita 2.8.0

Анонсирован релиз растрового графического редактора Krita 2.8.0, входящего в состав развиваемого проектом KDE офисного пакета Calligra. Krita поддерживает многослойную обработку изображений и обладает большим набором средств для цифровой живописи, создания скетчей и формирования текстур. Кроме варианта для настольных систем, в состав пакета включены две специализированные редакции: Krita Sketch, вариант редактора для планшетов, и Krita Gemini, вариант для гибридных субноутбуков, позволяющий на лету переключаться между десктоп-режимом и режимом для планшетов. Особенности нового выпуска: Задействована собственная реализация кода для поддержки графических планшетов для рисования от руки, которая теперь используется вместо API поддержки планшетов из состава Qt. Причинами создания собственной реализации, которая основана на примерах кода Qt, были ограничения в поддержке устройств, отличных от Wacom, и проблемы, проявляющиеся на платформе Windows. Новая реализация поддерживает работу в системах на базе X11 и в Windows, и отличается значительно более гладким рисованием и избавлением от проблем со смещением кисти; Добавлен новый режим высококачественного масштабирования для OpenGL холстов. Krita является одним из первых графических редакторов, обеспечивших поддержку отрисовки изображений с использованием OpenGL. OpenGL позволил обеспечить высокую производительность вращения, панорамирования и масштабирования, но при незначительной потере качества картинки, которая, в зависимости от выбора алгоритма, получалась размытой или зернистой. Теперь доступен новый алгоритм для высококачественного масштабирования, основанный на использовании шейдеров; В основной состав для Linux интегрирован оптимизированный для планшетных ПК интерфейс Krita Sketch и обеспечена возможность переключения между интерфейсом для настольных систем и интерфейсом для планшетов. Для пользователей гибридных ультрабуков на базе Windows через сервис Valve Steam доступен Krita Gemini, комбинирующий возможности настольной версии Krita и мобильной редакции Krita Sketch для планшетов и автоматически переходящий в планшетный или настольный режим в зависимости от текущего режима Windows; Добавлена полноценная поддержка платформы Windows. Krita 2.8 отмечен как первый стабильный выпуск для платформы Windows, пригодный для использования обычными пользователями; Режим мозаичного дублирования (Wrap Around), копирующий выбранный элемент изображения по окружающему пространству холста, что удобно использовать для создания текстур. Новый набор настроек кистей, используемый по умолчанию, и новый набор пиктограмм; Упрощённый режим выбора слоя. Слой можно выбрать нажав клавишу R и кликнув на нужный холст. Упрощена привязка тегов к ресурсам - тег теперь можно присвоить через меню по правому клику мышью; Новый докер палитры, позволяющий добавлять и удалять цвета непосредственно в докере. В состав включён набор преднастроенных палитр. Включить показ докера можно через меню Setting > Docker > Palette; Реализация псевдобесконечных холстов, позволяющая не задумываться о доступном месте на холсте. При значительной прокрутке холста в одном направлении теперь появляется специальная кнопка для быстрого расширения холста; В инструмент для обрезания краёв добавлен новый флагo "grow" для обрезания документа за пределами холста с последующим расширением холста до новой области. Также добавлена поддержка выбора типа декорирования рамки инструмента обрезания краёв; Новый фильтр цветового баланса, позволяющий при помощи ползунков корректировать цвета для всей работы. Фильтр идеально подходит для придания дополнительного настроения своей работе (например, можно сделать цвета теплее или холоднее); Интегрирована начальная поддержка фреймворка обработки изображений G’Mic, позволяющая применять в Krita сотни готовых фильтров Gmic через меню Layer > Apply Gmic; Инструмент для клонирования текущего слоя (Layer > Clone Array), в том числе для его повторяющейся отрисовки; Более компактная организация интерфейса и снижение насыщенности элементов управляющего интерфейса для концентрации внимания на холсте.
