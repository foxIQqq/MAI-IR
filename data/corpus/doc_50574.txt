Выпуск дистрибутива NomadBSD 1.2

Представлен выпуск Live-дистрибутива NomadBSD 1.2, представляющего собой редакцию FreeBSD, адаптированную для использования в качестве переносного рабочего стола, загружаемого с USB-накопителя. Графическое окружение основано на оконном менеджере Openbox. Для монтирования накопителей применяется DSBMD (поддерживается монтирование CD9660, FAT, HFS+, NTFS, Ext2/3/4), для настройки беспроводной сети - wifimgr, а для управления громкостью - DSBMixer. Размер загрузочного образа 2 Гб (x86_64, i386). В новом выпуске осуществлён переход на кодовую базу FreeBSD 12, включена по умолчанию поддержка TRIM, добавлена цветовая тема для оформления виртуального терминала, обеспечено динамическое создание настроек драйверов для GPU Intel. В состав включена реализация управляющего меню dmenu, вызываемая при нажатии Ctrl+Space. Представлен новый графический интерфейс для настройки параметров системы, построенный с использованием Qt (ранее предлагался консольный конфигуратор).
