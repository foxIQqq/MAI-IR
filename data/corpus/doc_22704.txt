Представлен новый интернет планшет Optima на базе платформы Maemo

Китайская компания Optima представила новый мобильный интернет планшет (MID) Optima OP5-E, основанный на Linux платформе Maemo, используемой в частности на планшетах Nokia N8xx. Характеристики устройства: CPU Marvell PXA320 @ 806MHz 4.3-дюймовый сенсорный экран (800 x 480 WVGA) ОЗУ 128MB RAM, Flash 256MB NAND microSD слот, MiniUSB Поддержка сотовых сетей CDMA2000 EV-DO (800MHz) WiFi 802.11b/g Bluetooth 2.0 GPS (Active Antenna) 3.2-мегапиксельная камера + 0.3-мегапиксельная камера для сетевого вещания Программное обеспечение Google search Firefox MSN, QQ, и Fetion IM VoIP Менеджер контактов ПО для обмена текстовыми сообщениями Медиа-плеер с поддержкой форматов MP3, WMA, WAV, AAC, AVI, MP4 Программа для чтения PDF Навигационный пакет Gao Network TV (beta) FBreader с поддержкой чтения книг в форматах FB2, HTML, CHM, RTF, plain text Фотоменеджер Программы для оформления заметок и создания простых рисунков Калькулятор Игры Аккумулятор 2600mAh Li-Ion; заявлено 8 часов автономной работы Размер 143 x 74 x 17.6 мм
