Выпуск дистрибутива Steam OS 3.4, используемого на игровой консоли Steam Deck

Компания Valve представила обновление операционной системы Steam OS 3.4, поставляемой в игровой консоли Steam Deck. Steam OS 3 основана на Arch Linux, использует для ускорения запуска игр композитный сервер Gamescope на базе протокола Wayland, поставляется с доступной только на чтение корневой ФС, применяет атомарный механизм установки обновлений, поддерживает пакеты Flatpak, использует мультимедийный сервер PipeWire и предоставляет два режима работы интерфейса (оболочка Steam и рабочий стол KDE Plasma). Обновления доступны только для Steam Deck, но энтузиастами развивается неофициальная сборка holoiso, адаптированная для установки на обычные компьютеры (в будущем сборки для ПК обещает подготовить и компания Valve). Среди изменений: Осуществлена синхронизация со свежей пакетной базой Arch Linux. Среди прочего до выпуска 5.26 (ранее поставлялся выпуск 5.23) обновлена версия рабочего стола KDE Plasma. Добавлена опция для отключения вертикальной синхронизации (VSync), применяемой для защиты от появления разрывов при выводе (tearing). В игровых программах после отключения защиты могут появиться артефакты, но с ними можно смириться, если борьба с ними приводит к дополнительным задержкам. Решены проблемы с зависанием некоторых игр после возвращения из спящего режима. Решены проблемы с подвисаниями на 100мс при включении режима адаптивной подсветки. Предложена новая прошивка для док-станции, в которой решены проблемы с определением экранов, подключённых через HDMI 2.0. Во всплывающей панели HUD (Heads-Up Display) применена горизонтальная компоновка для второго уровня статистики производительности. Включена поддержка операции TRIM для информирования внутренних накопителей о неиспользуемых в ФС блоках. В настройках "Settings → System → Advanced" появилась кнопка для принудительного выполнения операции TRIM в любой момент. В "Settings → Storage" для внешних устройств добавлена опция для извлечения устройства. Обеспечено автоматическое монтирование внешних накопителей с ФС ext4. При запуске Steam отключена эмуляция мыши для трекпадов DualShock 4 и DualSense. Когда Steam не запущен в режиме рабочего стола обеспечена загрузка драйвера gamepad. Улучшено использование виртуальной клавиатуры в играх. Добавлена поддержка беспроводных контроллеров 8BitDo Ultimate.
