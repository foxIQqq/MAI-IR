Вышла отечественная сборка Linux Mint 8 Росинка х86 и 64-разрядная сборка Linux Mint 8

Вышел релиз отечественной сборки Ubuntu-совместимого дистрибутива Linux Mint 8, развиваемой под кодовым именем "Росинка". Кроме отличной поддержки русского языка из коробки, в состав отечественной сборки добавлены следующие дополнительные пакеты: manpages-ru русские руководства ко многим программам. mc двухпанельный консольный файловый менеджер (наподобие FAR). arj rar unrar unace-nonfree p7zip-full архиваторы. ubuntu-tweak удобная настроечная утилита. qutim интернет мессенжер (ICQ, Mail-агент, Jabber). pidgin-plugins* различные плагины для Pidgin (интернет мессенжер по-умолчанию) flashplugin-nonfree adblock-plus полезные расширения для firefox msttcorefonts ttf-liberation шрифты Times New Roman, Arial и другие от MS, а также свободная альтернатива им. gxneur аналог PuntoSwitcher. gnome-games простые игры. sensors-applet дополнение к панели задач (мониторинг температуры процесссора и пр.). imagemagick пакет программ для неинтерактивной обработки графических файлов. audacious аудиоплеер с очень качественным звуком, поддержкой кодировки ср-1251 в тегах файлов и внешне напоминающим WinAmp. audacity аудиоредактор, заменяющий SoundForge, только для более простых задач. easytag редактор тегов (поддерживает перекодировку) в mp3 и прочих аудиофайлах. python-mutagen набор инструментов для работы с тегами в аудиофайлах. chmsee просмотрщик файлов .chm dia программа для работы с диаграммами. playonlinux wine1.2 приложения для запуска Windows программ и игр. smbfs возможность монтирования сетевых дисков SMB (Windows). sshfs монтирование по SSH. openprinting-ppds-extra большая база драйверов для принтеров. isomaster работа в образами дисков. nmap traceroute wireshark сетевые инструменты. поддержка сетевых подключений: network-manager-pptp network-manager-vpnc network-manager-openvpn pppoe pppoeconf pppstatus openvpn openvpn-blacklist pptp-linux vpnc wvdial blueman. Кроме того, в Росинке устранена ошибка в коде изменения раскладок клавиатуры, выключена по умолчанию поддержка протокола ipv6 и исправлена проблема с открытием рабочего стола из меню. Дополнение: Спустя 3 недели с момента выхода 32-разрядной сборки, официально вышел Linux Mint 8 в сборке для 64-разрядной архитектуры X86_64 (CPU Intel Core 2, AMD X2 64 и т.п.).
