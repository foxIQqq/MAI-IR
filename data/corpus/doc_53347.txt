Релиз оконного менеджера IceWM 1.7

Доступен выпуск легковесного оконного менеджера IceWM 1.7. Из особенностей IceWM можно отметить полноценное управление через клавиатурные комбинации, возможность использования виртуальных рабочих столов, панели задач и меню-приложений. Оконный менеджер настраивается через достаточно простой файл конфигурации, возможно использование тем оформления. Доступны встроенные апплеты для мониторинга CPU, памяти, трафика. Отдельно развивается несколько сторонних GUI для настройки, реализаций рабочего стола и редакторов меню. Код написан на языке С++ и распространяется под лицензией GPLv2. Основные изменения: Добавлена настройка KeyboardLayouts для управления переключением раскладки клавиатуры. KeyboardLayouts автоматизирует настройку переключения через setxkbmap и позволяет задать список поддерживаемых раскладок в форме 'KeyboardLayouts="de","ru","en"' без настройки ручного вызова setxkbmap. Обеспечено сохранение фокуса на окне приложения при перезапуске оконного менеджера и корректное восстановление прошлого фокуса при закрытии активного окна. Добавлена опция ignoreActivationMessages для игнорирования программных запросов приложений на изменение фокуса. Вместо вызова shell для раскрытия имён файлов по маске (например, "[a-c]*.c") задействована функция wordexp. В интерфейс просмотра списка окон добавлена команда раскрытия окон по горизонтали (Maximize Horizontal). Добавлена возможность детального отслеживания операций с системным лотком. Улучшено соответствие стандарту XEMBED. Обновлена тема оформления NanoBlue (Nano_Blu-1.3).
