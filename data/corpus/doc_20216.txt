В следующем релизе Mac OS X будет значительно улучшена поддержка ZFS

"Mac OS X Snow Leopard Server to pioneer ZFS ahead of desktop" - в следующем релизе Mac OS X 10.6 (Snow Leopard) будет значительно улучшена поддержка ZFS (в прошлых версия была возможность доступа в режиме только для чтения), будет добавлена возможность записи данных и создания дисковых разделов. Кроме того, для сокращения размера дистрибутива рад наборов драйверов (например, принтеров) будет исключен из базовой поставки и вынесен во внешний репозиторий и доступен через сервис Software Update.
