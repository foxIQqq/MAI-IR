Вышел дистрибутив Univention Corporate Server 3.1

Доступен релиз Univention Corporate Server 3.1, промышленного дистрибутива, основанного на Debian GNU/Linux 6 (Squeeze) и ориентированного на обеспечение поддержки корпоративной IT-инфраструктуры. Для установки задействован собственный инсталлятор, оптимизированный для быстрого развёртывания готовых сервисов. Для локального и централизованного администрирования используется специальный web-интерфейс. Для управления конфигурацией подготовлен инструментарий Univention Configuration Registry, позволяющий через наглядный интерфейс генерировать файлы конфигурации для разных сервисов и программ. В состав дистрибутива входит набор компонентов, позволяющих решать такие задачи, как ведение централизованного LDAP-каталога учетных записей пользователей, обеспечение единой системы аутентификации и авторизации, интеграция с MS Active Directory, автоматизация развёртывания серверов и рабочих станций, обеспечение работы тонких клиентов, мониторинг, создание серверов печати, резервное копирование, управление сетевыми службами (DHCP, DNS, прокси, почтовый сервер), обеспечение совместной работы сотрудников предприятия (groupware), управление работой виртуальных окружений на базе Xen и KVM. Размер загрузочного iso-образа 1.8 Гб. Ключевым улучшением новой версии является предоставление стека на базе Samba 4 для обеспечения работы сервисов, совместимых с Active Directory, а также поставка инструментов для упрощения миграции Microsoft Active Directory на решение Univention Corporate Server. В том числе доступны средства для автоматизации миграции пользователей, машин и групповых политик. Переработан интерфейс пользователя, добавлена возможность расширения функциональности и кастомизации через подключение модулей-дополнений. Добавлен новый модуль для настройки параметров сети. Обновлены версии поставляемых в дистрибутиве приложений. Обеспечено формирование образов для загрузки на системах с UEFI.
