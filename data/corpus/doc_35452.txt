Представлена Linux-версия системы удалённого доступа к рабочему столу Splashtop Streamer

Компания Splashtop выпустила Linux-версию системы для организации удалённого управления рабочим столом - Splashtop Streamer. В настоящее время сборки подготовлены только для Ubuntu Linux 12.04 (программа доступна для загрузки через Ubuntu Software Center). Программа даёт возможность пользователю получить доступ к рабочему столу по сети с внешних стационарных и мобильных устройств, работающих на базе платформ Android, iOS, Windows и Mac OS X. Для Linux пока доступна только реализация сервера, Linux-клиент Streamer ещё находится на стадии разработки, т.е. возможно предоставить доступ к рабочему столу Linux для других систем, но подключиться для управления внешними системами из Linux пока нельзя. Splashtop Streamer является по сути сервером потокового вещания звука и видео, в который интегрированы средства передачи событий ввода. При организации работы поддерживаются трансляция средств ввода удалённой системы, например, для работы с удалённым ПК могут быть использованы экранная клавиатура и сенсорный экран мобильного устройства. Все данные передаются с использованием шифрования (SSL с 256-разрядным AES). Важным достоинством Splashtop Streamer является высокая эффективность передачи данных и обеспечение минимальных задержек. В частности, по заявлению разработчиков Splashtop Streamer для Linux в 10 раз опережает по скорости VNC и позволяет обеспечить комфортную работу с интерактивными мультимедиа приложениями. Splashtop Streamer даёт возможность передачи данных на мобильное устройство с задержкой менее 30 мс и частотой обновления картинки 30 кадров в секунду, что позволяет использовать продукт в таких областях как просмотр видео и запуск игровых приложений. Используемый в Splashtop Streamer протокол обеспечивает динамическую адаптацию битрейта к качеству сетевого соединения, мягко сглаживая провалы и изменения пропускной способности. Все изменения на рабочем столе отображаются на удалённых системах синхронно, т.е. можно при подключении удалённого управления продолжать работу на локальной системе и использовать Splashtop Streamer как средство для организации совместного доступа. Поддерживаются организация управления и наблюдения за рабочим столом одновременно с нескольких устройств. Например, разработчиками продемонстрирована организация вещания пользовательского сеанса одновременно на 40 внешних устройств, среди которых были ПК, планшеты и телефоны. Среди дополнительных функций отмечается возможность организации прозрачного перекодирования отображаемых на базовой системе видео и звуковых потоков в другие форматы, пригодные для просмотра на удалённой системе.
