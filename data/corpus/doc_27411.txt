Tab Candy - переосмысление работы с закладками в веб-браузере Mozilla Firefox

Один из разработчиков Mozilla Firefox представил на суд пользователей новую концепцию работы с закладками - Tab Candy, которая позволяет упорядочить вашу работу с ними. После нажатия на кнопку предпросмотра, Tab Candy показывает обзор открытых вкладок, позволяя найти и выбрать любую из них, однако, главная особенность этой новой функции заключается в том, что появляется возможность сгруппировать вкладки по характеру выполняемой работы, а затем легко пользоваться и переключаться между группами вкладок. При переключении на интересующую группу, Firefox отображает только вкладки этой группы, скрывая все остальные. Обзор возможностей Tab Candy: Легкая группировка вкладок. Достаточно перетащить две вкладки в одну новую область пространства и они автоматически образуют группу. Для группы также можно указать имя. Только те вкладки, которые нужны. При переключении на группу вкладок, отображаются только они, что позволяет сфокусировать внимание пользователя на одной конкретной задаче. Все остальные вкладки будут скрыты. Переорганизация групп вкладок. Имеется возможность изменить размер каждой конкретной группы вкладок и расположить её так, чтобы она приняла наиболее удобное представление (одна иконка, включающая набор страниц, или расположение столиком, или расположение решёткой, а также перемещение их по всему полю предпросмотра). Отличительной особенностью Tab Candy является то, что она построена только на технологиях Web: HTML, Javascript и CSS - родной код Mozilla Firefox XUL не используется. Функция Tab Candy не является расширением (add-on) и она будет встроена в один из будущих выпусков веб браузера. Видеодемонстрация работы Tab Candy можно посмотреть здесь.
