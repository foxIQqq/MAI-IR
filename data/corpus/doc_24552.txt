Вышла девятая альфа версия Samba 4.0.0

Вышла девятая альфа версия проекта Samba 4.0.0, развиваемого параллельно с веткой 3.x и нацеленного на реализацию возможности работы в качестве Active Directory контроллера доменов. По сравнению с прошлой альфа версией, вышедшей полгода назад, можно отметить следующие улучшения: Samba4 теперь включает полный набор спецификаторов пользовательского интерфейса, которые требуется для управления через Microsoft Management Console; LDB, базовая библиотека для работы с БД в Samba4, переработана с целью увеличения производительности; Продемонстрирована возможность репликации между Samba4 и доменами на базе Active Directory при помощи нативного протокола для организации репликации данных DRS; Списки доступа теперь выставлены корректно для объектов в директории, в соответствии с правилами, принятыми в Windows 2008 (при поиске все еще используется схема администратор/не администратор). Подробный обзор ранее внесенных в ветку Samba 4.0.0 изменений доступен на данной странице.
