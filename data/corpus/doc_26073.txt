Началось бета-тестирование Mandriva Linux 2010.1

Вышла первая бета-версия дистрибутива Mandriva Linux 2010.1, релиз которого запланирован на 3 июня. Тестовая версия доступна в DVD-сборке "Free" (без проприетарных драйверов в комплекте) для архитектур x86-32 и x86-64, а также в LiveCD сборках "One" с GNOME и KDE. Из особенностей Mandriva Linux 2010 Spring Beta 1 можно отметить: Произведена интеграция десктоп окружения GNOME 2.30, для предварительного знакомства подготовлен пакет с поддержкой GNOME-Shell (будущий рабочий стол GNOME 3). В состав первой бета-версии включены приложения: KDE 4.4.2, Firefox 3.6.2, OpenOffice.org 3.2, Qt 4.6.2, Amarok 2.3, KTorrent 4.0beta1, Digikam 1.2, X.org 7.5, Mesa 7.8, Linux ядро 2.6.33.1, Glibc 2.11, ntfs-3g 2010.03.06, снапшот свободного Flash плеера Gnash 0.8.7, Xine 1.2, Deluge 1.2.3, Transmission 1.91, OpenShot 1.1, Songbird 1.4.3, Spamassassin 3.3.1, Postfix 2.7.0, Zabbix 1.8.1, Для организации начального этапа загрузки вместо mkinitrd теперь используется dracut (образ ram-диска с возможностью динамического обновления содержимого); В загрузчике Grub добавлена поддержка шифрованных паролей; В состав дистрибутива добавлен пакет USB/IP для организации доступа к USB устройствам на других машинах по сети; При сборке пакетов на этапе связывания задействованы опции "-Wl,-O1", что позволяет уменьшить время запуска приложений связанных с большим числом библиотек; Продолжение интеграции в десктоп-окружение KDE и пользовательские приложения системы семантических связей Nepomuk. Осуществлен переход Nepomuk на использование бэкенда для хранения данных Virtuoso, который позволяет ускорить работу с тегами и поддерживает индексацию хранимых данных; Значительно улучшена организация работы гостевых аккаунтов, позволяющих организовать одноразовый доступ к машине случайных людей, работа которых будет производиться в изолированной временной домашней директории, очищаемой при выходе гостя из системы. Например, для гостевых аккаунтов теперь используется специальный ограниченный shell, закрыт доступ к файлам других пользователей, заблокирован запуск административных системных приложений; Включение в комплект тестовой версии web-браузера Chromium и reKonq (на базе QtWebKit) В качестве базового свободного драйвера для видеокарт NVidia вместо драйвера nv теперь используется Nouveau, в котором реализована поддержка переключения видеорежимов на уровне ядра, 2D и 3D акселерация, поддержка RandR 1.2 для упрощения управления несколькими мониторами; Демон для ведения системных логов Sysklogd заменен на более продвинутую систему rsyslog, поддерживающую возможность хранения логов в СУБД, отправку email уведомлений при обнаружении в логах заданных масок, перенаправление логов на другой сервер с использованием TCP; В поставку интегрирован пакет мониторинга Munin 1.4 с плагинами для отслеживания работы MySQL, PostreSQL, Asterisk и т.п. Некоторые планы: Поддержка интерфейса KDE для нетбуков; Упрощение процесса шифрования содержимого домашних директорий; Улучшение работы утилит для управления установкой программного обеспечения, проведения обновлений и поиска нужных программ; Интеграция в дистрибутив наработок GNOME 3 и доступность для тестирования GNOME Shell, GNOME Color Manager, Gnome Activity Journal; Наращивание возможностей cloud-сервисов для пользователей дистрибутива и интеграция web-приложений на десктоп;
