Вышел шестой тестовый выпуск openSUSE 11.2

Вышел очередной тестовый выпуск openSUSE 11.2 Milestone 6. Релиз openSUSE 11.2 намечен на ноябрь. В новой тестовой версии представлены следующие изменения: В состав LiveCD с KDE добавлены новые программы: Choqok, KNode, Kompare, KSystemlog, Okteta; В состав GNOME LiveCD добавлен Bittorrent-клиент Transmission; Обновление версий программ: Python 3.1 PackageKit 0.5.1 Linux ядро 2.6.31-rc6 Firefox 3.5.2 GIMP 2.6.7 GNOME 2.28 Beta Konversation 1.2 Alpha 6 OpenOffice.org 3.1.1 Beta 2 VirtualBox 3.0.4 Wine 1.1.27 Базовые новшества openSUSE 11.2: В качестве десктоп-окружения по умолчанию будет использоваться KDE 4.3; Время поддержки дистрибутива (выпуска обновлений) будет сокращено с 24 до 18 месяцев; Пакеты дистрибутива собраны GCC 4.4.1 Образ LiveCD теперь может быть использован и на USB Flash; В состав LiveCD добавлен Midnight Commander, а в KDE-вариант LivCD программа YaKuake; Для YaST создан новый центр управления конфигурацией (YaST Control Center) с интерфейсом на базе Qt4; В качестве файловой системы по умолчанию задействована Ext4 В состав Live CD включен Gimp; Для сжатия данных на LiveCD отныне используется алгоритм сжатия LZMA; По умолчанию LiveCD содержит переводы для немецкого, французского, итальянского, польского и русского языков; В Qt версии конфигуратора YaST появилась поддержка настраиваемого внешнего вида. В YaST добавлены средства работы с кешем пакетов; В Zypp улучшены средства работы с зеркалами репозиториев пакетов; В интерфейсе для настройки параметров дисплея gnome-display-properties добавлена поддержка автоматического сброса настроек в случае проблем. Ведется разработка web-интерфейса к YaST для осуществления удаленного управления дистрибутивом Появление установочных образов для USB Flash и утилита для их подготовки Улучшение поддержки субноутбуков
