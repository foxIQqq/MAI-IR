Увидел свет Linux дистрибутив Mandriva 2010.0

Официально объявлено о выходе Linux дистрибутива Mandriva 2010.0. Дистрибутив распространяется в сборках для архитектур i586 и x86-64, для свободной загрузки доступны две редакции: Free (только свободное ПО, без проприетарных драйверов и кодеков, 1 DVD) и One (установочный LiveCD, KDE и GNOME версии). Для коммерческого распространения подготовлен Mandriva Linux Powerpack, который можно загрузить через интернет-магазин Mandriva Store. В Mandriva 2010.0 можно отметить следующие новшества: Обновление версий программ Десктоп окружения: KDE 4.3.2, GNOME 2.28.1, Xfce 4.6.1, LXDE; Системные приложения: Linux ядро 2.6.31, Virtualbox 3.0.8, Glibc 2.10.1, Bash 4.0, Cups 1.4; Пользовательские приложения: Firefox 3.5.3, Amarok 2.2, Go-OO сборка OpenOffice.org 3.1.1; В состав репозиторий contrib включен Python 3, но из соображений совместимости по умолчанию по прежнему присутствует Python 2.6. Кроме того, в комплект включен PHP 5.3.0; Переход на версию набора компиляторов GCC 4.4.1 для пересборки пакетов; Linux ядро для построения управляющего Xen окружения (Dom0) обновлено с версии 2.6.18 до 2.6.30; Новшества для пользователей В KDE интегрирована система Nepomuk, с помощью которой можно добавлять заметки ко всем файлам, добавлять аннотации и тэги. Возможно определение ассоциаций между файлами и объектами реального мира (например, привязка к mp3 файлу тэгов с характеристиками записанного произведения, времени модификации, имени человека, отправившего данный файл по email); Мультимедиа центр, для управления коллекцией фотографий, музыки и поддержания видеотеки, Elisa переименован в проект Moovida; В состав репозитория пакетов включена программа для ведения заметок GNote (аналог TomBoy на языке C++); KDE 3 больше не входит в состав дистрибутива как полное десктоп окружение, некоторые пакеты на базе KDE 3 остались, но в будущем поддержка будет полностью прекращена. Для обновления, для пользователей KDE 3 подготовлена специальная утилита, которая обеспечит перенос всех настроек и персональных данных (почта, заметки и т.п.) в окружение KDE 4; Подготовлен набор пакетов с окружением Moblin для субноутбуков, который автоматически предлагается к установке на слабых машинах и при низких экранных разрешениях. Расширена поддержка аппаратных особенностей новых моделей субноутбуков Asus EeePC, Acer Aspire One, Dell Mini, MSI Wind, HP Mini и т.д. Загрузка и установка Проведена дополнительная оптимизация системы инициализации с целью уменьшения времени загрузки; Все ISO-образы являются гибридными, т.е. их можно использовать для создания загрузочного флэш-брелока, например, с помощью программы Mandriva-seed; Задействован режим бесшовной графической загрузки через использование Plymouth, для карт не поддерживающих переключение видеорежимов на уровне ядра по прежнему будет использован Splashy; Переработка загрузочного меню Grub и начального меню инсталлятора (режим защиты от сбоев (failsafe), загрузка в консольном режиме (init3), восстановление после сбоя, проверка целостности инсталляционного диска); Настройка и поддержка оборудования В комплект добавлены проприетарные драйверы для интегрированных видеокарт Intel GMA500 (Poulsbo), поставляемых в новых устройствах на базе процессора Atom; Полностью переписан инструмент Netprofile, предназначенный для настройки профилей сетевых соединений. Теперь можно подготовить несколько сетевых профилей и использовать из в зависимости от ситуации; В апплет mdkonline добавлена возможность информирования об окончании времени поддержки дистрибутива и выпуска обновлений, связанных с безопасностью; Автоматическое определение наличия у пользователя локальных urpmi репозиториев; Безопасность Реализована поддержка гостевых аккаунтов для организации изолированной работы случайных людей. При входе под пользователем guest домашняя директория автоматически формируется через tmpfs и поддерживается в памяти, как только пользователь завершает сеанс - все данные удаляются; Переход на использование, появившихся в Linux ядре 2.6.30, модулей безопасности TOMOYO Linux, позволяющих привязать политику безопасности процесса к файловому пути, способом похожим с AppArmor. Tomoyo-gui - первая, написанная сотрудниками Mandriva, графическая оболочка для системы безопасности Tomoyo. Разное Осуществлен перевод процесса сборки дистрибутива на использование нового инструментария bcd, вместо ранее используемого mkcd, что должно положительно сказаться на качестве дистрибутива из-за значительного упрощения и усиления процесса тестирования.
