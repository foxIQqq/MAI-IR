Обновление инструментария Snapd 2.18 для самодостаточных пакетов Snap

Компания Canonical представила новый выпуск Snapd 2.18, инструментария для управлениями самодостаточными пакетами в формате snap. Новая версия уже включена в состав предварительных сборок Ubuntu Core ("snap refresh --candidate core") и в ближайшее время будет добавлена в штатные репозитории Ubuntu. Кроме того, можно отметить планы по переводу базовой сборки следующего LTS-релиза Ubuntu 18.04 полностью на технологию Snap и окружение рабочего стола Unity 8. Первый рабочий образ на базе snap и Unity 8 будет выпущен для ознакомления уже в рамках Ubuntu 17.04 и за оставшийся год доведён до полной готовности. Основные улучшения: Добавлены новые интерфейсы для работы raw-usb, avahi-observe, alsa, i2c, ofono и lxd-client; Представлена команда "snap info {name|path}" для показа детальной информации о заданном пакете; Вызов "snap find" без аргументов теперь приводит в отображению списка популярных пакетов; Добавлена возможность поиска только в заданных тематических разделах ("snap find --section=name"); Запуск "snap watch {change-id}" позволяет отследить прогресс выполнения ещё незавершённой операции; Реализованы возможности, которые в дальнейшем позволят реализовать delta-обновления, при которых вместо всего пакета при обновлении загружаются только изменившиеся данные; Повышена стойкость к возникновению сбоев в работе сети и реализована поддержка возобновления прерванной загрузки; Запуск "snap list --all" теперь показывает все ревизии установленных пакетов, в том числе отключенных; В API добавлена поддержка скриншотов.
