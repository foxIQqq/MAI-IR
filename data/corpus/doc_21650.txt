ELPF - новый Linux фреймворк для создания решений для встраиваемой техники

Компания Enea готовит к выходу новый проприетарный пакет для разработки встраиваемых Linux продуктов - Embedded Linux Project Framework (ELPF). Фреймворк объединит в одном пакете Linux ядро с поддержкой архитектур ARM, PPC и x86, набор стандартных утилит Busybox, библиотек (zlib, libxml, libpng и т.п.), средств разработки, готовых сервисов, средств конфигурации и графическую прослойку на основе SDL.
