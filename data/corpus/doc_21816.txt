Вышла новая версия Openbravo POS 2.30

20 мая 2009 вышел стабильный релиз POS-системы с открытым кодом Openbravo POS 2.30. Начиная с данной версии система распространяется под лицензией GNU GPL версии 3. Данная система разрабатывается в рамках Сообщества Openbravo, которое также разрабатывает систему для организации работы с ресурсами предприятия Openbravo ERP. Openbravo POS представляет собой фронт-офис для автоматизации бизнес-процессов предприятия на базе Openbravo ERP. Так как система хранит информацию в собственной базе данных, то Openbravo POS может работать и без развёртывания Openbravo ERP. Отличительной особенностью Openbravo POS является интерфейс системы, специально разработанный для использования с сенсорными экранами мониторов, коммуникаторов и т. д. В новую версию Openbravo POS были внесены следующие изменения: добавлен модуль для работы с КПК через вэб-интерфейс и соединение WiFi; появилась возможность вести учёт по характеристикам товара (размер, цвет, серийный номер и т.д.); переработан модуль синхронизации между Openbravo POS и Openbravo ERP, за его работу теперь отвечает Pentaho Data Integration; улучшена работа с платёжными шлюзами; расширена поддержка принтеров чеков; с исходными кодами стал распространяться профиль для инсталлятора Bitrock; обновлено лицензионное соглашение с GNU GPL v2 на GNU GPL v3; СУБД по умолчанию стала Apache Derby; также обновлены до последних стабильных релизов сторонние библиотеки и улучшен интерфейс пользователя. За период работы над системой было внесёно 53 исправления на замечания и предложения поступившие от пользователей. Изменились системные требования, для запуска Openbravo POS 2.30 необходимо чтобы был установлена Java 2 Platform Standard Edition 6.0 или выше. Подробности о новшествах Openbravo POS 2.30 и методике перехода с версии 2.20, размещены на русской вики-странице анонса версии Openbravo POS 2.30. Дополнение: доступен пакет с русским переводом для локализации Openbravo POS 2.30. Также, с данного релиза вся работа над русским пакетом перевода Openbravo POS перенесена в специально созданный проект на портале Openbravo Forge.
