Обновления систем видеоконференций Jitsi Meet Electron, OpenVidu и BigBlueButton

Опубликованы новые выпуски нескольких открытых платформ для организации видеоконференций: Выпуск клиента для проведения видеоконференций Jitsi Meet Electron 2.0, представляющего собой упакованный в отдельное приложение вариант Jitsi Meet. Из особенностей приложения отмечается локальное хранение настроек видеоконференций, встроенная система доставки обновлений, средства удалённого управления и режим закрепления поверх других окон. Из новшеств версии 2.0 выделяется возможность предоставления совместного доступа к воспроизводимому в системе звуку. Код клиента написан на JavaScript с использованием платформы Electron и распространяется под лицензией Apache 2.0. Готовые сборки подготовлены для Linux (AppImage), Windows и macOS. Jitsi Meet представляет собой JavaScript-приложение, использующее WebRTC и способное работать с серверами на базе Jitsi Videobridge (шлюз для трансляции видеопотоков участникам видеоконференции). В Jitsi Meet поддерживаются такие возможности как передача содержимого рабочего стола или отдельных окон, автоматическое переключение на видео активного докладчика, совместное редактирование документов в Etherpad, показ презентаций, стриминг конференции в YouTube, режим аудиоконференции, возможность подключения участников через телефонный шлюз Jigasi, защита подключения паролем, режим "можно говорить во время нажатия кнопки", отправка приглашений для подключения к конференции в форме URL, возможность обмена сообщениями в текстовом чате. Все передаваемые потоки данных между клиентом и сервером шифруются (подразумевается, что сервер работает на собственных мощностях). Jitsi Meet доступен как в форме отдельного приложения (в том числе для Android и iOS), так и в форме библиотеки для интеграции на сайты. Выпуск платформы для организации видеоконференций OpenVidu 2.12.0. Платформа включает в себя сервер, который может быть запущен на любой системе с реальным IP, и несколько вариантов клиентов на Java и JavaScript + Node.js для управления видеовызовами. Для взаимодействия с бэкендом предоставляется REST API. Видео передаётся при помощи WebRTC. Код проекта написан на Java и распространяется под лицензией Apache 2.0. Поддерживаются режимы переговоров между двумя пользователями, конференций с одним докладчиком и конференции, в которых все участники могут вести дискуссию. Параллельно с конференцией участникам предоставлен текстовый чат. Доступны функции записи мероприятия, трансляции содержимого экрана, наложения звуковых и видеофильтров. Предоставляются мобильные приложения для Android и iOS, настольный клиент, web-приложение и компоненты для интеграции функциональности проведения видеоконференций в сторонние приложения. Выпуск BigBlueButton 2.2.4, открытой платформы для организации web-конференций, оптимизированной для проведения обучающих курсов и online-обучения. Поддерживается трансляция нескольким участникам видео, звука, текстового чата, слайдов и содержимого экрана. Докладчик имеет возможность опрашивать участников и контролировать выполнение заданий на многопользовательской виртуальной доске. Возможно создание комнат для совместных обсуждений, в которых все участники видят друг друга и могут высказываться. Доклады и презентации можно записывать для последующей публикации видео. Для развёртывания серверной части подготовлен специальный скрипт.
