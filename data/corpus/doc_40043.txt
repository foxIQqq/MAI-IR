Компания Google представила Web Fundamentals и Web Starter Kit

Компания Google представила два проекта для web-разработчиков - Web Fundamentals и Web Starter Kit. Web Fundamentals представляет собой набор основополагающих рекомендаций по созданию переносимых web-интерфейсов, работающих на устройствах различных типов, независимо от используемых платформ, браузеров и фреймворков. Например, даются советы по обеспечению адаптации интерфейса для разных экранных разрешений, организации форм ввода, размещения изображений и видео, оптимизации производительности, локализации. Web Starter Kit является готовым набором CSS-шаблонов и инструментов для воплощения на практике методов, описанных в Web Fundamentals. Исходные тексты распространяются под лицензией Apache и доступны для внесения изменений через GitHub.
