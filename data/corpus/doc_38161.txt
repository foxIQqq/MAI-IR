Выпуск Chrome OS 30

Компания Google представила релиз операционной системы Chrome OS 30, основанной на ядре Linux, open source компонентах и web-браузере Chrome. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 30 доступна для для Acer Chromebook C7, Samsung Chromebox и Samsung Chromebook Series 5. Среди изменений, внесённых в Chrome OS 30: В меню системного лотка добавлен переключатель для выбора источника звука; По умолчанию в поставку включена бета-версия офисного пакета Quickoffice, позволяющего редактировать документы в форматах MS Word и Excel; Улучшен интерфейс для выбора канала получения обновлений; Для устройств с сенсорными экранами включена поддержка жестов для выделения текста и перетаскивания объектов (drag&drop); На странице, выводимой для нахождении в offline-режиме добавлен инструмент для диагностики проблем с сетевым соединением; Возможность организации совместного доступа к файлам и директориям из файлового менеджера; Улучшены механизмы изоляции в Native Client.
