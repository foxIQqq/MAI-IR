Представлен CorsixTH 0.20, открытый клон игры Theme Hospital

Вышла новая версия открытого проекта CorsixTH, в рамках которого развивается открытый клон экономического симулятора Theme Hospital («Частная клиника»), игровой процесс которого построен вокруг управления больницей. CorsixTH поставляется под свободной лицензией MIT, но требует для своей работы оригинальные графические файлы игры Theme Hospital (например из состава бесплатной демо-версии). Игра доступна для операционных систем Linux, FreeBSD, Mac OS X, Windows и Android. Работа над созданием собственных открытых ресурсов начнётся после доведения движка до уровня оригинальной игры. Из пока нереализованных возможностей движка отмечается поддержки эпидемий, крыс, многопользовательского режима, некоторые полноэкранные анимации, визуальные эффекты для некоторых видов пациентов (лучевая болезнь, размягчение тканей), графики страховой компании и графики заболеваемости.
