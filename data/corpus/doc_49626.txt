Новый выпуск Raspbian, дистрибутива для Raspberry Pi

Разработчики проекта Raspberry Pi опубликовали обновление дистрибутива Raspbian. Для загрузки подготовлены две сборки - сокращённая (351 Мб) для серверных систем и полная (1 Гб), поставляемая с пользовательским окружением PIXEL (ответвление от LXDE). Для установки из репозиториев доступно около 35 тысяч пакетов. В новом выпуске: В состав добавлен видеопроигрыватель VLС, в котором обеспечена поддержка средств аппаратного ускорения GPU VideoCore при декодировании форматов H.264, MPEG-2 и VC-1; До версии 3.0 обновлена интегрированная среда разработки Thonny, которая может использоваться для обучения новичков написанию приложений на языке Python. Thonny сочетает в одном приложении простой для освоения начинающими интерфейс и продвинутые возможности для опытных программистов, такие как пошаговое выполнение кода и инспектирование переменных. Версия Thonny 3 примечательна добавлением новых возможностей для отладки, таких как точки останова и функция Assistant для анализа кода на наличие ошибок, не выявляемых при простой проверке синтаксиса. По умолчанию предложен упрощённый режим интерфейса для новичков, предлагающий фиксированную раскладку элементов без меню. Для переключения в обычный режим достаточно кликнуть на специальную ссылку в правом краю панели инструментов; Унифицирован процесс настройки параметров рабочего стола при помощи приложения Appearance Settings. Большая часть приложений модифицирована для корректной загрузки глобальных файлов конфигурации из каталога /etc/xdg. В будущем выпуске в Appearance Settings планируется перейти к правке только глобальных настроек (/etc/xdg/) не трогая локальные (~/.config), что позволит решить проблему с потерей ручных настроек после изменения конфигурации через GUI. В текущей версии при обновлении конфигурации добавлено предупреждение, позволяющее не трогать локальные настройки. Для применения новых настроек по умолчанию в интерфейс Appearance Settings добавлены специальные кнопки и предусмотрена возможность возврата к ранее используемым настройкам (резервная копия старых локальных файлов конфигурации сохраняется в каталоге oldconffiles); Предложено два разных установочных образа: Образ по умолчанию предлагает минимальное окружение с рабочим столом LXDE, браузером Chromium, мультимедийным проигрывателем VLC, Python и рядом вспомогательных утилит. Полный образ включает все рекомендованные программы, в том числе LibreOffice, Scratch, SonicPi, Thonny и Mathematica. После установки урезанной сборки можно привести окружение к виду полной сборки, выбрав все опции в конфигураторе "Recommended Software";
