Релиз программы для управления фотографиями digiKam 2.6.0

Вышла новая версия программы для управления коллекцией фотографий digiKam 2.6.0, единственного свободного менеджера фотографий, в котором реализована функция распознавания лиц, позволяющая автоматически определять фотографии, на которых присутствуют определённые люди и присваивать им соответствующие теги. По сравнению с прошлым выпуском отмечено исправление 118 ошибок (12 приводили к краху программы). Во входящем в состав пакета наборе плагинов (Kipi-plugins) устранена 49 ошибок (7 крахов). Основные улучшения: Добавлен интерфейс для отслеживания прогресса выполнения параллельно запущенных в фоне задач. Отслеживается в том числе и работа Kipi-плагинов; Новый инструмент для упрощения выполнения работ по сопровождению коллекции фотографий и выполнения типовых работ над всей коллекцией. В состав пакета с Kipi-плагинами добавлена утилита для эскпорта изображений в web-сервисы ImageShack, MediaWiki и Imgur; digiKam портирован для использования библиотеки управления цветностью LCMS 2. Также добавлен новый инструмент для нормализации цветов на отсканированных плёночных негативах.
