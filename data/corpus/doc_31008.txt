Корректирующий релиз Pidgin 2.9.0 с устранением DoS-уязвимости

Спустя две недели с момента выхода Pidgin 2.8.0, представлена новая версия многоплатформенного клиента для быстрого обмена сообщениями - Pidgin 2.9.0. Релиз носит корректирующий характер, из исправленных ошибок можно отметить: Графическая оболочка Pidgin: Устранена уязвимость, позволяющая удаленно инициировать отказ в обслуживании (исчерпание доступной процессу памяти) при попытке отображения в списке контактов специально оформленной пиктограммы в формате GIF; Значительно увеличена производительность при работе в больших IRC-каналах (устранена регрессия в выпуске 2.8.0); Налажена работа опции создания беседы (Conversation->Add) для аккаунтов AIM и MSN; Исправлены проблемы с сортировкой участников чата (устранена регрессия в выпуске 2.8.0); В консольном клиенте Finch решены проблемы со входом в ICQ; В библиотеке libpurple при создании TCP-релеев теперь используется TCP-порт из конфигурации TURN; Устранен крах в реализации клиентов для сетей AIM и ICQ, проявляющийся на некоторых редкоиспользуемых операционных системах; Решена проблема с компиляцией плагина для интеграции с Evolution.
