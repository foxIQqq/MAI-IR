Третья альфа-версия Mageia 2, форка дистрибутива Mandriva

Представлена третья альфа-версия Linux-дистрибутива Mageia 2, в рамках которого независимым сообществом энтузиастов развивается форк проекта Mandriva. Для загрузки доступны 32- и 64-разрядные DVD-сборки (4 Гб) и минимальный образ для установки по сети (40 Мб). Набор LiveCD на базе GNOME и KDE с различными наборами локализации будет подготовлен примерно через неделю, после решения некоторых проблем, вызванных миграцией на инструментарий Dracut. По сравнению со второй альфа-версией добавлены следующие улучшения: Ядро Linux обновлено до версии 3.2. В релизе планируется использовать ядро 3.3; Продолжена интеграция системного менеджера systemd, который будет использоваться по умолчанию. Возможность загрузки с классической системой инициализации будет сохранена для тех, кто хочет повременить с переходом на systemd; В DVD-сборку кроме трёх базовых десктоп-окружений (KDE, GNOME, LXDE) интегрирована оболочка на базе Enlightenment e17; Обновление системных компонентов и средств разработки: glibc 2.14.1, gcc 4.6.2, python 2.7.2/3.2.2, perl 5.14.2, ruby 1.8.7, rails 2.3.14, php 5.3.9 RC4, smarty 2.6.26/3.1.5, java openjdk-1.6.0.0, sun-java-1.6.0.26. IDE: eclipse 3.7.1, netbeans 7.0.1, anjuta 3.3.2, ninja-ide Python IDE 2.0; Обновление графических окружений: KDE 4.7.97 (KDE 4.8 RC2), GNOME 3.3.3, свежая сборка LXDE (в том числе файловый менеджер Pcmanfm 0.9.10). В репозиториях также доступны для установки оконные менеджеры dwm, scrotwm, i3, icewm, awesome, enlightenment (e17), openbox, fvwm2; Обновление мультимедийных программ: редактор анимации Pencil-Planner 0.4.4b, Amarok 2.5beta1, Cantata music player 0.1.2, kdenlive 0.8.2.1, mplayer (+ GUI gnome-mplayer, kmplayer, mplayer-gui, smplayer), vlc, xine (+ GUI kaffeine и xine-ui); Вместо MySQL предлагается использовать MariaDB 5.5; Обновление серверного ПО: apache 2.2.21, postfix 2.8.6, drupal 7.10, mediawiki 1.16.5;
