Релиз распределенной системы управления версиями Mercurial 1.5

Вышел релиз распределенной системы управления версиями Mercurial 1.5. Из добавленных в новой версии улучшений можно отметить: Расширены возможности команды heads при работе с именованными ветками; Добавлен новый XML-шаблон для вывода логов; Появилась возможность импорта набора патчей из email, почтовых ящиков, файлов и URL; Для команд clone, bundle, incoming, outgoing, pull и push добавлена поддержка опции -b/--branch; Реализована опция server.validate для проверки полного завершения отправки изменений на сервер; В subrepos добавлена базовая поддержка Subversion; Добавлена поддержка аутентификации по SSL-сертификатам и улучшена поддержка IPv6; При указании переменной окружения HGPLAIN формируемый вывод теперь более корректен для использования внутри скриптов; Расширены возможности web-интерфейса; В комплект включено новое расширение для отображения прогресса выполнения операции (ProgressExtension). Об особенностях Mercurial можно прочитать в анонсе одного из прошлых релизов.
