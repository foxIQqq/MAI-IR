Выпуск дистрибутива EndeavourOS 2020.07.15, продолжившего развитие проекта Antergos

Опубликован выпуск проекта EndeavourOS 2020.07.15, пришедшего на смену дистрибутиву Antergos, развитие которого было прекращено в мае 2019 года из-за нехватки у оставшихся мэйнтейнеров свободного времени для поддержания проекта на должном уровне. Дистрибутив предлагает простой инсталлятор для установки базового окружения Arch Linux с рабочим столом Xfce по умолчанию и возможностью установить один из 9 типовых рабочих столов на основе i3-wm, Openbox, Mate, Cinnamon, GNOME, Deepin, Budgie и KDE. Endeavour OS даёт возможность пользователю без лишних усложнений установить Arch Linux с необходимым рабочим столом в том виде, каким он задуман в своей штатной начинке, предлагаемой разработчиками выбранного рабочего стола, без дополнительных предустановленных программ. Размер установочного образа 1.7 ГБ (x86_64). В новом выпуске обновлены версии программ, в том числе поставляется ядро Linux 5.7, Mesa 20.1.3, Firefox 78.0.2 и инсталлятор Calamares 3.2.26. В состав включена простая и компактная программа первоначальной настройки Welcome, через которую можно выполнить такие операции как очистка кэша установленных пакетов, установить альтернативные пакеты с ядром Linux из репозиториев Arch, добавить кнопки для запуска произвольных команд и сценариев. Из планов на ближайшее время также отмечается формирование сборки для устройств на базе архитектуры ARM.
