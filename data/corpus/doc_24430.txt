Вышла четвертая бета-версия Firefox 3.6

Вышла четвертая бета-версия Firefox 3.6, в которой исправлена 131 ошибка и представлена поддержка HTML 5 API для управления локальными файлами. Новая возможность позволяет разработчиком организовать обработку локальных файлов через JavaScript после их выбора пользователем в web-форме, но без загрузки на сервер. В качестве примера приводится скрипт, позволяющий локально отредактировать набор изображений перед их отправкой на сервер. Судя по всему, перед релизом ожидается выход ещё одной бета версии. Также можно отметить, что около 70% разработчиков сторонних дополнений уже адаптировали код для работы с Firefox 3.6. Статус поддержки дополнений можно посмотреть на странице "Add-on Compatibility Reporter". Подробный обзор новшеств Firefox 3.6, реализованных в прошлых тестовых версиях, можно увидеть в тексте данного анонса.
