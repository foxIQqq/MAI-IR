Вышел Calculate Linux 14.16.2

Доступен релиз отечественного дистрибутива Calculate Linux 14.16.2, построенного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в шести редакциях, собранных для архитектур i686 и x86_64: Calculate Directory Server (CDS), Calculate Linux Desktop с рабочим столом KDE (CLD) и XFCE (CLDX), Calculate Media Center (CMC), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS). Новый выпуск примечателен возвратом к классическим именам сетевых интерфейсов, за создание файла 70-persistent-net.rules в eudev отвечают шаблоны утилит. Проведена дальнейшая работа по сокращению времени загрузки, OpenRC теперь использует параллельный режим запуска сервисов. Состав пакетов десктопов CLD и CLDX был значительно почищен. Основные изменения: Вместо Plymouth для отображения декорации во время загрузки используется более легковесный и лучше совместимый с Gentoo Splashutils. В OpenRC по умолчанию используется параллельная загрузка. Возвращены классические имена сетевых интерфейсов. Файл 70-persistent-net.rules создаётся шаблонами утилит. Почищен состав пакетов в CLD и CLDX. Сокращены зависимости в CMC и CLS. Исправлено подвисание Chromium в CLDX при открытии большого количества закладок. Исправлена работа USB 3G/4G модема. Интервал проверки обновлений по умолчанию изменён на 24 часа. В CLD и CLDX добавлен SMPlayer YouTube Browser Редактор разделов GParted в CLD заменён на KDE Partition Manager. В дистрибутивы включён Python 3.4. Исправлено сообщение об ошибке в ПК с картридерами. Исправлено определение установленных ОС. Исправлена загрузка с USB-контроллеров некоторых моделей. Libreoffice обновлён до версии 4.4.3, которая отмечена как пригодная для внедрения на крупных предприятиях.
