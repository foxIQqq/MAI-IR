Доступна тестовая версия Opera 10.51 для Unix-подобных систем

Вышла новая тестовая версия web-браузера Opera 10.51 для Unix-подобных систем, отличающаяся полной переработкой процесса формирования установочных пакетов (.tar, .rpm и .deb), например, появилась возможность установки нескольких разных версий opera в разные директории. К сожалению, несмотря на выход финального релиза для платформы Windows, Opera 10.5 для Unix еще требует значительной доработки: не решен вопрос интеграции в десктоп-окружение KDE, наблюдаются проблемы со шрифтами, вводом не ASCII-текста, работой под управлением композитных оконных менеджеров. Главные улучшения Opera 10.5: вместо использования библиотеки Qt, пользовательский интерфейс формируется с помощью низкоуровневой библиотеки Xlib; интегрирован новый JavaScript-движок Carakan, отличающийся значительным приростом производительности; добавлена поддержка HTML5 тэга Video. Более полный список новшеств можно посмотреть в анонсе первой тестовой версии.
