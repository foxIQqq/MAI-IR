Ubuntu 9.04 сертифицирован на предмет полной совместимости с Java SE 6

Разработчики Ubuntu сообщили об официальной сертификации пакета OpenJDK 6 из состава Ubuntu 9.04 на предмет полной совместимости с оригинальной реализацией Java SE 6 на архитектурах amd64 (x86_64) и i386 (ix86). Комплект тестов Java SE Test Compatibility Kit (TCK) был пройден в конце мая, после чего месяц был потрачен на получение официального сертификата от компании Sun Microsystems. Java TCK — это набор утилит и документов, удостоверяющих, что данная реализация Java соответствует спецификации. Он содержит более 80 тыс. тестов и около миллиона строк кода. Прохождение тестов подтверждает, что в полностью открытом пакете OpenJDK из состава Ubuntu, все API Java SE 6 работают как положено, а исполняемый код может быть запущен на любой Java-платформе.
