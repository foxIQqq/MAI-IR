Релиз интерфейса для управления СУБД PostgreSQL - pgAdmin 1.16

После 16 месяцев разработки вышел релиз pgAdmin 1.16.0, многоплатформенного графического интерфейса для управления СУБД PostgreSQL. Проект написан на языке C++ с использованием тулкита wxWidgets и доступен для Windows, Linux, FreeBSD, Mac OS X и Solaris. Из добавленных в новой версии улучшений отмечается: Обеспечение поддержки новых возможностей PostgreSQL 9.2, например, SSL-соединений со сжатием трафика, дополнительной статистики, условий "non valid", опции "security_barrier", расширенной системы привилегий и т.п.; Новый инструмент для импорта данных из файлов с произвольной структурой (например, в виде, подобном СSV); Поддержка поиска по именам объектов и их типам; Поддержка изменения схемы для объекта; Переработанный диалог настройки параметров приложения; Возможность создания таблицы с копированием структуры из другой таблицы.
