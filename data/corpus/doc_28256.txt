Релиз системы виртуализации VirtualBox 3.2.10

Компания Oracle выпустила корректирующий релиз системы виртуализации VirtualBox 3.2.10, в котором исправлены 62 ошибки и добавлена поддержка Ubuntu 10.10. Основные исправления: Устранены проблемы с работой гостевых окружений, которым выделен большой объем памяти. Увеличена производительность виртуальных машин с объемом ОЗУ, превышающим 2 Гб; Налажена обработка клавиатурной комбинации Ctrl+Break на хостах с X11; В GUI-интерфейсе добавлена возможность минимизации виртуальной машины в мини-тулбар; Добавлена проверка на наличие в системе Linux-ядра 2.6.36 с целью обхода ошибки, проявляющейся при работе асинхронного ввода/вывода вкупе с ФС ext4 и xfs; В гостевых окружениях OpenSolaris по умолчанию задействован SATA-контроллер; Устранены ошибки ввода/вывода в гостевых системах, после операции уплотнения VDI-образов; При отмене операции слияния дисковые образы более не становятся поврежденными; Добавлена поддержка проброса ATAPI для аудио CD; В SATA драйвере решена проблема с зависанием Solaris в процессе загрузки гостевой ОС и улучшена обработка ситуации нехватки свободного места на диске; В коде поддержки сетевых мостов для Linux решена проблема с зависанием в процессе завершения работы VM; В 3D-подсистеме добавлена поддержка расширения GLX_ARB_get_proc_address, решены проблемы с работой в полноэкранном режиме, налажена работа 32-разрядных OpenGL-приложений в 64-разрядных Windows XP/Vista; Добавлена поддержка X.Org Server 1.9; Исправлены проблемы с компиляцией для Linux-ядра 2.6.36. Загрузить VirtualBox 3.2.10 для различных ОС (в том числе образ Guest Additions отдельным файлом) можно здесь.
