Релиз Novell Open Enterprise Server 11 SP1 и SUSE Cloud 1.0

Компания Novell выпустила релиз операционной системы Open Enterprise Server 11 SP1, основанной в 2005 году с целью обеспечения безболезненной миграции на Linux пользователей NetWare. Продукт базируется на SUSE Linux Enterprise Server 11 SP2 и предоставляет полную поддержку всех сервисов NetWare. Среди новшеств Open Enterprise Server 11 SP1: Улучшенная поддержка виртуализации. В дополнение к средствам виртуализации на базе Xen добавлена сертифицированная поддержка продуктов VMware; Расширенные возможности по установке патчей и обновлений. Добавлен режим быстрой установки обновлений, позволяющий значительно упростить применение патчей; Проведение оптимизации энергопотребления; Сокращение цикла подготовки новых выпусков. Новые возможности отныне будут интегрироваться в продукт по мере их готовности и будут доступны через систему обновлений помимо патчей с исправлением ошибок и проблем безопасности; Поддержка системы Novell Filr для организации совместного доступа к файлам с любых типов устройств и автоматизации выполнения резервного копирования; 8400 пакетов в репозитории. Одновременно можно отметить выпуск нового продукта SUSE Cloud 1.0, представляющего собой вариант SUSE Linux Enterprise Server, рассчитанный на быстрое развёртывание инфраструктуры для обслуживания приватных облачных сервисов на базе платформы OpenStack. Используя подготовленный дистрибутив корпорации имеют возможность создания локальных облачных платформ, подобных Amazon EC2, но выполняемых исключительно на собственных серверах и полностью подконтрольных компании. Дистрибутив поддерживает установку как сервера Cloud Administration Server для централизованного управления облачной инфраструктурой, так и узлов Cloud Control Nodes, Compute Nodes и Storage Nodes на которых осуществляется управление пользователями, хранение данных и выполнение виртуальных машин. Из компонентов OpenStack в продукте задействованы: инструментарий Nova для управления работой больших групп VPS-серверов, сервис доставки и хранения образов виртуальных машин Glance, система идентификации Keystone и управляющий web-интерфейс Dashboard. В комплект также входят средства интеграции с конструктором образов виртуальных окружений SUSE Studio и системой для управления инфраструктурой Linux-серверов SUSE Manager. Дополнительно подготовлен репозиторий, который можно использовать для установки пакетов с компонентами OpenStack в системах на базе SLES 11 SP3, openSUSE 12.1 и 12.2.
