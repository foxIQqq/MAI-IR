Вышел Calculate Linux Desktop 9.9 KDE и XFCE

Вышла новая версия Calculate Linux Desktop (CLD), современный десктоп на базе графического окружения KDE и Calculate Linux Desktop XFCE (CLDX) с рабочим столом XFCE, который может выполнять большинство офисных задач. Основной особенностью является быстрая установка, удобная система обновлений и возможность хранить учетные записи пользователей на сервере. CLD поддерживает работу в локальной сети совместно с Windows машинами, где в качестве контроллера домена выступает Calculate Directory Server. Основные изменения Начиная с этой версии Calculate Linux Desktop (KDE и XFCE), собирается на базе Calculate Linux Scratch, соответственно доступны все улучшения проекта CLS: поддержка интерактивной сборки системы, переход на aufs2. В CLD заменены следующие программы: клиент IRC - Xchat на Konversation, словарь GStarDict на GoldenDict. Добавлены: Partitionmanager (KDE Partition Manager). По умолчанию теперь используется браузер Firefox. В CLDX по умолчанию работает Compiz. По аналогии с CLD дистрибутив включает исходники ядра, портежи, проприетарные драйвера nvidia. Добавлены следующие программы: gnome-ppp, fusesmb, xfce4-taskmanager, xfce4-power-manager, thunar-volman. Оптимизация скорости работы за счет: флагов сборки, добавления prelink в крон, использования preload, перехода на ядро 2.6.30 и KDE 4.3, упаковки ядра и initrd в lzma архив. Новая утилита cl-unmask для демаскировки пакетов. Состав CLD 9.9: X.Org 7.4, Kernel 2.6.30.5, Firefox 3.5.3, KDE 4.3.1, OpenOffice 3.1.1, Calculate-client 2.1.6 Состав CLDX 9.9: X.Org 7.4, Kernel 2.6.30.5, Firefox 3.5.3, XFCE 4.6.1, OpenOffice 3.1.1, Calculate-client 2.1.7
