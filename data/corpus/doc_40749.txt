Новая версия свободной системы 3D-моделирования Blender 2.72

После трёх месяцев разработки представлен выпуск свободного пакета 3D-моделирования Blender 2.72. Последние выпуски Blender развиваются в процессе подготовки нового анимационного фильма Gooseberry и включают доработки, требуемые для реализации тех или иных идей. Например, сейчас разработчики сосредоточились на развитии средств симуляции волос и одежды. Основные новшества Blender 2.72: В движке рендеринга Cycles задействованы средства GPU при выполнении объёмного рендеринга, подповерхностного рассеивания и связанного дрожания разных объектов. Представлен ряд новых оптимизаций производительности, в том числе задействованы инструкции AVX2, доступные в CPU Intel Haswell. Внесены улучшения, связанные с объёмным рендерингом и обработкой теней. OSL (Open Shading Language) обновлён до версии 1.5. Добавлен оператор для быстрой настройки связки узлов для дыма и огня. В интерфейсе пользователя реализована поддержка кругового меню (Pie menu), элементы которого отображаются вокруг позиции мыши, а вместо указателя появляется индикатор направления. Изменено оформление всплывающих подсказок. Эскизы внутреннего рендеринга теперь приведены в соответствие с эскизами при рендеринге через Cycle, т.е. вначале отрисовываются в низком разрешении, а потом улучшаются; В инструментах для создания анимации упрощён процесс фильтрации каналов, внесены небольшие изменения в интерфейсы редактирования, проведена чистка блоков данных, улучшен процесс отмены изменений каркаса; В средства моделирования добавлен новый инструмент разделения грани по краям (Face split by edges). Реализованы новые опции в инструментах разделения вершин (Dissolve vertex), выбора коллектора (Select manifold) и скоса (Bevel). Добавлен инструмент для разделения геометрии в режиме редактирования мэшей (Editmesh Intersect); В режимах лепки и рисования представлен новый рабочий процесс создания текстур, более привязанный к материалам мэшей и упрощающий выбор доступных для рисования материалов. При подготовке текстур, реализована возможность использования градиента в качестве источника цвета. Добавлены новые инструменты для заострения (Sharpen), заполнения (Fill), наложения масок (Mask), управления непрозрачностью кисти (Brush Opacity Control). Добавлено 16 новых режимов смешивания. Во всех режимах рисования унифицированы цвета для кистей, добавлена функция штрихования линиями и кривыми, обеспечена поддержка палитр, внесены оптимизации для увеличения удобства работы; В compositor добавлен новый узел Sun Beams, симулирующий эффект струящегося яркого света в сумеречной среде; Средства линейного нефотореалистичного рендеринга (Freestyle) интегрированы в движок Cycles, который теперь можно использовать наряду со встроенным движком рендеринга (BI). Для Cycles и BI подготовлено новое свойство цвета Freestyle line. Значительно расширен Freestyle Python API; Добавлено дополнение для импорта данных в формате DXF. Улучшена поддержка формата FBX, для которого обеспечена возможность импорта каркаса, ключей форм (shapekeys) и анимации. Новые дополнения для UV-преобразований, управления именованными слоями и публикации игр.
