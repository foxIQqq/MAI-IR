Выпуск дистрибутива Nitrux 1.1 с рабочим столом Nomad

Доступен релиз дистрибутива Nitrux 1.1, построенного на пакетной базе Ubuntu и технологиях KDE. Дистрибутив развивает собственный рабочий стол Nomad, который представляет собой надстройку над пользовательским окружением KDE Plasma. Для установки дополнительных приложений продвигается система самодостаточных пакетов AppImages и собственный центр установки приложений NX Software Center. Размер загрузочного образа составляет 1.4 Гб. Наработки проекта распространяются под свободными лицензиями. Рабочий стол Nomad предлагает иное стилевое оформление, собственную реализацию системного лотка, центра вывода уведомлений и различных плазмоидов, таких как конфигуратор сетевых соединений и мультимедийный апплет, сочетающий средства для регулирования громкости с интерфейсом управления воспроизведением мультимедийного контента. Из развиваемых проектом приложений также выделяется интерфейс для настройки межсетевого экрана Nomad Firewall, позволяющий управлять сетевым доступом на уровне отдельных приложений. Среди входящих в базовую поставку приложений: файловый менеджер Index (можно использовать и Dolphin), текстовый редактор Kate, архиватор Ark, эмулятор терминала Konsole, браузер Chromium, музыкальный проигрыватель VVave, видеопроигрыватель VLC, офисный пакет LibreOffice и просмотрщик изображений Pix. В новом выпуске: Добавлен новый компонент znx, предоставляющий инструменты для управления операционной системой. В частности, znx позволяет решать такие задачи, как установка нескольких операционных систем на одном устройстве без ручной разбивки разделов, организация атомарных обновлений системы и откат обновлений в случае проблем; Представлен собственный фреймворк построения интерфейса пользователя MauiKit, основанный на Qt Quick Controls 2. Фреймворк предлагает готовый набор стилевых решений, виджетов и шаблонов элементов интерфейса, которые можно использовать на разных платформах и видах устройств. Построенные на базе MauiKit приложения могут быть собраны для стационарных рабочих столов Linux, Android и Plasma Mobile; Одним из приложений, построенных на базе MauiKit, является музыкальный проигрыватель VVave, предлагающий инструменты для управления музыкальной коллекцией и поддерживающий извлечение метаданных через online-сервисы. На базе MauiKit проектом также развивается файловый менеджер Index, имеющий встроенный эмулятор терминала, поддерживающий предпросмотр различных видов контента (музыка, видео, текст, изображения) и доступный не только для стационарных Linux-дистрибутивов, но и для Android. Реализована система ведения заметок Buho; Создан новый просмотрщик изображений Pix, включающий функции управления галереей фотографий. Программа также доступна для платформы Android; В качестве основного метода установки новых программ теперь позиционируется AppImages. Поддержка установки пакетов из PPA-репозиториев прекращена; Предложена альтернативная реализация меню приложений Nomad Simplemenu, позволяющая пользователю на своё усмотрение группировать программы по категориям; Прекращена поддержка установки на системах с BIOS и таблицей разделов MBR. Поддерживается только UEFI и GPT; Обновлены версии компонентов дистрибутива: KDE Plasma 5.14.2, KDE Frameworks 5.52.0, Qt 5.11.2, ядро Linux 4.19.0, Ubuntu 18.04.1 LTS.
