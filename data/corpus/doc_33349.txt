Релиз Calculate Linux 11.15 с GNOME 3

Вышел релиз Calculate Linux 11.15 - дистрибутива, основанного на Gentoo и полностью с ним совместимого, но отличающегося наличием своего установщика, фирменных утилит Calculate для настройки системы, готового гетерогенного клиент-серверного решения для быстрого развертывания системы на предприятии. Основным новшеством версии 11.15 является переход на GNOME 3 и оболочку GNOME-Shell, с сохранением привычного интерфейса рабочих столов Calculate Linux и поддержкой работы в домене под управлением Calculate Directory Server. Для организации пользовательского окружения в дистрибутив добавлен набор собственных расширений. Переключение между окнами выполняется привычным сочетанием Alt+Tab. Панель для часто запускаемых приложений, традиционно для дистрибутива, размещена внизу экрана. На верхней панели находятся меню запуска приложений, переключатель рабочих столов, панель задач, корзина и системный трей, часы, кнопки блокирования экрана и завершения работы. Системные уведомления перемещены в правую верхнюю часть экрана под системный трей. Activities по умолчанию выключен. Для пользователей, ценящих возможности GNOME 3, сохранена возможность отключить расширения, приведя рабочий стол в соответствие классическому поведению GNOME 3. В состав дистрибутива входит ядро Linux 3.2.8, GNOME 3.2, KDE 4.8.1, XFCE 4.8, Chromium 17, LibreOffice 3.5, XBMC 11.0, Gimp 2.6.11, OpenLDAP 2.4.24, Samba 3.5.11. Начиная с версии 11.15, все редакции дистрибутива Calculate Linux поддерживают репозитории пакетов для обновлений. Шаблоны настройки системы перенесены в оверлей, позволяя оперативно получать поддержку новых пакетов. Для загрузки доступны следующие редакции дистрибутива: Calculate Directory Server (CDS), Calculate Linux Desktop с рабочим столом KDE (CLD), GNOME (CLDG) и XFCE (CLDX), Calculate Media Center (CMC), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS).
