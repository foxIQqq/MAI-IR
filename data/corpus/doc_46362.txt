В Fedora Linux добавлена поддержка самодостаточных пакетов Snap

В штатные репозитории пакетов Fedora Linux 24, 25 и 26 включён пакет snapd с инструментарием для управления самодостаточными пакетами в формате snap. Таким образом, для начала работы с пакетами в формате Snap пользователям Fedora Linux теперь достаточно выполнить "sudo dnf install snapd", после чего можно использовать утилиту snap. В момент первой установки snap в систему также будет установлен snap-пакет core c набором базовых библиотек для функционирования пакетов snap. Так как для изоляции в snap используется механизм AppArmor, который не поддерживается в Fedora, то пакеты запускаются без применения изоляции, т.е. нужно соблюдать осторожность и не устанавливать непроверенные приложения. В будущем для обеспечения изоляции в Fedora будет подготовлен отдельный бэкенд на основе механизма SELinux.
