Выпуск Chrome OS 96

Опубликован релиз операционной системы Chrome OS 96, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 96. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 96 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 96: Значительно расширены возможности приложения для работы с камерой. В программу встроен отдельный режим для сканирования документов, позволяющий использовать фронтальную или заднюю камеру вместо сканера. В процессе сканирования программа автоматически определяет границы документа для обрезки лишнего фона. Полученный документ можно сохранить в формате PDF или JPEG, отправить в социальную сеть или Gmail, передать на смартфон при помощи функции Nearby Share. При подключении к Chromebook внешней камеры добавлена поддержка регулировки угла наклона и приближения/удаления при помощи блока настроек "Pan-Tilt-Zoom" для выбора видимой области снимка. В программе для работы с камерой также предложен режим "видео" для быстрой записи видео, возможность создания снимка по таймеру и режим сканирования QR-кодов. Все снимки и видео автоматически сохраняются в каталог "Camera" и доступны из файлового менеджера. В следующем году планируется добавить возможность создания анимированных GIF-ов и реализовать голосовое управление камерой через Google Assistant (например, для создания снимка будет достаточно сказать "сделай фото"). Предложена новая боковая панель, упрощающая сопоставление в браузере данных со страниц на разных сайтах, например, при работе с поисковой системой можно открыть интересующую страницу не закрывая список с результатами поиска - если информация не соответствует ожиданию можно сразу открыть другую страницу без возвращения назад и не теряя результаты поиска. Добавлена возможность использования функции Nearby Share из ARC++ (App Runtime for Chrome), прослойки для запуска Android-приложений в Chrome OS. Функции Nearby Share позволяет быстро и безопасно передавать файлы находящимися поблизости устройствам, на которых запущен браузер Chrome. Ранее Nearby Share можно было использовать из файлового менеджера, web-приложений и системных приложений Chrome OS. Теперь функция стала доступна и для приложений Android. Добавлена настройка, позволяющая использовать приложения в качестве обработчиков по умолчанию для различных видов ссылок. Например, можно настроить вызов PWA-приложения Zoom для обработки перехода по ссылкам на zoom.us. В экранную клавиатуру добавлен вывод рекомендации для вставки данных, в течение последних двух минут добавленных в буфер обмена. Если поместить данные в буфер обмена и открыть виртуальную клавиатуру, то добавленные данные будут показаны в верхней строке и для их вставки в текст достаточно одного клика. Улучшен интерфейс для установки обоев рабочего стола. В конфигуратор добавлен отдельный раздел с настройками вывода уведомлений (ранее уведомления настраивались только через меню Quick Settings). Ветка Chrome OS 96 будет сопровождаться 8 недель в рамках цикла LTS (Long-term support).
