Доступна публичная бета-версия Skype для Asterisk

Публичная бета-версия Skype для Asterisk теперь доступна для бесплатной загрузки через web-магазин Digium, после обязательной общедоступной регистрации. После оформления формального заказа (стоимость 0 долларов) на указанный при регистрации email будет отправлен лицензионный ключ и ссылка для прямой загрузки Skype для Asterisk. Бета-версия будет доступна для загрузки до 7 августа, программа имеет ограничение по времени работы, которое истекает 31 августа.
