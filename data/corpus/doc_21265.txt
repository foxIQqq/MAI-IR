Вышел релиз X Server 1.6.1

Доступен релиз X Server 1.6.1 с исправлением некоторых ошибок, обнаруженных с момента выхода X Server 1.6.0. В частности устранены проблемы, возникающие при работе на системах с SELinux, исправлены ошибки в расширениях randr, glx, Xi, xkb и Xext. Летом планируется выпустить X.Org 7.5 / X Server 1.7, в котором появится поддержка технологии Multi-Pointer X и переработанной подсистемы ввода X Input 2, что позволит организовать работу нескольких независимых устройств ввода, например, несколько управляемых разными мышами курсоров на экране или ввод в разные окна с разных клавиатур.
