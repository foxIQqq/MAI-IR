Релиз менеджера фотографий Shotwell 0.12.0

Вышел релиз программы для управления коллекцией фотографий Shotwell 0.12.0. Готовые бинарные сборки доступны для Linux-дистрибутивов Ubuntu и Fedora. Дополнительно можно отметить важное достижение - объем исходных текстов проекта превысил 100 тыс. строк кода. Из добавленных в новой версии улучшений, можно отметить: Интеграция инструмента для выпрямления фотографий, снятых под определённым углом, в которых изображение заваливается на бок; Поддержка сборки с использованием GTK+3; Использование механизма аутентификации OAuth для подключения к своему аккаунту в сервисе Flickr; Улучшенная поддержка устройств на базе платформы Android; Возможность исключения из результатов поиска фотографий с заданными тегами; Устранение ошибок, по словам разработчиков Shotwell 0.12 стабилен как никогда раньше. Базовые возможности Shotwell: Инструменты для импорта и конвертации новых фотографий; Удобные возможности каталогизации и навигации по имеющейся коллекции, поддержка группировки по времени и тегам; Поддержка выполнения типовых операций по обработке изображений (вращение, устранение эффекта красных глаз, корректировка экспозиции, оптимизация цветности и т.п.); Средства для публикации в социальных сетях, таких как Facebook, Flickr и Picasa.
