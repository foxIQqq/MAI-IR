Выпуск online-редакторов ONLYOFFICE Docs 6.3

Доступен новый выпуск ONLYOFFICE DocumentServer 6.3 с реализацией сервера для online-редакторов ONLYOFFICE и организации совместной работы. Редакторы можно использовать для работы с текстовыми документами, таблицами и презентациями. Код проекта распространяется под свободной лицензией AGPLv3. В ближайшее время ожидается обновление продукта ONLYOFFICE DesktopEditors, построенного на единой кодовой базе с online-редакторами. Десктоп-редакторы оформлены в виде приложений для рабочего стола, которые написаны на JavaScript с использованием web-технологий, но объединяют в одном наборе клиентские и серверные компоненты, оформленные для самодостаточного использования на локальной системе пользователя, без обращения к внешнему сервису. Для совместной работы на своих мощностях также можно использовать платформу Nextcloud Hub, в которой обеспечена полная интеграция с ONLYOFFICE. В OnlyOffice заявлена полная совместимость с форматами MS Office и OpenDocument. Среди поддерживаемых форматов: DOC, DOCX, ODT, RTF, TXT, PDF, HTML, EPUB, XPS, DjVu, XLS, XLSX, ODS, CSV, PPT, PPTX, ODP. Предусмотрена возможность расширения функциональности редакторов через плагины, например, доступны плагины для создания шаблонов и добавления видео с YouTube. Готовые сборки сформированы для Windows и Linux (deb- и rpm-пакеты). Наиболее заметные улучшения: Добавлена поддержка тем оформления интерфейса. Реализована тёмная тема и отдельная лёгкая тема. Темы можно изменить через меню "File tab -> Advanced Settings -> Interface theme". Для экранов с высокой плотностью пикселей реализована возможность увеличения масштаба интерфейса до 150% (помимо 100% и 200%). В будущих обновлениях обещают добавить поддержку уровней 125% и 175%. Компонент для проверки правописания переделан в форме SharedWorker, который работает на стороне браузера и не требует запуска отдельного бэкенда на сервере. Добавлена поддержка добавления файлов в секцию "избранное" напрямую из режимов редактирования. Реализована возможность защиты документов, электронных таблиц и презентаций паролем (File tab -> Protect -> Add password). Добавлена поддержка линейных диаграмм и диаграмм рассеяния. Предоставлена возможность назначения имён для анонимных пользователей. При открытии в Firefox реализована поддержка вывода на печать. Добавлены всплывающие подсказки с информацией о методах макросов. В редакторе документов появилась возможность экспорта в форматы html, fb2 и ePub. В боковую панель добавлены настройки отступа, а в верхнюю панель кнопка для обтекания изображений текстом. Изменена работа с многоуровневыми списками. Переделан режим отслеживания изменений в документе (Collaboration tab -> Track Changes), добавлена поддержка записи изменений в файл. В табличном процессоре: Добавлена поддержка форматов даты "YYYY-MM-DD" (ISO 8601), mm/dd, mm/dd/yyyy и mm/dd/yy. В боковую панель добавлены настройки для выравнивания ячеек. Реализована поддержка открытия файлов Microsoft Office XML 2003. Для сводных (pivot) таблиц реализованы операции создания и удаления групп. Добавлена функция XLOOKUP. Добавлена поддержка комбинированных диаграмм, переделаны настройки диаграмм и предоставлена возможность установки формата именования осей. В редактор презентаций добавлены настройки прозрачности слайда. Предложены кнопки для изменения регистра текста и выделения текста цветом. В верхнюю панель добавлена кнопка для настройки столбцов. Реализована возможность сохранения анимации после экспорта.
