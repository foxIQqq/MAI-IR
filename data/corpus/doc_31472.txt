Легковесные дистрибутивы: Puppy Linux 5.1.3, Toorox 08.2011 "Lite", ConnochaetOS 0.9.0, wattOS R4

Доступны новые версии трех легковесных дистрибутивов Linux: Релиз Wary Puppy Linux 5.1.3, классической сборки Puppy Linux, способной работать на устаревшем оборудовании и продолжающей развитие серии Puppy 4.x. Дистрибутив является самостоятельным проектом, использующим систему пакетов T2. Графическое окружение основано на оконном менеджере JWM. В поставку входит собственный набор GUI-конфигураторов (Puppy Control Panel) и виджетов (Pwidgets - часы, календарь, RSS, состояние соединения и т.п.). Для установки дополнительных программ используется Puppy Package Manager. Параллельно с Wary Puppy развивается еще четыре ответвления дистрибутива Puppy: основанный на пакетной базе Ubuntu Lucid Puppy, инновационный Quirky, оптимизированный для 64-разрядных систем FatDog64 и нацеленного на нетбуки Puppeee. Установочный Live-образ занимает 121 Мб. В состав новой версии добавлен апплет для выбора используемых по умолчанию приложений, Urxvtset (конфигуратор терминала rxvt-unicode) и новые утилиты picscale, pngoverlay, yad. В состав по прежнему сходит браузер SeaMonkey 1.1.18, так как в версии 2.2 наблюдается ряд нерешенных проблем в модуле Composer и ветка 1.1 лучше подходит для маломощных систем. В качестве ядра Linux используется версия 2.6.32. Представлены новые редакции "Xfce" и "Lite" дистрибутива Toorox 08.2011, использующего наработки проекта Gentoo и загрузочные скрипты KNOPPIX. Toorox позиционируется в роли переносного пользовательского окружения, работающего на USB Flash. Дистрибутив снабжен графическим инсталлятором, который позволяет задействовать /home раздел с жесткого диска или Flash-накопителя. При установке на Flash поддерживается три режима установки: статическая, персональная и шифрованная, отличающиеся способом организации хранения домашней директории пользователя. По умолчанию на Flash создается два раздела - ext2 для загрузки и FAT32 для использования в роли хранилища данных. Загрузочный iso-образ полной сборки с графическим окружением Xfce 4.8.0 занимает 1.3 Гб, а сборки "Lite" - 768 Мб. Lite-версия отличается набором входящих в комплект приложений: в комплект не входят программы Ardour, Cinelerra, Gimp, Samba, IcedTea6, Wine, вместо LibreOffice используется Abiword, вместо IceCat (Firefox) - Midori, вместо Thunderbird - Claws Mail, в качестве медиа-плеера используется parole 0.2.0.6, для просмотра фотографий задействован Mirage 0.9.5.2. В комплект также входят Transmission 2.33, pyNeighborhood 0.5.4, файловый менеджер Thunar 1.2.2, ePDF 0.1.7, текстовый редактор Leafpad. Вышел дистрибутив ConnochaetOS 0.9.0, ранее развивавшийся под именем DeLi Linux (прошлая версия вышла три года назад). Дистрибутив основан на наработках Arch Linux и рассчитан на работу на сильно устаревшем оборудовании, начиная с систем на базе CPU Pentium I и 64MB ОЗУ. Десктоп-окружение в ConnochaetOS построено на базе оконного менеджера IceWM 1.3.7. Из компонентов дистрибутива можно отметить: ядро Linux 2.6.32.43, минималистичный браузер xxxterm (на базе движка WebKit), текстовый процессор Abiword, табличный процессор Gnumeric. Дополнительно поддерживается установка ядра Linux-Libre 2.6.39.4, браузера GNU Icecat 3.6.16 и 5.0, графического окружения LXDE 0.5.x. Размер iso-образа составляет 391 Мб. Вышел релиз минималистичного Linux-дистрибутива wattOS 4, построенного на пакетной базе Ubuntu 11.04 и рассчитанного на минимальное потребление ресурсов. Дистрибутив поставляется с графическим окружением LXDE. Размер установочного iso-образа составляет 527 Мб, поддерживается как работа в Live-режиме, так и установка на жесткий диск. В рамках проекта поддерживается специальный PPA-репозиторий, позволяющий оперативно обновлять версии программ, поставляемых в комплекте. В новой версии вместо аудиоплеера foobnix использован audacious, в комплект включен набор кодеков, вместо Totem задействован видеоплеер VLC, в состав включен почтовых клиент Sylpheed, обновлен менеджер фотографий fotoxx, добавлена программа для управления питанием из состава xfce4, вместо Firefox задействован браузер Chromium.
