Проект XreaL переходит на игровой движок Enemy Territory

Через несколько дней после открытия компанией id Software движков игр "Wolfenstein: Enemy Territory" и "Return to Castle Wolfenstein", разработчики свободного движка для создания 3D-шутеров XreaL объявили о переходе проекта на движок "Wolfenstein: Enemy Territory". В ближайшие дни будет создан репозиторий ET-XreaL в котором будет произведена замена движка с Quake 3 Arena на Enemy Territory и произведена интеграция с кодом рендеринга XreaL. По заявлению разработчиков переход не должен вызвать проблем, так как движки, с точки зрения программного интерфейса, похожи между собой. Таким образом, энтузиастам свободных технологий не пришлось долго ждать момента, когда открытый компанией id Software будет дополнен необходимыми для создания работоспособной игры компонентами (графика, карты, текстуры, звуки и другие игровые данные не были открыты).
