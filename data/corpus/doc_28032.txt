Ubuntu вводит практику рецензирования новых программ

Менеджер Canonical по работе с сообществом Джоно Бэкон (Jono Bacon) анонсировал новую экспериментальную инициативу - Ubuntu Application Review Process, цель которой - упростить включение в состав Ubuntu новых приложений, введя в практику строгую проверку качества добавляемых программ для гарантии их безопасности, работоспособности и соответствия заявленным характеристикам. Инициатива обращена в первую очередь к разработчикам, которые хотели бы увидеть своё приложение в Ubuntu Software Center (без добавления программы в репозитории) и затрагивает только исключительно новые приложения, которых ещё нет в официальных репозиториях, таких как main и universe. Для авторов программ процедура рецензирования является упрощенным эквивалентном стандартного MOTU-процесса разработки, позволяющего не углубляться в тонкости поддержки пакетов начать распространять свою программу для пользователей Ubuntu. Новая инициатива не затрагивает новые версии уже существующих приложений, библиотеки и пакеты с данными и документацией, которые продолжают добавляться в репозитории только на прежних условиях. Рецензируемое приложение должно иметь открытый исходный текст и распространяться под одной из лицензий, одобренных организацией Open Source Initiative (OSI). Процесс добавления новой программы для получения одобрения состоит в подготовке PPA-репозитория с готовым обособленным пакетом (установка в /opt, пакет должен включать в себя все зависимости) для заданной программы, отправке запроса на проведение рецензирования и ожидания получения одобрения. С момента отправки заявки, до вынесения решения вносить изменения в пакет запрещено, как только разрешение получено - разработчик получает возможность внесения изменений и выпуска обновлений. При этом, так как пакет распространяется через PPA-репозиторий, обновления с новой версией программы могут выпускаться не дожидаясь выхода нового релиза Ubuntu. Решение о соответствии качества приложения заявленным требованиям будет принимать специальный комитет, состав которого будет состоять из избираемых представителей сообщества.
