Выпуск системы управления контентом Joomla 4.0

Доступен новый значительный выпуск свободной системы управления контентом Joomla 4.0. Из особенностей Joomla можно отметить: гибкие инструменты по управлению пользователями, интерфейс для управления медиа-файлами, поддержка создания многоязычных вариантов страниц, система управления рекламными кампаниями, адресная книга пользователей, голосования, встроенный поиск, функции категоризации ссылок и учета кликов, WYSIWYG-редактор, система шаблонов, поддержка меню, управление новостными потоками, XML-RPC API для интеграции с другими системами, поддержка кэширования страниц и большой набор готовых дополнений. Основные особенности Joomla 4.0: Реализация отдельного макета и контрастного представления для людей с ограниченными возможностями. Улучшенные интерфейсы редактора и медиа-менеджера. Настраиваемые шаблоны электронных писем, отправляемых с сайта. Более эффективные инструменты для поиска содержимого. Изменение архитектуры и кода для увеличения безопасности. Поддержка инструментов для поисковой оптимизации (SEO). Сокращение времени загрузки страниц. Новый компонент "Процессы" (Workflows) для управления действиями в процессе публикации.
