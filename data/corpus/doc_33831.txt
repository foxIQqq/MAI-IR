IBM и SAP портируют OpenJDK для архитектуры PowerPC

Компании IBM и SAP организовали совместный проект по обеспечению в OpenJDK поддержки процессоров PowerPC и подготовки прототипа JDK 7 для PPC64-версий Linux и AIX. В качестве причины портировния называется намерение создания на базе OpenJDK коммерчески лицензируемого Java-продукта для систем, поддерживающих архитектуру PowerPC, созданного поверх OpenJDK по аналогии с Oracle JDK. Отдельное внимание обращается на то, что OpenJDK будет использоваться IBM и SAP в качестве эталонной платформы Java для PowerPC, которая заменит собой проприетарный JDK от компании IBM. Кроме того, так как поддержку PowerPC планируется интегрировать непосредственно в OpenJDK, сообщество получит возможность создания бесплатных и свободно распространяемых сборок нового порта Java для независимых дистрибутивов Linux, собираемых для архитектуры PowerPC.
