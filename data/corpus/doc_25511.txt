Вышел Mono Tools for Visual Studio 1.1 beta

В рамках проекта Mono подготовлено расширение для удобной разработки кросплатформенных .Net приложений в Visual Studio 2008/2010RC. Поддерживается отладка по сети, а также на локальной и виртуальной машине с Windows или Linux. Продукт позволят разработчикам портировать их существующие Windows приложения на другие платформы, создавать новые приложения с нуля, оценивать степень совместимости приложений с Mono и Linux окружением, создавать RPM пакеты для установки в Linux. Новые функции: Поддержка автоматических обновлений в Mono Tools for VS Поддержка Visual Studio 2010 через vsix пакет расширения Поддержка обнаружения изменений в отлаживаемой на удаленной машине сборке. Теперь не нужно копировать каждый раз заново весь проект для отладки. Упрощение упаковки прекомпилированных веб-приложений. Раньше, при необходимости создания RPM для веб-приложения, нужно было сначала воспользоваться пунктом Publish для создания локальной сборки. Теперь в меню Mono появился для этого раздел "Create Linux Package for Project". Тем не менее в пакете до сих пор не хватает более глубокой интеграции с Visual Studio при создании проекта, для того, чтобы по умолчанию использовались сборки Mono. Для частичного устранения этого недостатка отдельно подготовлено специальное расширение для Visual Studio.
