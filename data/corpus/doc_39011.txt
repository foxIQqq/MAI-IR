Лучшие проекты 2013 года по версии сайта LinuxQuestions

Подведены итоги голосования среди участников форума LinuxQuestions.org по выбору лучших приложений 2013 года: Пользовательские приложения для рабочего стола Десктоп дистрибутив - Ubuntu (23.59%); Мобильная платформа - Android (59.15%); Офисный пакет - LibreOffice (85.50%); Десктоп среда - KDE (35.77%); Оконный менеджер - Openbox (18.88%); Средство виртуализации - VirtualBox (54.38%); Файловый менеджер - Dolphin (28.05%); Игровое приложение - 0 A.D. (13.55%), в прошлом году лидировал Battle for Wesnoth; Эмулятор терминала - Konsole (23.87%); Сетевые пользовательские приложения Web-браузер - Firefox (63.54%); ПО для обмена мгновенными сообщениями - Pidgin (47.83%); VoIP-клиент - Skype (44.95%); Мультимедиа Аудиопроигрыватель - VLC (26.85%), в прошлом году лидировал Amarok; Видеопроигрыватель - VLC (66.79%); ПО для обработки графики - GIMP (70.16%); Приложения для администратора Серверный дистрибутив - Slackware (31.83%), выбор связан с тем, что на форуме LinuxQuestions.org в основном общаются пользователи Slackware. Debian набрал 23.95%, CentOS -15.91%, Ubuntu LTS - 12.73%; Приложение для резервного копирования - rsync (39.71%); Приложение связанное с сетевой безопасностью - Wireshark (31.09%); Приложение связанное с безопасностью системы - SELinux (38.46%); Программа для мониторинга - Nagios Core (40.88%); Система управлением конфигурацией - Puppet (42.05%); Приложения для разработчика СУБД - MariaDB (36.41%); NoSQL база данных - MongoDB (46.15%); Интегрированная среда разработчика - Eclipse (21.67%); Текстовый редактор - vim (30.47%); Язык программирования - Python (28.63%); Система управления контентом - Wordpress (57.06%); Web-фреймворк - Django (30.07%); Система управления исходными текстами - Git (71.33%);
