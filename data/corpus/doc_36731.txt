Canonical прекращает производство DVD-дисков для не LTS-релизов Ubuntu

Компания Canonical приняла решение прекратить производство и распространение DVD-дисков для промежуточных выпусков Ubuntu, ограничившись распространением дисков с LTS-релизами. Данное изменение касается только физических носителей и вступит в силу начиная с выпуска Ubuntu 13.04, для которого не будут поставляться готовые диски. Формирование загружаемых с сайта образов для самостоятельного создания установочных DVD или USB Flash будет продолжено без изменений. Напомним, что в 2011 году компания Canonical закрыла сервис прямой безвозмездной рассылки дисков и делегировала функцию распространения дисков региональным группам пользователей Ubuntu, которым отправлялись большие партии дисков для дальнейшего вручения всем желающим. Единичные заказы дисков принимались через магазин shop.ubuntu.com по символической цене.
