Релиз системы управления web-контентом WordPress 3.3

Доступен релиз системы управления web-контентом WordPress 3.3, написанной на языке PHP и ориентированной на создание блогов. По сравнению с прошлым выпуском внесено 582 исправления. Отмечается, что с момента выпуска версии 3.0 с сайта проекта было загружено около 65 млн копий WordPress. В версии 3.3 продолжена работа по усовершенствованию системы навигации, повышению удобства использования, расширению средств импорта и загрузки. Из добавленных в новой версии улучшений можно отметить: Для пользователей: Реализован новый интерфейс загрузки медиа-данных, работающий в режиме drag-and-drop. Благодаря задействованию системы автоматического определения типа загружаемого контента для загрузки теперь достаточно одной кнопки; Всплывающее меню, позволяющее через один клик получить доступ к любому экрану; В Dashboard добавлен новый тулбар, совмещающий панель администратора (Admin Bar) и заголовок администратора (Admin Header); Расширены средства для организации совместного редактирования контента; Новая система импорта контента - Tumblr Importer; Проведена работа по повышению удобства работы в WordPress новичков, ранее не сталкивающихся с данной CMS: в секцию "About" добавлен отчёт о новшествах, представленных в последнем обновлении; переработана вкладка помощи; подготовлено информационное сообщение, выводимое для новых инсталляций; организован вывод подсказки по использованию недавно добавленных возможностей; Организация элементов в Dashboard оптимизирована для работы с использованием сенсорных экранов планшетных ПК; Для разработчиков: Новый редакторский API (Editor API); Обновление версий интегрированных в систему сторонних компонентов: jQuery 1.7.1, jQuery UI 1.8.16, Plupload 1.5.1.1, quicktags.js (редактор HTML), TinyMCE 3.4.5; Новый Screen API с улучшенной системой хуков, позволяющей добавлять дополнительные элементы в раздел помощи и адаптировать вывод в зависимости от контекста; Улучшенный API для работы с метаданными.
