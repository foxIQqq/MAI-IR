Выпуск программы для профессиональной обработки фотографий Darktable 3.0

После года активной разработки доступен релиз программы для организации и обработки цифровых фотографий Darktable 3.0. Darktable выступает в роли свободной альтернативы Adobe Lightroom и специализируется на недеструктивной работе с raw-изображениями. Darktable предоставляет большую подборку модулей для выполнения всевозможных операций по обработке фотографий, позволяет вести базу исходных фотографий, осуществлять наглядную навигацию по имеющимся снимкам и при необходимости выполнять операции корректировки искажений и улучшения качества, сохраняя при этом исходный снимок и всю историю операций с ним. Код проекта распространяется под лицензией GPLv3. Бинарные сборки подготовлены для Windows и macOS, а для Linux ожидаются в ближайшее время. Главные изменения: Полная переработка интерфейса и переход на GTK/CSS. Все элементы интерфейса теперь могут контролироваться при помощи CSS-тем. Подготовлена серия тем, оптимизированных для работы на мониторах с низким и высоким разрешением: darktable, darktable-elegant-darker, darktable-icons-darker, darktable-elegant-dark, darktable-elegant-grey, darktable-icons-dark, darktable-icons-grey. Требования к минимальной версии GTK подняты до 3.22. В истории изменений теперь отображаются ранее скрытые "системные" модули. Статус модулей в истории отображается иконкой. Поддержка переупорядочивания модулей в порядке применения к изображению (Ctrl+Shift+Перетащить). Поддержка назначения горячих клавиш для отдельных слайдеров. Например, управления экспокоррекцией. Это открывает возможность для быстрого редактирования с применением специализированных пультов. Поддержка операций undo/redo в режиме светового стола (lighttable) для меток, цветовых меток, рейтингов, метаданных, истории редактирования и применяемых стилей. Поддержка растровых масок (специальный тип параметрической маски). Переработана лента снимков и режимы гистограммы. Добавлен режим сохранения цветов в модуль "базовая кривая". Внимание! Этот режим включен по умолчанию (в режиме "Светлота") и может заметно изменять вид вновь импортированных файлов в сравнении с файлами JPEG, созданными камерой. Новые версии модулей "плёночная тональная кривая" и "эквалайзер тонов". Модули предоставляют мощные средства работы с изображениями и могут полностью заменить модули "базовая кривая", "тени и света" и "отображение тонов". Интерфейс модулей весьма непрост, поэтому с логикой работы проще познакомиться на реальных примерах из видео автора. Переработан модуль подавления шума по профилю. Добавлена поддержка профилей новых камер. Новый модуль "3D таблицы поиска цветов" с поддержкой форматов PNG Hald-CLUT и Cube. Наиболее популярный свободный набор СLUTов можно загрузить по ссылке, а с подробностями работы можно ознакомиться здесь. Новый модуль "основные настройки", позволяющий быстро настроить чёрную, белую и серую точки, изменить насыщенность и автоматически рассчитать экспозицию снимка. Новые модули "уровни RGB" и "тональная кривая RGB", поддерживающие работу с отдельными каналами в пространстве RGB в дополнение к уже существовавшим модулям, работающим в пространстве Lab. Инструмент "цветовая пипетка" в модулях смешивания, тональной кривой, цветовых зон и свечения, поддерживающий выборку среднего значения по выделенному участку (Ctrl+Щелчок по иконке пипетки). Поддержка быстрого поиска модулей по имени. Добавлен режим выбраковки изображений (попарное сравнение). Добавлен диалог настройки экспортируемых метаданных, позволяющий управлять экспортом данных Exif, меток, их иерархией и данных геотеггинга. Произведена миграция с POSIX threads на OpenMP. Произведены множественные оптимизации для SSE и OpenCL. Добавлена поддержка более 30 новых камер. Поддержка нового Google Photo API с возможностью создания альбомов напрямую из darktable (на данный момент не работает из-за блокировки со стороны Google). Значительно переработанное руководство пользователя будет опубликовано в ближайшее время.
