Бета-выпуск Opera 25 для Linux с возвращением поддержки закладок

Представлен бета-выпуск web-браузера Opera 25, который ознаменовал собой начало бета-тестирования поддержки платформы Linux в новой реинкарнации Opera Next, построенной на браузерном движке Chromium. Начиная с Opera 15 поддержка Linux была прекращена и вернулась спустя год в предварительном тестовом выпуске Opera 24 (релиз Opera 24 не был сформирован для Linux). В настоящее время сформированы только 64-разрядные сборки для Ubuntu Linux, рассчитанные на работу в окружениях Unity и GNOME Shell, а также неофициальные пакет для ArchLinux, ebuild для Gentoo и spec для Fedora. Выпуск Opera 25 также примечателен возвращением поддержки классической системы закладок, допускающей группировку ссылок по разделам, вместо которой в Opera 15 был предложен адаптивный интерфейс поиска избранных сайтов. Кроме того, в новом выпуске появился встроенный PDF-просмотрщик, переработана стартовая страница (в Speed Dial вместо скриншотов теперь выводятся пиктограммы), добавлена поддержка трансляции событий Web notifications API в системные уведомления, реализована поддержка форматов H.264 и MP3 (в Linux через FFmpeg).
