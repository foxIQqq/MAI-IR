Вышла бета-версия Mandriva Linux 2010

22 августа вышла первая бета-версия дистрибутива Mandriva Linux 2010. Нововведения: Splashy заменён на Plymouth, что позволило организовать графическую загрузку без мерцания и лишних переключений видеорежимов; Полностью переписан инструмент Netprofile, предназначенный для настройки профилей сетевых соединений; Tomoyo-gui - первая,написанная сотрудниками Mandriva, графическая оболочка для системы безопасности Tomoyo, появившейся в Linux ядре 2.6.30 и позволяющая привязать политику безопасности процесса к файловому пути, способом похожим с AppArmor. KDE 4.3, GNOME 2.27.5, X server 1.6.2, Virtualbox 3, Xfce 4.6.1; Linux ядро 2.6.31-rc6. В состав репозиторий contrib включен Python 3, но из соображений совместимости по умолчанию по прежнему присутствует Python 2.6 Подробный обзор новшеств Mandriva 2010 можно прочитать в анонсе альфа версии.
