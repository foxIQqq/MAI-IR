Новая версия web-фреймворка Pyramid 1.3

Увидела свет стабильная версия web-фреймворка Pyramid 1.3, написанного на языке Python и основанного в результате объединения проектов repoze.bfg и Pylons. Основные изменения в новом релизе: Обеспечение совместимости с Python 3 и прекращение поддержки Python 2.5 (поддержка Python 2.6 и 2.7 сохранена). Основные плагины Pyramid уже совместимы с Python 3, среди них отмечаются: pyramid_debugtoolbar, pyramid_jinja2, pyramid_exclog, pyramid_tm, pyramid_mailer, pyramid_handlers и другие. Добавлен модуль pyramid.compat, помогающий проверять совместимость компонентов pyramid с текущей версией интерпретатора Python. Следить и участвовать в процессе миграции на Python 3 можно здесь; Добавлен анализирующий модуль Introspection, помогающий при отладке и показывающий различные аспекты в конфигурации; Добавлен декоратор @view_defaults; Добавлены новые view-обработчики Not Found и Forbidden; Множество незначительных изменений. Дополнительно можно отметить о появлении с 27 февраля 2012 года в Google App Engine (GAE) поддержки версии Python 2.7.
