Выпуск дистрибутива для создания сетевых хранилищ FreeNAS 9.10

Спустя более года с момента прошлого выпуска сформирован релиз FreeNAS 9.10, дистрибутива для быстрого развёртывания сетевого хранилища (NAS, Network-Attached Storage). Дистрибутив основан на кодовой базе FreeBSD, отличается интегрированной поддержкой ZFS. Управление работой FreeNAS осуществляется через веб-интерфейс, построенный с использованием Python-фреймворка Django. Для организации доступа к хранилищу поддерживается FTP, NFS, Samba, AFP, rsync и iSCSI, для повышения надежности хранения может применяться программный RAID (0,1,5), для авторизации клиентов реализована поддержка LDAP/Active Directory. Установочный iso-образ (405 Мб) подготовлен только для архитектуры x86_64. Выпуск позиционируется как переходное звено между FreeNAS 9.3 и ожидаемым через несколько месяцев выпуском FreeNAS 10. Системное окружение FreeNAS 9.10 и jails-образов основано на FreeBSD 10.3-RC3 и портах FreeBSD 2016Q1, но пользовательский интерфейс и специфичные для дистрибутива компоненты перенесены из FreeNAS 9.3. Кроме расширения поддержки оборудования и обновления версий программ, из новшеств отмечается экспериментальная поддержка запуска виртуальных машин при помощи гипервизора bhyve. Для управления виртуальными машинами в состав включена утилита iohyve.
