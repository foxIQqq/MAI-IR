Вышел Calculate Linux 13.11

Анонсирован выход релиза Calculate Linux 13.11. Доступны для загрузки следующие редакции дистрибутива: Calculate Directory Server (CDS), Calculate Linux Desktop с рабочим столом KDE (CLD) и XFCE (CLDX), Calculate Media Center (CMC), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS). Основные изменения: Улучшена работа компьютера в домене ActiveDirectory: Устранено подвисание графического интерфейса при отключении от сети доменного компьютера. Исправлено восстановление подключения к сети с правами пользователя при продолжительной недоступности LDAP. Добавлено ожидание LDAP сервера при разблокировке сеанса доменного пользователя. Ускорена работа приложений за счет кэширования обращений к LDAP. Уменьшено время ожидания ответа от LDAP, если он не доступен. Добавлено автоматическое восстановление сетевых дисков при выходе из спящего и ждущего режимов. Исправления ошибок: Исправлена ошибка, приводящая к повторному запросу пароля при разблокировании сеанса. Исправлена установка на системах с UEFI. Добавлен параметр '--swap-size' для указания размера раздела подкачки при авторазметке. Исправлена установка с использованием PXE. Исправлен автовход в домен для livecd. Исправлена ошибка, иногда возникающая во время первой загрузки системы. Исправлены настройки шрифтов для корректного отображения сайтов в Firefox. Устранён запрет на изменение пользователем параметров сети и часового пояса в CLD. Исправлены спящий и ждущий режимы в CLD. Исправлено определение бездействия пользователя в Pidgin. Исправлена ошибка установки Calculate Directory Server и Calculate Scratch Server на машины с Intel видеокартами. Устранено удаление профилей пользователей Windows 7.
