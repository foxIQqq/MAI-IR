Доступны релизы LiveCD дистрибутивов SimplyMEPIS 8.0 и Foresight 2.1.0

Анонсирован выход популярного десктоп-ориентированного дистрибутива SimplyMEPIS 8.0, основанного на пакетной базе Debian Lenny и Debian Testing. В качестве графической оболочки используется KDE 3.5.10. В новом релизе по умолчанию включена поддержка IPv6 и DNSSEC. Дополнительно подготовлен, основанный на KVM и libvirt, комплект пакетов для запуска виртуальных окружений. Дистрибутив доступен в виде LiveCD с возможностью установки на диск для 32-разрядных и 64-разрядных x86 систем, размер дискового образа 680Мб. Из присутствующих в базовой поставке приложений можно отметить: Linux ядро 2.6.27.18, Xorg 7.4, OpenOffice 3.0, Firefox 3.0.6, Sun JavaSE 6u12, Amarok 1.4.7, mplayer 1.0.rc2, проприетарные драйверы для видеокарт NVIDIA и ATI, подборка дополнительных драйверов для беспроводных карт, шрифты и мультимедиа кодеки. Одновременно выпущен релиз Linux дистрибутива Foresight 2.1.0, изначально разработанного для демонстрации последних достижений десктоп окружения Gnome (2.24.1), без необходимости установки системы на диск, но в позднее превратившийся в дистрибутив общего назначения. Для установки доступны два LiveDVD образа размером 1.4 Гб - для архитектуры x86 и x86_64. В новой версии добавлена поддержка автоматической конфигурации подключаемых принтеров, осуществлен переход на Python 2.6 и Linux ядро 2.6.27.15, переработана система управления пакетами (в Conary по умолчанию включен режим предварительной загрузки пакетов (downloadFirst) и возможность исключения установки "dev" пакетов для разработчиков (excludeTroves));
