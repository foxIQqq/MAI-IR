Релиз набора образовательных приложений TQ 1.6 и программы для проведения тестирования Testy 1.0

Вышла новая версия пакета TQ и первая стабильная версия программы Testy. Пакет приложений TQ (TQ Suite) представляет собой набор приложений для образования, который включает набор средств для организации и управления тестами, для ведения журналов и создания баз данных. Testy — написанное с использованием Qt приложение для тестирования, полностью совместимое с TQ, поддерживающее кроссплатформенный мультиязычный интерфейс, а также средства разметки текста в тестах. Для всех компонентов доступны сборки для Windows и Linux, исходный код распространяется под лицензией GNU LGPL. Основные изменения: Новая версия формата тестов TQ (revision 3), убрано ограничение на длину вопроса (Tests management, Testing); «Файлы ключей» больше не поддерживаются, поддержка была убрана из Tests management; Поддержка автоматического и ручного создания резервных копий (Base); Улучшенная поддержка сеансов (Testing); TQ может установлен на Linux-системы, не поддерживающие DEB-пакеты; Поддержка изменения количества вопросов в тесте (Tests management).
