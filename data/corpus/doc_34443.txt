ttycast - инструмент для вещания консольных сессий в Web

В рамках проекта ttycast подготовлен открытый инструментарий для организации трансляции в Web консольных сеансов работы в эмуляторе терминала. Система может быть использована для проведения online-демонстраций в режиме реального времени или для подготовки консольных скринкастов. Для записи сеанса используется стандартная утилита ttyrec (в терминале повторить записанное при помощи утилиты ttyplay). Клиентская часть написана на JavaScript и запускается в любом браузере (задействован код эмулятора терминала tty.js). Серверная часть базируется на использовании сервера Node.js.
