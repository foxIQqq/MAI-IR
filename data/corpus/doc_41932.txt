Релиз открытой биллинговой системы Ubilling 0.6.5

Доступен релиз открытой биллинговой системы Ubilling 0.6.5, выступающий надстройкой над Stargazer, свободной системы учёта и авторизации в локальных, домашних и офисных сетях. Код Ubilling написан на PHP и распространяется под лицензией GPLv2. Основные изменения в новом выпуске: Изменения в структуре БД, см дамп. alter.ini: новая опция SWITCH_AUTOCONFIG для автоматического (по нажатию кнопки) поднятия влана на порту коммутатора. alter.ini: новая опция SNMPSET_PATH, указывающая путь к исполняемому файлу snmpset alter.ini: новая опция SNMP_MODE, указывающая режим работы с SNMP alter.ini: новая опция SNMPWALK_BACKGROUND, включающая возможность фонового выполнения запросов. Модуль «Карта оборудования»: улучшено прослеживание трассы аплинков по карте. Модуль «Справочник тарифов»: теперь уточняет название тарифа перед редактированием или удалением. Модуль «Изменения тарифа»: улучшено логирование. Модуль «Свитчи»: коммутаторы с NP теперь подсвечиваются желтой лампочкой. Модуль «Свитчи»: опрос работоспособности коммутаторов теперь происходит с сортировкой по их местонахождению. Поддержка FreeRADIUS: несколько незначительных исправлений, добавлены кастомизированные представления для блокирования должников. Модуль «События»: добавлен селектор администратора для фильтрации событий. Модуль «Опрос при помощи SNMP»: мигрировал на SNMPHelper в виде уровня абстракции. Модуль «Мониторинг сигнала на Mikrotik»: косметические изменения, переход на на SNMPHelper. Модуль «Регистрация пользователей»: проведена чистка кода. Модуль «UHW»: чистка кода. Модуль «Комплексные услуги»: добавлена возможность сортировки отчетов перед печатью. Глобальный поиск: исправлена проблема с поиском по неполному списку логинов. Глобальный поиск: исправлена локализация. Кабинет пользователя: новая необязательная опция ONLINELEFT_CREDIT, включающая предложение кредита пользователям с задолженностью. УКВ: небольшая косметика диалога редактирования пользователя. УКВ: в профиле пользователя появилась возможность заполнять метки кабеля. УКВ: в отчетах по должникам и антидолжникам, теперь показываются метки кабеля. Область уведомлений (DarkVoid): исправлена ошибка, вызывавшая в некоторых ситуациях проблемы при обновлении кэша. Добавлен геттер GetExtNets, позволяющий получить закрепленные за пользователями расширенные пулы адресов. Remote API: добавлено новое действие uhwbrutecleanup, очищающее реестр попыток перебора UHW.
