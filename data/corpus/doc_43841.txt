Новый выпуск Raspbian, дистрибутива для Raspberry Pi

Доступно обновление дистрибутива Raspbian, официально рекомендуемого для использования на Raspberry Pi и специально оптимизированного для данной платформы (сборка для ARMv6 с расширениями "hard float"). Размер образа для записи на SD-карту составляет 1.4 Гб. Для установки из репозиториев доступно около 35 тысяч пакетов. Ключевые новшества: В поставку включен экспериментальный OpenGL-драйвер vc4, позволяющий задейстовать средства аппаратного ускорения GPU VideoCore IV. По умолчанию драйвер отключен, для активации следует использовать конфигуратор raspi-config (раздел Advanced Options/GL Driver). Из-за требований к памяти, драйвер работает только в Raspberry Pi 2. В отличие от штатного драйвера OpenGL, который использует программную отрисовку, новый драйвер позволяет добиться существенного повышения производительности приложений и игр, использующих OpenGL. Например, со старым драйвером glxgears выдаёт 23 FPS, а с новым 60 FPS; Обновлены версии приложений, в том числе визуальных систем программирования Scratch 20160115 и Node-RED 12.5, среды программирования Sonic Pi 2.9, математического пакета Mathematica 10.3; Обновлены библиотеки, в том числе Java 8 update 65, RPi.GPIO 0.6.1 и WiringPi 2.31 (доступ к GPIO); Решены проблемы с поддержкой в апплете управлений громкости устройств с интерфейсом USB; В редакторе меню добавлена возможность создания новых меню (раньше можно было только редактировать существующие меню LXDE); В конфигураторах GUI Raspberry Pi Configuration и raspi-config добавлены опции для оверклокинга плат Raspberry Pi 1, Pi 2 и Pi Zero; Утилита ping теперь работает без необходимости её запуска под пользователем root.
