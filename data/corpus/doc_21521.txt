Тонкий клиент Linux в формате PCI-карты

Компания Igel выпустила реализацию платы с прошивкой на базе Linux (дистрибутив IGEL Linux) для создания тонких клиентов, основанных на устаревшем оборудовании. Igel TC Card продается по цене 99$, устанавливается в обычный PCI слот и подсоединяется к материнской плате в виде виртуального жесткого диска, через IDE интерфейc. Минимальные требования к клиентскому ПК: Pentium II 500MHz, 128 Мб ОЗУ, графическая карта с 8 Мб видеопамяти. Прошивка хранится на CompactFlash карте и включает в себя Linux окружение, поддерживающее следующие возможности: Соединение с серверами по протоколам XDMCP, Citrix ICA, RDP, X11R6, VDI (virtual desktop infrastructure) или NoMachine NX; Возможность работы с виртуальными окружениями Citrix XenDesktop и VMware VDM; Эмулятор терминала с поддержкой SSH - Ericom PowerTerm LTC и ПО Ericom PowerTerm WebConnect; ПО для организации печати ThinPrint; VoIP (SIP клиент); Средства для организации VPN с поддержкой Cisco VPN; Набор 802.11b/g и ISDN драйверов; Поддержка протоколов PPPoE, PPTP, DHCP, DNS, LPD, PXE1; Наличие в комплекте web-браузера Firefox, программы для просмотра PDF (Acrobat Reader); Universal Management Suite (UMS) - пакет для централизованного управления тонкими клиентами на базе Igel TC Card. Поддержка обновления прошивки через FTP и определения привязанных к локальным машинам конфигураций.
