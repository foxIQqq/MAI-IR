Компания VIA опубликовала DRM-код для карт Chrome 9, но не планирует открыть 3d драйвер

Компания VIA опубликовала под лицензией GPL новый код DRM (Direct Rendering Manager) модуля для интегрированных видеокарт серии Chrome 9 (CN896, K8M890CE/K8N890CE, P4M900 и VN896). Код состоит из приблизительно 5 тыс. строк и разбит на три патча для Linux ядра 2.6.31-rc3. Разработчики VIA надеются продвинуть данные патчи для включения в состав основной ветки Linux ядра 2.6.32. Дополнение: В будущем компания VIA не планирует создание открытого 3D драйвера и публикацию соответствующего кода для проекта Mesa.
