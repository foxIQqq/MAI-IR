Доступен прототип дизайна OLPC XO-3 и представлен ноутбук с Linux за 99$

Один из участников проекта One laptop Per Child, рассказал о концептуальном дизайне новой модели портативного компьютера для детей. OLPC XO-3 будет выполнен в виде планшета с цветным сенсорным экраном. При толщине устройства 6 мм., размер будет близок к листу формата A4. Первые поставки OLPC XO-3 ожидаются в 2012 году, устройство планируется распространять по цене 75$. Из интересных особенностей OLPC XO-3 отмечается наличие герметичного корпуса, лишенного каких бы то ни было отверстий. Для зарядки аккумуляторов планшет будет помещаться в специальную рамку, для передачи энергии будет использован беспроводной индуктивный механизм. Экран будет построен на основе гибридной технологии Pixel Qi, отличающейся поддержкой мультитач и пониженным потреблением энергии. Общее энергопотребление OLPC XO-3 планируется свести к 1 Вт. До момента выпуска модели XO-3 проект OLPC намерен выпустить в продажу две промежуточные модели: XO-1.5 на базе CPU 1GHz Via C7M поступит в продажу уже в январе следующего года по цене близкой к 200$. Далее в начале 2011 года ожидается выход модели XO-1.75, имеющей уже не форму субноутбука, а бесклавиатурного складного планшета с двумя экранами. XO-1.75 будет построен на базе CPU Marvell ARM и продаваться по цене $150. Программное обеспечение устройств будет основано на Linux дистрибутиве Fedora, вместо ранее используемого ПО XO Software. В качестве графического интерфейса пользователя по прежнему будет использоваться оболочка Sugar, но пользователи получат возможность включения и стандартного десктоп окружения на базе GNOME. Тем временем, пока проект OLPC строит планы, компания Cherrypal уже выпустила в продажу первый субноутбук "Africa" c ценой 99$ (в Китае подобная модель продается за 80$). Устройство поставляется с предустановленным Linux дистрибутивом Green Maraschino, базирующимся на Debian GNU/Linux. Модель "Africa" снабжена 7-дюймовым экраном (800 x 480), 400MHz ARM CPU (Samsung S3C2450), 256 Мб ОЗУ, 2 Гб NAND-flash, SD-слот, 10/100 Ethernet, Wifi 802.11b/g, 3 USB порта, аудиовыход, вес 1.2 кг.
