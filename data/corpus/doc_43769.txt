Релиз операционной системы Chrome OS 48

Компания Google представила релиз операционной системы Chrome OS 48, основанной на ядре Linux, открытых компонентах и web-браузере Chrome 48. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 48 доступна для всех актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache. Основные изменения в Chrome OS 48: Улучшены уведомления о завершении загрузки; В состав доступных по умолчанию приложений включен менеджер фотографий Google Photos; Изменена горячая клавиша для перехода на следующую раскладку клавиатуры в списке. Вместо Shift+Alt теперь предлагается комбинация Ctrl+Shift+Space, которая близка к комбинации Ctrl+Space, используемой для переключения на предыдущую раскладку; Обеспечен показ адреса IPv6 в области состояния (ранее приходилось смотреть в chrome://system или запускать из терминала "ifconfig"). Обновлён экран восстановления после сбоя; Добавлены обои рабочего стола с изображением объектов культурного наследия. Дополнительно можно отметить публикацию энтузиастами сборки Chromium OS for Raspberry Pi 2 0.3, оптимизированный для использования на одноплатных ПК Raspberry Pi 2. Размер системного образа для установки на microSD - 2 Гб.
