Вторая альфа-версия Mageia 2, форка дистрибутива Mandriva

Представлена вторая альфа-версия Linux-дистрибутива Mageia 2, в рамках которого независимым сообществом энтузиастов развивается форк проекта Mandriva. Для загрузки доступны 32- и 64-разрядные DVD-сборки (2.9 Гб), минимальный образ для установки по сети (40 Мб) и набор LiveCD на базе GNOME и KDE с различными наборами локализации. Релиз Mageia 2 намечен на 3 мая. По сравнению с первой альфа-версией добавлены следующие улучшения: Продолжена интеграция системного менеджера systemd, который будет использоваться по умолчанию. Возможность загрузки с классической системой инициализации будет сохранена для тех, кто хочет повременить с переходом на systemd; Десктоп-окружение KDE обновлено до версии 4.7.4, а GNOME до 3.3.3. Также обновлены компоненты оболочки LXDE, включая файловый менеджер Pcmanfm 0.9.10; В репозиторий включена обучающая оболочка Sugar 0.95; Новые версии СУБД: mysql 5.5.15, postgresql 8.4.9 и 9.0.5; ГИС: geos 3.3.1, gdal 1.8.1, umn mapserver 6.0.1, qgis 1.7.2, postgis 1.5.3; Серверное ПО: apache 2.2.21, postfix 2.8.6; Web: drupal 7.10, mediawiki 1.16.5; Средства разработки: gcc 4.6.2, python 2.7.2 и 3.2.2, perl 5.14.2, ruby 1.8.7, rails 2.3.14, php 5.3.8, smarty 2.6.26 и 3.1.5, java openjdk-1.6.0.0 и sun-1.6.0.26; IDE: eclipse 3.7.1, netbeans 7.0.1, anjuta 3.3.2.
