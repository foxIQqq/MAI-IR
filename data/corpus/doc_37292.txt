Компания Valve выпустила Source SDK 2013 с поддержкой Linux

Компания Valve представила новый выпуск инструментария для разработки модифицированных игр на базе движка Source, примечательный обеспечением поддержки платформ Linux и Mac OS X. SDK поддерживает создание модификаций игр (модов), запускаемых в Windows, OS X и Linux, а также создание игровых серверов, работающих в Windows и Linux. Код SDK размещён на GitHub и распространяется под новой лицензией SOURCE 1 SDK LICENSE, ограничивающей применение некоммерческим использованием, но разрешающим создание модов игр на базе движка Source для целей, не преследующих получение коммерческой выгоды. Кроме того, новая лицензия вкупе с размещением кода на GitHub упрощает процесс приёма изменений в SDK от сторонних авторов. Инструкцию по установке SDK можно найти здесь.
