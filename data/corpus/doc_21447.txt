Увидел свет релиз платформы Android 1.5

Компания Google выпустила релиз платформы Android 1.5 и новую версию комплекта для разработки приложений Android SDK 1.5r1. В Испании уже начались продажи новой модели телефона HTC Magic (G2), построенного на базе Android 1.5. На прилавки Великобритании G2 поступит 5 мая. Распространяться на территории Европы новый телефон будет оператором связи Vodafone. Для обладателей первой модели телефона T-Mobile G1, объем продаж на территории США которого уже достиг отметки в 1 миллион экземпляров, доступно обновление прошивки. Новшества Android 1.5: Поддержка встроенной экранной клавиатуры, API для разработки альтернативных экранных клавиатур; Встроенная поддержка записи видео со встроенной камеры и работы в роли диктофона. В комплект включен видеопроигрыватель с поддержкой форматов MPEG-4 и 3GP; Поддержка Bluetooth профилей A2DP и AVRCP (стерео звук), средства для связывания устройств; Изменения в web-браузере: улучшен процесс скроллинга; добавлена возможность поиска в содержимом текущего окна; возможность ручной смены кодировки; управляющий экран с табами для истории посещений, закладок и часто открываемых страниц; функция помещения в буфер обмена статических областей текста (ранее можно было только выделять области в полях ввода); обновление версии движка WebKit и JavaScript движка SquirrelFish; API для создания виджетов для базового экрана. Ранее были доступны только виджеты - часы и Google Search. В новой версии появились: календарь, виджеты для прослушивания музыки и просмотра фотографий; Live Folders - возможность размещения на базовом экране обновляемой из web-информации (например, RSS или обычная упрощенная страница с какими-то данными); Интеграция фреймворка для распознавания речи (работа через обращение к внешним интернет-сервисам); Поддержка автоматического переключения в портретный или ландшафтный режим при изменения позиции телефона (на основе данных от акселерометра); Осуществлен переход на Linux ядро 2.6.27. Интегрирована поддержка автоматической проверки и восстановления целостности ФС на SD-карте; Увеличение производительности инициализации web-камеры и GPS; Изменена структура SDK: В комплект включен код для поддержки сборки под несколько версий SDK (приложения можно собирать для SDK 1.1 и 1.5). Добавлена поддержка подключения внешних расширений, разработанных OEM производителями для увеличения функциональности своих аппаратов. В качестве примера одного из таких расширений, представлен Google Maps API, вынесенный из базового SDK. В эмулятор телефона добавлена поддержка Android Virtual Devices (AVD), позволяющего определить параметры устройств отличающихся от текущих моделей. AVD также позволяет выполнять одновременно эмуляцию нескольких устройств; Обновлена версия Eclipse плагина ADT (0.9.0), в котором появилась поддержка новых мастеров (Wizards) для создания проектов, нацеленных на определенную конфигурацию оборудования. Добавлена возможность генерации XML ресурсов, экспорта и подписи приложений для публикации. Улучшена поддержка средств тестирования JUnit, возможностей по профилированию кода и управления локализацией приложений.
