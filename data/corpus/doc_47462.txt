Выпуск операционной системы Chrome OS 62

Компания Google опубликовала релиз операционной системы Chrome OS 62, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 62. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 62 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 62: В управляемых через Active Directory устройствах Chrome OS обеспечена интеграция с системами единого входа (SSO) на базе Kerberos; В режиме для планшетов с сенсорными экранами появилась возможность масштабирования окон старых приложений; Добавлено определение проброса на web-интерфейс для организации подключения пользователей к сети (Captive portal); Переработан стиль и методы взаимодействия с системой вывода уведомлений; В файловом менеджере улучшены средства управления с сенсорных экранов и добавлен раздел с перечнем недавно изменённых файлов, включая новые загрузки, мультимедийные файлы и содержимое Google Drive; В режиме Enterprise Public Session обеспечена возможность запуска приложений в полноэкранном режиме; Добавлена поддержка входящих сетевых соединений в ARC++ (Android Runtime for Chrome); Устранены уязвимости KRACK, позволяющие вклиниться в беспроводное соединение на базе технологии WPA2; Добавлена настройка для инвертирования направления прокрутки; Добавлена возможность переименования внешних USB-накопителей по аналогии с переименованием файлов и каталогов через контекстное меню или горячую клавишу Crtl+Enter (включается через chrome://flags/#enable-external-drive-rename);
