Компания Microsoft представила полностью переработанный клиент Skype для Linux

После двух лет с момента прошлого выпуска компания Microsoft объявила о начале альфа-тестирования новой версии Skype для Linux. Новый Skype примечателен полной переработкой с переходом на использование web-технологий для интерфейса и протокола WebRTC для организации канала связи. Новый клиент Skype построен с использованием платформы electron, т.е. по сути является упакованной в самодостаточное приложение надстройкой над Chromium и Node.js, в которой выполняется расширенный web-клиент. Работа над новым клиентом пока не завершена и не все запланированные возможности реализованы. Например, по сравнению со старым нативным клиентом пока не готовы видеовызовы, звонки во внешние телефонные сети (PSTN), отправка SMS, изменение настроек устройства и добавление участников для исходящих звонков. История вызовов ограничена последними 30 днями. Наиболее значительным ограничением является поддержка только новой версии протокола, т.е. новый клиент может взаимодействовать со свежими выпусками Skype для Windows, macOS, iOS и Android, но не может совершать и принимать звонки от старого клиента для Linux 4.3.0.37. Из новых возможностей, которые отсутствуют в старом клиенте называется новый набор пиктограмм. В новом клиенте также полностью переработан интерфейс пользователя, имеется возможность организации совместного доступа к файлам, фотографиям и видео. Поддержку видеовызовов, включая групповые звонки, планируют добавить в одном из следующих выпусков (новые выпуски будут выходить раз в две недели). Старый и новый клиенты для Linux могут сосуществовать на одной системе, т.е. можно тестировать новый клиент продолжая использовать старый. Для загрузки подготовлены пакеты в формате deb и rpm, которые протестированы в Fedora 23, Debian 8.5, Ubuntu 16.04, Ubuntu GNOME 16.04, openSUSE 13.2 и 42.1. Работа проверена в GNOME, Unity, MATE, Cinnamon и KDE. Пакеты формируются только для 64-разрядных систем, но не исключено, что в будущем будут доступны и 32-разрядные сборки. Одновременно объявлено о полной поддержке выполнения web-клиента web.skype.com на платформе Chrome OS и в браузере Chrome с возможность совершения индивидуальных и групповых голосовых вызовов. Видеовызовы, а также звонки в мобильные и стационарные телефонные сети пока не поддерживаются. Отмечается, что Skype постепенно уходит от разработки индивидуальных клиентов для разных систем в пользу единого web-ориентированного решения, благодаря WebRTC, способного работать в браузерах без установки дополнительных плагинов.
