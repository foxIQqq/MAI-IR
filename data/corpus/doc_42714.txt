Доступен Brackets 1.4, открытый редактор для web-разработчиков

Компания Adobe представила релиз текстового редактора Brackets 1.4, предназначенного для редактирования JavaScript, HTML и CSS. Код Brackets написан с использованием web-технологий (JavaScript, HTML и CSS) и распространяется под лицензией MIT. Редактор оформлен в виде обособленного десктоп-приложения, для установки которого подготовлены deb-, dmg- и msi- пакеты для Linux, OS X и Windows. Brackets поддерживает режим Live-разработки, при котором редактируемый контент (JavaScript, HTML и CSS) по мере изменения сразу отображается в синхронизированном с редактором окне браузера - разработчик может изменять содержимое и сразу наблюдать к каким последствиям приводят данные изменения. Отладка также может выполняться синхронно с браузером, разработчик может установить точку останова или откатиться на шаг назад при просмотре результатов. Имеется встроенная поддержка препроцессоров LESS и SCSS. В интерфейсе применяется система контекстно-зависимых инструментов, появляющихся по мере необходимости в основном окне разработки. Для расширения возможностей редактора развивается система дополнений. Новый выпуск интересен главным образом обеспечением возможности работы с кириллицей - в используемый в редакторе шрифт Source Code Pro добавлены наборы кириллических и греческих символов. Среди других улучшений: Быстрый поиск в файлах. Результаты поиска выводятся и обновляются по мере ввода поискового запроса. Упрощённая система настроек редактирования. Все настройки снабжены подсказками, которые выводятся при редактировании конфигурационного файла brackets.json. Кроме того, при открытии конфигурационного файла (Debug-> Open Preferences file) во второй панели теперь отображается состояние настроек по умолчанию. Возможность выборочного отключения установленных дополнений. В Extension Manager добавлены соответствующие опции для включения, отключения и удаления каждого дополнения. Улучшено качество отрисовки текста на платформе OS X, благодаря включению межпиксельного сглаживания. Поведение отрисовки можно изменить через настройку fonts.fontSmoothing; Возможность изменения позиции открытых файлов в списках на панели, через нажатие CTRL+PageUp и CTRL+PageDown.
