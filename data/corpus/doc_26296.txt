Компания NVIDIA выпустила видеодрайвер с поддержкой OpenGL 4.0

Компания NVIDIA выпустила для платформ Linux, FreeBSD и Solaris драйвер 195.36.07.04 для своих видеокарт с полноценной поддержкой OpenGL 4.0 и GLSL 4.00. Драйвер не имеет статуса стабильного и доступен на странице для разработчиков - в случае обнаружения сбоев и ошибок в работе предлагается оформить уведомление через интерфейс nvdeveloper.nvidia.com. Новая версия OpenGL 4.0 совместима с видеоадаптерами начиная с nVidia Fermi. Для более старых карт, поддерживающих OpenGL 3, драйвер предоставляет возможность использовать OpenGL 3.3 и GLSL 3.30. В частности поддерживаются следующие карты: Для настольных компьютеров: Quadro FX 370, 570, 1700, 3700, 4600, 4700x2, 4800, 5600, 5800, Quadro VX200, Quadro CX. Видеочипсеты GeForce серии 8000 и выше; Geforce G100, GT120, 130, 220, GTS 150, GTS 250, GT310, 320, 330, 340, GeForce GTX 260 и выше, а также на всех продуктах из серии ION. Для ноутбуков: Quadro FX 360M, 370M, 570M, 770M, 1600M, 1700M, 2700M, 2800M, 3600M, 3700M, 3800M. Видеочипсеты GeForce серии 8000 и выше. Для видеокарт, совместимых со спецификациями OpenGL 2, драйвер предоставляет новые функции: ARB_texture_swizzle (также представлена в OpenGL 3.3) ARB_sampler_objects (также представлена в OpenGL 3.3) ARB_occlusion_query2 (также представлена в OpenGL 3.3) ARB_timer_query (также представлена в OpenGL 3.3) ARB_explicit_attrib_location (также представлена в OpenGL 3.3) Для видеокарт, совместимых со спецификациями OpenGL 4, драйвер предоставляет новые функции: ARB_texture_compression_bptc EXT_shader_image_load_store EXT_vertex_attrib_64bit NV_gpu_shader5 NV_gpu_program5 NV_tessellation_program5 NV_vertex_attrib_integer_64bit NV_shader_buffer_store Дополнительно, для видеочипсетов, совместимых со спецификациями OpenGL 3, предоставляется полный набор ARB-функций OpenGL 3.3. Для видеочипсетов, совместимыми со спецификациями OpenGL 4, предоставляются все возможности ARB и OpenGL 3.3, и OpenGL 4.0. На видеокартах nVidia функции, названные устаревшими, сохранятся и будут функционировать без потери в скорости. Загрузить 32-битную версию драйвера 195.36.07.04 для дистрибутивов Linux можно здесь, а 64-битную - здесь. Доступны также версии для операционных систем Windows (несколько вариантов для загрузки), Solaris, и FreeBSD (32-bit).
