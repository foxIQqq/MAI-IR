Релиз Linux-дистрибутива Mandriva 2010.1

Официально объявлено о выходе Linux дистрибутива Mandriva 2010.1 (Spring). Дистрибутив распространяется в сборках для архитектур i586 и x86-64, для свободной загрузки доступны две редакции: Free (только свободное ПО, без проприетарных драйверов и кодеков, 1 DVD) и One (установочный LiveCD, KDE и GNOME версии). Для коммерческого распространения подготовлен Mandriva Linux Powerpack, который можно загрузить через интернет-магазин Mandriva Store. В Mandriva 2010.1 можно отметить следующие новшества: Обновление версий программ Базовое пользовательское окружение построено с использованием KDE 4.4.3. Продолжение интеграции в десктоп-окружение KDE и пользовательские приложения системы семантических связей Nepomuk. Осуществлен переход Nepomuk на использование бэкенда для хранения данных Virtuoso, который позволяет ускорить работу с тегами и поддерживает индексацию хранимых данных; Произведена интеграция десктоп окружения GNOME 2.30.1, для предварительного знакомства подготовлены пакеты с поддержкой GNOME Shell (будущий рабочий стол GNOME 3), GNOME Color Manager (управление цветовыми профилями) и Gnome Activity Journal (интерфейс для наглядного отображения активности пользователя в системе); Системные приложения: Linux ядро 2.6.33.4, Glibc 2.11.1, ntfs-3g 2010.05.16, Spamassassin 3.3.1, Postfix 2.7.0, Zabbix 1.8.1, Python 2.6.5, OpenSSL 1.0.0; Пользовательские приложения: Firefox 3.6.6, OpenOffice.org 3.2, Qt 4.6.2, Amarok 2.3.1, KTorrent 4.0, Digikam 1.3.0, X.org 7.5, Mesa 7.8.1, снапшот свободного Flash плеера Gnash 0.8.7, Xine 1.2, Deluge 1.2.3, Transmission 1.92, OpenShot 1.1.3, Songbird 1.4.3; Включение в комплект тестовой версии web-браузера Chromium и reKonq (на базе QtWebKit) В дистрибутив интегрирована новая версия LDAP-сервера Mandriva Directory Server, из новшеств отмечаются средства для настройки пользовательских квот, возможность массового импорта пользователей, поддержка управления OpenSSH-ключами. В поставку включен пакет Zarafa Groupware для организации совместной работы, позиционируемый в роли открытой замены сервера Microsoft Exchange; В комплект включена большая порция обучающих и научных приложений, среди которых: математическая система Sagemath, пакет для численного моделирования Salome, генератор анатомических 3D-моделей на основе полученных с томографа данных - Invesalius, используемый в ЦЕРНе фреймворк для обработки данных Root, системы компьютерной алгебры Axiom и Singular, библиотека для организации распределенной на несколько машин параллельной трассировки лучей Tachyon, 3D-визуализатор химических структур Jmol, система для дискретных математических вычислений Gap, интерпретатор языка Common-Lisp ECL, библиотека для вычисления полиномов Polybori, система для расчетов алгебраической геометрии и коммутативной алгебры Macaulay2, библиотека связанных с линейной алгеброй функций Linalg; Новшества для пользователей Упрощение процесса организации шифрования домашней директории пользователя или дисковых разделов системы, настроить шифрование теперь можно в один клик; Средства для организации родительского контроля теперь не только позволяют ограничить детям доступ к нежелательным сетевым ресурсам, но и блокировать запуск определенных локальных приложений; Загрузка и установка В LiveCD-сборке "One" добавлена возможность установки без предварительной загрузки в Live-режиме, достаточно на начальном этапе загрузки выбрать опцию "Install Mandriva Linux 2010 Spring"; В инсталляторе, доступном на DVD-дисках "Free" переработан процесс создания шифрованных разделов, который теперь унифицирован с интерфейсом diskdrake. Добавлена возможность выбора типа ФС, который будет использован внутри зашифрованного раздела. В инсталлятор добавлена поддержка режима "KA" для клонирования уже установленной системы на другую машину. Изначально поддержка клонирования разрабатывалась для развертывания кластеров с типовой программной конфигурацией узлов на разнотипное оборудование (для колонирования на машины с разными сетевыми картами, дисками и другими устройствами в процессе установки автоматически создавался подходящий для каждого компьютера modprobe.conf); Для организации начального этапа загрузки вместо mkinitrd разработан dracut (образ ram-диска с возможностью динамического обновления содержимого), который не используется по умолчанию, но доступен для установки; В загрузчике Grub добавлена поддержка шифрованных паролей; Настройка и поддержка оборудования В состав дистрибутива добавлен пакет USB/IP для организации доступа к USB устройствам на других машинах по сети; В качестве базового свободного драйвера для видеокарт NVidia вместо драйвера nv теперь используется Nouveau, в котором реализована поддержка переключения видеорежимов на уровне ядра, 2D и 3D акселерация, поддержка RandR 1.2 для упрощения управления несколькими мониторами; В состав включен полнофункциональный плагин Network-Manager, позволяющий использовать через интерфейс NetworkManager все возможности, доступные в Mandriva Network Tools (drakx-net). В сетевые профили (netprofile) теперь можно добавить для конфигурирования собственные сетевые сервисы. В системе печати прекращено использование подсистемы HAL; Система и безопасность Расширены возможности GUI-интерфейса MSEC (Mandriva Security Center) для управления настройками, связанными с безопасностью: добавлен отчет со сводной информацией, реализована поддержка настройки ACL, добавлена возможность вызова и оценки результатов выполнения периодически выполняемых в системе тестов; При сборке пакетов на этапе связывания задействованы опции "-Wl,-O1", что позволяет уменьшить время запуска приложений связанных с большим числом библиотек; Значительно улучшена организация работы гостевых аккаунтов, позволяющих организовать одноразовый доступ к машине случайных людей, работа которых будет производиться в изолированной временной домашней директории, очищаемой при выходе гостя из системы. Например, для гостевых аккаунтов теперь используется специальный ограниченный shell, закрыт доступ к файлам других пользователей, заблокирован запуск административных системных приложений; Демон для ведения системных логов Sysklogd заменен на более продвинутую систему rsyslog, поддерживающую возможность хранения логов в СУБД, отправку email уведомлений при обнаружении в логах заданных масок, перенаправление логов на другой сервер с использованием TCP;
