Определено кодовое имя для следующего релиза Debian GNU/Linux - "wheezy"

Разработчики Debian утвердили кодовое имя для релиза Debian 7.0, выход которого ожидается в следующем году. По традиции имя заимствовано от одного из персонажей фильма "История игрушек": релиз Debian 7.0 будет назван "wheezy", в честь резиновой игрушки-пингвина с красной бабочкой. Что касается подготовки релиза Debian 6.0 (Squeeze), то несмотря на месячное нахождение в фазе заморозки, неисправленными остается 435 блокирующих релиз критических ошибок (месяц назад было 600). С настоящего момента, пакетная база Debian 6.0 переводится в фазу жесткой заморозки, при которой изменение пакетов допустимо только для исправления критических ошибок, для внесения правок, необходимых для достижения целей релиза, а также при корректировке документации и файлов с переводами.
