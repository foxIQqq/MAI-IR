Релиз дистрибутивов Parted Magic 2012_05_30 и Clonezilla Live 1.2.12-60

Доступны обновления двух дистрибутивов для системных администраторов: Parted Magic 2012_05_30 - компактный LiveCD/LiveUSB дистрибутив для восстановления системы после сбоя. Графический интерфейс основан на окружении LXDE. В минимальном консольном режиме дистрибутив может работать на машинах с 48 Мб ОЗУ, для работы в Live-режиме требуется 312 Мб ОЗУ. Размер iso-образа 196 Мб. В новом выпуске добавлены новые программы: Pyneighborhood 0.5.4, Shorewall 4.4.27, Shorewall6 4.4.27.3, Gptsync 0.14, и Udevil 0.2.4. Произведено обновление пакетов: SpaceFM 0.7.7, Coreutils 8.17, File 5.11, Lftp 4.3.6, Mbr 1.1.11, Openssh 6.0p1, Openssl 0.9.8x, Rdesktop 1.7.1, Sshfs-fuse 2.4, Unetbootin 575, Zerofree 1.0.2, Clamav 0.97.4, E2fsprogs 1.42.3, Fuse 2.9.0, Rsync 3.0.9, Cifs-utils 5.4, Keyutils 1.5.5, Krb5 1.7.1, Curl 7.25.0, Libidn 1.25, Openldap-client 2.4.31. Полностью переписана программа загрузки Java. Началась работа по созданию GUI-интерфейса для утилиты настройки межсетевого экрана Shorewall. Clonezilla Live 1.2.12-60 - дистрибутив для быстрого клонирования дисков (копируются только используемые блоки). Задачи выполняемые дистрибутивом сходны с проприетарным продуктом Norton Ghost. Дистрибутив основан на Debian GNU/Linux, в своей работе использует код проектов DRBL, Partition Image, ntfsclone, partclone, udpcast. Возможна загрузка с CD, USB Flash и по сети (PXE). Поддерживаются ФС: ext2, ext3, ext4, reiserfs, reiser4, xfs, jfs, FAT, NTFS, UFS, HFS+ (Mac OS X). Поддерживается режим массового клонирования в multicast режиме, например, при тестировании, за 10 минут удалось провести клонирование исходного 5.6 Гб диска на 41 клиентскую машину. Размер iso-образа дистрибутива - 107 Мб. В новой версии проведена синхронизация с пакетной базой Debian Sid по состоянию на 25 мая. Обновлены версии компонентов: ядро Linux 3.2.18-1, drbl 1.12.9-1drbl, clonezilla 2.5.36-1drbl, Partclone 0.2.48, Gdisk 0.8.4-1.1drbl. В состав дистрибутива включён пакет vlan. При обнаружении пустого целевого диска в режиме восстановления разделов добавлена возможность перехода в режиме ввода команд для ручного создания таблицы разделов. Добавлена опция "autohostname" для выбора имени образа на основе имени хоста. Возвращена поддержка файловой системы vmfs5.
