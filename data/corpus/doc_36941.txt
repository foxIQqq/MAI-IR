Обновление программы для управления фотографиями digiKam 3.2.0

Доступна новая версия программы для управления коллекцией фотографий digiKam 3.2.0. По сравнению с прошлым выпуском отмечено исправление 46 ошибок. Во входящем в состав пакета наборе плагинов (Kipi-plugins) устранено 10 ошибок. Из новшеств отмечается добавление в интерфейс для работы с альбомом фотографий нового режима "Table-View" для отображения элементов в виде списка, напоминающего вывод в файловом менеджере. Для изменения порядка вывода допускается сортировка по произвольному свойству фотографий. Состав списка можно изменять, например, можно показать имя файла, изображение, метаданные и свойства digiKam. Другим новшеством является предоставление возможности фильтрации и поиска элементов на основе свойства Aspect-Ratio, определяющего пропорции изображения. Добавлен новый инструмент Auto-Crop, позволяющий выявлять наличие тёмной рамки и автоматически кадрировать по ней изображения. В плагин FacebookExport добавлена опция для создания приватного альбома. Добавлен новый плагин JAlbumExport для экспорта контента в серивс JAlbum.
