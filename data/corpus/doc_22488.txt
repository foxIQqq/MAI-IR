Релиз музыкального проигрывателя Audacious 2.1.0

Вышел новый стабильный релиз музыкального проигрывателя Audacious - 2.1.0, ответвившегося в свое время от проекта Beep Media Player (BMP), являющегося форком классического плеера XMMS. В новой версии можно отметить следующие улучшения: Пользовательский интерфейс Newui оформлен в виде плагина 'gtkui' Добавлена возможность сортировки композиций по альбомам; Написанный с нуля новый плагин alsa-ng, для вывода звука через подсистему ALSA, в котором решены проблемы с высокой нагрузкой на CPU при работе с Intel-HDA устройствами; Для просмотра доступных интерфейсов можно использовать команду 'audacious -i list'; Новый фильтр bs2b для улучшения звучания в наушниках; Из состава удален плагин rootvis, имевший проблемы при работе на большинстве систем; Обновлены файлы с многоязычным переводом элементов интерфейса; Исправлено большое число ошибок.
