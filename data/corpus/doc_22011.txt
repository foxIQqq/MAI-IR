Red Hat представила JBoss Operations Network 2.2 и JBoss Enterprise Application Platform

Компания Red Hat выпустила релиз open source продукта JBoss Operations Network 2.2, представляющего собой модульную платформу для управления разработкой кросс-платформенных приложений, а также тестирования, развертывания и мониторинга. Представленная версия снабжена новым web-интерфейсом, основанным на использовании AJAX, имеет поддержку управления кластерами JBoss, реализацию упрощенных списков контроля доступа и систему автообновления агентов, управляющих удаленными машинами. Кроме того, Red Hat анонсировала новую стратегию развития продуктов линейки JBoss, в рамках которой предусмотрен выпуск единой унифицированной платформы JBoss Enterprise Application Platform, поддерживающей одновременно несколько подходов к созданию промышленных Java приложений (Spring, Seam, Struts, Google Web Toolkit и Java EE) и развертыванию решений на их основе.
