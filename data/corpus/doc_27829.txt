Вышел Element 1.4, Linux-дистрибутив для создания домашнего медиа-центра

Вышел релиз Linux-дистрибутива Element 1.4, предназначенного для организации работы домашнего кинотеатра или медиа-центра, собранного своими руками на основе обычного ПК, подключенного к HDTV-телевизору. Дистрибутив основан на Ubuntu Linux и включает в себя пользовательский интерфейс на базе XFCE с возможностью использования в качестве интерфейса медиа-центра XBMC. Дистрибутив оформлен в виде LiveCD, размер iso-образа составляет 626 Мб. В комплект входит подборка приложений для просмотра фотографий, прослушивания музыки, просмотра видео и навигации по сетевым медиа-ресурсам. Из программ можно отметить: Firefox 3.6.8, аудиопроигрыватель Decibel, клиент для мгновенного обмена сообщениями Pidgin, bittorrent-клиент Transmission, AllmyApps.com. В новой версии отмечены незначительные улучшения в работе апплетов E-Bar Editor, HDMI Audio Switch и Element Slim Settings.
