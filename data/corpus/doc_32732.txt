Релиз дистрибутива для создания сетевых хранилищ FreeNAS 8.0.3

Доступен релиз дистрибутива для быстрого развёртывания сетевого хранилища (NAS, Network-Attached Storage) - FreeNAS 8.0.3. Дистрибутив основан на FreeBSD 8 и полностью настраивается через веб-интерфейс, поддерживает FTP, NFS, Samba, AFP, rsync и iSCSI протоколы, программный RAID (0,1,5). Для авторизации может быть использован LDAP/Active Directory. Размер iso-образа составляет 95 Мб. Основные улучшения: Для разделов /etc и /var уменьшен размер блоков. Размер рамдиска для раздела /var увеличен с 74 до 160 Мб, что позволило решить проблемы с копированием больших наборов файлов через Samba; Около 7 Мб освобождено за счет чистки /var/db/pkg на этапе сборки iso-образа. Дополнительно на этапе сборки из iso удалены ненужные функции и пакеты; Обновлены версии программ: ataidle 2.6, nut 2.6.2, net-snmp 5.7.1, В web-интерфейс добавлена поддержка настройки sysctl и параметров ядра; Пакет Samba обновлён с 3.5.11 до версии 3.6.1. Решены проблемы с Kerberos и подключением к доменам Active Directory. Сachetool улучшен в плане предотвращения эффекта гонки на этапе загрузки между winbindd и ix-cache. Группа по умолчанию изменена с FREENAS на WORKGROUP; Добавлена возможность для включения и отключения поддержки Zeroconf/Bonjour (avahi) из web-интерфейса; Netatalk обновлён до версии 2.2.1. Добавлена возможность настройки максимального числа соединений. По умолчанию информация о сервисах передаётся только через avahi; В Rsync добавлена поддержка NFSv4 ACL; Решены проблемы с импортом NTFS-разделов.
