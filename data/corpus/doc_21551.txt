В видеоподсистему Gallium3D интегрирована поддержка OpenVG 1.0

В Gallium3D добавлено около 25 тыс. новых строк кода с реализацией поддержки OpenVG 1.0 - API для низкоуровневой прорисовки 2D векторной графики с задействованием аппаратной акселерации. Данное новшество позволит использовать API OpenVG для любого, основанного на Gallium3D, видеодрайвера. Gallium3D является реализацией новой архитектуры для разработки 3D драйверов, представляет собой попытку адаптировать OpenGL реализацию Mesa 3D к современным аппаратным архитектурам, создав единый промежуточный интерфейс для упрощенной разработки драйверов. В Gallium3D, для оптимизации работы, используется два модуля ядра - DRM2 (Direct Rendering Manager) для управления памятью, и DRI (Direct Rendering Interface) для взаимодействия с GPU.
