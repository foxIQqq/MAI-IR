Релиз медиаплеера GNOME Media Player 0.1

Представлен первый стабильный релиз медиаплеера GNOME Media Player 0.1, отличающегося минималистичным интерфейсом и возможностью использования движков VLC, Xine и GStreamer. По задумке разработчиков интерфейс GNOME Media Player сосредоточен только на проигрывании видео, без лишних настроек и усложнений. Особенности: Интерфейс базируется на технологиях GNOME/GTK; Для перемещения по потоку может быть использовано колесо мыши, для временной остановки - средняя кнопка, для перехода в полноэкранный режим - двойной клик; Пользовательский интерфейс не мешает во время проигрывания; Возможность быстрого переключения движков VLC, Xine и GStreamer; Поддержка списков воспроизведения с возможностью зацикливания; Управление громкостью производится только через стандартные системные механизмы, в самом плеере громкость не регулируется; В движках VLC и Xin поддерживается операция деинтерлейсинга (создание одного кадра из двух полукадров чересстрочного формата для дальнейшего вывода на экран с прогрессивной развёрткой). Дополнение: сразу вслед за версией 0.1 был выпущен корректирующий релиз 0.1.1 в котором было исправлено несколько неприятных ошибок. Для установки плеера в Ubuntu подготовлен специальный PPA-репозиторий (add-apt-repository ppa:gnome-media-player-development/development).
