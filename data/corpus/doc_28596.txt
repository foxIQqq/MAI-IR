Вышел Linux-дистрибутив для создания межсетевых экранов IPFire 2.7 Core 41

Вышел релиз специализированного Linux-дистрибутива IPFire 2.7 Core 41, предназначенного для создания межсетевых экранов. IPFire отличается предельно простым процессом установки и организацией настройки через интуитивно понятный web-интерфейс, изобилующий наглядными графиками. Размер установочного iso-образа составляет всего 62 Мб. Для установки дополнений в IPFire используется специальный пакетный менеджер Pakfire. Система модульная, кроме базовых функций пакетной фильтрации и управления трафиком для IPFire доступны модули с реализацией системы для предотвращения атак на базе Snort, для создания файлового сервера (Samba, FTP, NFS), почтового сервера (Cyrus-IMAPd, Postfix, Spamassassin, ClamAV и Openmailadmin) и сервера печати (CUPS), организации VoIP шлюза на базе Asterisk и Teamspeak, создания беспроводной точки доступа, организации потокового аудио и видео-сервера (MPFire, Videolan, Icecast, Gnump3d, VDR). Изменения: Устранены уязвимости в glibc и bzip2; Обновлены драйверы r8169, r8168 и r8101; В StrongSwan для загрузки списка отозванных сертификатов задействована утилита Curl; Добавлен PAM-модуль для OpenVPN (pam.so); Отключен по умолчанию встроенный в Dnsmasq DNS-сервер; Исправлены ошибки в web-интерфейсе; Добавлены примеры заполнения черных и белых списков на странице настройки прокси; Налажена возможность деактивации исходных правил пакетного фильтра.
