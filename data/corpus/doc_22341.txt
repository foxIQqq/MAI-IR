Проект по измерению производительности Linux десктопов

Организация Linux Solution Group (LiSoG) опубликовала спецификации с описанием методики тестирования производительности десктоп-приложений в Linux и продемонстрировала прототип нового тестового комплекта OSDtBench, предназначенного для сравнительной оценки различных настольных компьютеров и ноутбуков с точки зрения производительности десктоп-окружений на базе свободного ПО. Пакет OSDtBench выступает в роли открытой альтернативы системе BAPCo Sysmark, используемой на платформе Windows. Опубликованные спецификации (PDF, 2Мб, 96 стр.) описывают сценарии измерения производительности с точки зрения работы типичного десктоп-окружения, используемого для выполнения офисных операций, редактирования графики, выхода в Интернет, выполнения ежедневных задач по обработке данных. Для симулирования реальной рабочей нагрузки, когда пользователь одновременно работает с несколькими программами, тестовые задания выполняются с частичным перекрытием. Одной из основных целей создания данного тестового комплекта послужила задача упрощения процесса беспристрастного выбора поставщика Linux решений, при проведении тендеров гос. учреждениями Евросоюза, а также для формирования ряда эталонных решений на которые могут равняться производители оборудования и программного обеспечения.
