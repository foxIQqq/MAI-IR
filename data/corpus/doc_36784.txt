Началось тестирование MySQL 5.7

Компания Oracle выпустила первую тестовую версию новой значительной ветки СУБД MySQL 5.7. Выпуск рассчитан на начальное ознакомление с новыми возможностями, развиваемыми для новой ветки MySQL. Разработка функциональности ветки 5.7 ещё не завершена, поэтому не все возможности запланированные для финального релиза доступны в представленном тестовом выпуске. Среди изменений: Улучшение движка InnoDB: увеличена производительность работы с временными таблицами; добавлена возможность менять размер строк VARCHAR без перестроения таблицы (ALTER TABLE ...ALGORITHM=INPLACE, CHANGE COLUMN...VARCHAR()); добавлена поддержка типов данных для задания пространственных координат; введено отдельное пространство имён для таблиц с несжатыми данными; В ALTER TABLE добавлена поддержка выражения RENAME INDEX для переименования индексов; Расширены средства обработки состояния и диагностики работы СУБД; В клиент mysql добавлена опция "--syslog" позволяющая отправлять в лог все вводимые команды; Прекращена поддержка отложенного выполнения операций, таких как "INSERT DELAYED" и "REPLACE DELAYED".
