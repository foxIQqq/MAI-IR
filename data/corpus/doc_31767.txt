Доступен тестовый выпуск Oracle Solaris 11 EA

Представлен предварительный выпуск Oracle Solaris 11 EA (Early Adopter), в котором доступны для тестирования все возможности, запланированные для финальной версии Solaris 11, которая ожидается в конце года. Официально, загрузка открыта только для пользователей, имеющих аккаунт в Oracle PartnerNetwork (OPN). Сборки доступны для 64-разрядных систем x86 и SPARC, при этом поддерживаются только процессоры SPARC серии M и T, поддержка 32-битного ядра и старых моделей процессоров SPARC, включая UltraSparc IV+, прекращена. Список пакетов, добавленных и удаленных с момента выпуска Solaris 11 Express, можно посмотреть на данной странице. В частности, в поставку добавлены GCC 4.5, Perl 5.12, Python 2.7, JDK 1.6.0_26 и 1.7.0-b148, значительно обновлены драйверы, включены библиотеки Open MPI, VIM, libFuse, ISC DHCP, HAL, новые инструменты администрирования и конфигрурирования, COMSTAR, новая версия ZFS, новый интерфейс для организации удаленного доступа к дисковым разделам через NFS и SMB. Из поставки удалена поддержка основанного на Xen гипервизора xVM, virt-manager, libvirt, MySQL 5.0.x (5.1.x оставлен), драйверов для устаревшего оборудования (Sun GigaSwift Ethernet, LSI MegaRAID, QLogic Ultra3, Via 82C686 и т.п.), jEdit, VirtualGL, GTK 1.x, Perl 5.10, Python 2.4, Tor, LP (теперь используется CUPS), Webmin, Apache 1.3.x,
