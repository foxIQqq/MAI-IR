Доступен Roundcube Webmail 1.2.0 с поддержкой PGP

Представлен релиз web-ориентированного почтового клиента Roundcube Webmail 1.2.0, предоставляющего интерфейс, напоминающий классические десктоп-приложения для работы с электронной почтой. Roundcube написан на языке PHP и использует TinyMCE для WYSIWYG-редактирования писем в формате HTML, Googiespell для проверки орфографии и компоненты IlohaMail для работы с IMAP. В качестве СУБД может применяться MySQL, PostgreSQL, SQLite, MSSQL и Oracle. Исходные тексты распространяются под лицензией GPLv3. Основные возможности: Многоязычный интерфейс с поддержкой более 70 языков и интернационализированных доменов (".рф" и т.п.); Адресная книга c поддержкой групп, возможностью интеграции с LDAP и поиском адресата по мере набора в полях ввода; Поддержка составления сообщений с использованием разметки HTML; Гибкая система поиска сообщений и контактов; Шаблоны для типовых ответов; Возможность подключения плагинов; Управление сообщениями в режиме Drag&drop Особое внимание к соблюдению приватности пользователя; Режим нитевидной (Threaded) навигации по переписке; Встроенная система проверки орфографии; Средства для управления папками на сервере IMAP, поддержка совместно используемых почтовых папкок; Поддержка организации многопользовательской работы; Возможность разграничения доступа при помощи ACL; Встроенная система кэширования для быстрого доступа к почтовым ящикам; Функции импорта и экспорта переписки; Настройка внешнего вида через шаблоны. Главным новшеством версии 1.2.0 является интеграция поддержки шифрования писем с использованием PGP. При этом, PGP-шифрование может быть организовано как на стороне сервера (используется плагин Enigma и GnuPG), так и на стороне браузера клиента через дополнение Mailvelope. Кроме того, в новой версии: Обеспечена совместимость с PHP 7; Добавлена возможность перемещения вложений мышью (режим Drag-n-drop) из окна просмотра письма в окно составления нового сообщения. Реализована функция поиска сообщений с учётом заданного интервала времени их получения; Улучшена защита от атак по подбору паролей к учётным записям. В планах на будущие выпуски отмечены: поддержка S/MIME, интерфейс администратора, адаптация оформления для комфортной работы с мобильных устройств, предпросмотр вложений, расширенный поиск по всем папкам, виртуальные папки, горячие клавиши, режим просмотр переписки XCONVERSATIONS (как в gmail), интеграция модуля с календарём-планировщиком.
