Доступно восьмое обновление дистрибутива Debian GNU/Linux 4.0 (r8)

Представлен выход обновления для прошлой стабильной ветки дистрибутива Debian GNU/Linux - 4.0r8 (кодовое имя etch), представляющее собой сборку с включением всех текущих обновлений пакетов. В 32 пакетах устранены проблемы безопасности, в 4 пакетах отмечены незначительные изменения. Из изменений можно отметить: Обновлен модуль libweather-com-perl для получения данных о погоде через ресурс weather.com, в связи с изменением API сервиса; PostgreSQL в пакетах обновлен до версии 8.1.17; Обновлен пакет с PAM из-за обнаружения уязвимости; В пакете sleuthkit устранены проблемы с лицензией; В debian-installer и debian-archive-keyring добавлен новый ключ шифрования для проверки пакетов из архива; Устранены опасные уязвимости в пакетах: newsx, rt2400, rt2500, rt2570, vnc4, gnutls13, libpam-krb5, libpam-heimdal, phpmyadmin, ndiswrapper, znc, libsndfile, libtk-img, gs-gpl, glib2.0. Выпуск обновлений с исправлением проблем безопасности для Debian 4.0 (Etch) будет прекращен в феврале 2010 года, т.е. через год после релиза Debian 5.0 (Lenny).
