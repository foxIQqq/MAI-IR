Вышел Linuх-дистрибутив VortexBox 2.3

Вышел релиз Linuх-дистрибутива VortexBox 2.3, основанного на Fedora Linux и предназначенного для быстрого создания серверов для организации музыкального вещания в сети, сетевых хранилищ музыки или для превращения любого компьютера со звуковой картой в интернет-радио. Размер iso-образа дистрибутива составляет 38 Мб (минимальный образ для установки по сети). В качестве музыкального плеера по умолчанию в дистрибутиве используется программа собственной разработки VortexBox Player, представляющая собой надстройку над mpd (Music Player Daemon). Программа отличается поддержкой аппаратных интернет-плееров SqueezeBox и Sonos, может работать со SqueezeBox-совместимыми серверами, поддерживает вывод звука как на локальную звуковую карту, так и на аудиоустройства, подключённые через USB и S/PDIF порты. Управление работой системы производится через удобный web-интерфейс. Из возможностей можно отметить функцию автоматического копирования и перекодирования данных при вставке в привод аудио-CD, композиции сохраняются локально в форматах FLAC или MP3 и затем становятся доступными для потокового вещания. При необходимости для музыкальных файлов автоматически заполняются ID3-тэги и загружаются изображения обложек альбомов (через обращение к музыкальным онлайн-сервисам). Поддерживается возможность организации работы файлового хранилища музыки, доступного по протоколам SMB и NFS. Дистрибутив может выступать в качестве бэкенд-сервера для XBMC (XBOX Media Center), AppleTV и большого числа аппаратных интернет-плееров. Исходные тексты всех созданных в рамках проекта компонентов распространяются в рамках лицензии GPLv3 и доступны в виде пакетов для Fedora Linux. Из добавленных в новой версии улучшений можно отметить: Осуществлён переход на пакетную базу Fedora 20; Сервер SqueezeBox обновлен до версии 7.8.0; В качестве встроенного музыкального плеера задействован Squeezelite; Полная поддержка формата DSD с возможностью прямого вещания на устройства с поддержкой DSD или перекодирования в другие форматы; Переход к поставке iso-образа для установки по сети; Поставка новой версии PlexMedia Server для организации вещания на телевизоры и видеоприставки; В SqueezeBox Server добавлены плагины для поддержки Google Music и Shairport (Airplay).
