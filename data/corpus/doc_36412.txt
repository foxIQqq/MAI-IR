Корректирующий релиз системы виртуализации VirtualBox 4.2.10

Компания Oracle выпустила корректирующий релиз системы виртуализации VirtualBox 4.2.10, в котором отмечено 10 исправлений, направленных на увеличение стабильности и устранение регрессивных изменений. Среди изменений: Устранены проблемы со сборкой модулей для ядра Linux 3.9-rc0; В дополнениях для гостевых систем на базе Linux устранён крах VBoxService в случае недоступности библиотеки libdbus и исправлено несколько предупреждений в модулей ядра для организации совместного доступа к директориям; При работе GUI на платформе Mac OS X устранены проблемы с работой клавиатуры в XQuartz X server; Устранён крах при работе в многомониторной конфигурации (проявляется только на хостах Mac OS X); В дополнениях для Windows решены проблемы с обновлением MultiMedia Redirection (MMR); Исправлена недоработка, приводившая к потере настроек внутренней сети, если они не были активны; В системе хранения устранена несовместимость с VHD-образами Hyper-V.
