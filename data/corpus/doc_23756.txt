Вышел корректирующий релиз VirtualBox 3.0.8

Вышел релиз системы виртуализации VirtualBox 3.0.8 с исправлением 31 ошибки. Некоторые изменения: Устранена уязвимость, позволяющая выполнить команды с привилегиями суперпользователя; Исправлены проблемы с выполнением 64-разрядных гостевых окружений на 32-разрядных системах при использовании механизма аппаратной виртуализации VT-x; Для хост-систем на базе Linux изменен режим доступа к USB устройствам для нормальной работы в некоторых дистрибутивах; устранены проблемы при работе на Gentoo с hardened-патчами; Для хост-систем на базе Solaris устранен крах при использовании сети в режиме host-only или сетевого моста. Windows драйверы теперь поддерживают Vista 64-bit Home; В утилиту VBoxManage добавлена опция "controlvm vrdpport"; Добавлена поддержка виртуальных SCSI дисков размером более 2 Тб; Устранен потенциальный крах при отключении USB2 устройств; Исправлена проблема, мешающая работе IPSEC в гостевых системах на базе Linux
