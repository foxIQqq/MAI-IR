Выпуск свободной системы 3D-моделирования Blender 3.3

Организация Blender Foundation опубликовала выпуск свободного пакета 3D-моделирования Blender 3.3, подходящего для решения различных задач, связанных с 3D-моделированием, 3D-графикой, разработкой компьютерных игр, симуляцией, рендерингом, композитингом, трекингом движений, скульптурным моделированием, созданием анимации и монтажом видео. Код распространяется под лицензией GPL. Готовые сборки сформированы для Linux, Windows и macOS. Выпуск получил статус релиза с расширенным сроком поддержки (LTS) и будет поддерживаться до сентября 2024 года. Среди добавленных улучшений: Предложена полностью переработанная система моделирования волос, в которой задействован новый тип объекта - "Curves", пригодный для использования в режиме скульптинга и применения в геометрических нодах. Возможность использования старой системы генерации волос на основе частиц сохранена, созданные в разных системах волосы могут переноситься из одной системы в другую. Добавлен режим скульптурного моделирования кривых, который можно использовать для управления генерацией волос и причёски. Возможно использование кривых, деформированных при помощи геометрических нод, а также определение контрольных точек или контрольных кривых, настройка симметрии и создание фильтров в табличном редакторе. Реализованы следующие инструменты: Add/Delete, Density (Плотность), Comb (Расчёска), Snake Hook (Подвес), Pinch (Щипок), Puff (Раздувание), Smooth (Сглаживание) и Slide (Сдвиг). Для рендиренга могут использоваться движки EEVEE и Cycles. В реализации геометрических нод добавлены новые ноды для поиска путей по краям сетки, которые можно использовать для генерации лабиринтов, молний и растений - Shortest Edge Path (кратчайший путь между вершинами), Edge Paths to Selection (отбор рёбер, через которые проходит путь) и Edge Paths to Curves (генерация кривой, включающей все рёбра в пути). Расширена поддержка процедурной UV-разврёртки - для создания и изменения UV-карт с использованием геметрических нод предложены новые ноды UV Unwrap (UV-развёртка) и Pack UV Islands (Упаковать UV-острова). Значительно повышена производительность нод UV Sphere (в 3.6 раз быстрее при высоком разрешении), Curve (в 3-10 раз быстрее), Separate XYZ и Separate Color (на 20% быстрее). Расширены возможности системы двумерного рисования и анимации Grease Pencil, позволяющей создавать эскизы в 2D и затем использовать их в 3D-окружении как трёхмерные объекты (на основе нескольких плоских эскизов в разных ракурсах формируется 3D-модель). Добавлена поддержка распознавания силуэтов вокруг объектов и коллекций, назначения разных приоритетов при пересечении объектов и вычисления линий разделения света и тени. В редакторе экспозиционного листа (Dopesheet Editor) обеспечено отображение ключевых кадров Grease Pencil, которые можно использовать вместе с обычными объектами при анимировании и настройке свойств. Значительно (в 4-8 раз) сокращено время загрузки объектов арт-линий и повышена производительность (вычисление модификатора теперь производится в многопоточном режиме). В системе рендеринга Cycles реализована поддержка аппаратного ускорения вычислений с использованием интерфейса oneAPI, реализованного в GPU Intel Arc. На платформах Linux и Windows включена поддержка аппаратного ускорения на GPU и APU на базе архитектуры AMD Vega (Radeon VII, Radeon RX Vega, Radeon Pro WX 9100). Добавлены оптимизации для чипов Apple Silicon. Снижено потребление памяти при обработке объёмных данных в формате OpenVDB. Значительно переделан интерфейс переопределения библиотеки (Library Overrides), все переопределённые свойства в котором теперь отображаются в иерархическом представлении с показом доступных меток и пиктограмм. Добавлена возможность быстрого переключения между редактируемыми и не редактируемыми переопределениями. В контекстное меню интерфейса Outliner (Структура проекта) добавлено подменю для переопределения библиотеки. В системе трекинга движения предоставлена возможность создания и обновления изображения из пикселей за маркером плоскости, что можно использовать для создания неискажённой текстуры на основе имеющегося видеоматериала и проецирования этой текстуры обратно в видеометериал после редактирования во внешних приложениях. В нелинейном видеоредакторе (Video Sequencer) предложена новая система пересчёта для изменения скорости воспроизведения или подгонки к нужному FPS. В интерфейсе пользователя предоставлена возможность прикрепления сцены к рабочему пространству. Сделаны постоянно видимыми полосы прокрутки. Обеспечен показ манипулятора (Gizmo) при преобразованиях.
