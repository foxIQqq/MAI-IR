Выпуск дистрибутива Calculate Linux 17.12

Состоялся релиз отечественного дистрибутива Calculate Linux 17.12, который построен на основе Gentoo Linux, поддерживает непрерывный цикл выпуска обновлений и оптимизирован для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в нескольких редакциях, собранных для архитектур i686 и x86_64: Calculate Linux Desktop с рабочим столом KDE (CLD), MATE (CLDM) и Xfce (CLDX), Calculate Directory Server (CDS), Calculate Linux Scratch (CLS), Calculate Scratch Server (CSS) и редакция для контейнеров Calculate Container Scratch (CCS). Все версии дистрибутива распространяются в виде загрузочного Live-образа с возможностью установки на жесткий диcк или USB-накопитель. Calculate Linux совместим с портежами Gentoo, использует систему инициализации OpenRC и применяет Rolling-модель обновлений. Репозиторий насчитывает 11493 бинарных пакетов. Live USB включает как открытые, так и проприетарные видеодрайверы. Поддерживается мультизагрузка и модификация загрузочного образа при помощи утилит Calculate. Система поддерживает работу с доменом Calculate Directory Server с централизованной авторизацией в LDAP и хранением профилей пользователей на сервере. В состав входит специально разработанная для проекта Calculate подборка утилит для настройки, сборки и установки системы. Предоставляются средства для создания специализированных ISO-образов, адаптированных под задачи пользователя. Основные изменения: В инсталляторе добавлена поддержка установки системы на программные RAID (SoftRAID), исправлено поведение авторазметки, добавлена поддержка закрытых оверлеев; Проведена работа по оптимизации производительности: Запуск сервера утилит Calculate теперь по умолчанию выполняется через D-Bus, не требуя постоянной работы демона. Добавлена версия ядра Linux с патчем планировщика задач MuQSS, улучшающего отзывчивость приложений на рабочем столе. Добавлена версия ядра с патчем UKSM, оптимизирующим использование памяти. Для 32-разрядных систем добавлены сборки ядра с PAE. Оптимизированы настройки ядра для серверов; Выполнен переход на GCC 6.4 и ядро Linux 4.14. Вместо клиента Pidgin в состав CLDC, CLDM и CLDX для мгновенного обмена сообщениями включён Gajim с поддержкой плагинов "httpupload, "image" и "url image preview"; Дистрибутив для установки в LXC/LXD контейнере переименован в Calculate Container Scratch (CCS); Состав пакетов (iso-образы и torrent-ы): CLD (KDE, i686 - 1.8 G, x86_64 - 2.0 G): KDE Frameworks 5.41, KDE Plasma 5.10.5, KDE Applications 17.08.3, LibreOffice 5.4.2.2, Firefox 57.0.1; CLDC (Cinnamon, i686 - 1.6 G, x86_64 - 1.8 G): Cinnamon 3.4, LibreOffice 5.4.2.2, Firefox 57.0.1, Evolution 3.22.6, Gimp 2.8.22, Rhythmbox 3.4.1; CLDM (MATE, i686 - 1.7 G, x86_64 - 1.8 G): MATE 1.18, LibreOffice 5.4.2.2, Firefox 57.0.1, Claws Mail 3.15.1, Gimp 2.8.22, Clementine 1.3.1; CLDX (Xfce, i686 - 1.6 G, x86_64 - 1.7 G): Xfce 4.12, LibreOffice 5.4.2.2, Firefox 57.0.1, Claws Mail 3.15.0, Gimp 2.8.22, Clementine 1.3.1; CDS (Directory Server, i686 - 731 M, x86_64 - 780 M): OpenLDAP 2.4.44, Samba 4.5.10, Postfix 3.2.4, ProFTPD 1.3.5e, Bind 9.11.1_p3 CLS (Linux Scratch, i686 - 782 M, x86_64 - 900 M): Xorg-server 1.19.5, ядро Linux 4.14.9; CSS (Scratch Server, i686 - 509 M, x86_64 - 559 M): ядро Linux 4.14.9, Calculate Utilities 3.5.8.2.
