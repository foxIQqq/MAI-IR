Релиз утилиты cURL 7.21.0 с поддержкой LDAP и RTMP

Вышла новая версия удобной утилиты для организации выборки данных из web - cURL 7.21.0, предоставляющей возможность гибкого формирования запроса с заданием таких параметров, как cookie, user_agent, referrer и любых других заголовков. Кроме web-запросов, cURL поддерживает почтовые протоколы, telnet, ftp, rtsp и другие сетевые протоколы. Одновременно вышло обновление параллельно развиваемой библиотеки libcurl, предоставляющей API для задействования всех функций cURL в программах на таких языках, как Си, Perl, PHP, Python. Кроме исправления ошибок в новой версии представлены следующие улучшения: Представлена поддержка кода для осуществления запросов к LDAP; Добавлена возможность использования масок при выполнении запросов по FTP; Добавлена поддержка протокола RTMP ( Real Time Messaging Protocol), используемого для организации потокового вещания, например, на базе Flash Media Server; Добавлены опции --proto и -proto-redir; Новая опция скрипта configure: --enable-threaded-resolver; Улучшена поддержка возможностей libcurl по формированию запросов с использованием протокола TELNET; Добавлена поддержка PolarSSL; Добавлены настройки libcurl - CURLINFO_PRIMARY_PORT, CURLINFO_LOCAL_IP и CURLINFO_LOCAL_PORT.
