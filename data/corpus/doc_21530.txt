Увидел свет Linux дистрибутив Mandriva 2009.1 (Spring). Обзор новшеств

Объявлено о выходе релиза Linux дистрибутива Mandriva 2009.1, который доступен в сборках для архитектур i586 и x86-64 для свободной загрузки в редакциях Free (только свободное ПО, без проприетарных драйверов и кодеков, 1 DVD) и One (установочный LiveCD, KDE и GNOME версии). Для коммерческого распространения подготовлен Mandriva Linux Powerpack, который можно загрузить через интернет-магазин Mandriva Store. Основные новшества: Обновление версий программ, например: KDE 4.2.2, GNOME 2.26.1, OpenOffice.org 3.0.1 с Go-OO патчами, Firefox 3.0.8, Linux ядро 2.6.29, Xfce 4.6, X.Org server 1.6, Qt 4.5.0 + Qt Creator 1.0, Tcl/Tk 8.6, Python 2.6.1, VirtualBox 2.2, Wine 1.1.19, Songbird 1.1.2; Подготовлены гибридные iso-образы LiveCD ("One") сборки Mandriva, которые теперь подходят и для создания загрузочных USB FLash путем простого копирования содержимого ("dd if=mandriva-linux-one-2009.1-GNOME-int-cdrom-i586.iso of=/dev/sdX bs=8M"); Проведена интеграция технологии speedboot, позволяющей значительно сократить время загрузки дистрибутива - вначале грузятся компоненты экранного менеджера, позволяющего начать работать с системой, а затем в фоне загружаются дополнительные сервисы. Speedboot, если конфигурация системы позволяет (несовместимо с шифрованной ФС, программным RAID, сетевой аутентификацией), активируется автоматически. Улучшения инсталлятора: В качестве дополнительной опции добавлена возможность установки на ext4 раздел. В состав установочного диска включен модуль HDT (Hardware Detection Tool) из состава пакета Syslinux, позволяющий на раннем этапе загрузки (до начала загрузки ОС) сформировать полное описание состава текущего аппаратного обеспечения. Интегрирована поддержка Draksnapshot - инструмента для быстрого восстановления всей системы из резервной копии по состоянию на указанный момент времени. При разбиении диска стало возможным манипулирование разделами размером больше 1 Тб и просмотр содержимого ранее созданных разделов. В качестве легковесного окружения теперь используется LXDE вместо icewm. Пакеты с KDE 3 лишены официальной поддержки и перемещены в "contrib" репозиторий, где они будут доступны до 2010 года; Проведена дополнительная оптимизация для работы на субноутбуках с низким экранным разрешением, из коробки полностью поддерживается все оборудование Asus Eee PC, Acer Aspire One и MSI Wind; Осуществлен редизайн системы управления безопасностью MSEC (Mandriva Security Center): переработан пользовательский интерфейс, добавлена поддержка плагинов, разработан модуль для настройки AppArmor. MSEC используется для настройки пользователями различных параметров безопасности - от прав доступа и ограничения сетевой активности до лимитирования отдельных приложений; Улучшения десктоп оболочки: Новая визуальная тема и набор графических элементов. По умолчанию при вставке съемных устройств (web-камера, tv-тюнер) пиктограммы для доступа к ним не помещаются на десктоп (для включения нужно добавить DYNAMIC=yes в /etc/sysconfig/system). Файловый менеджер Nautilus по умолчанию переведен в режим "браузера". Заблокирована возможность перезапуска X-сервера через сочетание клавиш Ctrl+Alt+backspace; Реализована возможность установки обучающего программного окружения Sugar, используемого на ноутбуках OLPC; Улучшения GUI для настройки системы: В графическом интерфейсе для настройки сети улучшена поддержка 3G, уменьшено потребление памяти, добавлена возможность настройки глобальных сетевых параметров (включение поддержки IPv6, размер TCP-окна и т.п.), реализован режим ATM бриджа для PPPOE соединений, поднятых поверх USB xDSL модем. В Mandriva Personal Firewall (drakfirewall) появилась поддержка примитивов для управления доступом к NFS, syslog, dhcp и bacula. В Drakauth улучшена поддержка аутентификации через LDAP и winbind; В Diskdrake появилась возможность изменение метки дискового раздела без его переформатирования.
