Выпуск дистрибутива Mageia 5, форка Mandriva Linux

После года разработки доступен релиз Linux-дистрибутива Mageia 5, в рамках которого независимым сообществом энтузиастов развивается форк проекта Mandriva. Для загрузки доступны 32- и 64-разрядные DVD-сборки (3 Гб), универсальная DVD-сборка (1 Гб), минималистичный образ для установки по сети (40 Мб) и набор Live-сборок на базе GNOME и KDE с различными наборами локализации. Ключевые улучшения: Поддержка загрузки и установки на системы с UEFI. Сообщается о возможности установки Mageia 5 на большинство современных систем с UEFI, выпущенных за последние три года. В инталляторе, кроме возможности загрузки с UEFI, реализован новый алгоритм автоматического разбиения дисковых разделов (выделяется больше места под корневой раздел), добавлена возможность настройки RAID (dmraid и Intel soft raid), интегрирован загрузчик GRUB 2, добавлены средства отладки для URPMI, улучшен процесс автозагрузки драйверов, для обработки ввода задействован evdev, улучшена работа с разделами GPT. Обеспечена первичная поддержка файловой системы Btrfs, которая теперь может применяться к корню и разделу /boot. Задействована стандартная тема оформления Adwaita, вместо ранее предлагаемой темы Oxygen-gtk. Прекращена поддержка платформ Alpha, IA64, PPC и SPARC. Представлен предварительный выпуск ManaTools, нового интерфейса для запуска различных внутренних и внешних модулей настройки. Модули могут использоваться как часть интерфейса ManaTools или как обособленные программы настройки. ManaTools был создан как порт MCC (Mageia/Mandriva Control Center) для набора виджетов libYui, и способен использовать Qt, GTK и ncurses для отображения интерфейса каждого модуля. В состав включены пакеты для быстрого развёртывания LDAP-сервера 389-Directory Server и платформы для организации совместной работы Kolab. Добавлен пакет bumblebee для поддержки технологии NVIDIA Optimus. В состав включен просмотрщик изображений Phototonic, перекодировщики видео Transmageddon и HandBrake, NPAPI-обвязка freshplayerplugin для запуска Adobe Flash в Firefox из Chrome, система управления контейнерами Docker. Обновлены версии программ, в том числе LibreOffice 4.4.2.2, задействовано ядро Linux 3.19.8, systemd 217, RPM 4.12.0.1, GCC 4.9.2, X.org-сервер 1.16.4, тулкиты Qt 5.4.0, GTK+ 3.14.8. Общее число пакетов в официальных репозиториях доведено до 25 тысяч. Для формирования рабочего стола на выбор предложены пользовательские окружения KDE 4.14.3, KDE Plasma 5.1.2, GNOME 3.14, Cinnamon 2.4.5, MATE 1.8.0, Xfce 4.12, Enlightenment 18 и LXQt 0.9.0.
