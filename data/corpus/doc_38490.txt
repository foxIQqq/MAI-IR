Вышел дистрибутив Univention Corporate Server 3.2

Доступен релиз Univention Corporate Server 3.2, промышленного дистрибутива, основанного на Debian GNU/Linux 6 (Squeeze) и ориентированного на обеспечение поддержки корпоративной IT-инфраструктуры. Для установки задействован собственный инсталлятор, оптимизированный для быстрого развёртывания готовых сервисов. Для локального и централизованного администрирования используется специальный web-интерфейс. Для управления конфигурацией подготовлен инструментарий Univention Configuration Registry, позволяющий через наглядный интерфейс генерировать файлы конфигурации для разных сервисов и программ. В состав дистрибутива входит набор компонентов, позволяющих решать такие задачи, как ведение централизованного LDAP-каталога учетных записей пользователей, обеспечение единой системы аутентификации и авторизации, интеграция с MS Active Directory, автоматизация развёртывания серверов и рабочих станций, обеспечение работы тонких клиентов, мониторинг, создание серверов печати, резервное копирование, управление сетевыми службами (DHCP, DNS, прокси, почтовый сервер), обеспечение совместной работы сотрудников предприятия (groupware), управление работой виртуальных окружений на базе Xen и KVM. Размер загрузочного iso-образа 1.8 Гб. В новом выпуске: Упрощён и оптимизирован управляющий интерфейс UMC (Univention Management Console), расширено число мастеров для быстрой настройки и заведения пользователей. Добавлена возможность использования единой аутентификации для разных экземпляров UMC. Добавлены средства для настройки VLAN, сетевых мостов и агрегации сетевых интерфейсов; Добавлен провайдер идентификации, поддерживающий стандарт SAML (Security Assertion Markup Language), который может быть использован для организации единой точки аутентификации пользователей (single sign-on) с использованием сторонних сервисов, например, используя аккаунты в Salesforce; Расширены возможности центра установки приложений (Univention App Center), добавлены средства для распространения программ от сторонних поставщиков приложений; Реализации контроллера домена и сервиса Active Directory обновлена до Samba 4.1; Пакеты обновлены до состояния Debian 6.0.8; Ядро Linux обновлено до версии 3.10.15; Упрощен процесс настройки совместного доступа к файлам. Добавлена поддержка NFSv4; Добавлена возможность аутентификации в proxy с использованием Kerberos.
