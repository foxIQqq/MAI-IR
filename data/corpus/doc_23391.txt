Кандидат в релизы неофициальной сборки ALT Linux - "Просто линукс"

Денис Корявов представил кандидат в релизы проекта "Просто линукс", в рамках которого на базе платформы ALT Linux 5.0 сформирована легковесная сборка с пользовательским интерфейсом на основе XFce, нацеленная на дружественность к начинающему пользователю и содержащая сбалансированный обособленный набор приложений (не нужно что-то доустанавливать из сети). В комплекте: XFce 4.6.1, набор полезных дополнений для файлового менеджера thunar, OpenOffice.org, Brasero для записи дисков, словари StarDict, Evince для просмотра PDF, Firefox с плагинами для Adobe Flash, Sun Java и totem-gstreamer, Thunderbird, Pidgin, NetworkManager для управления сетью, видео-проигрыватель Totem, аудио-проигрыватель Sonata, GIMP, Inkscape, Gqview.
