В Chrome OS 36 представлена поддержка сенсорных экранов

Компания Google представила релиз операционной системы Chrome OS 36, основанной на ядре Linux, открытых компонентах и web-браузере Chrome. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 36 доступна для всех актуальных моделей Chromebook. Основные новшества: Оконный менеджер адаптирован для использования на устройствах с сенсорными экранами; На устройствах с сенсорными экранами обеспечена возможность использования управляющих жестов, таких как масштабирование щипком; Возможность корректировки времени локальных часов без наличия активного сетевого соединения (ранее часы синхронизировались только по NTP). В состав включена реализация экранной клавиатуры, которая может быть использована для ввода расширенных символов, которых нет на клавиатуре; Поддержка импорта списка существующих подконтрольных пользователей, которым разрешено открытие только конкретных сайтов, и возможность установки таким пользователям нового пароля через интерфейс chrome.com/manage; Кроме того, команда разработчиков Chromium OS опубликовала первый черновой макет нового интерфейса пользователя, который может появиться в будущих выпусках Chrome OS. Новый интерфейс развивается в рамках проекта Athena, в котором проводятся эксперименты с созданием для Chrome OS новых методов взаимодействия с пользователем. Для упрощения оценки нового интерфейса создано небольшое приложение, использующее графический стек Aura и позволяющее через запуск одного исполняемого файла получить прототип пользовательского окружения Athena. Интересной особенностью нового интерфейса является размещение окон в форме горизонтально перекрывающих друг друга карт, вместо традиционной системы вкладок. Если ранее центральное место Chrome OS занимал web-браузер, то в предложенном интерфейсе акцент смещён на вынос разных задач в отдельные окна.
