Релиз OpenIndiana oi_151a, продолжающего развитие OpenSolaris

Спустя год с момента первого выпуска представлен третий публичный релиз проекта OpenIndiana (oi_151a), в рамках которого независимым сообществом развивается полностью свободный дистрибутив, пришедший на смену бинарному дистрибутиву OpenSolaris, развитие которого было прекращено компанией Oracle в пользу тестовых выпусков Solaris 11 Express. Размер установочного iso-образа OpenIndiana oi_151a составляет 802 Мб. OpenIndiana предоставляет пользователю полностью работающее из коробки окружение, построенное на базе наработок проекта Illumos. Непосредственно разработка технологий OpenSolaris продолжается проектом Illumos, в котором развивается ядро, сетевой стек, файловые системы, драйверы, а также базовый набор пользовательских системных утилит и библиотек. Из других продуктов на базе Illumos можно отметить дистрибутив для создания сетевых хранилищ NexentaStor, отличающийся использованием утилит и библиотек GNU в сочетаниии с пакетным менеджером APT, и операционную систему SmartOS, ориентированную на облачные вычисления и виртуализацию. Дополнительно поддерживается система сборки illumos-live, позволяющая создавать собственные системные образы на кодовой базе Illumos. Из улучшений, добавленных по сравнению с выпуском oi_148, можно отметить: Поддержка системы виртуализации KVM (Kernel Virtual Machine), которая была портирована из Linux компанией Joyent для своей ОС SmartOS и впоследствии перенесена в Illumos (за исключением поддержки KVM branded zone, которая пока не добавлена). В качестве инструментария для управления виртуальными машинами используется пакет QEMU. Из аппаратных механизмов виртуализации пока поддерживается только Intel VT. Используя KVM в OpenIndiana можно обеспечить запуск нескольких полностью изолированных виртуальных машин, в которых можно запустить другие немодифицированные операционные системы, такие как Linux, *BSD и Windows. Каждой виртуальной машине выделяется собственное виртуальное оборудование, такое как сетевой адаптер, диск, видеокарта и т.п. Поддержка SCSI-команды UNMAP для файловой системы ZFS и фреймворка COMSTAR (Common Multiprotocol SCSI Target), который обеспечивает работу системы в режиме SCSI target, позволяющем организовать доступ к сетевым хранилищам данных посредством протоколов FC (Fibre Channel), FCoE (Fibre Channel over Ethernet), iSCSI или iSER. Используя SCSI-команду UNMAP, файловая система может информировать хранилище об освобождении блоков данных и возможности их возвращения в пул свободных блоков физического хранилища. Например, когда для хранения задействован SSD-накопитель, производится не только изменение мета-данных ФС, но и уведомление контроллера SSD о возможности перераспределения указанных блоков, что позволяет заметно повысить эффективность работы сборщика мусора; Поддержка новых опций для TCP-сокетов: TCP_KEEPCNT, TCP_KEEPIDLE и TCP_KEEPINTVL; Поддержка в загрузчике Grub секторов большого размера (4К); Поддержка в ZFS свойства "aclmode"; Улучшение средств для администрирования Solaris Zone. Возможность выполнения действий через утилиты svcadm и svcprop для сервисов, специфичных для указанных зон; Команда "svcs -L" для упрощения доступа из скриптов к файлам с логами работы сервисов; Реализован драйвер для RAID-контроллеров Areca ARC-1880; Обновленная утилита whois, использующая корректный список серверов для формирования запросов; Обновление БД terminfo с параметрами терминалов. Исправлены данные об 294 терминалах и устранено 1149 предупреждений о именах файлов длиннее 16 символов; При выполнении команды "iostat -E" теперь выводится серийный номер диска; В состав включены открытые аналоги многих системных утилит, локалей и реализация поддержки UTF-8 в libc. Дополнительные пакеты можно установить из репозитория пакетов Spec Files Extra, в котором можно найти такие приложения, как: Десктоп: Xfce 4.8, AbiWord 2.8.6, Arora 0.11.0, Nvidia 280.13, Qt 4.7.2, Sylpheed 3.1.1, Wine 1.1.44/1.3.28; Мультимедиа и графика: Blender 2.49.2, Inkscape 0.48.1, VLC 1.1.11, Mplayer 2.0.99, mpg123 1.13.3, Scribus 1.4.0, x264 20110827; Серверные приложения: Apache 2.2.16, BIND 9.8.1, ClamAV 0.97.1, Drupal 7.7, MySQL 5.1.37, PostgreSQL 9.0.4, OpenSSL 1.0.0d, Samba 3.5.5; Средства разработки: GCC 4.6.1, Java JDK 1.6.0_26, Perl 5.12.3, PHP 5.2.12, Python 3.2; Дополнительно силами сообщества поддерживаются пакеты с Firefox (6.0.2), KDE (4.6.1), OpenOffice (3.3.0), Thunderbird (6.0.2) и Flash Player (10.3.183.7).
