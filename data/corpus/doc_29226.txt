Обновление VirtualPBX - продукта для развертывания голосового сервиса на базе Asterisk

Подготовлена новая сборка продукта VirtualPBX, предназначенного для быстрого развертывания публичного Voice2.0-сервиса на базе ПО Asterisk. С обзором возможностей VirtualPBX можно познакомиться в тексте первого анонса. По сравнению с прошлой сборкой, можно отметить следующие изменения: Добавлена поддержка HD-кодека (g722); Добавлено изменение голоса (PITCH_SHIFT); Переход на asterisk 1.8; Добавлены звуковые эффекты (подложенная музыка, звуки) для исходящих звонков/очередей при нажатии определенной последовательности DTMF; Добавлена возможность записи логов в Syslog; Добавлена возможность разделить DB инстансы - запись.чтение; Добавлен биндинг интернациональных раскладок на имена/фамилии из списка Directory; Добавлены лимиты на количество подкастов для группы Добавлена изолированная парковка/подбор вызовов; Добавлен трансфер на внешние номера (*123 - IVR, 123 - внешний номер); Добавлен SIP-заголовок в исходящие звонки, через общий контекст, для аутентификации клиента; Добавлен radius accounting; Добавлена опция включить/выключить API в настройку группы; Добавлена проверка на зацикливание между внутренними номерами; Лимиты на количество внутренних номеров определенного типа для группы; AGI failover для FastAGI сервера; Добавлена поддержка глобальных внутренних номеров (доступ из любого уровня IVR); Добавлена поддержка MP3 Streaming VB; Добавлена поддержка отправки уведомлений чрез Twitter; Добавлена поддержка Basic auth для WebRequest & WebVars; Исправлены многочисленные ошибки.
