Релиз Wine 1.5.16

Вышел очередной экспериментальный релиз открытой реализации Win32 API - Wine 1.5.16. С момента выпуска версии 1.5.15 было закрыто 77 отчётов об ошибках. Основные изменения в новой версии: Обновлен встроенный пакет Mono для поддержки .Net приложений; Добавлена поддержка управлениями службами через подсистему WMI; Множество улучшений интерпретатора команд cmd.exe; Добавлены некоторые недостающие классы C++ для работы с потоками; Улучшено определение параметров центрального процессора; Обновлены переводы; Исправлено множество ошибок.
