Доступен Plasma Media Center 1.3 - медиацентр от проекта KDE

После семи месяцев разработки проект KDE представил новый выпуск Plasma Media Center 1.3 (PMC), унифицированного интерфейса для работы с мультимедиаконтентом на стационарных ПК, нетбуках, планшетах, телевизорах и других устройствах, на которых может быть запущено окружение KDE. PMC обладает типичными для медиацентров функциями и позволяет показывать изображения, проигрывать музыку и воспроизводить видео. Медиацентр базируется на технологиях Plasma и KDE, и использует QML для формирования интерфейса, адаптированного для различных классов устройств. Для расширения функциональности предоставляется API для создания плагинов. Поддерживается работа как с содержимыми локальных файловых систем, так и внешних онлайн-сервисов, таких, как YouTube, Flickr и Picasa. Управление воспроизведением и навигация по контенту могут осуществляться через клавиатуру/мышь, сенсорный экран или пульты дистанционного управления. Для унифицированного просмотра всего медиаконтента доступна возможность использование средств поиска на десктопе KDE (KDE Desktop Search). Ключевые новшества: Поддержка извлечения медиаколлекции из системы семантического поиска Baloo, которая пришла на смену Nepomuk в выпуске KDE 4.13. С практической стороны изменение привело к существенному ускорению загрузки медиабиблиотеки. Поддержка Nepomuk оставлена в качестве опции; Возможность взаимодействия со сторонними интерфейсами управления, поддерживающими стандарт MPRIS (Media Player Remote Interfacing Specification). Для управления Plasma Media Center теперь можно использовать такие компоненты как плазмоид Now Playing, отображающий проигрываемую в текущий момент композицию, или модуль для предпросмотра из панели задач; Экспериментальная поддержка простого сканера медиаконтента в файловой системе, который можно использовать в конфигурациях без Nepomuk и Baloo; Отображение дополнительной информации в режиме работы с музыкальной коллекций ("All Music"), такой как данные о музыканте, альбоме и длительности композиции; Улучшена совместимость со звуковым сервером GStreamer, устранён конфликт между пакетами gstreamer0.10 и gstreamer1.0, из-за которого в некоторых дистрибутивах наблюдались проблемы с воспроизведением; Реализован тестовый комплект для контроля качества базовых библиотек Plasma Media Center; Проведены оптимизации производительности и полировка интерфейса. Из планов на следующую версию: Реализация средств для локального поиска файлов; Экран для настройки параметров индексации, настройки интерфейса и поддержания списка игнорируемого контента (например, можно будет игнорировать мелкие картинки и короткие звуковые файлы); Вывод детальной информации о роликах YouTube; Замена боковой панели на полноэкранную; Поддержка просмотра телепрограмм через DVB; Средства звукозаписи; Создание редакции Plasma Media Center для рабочего стола Plasma 5.
