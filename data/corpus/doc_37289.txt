Компания NVIDIA начала публикацию видеодрайверов для Linux-систем на базе ARM

В дополнение к ранее поддерживаемым платформам x86 и x86_64 началась публикация сборок проприетарного драйвера NVIDIA для Linux-систем на базе архитектуры ARM. В настоящее время для загрузки уже доступна сертифицированная ARM-версия драйвера NVIDIA 319.32, полностью аналогичная представленному несколько дней назад выпуску для других архитектур. Компоненты драйвера собраны с использованием набора инструкций Thumb-2, модуль ядра и драйвер CUDA собраны с использованием набора инструкций ARMv7 (HardFP). Выпуск версии драйверов для ARM последовал после объявления о начале лицензирования технологий GPU Kepler для сторонних производителей SoC. Для собственных SoC Tegra компания NVIDIA использует отдельный набор драйверов.
