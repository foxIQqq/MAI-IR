Вышел новый дистрибутив Calculate Linux Scratch 9.8

Вышел дистрибутив Calculate Linux Scratch 9.8, доступный в двух редакциях: базовой (CLS) и с рабочим столом Gnome (CLSG) под 32- и 64-разрядные системы. Calculate Linux Scratch (CLS) - это установочный livecd диск, предназначенный для сборки собственного дистрибутива с необходимым программным обеспечением с возможностью записи его на LiveCD или USB-Flash. CLS имеет базовый набор компонентов, необходимых для создания livecd с поддержкой установки системы на жесткий диск. В дистрибутиве применен оригинальный подход для создания среды сборки новой системы, основанной на разделении уровней работы по слоям unionfs. Возможности: Дистрибутив включает только базовый набор приложений: Xorg, Firefox, драйвера, библиотеки, портежи и исходники ядра. Версия CLSG включает дополнительно сокращенную версию рабочего стола Gnome, gdm и wicd. Сборка приложений может осуществляться в отдельном unionfs-слое в интерактивном режиме. Собранная система пакуется в ISO образ, пригодный для записи на CD или USB-Flash, с поддержкой установки на жесткий диск. Исходный squashfs образ системы установленный на USB-Flash можно переписывать, обновляя программное обеспечение без использования жесткого диска. Используя профили установки можно настроить дистрибутив под любые задачи. CLS на 100% совместим с Gentoo. В качестве менеджера пакетов используется Portage.
