Релиз программы для записи оптических дисков K3B 2.0

После двух лет разработки вышла обновлённая версия K3B 2.0, программы для записи оптических дисков, которая использует возможности среды рабочего стола KDE. Новая версия реализует не просто очередной набор улучшений: в рамках перевода K3B на использование библиотеки Qt4 и технологий KDE4, большое число подсистем было переписано практически с нуля. В частности, большой объем работы был связан с переходом на новый KDE-фреймворк для взаимодействия с оборудованием и переработкой организации работы интерфейса пользователя. Внешний вид интерфейса K3B по своей сути не изменился и сохранил все элементы прошлой версии для KDE3, тем не менее интерфейс базируется на новом наборе виджетов и использует более современные пиктограммы. Значимые изменения K3B 2.0 по сравнению с версией 1.0.5 включают: Произведено объединение диалогов копирования, прожига и форматирования CD и DVD-носителей. Проекты по записи CD и DVD объединены, к ним добавлена поддержка записи BluRay дисков. Добавлена поддержка записи файлов больше чем 4GB при наличии mkisofs 2.01.01a32 и выше. Улучшен внешний вид панелей управления. Добавлена возможность ручного выбора скорости записи. Программа теперь позволяет дописывать данные к любой предыдущей сессии (раньше разрешалось дописывать только к самой последней). Убрана опция "Использовать существующий CD-Text" для копирования CD дисков, теперь при копировании данные параметры запрашиваются индивидуально. "Очистить проект" теперь удаляет только данные, оставляя нетронутыми опции проекта. Стало возможным проверять проект без перезагрузки носителя, если привод поддерживает эту возможность. Добавлена поддержка записи носителей Blu-ray. Добавлена поддержка записи Blu-ray и DVD с помощью cdrecord. Новая переменная %e при захвате аудио носителей подставляет расширение файла. Улучшена поддержка языков с записью справа налево (арабский и пр.).
