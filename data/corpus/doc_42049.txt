Cyanogen объявляет о стратегическом соглашении с Microsoft

Компании Cyanogen Inc и Microsoft объявили о партнёрстве с целью интеграции популярных сервисов Microsoft в прошивку Cyanogen OS, основанную на наработках CyanogenMod, альтернативной сборки платформы Android. По условиям соглашения Cyanogen будет интегрировать и распространять пользовательские мобильные приложения и сервисы Microsoft всех уровней, включая инструменты для создания контента, обмена сообщениями, утилиты и облачные сервисы. В рамках совместной работы, компания Microsoft также обеспечит бесшовную интеграцию своих решений и подготовит для Cyanogen новый класс мобильных сервисов. На данный момент планируется произвести интеграцию со следующими сервисами Microsoft: Bing, Skype, OneDrive, OneNote, Outlook и Microsoft Office. По опубликованной в январе неофициальной информации, Microsoft станет держателем акций компании и выступит одним из инвесторов, в $70-миллионном раунде инвестиций в Cyanogen Inc. В марте данная информация не подтвердилась, но было упомянуто, что вместо прямых инвестиций Microsoft выразил интерес к коммерческой сделке с Cyanogen Inc, результатом которой стала бы интеграция программ Microsoft в прошивку Cyanogen. Финансовая сторона соглашения не разглашается. Представители Cyanogen Inc обратили внимание на то, что компоненты с сервисами Microsoft не будут встроены в CyanogenMod (не путать с коммерческим продуктом Cyanogen OS) и не будут предустанавливаться по умолчанию. Сервисы Microsoft будут предложены в качестве отдельной опции к CyanogenMod, как это сейчас делается для сервисов Google, Amazon и Fdroid. Что касается коммерческого продукта Cyanogen OS, то в нём сервисы Microsoft будут предустановлены, но пользователь получит возможность удалить их при необходимости. Из встроенных в основную платформу доработок отмечается появление нового API, позволяющего использовать CyanogenMod в качестве платформы для разработчиков и дающего возможность использовать оригинальные новшества, которых пока нет в Android. Также объявлено о начале формирования ночных сборок CyanogenMod 12.1, основанных на Android 5.1. В настоящее время в сборки CyanogenMod 12 подготовлены для 31 модели устройств, для которых обеспечена полная поддержка оборудования. Наиболее интересным новшеством ветки CyanogenMod 12, специфичным для CyanogenMod, является отказ от поставки отдельного приложения для выполнения программ с правами root (SU App). Функции предоставления дополнительных привилегий теперь встроены непосредственно в менеджер управления приватностью (Privacy Guard), через который ранее осуществлялось ограничение и контроль доступа приложений к системным ресурсам.
