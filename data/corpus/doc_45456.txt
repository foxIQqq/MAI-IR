Доступен дистрибутив SUSE Linux Enterprise 12 SP2

Компания SUSE представила релиз промышленного дистрибутива SUSE Linux Enterprise 12 SP2. Пакеты SUSE 12 SP2 использованы в качестве основы в поддерживаемом сообществом дистрибутиве openSUSE Leap 42.2, который намечен на 16 ноября. На основе платформы SUSE Linux Enterprise также сформированы такие продукты, как SUSE Linux Enterprise Server, SUSE Linux Enterprise Desktop, SUSE Linux Enterprise High Availability Extension, SUSE Linux Enterprise Point of Service и SUSE Linux Enterprise Real Time Extension. Дистрибутив можно загрузить и использовать бесплатно, но доступ к получению обновлений и исправлений ограничен 60-дневным пробным периодом. Выпуск доступен в сборках для архитектур x86_64, ARM64, IBM POWER8 LE и IBM System z. Основные изменения: Ядро Linux обновлёно до версии 4.4, а системный менеджер systemd до версии 228; Рабочий стол GNOME обновлён до версии 3.20 (ранее поставлялся GNOME 3.10) с поддержкой Google Drive, новым оформлением системы уведомлений и доступностью интегрированной среды разработки GNOME Builder. Представлен новый режим обновления дистрибутива "skip service packs", позволяющий пропустить переход на промежуточные выпуски при обновлении со старого выпуска, т.е. можно с SUSE Linux Enterprise Server 12 сразу обновить систему до SLE 12 SP2, минуя SLE 12 SP1; Началось формирование сборок для архитектуры ARMv8-A (AArch64), отдельно подготовлена сборка для 64-разрядных устройств Raspberry Pi 3; Интеграция с репозиторием SUSE Package Hub, предоставляющим доступ к свежим версиям пакетов и модулей, поддерживаемых представителями сообщества для openSUSE Leap. Обеспечена возможность задействования фреймворка DPDK (Data Plane Development Kit) для значительного увеличения производительности программных сетей SDN (Software-Defined Networking) и виртуальных сетей NFV (Network Functions Virtualization), основанных на применении Open vSwitch. DPDK позволяет создавать высокопроизводительные сетевые приложения, напрямую обрабатывающие пакеты, минуя сетевой стек ядра; Улучшена производительность на системах с процессорами Intel Skylake. Добавлена поддержка архитектуры Intel Omni-Path, обеспечивающей скорость передачи данных между узлами кластера до 100 Гбит/c; Поддержка стандарта TPM (Trusted Platform Module) 2.0, определяющего средства для задействования криптопроцессоров; Представлена новая web-консоль управления кластером для обеспечения высокой доступности - Hawk 2 (HA Web Konsole); Поддержка постоянной памяти NVDIMM, сохраняющей данные после перезагрузки и выключения питания; В установщике добавлена по умолчанию проверка наличия обновлений и их применения; Задействован PID cgroup controller, позволяющий задать лимит на максимальное число процессов в группе, при превышении которого блокируется создание новых задач через вызовы fork() и clone(), что может применяться как защита от fork-бомб; В YaST добавлен новый инструмент User Logon Management (ранее Authentication Client), предоставляющий интерфейс для подключения к домену Active Directory и позволяющий настроить аутентификацию пользователей через AD, Kerberos или LDAP; В Btrfs добавлена поддержка резервирования пустых областей через вызов fallocate() и включена реализация операций Send/Receive; В состав включена утилита Devilspie 2, позволяющая привязывать действия к операциям с окнами; Из репозиториев SUSE удалён плагин Adobe Flash, который теперь следует устанавливать из репозитория Adobe; Удалён пакет Xemacs, вместо которого рекомендован GNU Emacs; Добавлен пакет virt-p2v, позволяющий преобразовать образы различных виртуальных машин в формат гостевой системы для гипервизора KVM.
