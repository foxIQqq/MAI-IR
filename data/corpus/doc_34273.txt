Компания Google открыла код Collide, IDE для совместной работы над проектами

В связи с закрытием инженерного подразделения в Атланте (США), компания Google перевела некоторые развиваемые в данном отделении разработки в разряд открытых. В частности, под лицензией Apache код экспериментального проекта "Collide" (Collaborative IDE), в рамках которого велась работа по созданию прототипа интегрированной среды разработки приложений, ориентированной на одновременное совместное редактирование кода несколькими разработчиками. Система построена с использованием web-технологий, интерфейс реализован через браузер и использует серверные компоненты на языке Java (для сборки требуется Java 7 и Ant). Серверная часть может быть запущена на любой машине разработчика, после чего участникам разработки достаточно открыть в браузере http://IP:8080. В настоящее время Collide находится на стадии демонстрационного прототипа, обкатывающего некоторые новые принципы совместного редактирования кода. На базе представленного кода построена одна из внутренних систем Google для совместной работы над проектами. Collide является существенно урезанным вариантом данной системы, тем не менее сохранившим ключевые функции. Предполагается, что независимое сообщество может заинтересоваться кодом Collide и создать на его основе полноценную web IDE.
