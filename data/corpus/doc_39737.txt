В Екатеринбурге состоится встреча разработчиков, использующих язык Java

15 мая в Екатеринбурге состоится первая встреча cообщества JUG.EKB (Java User Group), на которой состоится обсуждение различных технологий из мира Java, обмен опытом, а также просто интересное общение. Подобные встречи с лекциями Java экспертов планируется проводить регулярно. Доклады первой встречи: Андрей Старовойт из JetBrains — "Практики тестирования кода"; Сергей Целовальников из Naumen — "Как не наступить на грабли, внедряя скрипты в java приложение" Мероприятие состоится 15 мая в 19:30 во freelance cafe (ТЦ Мытный Двор), 8 Марта ул., 8Д, г. Екатеринбург.
