Список достижений в разработке СУБД Drizzle (легковесный MySQL)

Один из разработчиков СУБД Drizzle подвел итоги годичной работы над проектом, рассмотрев, какие из пожеланий разработчиков реализованы на текущий момент. Например, осуществлен переход на 6-месячный цикл подготовки релиза, разработан комплект для автоматизированного тестирования, оформлен внутренний API для разработки дополнений, создана инфраструктура для общения разработчиков, добавлены средства для мониторинга запросов в реальном режиме времени и т.д. Drizzle развивается силами независимого сообщества разработчиков и представляет собой упрощенный и более быстрый вариант MySQL, в котором убрана поддержка некоторых типов данных, хранимых процедур, триггеров, кэша запросов (query cache), представлений (view), операции GRANT и системы ACL, команды SHOW, предварительно подготовленных запросов (prepared statement) и других утяжеляющих работу MySQL возможностей. В качестве хранилища используется InnoDB и PBXT.
