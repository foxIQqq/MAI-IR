Релиз torrent-клиента Transmission 2.50

Представлен релиз Transmission 2.50, относительно легкого и нетребовательного к ресурсам BitTorrent-клиента, написанного на языке Cи и поддерживающего разнообразные интерфейсы пользователя: GTK, Qt, native Mac, Web-интерфейс, daemon, command-line. Наиболее заметные изменения: Все платформы: Более честное распределение полосы пропускания между подключенными пирами. Теперь используется меньше времени CPU при вычислении не скачанных частей у больших торрентов. Для выбора пиров-кандидатов используется алгоритм отбора вместо сортировки. Используются кратные 10 показатели при отображении пропускной способности и места на диске. В случае если в ОС есть своя библиотека natpmp, она будет по возможности использоваться вместо встроенной. Исправлен крах при добавлении некоторых magnet-ссылок. Улучшена поддержка скачки больших файлов с webseed'ов. Теперь transmission правильно обрабатывает ситуацию когда webseed присылает некорректные сжатые данные. Исправлены ошибки компиляции в Solaris 10 из-за mkdtemp() и во FreeBSD из-за alloca(); Исправлена компиляция в ситуации когда отсутствовал компилятор C++ но производилась попытка сборки libuTP (нуждающейся в компиляторе C++). GTK+: Исправлена регрессия, сломавшая работу параметра "--minimized" в командной строке. Вместо запуска notify-send (отсутствующего в некоторых конфигурациях) используется стандартное org.freedesktop.Notifications DBus API. Исправлено несколько мелких утечек памяти. Qt: Исправлена ошибка компиляции при сборке без поддержки библиотеки libuTP в Debian. Web Client: Реализована фильтрация по состоянию и трекеру. Реализована сортировка по размеру. Более крупные кнопки тулбара, которые проще нажимать в некоторых случаях. Исправлено отображение размера торрента и оставшегося времени в вкладке "details" у инспектора. Библиотека jQuery и CSS-файлы теперь поставляются в комплекте, для независимости веб-интерфейса от доступности CDN-серверов третьих лиц. jQuery обновлен до версии 1.7.1 Исправлены ошибки работы веб-интерфейса в Internet Explorer 8, Internet Explorer 9 и Opera. Таблицы стилей CSS используют SASS Мелкие улучшения интерфейса. Mac: Теперь требуется Mac OS X 10.6 Snow Leopard или новее. Анимированные столбцы в главном окне (только в Lion) Скачанные файлы помещаются в карантин (для защиты от вредоносного ПО) Инспектор более не плавает поверх остальных окон (по запросам пользователей) Уведомления: минимальная поддержка уведомлений для пользователей без Growl. Поддержка вставки из буфера обмена URL с файлом торрента в главное окно программы (только в Lion) Мелкие улучшения интерфейса и исправления ошибок. Daemon: Исправлено разрушение строки состояния в transmission-remote.
