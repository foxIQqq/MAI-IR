Релиз свободного торрент-клиента qBittorrent 3.1.0

Доступна новая версия торрент-клиента qBittorrent 3.1.0, написанного с использованием тулкита Qt, и, по заверению автора, наиболее полно из существующих свободных торрент-программ приближённого к интерфейсу и функциям популярного µTorrent. Из добавленных улучшений, можно отметить: Опция командной строки для запуска qbittorrent-nox в виде фонового процесса; В web-интерфейс добавлена кнопка "Shutdown qBittorrent"; Настройки для копирования .torrent-файлов завершённых загрузок; В интерфейс для создания торрентов добавлена поддержка групп и уровней трекера; Поддержка очистки логов запуска и блокировки; Опция для использования случайного номера порта при каждом запуске; Поддержка новостных лент в формате Atom; Добавлен столбец с указанием пути для сохранения; Во вкладку параметров пира добавлен столбец с μTorrent-совместимыми флагами; Обеспечена фоновая загрузка magnet-метаданных в процессе отображения диалога "Add New Torrent"; Возможность разбора только info-hash для загрузок; Диалог "Add New Torrent" выведен на передний план; Увеличена скорость отрисовки списка торрентов, содержащего более 100 элементов, а также больших списков пиров. qBittorrent обладает следующими особенностями: μTorrent-подобный интерфейс Интегрированные поисковые движки Поддержка всех расширений протокола BitTorrent: DHT, Peer Exchange, шифрование, Magnet/BitComet-ссылки Удалённое управление через Web-интерфейс Очереди и приоритеты как торрент-передач, так и отдельных файлов и каталогов внутри торрента Поддержка UPnP/NAT-PMP Доступность на 36 языках, включая русский, белорусский и украинский Инструмент создания торрентов Продвинутая поддержка RSS Расписание ограничений скоростей Фильтрация IP-адресов Совместимость с IPv6 Последовательное скачивание Кроссплатформенность. Поддерживаются: GNU/Linux, Mac OS X, Microsoft Windows, OS/2, FreeBSD Возможность использования без X-сервера
