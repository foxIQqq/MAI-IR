Релиз системы управления web-контентом WordPress 3.1

Вышел релиз WordPress 3.1, системы управления web-контентом, написанной на языке PHP и ориентированной на создание блогов. По сравнению с выпущенной в июне версией 3.0 в WordPress 3.1 внесено более двух тысяч коммитов, исправлено 820 ошибок. Из улучшений можно отметить: Упрощен процесс установки внутренних ссылок - для установки ссылки достаточно кликнуть на кнопку и через функцию поиска выбрать сообщение для ссылки; Добавлена плавающая панель Admin Bar, позволяющая просматривая сайт иметь под рукой наиболее полезные ссылки на страницы интерфейса администрирования; Упрощен интерфейс написания сообщений, в котором по умолчанию отображаются только базовые элементы, а дополнительные опции скрыты и отображаются после нажатия специальной кнопки; Возможность привязки разных визуальных тем к отдельным сообщениям, что позволяет использовать нестандартное стилевое оформление для определенных типов публикаций в блоге; Меню Super Admin и близкие по тематике страницы вынесены из базового административного блока на отдельную страницу Network Admin; Переработана система импорта и экспорта; Улучшена система определения пользовательских типов контента, для которых теперь можно создавать архивные страницы. Поддержка расширенных запросов, которые могут охватывать несколько пользовательских полей и таксономий; Обновлена цветовая схема Blue Admin с целью акцентирования большего внимания на контент.
