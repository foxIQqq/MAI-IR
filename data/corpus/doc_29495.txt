Вышла вторая альфа-версия Ubuntu 11.04

Вышла вторая альфа-версия Ubuntu 11.04 "Natty Narwhal". Загрузочные iso-образы с тестовой версией Ubuntu 11.04 можно получить на данной странице. По сравнению с прошлой тестовой версией в Ubuntu 11.04 Alpha-2 отмечены следующие улучшения: Осуществлен переход на Linux-ядро 2.6.38-rc2 (в которое интегрирован патч, существенно повышающий интерактивность); Графическая подсистема обновлена до X.Org Server 1.10 и Mesa 7.10. Для видеокарт ATI Radeon улучшена поддержка 3D. Интегрирована новая версия драйвера xf86-video-intel, в котором обеспечена полноценная поддержка нового поколения процессоров Intel, имеющих архитектуру Sandy Bridge, которая подразумевает наличие GPU (Intel HD Graphics 2000/3000) на одном чипе с CPU; В качестве базовой версии языка программирования Python теперь используется Python 2.7; Вместо OpenOffice.org в дистрибутиве теперь используется LibreOffice 3.3; В качестве рабочего окружения по умолчанию используется оболочка Unity, разработка которой еще не завершена. Проект Unity развивается интенсивно, поэтому желающим более полно отслеживать изменения в Unity рекомендуется использовать ежедневные сборки, в которые каждый день добавляются новые возможности и исправляются ошибки. Интегрирован новый Unity Launcher, предназначенный для запуска приложений и переключения между активными задачами. В Launcher реализовано контекстное меню со списками быстрого вызова и добавлена возможность автоматического скрытия с экрана. Для обеспечения группировки приложений на экране используется начальный прототип системы Places, который пока не поддерживает запланированные функции поиска и прокрутки экрана. Интерфейс Unity оптимизирован для максимального увеличения доступного вертикального пространства на рабочем столе и поддерживает технологию глобального меню, при котором строка с меню для всех приложений отображается не в окне программы, а в верхней управляющей панели. Unity соответствует стандартам freedesktop.org, активно использует GPU для акселерации выполнения графических операций и базируется на ключевых технологиях GNOME 3.0: библиотеке Clutter и фреймворке для организации работы с документами и пользовательской информацией Zeitgeist. Для управления окнами и выводом на экран используется Compiz. Для обеспечения поддержки сенсорных экранов и мультитач-навигации в Unity задействован фреймворк uTouch. Апплет для настройки сети NetworkManager переведен на использование фреймворка для генерации индикаторов AppIndicator. Классические панельные GNOME-апплеты более не поддерживаются, доступна только новая система индикаторов Ubuntu; В экранном менеджере gdm представлено три вида сессий: Ubuntu Desktop на базе Unity, Ubuntu Classic на базе классической панели GNOME и "2D-режим", отличающийся от Ubuntu Classic отключением использования 3D-элементов. Для работы Unity обязательно наличие современных 3D-драйверов, интеграция проекта Unity 2D ожидается только в осеннем релизе; Редакцию для нетбуков Ubuntu Netbook Еdition решено выпустить только для armel-платформ OMAP3 и OMAP4, для остальных архитектур можно будет использовать стандартную сборку Ubuntu Desktop Edition, в которую добавлены дополнительные средства для работы на системах с низкими экранными разрешениями. Интерфейс Ubuntu для архитектуры ARM по прежнему построен на базе библиотек EFL (Enlightenment Foundation Library) и использует Linux-ядро 2.6.35 (в 2.6.38 не доведена до рабочего состояния поддержка OMAP 4); В центре для управления установкой приложений Software Center добавлена функция "Ratings and Reviews", позволяющая пользователям определять и просматривать рейтинг приложений, а также оставлять свои комментарии; В Ubuntu Server интегрирована поддержка системы для организации сетевой загрузки и обновления группы машин cobbler и системы для организации параллельного выполнения заданий на разнвх системах mcollective. Программа Powernap, используемая для автоматического тюнинга системы с целью минимизации потребления энергии, обновлена до версии 2.0. В среднем при использовании Powernap потребление энергии удается снизить на 14%. Powernap оперирует результатами мониторинга за активностью пользователя (консоль, клавиатура, мышь), сетевой активностью и состоянием системы (LA, ввод/вывод); В Kubuntu произведен переход на KDE 4.6. Для работы со звуком использован Phonon GStreamer-бэкенд. Благодаря интеграции визуальной темы GTK Oxygen, GTK-приложения теперь значительно лучше вписываются в KDE и почти неотличимы от обычных KDE-программ. Вместо HAL (Hardware Abstraction Layer) используется udev в сочетании с udisks (DeviceKit-disks, взаимодействие с блочными устройствами) и upower (DeviceKit-power, управление питанием); Проект Xubuntu завершил переход на десктоп-окружение Xfce 4.8. Добавлена новая визуальная тема "greybird". Для обеспечения совместимости с Xfce 4.8 сборка Xubuntu продолжит использовать GTK2 и не перейдет на GTK3; Общий обзор планов, озвученных на саммите разработчиков Ubuntu, был представлен в прошлом анонсе.
