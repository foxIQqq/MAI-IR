Выпуск Chrome OS 118

Доступен релиз операционной системы Chrome OS 118, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 118. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Исходные тексты распространяются под свободной лицензией Apache 2.0. Сборка Chrome OS 118 доступна для большинства актуальных моделей Chromebook. Для использования на обычных компьютерах предлагается редакция Chrome OS Flex. Основные изменения в Chrome OS 118: Предоставлена возможность восстановления забытого пароля и возвращения доступа к данным, привязанным к учётной записи. Добавлен режим отображения самодостаточных web-приложений (PWA, Progressive Web Apps) в виде вкладок, позволяющих одновременно работать с несколькими документами в одном окне. При использовании данного режима разработчик приложения также может определить домашнюю вкладку, на которой, например, можно разместить интерфейс выбора документов и настройки. Упрощена настройка вывода на печать. Добавлен вывод контекстной справки по использованию принтера, упрощено сохранение выбранного принтера, предложена инструкция по настройке. На страницу с настройками добавлена информация о состоянии принтера. Предложен набор опций для управления принтером в случае возникновения проблем с выводом на печать. Обеспечена интеграция с сервисом Imprivata OneSign. Изменено оформление интерфейса для редактирования текста на сенсорных экранах. Предложены более интуитивно понятные управляющие жесты. Добавлен новый режим увеличения, автоматически показывающий точную позицию курсора. Предложен новый Bluetooth-стек, основанный на стеке Fluoride, перенесённом из платформы Android и продолжающий развитие стека BlueDroid от компании Broadcom. Для управления включением нового стека добавлена настройка "chrome://flags/#bluetooth-use-floss". Устранено 6 уязвимостей в компоненте virglrenderer, позволяющем задействовать virtio-gpu (виртуальный GPU Virgil3D) для использования 3D-акселерации в виртуальных окружениях без эксклюзивного проброса видеокарты в гостевую систему. 3 уязвимости приводят к переполнению буфера, а одна к обращению к памяти после её освобождения. Также устранена use-after-free уязвимость (CVE-2023-4921) в подсистеме net/sched ядра Linux, позволяющая поднять свои привилегии в системе.
