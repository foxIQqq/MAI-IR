Выпуск программы для профессиональной обработки фотографий Darktable 1.6

После года разработки представлен новый значительный релиз программы для редактирования фотографий Darktable 1.6, выступающей в роли свободной альтернативы таким продуктам, как Adobe Lightroom и Apple Aperture. Darktable поддерживает работу с RAW-изображениями и предоставляет большую подборку модулей для выполнения всевозможных операций по обработке фотографий. Darktable позволяет вести базу исходных фотографий, осуществлять наглядную навигацию по имеющимся снимкам и при необходимости выполнять операции корректировки искажений и улучшения качества, сохраняя при этом исходный снимок и всю историю операций с ним. В новой версии: Поддержка экранов высокого разрешения (high DPI); Режим слайд-шоу; В map view обеспечена возможность просмотра изображений только из текущей коллекции; Инструмент darktable-cli теперь может использоваться без использования X-сервера на системах без монитора; Поддержка воспроизведения звуковых примечаний; Прикрепляемый предпросмотр (sticky preview); Добавлена опция для перезаписи файлов в процессе экспорта; Добавлен crawler, который синхронизирует все файлы xmp при запуске; Поддержка больших изображений с пиксельным индексом более 32 бит, например, Darktable теперь способен открывать TIFF-файлы с разрешением 26770x13385; Переработан код чтения и записи TIFF-файлов, добавлена поддержка сжатия; Возможность установки PPI для экспортируемых JPEG-изображений; Для экспорта в Flickr теперь используется защищённый канал связи (HTTPS); В darkroom добавлена новая операция чистки краёв (defringe), реализована возможность отключения баланса белого, добавлен новый режим реконструкции цветов, улучшена работа инструмента basecurves; Ускорены операции преобразования цветов, инвертирования, баланса белого, записи pfm, задействованы инструкции SSE в коде обработки изображений; Для png/tiff добавлена поддержка встроенных профилей icc; Внесены многочисленные улучшения в движок для выполнения скриптов на языке Lua.
