Вышел дистрибутив Chakra GNU/Linux 2014.05

Представлен релиз Linux-дистрибутива с непрерывным циклом выпуска обновлений Chakra 2014.05, основанного на наработках проекта Arch Linux. Дистрибутив поставляется с десктоп-окружением KDE 4.13, снабжён графическим инсталлятором и конфигураторами, упрощающими настройку оборудования и системных параметров. Chakra GNU/Linux может работать в Live-режиме и устанавливаться на жёсткий диск. Для загрузки доступна DVD-сборка для архитектуры x86_64, размером 1.7 Гб. В репозитории проекта представлено более 5 тыс. пакетов, которые могут быть установлены при помощи пакетного менеджера pacman или через специальный web-каталог и систему Click'n'Run. Основные улучшения: Представлена полностью новая тема оформления Sirius, привнёсшая единый стиль в оформление всех стадий работы дистрибутива, в том числе меню загрузчика GRUB, KDM, KSplash и Yakuake. Представлен новый логотип проекта; Осуществлён перевод инфраструктуры проекта на новый сервер, упрощено сопровождение дистрибутива, улучшена система сборки, выполнена подготовка для перехода на пакетный менеджер Akabei; Компоненты KDE обновлены до выпуска 4.13.1. Вместо Nepomuk по умолчанию задействован движок семантического поиска Baloo. Добавлена опция для отключения Baloo; Обновлены компоненты графического стека: xorg-server 1.14.5, mesa 10.0.5, Nvidia 331.38 Catalyst 13.12, xf86-video-nouveau 1.0.10, xf86-video-ati 7.2.0, xf86-video-intel 2.21.15. Из репозитория удалён вариант драйвера Catalyst для устаревших видеокарт, так как он не поддерживает работу с новыми выпусками X-сервера. Вместо него следует использовать открытый драйвер xf86-video-ati; Обновлены системные компоненты: ядро Linux 3.12.15, systemd 212; По умолчанию включены новые приложения: интерфейс для управления пакетами octopi, kcm-pacman-repoeditor для редактирования параметров репозиториев, kcm-about-distro для вывода информации о дистрибутиве, система резервного копирования Kup и калькулятор kcalc.
