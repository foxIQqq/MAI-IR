Представлен GitSharp, реализация Git на C# и .Net

Представлен проект GitSharp, представляющий собой попытку реализации на языке C# библиотеки и утилиты для работы с Git репозиториями, адаптированной для работы в окружении .NET и Mono. Исходные тексты GitSharp распространяются в рамках лицензии BSD. Продукт развивается с целью создания полноценного Git-инструментария для Windows. От популярной Git-системы для Windows msysGit новый проект отличается наличием библиотеки функций, предоставляющей возможность встраивания Git-функциональности в другие приложения. Например, на базе GitSharp можно создать распределенное хранилище данных или P2P систему обмена контентом.
