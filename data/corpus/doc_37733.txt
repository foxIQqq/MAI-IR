Релиз Ubuntu 12.04.3 LTS c обновлённым графическим стеком и ядром 3.8

Опубликован промежуточный выпуск дистрибутива Ubuntu 12.04.3 LTS, примечательный не только включением накопившихся исправлений и обновлением установочного образа, связанных с устранением уязвимостей и проблем, влияющих на стабильность, но и поставкой для архитектуры x86 новой версии ядра Linux, свежего графического стека и драйверов с поддержкой нового оборудования. Одновременно представлены аналогичные обновления Kubuntu 12.04.3, Edubuntu 12.04.3, Xubuntu 12.04.3, Mythbuntu 12.04.3 и Ubuntu Studio 12.04.3. Использовать представленную сборку имеет смысл только для новых установок, системы установленные ранее могут получить все присутствующие в Ubuntu 12.04.3 изменения через штатную систему установки обновлений. Поддержка выпуска обновлений и исправлений проблем безопасности для серверной и десктоп редакций Ubuntu 12.04 LTS продлится до апреля 2017 года. Основные особенности Ubuntu 12.04.3: В состав дистрибутива включено новое ядро Linux 3.8, протестированное в рамках выпуска Ubuntu 13.04. Поддержка и поставка пакетов с ранее доступными ядрами 3.2 и 3.5 сохранена в полном объёме, тем не менее для новых установок ядро 3.8 теперь предлагается по умолчанию. Использование новой версии ядра ознаменовало не только значительное расширение поддержки оборудования, но и появление существенных новшеств, таких как: Прекращена поддержка процессоров 386-DX/SX; Добавлена новая файловая система F2FS; Реализован механизм проверки целостности метаданных для XFS; Обеспечена поддержка inline-хранения данных в Ext4; Подготовлен режим быстрой замены дисков для Btrfs; Реализована возможность применения пространств имён для непривилегированных пользователей; В cgroup добавлена поддержка ограничения памяти, потребляемой ядром; Реализация пространства имён для идентификаторов пользователей; Возможность проверки цифровой подписи при загрузке модулей ядра; Использование инструкций SMAP для защиты от эксплуатации уязвимостей в ядре; Поддержка протоколов SMBv2 и VXLAN; Поддержка режима быстрого открытия TCP-соединений (TFO - TCP Fast Open) на стороне сервера и клиента; Поддержка коротких очередей TCP пакетов (TSQ - TCP Small Queues); Реализация подсистемы VFIO (Virtual Function I/O) для создания виртуализированных драйверов устройств, работающих в пространстве пользователя. Поддержка групп IOMMU (IOMMU Groups), позволяющих обеспечить изоляцию устройств PCI и PCIe с использованием технологий виртуализации ввода/вывода AMD-Vi и Intel VT-d; Добавлен режим "Suspend to both", комбинирующий спящий и ждущий режимы, путем перехода в ждущий режим после предварительного сохранения образа памяти на диск; Обновлённый графический стек, включающий новые версии X.Org-сервера, Compiz, Mesa и видеодрайверов для чипов Intel, AMD и NVIDIA. Обновлённые компоненты графического стека соответствуют версиям из состава Ubuntu 13.04 и достаточно хорошо протестированы; В Xubuntu, Mythbuntu, Ubuntu Studio и Kubuntu добавлена поддержка загрузки на системах с активным режимом UEFI Secure Boot c использованием штатного загрузчика, подписанного ключом Microsoft (в Ubuntu и Edubuntu подобная поддержка была добавлена в прошлом обновлении); Интегрирована значительная порция исправлений для оболочки Unity; Обновлены версии нескольких десятков приложений.
