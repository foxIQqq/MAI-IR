Вышел TeX Live 2015

Доступен релиз популярного LaTeX-дистрибутива TeX Live 2015, созданного в 1996 году на базе проекта teTeX. TeX Live является наиболее простым способом для развертывания инфраструктуры для подготовки научной документации, независимо от используемой операционной системы. Для загрузки доступна DVD-сборка (2,8 Гб) TeX Live 2015, которая содержит рабочее Live-окружение, полный набор установочных файлов для различных операционных систем, копию репозитория CTAN (Comprehensive TeX Archive Network), подборку документации на разных языках (включая русский). Из новшеств можно отметить включение в LaTeX2e по умолчанию сторонних изменений, ранее поставляемых в форме внешнего пакета fixltx2e. В LaTeX2e также включены связанные с движком Unicode настройки, такие как определение какие символы являются буквами. В pdfTEX добавлена поддержка JPEG Exif и JFIF. В LuaTEX представлена новая библиотека newtokenlib для поиска токенов. В XeTEX добавлен новый исполняемый файл numbersystem. Прекращена поддержка платформы *-kfreebsd, так как TEX Live теперь легко доступен через механизмы системной платформы.
