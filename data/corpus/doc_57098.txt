Доступна мобильная платформа KDE Plasma Mobile 22.04

Опубликован выпуск мобильной платформы KDE Plasma Mobile 22.04, основанной на мобильной редакции рабочего стола Plasma 5, библиотеках KDE Frameworks 5, телефонном стеке ModemManager и коммуникационном фреймворке Telepathy. Для вывода графики в Plasma Mobile используется композитный сервер kwin_wayland, а для обработки звука применяется PulseAudio. Одновременно подготовлен выпуск набора мобильных приложений Plasma Mobile Gear 22.04, формируемого по аналогии с набором KDE Gear. Для создания интерфейса приложений применяется Qt, набор компонентов Mauikit и фреймворк Kirigami из состава KDE Frameworks, позволяющий создавать универсальные интерфейсы, пригодные для смартфонов, планшетов и ПК. В состав входят такие приложения, как KDE Connect для сопряжения телефона с рабочим столом, просмотрщик документов Okular, музыкальный проигрыватель VVave, просмоторщики изображений Koko и Pix, система ведения заметок buho, календарь-планировщик calindori, файловый менеджер Index, менеджер приложений Discover, программа для отправки SMS Spacebar, адресная книга plasma-phonebook, интерфейс для осуществления телефонных вызовов plasma-dialer, браузер plasma-angelfish и мессенджер Spectral. В новой версии: В мобильную оболочку перенесены изменения, развиваемые в ветке KDE Plasma 5.25, релиз которой ожидается 14 июня. В интерфейсе для переключения между запущенными приложениями (Task Switcher) улучшена анимация при активации и сворачивании приложений. Добавлена возможность сортировки запущенных приложений в порядке их открытия, а не только в алфавитном порядке. В панели задач улучшена адаптация к ширине экрана. В панели навигации обеспечено отключение прозрачности при открытии экранной клавиатуры, но не других окон оболочки. Добавлена возможность вызова урезанного варианта выпадающей панели быстрых настроек (Action Drawer) при блокировке экрана. Обеспечено закрытие панели при касании к внешней пустой области. Предоставлена поддержка перегруппировки настроек оболочки. Улучшена анимация при нажатии на кнопки быстрых настроек. Добавлена возможность переключения между реализациями домашнего экрана. Новые типы домашних экранов пока не добавлены, но в каталоге KDE Store будет предоставлена возможность размещения альтернативных вариантов домашнего экрана для KDE Plasma Mobile. В базовом домашнем экране добавлена анимация увеличения и уменьшения пиктограмм приложений при действиях с ними пользователя. Текст с названиями приложений теперь выделен жирным шрифтом для улучшения читаемости. В мультимедийном проигрывателе реализована поддержка параллельных потоков (одновременно несколько приложений могут выводить звук). В конфигураторе обновлено оформление модуля для настройки параметров сотовых сетей. Улучшена страница для настройки APN. Интерфейс для совершения звонков (Plasma Dialer) переведён на использование фонового процесса callaudiod, развиваемого проектом Mobian, что позволило избавиться от собственных обработчиков звука и обеспечить использование общего кода для разных устройств и дистрибутивов. В виджет часов добавлена поддержка порталов Flatpak, обеспечивающих фоновое выполнение, что позволило реализовать автозапуск процесса kclockd в режиме sandbox-изоляции. В Tokodon, клиенте к децентрализованной микроблогинговой платформе Mastodon, обеспечен вывод всей доступной информации о профиле пользователя. Добавлена поддержка блокировки, приглушения и отслеживания (follow) других пользователей. Переделаны интерфейс выбора учётной записи и боковая панель. Улучшена работа календаря-планировщика Kalendar. Продолжена работа над клиентом к коммуникационной системе Nextcloud Talk, в котором реализована большая часть API комнат общения. Улучшен интерфейс Spacebar, программы для отправки SMS/MMS. Переработаны верхняя панель, навигация по приложению и интерфейс управления вложениями. Добавлены настройки уведомлений, позволяющие скрыть отправителя и содержимое сообщения при показе уведомлений во время блокировки экрана. Добавлена возможность выбора активного номера телефона для записей в адресной книге, к которым привязано несколько номеров. Добавлена поддержка перехода по ссылкам при просмотре сообщений.
