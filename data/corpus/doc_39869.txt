Разработка MariaDB переводится c Bazaar на Git

Разработчики СУБД MariaDB объявили о решении перевести разработку проекта на использование системы управления версиями Git и платформы совместной работы GitHub. Ранее, продолжая традиции MySQL, разработка MariaDB велась в поддерживаемом компанией Canonical сервисе Launchpad с использованием системы контроля версий Bazaar (bzr). Находящаяся в разработке ветка MariaDB-10.1 уже полностью переведена на GitHub. Стабильные ветки 10.0, 5.5, 5.3, 5.2 и 5.1 пока остаются на Launchpad, но в ближайшее время их ожидает миграция на GitHub. В процессе перехода с Bazaar на Git также находится проект Bugzilla.
