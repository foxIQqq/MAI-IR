Обновление Raspbian, дистрибутива для Raspberry Pi

Доступно обновление дистрибутива Raspbian, официально рекомендуемого для использования на Raspberry Pi и специально оптимизированного для данной платформы (сборка для ARMv6 с расширениями "hard float"). Размер образа для записи на SD-карту составляет 1.3 Гб. Для установки из репозиториев доступно около 35 тысяч пакетов. Ключевые новшества: Поддержка платы Raspberry Pi Zero; Включение в состав приложения Node-RED, предоставляющего средства для визуального проектирования приложений для IoT-устройств (Internet Of Things); Представлен новый графический интерфейс для управления пакетами, созданный на основе программы GNOME Packages; В состав включены библиотеки GPIO Zero, позволяющие организовать доступ к расширениям GPIO из скриптов на языке Python; Обновлён web-браузер Epiphany, решены проблемы с отображением видео на сайтах YouTube и Vimeo; Расширены возможности интерфейса для настройки системы.
