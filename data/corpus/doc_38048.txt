Вышел дистрибутив Red Hat Enterprise Linux 5.10 и платформа Red Hat Satellite 5.6

Компания Red Hat анонсировала релиз дистрибутива Red Hat Enterprise Linux 5.10, в котором представлены исправления серьёзных ошибок, устранения проблем безопасности и обновления с ограниченным бэкпортированием поддержки нового оборудования. Наращивание функциональности для нового оборудования в ветке RHEL 5 прекращено. Из улучшений отмечается: Интегрирована новая версия фреймворка OpenSCAP с поддержкой протокола SCAP 1.2 (Security Content Automation Protocol) и набором инструментов для автоматической проверки системы на соответствие требованиям безопасности. В составе дистрибутива поставляется MySQL 5.5. Для перехода на него с mysql 5.0 предоставляется (без поддержки в дальнейшем) mysql 5.1. Дальнейшая поддержка mysql 5.0 в RHEL 5.x прекращена. Обновлен драйвер cciss, в который добавлена поддержка серверов Gen8 Snap IVB-EP с новым контроллером HP P430i 12GB Smart Array. Обновлён драйвер ALSA HDA, в который добавлена поддержка звуковых карт Creative Recon3D. В поставку включена утилита fsfreeze для временной приостановки активности в ФС. Предоставлен доступ к набору Red Hat Developer Toolset 2.0, включающему пакеты с более новыми выпусками средств для разработчиков приложений, в том числе доступны Eclipse 4.3.0, Dyninst 8.0, Strace 4.7 и GCC 4.8. Одновременно представлен новый выпуск платформы для управления инфраструктурой Linux-серверов - Red Hat Satellite 5.6, которая основана на свободном проекте Spacewalk и развивается синхронно с ним. Продукт позволяет организовать инвентаризацию оборудования и программного обеспечения, автоматизировать установку и обновление ПО на группе машин, централизовано управлять файлами конфигурации, осуществлять мониторинг, управлять работой виртуальных машин, автоматизировать установку типовых конфигураций системы, организовать сетевую загрузку и обновление группы машин. В новом выпуске Red Hat Satellite предоставлена возможность использования в качестве внешней СУБД как Oracle, так и PostgreSQL. Добавлены средства для автоматизации развертывания систем в окружениях без PXE-загрузки через поддержку формирования кастомизированных ISO-образов. Реализована поддержка синхронизации между несколькими системами Red Hat Satellite. Обеспечена интеграция с ABRT (Automatic Bug Reporting Tool) для централизованного агрегирования информации об ошибках и крахах.
