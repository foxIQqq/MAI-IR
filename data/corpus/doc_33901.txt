Увидела свет платформа виртуализации Oracle VM 3.1

Компания Oracle представила новую версию Oracle VM 3.1, дистрибутива для развертывания решений виртуализации, построенного на базе Oracle Enterprise Linux и гипервизора Xen. Основу Oracle VM составляют два компонента Oracle VM Server для серверов виртуальных машин и Oracle VM Manager для управляющих узлов. Продукт нацелен на упрощение сопровождения виртуальных инфраструктур, организации централизованного управления и обеспечение развёртывания промышленных приложений на виртуальных и физических серверах. В поставку входит более ста шаблонов для быстрого создания виртуальных окружений, учитывающих специфику различных приложений. Oracle VM доступен для бесплатной загрузки и не требует при использовании лицензионных отчислений (компания зарабатывает на предоставлении услуг платной поддержки). В новой версии задействовано Linux-ядро Unbreakable Enterprise Kernel 2. Для упрощения выполнения текущих задач через web-интерфейс подготовлено несколько новых мастеров для генерации виртуальных машин и настройки параметров окружений. Добавлена поддержка подключения дополнительных виртуальных CPU к работающим виртуальным окружениям. Расширены возможности по конфигурированию систем хранения: добавлена поддержка использования внешних NFS-разделов, улучшены встроенные средства резервного копирования и восстановления после сбоев, обеспечена возможность копирования и перемещения хранилищ на базе iSCSI и Fibrechannel с одного кластера на другой. Обеспечена интеграция с продуктами Oracle Enterprise Manager и Oracle Enterprise Manager Ops Center.
