В Chrome 86 будет по умолчанию включено контекстное меню для возвращения полного URL

В кодовой базе, которая будет использована при формировании релиза Chrome 86, удалён экспериментальный флаг, позволяющий отобразить в контекстном меню пункт для показа в адресной строке полного URL. Элемент "Всегда показывать URL полностью" будет доступен в контекстном меню по умолчанию, без необходимости изменения параметров на странице about:flags. Полный URL также можно просмотреть кликнув два раза на адресной строке. Напомним, что в Chrome 69 компания Google провела эксперимент по скрытию "https://", "http://" и "www." в адресной строке. Первая попытка оказалась неудачной и скрытие было отменено. Позднее эксперименты продолжились и в Chrome 76 по умолчанию адрес стал показываться без протокола и поддомена www, а в Chrome 79 была удалена настройка по возвращению старого поведения. После недовольства пользователей в Chrome 83 был добавлен новый экспериментальный флаг, добавляющий в контекстное меню пункт для отключения скрытия и показа полного URL в любых условиях. В прошлом месяце эксперименты были продолжены добавлением флага, оставляющего в адресной строке только домен, без элементов пути и параметров запроса.
