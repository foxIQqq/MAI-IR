Вышел восьмой тестовый выпуск openSUSE 11.2

Вышел восьмой тестовый выпуск openSUSE 11.2. Релиз openSUSE 11.2 намечен на ноябрь. По сравнению с прошлой тестовой сборкой в состав включены следующие новые версии программ: Linux ядро 2.6.31 GNOME 2.28 RC OpenOffice.org 3.1.1 PulseAudio 0.9.17 Samba 3.4.1 postgresql 8.4.1 bluez 4.5.1 php 5.3.0 Evolution 2.27.92 Firefox 3.5.3 Gnumeric 1.9.12 KTorrent 3.2.4 Pidgin 2.6.2 SeaMonkey 2.0b2 Thunderbird 3.0 Beta 4 VirtualBox 3.0.6 Кроме того отмечен прогресс в разработке новой визуальной темы и проведена адаптация Firefox для более полной интеграции в интерфейс KDE. Базовые новшества openSUSE 11.2: В качестве десктоп-окружения по умолчанию будет использоваться KDE 4.3; Время поддержки дистрибутива (выпуска обновлений) будет сокращено с 24 до 18 месяцев; Пакеты дистрибутива собраны GCC 4.4.1 Образ LiveCD теперь может быть использован и на USB Flash; В состав LiveCD добавлен Midnight Commander, а в KDE-вариант LiveCD программа YaKuake; Для YaST создан новый центр управления конфигурацией (YaST Control Center) с интерфейсом на базе Qt4; В качестве файловой системы по умолчанию задействована Ext4 В состав Live CD включен Gimp; Для сжатия данных на LiveCD отныне используется алгоритм сжатия LZMA; По умолчанию LiveCD содержит переводы для немецкого, французского, итальянского, польского и русского языков; В Qt версии конфигуратора YaST появилась поддержка настраиваемого внешнего вида. В YaST добавлены средства работы с кешем пакетов; В Zypp улучшены средства работы с зеркалами репозиториев пакетов; В интерфейсе для настройки параметров дисплея gnome-display-properties добавлена поддержка автоматического сброса настроек в случае проблем. Ведется разработка web-интерфейса к YaST для осуществления удаленного управления дистрибутивом Появление установочных образов для USB Flash и утилита для их подготовки Улучшение поддержки субноутбуков; В состав LiveCD с KDE добавлены новые программы: Choqok, KNode, Kompare, KSystemlog, Okteta; В состав GNOME LiveCD добавлен Bittorrent-клиент Transmission; Новое визуальное оформление инсталлятора YaST2 и новая визуальная тема для KDE; Пакеты сформированы с использование RPM версии 4.7.1 При установке c DVD, десктоп-окружение KDE отныне выбрано по умолчанию; В zypper добавлена опция "--download-only", позволяющая осуществить только загрузку компонент, без выполнения действи. Опцию можно использовать совместно с командами install, update, patch, dup и verify;
