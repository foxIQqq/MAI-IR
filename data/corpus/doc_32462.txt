Макеты возможных вариантов интерфейса Ubuntu для смартфонов

Опубликовано несколько неофициальных макетов с предложениями по организации интерфейса Ubuntu для смартфонов. Макеты подготовлены независимыми дизайнерами, заинтересовавшимися проектом по созданию версии Ubuntu для телефонов. Ник Ратлидж (Nick Rutledge) подготовил дизайн интерфейса в традиционном для смартфонов стиле, отличающемся использованием одновременного вертикального и горизонтального скроллинга: Макеты Сэма Хорна (Sam Horne) дизайнера под ником Musl1m отталкиваются от текущих особенностей оболочки Unity, например, используют боковую панель и интерфейс для поиска/запуска приложений:
