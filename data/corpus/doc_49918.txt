Выпуск дистрибутива Funtoo 1.3, развиваемого основателем Gentoo Linux

Представлена новая версия дистрибутива Funtoo 1.3, который развивает Дэниель Роббинс (Daniel Robbins), основатель дистрибутива Gentoo, с момента своего ухода из проекта в 2009 году. Funtoo основан на пакетной базе Gentoo и ставит перед собой цель дальнейшего усовершенствования имеющихся технологий. Для загрузки формируются установочные образы, специально оптимизированные для достижения максимальной производительности на системах с различными моделями процессоров. Из ключевых особенностей Funtoo выделяется поддержка автоматической сборки пакетов из исходных текстов (пакеты синхронизируются из Gentoo), использование Git при разработке, распределённое дерево portage-ей, более компактный формат сборочных манифестов, применение инструментария Metro для создания live-сборок. Выпуск 1.3 позиционируется как LTS (версия с длительной поддержкой) и отличается дополнительной стабилизацией (ранее применялась полная синхронизация с репозиториями Gentoo) и переходом к модели разработки на основе снапшотов наборов пакетов (kit) с заморозкой и самостоятельной поддержкой форков части из них (в том числе наборов xorg-kit и gnome-kit). Базовая часть заморожена в виде, синхронизированном из Gentoo 13 ноября. В новом выпуске объявлена устаревшей поддержка "multilib", обеспечивающая возможность запуска 32-разрядных исполняемых файлов в 64-разрядном окружении (сборки Funtoo для 64-разрядных CPU теперь поддерживают выполнение только 64-разрядых приложений). В состав включено LTS-ядро Linux 4.9 и выпуски GCC 7.4.1, binutils 2.31.1, glibc 2.27, openrc 0.40.2, perl 5.26, lxd 3.6. Добавлены сборки для AWS (Amazon Web Services). Предложены новые наборы: core-server-kit с подборкой серверных приложений, core-ui-kit c фреймворками для разработки графических приложений и core-gl-kit с Mesa и библиотеками, связанными с OpenGL.
