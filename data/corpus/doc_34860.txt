Компания Canonical представила новую версию Landscape и непрерывно обновляемый репозиторий для OpenStack

Компания Canonical анонсировала значительное обновление Landscape, проприетарной системы управления большими группами рабочих станций, серверов и облачных окружений, работающих на базе Ubuntu Linux. Управление Landscape производится через web-интерфейс, позволяющий автоматизировать выполнение типовых операций по администрированию системы (например, добавление пользователей, установка и обновление пакетов), применительно к группе машин. Пакет поддерживает функции мониторинга, централизованного управления пакетами, обеспечения единой точки аутентификации (SSO), интеграции с cloud-инфраструктурой. Landscape позволяет управлять тысячами машин также легко как одной машиной. Исследование компании Enterprise Management Associates показало, что при управлении инфраструктурой из тысячи машин с использованием Landscape стоимость обслуживания понизилась примерно на 2/3, а коэффициент возврата инвестиций (ROI) за 5 лет составил 1121%. Среди ключевых улучшений в новой версии Landscape: Обеспечение поддержки разделения привилегий при помощи RBAC (Role-based access control); Инструменты для формирования отчётов об обеспечении требований к инфраструктуре; Средства для управления локальными репозиториями; API для расширения возможностей web-интерфейса и обеспечения интеграции с другими продуктами через подключение скриптов-дополнений; Поддержка MAAS, инструментария для быстрого и удобного развертывания Ubuntu-конфигураций на множестве физических серверов с использованием техник, используемых в облачных платформах; Улучшение интеграции с Ubuntu 12.04, добавление апплета Management Service для обеспечения регистрации в Landscape; Увеличение масштабируемости, Landscape может обслуживать до 40 тысяч устройств; Возможность задания политики автоматической установки обновлений для разных категорий машин. Одновременно компания Canonical объявила о введении в строй для LTS-релизов Ubuntu непрерывно обновляемого репозитория Ubuntu Cloud Archive из которого администраторы могут загрузить последние версии облачной платформы OpenStack. Пользователям предоставлена возможность использования свежей версии OpenStack на протяжении всего цикла поддержки LTS-релизов Ubuntu, не ограничиваясь изначально поставляемой версий. В настоящее время в репозитории обеспечена поддержка Ubuntu 12.04 LTS Server.
