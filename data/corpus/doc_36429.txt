Конфликт, связанный правами на торговую марку Python, урегулирован мирным путём

Организация Python Software Foundation (PSF) сообщила о мирном урегулировании конфликта с компанией Veber, подавшей в Европейский комитет по торговым маркам заявку на получение торговой марки Python. Компания Veber согласилась отозвать свою заявку на торговую марку Python и поменять название для своих облачных сервисов и продуктов. Таким образом сообщество разработчиков Python сохранит все права на имя Python на территории Евросоюза, а компания Veber избежит возможной путаницы между языком программирования и развиваемыми компанией сервисами. Напомним, что английская компания Veber, владеющая сайтом python.co.uk, зарегистрированным 13 лет назад, попыталась получить эксклюзивные права на использование торговой марки Python в привязке к любым компьютерным технологиям, включая программные продукты и сервисы. Кроме того, Veber была намерена использовать бренд Python для распространения своих серверных продуктов. Язык Python существует уже более 20 лет и права на торговую марку Python в США изначально были зарегистрированы на организацию Python Software Foundation, которая, тем не менее, упустила вопрос регистрации торговой марки в Евросоюзе. Компания Veber не предъявляла каких-либо претензий к Фонду Python и не пыталась оказывать давление, но сообщество было обеспокоено потенциальным негативным влиянием причисления имени Python к интеллектуальной собственности Veber.
