Выпуск KDE Gear 21.08, набора приложений от проекта KDE

Представлено августовское сводное обновление приложений (21.08), развиваемых проектом KDE. Напомним, что сводный набор приложений KDE c апреля публикуется под именем KDE Gear, вместо KDE Apps и KDE Applications. Всего в рамках обновления опубликованы выпуски 226 программ, библиотек и плагинов. Информацию о наличии Live-сборок с новыми выпусками приложений можно получить на данной странице. Наиболее заметные новшества: Изменения в файловом менеджере Dolphin: Улучшена возможность оценки содержимого каталогов через показ миниатюр - если в каталоге большое число файлов, то при наведении курсора эскизы с их содержимым теперь прокручиваются, что упрощает определение наличия нужного файла. Добавлена поддержка предпросмотра для файлов, размещённых в зашифрованных областях, таких как Plasma Vaults. В информационной панели, включаемой через нажатие F11 и показывающей детальные сведения о файлах и каталогах, реализовано обновление данных о размере и времени доступа в режиме реального времени, что удобно для отслеживания прогресса загрузки и появления изменений. Упрощён интерфейс для переименования нескольких файлов: после переименования выбранного файла по клавише F2 теперь можно нажать клавишу Tab для перехода к переименованию следующего файла или Shift + Tab - предыдущего. Предоставлена возможность выделения имени файла по аналогии с текстом для помещения имени в буфер обмена. В контекстное меню, показываемое при нажатии правой кнопки мыши на корзине в боковой панели Places, добавлена возможность вызова настроек корзины. Проведена чистка меню "гамбургер", показываемого в верхнем правом углу. В просмотрщике документов Okular появилась возможность добавления в панель инструментов кнопки для переключения цвета текста и фона страницы с чёрных букв на белом фоне на тёмнокрасные буквы на сером фоне, что более комфортно для чтения (кнопка добавляется через секцию Configure Toolbars в контекстном меню). Предоставлена опция для отключения вывода всплывающих окон с уведомлениями о встроенных в документ файлах, формах и подписях. Также добавлены настройки для выборочного скрытия различных типов аннотаций (подсветка, подчёркивание, обрамление и т.п.). Во время добавления аннотации реализовано автоматическое отключение режимов навигации и выделения, чтобы случайно не перейти к другой области и не выделить текст для буфера обмена вместо пометки для аннотации. В эмуляторе терминала Konsole добавлена поддержка предпросмотра изображений и каталогов - при наведении курсора на имени файла с картинкой пользователю теперь будет показан эскиз изображения, а при наведении курсора на имя каталога - появятся сведения о содержимом. При клике на имени файла будет запущен обработчик, связанный c типом данного файла (например, Gwenview для JPG, Okular для PDF и Elisa для MP3). Более того, удерживая клавишу Alt при клике мышью на имени файла, данный файл теперь можно переместить в другое приложение в режиме drag-and-drop. При необходимости одновременного отображения нескольких вкладок в панели инструментов предложена новая кнопка, а также добавлены комбинации Ctrl + "(" и Ctrl + ")", позволяющие разделить окно и показать сразу несколько вкладок. Размер каждой области можно настроить мышью, а итоговый макет сохранить для последующего использования через меню "View > Save tab layout to file…". Из новшеств отдельно выделяется SSH-плагин, который позволяет выполнять действия на внешних хостах, например, с его помощью можно создать каталог на другой системе, с которой настроено соединение через SSH. Для включения плагина следует использовать меню "Plugins > Show SSH Manager", после чего появится боковая панель со списком добавленных в ~/.ssh/config SSH-хостов. В просмотрщике изображений Gwenview проведена работа по увеличению производительности и модернизации интерфейса. В нижнем правом углу предложен новый компактный набор кнопок, позволяющих быстро изменить масштаб и размер, а также изменить цвет фона. В процессе навигации для перехода от одного изображения к другому теперь можно использовать размещённые в панели кнопки со стрелками и клавиши управления курсором. Для остановки и продолжения воспроизведения видео можно использовать клавишу "пробел". Добавлена поддержка показа изображений с 16-битами цвета на канал и чтения цветовых профилей из файлов в различных форматах. Проведена реструктуризация меню "гамбургер", показываемого в верхнем правом углу, через которое предоставлен доступ ко всем имеющимся возможностям. В музыкальном проигрывателе Elisa добавлен режим вечеринки, включаемый клавишей F11. При выходе из программы обеспечено запоминание параметров трека для продолжения воспроизведения с прерванной позиции после запуска. В программе для создания скриншотов Spectacle предоставлена возможность создания снимка экрана окна над которым находится курсор мыши (активируется нажатием Meta + Ctrl + Print). Значительно повышена надёжность работы в окружениях на базе Wayland. В текстовом редакторе Kate упрощена работа с шаблонами готовых кусков кода (Snippets), которые теперь можно загрузить через менеджер управления приложениями Discover. На базе LSP (Language Server Protocol) реализована поддержка языка программирования Dart. В видеоредакторе Kdenlive осуществлён переход на новый выпуск фреймворка MLT 7, который позволил реализовать такие возможности, как добавление изменения скорости клипа в число эффектов с ключевыми кадрами. Улучшен менеджер задач. Ускорены операции импорта файлов и открытия проектов. Обновлено приложение KDE Connect, обеспечивающее интеграцию рабочего стола KDE со смартфоном. В новой версии реализована поддержка отправки ответов прямо из уведомлений о поступлении сообщения. Добавлена официальная поддержка платформы Windows, а само приложение предложено в каталоге Microsoft Store. Во всплывающем по клавише F12 терминале Yakuake добавлен режим разделения окна для показа сразу нескольких вкладок. Предоствлена возможность переключения между панелями через комбинацию клавиш Ctrl+Tab. В утилите для работы с архивами (Ark) добавлена заставка, показываемая при запуске без указания файлов. Реализована поддержка распаковки zip-архивов, в которых для разделения каталогов используется обратный слэш вместо прямого.
