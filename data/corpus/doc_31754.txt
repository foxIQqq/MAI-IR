Релиз Proxmox VE 1.9, дистрибутива для развертывания виртуальных серверов

Вышел релиз Proxmox Virtual Environment 1.9, специализированного Linux-дистрибутива на базе Debian GNU/Linux, нацеленного на развертывание виртуальных серверов с использованием OpenVZ и KVM. Сразу после установки пользователь получает полностью готовую систему виртуальных серверов промышленного уровня с управлением через web-интерфейс, работающий поверх libvirt. Дистрибутив имеет встроенные инструменты для организации резервного копирования виртуальных окружений и доступную из коробки поддержку кластеризации, включая возможность миграции виртуальных окружений с одного узла на другой без остановки работы. Размер установочного iso-образа 342 Мб. Основные улучшения Proxmox 1.9: Инструментарий KVM обновлен до версии 0.15. Добавлена поддержка менеджера памяти KSM ("Kernel Samepage Merging"), определяющего факты использования разными процессами идентичных по содержанию областей памяти и производящего их объединение в один блок; Обновлено Linux-ядро 2.6.32 (vzkernel-2.6.32-042stab036.1.src.rpm) c поддержкой OpenVZ и патчами из Debian Squeeze. Добавлены новые драйверы e1000e, ARECA RAID, megaraid_sas, bnx2, igb, ixgbe и т.д. DRBD обновлен до версии 8.3.10; Утилиты OpenVZ (vzctl) обновлены до версии 3.0.28-1pve5. В качестве шаблона контейнеров по умолчанию задействован debian-6.0-standard; В поставку включены новые версии qemu-server 1.1-32, pve-manager 1.9-24, pve-qemu-kvm 0.15.0, libpve-storage-perl 1.0-19 и vzdump 1.2-15.
