Представлена бета-версия нового выпуска клиента Steam для Linux

Компания Valve анонсировала начало бета-тестирования нового выпуска клиента Steam для платформы Linux. Из изменений можно отметить поддержку ограничения пропускной способности для загружаемых игр, добавление кнопки для быстрого просмотра наличия обновлений на странице загрузки, новый диалог для управления выводом уведомлений и звуковыми эффектами в web-чате, налажена поддержка настройки работы через прокси через переменную окружения http_proxy, обеспечена корректная обработка горячих клавиш при удержании Shift. Тем временем, общее число доступных в каталоге Steam игр для Linux за месяц увеличилось с 79 до 104.
