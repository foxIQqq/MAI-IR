GitHub преодолел рубеж в 10 млн репозиториев

Хостинг свободных проектов GitHub.com, базирующийся на системе управления исходными текстами Git и позволяющий разработчикам общаться в формате единой социальной сети, объявил о преодолении отметки в десять миллионов репозиториев. Примечательно, что около пяти с половиной миллионов репозиториев было создано в 2013 году, в то время как для накопления первого миллиона репозиториев проекту потребовалось почти четыре года. Из наиболее интересных проектов, появившихся на GitHub в 2013 году отмечены: Project Open Data - развиваемый Белым Домом (США) набор инструментов для организации открытого доступа к информации госучреждений; Docker - инструментарий для управления изолированными Linux-контейнерами; Tate Metadata - коллекция метаданных о более чем 70 тысячах произведений искуств и 3.5 тысячах художниках, представленных в галерее Tate; Серия открытых проектов компаний Amazon, Facebook, Google, Microsoft, Netflix, Square, Twitter и Walmart.
