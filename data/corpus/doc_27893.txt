Разработчики Mozilla выступили с инициативой по разработке игр на основе web-технологий

Подразделение Mozilla Labs представило новый проект - Mozilla Labs Gaming, связанный с развитием игр на основе технологий Open Web. Современные технологии Open Web состоят из множества направлений, таких, как html5-теги Video и Audio, WebGL (стандарт для формирования аппаратно ускоренного 3D контента), API для сенсорных экранов, API для реагирования на изменения ориентации экрана, учет местоположения и быстрые JavaScript-движки, которые делают возможным создание сложных и не очень сложных игр в Web. Сегодня, с возможностями современных браузеров, настало самое время для продвижения этой платформы, а лучшего способа, чем игры, для этого не найти, поскольку традиционно разработчики игр находятся на переднем плане развития технологий. В качестве начального шага по продвижению инициативы Mozilla Labs Gaming стартует первый интернациональный конкурс - Game On 2010. К участию в конкурсе призываются разработчики, заинтересованные в создании грандиозных игр на основе новейших технологий Open Web.
