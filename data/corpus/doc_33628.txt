Новый тестовый выпуск Chrome для Android

Компания Google представила второй тестовый выпуск браузера Chrome для платформы Android, базирующийся на кодовой базе Chromе 18. В отличие от существенно урезанного по возможностям штатного браузера для платформы Android, Chrome для Android позволяет воспользоваться на телефонах и планшетах полным набором функций, свойственных современным браузерам для настольных систем, обладая при этом интерфейсом, оптимизированным для мобильных устройств. Из новшеств, добавленных с момента выхода первой тестовой версии, можно отметить: Режим представления десктоп-браузером, что позволяет загрузить вместо специальной мобильной версии полноценный вариант сайта; Добавление закладок на домашний экран в форме ярлыков; Возможность определения MIME-обработчиков для разных типов контента; Поддержка работы через системный прокси, конфигурируемый в настройках Android; Поддержка загрузки файлов на локальное хранилище устройства; Поддержка режимов вывода текста CTL (Complex Text Layout)) и RTL (Right to Left) для языков с двунаправленным и обратным направлением письма; Поддержка проигрывания старых встраиваемых блоков YouTube через штатное приложения для просмотра контента YouTube; Поддержка специфичных для разных стран поисковых систем.
