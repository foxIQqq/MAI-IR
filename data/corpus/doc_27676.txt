Google вводит новые правила для публикации расширений и объявляет о создании Web-каталога приложений

Учитывая, что уже не раз в каталоге расширений для веб-браузера Chrome оказывались вредоносные дополнения, которые копировали личные данные пользователей, корпорация Google решила ужесточить требования для публикации расширений, тем и мобильных приложений. Начиная с этого момента, для возможности публикации дополнений с разработчиков будет однократно взыматься плата в размере пяти американских долларов, а также будет в обязательном порядке производиться привязка дополнения к доменному имени разработчика. С уже зарегистрированных разработчиков плата взыматься не будет. Google предполагает, что эти меры сократят поток зловредных расширений, распространяемых для веб-бразуера Google Chrome. Также разработчики Google объявили о предварительном открытии (пока только для разработчиков) каталога-магазина приложений для Google Chrome - Chrome Web Store. Публикуемые Web-приложения могут представлять из себя всё что угодно, начиная от встраиваемых веб-страниц, работающих внутри веб браузера, заканчивая версиями приложений для мобильных телефонов, которые теперь могут использовать большое пространство обычного монитора. Для написания веб-приложений Google Chrome можно использовать его API и все поддерживаемые браузером технологии (HTML5/JS/DOM/etc). Презентацию Chrome Web Store можно посмотреть на сайте YouTube.
