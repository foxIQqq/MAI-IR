Релиз Linux дистрибутива Salix 13.0.2

Доступно обновление Linux дистрибутива Salix 13.0.2, развиваемого создателем Zenwalk Linux, покинувшим проект в мае в результате конфликта с другими разработчиками, отстаивавшими политику максимальной похожести на Slackware. Дистрибутив Salix остался на 100% совместим со Slackware Linux 13.0, исповедует подход "одно приложение на одну задачу" и поддерживает три режима установки: Полная - десктоп окружение на основе Xfce, Firefox, почтовый клиент Claws-mail, полная версия офисного пакета OpenOffice.org, Java Runtime Environment, мультимедиа плеер Totem, ПО для ведения архива музыки Exaile, пакетный менеджер gslapt; Основная - Xfce, Firefox и пакетный менеджер gslapt; Базовая - только минимальный набор консольных утилит, подходит для создания серверов. Изменения в Salix 13.0.2, по сравнению с прошлым выпуском: Начата подготовка 64-разрядной сборки Salix64, совместимой со Slackware64; В комплект 32-разрядной сборки включено Linux ядро huge.s, позволяющее использовать дистрибутив на устаревших системах с процессорами AMD K6, Intel Pentium и даже i486; Добавлен режим упрощенного заведения пользователей в процессе установки и внесены другие небольшие улучшения в инсталлятор; Обновлены пакеты с десктоп-окружением XFCE, в которых исправлена неприятная ошибка, приводящая к монтированию USB накопителей с файловой системой VFAT в синхронном режиме; В комплект включена утилита salix-update-notifier для периодической проверки на наличие обновлений и информировании пользователей через системный трей. Добавлена поддержка локализации кнопок на рабочем столе XFCE.
