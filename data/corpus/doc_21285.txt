Вышла J2SE платформа Apache Harmony 5.0m9 и кластерный JVM Terracotta 3.0

Вышла девятая стабильная сборка свободной J2SE 5 (Java 2, Standard Edition) платформы Apache Harmony. В рамках проекта Apache Harmony разработаны независимые от Sun Microsystems реализации Java SE 5 JDK, виртуальной машины и библиотеки классов. В новом выпуске Apache Harmony отмечены следующие улучшения: Улучшение производительности в библиотеке классов, оптимизация работы JIT компилятора, сокращение интенсивности MUL/DIV/REM для целочисленных вычислений; Новая реализация технологии сжатия Java-байткода, описанной в JSR-200; Библиотека классов портирована для работы в операционных системах AIX и zOS; Исправления ошибок, улучшение прохождения текстов на совместимость, устранение недоработок, вскрывшихся при испытаниях в тестовом комплекте Eclipse Test Suite. Кроме того, выпущен релиз пакета для кластеризации Java приложений - Terracotta 3.0. Код Terracotta распространяется в рамках лицензии Terracotta Public Licence, основанной на Mozilla Public Licence. Продукт позволяет организовать работу виртуальной Java-машины распределенной на несколько узлов кластера и имеющей встроенную поддержку кеширования Java-объектов.
