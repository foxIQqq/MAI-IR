Web-браузер Opera 10 перешел на стадию бета-тестирования

Анонсирован выход первой бета-версии web-браузера Opera 10. По сравнению с прошлыми тестовыми версиями Opera 10, в бета-версии появилась поддержка визуального режима переключения табов, при котором мини-изображение содержимого страницы отображается непосредственно на вкладках (переключение между визуальным и стандартным режимом производится в один клик). В Speed Dial появилась новая система настройки содержимого окна, добавлена возможность определения фона и указания более 9 сайтов (режим 4 x 4). Другие новшества Opera 10: Новый движок Presto 2.2, позволивший реализовать: Прохождение 100 из 100 HTML/CSS/JavaScript тестов Acid3 + прохождение дополнительного теста на плавную анимацию; Новая система для обработки регулярных выражений, отличающаяся значительным увеличением производительности. Особенно данная оптимизация должна повлиять на результаты прохождения тестов SunSpider; Улучшение производительности при обработке CSS; При просмотре XML без определения стиля теперь производится автоформатирование вывода; Поддержка возможности загрузки TrueType (TTF), OpenType (OTF) и SVG шрифтов из сети. Управление загрузкой осуществляется через CSS элемент @font-face (свойства font-family и src); Поддержка цветовых моделей RGBA и HSLA в CSS3, поддержка "color: transparent"; Реализация JavaScript-селекторов Selectors API. Поддержка интерфейсов querySelectors и querySelectorsAll, позволяет значительно ускорить доступ к некоторым элементам web-страниц. Функциональность для совершения автообновлений, без необходимости переустановки; Средство для проверки орфографии в web-формах на лету, по мере ввода текста. Словарь для русского языка можно загрузить здесь; Поддержка виджетов в сборке для платформы Linux/UNIX. Размещение прозрачных виджетов на десктопе поддерживается только при использовании композитного менеджера и видеорежиме с 32-битным цветом; Во встроенном почтовом клиенте появилась возможность создания HTML-сообщений, с картинками, шрифтовым оформлением текста и ссылками; Добавлена поддержка TLS 1.2 и удалена SSL v2.
