Вышел дистрибутив Calculate Linux 15

Доступен релиз отечественного дистрибутива Calculate Linux 15, построенного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в нескольких редакциях, собранных для архитектур i686 и x86_64: Calculate Linux Desktop с рабочим столом KDE (CLD), Mate (CLDM) и Xfce (CLDX), Calculate Directory Server (CDS), Calculate Linux Scratch (CLS), Calculate Scratch Server (CSS) и Calculate Media Center (CMC). Все версии дистрибутива распространяются в виде загрузочного Live-образа с возможностью установки на жесткий диcк или USB-накопитель. Основные изменения: Новый метод обновления пакетов, для которого свойственны: Повышенная надежность - каждое зеркало бинарных пакетов теперь хранит информацию о версии модификации, что позволяет использовать для обновления любое из доступных зеркал, а так же создавать зеркало обновлений в локальной сети. Более высокая скорость обновлений - утилита обновления оценивает скорость канала каждого зеркала, выбирая наиболее быстрое зеркало. Установка пакетов теперь по умолчанию выполняется параллельно. По умолчанию в cl-update отключен вызов revdep-rebuild Дополнительный уровень тестирования - подготовленные бинарные пакеты проходят дополнительную проверку в тестовом зеркале до загрузки на master-зеркало. Экономия трафика - при отсутствии обновлений убраны запросы для синхронизации портежей. Представлены общие репозитории бинарных пакетов. Профили дистрибутивов теперь унифицированы. Для сборки бинарных пакетов создано три новых дистрибутива с USE-флагами под различные конфигурации. Репозитории насчитывают более 2500 популярных бинарных пакетов. Новая версия утилит Calculate 3.4 Полностью переписаны утилиты сборки системы. Добавлена поддержка графического интерфейса в Calculate Console. Добавлена поддержка overlayfs для мгновенного разворачивания системы. Добавлена поддержка использования списка бинарных зеркал для обновления. Изменён алгоритм загрузки обновлений Добавлена поддержка создания мультизагрузочной Flash. Репозиторий бинарных пакетов включает, помимо стабильной, всегда свежую версию ядра.
