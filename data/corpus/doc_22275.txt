Определены финалисты конкурса самых популярных проектов на SourceForge.net

Завершен первый этап конкурса по выбору самых популярных проектов, размещенных на SourceForge.net - Community Choice Awards 2009. Из более 47 тыс. проектов, номинированных на участие в конкурсе, было отобрано 85 финалистов, из которых в последствии будут выбраны 12 победителей. Примечательно, что в этом году отмечается всплеск интереса к конкурсу: число номинированных работ примерно в два раза превышает показатели прошлого года. Выбор победителей будет произведен путем открытого голосования, к участию в котором допускаются все желающие (для незарегистрированных пользователей требуется подтверждение голоса по email). Голосование продлится до 20 июля, награждение победителей состоится 23 июля на конференции OSCON. Конкурсные категории: "Лучший проект"; "Лучший проект для бизнеса"; "Лучший проект для любителей компьютерных игр"; "Лучший мультимедиа-проект"; "Лучший новый проект"; "Лучшая утилита для сисадмина"; "Лучшая утилита для разработчиков"; "Лучший коммерческий open source проект"; "Лучший проект для учёбы"; "Лучший проект для правительственных учреждений"; "Проект способный изменить привычный образ действий"; "Лучшее визуальное оформление". Из числа финалистов, одновременно в несколько конкурсных категорий попали следующие проекты: аудиоредактор Audacity; web-форум Beehive Forum; Cyn.in Groupware; СУБД Firebird; FreeNAS; GoldenDict - ПО для работы со словарями от отечественных разработчиков; KeePass Password Safe - менеджер паролей для Windows; Lancelot - альтернативная реализация меню и ланчера для KDE 4; phpMyAdmin; PortableApps.com: Portable Software/USB - ПО для создания переносимых на USB Flash установок Windows программ, проект номинирован сразу в 8 категорий; ProcessMaker Open Source - коммерческое open source ПО для управления бизнес процессами; ReactOS поисковый движок Sphinx symfony PHP framework; TortoiseSVN - Subversion клиент для Windows; XAMPP - пакет для развертывания web-сервера на базе Apache, MySQL, PHP, Perl и phpMyAdmin; XBMC Media Center; XMind - программа для составления интеллект-карт (mind map, карт-памяти); Ajax фреймворк ZK1.
