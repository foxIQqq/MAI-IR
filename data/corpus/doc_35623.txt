Проект Raspberry Pi запустил свой каталог-магазин приложений

Разработчики одноплатного компьютера Raspberry Pi объявили о введении в строй собственного каталога-магазина приложений - Pi Store, позиционируемого как площадка для распространения платных и бесплатных приложений, представляющих интерес для пользователей Raspberry Pi. В настоящее время в каталоге опубликовано 5 игр (Freeciv, OpenTTD, Iridium Rising, Storm in a Teacup, POWDER), 7 приложений (LibreOffice, Asterisk for Raspberry Pi, Code::Blocks IDE и т.п.) и 4 инструмента разработки (Pi3D, Audio Pack, Hud Sprite Pack, Effects Sprite Pack). Все поставляемые в каталоге приложения опубликованы в сборках для Linux-дистрибутива Raspbian и доступны для быстрой установки с сайта. Для работы с каталогом из дистрибутива подготовлено специальное приложение pistor, которое можно установить командой "sudo apt-get update && sudo apt-get install pistor".
