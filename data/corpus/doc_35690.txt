Релиз Apache Isis 1.0

Вышел релиз фреймворка для быстрой разработки проблемно-ориентированных приложений на языке Java - Apache Isis 1.0, первый выпуск после включения Isis в число первичных проектов Apache. Apache Isis даёт возможность определить бизнес-логику в виде сущностей, предметных сервисов и репозиториев, на основании которых фреймворк динамически сгенерирует готовое представление предметной модели в форме web-приложения или RESTful API. Наглядный пример процесса создания простейшего приложения с использованием Apache Isis можно оценить на данной странице.
