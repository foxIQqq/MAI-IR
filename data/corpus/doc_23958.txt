Новые версии видеодрайверов: AMD Catalyst 9.10, NVIDIA 190.42 и S3 14.02.26

Компания AMD выпустила новую версию проприетарного видеодрайвера для видеокарт ATI - AMD Catalyst 9.10. По сравнению с прошлым выпуском, в AMD Catalyst 9.10 отмечены следующие изменения: Реализована поддержка графических карт Radeon HD 5750/5770 Juniper; Добавлена поддержка отмеченных в спецификации RandR 1.3 произвольных преобразований Добавлена возможность загрузки fglrx модуля ядра без выгрузки DRM-модуля свободных драйверов; Исправлены проблемы с работой модуля ядра fglrx в случае использования DKMS 2.1.0; Устранены недоработки при выводе меню в центре управления драйвером; Одновременно, компания NVIDIA представила новый выпуск своих проприетарных драйверов 190.42, пока имеющих статус кандидата в релизы, но уже достаточно стабильных, чтобы занять место ветки 185.18.x. Главное отличие ветки состоит в поддержке спецификации OpenGL 3.2. Из улучшений внесенных с момента выпуска версии 190.36 можно отметить следующие новшества: Поддержка карт видеозахвата NVIDIA Quadro SDI; Поддержка GPU: GeForce G102M, GT 220, G210, G210M, GT 230M, GT 240M, GTS 250M, GTS 260M; Возможность для GeForce 8 и более новых GPU использовать расширенные видеорежимы, выходящие за пределы указанных в EDID списке диапазонов HorizSync или VertRefresh; Увеличен допустимый уровень компенсации переразвертки со 100 до 200; Улучшена работа акселерации обработки H.264 и MPEG для карт с поддержкой VDPAU категории "B"; Устранены проблемы с картами старее GeForce 8 GPU, мешающие работе расширения GLX_EXT_texture_from_pixmap, что приводило к некорректному выводу данных на экран в Compiz и KDE 4. Дополнение: Компания Nvidia сменила статус драйвера NVIDIA 190.42 с кандидата в релизы на стабильный релиз. В заключение, можно упомянуть выход еще одного проприетарного дарайвера для Linux - S3 14.02.26, поддерживающего видеокарты серии S3 Chrome 4xx и 5xx. Основные новшества: Поддержка видеокарт S3 Graphics 5400E; Доступность сборки для 64-разрядных систем; Поддержка Ubuntu9.04/9.10 (i386 и x86_64) Поддержка OpenGL 3.1. Модуль для переключения видеорежимов на уровне Linux ядра и поддержка VDPAU.
