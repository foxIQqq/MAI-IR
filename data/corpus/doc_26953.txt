Запуск классических приложений в Chromium OS будет организован в режиме тонкого клиента

Как известно, компания Google изначально ограничила функциональность открытой операционной системы Chromium OS возможностью выполнения только web-приложений. Операционная система Chromium OS базируется на Linux ядре, открытых компонентах и web-браузере Google Chromium, на базе которого сформирован интерфейс пользователя. Тем не менее разработчики проекта Chromium OS планируют реализовать возможность работы в окружении Chromium OS со стандартными программами, созданными для других ОС, но сделать это планируют не совсем обычным образом. Рассматривается возможность организовать выполнение дополнительных программ, используя устройство с Chromium OS в виде тонкого клиента, на котором производится только отображение информации и организация взаимодействия с пользователем. При этом непосредственно программа будет выполняться на внешнем сервере приложений. Для организации работы будет использована созданная в Google технология "Chromoting", детали по её реализации пока не опубликованы, упомянуты лишь общие принципы. Из предварительных заявлений можно понять, что данная технология будет аналогом классических средств для работы с приложениями удалённого рабочего стола (VNC, NoMachine NX, Remote Desktop Connection), расширенных механизмами для формирования VPN для соединения удалённого устройства на базе Chrome OS с настольным ПК или сервером приложений. Для интеграции Chromoting в систему поддержка данной технологии будет реализована в виде плагина к браузеру Google Chrome.
