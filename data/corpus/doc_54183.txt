Доступен серверный дистрибутив NethServer 7.9

Представлен релиз дистрибутива NethServer 7.9, предлагающего модульное решение для быстрого развёртывания серверов в небольших офисах или на предприятиях среднего размера. Дистрибутив основан на пакетной базе CentOS 7.9 и предоставляет web-интерфейс для управления доступными серверными компонентами. Размер установочного образа 1.2 ГБ. Для ознакомления с возможностями интерфейса предоставляется online-демонстрация. Наработки проекта распространяются под свободными лицензиями. Пользователю предлагаются готовые модули для организации работы почтового сервера (Postfix, Dovecot, Amavis, ClamAV + web-клиент Roundcube), системы совместной работы (SOGo), межсетевого экрана (Shorewall), web-сервера (LAMP), файлового сервера и контроллера домена Active Directory (Samba), фильтрующего прокси (Squid, ClamAV и SquidGuard), VPN-сервера (OpenVPN, L2TP), облачного хранилища (ownCloud), систем обнаружения и предотвращения вторжений. Установка и введение в строй нужного сервиса производится в один клик и не требует знания особенностей настройки каждого серверного компонента. Типовые работы по администрированию можно выполнять через web-интерфейс. Основные новшества: Пакетная база обновлена до CentOS 7.9. Прекращена поставка старого интерфейса NethGui (Server Manager), на смену которому пришёл интерфейс пользователя, построенный на базе Cockpit и предлагающий более современное оформление. По умолчанию NethGui больше не поставляется, но его можно установить из Software Center. Новый интерфейс запускается на сетевом порту 9090, но приложения CGP (Collectd Graph Panel), EveBox, Rspamd UI, Lightsquid и Ntopng остаются доступны через порт 980, независимо от выбранного интерфейса. В SSH по умолчанию отключены ненадёжные алгоритмы шифрования. По умолчанию отключена поддержка TLS 1.1, TLS 1.0, SSL v3 и SSL v2. В Threat Shield реализована поддержка блокировки DNS-запросов по чёрным спискам доменных имён (ранее блокировка производилась по IP-адресам). По умолчанию обеспечена установка агента мониторинга Netdata, на базе которого формируются графики, отражающие активность web-прокси (squid), почтового сервера (postfix) и межсетевого экрана. Добавлена новая панель для синхронизации почты через IMAP при выполнении миграции учётной записи с внешнего сервера на NethServer. Добавлена опция для отключения SIP-ALG, из-за которого возникают проблемы проброса через межсетевой экран Shorewall и протоколов SIP и H.323 Обновлены версии NtopNG 4.2, Nextcloud 20.0.1, Rspamd 2.5 и Asterisk 13.37.1.
