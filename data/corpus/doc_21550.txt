Пакет 2D/3D моделирования MEDUSA4 за 3 месяца удвоил пользовательскую базу

Немецкий разработчик CAD/CAM решений, компания CAD Schroer, объявила о рекордном росте количества пользователей бесплатной версии ПО для 2D/3D моделирования MEDUSA4 Personal. Продукт, версии которого существуют как для Windows так и для Linux и Solaris, за первый квартал удвоил пользовательскую базу по сравнению с аналогичным периодом прошлого года. Отрадно заметить, что из 109 стран, откуда пришли заявки на загрузку MEDUSA4, больше половины пользователей отдали предпочтение именно Linux версии. MEDUSA4 Personal — это полнофункциональная версия ПО для создания и редактирования инженерной графики, имеющая достаточно мощный инструментарий и удобный пользовательский интерфейс. Предназначение данной версии ограничивается личным пользованием и нуждается в обновлении лицензии каждые пол-года. Продукт может оказаться полезным как для целей самостоятельного обучения CAD/CAM технологиям, так и для энтузиастов, занимающихся созданием различных технических устройств. «Наши разработчики очень ценят те положительные отзывы, которые приходят в адрес MEDUSA4 Personal» - говорит управляющий директор копании Michael Schroer. «Определенно, существует множество увлеченных любителей, которые также хотят работать с профессиональными CAD системами». По мнению разработчиков MEDUSA4 Personal одна из немногих инженерных программ, бесплатная полнофункциональная версия которой доступна для самого широкого круга пользователей. Более того, это единственное в своем роде профессиональное решение, которое работает под Linux. В настоящее время пакет может быть установлен на следующие дистрибутивы: CentOS, Fedora, Mandriva, RedHat, SuSE и Ubuntu.
