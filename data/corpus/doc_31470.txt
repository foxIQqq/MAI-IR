Вышел кластерный дистрибутив Rocks 5.4.3

Доступен релиз Rocks 5.4.3, основанного на CentOS дистрибутива для создания вычислительных кластеров, grid-сетей и инфраструктур для обеспечения работы виртуальных окружений на базе Xen. Дистрибутив может быть использован для быстрого развертывания кластеров в локальной сети с загрузкой основного управляющего узла с LiveCD/LiveUSB. После загрузки управляющего узла, дополнительные узлы могут быть загружены по сети через PXE и автоматически настроены. В комплект входит набор специально разработанных средств для централизованного управления кластером и мониторинга его работы. Размер загрузочного iso-образа 2.7 Гб. Основные изменения: Новая версия построена на пакетной базе CentOS 5.6 по состоянию на 7 августа. Изменен подход в установке паролей, если раньше выбранный для фронтэнда пароль использовался для конечных узлов и запущенных сервисов, таких как MySQL и Wordpress, то сейчас указанный пароль применяется только для фронтэнда, а для остальных систем пароли генерируются случайным образом и могут быть изменены после установки. Возможность определения в БД произвольных категорий для группировки хостов; Изменена структура команд для управления межсетевым экраном, для каждой команды теперь требуется определять имя правила; Представлены две новые связки - /opt/perl с Perl 5.14.1 и подборкой модулей из CPAN, и /opt/python c Python 2.7.2 и 3.2.1 (в штатной поставке представлены старые версии Perl и Python, указанные связки позволяют использовать последние выпуски). Инсталлятор anaconda обновлен до версии 11.1.2.224.
