Релиз Qt SDK 1.2 и Qt Creator 2.4.1. Проект по созданию визуальных эффектов для Qt

Компания Nokia выпустила новую версию пакета Qt SDK 1.2, который ориентирован на упрощение разработки, сборки, отладки и тестирования как настольных программ, так и кросс-платформенных мобильных приложений, основанных на Qt. Nokia Qt SDK базируются на фреймворке Qt, интегрированной среде разработки Qt Creator, наборе библиотек Qt Mobility и симуляторе Qt Simulator. Подготовленное с использованием Nokia Qt SDK приложение без дополнительных модификаций можно скомпилировать для устройств на базе платформ Symbian, Maemo и MeeGo, а также для широкого круга настольных операционных систем. В новой Qt SDK версии осуществлён переход на Qt 4.8, обновлена версия Qt Creator 2.4.1, добавлено 20 новых примеров создания мобильных приложений, обновлён пакет для поддержки ОС Symbian в котором представлен новый плагин для анализа кода и инсталляционный пакет CODA 1.0.6. В средствах разработки для Nokia N9 внесены улучшения в Notifications API. Одновременно доступен корректирующий выпуск интегрированной среды разработки Qt Creator 2.4.1, в котором, по сравнению с версией 2.4.0, исправлено несколько ошибок. Дополнительно можно отметить анонс нового проекта Qt Labs - Qt Graphical Effects, нацеленного на предоставление средств для разработки и применения визуальных эффектов для Qt5/Qt Quick 2.0. В рамках проекта создан модуль, QtGraphicalEffects который позволяет легко применять готовые графические эффекты в QML-скриптах, например, затенение, размывание, изменение цвета, создание тени, искажение и т.п. В настоящий момент подготовлено 20 готовых эффектов, но пользователем также доступна возможность создания собственных. Для создания эффектов используется элемент QML ShaderEffect, позволяющий выполнять сложные графические манипуляции, комбинируя GLSL (OpenGL Shading Language) и QML, без привлечения дополнительного API на C++.
