Проект Mozilla начал тестирование каталога приложений Firefox Marketplace для платформы Android

Разработчики проекта Mozilla объявили о помещении в тестовую Aurora-ветку Firefox для Android поддержки магазина web-приложений Firefox Marketplace. В каталог добавлено несколько тестовых приложений, используя которые предлагается оценить работу магазина, перед тем как он будет интегрирован в другие версии Firefox и будет задействован в Firefox OS. Во время начального тестового периода установка программ из Firefox Marketplace поддерживается только в редакции Firefox для платформы Android. Из пока нереализованных возможностей также отмечается распространение платных приложений, ведение рейтинга и публикации рецензий. Все размещаемые в каталоге приложения написаны с использованием стандартных web-технологий (HTML5, CSS3, JavaScript) и будут работать как в мобильной, так и в настольной версиях Firefox. Firefox Marketplace также рассматривается как основной источник установки приложений для смартфонов на базе операционной системы Firefox OS. Размещённые в Firefox Marketplace приложения, несмотря на то, что они построены только с использованием web-технологий, после установки функционируют как отдельные программы в окружении платформы Android. Запуск подобных программ производится из специального меню браузера, после чего они видны в менеджере управления приложениями Android как отдельные задачи. Запущенные таким образом приложения отображаются без дополнительных элементов управления, в частности, убраны область вкладок и другие элементы пользовательского интерфейса браузера - формирование интерфейса целиком производится силами самого web-приложения, как и в случае обычных мобильных программ. Каталог приложений является частью более масштабной платформы Web Apps. В число компонентов платформы также входит универсальный API для управления установкой web-приложений и предоставления доступа к локальным ресурсам; набор программных интерфейсов Web API, продвигаемых в качестве web-стандартов через организацию W3C; сервис идентификации Mozilla Persona, основанный на технологии BrowserID и дающий пользователям возможность контролировать свой контент.
