The-M-Project - HTML5-фреймворк для создания универсальных мобильных приложений

Представлен новый фреймворк для разработки приложений для мобильных устройств - The-M-Project, базирующийся на использовании JavaScript и технологий HTML5. Весь цикл разработки приложения производится на языке JavaScript, итоговая программа может работать на таких мобильных платформах, как Apple iOS, Android, Palm webOS и BlackBerry. Код проекта распространяется в рамках лицензии MIT. Приложение создается с использованием парадигмы MVC (Model-View-Controller) с задействованием для построения интерфейса фреймворка jQuery Mobile. Несмотря на то, что программы оформляются в виде web-приложений, поддерживается режим работы в offline и локальное хранение данных, организованное с использованием средств HTML5 по организации локальных хранилищ. Для создания типового каркаса проекта, управления ресурсами, сборки приложений и запуска кода на локальной системе подготовлен специальный инструмент Espresso, базирующийся на использовании серверной JavaScript-платформы node.js.
