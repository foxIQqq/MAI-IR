Замена Network Manager на Wicd для настройки параметров сети в Ubuntu

"Wicd - замена многострадального Network Manager" - замена Network Manager на Wicd для настройки проводной и беспроводной сети в Ubuntu Linux. Wicd доступен для всех популярных Linux дистрибутивов, не зависит от GNOME и GConf, хорошо работает в разных десктоп-окружениях, поддерживает создание профилей соединений, совместим с wireless-tools, поддерживает WEP/WPA/WPA2, сворачивается в трей и отображает качество сигнала и интенсивность трафика.
