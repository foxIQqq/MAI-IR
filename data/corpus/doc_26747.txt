Для iPhone представлено приложение для синхронизации данных с Firefox

Не имея возможности подготовить вариант мобильной сборки Firefox для iPhone, из-за лицензионных ограничений на поставку программ для данной платформы, разработчики Mozilla представили для пользователей iPhone приложение Firefox Home. Через задействование возможностей системы синхронизации Wеave (Firefox Sync) приложение Firefox Home позволяет просмотреть на iPhone актуальные данные из последней сессии Firefox на настольной системе. Поддерживается навигация по системе закладок, выборка данных из истории посещений и просмотр списка текущих открытых вкладок (табов).
