HP готовит версию WebOS для ноутбуков и настольных ПК

Корпорация Hewlett-Packard, весной 2010 года поглотившая компанию Palm Inc, анонсировала три новых устройства на базе мобильной Linux-платформы WebOS: два новых смартфона (HP Pre 3 и HP Veer) и интернет-планшет HP TouchPad. Более того, в процессе презентации новых продуктов Тод Бредли (Todd Bradley), вице-президент HP, заявил о планах по превращению WebOS в операционную систему для всех типов устройств. Первые ноутбуки и настольные ПК, поставляемые с WebOS, поступят в продажу в течение 2011 года. Подробности не сообщаются, но можно предположить, что HP планирует использовать WebOS не как поставляемую вместо Windows систему общего назначения, а в роли быстро загружаемого программного окружения, доступного наряду с обычной операционной системой. Сейчас в качестве такого окружения на некоторых ноутбуках HP поставляется Splashtop. Одним из наиболее инновационных улучшений WebOS, продемонстрированных в процессе презентации, является поддержка бесшовного обмена данными между устройствами на базе WebOS. Например, пользователь планшета Touchpad может передать ссылку пользователю телефона Pre 3 просто приблизив аппараты и перетащив ярлык. Также планируется обеспечить широкие возможности по синхронизации информации и обмену данных между различными типами мобильных и настольных устройств, работающих на базе различных программных платформ. WebOS является многозадачной ОС, которая отличается тесной интеграцией с интернет-технологиями и широким использованием web-приложений. Весь пользовательский интерфейс WebOS оформлен с использованием CSS, HTML5 и JavaScript. Для разработки приложений на языках C/C++ и для организации прямого доступа к низкоуровневому API платформы доступен Plug-in Development Kit. Вместо окон используется парадигма сменяющих друг друга карт. Обмен данными между приложениями построен на основе шины, манипулирующей данными в формате JSON. Для доступа web-приложений к системному API используется D-Bus Bridge. Из стандартных программ присутствует основанный на Webkit web-браузер, близкий по возможностям к браузерам настольных систем (поставляется с Adobe Flash Player 10.1), приложения для чтения электронной почты (c функцией синхронизации с Microsoft Exchange), календарь-планировщик, калькулятор, заметки, клиент для мгновенного обмена сообщений, мультимедиа плеер. Для синхронизации данных (календарь, почта, адресная книга, фотографии и т.п.) с web-сервисами (Facebook, Gmail и т.п.) используется технология Synergy. Поддерживается навигация через жесты и одновременные касания к экрану (мультитач режим). Характеристики представленных новых устройств на базе WebOS: Интернет-планшет HP TouchPad: 9.7-дюймовый сенсорный экран (1024 x 768, цветность 18-bit) с поддержкой технологии multitouch; Двухядерный CPU Qualcomm Snapdragon APQ8060 (1.2GHz) Встроенная память 16 Гб или 32 Гб (размер ОЗУ не указан) WiFi 802.11b/g/n with WPA, WPA2, WEP, 802.1X Bluetooth2.1 + EDR с A2DP-стереовыводом; 3G-модем (опционально) aGPS (опционально) Micro-USB порт, совмещающий в себе разъем для зарядки аккумулятора; 1.3-мегапиксельная камера; 3.5мм аудио вход/выход; Сенсор освещенности, акселерометр, компас, гироскоп, функция HP touch-to-share Аккумулятор 6300mAh; Размер: 190 x 242 x 13.7 мм. Вес 740 гр. Цена не сообщается. Смартфоны HP Pre 3 и HP Veer продолжают развитие моделей Palm Pre и Palm Pixi. Аппараты снабжены выдвижной QWERTY-клавиатурой, поддерживают Wi-Fi 802.11b/g/n, GPS, Bluetooth, поставляются с 5-мегапиксельной камерой. HP Pre 3 оснащен 3.6-дюймовым экраном (800 x 480), процессором Qualcomm Snapdragon MSM8x55 с частотой 1.4GHz, 512 Мб ОЗУ, 16 или 8 Гб Flash, аккумулятором 1230mAh и укомплектован дополнительной фронтальной web-камерой. HP Veer является упрощенным вариантом и снабжен 800MHz CPU, 2.6-дюймовым экраном (400 x 320), аккумулятором 910mAh, Flash 8 Гб. В продажу устройства поступят весной, цена не сообщается.
