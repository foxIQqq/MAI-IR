Демонстрация телефона ZTE, основанного на платформе Firefox OS

Разработчики Mozilla опубликовали видеодемонстрацию с обзором находящейся в разработке модели смартфона ZTE, укомплектованного открытой платформой Firefox OS. В демонстрации, кроме ознакомления с базовыми возможностями Firefox OS и штатными приложениями, больше внимание уделяется каталогу-магазину приложений Mozilla Marketplace. Компания ZTE намерена первой выпустить телефоны с Firefox OS, поступление в продажу которых запланировано на начало следующего года. Находящийся в разработке телефон базируется на аппаратной платформе Qualcomm Snapdragon. Платформа Firefox OS развивается сообществом Mozilla и нацелена на создание мобильной операционной системы, загружающейся в браузерное окружение и предлагающей унифицированный Web API для разработки мобильных приложений с использованием HTML5-технологий, CSS и JavaScript. Оценить текущее состояние развития Firefox OS можно воспользовавшись экспериментальными ежедневными сборками, которые формируются для Linux, Windows и Mac OS X.
