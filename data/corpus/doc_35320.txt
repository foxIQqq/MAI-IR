Вышел Linux дистрибутив Parsix 4.0

Вышел Parsix 4.0, пользовательский дистрибутив, построенный на пакетной базе Debian Testing и продолжающий развитие LiveCD-дистрибутива KANOTIX. Размер загрузочного iso-образа составляет 1028 Мб. Загрузка с LiveDVD осуществляется при помощи GRUB 2, в качестве файловой системы задействована Squashfs 4.0 c Lzma методом сжатия (при установке на диск используется ext4). В комплект включен установщик дистрибутива на USB-накопители. Среди новшеств Parsix 4.0 : Переход на использование графического окружения GNOME 3.4.2 (ранее использовался GNOME 2); Linux-ядро 3.2.28 с TuxOnIce-патчами для более корректного перехода в спящий и ждущий режимы и с включением планировщика задач BFS, позволяющим увеличить отзывчивость пользовательских приложений; Синхронизации с пакетной базой Debian testing по состоянию на 7 ноября; Через стандартный репозиторий доступна быстрая установка свежих версий проприетарных видеодрайверов AMD и NVIDIA. По умолчанию вместо OpenOffice.org поставляется LibreOffice 3.5.4; Обновлены версии программ: X.Org 7.7, GNU Iceweasel 16.0.2, GParted 0.12.1, Empathy 3.4.2.3, VirtualBox 4.1.18, Chromium 22.0, Glibc 2.13, GIMP 2.8.2, VLC 2.0.4, Adobe Flash 11.2
