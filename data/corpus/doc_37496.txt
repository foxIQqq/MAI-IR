Вышли платформонезависимые мобильные фреймворки Apache Cordova 3.0 и PhoneGap 3.0

Увидел свет Apache Cordova 3.0, фреймворк для создания мобильных приложений, продолжающий развитие платформы PhoneGap, после передачи проекта компанией Adobe в руки фонда Apache. Одновременно компания Adobe представила основанный на единой кодовой базе с Apache Cordova продукт PhoneGap 3.0, функционально идентичный Apache Cordova 3.0. Apache Cordova позволяет создавать универсальные мобильные приложения, работающие на различных мобильных платформах, с использованием стандартных web-технологий (HTML5, CSS3 и JavaScript). Использование Apache Cordova в качестве базового фреймворка, позволяет создавать приложения, функционирующие на широком спектре мобильных платформ, включая Tizen, webOS, Android, iOS, Blackberry, Samsung Bada и Windows Phone. Процесс установки и внешний вид таких приложений ничем не отличается от программ, созданных с использованием родного для каждой мобильной платформы инструментария. Через специальную прослойку поддерживается использование специфичных для каждой платформы функций, а также доступ к API для взаимодействия с оборудованием, телефонным стеком, адресной книгой, GPS, звуковой подсистемой, камерой и другими компонентами мобильных платформ. Через создание плагинов поддерживается добавление дополнительной функциональности и расширение API. Основные улучшения, добавленные в Apache Cordova 3.0: Новая архитектура создания плагинов, позволяющая обеспечить максимальную производительность приложений при минимальном размере, за счёт подключения только необходимых API и функций автоматической установки и удаления плагинов; Улучшенный инструментарий, позволяющий сгладить различия между целевыми мобильными платформами за счёт использования промежуточной прослойки Cordova-CLI для сборки, развёртывания и управления создаваемыми мобильными приложениями. Cordova-CLI унифицирует доступ к специфичным для каждой мобильной платформы утилитам, что избавляет от необходимости установки SDK и изучения особенностей каждой из платформ для которых производится сборка приложения; Поддержка новых платформ: iOS6, Windows Phone 8 и BlackBerry 10. В ближайшее время ожидается поддержка Firefox OS и Ubuntu Touch; Новые API: InAppBrowser и Globalization.
