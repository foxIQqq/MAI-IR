Вышла вторая бета-версия Ubuntu 12.10

Доступна для тестирования вторая бета-версия Ubuntu 12.10 "Quantal Quetzal". Релиз запланирован на 18 октября. Загрузочные iso-образы с тестовой версией Ubuntu 12.10 можно получить на данной странице. CD и DVD образы десктоп-редакции Ubuntu 12.10 объединены в единый образ размером 751 Мб, пригодный для установки на USB Flash и DVD. По сравнению с первой бета-версией, в Ubuntu 12.10-beta2 отмечены следующие улучшения: В состав включено обновление пользовательской оболочки Unity 6.6, в которой улучшены средства управления окнами и рабочим столом. При переключении между несколькими окнами одного приложения для увеличения наглядности окна подвергаются адаптивному масштабированию. Обеспечено корректное следование декораций окна выбранной визуальной теме. Добавлена поддержка перегруппировки пиктограмм на боковой панели запуска приложений (иконки теперь можно свободно перетаскивать мышью). Расширены возможности линзы для выборки видео (например, опциональная поддержка поиска в YouTube) и улучшена работа предпросмотра в линзах (например, предпросмотр появился в линзе Gwibber); В Unity добавлена линза для вывода контекстных предложений о покупке музыки и видео в магазинах Amazon и Ubuntu One. Подобные коммерческие предложения будут выводиться в специальной секции "Дополнительные предложения", выводимой при выполнении операции глобального поиска в Dash. Пользователь получит возможность не только найти локально хранящийся контент, но и купить контент из online-архивов музыки и видео. Кроме того, по умолчанию на панель будут добавлены ярлыки приложений для покупки в магазинах Amazon и Ubuntu One. Отдельно отмечается два важных момента: Во-первых, в сторонние сервисы (в настоящее время только Amazon) данные о поисковых запросах будут передаваться в обезличенном виде через транзитный сервер Canonical, что позволит не допустить отслеживание предпочтений конкретных пользователей. Запросы с машины клиента на сервер Canonical будут передаваться в зашифрованном виде. Во-вторых, в блоке настроек управления приватными данными будет предусмотрена специальная настройка, дающая возможность отключить функцию обращения к внешним ресурсам при поиске в Dash; Ядро Linux обновлено до версии 3.5.4; GNOME обновлён до версии 3.5.92 (некоторые компоненты обновлены до версии 3.6); По умолчанию активирован стек для обеспечения работы людей с ограниченными возможностями; В образы с Ubuntu Cloud добавлена поддержка кнопки быстрого запуска. Сформированы нативные образы для Amazon AWS RDS (rdscli), Identity Management(iamcli), Elastic LoadBalancing (elbcli), ElastiCache (elasticache) и CloudWatch (moncli); В инсталлятор, используемый в Kubuntu, интегрированы функции задействования шифрованных разделов и поддержка LVM (для Ubuntu указанные возможности были представлены в прошлом тестовом выпуске); В Edubuntu при выборе LTSP по умолчанию задействован классический режим GNOME (Gnome Fallback session); Продолжен перевод дистрибутива на использование по умолчанию Python 3.2 в десктоп-сборке 12.10 (на время тестирования Python 2 пока остаётся в поставке до момента завершения портирования всех приложений);
