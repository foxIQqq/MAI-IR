Выпуск растрового графического редактора Krita 4.3.0

Представлен релиз растрового графического редактора Krita 4.3.0, развиваемого для художников и иллюстраторов. Редактор поддерживает многослойную обработку изображений, предоставляет средства для работы с различными цветовыми моделями и обладает большим набором средств для цифровой живописи, создания скетчей и формирования текстур. Для установки подготовлены самодостаточные образы в формате AppImage для Linux, экспериментальные APK-пакеты для ChromeOS и Android, а также бинарные сборки для macOS и Windows. Основные улучшения: Добавлена возможность выноса центральной области с холстом за пределы текущего окна и размещения в отдельном окне. Например подобный режим можно использовать при наличии в системе двух разных мониторов - изображение можно разместить на большом экране, а инструменты на маленьком. Расширены инструменты для создания анимации. В диалог "Render Animation" добавлена опция, позволяющая экспортировать только уникальные кадры анимации. Сам диалог отделён от диалога "Export" и больше не конфликтует с ним при вызове. Добавлена возможность привязки горячих клавиш для выбора предыдущего или следующего слоя одного уровня, что упрощает работу с одной группой слоёв. Повышена эффективность кэширования анимации, позволившая добиться более плавного и согласованного воспроизведения. Решены проблемы с редактированием скрытых слоёв в режиме активной изоляции и проблемы с индикацией текущих кадров на шкале времени при загрузке нового документа. Функция изоляции слоя ("Isolate Layer") для более точного отражения сути переименована в функцию изоляции активного слоя ("Isolate Active Layer"). Добавлен новый набор пресетов кистей с эффектом акварели. Для любителей пиксельной графики в фильтре "Gradient Map" добавлен режим настройки промежуточных цветов, используемых в качестве шаблона дизеринга или для ограничения цветов ближайшим запрещённым цветом. В фильтре "Gradient Map" также устранены утечки памяти. Добавлен фильтр сопоставления с палитрой (Palettize), который напоминает "Gradient Map", но использует палитру для определения цветов. Новый фильтр также поддерживает дизеринг. В состав включено несколько новых шаблонов, хорошо подходящих для использования вместе с новым фильтром. Предложен фильтр "High Pass", который можно применять для повышения чёткости изображений через наложение фильтрующего слоя. Фильтр может быть полезен на последнем этапе при загрузке изображений в социальные сети для компенсации появления размытости из-за масштабирования на сайтах для хостинга изображений. Все фильтры управления резкостью и размытостью адаптированы для корректной работы с изображениями с непрозрачным фоном. В фильтр HSV добавлена поддержка работы с изображениями в оттенках серого. В фильтрах определения краёв и нормализации высоты устранены артефакты линий, похожие на лестницу. В фильтре размытия движущихся объектов (motion blur) уменьшено число артефактов, а в фильтре обычного размытия (blur) обеспечен правильный расчёт коэффициента соотношения сторон. В системе слоёв возобновлена работа функции разделения каналов. Повышена производительность применения стилей. В диалоге разделения изображения добавлена поддержка разделения по направляющим и предоставлен удобный предпросмотр. Повышена стабильность работы функции клонирования слоёв, добавлен диалог для изменения источника клонируемого слоя (клонированный слой обрабатывается как экземпляр исходного слоя и теперь этот исходный слой можно поменять). Обработчики действий в инструментах, таких как увеличение/уменьшение размера кисти, теперь создаются до того, как изображение загружено, что позволяет разместить их на панели. Расширен API для написания дополнений на языке Python. В класс View добавлен метод setDocument. Действия, создаваемые в Python-дополнениях, теперь загружаются до компоновки меню и панелей инструментов. В цветных (RGBA) кистях появилась возможность раздельной настройки непрозрачности и яркости, что позволяет получить текстуру, напоминающую масляный или акриловый импасто. Производительность движка пиксельных кистей увеличена приблизительно на 20%. Добавлена возможность указания нескольких измерений при экспорте кисти в формате GIH (GIMP Image Hose), например, можно сохранить несколько рядов кистей, каждый из который может случайно размещаться по горизонтали и увеличивается по вертикали. В системе обработки цветов добавлена возможность загрузки и отображения палитр, включающих более 40196 столбцов. Значительно улучшена и упрощена работа в интерфейсе выбора цветов. В рамках проекта по полной переработке системы отката изменений реализована панель управления снапшотами (Snapshot Docker), позволяющая сохранять определённые стадии работы и переключаться между ними; Добавлен инструмент магнитного выделения (Magnetic Selection), упрощающий выделение произвольных областей за счёт автоматического выравнивание точек по границе выделяемой области. В инструменте создания градиентов добавлены новые режимы: cпираль, обратная спираль и билинейный режим. Расширены инструменты заливки и непрерывного выделения (Magic Wand), в которых для определния области заливки или выделения появилась возможность выбора между всеми слоями, текущим слоем и помеченными слоями, что может быть удобным при рисовании многослойных штриховых картинок. Значительно ускорено создание выделенных областей. Проведена работа по повышению надёжности сохранения файлов - после сохранения теперь проверяется размер и время, а также выполняется контрольное открытие файла сразу после его сохранения. Добавлена опция для открытия изображения в отдельном слое внутри ранее загруженного изображения. Добавлена опция для сохранения изображений в формате "kra" с кадрированием всех слоёв по заданному размеру изображения (по умолчанию данные в слоях за видимой границей продолжают сохраняться). Предложен второй тестовый выпуск Krita для Android, в котором появилась поддержка технологии Samsung Air для рисования с использованием пера ввода S Pen, поддерживаемого в таких устройствах, как Samsung Tab S6, Note 9 и Note 10.
