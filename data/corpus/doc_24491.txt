Вышел GNOME 3 фреймворк Zeitgeist 0.3.0

Разработчики GNOME выпустили экспериментальный релиз Zeitgeist 0.3.0, фреймворка для организации работы с документами и пользовательской информацией в GNOME 3.0. Сообщается, что Zeitgeist 0.3 является последним нестабильным выпуском, следующий релиз Zeitgeist 0.4 будет иметь статус стабильного. По сравнению с версией 0.2 в новом выпуске переработан D-Bus API, представлен API для разработки дополнений и реализация клиентского API для языка Python. Для организации семантических связей модуль Ontology переведен с кодовой базы XESAM на Nepomuk. Zeitgeist представляет собой систему для организации доступа пользователя к документам, производящую отслеживание и журналирование пользовательской активности, направленное на то, чтобы позже забывчивый пользователь смог найти URL в истории браузера или отыскать ранее сохранённые файлы. Ключевым компонентом системы является демон регистратор событий, хранящий данные в SQLite и обеспечивающий доступ к накопленной информации через D-Bus.
