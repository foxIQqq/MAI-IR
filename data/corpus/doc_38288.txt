Обновление Python 2.6.9

Представлен корректирующий выпуск Python 2.6.9, последний в ветке 2.6.x. В выпуске 2.6.9 устранено 7 уязвимостей, среди которых исчерпание памяти через обработку очень длинных строк в функции readline, возможность организации выполнения кода через tkinter и проблема в SSL-модуле с обработкой символов с нулевым кодом в поле subjectAltName. Вся активность по поддержанию выпусков 2.6.x полностью прекращена, обновления, в том числе для критических проблем безопасности, для данной ветки выпускаться больше не будут. Пользователям Python 2, которые по тем или иным причинам не могут перейти на Python 3, рекомендуется спланировать переход на Python 2.7. Прекращение поддержки ветки 2.7.x, последней в серии 2.x, ожидается в 2015 году.
