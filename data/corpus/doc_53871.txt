Выпуск видеоредактора Pitivi 2020.09

После двух лет разработки доступен релиз свободной системы нелинейного видеомонтажа Pitivi 2020.09, предоставляющей такие возможности, как поддержку неограниченного числа слоёв, сохранение полной истории операций с возможностью отката, отображение эскизов на шкале времени, поддержку типовых операций обработки видео и звука. Редактор написан на языке Python с использованием библиотеки GTK+ (PyGTK), GES (GStreamer Editing Services) и может работать со всеми форматами звука и видео, поддерживаемыми в GStreamer, включая формат MXF (Material eXchange Format). Код распространяется под лицензией LGPL. Проектом задействована новая схема наименования выпусков с нумерацией "год.месяц". Следом за версией 0.999 опубликован не ожидавшийся релиз 1.0, а выпуск 2020.09. Также изменён подход к разработке - созданы две ветки: "stable" для формирования стабильных выпусков и "development" для приёма и проверки новой функциональности. В процессе длившейся с 2014 года стадии стабилизации перед выпуском 1.0 в основной состав принимались только критически важные изменения, но за бортом оставались многие интересные возможности. В выпуск Pitivi 2020.09 вошла большая порция новшеств, разработанных студентами в рамках программ Google Summer of Code, проводимых с 2017 года. Для стабилизации указанных новшеств задействовано unit-тестирование и рецензирование. Основные новшества: Стабилизирована и достигла версии 1.0 библиотека GStreamer Editing Services (GES), лежащая в основе Pitivi. Добавлена поддержка плагинов для расширения функциональности Pitivi. Добавлен плагин для управления из консоли. Реализован механизм для реализация собственных интерфейсов для различных эффектов, который можно использовать вместо автоматической генерации интерфейса. Подготовлены отдельные интерфейсы для эффектов frei0r-filter-3-point-color-balance и прозрачности. Добавлен новый экран с приветствием запуска приложения, пришедший на смену диалогу Welcome и позволяющий сразу перейти к недавно открытым проектам. Добавлена возможность создания вложенных шкал времени при импортировании файлов XGES. Добавлена поддержка размещения меток на шкале времени. Полностью переделано оформление библиотеки эффектов. Добавлена возможность закрепления часто используемых эффектов для ускорения их выбора. Упрощён процесс добавления эффектов. Добавлена возможность работы одновременно с несколькими эффектами. Переработана медиабиблиотека, в которой появилась возможность использования разных представлений. Переработан диалог рендеринга. Обеспечено восстановление состояния редактирования после повторного открытия проекта. В просмотрщике добавлена визуализации безопасных областей. Упрощено выравнивание клипов. Добавлены возможности отключения звука для всего слоя и скрытия всего слоя. Предложено интерактивное руководство для знакомства с программой новичков.
