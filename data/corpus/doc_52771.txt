Выпуск DXVK 1.6.1, реализации Direct3D 9/10/11 поверх API Vulkan

Сформирован выпуск прослойки DXVK 1.6.1, предоставляющей реализацию DXGI (DirectX Graphics Infrastructure), Direct3D 9, 10 и 11, работающую через трансляцию вызовов в API Vulkan. Для использования DXVK требуется наличие драйверов с поддержкой API Vulkan 1.1, таких как AMD RADV 18.3, NVIDIA 415.22, Intel ANV 19.0 и AMDVLK. DXVK может применяться для запуска 3D-приложений и игр в Linux при помощи Wine, выступая в качестве более высокопроизводительной альтернативы встроенной в Wine реализации Direct3D 11, работающей поверх OpenGL. Основные изменения: Добавлена возможность экранирования значений настроек при помощи кавычек, например d3d9.customDeviceDesc = "ATi Rage 128"; Добавлена опция dxgi.tearFree для явного включения защиты от мерцания при отключенном Vsync; Реализована функциональность DXGI, необходимая для некоторых модов SpecialK; Устранена порция ошибок, приводящих к нарушению отрисовки или крахам при использовании Direct3D 9; Устранены ошибки проверки поддержки Vulkan на системах с видеокартами NVIDIA; Исправлена ошибка в скрипте настройки, который не работал с Wine 5.6; Решены проблемы с отрисовкой и крахами, проявляющиеся в играх "Blue Reflection", "Battlefield 2", "Crysis", "Half-Life Alyx", "L.A. Noire", "Prince of Persia", "Yooka-Laylee and Impossible Lair"; Улучшена производительность игры Heavy Rain на GPU NVIDIA.
