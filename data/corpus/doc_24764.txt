Релиз программы для управления фотографиями digiKam 1.0.0

Проект по созданию программы для управления коллекцией фотографий digiKam отметил переход в фазу зрелости выходом релиза 1.0.0. Для загрузки доступны как пригодные для сборки в большинстве Linux дистрибутивов исходные тексты, так и сборки для MacOS X и Windows. По сравнению с digiKam 0.10 в новой версии можно отметить следующие улучшения: Исправлено 485 ошибок; Добавлен мастер первоначальной настройки (First Run Assistant), после первого запуска digiKam задает несколько наводящих вопросов на основе которых формируются базовые настройки, оптимизированные для конкретного пользователя; Добавлен менеджер очередей автоматизированного выполнения заданий, позволяющий осуществить такие типовые операции, как преобразование форматов, устранение эффекта красных глаз и изменение размера для группы файлов. Из новых фильтров для автоматической обработки можно обратить внимание на утилиту для устранения шумов на фотографиях при помощи вейвлетов; Добавлена поддержка формата PGF (Progressive Graphics File), обеспечивающего эффективное сжатие изображений без потерь за счет использования основанного на вейвлетах алгоритма; Полностью переписан код для управления цветностью, добавлена поддержка предварительного просмотра; В комплект включен инструмент для плавного масштабирования фотографий. На фотографии выделяются объекты (люди, дома и т.п.), изменение масштаба которых может нарушить их пропорции, далее осуществляется неравномерное изменение масштаба по вертикали или горизонтали, при этом сама фотография становится шире или выше, но выделенные объекты сохраняют свои пропорции; Добавлена возможность формирования шаблонов для базовых мета-данных, таких как контакты, место и IPTC-данные. Добавлен шаблон для быстрой установки метки с авторскими правами. В настройках также можно определять методы фильтрации по тэгам, привязанным к Exif, Makernotes, Ipt и Xmp мета-данным; Улучшена поддержка многоязычности, например, поле автора каждой дополнительной заметки к изображению можно заполнять на разных языках. На разных языках можно заполнять аннотации и тэги в правой панели; Добавлен новый инструмент для создания анимированых баннеров; Реализовано диалоговое окно со статистикой базы данных; Для кэширования эскизов фотографий вместо ~./thumbnails теперь используется новая БД, эскизы при этом сохраняются в формате PGF. Кроме того, эскизы в любом месте программы можно сворачивать в док; В панель геопозиционирования интегрирована поддержка сервиса OpenStreetMap; Утилиту для автоматического переименования файлов AdvancedRename теперь можно вызывать из интерфейса digiKam (AlbumUI, CameraUI, BQM); Действие инструмента для выявления дубликатов изображений теперь может быть ограничено альбомом или группой тэгов; Во встроенном редакторе изображений появилась опция для быстрого переключения на другое цветовое пространство. В конвертер черного и белого добавлены дополнительные цветовые фильтры для синего и желто-зеленного. Добавлен инструмент для изменения контрастности через симулирование HDR изображений (тоновый маппинг); В CameraGUI добавлена возможность просмотра истории операций с камерой.
