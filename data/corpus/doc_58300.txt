Доступна платформа Android TV 13

Спустя четыре месяца после публикации мобильной платформы Android 13 компания Google сформировала редакцию для умных телевизоров и телеприставок Android TV 13. Платформа пока предлагается только для тестирования разработчиками приложений - готовые сборки подготовлены для телеприставки Google ADT-3 и эмулятора Android Emulator for TV. Публикация обновлений прошивок для потребительских устройств, таких как Google Chromecast, ожидается в 2023 году. Основные новшества, специфичные для Android TV 13: В API InputDevice добавлена поддержка разных раскладок клавиатуры и возможность привязки к физическому расположению клавиш для обработки нажатий независимо от активной раскладки. На внешних клавиатурах теперь можно использовать раскладки для разных языков. API AudioManager расширен возможностью упреждающего определения атрибутов активного звукового устройство и их использования для выбора оптимального формата не переходя к воспроизведению. Например, приложение теперь может определить устройство, через которое будет передаваться звук и поддерживаемые им форматы на стадии до создания объекта AudioTrack. Предоставлена возможность изменения разрешения и частоты обновления кадров для передающих устройств, подключённых через HDMI. Улучшен выбор языка для передающих HDMI-устройств. В API MediaSession обеспечена обработка изменения состояния HDMI, что может быть использовано для экономии энергопотребления компактными TV-адаптерами (TV Dongle) и другими передающими HDMI-устройствами, и приостановки воспроизведения содержимого в ответ на изменение состояния. В AccessibilityManager добавлен API для организации выдачи пользователям с ограниченными возможностями звуковых описаний, в соответствии с их предпочтениями. Добавлены общесистемные настройки включения звуковых описаний в приложениях. Проведена работа по улучшению управления питанием при переходе в ждущий режим с низким энергопотреблением. В настройках конфиденциальности отражено состояние аппаратных выключателей звука. Обновлён интерфейс управления ассистентом доступа к микрофону на пульте дистанционного управления. Предложен фреймворк для взаимодействия с TV-тюнерами Tuner HAL 2.0, в котором реализованы оптимизации производительности, обеспечена работа со сдвоенными тюнерами и добавлена поддержка спецификации ISDB-T Multi-Layer. Добавлен фреймворк для применения в области интерактивного телевидения, оформленный в виде расширения к TIF (Android TV Input Framework).
