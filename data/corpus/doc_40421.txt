Подготовлен неофициальный PPA-репозиторий клиента uTox, свободной замены Skype

Для пользователей Ubuntu доступен для использования неофициальный PPA-репозиторий с uTox, основным графическим клиентом для распределённой системы обмена сообщениями Tox, которая развивается в качестве свободной альтернативы Skype. Программа уже поддерживает чат, голосовую и видеосвязь, передачу файлов и организацию телеконференций. На днях была добавлена возможность работы через анонимную сеть Tor. Для установки uTox в Ubuntu теперь можно использовать команды (сборки для других дистрибутивов можно загрузить здесь): sudo add-apt-repository ppa:v-2e/tox sudo apt-get update sudo apt-get install utox
