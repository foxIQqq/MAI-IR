Релиз библиотеки LibVirt 1.2.16 для управления виртуальными окружениями

Компания Red Hat представила релиз проекта libvirt 1.2.16, в рамках которого развивается библиотека и набор инструментов для унифицированного локального и удаленного управления виртуальными окружениями. Поддерживаются системы виртуализации Xen, QEMU/KVM, LXC, OpenVZ, User Mode Linux, VirtualBox, VMware ESX/Workstation/Player, Hyper-V, Virtuozzo (драйвер parallels, поддерживается управление и контейнерами и виртуальными машинами), IBM PowerVM, BHyve. Статус поддержки различных систем виртуализации можно посмотреть в данной таблице. С момента выхода прошлой версии (1.2.15) добавлено около 150 изменений. Наиболее значительные изменения в Libvirt 1.2.16: Добавление поддержки нового устройства PCI-Serial В API добавлена поддержка нового вызова virDomainSetUserPassword для изменения пароля пользователя Добавлена возможность изменения номера порта ввода-вывода VMWare Исправлено около семидесяти дефектов в разных компонентах LibVirt: storage, virsh, qemu, parallels, network, xenconfig, lxc, tools Сделаны улучшения в следующих компонентах: internal, virsh, polkit, libxl, util, daemon, parallels, qemu, xenconfig, interface В дополнение к существующим зеркалам исходного текста на repo.or.cz и gitlab.com было добавлено зеркало на Github. Зеркала работают только в режиме чтения.
