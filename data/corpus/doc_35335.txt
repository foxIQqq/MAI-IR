Релиз Skype 4.1 для Linux (дополнено)

Увидел свет стабильный релиз Skype 4.1 для платформы Linux. Указанная версия является вторым значительным выпуском после поглощения бизнеса Skype компанией Microsoft. Возможности версии для Linux близки к функциональности клиента Skype для других платформ, в частности, доступна организация конференц-связи, перенаправление звонков, отправка SMS на мобильные телефоны, организация совместного доступа к рабочему столу, осуществление звонков на стационарные номера, выполнение видеовызовов, отправка файлов и т.п. Сборки Skype 4.1 подготовлены для Ubuntu 10.04/12.04 (32-битная версия в отдельных пакетах для 32-бит и 64-бит систем), Debian 7 (32-битная версия в отдельных пакетах для 32-бит и 64-бит систем), Fedora 16+ (32-битная версия) и openSUSE 12.1+ (32-битная версия). В качестве системных требований приводятся следующие версии программ: Qt 4.7.0, D-Bus 1.0.0, libasound2 1.0.18, а также опционально BlueZ 4.0.0 и PulseAudio 1.0. При установке некоторыми пользователями Ubuntu отмечается вывод ошибки из-за проблем с зависимостями. Указанная проблема решается предварительным удалением прошлой версии Skype. В новой версии, кроме обычной опции входа в Skype, пользователю предоставляется дополнительная возможность подключения к Skype через аккаунт Microsoft (Windows Live Messenger). После привязки к аккаунту в Windows Live становятся доступными средства для импорта контактов из аккаунтов Windows Live Messenger, Xbox, Hotmail и Outlook.com. Кроме того, у пользователя Skype теперь появилась возможность прямого общения с пользователями сервиса Windows Live Messenger. Дополнение: На сайте Skype опубликован обзор изменений: Возможность установки разных звуковых потоков для уведомлений; Поддержка отображения панели набора номера в процессе звонка или в момент проведения видеоконференции. Панель появляется при нажатии клавиши "D", даже если на текущем экране нет кнопки показа панели; Обеспечена возможность выхода из полноэкранного режима звонка через нажатие ESC; Во время звонка теперь выводятся уведомления о непрочитанных сообщениях; Разделено представление между локальным и удалённым переводом звонка в режим ожидания; Обеспечена возможность одновременного запуска нескольких копий Skype через указание опции "--secondary" при запуске дополнительных экземпляров программы; Полностью реализована поддержка команды "/golive" для создания конференций; Поддержка использования Skype URI на web-сайтах, при помощи которых можно инициировать звонок или чат через нажатие ссылки на web-странице; Проведена работа по улучшению качества звука и видео; Исправлено большое число ошибок, увеличена стабильность приложения.
