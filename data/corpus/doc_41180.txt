Браузер Opera 26 вышел с поддержкой Linux

Спустя полтора года после перехода на браузерный движок Chromium и прекращения формирования выпусков для Linux, компания Opera Software представила браузер Opera 26, который доступен в сборках и для платформы Linux. Opera 26 является первым стабильным выпуском новой редакции Opera, в который возвращена поддержка Linux. В отличие от Opera 12.16, новая версия доступна только в форме 64-разрядной сборки для Ubuntu Linux. Сообщается, что несмотря на то, что официально поддерживается только Ubuntu Linux, правила распространения пакета позволяют адаптировать и использовать его в любых других дистрибутивах. Например, пользователи Arch Linux уже сформировали неофициальный пакет. Для обновления с прошлой стабильной ветки Opera 12.16 следует явно установить пакет opera-stable (ранее предлагался пакет "opera"), который уже можно найти в Ubuntu Software Center. Автоматически уже установленные экземпляры Opera 12.16 обновлены не будут. После установки Opera 26 будет активирован репозиторий для установки обновлений, который обеспечит автоматическую загрузку и установку новых выпусков. Версия для Linux содержит все значительные возможности, присутствующие в выпусках для Windows и OS X. Из ограничений можно отметить отсутствие поддержки Flash и NPAPI-плагинов в сборке для Linux. Для использования Flash-плагина следует воспользоваться PPAPI (Pepper API) плагином Flash, для этого можно установить Google Chrome или пакет pepperflashplugin-nonfree. Встроенная поддержка видеокодека H.264 работает только при наличии FFmpeg 2.3 или более новой версии. Так как FFmpeg не входит в состав актуальных выпусков Ubuntu, следует воспользоваться одним из неофициальных PPA-репозиториев. Главные отличия от версии 12.16: Переход с собственного движка Presto на браузерный движок Blink (форк WebKit) и JavaScript-движок V8, развиваемые в рамках проекта Chromium. Для запуска плагинов, таких как Adobe Flash, предлагается поддержка PPAPI вместо NPAPI. Новый API для разработки дополнений, основанный на API Chrome и позволяющий использовать дополнения, созданные для Chrome; Полностью переработанный пользовательский интерфейс. Проведение чистки расширенной функциональности, удалены такие компоненты, как боковая панель, почтовый клиент и интерфейс для чтения RSS. Интерфейс реализован с использованием графического стека Aura. Новая панель Discover, предназначенная для обнаружения нового контента в Сети и слежения за новостными лентами по интересам. При этом браузер сам формирует подборку источников в зависимости от выбранной тематики и производит фильтрацию популярного и востребованного контента; Переработанная экспресс-панель, которая тесно интегрирована с системой ведения закладок и комбинирует при выводе как ссылки для быстрого запуска, так и разделы системы закладок. Поддерживаются средства группировки, переименования и перемещения содержимого экспресс-панели, а также объединение ссылок по разделам; Для ускорения работы на низкоскоростных каналах связи или при необходимости экономить трафик, предусмотрена новая реализация режима Opera Turbo, переведённая на использование протокола SPDY. При включении режима Turbo обработка контента производится на стороне серверной инфраструктуры Opera Software с передачей данных клиенту в компактном виде, что позволяет снизить трафик и обеспечить комфортную работу при роуминге или при работе в 2G-сетях; Режим Tab Peek для быстрого предпросмотра вкладки при наведении курсора, без явного перехода на вкладку; Комбинированная адресная строка, в которой объединены поле для ввода адресов и форма для отправки поисковых запросов. Встроенный PDF-просмотрщик; Поддержка форматов H.264 и MP3 (в Linux через FFmpeg); В качестве поисковой системы по умолчанию выбран DuckDuckGo. Начиная с Opera 15 в интерфейсе вместо классической системы закладок была предложена система адаптивного поиска избранных сайтов, но после многочисленных пожеланий пользователей, классические закладки, допускающие группировку ссылок по разделам, были возвращены в Opera 25. В версии Opera 26 система закладок была расширена функцией обмена закладками с друзьями - достаточно выбрать одну или группу закладок и нажать кнопку Share, после чего будет подготовлена и помещена в буфер обмена специальная ссылка, открыв которую в браузере будут показаны выбранные пользователем закладки. Также появилась функция импорта закладок, истории помещений и настроек из других браузеров.
