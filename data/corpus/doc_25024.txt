Вышло крупное обновление системного ПО коммуникатора Nokia N900

Через непродолжительное время после выпуска подготовительного обновления, компания Nokia выпустила основанное на открытой Linux платформе Maemo 5 крупное обновление системного ПО для устройств Nokia N900 (версия ПО 2.2009.51-1 также известная, как PR1.1). Наиболее заметные изменения в этом релизе: Новые возможности и функциональность: Activesync теперь работает и с MS Exchange 2003 и корректно обрабатывает операцию перенаправления писем с аттачами (показывался неверный размер аттача (Exchange server 2003)). В состав включен копировщик PC Suite. Теперь через PC-Suite можно восстановить полную копию содержимого, включая Notes. Добавлена совместимость с некоторыми сим-картами, с которыми ранее были проблемы. Обновления (Maemo Update): Теперь при необходимости система установки обновлений cможет выполнить операцию обновления загрузчика (NOLO) и микропрограммы GSM/3G модема; Кэш значков (Icon cache) был ликвидирован с целью увеличения свободного места для обновлений ПО. Диспетчер приложений (Application Manager): проверка свободного места на разделах перенесена в apt-worker для более аккуратного результата проверки. Надежность: Улучшения скорости работы и надежности SGX (акселератор графики). Время работы и потребление энергии: Добавлен API для апплетов строки статуса, чтобы они могли определить когда они скрыты или невидимы. Исправления в таймере PulseAudio улучшают экономию энергии при старте и выходе из программ. Уменьшена нагрузка на СPU при записи видео. Изменены пороги обнаружения неактивности CPU и размер соответствующих задержек. Уменьшено потребление энергии сотовым модемом (CellMO); Ускорена работа браузера, убран ненужный отладочный вывод в последовательный порт. Уменьшено количество просыпаний, необходимых для работы компонентов Desktop, Connectivity, Сonversations и RSS Feed Reader. Добавлена фиксация данных на MMC карте при засыпании. Резервное копирование (Backup) теперь правильно работает в случае каталогов с парой симлинков. Поддержка установки запрета на всплытие виртуальной клавиатуры в полях ввода. Блокировка ввода: теперь экранный слайдер разблокировки ("swipe to unlock") не будет оставлять устройство включенным. Юзабилити: Работа кнопок регулировки громкости звука меняется на основании пути звука, а не ориентации экрана. Загрузка страницы и начало рендеринга теперь происходит более отзывчиво. Исправления стабильности и юзабилити профилей. Исправлен выбор, использование и восстановление рингтонов. Появились ярлыки для быстрого звонка с домашнего экрана (home). Возможность быстрой отсылки шаблонной SMS при отвергании входящего звонка. Возможно выключение звука входящего звонка путем переворота устройства экраном вниз. Скорость работы: X-сервер: улучшена скорость GLES приложений. Уменьшено потребление ОЗУ при съемке видео. Интерфейсы Phone-UI и Msg-UI всегда готовы к появлению. Увеличена скорость работы пользовательского интерфейса Application Manager. Приложение для Email работает быстрее с большими майлбоксами. Теперь копирование больших файлов с SD карты на внутреннюю MMC не тормозит работу системы. Навигация (OVI Maps): Ускорено вычисление маршрутов и улучшена скорость работы поиска. Связь с внешним миром: Исправлена работа профиля "BT Secure Simpue Pairing" с ОС Windows Vista. Теперь показывается индикатор прогресса при передаче файлов по Bluetooth. Улучшение стабильности WLAN соединений и поведения согласования WLAN при выходе за пределы покрытия точки доступа. Телефонные звонки: Добавлена поддержка ярлыка для вызова голосовой почты. Корректно отображается перенаправление звонка когда не установлено перенаправление по условию "занято". Улучшение отображения лога звонков. Аудио и видео: Управление громкостью кнопками "+" и "-" теперь работает и при проигрывании полноэкранных флеш-роликов. Улучшено поведение в случае испорченных файлов рингтонов. Аппаратные кнопки громкости звука всегда ориентированы одинаково. Интерфейс медиа-плеера теперь наглядно реагирует при нажатии на кнопки в интерфейсе проигрывания видео. Исправлено отсутствие ограничения сигнала с микрофона при громком голосе в VoIP вызовах. Рингтон проигрывается в наушники, bluetooth устройства и штатный громкоговоритель. Камера: Значение ISO теперь пишется в EXIF и при съемке с автоматической чувствительностью ISO. Тест "vignetting" у второй камеры исправлен, улучшена стабильность VGA камеры. Теперь запись информации об ориентации камеры в EXIF тегах происходит правильно. Налажена синхронизация звука и видео в снимаемом камерой видео. Батарея и заряд: Улучшена процедура заряда в случае использования маломощного источника тока на USB (например, заряд от usb-порта). USB: Исправлена проблема определения usb и монтирования карт памяти в случаях подключения и потом быстрого отключения usb-кабеля. Теперь можно переформатировать разрушенную eMMC карту. Исправлено зависание в случае когда копировался файл по usb и при этом происходило отключение от wi-fi сети. Приложения: В Policy-settings для сторонних приложений исправлен переход при перевороте экрана. По умолчанию ограничения для сторонних приложений теперь такие же как для программ от Nokia; Календарь: добавлен публичный API для color picker и API для установки и получения списка будильников. Виджет для десктопа теперь прозрачный. Поддерживается синхронизация оповещений о задачах. Просмтрщик картинок: генерация эскизов теперь происходит быстрее. Диспетчер приложений (Application Manager): каталоги Maemo Extras переименованы в maemo.org и теперь активны по умолчанию. Для категорий приложений в режиме загрузки применен Grid view. Добавлено API для применения фильтров при отображении пакетов. Платформа для разработки: Обновлено содержимое SDK и утилит. Обновлен графический установщик SDK. Новый пакет "sharing-dialog-dev" позволяющий отправлять файлы по Bluetooth и email. Новый пакет "wappushd-dev", документирующий интерфейсы для MMS. Исправления: Исправление уязвимости CVE-2009-2417 в curl. Исправлено множество ошибок о которых разработчикам сообщили пользователи.
