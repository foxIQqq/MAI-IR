Доступен дистрибутив ROSA Desktop Fresh KDE R7

Компания НТЦ ИТ РОСА представила выпуск дистрибутива ROSA Desktop Fresh KDE R7, основанного на платформе rosa2014.1. Для свободной загрузки подготовлены DVD-сборки, размером 1.9 Гб, оформленные в вариантах с KDE 4 и KDE Plasma 5 для платформ i586 и x86_64. Основные новшества: Изменен состав приложений в образе с KDE4: добавлены дополнительные мультимедийные-кодеки (в том числе h.265), программа для резервного копирования Kup и программа для работы с веб-камерой Kamoso. Из состава удалён KWallet. По умолчанию предлагается ядро Linux 4.1.15, в качестве опции доступны сборки ядра 4.2.x (x86_64, i586) и 4.3.x (x86_64, i586). Обновление версий программ: X Server 1.17.4, новые версии видеодрайверов (AMD Catalyst 15.9, NVIDIA 352.63, 340.96, 304.131, 358.16), Mesa 11.0.6, PulseAudio 7.1, Chromium 47.0, Firefox 43.0.4, Thunderbird 38.3, LibreOffice 4.4.7, Qt5 5.5.1, KDE Frameworks 5.18.0, GTK+3 3.16.7, FFmpeg 2.8.5, GStreamer 1.6.2. Исправлены ошибки при установке ROSA Linux с USB3-носителей. Значительно улучшена поддержка систем с видеокартами AMD и Intel. Улучшена поддержка USB-клавиатур ThinkPad. Изменена политика обновления ядра Linux - автоматические обновления теперь работают только внутри одной ветки ядра, переход на следующую ветку осуществляется вручную. Исправлена возможность раздачи интернета через Wi-Fi средствами NetworkManager. Исправлена невозможность переключения на виртуальную консоль при ошибке инициализации графической подсистемы. Исправлено поведение заставки при запуске-завершении работы системы. KLook теперь нормально воспроизводит видео с кодеком h.265. В Clementine добавлена возможность играть музыку целыми папками из контекстного меню в файловом менеджере.
