Релиз Wine 1.5.6

Вышел очередной экспериментальный релиз открытой реализации Win32 API - Wine 1.5.6. С момента выпуска версии 1.5.5 было закрыто 36 отчётов об ошибках. Основные изменения в новой версии: Автоматическая установка Mono (Open Source аналог .Net) в качестве дополнительного пакета; Добавлен апплет панели управления для настройки джойстика; Вывод bitmap изображений переведён на DIB движок; Добавлена поддержка вывода видео через DirectX (VMR-9); Начата реализация компилятора шейдеров для Direct3D; Исправлены проблемы компиляции под DragonFly BSD; Обновлены переводы; Исправлено множество ошибок.
