Релиз дистрибутива Bodhi Linux 2.2.0 с окружением на основе Enlightenment E17

Представлен релиз дистрибутива Bodhi Linux 2.2.0, построенного на пакетной базе Ubuntu 12.04. Графическое окружение в Bodhi Linux основано на оконном менеджере Enlightenment 17, в качестве файлового менеджера используется EFM, а браузера - Midori. Для упрощения установки дополнительных приложений, наиболее интересные программы из различных категорий, представлены в специальном web-каталоге. Размер загрузочного iso-образа 573 Мб. В качестве минимальных требований к оборудованию упоминается CPU 300 Mhz и 128 Мб ОЗУ. Новая версия примечательна включением релиза пользовательской оболочки Enlightenment E17, который можно опробовать загрузив дистрибутив в Live-режиме. Кроме 32- и 64-разрядных сборок для архитектуры x86, отныне также подготовлена сборка для платы Raspberry Pi и мобильных устройств на базе архитектуры ARM (hard float), например, для планшета Nexus 7. Для 32-разрядных систем x86 подготовлено два варианта сборок: с ядром Linux 3.2 (без включения режима PAE) и с ядром 3.7 (с включенным PAE). Для x86_64 доступна на сборка с ядром 3.7. Впервые сборки оформлены в гибридном виде и подходят как для копирования на USB-накопитель, так и на CD/DVD. Для пользователей планшетных ПК предусмотрен режим работы десктопа, оптимизированный для использования с сенсорных экранов (см. скриншот ниже).
