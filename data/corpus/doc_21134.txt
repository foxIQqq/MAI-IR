Вышел второй кандидат в релизы Linux дистрибутива Mandriva Linux 2009.1

Вышел второй кандидат в релизы Linux дистрибутива Mandriva Linux 2009.1 (Spring). Главные изменения RC2: завершение доработки новой визуальной темы, обновление версий KDE 4.2.2, GNOME 2.26 и Linux ядра 2.6.29. Кроме того, завершена вторая фаза интеграции технологии speedboot, позволяющей значительно сократить время загрузки дистрибутива - вначале грузятся компоненты экранного менеджера, позволяющего начать работать с системой, а затем в фоне загружаются дополнительные сервисы. Speedboot, если конфигурация системы позволяет (не совместимо с шифрованной ФС, программным RAID, сетевой аутентификацией), активируется автоматически. Тестовая версия доступна в виде "DVD Free" сборки (без проприетарных драйверов в комплекте) и двух LiveCD сборок с KDE и GNOME. Согласно планам разработки, финальный релиз намечен на 29 апреля. Новшества добавленные ранее: Обновление версий: Xfce 4.6, X.Org server 1.6, OpenOffice.org 3.0.1, Qt 4.5.0, Tcl/Tk 8.6. Python 2.6.1; В качестве легковесного окружения теперь используется LXDE, вместо icewm; Осуществлен редизайн системы управления безопасностью msec; В инсталлятор, в качестве дополнительной опции, добавлена возможность установки на ext4 раздел. В Diskdrake появилась возможность изменение метки дискового раздела без его переформатирования; Заблокирована возможность перезапуска X-сервера через последовательность клавиш Ctrl+Alt+backspace; В инсталляторе добавлена опциональная возможность использования файловой системы ext4. Но используемая версия загрузчика пока не поддерживает ext4, необходимо подождать обновления GRUB; Подготовлены гибридные iso-образы LiveCD ("One") сборки Mandriva, которые теперь подходят и для создания загрузочных USB FLash путем простого копирования содержимого ("dd if=mandriva-linux-one-2009.1-rc1-GNOME-int-cdrom-i586.iso of=/dev/sdX bs=8M"); В состав установочного диска включены два дополнительных пакета: 1. Модуль HDT (Hardware Detection Tool) из состава пакет Syslinux , позволяющий на раннем этапе загрузки сформировать полное описание состава текущего аппаратного обеспечения, до начала загрузки ОС. 2. Draksnapshot - инструмент для быстрого восстановления всей системы из бэкапа по состоянию на указанный момент времени.
