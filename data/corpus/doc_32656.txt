Вышел релиз Live-дистрибутива Grml 2011.12

Анонсирован релиз Live-дистрибутива grml 2011.12, основанного на пакетной базе Debian GNU/Linux и содержащего подборку программ для выполнения операций по обработке текстовых данных средствами пакета texttools и для выполнения работ, возникающих в практике системных администраторов (восстановление данных после сбоя, разбор инцидентов и т.д.). Размер iso-образа 315 Мб. Основные изменения: Поддержка загрузки с CD-ROM и USB Flash с использованием EFI; Поддержка mDNS (Multicast DNS) с использованием libnss-mdns и avahi-daemon; Поддержка NFS-клиента и автоматического запуска D-Bus; Утилита grml-debootstrap, позволяющая установить Debian, теперь может использоваться для непосредственной сборки готовых образов виртуальных машин; Инструментарий для сборки Live-системы Grml-live расширен в плане улучшения интеграции с Jenkins, который, например, теперь задействован для формирования ежедневных тестовых сборок; Проведена чистка конфигурации Z-Shell; Уменьшен размер компонентов X11, в качестве оконного менеджера осуществлён переход на fluxbox. Удалён idesk, вместо которого теперь используется стандартное меню fluxbox; Добавлена поддержка iSCSI Target через LIO, конфигурация осуществляется утилитой targetcli; В xterm по умолчанию используется шрифт Terminus, как и для виртуальных консолей; Обновлено ядро Linux до версии 3.1.6; Из пакета grml-network удалена поддержка модемов; Сборка осуществлена на пакетной базе Debian testing, а не на Debian unstable, как раньше; Новая визуальная тема для загрузчика GRUB.
