SparkleShare - новый свободный проект для создания online-хранилищ

Представлена вторая тестовая версия проекта SparkleShare, в рамках которого ведется работа по созданию свободного движка для организации собственных online-хранилищ, похожих на Dropbox. Хранение данных осуществляется в любом Git-репозитории, что делает SparkleShare универсальной системой, не требующей для работы развертывания специальной серверной инфраструктуры (например, можно использовать сервисы Gitorious и GitHub). По сути SparkleShare является Git-клиентом, оптимизированным для хранения и обмена файлами. Интерфейс SparkleShare-клиента написан с использованием Mono и библиотеки GTK+. Из особенностей программы можно отметить поддержку шифрования хранимых данных, что позволяет использовать публичные Git-репозитории не опасаясь возможной утечки информации. Программа может использоваться не только для хранения своих файлов, но и для организации обмена файлами с другими людьми. Использование Git позволяет задействовать для отслеживания изменений в файлах элементов версионного контроля, давая возможность отследить все изменения и при необходимости вернуться к состоянию определенного файла в прошлом.
