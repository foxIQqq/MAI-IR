Началось тестирование первой альфа-версии Ubuntu 12.04

Вышла первая альфа-версия будущего LTS-релиза Ubuntu 12.04 "Precise Pangolin", в которой проведена синхронизация пакетной базы с Debian Testing (не-LTS релизы базируются на Debian Unstable). Начиная с выпуска 12.04 время поддержки десктоп-редакции для LTS-выпусков Ubuntu будет увеличено с трёх до пяти лет. Загрузочные iso-образы с тестовой версией Ubuntu 12.04 можно получить на данной странице. Релиз запланирован на 28 апреля, после выхода еще одной альфа-версии и двух бета-выпусков. Кроме синхронизации пакетов с Debian Testing из изменений в первой бета-версии можно отметить: В качестве ядра Linux использована версия 3.2-rc3. С целью оптимизации поддержки пакетов с ядром generic и server для платформы amd64 решено использовать один набор настроек. Добавлены дополнительные драйверы для тачпадов ALPS, обновлён набор патчей seccomp, изменены некоторые настройки ядра (например, CONFIG_PANEL_DVI=y для omap, CONFIG_EXT2_FS=m (поддержка ext2 будет в виде модуля) и CONFIG_SATA_AHCI=y); Интегрированы новые версии программ: Firefox 9-beta, Thunderbird 9-beta, Shotwell 0.11.6 и LibreOffice 3.4.4; Вместо Banshee в качестве музыкального плеера по умолчанию задействован Rhythmbox. Поддержка UbuntuOne Music Store пока не реализована, так как не завершено портирование на GTK+ 3; Размер поставляемого по умолчанию iso-образа преодолел барьер в 700 Мб и составляет 703 Мб. Напомним, что принято решение увеличить размер образа до 750 Мб; Для увеличения совместимости с Debian группа, вхождение в которую позволяет пользователям выполнять привилегированные операции через утилиту sudo, изменена с "admin" на "sudo". Поддержка группы admin оставлена для обеспечения совместимости с прошлыми выпусками; В серверной редакции вместо squid 2.7 теперь используется пакет squid3. Cobbler теперь базируется на upstream-версии 2.2.2; Компоненты Openstack (nova, glance, swift, keystone, opestack-dashboard) основаны на релизе Essex 2; В качестве дисплейного менеджера по умолчанию в Lubuntu теперь используется Lightdm. О грядущих новшествах Ubuntu 12.04 можно прочитать в заметке про цели развития нового релиза и в сводной новости с обзором решений, принятых на саммите разработчиков Ubuntu.
