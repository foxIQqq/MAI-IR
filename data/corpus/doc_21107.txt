xPUD - минималистичный LiveCD Linux дистрибутив новой формации

Вышел новый минималистичный LiveCD Linux дистрибутив xPUD 0.8.9, iso-образ которого занимает всего 48 Мб, а время полной загрузки составляет 10 секунд. Дистрибутив предельно прост в эксплуатации, в нем нет ничего лишнего, только web-браузер Firefox, файловый менеджер, медиа-плеер gmplayer, терминал lilyterm, тектовый редактор leafpad, программа для просмотра фотографий gpicview, Bittorrent-клиент Transmission и средства для управления конфигурацией. Интерфейс пользователя Plate написан на XUL. Для установки дополнительных программ задействован специально разработанный механизм Opt-Get плагинов. Например, можно одним щелчком мыши установить OpenOffice.org, wine, дополнительные мультимедиа кодеки и проприетарные драйверы, skype, opera flash-plugin, smplayer, amarok, thunderbird, qemu, java и другие программы. Система конфигурирования поддерживает GUI интерфейсы для настройки сети (NetworkManager), резервного копирования, часов, звука, межсетевого экрана, параметров дисплея, устройств ввода. В интерфейсе предусмотрена возможность быстрой организации совместного доступа к файлам с нескольких машин. В дистрибутиве используется Linux ядро 2.6.28 с поддержкой Squashfs-LZMA и дополнительными патчами, разработанными в рамках проекта Moblin. Для построения собственных сборок, например для включения в поставку дополнительных приложений или быстрого создания интернет-киосков, подготовлена специальная система сборки mkxpud, сильно упрощающая создание нужных конфигураций.
