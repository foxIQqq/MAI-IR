Доступен серверный дистрибутив NethServer 7.6

Представлен релиз дистрибутива NethServer 7.6, предлагающего модульное решение для быстрого развёртывания серверов в небольших офисах или на предприятиях среднего размера. Дистрибутив основан на пакетной базе CentOS 7.6 и предоставляет web-интерфейс для управления доступными серверными компонентами. Размер установочного образа 1 Гб. Для ознакомления с возможностями интерфейса предоставляется online-демонстрация. Наработки проекта распространяются под свободными лицензиями. Пользователю предлагаются готовые модули для организации работы почтового сервера (Postfix, Dovecot, Amavis, ClamAV + web-клиент Roundcube), системы совместной работы (SOGo), межсетевого экрана (Shorewall), web-сервера (LAMP), файлового сервера и контроллера домена Active Directory (Samba), фильтрующего прокси (Squid, ClamAV и SquidGuard), VPN-сервера (OpenVPN, L2TP), облачного хранилища (ownCloud), систем обнаружения и предотвращения вторжений. Установка и введение в строй нужного сервиса производится в один клик и не требует знания особенностей настройки каждого серверного компонента. Типовые работы по администрированию можно выполнять через web-интерфейс. Основные новшества: Расширены средства резервного копирования. Добавлена поддержка создания нескольких резервных копий в разное время суток. Добавлены новые бэкенды для хранения бэкапов (duplicity, restic, rsync) и реализована опция для сохранения резервных копий в зашифрованном виде; Предложен для тестирования новый интерфейс пользователя, построенный на базе Cockpit и предлагающий более современное оформление. В базовом наборе модулей предоставляются инструменты для управления учётными записями, DNS, DHCP, FQDN, установки времени, создания резервных копий, настройки сети, применения TLS-шифрования, управления системой, установки приложений, управления хранилищами и сертификатами SSL; В базовую поставку добавлена система фильтрации спама Rspamd с предоставлением возможности отдельного сохранения отфильтрованных писем для анализа администратором (перенос в карантин); Расширены возможности модуля развёртывания инфраструктуры для организации совместной работы на базе платформы WebTop (Web Desktop). Добавлена реализация CalDAV/CarDAV-сервера (SabreDAV), обеспечена автоматическая синхронизация календаря-планировщика, добавлена поддержка голосовых и видео конференций, реализован оптимизированный для планшетов вариант интерфейса, добавлена возможность прикрепления вложений для событий, контактов и задач; Добавлена страница, отражающая активность блокировки попыток подбора паролей к сервисам в изолированных окружениях (используется Fail2Ban); Система глубокого инспектирования пакетов nDPI обновлена до выпуска 2.4, в котором увеличена производительность и повышена безопасность; Облачное хранилище Nextcloud обновлено до версии 14.0.4, а платформа для обмена сообщениями между разработчикам Mattermost до выпуска 5.5.0; В интерфейсе управления учётными записями упрощена операция удаления списка пользователей, улучшены возможности для определения срока действия паролей, контейнер с реализация контроллера домена обновлён до Samba 4.8.6.
