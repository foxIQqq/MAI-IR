Релиз CrossOver 12.0 для Linux и Mac OS X

Компания CodeWeavers выпустила релиз пакета Crossover 12.0, основанного на коде Wine 1.5.15 и предназначенного для выполнения программ и игр, написанных для платформы Windows. CodeWeavers входит в число ключевых участников проекта Wine, спонсирует его разработку и возвращает в проект около 98% всех новшеств, реализованных для своих коммерческих продуктов. По сравнению с прошлым выпуском отмечается более пяти тысяч изменений и улучшений, из которых можно отметить: Осуществлён переход на кодовую базу Wine 1.5.15 (прошлая версия была основана на Wine 1.4.1); Представлена технология "Mac Driver", обеспечивающая бесшовную интеграцию Windows программ в окружение Mac OS X, без необходимости запуска X Window System; Для платформы Linux проведена реструктуризация меню и осуществлён переход к одноуровневому меню, которое лучше работает в GNOME 3 и Unity. Улучшена работа под управлением композитных менеджеров и обеспечена возможность использования прозрачных и многослойных окон в запускаемых Windows программах, например, в MS Office и играх; Улучшена поддержка различных популярных программ, в том числе внесены многочисленные улучшения, связанные с поддержкой Microsoft Office; Добавлена поддержка новых игр, таких как Quicken 2013, World of Tanks и Guild Wars 2; Добавлена поддержка установки .Net 4.0. В базовую поставку включён пакет Wine-Mono с альтернативной реализацией .Net API; Во встроенном браузере появилась возможность загрузки и установки компонентов ActiveX; Добавлена поддержка Raw Input API, используемого в некоторых играх, желающих получить эксклюзивный доступ к мыши; В реализацию DirectSound добавлен новый модуль изменения частоты дискретизации, обеспечивающий поддержку звука высокого качества; Интегрирована поддержка субпиксельного сглаживания шрифтов.
