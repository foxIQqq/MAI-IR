Началось бета-тестирование открытого медиа-плеера Boxee

Разработчики кроссплатформенного медиа-плеера Boxee объявили о выпуске первой общедоступной бета-версии продукта. По сравнению с ранее доступной альфа-версией в новом выпуске переработан интерфейс пользователя, добавлена система установки закладок на интересные медиа материалы, заключены соглашения об интеграции поддержки сервисов TV.com, Blip.tv и IGN. Код Boxee основан на открытом проекте XBMC, расширенном в направлении создания дополнительных проприетарных компонентов, обеспечения возможности интеграции с социальными сетями и online-сервисами, расширении средств для упрощения интеграции и разработки внешних дополнений. Программное обеспечение поддерживает все популярные мультимедиа форматы и сервисы потокового видео и аудио вещания. Например в Boxee интегрирована поддержка таких ресурсов, как Netflix, MLB.TV, Comedy Central, Pandora, Last.fm, YouTube, Hulu и Flickr. Кроме того в платформу интегрированы средства для работы с социальными сетями, такими как Twitter или Facebook. Для загрузки файлов из сети в медиа центр интегрирован BitTorrent-клиент. Разработка платформы ведется на языке C++ с использованием Python плагинов. Сборки Boxee доступны для платформ Windows, Linux и MacOS X. Часть исходные текстов Boxee распространяется под лицензией GPL, но некоторые новые подсистемы, например, библиотека libboxee, доступны только в бинарном виде. Дополнительно оглашены подробности о начинке домашнего мультимедиа центра Boxee Box DM-380, готовящегося к выпуску компанией D-Link. Кроме ранее известных фактов в поддержке Wifi 802.11n и наличию портов HDMI, S/PDIF, Ethernet RJ-45, 2 x USB 2.0 и SD Card, сообщается, что устройство будет основано на платформе NVIDIA Tegra 2 (T20) с двухядерным CPU ARM Cortex A9, мощности которого достаточно для отображения 1080p HD видео. Вопреки ожиданиям, в устройстве не будет встроенного жесткого диска, но жесткий диск можно будет подключить через USB порт. Отдельно отмечается поставка в комплекте Boxee Box двухстороннего пульта дистанционного управления, с одной стороны имеющего обычные кнопки управления воспроизведением, а с другой - полноценную QWERTY-клавиатуру.
