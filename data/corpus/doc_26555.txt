Подведены итоги конкурса разработчиков расширений для мобильной версии Firefox

Представлен список победителей конкурса по разработке дополнений для мобильной версии Firefox (Firefox Mobile). В качестве приза авторам 10 лучших дополнений вручены смартфоны Nokia N900. Победители: Find In Page - подсветка на странице всех элементов, соответствующих указанному пользователем ключевому слову; Image Exposer - предоставляет возможность просмотреть список всех изображений на данной странице; Low Quality Flash - переводит все Flash-объекты в режим низкого качества, что позволяет повысить производительность при их просмотре на маломощных устройствах; Papaya - дополнение для обмена ссылками с друзьями через Jabber/Gtalk; Readability - вычищает перед отображением излишне нагроможденные страницы; Reporter - система оповещения разработчиков о возникших проблемах с работой браузера; Shareaholic - дополнение для публикации интересных ссылок в таких системах, как Facebook, Twitter, Email, Gmail, Reader, Bookmarks, Buzz, Evernote, Bit.ly и т.д. Speed Dial for Mobile - реализация системы визуальных закладок, похожим на Opera способом; Tabs Utility - позволяет автоматически перечитывать содержимое последней открытой вкладки, перетаскивать табы, выполнять группировку и сортировку по различным критериям; Where Is My Car - помогает найти место, где оставлена машина.
