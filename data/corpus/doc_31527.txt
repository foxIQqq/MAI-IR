Новые версии десктоп-дистрибутивов: Frugalware Linux 1.5, Parsix GNU/Linux 3.7 и Linux Mint 11 "LXDE"

Вышли новые версии нескольких специализированных десктоп-дистрибутивов Linux: Frugalware 1.5 - дистрибутив общего назначения, разработанный для опытных пользователей, не боящихся командной строки. Основное направление развития дистрибутива - максимальное упрощение. Концепция построения дистрибутива похожа на Slackware, в качестве пакетного менеджера задействован pacman из Arch Linux. При создании пакетов разработчики стараются публиковать программы как есть, избегая внесения дополнительных патчей. Размер установочного iso-образа - 4.3 Гб. В состав новой версии включены: ядро Linux 2.6.39.3, XOrg server 1.10.3, GNOME 3, KDE SC 4.6.5, XFCE 4.8, LibreOffice 3.4.2, Mozilla Firefox 5.0.1. Осуществлен переход на системный менеджер systemd и новую версию пакетного менеджера Pacman-g2. Реализована начальная поддержка порта для архитектуры ARM. Добавлена поддержка загрузки с использованием EFI и возможность использования таблицы разделов GPT. Parsix GNU/Linux 3.7 - пользовательский дистрибутив, построенный на пакетной базе Debian testing и продолжающий развитие LiveCD-дистрибутива KANOTIX. Размер загрузочного iso-образа составляет 947 Мб. В комплект включен экспериментальный установщик дистрибутива на USB-накопители. Загрузка с LiveDVD осуществляется при помощи GRUB 2, в качестве файловой системы задействована Squashfs 4.0 c Lzma методом сжатия (при установке на диск используется ext4). Графическое окружение построено на основе GNOME 2.32. В состав новой версии включены такие приложения, как Linux-ядро 2.6.37.6, GNOME 2.32.1, Xserver-xorg 1.7.7, GNU Iceweasel 5.0, Chromium 11.0.696.71, OpenOffice.org 3.2.1, GParted 0.8.1, Empathy 2.30.3, GIMP 2.6.8, CompizFusion 0.8.4, Synaptic 0.70.1, VirtualBox-OSE 4.0.4, VLC 1.1.3, xFarDic 0.11.7, Adobe flash 10.3. Через стандартный репозиторий теперь доступна быстрая установка проприетарных видеодрайверов AMD и NVIDIA. Linux-ядро поставляется с TuxOnIce-патчами для более корректного перехода в спящий и ждущий режимы и с включением планировщика задач BFS, позволяющим увеличить отзывчивость пользовательских приложений. Linux Mint 11 "LXDE" - редакция дистрибутива Linux Mint 11 с графическим окружением на базе LXDE. Дистрибутив основан на пакетной базе Ubuntu 11.04, дополненной рядом оригинальных приложений, упрощающих настройку и работу в системе начинающих пользователей. Размер iso-образа - 693 Мб. Минимальные требования к системе - 256 Мб ОЗУ. Новшества Linux Mint 11 "LXDE" в основном идентичны изменениям, представленным в основной редакции дистрибутива, познакомиться с которыми можно в прошлом анонсе.
