Выпуск Wine 1.7.1

Вышел очередной экспериментальный выпуск открытой реализации Win32 API - Wine 1.7.1. С момента выпуска версии 1.7.0 было закрыто 56 отчётов об ошибках. Наиболее важные изменения: Добавлена поддержка объектов типа "keyed event", которые появились в ядре Windows Vista; Реализован механизм синхронизации однократной инициализации (включая InitOnceExecuteOnce); Добавлена поддержка активации контекста для DLL, typelibs и COM классов; Реализована возможность загрузки 32-разрядных TypeLib в 64-разрядной среде; Множество улучшений для драйвера вывода OS X; Некоторые исправления для устройств последовательного порта;
