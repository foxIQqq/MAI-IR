Первый телефон на базе ОС Sailfish будет продемонстрирован в мае

В интервью финскому изданию Taloussanomat представители компании Jolla, основанной бывшими сотрудниками Nokia с целью разработки смартфонов, продолжающих развитие проекта MeeGo, рассказали о разработке проекта Sailfish с опережением графика, поэтому первое потребительское устройство на базе новой ОС планируется продемонстрировать уже в мае. Вскоре после презентации, в середине мая устройство станет доступно для предзаказов и будет опубликовано расписание выпусков. Отправка телефонов по предзаказам ожидается не раньше второй половины 2013 года. В рамках проекта Sailfish развивается операционная система с открытым исходным кодом, которая опирается на развиваемого сообществом потомка платформы MeeGo - проект Mer, и делает ставку на отличающийся от Android и iOS пользовательский интерфейс. Финская фирма Jolla получила от Nokia внушительный пул патентов на MeeGo-технологии. Для создания мобильных приложений поддерживается использование технологий HTML5 и Qt/QML. Присутствует потенциальная совместимость с нынешними программами для платформы MeeGo. В платформу интегрированы прослойки для обеспечения бесшовного запуска приложений, написанных для платформы Android, а также мобильных программ, базирующихся на web-технологиях (HTML5, JavaScript и CSS) и созданных с использованием API Firefox OS или API Tizen. Напомним, что поступление в продажу первых телефонов на базе Firefox OS ожидается в июне (geeksphone.com начнёт отправку моделей для разработчиков на этой неделе), устройства на базе Tizen ожидаются в конце лета, прототип телефона с Ubuntu будет показан в октябре, а серийный выпуск начнётся в начале 2014 года.
