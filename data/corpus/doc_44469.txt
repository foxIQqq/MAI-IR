Выпуск дистрибутива Calculate Linux 15.17

Доступен релиз отечественного дистрибутива Calculate Linux 15.17, построенного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в нескольких редакциях, собранных для архитектур i686 и x86_64: Calculate Linux Desktop с рабочим столом KDE (CLD), Mate (CLDM) и Xfce (CLDX), Calculate Directory Server (CDS), Calculate Linux Scratch (CLS), Calculate Scratch Server (CSS) и Calculate Media Center (CMC). Все версии дистрибутива распространяются в виде загрузочного Live-образа с возможностью установки на жесткий диcк или USB-накопитель. Основные изменения: Calculate Linux Desktop обновлён до KDE Plasma 5. Добавлено разделение на тестовые и стабильные обновления системы. Пакеты, собранные с различными USE-флагами объединены в один репозиторий (binpkg-multi-instance). Основное зеркало обновлений Git перенесено на Github. Ускорена работа архивирования XZ на многопроцессорных системах. Для запаковки initramfs, ядра и модулей используется формат компрессии XZ. Исправлена работа OverlayFS (перемещение и переименование файлов). По умолчанию задействован Python 3.4. Улучшена проверка на доступность зеркала в период обновления. Изменена структура зеркал. Состав пакетов: CLD (KDE desktop): KDE Frameworks 5.22, KDE Plasma 5.6.4, KDE Applications 16.04.1, LibreOffice 5.1.2.2, QupZilla 2.0.0, Gimp 2.8.16, Amarok 2.8.0 CLDM (MATE desktop): MATE 1.14, LibreOffice 5.1.2.2, QupZilla 2.0.0, Claws Mail 3.13.2, Gimp 2.8.16, Clementine 1.3.1 CLDX (Xfce desktop): Xfce 4.12, LibreOffice 5.1.1.2, QupZilla 2.0.0, Claws Mail 3.13.2, Gimp 2.8.16, Clementine 1.3.1 CDS (Directory Server): OpenLDAP 2.4.38, Samba 4.2.11, Postfix 3.0.3, ProFTPD 1.3.5b, Bind 9.10.3 CLS (Linux Scratch): Xorg-server 1.17.4, Kernel 4.4.9 CSS (Scratch Server): Kernel 4.4.9, Calculate Utilities 3.4.6.12 CMC (Media Center): Kodi 16.0, Xorg-server 1.17.4
