Вышел релиз контейнера для выполнения Java-сервлетов Jetty 7

Вышел релиз Jetty 7, свободного контейнера для выполнения Java-сервлетов, распространяемый под лицензиями Apache 2 и Eclipse Public Licence (EPL). Начиная с данной версии Jetty разрабатывается в составе первичного проекта Eclipse Runtime. Основные новшества: поддержка спецификации Servlet 2.5 API, полная поддержка Java 1.5, поддержка JASPI для работки внешних модулей аутентификации, улучшение интеграции OSGi, добавление модульного механизма start.jar, проведение оптимизаций и работа по увеличению производительности.
