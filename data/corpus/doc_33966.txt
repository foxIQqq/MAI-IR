Организация Linux Foundation выпустила инструмент для отслеживания использования открытых компонентов

Некоммерческая организация Linux Foundation, курирующая широкий спектр работ, связанных с развитием Linux, объявила о выпуске проекта FOSS Bar Code Tracker. Данное программное обеспечение нацелено на упрощение отслеживания и декларирования информации об использовании Open Source компонентов в коммерческих продуктах, путем автоматической генерации QR-кодов для каждого продукта. Сгенерированный QR-код содержит важную информацию об Open Source программах и библиотеках, включая их наименования, версии, лицензии и ссылки на загрузку. Приложение развивается как часть инициативы Open Compliance, нацеленной на оказание компаниям помощи по обеспечению соблюдения требований открытых лицензий. Даже для наиболее прилежных поставщиков оборудования отслеживание использования Open Source до последнего времени было достаточно сложным заданием. Приложение FOSS Bar Code Tracker позволяет командам разработчиков создавать свою собственную накладную открытых "материалов", идентифицировать содержащиеся FOSS (Free Open Source Software) компоненты, включенные в каждый конкретный продукт, а также легко обновлять и публиковать эту критическую информацию по всей цепочке линии поставок для обеспечения соответствия условиям лицензирования. FOSS Bar Code Tracker поддерживает генерацию одноразмерного штрих-кода, QR-кода с данными или QR-кода с "горячей ссылкой", позволющей заинтересованным лицам получить полную информацию о включённых Open Source компонентах в одно нажатие. Среди дополнительных возможностей программы отмечается обработка записей об используемых лицензиях (создание, поиск, изменение, удаление), определение информации о компании, панель для управления лицензиями, возможности импорта сведений об используемых открытых компонентах из информационной базы SPDX. Исходные тексты FOSS Bar Code Tracker распространяются под открытой лицензией MIT и доступны для загрузки из Git-репозитория.
