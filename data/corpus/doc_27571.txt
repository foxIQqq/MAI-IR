Kronikarz - утилита для резервного копирования и синхронизации настроек

Представлен первый выпуск проекта Kronikarz, разработанного в рамках инициативы Fedora Summer Code 2010. Программа представляет собой GUI-интерфейс для резервного копирования и синхронизации между разными компьютерами параметров рабочего окружения пользователя. Программа написана на языке Python с использованием библиотеки PyQt4. Готовые к установке пакеты подготовлены для Fedora 12/13, Slackware, openSUSE 11.3 и в скором времени ожидается для ArchLinux.
