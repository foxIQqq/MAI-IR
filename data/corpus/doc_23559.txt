Вышлa четвертая бета версия почтового клиента Thunderbird 3.0

Вышлa четвертая бета версия почтового клиента Thunderbird 3.0. По сравнению с прошлой бета версией добавлено около 200 изменений. Основные новшества: Средства поиска Новая система поиска с расширенными функциями фильтрации полученных результатов поиска. Например, можно отфильтровать вывод по отправителю, диапазону времени отправки, ассоциированным тэгам, типу приложения к письму, папке, принадлежности к списку рассылки и т.п. При наборе текста в поле глобального поиска, Thunderbird теперь автоматически предлагает наиболее релевантные ключевые слова на основе данных в адресной книге (например, дополняет email и имена). Интерфейс пользователя: Новый мастер для настройки почтовых аккаунтов, учитывающий типовые настройки для популярных почтовых сервисов (достаточно ввести email и пароль. IMAP, POP3 и другие настройки подставятся автоматически). Редизайн панели инструментов (тулбара), на которую добавлена форма быстрого поиска. Кнопки для ответа, пересылки и удаления убраны из тулбара и теперь привязаны к каждому почтовому сообщению. Навигация через систему вкладок (табов) - каждое почтовое сообщение, папки или страницы календаря могут быть открыты в отдельном табе (новый таб открывается после двойного клика на сообщение в списке). По умолчанию активен режим "умных" папок (Smart Folders), при котором в одном месте комбинируются данные из специализированных ящков, подобных Inbox, от разных аккаунтов. Для каждой папки поддерживается выбор своего метода сортировки списка сообщений; Новый режим сводного просмотра сообщений, позволяющий в одном окне посмотреть данных из нескольких выделенных email; Менеджер архивов - позволяет поддерживать общий архив всех сообщений (как в GMail), в котором они остаются и после из перемещения из входящей почты и других папок. В архиве сообщения группируются по месяцам; Менеджер активности - записывает и отображает через единый интерфейс все операции, осуществляемые между Thunderbird и почтовыми серверами, а также прочие перемещения сообщений; Менеджер дополнений, который теперь может быть использован для поиска, загрузки и выбора расширяющих функциональность расширений. При первом открытии "Tools > Add-ons" будет предложен список рекомендованных для установки дополнений. Упрощен процесс изменения и добавления элементов в адресной книге. Изменить или добавить параметры можно через нажатие на специальную пиктограмму рядом с email в окне с сообщением, без необходимости открытия адресной книги. Улучшена интеграция с сервисом Gmail, добавлено распознавание специальных папок (Sent, Trash и т.п.) для не англоязычных локализаций. Дополнительно Gmail папка "All Mail" теперь ассоциируется с архивом сообщений. Для платформы Windows Vista добавлена поддержка индексации почтовых и новостных сообщений встроенным в данную ОС механизмом поисковой индексации. Прекращена поддержка платформ Windows 95/98/Me. Для пользователей Mac OS X добавлена возможность чтения системной адресной книги и вывода уведомления о приходе почты через Growl. Производительность По умолчанию включена функция фоновой загрузки сообщений с IMAP сервера. Возможность можно включить или отключить в привязке к отдельным папкам на IMAP сервере. Интегрирована новая версия движка Gecko 1.9.1.3, используемого в Firefox 3.5, что привело у увеличению быстродействия и понижению потребления памяти.
