Выпуск Chrome OS 78 с поддержкой виртуальных рабочих столов

Компания Google представила релиз операционной системы Chrome OS 78, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 78. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 78 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 78: Добавлена поддержка виртуальных рабочих столов. Пользователь теперь может создать до четырёх виртуальных рабочих столов, произвольно переносить приложения между ними и переключаться на любой из них. Для создания виртуального рабочего стола в режиме с обзором запущенных приложений в верхнем правом углу добавлена кнопка "New Desk"; Обеспечен автоматический выход из спящего режима при подключении док-станции к порту USB-C, что позволяет сразу начать работу с внешним монитором без необходимости открытия крышки ноутбука и нажатия на кнопку включения питания; В подсистеме Crostini, предназначенной для запуска Linux-приложений, по умолчанию активирована функция предоставления доступа к GPU для повышения производительности графических операций в Linux-приложениях. Кроме того в Crostini реализован вывод предупреждения в случае невозможности использовать в запущенном приложении выбранного метода ввода или экранной клавиатуры; В окружение для запуска Linux-приложений добавлена функция создания резервной копии всех приложений и связанных с ними файлов. Резервная копия может быть сохранена в локальное хранилище, на внешний накопитель или в облачный сервис Google Drive. В случае проблем или при переходе на другое устройство сохранённая резервная копия может быть использована для восстановления предыдущего состояния или клонирования сформированного окружения; В файловом менеджере обновлены индикаторы операций. Информация о прогрессе выполнения операций перенесена из нижнего левого угла в область обратной связи в основном окне; Обновлён интерфейс для настройки принтера. В разделе настройки принтера в конфигураторе теперь сразу можно посмотреть список доступных принтеров, поддерживающих IPP/IPPS, и без дополнительных настроек выбрать принтер для вывода на печать или для использования в качестве принтера по умолчанию (после нажатия Ctrl + P достаточно просто выбрать принтер); В экранном ридере ChromeVox добавлена опция для озвучивания стиля текста; В конфигураторе разделены настройки Chrome OS и браузера Chrome. Приложение Settings, запускаемое из списка App Launcher или через меню быстрого вызова настроек, теперь предлагает настойки операционной системы. Настройки браузера вынесены в секцию "Дополнительно" в верхнем правом углу окна конфигуратора или доступны через "chrome://settings" в адресной строке. Для вызова системных настроек из браузера предложен новый URL "chrome://os-settings"; В окружение ARC++ (App Runtime for Chrome, прослойка для запуска Android-приложений в Chrome OS) добавлена поддержка просмотра видео в приложении YouTube в режиме картинка в картинке, параллельно работая с другими Android-приложениями; Добавлена возможность инициирования звонка с Android-устройства, привязанного к той же учётной записи. Пользователь теперь может выделить номер телефона в браузере и из контекстного меню перенаправить операцию совершение звонка на Android-устройство, после чего на телефоне всплывёт уведомление, позволяющее инициировать звонок; Упрощён процесс отправки отзывов и пожеланий. Кнопка для отправки отзыва добавлена в диалог завершения работы, отображаемый при нажатии кнопки выключения питания.
