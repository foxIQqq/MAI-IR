Лучшие проекты 2012 года по версии LinuxQuestions.org

Подведены итоги голосования среди участников форума LinuxQuestions.org по выбору лучших приложений 2012 года: Пользовательские приложения для рабочего стола Десктоп дистрибутив - Slackware (20.59%). За Ubuntu и Linux Mint в сумме проголосовало около 40%, но по отдельности они отстали от Slackware, высокие показатели которого связаны с тем, что на форуме LinuxQuestions.org в основном общаются пользователи Slackware; Мобильная платформа - Android (66.86%); Офисный пакет - LibreOffice (85.14%); Десктоп среда - KDE (31.31%), в прошлом году лидировал GNOME; Оконный менеджер - Openbox (17.53%), в прошлом году лидировал Compiz; Средство виртуализации - VirtualBox (53.79%); Файловый менеджер - Dolphin (23.77%), в прошлом году лидировал Nautilus; Игровое приложение - Battle for Wesnoth (11.72%); Эмулятор терминала - Konsole (22.68%); Сетевые пользовательские приложения Web-браузер - Firefox (52.76%); ПО для обмена мгновенными сообщениями - Pidgin (52.06%); VoIP-клиент - Skype (51.82%); Мультимедиа Аудиопроигрыватель - amaroK (15.68%); ПО для обработки звука- Audacity (74.58%, в прошлом году 77.26%); Видеопроигрыватель - VLC (64.99%); ПО для обработки графики - GIMP (69.85%); Приложения для администратора Серверный дистрибутив - Debian (28.74%); Приложение для резервного копирования - rsync (42.82%); Приложение связанное с сетевой безопасностью - Wireshark (25.79%); Приложение связанное с безопасностью системы - SELinux (34.07%); Программа для мониторинга - Nagios (55.29%); Система управлением конфигурацией - Puppet (61.70%); Web-сервер - Apache httpd (74.04%); Открытый аппаратный проект - Raspberry Pi (79.29%); Приложения для разработчика СУБД - MySQL (40%); NoSQL база данных - MongoDB (38.96%); Интегрированная среда разработчика - Eclipse (23.62%); Текстовый редактор - vim (34.35%); Язык программирования - Python (28.03%); Система управления контентом - WordPress (56.63%); Web-фреймворк - Django (34.31%); Система управления исходными текстами - Git (61.35%);
