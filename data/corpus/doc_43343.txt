Доступны Red Hat Developer Toolset 4 и Red Hat Software Collections 2.1

Компания Red Hat объявила о выпуске продукта Red Hat Software Collections 2.1, позволяющего использовать в Red Hat Enterprise Linux новые технологии разработки и компоненты web-стека, не ожидая появления следующей ветки RHEL и не устанавливая сторонние пакеты из неофициальных внешних источников. Время поддержки Red Hat Software Collections составляет три года, в течение которых пользователи будут снабжаться всеми необходимыми обновлениями с устранением проблем безопасности и критических ошибок. Red Hat Software Collections 2.1 включает в себя пакеты с более свежими версиями программ, чем входящие в состав штатного RHEL. Например, для установки предлагаются свежие выпуски Python, PHP, Perl, Ruby, MySQL, MariaDB , PostgreSQL, MongoDB, Maven, Passenger, Apache httpd, nginx, Node.js, Git, Common Java Packages. Из наиболее заметных обновлений отмечается включение пакетов с nginx 1.8, node.js 0.10.40, Maven 3.0 и прокси-сервером Varnish 4.0.3. Для популярных коллекций программ, востребованных разработчиками, подготовлены шаблоны для создания контейнеров Docker и окружений для Linux Atomic Host и PaaS-платформы OpenShift. Одновременно представлен похожий по своей сути набор Red Hat Developer Toolset 4, включающий пакеты с более новыми выпусками средств для разработчиков приложений на языках C и C++. В комплект входят: GCC 5.2, GDB 7.10, Eclipse 4.5, binutils 2.25, SystemTap 2.8 и т.п.
