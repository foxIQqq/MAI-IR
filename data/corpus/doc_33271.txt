Для Raspberry Pi портирован дистрибутив Arch Linux

Разработчики проекта Raspberry Pi, в рамках которого на базе SoC Broadcom BCM2835 700 МГц (ARM11 ARM1176JZF-S) создан одноплатный компьютер ценой 25 долларов, сообщили о готовности для данного устройства порта дистрибутива Arch Linux ARM. Кроме Raspberry Pi дистрибутив также доступен для широкого спектра ARM-платформ, таких как ARMv5te, ARMv6 PLX OxNAS и ARMv7 Cortex-A8/Cortex-A9 (OMAP3/4, Tegra2). Подготовленный для загрузки с SD-карты образ (215 Мб) содержит только базовое консольное окружение. Тем не менее, в штатном репозитории подготовлены пакеты с рабочим окружением LXDE, для установки которых достаточно выполнить команду "pacman -S lxde xorg-xinit xf86-video-fbdev" (для запуска LXDE - "xinit /usr/bin/lxsession"). Пакеты в репозитории обновляются непрерывно по мере выхода новых версий программ. Кроме Arch Linux для Raspberry Pi также доступны установочные образы на базе Debian Squeeze, которые из коробки содержат рабочее окружение LXDE, браузер Midori и мультимедиа приложения.
