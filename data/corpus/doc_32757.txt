Netbook-live - дистрибутив для нетбуков, основанный на ALT Linux

Николай Гречух представил рождественский выпуск дистрибутива для нетбуков Netbook-live на базе ALT Linux p6 branch с рядом дополнительных пакетов (ядро Linux 3.1.7, Firefox 8, skype-call-recorder, redis). В качестве графического окружения используется LXDE. Особенностью дистрибутива является ориентация на повседневную работу профессионального пользователя в Live-режиме без установки системы на жёсткий диск и с возможностью возвращения системы в первоначальное состояние после экспериментов простой перезагрузкой (корень системы доступен в сжатом виде в режиме только для чтения). Размер iso-образа 653 Мб. Для модификации рабочей среды предусмотрены хуки, выполняемые при загрузке и позволяющие изменить настройки загрузки по умолчанию (например, смонтировать внешний раздел). При необходимости внесения более глубоких изменений можно взять сборочный профиль дистрибутива вместе с инструкциями и собрать новый образ, пользуясь инструментарием в комплекте. Для входа в систему следует использовать логин altlinux и пустой пароль. Среди поддерживаемых устройств: Acer Aspire TimelineX 1830T Acer Aspire One NAV50 ASUS Eee Box 1007 (неттоп) ASUS Eee PC 701, 901, 1000 ASUS Eee PC 1215N ASUS Z99H BENQ R55V IBM ThinkPad R52 IBM ThinkPad T43 LG LS 50
