Вышла официальная консольная версия 7-zip для Linux

Игорь Павлов выпустил официальную консольную версию 7-zip для Linux вместе с выпуском версии 21.01 для Windows в связи с тем, что проект p7zip не видел обновлений уже пять лет. Официальная версия 7-zip для Linux схожа с p7zip, но не является копией. О разнице проектов не сообщается. Программа выпущена в версиях для x86, x86-64, ARM и ARM64. Лицензия на код 7-Zip для Linux - GNU LGPL. Помимо портирования для Linux из улучшений в 7-zip 21.01 отмечается повышение производительности на оборудовании ARM64 за счёт задействования специализированных инструкций CPU для ускорения вычислений AES, CRC-32, SHA-1 и SHA-256.
