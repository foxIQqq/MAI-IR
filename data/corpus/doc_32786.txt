Рейтинг наиболее активно внедряемых открытых проектов возглавили HBase, Node.js и nginx

Компания OpenLogic, занимающаяся оказанием коммерческой технической поддержки для дистрибутива CentOS и отдельных открытых пакетов, а также развивающая сервис, позволяющий коммерческим предприятиям избежать нарушения лицензии GPL, представила отчёт с результатами анализа тенденций востребованности открытых проектов в 2011 году. Отчёт построен на основе изучения внедрения популярных и новых открытых проектов для обеспечения работы ключевых служб предприятий. Итоговый рейтинг построен с учётом восьми метрик для каждого открытого проекта, включающих как общедоступные данные, так и статистику собранную на 250 предприятиях в разных частях света, использующих инструментарий OpenLogic. Проекты, продемонстрировавшие наиболее значительный рост внедрений в 2011 году: HBase, реализованная поверх Hadoop распределённая БД, претендующая на роль аналога Google Bigtable; Node.js, серверная платформа для написания эффективных интернет-приложений на языке JavaScript; nginx, высокопроизводительный http-сервер; Hadoop, платформа для организации распределённой обработки больших объёмов данных (петабайты) с использованием парадигмы map/reduce; Ruby on Rails, фреймворк для разработки web-приложений на языке Ruby с использованием парадигмы MVC (Model-View-Controller). Тенденции внедрения серверов приложений и web-серверов: Рост (общий рост внедрений по всем всех метрикам): Node.js и nginx; Cохранение уровня (удержание прошлых позиций, падение и рост примерно одинакового числа метрик): Tomcat и Apache httpd; Спад (общее снижение всех метрик): JBoss и GlassFish; Тенденции внедрения фреймворков: Рост: Rails Сохранение уровня: Spring, Grails, Struts Спад: нет Тенденции внедрения баз данных и средств для работы с большими объёмами данных: Рост: HBase, Hadoop, MongoDB Сохранение уровня: MySQL, PostgreSQL Спад: CouchDB Общий рейтинг 16 выбранных открытых проектов по степени роста внедрений: HBase Node.js nginx Hadoop Rails MongoDB Tomcat MySQL Apache Spring PostgreSQL Grails Struts JBoss GlassFish CouchDB Дополнительно можно отметить рейтинг 10 лучших открытых продуктов 2011 года по мнению издания ChannelWeb: LibreOffice; Ubuntu 11.10; Linux Mint 12; Firefox For Android; Android 3.x (платформа была открыта, только после выхода Android 4); MySQL 5.6 (релиз ещё не вышел); Asterisk 10; OpenStack; Ubuntu 11.04; Fedora 16.
