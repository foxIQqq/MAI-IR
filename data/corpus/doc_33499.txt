Первоапрельские шутки 2012 года (дополнено)

Подборка первоапрельских шуток: Проект NetBSD сообщил о переходе к использованию графической консоли, реализованной поверх драйвера genfb(4) и использующей шрифты со включенным антиалиасингом, которые выглядят лучше консольных монохромных шрифтов. В качестве шрифтов решено использовать пакет Comic Sans, который компания Microsoft разрешила использовать во всех будущих выпусках NetBSD. Distrowatch объявил о выходе первой тестовой версии десктоп-окружения GNOME 4.0, отличающегося полностью переработанной концепцией организации рабочего стола, унифицированным внешним видом для всех типов устройств, упрощённым интерфейсом пользователя. Опубликован достаточно подробный обзор первого тестового выпуска GNOME 4.0. Как всегда отличилась компания Google, которая представила многозадачный режим для браузера Chrome, позволяющий существенно увеличить эффективность работы. При навигации человек одновременно действует только одной рукой, в то время как новая функций Chrome позволяет работать в браузере сразу двумя руками, путем подключения к ПК нескольких мышей, каждая из которых связывается с отдельным курсором. Среди других розыгрышей Google: Создан работающий вариант Google Maps для экранов 8-битных приставок Nintendo, карты отображаются в стиле, похожем на первые версии игры Simcity (видеодемонстрация); Представлен полностью автоматически управляемый гоночный автомобиль, при этом на сайте NASCAR (Национальная Ассоциация гонок серийных автомобилей) можно посмотреть видеоролик с презентацией данного автомобиля (в роли пилота-испытателя выступает Сергей Брин). Началось распространение DVD с материалами YouTube; Для создания панорам Street View диких мест работники Google прикрепили к голове более тысячи кенгуру автономные камеры; Для Gmail представлен сервис Gmail Tap для быстрого написания сообщений, используя всего две кнопки. Слова набираются с использованием азбуки морзе. На сайте очень качественный рекламный ролик новой функции; Создатели AdBlock представили дополнение CatBlock, защищающее от случайного нажатия клавиш кошкой; Разработчики Bitbucket опубликовали новую систему для организации параллельного программирования, при которой над написанием кода для реализации одной задачи синхронно могут работать несколько программистов; Sony анонсировала ультрабук VAIO Q, размером с почтовую марку. Ультрабук поставляется с экраном 0.75x1.25 дюймов (среди аксессуаров линза как для первых моделей телевизоров), способным отображать Full HD 1080p видео. Устройство оснащено 8 Гб ОЗУ и видеокартой AMD Radeon HD 6650M; Компания Apple получила патент на прямоугольник; Использование Git в качестве системы для передачи сообщений: пишется сообщение и делается коммит, для отслеживания появления сообщений проверяется наличие новых коммитов. Новая парадигма массиво-ориентированного программирования для Web с реальным примером на PHP; Для HTML5 представлены тэги для определения запаха, текстуры поверхности и вкуса; Компания Skype представила метод бесплатной передачи аудиосообщений на небольшие расстояния без затрат энергии и без доступа к сети (используется метод связи через соединение натянутой ниткой двух спичечных коробков или пластиковых стаканов); Ричард Столлман выпускает собственную коллекцию стильной одежды для любителей свободного ПО; Решена проблема с повышенным потреблением памяти БД Virtuoso (пакет virtuoso-nepomuk). В связи с частыми нареканиями на Virtuoso, связанными с чрезмерным потреблением памяти и CPU, разработчики KDE подготовили патч, решающий эти проблемы. В новой версии KDE утилита ksysguard больше не будет показывать процессы Virtuoso, и разработчики надеются, что число жалоб на его прожорливость заметно снизится. В настоящее время Себастьян Трюег (Sebastian Trueg), основной разработчик Nepomuk, готовит аналогичный патч для утилиты top, также ведутся переговоры с разработчиками ядра о возможности удаления PIDов Virtuoso из /proc. Дополнение: Компания Canonical представила сборку Ubuntu for Eyewear с интерфейсом, оптимизированным для очков; Retro Fedora - репозиторий пакетов для Fedora 16 с подборкой старого софта для пользователей, испытывающих чувство ностальгии. В репозитории Gnome 1.0, Gnome 2.0, KDE 1 - 3.5, старые версии Open Office и Gimp; Линус Торвальдс признал Arch Linux дистрибутивом, наиболее дружественным к пользователю; Проект olpc-antarctica.com по распространению ноутбуков OLPC в Антарктиде; Опубликован скриншот версии MS Office 2013 для Linux; Патрик Фолькердинг решил остановить гонку за новыми версиями и использовать в Slackware подборку старых и проверенных программ, а также оставить поддержку только интерфейса на базе Window Maker и FVWM; Microsoft подала в суд на Canonical. Другой вариант - Microsoft и Canonical заключили партнёрское соглашение по совместному созданию нового пользовательского интерфейса; Проект KDE переключается на использование старых мультиедиа фреймворков Akode и aRts, вместо Phonon и Pulseaudio ; Разработчики Lubuntu подготовили сборку для домашнего компьютера Commodore 64k, образца 1982 года; Xubuntu объявил о переходе на пакетную базу Debian; Два новых Linux-дистрибутива: Hsiloof Linux, обеспечивающий плотную интеграцию с приложениями для Windows; Biebian (загрузить можно с SourceForge), дистрибутив на основе Puppy Lucid для тинейджеров; Опубликован эксплоит для 0-day уязвимости в Apache 2.2.x, позволяющий удалённо выполнить код на сервере; В течение дня текст новости будет пополняться новыми первоапрельскими шутками.
