Mozilla не будет портировать Firefox для iPhone и сосредоточит усилия на развитии Firefox Home

Разработчики Mozilla не намерены в будущем портировать мобильную версию Firefox для телефонов iPhone, сосредоточив свои усилия на развитии приложения для синхронизации данных - Firefox Home. По заявлению разработчиков, портированию Firefox Mobile мешает ряд технических и юридических проблем, которые значительно затрудняют и даже делают невозможным создание полной версии Firefox для iPhone. Программа Firefox Home, используя возможности системы синхронизации Firefox Sync, позволяет перенести на iPhone актуальные данные из последней сессии Firefox на настольной или мобильной системе. Поддерживается синхронизация закладок, просмотр данных из истории посещений и открытие списка текущих открытых вкладок (табов). В ближайшее время разработчики планируют выпустить Firefox Home для других мобильных платформ, таких как BlackBerry и Symbian. Кроме того, планируется заметно расширить функциональность приложения: обеспечить синхронизацию паролей, провести более тесную интеграцию с новыми возможностями iOS4 и Mobile Safari, предоставить сервис для быстрого обмена ссылками и комментариями, добавить возможность прямой работы с социальными сетями.
