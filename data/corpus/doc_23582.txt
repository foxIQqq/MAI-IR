Microsoft и Intel портируют Silverlight на Linux

Компании Intel и Microsoft объявили о начале работы над созданием новой реализации технологии Silverlight для Linux, оптимизированной для использования совместно с мобильной платформой Moblin. Intel уже получила от Microsoft исходный код и набор тестов для Silverlight. Как известно, реализация Silverlight для Linux уже существует (см. проект Moonlight), в связи с этим руководитель Developer Platform Group Брайан Голдфарб (Brian Goldfarb) сказал, что в данном случае делаются различия между "Linux в целом", на который ориентирован Moonlight, и конкретными Linux-устройствами, для которых Microsoft может обеспечить поддержку других реализаций. Ожидается, что продукт будет готов к лету следующего года, а более подробная техническая информация будет предоставлена не позднее окончания следующей конференции MIX10. Поддержка от Intel, обещающая Microsoft долгосрочное партнёрство, и связанная с этим PR-акция пришлась очень кстати для Silverlight, который, возможно, уравняет шансы с Adobe AIR, в настоящее время явно гораздо лучше приспособленного для таких устройств, как нетбук с Moblin на борту. Несколько не вписывающимся в общую радостную картину остаётся вопрос: "если портировать Microsoft Silverlight на Linux настолько просто, то для чего вообще существует проект Moonlight?".
