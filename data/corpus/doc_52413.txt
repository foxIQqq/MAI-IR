Релиз видеоредактора Shotcut 20.02

Опубликован релиз видеоредактора Shotcut 20.02, который развивается автором проекта MLT и использует данный фреймворк для организации редактирования видео. Поддержка форматов видео и звука реализована через FFmpeg. Возможно использование плагинов с реализацией видео и аудио эффектов, совместимых с Frei0r и LADSPA. Из особенностей Shotcut можно отметить возможность многотрекового редактирования с компоновкой видео из фрагментов в различных исходных форматах, без необходимости их предварительного импортирования или перекодирования. Имеются встроенные средства для создания скринкастов, обработки изображения с web-камеры и приёма потокового видео. Для построения интерфейса применяется Qt5. Код написан на C++ и распространяется под лицензией GPLv3. В новом выпуске: Добавлена возможность обработки видео в процессе редактирования с разрешением, установленным для предпросмотра. Предложенный режим активируется через настройку "Preview Scaling" и позволяет экономить процессорные ресурсы, благодаря промежуточной обработке видео с разрешением ниже целевого (например, для целевого видео 1080p в процессе редактирования будут выполняться манипуляции с разрешением 640x360). Некоторые фильтры не поддерживают новый режим и по-прежнему всегда обрабатывают картинку с полным разрешением проекта. Дополнительно предусмотрен режим быстрого экспорта, позволяющий сохранить черновой вариант с более низким разрешением. Добавлен фильтр для изменения высоты звука, который можно использовать для компенсации изменения скорости видео, для создания неузнаваемого голоса или для создания комичных голосов. Расширены эффекты перехода от одного изображения к другому. Общее число предлагаемых эффектов перехода превысило 150. Добавлен новый режим визуализации видео "Video Vector" (Views > Scopes > Video Vector). Добавлены преднастройки для экспорта в форматы ALAC, FLAC, DNxHR HQ, ProRes HQ и ProRes 422.
