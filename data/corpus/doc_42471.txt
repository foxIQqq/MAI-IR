Открыт код статического анализатора Code Climate

Компания Code Climate объявила об открытии своей платформы статического анализа кода. В настоящий момент под лицензией MIT опубликован инструментарий командной строки и движки статического анализа, лежащие в основе новой платформы Code Climate. В дальнейшем планируется открыть под открытыми лицензиями все наработки связанные со статическим анализом кода. Открытая версия Code Climate позволяет проверить качество кода и выявить возможные ошибки при помощи инструментария командной строки. Изначально система создавалась для анализа кода на языке Ruby, но также поддерживает проверку стиля и выявление ошибок для языков EcmaScript/JavaScript и Go. Движки проверки стиля реализованы для языков PHP, CoffeeScript и CSS. Для языка Ruby дополнительно поставляется движок для поиска уязвимостей с учётом используемых в проекте зависимостей.
