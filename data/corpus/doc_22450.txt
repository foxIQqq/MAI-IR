Выпущен июльский номер электронного журнала Gnome Journal

Выпущен июльский номер электронного журнала Gnome Journal, в котором представлены следующие статьи: "Tracking Your Time with Project Hamster" - обзор приложения для планирование времени - Hamster; "Working With Upstream: An Interview with Laszlo Peter" - интервью с Laszlo Peter, релиз-инженера из Sun Microsystems, занимающегося поддержкой Sun Desktop на базе GNOME; "Git for GNOME Translators" - примеры использования системы управления исходными текстами Git для переводчиков элементов интерфейса Gnome; "An Introduction to GNOME Zeitgeist" - обзор проекта GNOME Zeitgeist, нового интерфейса для работы с документами и пользовательской информацией, который будет включен в состав GNOME 3.0; "Do-licious" - обзор возможностей GNOME Do, среды для для быстрого запуска программ в GNOME; "Behind the Scenes with Owen Taylor" - интервью с Owen Taylor, мантейнера проекта GNOME Shell и один из системных администраторов управляющих инфраструктурой GNOME. В беседе затрагивается вопрос миграции GNOME на систему управления исходными текстами Git.
