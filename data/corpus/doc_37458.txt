Доступна публичная бета-версия дистрибутива ROSA Fresh R1 на базе GNOME 3.8

Компания РОСА представила публичный бета-выпуск дистрибутива ROSA Fresh R1 GNOME версии 3.8. Размер установочных iso-образов 1.3 Гб. Основные особенности сборки: Добавлена возможность использования типографской раскладки Бирмана. По умолчанию используется белая тема оформления GNOME Shell. Из коробки доступны некоторые расширения, такие как Dash to Dock, Weather, Notification Alert, Alternative Status Menu, классический режим GNOME и др. В виду изменения политики Google относительно ключей для API Chromium, выполнен обратный переход на FF . Использование утилиты gnome-initial-setup для первоначальной настройки окружения при первом входе нового пользователя в систему. В GNOME Control Center перенесена возможность настройки отображения шрифтов из утилиты gnome-tweak-tool. По умолчанию используются патчи из Ubuntu для улучшенного отображения шрифтов на LCD-мониторах. Базовый состав ПО: Ядро Linux версии 3.8.12 Firefox 22 Evolution 3.8.3 Libreoffice 3.6.6 ROSA Media Player 1.6.
