Релиз Lubuntu 10.10, сборки Ubuntu на базе LXDE

Доступен релиз Lubuntu 10.10, неофициальной легковесной сборки Ubuntu 10.10 c рабочим столом на основе десктоп-окружения LXDE. Цель развития Lubuntu - создание нетребовательного к системным ресурсам варианта Ubuntu, более легкого чем Xubuntu и способного работать на устаревших ПК с 128 Мб ОЗУ. Размер iso-образа составляет 546 Мб. Из улучшений можно отметить: Полностью переработанная визуальная тема; Поддержка размещения апплетов-индикаторов Ubuntu на панели Lxpanel; В состав включены программы: Update-notifier для вывода уведомлений о наличии обновлений, Xpad для написания кратких заметок и набор игр Ace-of-penguins; Для управления задачами вместо ранее используемого Xfce4-taskmanager задействован LXtask; Для просмотра PDF теперь используется Evince; Инсталлятор переработан для демонстрации слайдов, показывающих возможности системы; Добавлены мета пакеты lubuntu-core для установки только базового набора пакетов Lubuntu и lubuntu-restricted-extras для установки дополнительных пакетов с проприетарными приложениями (flash, java и набор мультимедиа кодеков). Приложения, входящие в базовую поставку: файловый менеджер PCManFM (переписанный для задействования gio/gvfs); web-браузер Chromium, почтовый клиент Sylpheed, программа для мгновенного обмена сообщениями Pidgin, Bittorrent-клиент Transmission, табличный процессор Gnumeric, редактор текстов Abiword, программа для просмотра фотографий GPicView, дисплейный менеджер Lxdm, программа для рисования MTPaint, музыкальный проигрыватель Aqualung, видеоплеер Gnome-mplayer, ПО для записи CD/DVD - xfBurn.
