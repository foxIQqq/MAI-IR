Релиз Plop Linux 4.2, Live-дистрибутива для нужд системного администратора

Представлен релиз Plop Linux 4.2, Live-дистрибутива с подборкой утилит для выполнения текущих задач системного администратора, таких как восстановление системы после сбоя, проведение резервного копирования и восстановления операционной системы, проверки защищенности системы, автоматизации выполнения типовых задач. В составе дистрибутива на выбор предлагается два графических окружения - Fluxbox и GNOME. Поддерживается загрузка дистрибутива на соседней машине через PXE. Размер iso-образа 976 Мб, дополнительно поставляется сокращённый консольный вариант размером 82 Мб. Минимальные требования к системе - CPU i486 и 64 Мб ОЗУ. Основные особенности новой версии: Из поставляемых в составе дистрибутива пакетов можно отметить: ядро Linux 3.2.7, binutils 2.22, btrfs 0.19, hfsprescue 0.2, ntfs-3g 2012.1.15AR.1, obexfs 0.12, plop boot manager 5.0.14, testdisk 6.13, photorec, chntpw, reged, chkrootkit, wine, vnc, F-PROT Antivirus, AVAST Antivirus, утилиты для работы с различными RAID-контроллерами, подборка сетевых утилит и т.д. Среди компонентов, работающих в графическом режиме: Fluxbox 1.3.2, GNOME 3.3.91 (тестовый выпуск 3.4), Mplayer 2012-02-11, GQView 2.0.4, gFTP 2.0.19, gParted 0.11.0, Firefox 10.0.2, Wine 1.3.37, mp3blaster 3.1.2, GtkOfflineRegistry 0.1 ushare 1.1a, xchat 2.8.8. Приложения, добавленные в версии 4.2: cpuutils 002, hddtemp 0.3 beta 15, hibernate 1.97, kexec-tools 2.0.2, kmod5, laptop-mode-tools 1.55, lm-sensors 3.3.1, plopKexec 0.3, plpmkboot. В состав включены компоненты, для обеспечения работы людей с ограниченными возможностями (brltty, Orca); Подготовлены примеры скриптов для создания специализированных Live-сборок: NAS Server, Media Server, Printer Server Добавлены новые опции загрузки по сети: TFTP, FTP, HTTP. Обеспечена загрузка из ISO-файла, в том числе доступного по сети;
