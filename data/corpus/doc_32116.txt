Вышел Finnix 103, Live-дистрибутив для системных администраторов

Увидел свет релиз Live-дистрибутива Finnix 103, основанный на срезе пакетной базы Debian testing по состоянию на 11 октября. Дистрибутив поддерживает только работу в консоли, но содержит хорошую подборку утилит для нужд администратора. Размер iso-образа - 113 Мб, при этом без учета сжатия в комплект входит около 350 Мб утилит, например, в состав включены LVM2, cryptsetup, cdpr, iftop, irssi, OTP calculator, VLAN tools и т.д. Кроме сборок для x86 и x86_64 дистрибутив доступен в сборке для архитектуры PowerPC. В новой версии: Добавлен режим криминалистической экспертизы (forensic mode), при котором в процессе загрузки минимализируется влияние на имеющиеся носители: блокируется работа с блочными устройствами, проверяются контрольные суммы для файлов на загрузочном CD, отключается создание раздела подкачки, и автоопределение LVM, RAID и параметров сети. Режим включается через указание флага "forensic" на этапе загрузки; Реализован криптографически более надежный метод инициализации генератора случайных чисел; Задействовано ядро Linux 3.0.6; В комплект добавлены новые утилиты: htop, iotop, emad, oui, vanityhash, uuid-runtime, twuewand; Из поставки удалены приложения: opie-client, console-common, hello; В initrd добавлена возможность проверки корректности сборки CD перед началом загрузки; Улучшены средства автоматического определения файловых систем в процессе создания /etc/fstab; Ускорен процесс вычисления зависимостей для сетевых модулей ядра в finnix-netboot-server.
