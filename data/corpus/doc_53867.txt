Релиз шрифтового движка FreeType 2.10.3

Представлен релиз FreeType 2.10.3, модульного шрифтового движка, предоставляющего единый API для унификации обработки и вывода шрифтовых данных в различных векторных и растровых форматах. Из изменений выделяется: Улучшена поддержка глифов TrueType с перекрывающимися контурами. При инициализации включена по умолчанию фильтрация для экранов LCD. Выполнена синхронизация кода автоматического хинтинга с ttfautohint. Добавлена поддержка сборки с использованием инструментария Meson. Улучшен код растеризатора и переработан код цветовой компенсации. Добавлена поддержка компиляторов Intel (icc).
