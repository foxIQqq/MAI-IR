Клиентское ПО для сервиса On My Disk доступно для Linux

Команда разработчиков сервиса On My Disk сообщила о готовности клиента On My Disk для Fedora, Ubuntu и openSUSE, а также об открытии исходных текстов компонентов сервиса под лицензией Apache 2.0. On My Disk представляет собой сервис для организации доступа к большим файлам без загрузки данных во внешние облачные хранилища (данные отдаются непосредственно с диска пользователя). Сервис основан на свободой платформе ownCloud и собственном протоколе, позволяющем монтировать клиентскую файловую систему удалённо по безопасному соединению. После запуска приложения пользователю достаточно выбрать директорию с данными и нажать кнопку "Connect", после чего появится возможность предоставления другим пользователям доступа к содержимому выбранной директории или организации доступа к своей системе из другого места, используя штатные установки ownCloud. Для подключения локального диска к удалённому серверу ownCloud подготовлен специальный FUSE-модуль и бэкенд для запуска на стороне клиента. При подключении клиента осуществляется монтирование выбранных директорий на сервер по сети. Трафик между клиентом и сервером шифруется. Протокол, клиентское ПО и дополнение для ownCloud распространяются под свободными лицензиями. В будущем планируется реализовать поддержку кросс-монтирования для подключения удалённых хранилищ к локальной системе, подготовить мобильный клиент и обеспечить поддержку потребительских устройств, таких ка медиаплееры (XBMC/Android) и домашние маршрутизаторы с подключенным диском (OpenWrt/Android).
