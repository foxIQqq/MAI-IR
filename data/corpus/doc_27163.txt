Вышел релиз Firefox Mobile 1.1 для платформы Maemo

Разработчики проекта Mozilla представили релиз мобильной версии web-браузера Firefox 1.1 (Fennec) для платформы Maemo, основанного на браузерном движке, едином с Firefox 3.6. Для предоставления возможности оценки функций браузера для пользователей не обладающих мобильными устройствами Nokia N810/N900, подготовлены специальные сборки Firefox Mobile 1.1 для запуска в стандартном десктоп-окружении Windows, Mac и Linux. Основные новшества Firefox Mobile 1.1: Реализация функции автозаполнения web-форм; Изменение оформления стартовой страницы, содержимое которой отражает активность пользователя в сети; Поддержка автоматического обновления дополнений; Реализация для смартфона N900 портретного режима просмотра и возможности изменения масштаба через нажатие кнопок управления громкостью; Добавление контекстного меню с функциями по открытию ссылки в новом табе и сохранения изображений; Набор настроек для управления параметрами сайта (очистка сохраненных паролей и т.п.) Возможность записи содержимого страницы в PDF-формате; Новая система обработки прикосновений к экрану, позволяющая более точно выполнять такие операции как клик по ссылке и нажатие кнопок. Проведена оптимизация производительности, интерфейс стал более интерактивным и более оперативно реагирующим на действия пользователя; При длительном удержании нажатия на элемент появляется контекстное меню, позволяющее выполнить такие операции как сохранение изображения или открытие ссылки в новой вкладке; В следующей версии Fennec 2.0 (Firefox Mobile) планируется переход на многопроцессную модель работы, при которой пользовательский интерфейс и обработка контента в разных вкладках, сможет обрабатываться параллельно, разными процессами. Также в будущем ожидается создание версии для платформы Android; поддержка мультитач навигации; виртуальная клавиатура; расширение средств синхронизации с настольной системой; редизайн стартовой страницы; поддержка восстановления случайно закрытых табов; поддержка RSS; разработка Qt-виджета; оптимизации для снижения энергопотребления.
