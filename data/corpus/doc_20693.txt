Выпущена LiveDVD вариант Linux дистрибутива Bluewhite64 12.2

Выпущен LiveDVD вариант Linux дистрибутива Bluewhite64 12.2, являющегося неофициальной сборкой Slackware Linux 12.2 для 64-разрядных систем (AMD64 и Intel EM64T) с упрощенным процессом установки. В состав LiveDVD вошло Linux ядро 2.6.27.10, десктоп-окружение KDE 4.2, web-браузеры Firefox 3.0.6 и Seamonkey 1.1.14 (включая Flash и 64-bit Java плагины), почтовые клиенты Thunderbird и K-mail, офисный пакет OpenOffice 3.0, игры Extreme Tux Racer, Battle for Wesnoth и Super Tux games. Для запуска 32-битных программ дистрибутив поддерживает режим эмуляции 32-разрядного окружения. В состав входит система настройки "ALICE" (Advanced Linux Installation and Administration Environment) из DarkStar Linux и инсталлятор BW64Installer для установки содержимого LiveDVD на жесткий диск. Напомню, что релиз стандартной (не "Live") редакции Bluewhite64 12.2 вышел 15 декабря.
