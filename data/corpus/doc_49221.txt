Релиз операционной системы NetBSD 7.2

Подготовлен релиз операционной системы NetBSD 7.2, который рекомендован для пользователей ветки NetBSD 7, которые по тем или иным причинам не могут перейти на NetBSD 8.0. Для загрузки подготовлены установочные образы размером 396 Мб. Релиз NetBSD 7.2 официально доступен в сборках для 58 системных архитектур и 16 различных семейств CPU. Выпуск NetBSD 7.2 включает ряд улучшений, некоторые из которых были бэкпортированных из NetBSD 8.0: Добавлен драйвер xhci с поддержкой USB 3.0; Появилась поддержка платы Raspberry Pi 3; Улучшенная подсистема эмуляции Linux, обеспечена эмуляция системного вызова pselect6; Решены проблемы с бинарной совместимостью с исполняемыми файлами, собранными для устаревших версий NetBSD; Новый драйвер iwm с поддержкой беспроводных чипов Intel Wireless 726x, 316x, 826x и 416x; Обеспечена возможность работы в окружении гипервизора Hyper-V с Legacy Network Adapter; Отключены по умолчанию подсистемы совместимости с SVR4 и IBCS2; Повышена стабильность USB-стека; Обновлены версии dhcpcd 7.0.8, libexpat 2.2.1 и lua 5.3.4.
