Релиз офисного пакета Calligra 2.5, развиваемого проектом KDE

Представлен релиз свободного офисного пакета Calligra 2.5, второй значительный выпуск с момента ответвления разработки от проекта KOffice в декабре 2010 года. Готовые бинарные сборки подготовлены для Arch Linux, Fedora, FreeBSD, Gentoo, openSUSE, Ubuntu и Windows. В качестве базового формата используется Open Document (ODF). Если первый стабильный выпуск Calligra содержал ряд кардинальных изменений, касающихся внутренней архитектуры, поддерживаемых платформ и организации пользовательского интерфейса, то Calligra 2.5 примечателен переходом на новую модель разработки, в рамках которой новые возможности помещаются в основное дерево исходных текстов только после полной готовности и стабилизации (до момента готовности новшества развиваются в отдельных ветках), что позволило сократить цикл подготовки релиза. Например, версия Calligra 2.5 была подготовлена за 4 месяца, в то время как на разработку прошлой версии ушло больше года. Основные новшества Calligra 2.5: В Words улучшена поддержка редактирования таблиц, обеспечено компактное обтекание текста вокруг изображений, расширены библиографические инструменты, добавлена возможность манипулирования границами таблиц, реализована функция перетаскивания блоков текста мышью. В табличном процессоре Sheets реализована новая обособленная панель для редактора ячеек, которая может размещаться поверх рабочей области таблицы, показываться в отдельном окне или прикрепляться с краю базового окна. Кроме того добавлено новое окно с инструментарием для управления форматированием ячеек, оформленное в стиле интерфейса управления форматированием текста в Words, которое предлагается вместо настроек в меню и специальных диалоговых окон; В Kexi, программе для работы с БД, добавлена поддержка полноэкранного режима, вызываемого через клавишу F11. В интерфейс проектирования формы ввода добавлена возможность использования дополнительных виджетов: кнопка привязки команд, слайдер, информатор о ходе выполнения операции, область получения данных; В Stage, программе для подготовки презентаций, проведена работа по улучшению юзабилити; Во Flow, программе для построения диаграмм, добавлена поддержка новых шаблонов для создания определённых пользователем очертаний элементов; В программе для рисования Krita добавлена новая панель для формирования композиций, полезная при выполнении задач раскадровки видео. Добавлена поддержка нового режима рисования с использованием текстур. Улучшены средства вращения содержимого холста. Расширены инструменты для управления автоматически сохранёнными файлами. Проведены оптимизации производительности; Добавлены новые фильтры импорта для файлов Visio, MS Works и xfig. Улучшена работа ранее добавленных фильтров для форматов MS Office, особенно улучшена поддержка формата OOXML; Представлен новый набор фильтров для обработки изображений, таких как преобразование в серые тона или чёрно-белый вид. Обеспечено зеркалирование изображений по произвольной оcи; Значительно улучшен код управления автоматическим сохранением данных; Добавлена новая система управления профилями пользователей. Пользователь теперь может определить несколько разных профилей (работа, личный ПК и т.п.) со своим набором информационных полей при создании и записи документа. По умолчанию создаётся два профиля: анонимный и профиль с данными, которые удалось получить из системы; Устранена критическая уязвимость, позволяющая организовать выполнение кода при открытии специально сформированного ODF-документа. Одновременно с десктоп-версией пакета выпущен вариант Calligra Active для планшетов с сенсорными экранами и смартфоров. Принято решение о развитии Calligra Active как основного варианта для мобильных устройств (параллельно развивался упрощённый Calligra Mobile для Nokia N900). В состав офисного пакета Calligra включены следующие базовые приложения: Calligra Words - текстовый процессор (ранее KWord). Поддерживается открытие и сохранение документов в форматах ODF и MS Word (.doc, .docx). Возможна интеграция изображений, таблиц и диаграмм в документы, перетаскивание объектов из других приложений Calligra через интерфейс drag&drop Calligra Sheets - табличный процессор (ранее KSpread), ориентированный на различного рода вычисления и бизнес-расчёты. Поддерживается использование шаблонов, интеграция диаграмм и графиков, возможности вставки формул. В наличии большая коллекций математических и статистических функций. Calligra Stages - система для подготовки презентаций (ранее KPresenter). Поддерживается использование в презентациях текcта, изображений, графиков и другого контента, поддерживаемого приложениями Calligra. Поддержка новых эффектов, типов контента и методов обработки презентаций может быть реализована через плагины; Kexi - среда для визуального управления данными (аналог MS Access, FileMaker и Oracle Forms), позволяющая быстро создавать формы ввода, обработки и редактирвоания данных, формировать запросы и генерировать отчёты; Calligra Plan - система управления проектами (ранее KPlato), позволяющая координировать выполнение задач и управлять распределением ресурсов; Krita - растровый графический редактор, поддерживающий многослойную обработку изображений и обладающий большим набором средств для цифровой живописи, создания скетчей и формирования текстур. Поддерживается большой набор графических планшетов для рисования от руки; Karbon - векторный графический редактор, отличающийся гибкими возможностями по кастомизации и поддерживающий расширение функциональности через плагины. Редактор подходит для создания логотипов, иллюстраций и фотореалистичных векторных изображений; Brainstorm - система ведения заметок, визуализации идей и структурирования контента. Для хранения информации используется формат openDocument, что позволяет включать в заметки не только текст, но и изображения и мультимедиа-вставки; Calligra Flow - редактор диаграмм, блок-схем и карт сети, напоминающий по своему назначению проприетарный пакет Visio.
