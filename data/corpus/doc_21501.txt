Вышел релиз PHP-фреймворка Adventure PHP 1.9

Вышел релиз Adventure PHP Framework 1.9, фреймворка для создания web-приложений на языке PHP, поддерживающего несколько объектно-ориентированных моделей разработки: MVC (Model-View-Controller), композитный шаблон, трехуровневую архитектуру, маппинг данных. В новой версии предпринята работа по упрощению построения проектов на базе готовых компонентов, интеграции с web-сервисами. С целью повышения безопасности переработан API фильтров входящих данных, улучшен модуль для управления пользователями.
