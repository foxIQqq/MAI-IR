Финальный релиз проекта m0n0wall 1.3

После трех лет разработки представлен финальный релиз m0n0wall 1.3, мини-дистрибутива на базе FreeBSD для создания сетевых шлюзов. Дистрибутив снабжен простым и удобным web-интерфейсом для настройки всех параметров системы (скриншоты), поддерживает сохранение всей конфигурации в виде единого XML файла. Из возможностей можно отметить: поддержка работы в качестве беспроводной точки доступа, 802.1Q VLAN, firewall, NAT, ограничение трафика, мониторинг трафика с генерацией SVG графиков, SNMP-агент, DNS cache, DynDNS клиент, IPSec, клиент/сервер для PPTP VPN, PPPoE, DHCP. Для загрузки доступны 4 установочных образа: embedded (7.11 Мб) для плат Soekris и PC Engines; generic-pc (8.41 Мб) для установки на CF/HDD/USB Flash накопители и использования на обычных ПК; cdrom (17.53 Мб) для использования в роли LiveCD; generic-pc-vm (8.51 Мб) образ виртуальной машины vmWare для тестирования; Основные новшества m0n0wall 1.3: Переход на кодовую базу FreeBSD 6.4 (прошлая ветка m0n0wall 1.2 базировалась на FreeBSD 4.11); Поддержка платформ net45xx, net48xx и WRAP консолидирована в одном установочном образе "embedded"; Работа сетевых мостов организована через модуль if_bridge; Поддержка IPv6 (включается отдельно, через вкладку "advanced setup"); В межсетевом экране реализована возможность работы с IPSec трафиком; Реализована поддержка создания динамических туннелей, IPsec NAT-T, и DPD (Dead peer detection); При обновлении с версии m0n0wall 1.2 необходимо вначале установить 1.3b7, а уже потом обновить его до релиза m0n0wall 1.3 (так как рабочий MFS образ, созданный для версии 1.2 слишком мал для прямого обновления до версии 1.3).
