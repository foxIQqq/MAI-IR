Вышел Linuх дистрибутив VortexBox 1.2

Вышел релиз Linuх дистрибутива VortexBox 1.2, основанного на Fedora 12 и предназначенного для быстрого создания серверов для организации музыкального вещания в сети, сетевых хранилищ музыки или для превращения любого компьютера со звуковой картой в интернет-радио. Размер iso-образа дистрибутива составляет 537 Мб. В состав VortexBox 1.2 включена новая версия пакета Logitech SquuezeBox Server 7.4.2, исправлено несколько критических ошибок в VortexBox Player, добавлен новый GUI-интерфейс для установки пакетов, улучшены средства перекодирования mp3-файлов, упрощен процесс установки Sonos web GUI для использования дистрибутива в качестве сетевого хранилища для плееров Sonos. В качестве музыкального плеера по умолчанию в дистрибутиве используется программа собственной разработки VortexBox Player, представляющая собой надстройку над mpd (Music Player Daemon). Программа отличается поддержкой аппаратных интернет-плееров SqueezeBox и Sonos, может работать со SqueezeBox совместимыми серверами, поддерживает вывод звука как на локальную звуковую карту, так и на аудиоустройства, подключенные через USB и S/PDIF порты. Управление работой системы производится через удобный web-интерфейс. Из возможностей можно отметить функцию автоматического копирования и перекодирования данных при вставке в привод аудио-CD, композиции сохраняются локально в форматах FLAC или MP3 и затем становятся доступными для потокового вещания. При необходимости, через обращение к музыкальным online-сервисам, для музыкальных файлов автоматически заполняются ID3 тэги и загружаются изображения обложек альбомов. Поддерживается возможность организации работы файлового хранилища музыки, доступного по протоколам SMB и NFS. Дистрибутив может выступать в качестве бэкенд-сервера для XBMC (XBOX Media Center), AppleTV и большого числа аппаратных интернет-плееров. Исходные тексты всех созданных в рамках проекта компонентов распространяются в рамках лицензии GPLv3 и доступны в виде пакетов для Fedora 12.
