Вышел релиз комьюнити сборки MySQL 5.1.41

Вышел релиз комьюнити сборки MySQL 5.1.41, содержащий исправление 35 ошибок, из которых 2 ошибки приводили к краху сервера: выполнение "ALTER TABLE ... ADD PARTITION" при истечении лимита на число открытых файлов и переименование столбцов в InnoDB Plugin с последующим созданием индекса для переименованного столбца. Из улучшений, добавленных в новой версии, можно отметить обновление InnoDB Plugin до версии 1.0.5, которая имеет статус кандидата в релизы. Из добавленных в InnoDB Plugin новшеств, можно отметить: Появление настроек для определения методов помещения блоков данных в промежуточные буферы (innodb_old_blocks_pct и innodb_old_blocks_time); Для команды SHOW STATUS добавлены две новых переменные: Innodb_buffer_pool_read_ahead и Innodb_buffer_pool_read_ahead_evicted.
