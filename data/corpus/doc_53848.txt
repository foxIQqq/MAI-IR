Открыты исходные тексты GitHub Docs

GitHub объявил об открытии исходных текстов, обеспечивающих работу сервиса docs.github.com, а также опубликовал размещённую там документацию в формате Markdown. Код может использоваться для создания интерактивных разделов для просмотра и навигации по документации проекта, изначально оформленной в формате Markdown и переведённой на разные языки. Пользователи также могут предлагать свои правки и новые документы. Кроме GitHub указанный код также используется проектами Atom и Electron для организации структурированного доступа к документации. Код написан на JavaScript и открыт под лицензией MIT, а документация и прочие данные доступны под лицензией CC-BY.
