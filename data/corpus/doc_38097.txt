Выпуск десктоп-ориентированной ОС PC-BSD 9.2

Увидел свет релиз десктоп-ориентированной операционной системы PC-BSD 9.2, основанной на кодовой базе FreeBSD 9.2. В качестве пользовательского окружения по умолчанию предлагается KDE 4.11. Для систем с ограниченными ресурсами предусмотрена возможность использования окружений GNOME2, LXDE и Xfce. Развитие PC-BSD курирует компания iXsystems, которая активно участвует в развитии FreeBSD, например, способствует улучшению поддержки ZFS и спонсирует многие инициативы FreeBSD Foundation. Для загрузки подготовлены iso-образы инсталляционных DVD (3.5 Гб), USB Flash (на выбор 1.4 Гб и 3.3 Гб), а также несколько видов Live-систем и образов для запуска внутри виртуальных окружений VirtualBox и VMware. Сборки доступны только для архитектуры AMD64 (x86_64). В качестве минимальных требований к системе отмечается CPU Pentium II и 512 Мб ОЗУ. Система PC-BSD снабжена графическим инсталлятором, системой автоопределения установленного оборудования и набором универсальных графических конфигураторов. Инсталлятор поддерживает создание шифрованных разделов и установку с использованием файловой системы ZFS. Программы упакованы в собственный формат пакетов - PBI, суть которого состоит в избавлении от зависимостей, которое достигается посредством помещения всех необходимых библиотек в каждый пакет. Поддерживается автоматическая загрузка обновлений, установка пакетов через один клик через сервис AppCafe и установка стандартных FreeBSD-портов. Кроме изменений, свойственных FreeBSD 9.2, в новой версии PC-BSD отмечается несколько специфичных улучшений: Полностью переписана утилита Life Preserver, позволяющая в плановом порядке периодически создавать снапшоты ZFS и при необходимости восстанавливать отдельные файлы, зеркалировать данные на другой диск или реплицировать изменения на другую систему по SSH для поддержания актуальной резервной копии; В инсталлятор добавлена поддержка восстановления всей системы из реплицированной резервной копии; Загрузочные ZFS-окружения, созданные командой beadm, автоматически добавляются в меню загрузчика GRUB2. В качестве загрузочного окружения может быть использован снапшот ZFS с определённым срезом состояния системы. Например, перед обновлением пакетов можно создать снапшот и в случае проблем загрузиться с использованием состояния ФС до установки обновлений; Новый графический интерфейс для управления загрузчиком, позволяющий в одном месте управлять загрузочными окружениями ZFS и настраивать меню GRUB; В дополнение к PBI и AppCafe в PC-BSD интегрирована поддержка пакетного менеджера pkgng и обеспечена возможность использования pkgng-репозитория пакетов. В репозитории представлен набор постоянно обновляемых бинарных пакетов со свежими версиями всех приложений, доступных через дерево портов FreeBSD. Как правило, обновление пакетов производится два раза в неделю; Задействование сети доставки контента ScaleEngine для организации загрузки сборок, обновлений и пакетов; Новая версия интерфейса управления пакетами AppCafe, в которой увеличена скорость работы и добавлена поддержка управления PBI; Переход на использование GitHub в качестве основной платформы разработки PC-BSD и TrueOS (серверная редакция PC-BSD); Обновлён интерфейс для управления работой изолированных окружений Warden. Добавлена поддержка установки через GUI различных опций для Jail-окружений, возможность создания шаблонов Jail-окружений на основе разных версий FreeBSD, поддержка Jail-окружений с задействованием средств виртуализации VIMAGE, возможность создания Jail-окружений без привязки адреса IPv4 или IPv6; В графическом интерфейсе управления пакетами реализован дополнительный расширенный режим, позволяющий полностью контролировать все аспекты управления пакетами и обновлениями; Добавлена поддержка шифрования домашних директорий с использованием файловой системы PEFS (Private Encrypted File System) с применением в качестве ключа пароля для текущего аккаунта и обеспечением доступа к расшифрованному контенту только во время активности сеанса пользователя; Существенно сокращено время загрузки; Добавлена поддержка установки бинарных обновлений до различных веток PC-BSD с использованием сервера freebsd-update.
