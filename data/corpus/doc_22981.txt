Nokia провела ребрендинг технологий, связанных с Qt

Компания Nokia анонсировала изменение названия подразделения Qt Software, занимающегося разработкой Qt, вместо которого отныне будет фигурировать "Qt Development Frameworks". Nokia также намерена избавиться от сайта qtsoftware.com, вся официальная информация о Qt перенесена на сайт qt.nokia.com, который теперь будет отправной точкой распространения Qt. Дэниель Киглберг, директор по продажам, маркетингу и сервисам компании Nokia, в качестве причин проведения ребрендинга назвал желание увеличить число заинтересованных в использовании тулкита Qt разработчиков мобильных устройств, чему может помочь предпринятое укрепление ассоциации Qt с брендом Nokia. Nokia заинтересована в форсировании продвижения библиотеки Qt в качестве инструмента для создания приложений для мобильных устройств, желая добиться популярности, имеющейся среди разработчиков десктоп-приложений. Планы на будущее компании Nokia связаны с переводом Qt в разряд основного фреймворка для разработки приложений для платформ S60 и Maemo. Начиная с версии Symbian^4 и Maemo 6 "Harmattan", официальным API для создания пользовательского интерфейса станет Qt.
