Код X.Org 7.5 перешел на стадию заморозки перед релизом

После многочисленных переносов сроков выпуска, код X.Org 7.5 в процессе подготовки релиза перешел на стадию заморозки. В соответствии с новыми планами, релиз ожидается в конце сентября. К сожалению, решено не включать в X Server 1.7 подсистему XKB2, которая отложена до выпуска X Server 1.8. Тем не менее в X.Org 7.5 / X Server 1.7 все же появится поддержка технологии Multi-Pointer X и переработанной подсистемы ввода X Input 2, что позволит организовать работу нескольких независимых устройств ввода, например, несколько управляемых разными мышами курсоров на экране или ввод в разные окна с разных клавиатур.
