Вышло обновление Linux XP SMB Server

Компания "ТрастВерс" объявила о выходе пакета обновлений Service Release 1 для программного продукта Linux XP SMB Server, ориентированный на развертывание внутреннего сервера малого предприятия или рабочей группы, например, корпоративного веб-сервера, обслуживающего сайт компании и почтовую систему (управление через web-интерфейс). Пакет обновлений позволяет пользователям системы Linux XP SMB Server актуализировать пакетную базу своих рабочих серверов и получить новые дополнительные возможности, предоставляемые обновленными пакетами программ. Основные новшества Linux XP SMB Server Service Release 1: Включен пакет wine-lxp (позволяет запускать Windows-приложений в системе Linux XP, такие как 1С, Консультант+ и т.д.). Полноценная поддержка FUSE. Включение в состав дистрибутива GnuPG 2. Добавлена поддержка RAID 4/5/10 в dm-raid для 34 новых аппаратных устройств. Обновлены драйверы для 43 RAID контроллеров; Ключевые особенности продукта Linux XP SMB Server: Средства групповой работы - WEB-почта, расписание, совместная разработка документов; Простая интеграция в корпоративные сети, поддержка доменов MS, создание собственного домена; Trustverse Directory – интегрированная служба централизованного управления инфраструктурой, пользователями, сервисами; Русскоязычная документация от разработчика и обучение. Linux XP основан на переработанной пакетной базе Fedora Linux и RHEL. Без активации десктоп-редакция дистрибутива работоспособна в течение 30-дневного пробного периода, для серверной редакции подобных ограничений нет. Исходные тексты измененных пакетов как серверной, так и десктоп редакции дистрибутива доступны для свободной загрузки с сайта downloads.linux-xp.com, iso-образы для ознакомления можно загрузить с данной страницы.
