OpenOffice.org pro 3.2.1 доступен для загрузки и использования

Компания Инфра-Ресурс выпустила оптимизированную для русскоязычных пользователей сборку офисного пакета OpenOffice.org pro 3.2.1. Сборка OpenOffice.org pro 3.2.1 выполнена для платформ GNU/Linux i386 и x86-64 в пакетах DEB, RPM и Generic, Windows и Windows Portable. Офисный пакет OpenOffice.org pro отличается от базовой версии OpenOffice.org большей совместимостью с кириллическими документами, содержит все необходимые модули для проверки орфографии, обладает множеством других отличий, повышающих комфорт работы русскоговорящих пользователей. OpenOffice.org pro 3.2.1, содержит все реализованные в прошлых версиях улучшения, а также содержит следующие новшества: Общие улучшения Улучшена совместимость с файлами MS Office; Улучшена поддержка VBA; Применён новый набор иконок для файлов документов; Произведена оптимизация для работы на нетбуках; Улучшения Calc Добавлена Вкладка-Кнопка [ + ] для быстрого добавления новых листов в Calc; Улучшена работа со ссылками в нотации R1C1; Улучшения Writer В меню «Правка» добавлена команда «Вставка из веба», копирующая из буфера обмена содержание HTML страниц с одновременной загрузкой изображений в документ; Добавлены новые автоматизированные шаблоны для сотрудников электронных магазинов: Письмо заказное с уведомлением; Комплект почтовых бланков для бандеролей и посылок; Улучшения StartCenter Исправлены ссылки на русскоязычные ресурсы OpenOffice.org;
