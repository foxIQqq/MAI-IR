В очередном обновлении игры Quake Live представлена поддержка Linux

Компания Id Software завершила портирование на платформы Linux и MacOS X игры Quake Live, бесплатного варианта игры Quake III Arena на базе движка id Tech 3 для проведения командных игр не выходя из web-браузера. В настоящий момент на сайте quakelive.com доступно для загрузки специальное дополнение для браузера Firefox, позволяющее пользователям ОС Linux подключиться к игровому процессу.
