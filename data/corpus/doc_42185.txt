Microsoft выпустил PowerShell DSC для Linux

Компания Microsoft анонсировала доступность платформы PowerShell DSC (Desired State Configuration) для Linux. PowerShell DSC представляет собой набор расширений к PowerShell для централизованного управления конфигурацией, выполняя действия похожие на такие системы, как Chef и Puppet. Продукт доступен в пакетах RPM и DEB, и нацелен на работу совместно с реализациями промышленного стандарта OMI (Open Management Infrastructure), что позволяет использовать интерфейс PowerShell для управления конфигурацией Windows, Linux и различного сетевого оборудования, такого как коммутаторы. Код системы написан на C++ и открыт под лицензией MIT. Из доступных в первом выпуске PowerShell DSC возможностей отмечается поддержка дистрибутивов CentOS, Debian GNU/Linux, Oracle Linux, Red Hat Enterprise Linux, SUSE Linux Enterprise Server и Ubuntu Server, поставка встроенного набора ресурсов для работы с типовыми конфигурациями Linux, поддержка методов Push (WS-Man) и Pull (HTTP/HTTPS) для доставки конфигурации, автоматическая защита от смещения конфигурации (drift). Из доступных для использования ресурсов можно выделить: развёртывание отдельно подготовленных архивов, управление переменными окружения, управление файлами и директориями, слежение за правами доступа, синхронизация файлов и директорий, проверка наличия заданного контента в файле, управление группами и пользователями, установка пакетов и обновлений (yum, apt, zypper), организация выполнения пользовательских скриптов, управление системными сервисами, управление ssh-ключами.
