Новая система для подсчета трафика для FreeBSD - Admin Panel

После месяца разработки представлен проект Admin Panel, в рамках которого создана система для подсчета трафика и управления пользователями в сети. Реализованные возможности: Подсчет трафика через ng_ipacct. Синхронизация пользователей с доменным контроллером. Квоты с уведомлением пользователя по истечению квоты, по email. Группы пользователей(квоты накладываются на группу). Аутентификация пользователей в доменном контроллере. Списки людей которые могут иметь доступ на сервисы ICQ, Mail.ru Agent.
