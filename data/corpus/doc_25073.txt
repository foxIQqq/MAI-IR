Вышел релиз NexentaStor 2.2.1, дистрибутива для создания сетевых хранилищ

Вышел релиз NexentaStor 2.2.1, дистрибутива для создания сетевых хранилищ, сочетающего в себе ядро OpenSolaris и программное окружение Ubuntu 8.04. В качестве файловой системы используется ZFS. От коммерческой версии "Developer Edition" отличается ограничением размера поддерживаемых хранилищ, которые не могут превышать 4 Тб. Версия 2.2.1 позиционируется как корректирующий релиз с исправлением накопившихся ошибок. Сборка NexentaStor начиная с данного выпуска произведена с использованием нового инструментария NABAT (Nexenta Automated Build and Testing). Из новшеств добвлен плагин Auto-sync, улучшена поддержка LDAP и сервиса Delorean. Из добавленных в NexentaStor 2.2 улучшений можно отметить: Добавление средств для антивирусной проверки хранимого контента (используется ClamAV); Поддержка бэкапов СУБД Oracle и MySQL через создание непротиворечивых снапшотов ФС; Поддержка устройств хранения Citrix StorageLink; Поддержка асинхронной обратной репликации; Реализация механизма поиска данных, учитывающего содержимое отдельных снапшотов; Поддержка удаленного доступа.
