Началось тестирование оболочки KDE Plasma 5.2. Выпуск KDE Applications 14.12.1

Доступна для тестирования первая бета-версия пользовательской оболочки Plasma 5.2. Окружение Plasma 5 примечательно унификацией режимов работы интерфейса, использованием платформы KDE Frameworks 5 и библиотеки Qt 5, применением OpenGL/OpenGL ES для ускорения отрисовки. С особенностями Plasma 5 можно познакомиться в анонсах первого и второго выпусков. Протестировать новый выпуск можно через Live-сборку от проекта Kubuntu, пакеты для различных дистрибутивов можно найти на данной странице. Релиз ожидается 27 января. Ключевые улучшения: Для организации входа и переключения между сеансами задействован дисплейный менеджер SDDM (Simple Desktop Display Manager). Для SDDM разработчиками KDE подготовлена стилизованная под Plasma 5 тема оформления, а в конфигуратор KDE добавлен модуль для настройки SDDM. SDDM написан на языке C++ и состоит из бэкенда, работающего в режиме фонового процесса, и фронтэнда, написанного с использованием Qt/QML и отличающегося большой гибкостью в плане смены оформления. SDDM практически не ограничивает дизайнеров и позволяет создать любое оформление, построенное вокруг простого программного интерфейса, предоставляющего несколько callback-вызовов для аутентификации, перехода в спящий режим, завершения работы и т.п. Использование QtQuick для построения интерфейса даёт возможность применять анимационные эффекты и задействовать аппаратное ускорение; В базовый состав добавлена системы упрощённой настройки многомониторных конфигураций KScreen. KScreen отличается кардинальным упрощением процесса настройки параметров размещения десктопа на нескольких мониторах, позволяя в наглядном режиме на лету менять настройки без использования классических предварительных конфигураторов. KScreen, насколько это возможно, автоматизирует процесс настройки - достаточно подключить второй монитор и пространство рабочего стола автоматически будет расширено вправо на новый экран. Для каждой выбранной конфигурации по умолчанию будет выбрано максимально возможное экранное разрешение. При закрытии крышки ноутбука, если к нему подключён внешний монитор, экран этого монитора автоматически становится первичным, а экран ноутбука временно отключается. После отключения внешнего экрана настройки запоминаются и при дальнейшем подключении экрана используется ранее выбранный режим; В состав включен BlueDevil, набор компонентов для управления подключением Bluetooth-устройств. Например, через BlueDevil можно настроить мышь и клавиатуру с интерфейсом Bluetooth, организовать сопряжение системы с Bluetooth-устройством, просмотреть содержимое файловой системы устройства и организовать получение и отправку файлов; Интегрирован пакетный менеджер Muon, предоставляющий интерфейс для поиска, установки и обслуживания программ, поставляемых в доступных для текущего дистрибутива репозиториях пакетов; Добавлен KSSHAskPass, интерфейс для организации ввода пароля в процессе установки SSH-соединений на основе открытых ключей, доступ к которым защищён паролем; Представлен новый модуль GTK Application Style, предоставляющий средства для адаптации темы оформления программ, написанных с использованием фреймворка GTK+, для их оптимального сочетания с интерфейсом KDE. Добавлена библиотека KDecoration, предоставляющая простые и надёжные средства для создания тем оформления для оконного менедежра KWin; Реализована функций отмены изменений (Undo) в размещении элементов на рабочем столе. Например, теперь легко может быть восстановлен случайно удалённый с рабочего стола виджет; Улучшены средства сортировки результатов поиска KRunner, которые теперь группируются по категориям. Для быстрого поиска на компьютере нужно нажать Alt-space. Добавлены инструменты для управления воспроизведением музыки из KRunner, например, можно сменить проигрываемую композицию нажав Alt-Space и введя слово "next"; Представлен новый внешний вид обрамления окон в теме оформления Breeze. Детально проработаны пиктограммы. Добавлена новая белая тема оформления курсора мыши. Добавлены новые виджеты Plasma (плазмоиды): web-браузер, игра 15 puzzle и режим обзора рабочего стола (show desktop); В альтернативном меню приложений Kicker обеспечена возможность редактирования меню и установки приложений не покидая меню. Одновременно представлен корректирующий выпуск набора пользовательских приложений KDE Applications 14.12.1, в котором исправлено более 50 ошибок, внесены небольшие улучшения и проведена работа по обновлению переводов. Наиболее важные исправления затронули программу Ark, систему UML-моделирования Umbrello, просмотрщик документов Okular, обучающий пакет Artikulate и клиент удалённого доступа к рабочему столу KRDC. В рамках новой раздельной схемы подготовки выпусков компонентов KDE также подготовлены корректирующие обновления Plasma Workspaces 4.11.15, KDE Development Platform 4.14.4 и Kontact Suite 4.14.4.
