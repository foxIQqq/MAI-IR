Порт библиотеки Qt4 для ОС Haiku достиг стабильного состояния

Пользователи ОС Haiku, продолжающей развитие технологий BeOS, получили возможность сборки и использования любых графических приложений, написанных с использованием Qt, - разработчики проекта Qt-Haiku сообщили о достижении стабильного состояния порта библиотеки Qt для Haiku. Для установки Qt 4.5.1 в Haiku подготовлен бинарный пакет, кроме самой библиотеки на сайте проекта создан каталог готовых сборок Qt-приложений, в котором пока присутствуют несколько игр, программа для работы с периодической таблицей химических элементов QPeriodicTable, web-браузер Arora, клиент для мгновенного обмена сообщениями qutIM, утилита для наглядной оценки различий в нескольких файлах kdiff3, программа для просмотра CHM файлов и система для ведения заметок Quicky.
