Компания Oracle существенно повысила цены на коммерческую версию MySQL

Компания Oracle ввела в строй новые расценки на коммерческую версию MySQL. Отныне доступны только три варианта коммерческой поддержки: MySQL Standard Edition - 2000 долларов в год; MySQL Enterprise Edition - 5000 долларов в год; MySQL Cluster Carrier Grade Edition (с движком Cluster NDB) - 10000 долларов в год. Базовый и серебряный планы поддержки, стоимостью 599 и 1999 долларов, были устранены. Увеличение минимальной цены поддержки более чем в три раза значительно снижает привлекательность MySQL для небольших и средних компаний. Более того, все цены приведены в расчете использования MySQL на сервере с числом CPU не превышающим 4, клиентам желающим использовать MySQL в системах с пятью и более CPU должны уточнить цену непосредственно в Oracle. Никаких изменений в распространении свободного MySQL Community Server не отмечено. На повышение цен сразу отреагировала компания SkySQL, основанная уволившимися из Oracle разработчиками MySQL и возглавляемая Ульфом Санбергом (Ulf Sandberg), бывшим вице-президентом по предоставлению глобальных сервисов MySQL. SkySQL заявила, что готова предоставить качественную поддержку MySQL серверов по более низким ценам и с некоторыми дополнительными опциями. Дополнение: компания Oracle опубликовала свое видение ситуации, в которой указано, что базовый план поддержки за 599$ не пользовался спросом, подразумевал решение только двух инцидентов в год и оказание услуг технической поддержки в рабочее время. Новый план поддержки за 2000$ в год подразумевает круглосуточную поддержку и неограниченное число инцидентов. Ранее за такой сервис компании вынуждены были платить 3000$ вместо $2000, т.е. для крупных клиентов цена стала на 30% меньше.
