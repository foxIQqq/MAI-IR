Вышел Calculate Directory Server 9.5

Calculate Directory Server - это свободная операционная система на базе Gentoo Linux, представляющая собой LDAP-сервер для централизованного хранения настроек приложений, профилей и настроек пользователей, прав доступа в едином сетевом хранилище. Возможности: Поддерживает работу как с Windows, так и с Linux клиентами. Имеет встроенные средства для настройки сервисов LDAP, Mail, Samba, FTP, Jabber. Система распространяется на LiveCD (500 Мб), позволяя оценить все возможности системы без инсталляции на ПК. Поддерживается установка как на HDD, так и на USB-HDD. Основные изменения в версии 9.5: Добавлена репликация почтовых записей LDAP сервера. Добавлена утилита cl-info для отображения состояния учетных записей пользователей и групп в LDAP сервере. Добавлена LZMA компрессия загрузочного образа диска. Переход на менеджер загрузки OpenRC. Поддерживается установка на файловые системы: ext2, ext3, ext4, jfs, reiserfs, xfs. Для получения технической поддержки на канале #calculate сервера FreeNode достаточно набрать команду irc. На LiveCD добавлен автологин пользователем root. Добавлена поддержка автоматического разбиения жесткого диска (calculate -d /dev/sdX). Добавлены программы: fetchmail, screen, vnstat. Состав: Bind 9.4.3-P2, Kernel 2.6.28.9, OpenLDAP 2.4.16, Postfix 2.5.5, ProFTPD 1.3.2, Samba 3.0.33.
