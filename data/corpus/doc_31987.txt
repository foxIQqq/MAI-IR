Первый выпуск дистрибутива IPFire для платформы ARM

Для платформы ARM представлен первый выпуск сборки дистрибутива IPFire, предназначенного для создания маршрутизаторов, VPN-шлюзов, серверов и межсетевых экранов. По словам разработчиков, выпуск имеет статус бета-версии. Размер загрузочного образа 96 Мб. Возможна как установка дистрибутива на встроенный накопитель, так и загрузка с USB Flash или SD-карты. Из поддерживаемых устройств отмечены компьютеры SheevaPlug, DreamPlug и GuruPlug+, выпускаемые компанией Globalscales на базе CPU Marvell Feroceon 1.2 GHz (ARMv5tel). В будущем планируется адаптировать дистрибутив для процессоров ARMv7hl. По функциональности вариант для платформы ARM идентичен версии для архитектуры x86. IPFire отличается предельно простым процессом установки и организацией настройки через интуитивно понятный web-интерфейс, изобилующий наглядными графиками. Система модульная, кроме базовых функций пакетной фильтрации и управления трафиком для IPFire доступны модули с реализацией системы для предотвращения атак на базе Snort, для создания файлового сервера (Samba, FTP, NFS), почтового сервера (Cyrus-IMAPd, Postfix, Spamassassin, ClamAV и Openmailadmin) и сервера печати (CUPS), организации VoIP шлюза на базе Asterisk и Teamspeak, создания беспроводной точки доступа, организации потокового аудио и видео-сервера (MPFire, Videolan, Icecast, Gnump3d, VDR). Для установки дополнений в IPFire используется специальный пакетный менеджер Pakfire.
