Доступен OpenIndiana 2016.10, продолжающий развитие OpenSolaris

После 6 месяцев разработки подготовлен выпуск проекта OpenIndiana 2016.10, в рамках которого независимым сообществом развивается полностью свободный дистрибутив, пришедший на смену бинарному дистрибутиву OpenSolaris, развитие которого было прекращено компанией Oracle. OpenIndiana предоставляет пользователю рабочее окружение, построенное на базе свежего среза кодовой базы проекта Illumos. Непосредственно разработка технологий OpenSolaris продолжается проектом Illumos, в котором развивается ядро, сетевой стек, файловые системы, драйверы, а также базовый набор пользовательских системных утилит и библиотек. Для загрузки сформировано три вида iso-образов - серверная редакция с консольными приложениями (637 Мб), минимальная сборка (396 Мб) и сборка с графическим окружением MATE (1.5 Гб). Основные изменения в OpenIndiana 2016.10: Прекращено формирование сборок для 32-разрядных систем x86 (возможность запуска 32-программ на 64-разрядных системах сохранена); Вместо устаревшей версии GNOME 2 задействовано окружение рабочего стола MATE 1.14. В качестве менеджера входа в систему вместо GDM применён LightDM; Вместо GRUB задействован загрузчик от проекта FreeBSD, применение которого позволило расширить возможности текстового инстяллятора и обеспечить поддержку установки системы на разделы с ZFS RAIDZ/mirror; В инсталлятор добавлена генерация рабочих системных образов для размещения на USB-накопителях, которые можно напрямую скопировать на Flash; Для OpenIndiana портирован KMS-драйвер для видеокарт Intel, обеспечивающий переключение видеорежимов на уровня ядра. Поддержка KMS необходима для использования свежих версий видеодрайверов Intel, в которых была удалена поддержка переключения видеорежимов на пользовательском уровне (UMS) и оставлена только поддержка KMS (Kernel Mode Setting). Драйвер пока ограничен в функциональности и поддерживает только GPU Intel 6 и 7 поколения. По умолчанию обеспечена поставка Python 2.7. Ранее предлагаемый Python 2.6 удалён из системы. Добавлена большая подборка модулей для Python 3.4; Из системы удалён Sun SSH, который больше не поддерживается. Вместо Sun SSH теперь применяется OpenSSH; Из-за проблем с сохранением документов из репозиториев удалён OpenOffice, вместо которого предлагается использовать LibreOffice; В состав включен FUSE-модуль ntfs-3g для монтирования файловой системы NTFS; Обновлены версии пользовательских программ, в том числе Mesa 12.0.3, CUPS 1.7.5, Inkscape 0.91, Thunderbird 45.4.0, Firefox 45.4.0, mc 4.8.18, драйвер NVIDIA 340.96, tmux 2.3, nmap 7.31 и т.д. В репозитории доступен Xorg-сервер 1.18.4; Обновлены средства для разработчиков: GCC 5.4 (опционально, в дополнение к основному GCC 4.9), GNU Make 4.2.1, CMake 3.5.2, OpenJDK 8 1.8.92 (опция, по умолчанию оставлен OpenJDK 7), Golang 1.6, Node.js 6.9.1, Lua 5.3, PHP 7.0 (удалены PHP 5.4 и 5.5), Tcl/Tk 8.5.19, OCaml 3.11.2, git 2.9.2, mecurial 3.8.4; Обновлено серверное ПО: Sendmail 8.15.2, Postfix 3.1.1, Samba 4.4.6 (samba 3 удалён), MongoDB 3.3, nginx 1.11.5, Tor 0.2.8.8, isc dhcp 4.3.4, zabbix-agent 2.2.13, Bind 9.10.4-P3.
