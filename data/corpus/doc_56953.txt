Выпуск платформы Lutris 0.5.10 для упрощения доступа к играм из Linux

После шести месяцев разработки состоялся выпуск игровой платформы Lutris 0.5.10, предоставляющей инструменты для упрощения установки, настройки и управления играми в Linux. Код проекта написан на языке Python и распространяется под лицензией GPLv3. Проектом поддерживается каталог для быстрого поиска и установки игровых приложений, позволяющий через единый интерфейс одним кликом запускать в Linux игры, не заботясь об установке зависимостей и настройках. Runtime-компоненты для запуска игр поставляются проектом и не привязываются к используемому дистрибутиву. Runtime представляет собой независимый от дистрибутива набор библиотек, включающий компоненты из SteamOS и Ubuntu, а также различные дополнительные библиотеки. Предоставляется возможность установки игр, распространяемых через сервисы GOG, Steam, Epic Games Store, Battle.net, Origin и Uplay. При этом сам по себе Lutris выступает лишь в роли посредника и не продаёт игры, поэтому для коммерческих игр пользователь должен самостоятельно приобрести игру в соответствующем сервисе (бесплатные игры можно запускать одним кликом из графического интерфейса Lutris). Каждая игра в Lutris привязывается к сценарию загрузки и обработчику, описывающему окружение для запуска игры. В том числе предоставляются готовые профили с оптимальными настройками для запуска игр под управлением Wine. Кроме Wine игры могут запускаться при помощи эмуляторов игровых консолей, таких как RetroArch, Dosbox, FS-UAE, ScummVM, MESS/MAME и Dolphin. Ключевые новшества Lutris 0.5.10: Добавлена поддержка запуска Lutris на игровой консоли Steam Deck. В настоящее время протестирована установка из репозиториев Arch Linux и AUR, что требует перевода системного раздела в режим записи и переустановки после применения значительных обновлений SteamOS. В дальнейшем планируется подготовить и самодостаточный пакет в формате Flatpak, на работу которого не будут влиять обновления Steam Deck. Предложен новый раздел для ручного добавления игр. В разделе предложены интерфейсы для: добавления и настройки игр, уже установленных в локальной системе; сканирования каталога с играми, ранее установленными через Lutris, но не проинспектированными в клиенте (при выполнении операции осуществляется сопоставление имён каталогов с идентификаторами игр); установки Windows-игр с внешнего носителя; установки при помощи YAML-инсталляторов, имеющихся на локальном диске (GUI-версия для флагов "--install"); поиска в библиотеке игр, предлагаемой на сайте lutris.net (ранее данная возможность предлагалась во вкладке "Community installers"). Добавлены компоненты для интеграции с сервисами Origin и Ubisoft Connect. По аналогии с поддержкой каталога Epic Games Store для работы новых модулей интеграции требуется установка клиентов Origin и Ubisoft Connect. Добавлена опция для добавления игр Lutris в Steam. Реализована поддержка формата обложек (coverart). Обеспечена загрузка недостающих компонентов во время запуска. Для Linux и Windows игр задействован отдельный кэш шейдеров на системах с GPU NVIDIA. Добавлена опция для поддержка античит-системы BattleEye. Добавлена возможность загрузки патчей и DLC для игр GOG. Добавлены флаги "--export" и "--import" для экспорта и импорта игр. Добавлены флаги "--install-runner", "--uninstall-runners", "--list-runners" и "--list-wine-versions" для управления runner-ами. Изменено поведение кнопки "Stop", удалено действие по завершению всех процессов Wine. На GPU NVIDIA обеспечено отключение опции Gamescope. По умолчанию включён механизм fsync. Дополнительно можно отметить, что для игровой консоли Steam Deck, основанной на Linux, подтверждена поддержка 2039 игр. 1053 игр помечены как проверенные вручную сотрудниками Valve (Verified), а 986 как поддерживаемые (Playable).
