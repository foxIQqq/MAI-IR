Вышел Mageia 3, развиваемый сообществом форк Mandriva Linux

После года разработки представлен третий релиз Linux-дистрибутива Mageia, в рамках которого независимым сообществом энтузиастов развивается форк проекта Mandriva. Для загрузки доступны 32- и 64-разрядные DVD-сборки (3.7 Гб), универсальная CD-сборка (699 Мб), минималистичный образ для установки по сети (40 Мб) и набор LiveCD- и LiveDVD-сборок на базе GNOME и KDE с различными наборами локализации. Выпуск посвящён Евгению Додонову, трагически погибшему в результате дорожно-транспортного происшествия в июле прошлого года. Ключевые улучшения: Поставка ядра Linux 3.8; Переход на пакетный менеджер RPM 4.11 (в прошлом выпуске использовалась ветка 4.9) и новую версию инструментария urpmi с поддержкой delta-изменений и опции "--downgrade". Для сжатия пакетов задействован формат XZ вместо LZMA; Обновление системного менеджера systemd до выпуска 195. По умолчанию задействованы встроенные в systemd средства ведения системных логов, в качестве опции сохранена возможность установки rsyslog; Из коробки доступен Linux-клиент для сервиса цифровой дистрибуции игр Steam; Каталоги /lib* и /*bin из корня перемещены в /usr, что упрощает монтирование в режиме только для чтения всех файлов уже установленной ОС, а также монтирование снапшотов системы и организацию распространения единых базовых частей ОС на несколько машин по сети; В качестве загрузчика по умолчанию задействован классический GRUB. Для тестирования доступна возможность использования GRUB2. В Harddrake реализована автоматическая установка драйверов при обнаружении подключения новых проводных и беспроводных сетевых адаптеров; В Live-образах задействована файловая система OverlayFS; Улучшение в системе установки: Реорганизовано разбиение пакетов на группы в инсталляторе и программе установки пакетов RPMdrake; В инсталляторе добавлена поддержка управления Btrfs-разделами и установки на разделы с файловой системой Btrfs; В процессе установки добавлена возможность подключения репозиториев nonfree и tainted, с которых размещены несвободные или пересекающиеся с патентами приложения; Улучшена поддержка установки системы в виртуализированных окружениях; Обновление версий приложений (всего в дистрибутиве доступно 10500 исходных пакетов): Поставка новых версий десктоп-окружений: KDE 4.10.2, GNOME 3.6, Xfce 4.10, LXDE 0.5.5, Enlightenment E17, Razor-qt 0.5.2. В репозиториях также доступны для установки оконные менеджеры dwm, scrotwm, i3, pekwm, icewm, awesome, openbox, fvwm2 и Window Maker; Медиаплееры: VLC 2.0.6, xine 1.1.21 (+ GUI kaffeine и xine-ui), XBMC 12.1, MythTV 0.26, Miro 5.0.4; Браузеры: Firefox ESR 17.0.6, Chromium 26; Офисные пакеты: LibreOffice 4.0.3, Calligra 2.6.2; Проприетарные приложения: Skype 4.1.0.20, драйверы для видеокарт NVIDIA 295.17 и fglrx 12.104 (AMD Catalyst 13.4); Растровый графический редактор GIMP 2.8.2; Музыкальные плееры: Amarok 2.7, Cantata music player 0.9.2, mplayer (+ GUI gnome-mplayer, kmplayer, mplayer-gui, smplayer); СУБД MariaDB 5.5.28 и PostgreSQL 9.2; Серверное ПО: apache 2.4.4, nginx 1.2.9, postfix 2.9.6, Cyrus-imapd 2.4.17, Dovecot 2.1.15, puppet 2.7.21, Samba 3.6.15; Средства виртуализации: VirtualBox 4.2.12, QEMU 1.2.0, virt-manager 0.9.5, Xen 4.2.1; Системные компоненты и средства разработки: glibc 2.17, gcc 4.7.2, python 3.3, perl 5.16.3, ruby 1.9.3.p392, rails 3.2.13, php 5.4.13;
