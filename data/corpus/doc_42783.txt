Выпуск IPython 4.0

Доступен релиз IPython 4.0, интерактивной оболочки для языка Python, которая сочетает возможности интерактивной консоли Python и командной оболочки Unix, предоставляет гибкие средства отладки, редактирования кода и визуализации данных. IPython активно используется в научной среде для разработки, обработки данных и интерактивного выполнения приложений, связанных с библиотеками numpy, matplotlib, sympy и scipy. Кроме языка Python через подключение дополнительных модулей в IPython доступны средства разработки на Bash, R, Ruby и Perl. Версия IPython 4.0 примечательная уходом от поставки в форме одного монолитного пакета. Отныне продукт разделён на две части: универсальные компоненты для работы с любыми языками программирования (notebook, qtconsole и т.п.) выделены в пакет Jupyter, а специфичные для языка Python части (интерактивный Python shell, Python kernel, IPython.parallel) продолжили поставляться под именем IPython.
