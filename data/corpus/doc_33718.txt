В Омске состоится мероприятие "Битва дистрибутивов 2012"

12 мая в 14.30 в Омске будет проводиться Битва дистрибутивов: обсуждение и выбор лучшего дистрибутива GNU/Linux. После чего будет обычный Linux Install Fest, когда пользователь сможет установить понравившийся дистрибутив с помощью активистов OmskLUG, получить понравившийся дистрибутивов GNU/Linux, проконсультироваться по тому или иному свободному программному обеспечению. В настоящее время на Битву заявлены дистрибутивы Ubuntu Linux, Debian GNU/Linux, Arch Linux, IPCop. Не заявленным на Битву дистрибутивам будет засчитано техническое поражение. У пользователей других систем есть шанс защитить свой дистрибутив, для этого нужно подготовить слова в его защиту и внятно их озвучить собравшимся зрителям. Мероприятие пройдет в 301 аудитории первого корпуса Омского государственного университета им. Ф.М. Достоевского (г. Омск, пр. Мира, д. 55А). Запланирована видеотрансляция. Для оценки числа посетителей предлагается зарегистрироваться.
