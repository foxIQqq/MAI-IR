Начало альфа-тестирования Linux дистрибутива Mandriva 2010

Выпущена первая альфа версия Linux дистрибутива Mandriva 2010, доступная в виде 32- и 64-рязрядной DVD Free сборки без проприетарных драйверов в комплекте. Это первая сборка Mandriva подготовленная с помощью нового инструментария bcd, вместо ранее используемого mkcd, что должно положительно сказаться на качестве дистрибутива из-за значительного упрощения и усиления процесса тестирования. В состав первой альфа версии включены такие приложения, как KDE 4.3 Beta 2, GNOME 2.27.3, Xfce 4.6.1, Linux ядро 2.6.30, X.org server 1.6.1.901, Koffice 2.0, k3b 2.0 alpha 2, Digikam 1.0 beta 1, Cups 1.4 RC 1. Основные направления развития Mandriva 2010, релиз которого запланирован на 21 октября: Продолжение работы по уменьшению времени загрузки; Переработка загрузочного меню Grub и начального меню инсталлятора (режим защиты от сбоев (failsafe), загрузка в консольном режиме (init3), восстановление после сбоя, проверка целостности инсталляционного диска); Переход на версию набора компиляторов GCC 4.4 для пересборки пакетов; Задействование режима бесшовной графической загрузки через использование Plymouth, для карт не поддерживающих переключение видеорежимов на уровне ядра по прежнему будет использован Splashy; Переход на использование, появившихся в Linux ядре 2.6.30, модулей безопасности TOMOYO Linux, позволяющих привязать политику безопасности процесса к файловому пути, способом похожим с AppArmor. Для управления будет создан специальный GUI интерфейс, который будет интегрирован в интерфейс msec (Mandriva Security Center); Поддержка создания гостевых аккаунтов на лету, при возникновении необходимости предоставления доступа к машине случайному пользователю; Создания набора пакетов с окружением Moblin для субноутбуков, который будет автоматически предлагаться к установке на слабых машинах и при низких экранных разрешениях; В апплет mdkonline будет добавлена возможность информирования об окончании времени поддержки дистрибутива и выпуска обновлений, связанных с безопасностью; Автоматическое определение наличия у пользователя локальных urpmi репозиториев; Linux ядро для построения управляющего Xen окружения (Dom0) будет обновлено с версии 2.6.18 до 2.6.27; Улучшение поддержки гибридных iso-сборок, подходящих как для Flash, так и для CD/DVD; Мультимедиа центр, для управления коллекцией фотографий, музыки и поддержания видеотеки, Elisa переименован в проект Moovida; KDE 3 больше не будет входить в состав дистрибутива как полное десктоп окружение, некоторые пакеты на базе KDE 3 останутся, но в будущем поддержка будет полностью прекращена, как только для KDE 4 будут портированы все самые интересные приложения; Основные программные компоненты будущей версии: Linux ядро 2.6.31, X.org 7.5 (xorg-server 1.7.0 или 1.6.2+ в случае задержки выхода), KDE 4.3.x, GNOME 2.28, GCC 4.4.x, Glibc 2.10.1 или 2.10.2+, Go-OO сборка OpenOffice.org 3.1.
