Консорциум W3C провел тестирование уровня поддержки HTML5 в web-браузерах

Консорциум W3C представил результаты первого официального тестирования поддержки реализации элементов будущего стандарта HTML5 в современных web-браузерах. Наиболее полный комплект поддерживаемых возможностей HTML5 продемонстрировала шестая тестовая сборка Internet Explorer 9, на втором месте Firefox 4 Beta 6, на третьем Google Chrome 7.0.517.41 beta. Например, по данным теста caniuse.com Internet Explorer 9 продемонстрировал поддержку 81% всех элементов HTML5, Firefox - 96%, Safari - 96%, Chrome - 96% и Opera - 77%. Объяснить такое расхождение можно тем, что тестовый комплект W3C содержит проверку только части элементов будущего стандарта, не учитывая такие функции как работа с локальным хранилищем, "web workers", File API, расширенные типы для тега input, WebGL, IndexedDB, WebSocket, Geolocation API.
