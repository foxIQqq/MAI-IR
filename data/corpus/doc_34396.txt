Релиз открытого драйвера для видеокарт Intel - xf86-video-intel 2.20

Вышел релиз свободного видеодрайвера для видеокарт Intel - xf86-video-intel 2.20.0 в котором отмечено более 470 изменений, следом за которым представлен корректирующий релиз 2.20.1 с исправлением 80 ошибок, некоторые из которых помечены как критические. Подавляющее большинство изменений в обоих выпусках связано с поддержкой архитектуры 2D-акселерации SNA, которая базируется на традиционной модели DDX (Device Dependent X) и нацелена на достижение максимальной производительности, которой можно добиться от имеющейся видеокарты, в сочетании с более низкой нагрузкой на CPU по сравнению с архитектурой UXA. Наиболее значительным изменением в новой версии является переход по умолчанию к сборке драйвера с поддержкой SNA (ранее SNA включался только при сборе с опцией "--enable-sna"). Тем не менее для включения SNA требуется явное указание "Option AccelMethod sna" в xorg.conf, иначе по умолчанию используется UXA. Другим заметным улучшением является обеспечение поддержки следующего поколения X.org API для взаимодействия с драйверами, которое будет интегрировано в релиз X Server 1.13. Новый API позволит переключать на лету вывод текущего активного X-сервера между разными видеодрайверами, что даст возможность реализовать поддержку технологии NVIDIA Optimus и обеспечить горячее подключение видеоадаптеров c интерфейсом USB. Из изменений в выпуске 2.20.1 можно отметить возобновление поддержки DRI для i810, увеличение производительности рендеринга для устройств на базе GT2 SandyBridge и IvyBridge, улучшение работы в условиях отключенной акселерации.
