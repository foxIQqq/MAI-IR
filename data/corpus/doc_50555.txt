Проект Ruby перешёл с Subversion на Git

Разработчики языка программирования Ruby объявили о миграции основного репозитория с централизованной системы контроля версий Subversion на распределённую систему управления исходными текстами Git. Разработка новой стабильной ветки ruby_2_7 и ветки trunk переведена на Git, но сопровождение веток ruby_2_4, ruby_2_5 и ruby_2_6 оставлено в SVN. Для навигации по коду и изменениям в основном репозитории предложен web-интерфейс, основанный на cgit. На использование кода из Git также переведены компоненты сборочной системы и система отслеживания ошибок (на основе Redmine). Репозиторий Ruby на GitHub продолжает позиционироваться как зеркало и пока не поддерживает приём pull-запросов.
