Релиз Wine 1.3.20

Вышла новая версия реализации Win32 API - Wine 1.3.20. С момента выпуска версии 1.3.19 было закрыто 77 отчётов об ошибках. Основные изменения в новой версии: Добавлена возможность удержания курсора мыши в полноэкранных приложениях; Добавлена поддержка постоянных HTTP-соединений; Добавлена начальная реализация экспорта XML-данных; DIB-движок стал поддерживать отрисовку прямоугольников; В API MMDevAPI добавлена возможность регулирования громкости звука; Реализованы некоторые недостающие функции библиотеки MSVC; Обновлены переводы; Исправлено множество ошибок.
