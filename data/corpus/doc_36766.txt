Релиз медиаплеера Nightingale 1.12

После года разработки представлен новый выпуск медиаплеера Nightingale, в рамках которого развивается форк проекта Songbird, обеспечивающий полноценную поддержку платформы Linux, наряду с платформами Windows и Mac OS X. Базовая часть плеера доступна под лицензий GPL. Компоненты, заимствованные у проекта Mozilla, распространяются под лицензиями MPL и BSD. Представленный выпуск Nightingale синхронизирован с кодовой базой Songbird 2.2.0. Проект Nightingale был основан независимыми энтузиастами в апреле 2010 года, вскоре после заключения сделки курирующей Songbird фирмы с компанией Philips, в результате чего основные усилия были направлены на развитие поддержки платформ Windows и Mac OS X, а с Linux был снят статус официально поддерживаемой платформы. Неофициальные сборки Songbird для Linux по-прежнему собираются энтузиастами, но отстают по реализованным возможностям и не проходят полноценный цикл тестирования. Как и Songbird, плеер Nightingale построен на базе технологий Mozilla (XULRunner), оптимизирован для каталогизирования и навигации по большим музыкальным коллекциям, а также же для поиска музыки в Сети, интеграции с музыкальными web-сервисами и социальными сетями. Среди других возможностей: система динамических плейлистов; эквалайзер; поддержка изменения внешнего вида через скины; режим проигрывания без пауз; мощные средства для импорта файлов с локального диска или из сети; удобный интерфейс для работы с большими музыкальными коллекциями, с возможностью разбиения данных на разделы (по авторам, жанрам и т.п.), поиском и сортировкой. Из изменений в новом выпуске отмечается проведение большой работы по устранению ошибок и увеличению стабильности. Осуществлён уход от поставки сторонних библиотек в комплекте с плеером - Nightingale теперь использует штатные библиотеки дистрибутива. Включены средства для интеграции с меню управления звуком и системой нотификации десктоп-окружений Unity и GNOME. Добавлен новый инсталлятор для платформы Windows, не требующий для установки привилегий администратора. Расширен диапазон поддерживаемых форматов для импорта видео (3gp, ogv, avi, divx, flv, mov, mp4, m4v, webm, wmv). Увеличено число портированных из Songbird дополнений, например, портировано дополнение для интеграции с музыкальной социальной сетью soundcloud.com. Движок XULRunner обновлён до версии 1.9.2.28.
