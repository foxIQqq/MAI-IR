Первая тестовая версия openSUSE 11.4. VMware начинает дарить клиентам SUSE Linux

Доступен для загрузки первый тестовый выпуск OpenSUSE 11.4, финальный релиз ожидается 10 марта 2011 года. В представленном выпуске обновлены версии большого числа программ, например осуществлен переход на XOrg 1.9, KDE 4.5 и GNOME 2.32.0 Beta 1. В дистрибутив интегрированы наработки по увеличению производительности сетевых операций, связанных с управлением пакетами: обновлена версия библиотеки libzypp (8.1), в которой представлен новый бэкенд для загрузки пакетов по http и ftp. Вместо бекенда MediaAria теперь используется MultiCurl, в котором реализована поддержка zsync-передачи и улучшена работа Metalink-загрузок. С точки зрения пользователя данные изменения будут заметны в ускорении обновления данных о состоянии репозиториев, уменьшении времени на установку пакетов и выполнение операций обновления. Поддержка Metalink позволяет организовать параллельную загрузку пакетов, при этом разные части пакетов могут загружаться с разных зеркал. Поддержка ZSync сокращает объем передаваемых данных, за счет того что осуществляется загрузка только измененных частей файлов. Кроме того, сообщается, что компании VMware и Novell достигли соглашения, в соответствии с которым VMware будет поставлять свои продукты в комплектации со специально оптимизированной для VMware сборкой SUSE Linux Enterprise Server (SLES). При этом SLES (с доступом к загрузке дополнений) для пользователей продукта VMware vSphere будет поставляться бесплатно. Клиенты, желающие получить сервис технической поддержки, теперь могу получить данную услугу из рук VMware, без необходимости заключения контракта с Novell.
