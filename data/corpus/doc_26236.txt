Первый стабильный релиз IronRuby 1.0

Команда разработчиков IronRuby объявила о выходе первой стабильной версии языка Ruby для платформы Microsoft .Net, обеспечивающей максимальную совместимость со стандартным интерпретатором Ruby 1.8.6. По сравнению с первым публичным релизом IronRuby было исправлено 455 ошибок. IronRuby поставляется в двух версиях: для платформы .Net 2.0 SP1 (совместимая также с Mono) и для .Net 4.0, предоставляющая улучшенную скорость запуска, совместимость с динамическими ключевыми словами языка C# и другими возможностями новой платформы. Загрузить IronRuby 1.0 можно с сайта Microsoft Codeplex.
