Вышла система сборки Apache Ant 1.9.0

После трёх лет с момента прошлого значительного выпуска представлен релиз свободной системы сборки Java-проектов Apache Ant 1.9. В новом выпуске подняты требования к минимально поддерживаемой версии Java: поддержка Java 1.4 прекращена, для работы требуется минимум Java 1.5. В набор тестов JUnit 4 добавлена поддержка аннотаций "@Ignore". Улучшена поддержка вызова утилит zip, bzip2 и tar в процессе сборки. Прекращена встроенная поддержка Perforce, пользователям данной системы контроля версий рекомендуется использовать более функциональные модули поддержки Ant от компании Perforce Inc.
