Релиз открытого медиацентра XBMC 10.0

Выпущена стабильная версия открытого медиацентра XBMC 10.0 "Dharma". Для быстрой оценки возможностей подготовлена XBMC Live-сборка на базе Ubuntu Linux 10.10 (430 Мб), готовые установочные пакеты доступны для MacOS X, Linux и Windows. Проект XBMC несколько раз был назван лучшим мультимедиа приложением и лучшим проектом для любителей компьютерных игр, в рейтинге самых популярных проектов на SourceForge.net. Из интересных функций XBMC, можно отметить возможность проигрывания файлов по FTP, без их предварительной загрузки, наличие гибкой система плагинов, реализованных на языке Python. Ключевые улучшения: Новый унифицированный фреймворк для разработки дополнений, в который добавлена поддержка создания скинов, систем визуализации, загрузчиков мета-данных из web (скрапперов), хранителей экрана, python-скриптов и т.п. В GUI-интерфейсе обеспечена возможность быстрого поиска и установки дополнений. Многие из базовых функций (визуализация, хранители экрана, SHOUTcast) переделаны в виде дополнений; Подготовлен web-интерфейс, который может быть использован для управления медиацентром с компьютера в другой комнате. Для удаленного управления задействованы API на базе JSON-RPC и JSON, в качестве встроенного http-сервера используется microhttpd; В поставку включено 11 различных визуальных оформлений (скинов), оптимизированных для различных областей применения. Начальная поддержка управления жестами в GUI-интерфейсе, в guilib добавлены функции для работы с сенсорным экраном. Для акселерации проигрывания видео на платформе Linux улучшена поддержка API VDPAU, добавлена возможность использования VA-API, обеспечена поддержка технологий NEON и OpenMAX, используемых в платформах TI OMAP3 / ARM NEON и NVIDIA Tegra 2; Поддержка проигрывания незашифрованных Blu-ray дисков и локальных bdmv-файлов. Поддержка Blu-ray/AVCHD PGS субтитров в контейнере MKV; Поддержка видеокодека VP8 и медиаконтейнера WebM; Добавлен кодировщик реального времени ATSC A/52 для формирование выходного потока в формате Dolby Digital 5.1 (AC-3). Добавлена поддержка многоканального аналогового вывода (5.1); Начальная поддержка OpenGL ES 2.0 для формирования интерфейса на платформе Linux; Поддержка проигрывания файлов с удаленной машины, используя доступ по протоколам SSH, SFTP и WebDAV; Добавлено 12 новых скрапперов для получения информации о видео из таких сервисов, как Freebase, videobuster, kino, TheMovieDB.org и IMDb. Поддержка извлечения текстов песен из музыкальных файлов, содержащих Lyrics3v2 (Lyrics3 2.00) ID3-теги; Обновление декодеров, заимствованных из проекта FFmpeg; Добавлена поддержка RTMPE/RTMPTE потоков и улучшена поддержка RTMP/RTMPT/RTMPS, благодаря использованию librtmp из пакета RTMPDump; Начальная поддержка MySQL в качестве бэкенда для хранения данных о музыкальной и видео коллекции; Поддержка задействования аппаратного акселератора Broadcom Crystal HD для декодирования видео.
