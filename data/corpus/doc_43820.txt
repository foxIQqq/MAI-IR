Опубликован официальный план прекращения разработки Firefox OS для смартфонов

Проект Mozilla довёл до разработчиков ряд решений, связанных с прекращением разработки Firefox OS для смартфонов. Последним выпуском для смартфонов станет Firefox OS 2.6, запланированный на май 2016 года, после чего задействованный в разработке персонал будет переключен на решение других задач. Разработка Firefox OS будет продолжена для сетевых потребительских устройств, таких как умные телевизоры и устройства, относящиеся к категории Интернет-вещей (IoT). В настоящее время ведётся работа по формированию прототипов новых продуктов на базе Firefox OS, три проекта уже прошли первичный отбор и многие предложения находятся на рассмотрении. В течение первой половины 2016 года процесс разработки данных продуктов будет раскрыт для широкой публики, сейчас же работа ведётся в узком кругу сотрудников компании Mozilla. Из раскрытых в настоящее время проектов пока отмечается только Firefox OS для SmartTV. Платформа Boot to Gecko (b2g) была и останется открытой операционной системой, открытой для сторонних изменений. Значительные изменения ожидают магазин-каталог приложений Firefox Marketplace. Начиная с 29 марта 2016 года Firefox Marketplace перестанет принимать приложения для Android, рабочих столов и планшетов, так как они потеряли смысл после удаления из Firefox компонента Web Runtime (WebRT), предназначенного для выполнения обособленных web-приложений. В дальнейшем, все не связанные с Firefox OS приложения будут удалены из Firefox Marketplace. Приложения для Firefox OS продолжат приниматься как минимум и в следующем году (точная дата прекращения приёма приложений для смартфонов на базе Firefox OS пока не определена). Программа Foxfooding, в рамках которой для внутреннего использования сотрудниками Mozilla распространяется эталонный Firefox OS-смартфон, построенный на базе Sony Z3C, продолжит своё существование, но будет переориентирована на развитие возможностей для новых потребительских продуктов, вместо улучшения интерфейса Firefox OS для смартфонов.
