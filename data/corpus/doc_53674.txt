Релиз среды разработки приложений KDevelop 5.6

После шести месяцев разработки представлен релиз интегрированной среды программирования KDevelop 5.6, полностью поддерживающей процесс разработки для KDE 5, в том числе с использованием Clang в качестве компилятора. Код проекта распространяется под лицензией GPL и использует библиотеки KDE Frameworks 5 и Qt 5. В новом выпуске: Улучшена поддержка проектов CMake. Добавлена возможность группировки сборочных целей cmake по разным подкаталогам. При импорте проектов задействован cmake-file-api. Улучшена обработка ошибок. Улучшены средства для разработки на языке С++. Добавлена возможность передачи произвольных флагов компиляции при вызове clang. Улучшена поддержка языка PHP. Актуализирован файл phpfunctions.php. Добавлена обработка синтаксиса PHP 7.1 для перехвата нескольких исключений. Добавлена поддержка Python 3.9. Реализована поддержка сборки с MSVC++ 19.24. Оптимизировано раскрытие переменных окружения и добавлена возможность экранирования символа доллара обратным слешем в переменных окружения.
