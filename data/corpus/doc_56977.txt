Выпуск TeX-дистрибутива TeX Live 2022

Подготовлен релиз дистрибутива TeX Live 2022, созданного в 1996 году на базе проекта teTeX. TeX Live является наиболее простым способом развертывания инфраструктуры для подготовки научной документации, независимо от используемой операционной системы. Для загрузки сформирована сборка (4 ГБ) TeX Live 2022, которая содержит рабочее Live-окружение, полный набор установочных файлов для различных операционных систем, копию репозитория CTAN (Comprehensive TeX Archive Network), подборку документации на разных языках (включая русский). Из новшеств можно отметить: Предложен новый движок hitex, формирующий вывод в формате HINT, специально спроектированном для чтения технической документации на мобильных устройствах. Просмотрщики формата HINT доступны для GNU/Linux, Windows и Android. Добавлены новый примитивы: "\showstream" (для перенаправления вывода команды "\show" в файл), "\partokenname", "\partokencontext", "\vadjust", "\lastnodefont", "\suppresslongerror", "\suppressoutererror" и "\suppressmathparerror". В LuaTeX улучшена поддержка шрифтов TrueType и добавлена возможность использования вариативных шрифтов в luahbtex. pdfTeX и LuaTeX добавлена поддержка структурированных ссылок, определённых спецификации PDF 2.0. Компонент pTeX обновлён до версии 4.0.0 с более полной поддержкой актуальной разметки LaTeX.
