Вышел редактор диаграмм DRAKON Editor 1.9 с поддержкой Java и С#

Доступен релиз DRAKON Editor 1.9, свободного кросс-платформенного редактора диаграмм для визуального языка ДРАКОН, поддерживающего генерацию результирующего кода на языках программирования Java, C, C++, Python, C# и Tcl. Язык ДРАКОН разработан в рамках космической программы «Буран» и оперирующего созданием наглядных блок-схем, обеспечивая наглядное и простое для понимания представление алгоритмов. Среди областей применения отмечается составление спецификаций для ПО и технических заданий, документирование ПО и собственно программирование. ДРАКОН позволяет переложить работу по созданию технологичных программ на плечи инженеров, которые не обладая должными навыками программирования, досконально владеют материалом и прекрасно разбираются в сущности процессов в прикладной области для которой создаётся программа. Код DRAKON Editor распространяется как общественное достояние (Public Domain), поддерживается работа в Linux, Mac OS X и Windows. Особенности новой версии: Поддержка генерации кода на языках Java и C# (ранее обеспечена поддержка C, C++, Python и Tcl); Новый генератор кода для скриптовых языков; Возможность вставки пиктограмм горячими клавишами; Реализован список ранее открытых файлов; Возможность менять шрифты.
