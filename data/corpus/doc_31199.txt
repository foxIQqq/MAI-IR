Компания SAP подключилась к работе над OpenJDK

Компания SAP подключилась к разработке открытого проекта OpenJDK, в рамках которого развивается официальная открытая реализация Java. Разработчики из компании SAP, принимающие участие в развитии продукта SAP JVM, отныне будут участвовать в публичных обсуждениях в списках рассылки, начнут присылать патчи и улучшения. Компания SAP является давним лицензиатом Sun/Oracle, имеющим право использования исходных текстов Java и поддерживающим собственный вариант Java, доступный для 14 платформ, среди которых Win/IA64, AS400/PPC64, HPUX/PARISC, HPUX/IA64, Linux/IA64, Linux/PPC64, Linux/S390, AIX/PPC64. Все формальности, необходимые для включения в работу над OpenJDK уже улажены - компания SAP подписала c Oracle соглашение о передаче прав на передаваемые проекту разработки (Oracle Contributor Agreement). За последний год к работе над OpenJDK, кроме SAP, присоединились такие крупные поставщики, как IBM и Apple, ранее развивавшие обособленные Java-решения за закрытыми дверями. Вместо разработки собственных вариантов Java компании Oracle, IBM, Apple и SAP переключились на совместное развитие единого проекта OpenJDK, который будет использован в качестве эталонной реализации Java SE 7 (прошлые выпуски Java SE базировались на проприетарном Sun JDK).
