Релиз свободного LDAP-сервера Mandriva Directory Server 2.4.1

После 14 месяцев разработки представлен релиз сервера директорий Mandriva Directory Server 2.4.1. Для управления данными в LDAP-каталоге поставляется специальный web-интерфейс MMC (Mandriva Management Console), поддерживающий управление пользователями и группами, настройку аккаунтов в Samba, управлением совместным доступом, настройку инфраструктуры печати на базе CUPS, доставки почты (Postfix) и конфигурирование прокси (Squid) и служб DNS/DHCP. Используемый в сервере фреймворк "MMC-CORE", обеспечивающий работу управляющей web-консоли, также используется в системе Mandriva Pulse 2. Для организации управления LDAP, Samba и Squid представлен специальный API для разработки дополнений на языке Python. Код проекта доступен под лицензией GPLv2. Из добавленных в новой версии улучшений можно отметить: В модуль управления почтовыми службами добавлена поддержка управления аккаунтами пользователей в платформе для организации совместной работы Zarafa; Добавлена поддержка настройки расширенных ACL для Samba-разделов, в экспертном режиме администратор имеет возможность назначить пользователям индивидуальные параметры доступа; Возможность определения собственных параметров для Samba-разделов; В пользовательском интерфейсе улучшен вывод списков (нумерация страниц, сохраняемые фильтры поиска); Поддержка отключения аккаунтов на машине с Samba; Расширены возможности управления Samba-аккаунтами (например, реализована опция принуждения к смене пароля и задание сроков истечения действия пароля); В сетевой модуль добавлена поддержка DHCP-опции local-pac-server , через который задается URL для автоматической конфигурации настроек прокси-сервера; Для сборки кода MDS и MMC-CORE теперь используется стандартный пакет autotools.
