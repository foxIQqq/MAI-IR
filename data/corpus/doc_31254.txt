Обновление online-конструктора дистрибутивов - SUSE Studio 1.2

Подразделение SUSE объявило о выпуске обновленного варианта системы SUSE Studio, которая предоставляет разработчикам набор инструментов для формирования сконфигурированной начинки виртуальных серверов и подготовки установочных образов (Software Appliance). Созданные окружения можно опубликовать в каталоге SUSE Gallery. Пользователю достаточно загрузить образ системы из каталога и получить полностью работающее и сконфигурированное решение, требующее минимальных усилий для внедрения. За два года существования online-сервиса SUSE Studio в нем было зарегистрировано более 150 тыс. пользователей и сформировано более 800 тысяч виртуальных окружений. В каталоге SUSE Gallery размещено около 7 тысяч специализированных окружений. Новая версия SUSE Studio доступна в двух вариантах: бесплатная стандартная версия для независимых поставщиков программного обеспечения и платная расширенная версия для корпораций. Основным отличием расширенной версии, которая теперь поставляется вместо продукта SUSE Appliance Toolkit, является поддержка формирования сборок для мэйнфреймов IBM System Z и дополнительные средства для автоматизации развертывания и управления. В качестве поддерживаемых форматов виртуальных окружений отмечаются VMWare, Xen, KVM и OVF, имеются средства для развертывания окружений в публичных облачных инфраструктурах, таких как Amazon EC. Примечательно, что информация о параметрах сборки и всех внесенных в процессе подготовки изменениях сохраняется внутри сгенерированного окружения, позволяя при необходимости сформировать похожую сборку и проследить за процессом её формирования. Уже сформированные окружения могут без пересборки быть преобразованы для работы в виде LiveCD/LiveUSB, для установки на жесткий диск или для использования в системах виртуализации. Из новшеств SUSE Studio 1.2 отмечается интеграция систем SUSE Lifecycle Manager и WebYaST. SUSE Lifecycle Manager позволяет организовать полный контроль за выполнением операций по обновлению установленных окружений, комбинируя в себе инструменты для обновления компонентов операционной системы, промежуточного ПО и запускаемых в окружении рабочих приложениий, со средствами для контроля целостности, тестирования обновлений и их доставки до конечных пользователей. WebYaST представляет собой настраиваемый web-интерфейс для конфигурирования различных параметров работы системы, от управления пользователями и настройки сетевой конфигурации, до средств мониторинга и установки обновлений. Поддержка WebYaST интегрирована в систему сборки SUSE Studio, что позволяет индивидуально формировать перечень доступных настроек для каждого создаваемого окружения.
