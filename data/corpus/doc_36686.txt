Выпуск Wine 1.5.28

Вышел очередной экспериментальный выпуск открытой реализации Win32 API - Wine 1.5.28. С момента версии 1.5.27 был закрыт 21 отчёт об ошибках. Из наиболее важных изменений отмечается: Для обеспечения работы всех видов защищённых соединений теперь используется GnuTLS, в наличии OpenSSL больше нет необходимости; По умолчанию включен драйвер для платформы Mac OS X; В состав встроены шрифты FixedSys; Для панели управления джойстиком добавлена новая пиктограмма; Улучшена работа драйвера Postscript.
