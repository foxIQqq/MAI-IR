Обновление Python 2.6.4 с исправлением критических ошибок

Вышло внеплановое обновление Python 2.6.4, ветки, выступающей в роли промежуточной версии, облегчающей переход с Python 2.x на Python 3. В релизе 2.6.4 исправлены критические ошибки, появившиеся в версии Python 2.6.3, и приводящие к нарушению совместимости в системе сборки модулей и краху при использовании пакета для ведения логов в окружении Google App Engine (GAE).
