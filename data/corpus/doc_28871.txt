Доступна для тестирования новая версия быстро загружаемого Linux-окружения Splashtop

Представлена тестовая версия новой редакции Linux-окружения Splashtop, предназначенного для предустановки на встроенный на материнские платы Flash-накопитель или загрузки со специального раздела на жестком диске. Splashtop предоставляет пользователю возможность получить готовое для выполнения типовых задач рабочее окружение, через несколько секунд после включения компьютера, не дожидаясь загрузки основной системы. Ключевым изменением новой версии является переход к использованию web-браузера Chromium (c Flash-плагином) вместо ранее используемого Firefox. Концепция новой версии основана на тесной интеграции с web-сервисами и напоминает Chrome OS. Выпуск подготовлен в сотрудничестве с компанией Microsoft, что отложило свой отпечаток на систему: использование по умолчанию поискового сервиса Bing, поддержка импорта закладок и сетевых настроек WiFi из Windows, работающий только из Windows инсталлятор. Для настройки системы используется специальная конфигурационная панель. Из поддерживаемых устройств отмечены ноутбуки: HP Mini, Compaq Mini CQ10 и HP Pavilion.
