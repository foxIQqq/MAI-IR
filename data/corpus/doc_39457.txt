Первоапрельские шутки 2014 года

Подборка первоапрельских шуток: Крис Мейсон (Chris Mason), автор файловой системы Btrfs, представил инициативу по использованию социальной сети Facebook в качестве платформы для обсуждения и рецензирования патчей к ядру Linux. По мнению Криса, возможности Facebook по использования картинок и видео в процессе обсуждения позволят существенно оптимизировать рабочий процесс по продвижению патчей в ядро. Для рассмотрения патчей в Facebook создана специальная группа, к которой уже присоединилось множество известных разработчиков ядра Linux. Куратор BSDCan, крупнейшей конференции пользователей и разработчиков BSD-систем, объявил о продаже бизнеса компании Oracle и трансформации конференции в отдельную секцию в рамках мероприятия Oracle OpenWorld. Среди запланированных связанных с BSD-системами докладов, которые планируется прочитать на конференции Oracle OpenWorld, отмечены "Миграция BSD-систем на Oracle Linux" и "Выгода заимствования кода под лицензией BSD". В ответ на продажу BSDCan группа энтузиастов, под руководством Michael W Lucas, автора многочисленных книг по BSD-системам, основала новую конференцию DetroitBSDCon. Конференцию планируется провести с 14 по 17 мая на заброшенной фабрике в городе Детройт. Компания, курирующая систему управления изолированными контейнерами Docker, объявила о диверсификации бизнеса и выходе на новый рынок, связанный с производством и продажей футболок и кепок. Первая коллекция фирменных футболок Docker уже доступна для заказа. Facebook заключил сделку с компанией Canonical о поглощении проекта Ubuntu за 3 млрд долларов. Для Facebook главный интерес к Ubuntu связан с последними достижениями проекта по созданию варианта Ubuntu для мобильных устройств. Facebook считает Ubuntu Touch перспективной платформой и намерен использовать её в своём смартфоне, который вскоре будет выпущен на рынок. Компания Google представила соревнование Pokémon Master, в рамках которого объявлена охота на покемонов. Для выявления скрывающихся покемонов предлагается новая технология виртуальной реальности, базирующаяся на использовании Google Maps и реализованная через наложение виртуальных персонажей при просмотре местности при помощи камеры смартфона. В Google Translate добавлен перевод в визуальный язык смайликов Emoji. Поддержка Emoji также интегрирована в браузер Chrome для Android и iOS, что позволяет просматривать сайты не в виде скучных текстов, а в форме отражающих суть эмоциональных иллюстраций. Сайт проекта Raspberry Pi переделан в стиле отображения в консольном браузере Lynx. Проект Apache Subversion начал перевод своей кодовой базы на использование Git. Разработка системы контроля версий Subversion отныне будет вестись в Git-репозитории. Миграция позволит проекту уйти от централизованной организации разработки к децентрализованной модели. Для Subversion также представлен бэкенд для работы через Git. Пакетный менеджер GNU Guix переходит с Guile (реализация языка Scheme) на JavaScript; Опубликован код консольного варианта phpMyAdmin; Эдвард Сноуден присоединился к разработке открытого эмулятора классических квестов ScummVM; Дополнение: Fyodor (автор nmap) объявил о закрытии недавно возрождённого им списка рассылки Full Disclosure, указав на невозможность борьбы с троллингом и апатию из-за обилия требований по удалению неугодных компаниям сообщений. Очередной номер журнала для разработчиков на Objective-C был посвящён переходу на платформу Android и миграции с iOS и OS X. В OpenSMTPD добавлена поддержка звукового уведомления о доставке сообщений; С cайта OpenBSD убран раздел FAQ и переводы страниц на разные языки. Проект Qt возвращается к модели формирования веток в стиле Qt 4. Вместо отдельных веток dev, stable и release развитие Qt будет вестись в одной общей ветке, которая в зависимости от стадии подготовки релиза будет переводиться в состояние приёма новшеств, стабилизации или приёма правок к релизу. Для PostgreSQL представлена поддержка "корзины", в которую будут помещаться удалённые таблицы. Выпущен RFC 7169, определяющий расширение АНБ (NSA, No Secrecy Afforded) для создания пригодных для прослушки SSL-сертификатов; В ночных сборках Cyanogenmod любой звук заменяется на блеяние барана; IRC-сеть FreeNode объявила об обеспечении интеграции с сервисом Google+. Сеть synIRC объявила об объединении с сетью Freenode. По мере выявления новых розыгрышей текст новости будет пополняться новыми первоапрельскими шутками.
