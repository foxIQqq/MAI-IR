Выпуск мобильной ОС Sailfish 3.1

Компания Jolla опубликовала релиз операционной системы Sailfish 3.1. Сборки подготовлены для устройств Jolla 1, Jolla C, Sony Xperia X, Gemini, и уже доступны в форме OTA-обновления. Sailfish использует графический стек на базе Wayland и библиотеки Qt5, системное окружение построено на основе Mer, который с апреля развивается как составная часть Sailfish, и пакетов Mer-дистрибутива Nemo. Пользовательская оболочка, базовые мобильные приложения, QML-компоненты построения графического интерфейса Silica, прослойка для запуска Andrоid-приложений, движок умного ввода текста и система синхронизации данных являются проприетарными, но их код планировалось открыть ещё в 2017 году. В новой версии: Переработан интерфейс многих базовых приложений, включая People, Phone, Messages и Clock, которые переделаны с учётом современных тенденций в дизайне мобильных интерфейсов; Добавлена поддержка шифрования пользовательских данных на накопителе (раздел Home); Добавлена аутентификация по отпечаткам пальцев; Усовершенствован режим работы через VPN, который теперь может быть активирован на этапе отсутствия сетевого соединения для обеспечения защиты трафика с первого использования телефона. Расширены средства управления VPN-соединениями. WPA-EAP расширен возможностью включения сертификатов удостоверяющих центров и метода PEAP. Для доступа к точке доступа и паролям VPN теперь требуется аутентификация по коду доступа; Усилена изоляция системных API и различных подсистем; В браузере включена поддержка WebGL; В календаре-планировщике появилась возможность отправки приглашений через ActiveSync; В приложении для работы с камерой добавлена функция масштабирования фотографии одним касанием; В часах таймеры, будильники и секундомеры оформлены в виде отдельных вкладок; Переработаны просмотрщики документов, PDF, электронных таблиц и презентаций. Добавлена поддержка открытия файлов с простым текстом. Решены проблемы с кодировкой RTF-файлов; В приложении для работы с электронной почтой добавлена опциональная возможность заверения сообщений цифровой подписью при помощи PGP; В настройки добавлена возможность ("Settings > Gestures > Show hints and tips") отключения вывода подсказок и рекомендаций; В программе для обмена сообщениями переработано оформление нити беседы, добавлен заголовок с данными адресата и реализована поддержка сохранения или редактирования записи в адресной книге не покидая приложения; Переработана адресная книга People, в которой разделены секции для поиска, просмотра и редактирования адресата. Список контактов реорганизован в алфавитном порядке; Переработан интерфейс для отправки и приёма звонков, который разделён на три вкладки: Dialer, History и People. Номеронабиратель оптимизирован для управления рукой, держащей телефон. Добавлены режимы простого и расширенного просмотра истории звонков. В диалог поступления нового звонка добавлена кнопка для быстрой отправки в ответ предварительно определённого сообщения; Улучшена прослойка для запуска Andrоid-приложений, в которой появилась возможность аутентификации по отпечаткам пальцев, включено по умолчанию применение TLS 1.2, реализована возможность добавления контактов из Android-приложений (например, Whatsapp) в приложение People, решены проблемы при запуске WhatsApp и Telegram; Bluetooth-стек Bluez обновлён до версии 5.50.
