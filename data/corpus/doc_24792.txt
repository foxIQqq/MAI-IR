Релиз нового промышленного Linux дистрибутива - ClearOS 5.1

Объявлено о выходе финального релиза Linux дистрибутива ClearOS 5.1, ориентированного на создание серверов и построенного на пакетной базе Red Hat Enterprise Linux, Fedora и CentOS. Дистрибутив был ранее известен как коммерческий продукт ClarkConnect, который со сменой названия полностью перешел в разряд свободного ПО, включая открытие всех ранее распространявшиеся проприетарных компонентов. Для загрузки доступен установочный образ, размером 690 Мб. По сравнению с прошлыми версиями линейки дистрибутивов ClarkConnect в ClearOS 5.1 отмечены следующие улучшения: Добавление средств для интеграции клиентов с Windows 7, например, переход на Samba 3.4.3; Интеграция технологии фильтрации фишинга для машин в локальной сети; Создание новой графической консоли для управления дистрибутивом; Переработка и улучшение юзабилити управляющего web-интерфейса; Синхронизация основного костяка системы с CentOS 5.4; Создание расширенных сервисов обновлений, например, доступна подписка на обновление баз сигнатур для блокирования распространения в сети вредоноского ПО, антиспам баз. Представлен сервис для организации резервного копирования; Из изменений, важных при переходе с продукта ClarkConnect 4.3, можно отметить переход от использования apt к yum, изменение организации системы ограничения пропускной способности, прекращение поддержки старых CPU (i586 / Pentium 2), система резервного копирования на базе Bacula объявлена устаревшей, полностью заменена система для блокирования P2P трафика, для управления устройствами осуществлен переход на udev, ПО для работы с Windows/Samba полностью интегрировано с ClearOS LDAP каталогом. Основное направление применения ClearOS – это небольшие организации, имеющие удаленные офисы и полагающиеся на единую сетевую инфраструктуру. Ключевыми особенностями дистрибутива называются полностью открытая архитектура, исключающая зависимость от одного проприетарного вендора, низкая стоимость обслуживания, возможность наращивания компонентов и расширенная поддержка. ClearOS предлагает защиту локальной сети, унифицированную систему отслеживания внешних угроз, фильтрацию web-контента и спама, обмен сообщениями и файлами. Собственная разработка компании - MultiWAN, позволяет осуществлять выход в интернет, используя сразу несколько подключений. Программный интерфейс позволяет компаниям самостоятельно добавлять интересующие их клиентов локальные и сетевые приложения в инфраструктуру дистрибутива.
