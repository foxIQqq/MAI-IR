Oracle считает, что Google Android помешал продвижению Java для мобильных устройств

В рамках возобновившейся судебной тяжбы между Oracle и Google, компания Oracle подала дополнительный иск, в котором расширены критерии нанесённого ей ущерба в результате применения в платформе Android Java-технологий. В частности, к разбирательству подключены шесть версий платформы Android, вышедших с момента первого иска в 2010 году. Утверждается, что данные выпуски Android включают тысячи строк кода, заимствованных из платформы Java (в числе доказательств пока только заголовочные описания методов), а также используется структура и организация Java-платформы. Кроме того, Oracle считает, что доминирующее положение Android, в котором используется Java API, необратимо подорвало фундаментальную ценность возможного превращения Java в операционную систему для мобильных устройств и разрушало рынок мобильных Java-приложений.
