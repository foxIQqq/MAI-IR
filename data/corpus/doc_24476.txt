Вышел Pidgin 2.6.4

Вышел релиз клиента для мгновенного обмена сообщениями Pidgin 2.6.4, с исправлением более 60 ошибок. Из изменений можно отметить: libpurple: устранена проблема сборки с GnuTLS; налажена работа поверх HTTP прокси; добавлен плагин с реализацией протокола "MXit"; Добавлена новая отладочная команда '/debug plugins', выводящая список загруженных плагинов; AIM/ICQ: улучшен метод ограничения пропускной способности; при возникновении проблемы при отправке данных теперь выводится более детальный текст ошибки; исправлены ошибки при отправке offline-сообщений; устранены проблемы с разметкой в тексте сообщений; исправлена ошибка при отправке и приеме Unicode символов в режиме прямого соединения (Direct IM); XMPP: при подключении к Google Talk для контактов добавлено контекстное меню "Initiate Chat"; устранен крах при попытке проверки некорректного JID; при помощи SRV запросов для текущего домена аккаунта автоматически производится поиск STUN сервера, который будет использован в случае передачи голоса или видео; устранен крах при попытке указания адресата без символа '@'; В MSN плагине исправлено несколько приводящих к краху ошибок; Проведена глобальная реорганизация диалога настройки параметров Pidgin.
