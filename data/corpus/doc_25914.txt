Политика Microsoft привела к прекращению разработки версии Firefox для Windows Mobile

Разработчики проекта Mozilla объявили о прекращении развития мобильной версии Firefox для платформы Windows Mobile. Решение о прекращении разработки накануне готовящегося релиза связано с анонсированием компанией Microsoft новой мобильной платформы Windows Phone 7, условия распространения SDK для которой накладывают существенные ограничения на запуск стороннего ПО, повторяя в своей практике запреты, ранее применяемые компанией Apple в отношении iPhone. Тем не менее, разработчики заявили, что продолжат развитие браузера в случае, если Microsoft предоставит проекту возможность нативной сборки Firefox для Windows Phone 7. В настоящий момент силы разработчиков перераспределены в направлении развития браузера для других мобильных платформ. Напомню, что в январе первый релиз мобильной версии Firefox вышел для устройств на базе Maemo и теперь готовится для платформы Android.
