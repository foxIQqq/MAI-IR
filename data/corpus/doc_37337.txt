Выпуск интегрированного набора интернет-приложений SeaMonkey 2.19

Представлен релиз пакета SeaMonkey 2.19, объединяющего в рамках одного продукта набор приложений для работы в сети, разрабатываемых под эгидой проекта Mozilla: web-браузер, почтовый клиент, систему агрегации новостных лент (RSS/Atom), WYSIWYG-редактор html-страниц Composer, IRC-клиент ChatZilla, набор средств для web-разработчиков, включающих DOM inspector и JavaScript-отладчик. Пакет доступен в сборках для платформ Windows, Mac OS X и Linux. Кроме синхронизации с кодовой базой Firefox 22 с заимствованием свойственных для данного продукта новшеств, в SeaMonkey 2.19 реализованы следующие улучшения: Для текстовых полей и полей ввода в контекстное меню добавлена функция поиска текста внутри формы; Действие "Пометить прочитанным" (Mark -> As Read) теперь проверяет состояние всех выбранных сообщений, а не только первых; Добавлены уведомления при открытии страниц со смешанным контентом (HTTPS и HTTP); В интерфейс управления Cookie добавлена опция для ограничения внешних Cookie на открываемых сайтах (возможность принимать только Cookie от текущего сайта); В Data Manager добавлена поддержка управления локальными хранилищами (localStorage, indexedDB и т.п.) Опция открытия каталога в который сохраняется файл ("Open Containing Folder") теперь доступна и во время загрузки, а не только после её завершения.
