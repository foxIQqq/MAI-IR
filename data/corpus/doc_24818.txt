Рейтинг 10 самых важных Linux разработок по мнению компании IBM

Компания IBM в рамках празднования десятилетия ресурса IBM developerWorks представила рейтинг 10 самых важных Linux разработок. Отмечены следующие проекты, которые непосредственно повлияли на бизнес IBM: Создание международной организации Linux Professional Institute (LPI) для проведения Linux сертификации специалистов и контроля за соблюдением требований при их подготовке; Проект Samba со свободной реализацией протокола Server Message Block (SMB), позволивший обеспечить интеграцию Linux систем в Windows сети, создавать файловые и принт-сервера на базе Linux; Портирование Linux на майнфреймы S/390 (System Z); Разработка, открытие исходных текстов и включение в состав Linux ядра системы принудительного контроля доступа SELinux; Развитие LiveCD дистрибутивов, позволяющих использовать Linux без установки на жесткий диск; Создание Linux решений для обеспечения работы кластеров; Успехи Linux для организации параллельных вычислений, на суперкомпьютерах подобных Blue Gene/L и Blue Gene/P Способствование Sony установке Linux на игровых приставках Playstation 3; Развитие средств виртуализации в Linux, таких как KVM, Xen, User-Mode Linux, QEMU, VMware, VirtualBox; Создание проекта One Laptop Per Child по разработке сверхдешевого ноутбука на базе Linux и разработка окружения Sugar для обучения детей.
