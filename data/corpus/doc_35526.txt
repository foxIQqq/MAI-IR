Релиз Linux-дистрибутива ZevenOS 5.0 с интерфейсом в стиле BeOS

После года разработки представлен релиз Linux-дистрибутива ZevenOS 5.0, снабженного не требовательным к ресурсам BeOS-подобным интерфейсом. Дистрибутив построен на пакетной базе Ubuntu 12.10. В качестве десктоп-окружения используется стилизованный под BeOS оконный менеджер Sawfish с использованием компонентов Xfce 4.10. Для настройки приложений в рамках проекта создано ПО MAGi 2. Размер iso-образа 720 Мб. Сформировать аналогичное окружение в любом Ubuntu-совместимом дистрибутиве можно через установку пакета zevenos-desktop, после подключения специально подготовленного репозитория. В новой версии используется ядро Linux 3.5. В качестве видеоплеера задействован Gnome-Mplayer 1.0.6, для просмотра изображений - Shotwell, для управления установкой программ - Software-Center, для проигрывания музыки - Deadbeef 0.5.6, для работы в Web - Firefox 17, для просмотра почты - Claws-Mail 3.8.1, для выполнения офисных задач - Abiword и Gnumeric 1.10.17, для работы с файлами - Thunar 1.5.3. Вместо ранее развиваемой собственной системы Deskbar задействована стандартная панель xfce4-panel (Deskbar по прежнему можно установить из репозитория).
