Зимнее обновление стартовых наборов ALT p10

Опубликован третий выпуск стартовых наборов на Десятой платформе ALT. Предложенные образы подходят для начала работы со стабильным репозиторием тем опытным пользователям, которые предпочитают самостоятельно определять список пакетов приложений и настраивать систему (вплоть до создания собственных производных). Как составные произведения они распространяются на условиях лицензии GPLv2+. Варианты включают в себя базовую систему и одно из окружений рабочего стола или набор специализированных приложений. Сборки подготовлены для архитектур i586, x86_64, aarch64 и armv7hf. Также собраны варианты Engineering на p10 (live/install-образ с инженерным ПО; установщик добавлен для возможности более точного выбора требуемых дополнительных пакетов) и cnc-rt (live с ядром реального времени и программным ЧПУ LinuxCNC) для x86_64, включая realtime-тесты. Изменения относительно осеннего выпуска: окружение собрано с использованием mkimage-profiles 1.4.22, mkimage 0.2.44; ядро Linux std-def 5.10.82, un-def 5.14.21; systemd 249.7; Firefox ESR 91.3; Chromium 96; NetworkManager 1.32.12; cinnamon 5.0.5; kde5: 5.87.0 / 5.23.2 / 21.08.3; lxqt: 1.0; builder: добавлен NetworkManager; cnc-rt: реал-тайм ядро обновлено до версии 5.10.78; добавлены сборки для плат ЭЛВИС mcom-02 (armh); прекращено формирование специализированных rootfs для Nvidia Jetson Nano. В будущем планируется обеспечить работу на этих платах rootfs с ядрами std-def и un-def; прекращено формирование сборок armh с ядром rpi-def для Raspberry Pi. прекращена сборка ядра Linux rpi-def для armh. Для armh можно собирать ядро от систем aarch64 на Raspberry Pi 4.
