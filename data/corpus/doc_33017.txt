Тестирование проприетарного драйвера NVIDIA 295.17. В Nouveau ожидается поддержка OpenCL

Компания NVIDIA представила бета-версию проприетарного видеодрайвера NVIDIA 295.17, в котором добавлена поддержка X.Org Server 1.12 RC1 (релиз ожидается в марте) и устранена ошибка, мешающая использованию встроенной LVDS-панели на ноутбуках с GPU серии GeForce 7. Дополнительно можно отметить публикацию видеозаписи доклада разработчиков Nouveau, свободного драйвера для видеокарт NVIDIA. В докладе упомянуто о работе по реализации поддержки спецификации OpenCL, определяющей API и расширения языка Си для организации кросс-платформенных параллельных вычислений с использованием GPU видеокарт и многоядерных CPU. В частности, уже подготовлен начальный рабочий прототип, позволяющий выполнить clinfo и ряд демонстрационных OpenCL-программ для карт GeForce 8/9/100/200/300 (NV50).
