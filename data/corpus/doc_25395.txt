Вышла вторая бета-версия web-браузера Chrome для Linux

Началось тестирование бета-версии 5.0.307.7 web-браузера Chrome для платформы Linux, в которой исправлено несколько неприятных ошибок, например, падение браузера при использовании внешних шрифтов и проблемы при обработке текста. Кроме того, улучшена поддержка визуальных тем GTK, добавлена загрузка плагинов из каталога ~/.config/$PRODUCT_NAME/Plugins, OOM killer настроен на автоматическое завершение работы повисших табов, не доводя развитие событий до краха браузера. Одновременно выпущена бета-версия Chrome для MacOS X в которой функциональность доведена до версии для Linux. В частности добавлена поддержка расширений, синхронизации закладок, реализованы менеджеры для управления задачами, закладками и cookie.
