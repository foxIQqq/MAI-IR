Релиз платформы для запуска игр Ubuntu GamePack 18.04

Доступна для загрузки сборка Ubuntu GamePack 18.04, включающая в себя средства для запуска более чем 55 тысяч игр и приложений, как специально разработанных для платформы GNU/Linux, так и игр для Windows, запускаемых при помощи PlayOnLinux, CrossOver и Wine, а также старых игр для MS-DOS. Дистрибутив собран на базе Ubuntu 18.04 и включает в себя все обновления, по состоянию на январь 2020 года. По умолчанию предлагается интерфейс GNOME Flashback, внешний вид которого напоминает классический GNOME, но в качестве опции можно выбрать и другие окружения. Размер iso-образа 4.1 ГБ (x86_64). Также доступно обновление прошлой ветки на базе Ubuntu 16.04, которая собрана и для 32-разрядных систем i386. В дистрибутиве присутствуют: Системы управления и доставки игр и приложений: Steam, Lutris, Itch; Средства запуска игр и приложений, разработанных для платформы Windows: PlayOnLinux, WINE и CrossOver Linux; Для запуска старых игр и программ, разработанных для платформы DOS, предлагаются утилиты DOSBox и DOSEmu; Для использования online-игр установлены Adobe Flash и Oracle Java; Дистрибутив уже подключен к репозиториям с большой коллекцией Linux-игр различного жанра: UALinux, SNAP и Flatpak (более 779); Поддержка Gnome Twitch для просмотра игровых видео и стриминга (киберспортивные турниры, всевозможные кибер соревнования и прочие стримы от обычных игроков) в отдельном окне приложения. Основные изменения, относительно прошлого выпуска: Дистрибутивы собраны на кодовой базе Ubuntu OEMPack 18.04 / 16.04; Добавлен онлайн-сервис инди-игр Itch; Убран Sparky APTus Gamer; WINE обновлен до версии 5.0; Steam Linux Client обновлен до версии 1.0.0.61; Lutris обновлен до версии 0.5.4; PlayOnLinux обновлен до версии 4.3.4; CrossOver Linux обновлен до версии 19.0.0; Flatpak обновлен до версии 1.6.0.
