Вышел Linux-дистрибутив Trinity Rescue Kit 3.4

Анонсирован релиз Live-дистрибутива Trinity Rescue Kit 3.4, основанного на Mandriva Linux и предназначенного для восстановления Linux и Windows систем после сбоя. Дистрибутив включает набор средств для восстановления удаленных файлов и разделов, обнуления забытых паролей, клонирования дисков, проверки на вирусы и т.п. Загрузочный iso-образ занимает 145 Мб. Кроме стандартных для подобного рода дистрибутивов программ в комплект входит несколько полезных приложений собственной разработки: fileserver для быстрого запуска файлового сервера на базе samba, открывающего доступ ко всем локальным дискам текущей машины; trkbootnet - утилита с реализацией PXE-сервера удаленной загрузки, позволяющая загрузить Trinity Rescue Kit c другой машины; mclone - инструмент для массового клонирования по сети операционной системы текущей машины на другие ПК. Релиз вышел после почти годового затишья в разработке. Новая версия содержит переработанный интерфейс пользователя, основанный на представлении всех поддерживаемых функций в виде консольного меню (дистрибутив работает в текстовом режиме). В состав также включено несколько новых инструментов, таких как утилита winclean для чистки мусора с Windows-разделов, скрипт virusscan для одновременного сканирования на вирусы пятью антивирусными пакетами (ClamAV, F-prot, BitDefender, Vexira и Avast) и утилита winpass для сброса паролей в Windows. Из обновлений версий программ можно отметить: Linux-ядро 2.6.35, util-linux-ng-2.16.2, e2fsprogs 1.41.9 с поддержкой ext4, wgetpaste, Truecrypt 6.2a, sysinit 2.87, gpart 0.1h, findutils 4.4.2, Mondoarchive 2.2.9.3-r2622, fuse 2.8.4, parted 2.1, LVM 2.02.61, dmraid-1.0.0.rc16, device-mapper 4.15, rkhunter 01-2010.
