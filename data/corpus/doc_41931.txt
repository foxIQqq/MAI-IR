Проект Linux Mint выпустил инструментарий для разработчиков

Разработчики дистрибутива Linux Mint подготовили набор инструментов mint-dev-tools, предназначенный для быстрого развёртывания изолированного окружения для сборки проектов, развиваемых Linux Mint, таких как пользовательское окружение Cinnamon, файловый менеджер Nemo и такие приложения, как mintmenu, mintupdate, mintinstall и т.д. Mint-dev-tools сформирует окружение, загрузит все компоненты, необходимые для сборки выбранного проекта, установит зависимости и соберёт пригодные для установки пакеты. Код загружается из git-репозитория, что позволяет принять участие в тестировании самых свежих наработок Linux Mint.
