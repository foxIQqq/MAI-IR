Компания Opera Software выпустила два открытых инструмента для web-разработчиков

Компания Opera Software представила два новых открытых проекта, направленных на упрощение отладки web-приложений и дополняющих ранее представленный проект Opera Dragonfly: OperaWatir - написанная на языке Ruby система для автоматизированного тестирования web-приложений через имитацию действий пользователя (написание скриптов для симуляции таких действий, как клики по ссылкам и заполнение форм, с последующей оценкой корректности реакции). OperaWatir представляет собой адаптированный для использования с браузером Opera инструментарий WATIR (Web Application Testing in Ruby); OperaDriver - бэкенд к OperaWatir, взаимодействующий с браузером Opera. OperaDriver написан на языке Java и позволяет создавать автоматизированные тесты, используя Java-фреймворк JUnit.
