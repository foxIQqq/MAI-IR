Релиз дистрибутива Network Security Toolkit 24-7977

Состоялся релиз Live-дистрибутива NST (Network Security Toolkit) 24-7977, предназначенного для проведения анализа безопасности сети и мониторинга её функционирования. Размер загрузочного iso-образа (x86_64) составляет 2.8 Гб. Для пользователей Fedora Linux подготовлен специальный репозиторий, дающий возможность установить все созданные в рамках проекта NST наработки в уже установленную систему. Дистрибутив построен на базе Fedora 24 и допускает установку дополнительных пакетов из внешних репозиториев, совместимых с Fedora Linux. В состав дистрибутива включена большая подборка приложений, имеющих отношение к сетевой безопасности (например: Wireshark, NTop, Nessus, Snort, NMap, Kismet, TcpTrack, Etherape, nsttracroute, Ettercap и т.д.). Для управления процессом проверки безопасности и автоматизации вызова различных утилит подготовлен специальный web-интерфейс, в который также интегрирован web-фронтэнд для сетевого анализатора Wireshark. Графическое окружения дистрибутива базируется на FluxBox. В новом выпуске обновлены версии программ, задействовано ядро Linux 4.6. В состав включена утилита SSLyze, предназначенная для анализа конфигурации сервера SSL. Прекращено формирование 32-разрядных сборок, для загрузки предлагается только образ для систем x86_64. Специально для дистрибутива создана новая реализация инструмента MTR (Multi-Traceroute), которая поддерживает выявление балансировщиков нагрузки и трансляторов адресов, а также предоставляет расширенные возможности, такие как отправка серии запросов на каждую целевую систему, показ RTT (Round Trip Time), выбор протокола (TCP, UDP и ICMP) и генерация наглядного отчёта в формате SVG. Новый инструмент интегрирован со многими компонентами web-интерфейса NST WUI. В web-интерфейс также добавлена возможность визуализации результатов проведённого через ntop глубокого инспектирования пакетов, результаты выводятся в форме интерактивной трёхмерной круговой диаграммы.
