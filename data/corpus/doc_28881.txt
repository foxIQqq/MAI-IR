Компания Alfresco выпустила систему управления бизнес-процессами Activiti BPM 5.0

Доступен первый стабильный релиз открытой системы для организации управления бизнес-процессами Activiti BPM 5.0, исходные тексты которой написаны на языке Java и распространяются в рамках лицензии Apache. Система реализует стандарт графического моделирования бизнес-процессов BPMN 2.0 и позволяет моделировать, анализировать и усовершенствовать процессы, происходящие в бизнес-инфраструктуре предприятия. Пакет состоит из шести базовых компонентов: Activiti Engine - отдельный JAR-файл, содержащий движок с реализацией стандарта BPMN 2.0; Activiti Explorer - пользовательское приложение для управления задачами и организации новых бизнес-процессов; Activiti Probe - управляющая консоль для управления развертыванием, работой с СУБД, архивированием и обработкой задач; Activiti Designer - Eclipse-плагин для визуального проектирования диаграмм; Activiti Cycle - интерфейс для организации совместной работы; Activiti Modeler - моделирующий интерфейс для аналитиков.
