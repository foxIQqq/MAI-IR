Первый онлайн-журнал, посвящённый Raspberry Pi

Вышел первый номер онлайн-журнала MagPi, первого периодического издания, целиком посвящённого одноплатному компьютеру Raspberry Pi и сообществу его пользователей. Редакторами журнала стали несколько активных членов сообщества пользователей форума Raspberry Pi. Они ставят перед собой цель рассказать читателям "о различных операционных системах для RP, о том, как писать для неё программы на разных языках и как начать свой собственный интересный проект". Первый номер позиционируется как подготовка к использованию RP и будет полезен читателям, которые ещё не приобрели собственный RP. Содержание первого номера: Введение - описание проекта, история создания и технические характеристики RP, всё в контексте истории доступной вычислительной техники, развития языков программирования и программного обеспечения. Развитие аппаратной части и роботостроение - описание нескольких хобби-проектов под управлением RP: плата расширения общего назначения (GPIO - General Purpose Input/Output), создание робота-манипулятора типа "рука" под управлением RP. RacyPy - набор софта для оффисной работы, навигации по интернету и программирования, доступный как в виде загрузочного образа для использование в качестве LiveCD/LiveUSB, так и для запуска в виртуальной машине. Debian - инструкции для запуска Linux Debian OS в виртуальной машине. Scratch - введение в Scratch, язык программирования для новичков. The Python Pit - программирование на Python и PyGame с примерами кода и объяснениями. Журнал доступен для скачивания в формате PDF, а также для чтения онлайн.
