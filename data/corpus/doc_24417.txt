Завершена сертификация во ФСТЭК России Linux XP Desktop 2008 Secure Edition

Компания «ТрастВерс» сообщает о получении сертификата соответствия ФСТЭК России №1921 на Защищенную операционную систему Trustverse Linux XP Desktop 2008 Secure Edition. Полученный сертификат удостоверяет, что Защищенная операционная система Trustverse Linux XP Desktop 2008 Secure Edition соответствует требованиям руководящих документов по 4-му уровню контроля отсутствия недекларированных возможностей и 5-му классу защищенности от НСД для СВТ и может использоваться при создании автоматизированных систем до класса защищенности 1Г включительно и информационных систем персональных данных до класса К2 включительно. Система Linux XP Desktop 2008 Secure Edition может использоваться для организации рабочих мест, тонких клиентов или файловых серверов рабочих групп. С дистрибутивом поставляется диск с комплектом прикладных приложений, включая средства для запуска Windows-приложений — Trustverse WINE-LXP. Linux XP основан на переработанной пакетной базе Fedora Linux и RHEL. Без активации десктоп-редакция дистрибутива работоспособна в течение 30-дневного пробного периода, для серверной редакции подобных ограничений нет. Исходные тексты измененных пакетов как серверной, так и десктоп редакции дистрибутива доступны для свободной загрузки с сайта downloads.linux-xp.com, iso-образы для ознакомления можно загрузить с данной страницы.
