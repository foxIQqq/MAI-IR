Первый выпуск Nightingale, форка медиаплеера Songbird

Представлен первый экспериментальный выпуск мультимедиа плеера Nightingale, в рамках которого развивается форк плеера Songbird, обеспечивающий полноценную поддержку платформы Linux, наряду с платформами Windows и Mac OS X (сборки плеера пока доступны для Linux и Windows). Код плеера доступен под лицензиями GPL, LGPL, MPL и BSD. Представленный выпуск Nightingale основан на кодовой базе Songbird 1.8.1 c бэкпортированием отдельных изменений из более новых веток. Из улучшений отмечены: переход на использования GStreamer на платформе Linux, переработанное оформление, поддержка синхронизации директорий и устранение проблем, например, мешавших созданию отдельных директорий для плейлистов. Проект поддерживает подключение дополнений, среди которых пока доступны только модули интеграции с Last.FM и модуль для вывода информации на LCD-дисплей клавиатуры G15. В разработке находятся дополнения для использования серивса SHOUTCast и обмена информацией о прослушиваемых композициях через Twitter (LiveTweeter). В ближайшее время планируется переход на использование браузерного движка XULRunner/Gecko 6, синхронизация с актуальной веткой разработки Songbird и портирование для Linuх нереализованных для данной платформы возможностей Songbird. Первый стабильный релиз Nightingale планируется выпустить в следующем году. Проект Nightingale был основан независимыми энтузиастами в апреле 2010 года, после того как разработчики Songbird, вскоре после заключения сделки с компанией Philips, заявили о прекращении официальной поддержки платформы Linux, сфокусировав внимание на поддержке Mac OS X и Windows. Неофициальные сборки Songbird по прежнему собираются энтузиастами, но отстают по реализованным возможностям и не проходят полноценный цикл тестирования. Как и Songbird плеер Nightingale построен на базе технологий Mozilla (XULRunner), оптимизирован для каталогизирования и навигации по большим музыкальным коллекциям, а также же для поиска музыки в сети, интеграции с музыкальными web-сервисами и социальными сетями. Из других возможностей можно отметить систему динамических плейлистов; эквалайзер; поддержку изменения внешнего вида через скины; режим проигрывания без пауз; мощные средства для импорта файлов с локального диска или из сети; удобный интерфейс для работы с большими музыкальными коллекциями, с возможностью разбиения данных на разделы (по авторам, жанрам и т.п.), поиском и сортировкой.
