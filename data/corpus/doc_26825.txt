Cardapio - альтернативная реализация меню для GNOME

В рамках проекта Cardapio подготовлена альтернативная реализация меню для GNOME. Cardapio поддерживает работу в двух режимах: в режиме апплета меню для панели GNOME и в режиме "Launcher" для организации быстрого запуска программ, выполняя функции похожие на упрощенный вариант GNOME Do. Кроме непосредственной организации запуска программ, в Cardapio предусмотрена возможность интеграции с базой tracker для выполнения поиска соответствующих определенным критериям фотографий, документов, музыки и других пользовательских данных. В отличие от GNOME Do, требующего для своей работы Mono, код Cardapio написан на языке Python и распространяется в рамках лицензии GPLv3. Для установки программы в Ubuntu достаточно выполнить команды: sudo add-apt-repository ppa:tvst-hotmail/cardapio sudo apt-get update && sudo apt-get install python-keybinder cardapio и добавить на панель апплет "Cardapio". Для запуска в обособленном режиме нужно выполнить "cardapio show", после чего программу можно вызвать нажав клавиши Super (Windows) и пробел.
