Релиз Linux-дистрибутива ZevenOS 4.0 с интерфейсом в стиле BeOS

Доступен релиз Linux-дистрибутива ZevenOS 4.0, снабженного не требовательным к ресурсам BeOS-подобным интерфейсом. Дистрибутив построен на пакетной базе Ubuntu 11.10. В качестве десктоп-окружения используется стилизованный под BeOS оконный менеджер Sawfish 1.8.2 с использованием компонентов Xfce 4.8. Для настройки приложений в рамках проекта создано ПО MAGi 2, в качестве панели для запуска программ задействован Zeebar, меню реализовано при помощи Deskbar 2. Размер iso-образа 696 Мб. Сформировать аналогичное окружение в любом Ubuntu-совместимом дистрибутиве можно через подключение специального репозитория и установку пакета zevenos-desktop. В новой версии используется ядро Linux 3.0, добавлена экспериментальная поддержка файловой системы btrfs. Переход на компоненты Xfce 4.8 ознаменовал ряд заметных улучшений, например, в файловом менеджере Thunar появилась возможность работы с удалёнными хранилищами через FTP, Samba, SSH и NFS. В состав включён видеоредактор OpenShot 1.4. Изменился состав поставляемых по умолчанию программ: в качестве видеоплеера вместо Parole отныне используется Gnome-Mplayer, для просмотра изображений - Shotwell, для управления установкой программ - Software-Center. Для проигрывания музыки задействована свежая версия Deadbeef, для работы в Web - Firefox, для просмотра почты - Claws-Mail, для выполнения офисных задач - Abiword и Gnumeric.
