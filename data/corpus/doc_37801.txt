Выход открытой платформы управления бизнес-процессами Camunda BPM 7.0.0

Увидела свет открытая платформа Camunda BPM 7.0.0, предоставляющей средства для автоматизации управления бизнес-процессами и организации рабочего процесса. В основе системы лежит движок моделирования бизнес-процессов, соответствующий концепции BPMN 2.0. Код платформы написан на языке Java с использованием Spring Framework и распространяется под лицензией Apache 2.0. Готовые дистрибутивы Camunda BPM подготовлены для серверов приложений Apache Tomcat, JBoss AS 7 и Glassfish AS 3. Версия camunda BPM 7.0.0 примечательна добавлением компонента camunda cockpit, с реализацией web-интерфейса для визуального слежения за бизнес-процессами и операциями. Кроме того, добавлен REST API на основе JAX-RS, улучшены средства аудита и кластеризации в движке моделирования бизнес-процессов, улучшены инструменты управления пользователями (реализована авторизация и поддержка LDAP).
