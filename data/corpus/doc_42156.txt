Выпуск Jython 2.7, реализации языка Python на Java

Представлен релиз Jython 2.7.0, реализации языка Python 2.x на Java. В отличие от стандартного CPython, Jython транслирует исходные тексты Python скриптов в Java-байткод, выполняющийся внутри виртуальной машины Java и не отличающийся от байткода Java-программ, что позволяет использовать готовые Java-библиотеки в Python-программах и наоборот. Кроме обеспечения совместимости языка и runtime с CPython 2.7, в новом выпуске добавлена поддержка инструментария pip/setuptools и подготовлен исполняемый файл для Windows (bin/jython.exe), позволяющий устанавливать Jython-скрипты в Windows.
