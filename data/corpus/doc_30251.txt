Компания MIPS открыла ресурс для разработчиков Linux и Android

Компания MIPS Technologies анонсировала введение в строй нового сайта developer.mips.com, предназначенного для обеспечения поддержки сообщества разработчиков, использующих Android и Linux на оборудовании с процессорами на базе архитектуры MIPS, включая портативные компьютеры и планшеты. Из представленной на сайте информации можно отметить: Исходные тексты разработок MIPS для Linux и Android, включая код для обеспечения работы Android на платформе MIPS, инструкции по портированию и адаптированный для платформы MIPS вариант Android NDK; Ресурсы для Linux-разработчиков, включая исходные тексты оптимизированного для платформы MIPS Linux-ядра; Набор инструментов для разработчиков, включая интегрированную среду разработки MIPS Navigator Integrated Component Suite (ICS) и вариант компиляторов GCC для MIPS; Документация и видеоинструкции по работе с MIPS Navigator ICS и отладке программ на многоядерных процессорах; Доступ к EJTAG отладчику; Информация о базирующихся на MIPS платах для разработчиков; Форумы поддержки Android, Linux и средств разработки; Блог с заметками инженеров MIPS на такие темы, как Linux-ядро, оптимизация производительности и отладка; Ссылки на близкие ресурсы, такие как mipsandroid.org и linux-mips.org.
