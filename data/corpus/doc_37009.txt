Sourceforge предложил пользователям Google Code свой сервис загрузки файлов

В связи с прекращением поддержки загрузки файлов в хостинге открытых проектов Google Code, конкурирующий сервис Sourceforge предложил использовать свои мощности для организации подобных загрузок. Пользователям предлагается разместить файлы на Sourceforge и поставить ссылку на раздел загрузки Sourceforge на своей странице в Google Code. Для GitHub аналогичное предложение действует с начала года и многие пользователи данного сервиса им воспользовались. Последнее время SourceForge, отставая от Google Code и GitHub по средствам разработки и работы с кодом, пытается оставаться на плаву, сосредоточивая внимание на нише распространения файлов и результатов работы проектов. Ежедневно Sourceforge обслуживает около 4.8 млн загрузок, ежемесячно отдавая около 2 Пб трафика.
