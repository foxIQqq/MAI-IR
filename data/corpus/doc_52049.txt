Релиз сетевого конфигуратора NetworkManager 1.22.0

Опубликован новый стабильный релиз интерфейса для упрощения настройки параметров сети - NetworkManager 1.22. Плагины для поддержки VPN, OpenConnect, PPTP, OpenVPN и OpenSWAN развиваются в рамках собственных циклов разработки. Основные новшества NetworkManager 1.22: В интерфейс nmcli добавлена команда "general reload" для перезагрузки настроек NetworkManager и параметров DNS; Добавлена утилита nm-cloud-setup, для автоматической настройки NetworkManager в облачных окружениях (пока поддерживаются только облака EC2 с IPv4); Предложен новый логотип NetworkManager; Встроенный плагин для DHCPv4 переведён с кодовой базы systemd на использование библиотеки n-dhcp4, развиваемой проектом nettools; Для IPv4-маршрутов добавлена поддержка атрибута "scope" (область достижимости); В DHCP-запросах обеспечена поддержка указания флагов IAID и FQDN; Добавлено свойство '802-1x.optional' для определения необязательности аутентификации 802.1X в проводных сетях; При определении состояния устройства обеспечен учёт сведений о стоимости беспроводного соединения (элемент Network Cost Wi-Fi); Предложена настройка main.auth-polkit=root-only для отключения PolicyKit и предоставления доступа только пользователю root; Состояние завершения запуска теперь выставляется сразу после соединения устройства (состояние "connected"), но не дожидаясь присвоения IP-адреса, что позволяет избежать блокировки "NetworkManager-wait-online.service" и "network-online.target". В случае возникновения проблем можно использовать параметры "ipv4.may-fail=no" и "ipv6.may-fail=no", позволяющие отложить присвоение состояния "connected" до получения адреса; Из libnm удалены API NMDeviceWimax и NMWimaxNsp, так как из самого NetworkManager поддержка WiMAX была удалена ещё в 2016 году; В libnm API для обращения к D-Bus в синхронном режиме объявлено устаревшим; Значительно переработаны внутренности NMClient, который может использоваться как урезанный вариант libnm; Прекращена поддержка Blutooth-стека BlueZ 4 (с 2012 года развивается BlueZ 5).
