Представлен комплект для создания мобильных платформ на базе кода Symbian^3

Компания Nokia выпустила инструментарий Symbian PDK 3.0.0, ориентированный на создание собственных сборок Symbian на базе исходных текстов мобильной платформы Symbian^3. PDK содержит полный набор средств сборки и исходных текстов, необходимых для формирования варианта Symbian^3 для различных ARMv5-совместимых мобильных устройств. Исходные тексты Symbian^3 полностью открыты, насчитывают более 200 тыс. файлов и 40 миллионов строк кода и распространяются в рамках свободной лицензии EPL (Eclipse Public License). Из особенностей Symbian^3 отмечается: Улучшенный интерфейс пользователя: переход к парадигме "действие через одно нажатие", поддержка мультитач-навигации; Поддержка нескольких персонализированных домашних экранов, для которых поддерживается несколько методов создания виджетов: C++/Cи, Qt, Javascript/HTML, Adobe FlashLite. Расширенные мультимедийные возможности: менеджер подкастов, поддержка интернет-радио, возможность вывода через HDMI, реализация профиля для удаленного управления Bluetooth Audio/Video Remote Control Profile; Улучшение графической подсистемы: реализация технологии NGG (Next Generation Graphics), поддержка OpenGL ES, 2D и 3D-акселерации; Улучшенная поддержка сетевых функций: поддержка соединения в один клик, возможность выделения повышенной полосы пропускания отдельным приложениям, готовность к переходу на 4G; Оптимизацию производительности и потребления памяти. В третьем квартале текущего года компания Nokia планирует выпустить основанный на Symbian^3 телефон Nokia N8, последнюю модель серии "N" поставляемую с прошивкой на базе ОС Symbian, в будущем аппараты данной серии будут поставляться с программным обеспечением MeeGo. Следующий релиз Symbian^4 будет снабжен интерфейсом пользователя, базирующемся на тесном использовании социальных web-сервисов и переработанной организации управления через сенсорный экран. В качестве первичного фреймворка для создания пользовательских приложений в Symbian^4 будет использован Qt. Разработка Symbian^4 будет завершена во втором полугодии 2010 года, а первые устройства поступят в продажу в первом полугодии 2011 года.
