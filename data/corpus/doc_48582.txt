Опубликованы макеты мобильного интерфейса GNOME для смартфона Librem 5

Компании Purism продемонстрировала первые макеты мобильного интерфейса GNOME, развиваемого для смартфона Librem 5, оснащённого прошивкой на базе Debian и свободного ПО. Программное обеспечение для смартфона должно быть готово к январю 2019 года. Работа над мобильной версией разделена на несколько этапов и ведётся совместно с разработчиками GNOME. Итоговые наработки будут предложены в качестве универсального GNOME Mobile Shell, который можно будет применять и для других мобильных устройств. Первый выпуск мобильной оболочки GNOME, который намерены поставлять при начале продаж Librem 5, будет обладать только простейшими возможностями и будет сосредоточен на оттачивании общей архитектуры, базовых принципов построения интерфейса, надёжности и эффективности. В составе первого выпуска будут предложены экран блокировки системы с защитой сеанса через PIN-код, домашний экран со списком установленных приложений, верхняя панель с полезной информацией (время, заряд аккумулятора, уровень сигнала, уровень громкости звука, статус сети и т.п.), нижняя панель с появляющимися при необходимости кнопками, виртуальная клавиатура, система уведомлений о входящих звонках. На второй фазе разработки платформу планируется расширить поддержкой многозадачности, средствами поиска и системой вывода уведомлений. Второй выпуск мобильной оболочки GNOME будет подготовлен с небольшим отставанием от первой версии. В качестве примера адаптации интерфейса для мобильных систем приводится приложение Contact с реализацией коммуникационного клиента и адресной книги. Программа будет отвечать за обработку звонков и сообщений. В качестве основы для Contact используется штатное одноимённое приложение GNOME, интерфейс которого переведён на другой шаблон и расширен специфичными для мобильных систем элементами. Что касается аппаратной начинки, то большое внимание уделяется времени автономной работы, для того чтобы использовать аккумулятор большей ёмкости принято решение увеличить размер экрана с 5.5 до 5.7 дюймов. Визуально смартфону попытались придать самобытность, вместо типичных для современных обтекаемых форм планируется использовать немного закруглённые углы со скошенной сзади кромкой, что выглядит более элегантно и позволяет вместить больше компонентов. Напомним, что для Librem 5 развиваются три альтернативные оболочки на основе GNOME, KDE и Ubuntu Touch (UBports). Устройство будет поставляться в металлическом корпусе, с 5.7-дюймовым экраном (18:9), CPU i.MX8 (окончательное решение ещё не принято), GPU Vivante (будет задействован свободный драйвер Etnaviv), 3 ГБ ОЗУ, 32 ГБ eMMC, MicroSD, USB Host, USB Type-C, WiFi 802.11, Bluetooth 4, 2G/3G/4G, GSM, UMTS и LTE. На устройстве будут блокироваться любые попытки отслеживания и сбора информации о пользователе. Например, будут жестко отделены компоненты Baseband-чипа, отвечающего за работу в сотовых сетях, от основного CPU, обеспечивающего работу пользовательского окружения. Также будут предусмотрены переключатели, которые на уровне аппаратного разрыва цепей позволят отключать камеру, микрофон, WiFi/Bluetooth и модуль Baseband. Для обмена сообщениями по умолчанию будет предложена система децентрализованных коммуникаций на базе протокола Matrix, обеспечивающая оконечное (end-to-end) шифрование.
