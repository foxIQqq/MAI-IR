Релиз Navicat 9 для Linux - утилиты по управлению СУБД

Вышла новая версия Navicat для Linux - мощного приложения по управлению СУБД, которое поддерживает MySQL, Oracle, Postgres и SQLite. Navicat удовлетворяет практически всем требованиям по управлению СУБД: поддерживается соединение по защищенному протоколу (SSL), импорт/экспорт, выполнение задач по расписанию, построение отчётов, backup данных. В версии 9 Navicat произошли следующие основные изменения: Появилась поддержка SQLite. Добавлен отладчик PL/SQL для СУБД Oracle. Улучшено автодополенение кода. Появилась возможность форматирования кода. Отображение кода в виде дерева. Восстановление документов после краха программы. Возможно сохранение любимых запросов в виде ярлыков. Улучшен пользовательский интерфейс. Linux версия программы стоит от 199 до 329 долларов США, доступна пробная версия на 30 дней.
