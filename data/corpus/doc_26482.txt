Подготовлен порт OpenSSH под Windows

Компания NoMachine, известная своим продуктом для удалённого администрирования NX, объявила, что силами её разработчиков был подготовлен нативный порт OpenSSH для платформы Windows. Проект уже работоспособен, но еще требует доработки, в первую очередь, в плане интеграции sshd в службу сервисов (сейчас он может работать в этом качестве только посредством cygrunsrv и только с разрешением службе взаимодействовать с рабочим столом пользователя) из комплекта Cygwin. На данный момент реализованы прослойки для fork() (через CreateProcess()), select() и других POSIX API. Аутентификация происходит через родные API Windows, то есть нет проблем с AD или его отсутствием.
