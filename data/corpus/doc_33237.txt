Релиз дистрибутива для создания сетевых хранилищ FreeNAS 8.0.4

Доступен релиз дистрибутива для быстрого развёртывания сетевого хранилища (NAS, Network-Attached Storage) - FreeNAS 8.0.4. Дистрибутив основан на FreeBSD 8 и полностью настраивается через веб-интерфейс, поддерживает FTP, NFS, Samba, AFP, rsync и iSCSI протоколы, программный RAID (0,1,5). Для авторизации может быть использован LDAP/Active Directory. Размер iso-образа составляет 101 Мб. Основные изменения: Медиасервер firefly обновлён до версии 1696_6 для обеспечения совместимости с iTunes 10.5.2+; Реализовано автоматическое создание домашних директорий для пользователей AD/LDAP; В конфигурации CIFS активирована настройка 'allow trusted domains'; Добавлена функциональность rsync+SSH; BitTorrent-клиент transmission обновлён до версии 2.42; В состав включена утилита MegaCLI; Отключены периодически запускаемые задачи, которые могли негативно повлиять на производительность системы на хранилищах с большим числом директорий; В web-интерфейсе 'loaders' переименованы в 'tunables' для обеспечения соответствия с терминологией FreeBSD (kernel tunables); При удалении снапшотов через web-интерфейс более не запускается collectd (бэкенд для формирования отчётов); В состав включены скрипты arcstat.py и arc_summary.py;
