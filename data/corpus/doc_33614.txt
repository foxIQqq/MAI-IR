Релиз ROSA Media Player 1.0

Компания РОСА представила первый стабильный релиз мультимедиа проигрывателя ROSA Media Player (ROMP), основанного на наработках проекта SMPlayer и являющегося надстройкой над MPlayer и FFmpeg. Среди ключевых целей проекта отмечено обеспечение максимального охвата поддерживаемых форматов в сочетании с разумными настройками по умолчанию и не перегруженным интерфейсом пользователя. Причиной создания форка является другой подход к формированию интерфейса и другая идеология развития проекта, что стало непреодолимым препятствием при попытках продвижения своих улучшений в SMPlayer. Код ROSA Media Player можно загрузить из Git-репозитория проекта или из репозитория пакетов main для Mandriva и ROSA Linux (пакеты rosa-media-player и rosa-media-player-plugin). Список поддерживаемых форматов идентичен возможностям MPlayer. Ключевые особенности ROSA Media Player: Более лаконичный и упрощённый интерфейс пользователя; Модуль интеграции с браузерами Firefox и Chromium, позволяющий просматривать потоковое видео; Возможность выделения и вырезания отдельных фрагментов видео. Функция обрезки видео по краям; Поддержка создания снимка экрана, позволяющая сохранить содержимое кадра в формате PNG; Возможность сохранения в отдельном файле аудио дорожки из видео. Звук можно сохранять в форматах ogg и mp3; Встроенные функции по созданию скринкастов (экранных презентаций). При выборе соответствующей опции в проигрывателе, в видеофайл записываются все действия, происходящие на экране. Для завершения записи нужно нажать на индикатор в системном лотке; Дополнительные фильтры для нормализации изображения и борьбы с искажениями. Например, фильтр для удаления эффекта "гребенки" при котором границы движущихся объектов становятся нечеткими, словно по объекту «прошлись» расческой. Среди других фильтров: смазывание границ квадратов, удаление краевых эффектов, несколько методов подавления шумов; Гибкие средства изменения размера области просмотра и скорости воспроизведения. Возможность настройки контрастности, яркости, оттенков, насыщенности; Средства для выбора заданной аудио дорожки и использования звука из внешнего файла. Наличие эквалайзера и фильтров; Поддержка поиска субтитров через сервис opensubtitles.org; Из планов на будущее отмечается продолжение усовершенствования интерфейса программы, новая система настроек, интеграция с онлайн-сервисами (например, YouTube) для поиска и просмотра видео, cоздание версии проигрывателя для Windows.
