Вышел Turnkey Linux 13.0, набор мини-дистрибутивов для быстрого развертывания приложений

Доступна новая версия проекта Turnkey Linux 13.0, в рамках которого сформирована коллекция из 1400 минималистичных сборок Debian общим размером 330 Гб, пригодных для использования в качестве гостевых ОС в системах виртуализации VMware, Xen, OpenVZ, KVM, VirtualBox или для быстрого развертывания в Cloud-окружениях Amazon EC2. Средний размер каждой сборки - 200 Мб. Сборки сформированы для 100 приложений, в 7 вариантах для 64- и 32-разрядных систем. Сразу после установки пользователь имеет возможность получить полностью работоспособные из коробки рабочие окружения с LAMP (Linux, Apache, MySQL, PHP/Python/Perl), Ruby on Rails, Joomla, MediaWiki, Wordpress, Drupal, Apache Tomcat, LAPP, Django, MySQL, PostgreSQL, Node.js, Jenkins, Typo3, Plone, SugarCRM, punBB, OS Commerce, ownCloud, MongoDB, OpenLDAP, gitlab, CouchDB и т.д. Управление программным обеспечением производится через специально подготовленный web-интерфейс. Сборки снабжены системой автоматического резервного копирования и средством для автоматической установки обновлений. Новая версия примечательна переходом на пакетную базу Debian Wheezy (7.2) и добавлением новых сборок с OpenVPN, Observium и Tendenci.
