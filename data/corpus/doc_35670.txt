Релиз распределенной платформы для обмена медиафайлами GNU MediaGoblin 0.3.2

Доступна новая версия GNU MediaGoblin 0.3.2, системы для организации обмена фотографиями, видеороликами и другими видами медиаконтента. В отличие от централизованных сервисов, подобных Fliсkr и Picasa, MediaGoblin нацелен на организацию обмена контентом без привязки к конкретному сервису, используя модель похожую на Jabber и StatusNet. Пользователь может поднять сервер на своих мощностях и связать его с другими аналогичными серверами, организовав таким образом единое поле для обмена данными. Ключевым новшеством представленного выпуска является поддержка загрузки в MediaGoblin 3D-моделей в форматах STL и OBJ, что позволяет создавать свой сервис для обмена проектами физических объектов, подобный Thingiverse. Для создания эскизов загружаемых 3D-объектов предусмотрена возможность формирования изображений через вызов Blender. Кроме того, для просмотра 3D-моделей в форматах STL и OBJ может быть использован просмотрщик thingiview.js, написанный на JavaScript с использованием Canvas и WebGL. Кроме того, теперь пользователь может создавать именованные коллекции (галереи), и добавлять в них ссылки на файлы, в том числе загруженные другими пользователями. Добавление производится нажатием на странице файла ссылки, пока что с изображением скрепки. Появился API и поддержка OAuth. В разработке — клиент для платформы Android.
