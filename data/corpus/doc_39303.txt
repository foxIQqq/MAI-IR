Переход Ubuntu на Mir может растянуться до 2016 года

На проходящем в эти дни саммите разработчиков Ubuntu Марк Шаттлворт высказал мнение, что несмотря на то, что дисплейный сервер Mir уже используется по умолчанию в вариантах Ubuntu для планшетов и смартфонов, переход Ubuntu для настольных систем на Mir может затянуться. В качестве целевого выпуска, в котором намечен переход на Mir, отмечается Ubuntu 16.04, который выйдет в апреле 2016 года. Внедрение Mir по умолчанию в более ранних выпусках не гарантируется. Решение о переходе по умолчанию будет зависеть от степени стабилизации решения на базе Mir. При этом, так как Ubuntu 16.04 является выпуском с длительным сроком поддержки для полной стабилизации работы новой графической подсистемы она должна быть протестирована при использовании по умолчанию хотя бы в рамках одного промежуточного выпуска Ubuntu. В настоящее время десктоп-окружение на базе Mir оценивается как уступающее по стабильности рабочему столу на основе X-сервера, состояние окружения Unity 8 также оставляет желать лучшего. Проблемы со стабильностью усугубляет отсутствие поддержки Mir в проприетарных драйверах AMD и NVIDIA, а также в upstream-проектах открытых драйверов (Mir пока поддерживается через внешние патчи). По мнению Шаттлворта, любовь к технологиям не должна перекрывать миссию проекта, ориентированную на обеспечение комфортной работы пользователя. В ситуации с версией Ubuntu для смартфонов, которая создаётся с нуля, продвигать новые технологии значительно проще, так как разработку не тяготит задача обеспечения совместимости. Для настольных систем внедрение Mir будет проходить постепенно: вначале пользователям будет предоставлена опциональная возможность установки варианта графической подсистемы на базе Mir и Unity 8. По мере роста числа пользователей, перешедших на Mir, и решения имеющихся проблем, будет оцениваться возможность внедрения Mir по умолчанию. Некоторые другие темы, упомянутые в выступлении Марка Шаттлворта или рассмотренные в процессе обсуждений на саммите: Миграцию на системный менеджер systemd планируется провести к выпуску Ubuntu 16.04. Как и в случае с Mir, переход дистрибутива на systemd по умолчанию будет зависеть от стабильности и обеспечения совместимости с ранее предлагаемым пользователям окружением. Возможность начать тестирования systemd в Ubuntu будет предоставлена в ближайшем будущем. Марк Шаттлворт упомянул, что стоимость первых смартфонов, поставляемых с Ubuntu, составит от 200 до 400 долларов. При этом смартфоны нацелены на занятие верхнего края рынка устройств средней ценовой категории, предлагая богатую по возможностям начинку по достаточно умеренной цене. Представлена новая концепция секционной компоновки заголовков мобильных приложений, позволяющая совместить информацию о текущем состоянии приложения, средства для переключения между экранами и список возможных действий. Планируется обеспечить поддержку разработки мобильных приложений для Ubuntu на языке Go, который разработчики из Canonial считают перспективным и уже используют в некоторых внутренних проектах; Пакет с HTTP-сервером nginx включён в состав main-репозитория Ubuntu 14.04 LTS, т.е. для пакета с nginx для Ubuntu теперь будет применяться длительный цикл выпуска обновлений.
