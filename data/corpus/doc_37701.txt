Представлен Plasma Media Center 1.1, медиацентр от проекта KDE

Проект KDE объявил о выходе Plasma Media Center 1.1 (PMC), унифицированного интерфейса для работы с мультимедиа-контентом на стационарных ПК, нетбуках, планшетах, телевизорах и других устройствах, на которых может быть запущено окружение KDE. PMC обладает типичными для медиацентров функциями и позволяет показывать изображения, проигрывать музыку и воспроизводить видео. PMC базируется на технологиях Plasma и KDE, и использует QML для формирования интерфейса, адаптированного для различных классов устройств. Для расширения функциональности предоставляется API для создания плагинов, Поддерживается работа как с содержимыми локальных файловых систем, так и внешних online-сервисов, таких как, YouTube Flickr и Picasa. Управление воспроизведением и навигация по контенту могут осуществляться через клавиатуру/мышь, сенсорный экран или пульты дистанционного управления. Для унифицированного просмотра всего медиаконтента доступна возможность использование средств поиска на десктопе KDE (KDE Desktop Search). Среди добавленных новшеств: Режим интеграции с сервисом YouTube, позволяющий искать и просматривать видеоролики, размещённые в YouTube; Улучшенный интерфейс для работы со списком воспроизведения, отличающийся показом большего числа деталей о композициях. Например, выводится имя исполнителя и продолжительность композиции. Для перегруппировки и удаления элементов в списке теперь можно использовать механизм Drag & Drop. Также доступен режим shuffling для случайной смены списков воспроизведения; Улучшен интерфейс пользователя. Полностью переработан код для управления с клавиатуры, для которой теперь используется стиль обработки событий похожий на управление с пульта, не привязанный к фокусу текущего ввода. Реализован режим навигации по музыкальной коллекции ("All Music") с использованием клавиатуры; В следующем выпуске планируется добавить настройки для управления индексацией медиаколлекции, реализовать поиск по локальным файлам, добавить предпросмотр картинок в директориях, полностью переработать режим работы с музыкальной коллекций ("All Music"), в том числе добавить вывод обложек альбомов.
