Вышел PelicanHPC 2.2, Linux дистрибутив для развертывания кластеров

Вышел PelicanHPC 2.2, Linux дистрибутив для быстрого развертывания кластеров. PelicanHPC построен на пакетной базе Debian testing и позволяет превратить обычный офисный компьютерный парк в вычислительный кластер, загрузив один управляющий компьютер с LiveCD/LiveUSB, а остальные машины использовав в качестве загружаемых через PXE узлов. Графический интерфейс управляющей станции построен на основе Xfce4. Размер установочного iso-образа 697 Мб. В состав входят такие кластерные приложения, как пакет Open MPI, х MPI-биндинги для GNU Octave, интерфейс мониторинга работы кластера Ganglia, набор для тестирования производительности кластеров Linpack. Из новшеств отмечается реализация упрощенного способа сохранения конфигурации загружаемых по сети узлов, что позволяет организовать полноценное централизованное удаленное управление кластером, используя дистрибутив для обеспечения функционирования постоянно работающих кластеров, а не только для проведения одноразовых экспериментов.
