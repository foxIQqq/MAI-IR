Релиз Linux-дистрибутива KNOPPIX 6.7.1

Увидел свет релиз одного из первых Live-дистрибутивов - KNOPPIX 6.7.1. Дистрибутив построен на основе оригинального набора загрузочных скриптов и пакетов, импортированных из Debian Squeeze, со вставками из веток Debian "testing" и "unstable". Для загрузки доступно три варианта: LiveCD (700 Мб), LiveDVD (3.7 Гб) и предназначенная для людей с ослабленным зрением сборка (700 Мб) с интегрированным звуковым меню ADRIANE. Торрент-файлы доступны здесь. Пользовательская оболочка дистрибутива основана на легковесном десктоп-окружении LXDE, построенном на базе библиотеки GTK+ и способном работать на маломощных системах. Вместо стандартной системы SysV инициализации задействована новая система загрузки Microknoppix, значительно ускоряющая процесс загрузки дистрибутива за счет параллельного запуска сервисов и отложенной инициализации оборудования. Время полной загрузки дистрибутива, с учетом установки WiFi-соединения, составляет около 50 секунд. При использовании USB Flash пользовательские настройки и дополнительно установленные программы не исчезают после перезагрузки системы. Сохраняемые между сессиями данные помещаются в файл KNOPPIX/knoppix-data.img, который при желании можно зашифровать с помощью алгоритма AES-256. В новой версии: В состав DVD-версии включено приложение eViacam, позволяющее двигать курсор, отслеживая перемещение головы через web-камеру; Осуществлено обновление ядра Linux до версии 3.0.4; В состав включен Nouveau, открытый драйвер для видеокарт NVIDIA; OpenOffice.org заменен на LibreOffice 3.4.3; Обновлены версии браузеров: Chromium 13.0.782.220 и Firefox 6.0.2; X.Org Server обновлен до версии 1.11; Добавлена опция загрузки 'knoppix64' позволяющая загрузить 64-разрядную сборку ядра Linux; Добавлена опция для загрузки с монтированием сжатой файловой системы из ISO-образа. Например, "bootfrom=/dev/sda1/knoppix-test.iso"; Добавлена опция 'grub' для перехода в оболочку загрузчика. Обновлено до версии 1.4 звуковое меню ADRIANE, которое включает в себя реализацию пользовательского окружения, основанного на идее звуковой навигации. Для голосового чтения содержимого страниц задействована система Orca. В новой версии в качестве движка распознания сканированного текста служит Cuneiform. В ADRIANE 1.4 также добавлена поддержка gammu, что позволило организовать поддержку работы с SMS, через подключенный к ПК телефон; Процесс загрузки модифицирован для обеспечения работы дистрибутива под управлением Virtualbox без дополнительных задержек.
