Internet Explorer 11 будет отправлять строку идентификации, схожую с Mozilla Firefox

Компания Microsoft в будущей версии Internet Explorer пошла на беспрецедентный шаг — новая 11 версия браузера, которая будет включена в Windows 9, будет отправлять сайтам следующую строку идентификации: Mozilla/5.0 (IE11.0; Windows NT 6.3; Trident/7.0; rv11.0) like Gecko Если присмотреться к новому идентификатору программы, станет видно, что Microsoft практически превратила Internet Explorer в Mozilla Firefox. Сравним идентификаторы Microsoft Internet Explorer 10 под Windows 7: Mozilla/5.0 (compatible; MSIE 10.0; Windows NT 6.1; Trident/6.0) и идентификатор Mozilla Firefox 19 под Windows 7: Mozilla/5.0 (Windows NT 6.2; rv:19.0) Gecko/20130104 Firefox/19.0 Как видно, Microsoft заменила MSIE на просто IE и добавила дополнение "like Gecko". Вероятно, таким образом корпорация хочет оставить в прошлом сайты и страницы, оптимизированные специально под Internet Explorer.
