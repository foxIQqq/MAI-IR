Доступна мобильная платформа KDE Plasma Mobile 22.02

Опубликован выпуск мобильной платформы KDE Plasma Mobile 22.02, основанной на мобильной редакции рабочего стола Plasma 5, библиотеках KDE Frameworks 5, телефонном стеке ModemManager и коммуникационном фреймворке Telepathy. Для вывода графики в Plasma Mobile используется композитный сервер kwin_wayland, а для обработки звука применяется PulseAudio. Одновременно подготовлен выпуск набора мобильных приложений Plasma Mobile Gear 22.02, формируемого по аналогии с набором KDE Gear. Для создания интерфейса приложений применяется Qt, набор компонентов Mauikit и фреймворк Kirigami из состава KDE Frameworks, позволяющий создавать универсальные интерфейсы, пригодные для смартфонов, планшетов и ПК. В состав входят такие приложения, как KDE Connect для сопряжения телефона с рабочим столом, просмотрщик документов Okular, музыкальный проигрыватель VVave, просмоторщики изображений Koko и Pix, система ведения заметок buho, календарь-планировщик calindori, файловый менеджер Index, менеджер приложений Discover, программа для отправки SMS Spacebar, адресная книга plasma-phonebook, интерфейс для осуществления телефонных вызовов plasma-dialer, браузер plasma-angelfish и мессенджер Spectral. В новой версии: В мобильную оболочку перенесены изменения из недавнего выпуска KDE Plasma 5.24. Основной репозиторий с мобильной оболочной переименован из plasma-phone-components в plasma-mobile. Переработана выпадающая панель быстрых настроек, в которую также добавлены новые виджеты для управления воспроизведением мультимедийного контента и показа уведомлений. Улучшена обработка управляющих жестов. Добавлен базовый вариант панели быстрых настроек для планшетов, который планируется улучшить в следующем выпуске. Переписан интерфейс для переключения между запущенными приложениями (Task Switcher), который переведён на использование одной строки с эскизами приложений и теперь поддерживает управляющие жесты. В панели навигации устранены ошибки, из-за которой панель временами становилась серой и нарушался показ эскизов приложений. В дальнейшем планируется реализовать возможность полного управления жестами без привязки к панели навигации. На домашний экран добавлена возможность вызова интерфейса поиска программ KRunner через проведение пальцем сверху вниз по сенсорному экрану. Повышена точность срабатывания экранных жестов для открытия и закрытия приложений. Решены проблемы, возникающие при размещении или удалении плазмоидов на домашнем экране. Индикатор запуска приложения и интерфейс для переключения между приложениями переведены на использование основного окна домашнего экрана, без создания новых окон, что заметно повысило плавность анимации на устройстве Pinephone. В конфигураторе реализована функция поиска и изменён стиль заголовка, в котором теперь применяется более компактная кнопка для возврата к предыдущему экрану. Добавлен вариант оформления конфигуратора для планшентов. Переработан бэкенд, отвечающий за срабатывание будильника. В будильнике переработан интерфейс редактирования списков и улучшена поддержка назначения собственных рингтонов. Добавлен встроенный диалог для настройки сигнала и таймеров. Началась модернизация интерфейса календаря-планировщика Calindori. Переработана навигация в программе PlasmaTube, предназначенной для просмотра видео с YouTube. Панель управления перемещена вниз экрана, а в заголовок добавлена кнопка для возврата к прошлому экрану. В программе для прослушивания подкастов Kasts проведена оптимизация элементов управления для ландшафтного режима. Добавлена возможность сворачивания в системный лоток программы для обмена сообщениями NeoChat (форк программы Spectral, переписанный с использованием фреймворка Kirigami для создания интерфейса и библиотеки libQuotient для поддержки протокола Matrix). В NeoChat также улучшена проверка наличия сетевого соединения, реализована возможность прикрепления меток к учётным записям (для наглядного разделения нескольких аккаунтов), добавлена поддержка прямого обмена файлами с другими приложениями и online-сервисами, такими как Nextcloud и Imgur. Предложен мобильный вариант диалогов, применяемых для получения полномочий при доступе к ресурсам при помощи порталов Freedesktop (xdg-desktop-portal). В эмуляторе терминала QMLKonsole налажена обработка кнопок Ctrl и Alt. В web-браузере Angelfish реализован вариант интерфейса для десктоп-систем, в котором доступны те же функции, что и в интерфейсе для мобильных устройств. Изменение позволит использовать одну кодовую базу для развития редакций Angelfish для мобильных и настольных систем. Для настройки задействован интерфейс Kirigami.CategorizedSettings. Обновлён список фильтров в блокировщике рекламы. Стиль кнопок в интерфейсе для осуществления телефонных вызовов (Plasma Dialer) приведён к стилю других приложений Plasma Mobile. Улучшена навигация между страницами, добавлена страница с настройками и страница About. Реализован диалог для очистки истории вызовов. Предоставлена возможность выбора телефонных номеров через меню Contacts, при привязке к одному пользователю нескольких номеров телефона. Обеспечена остановка мультимедийных проигрывателей при поступлении входящего вызова.
