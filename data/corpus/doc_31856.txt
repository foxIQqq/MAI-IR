Релиз минималистичного дистрибутива Tiny Core Linux 4.0

Вышел минималистичный Linux дистрибутив Tiny Core Linux 4.0, который может работать на системах с 48Мб ОЗУ. Загрузочный iso-образ занимает всего 12 Мб. Графическое окружение дистрибутива построено на основе X-сервера Tiny X, тулкита FLTK и оконного менеджера FLWM. Дистрибутив загружается целиком в ОЗУ и работает из памяти. Дополнительно поставляется консольная версия дистрибутива - Micro Core, занимающая 6 Мб и построенная на основе набора утилит BusyBox. Также доступна сборка MultiCore (45 Мб), снабженная графическим инсталлятором и поставляемая с набором дополнительных расширений. Кроме стандартной начинки Tiny Core и Micro Core в состав Multi Core включено два дополнительных загрузочных пакета: install.gz - инсталлятор с поддержкой установки Tiny Core Linux в виде файла на NTFS-раздел и с возможностью установки дополнительных расширений; network.gz - готовый набор инструментов для обеспечения выхода в сеть, включая менеджер для настройки Wifi-соединений. Основные улучшения: Обновление версий программ: ядро Linux 3.0.3, busybox 1.19.2, udev 173, e2fsprogs 1.41.14, gcc 4.6.1, util-linux 2.19.1. Кроме того обновлены библиотеки Xlibs (Xlibs.tcz); Системная библиотека glibc заменена на eglibc 2.13; Обновлены все разработанные в недрах проекта утилиты, которые переведены на использование нового репозитория; Обновлены tce-load и tce-audit для динамической обработки зависимостей ядра; Модифицирован .xsession для корректной обработки проблем с запуском X-сервера; Обновлены апплеты для поиска приложений (AppBrowser Search и AppBrowser Keyword), а также внутренние скрипты search.sh и keyword.sh, в которых реализован единый с AppBrowser метод поиска и работы с ключевыми словами; Задействована написанная на языке Си программа loadcpufreq; В busybox добавлен патч с поддержкой nbd-client. В rebuildfstab, tc-config и rc.shutdown добавлена поддержка nbd-client; Скрипт getTime.sh переписан для случайного выбора серверов синхронизации времени; Обновлены шрифты Luxi, отключен хинтинг шрифтов.
