Выпуск H3Droid 1.3.2, Android-прошивки для плат на процессоре Allwinner

Доступен новый выпуск прошивки H3Droid, построенной на основе платформы Android для плат на базе процессоров Allwinner H3, таких как Xunlong Orange Pi, FriendlyArm NanoPi, Beelink x2, pcDuino4 и BananaPi. Прошивка основана на платформе Android 4.4 (KitKat) и может функционировать на устройствах с 512 Mb ОЗУ. Размер загрузочного образа для записи на SD-карту 165 Мб. Пользователям предлагается готовое к работе графическое окружение, пригодное для выполнения любых Android-приложений, а также включающее подборку штатных утилит командной сроки. Из окружения доступен каталог Google Play, имеется поддержка воспроизведения видеороликов с YouTube. Поддерживается превращение H3Droid в мультимедийный центр через установку Kodi. Через SSH и ADB предоставляется доступ с правами root. По умолчанию предлагается языковой пакет для английского языка, но можно доустановить пакеты и для других языков. Из поддерживаемого оборудования отмечаются различные модели встроенных (Orange Pi PC Plus/Plus 2E) и внешних WiFi-адаптеров (realtek 8188cu, 8188eu, 8192cu, 8723, 8812au/8821au, atheros ath9k, ath6k, ralink rt2800usb), а также встроенные Ethernet-контроллеры. Встроенные Bluetooth-чипы не поддерживаются, но можно подключить внешние Bluetooth-адаптеры с интерфейсом USB. Для подключения экрана можно использовать HDMI и CVBS. Для переключения USB-порта в режим OTG Gadget в поставку входит приложение UsbModeSwitch. Для установки применяется единый инсталлятор, универсальный для всех плат. Инсталлятор встроен в базовый образ и вызывается при первой загрузке, давая возможность выбрать script.bin(fex) и uboot для текущей платы, а также внести изменения в настройки (выбрать разрешение экрана, добавить сервисы для автозапуска, сгенерировать SSH-ключи, изменить размер раздела и т.п.). Теоретически поддерживаются любые платы на базе Allwinner H3, но в первую очередь тестирование сборок проводится для плат Orange Pi и NanoPi. При удержании клавиши включения питания во время запуска активируется режим восстановления после сбоя. В новом выпуске: Модернизировано оформление окружения Launcher3 и навигационной панели. В настройки включены секции для контроля потребления трафика, управления средствами для людей с ограниченными возможностями и блокировкой экрана (keyguard). Добавлена настройка для использования файла подкачки /data/swapfile (128M) вместо zswap. В поставку возвращены звуковые файлы (кроме рингтонов). Устранены проблемы в работе интерфейса выбора обоев рабочего стола, установщика h3armbian (не вызывалась финальная стадия кастомизации) и кнопки изменения настроек. Из панели удалён индикатор заряда аккумулятора. Решены проблемы с работой WiFi и добавлена поддержка беспроводных адаптеров mt7601u с интерфейсом USB. Проведена оптимизация ядра Linux, изменены настройки потребления памяти, включены средства защиты stackprotector и seccomp. Обеспечена работа CSR-совместимых USB-брелоков Bluetooth (встроенные чипы Bluetooth пока не поддерживаются).
