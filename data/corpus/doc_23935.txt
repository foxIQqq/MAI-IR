Mozilla внедрит в Firefox поддержку формата Web Open Font Format (WOFF)

Разработчики Mozilla приняли решение внедрить поддержку формата Web Open Font Format (WOFF) в браузере Firefox 3.6, релиз которого ожидается в ноябре. Web Open Font Format - контейнер шрифтов OpenType, Open Font Format или TrueType. В последствии Mozilla будет продвигать WOFF как web-стандарт. В WOFF используется сжатие данных, поэтому его размер намного меньше, чем шрифты TrueType или OpenType, поддержка загрузки которых была интегрирована в Firefox 3.5. Кроме того, формат WOFF позволяет добавлять при компоновке файла с шрифтом дополнительные метаданные, например, информацию об авторах или определение допустимых рамок использования шрифтов.
