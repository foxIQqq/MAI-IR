Релиз Linux-дистрибутива Slackware 13.37

После года разработки Патрик Фолькердинг (Patrick Volkerding) представил релиз Linux-дистрибутива Slackware 13.37. При подготовке нового выпуска разработчики впервые за восемнадцатилетнюю историю существования дистрибутива отступили от привычной нумерации версий и решили вместо версии 13.2 использовать знаковое число 13.37. Для загрузки доступен установочный DVD (4.3 Гб), который подготовлен для архитектур i386 и x86_64. Подборку дополнительных пакетов с программами, отсутствующими в стандартной поставке, можно найти в репозитории slackbuilds.org. В рамках проекта ARMedslack развивается порт Slackware для архитектуры ARM. Особенности Slackware 13.37: В новой версии присутствует поддержка двух графических окружений - легковесного на базе Xfce 4.6.2 и полнофункционального на основе KDE 4.5.5. Поддержка GNOME развивается в рамках отдельного проекта GNOME SlackBuild (доступны пакеты с GNOME 3.0.0). Для обеспечения работы графических приложений используется X.Org Server 1.9.4; По умолчанию задействовано Linux-ядро 2.6.37.6, так как при использовании ветки 2.6.38 наблюдались проблемы в работе дистрибутива. Использование ядра 2.6.37 позволило расширить поддержку в Slackware журналируемых и шифрованных ФС, SCSI и ATA RAID, SATA, Software RAID, LVM (Logical Volume Manager) и DRI (the Direct Rendering Interface). Ядро собрано с дополнительными патчами от проекта Speakup, что позволяет задействовать синтезатор речи для обеспечения работы людей, имеющих проблемы со зрением. Для желающих использовать более новые версии, пакеты с ядрами 2.6.38.4 и 2.6.39-rc4 можно найти в директории в /testing; Пакеты собраны с использованием компилятора GCC 4.5.2, система работает под управлением системной библиотеки Glibc 2.13; Продолжена работа по интеграции в дистрибутив системы udev, позволяющей упростить подключение различных устройств, таких как USB-накопители и web-камеры, без необходимости выполнения каких-либо действий вручную. Для инициализации оборудования и автоматической настройки сетевых устройств в инсталляторе Slackware также задействован udev; Улучшена интеграция подсистем ConsoleKit и PolicyKit, первая для управления сессиями и трекингом работы пользователей, а вторая для предоставления расширенных прав доступа, давая возможность пользователям выполнять некоторые привилегированные действия, не имея прав суперпользователя; Обновлены инструменты для управления пакетами, позволяющие упростить создание, установку, удаление и обновление собственных пакетов программ. Функции отслеживания пакетов позволяют легко обновить Slackware 13.1 до релиза 13.37. Для обновления с более ранних выпусков следует использовать утилиту slackpkg; Для повышения качества рендеринга векторных шрифтов пакет freetype собран с включенным интерпретатором TrueType-байткода, который ранее был отключен из-за патентных ограничений; Стабилизирована работа KMS-модулей (Kernel Mode Setting). По умолчанию KMS-драйверы используются для видеокарт Intel, ATI и NVIDIA. В состав дистрибутива включен драйвер Nouveau (пакет xf86-video-nouveau) и прошивки с микрокодом для карт ATI Radeon (radeon_ucode); В состав дистрибутива включены пакеты с прошивками для беспроводных карт Intel iwlwifi-100-ucode и iwlwifi-6xxx-ucode; Новые версии программ: Web-браузеры и почтовые клиенты: Konqueror 4.5.5, SeaMonkey 2.1b3, Firefox 4.0, Thunderbird 3.1.9; Пользовательские приложения на базе технологий KDE: amarok 2.4.0, koffice 2.3.3, k3b 2.0.2, ktorrent 4.0.5; Коллекция приложений на базе GTK+: pidgin 2.7.11, gimp 2.6.11, gkrellm 2.3.5, xchat 2.8.8, xsane 0.998, pan 0.134; Web-технологии: Apache (httpd) 2.2.17 и PHP 5.3.6; Шифрование: OpenSSL 0.9.8r, OpenSSH 5.8, OpenVPN 2.1.4 и GnuPG 2.0.17; Средства разработки: Perl 5.12.3, Python 2.6.6, Ruby 1.9.1-p431, Subversion 1.6.16, git 1.7.4.4, mercurial 1.8.2, KDevelop 4.2.0; В extra-репозитории представлены Java JDK 6 update 25, набор библиотек для совместимости с kde3, браузерный плагин на базе mplayer и т.д.
