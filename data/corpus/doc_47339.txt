Выпуск операционной системы Chrome OS 61

Компания Google представила релиз операционной системы Chrome OS 61, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 61. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 61 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 61: Улучшены средства управления окнами в режиме для планшетов с сенсорными экранами; Новое меню приложений (App Launcher), оптимизированное для управления с сенсорного экрана; Новый аватар по умолчанию и интерфейс изменения аватара; Новое оформление экранов входа в систему и блокировки. В секцию настроек управления энергопотреблением добавлены опции для управления переходом в спящий режим при закрытии крышки ноутбука (теперь можно выбрать перейти в спящий режим или продолжить работу в фоне) и для выбора действия при длительной неактивности (выключить экран, перейти в спящий режим или продолжить работу); Улучшенный интерфейс для просмотра системной справки; В число экспериментальных возможностей (ash-enable-night-light в about:flags) добавлен режим ночной подсветки ("Night Light"), при котором вечером цветовая гамма оформления меняется на более тёплые тона, снижающие утомление глаз и минимизирующие факторы возникновения бессонницы после работы перед сном;
