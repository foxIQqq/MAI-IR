Открыт код системы для проверки корректности оформления CSS-файлов - CSS Lint

Открыт исходный код online-сервиса CSS Lint, позволяющего выполнить анализ CSS-блоков с целью выявления возможных проблем. Система проверяет корректность синтаксиса CSS, пытается выявить использование неэффективных конструкций и недопустимых сочетаний параметров, определяет типичные логические ошибки и элементы, которые вызывают проблемы в определенных браузерах. CSS Lint поддерживает подключение дополнений, что может быть использовано для написания собственных модулей для выполнения дополнительных проверок. Код проекта написан на JavaScript, предназначен для выполнения на серверной платформе node.js и распространяется в рамках лицензии MIT. Поддерживается работа не только в виде web-сервиса, но и как утилиты командной строки.
