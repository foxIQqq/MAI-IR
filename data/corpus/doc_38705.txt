Выпуск дистрибутива Sabayon Linux 14.01

Представлен релиз дистрибутива Sabayon Linux 14.01, сочетающего в себе удобство для конечного пользователя с гибкостью Gentoo. Sabayon Linux оформлен в виде Live-дистрибутива, снабжённого возможностью лёгкой установки на жёсткий диск. Sabayon использует модифицированную rolling-модель подготовки обновлений, которая делает выпуск обновлений более предсказуемым и менее болезненным. Дистрибутив доступен в сборках с KDE 4.11.4 (2.4 Гб), GNOME 3.10.3 (1.9 Гб) и Xfce 4.10 (1.6 Гб). Пакетная база дистрибутива синхронизирована с коллекцией портажей Gentoo и полностью совместима с Portage 2.2 (EAPI2), пользователь имеет возможность установки программ как из исходных текстов, так и используя готовые бинарные пакеты, собранные для платформ ARMv7, x86 и x86-64. Для x86 и x86-64 в репозитории насчитывается около 14000 пакетов, а для ARMv7 - более 2000. Поддерживаемые пакеты обновляются непрерывно (rolling-релиз), предоставляя пользователю доступ к самым свежим версиям программ. Для управления пакетами подготовлен GUI интерфейс "Rigo" и консольная программа "equo". Для установки дистрибутива используется инсталлятор, основанный на исходных текстах проекта Anaconda, развиваемого в рамках Fedora Linux. Для инициализации и управления службами применяется systemd. В комплекте поставляется полный набор всех необходимых мультимедиа кодеков и проприетарных драйверов для видеокарт, беспроводных адаптеров и софт-модемов. Особенности нового выпуска: Дистрибутив поставляется с ядром Linux 3.12.5 с планировщиком ввода/вывода BFQ и поддержкой файловой системы ZFS. Дополнительно доступны для установки пакеты с ядрами Linux 3.4 и 3.10; Обновление версий программ, в том чиле GNOME 3.10.3, KDE 4.11.4, Xfce 4.10, LibreOffice 4.1.3. В репозиториях доступен Cinnamon 2.0; Включение в базовую поставку сборок с KDE и GNOME клиента для сервиса доставки игр Steam и подготовка профиля для запуска клиента Steam в режиме Big Picture (запуск производится по аналогии с входом в сеанс медиа-центра на базе XBMC); Обновление пакетного менеджера Entropy 254 с поддержкой параллельного выполнения операций, например, теперь можно одновременно выполнять установку и удаление пакетов. Поддержка параллелизма также добавлена в бэкенд PackageKit 0.8 Entropy. Расширены возможности Rigo, интерфейса к Entropy для управления установкой программ, пришедшего на смену приложению Sulfur. Rigo отличается современным дизайном интерфейса, выполненным на базе Gtk3+ и ориентированным на активное использование поиска. Непосредственным выполнением операций с пакетам занимается специальный демон RigoDaemon, взаимодействие с которым организовано через DBus. В новой версии Rigo реализована возможность оперативного вывода уведомлений о наличии обновлений с поддержкой установки обновлений в один клик, без необходимости ввода пароля администратора (для обновлений без пароля пользователь должен быть включен в группу "entropy"). Кроме Python 2.7 предоставлена возможность использования Python 3.3.
