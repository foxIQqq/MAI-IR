Вышел Linux-дистрибутив для создания межсетевых экранов IPFire 2.7-40

Вышел релиз специализированного Linux-дистрибутива IPFire 2.7 Core Update 40, предназначенного для создания межсетевых экранов. IPFire отличается предельно простым процессом установки и организацией настройки через интуитивно понятный web-интерфейс, изобилующий наглядными графиками. Размер установочного iso-образа составляет всего 62 Мб. Для установки дополнений в IPFire используется специальный пакетный менеджер Pakfire. Система модульная, кроме базовых функций пакетной фильтрации и управления трафиком для IPFire доступны модули с реализацией системы для предотвращения атак на базе Snort, для создания файлового сервера (Samba, FTP, NFS), почтового сервера (Cyrus-IMAPd, Postfix, Spamassassin, ClamAV и Openmailadmin) и сервера печати (CUPS), организации VoIP шлюза на базе Asterisk и Teamspeak, создания беспроводной точки доступа, организации потокового аудио и видео-сервера (MPFire, Videolan, Icecast, Gnump3d, VDR). В представленном обновлении добавлены следующие новшества: Обновлены входящие в комплект программы: strongswan 4.4.1, openvpn 2.1.2, snort 2.8.6.1, python 2.7, cpio 2.11; Обновлены драйверы устройств: Intel igb 2.3.4, compat-wireless 2.6.35-1, добавлена поддержка usbid Huawei Android; В модуль для обеспечения работы межсетевого экрана добавлен фильтр MAC-адресов и налажена работа с группами фильтрации; Устранены проблемы в QoS-модуле; Добавлена поддержка IPTV over ADSL; Добавлена поддержка внесения индивидуальных настроек для DHCPd и dnsmasq; Проведена чистка кода в инсталляторе: удален модуль reiser4progs и сокращен размер initrd, благодаря исключения поддержки неиспользуемых файловых систем.
