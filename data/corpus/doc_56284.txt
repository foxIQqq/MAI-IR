Выпуск дистрибутива EndeavourOS 21.4

Опубликован выпуск проекта EndeavourOS 21.4 "Atlantis", пришедшего на смену дистрибутиву Antergos, развитие которого было прекращено в мае 2019 года из-за нехватки у оставшихся мэйнтейнеров свободного времени для поддержания проекта на должном уровне. Размер установочного образа 1.9 ГБ (x86_64, отдельно развивается сборка для ARM). Endeavour OS даёт возможность пользователю без лишних усложнений установить Arch Linux с необходимым рабочим столом в том виде, каким он задуман в своей штатной начинке, предлагаемой разработчиками выбранного рабочего стола, без дополнительных предустановленных программ. Дистрибутив предлагает простой инсталлятор для установки базового окружения Arch Linux с рабочим столом Xfce по умолчанию и возможностью установить из репозитория один из типовых рабочих столов на основе Mate, LXQt, Cinnamon, KDE Plasma, GNOME, Budgie, а также мозаичных оконных менеджеров i3, BSPWM и Sway. Ведётся работа по добавлению поддержи оконных менеджеров Qtile и Openbox, рабочих столов UKUI, LXDE и Deepin. Также одним из разработчиков проекта развивается собственный оконный менеджер Worm. В новом выпуске: Инсталлятор Calamares обновлён до версии 3.2.47. Налажена возможность отправки логов в случае сбоя установки. Обеспечен показ более детальной информации об устанавливаемых в текущим момент пакетах. Возвращена возможность одновременной установки Xfce и i3. В по умолчанию устанавливаемом проприетарном драйвере NVIDIA включён модуль nvidia-drm, использующий подсистему ядра DRM KMS (Direct Rendering Manager Kernel Modesetting). В файловой системе Btrfs задействовано сжатие zstd. Обновлены версии программ, в том числе ядро Linux 5.15.5, Firefox 94.0.2, Mesa 21.2.5, nvidia-dkms 495.44. Добавлены дополнительные проверки для исключения проблем с загрузкой после обновления драйверов NVIDIA и ядра Linux. На экране приветствия входа в систему добавлена новая кнопка с информацией об установленном окружении рабочего стола. По умолчанию добавлен пакет eos-apps-info и расширен спектр программ, о которых имеется информация в eos-apps-info-helper. В paccache-service-manager добавлена опция для удаления кэша удалённых пакетов. В eos-update-notifier улучшен интерфейс настройки расписания проверки обновлений. Возвращена установка OS prober для улучшения работы при наличии нескольких установленных операционных систем. В ISO-образе предоставлена возможность определения через файл user_commands.bash собственных bash-команд для выполнения после установки. В ISO-образе реализована функция "hotfix", позволяющая распространять исправления пробоем без обновления iso-образа (приложение Welcome проверяет наличие hotfix-ов и загружает их перед запуском инсталлятора). С оконным менеджером Sway задействован дисплейный менеджер ly DM. По умолчанию включён мультимедийный сервер Pipewire.
