Релиз JPype 0.7, библиотеки для доступа к Java-классам из Python

Спустя более четырёх лет с момента формирования прошлой значительной ветки доступен релиз прослойки JPype 0.7, позволяющей организовать полный доступ Python-приложений к библиотекам классов на языке Java. При помощи JPype из Python можно использовать специфичные для Java библиотеки, создавая гибридные приложения, сочетающие код на Java и Python. Код проекта распространяется под лицензией Apache 2.0. В отличие от Jython, интеграция с Java достигается не через создание варианта Python для JVM, а через взаимодействие на уровне обеих виртуальных машин, используя разделяемую память. Предложенный подход позволяет не только добиться хорошей производительности, но и предоставляет доступ ко всем библиотекам CPython и Java. В новом выпуске полностью переписан код основного модуля, добавлена поддержка unattached-потоков, улучшена защищённость, обеспечена трансляция Java-исключений в Python-исключения, изменено поведение при преобразовании строк.
