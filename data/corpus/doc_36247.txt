Тестирование плагина ОРФО для Linux версии LibreOffice и OpenOffice

Началось открытое бета-тестирование Linux-версии плагина ОРФО-Lite для офисных пакетов LibreOffice и Apache OpenOffice. Плагин ОРФО (пакет, инструкция по установке) содержит в своем составе модуль проверки орфографии русского языка, разработанный в сотрудничестве с Институтом русского языка имени В.В. Виноградова в соответствии со всеми действующими рекомендациями орфографической комиссии РАН. В плагине реализован интеллектуальный механизм многоместной подсказки для слов с ошибками. После завершения разработки плагин проверки орфографии для LibreOffice останется бесплатным для личного некоммерческого использования, для коммерческих предприятий он будет поставляться на платной основе. Русскоязычная сборка LibreOffice комплектуется средствами проверки орфографии русского языка, но словари, по которым происходит проверка, включают относительное небольшое число слов, содержат большое количество ошибок и ограничений, например, не проверяется сочетаемость и согласованность частей сложных слов, написанных через дефис. Тестовая версия плагина ОРФО для Linux пока тоже не лишена ограничений: в отличие от ОРФО 2012 не поддерживается проверка и расстановка буквы Ё, словари проверки не обновляются, нет проверки грамматики. Данные возможности будут реализованы в полной версии плагина. Примечательно, что представленный плагин – первая с момента основания компании в 1989 году версия ОРФО для операционной системы Linux.
