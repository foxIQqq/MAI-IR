Адаптация ALT Linux для людей, имеющих проблемы со зрением

В рамках проекта Homeros начата работа по адаптации дистрибутива ALT Linux для людей с ограничениями по зрению. В итоге планируется подготовить набор пакетов и специализированную сборку ALT Linux, снабженную средствами речевого управления и озвученной программой установки. Для организации ввода речевых команд и взаимодействия с синтезаторами речи в рамках проекта ведется разработка специального речевого сервера Voiceman, первые версии которого уже доступны для загрузки. Исходные тексты Voiceman распространяются в рамках лицензии GPLv3. В качестве экранного чтеца использован проект Orca.
