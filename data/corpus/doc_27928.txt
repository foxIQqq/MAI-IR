Обновление свободного Java-пакета IcedTea6 1.9

Анонсирован выход релиза проекта IcedTea6 1.9, полностью открытой реализации Java SE 6, построенной на базе OpenJDK6 20 и виртуальной машины HotSpot 17, с использованием свободных средств сборки, интеграцией компилятора gcj (GNU Compiler for Java), реализацией некоторых дополнительных возможностей (64-разрядный плагин для LiveConnect и Java Web Start) и расширенной поддержкой платформ. По сравнению с прошлым выпуском в IcedTea6 1.9 внесено более 1100 исправлений. Особенностью релиза является исключение из поставки инструмента VisualVM и оформления его в виде отдельного пакета, который поддерживает интеграцию с NetBeans 6.8 и NetBeans 6.9. Причина выделения VisualVM из основного состава IcedTea6 в упрощении формирования пакетов и снижении зависимости от проекта NetBeans.
