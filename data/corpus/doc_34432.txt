Доступна для тестирования третья альфа-версия Ubuntu 12.10

Представлена третья альфа-версия Ubuntu 12.10 "Quantal Quetzal". Загрузочные iso-образы с тестовой версией Ubuntu 12.10 можно получить на данной странице. Напомним, что CD и DVD образы десктоп-редакции Ubuntu 12.10 объединены в единый образ размером 764 Мб (к релизу будет 800 Мб), пригодный для установки на USB Flash и DVD. Релиз запланирован на 18 октября, после выхода двух бета версий. По сравнению со вторым альфа-выпуском, в Ubuntu 12.10-alpha3 отмечены следующие улучшения: В состав включён финальный релиз ядра Linux 3.5. Из специфичных для Ubuntu 12.10-alpha3 изменений отмечается возврат к использованию по умолчанию планировщика ввода/вывода Deadline, как для серверных систем, так и для десктопов. К более однородному виду приведены файлы конфигурации ядра для разных типов задач. Удалены устаревшие файлы с прошивками; Менеджер обновлений Update Manager модернизирован и переименован в Software Updater. Добавлена поддержка проверки наличия обновлений при запуске. Обновлён графический стек X.org, включая xserver 1.12, mesa 8.0.3, обновлённые драйверы и библиотеки. Обновление X-сервера позволило задействовать такие возможности, как режим плавной прокрутки для тачпадов, API для сенсорных экранов и протокол для обработки событий мультитач. Пользователям предоставляется возможность установки новой версии проприетарного драйвера NVIDIA, в котором обеспечена поддержка стандарта Randr для конфигурации подключённых дисплеев. До шестой версии обновлена пользовательская оболочка Unity, в которой исправлена порция ошибок и проведена подготовка к обеспечению поддержки новых возможностей, добавление которых последует в следующих выпусках. Libreoffice обновлён до версии 3.6.0rc2, компоненты GNOME обновлены до версии 3.5.4; Продолжена работа по сокращению числа установочных образов для серверной редакции, в которой будет прекращено формирование CD-образов для архитектуры i386, а в качестве первичной платформы будет X86_64 (AMD64). Для установки на i386 будет продолжена поддержка mini-образа и образа для установки по сети. Установочный образ серверной версии по прежнему будет вмещаться на CD (703MB); Изменения в серверной редакции: Реализована политика доступа к облачным хранилищам Juju Charm Store и возможность назначения мейнтейнера для Juju Charm. Осуществлена интеграция Juju с Facter; Tomcat7 заменил собой Tomcat6 в репозитории Main и в серверном iso; Добавлена поддержка MAAS IPMI; Инструментарий для файловой системы Ceph обновлён до версии 0.48; Подготовлены тестовые пакеты с платформой OpenStack Folsom. Добавлена поддержка автоопределения архитектуры окружений на базе Openstack, добавлен гетерогенный планировщик для ARM; Проведена работа по увеличению надёжности apt (избавление от эффекта гонки при проверке хэшей, приводящего к ошибке 'Hash Sum Mismatch'); Для архитектуры armhf сформированы образы для облачных систем. В Kubuntu обновлена версия облачного хранилища OwnCloud, добавлены новые версии менеджера фотографий Digikam 2.7 и браузера Rekonq 1.0, улучшена интеграция Kontact с сервисами Google и Facebook; В Lubuntu добавлены новые версии менеджера сессий, файлового менеджера pcmanfm, представлено новое визуальное оформление, notification-daemon заменён на xfce4-notifyd, в состав добавлена поисковая утилита catfish.
