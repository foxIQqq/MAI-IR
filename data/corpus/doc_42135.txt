В Екатеринбурге состоится встреча разработчиков, использующих язык Java

14 мая в Екатеринбурге состоится первая встреча cообщества JUG.EKB (Java User Group), на которой состоится обсуждение различных технологий из мира Java и обмен опытом. Мероприятие пройдёт в коворгинге «Соль» (переулок Химиков, 3, 4 этаж ), начало в 19:30. Встреча бесплатная, но необходимо зарегистрироваться. На встрече ожидается гость из Санкт-Петербурга – Андрей Паньгин. Андрей занимается платформой портала ok.ru с 2011 года. До этого работал в компании Oracle над виртуальной Java-машиной HotSpot. Отлично разбирается во внутренностях JVM и увлекается низкоуровневым системным программированием. Автор проекта one-nio на github — фреймворка для разработки высокопроизводительных серверов. Программа встречи: Андрей Паньгин (компания Одноклассники) c докладом «Разработка высоконагруженного сервера на Java» Денис Кирпиченков (компания Naumen) c докладом «10 человеко-лет GWT» Afterparty
