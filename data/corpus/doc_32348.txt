Релиз Wine 1.3.33

Вышла новая версия реализации Win32 API - Wine 1.3.33. С момента выпуска версии 1.3.32 было закрыто 84 отчёта об ошибках. Основные изменения в новой версии: В DIB движке добавлена поддержка вывода текста; Улучшена поддержка HTTP прокси; Обновлен встроенный HTML движок на основе Mozilla Gecko; Исправлены некоторые ошибки в работе курсора мыши в играх; Добавлено множество недостающих вызовов библиотеки MSVCRT; Обновлены переводы; Исправлено множество ошибок.
