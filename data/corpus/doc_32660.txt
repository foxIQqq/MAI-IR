Обновление Linux-дистрибутивов Pear OS 3.0, Bodhi 1.3.0 и Semplice 2.0.0

Доступны новые версии трёх Linux-дистрибутивов, основанных на пакетной базе Debian и Ubuntu: Вышел первый стабильный релиз дистрибутива Pear OS 3.0, построенного на базе Ubuntu и графического окружения GNOME 3.2, оформление которого стилизовано под Mac OS X. В системе использовано ядро Linux 3.0.0.14, в качестве браузера предлагается Opera, для управления фотографиями задействован Shotwell, для просмотра видео - Totem, для прослушивания музыки и управления музыкальной коллекцией - Clementine. В поставку входит похожий на Mac OS X TimeMachine менеджер резервных копий - Back in Time, чистильщик дисков - Bleachbit, а так же каталог-приложений Pear Appstore, система запуска программ Launchpad и пакетный менеджер Y-ppa-manager. Для загрузки доступны образы для платформ x86 и x86_64 (размер 1 Гб), дополнительно подготовлена сборка для VirtualBox. Одновременно сообщается о подготовке выпуска Pear OS Linux Debian Edition, построенного на базе Debian Squeeze 6.0.3 и основанный на графическом окружении GNOME 2.30 и ядре Linux 2.6.32. Представлен релиз легковесного дистрибутива Bodhi Linux 1.3.0, построенного на пакетной базе Ubuntu 10.04. Графическое окружение в Bodhi Linux основано на оконном менеджере Enlightenment 17, в качестве файлового менеджера используется PCManFM. Для упрощения установки дополнительных приложений наиболее интересные программы из различных категорий представлены в специальном web-каталоге. Размер загрузочного iso-образа 380 Мб. В качестве минимальных требований к оборудованию упоминается CPU 300 Mhz и 128 Мб ОЗУ. В новом выпуске использована свежая сборка Enlightenment из SVN-репозитория, браузер Midori 0.4.2, в состав интегрирован модуль "itask", который выполняет функции панели задач. Также можно отметить прогресс в создании варианта дистрибутива для архитектуры ARM, уже готовится вариант Bodhi с оболочкой Enlightenment для планшета HP TouchPad и коммуникатора Nokia N900; Увидел свет новый легковесный Live-дистрибутив Semplice Linux 2.0.0, построенный на пакетной базе Debian "unstable" и оконном менеджере Openbox 3.5.0. Размер iso-образа 560 Мб. Из входящих в состав приложений можно отметить: ядро Linux3.1.6, браузер Chromium 15, музыкальный плеер Exaile 0.3.2.2, видеоплеер GNOME MPlayer 1.0.4, текстовый редактор AbiWord 2.9.1, табличный процессор Gnumeric 1.10.17, клиент для мгновенного обмена сообщениями Pidgin 2.10.1, эмулятор терминала Guake 0.4.2.
