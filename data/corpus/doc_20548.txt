Вышла седьмая альфа версия Samba 4.0.0

"Samba 4.0.0alpha7" - вышла седьмая альфа версия Samba 4.0.0. Некоторые новшества, внесенные с момента выпуска прошлой альфа версии: Поддержка генерации для OpenLDAP бэкенда конфигурации для мульти-мастер репликации (MMR - Multi Master Replication); Поддержка генерации olc-конфигурации (OpenLDAP-Online-Configuration) для OpenLDAP бэкенда; Поддержка работы Windows 7-beta в домене под управлением Samba4; Обновлен ряд внутренних библиотек, с целью упрощения использования приложениями не из комплекта samba4; Возможность использования для обеспечения работы TLS/SSL последних версий GnuTLS. Подробный обзор ранее внесенных изменений доступен на данной странице.
