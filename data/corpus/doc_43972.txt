Новые проекты Mozilla для умного дома и сетевых потребительских устройств

После прекращения разработки Firefox OS для смартфонов внимание организации Mozilla сосредоточилось на развитии технологий для сетевых потребительских устройств (Connected Devices). Начиная эксперименты в этой области разработчики Mozilla представили четыре новых проекта, к разработке и тестированию которых планируется привлечь энтузиастов из сообщества. Project Link (Foxlink) - персональный пользовательский агент (User Agent) для умного дома, который распознаёт предпочтения пользователя по взаимодействию с находящимися в доме умными устройствами и автоматизирует работу с ними, предоставляя пользователю удобный и безопасный интерфейс, но сохраняя возможность полного контроля за системой. Код проекта разрабатывается преимущественно на языке Rust. Центральным звеном проекта является серверный процесс FoxBox, координирующий взаимодействие с устройствами в доме. В состав также входит сервер регистрации и обнаружения новых устройств, движок для выполнения сценариев автоматизации и web-интерфейс для управления умным домом. Project Smart Home - решение, которое пытается занять промежуточную нишу между готовыми специализированными системами домашней автоматизации, такими как платформа Apple Homekit, и самодельными разработками, основанными на таких компонентах, как платы Raspberry Pi. Smart Home развивается на базе фреймворка Eclipse Smart Home и предоставляет простое решение для создания умного дома для решения типовых ежедневных проблем пользователей, сочетая применение недорогого модульного оборудования и простых методов определения правил автоматизации. Project Vaani - пакет c реализацией интерфейса голосового управления IoT-устройствами (Internet of Things). Vaani даёт возможность разработчикам и производителям оборудования интегрировать в свои продукты реализацию похожего на Siri или Google Now виртуального голосового помощника, позволяющего организовать управление через диалог на естественном языке, но без привязки к внешним сервисам. В ближайшее время на базе Vaani планируется подготовить рабочий прототип системы голосового управления умным домом, а также продемонстрировать возможность доступа к сервисам из Web. Vaani использует наработки проекта OpenHAB (Home Automation Bus). Project Sensor Web - платформа для построения сети сбора данных с различных датчиков. В настоящий момент развивается прототип сети PM 2.5, охватывающей множество датчиков загрязнения окружающей среды по всему миру. Информация доступна любому желающему, который может использовать поступающие данные, например, для мониторинга загрязнения воздуха на своей улице.
