Выпуск qBittorrent 4.4 с поддержкой протокола BitTorrent v2

Спустя больше года с момента публикации прошлой значительной ветки представлен релиз торрент-клиента qBittorrent 4.4.0, написанного с использованием тулкита Qt и развиваемого как открытая альтернатива µTorrent, приближенная к нему по интерфейсу и функциональности. Среди возможностей qBittorrent: интегрированный поисковый движок, возможность подписки на RSS, поддержка многих BEP-расширений, удалённое управление через web-интерфейс, режим последовательной загрузки в заданном порядке, расширенные настройки для торрентов, пиров и трекеров, планировщик пропускной способности и IP-фильтр, интерфейс для создания торрентов, поддержка UPnP и NAT-PMP. В новой версии: Добавлена поддержка протокола BitTorrent v2, в котором хэш вычисляется для каждого файла, а не для всей раздачи, и осуществлён уход от использования алгоритма SHA-1, имеющего проблемы с подбором коллизий, в пользу SHA2-256. Для работы с новой версией torrеnt-ов задействована библиотека libtorrent 2.0.x. Добавлена поддержка фреймворка Qt6. Добавлены новые настройки, такие как ограничение пропускной способности соединений, таймаут для уведомлений и опции hashing_threads для libtorrent. Обеспечена отправка анонсов для всех трекеров при смене IP-адреса. Для разных колонок в интерфейсе добавлены всплывающие подсказки. Добавлено контекстное меню для переключения столбцов вкладок. В боковую панель добавлен фильтр состояния "Checking". В настройках обеспечено запоминание последней просматриваемой страницы. Для отслеживаемых каталогов предоставлена возможность пропуска проверки хэшей (опция "Skip hash check"). При двойном клике реализован просмотр опций торрента. Добавлена возможность привязки разных каталогов с временными файлами для отдельных торрентов и категорий. Добавлена поддержка тем оформления, разнесённых по разным каталогам. В виджете поиска реализовано контекстное меню и увеличено число режимов загрузки. В web-интерфейсе предоставлена возможность навигации по таблицам и каталогам при помощи клавиш управления курсором. В основной вкладке реализован индикатор прогресса выполнения операций. Для Linux обеспечена установка векторных пиктограмм. В сборочном скрипте реализовано определения ОС OpenBSD и Haiku. Добавлена экспериментальная настройка для хранения файлов fastresume и torrent в СУБД SQLite.
