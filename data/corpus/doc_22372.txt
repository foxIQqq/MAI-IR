Выпущена новая версия Linux дистрибутива BSL для небольших предприятий

Выпущена новая версия Linux дистрибутива BSL, ориентированного для использования на небольших предприятиях. Изменения: добавлен DNS-сервер с поддержкой динамических обновлений с TSIG-подписями в DHCP-сервер добавлена поддержка динамических обновлений DNS с использованием TSIG-подписей добавлен журнальный сервер с возможностью поиска системный планировщик: планировщик отслеживает задания в каталогах cron.d, cron.daily, cron.hourly, cron.monthly, cron.weekly, находящиеся в каталогах /etc/ и /usr/local/etc/ только пользователям группы wheel разрешено создавать собственные расписания заданий сетевые настройки: шлюз по умолчанию можно настроить через web-интерфейс сетевые адаптеры отображаются в web-интерфейсе в порядке их расположения на шине PCI при первом после установки старте системы адрес 192.168.1.100 присваивается первому по порядку на шине PCI сетевому устройству, а не /dev/eth0 исправлена ошибка, из-за которой для всех сетевых адаптеров отображались одни и те же значения полей производитель/модель добавлена контактная информация пользователя: домашний адрес, телефоны домашний, мобильный и рабочий для файлового сервера SMB/CIFS добавлен ряд настроек, позволяющих получить более привычное для Windows-пользователей поведение в части разрешений для файлов/каталогов текстовой web-обозреватель elinks заменён на links2 установка системы на программный RAID1 ("зеркало"); позже будет опубликована инструкция по переносу уже установленной системы на RAID1 установка системы на SCSI LSI logic контроллере в эмуляторе VirtualBox
