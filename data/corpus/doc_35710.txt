Увидел свет дистрибутив Calculate Linux 13

Доступен релиз отечественного дистрибутива Calculate Linux 13.0, построенного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в 7 редакциях: Calculate Directory Server (CDS), Calculate Linux Desktop с рабочим столом KDE (CLD), GNOME (CLDG) и XFCE (CLDX), Calculate Media Center (CMC), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS). Особенности Calculate Linux: 100% совместимость с Gentoo с поддержкой бинарных репозиториев обновлений. Поставка специально разработанных в рамках проекта Calculate утилит для настройки, сборки и установки системы. Готовое клиент-серверное решение. Быстрое развёртывание на предприятии. Полноценная работа в гетерогенных сетях. Удобство администрирования. Rolling-release модель обновлений. Поддерживается интерактивная сборка системы - подготовка ISO-образа системы под свои задачи. Состав версий: CDS (Directory Server, i686 - 637 M, x86_64 - 661 M): Bind 9.9.2, ядро Linux 3.6.7, OpenLDAP 2.4.30, Postfix 2.9.4, ProFTPD 1.3.4a, Samba 3.6.9 и компоненты для организации работы LDAP-сервера для централизованной авторизации пользователей с хранением профилей на сервере, в том числе web-интерфейс для удалённого управления сервером и инструменты для организации обновления программного обеспечения сетевых клиентов; CLD (KDE desktop, i686 - 1.97 G, x86_64 - 2.18 G): KDE 4.9.4, Amarok 2.6.0, digiKam 2.9.0, Chromium 23.0.1271.97, ядро Linux 3.6.7, LibreOffice 3.6.4.3, набор компонентов для обеспечения хранения настроек рабочего стола пользователя на сервере и централизованной настройки приложений на клиентских машинах; CLDG (GNOME desktop, i686 - 1.51 G, x86_64 - 1.72 G): GNOME 3.4, GIMP 2.8.2, Chromium 23.0.1271.97, ядро Linux 3.6.7, LibreOffice 3.6.4.3; CLDX (XFCE desktop, i686 - 1.40 G, x86_64 - 1.60 G): Xfce 4.10, Gimp 2.8.2, Chromium 23.0.1271.97, ядро Linux 3.6.7, LibreOffice 3.6.4.3; CMC (Media Center, i686 - 933 M, x86_64 - 1.09 G): XBMC 11.0, ядро Linux 3.6.7, Xorg-server 1.13.0; CLS (Linux Scratch, i686 - 648 M, x86_64 - 669 M): Openbox 3.5.0, ядро Linux 3.6.7, Xorg-server 1.13.0, минимальный набор драйверов и утилит для сборки пользовательских Live-систем под свои задачи; CSS (Scratch Server, i686 - 370 M, x86_64 - 386 M): ядро Linux 3.6.7 и минимальный набор компонентов для сборки серверных Live-систем под свои задачи. Основные изменения в Calculate Linux 13 Система Мета-пакеты более не используются для построения зависимостей состава пакетов дистрибутива. Добавлена распаковка портежей при первой загрузке установленной системы. Улучшено определение NTFS-разделов. Исправлена настройка пользовательского профиля на VMware. Для VMware по умолчанию включён composite. Исправлено создание ярлыков запуска основных программ в случае их замены. Добавлено восстановление из Git локальной копии портежей и оверлея в случае внесения изменений пользователем. Утилиты Calculate Утилиты Calculate обновлены до версии 3.1. В Консоль Calculate добавлены новые настройки: ввод компьютера в домен Calculate Directory Server; настройка сеанса пользователя; изменение пароля пользователя; просмотр и редактирование значений переменных утилит; обновление world-файла. Добавлены новые события настройки системы: обновление портежей; удаление пакета. Добавлен новый пакет утилит calculate-update, в задачу которого пока входит только работа с world-файлом. Расширены возможности шаблонов настройки. Десктоп Calculate Linux Desktop KDE: KDE обновлен до версии 4.9.4. Calculate Linux Desktop GNOME: GNOME обновлен до версии 3.4; меню заменено на AxeMenu; используется буфер обмена Parcellite. В Calculate Linux Desktop XFCE: по умолчанию задействована модифицированная тема Adwaita; значки заменены на Elementary; добавлена среда разработки Geany; добавлен менеджер фотографий Shotwell; в файловый менеджер интегрирован поисковик файлов Catfish; для воспроизведения видеофайлов теперь используется SMplayer; в почтовый клиент Claws-Mail включён плагин Address Keeper для автоподстановки адреса получателя. Каждый дистрибутив использует свою тему оформления. Из состава десктопов удалены словари. Сервер В Calculate Directory Server: добавлена возможность подключения к Jabber используя StartTLS; добавлена поддержка настройки SMTP-аутентификации.
