Plasmate - новая интегрированная среда для создания плазмоидов для KDE4

Аарон Сеиго (Aaron Seigo), основной разработчик десктоп-оболочки Plasma из состава KDE4, представил первый экспериментальный выпуск 0.1-alpha1 минималистичной IDE-оболочки Plasmate, предназначенной для упрощения разработки дополнений к Plasma с использованием динамических языков программирования Javascript, Ruby и Python. В качестве основы для представленной мини-IDE использован текстовый редактор Kate. В Plasmate плотно интегрирована поддержка Git, имеется интерактивная система просмотра документации. Среда позволяет создавать плазмоиды, движки для обработки данных (DataEngines), обработчики событий (Runner) и визуальные темы.
