Выпуск почтового клиента Mutt 2.0

Доступен релиз консольного почтового клиента Mutt 2.0. Переход на новый значительный номер версии обусловлен внесением изменений, нарушающих обратную совместимость. Например, изменено поведение при прикреплении файлов во вложение, по умолчанию выставлен режим $ssl_force_tls, отключена чистка заголовков при выполнении операций decode-copy и decode-save, установка параметра $hostname теперь производится после обработки muttrc и опций "-e" в командной строке. Среди новых возможностей: Возможность указания IPv6-адреса вместо имени хоста в email, например "user@[IPv6:fcXX:....]". Добавлена команда "cd" для смены рабочего каталога. Добавлена поддержка XOAUTH2 (аутентификация в IMAP, POP и SMTP при помощи OAuth), активируемая через установку параметра "xoauth2" в настройках $imap_authenticators, $smtp_authenticators и $pop_authenticators. Обеспечена автоматическая установка повторного соединения к IMAP в случае сбоя установленного соединения (решило проблему с потерей не записанных из-за сбоя изменений). При вводе модификатора шаблона после символа "~" теперь можно нажать табуляцию для просмотра списка имеющихся модификаторов. Добавлен MuttLisp, позволяющий использовать Lisp-подобные выражения в файле конфигурации. Например: run (if (equal $USER "kevin8t8") \ "set arrow_cursor") Добавлена переменная $cursor_overlay, которую можно использовать для сохранения цветовых индикаторов для строк, на которые указывает курсор. Например, при выставлении следующих настроек подчёркнутый курсор будет подсвечен красным на строках с новыми сообщениями. color indicator underline default default color index red default ~N set cursor_overlay Добавлена переменная $attach_save_dir, позволяющая указать каталог для сохранения вложений.
