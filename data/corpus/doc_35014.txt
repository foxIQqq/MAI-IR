Релиз Jabber/XMPP клиента Psi 0.15

После двух лет разработки вышла новая версия открытого многоплатформенного Jabber/XMPP клиента Psi 0.15, написанного на C++ с использованием библиотеки Qt. По сравнению с предыдущей версией в новом релизе представлены следующие улучшения: Принято множество патчей из проекта Psi+. Новый виджет для просмотра истории сообщений. Улучшено окно со списком контактов (ростер). Поддержка TURN-прокси для голосовых вызовов. Более стандартное размещение данных в файловой системе. (Например, для GNU/Linux систем используются стандарты freedesktop.org) Переписан старый код, библиотека Qt3Support больше не используется. (Это тоже вклад от команды Psi+) Добавлена поддержка 64-битных MS Windows и Mac OS X и прекращена поддержка сборок для архитектуры PPC под Mac OS X. Множество мелких улучшений и исправлений. Внимание! Psi 0.15 использует новый формат профиля. При обновлении с версии 0.14 и более ранних будет произведена конвертация настроек при первом запуске Psi.
