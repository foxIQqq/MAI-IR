Первая альфа-версия Ubuntu 14.04

Объявлено о начале альфа-тестирования дистрибутива Ubuntu 14.04 "Trusty Tahr", который отнесён к категории выпусков с длительным сроком поддержки (LTS), обновления для которых формируются в течение 5 лет. В соответствии с планом разработки, уже опробованном при подготовке Ubuntu 13.04 и 13.10, отдельные альфа-сборки не будут сформированы, вместо них для тестирования предлагается использовать ежедневные экспериментальные сборки. Готовые тестовые образы созданы только для Edubuntu, Ubuntu GNOME, Xubuntu и Kubuntu. Бета-версия Ubuntu 14.04, запланированная на 27 марта, будет сформирована в виде полноценной сборки. Релиз намечен на 17 апреля. Основные изменения: Синхронизация новых версий пакетов из репозиториев Debian Unstable. Включение в поставку ядра Linux 3.12; Kubuntu: Обновление компонентов десктоп-окружения до выпуска KDE 4.12. Поставка в качестве браузера по умолчанию Firefox 26. Поддержка установки плагинов для просмотрщика изображений Gwenview из штатного интерфейса. Интеграция системы для автоматической генерации отчётов о сбоях; Ubuntu GNOME: Обновление некоторых компонентов графического окружения до версий из состава GNOME 3.10. Реализация поддержки работы в режиме классического рабочего стола, в менеджер входа добавлен отдельный сеанс GNOME Classic; Xubuntu: Включена новая версия конфигуратора xfce4-settings; Edubuntu: добавлена поддержка бездисковой загрузки. Интегрирован сервер директорий для организации единой точки входа пользователей. Основной задачей при подготовке нового выпуска является обеспечение стабильности и производительности рабочего стола на основе Unity 7. Усилия по расширению функциональности будут направлены на развитие редакции для планшетов и смартфонов Ubuntu Touch. Ubuntu 14.04 LTS будет поставляться по умолчанию с классическим графическим стеком на основе X.Org, Compiz и Unity 7, без прямого задействования Mir или использования прослойки XMir. Ubuntu Touch будет по умолчанию поставляться с графическим стеком на базе Mir. Из общих планов развития Ubuntu 14.04 можно отметить: формирование сборки для процессоров на базе архитектуры AArch64 (ARM64), включение по умолчанию операции TRIM для SSD-накопителей, улучшение поддержки UEFI Secure Boot (поддержка записи сигнатуры ядра в NVRAM без входа пользователя в интерфейс прошивки и включение в поставку запасного загрузчика fallback.efi для ситуаций переноса диска на другую систему или при повреждении прошивки), поддержка cgroups в системном менеджере upstart, переход с DMRAID на MDADM для обеспечения работы программных RAID. Из изменений в Ubuntu Touch отмечается работа по обеспечению возможности использования приложений, написанных для Ubuntu Touch, в обычном десктоп-оркужении Ubuntu, переработка интерфейса запуска приложений, расширение возможностей предлагаемых по умолчанию приложений.
