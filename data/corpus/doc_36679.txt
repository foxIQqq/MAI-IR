Вышел мультимедиа проигрыватель Qmmp 0.7

Доступна очередная версия плеера Qmmp, написанного с использованием библиотеки Qt. Из наиболее значительных изменений можно отметить следующие: Поддержка Ogg Opus; Возможность запускать внешние скрипты при смене трека; Формирование списков воспроизведения с использованием копирования/вставки; Автосохранение списков воспроизведения; Эквалайзер поддерживает работу в 24/32-х битных режимах; Значительное улучшение поддержки MS Windows, включающее регулировку громкости с помощью winapi, глобальные горячие клавиши, поддержку файлов звукогенератов; Переработана авторизация на last.fm Использование UTF-8 по умолчанию для cue-файлов и shoutcast-метаданных; В модуле ffmpeg по умолчанию включено большее число форматов; Поддержка устаревших версий ffmpeg 0.6-0.8 перемещена в отдельный модуль ffmpeg_legacy и будет удалена в следующей мажорной версии; Анализатор спектра теперь имеет изменяемый размер и полноэкранный режим (клавиша "F"); Добавлена поддержка udisks2, модуль с поддержкой udisks1 больше не собирается по умолчанию и будет удалён в следующей мажорной версии; Обновлены переводы, включая русский и украинский; В рамках возможного перехода на Qt5 проведена работа по удалению зависимости от модуля QtXML, который объявлен устаревшим. Кроме того, вышел Qmmp Plugin Pack 0.7.0 — набор дополнительных модулей для Qmmp. В новой версии осуществлён полный переход на API qmmp 0.7.x, добавлена поддержка потокового вещания в модуль mpg123, а также проведено значительное улучшение "стандартного" интерфейса Qmmp Simple Ui (qsui), заключающееся в следующем: Добавлена панель с анализатором спектра и отображением обложек; Сортировка по времени создания файла; Настройки цветов списка воспроизведения; Добавлены предустановки для 16-полосного эквалайзера; Улучшено управление вкладками (возможности перемещения и закрытия с помощью средней кнопки мыши). Следует также отметить, что благодаря усилиям сообщества Qmmp Plugin Pack перестал быть экспериментальным и появился в большинстве ведущих дистрибутивов, таких как Alt, Agilia, Arch, Gentoo и др. Для пользователей MS Windows интерфейс Simple Ui поставляется вместе с основным установочным архивом. Кроме того, в связи с нежеланием разработчиков Ubuntu предоставлять актуальную версию, был создан сторонний PPA.
