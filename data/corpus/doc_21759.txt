Релиз открытого ETL пакета Pentaho Data Integration 3.2.0

Вышел релиз Pentaho Data Integration 3.2.0 (PDI, также называемая Kettle) - компонент комплекса Pentaho отвечающий за процесс Извлечения, Преобразования и Загрузки (Extract, Transform and Load - ETL). Данная система чаще всего используется при работе с хранилищами данных, но так же её возможности позволяют осуществлять: Обмен данными между приложениями или базами данных Экспорт информации из баз данных в файлы различных типов Загрузка массивов данных в базы данных Обработка данных Интеграция с другими приложениями Новое в данной версии: Улучшена визуальная составляющая: реализована цветовое деление с мини-иконками для различных типов переходов, также стала более интуитивной система подсказок. Добавлены новые виды этапов обработки и заданий. Импортирован этап обработки по формуле использующий библиотеку libformula из комплекса Pentaho. Импортирован этап случайной выборки из массива. Улучшены этапы по выгрузке данных. Исправлено большинство выявленных ошибок в коде. Усовершенствованна работа всей системы в целом. Добавлено динамическое объединение в кластеры (только по XML без заголовка). По сравнению с версией 3.1.0 исходный код увеличился на 72 505 строк и в сумме составляет 529 277 строк в версии 3.2.0. Проект Openbravo POS ru ищет единомышленников для объединения усилий по локализации Pentaho Data Integration. Круг работающих с данной системой не очень велик, но её потенциал очень высок, к тому же стабильность работы системы повысилась в разы. Данное приложение рассчитано на не очень сведущего в программировании пользователя, но на данный момент нет сложившейся русской терминологии в области ETL. Для начала работы по локализации PDI требуется создание определённого круга единомышленников, который и будет создавать (согласовывать) эту терминологию.
