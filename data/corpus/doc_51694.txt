Выпуск дистрибутива Ubuntu 19.10

Доступен релиз дистрибутива Ubuntu 19.10 "Eoan Ermine". Готовые образы созданы для Ubuntu, Ubuntu Server, Lubuntu, Kubuntu, Ubuntu Mate, Ubuntu Budgie, Ubuntu Studio, Xubuntu и UbuntuKylin (редакция для Китая). Основные новшества: Рабочий стол GNOME обновлён до выпуска 3.34 c поддержкой группировки пиктограмм приложений в обзорном режиме, улучшенным конфигуратором беспроводных соединений, новой панелью выбора обоев рабочего стола и проведением работы по повышению отзывчивости интерфейса и снижению нагрузки на CPU. Вместо ранее предлагаемой темы оформления с тёмными заголовками по умолчанию задействована светлая тема оформления, близкая к штатному внешнему виду GNOME. В качестве опции предложена полностью тёмная тема, в которой использован тёмный фон внутри окон. Для переключения темы оформления можно использовать GNOME Tweaks; Добавлена возможность доступа к подключённым сменным USB-носителям непосредственно из панели. Для подключённых накопителей на панели теперь отображаются соответствующие значки, при помощи которых можно открыть содержимое в файловом менеджере или отмонтировать накопитель для безопасного извлечения устройства; Включена по умолчанию возможность организации доступа к мультимедийным данным при помощи протокола DLNA. Например, пользователь может открыть совместный доступ к коллекции видео для просмотра на SmartTV; В окружении на базе Wayland обеспечена возможность запуска X11-приложений с правами root под управлением Xwayland; Добавлена поддержка технологии защиты беспроводных сетей WPA3; Ядро Linux обновлено до выпуска 5.3. Для сжатия ядра Linux и начального загрузочного образа initramf задействован алгоритм LZ4, что позволит сократить время загрузки за счёт более быстрой распаковки данных. Предлагаемый в Ubuntu 19.10 пакет с ядром содержит неисправленную уязвимость в IPv6-стеке, позволяющую непривилегированному локальному атакующему инициировать крах ядра. Инструментарий обновлён до glibc 2.30, GCC 9.2, OpenJDK 11, rustc 1.37, Python 3.7.5, ruby 2.5.5, php 7.3.8, perl 5.28.1, go 1.12.10. Добавлены пакеты с MySQL 8.0; Офисный пакет LibreOffice обновлён до выпуска 6.3. Звуковой сервер PulseAudio обновлён до выпуска 13.0. Обновлены QEMU 4.0, libvirt 5.6, dpdk 18.11.2, Open vSwitch 2.12, cloud-init 19.2; Улучшена поддержка кросс-компиляции - инструментарий для архитектур POWER и AArch64 теперь поддерживает кросс-компиляцию для платформ ARM, S390X и RISCV64; Все пакеты пересобраны GCC с опцией "-fstack-clash-protection", при указании которой компилятор подставляет проверочные вызовы (probe) при каждом статическом или динамическом выделении места для стека, которые позволяют выявлять факты переполнения стека и блокировать методы атак на основе пересечения стека и кучи, связанные с пробросом потока выполнения через guard-страницы защиты стека. При сборке также включена техника защиты Control Flow Integrity (CFI), обеспечивающая выявление некоторых форм неопределённого поведения, которые потенциально могут привести к нарушению нормального потока управления (control flow) в результате выполнения эксплоитов; Для систем с GPU Intel обеспечен режим загрузки без мерцания при переключении видеорежимов; В состав установочных iso-образов по согласованию с компанией NVIDIA включены пакеты с проприетарными драйверами NVIDIA. Для систем с графическими чипами NVIDIA по умолчанию продолжают предлагаться свободные драйверы "Nouveau", а проприетарные драйверы доступны в качестве опции для быстрой установки после завершения инсталляции. В дистрибутиве также проведена работа по повышению стабильности запуска при использовании проприетарного драйвера NVIDIA и оптимизации производительности и качества отрисовки на системах с видеокартами NVIDIA; Прекращена поставка deb-пакетов с браузером Chromium, вместо которых теперь предлагаются только самодостаточные образы в формате snap; В репозитории прекращено распространение пакетов для 32-разрядной архитектуры x86. Для запуска 32-разрядных приложений в 64-разрядном окружении будет обеспечена сборка и поставка отдельного набора 32-разрядных пакетов, включающих компоненты, необходимые для продолжения работы устаревших программ, остающихся только в 32-разрядном виде или требующих 32-разрядных библиотек; В инсталлятор добавлена экспериментальная возможность установки на корневой раздел с ZFS. В установщик добавлена поддержка создания и разбивки разделов с ZFS. Для управления ZFS развивается новый демон zsys, позволяющий запускать несколько параллельных систем с ZFS на одном компьютере, автоматизирующий создание снапшотов и управляющий разнесением системных и изменяемых в процессе пользовательского сеанса данных. Основная идея в том, что в разных снапшотах можно содержать разные состояния системы и переключаться между ними. Например, в случае проблем после установки обновлений можно будет вернуться к старому стабильному состоянию, выбрав прошлый снапшот. Снапшоты также могут быть использованы для прозрачного автоматического создания резервных копий пользовательских данных. Добавлены сборки для плат Raspberry Pi 4, которые дополнили собой сборки для Raspberry Pi 2, Pi 3B, Pi 3B+, CM3 и CM3+; В Kubuntu предложен рабочий стол KDE Plasma 5.16, набор приложений KDE Applications 19.04.3 и фреймворк Qt 5.12.4. Обновлены версии latte-dock 0.9.2, Elisa 0.4.2, Kdenlive 19.08.1, Yakuake 19.08.1, Krita 4.2.7, Kdevelop 5.4.2, Ktorrent. Продолжается тестирование сеанса на базе Wayland (после установки пакета plasma-workspace-wayland появляется опциональный пункт "Plasma (Wayland)" на экране входа); В Xubuntu предложен новый выпуск рабочего стола Xfce 4.14. Вместо Light Locker для блокировки экрана задействован Xfce Screensaver, обеспечивающий интеграцию с Xfce Power Manager и улучшенную поддержку спящего и ждущего режимов; В Ubuntu Budgie добавлены новые апплеты Window Previews (замена менеджера задач (Alt+Tab)), QuickChar (просмотр таблиц символов), FuzzyClock, Workspace Stopwatch (секундомер) и Budgie Brightness Controller (управление яркостью экрана). Улучшена интеграция с GNOME 3.34. В Ubuntu MATE проведена работа по устранению недоработок и повышению качества интерфейса. Рабочий стол MATE обновлён до выпуска 1.22.2. Добавлен новый индикатор для вывода уведомлений, поддерживающий функцию "не беспокоить". Вместо Thunderbird по умолчанию задействован почтовый клиент Evolution, а вместо VLC - Celluloid (бывший GNOME MPV). Из базовой поставки удалены Qt4 и программа записи CD/DVD Brasero. В установочный образ включены проприетарные драйверы NVIDIA и комплект локализации для русского языка; В Ubuntu Studio добавлены пакет для организации потокового видеовещания OBS Studio и менеджер сеансов Raysession для управления программами обработки звука. В Ubuntu Studio Controls добавлено несколько прослоек для PulseAudio, реализован индикатор запуска Jack и добавлена возможность выбора бэкенда для Jack (Firewire, ALSA или Dummy). Обновлены версии компонентов: Blender 2.80, KDEnlive 19.08, Krita 4.2.6, GIMP 2.10.8, qJackCtl 0.5.0, Ardour 5.12.0, Scribus 1.4.8, Darktable 2.6.0, Pitivi 0.999, Inkscape 0.92.4, Carla 2.0.0, Ubuntu Studio Controls 1.11.3, В Lubuntu отмечено только исправление ошибок.
