Релиз Linux-дистрибутива moonOS 4

Вышел релиз Linux-дистрибутива moonOS 4, основанного на Ubuntu Linux 10.10, пользовательском окружении GNOME 2.32, панели Docky и ланчере Synapse (замена GNOME Do, написанная на Vala). Дистрибутив поставляется с рядом оригинальных приложений: системой конфигурирования moonControl, интерфейсом для настройки загрузчика - moonGrub, средством для просмотра доступных для установки приложений - moonSoftware. Размер установочного iso-образа 832 Мб. В новой версии проведена реструктуризация файловой системы (данные распределены по каталогам /AppFiles, /System, /Temporary, /Users, /Volumes, стандартные каталоги (/usr, /var и т.п.) присутствуют, но скрыты от пользователя) и представлен новый интерфейс для обеспечения работы с файлами - File Hierarchy System, напоминающий MacOS X. Также добавлен новый фреймворк для управления пакетами Appshell, позволяющий загрузив самодостаточный пакет, установить его в дальнейшем на любых системах. Из входящих в состав пакетов можно отметить: Linux-ядро 2.6.35, Firefox 3.6.13, OpenOffice.org 3.2.1, banshee 1.8.0, evince 2.32, evolution 2.30.3, brasero 2.32, fglrx (AMD Catalyst) 8.780, nvidia drv 73.14.28/96.43.18/260.19.06, Gimp 2.6.11, Mono 2.0, Totem 2.32, zeitgeist 0.6.
