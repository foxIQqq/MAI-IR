Python Software Foundation начинает финансирование неформальных встреч разработчиков

Организация Python Software Foundation объявила о проведении акции, направленной на оказание небольшой финансовой помощи разработчикам, выполняющим работу по доработке или исправлению ошибок в проектах, связанных с языком программирования Python. На проведение каждой неформальной встречи энтузиастов (Sprint, Hackfest, Hack-a-thon), направленной на решение задач, поставленных разработчиками Python, отныне будет выплачиваться символическое денежное довольствие, размером до 250 долларов. Python Software Foundation планирует проводить 2-3 подобные встречи каждый месяц, среди предложенных для подобных встреч задач отмечается: сортировка сообщений об ошибках, разбор патчей, улучшение документации, портирование библиотек и приложений на Python 3, доработка сайта и wiki проекта.
