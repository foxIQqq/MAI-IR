Вышел кластерный дистрибутив Rocks 6.1

Анонсирован релиз Rocks 6.1, основанного на пакетной базе CentOS специализированного дистрибутива для создания вычислительных кластеров, grid-сетей и инфраструктур для обеспечения работы виртуальных окружений на базе Xen и KVM. Дистрибутив может быть использован для быстрого развертывания кластеров в локальной сети с загрузкой основного управляющего узла с LiveCD/LiveUSB. После загрузки управляющего узла, дополнительные узлы могут быть загружены по сети через PXE и автоматически настроены. В комплект входит набор специально разработанных средств для централизованного управления кластером и мониторинга его работы. Размер универсального загрузочного iso-образа 2.5 Гб. Дополнительно подготовлено около десятка специализированных roll-образов, позволяющих упростить изменение конфигурации различных приложений (изменённые приложения ставятся в /opt) при развёртывании узлов на большом числе машин (например, созданы образы c пакетами для web-сервера, bio, condor, ganglia, perl, python, hpc, java, kvm, sge). Из особенностей нового выпуска можно отметить: Использование пакетной базы CentOS 6.3 с учётом всех обновлений, выпущенных до 27 ноября; Включение по умолчанию SSH-аутентификации на уровне хостов (HostbasedAuthentication) при которой к серверу может подключиться под своим идентификатором любой пользователь доверительного хоста; Поддержка двухуровневой SSH-аутентификации (кроме параметров входа используется дополнительный ключ верификации, ограниченный по сроку использования) с использованием приложения Google Authenticator для платформ Android и iPhone; Для генерации QR-кодов задействованы libqrencode.so и qrencode; Подготовлен roll-образ ZFS-linux с компонентами для поддержки файловой системы ZFS в Linux; Подготовлен roll-образ kernel-org с ванильным немодифицирвоанным ядром Linux; Добавлена поддержка Infiniband на оборудовании Mellanox.
