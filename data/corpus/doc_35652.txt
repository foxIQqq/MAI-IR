Игра Doom 3 BFG портирована для Linux

Спустя всего несколько недель с момента открытия исходных текстов игры Doom 3 BFG, изначально доступной только для Windows и Mac OS X, один из энтузиастов своими силами подготовил порт для платформы Linux. Сборка Linux порта успешно опробована в Debian, Ubuntu и Fedora. Для вывода графики в настоящий момент поддерживается только библиотека SDL. Поддержка звука пока отсутствует.
