Релиз программы для управления фотографиями digiKam 1.6.0

Вышел релиз программы для управления коллекцией фотографий digiKam 1.6.0 и набора digiKam-плагинов Kipi-plugins 1.6.0. В новой версии digiKam исправлено 31 ошибка и добавлено несколько новшеств: Обеспечена возможность сборки на платформе MacOS X, благодаря сотрудничеству с проектом MacPorts; При наличии в системе библиотек libPGF и Clapack они теперь используются в качестве внешних зависимостей, вместо встроенной в digiKam реализации определенных функций; В интерфейсе для управления альбомом добавлена возможность фильтрации фотографий по критерию отсутствия привязки к ним определенных тегов (исключение заданных тегов из вывода); При включении режима слайд-шоу, пользователь теперь может на лету изменить рейтинг фотографий не прерывая показ; Во встроенном редакторе изображений появилась опция для отображения индикатора изменения экспозиции в случае определения полной цветопередачи или соответствия одного из компонентов цвета заданным условиям; В редактор изображений добавлена опция для корректировки уровня экспозиции пикселов по гистограмме; В инструменте для обрезания изображения с учетом соотношения сторон появилась опция для сохранения оригинальных пропорций изображения.
