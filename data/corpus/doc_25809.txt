Увидел свет Linux-дистрибутив Mandriva Enterprise Server 5.1

Компания Mandriva объявила о выходе коммерческого серверного дистрибутива Mandriva Enterprise Server 5.1. Одним из основных направлений развития дистрибутива является организация централизованного управления над инфраструктурой серверов и рабочих станций предприятия (поддерживаются разные Linux дистрибутивы, Unix и Windows). Для организации управления используется платформа Pulse2, поддерживающая такие функции, как поддержание инвентаризационной базы данных компьютеров, установка и обновление приложений на группе машин, диагностика служб, удаленное управление. В новой версии значительные изменения коснулись поддержки технологии виртуализации KVM, в стандартный административный набор инструментов дистрибутива интегрированы средства для управления виртуальными окружениями и создания новых окружений. Кроме того, в новой версии упрощен процесс установки: добавлена возможность установки уже сконфигурированных сервисных наборов программ, нацеленных на определенный вид функциональности (почтовый сервер, файловый сервер, сервер печати, служба каталогов и т.п.). В состав дистрибутива также входит сервер каталогов Mandriva Directory Server, позволяющий организовать централизованную базу пользователей с интеграцией с такими серверными приложениями, как Samba, Postfix, Squid и CUPS. Для управления LDAP-сервером подготовлен специальный web-интерфейс. Оценить возможности Mandriva Enterprise Server можно загрузив образ для запуска дистрибутива в виртуальных окружениях. Стоимость дистрибутива равна 299 евро на одну машину с годовым контрактом технической поддержки или 500 евро с трехгодичным контрактом поддержки. Время жизни дистрибутива составляет 5 лет (обновления будут выходить до 2014 года). Mandriva Enterprise Server 5.1 прошел сертификацию на предмет совместимости со стандартом LSB 4.0.
