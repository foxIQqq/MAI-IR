GitHub добавил поддержку отслеживания уязвимостей в проектах на языке Rust

Компания GitHub объявила о добавлении поддержки языка Rust в каталог GitHub Advisory Database, в котором публикуются сведения об уязвимостях, затрагивающих проекты, размещённые на GitHub, а также выполняется отслеживание проблем в пакетах, связанных зависимостями с уязвимым кодом. В каталог добавлена новая секция, позволяющая отслеживать появление уязвимостей в пакетах, содержащих код на языке Rust. В настоящее время предоставлены сведения о 318 уязвимостях в проектах на языке Rust. Ранее в каталоге была предоставлена поддержка репозиториев, в которых развиваются пакеты на базе Composer (PHP), Go, Maven (Java), npm (JavaScript), NuGet (С#), pip (Python) и RubyGems (Ruby).
