Релиз Mandriva Linux 2011 и ROSA Desktop 2011

Спустя более года с момента выпуска прошлой значительной версии анонсирован выпуск релиза Mandriva Linux 2011. Релиз Mandriva 2011 подготовлен международной командой разработчиков, состоящей из специалистов российской компании "РОСА", бразильской Conectiva и французской Mandriva. Дистрибутив распространяется в виде одной DVD-сборки, поддерживающей работу в Live-режиме с возможностью установки на диск и доступной для архитектур i586 (1.6 Гб) и x86-64 (1.7 Гб). Подготовленные iso-образы являются гибридными, т.е. их можно использовать как на DVD, так и на USB Flash. Представленный выпуск как никогда раньше богат значительными нововведениями, как на уровне организации интерфейса пользователя, так и на уровне системных компонентов. Например, осуществлен переход на пакетный менеджер RPM5 и систему инициализации Systemd, добавлен новый интерфейс запуска программ и переработаны различные части десктоп-окружения. Пересмотрен процесс установки: вместо выбора компонентов в инсталляторе, сформирован устанавливаемый по умолчанию базовый состав приложений, для которого отобрано по одному тщательно подобранному и протестированному приложению для каждого вида задач. В соответствии с новым циклом подготовки релизов, новые значительные версии Mandriva будут выпускаться раз в год. Базовый срок поддержки и выпуска обновлений будет составлять полтора года. Раз в несколько лет будет формироваться LTS-выпуск, отличающийся длительным временем выпуска обновлений (3 года). В частности, ближайший LTS-релиз планируется сформировать на базе Mandriva Linux 2011 и выпустить в конце года. Одновременно выпущен отечественный вариант Mandriva - ROSA Desktop 2011, отличающийся сборкой с учетом Российской специфики (русскоязычный интерфейс, другая визуальная тема, наличие мультимедиа кодеков и проприетарных драйверов из коробки, подборка приложений для обеспечения рабочих мест в отечественных государственных и коммерческих учреждениях). Все созданные в ROSA наработки развивались в рамках основного репозитория Mandriva и доступны пользователям данного дистрибутива. В частности, разработчиками ROSA подготовлены такие компоненты, как главное интерфейс запуска программ SimpleWelcome, панель KDE ROSA Panel и новый интерфейс для сервиса синхронизации данных MandrivaSync. Основные новшества: Системные изменения Вместо развиваемого компанией Red Hat пакетного менеджера RPM 4 для сборки пакетов теперь используется независимый проект RPM 5 (5.3.8). Переход на новый пакетный менеджер требует более внимательного отношения при проведении обновления старых выпусков дистрибутива до версии Mandriva 2011. Необходимые инструкции по обновлению и описание известных проблем можно найти на данной странице; Интегрирован и включен по умолчанию системный менеджер Systemd, который заменил собой классическую систему инициализации. Systemd нацелен на более интенсивную параллелизацию выполнения сервисов на этапе загрузки системы, вобрав в себя лучшие черты таких систем, как launchd (Mac OS X), SMF (Solaris) и Upstart (Ubuntu, Fedora). Кроме запуска сервисов, systemd обеспечивает функции контроля за выполнением фоновых процессов. Подробнее о systemd можно прочитать в ранее опубликованных обзорах (здесь, здесь, здесь, здесь и здесь). Сравнение систем инициализации systemd, upstart и SysVinit можно посмотреть здесь; Обновление базовых приложений: ядро Linux 2.6.38.7 (дополнительно подготовлена неофициальная сборка с ядром 3.0), X.org Server 1.10.3, X.Org 7.6, подборка последних выпусков видеодрайверов; Обновление интернет-приложений: web-браузер Firefox 5, почтовый клиент Thunderbird 5, клиент для мгновенного обмена сообщениями Pidgin 2.9, клиент для публикации сообщений в микро-блогах Choqok 1.1; Обновление мультимедийных приложений. Вместо ранее используемого Digikam в новой версии для управления фотографиями задействован просмотрщик Shotwell 0.10.1. Вместо Amarok используется Clementine 0.7.1 (переведенный на использование Qt 4 форк Amarok 1.4). В качестве видеоредактора по умолчанию задействован PiTiVi 0.14.2. В роли видеоплеера вместо VLC в поставку включен SMPlayer 0.6.9; Вместо OpenOffice.prg в дистрибутиве поставляется LibreOffice 3.4.2; Новые версии программ для разработчиков: gcc 4.6.1, git 1.7.6, kdevelop 4.2.3, sqlite 3.7.7, php 5.3.8, perl 5.12.3, python 2.7.2, MySQL 5.5.15, PostgreSQL 9.0.4; Для настройки сети вместо drakx-net-applet задействован NetworkManager, апплет KNetworkManager и плагин ifcfg-mdv. Старый конфигуратор drakx-net-applet можно использовать в качестве альтернативного и совместимого с прошлыми выпусками решения; Интеграция облачной платформы OpenStack; Пользовательское окружение Обновление десктоп-окружения KDE с версии 4.4.5 до 4.6.5. Начиная с данного выпуска KDE будет единственным официально поддерживаемым пользовательским окружением. Все усилия компании Mandriva теперь сосредоточены только на KDE. Поддержка пакетов и сборок с GNOME, XFCE, LXDE и другими пользовательскими оболочками будет осуществляться только силами независимого сообщества; Новая система установки дистрибутива, работающая после загрузки в Live-режиме и копирующая текущий Live-образ на жесткий диск. Новый подход позволил заметно сократить время установки и свести к минимуму число выполняемых в инсталляторе шагов. Установка дополнительных пакетов и настройка дополнительных параметров производится после инсталляции. Переработанная графическая тема ROSA и новый набор пиктограмм. Новая комплексная визуальная тема охватывает все части дистрибутива, включая загрузчик, оформление для приложений KDE и GNOME, а также содержит новый набор пиктограмм и курсоров. Унифицировано оформление и поведение для диалогов базовых приложений, например, диалогов записи и чтения файлов, вывода на печать. Отдельно отмечается, приведение к общему виду оформление GTK+ и Qt приложений, которые при использовании темы ROSA выглядят неотличимо; Переработанное оформление менеджера входа в систему (KDM), отличающееся большей наглядностью, минимализмом и использованием графических эффектов. Для уменьшения числа ошибок при вводе пароля на экран добавлены индикаторы раскладки клавиатуры и нажатия клавиши CapsLock. Проведена оптимизация для управления только с клавиатуры: переключаться между пользователями можно через клавишу Tab, по умолчанию фокус ввода указывает на поле ввода пароля. Добавлен новый набор аватаров (свою фотографию можно добавить через меню "Учетная запись -> Профиль пользователя"); Доработан файловый менеджер Dolphin: Удалены лишние элементы с панели инструментов; Меню скрыто и доступно через отдельную кнопку в правой части строки инструментов; Реструктуризировано меню, вызываемое при нажатии правой кнопки мыши; Более наглядная информация в строке статуса; При создании ярлыков новая ссылка создается по умолчанию с именем "Ссылка на имя_файла"; Не включен по умолчанию, но доступен из репозиториев новый интерфейс для управления пакетами MPM (Mandriva Package Manager). MPM построен с использованием клиент-серверной архитектуры (пользовательский интерфейс взаимодействует с управляющим сервисом mdvpkgd через D-Bus). Поддерживается разбиение приложений по категориям, например, по типу приложений (Мультимедиа, офис и т.п.) и источнику их получения (официальные и внешние репозитории); SimpleWelcome - заменяющий стандартное меню интерфейс для запуска приложений. Приложения выводятся сгруппированными по базовым категориям (интернет, офис, графика, игры). Имеется поддержка просмотра последних запущенных приложений и открытых документов, а также возможность более детально проследить историю работы с файлами через систему TimeFrame. Предусмотрена функция быстрого поиска по ключевым словам в названии и описании программ и документов; TimeFrame - новый интерфейс для отслеживания активности пользователя в системе, построенный с использованием системы семантических связей Nepomuk. Поддерживается отслеживание работы с фотографиями, видеофайлами, документами. Возможен поиск нужных файлов и документов с учетом даты их создания и модификации, сортировка по типам файлов и просмотр истории работы в указанный период времени; StackFolder - апплет для организации быстрого доступа к наиболее используемым каталогам и файлам. Создание стековой папки происходит через выбор пункта "Стековая папка" в выпадающем меню на рабочем столе или панели; RocketBar - интерфейс для быстрого запуска и переключения между запущенными приложениями. RocketBar отображает несколько областей: часто используемые приложения, запущенные пользовательские приложения, запущенные системные приложения и доступные стековые папки; MandrivaSync - похожий на Ubuntu One и Dropbox сервис, который позволяет сохранить во внешнем облачном окружении настройки и файлы, обеспечив их синхронизацию между всеми машинами пользователя. Облачное хранилище размером 2 Гб предоставляется бесплатно. Имеется возможность указания конкретных объектов для синхронизации и выбора времени автоматической синхронизации; Sphere - интерфейс для упрощения доступа к технической поддержке, позволяющий быстро оформить заявку, обсудить детали и проследить за получением ответа; Improver - комплекс для поиска ошибок и тестирования работоспособности дистрибутива на аппаратном обеспечении пользователя. Клиентская часть Improver позволяет загрузить тестовый набор и отправить информации об аппаратном обеспечении и характеристиках системы. Общую статистику можно посмотреть через централизованный web-интерфейс. Полная интеграция системы для управлений персональной информацией KDE PIM 4.6.0 (Kontact Suite); Новый центр управления (mandriva-control-center), написанный с использованием библиотеки Qt и интегрированный в систему управления конфигурацией KDE;
