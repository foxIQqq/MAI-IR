Intel и Nokia представили совместный свободный проект oFono

Компании Intel и Nokia объединили свои усилия в создании нового свободного стека приложений для решения задач, связанных с телефонией. Новый проект получил название oFono и ориентирован на разработку программ для работы на мобильных телефонах, поддерживающих стандарты GSM или UMTS. Исходные тексты oFono распространяются в рамках лицензии GPLv2. В качестве связующей шины и высокоуровневого API для использования функций телефонии в программах используется D-Bus, что позволяет коммерческим производителями комбинировать с приложениями oFono собственные проприетарные разработки. D-Bus API предоставляет интерфейс к функциям базового ядра oFone Core Stack, которое непосредственно осуществляет выполнение сетевых операций, обслуживание звонков, работу с данными на SIM картах. Для взаимодействия oFone Core Stack с оборудованием подготовлен специальный низкоуровневый API, позволяющий интегрировать oFono в окружение сторонних телефонных стеков, использовать возможности GSM-модемов и устройств чтения SIM-карт. Данный API (plug-in API) построен с учетом существующих открытых стандартов, например 3GPP TS 27.007 "Набор AT-команд для оборудования пользователя (UE)." Разработчикам конечных приложений oFono позволит полностью сосредоточится на создании пользовательского интерфейса, без углубленного изучения тонкостей работы телефонии. Так как в качестве API задействован D-Bus, каких-либо ограничений на выбор языков программирования не накладывается. API достаточно простой для изучения и минималистичный. Предусмотрены функции для голосовых вызовов, SMS, операций с адресной книгой на SIM-карте, широковещательной передачи данных и использования сопутствующих сервисов. В настоящий момент управление и развитие проектом осуществляется совместными усилиями компаний Nokia и Intel, которые приветствуют подключение к работе oFono заинтересованных разработчиков и компаний, стремятся сформировать вокруг проекта независимое комьюнити.
