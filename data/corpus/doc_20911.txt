Компания Novell представила релиз дистрибутива SUSE Linux Enterprise 11

Компания Novell выпустила релиз дистрибутива SUSE Linux Enterprise 11, доступный в серверной и десктоп редакциях, которые можно загрузить и использовать совершенно бесплатно, но с 60-дневным ограничением на получение обновлений и исправлений. Для линейки продуктов SUSE поставляются комплекты расширений, для каждого из которых может предоставляться коммерческая техническая поддержка: SUSE Linux Enterprise Server for System z - сборка оптимизированная для работы на System z (майнфремы IBM), может функционировать как обособленно в LPAR режиме, так и в роли гостевой системы в окружении z/VM; SUSE Linux Enterprise High Availability Extension - расширения, нацеленные на создание высоконадежных кластеров, базируются на кластерной файловой системе OCFS2 (Oracle Cluster File System 2), системе репликации блочных устройств DRDB 8, менеджере логических томов cLVM2, менеджере ресурсов Pacemaker и пакете для переключения нагрузки на запасной сервер OpenAIS; SUSE Linux Enterprise Mono Extension - расширение для выполнения серверных приложений написанных с использованием .NET. Для выполнения .NET программ задействован Mono 2.4; Инфраструктура виртуализации основана на Xen 3.3.1 и представлена в трех сборках - сервер для развертывания виртуальных окружений, комплект драйверов для запуска гостевых ОС в режиме паравиртуализации (имеется поддержка Windows) и SUSE Linux Enterprise JeOS, легковесная сборка дистрибутива для создания минималистичных систем, работающих в внутри виртуальных окружений. Для простого быстрого создания и управления виртуальными окружениями в дистрибутив интегрировано специализированное ПО на базе проектов ZENworks и PlateSpin. Кроме того, доступны расширения для выполнения приложений в реальном режиме времени, для организации POS-терминалов и для создания тонких клиентов. Главные новшества: Усовершенствование дистрибутива в плане обеспечения работы систем требующих особой надежности (mission-critical). В состав включены инструменты для самодиагностики и определения аппаратных сбоев, добавлена поддержка размещения раздела подкачки на удаленном NFS сервере (Swap over NFS для бездисковых станций), режим кластера из одного узла (для выполнения и отладки кластерных приложений). Из программ, входящих в комплект серверной системы, можно отметить: Linux ядро 2.6.27, GCC 4.3.2, glibc 2.9, perl 5.10, php 5.2.6, python 2.6.0, ruby 1.8.7, Apache 2.2.10, Samba 3.2.5 В базовый состав дистрибутива интегрирована система мандатного контроля доступа SELinux, при этом поддержка AppArmor сохранена в полном объеме. Возможность имеет статус экспериментального дополнения, без обеспечения "enterprise" класса поддержки. Кроме того, хотя в состав дистрибутива включены все необходимые утилиты, сама политика SELinux не поставляется. Таким образом для того чтобы включить SELinux требуется скачать исходный код политики из независимого источника и обладать умением ее правильно собрать на хосте; В комплект десктоп приложений интегрировано ПО для организации групповой работы (Novell Teaming + Conferencing), основанного на открытом проекте ICEcore. В поставку включена расширенная версия OpenOffice.org 3.0 (Novell Edition) и десктоп окружения KDE 4.1.3 и GNOME 2.24; Осуществлен переход десктоп системы на аудиосервер Pulseaudio, который, например, позволяет определять разный уровень громкости для различных приложений, работать с несколькими источниками и приемниками звука, осуществлять преобразование из одного звукового формата в другой; Упрощен процесс установки дистрибутива и изменен внешний вид инсталлятора. Полностью заменен код системы разбиения диска. Переписаны YaST модули для управления печатью, дисковыми разделами, настройками связанными с безопасностью, добавлена возможность настройки соединения с GSM/CDMA/3G сетями, упрощена настройка многомониторных конфигураций. Обновлен внешний вид YaST; В качестве файловой системы по умолчанию используется Ext3, вместо ReiserFS. В экспериментальном режиме возможно задействовать файловую систему ext4. Кроме того менеджер логических томов EVMS2 заменен на LVM2.
