Релиз Bootstrap 2.0, развиваемого компанией Twitter набора web-шаблонов

Доступен релиз Bootstrap 2.0, открытого компанией Twitter набора шаблонов и готовых компонентов пользовательского интерфейса, в рамках которого сформирована коллекция CSS-стилей, Javascript-функций и HTML-кода для быстрого оформления типовых элементов web-страниц и организации интерактивного взаимодействия с пользователем. Для формирования стилей задействован препроцессор Less. Код Bootstrap распространяется в рамках лицензии Apache. Работа шаблонов проверена на корректность работы в последних выпусках Safari и Google Chrome, Firefox 4+ и Internet Explorer 7+. Bootstrap включает в себя набор элементов типографики, различные представления web-форм, кнопок, таблиц и списков, средства для формирования сложной разбивки элементов страниц, блоки навигации, вкладки и т.п. Для интеграции данных элементов достаточно подключить нужный CSS и добавить на страницу определенные теги div. Если первый выпуск Bootstrap был создан в основном для реализации внутренних нужд Twitter, то второй релиз вобрал в себя пожелания и улучшения, предложенные сообществом (Bootstrap является одним из самых популярных проектов на Github), а также наработки, созданные в рамках подготовки последнего редизайна сайта Twitter.com. В новой версии: полностью переработана документация; представлена новая реализация инструмента кастомизации для создания вариантов Bootstrap только с необходимым набором компонентов; представлены новые стили для таблиц и форм; осуществлён переход к 12-колоночной базовой сетке наложения элементов контента; дополнительный набор пиктрограмм; новые компоненты для создания навигационных элементов; новые кнопки, группы кнопок и выпадающие управляющие элементы; новый упрощенный диалог вывода уведомлений; новые JavaScript-плагины, такие как карусель (для отображения слайдшоу), сворачивание и упреждающий ввод.
