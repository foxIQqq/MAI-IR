Компания Adobe начала тестирование Flash 10.3 и конвертера FLA-файлов в HTML5

Компания Adobe выпустила бета-версию Flash 10.3 для Windows, Mac OS X и Linux. Финальный релиз запланирован на второй квартал 2011 года. Из новшеств можно отметить: В API добавлены функции для централизованного измерения характеристик медиаконтента, позволяющие в режиме реального времени сформировать на сервере SiteCatalyst статистику об особенностях доставки контента, числе загрузок и аудитории; Расширено API для обработки звука, позволяющее создавать такие системы как голосовой чат, приложения для телефонии и организации телеконференций. Реализованы механизмы акустического гашения эхо, подавления шумов, определения наличия голоса, компенсации уровня громкости при передаче звука с микрофона. Реализован блок настроек для легкого удаления содержимого локального кэша и генерируемых средствами Flash Cookie-идентификаторов, которые могут быть использованы для отслеживания перемещения пользователя в сети. В отличие от традиционных браузерных Cookie, формируемые средствами Flash идентификаторы невозможно удалить традиционными средствами браузера. Добавлена встроенная панель управления настройками Flash Player, через которую можно управлять параметрами приватности, безопасности и управлять локальным хранилищем. Для платформы Mac OS X добавлена система уведомления о наличии более новых версий Flash Player. Дополнительно анонсирован выход предварительной версии проекта Wallaby, в рамках которого подготовлен инструментарий для преобразования медиаконтента и анимации из FLA-файлов (Adobe Flash Professional CS5) в HTML5-представление, которое может быть просмотрено любым современным браузером на базе движка WebKit, даже без наличия Flash-плеера. В качестве основной причины создания инструментария называется желание обеспечить доступность FLA-контента для платформы Apple iOS (iPad, iPhone и iPod), для которой вследствие политики Apple не доступен Flash-плеер.
