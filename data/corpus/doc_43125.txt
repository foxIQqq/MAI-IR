Выпуск почтового клиента Claws Mail 3.13.0

Представлен релиз лёгкого и быстрого почтового клиента Claws Mail 3.13.0, который отделился от проекта Sylpheed в 2005 году (с 2001 по 2005 г. проекты развивались совместно, Claws использовался для тестирования будущих новшеств Sylpheed). Интерфейс Claws Mail построен с использованием GTK+, код распространяется под лицензией GPL. Ключевые новшества: Изменение настроек теперь записывается на диск сразу после нажатия OK или "Применить". Добавлен новый блок настроек для определения используемых по умолчанию параметров сортировки в списках сообщений; Функция удаления дискуссии перемещена из основного меню в меню управления сообщением и теперь приводит к фактическому удалению, а не перемещению в корзину. Для корзины добавлен отдельный пункт "Переместить в корзину"; В окне составления сообщений прекращена поддержка ввода произвольных заголовков, вместо чего предлагается использовать только ранее предопределённые заголовки. Для возвращения старого поведения в файле конфигурации предусмотрена опция "type_any_header"; Автоматический выбор аккаунта теперь применяется и при ответе на своё собственное сообщение в папке "Отправленные"; Для раскрытия и сворачивания древовидного представления текущего элемента папки теперь можно использовать клавиши курсора "Влево" и "Вправо"; В списке папок для перехода к первой и последней папке теперь можно использовать клавиши Home и End; Обеспечен вывод более детальных диагностических сообщений при возникновении ошибок при автоматической конфигурации учётной записи; В плагине для работы с GnuPG добавлена возможность определения пути к исполняемому файлу GnuPG, а также добавлена настройка 'autocompletion_limit' для ограничения числа выводимых ключей при автодополнении ввода; Из поставки удалён плагин GeoLocation для определения местоположения.
