Google выпустил плагин для форматирования кода в Vim

Разработчики из компании Google анонсировали новый открытый проект vim-codefmt, в рамках которого подготовлена утилита для автоматической расстановки отступов и форматирования кода с учётом синтаксиса используемого языка программирования. Утилита может использоваться как плагин для обеспечения форматирования кода в редакторе Vim. Поддерживаются две команды ":FormatLines" для форматирования диапазона строк и ":FormatCode" для форматирования буфера целиком. Обеспечена поддержка языков C++, JavaScript, Protocol Buffers и Go. Например, при использовании vim-codefmt строка int foo(int * x) { return * x** x ; } будет переоформлена в int foo(int* x) { return *x * *x; }
