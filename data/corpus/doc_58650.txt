Выпуск Альт Сервер Виртуализации 10.1

Состоялся выпуск операционной системы «Альт Сервер Виртуализации» 10.1 на 10 платформе ALT (ветка p10 Aronia). Операционная система предназначена для использования на серверах и реализации функций виртуализации в корпоративной инфраструктуре. Доступен сервис для работы с образами Docker. Сборки подготовлены для архитектур x86_64 и AArch64. Продукт поставляется в рамках Лицензионного договора, который предоставляет возможность свободного использования физическими лицами, но юридическим лицам допускается только тестирование, а для использования требуется приобрести коммерческую лицензию или заключить лицензионный договор в письменной форме. Новшества: Системное окружение построено на базе ядра Linux 5.10 и systemd 249.13. В инсталлятор добавлен пакет kernel-modules-drm, обеспечивающий более высокую производительность для графического оборудования (актуально для платформ AArch64). Использование загрузчика GRUB (grub-pc) вместо syslinux в образе для Legacy BIOS. Добавлена поддержка оптимизации памяти NUMA (numactl) при использовании сценария базовой виртуализации на основе kvm+libvirt+qemu. Улучшена поддержка multipath для создания сетевого хранилища хранения данных (multipathd включен в исталляторе по умолчанию). В Настройках сети по умолчанию используется etcnet, который позволяет настроить сеть ручным способом. Для работы с конфигурационными файлами необходимы полномочия администратора (root). Использование CRI-O вместо Docker в Kubernetes. В системе управления виртуализацией PVE 7.2 (Proxmox Virtual Environment) добавлена поддержка новых сервисов и настроек, а также обновлены QEMU 7.1.0, LXC 4.0, Ceph 16.2.10 и OpenZFS 2.1.5. Увеличено число поддерживаемых CPU для хостов гипервизора, что позволяет использовать более мощное оборудование для развертывания системы управления виртуализацией. Обновлены версии ключевых компонентов для создания, управления и мониторинга виртуального контура. Обновлены официальные образы контейнеров в реестре контейнеров, а также обновлены образы на ресурсах hub.docker.com и images.linuxcontainers.org. Новые версии приложений CRI-O 1.22. Docker 20.10. Podman 3.4. Apache 2.4. SSSD 2.8. PVE 7.2. FreeIPA 4.9. QEMU 6.2. Ansible 2.9. Libvirt 8.0. MariaDB 10.6. Улучшена документация, в том числе добавлено описание новых функциональных возможностей и лимитов.
