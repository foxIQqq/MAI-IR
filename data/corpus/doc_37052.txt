Релиз дистрибутива Linux Mint 15

Увидел свет дистрибутив Linux Mint 15, основанный на пакетной базе Ubuntu 13.04, полностью совместимого с ним, но существенно отличающегося подходом к организации интерфейса пользователя и подбором используемых по умолчанию приложений. Разработчики Linux Mint предоставляют десктоп-окружение, соответствующее классическим канонам организации рабочего стола, которое является более привычным для пользователей не принимающих новые методы построения интерфейса Unity и GNOME 3. Для загрузки доступны DVD-сборки на базе оболочек MATE 1.6 (1 Гб) и Cinnamon 1.8 (928 Мб). Кроме отличий в организации рабочего стола, в дистрибутиве задействован ряд оригинальных приложений, упрощающих настройку и работу в системе начинающих пользователей. Например, в Linux Mint используется переработанная система меню, собственный менеджер установки и обновления приложений, оригинальный интерфейс для настройки системы mintConfig, "мастер" для настройки различных параметров системы для начинающих пользователей в стиле вопрос-ответ, интерфейс для настройки параметров десктопа, средство для выполнения резервного копирования mintBackup и т.п. Ключевые особенности Linux Mint 15: В состав включены новые версии десктоп-окружений MATE 1.6 и Cinnamon 1.8, оформление и организация работы в которых продолжает развитие идей GNOME 2 - пользователю предлагается рабочий стол и панель с меню, областью быстрого запуска, перечнем открытых окон и системным лотком с запущенными апплетами. MATE продолжает развитие кодовой базы GNOME 2.32, основан на GTK2+ и полностью избавлен от пересечений с GNOME 3, что позволяет использовать традиционный рабочий стол GNOME 2 параллельно с рабочим столом GNOME 3. В новой версии MATE проведена большая работа по чистке от устаревших компонентов и их замене на актуальные технологии, многие из которых используются в GNOME 3; внесена большая порция улучшений в файловый менеджер Caja; расширены возможности панели. Cinnamon основан на GTK3+ и технологиях GNOME 3. Проект развивает оболочку GNOME Shell и оконный менеджер Mutter в направлении предоставления окружения в стиле GNOME 2 с более современным оформлением и задействованием элементов из GNOME Shell, дополняющих средства классического десктопа. В новой версии Cinnamon представлена поддержка десклетов (Desklets) - интегрируемых на рабочий стол мини-приложений, которые выступают аналогом плазмоидов KDE и виджетов Android; добавлен отдельный менеджер для установки внешних компонентов (апплеты, десклеты, темы оформления, дополнения) и слежением за появлением обновлений; расширены возможности файлового менеджера Nemo; добавлена новая реализация хранителя экрана; Обновлён используемый в дистрибутиве дисплейный менеджер MDM 1.2, представляющий собой форк GDM 2.20. Как и GDM новый дисплейный менеджер предоставляет графические инструменты настройки параметров входа, возможность использования визуальных тем, средства организации удалённой работы, подключения своих скриптов для обработки событий, режим автоматического входа в систему, возможность выбора языка. В новой версии MDM добавлена поддержка трех новых дополнений для вывода экрана входа в систему: простой GTK greeter, GDM greeter с поддержкой использования тем оформления, созданных для GDM, и HTML greeter, поддерживающий новые анимированные и интерактивные темы оформления, основанные на использовании web-технологий (HTML5, CSS, JavaScript и WebGL). В состав нового выпуска дистрибутива включена большая коллекция готовых тем оформления MDM, в том числе доступна тема "Clouds GL", созданная с использованием WebGL и симулирующая полёт в облаках. Кроме того внесена большая порция изменений под капотом и упрощено использование конфигуратора mdmsetup; В состав включено новое приложение Software Sources (MintSources), созданное специально для Linux Mint и предназначенное для настройки источников установки приложений, выступая заменой ранее используемого компонента software-properties-gtk. Из основного экрана Software Sources пользователь может активировать и выключать опциональные компоненты, включить доступ к репозиториям с бэкпортами и нестабильными версиями пакетов, получить пакеты с исходными текстами приложений, подключить внешние репозитории и PPA, управлять ключами аутентификации. Кроме того, программа предоставляет наглядный интерфейс для выбора наиболее быстрого зеркала, отображая для каждого из зеркал результаты выполнения теста для оценки реальной скорости загрузки на текущую машину. В рамках приложения Driver Manager (MintDrivers) подготовлен новый интерфейс для установки имеющихся в наличии проприетарных и альтернативных драйверов для популярных устройств (NVIDIA, AMD, Broadcom, Samsung и т.п.); Расширены возможности менеджера установки приложений: добавлена поддержка автоматической загрузки очередной порции элементов при приближении просмотра к концу страницы со списком; возможность поиска только в отдельных категориях приложений; пакеты из секции Java теперь видны и в категории Программирование; Менеджер установки обновлений теперь не требует прав суперпользователя для обновления кэша;
