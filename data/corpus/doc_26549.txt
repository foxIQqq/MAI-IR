Бинарные приложения с GUI для Solaris 10 x86

Открылся новый проект по подготовке сборок свободно распространяемого программного обеспечения для Solaris 10 x86. В данный момент для загрузки и установки подготовлены следующие приложения: FUSE - эмулятор ZX Spectrum; DOS Box - эмулятор MS DOS; Wine - среда для запуска MS Windows приложений; Sane - среда для поддержки сканеров; CUPS - среда для поддержки принтеров (альтернатива lpd); UFRAW - средство для конвертирования RAW-форматов цифровых фотокамер; BigBand - музыкальный редактор; TuxType - клавиатурный тренажёр; MHWaveEdit - редактор звуковых файлов; игры HHexen, ABE, BlackBox, BugsSquish, Fillets. Другие активные репозитории для Solaris: Sunfreeware - около 300 пакетов для Solaris 10 в стандартном формате System V Package, средства автоматического разрешения зависимостей отсутствуют; Blastwave.org - более 1500 пакетов в формате System V Package. Для установки с учетом зависимостей используется напоминающая APT утилита pkg-get. pkg.opensolaris.org - репозиторий пакетов для OpenSolaris для использования совместно с пакетным менеджером IPS pkgsrc - проект NetBSD pkgsrc поддерживает репозиторий пакетов для Solaris. Conary - Solaris порт инновационной системы управления пакетами Conary, главное отличие которой от традиционных систем - при обновлении пакета копирование только изменившихся данных. sunware.programist.ru; sunfreepacks.com; pkg.opensolar.ru + karakuts.livejournal.com
