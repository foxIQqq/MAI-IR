Вышла третья альфа-версия Ubuntu 11.10

Доступна третья альфа-версия Ubuntu 11.10 "Oneiric Ocelot". Загрузочные iso-образы с тестовой версией Ubuntu 11.10 можно получить на данной странице. Релиз запланирован на 13 октября. Это последняя альфа-версия, в дальнейшем планируется выпустить ещё две бета версии. Основные новшества, добавленные в Ubuntu 11.10 Alpha 3: Началось распространение нескольких новых сборок: Lubuntu на базе десктоп-окружения LXDE. Для быстрой установки готового окружения на базе LXDE подготовлен мета-пакет lubuntu-desktop; Ubuntu Core, базовый костяк платформы для создания собственных сборок Ubuntu для встраиваемых устройств (пока поддерживается только архитектура ARM). Например, Ubuntu Core используется в качестве основы в Ubuntu IVI Remix, платформы для автомобильных информационно-развлекательных систем; В состав дистрибутива интегрировано Linux-ядро 3.0. Из связанных с ядром изменений отмечается включение Overlayfs, интеграция WiFi-драйверов Realtek RTL8192CU/RTL8188CU, поддержка беспроводного чипсета rt53xx, обновление конфигурации ядра, поддержка режима Intellimouse для ноутбука Lenovo Zhaoyang E47; Задействование новых экспериментальных выпусков Compiz и оболочки Unity. В новой версии доступен измененный диалог переключения окон при нажатии alt + tab. Улучшена производительность ланчеров и панели, осуществлено портирование на Gtk3 и задействован стек индикаторов Gtk3; Сокращено число отличий между Unity и упрощенной оболочкой Unity 2D, не требующей для своей работы современных видеокарт и способной работать без поддержки OpenGL. В Unity 2D добавлена поддержка средств для людей с ограниченными возможностями (accessibility); В качестве почтового клиента по умолчанию вместо Evolution в поставку включен Thunderbird 5.0 (Evolution можно будет установить из репозиториев). Произведена интеграция Thunderbird с ланчером и меню Unity; Полностью переработан пользовательский интерфейс клиента микроблогинга Gwibber, предназначенного для интеграции с социальными сетями (Twitter, Identi.ca, Facebook, Flickr и т.п.). Новый Gwibber написан на языке Vala (прошлые версии Gwibber были написаны на языке Python с использованием PyGTK); В дистрибутив интегрированы компоненты экспериментальной ветки GNOME (3.1.4), на базе которой в сентябре будет выпущен релиз GNOME 3.2; Интегрированный в прошлых тестовых выпуска Ubuntu дисплейный менеджер LightDM, который используется для запуска пользовательской сессии (ввода логина/пароля, выбор графической оболочки и языковых настроек), портирован для работы с библиотекой Gtk3. Обновлено оформление отображаемых в панели индикаторов, полностью изменены индикаторы сессий и контроля за питанием; В Центре установки приложений (Software Center) добавлена переработанная система рейтинга и рецензирования приложений. Свои рейтинги наиболее популярных приложений теперь отображаются как для основных категорий, так и для подкатегорий. Значительно увеличена скорость установки отдельных deb-пакетов через gdebi; В серверный вариант Ubuntu для портативных серверных платформ на базе архитектуры ARM (armel+omap, armel+omap4 и armel+mx5) добавлена поддержка предустановленного пула приложений, дающего возможность полностью укомплектовать сервер без наличия доступа в Интернет. Исправлено множество ошибок и недоработок, включая налаживание работы стека IPsec; В Kubuntu осуществлен переход на KDE 4.7, произведена интеграция KDEPIM 4.7 и переход на использование хранилища Akonadi. Задействован новый набор пиктограмм для темы Oxygen. В состав включен релиз Amarok 2.4.3. Для управления пакетами по умолчанию теперь используются Muon Software Center и Muon Package Manager; Ubuntu переходит на использование OpenStack в качестве основной cloud-платформы и больше не будет базироваться на Eucalyptus. Вместо "Ubuntu Enterprise Cloud" (UEC) будет использоваться название "Ubuntu Cloud", а сборки виртуальных машин будут распространяться как 'Ubuntu Cloud Images'.
