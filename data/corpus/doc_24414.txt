Вышел релиз NexentaStor 2.2.0, дистрибутива для создания сетевых хранилищ

Вышел релиз NexentaStor 2.2.0 Developer Edition, дистрибутива для создания хранилищ данных. В качестве файловой системы используется ZFS. От коммерческой версии "Developer Edition" отличается ограничением размера поддерживаемых хранилищ, которые не могут превышать 4 Тб. NexentaStor построен на кодовой базе NexentaCore, сочетающей в себе ядро OpenSolaris и программное окружение Ubuntu 8.04. Из добавленных в новой версии улучшений можно отметить: Добавление средств для антивирусной проверки хранимого контента (используется ClamAV); Поддержка бэкапов СУБД Oracle и MySQL через создание непротиворечивых снапшотов ФС; Поддержка устройств хранения Citrix StorageLink; Поддержка асинхронной обратной репликации; Реализация механизма поиска данных, учитывающего содержимое отдельных снапшотов; Поддержка удаленного доступа.
