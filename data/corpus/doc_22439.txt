Вышел Calculate Linux Desktop 9.7 KDE

Вышла новая версия Calculate Linux Desktop, свободной операционной системы на базе Gentoo Linux для ПК или ноутбука. Система поддерживает работу в локальной сети совместно с Windows машинами, где в качестве контроллера домена выступает Calculate Directory Server. Возможности: Дружественный рабочий стол. 100% совместимость с Gentoo. Возможность работать с livecd с полной загрузкой в память. Поддерживаемые языки: английский, испанский, немецкий, португальский, итальянский, русский, украинский, польский и французский. Поддерживается установка на HDD, USB-Flash и USB-HDD с файловой системой: ext4, ext3, ext2, reiserfs, xfs и jfs. Основные изменения: Добавлена поддержка итальянского и польского языков. Добавлена поддержка установки системы на USB Flash с DVD либо HDD. Для установки достаточно 2 Гб Flash. В 2.5 раза ускорен процесс создания учетной записи в KDE. Добавлена настройка звуковой карты. Добавлено распознавание ПК и ноутбуков с двумя видеокартами. Размаскировка нестабильных версий программ выполнена с учетом последней доступной на момент сборки версии. Добавлена опция загрузки образа LiveDVD в оперативную память. Этот режим будет работать только на ПК с 2 Гб оперативной памяти и более. На рабочий стол livecd добавлен файл README.html с описанием процесса установки. Все дополнительные пакеты собираются из ebuild файлов Calculate Overlay. Таким образом Calculate Linux Desktop KDE на 100% совместим с Gentoo. Состав: KDE 4.2.4, Kernel 2.6.28.10, OpenOffice 3.0.1, X.Org 7.4.
