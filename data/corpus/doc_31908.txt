Вышла новая версия Bittorrent клиента и сервера Vuze 4.7

Вышла новая версия многоплатформенного торрент клиента/сервера Vuze 4.7, разработка которого ранее велась под именем Azureus. Код программы написан на языке Java и распространяется под лицензией GPL. Графический интерфейс построен с использованием тулкита SWT, базирующегося на Gtk+ в Linux. Vuze является одним из наиболее богатых по возможностям торрент-приложений, поддерживает удаленное управление, снабжен интегрированными средствами поиска, обладает гибкими возможностями по управлению пропускной способностью, поддерживает работу через анонимные сети I2P, Tor и Nodezilla. Для загрузки доступны сборки для Linux (32- и 64-битные сборки), Windows и Mac OS X, а также Jar-файл и исходный код. Основные улучшения: В Device Playback (функции копирования и перекодирования контента для устройств) добавлена поддержка новых устройств и увеличена совместимость с ранее поддерживаемыми медиаплеерами, телефонами, телеприставками и прочей портативной техникой. Пока эта функция доступна только для платформ Windows и Mac OS на системах с CPU Intel. Часть кода определения устройств выведена из ядра Vuze; Удалены IView, AbstractIView, SWTManager и другие устаревшие API плагинов; Добавлено отображение кнопки боковой панели для устройств и кнопки "Do Not Transcode" ("Не перекодировать"); Все виды интерфейсов пользователя переведены с IView на UISWTVIew; Весь не связанный с обновлением интерфейса код вынесен из потока SWT, что позволило добиться улучшение отзывчивости интерфейса; Удалён остаточный и неиспользуемый код (в английской новости авторы выразились так: "это освободило жалкие 200 килобайт")
