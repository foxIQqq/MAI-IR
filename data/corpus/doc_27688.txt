Ginn - система для обеспечения управления жестами в приложениях, не поддерживающих мультитач API

Недавно представленный стек uTouch, предназначенный для организации мультитач-навигации и управления жестами, вызвал у некоторых энтузиастов скептическую реакцию, связанную с необходимостью модификации приложений для задействования функций управления жестами. Для преодоления данного препятствия представлен проект Ginn (Gesture Injector), в рамках которого создан специальный процесс-демон, позволяющий обеспечить управление через систему экранных жестов в любом графическом приложении, не поддерживающем напрямую API uTouch-GEIS или функции multi-touch библиотеки GTK+. Ginn работает поверх стека uTouch и в фоне распознает управляющие жесты, транслируя их в активное приложение в виде эмуляции нажатия комбинаций клавиш, которые формируются для каждой программы индивидуально на основе предварительно оформленного набора правил. Видеодемонстрации возможностей, связанных с управлением жестами, представлены для Evince и Inkscape.
