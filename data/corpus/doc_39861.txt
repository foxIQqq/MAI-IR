Демонстрационный выпуск игрового движка Leadwerks для Linux

Компания Leadwerks Software, которая в прошлом году на Kickstarter собрала 42 тысячи долларов на создание варианта игрового движка Leadwerks Engine для платформы Linux, опубликовала первый демонстрационной выпуск движка для Linux. Сборки движка подготовлены для Ubuntu 14.04, для работы необходимы проприетарные видеодрайверы NVIDIA и ATI и видеокарта с поддержкой OpenGL 4.0. Движок также доступен для загрузки через штатный Ubuntu Software Center. Целью проекта на Kickstarter была подготовка игрового движка и связанной с ним визуальной среды разработки, предоставляющих полноценный набор средств для создания и выполнения современных 3D-игр в Linux. Для разработки 3D-моделей в Leadwerks обеспечена интеграция со свободным пакетом 3D-моделирования Blender. Интерфейс среды разработки построен с использованием GTK+. Разработка игр осуществляется на языках программирования C++ и Lua. Компоненты движка Leadwerks поставляются под проприетарной лицензией, предусматривающей возможность получения доступа к исходным текстам движка для разработчиков игр.
