Доступна интегрированная среда разработки NetBeans IDE 8.2

Компания Oracle опубликовала релиз интегрированной среды разработки приложений NetBeans IDE 8.2, поддерживающей разработку на языках Java, JavaFX, PHP, JavaScript/HTML5, Groovy и C/C++. NetBeans распространяется в исходных текстах под лицензиями GPLv2 и CDDL. В прошлом месяце компания Oracle инициировала процесс передачи проекта организации Apache Software Foundation. На днях в сообществе Apache завершилось голосование, в соответствии с которым NetBeans принят в инкубатор Apache. В настоящее время началась работа по подготовке инфраструктуры. Таким образом, NetBeans IDE 8.2 является последним релизом от компании Oracle, следующий выпуск выйдет под именем Apache NetBeans и будет доступен под лицензией Apache 2.0, вместо GPLv2 и CDDL. Основные улучшения NetBeans 8.2: В средствах для разработки на JavaSript добавлена поддержка стандарта ЕCMAScript 6 и экспериментальная поддержка спецификации ECMAScript 7. В том числе добавлена поддержка литералов hexa, binary и octal, а также обеспечено автодополнение кода для генераторов; Улучшены средства разработки на языке JavaScript с использованием платформы Node.js: обеспечена совместимость с Node.js 4 и более новыми выпусками, добавлена поддержка запуска задач для систем сборки Grunt и Gulp, реализована корректная обработка операции return в глобальном контексте; Добавлена поддержка языка PHP 7, а также возможность определения собственных аннотаций для проектов PHP, поддержка Symfony 3 и PHPUnit 5, и возможность использования файла конфигурации phpcs.xml; Добавлена поддержка системы управления контейнерной изоляцией Docker. В NetBeans теперь можно редактировать файлы Dockerfile с подсветкой синтаксиса, добавлена полная поддержка терминала и лога Docker; Улучшена поддержка Oracle JET (JavaScript Extension Toolkit): добавлен шаблон Oracle JET Base Distribution и пример приложений для Oracle JET Quickstart Basic и Oracle JET Component Interaction; В редактор добавлен режим для работы с несколькими курсорами (например, синхронно можно отредактировать несколько типовых строк); При отладке кода на Java, JavaScript и C/C++ реализована возможность закрепления на видном месте в окне подсказки с содержимым отслеживаемой переменной или выделенного выражения; Добавлен режим профилирования SQL-запросов с возможностью подсветки результатов выполнения запросов на основе заданного пользователем фильтра; Улучшены средства для разработки на C/C++: изменён интерфейс настройки команд Run и Debug, реализовано автоматическое определение компилятора, применяемого в используемой пользователем системе сборки; Добавлена возможность поиска в терминале;
