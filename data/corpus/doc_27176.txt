Компания Oracle вошла в состав консорциума, развивающего БД SQLite

Компания Oracle объявила о присоединении к работе ассоциации SQLite Consortium, финансирующей проект SQLite, наработки которого переведены в разряд общественного достояния и распространяются безвозмездно (Public Domain). В ответ разработчики SQLite готовы обеспечить входящим в ассоциацию компаниям техническую поддержку из первых рук, внеплановое исправление ошибок и реализацию новшеств, а также предоставить правовые гарантии того, что SQLite останется независимым и открытым проектом. При этом ассоциация оставляет за разработчиками SQLite полное право управлять проектом и развивать его на свое усмотрение. По заявлению разработчиков, участие в ассоциации и делегирование процесса доработки и поддержки критически важных решений на базе SQLite дешевле, чем наем дополнительного персонала для обслуживания SQLite силами компании. Кроме Oracle, спонсорами разработчиков SQLite выступают такие компании, как Adobe, Mozilla, Symbian и Bloomberg. Ранее компания Oracle интегрировала в БД Berkeley DB 5.0 поддержку режима совместимости с SQLite и выпустила продукт для синхронизации СУБД Oracle с SQLite. Участие в консорциуме позволит Oracle получить доступ к более полному тестовому комплекту, для проверки степени совместимости BerkeleyDB SQL API с SQLite 3 ANSI C API. По заявлению представителей Oracle, несмотря на реализацию эмуляции SQLite API в BerkeleyDB компания не планирует создавать форк проекта, а намерена активно участвовать в развитии основной кодовой базы и укрепить сотрудничество с разработчиками БД SQLite.
