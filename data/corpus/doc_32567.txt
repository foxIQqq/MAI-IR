В Windows Azure обеспечена интеграция с Node.js и другими открытыми проектами. В Messenger появилась поддержка XMPP

Компания Microsoft объявила о реализации в облачной платформе Windows Azure поддержки ряда открытых проектов. Для разработчиков выпущен SDK для обеспечения работы серверной JavaScript-платформы Node.js, в который включены библиотеки, позволяющие использовать из программ на Node.js сервисов и хранилищ Windows Azure. Кроме .Net и Node.js для разработки приложений также обеспечена поддержка языка Java. Дополнительно отмечается намерение в скором времени обеспечить поддержку Apache Hadoop для организации распределенной обработки больших объемов данных с использованием парадигмы map/reduce. Все библиотеки для интеграции вышеупомянутых систем для Windows Azure, а также утилиты из состава SDK, опубликованы под лицензией Apache. Кроме того, анонсирован выход значительного обновления плагина на базе Eclipse, в котором обеспечена поддержка разработки для Windows Azure с использованием таких открытых технологий, как БД MongoDB, системы кэширования в памяти Memcached, поисковых механизмов Apache Solr и Apache Lucene. Дополнительно компания Microsoft сообщила о реализации в клиенте для мгновенного обмена сообщениями Messenger поддержки открытого стандарта XMPP, что делает ранее обособленный продукт совместимым с различными Jabber-клиентами.
