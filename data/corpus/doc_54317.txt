Выпуск Icaros Desktop 2.3, дистрибутива операционной системы AROS

Доступен релиз дистрибутива свободной AmigaOS-совместимой десктоп-ориентированной операционной системы AROS - Icaros Desktop 2.3. Icaros Desktop позволяет на обычном ПК запустить Amiga-подобное десктоп-окружение, для которого в комплект включена подборка развлекательных приложений и игр. Для загрузки доступны три загрузочных образа: сокращённый (384 МБ), полный (2.3 ГБ) и редакция для USB Flash (1.3 ГБ). Основные изменения: В состав включены новые приложения, включая табличный процессор Leu, просмотрщик изображений SilkRAW, поддерживающий импорт с цифровых камер в формате RAW, утилиту для создания пиктограмм IconPoser, шахматную игру Amifish, программу для создания PDF-документов RNOPUblisher и интерфейс для просмотра комиксов ComicOn. Улучшен рабочий стол Magellan, в котором появилась возможности определения обработчиков для новых типов файлов (lua-скрипты, электронные таблицы, ISO, BMP и т.п.). Переделана работа с архивами. Встроены календарь-планировщик LilCalendar и интерфейс для создания скриншотов и скринкастов SnapIT. Добавлены новые шрифты Terminus и Petme. Предоставлена возможность установки Icaros Desktop в окружениях Linux и Windows, и работы в качестве оболочки для запуска установленных в основной системе приложений Windows и Linux (файлы .desktop и .lnk импортируются при помощи компонента HostBridge). Помимо образов для CD/DVD началось формирование сборки для загрузки с USB-накопителей.
