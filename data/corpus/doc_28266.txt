Новые версии дистрибутивов SystemRescueCd 1.6.2 и Parted Magic 5.6

Доступны обновленные версии двух популярных Live-дистрибутивов, предназначенных для восстановления системы после сбоя: Parted Magic 5.6 - компактный LiveCD/LiveUSB дистрибутив для восстановления системы после сбоя. Графический интерфейс основан на окружении Xfce. В минимальном режиме дистрибутив может работать на машинах с 48 Мб ОЗУ. Размер iso-образа 134 Мб. Из базовых программ можно отметить: web-браузер Chromium, GParted, TestDisk, PhotoRec, Partition Image, Xarchive, Xfburn, ISO Master, Leafpad, Grsync. Из утилит командной строки присутствуют dd, ddrescue , cfdisk, fdisk, tcpdump. Имеется поддержка восстановления файловых систем ext2, ext3, ext4, btrfs, FAT16, FAT32, HFS, HFS+, JFS, NTFS, ReiserFS, Reiser4, XFS. В новой версии добавлен X.Org-драйвер Nouveau X.Org и тестовый комплект Systester. Для удобства входа по SSH по умолчанию для пользователя root установлен пароль "partedmagic". Обновлены версии программ: Linux-ядро 2.6.35.7, GParted 0.6.4, gDisk 0.6.11, hdparm 9.33, stress 1.0.4, UNetbootin 490, NTFS-3G 2010.10.2, BusyBox-1.17.3 и udev 163. SystemRescueCd 1.6.2 - специализированный LiveCD-дистрибутив на основе Gentoo Linux, предназначенный для восстановления системы после сбоя. В качестве графического окружения используется Xfce. Размер iso-образа - 270 Мб. В новой версии обновлены следующие программы: Linux-ядро 2.6.35.6 (rescuecd + rescue64), GParted 0.6.4, ntfs3g 2010.10.2, X.org-server 1.9.0.901 и xf86-video-nv 2.1.18. В комплект добавлены дополнительные шрифты (font-adobe-75dpi, font-bh-ttf, font-bh-type1), для утилиты fsarchiver-0.6.10 применен патч, устраняющий проблемы при работе с btrfs-разделами.
