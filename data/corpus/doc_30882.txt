Разработчики медиа-плеера Boxee планируют возобновить выпуск открытых релизов

Разработчики проекта Boxee сообщили о намерении выпустить осенью доступный в исходных текстах релиз медиа-плеера Boxee для платформ Linux, Mac OS X, Windows и Apple TV. Открытая версия плеера не обновляется уже более 8 месяцев, в то время как обновления проприетарной прошивки для выпускаемого компанией D-Link медиаплеера Boxee Box выходят регулярно. Осенний релиз будет содержать ранее добавленные в прошивку улучшения и исправления, включая обновление медиа-браузера, улучшение проигрывания звука и видео, поддержку новых провайдеров контента. Дополнительно разработчики планируют провести работу по упрощению использования кода открытой версии Boxee для создания собственных модификаций продукта. Тем не менее, по возможностям доступная для свободной загрузки версия скорее всего будет немного отставать от версии, на базе которой формируются проприетарные прошивки, в частности, в области, связанной с обеспечением доступа к платному контенту. В качестве причин стагнации развития версии Boxee для настольных систем отмечается нехватка ресурсов разработчиков, которые были вынуждены посвятить свое время адаптации пакета для работы на устройстве Boxee Box, для которого пришлось по сути создать отдельный дистрибутив Linux, обеспечить поддержку совершенно иного процесса настройки, внедрить новые механизмы обеспечения защиты платного контента, создать новый медиа-браузер, интегрировать взаимодействующие с оборудованием библиотеки для обработки звука, видео и графики. После получения в марте значительных инвестиций, у проекта появились недостающие для развития настольной версии Boxee ресурсы. Код Boxee основан на открытом проекте XBMC, расширенном в направлении создания дополнительных проприетарных компонентов, обеспечения возможности интеграции с социальными сетями и online-сервисами, расширении средств для упрощения интеграции и разработки внешних дополнений. Программное обеспечение поддерживает все популярные мультимедиа форматы и сервисы потокового видео и аудио вещания. Например в Boxee интегрирована поддержка таких ресурсов, как Netflix, MLB.TV, Comedy Central, Pandora, Last.fm, YouTube, Hulu и Flickr. Кроме того в платформу интегрированы средства для работы с социальными сетями, такими как Twitter или Facebook. Для загрузки файлов из сети в медиа центр интегрирован BitTorrent-клиент. Разработка платформы ведется на языке C++ с использованием Python плагинов. Сборки Boxee доступны для платформ Windows, Linux и MacOS X. Часть исходных текстов Boxee распространяется под лицензией GPL, но некоторые новые подсистемы, например, библиотека libboxee, доступны только в бинарном виде.
