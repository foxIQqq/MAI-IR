Релиз тестового комплекта Phoronix Test Suite 2.4

Вышел релиз пакета для измерения производительности системы - Phoronix Test Suite 2.4. Пакет поддерживает работу в Linux, OpenSolaris, системах семейства BSD и Mac OS, распространяется под лицензией GPLv3 и призван обеспечить потребность простых пользователей в удобном инструменте для измерения производительности системы. По сравнению с прошлой версией в пакет внесено около 180 изменений, среди которых: Улучшена совместимость с ОС OpenSolaris, FreeBSD 8.0, OpenBSD и NetBSD. Добавлена поддержка мобильных платформ Palm webOS и Optware, а также начальная поддержка ОС Microsoft Windows 7. Добавлен новый тип тестовых профилей для оценки качества формирования изображений (покадровое сравнение идентичных по содержанию скриншотов). Данный метод тестирования пока доступен только для тестов Nexuiz и Enemy Territory: Quake Wars; Добавлена поддержка новых тестов GCrypt, Bullet Physics Engine, Himeno Pressure Solver, FS-Mark, CompileBench, Unpack-Linux, Unigine Heaven, Battery-Power-Usage, Idle-Power-Usage, x264 и VDrift FPS Monitor; Phoronix Test Suite 2.4 поддерживает 130 тестовых профилей и 53 различных тестовых наборов. Спецификация проведения каждого теста оформлена в виде XML файла, в качестве тестов используются внешние приложения, например, измеряется FPS в различных играх, оценивается время сборки Linux ядра, скорость кодирования mencoder, время сжатия bzip2 и т.д. Во время проведения тестов осуществляется контроль за изменением состояния оборудования (скорость вращения кулера, температура, напряжение). Итоги тестирования можно опубликовать в каталоге PTS Global, представляющем собой централизованную базу данных с результатами, полученными от различных пользователей.
