Вышел OpenJDK 6 b18, open source вариант Java SE

Увидела свет 18 сборка OpenJDK 6, открытого варианта Java SE, распространяемого в исходных текстах (за редким исключением используется лицензия GPLv2) и поставляемого в качестве Java-окружения по умолчанию в таких дистрибутивах, как Fedora и Ubuntu. В новой версии внесены исправления, связанные с безопасностью (проблемы в Swing, уязвимость в MessageDigest.isEqual), из ветки JDK 7 бэкпортировано визуальное оформление Nimbus, для компонентов jaxp и jax-ws задействована новая модель доставки.
