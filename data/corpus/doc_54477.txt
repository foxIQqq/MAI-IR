Релиз браузера Vivaldi 3.6

Компания Vivaldi Technologies анонсировала выход финальной версии проприетарного браузера Vivaldi 3.6, разрабатываемого на базе движка Chromium. Браузер развивается силами бывших разработчиков Opera Presto и ставит своей задачей создание настраиваемого и функционального браузера, сохраняющего приватность данных пользователей. В число основных функций входит блокировщик слежки и рекламы, менеджеры заметок, истории и закладок, приватный режим просмотра, синхронизация, защищённая сквозным шифрованием, режим группировки вкладок, боковая панель, конфигуратор с большим числом настроек, режим горизонтального отображения вкладок, а также в тестовом режиме встроенный почтовый клиент, RSS-ридер и календарь. Интерфейс браузера написан на языке JavaScript с использованием библиотеки React, платформы Node.js, Browserify и различных готовых NPM-модулей. Сборки Vivaldi подготовлены для Linux, Windows, Android и macOS. Для прошлых выпусков проект под открытой лицензией распространяет исходные тексты изменений к Chromium. Реализация интерфейса Vivaldi написана на JavaScript, доступна в исходных текстах, но под проприетарной лицензией. В новом выпуске: Значительно улучшена работа с группами вкладок - теперь для отображения входящих в группу страниц автоматически активируется дополнительная панель вкладок, что позволяет сделать работу с вкладками более удобной и интуитивной. При необходимости дополнительную панель можно закреплять для постоянного использования. Продолжена работа по расширению возможностей настройки контекстных меню браузера - в список доступных для изменения были добавлены контекстные меню боковых панелей. Для ускорения запуска браузера добавлена опция отложенной загрузки веб-панелей. Ранее при большом числе пользовательских веб-панелей при старте браузера осуществлялась загрузка всего их содержимого, что замедляло процесс запуска, при использовании новой опции загрузка содержимого веб-панели будет осуществляться только в момент её активации. Список исправленных ошибок включает в себя проблемы выхода из полноэкранного режима при просмотре видео, некорректное переключение между вкладками при закрытии активной, а также создание неправильного имени для сохраняемого на рабочем столе ярлыка веб-страницы. Для Linux-версии библиотека проприетарных медиа-кодеков обновлена до версии 87.0.4280.66. Движок Chromium обновлён до версии 88.0.4324.99.
