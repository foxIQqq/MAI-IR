Релиз программы для рисования MyPaint 1.0

После года разработки вышла новая версия специализированной программы для цифровой живописи с использованием планшета или мыши - MyPaint 1.0. Программа распространяется под лицензией GPLv2, разработка ведётся на языках Python и C++ с использованием тулкита Gtk+. Данное приложение интересно тем, что является хорошим подспорьем для художников, занимающихся цифровой живописью и, в ряде случаев, может составить достойную конкуренцию общепризнанным проприетарным программам для рисования Corel Painter и Artrage. Программа имеет простой, удобный в работе интерфейс и не позиционируется как графический редактор для обработки изображений. По сравнению с графическим редактором Gimp, MyPaint имеет большой набор кистей качественно симулирующих настоящие художественные инструменты, такие как карандаш, масляные краски, акварель, мастихин и другие. Одна из интересных особенностей программы — это бесконечный холст, который можно прокручивать и масштабировать. Для загрузки пока доступны только исходные тексты MyPaint 1.0, статус подготовки бинарных пакетов для различных дистрибутивов Linux, Windows и Mac OS X можно проследить на данной странице. Для Ubuntu поддерживается специальный PPA-репозиторий. Ключевые новшества версии 1.0: Реализована настраиваемая панель с выпадающими виджетами для управления цветами, выбора кистей и их настройки; Плавающие окна с инструментами теперь могут быть прикреплены к боковой панели основного окна; Для типов мазков кистей добавлен режим блокирования альфа-канала (Lock Alpha); В настройках можно определить разные действия для кнопок мыши и стилуса; Новый инструмент для работы с набросками, который можно использовать для хранения заметок, эскизов, записей цветового оформления и мазков кистей; Расширена коллекций кистей, проведены различные оптимизации интерфейса, увеличена производительность. Дополнение: Александр Прокудин опубликовал интервью с несколькими разработчиками MyPaint.
