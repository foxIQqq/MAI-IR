Вышел релиз OpenSolaris 2009.06

На конференции CommunityOne был официально представлен релиз десктоп-ориентированного дистрибутива OpenSolaris 2009.06. Основные новшества: Улучшение пакетного менеджера IPS (Image Packaging System): Возможность упаковки в один пакет исполняемых файлов одновременно для платформы x86 и Sparc; Возможность использования логических операций при поиске, например, "pkg search ключ1 AND ключ2"; Поддержка структурированного поиска, с заданием запроса в формате имя:тип_действия:ключ:признак, например, чтобы найти пакет с драйвером для определенного pci устройства - "pkg search :driver:alias:pci\*8086,\*" Улучшение GUI-интерфейса для управления пакетами. На порядок уменьшено время загрузки интерфейса, время установки сокращено с минут до секунд. Возможность установки пакетов через web в один клик. Поддержка категорий пакетов, средства управления репозиториями и возможность осуществлять поиск пакетов в нескольких репозиториях; Поддержка платформы SPARC, включая машины M-серии и серверные CMT (sun4v) системы T5xxx, Tx000, на которых поддерживается создание логических виртуальных окружений для организации работы рабочих станций в виде гостевых систем; Программа установки Caiman поддерживает на платформе Sparc автоматическую инсталляцию, конструктор дистрибутивов и систему обновлений SNAP Upgrade; Интеграция Crossbow, системы виртуализации IP стека и сетевых устройств. Отличительной особенностью Crossbow является возможность установки приоритетов в обработке трафика и гибкое управление пропускной способностью для изолированной зоны, протокола или сервиса. Трафик определенного сервиса или зоны при этом не не оказывает влияние на другие подсистемы, что, например, позволит сохранить работоспособность других сервисов и зон в момент проведения DoS атаки; Улучшение поддержки оборудования, добавление новых драйверов, улучшение в плане управления энергопотреблением и функционирования спящего/ждущего режимов. Работа OpenSolaris проверена на 2643 ноутбуках. Добавлены Wifi драйверы для карт на базе чипов ZyDAS ZD1211/ZD1211B и Ralink RT2501/RT2601/RT73USB; Улучшение систем хранения: возможность загрузки с iSCSI разделов, поддержка iSCSI для сборки SPARC; улучшение поддержки CIFS; улучшение производительности ZFS; Обновление версий программ: Пользовательское ПО: Firefox 3.1b3, Adobe Flash 10 (в extra репозитории), медиаплееры Songbird 1.0.0 и Elisa, Xorg 1.5, vim 7.1; Сресдства разработки: cmake 2.4.8, git 1.5.4.3, grails 1.0.3, lua, MySQL 5.1, Ruby 1.8.7, Python 2.6, Sun Studio/Studio Express 3/09, GNU Developer Collections 4.3.2, NetBeans 6.5.1, Eclipse 3.4.2, ClusterTools 8.1, Exuberant Ctags 5.7; Java и middleware: JDK 6u13, JavaFX, OpenDS v1.2, GlassFish 2.1, OpenMQ 4.1; Добавлена технология Clearview унифицирующая работу с сетевыми интерфейсами и предоставляющая единую утилиту для конфигурирования dladm, заменяющую /etc/system, driver.conf и ndd; В ПО для резервного копирования Timeslider появилась возможность ручного создания и удаления снапшотов, а также автоматической чистки снапшотов не содержащих изменений; Улучшение визуальных панелей: улучшение аплета для настройки SMF сервисов и конфигуратора Coreadm, новый аплет для управлением датой и временем; Coolstart - поддержка переключения видеорежимов на уровне ядра (Kernel Mode-Setting) для видеокарт Intel, что позволяет организовать бесшовный процесс загрузки, лишенный мерцания и скачков, дает возможность избавить X сервер от необходимости запуска с повышенными привилегиями, упрощает поддержку спящего и ждущего режима.
