Вышел мультимедийный проигрыватель MPlayer 1.2

Спустя более двух лет с момента прошлого выпуска представлен релиз мультимедийного проигрывателя MPlayer 1.2. В новой версии обеспечена совместимость со свежей версией мультимедиа-пакета FFmpeg 2.8. Копия нового FFmpeg включена в базовую поставку MPlayer, что позволяет обойтись без доустановки зависимостей при сборке. Из-за грядущего изменения API текущая master-ветка и будущие выпуски FFmpeg в MPlayer 1.2 поддерживаться не будут, для работы с ними необходимо использовать HEAD-ветку MPlayer из SVN-репозитория проекта. Как следствие обновления FFmpeg в MPlayer 1.2 появилась поддержка множества новых форматов и кодеков, в том числе HEVC, VP9, Prores, Opus, WebP, JPEG2000, улучшенных декодеров AAC и DTS. Для просмотра потокового контента, использующего протокол RTSP, осуществлён переход по умолчанию на реализацию RTSP из FFmpeg. Реализована возможность использования средств аппаратного ускорения VDPAU для декодирования видео HEVC/h.265. Добавлена возможность использования аппаратного ускорения VDA на платформе OS X. VAAPI формально пока не поддерживется, но может быть использован для ускорения декодирования H.264 через обёртку для VDPAU. Внесены многочисленные улучшения в графический интерфейс, расширена функциональность, связанная с управлением просмотром TV и DVB. Представлена подборка новых тем оформления. Для сборки GUI по-прежнему достаточно GTK+ 2.x. Из базовой поставки исключены библиотеки libdvdnav и libdvdcss, которые теперь нужно использовать как внешние зависимости.
