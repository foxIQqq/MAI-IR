Кандидат в релизы инсталлятора Debian 7.0

Процесс подготовки Debian 7.0, после двух лет разработки, перешёл на финишную прямую - началось тестирование кандидата в релизы программы установки Debian GNU/Linux для грядущего выпуска "Wheezy" (7.0). В настоящее время в Wheezy насчитывается 230 блокирующих релиз ошибок. Если отбросить проблемы, для которых уже почти готовы исправления или которые устраняются элементарно, для выхода релиза остаётся исправить 66 ошибок. Основные изменения, по сравнению с четвёртой бета-версией инсталлятора Debian 7.0: Добавлена большая порция новых драйверов: 8021q, adm8211, at76c50x-usb, b43legacy, bnx2fc, cxgb4, cxgb4vf, fnic, igbvf, int51x1, isci, iwl4965, ixgbevf, libertas_tf_usb, micrel, mlx4_en, mwifiex_pcie, mwl8k, orinoco_usb, pata_piccolo, pch_gbe, pmcraid, prism2_usb, qlge, r8187se, r8192e_pci, r8712u, rtl8192ce, rtl8192cu, rtl8192de, rtl8192se, smsc75xx, smsc9420, smsc95xx, tehuti, ums-eneub6250, ums-realtek, vt6656_stage, vxge; Обеспечена поддержка идентификаторов новых моделей беспроводных устройств на базе чипов Ralink; Добавлена поддержка Ethernet-адаптера Lenovo 10/100 с интерфейсом USB; Для сессии GNOME 3 включена поддержка экранного ридера Orca, улучшена поддержка синтеза речи; Улучшена установка на системы с EFI. Переработаны меню GRUB, используемые при загрузке в режиме UEFI. Разрешение экрана при загрузке с UEFI по умолчанию изменено на 800x600; Конфигуратор сети теперь сохраняет конфигурацию проводной и беспроводной сети для network-manager, если данные настройки обнаружены в уже установленной системе. Иначе, осуществляется настройка проводной сети через ifupdown; Устранены проблемы с монтированием procfs в GNU/kFreeBSD; Обзор возможностей, запланированных для Debian 7.0, можно посмотреть здесь, список новшеств альфа-версии инсталлятора, первой бета-версии, второй бета-версии, третьей бета-версии, четвёртой бета-версии.
