Представлен сайт debtags.debian.net

Анонсировано введение в строй сайта debtags.debian.net, на котором реализован новый web-интерфейс для навигации, редактирования и рецензирования системы каталогизации пакетов по тегам (debtags). Среди наиболее интересных возможностей debtags.debian.net называется поддержка поиска пакетов, для которых привязаны теги, прототип интерфейса для навигации по каталогу тегов, статистика присвоения тегов и публичный API для интеграции с внешними системами. Новый web-интерфейс написан на Python и базируется на Python-фреймворке Django и JavaScript-библиотеке JQuery.
