Подборка первоапрельских шуток 2010 года

Подборка первоапрельских шуток: Разработчики Midnight Commander выпустили экспериментальную версию консольного файлового менеджера Midnight Commander 5, которая включает в себя следующие изменения: переход на лицензию BSD; реализация поддержки плагинов (совместимых с Far); вместо технологий UNIX и POSIX код будет базироваться на JVM/.NET; поддержка multitouch-интерфейса ввода; плагин для "облачной" навигации с поисковой панелью инструментов Google; плагин для работы в социальной сети Facebook. Загрузить MC 5.0 можно здесь (код написан на C#, его можно без проблем запустить в Mono). Изображение на сайте kernel.org зеркально отразили (в CSS добавили "direction: rtl;"). В рамках проекта по написанию руководства пользователя Ubuntu 10.04 анонсирована подготовка аудиокниги "Getting Started with Ubuntu 10.04", озвучивать которую согласилась Марли Матлин Microsoft и Apple подписали соглашение, предусматривающее возможность запуска Windows 7 на компьютерах производства Apple и установки Mac OS X (Snow Leopard) на обычных ПК. На LOR изменена цветовая гамма сайта и заявлено об объединении с OpenNet. На OpenNet опубликовано подтверждение этой информации. Также на LOR появилась новость с планом разработки новой ветки Linux ядра 2.8; опубликовано интервью с Ларри Эллисоном в котором тот признался, что Oracle поглотил Sun с целью недопустить покупку Sun-ом компании Canonical; Компания Opera анонсировала выпуск версии веб-браузера для использования в космосе (доступен перевод на русский язык). Для доступа к сети задействован специальный протокол INTP (Interplanetary Network Transport Protocol), изображение может проецироваться на сейтчатку глаза, при работе с сенсорным экраном в скафандре браузер рассчитывает наиболее вероятную область на которую хочет нажать пользователь. Разработка полностью протестирована на пригодность использование в вакууме. Линус Торвальдс перешел на работу в компанию Microsoft на должность директора по развитию программных архитектур, где ему предоставили возможность создать принципиально новую операционную систему. Компания Google представила систему перевода человеческой речи в воспринимаемые животными звуковые сигналы (язык животных). Для платформы Android через каталог Android Market любой желающий может загрузить соответствующее приложение "Translate for Animals". Google поглотил компанию, разработавшую новый способ обогащения урана и планирует выйти на рынок атомной энергетики. Microsoft прекращает разработку браузерного движка Trident и планирует использовать движок текстового браузера Lynx в Internet Explorer 9.0. Приводится большой список достоинств Lynx, таких как кроссплатформенность, неподверженность уязвимостям и XSS-атакам, непревзойденная скорость рендеринга, возможность использования без мыши. В Google Docs кроме дискового пространства будет предоставляться место на складе для хранения реальных вещей. В Google Docs появились книги со стереоизображением, которые нужно читать надев специальные очки; По традиции рейтинг Linux-дистрибутивов на distrowatch.com отображен в обратном порядке; Проект KDE объявил о том, что начиная с KDE SC 5 будут выпускаться три различные сборки: Полностью свободная, карликовая и базовая (Ultimate, Dwarfed и Basic Freedom Edition). Первый вариант будет давать пользователю возможность полного изменения оформления в зависимости от личных предпочтений. Карликовая версия предоставит пользователю набор только базовых настроек, а базовая версия будет представлять из себя черный ящик и не даст менять какие-либо параметры системы. Апрельский номер PDF-журнала Arch Linux Magazine опубликован с зеркально отраженным текстом. В Google Code появился проект zfs-linux, авторы которого намерены написать с нуля реализацию ZFS для Linux, распространяемую под лицензией GPL. Представлены новые элементы брендинга Fedora, выполненные в фиолетовых тонах; Организация Apache Software Foundation опубликовала пресс-релиз в котором сообщила о согласовании сделки по продаже всей интеллектуальной собственности и активов компании Oracle за 1.5 млрд. долларов. На сайте форума phpBB появился анонс проекта ASPbb, в рамках которого будет подготовлен аналог phpBB 3, написанный на ASP.NET Разработчики открытой игры Tremulous анонсировали о кооперации с Microsoft в плане разработки нового игрового движка, который будет закрытый и будет распространяться через сервис XBox Live; Разработчики дистрибутива Linux Mint объявили о заключении партнерских соглашений с компаниями Mattel и Microsoft, создании "Barbie Edition" для предустановки дистрибутива на некоторые игрушки, программы для настройки будут переписаны на Mono, вместо APT будет задействован пакетный менеджер от компании Microsoft. Открытый LaTeX-редактор TexMakerX преобразился в LaTeXtrix (игра в тетрис); Библиотека libspotify для работы с музыкальным сервисом Spotify портирована для операционной системы CP/M; После длительного затишья ожил] ресурс kerneltrap.org, причем одна из новостей про nftables на самом деле опоздала на год. Представители PostgreSQL Core Team приняли решение уйти в будущих версиях PostgreSQL от использования SQL в пользу языка QUEL, соответствующий философии 'NoSQL'. Соответственно поменяется и название проекта - PostgreQUEL. Марк Шаттлворт окончательно утвердил перенос в Ubuntu 10.04 кнопок управления в левый верхний угол окна, причем кнопка закрытия окна теперь будет первой, затем будет кнопка минимизации и в конце раскрытия окна на весь экран. Во FreeBSD собираются удалить DNS-сервер BIND из базовой поставки. Разработчики Debian подготовили пакет "cpan", содержащий в себе все модули, доступные в архиве CPAN. "Проекты OpenBSD и NetBSD объединяются, новая система будет изменяться под именем NetOpenBSD или NoBSD", интересно, что эта шутка была перепечатана на полном серьезе второго апреля на сайте cnews.ru, при этом никакой ссылки на источник они как всегда не поставили, что породило вторичную волну "сенсаций" на страницах других отечественных компьютерных изданий. В течение дня текст новости будет пополнятся новыми первоапрельскими шутками. Ссылки на интересные первоапрельские розыгрыши просьба присылать в комментариях.
