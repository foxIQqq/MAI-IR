Фонд СПО опубликовал петицию с призывом опубликовать код Windows 7

В связи с прекращением 14 января поддержки Windows 7, Фонд свободного ПО адресовал компании Microsoft петицию с призывом перевести Windows 7 в категорию свободного программного обеспечения, чтобы дать возможность сообществу изучать и улучшать данную ОС. Отмечается, что Microsoft уже переводила некоторые свои программы в категорию СПО, а также, что раз поддержка уже завершена, то и терять Microsoft нечего. По мнению Фонда СПО, окончание поддержки Windows 7 предоставляет прекрасную возможность для Microsoft опубликовать исходный код, "замолив" тем самым грехи Windows 7, в числе которых упоминаются препятствование обучению, нарушение приватности и безопасности пользователей. Целью кампании является сбор по меньшей мере 7777 подписей (на момент написания новости уже собралось 5007 подписей). Обращение состоит из трёх пунктов: Перевод Windows 7 в категорию СПО. По мнению Фонда, жизненный цикл данной ОС не должен заканчиваться, Windows 7 все ещё может использоваться сообществом для обучения и получать улучшения в процессе совместной разработки. Уважать свободу и приватность пользователей, не переводить их насильно на новые версии Windows. Предоставление доказательств, что Microsoft действительно уважает пользователей и их свободу, вместо слов и маркетинговых материалов.
