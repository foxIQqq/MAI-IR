Вышла вторая бета-версия Ubuntu 11.10

Началось тестирование второй бета-версии Ubuntu 11.10 "Oneiric Ocelot". Загрузочные iso-образы с тестовой версией Ubuntu 11.10 можно получить на данной странице. На следующем этапе будет выпущен релиз, который запланирован на 13 октября. Основные новшества, добавленные в Ubuntu 11.10 Beta 2, по сравнению с первой бета-версией: В Центр установки приложений (Software Center) интегрирован сервис OneConf, который можно использовать для синхронизации установленных программ между компьютерами; До релиза планируется подготовить несколько новых iso-образов для архитектуры ARM. Образ armel+ac100 для нетбука Toshiba ac100 доступен уже сейчас, а образ armel+mx5 для плат Freescale i.MX53 будет размещен в ближайшие дни; Обновлены интегрированные в дистрибутив компоненты последней тестовой версии GNOME (3.1.92), на базе которой 28 сентября будет выпущен релиз GNOME 3.2; Улучшена поддержка установки 32-разрядных библиотек и приложений в 64-разрядных системах; Ядро Linux обновлено до версии 3.0.4; Интерфейс для управления пакетами Synaptic и нелинейный редактор видео PiTIVi исключены из базовой поставки, но их можно установить из стандартных репозиториев; Очень много улучшений внесено в Kubuntu 11.10: декстоп-окружение KDE обновлено до версии 4.7.1; в дистрибутив интегрирован менеджер персональной информации KDEPIM 4.7, включая почтовый клиент Kmail 2; задействован новый набор пиктограмм для темы Oxygen; переработано оформление файлового менеджера Dolphin; расширены возможности виджета для настройки сетевых подключений; для управления пакетами по умолчанию теперь используются Muon Suite 1.2, включающий Muon Software Center и Muon Package Manager; В серверной редакции Ubuntu возобновлена возможность использования из коробки гипервизора Xen; Были добавлены в прошлых тестовых выпусках Ubuntu Server, но отмечены в примечании к beta 2 как требующие дополнительной проверки инструментарии Orchestra и Juju (Ensemble). В рамках Orchestra подготовлен набор основанных на Cobbler инструментов, позволяющих обеспечить быстрое развертывание управляющих сервисов, ориентированных для использования в серверной инфраструктуре предприятий. Например, подготовлен готовый комплект для обеспечения установки, развертывания и обновления программ, управления конфигурацией, хостинга, мониторинга и обеспечения слаженной работы всех компонентов облачной инфраструктуры (оркестровка). Фреймворк Juju (Ensemble) выполняет функции, сходные с пакетным менеджером APT, но расширенные контекстом облачных систем. Ensemble позволяет обеспечить запуск внутри обычных и облачных окружений сервисов, обслуживающих выполнение конкретных пользовательских задач. Используя Ensemble разные авторы могут независимо создавать свои сервисы и обеспечивать связь с этими сервисами через простой протокол управления конфигурацией.
