Вторая бета-версия Mageia 2, форка дистрибутива Mandriva

Представлена вторая бета-версия Linux-дистрибутива Mageia 2, в рамках которого независимым сообществом энтузиастов развивается форк проекта Mandriva. Отмечается, что основная функциональность дистрибутива уже сформирована - неделю назад проект перешёл на стадию полной заморозки от внесения новшеств и переключился на выявление и исправление ошибок. Релиз Mageia 2 намечен на 3 мая. Для загрузки доступны 32- и 64-разрядные DVD-сборки (3.5 Гб) и минимальный образ для установки по сети (40 Мб). Набор LiveCD на базе GNOME и KDE с различными наборами локализации по прежнему недоступен и будет подготовлен позднее, после решения некоторых проблем, вызванных миграцией на инструментарий Dracut. По сравнению с первой бета-версией добавлены следующие улучшения: Ядро Linux обновлено до версии 3.3-rc7; Продолжена интеграция системного менеджера systemd, который будет использоваться по умолчанию. Возможность загрузки с классической системой инициализации будет сохранена для тех, кто хочет повременить с переходом на systemd; В дистрибутив интегрирована вторая бета-версия звукового сервера PulseAudio 2.0. В финальную версию Mageia 2 планируется включить релиз PulseAudio 2.0, но если будут выявлены какие-то недопустимые проблемы, то предусмотрена возможность отката на прошлую версию PulseAudio. В качестве причины перехода на PulseAudio 2.0 называется желание задействовать добавленных в ветку ядра 3.3 возможностей по определению аудиоразъёма при подключении наушников, что даст возможность автоматически перенаправить звуковой поток и откорректировать громкость; В состав включены библиотеки и компоненты Nomachine NX 3.5.0.11. Кроме того, в состав включена система организации удалённого доступа к рабочему столу X2Go: x2goserver 3.1.0.0, x2goclient 3.99.1.0 и pyhoca-gui 0.1.0.10; Обновление графических окружений: KDE 4.8.1, GNOME 3.4-beta, свежая сборка LXDE (в том числе файловый менеджер Pcmanfm 0.9.10), обучающее окружение Sugar 0.95. В репозиториях также доступны для установки оконные менеджеры dwm, scrotwm, i3, pekwm, icewm, awesome, enlightenment (e17), openbox, fvwm2 и Window Maker; Обновление мультимедийных программ: редактор анимации Pencil-Planner 0.4.4b, Amarok 2.5, Cantata music player 0.3, kdenlive 0.8.2.1, mplayer (+ GUI gnome-mplayer, kmplayer, mplayer-gui, smplayer), vlc, xine (+ GUI kaffeine и xine-ui); Вместо MySQL предлагается использовать MariaDB 5.5. Обновлены пакеты с PostgreSQL - 8.4.10, 9.0.6, 9.1.2; Обновление серверного ПО: apache 2.2.22, postfix 2.8.9, drupal 7.11, mediawiki 1.16.5; Обновление системных компонентов и средств разработки: glibc 2.14.1, gcc 4.6.2, python 2.7.2/3.2.2, perl 5.14.2, ruby 1.8.7, rails 2.3.14, php 5.3.10, smarty 2.6.26/3.1.5, java openjdk-1.6.0.0, sun-java-1.6.0.26. IDE: eclipse 3.7.1, netbeans 7.0.1, anjuta 3.3.90, ninja-ide Python IDE 2.0; Игры. Аркадные: abuse-sdl, flaw, hedgewars, supertux, supertuxkart, virtualjaguar. Шутеры от первого лица: ioQuake3 1.36, Quake III Arena demo, Smokin' Guns 1.0, World of Padman 1.5, Urban Terror 4.1.1; Браузеры: Firefox, Chromium, Epiphany, Opera, Konqueror, Midori; Почтовые клиенты: Thunderbird, Kmail, Evolution, Claws mail.
