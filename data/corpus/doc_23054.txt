Вышел экспериментальный релиз GIMP 2.7.0 с изменением лицензии на GPLv3

Спустя несколько дней после публикации стабильного корректирующего релиза GIMP 2.6.7 вышел GIMP 2.7.0, первый релиз в экспериментальной ветке GIMP, выступающей в роли тестового полигона для формирования стабильной ветки GIMP 2.8. Начиная с данного выпуска кодовая база проекта переведена на использование лицензии GPLv3. Некоторые другие новшества: Улучшение интерфейса пользователя: Поддержка прямого ввода текста на холсте, без открытия промежуточного окна ввода; Возможность привязки ключевых слов (тегов) к кистям, текстурам и градиентам. Поддержка фильтрации, группировки и поиска элементов по сопоставленным с ними тэгами. База тэгов хранится в XML файле, что упрощает их обработку внешними приложениями; Полное разделение операций по сохранению и экспорту изображений: сохранить изображение отныне можно только в формате XCF, во все остальные форматы сохранение возможно через меню экспорта 'File->Export...'. Тем не менее будет предусмотрен режим быстрого сохранения в изначальный формат, например, когда нужно открыть jpeg, немного отредактировать и сохранить обратно в jpeg, при этом число промежуточных диалогов будет минимизировано; В функции оперирующих изменением размера, добавлена поддержка указания относительных величин, например, в процентах (50%) или через выражение (30in + 40px); Возможность группировки слоев, через логическое объединение слоев на основании их вклада в общее изображение. Поддерживается быстрое применение определенной операции к группе в целом, например, можно изменить уровень яркости целиком для группы; Улучшение работы движка формирования динамических кистей. Поддержка динамического вращения, наклона или масштаба кистей; Ведется работа по пересмотру представленного по умолчанию набора кистей и других ресурсов; Добавлены плагины для загрузки изображений в формате JPEG2000, файлов с курсорами мыши для X11 и raw-данных с 16-битным представлением цвета (RGB565). Плагин для экспорта палитры для CSS, PHP, Python, txt и Java; Продолжение перевода кода на GEGL (Generic Graphics Library), что позволит увеличить глубину цвета на канал (сейчас только 8 бит на канал ), использовать цветовые схемы CMYK и LaB. В скором времени ожидается появление поддержки слоев, изображение в которых представлено в векторном виде; Некоторые планы на будущее (скорее всего не будет включено в 2.8): Вейвлетные фильтры, которые формируют для обрабатываемого изображения заданное число слоев, в каждом из которых сохранен вариант с различным уровнем детализации (от контуров к мелочам); Новый однооконный интерфейс, базирующийся на концепции вкладок (табов) Реализация слоев выполняющих корректировку и задающих определенные эффекты. Объединение в один инструмент функций проекции, искажения, поворота и масштабирования. Дополнение: Доступен подробный обзор Gimp 2.7.0 на русском языке.
