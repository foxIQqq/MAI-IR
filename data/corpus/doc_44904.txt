Выпуск Remix OS 3.0, Android-окружения для ПК

Доступен выпуск проекта Remix OS 3.0, в рамках которого развивается редакция платформы Android, оптимизированная для использования на рабочем столе персональных компьютеров и ноутбуков. Разработкой занимается компания Jide Technology, основанной несколькими бывшими инженерами из Google, совместно с разработчиками из сообщества Android-x86. Сборки Remix OS сформированы в виде 64- и 32-разрядных Live-окружений (750MB и 600MB), пригодных для установки на жесткий диск или USB Flash. Новый выпуск примечателен обновлением компонентов платформы Android до версии Android 6.0 "Marshmallow". В Remix OS 3 также представлены новые возможности многооконного режима - в заголовок окна добавлены кнопки для разворачивания видео на полный экран и для автоматической корректировки размера окна для полного отображения контента приложения. Захват для изменения размера окна теперь может производится не только за нижний правый угол, но и в любом месте вдоль края окна. В мастере начальной настройки появилась возможность выбора дополнительных приложений для установки. Расширена поддержка GPU NVIDIA и AMD. Добавлена возможность выполнения приложений и игр, использующих графический API Vulkan. Представлен экспериментальный набор для любителей компьютерных игр (Gaming Toolkit), в котором пока находится только утилита для настройки раскладки клавиатуры. Общие особенности Remix OS: Кардинальная переработка интерфейса Android в плане замены методов работы, специфичных для мобильных устройств, на решения, традиционные для рабочего стола ПК. Полноценная поддержка управления с использованием мыши и клавиатуры; Поддержка многооконного режима и возможность произвольного изменения размера окон. Наличие кнопок раскрытия и сворачивания окон; Наличие меню приложений, системного лотка и панели инструментов. Вывод контекстного меню, показываемого при клике правой кнопкой мыши; Наличие традиционного файлового менеджера. Опциональная поддержка установки приложений из Google Play (требуется установка пакета с приложениями Google); Наличие системы автоматической установки обновлений. Адаптация системы вывода уведомлений для больших экранных разрешений. Вывод уведомлений производится в форме боковой всплывающей панели.
