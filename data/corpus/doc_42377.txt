Бета-выпуск июньского набора ALT Linux p7 starterkits

Доступен предварительный комплект ежеквартальных стартовых наборов Седьмой платформы Альт Линукс. Желающие приглашаются к проверке, критике и доработке выпуска, который запланирован на 12 июня 2015 года. Изменения за весну в целом невелики: обновление пакетной базы, в том числе Firefox 38.0.1; переработку запуска сессии IceWM; добавление в rescue цели загрузки "под iKVM", при котором автоматически поднимается сеть по DHCP, генерируется пароль root и данные для доступа выдаются в консоль; вариант server-ovz пополнился утилитой vzstats с возможностью включить её при желании во время установки наравне с уже предлагавшимися группами пакетов. Список вариантов с марта не изменился: устанавливаемые LiveCD (cinnamon, e17, gnome3, gnustep, icewm, kde4, lxde, lxqt, mate, tde, wmaker, xfce) с ядрами 3.14/3.19 и системами инициализации (systemd, sysvinit); классические инсталяторы (jeos, server, server-hyperv, server-ovz, sysv-tde) с sysvinit; образ диска для виртуальных машин (vm-net) и шаблон контейнера OpenVZ (ovz-generic); инструментальный сборочный LiveCD (builder); текстовый спасательный/исследовательский образ (rescue). Анонсированная ранее вариация на тему UPR достигла бета-состояния и, возможно, в таком статусе будет опубликована вслед за основным комплектом (решалась задача предоставления окружения, где пользователь не может организовать доступ к локальным фиксированным носителям информации либо утечку через сетевые средства - "доверенная пишмашинка/слайдоскоп"; какие-либо усилия для обеспечения работы криптофлэшек, тем более совместимых с UPR в части TrueCrypt, на данный момент не предпринимались). Стартовые наборы — сборки на базе стабильного репозитория ALT Linux, предназначенные для опытных пользователей и предоставляющие им возможность быстро и удобно ознакомиться с ещё не опробованными DE/WM либо развернуть очередную машинку с минимальным расходом времени на установку и настройку под себя. Технология "под капотом" предназначена для простого создания производных и открыта; данный комплект собран по тегу 1.1.66-alt1.
