Релиз офисного пакета ONLYOFFICE Docs 7.1

Опубликован выпуск ONLYOFFICE DocumentServer 7.1 с реализацией сервера для online-редакторов ONLYOFFICE и организации совместной работы. Редакторы можно использовать для работы с текстовыми документами, таблицами и презентациями. Код проекта распространяется под свободной лицензией AGPLv3. Одновременно сформирован выпуск продукта ONLYOFFICE DesktopEditors 7.1, построенного на единой кодовой базе с online-редакторами. Десктоп-редакторы оформлены в виде приложений для рабочего стола, которые написаны на JavaScript с использованием web-технологий, но объединяют в одном наборе клиентские и серверные компоненты, оформленные для самодостаточного использования на локальной системе пользователя, без обращения к внешнему сервису. Для совместной работы на своих мощностях также можно использовать платформу Nextcloud Hub, в которой обеспечена полная интеграция с ONLYOFFICE. Готовые сборки сформированы для Linux, Windows и macOS. В ONLYOFFICE заявлена полная совместимость с форматами MS Office и OpenDocument. Среди поддерживаемых форматов: DOC, DOCX, ODT, RTF, TXT, PDF, HTML, EPUB, XPS, DjVu, XLS, XLSX, ODS, CSV, PPT, PPTX, ODP. Предусмотрена возможность расширения функциональности редакторов через плагины, например, доступны плагины для создания шаблонов и добавления видео с YouTube. Готовые сборки сформированы для Windows и Linux (deb- и rpm-пакеты). Основные новшества: Предоставлена поддержка установки ONLYOFFICE на системы с архитектурой ARM. Опубликована отдельная сборка ONLYOFFICE Docs для ARM. Предложен новый просмотрщик документов в форматах PDF, XPS и DJVU, который отличается высокой производительностью и обработкой всех операций на стороне клиента. Из особенностей нового просмотрщика также отмечается боковая панель с эскизами страниц документа, панель навигации, режим ручного выделения областей в документе, секция с информацией о файле и возможность перехода по внешним и внутренним ссылкам. Во все редакторы добавлено новое меню для вставки и изменения фигур. Добавлены пиктограммы для всех предлагаемых фигур. Обеспечен показ списка ранее использованных фигур. Добавлен режим редактирования геометрии фигур через расстановку мышью опорных точек. Изменён инструмент для выбора направления заливки фигуры градиентом. На пиктограмме заливки градиентом обеспечено отображение выбранных цветов. Предоставлена возможность обрезки изображения по контуру фигуры. Добавлена поддержка новых типов диаграмм: пирамидальных, столбчатых, цилиндрических и конусных. В левую боковую панель добавлена поддержка сортировки комментариев по группам. При вводе пароля защиты файла добавлена опция для показа символов пароля во время ввода. Добавлена поддержка объектов SmartArt, работающая без их преобразования в группы объектов. Реализован вывод уведомления об обрыве и восстановлении соединения. В редакторе документов добавлена поддержка преобразования файлов PDF/XPS в редактируемый документ в формате DOCX. Предоставлена возможность принятия и отклонения изменений через контекстное меню. Добавлена поддержка указания спецсимволов при поиске документов. Добавлена новая вкладка инструментов View, предоставляющая настройки отображения документов и презентаций, такие как тема оформления, позиционирование документа, уровень масштабирования, показ панели инструментов и статусной строки. Изменения в табличном процессоре: Реализован интерфейс для предпросмотра таблиц перед выводом на печать. Во вкладку View добавлены опции для включения строки состояния, объединения строки состояния и панели управления электронной таблицей, постоянного отображения панели инструментов, выбора темы оформления интерфейса и отображения теней для панелей. В диалог выбора формата чисел в ячейках добавлена секция с символами валют. При вводе формул обеспечен вывод всплывающих подсказок, предлагающих подходящие варианты формул. В диалоге настройки параметров импорта в форматах TXT и CSV добавлена опция для выбора символов определения начала и конца блока с текстом (кавычки). Добавлена поддержка открытия файлов в формате XLSB. Добавлено контекстное меню для перемещения электронных таблиц. Предоставлена возможность открытия и закрытия групп в режиме просмотра и комментирования. Изменения в редакторе презентаций: Добавлена поддержка вставки анимации в презентацию. В панель инструментов добавлены новые вкладки Animation и View. В меню предоставлены инструменты для дублирования слайдов и перемещения слайдов в начало и конец списка. Во вкладке Insert появилась возможность вставки недавно использованных фигур. Добавлена поддержка масштабирования форм. В варианте редакторов и просмотрщиков для мобильных устройств реализована поддержка тёмной темы оформления и добавлена кнопка для показа списков в электронной таблице.
