Релиз дистрибутива Arch Linux 2011.08.19

После 15 месяцев разработки представлен обновленный установочный образ дистрибутива Arch Linux 2011.08.19. Представленный выпуск имеет смысл только для формирования новых установок, так как дистрибутив Arch Linux разрабатывается с использованием модели непрерывных обновлений (Rolling-release), при которой обновления пакетов выходят постоянно и пользователь в любой момент имеет возможность перейти на последние версии программ без ожидания формирования очередного релиза дистрибутива. Таким образом, данный выпуск интересен не новыми версиями программ, а обновленным инсталлятором и набором утилит для формирования сборок дистрибутива. Установочные сборки дистрибутива доступны для архитектур i686 и x86_64 в двух вариантах: для установки с задействованием сети (181 Мб) и для автономной установки (371 Мб). Кроме того, опубликован комбинированный установочный iso-образ с набором пакетов для 32- и 64-разрядных архитектур. Все представленные iso-образы оформлены в гибридном виде, пригодном для установки как с CD, так и с USB Flash, а также поддерживают загрузку по сети. Особенности нового выпуска: Установочный образ построен на слепке core-репозитория от 19 августа, включающем Linux-ядро 3.0, набор компиляторов gcc 4.6.1, пакетный менеджер pacman 3.5.4, системную библиотеку glibc-2.14, пакеты mkinitcpio 0.7.2, initscripts 2011.07.3 и netcfg 2.6.7. Пакеты типичных пользовательских приложений в других репозиториях находятся в процессе непрерывного обновления, т.е. поставляются самые свежие на момент установки программы; Улучшения программы установки AIF: Экспериментальная поддержка файловых систем Btrfs и nilfs2; Поддержка загрузчика syslinux, наряду с grub; Изменение форматов конфигурации для обеспечения поддержки нового rc.conf и ядра Linux 3.0; Увеличена гибкость при выборе источников установки, теперь можно выбрать сразу несколько локальных или удаленных репозиториев; В процессе установки пакетов теперь отображаются их описания; По умолчанию активирована возможность запуска в отладочном режиме и в режиме ведения полных логов. Добавлен скрипт /arch/report-issues; Автоматический выбор источников в зависимости от используемого установочного образа (core или netinstall); Удалена поддержка tcp_wrappers из-за расхожения с правилами разработки Arch Linux; Улучшения в Archiso, инструментарии для формирования сборок: Вместо aufs2 для задействован dm-snapshot; Все файлы, необходимые для загрузки с USB-накопителей, теперь устанавливаются в директорию /arch; Возможность изменения имени экспорта для NBD; Возможность использования консоли через последовательный порт (опция для ядра - console); Возможность запуска пользовательского скрипта после загрузки (опция для ядра - script); Поддержка проверки целостности компонентов, через сверку контрольных сумм (опция для ядра - checksum=y); Возможность примонтировать iso-образ в режиме loopback; Из загрузочного меню удален x86test, вместо него добавлен HDT (Hardware Detection Tool) из состава Syslinux, позволяющий на раннем этапе загрузки сформировать полное описание состава текущего аппаратного обеспечения, до начала загрузки ОС; Поддержка загрузки с предварительным копированием содержимого образа в RAM-диск; Использование сжатия XZ для SquashFS и initramfs; Из состава установочного диска удалены пакеты: aufs2, joe, linux-atm, mtools, ndiswrapper, sqlite3, squashfs-tools, tcp_wrappers, tiacx. Многие обособленно поставляемые прошивки теперь заменены на единый пакет linux-firmware. Вместо dcron задействован cronie, вместо heimdal - krb5; По сравнению с прошлым выпуском в состав сборки добавлены такие пакеты, как: btrfs-progs-unstable, curl, dnsmasq, hdparm, nbd, nilfs-utils, openconnect, rsync, tre, vpnc, wireless-regdb, wpa_actiond. Расширены возможности libui-sh, библиотеки для построения пользовательского интерфейса инсталлятора.
