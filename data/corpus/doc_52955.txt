Новая версия российского дистрибутива Astra Linux Common Edition 2.12.29

Компании ООО "РусБИТех-Астра" опубликовала выпуск дистрибутива Astra Linux Common Edition 2.12.29, построенного на пакетной базе Debian GNU/Linux 9 "Stretch" и поставляемого с собственным проприетарным рабочим столом Fly (интерактивная демонстрация), использующим библиотеку Qt. Iso-образы для загрузки пока не доступны, но предложены бинарный репозиторий и исходные тексты пакетов. Дистрибутив распространяется в рамках лицензионного соглашения, которое накладывает ряд ограничений на пользователей, в частности, запрещены коммерческое использование, декомпиляция и дизассемблирование продукта. Основные изменения: Добавлено приложение fly-admin-ltsp для создания инфраструктуры работы с "тонкими клиентами" на базе сервера LTSP (создание сервера и генерирование образов клиентов); Добавлено приложение fly-admin-repo для создания собственных репозиториев c deb-пакетами; Добавлено приложение fly-admin-sssd-client для ввода в домен Active Directory с использованием системного сервиса sssd, позволяющего обращаться к удалённым механизмам авторизации; Предложен набор утилит Astra OEM Installer для облегчения OEM-установки ОС через настройку системы при первом старте (задание имени администратора и пароля, часового пояса и доустановки необходимых компонентов); Добавлен сервис для создания цифровых подписей документов и проверки электронной подписи с применением КриптоПро CSP (fly-csp); Добавлена утилита fly-admin-touchpad для настройки сенсорной панели в ноутбуках; В меню "Пуск" и "Панель управления" добавлен поиск по элементам, который работает независимо от наличия соответствующего пункта в меню; Обновлены реализация и оформление средств для группировки приложений на панели задач, добавлена возможность закрытия группы окон; Реализована настраиваемая группировка значков приложений в системном лотке. В правой части панели задач появилось поле "Свернуть все"; Реализована настройка открытия ярлыков по одинарному щелчку на рабочем столе; Добавлена настройка включения вертикальной синхронизации для борьбы с тирингом; В файловом менеджере добавлена возможность отображения размеров файлов в байтах, оптимизирована работа с большим количеством файлов в каталоге, появилась возможность работы с шаблонами документов из контекстного меню, реализован быстрый переход к внешним ресурсам (ftp, smb) через адресную строку. Внесены улучшения в работу с SMB-ресурсами, в том числе, скорректировано определение размера свободного места для SMB-ресурсов; Полностью переработана утилита регулировки яркости fly-brightness, добавлена поддержка PowerDevil, реализовано автоматическое управление яркостью при наличии датчика освещения; В политике безопасности (fly-admin-local) реализован запрос пароля сразу после создания учётной записи пользователя; В утилиту управления принтерами (fly-admin-printer) добавлена возможность просмотра атрибутов задачи и принтера, значительно повышена стабильность работы; Улучшено поведение приложения для работы с камерой (fly-camera) при повороте устройства; В утилите для смены ориентации экрана (fly-orientation) переработано приложение по повороту устройства, добавлена поддержка нескольких экранов, добавлена калибровка сенсоров и реализован выбор ориентации по умолчанию; В менеджер входа (fly-qdm) добавлена экспериментальная поддержка новой темы; В интерфейс обновление системы (fly-update-notifier) добавлена возможность отложить напоминание об обновлении; В fly-admin-samba добавлены настройки для работы со списками контроля доступа, а также небольшие функциональные изменения; Реализован новый сервис двухфакторной аутентификации на базе libpam-csp и csp-monitor; В fly-xkbmap добавлена возможность настройки более 2 раскладок клавиатуры; В мобильной сессии доработан диалог выбора файлов, улучшена работа с контактами; Произведена адаптация для планшетов MIG Т10 на процессорной архитектуре x86_64; В репозиторий добавлены пакеты: audacious 3.7.2 clang-9 9.0.1 conky 1.10.8 csp-monitor 0.0.1 golang-1-10 1.10.4 gcc-mozilla 7.5.0 nvidia 440 pam-usbguard-astra 1.3 tlp 1.1 x2goclient 4.1.2 stoken 0.91 vagrant-libvirt 0.0.37 xfsdump 3.1.6 xmms2 0.8 kolourpaint 4:19.12 icedtea 1.7.2 z3 4.4.1 и другие Обновлено более 300 пакетов, из них более 90 из состава графической оболочки Fly, включая fly-wm (до версии 2.30.4) и fly-fm (до версии 1.7.39). В том числе обновлены: ansible 2.7.7 astra-openvpn-server 0.3.02 bsign 1.1.7 chromium 80 firefox 72 hplip 3.20.3 freeipa 4.6.4-24 lighttpd 1.4.53 libreoffice 6.3.5 libsane 1.0.27 libgost-astra 0.0.19 live-build-astra 0.4.19 lkrg 0.7 lxc 3.1.0 linux kernel 4.15.3-2 network-manager 1.10.14 nss 3.45 opensc 0.19 openssl 1.1.1d puppet-agent 6.12 puppetserver 6.9 qmmp 1.3.7 remmina 1.3.3 sqlite3 3.30.1 vagrant 2.0.2 vlc 3.0.10
