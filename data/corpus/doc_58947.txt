Выпуск дистрибутива 4MLinux 42.0

Представлен выпуск 4MLinux 42.0, минималистичного пользовательского дистрибутива, не являющегося ответвлением от других проектов и использующего графическое окружение на базе JWM. 4MLinux может использоваться не только в качестве Live-окружения для воспроизведения мультимедийных файлов и решения пользовательских задач, но и в роли системы для восстановления после сбоев и платформы для запуска серверов LAMP (Linux, Apache, MariaDB и PHP). Для загрузки подготовлена два iso-образа (1.2 ГБ, x86_64) с графическим окружением и с подборкой программ для серверных систем. В новой версии: Обновлены версии пакетов: ядро Linux 6.1.10, LibreOffice 7.5.2, AbiWord 3.0.5, GIMP 2.10.34, Gnumeric 1.12.55, Firefox 111.0, Chromium 106.0.5249.91, Thunderbird 102.8.0, Audacious 4.3, VLC 3.0.18, SMPlayer 22.7.0, Mesa 22.2.3, Wine 8.3, Apache httpd 2.4.56, MariaDB 10.6.12, PHP 8.1.17, Perl 5.36.0, Python 2.7.18, Python 3.10.8, Ruby 3.1.3. В число доступных для загрузки дополнительных пакетов включены графический редактор Krita и игра Hex-a-Hop. Улучшена поддержка различных форматов изображений, видео и звука. В базовую поставку включены мультимедийные проигрыватели AlsaPlayer, Baka MPlayer, GNOME MPlayer, GNOME MPV и mp3blaster. В качестве мультимедийного проигрывателя по умолчанию задействован XMMS.
