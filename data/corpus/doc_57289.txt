Выпуск дистрибутива Nitrux 2.2 с рабочим столом NX Desktop

Опубликован выпуск дистрибутива Nitrux 2.2.0, построенного на пакетной базе Debian, технологиях KDE и системе инициализации OpenRC. Дистрибутив развивает собственный рабочий стол NX Desktop, который представляет собой надстройку над пользовательским окружением KDE Plasma, а также фреймворк построения интерфейсов пользователя MauiKit, на основе которого развивается набор типовых пользовательских приложений, которые можно использовать как на настольных системах, так и на мобильных устройствах. Для установки дополнительных приложений продвигается система самодостаточных пакетов AppImages. Размер полного загрузочного образа составляет 2.5 ГБ, а сокращённого с оконным менеджером JWM - 1.4 ГБ. Наработки проекта распространяются под свободными лицензиями. Рабочий стол NX Desktop предлагает иное стилевое оформление, собственную реализацию системного лотка, центра вывода уведомлений и различных плазмоидов, таких как конфигуратор сетевых соединений и мультимедийный апплет для регулирования громкости и управления воспроизведением мультимедийного контента. Из приложений, созданных при помощи фреймворка MauiKit, можно отметить файловый менеджер Index (можно использовать и Dolphin), текстовый редактор Note, эмулятор терминала Station, музыкальный проигрыватель Clip, видеопроигрыватель VVave, центр управления приложениями NX Software Center и просмотрщик изображений Pix. Отдельно проектом ведётся разработка пользовательского окружения Maui Shell, которое автоматически адаптируется к размеру экрана и доступным методам ввода информации, и может применяться не только на настольных системах, но и на смартфонах и планшетах. Окружение развивает концепцию "Convergence", подразумевающую возможность работы с одними и теми же приложениями как на сенсорных экранах смартфона и планшета, так и на больших экранах ноутбуков и ПК. Maui Shell может запускаться как со своим композитным сервером Zpace, использующим Wayland, так и через запуск отдельно оболочки Cask внутри сеанса на основе X-сервера. Основные новшества Nitrux 2.2: Компоненты рабочего стола NX Desktop обновлены до версий KDE Plasma 5.24.5, KDE Frameworks 5.94.0 и KDE Gear (KDE Applications) 22.04.1. Пакет Mesa обновлён до ветки 22.2. Настройки KWin изменены для показа сетки открытых окон в обзорном режиме. Обновлены версии программ, включая Firefox 101 и LibreOffice 7.3.1.3. По умолчанию задействовано ядро Linux 5.17.12 с патчами Xanmod. Для установки также предложены пакеты с ванильными, Libre- и Liquorix-сборками ядра Linux. Обновлены пакеты с микрокодом amd64-microcode и linux-firmware. Добавлены дополнительные пакеты с прошивками для чипов Broadcom. В инсталлятор добавлена возможность шифрования всего диска. Реализована сборка двух новых ISO-образов, включающих проприетарные драйверы NVIDIA. Первый iso поставляется с версией драйверов 510.73.05, а второй - с версией 390.151, обеспечивающей поддержку устаревших видеокарт. Улучшена поддержка графического API Vulkan. Обновлён драйвер AMDVLK. В состав включена прослойка vkBasalt для улучшения качества графики в играх. Обновлены приложения из набора Maui Apps. Библиотеки MauiKit обновлены до выпуска 2.1.2. Обновлён Центр управления приложениями (NX Software Center). В дополнение к AppImageHub добавлен ещё один репозиторий пакетов в формате AppImage - AppRepo. Добавлено приложение Bonsai, написанное с использованием фреймворка Maui и предлагающее интерфейс для управления Git-репозиториями, напоминающий GitHub Desktop client. Добавлены утилиты для автоматического определения и настройки принтеров, а также обработчик на базе PolicyKit для настройки привилегий системы печати CUPS.
