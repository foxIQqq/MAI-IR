Sega выпустит игру Football Manager 2014 для платформы Linux

Miles Jacobson, создатель популярного спортивного симулятора Football Manager, сообщил в своём блоге планы, касающиеся будущего выпуска игры. Релиз Football Manager 2014 выйдет в октябре и будет доступен в версии для платформы Linux. Поставка игры будет осуществляться через сервис Steam. Кроме Windows и Mac OS X, игра будет официально поддерживать дистрибутив Ubuntu 12.04. Про остальные дистрибутивы Linux ничего не сообщается, тем не менее, неофициальная поддержка поставляемых через Steam игр успешно реализуется энтузиастами для большинства популярных дистрибутивов Linux.
