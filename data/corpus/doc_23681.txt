Новый релиз системы защиты от внешних угроз Astaro Gateway 7.500

Представлена новая версия системы Astaro Security Gateway, которая представляет из себя программно-аппаратное решение, основанное на собственном Linux дистрибутиве, сочетающем в себе механизмы защиты сети, фильтрации интернет-трафика, обеспечения безопасности переписки по электронной почте. Управление системой производится с единой консоли через веб-интерфейс. Astaro Security Gateway распространяется в трех вариантах: как аппаратное, программное или виртуальное устройство. Astaro Security Gateway Hardware Appliance – набор сетевых устройств на базе Интел-совместимых серверов. Astaro Security Gateway Software Appliance – программное обеспечение, которое может быть развернуто на любых серверах с процессорами Интел (или его аналогами), PC со следующими минимальными параметрами: Intel Pentium III 900 MHz, 512MB RAM, 10 GB HDD. Astaro Security Gateway Virtual Appliance – виртуальные машины для сред VMware. Компания Astaro представляет свои решения в данной области с 2000 года и является одним из лидеров в области интегрированных платформ безопасности, известных как XTM-устройства (eXtensible Threat Managment). Ранее данный продукт назывался Astaro Security Linux. Релиз 7.500 включает в себя новые функции и возможности, многие из которых появились благодаря заявкам клиентов компании. Например, новый модуль защиты от несанкционированного доступа использующий все преимущества многоядерных процессоров (устройств), модуль мониторинга канала связи, модуль Proxy с порталом аутентификации, возможность импортировать и экспортировать различные списки (например, черный список URL), улучшенная система формирования отчетов о работе устройств защиты сети, переработка DHCP-сервера, поддержка PIM-SM маршрутизации. Кроме того, добавлена возможность использования trustedsource.org для организации фильтра веб-ресурсов в сочетании с дополнительной проверкой разрешенных URL, исходя из пяти уровней безопасности (от "Надежный" до "Вредоносный").
