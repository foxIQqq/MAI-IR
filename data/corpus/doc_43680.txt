Релиз программы для рисования MyPaint 1.2.0

После трёх лет разработки вышла новая версия специализированной программы для цифровой живописи с использованием планшета или мыши - MyPaint 1.2.0. Программа распространяется под лицензией GPLv2, разработка ведётся на языках Python и C++ с использованием тулкита Gtk+. Статус подготовки бинарных пакетов для Linux, Windows и OS X можно проследить на данной странице. Для Ubuntu поддерживается специальный PPA-репозиторий. MyPaint может использоваться художниками, занимающимися цифровой живописью и в некоторых применениях способен выступить конкурентом общепризнанным проприетарным программам для рисования Corel Painter и Artrage. Программа имеет простой, удобный в работе интерфейс и не позиционируется как графический редактор для обработки изображений. MyPaint имеет большой набор кистей качественно симулирующих настоящие художественные инструменты, такие как карандаш, масляные краски, акварель, мастихин и другие. Одна из интересных особенностей программы — это бесконечный холст, который можно прокручивать и масштабировать. Основные улучшения: Поддержка GTK+3; Новый инструмент для нанесения сглаженных контуров (smooth-stroke inking); Новый инструмент заливки "Flood Fill"; Автоматическое создание резервных копий для редактируемых файлов c возможностью восстановления при запуске; Улучшение режимов определения симметрии (symmetry-setting) и рамок (frame-setting); Новое оформление рабочего стола: две боковые панели и две перемещаемые верхние панели со вкладками; Поддержка плавной прокрутки и панорамирования; Новый набор кистей; Новая панель для просмотра истории выбора кистей и цвета; Поддержка группировки слоёв; Поддержка векторных слоёв, которые можно редактировать во внешнем приложении (рекомендован Inkscape); Новые запасные типы слоёв: не-PNG картинки и данные; В настройках рамки появилась новая команда для обрезания слоя; Новые режимы слоёв с реализацией различных шаблонов; Новые фильтры: оттенки серого и имитация дихроматического зрения; Новые сочетания колеса выбора цвета: Red/Yellow/Blue, Red-Green/Blue-Yellow. Применение по умолчанию тёмной темы оформления; Проведение чистки пиктограмм и курсоров; Возможность отключения проблемных устройств в настройках; Многие типы операций с изображениями теперь выполняются в фоне; Улучшена поддержка Windows.
