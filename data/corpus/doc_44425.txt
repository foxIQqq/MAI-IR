Новый выпуск Raspbian, дистрибутива для Raspberry Pi

Увидел свет выпуск дистрибутива Raspbian 2016-05-10, официально рекомендуемого для использования на Raspberry Pi и специально оптимизированного для данной платформы (сборка для ARMv6 с расширениями "hard float"). Размер образа для записи на SD-карту составляет 1.3 Гб. Для установки из репозиториев доступно около 35 тысяч пакетов. В выпуске реализованы новые возможности для поддержки аппаратных особенностей новой платы Raspberry Pi 3, которая поставляется со встроенными Wi-Fi 802.11n, Bluetooth 4.1 и Bluetooth Low Energy (BLE). На системном уровне данные возможности поддерживались изначально, но на рабочем столе недоставало удобных средств для настройки Bluetooth. В Raspbian 2016-05-10 в состав рабочего стола интегрирован написанный с нуля плагин для панели lxpanel, который размещается слева от пиктограммы настройки сети. Интерфейс предоставляет средства для поиска других устройств, их добавления, удаления и сопряжения с текущей системой. Из временных ограничений текущего выпуска отмечается возможность настройки подключения только звуковых и HID-устройств, таких как беспроводные клавиатуры и мыши. Для прошлых моделей Raspberry Pi интерфейс позволяет организовать работу при подключения внешних Bluetooth-брелоков. При подключении звуковых Bluetooth-устройств, таких как беспроводные колонки или гарнитуры, в интерфейсе настройки звука предоставляется возможность выбора данных устройств для вывода звука по умолчанию. Другие новшества: Подготовлено приложение для копирования SD-карт, которое может применяться в качестве инструмента для резервного копирования начинки текущей системной SD-карты или клонирования окружения на карту большего размера. Карта на которую осуществляется запись подключается через внешний USB SD card writer (запись на внутреннюю карту запрещена чтобы не повредить систему). В базовую поставку включена библиотека pigpio, предоставляющая функции для доступа к GPIO из программ на Python, C и других языков. Программы, использующие pigpio, не требуют получения повышенных привилегий для работы с GPIO (например, Scratch теперь можно запускать без sudo); В состав включен текстовый редактор Geany, обладающий такими возможностями как вкладки, подсветка синтаксиса и автоматическая расстановка отступов; Новый диалог завершения работы; В настройках устройств ввода добавлена поддержка определения задержки для двойного клика мышью; В конфигураторе Raspberry Pi добавлена поддержка включения/выключения отдельных интерфейсов и включения/выключения доступа к демону pigpio; В контекстное меню корзины добавлена опция для очистки корзины; Добавлена горячая клавиша Ctrl-Alt-T для открытия терминала; После записи образа Raspbian при первой загрузке файловая система автоматически будет расширена для занятия всего свободного пространства накопителя; Обновлены версии многих штатных приложений, включая Scratch, Sonic Pi, Node-RED, BlueJ и PyPy; В дистрибутиве задействовано ядро Linux 4.4 c применением механизма device tree для определения оборудования.
