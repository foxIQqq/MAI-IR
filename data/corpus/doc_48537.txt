Выпуск проприетарного драйвера NVIDIA 396.24

Компания NVIDIA представила первый выпуск новой стабильной ветки проприетарного драйвера NVIDIA 396.24. Драйвер доступен для Linux (ARM, x86_64), FreeBSD (x86_64) и Solaris (x86_64). Начиная с ветки 396.x выпуск 32-разрядных драйверов прекращён и поддержка новых GPU теперь осуществляться только в 64-разрядных драйверах. Основные новшества: Добавлена поддержка GPU GeForce GTX 1050 с Max-Q Design, Quadro GV100, Quadro P3200, Quadro P4200, Tesla V100-SXM2-32GB, Tesla V100-PCIE-32GB, Tesla V100-DGXS-32GB и Tesla V100-FHHL-16GB; Добавлена поддержка X.Org xserver ABI 24 (xorg-server 1.20); Для API Vulkan добавлен новый оптимизированный компилятор промежуточного представления шейдеров SPIR-V, примечательный уменьшением потребления памяти и сокращением времени компиляции шейдеров. Новый компилятор включен по умолчанию, для отключения можно воспользоваться переменной окружения __GL_NextGenCompiler=0; Добавлена поддержка 32-разрядных доменов PCI, необходимых для использования драйвера NVIDIA в запущенных под управлением гипервизора Hyper-V виртуальных машинах с Linux, в которых применяется драйвер pci_hyperv; В nvidia-settings добавлен раздел с информацией о EGL и обновлена страница настройки SLI Mosaic; Обновлён инсталлятор, добавлена поддержка определения частичных установок libglvnd (когда присутствует libglvnd, но нет libOpenGL.so); Увеличена производительность Vulkan API vkAllocateMemory(), vkBindBufferMemory() и vkBindImageMemory(); Для X-сервера реализована новая опция "AllowExternalGpus" для настройки съёмных GPU.
