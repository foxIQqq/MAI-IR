Выпуск GNU wget 1.17 с поддержкой FTPS и HSTS

Состоялся релиз GNU Wget 1.17, программы для автоматизации загрузки контента с использованием протоколов HTTP и FTP. В новом выпуске: Поддержка FTPS; Поддержка механизма HSTS (HTTP Strict Transport Security) для автоматической трансляции ссылок в HTTPS. Управление включением производится через опции "--hsts" и "--hsts-file"; Поддержка передачи ссылок через файлы в формате metalink. Для задания файлов добавлены опции "--input-metalink" и "--metalink-over-http"; Поддержка опции "--if-modified-since" для установки соответствующего HTTP-заголовка; Прекращена поддержка отката от пассивного FTP-режима к активному из соображений обеспечения приватности; Добавлена опция "--rejected-log" для записи отклонённых запросов в отдельный лог-файл; В режиме "--spider" запрещены загрузка и сохранение файлов, если запрос привёл к выводу кода ошибки; Добавлена опция "--convert-file-only" для конвертации только части URL, содержащей имя файла.
