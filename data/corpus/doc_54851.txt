Выпуск программы для управления фотографиями digiKam 7.2

Опубликован релиз программы для управления коллекцией фотографий digiKam 7.2.0, развиваемой проектом KDE. Программа предоставляет исчерпывающий набор инструментов для импорта, управления, редактирования и публикации фотографий, а также изображений с цифровых камер в формате raw. Код написан на языке C++ с использованием Qt и библиотек KDE, и распространяется под лицензией GPLv2. Установочные пакеты подготовлены для Linux (AppImage, FlatPak), Windows и macOS. Среди добавленных улучшений: В движке для распознавания лиц и инструменте устранения эффекта красных глаз задействована новая модель машинного обучения (Yolo), более качественно определяющая лица на изображениях со сложным ракурсом съёмки. Повышена скорость обработки данных и реализована возможность распараллеливания операций. Из базовой поставки удалены файлы с данными модели машинного обучения, которые теперь загружаются во время работы. Модернизирован графический интерфейс для работы с лицами и привязки к ним тегов, а также связанные с ним виджеты. Улучшен процесс управления альбомом фотографий, расширены возможности группировки информации, ускорена работа движка для фильтрации вывода по маске, оптимизировано отображение свойств и улучшена поддержка извлекаемых носителей. В бинарные сборки добавлена утилита для проверки наличия обновлений с возможностью их автоматической загрузки и установки. Значительно улучшены сборки для macOS. Проведена оптимизация кода для работы с БД и схемы хранения, используемые для поиска, хранения метаданных, распознавания лиц и работы различных инструментов. Повышена скорость сканирования коллекций во время запуска. Улучшена поддержка интеграции с движком семантического поиска и с MySQL/MariaDB. Расширены инструменты для сопровождения БД. Проведена работа по повышению стабильности и удобства использования инструмента для переименования группы файлов в пакетном режиме. Добавлена возможность сохранения в метаданных информации о местоположении и улучшена поддержка файлов GPX. До версии libraw 0.21.0 обновлён внутренний движок для обработки RAW-изображений. Добавлена поддержка форматов CR3, RAF и DNG. Добавлена поддержка новых моделей камер, включая iPhone 12 Max/Max Pro, Canon EOS R5, EOS R6, EOS 850D, EOS-1D X Mark III, FujiFilm X-S10, Nikon Z 5, Z 6 II, Z 7 II, Olympus E-M10 Mark IV, Sony ILCE-7C (A7C) и ILCE-7SM3 (A7S III). Улучшен инструмент импорта фотографий с камер, добавлена поддержка автоматического назначения имён альбомам и переименования во время загрузки.
