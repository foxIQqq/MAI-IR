Дистрибутив Ubuntu MATE сформировал сборки для плат Raspberry Pi

Разработчики дистрибутива Ubuntu MATE, построенного на пакетной базе Ubuntu и предлагающего десктоп-окружение на основе проекта MATE, объявили о формировании сборок для плат Raspberry Pi. Сборки основаны на выпуске Ubuntu MATE 22.04 и подготовлены как для 32-разрядных, так и для 64-разрядных плат Raspberry Pi. Из особенностей выделяется: Включение по умолчанию механизма zswap и алгоритма lz4 для сжатия информации в разделе подкачки. Поставка KMS-драйверов для GPU VideoCore 4, а также драйвера v3d для графического ускорителя VideoCore VI. Активация по умолчанию композитного оконного менеджера. Проведение оптимизации состава загрузочного образа. Дополнительно можно отметить намерение разработчиков дистрибутива Fedora Linux обеспечить официальную поддержку сборок для плат Raspberry Pi 4. До сих пор платы Raspberry Pi 4 официально не поддерживались проектом из-за отсутствия открытых драйверов для графического ускорителя. С включением драйвера v3d в состав ядра и Mesa проблема с отсутствием драйверов для VideoCore VI решена, поэтому ничего не мешает реализовать официальную поддержку данных плат в Fedora 37.
