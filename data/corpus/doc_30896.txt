Вышла Fluxbox-редакция Linux дистрибутива Salix 13.37

Представлена легковесная редакция Linux-дистрибутива Salix 13.37, базирующаяся на оконном менеджере Fluxbox. Дистрибутив на 100% совместим со Slackware Linux 13.37 и исповедует подход "одно приложение на одну задачу". Salix развивается создателем дистрибутива Zenwalk Linux, покинувшим проект в результате конфликта с другими разработчиками, отстаивавшими политику максимальной похожести на Slackware. Размер загрузочного iso-образа составляет 574 Мб. В состав новой версии включены такие приложения, как Linux-ядро 2.6.37.6, оконный менеджер Fluxbox 1.3.1, веб-браузер Firefox 4.0.1, почтовый клиент Claws-mail 3.7.8 и офисный пакет Libreoffice 3.3.2, графический редактор Gimp 2.6.11, музыкальный плеер Exaile 0.3.2, видеопроигрыватель Whaaw Media Player 0.2.14, пакет для записи CD/DVD Brasero 2.32.1, BitTorent-клиент Transmission, приложение для сканирования документов simple-scan, интерфейс для настройки параметров экрана LXRandr, набор инструментов для разработчиков, включая GCC, Python, Perl и Geany IDE. В меню добавлена опция "Install multimedia codecs" позволяющая одним кликом установить комплект дополнительных медиа-кодеков. Для управления пакетами в командной строке задействован пакетный менеджер gslapt, являющийся эквивалентом slapt-get, в качестве графического интерфейса для установки программ из SlackBuilds в дополнение к gslapt поставляется программа Sourcery, являющаяся специально разработанным в рамках проекта Salix фронтэндом к slapt-src. В настоящий момент в SlackBuilds репозитории доступно более 2600 пакетов. Стандартные инструменты для управления пакетами Slackware модифицированы для обеспечения поддержки Spkg, что позволяет использовать внешние приложения, подобные sbopkg, без нарушения совместимости со Slackware. Дистрибутив поддерживает три режима установки: Полная - все приложения для десктоп окружения; Основная - FluxBox, файловый менеджер PCManFM, web-браузер Firefox, пакетный менеджер gslapt, SlackBuild-менеджер Sourcery и системные программы Salix; Базовая - только минимальный набор консольных утилит, подходит для создания серверов.
