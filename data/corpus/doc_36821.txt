Проект Remastersys прекратил своё существование

Автор Remastersys, после 7 лет разработки, объявил о закрытии проекта и сворачивании всей связанной с Remastersys инфраструктуры. Один из энтузиастов уже выразил желание продолжить разработку и основал форк инструментария, который будет распространяться под именем System Imager. Кроме того можно отметить проект Relinux, в рамках которого с 2011 года активно развивается ответвление от Remastersys. Remastersys представляет собой инструментарий для создания загружающихся в Live-режиме систем на основе текущего среза уже установленной на диск и настроенной системы, что удобно использовать для создания работающих в режиме только для чтения резервных копий, для подготовки демонстрационных версий или для формирования Live-дистрибутивов. Например, используя Remastersys можно записать текущую систему со всеми пользовательскими данными на DVD и использовать ее на другом компьютере для демонстрации или в качестве временного рабочего окружения. В случае потери данных на основной системе, созданный диск можно использовать для восстановления изначальной системы.
