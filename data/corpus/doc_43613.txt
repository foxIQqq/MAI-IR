Вышел Calculate Linux 15.12

Доступен релиз отечественного дистрибутива Calculate Linux 15.12, построенного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в нескольких редакциях, собранных для архитектур i686 и x86_64: Calculate Linux Desktop с рабочим столом KDE (CLD), Mate (CLDM) и Xfce (CLDX), Calculate Directory Server (CDS), Calculate Linux Scratch (CLS), Calculate Scratch Server (CSS) и Calculate Media Center (CMC). Все версии дистрибутива распространяются в виде загрузочного Live-образа с возможностью установки на жесткий диcк или USB-накопитель. Основные изменения: Браузер Chromium в редакциях дистрибутива для рабочего стола заменён на QupZilla. В Grub добавлена поддержка формирования меню "Live HDD" для загрузки ISO-образов Calculate Linux с раздела жёсткого диска. По умолчанию используются настройки текущей системы для видео, локали и часового пояса. Добавлена поддержка открытого видеодрайвера AMDGPU. Изменены параметры авторазметки. По умолчанию при авторазметке возвращено создание резервного раздела для обновления системы. Eudev обновлён до версии 3.1.5 с восстановлением удалённого из udev кода, отвечающего за корректное переименование сетевых устройств со схожими с ядром именами. Репозиторий насчитывает более 2900 бинарных пакетов и включает теперь как пакеты с KDE 4, так и пакеты с KDE 5. Улучшена работа с Live USB. Добавлена поддержка установки системы на Flash без форматирования FAT-раздела, что позволяет выполнять обновление основной системы после загрузки с флеш-накопителя. Для установки системы на флеш-накопители добавлена отдельная команда cl-install-flash. Корневой раздел USB Flash после загрузки системы теперь доступен для записи. Многократно выросла скорость работы с архивами bzip2 и gzip в особенности на многоядерных процессорах за счёт использования по умолчанию lbzip2 и pigz. Ускорение отразилось в том числе и на времени установки программ. Прекращено использование демона preload для SSD и RAID с использованием планировщика 'noop'; Реализовано удаление неиспользуемых локалей во время установки. Удалены systemd-настройки пакетов. Оптимизирована установка проприетарного драйвера во время загрузки livecd за счёт исключения обращений к сети. Оптимизировано обновление портежей. Проверка целостности теперь выполняется опционально, оптимизирован вызов. Улучшен алгоритм выбора оптимального зеркала репозитория бинарных пакетов. Добавлена проверка на доступность Git-репозитория при обновлении портежей. Добавлена опция "--one-depth" в утилитах cl-update, cl-builder-update и cl-builder-image для освобождения места, занимаемого локальным репозиторием портежей. Ускорена работа SSH за счёт преднастроенного мультиплексирования. Добавлена поддержка большинства европейских языков и расширен выбор раскладок. Изменён выбор языка на Live USB. Вместо отдельного выбора языка и раскладки, добавлено меню выбора локали. Утилиты Calculate и описания бинарных пакетов переведены на украинский язык. Состав версий CLD (KDE desktop, i686 - 2.18 G, x86_64 - 2.34 G): KDE 4.14.15, LibreOffice 5.0.3.2, QupZilla 1.8.9, Gimp 2.8.16, Amarok 2.8.0 CLDM (MATE desktop, i686 - 1.76 G, x86_64 - 1.92 G): MATE 1.12, LibreOffice 5.0.3.2, QupZilla 1.8.9, Claws Mail 3.13.1, Gimp 2.8.16, Clementine 1.2.3; CLDX (Xfce desktop, i686 - 1.70 G, x86_64 - 1.86 G): Xfce 4.12, LibreOffice 5.0.3.2, QupZilla 1.8.9, Claws Mail 3.13.1, Gimp 2.8.16, Clementine 1.2.3; CDS (Directory Server, i686 - 751 M, x86_64 - 774 M): OpenLDAP 2.4.38, Samba 3.6.25, Postfix 3.0.2, ProFTPD 1.3.5a, Bind 9.10.2 CLS (Linux Scratch, i686 - 855 M, x86_64 - 948 M): Xorg-server 1.17.4, ядро Linux 4.2.8 CSS (Scratch Server, i686 - 542 M, x86_64 - 565 M): ядро Linux 4.2.8, Calculate Utilities 3.4.5.22 CMC (Media Center, i686 - 1.16 G, x86_64 - 1.31 G): Kodi 15.2, Xorg-server 1.17.4 Особенности Calculate Linux: 100% совместимость с Gentoo с поддержкой бинарных репозиториев обновлений. Поставка специально разработанных в рамках проекта Calculate утилит для настройки, сборки и установки системы. Готовое клиент-серверное решение. Быстрое развёртывание на предприятии. Полноценная работа в гетерогенных сетях. Удобство администрирования. Rolling-release модель обновлений. Поддерживается интерактивная сборка системы - подготовка ISO-образа системы под свои задачи.
