Вышла бета версия нового сервера приложений Zend для языка PHP

"Zend Server Community Edition" - вышла бета версия нового сервера приложений Zend для быстрого развертывания среды для выполнения PHP приложений. В состав пакета входят: PHP, Zend Framework, Apache, MySQL, модули для взаимодействия с СУБД и Java, web-интерфейс для управления, средства для оптимизации производительности. Дополнительно развивается коммерческая версия Zend Server, отличающаяся от бесплатной версии наличием средств для кеширования страниц и мониторинга.
