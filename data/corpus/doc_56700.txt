Опубликован AV Linux MX-21, дистрибутив для создания аудио- и видеоконтента

Доступен дистрибутив AV Linux MX-21, содержащий подборку приложений для создания/обработки мультимедийного контента. Дистрибутив основан на пакетной базе проекта MX Linux и дополнительных пакетов собственной сборки (Polyphone, Shuriken, Simple Screen Recorder и т.п.). Дистрибутив может функционировать в Live-режиме и доступен для архитектуры x86_64 (3.4 ГБ). Пользовательское окружение основано на Xfce4 с оконным менеджером OpenBox вместо xfwm. В поставку входят звуковые редакторы Ardour, ArdourVST, Harrison, Mixbus, система 3D-проектирования Blender, редакторы видео Cinelerra, Openshot, LiVES и инструменты для преобразования мультимедийных форматов файлов. Для коммутации звуковых устройств предлагается JACK Audio Connection Kit (используется JACK1/Qjackctl, а не JACK2/Cadence). Дистрибутив снабжён подробным иллюстрированным руководством (PDF, 74 стр.) В новом выпуске: Изменён метод формирования дистрибутива, вместо переупаковки готовых пакетов MX Linux (Respin) дистрибутив собран из исходных текстов с использованием инструментария, применяемого для сборки MX Linux и antiX. Пакетная база синхронизирована с пакетными базами MX Linux 21 и Debian 11 (Bullseye). Изменена схема нумерации выпусков, вместо номера YYYY.MM.DD теперь используется номер, соответствующий версии MX Linux. Прекращено использование внешних репозиториев (KXStudio), отличных от репозиториев MX Linux. Прекращено формирование сборок для 32-разрядных систем x86. Прекращена поставка варианта ядра Linux с набором RT-патчей для увеличения отзывчивости системы. Новое ядро AV Linux основано на выпуске 5.15 и поставляется с патчами Liquorix, нацеленными на повышение производительности при выполнении мультимедийных приложений. Добавлена поддержка AHS (Advanced Hardware Support), варианта настройки репозиториев дистрибутива MX Linux, в котором предложены последние обновления подсистем графического стека и микрокода для новых процессоров. Пакеты с улучшенной поддержкой оборудования могут быть установлены по мере их выпуска, используя штатные инструменты установки и обновления. Предложена новая тема оформления для Xfce и Openbox с набором обоев рабочего стола с разрешением 4K. Также предложены новые темы оформления пиктограмм Suru++ и Papirus. Предложен новый модульный интерфейс AV Linux Assistant, основанный на YAD. Добавлена заставка (Splash Screen), показываемая при первом запуске. Добавлена поддержка Yabridge для загрузки VST-плагинов, собранных для Windows. Реализован отдельный графический интерфейс для работы с Yabridge. Добавлены новые утилиты: BPM Converter и Gradient Wallpaper generator. Расширены пакеты с плагинами AVL-MXE и шрифтами. Добавлена поддержка пакетов в формате AppImage. Для установки пакетов вместо GDebi задействован обработчик на базе Thunar Custom Action.
