API Media Source Extensions переведён в разряд рекомендованных стандартов

Консорциум W3C представил финальный вариант спецификации Media Source Extensions, получившей статус рекомендованного стандарта. API Media Source Extensions предоставляет средства для генерации мультимедийных потоков из JavaScript-приложений. В том числе, данный API даёт возможность динамически формировать контент для тегов video и audio, определять свои модели буферизации и состыковки данных, что позволяет реализовывать такие системы, как подстановка рекламы, live-вещание со сдвигом во времени, адаптивное изменение битрейта и разрешения, редактирование видео и обеспечение просмотра в формате "360 градусов".
