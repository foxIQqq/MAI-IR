Проект Raspberry Pi выпустил микроконтроллер RP2040 стоимостью 1 доллар

Проект Raspberry Pi объявил о поступлении в продажу микроконтроллеров RP2040, разработанных для платы Raspberry Pi Pico и также задействованных в новых продуктах Adafruit, Arduino, Sparkfun и Pimoroni. Стоимость чипа - 1 доллар США. Микроконтроллер RP2040 включает в себя двухъядерный процессор ARM Cortex-M0+ (133MHz) c 264 КБ встроенной оперативной памяти, датчиком температуры, контроллерами USB 1.1, DMA, UART, SPI и I2C. Для создания приложений может применяться язык Си, C++ или MicroPython. Порт MicroPython для RP2040 подготовлен совместно с автором проекта и поддерживает все возможности чипа, включая собственный интерфейс подключения расширений PIO. Для разработки для чипа RP2040 с использованием MicroPython адаптирована интегрированная среда программирования Thonny. Возможностей чипа достаточно для выполнения приложений для решения задач машинного обучения, для разработки которых подготовлен порт фреймворка TensorFlow Lite. Поддерживается запуск на чипе FreeRTOS.
