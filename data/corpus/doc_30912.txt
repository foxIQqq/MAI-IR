Релиз пакета для организации групповой работы Horde 4.0

Проект Horde анонсировал выпуск продуктов Horde Groupware 4.0 и Horde Groupware Webmail Edition 4.0. Код продуктов написан на языке PHP, распространяется в рамках лицензии LGPL и поддерживает работу с СУБД MySQL, PostgreSQL, Oracle и MS SQL. Horde Groupware представляет собой готовый для промышленного применения, web-ориентированный пакет для организации совместной работы пользователей, которым предоставлена возможность совместного использования календаря-планировщика, адресной книги, списка задач и заметок. Horde Groupware Webmail Edition расширяет первый пакет поддержкой интерфейса для чтения электронной почты, при этом на выбор предоставляется пять различных web-интерфейсов (платформа одна, отличается внешнее оформление), в которые интегрирована поддержка вышеупомянутых средств для совместной работы. Ключевым отличием Horde Groupware 4.0 от ветки 3.x, развивающейся с 2005 года, является проведение рефакторинга кода для перехода на использование возможностей PHP 5. Из других изменений можно отметить: Реализован новый процесс установки, привязанный к использованию PEAR-инсталлятора. Упрощен процесс настройки. Добавлена поддержка ActiveSync. Интерфейс разделен на два режима - базовый и для продвинутых пользователей. Добавлены новые web-фронтэнды с реализацией интерфейса на базе активного использования технологии Ajax и для использования на мобильных устройствах. Реализованы дополнительные оптимизации, усилено использование кэширвоания, сокращено число внешних зависимостей. Значительно расширены функции календаря, добавлена поддержка CalDAV-клиента, возможность планирования ресурсов, новая система подсветки, поддержка создания системных календарей, установка тегов. В почтовом клиенте задействована новая библиотека для работы с IMAP, расширены возможности поиска и фильтрации, расширены функции нотификации, улучшено отображение сообщений в формате HTML, сокращено потребление памяти и реализована функция автозаписи.
