Разработчики Python заявили о прекращении поддержки ветки Python 3.0.x

Разработчики Python заявили о прекращении поддержки ветки Python 3.0 в пользу Python 3.1. Обновления Python 3.0.2 не будет, рекомендуется перейти на использование версии Python 3.1, которая будет поддерживаться в полном объеме. Через 18-24 месяцев выйдет релиз Python 3.2, после чего будет выпущен как минимум один корректирующий релиз Python 3.1.x и только тогда ветка Python 3.1 будет закрыта.
