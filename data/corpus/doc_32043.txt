Релиз Linux-дистрибутива Wary Puppy 5.2

Доступен релиз Wary Puppy 5.2, классической сборки Puppy Linux, способной работать на устаревшем оборудовании и продолжающей развитие серии Puppy 4.x. Дистрибутив является самостоятельным проектом, использующим систему пакетов T2. Графическое окружение основано на оконном менеджере JWM. В поставку входит собственный набор GUI-конфигураторов (Puppy Control Panel) и виджетов (Pwidgets - часы, календарь, RSS, состояние соединения и т.п.). Для установки дополнительных программ используется Puppy Package Manager. Загрузочный iso-образ занимает 124 Мб. Для обеспечения возможности работы на устаревшем оборудовании в базовой поставке Wary Puppy совмещен набор современных приложений с некоторыми старыми версиями пакетов. Кроме того, до минимума сокращено число запущенных в фоне процессов (например, не используется Dbus). Из входящих в комплект приложений, можно отметить: браузер/почтовый клиент SeaMonkey, текстовый процессор Abiword, табличный процессор Gnumeric, растровый редактор mtPaint, векторный редактор Inkscape, медиаплееры (MPlayer, Ogle DVD и Pmusic), файловый менеджер ROX Filer, ПО для записи CD/DVD Pburn, менеджер фотографий PupCamera, а также подборка мультимедиа кодеков и набор драйверов для видеокарт, звуковых карт, принтеров и сканеров. Параллельно с Wary Puppy развивается еще четыре ответвления дистрибутива Puppy: основанный на пакетной базе Ubuntu Lucid Puppy, инновационный Quirky, оптимизированный для 64-разрядных систем FatDog64 и нацеленного на нетбуки Puppeee. В Wary Puppy 5.2 по умолчанию по прежнему используется Xorg 7.3, что позволяет сохранить универсальный драйвер Xvesa для устаревших систем, но нарушает совместимость с некоторыми новыми дрaйверами, например, xf86-video-intel. Для современных систем подготовлен специальный мета-пакет, который позволяет в один клик перейти к использованию Xorg 7.6 и установить более свежий набор видеодрайверов для видеокарт Intel, nVidia и ATI. Таким образом система содержит Xorg 7.3 для старых систем, с возможностью быстрого обновления до версии 7.6 для систем с современными видеокартами. В новом выпуске из поставки удалены Adobe Flash, Gtksourceview/Nicoedit, Xfprot. Gtkam заменен на PupCamera. Библиотека GTK+ обновлена с 2.21.1 до версии 2.24.5. Обновлен пакет usb-modeswitch, что позволило расширить поддержку современных 3G-модемов. Созданы PET-пакеты для антивирусной системы Avast, JWMThemeMaker и GTKThemeMaker. Обновлены версии программ: SeaMonkey 2.3.2, ePDFView 0.1.7, jwm_tools 0.7, JWM 2-svn505, Peasyscan 1.7, PeasyPDF 1.4, Busybox 1.17.2, Pmusic 2.1.8, speex 1.2rc1, ffmpeg 20111002, mplayer 20110929, ogle 0.9.2. Расширены возможности VoIP-телефона PuppyPhone (Psip).
