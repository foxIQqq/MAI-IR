Релиз web-фреймворка Django 1.2

Вышла очередная стабильная версия фреймворка Django 1.2, написанного на языке Python и предназначенного для разработки веб-приложений. Основные особенности новой версии: Прекращение поддержки Python 2.3; Возможность одновременного обращения к разным БД в рамках одного рабочего процесса; Механизм проверки корректности форм и модели данных (model validation); Поддержка определения прав доступа на уровне отдельных объектов; Усилена защита от CSRF-атак; Новый фреймворк для обработки сообщений пользователей, позволяющих использовать cookie и сессии для анонимных и аутентифицированных посетителей; Поддержка определения собственного бэкенда для отправки email-сообщений.
