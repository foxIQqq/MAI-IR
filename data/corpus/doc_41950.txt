Новая версия свободной системы 3D-моделирования Blender 2.74

После трёх месяцев разработки представлен выпуск свободного пакета 3D-моделирования Blender 2.74. Наиболее значительные улучшения: В движок рендеринга Cycles добавлен новый атрибут Pointness, похожий на инструмент работы с вертексным цветом "Dirty Vertex Colors"; Добавлены новые функции композитинга для вьюрпорта, "Структура проекта" (Outliner) теперь лучше справляется с датаблоками, не имеющими активных пользователей; Появился инструмент редактирования нормалей, добавлена возможность переноса слоёв данных между мешами; Серьёзно улучшена динамика и инструменты редактирования частиц, таких как волосы; Для режимов скульптурирования и рисования добавлен инструмент "Маска неровностей" (Cavity Mask); Проведена большая работа по уменьшению потребления памяти при рендеринге штрихов в движке Freestyle; Внесены улучшения в инструмент Grease Pencil и игровой движок. Подробнее о новшествах: Движок рендеринга Cycles: Улучшен алгоритм рендеринга, что позволило решить проблемы с точностью и уменьшить шум; Оптимизация: Черный фон исключён из шейдера, что значительно повысило скорость отрисовки. Также проведено множество оптимизаций потребления памяти; В ноду Geometry добавлен новый атрибут Pointness, появилась возможность контролировать текстурное пространство одного объекта при помощи другого. Пользовательский интерфейс Добавлены новые функции композитинга для вьюпорта: реализованы взаимное затенение и глубина резкости в реальном времени (Ambient Occlusion & Depth Of Field)! Глубину фокуса теперь можно получить при помощи пипетки В структуре проекта появилась возможность полностью удалять иерархию объектов и переносить объекты в группы. В нодовом редакторе добавлена возможность показа во фреймах текстовых блоков. Появилась возможность редактирования "безопасных областей" камеры. Моделирование Добавлены два модификатора: инструмент редактирования нормалей и перенос данных между мешами (Normal Edit & Data Transfer). Улучшено поведение инструмента для "скольжения" ребра/вершины. Добавлен новый тип интерполяции - обратный квадрат - для режима пропорционального редактирования. "Fit Camera View" теперь работает и для камер с ортографической проекцией. Добавлен новый инструмент "Split Concave Faces" для обеспечения выпуклости объекта. Частицы Значительно улучшена динамика волос и инструменты её редактирования: Поддержка соударений волос с сеткой объекта. Симуляция теперь использует волюметрику, что даёт большую реалистичность. Пряди теперь используют более реалистичный подход для расчёта изгиба. Возможность изменять форму дочерних (child) волос с помощью кривой. Новый режим спиральной модификации волос. Добавлены кнопки переключения видимости в рендере и вьюпорте для систем частиц. Скульптурирование/Рисование по текстуре Улучшены инструменты скульптурирования и рисования по текстуре. Штрихи типа “Линия” теперь можно проводить под 45 градусов. Поддержка дизеринга при рисовании на изображениях с байтовой цветностью. Добавлен новый инструмент “Cavity Mask”, который позволяет создавать маски неровностей меша. В системе линейного нефотореалистичного рендеринга (Freestyle) значительно оптимизирован рендер при рисовании линий. Анимация, Grease Pencil Новые инструменты редактирования, такие как копирование штрихов, дублирование активного слоя и т.д. При рисовании в режиме «Continuous Drawing» теперь можно быстро воспользоваться ластиком «на лету» без выхода из режима рисования. Различные улучшения интерфейса, такие как палитры цветов для штрихов и для заливки. Слои Grease Pencil теперь отображаются в Outliner-е. Улучшенное управление набором анимаций объекта. Ключи анимации теперь можно копировать перевёрнутыми (flipped). Добавлена опция «Follow» для редакторов анимации. В игровом движке с помощью Python API теперь можно получить первую точку столкновения. Множество небольших изменений, повышающих удобство работы с BGE. Многочисленные улучшения в дополнениях. Представлено новое дополнение Auto Tile Size, устанавливающее оптимальный размер ячейки (tile) при рендеринге в Cycles.
