В рамках проекта Homerun для KDE подготовлен интерфейс для запуска приложений в стиле GNOME 3

Для KDE представлен апплет Homerun, реализующий функции полноэкранного интерфейса для запуска программ, работающего в стиле ланчера из состава GNOME 3. В Homerun используется разбиение типов контента по иерархии вкладок (программы, файлы и т.п., во вкладке с программами в свою очередь доступны фильтры для мультимедия, интернет-приложений и т.д.). В базовой вкладе выводится сводный список избранных приложений, недавно открытых документов и часто используемых файлов/директорий. Доступны средства для поиска контента и фильтрации вывода. Готовые установочные пакеты подготовлены для Kubuntu и openSUSE. Из планов по дальнейшему развитию отмечается обеспечение интеграции с системой комнат KDE (Activity), что позволит выводить для каждой комнаты свой набор рекомендованных приложений. Также планируется расширить число поддерживаемых источников данных.
