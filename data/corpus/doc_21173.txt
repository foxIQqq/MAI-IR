Вышел релиз системы виртуализации VirtualBox 2.2

Компания Sun Microsystems представила релиз виртуальной машины VirtualBox 2.2.0, в котором добавлена долгожданная реализация поддержки OpenGL 3D акселерации для гостевых Linux и Solaris окружений. Кроме исправления более 50 ошибок, можно отметить следующие новшества: Возможность импорта и экспорта виртуальных окружений в формате OVF (Open Virtualization Format); Лимит памяти на одну виртуальную машину увеличен до 16 Гб; Host-only режим работы сети; Оптимизация кода гипервизора для повышения производительности в условиях интенсивного переключения контекста; Для создаваемых виртуальных машин средства аппаратной виртуализации VT-x/AMD-V и поддержка USB (OHCI & EHCI) теперь активируются по умолчанию; Экспериментальная поддержка USB для OpenSolaris хостов; Поддержка импортирования директорий корневой системы (Shared folders) в гостевое окружение с Solaris и OpenSolaris; Экспериментальная поддержка Mac OS X 10.6 (Snow Leopard) хостов, работающих с 64- или 32-разрядным ядром. В дополнение к управляющему API для C++, Java, Python и Web-сервисов, представлен программный интерфейс для языка Си.
