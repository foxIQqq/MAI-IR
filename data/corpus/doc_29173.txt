Вышел новый русскоязычный дистрибутив FIDOSlax Linux 1.0

Представлен FIDOSlax Linux, компактный и быстрый русскоязычный дистрибутив Linux, который поддерживает как установку на жесткий диск, так и работу Live-режиме с USB-Flash или CD. FIDOSlax Linux является гибридным дистрибутивом и занимает промежуточное положение между обычными дистрибутивами и дистрибутивами Live, позволяя использовать FIDOSlax Linux и приложения Open Source на компьютере параллельно с установленной на жестком диске MS Windows на разделах NTFS и FAT32, а также на разделах с родными для Linux файловыми системами. FIDOSlax Linux основан на дистрибутиве Remix Linux, однако его функционал расширен дополнительными возможностями: Расширенная и упрощенная для пользователя поддержка русского языка - быстрая замена кодировки CP1256, KOI8-R и UTF-8 при помощи команды lang; Репозиторий программного обеспечения - на данный момент количество собранных модулей ПО для 32-х разрядной версии дистрибутива - 1168 модулей, а для 64-разрядной бета-версии FIDOSlax - 371 модуль. В дистрибутив FIDOSlax встроен простой пакетный менеджер lzm-модулей - для поиска, получения информации, загрузки и активации дополнительных модулей программного обеспечения; Добавлены средства для автоматического создания модулей программного обеспечения из исходных текстов, разработанные на основе портированных для FIDOSlax Linux средств crux4slack, Arch Linux ABS/AUR и репозиториев SlackBuilds. Также добавлены средства для массового создания модулей; Новый метод сохранения данных «Волшебные папки» позволяет хранить данные и настройки FIDOSlax на USB-Flash или разделе жесткого диска; Добавлены средства для установки на жесткий диск загрузчика Grub4DOS для совместной установки с MS Windows и загрузчика EXTLINUX установки дистрибутива на разделы ext3, ext4 или в качестве единственной ОС; FIDOSlax уже содержит предварительно настроенное ПО для удаленного доступа к рабочему столу. После задания пароля на сессию, можно использовать любой vnc-клиент для удаленной работе в FIDOSlax; 32-битная версия дистрибутива FIDOSlax 1.0 включает базовый набор приложений - ядро 2.6.31.1, Xorg 1.9.2, KDE 3.5.10, Firefox 3.6.13 и т.д. Также FIDOSlax является дистрибутивом, в котором уже встроено программное обеспечение для работы с сетью FIDONet при помощи метода FIDO over IP(binkd, husky hpt и GoldED-NSF). Используя FIDOSlax Linux можно легко и быстро подключиться к сети Фидонет - всё что нужно, это связаться с системными операторами сети, список которых приведён в подробной документации на сайте проекта.
