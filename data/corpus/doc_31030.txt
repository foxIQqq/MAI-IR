GitHub открыл код библиотеки для определения языка программирования и подсветки синтаксиса

Хостинг свободных проектов GitHub объявил об открытии исходных текстов библиотеки Linguist, выполняющей задачу определения используемого языка программирования, подсветки синтаксиса и определения MIME-типа для бинарных файлов. Код библиотеки написан на языке Ruby и распространяется в рамках лицензии MIT. Определение языка программирования производится на основании расширения файла или через поиск свойственных для каждого языка ключевых слов. Для подсветки синтаксиса задействован ruby-модуль Albino и Python-библиотека Pygments.
