Релиз mod_rivet 2.0.5, apache-модуля с поддержкой языка Tcl

После длительного периода (предыдущая версия вышла 23 сентября 2011 года) вышел релиз модуля mod_rivet 2.0.5, предназначенного для использования совместно с Apache и реализующего возможность разработки веб-проектов на языке Tcl. Релиз носит в основном корректирующий характер и включает в себя небольшое количество нововведений. В ближайшее время ожидаются бинарные пакеты для таких дистрибутивов Linux как openSUSE, Red Hat, CentOS 5, Debian и Ubuntu. Основные библиотеки, входящие в состав mod_rivet: DIO - библиотека для взаимодействия с базами данных, предоставляющая единый API, вне зависимости от используемой БД (заявлена поддержка таких БД как MySQL, Oracle, PostgreSQL, SQLite). Session - Библиотека для работы с сессиями, тесно взаимодействующая с библиотекой DIO и использующая для хранения данных БД. Form - библиотека для создания веб-форм.
