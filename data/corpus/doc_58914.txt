Выпуск программы для детского рисования Tux Paint 0.9.29

Опубликован релиз графического редактора для детского творчества - Tux Paint 0.9.29. Программа предназначена для обучения рисованию детей в возрасте от 3 до 12 лет. Бинарные сборки сформированы для Linux (rpm, Flatpak), Haiku, Android, macOS и Windows. В новом выпуске: Добавлено 15 новых "волшебных" инструментов, эффектов и фильтров. Например, добавлен инструмент Fur для создания меха, Double Vision для симуляции раздвоения в глазах (диплопия), Remove Color и Keep Color для обесцвечивания части изображения, Maze для генерации случайных лабиринтов. Googly Eyes для прикрепления глаз к нарисованным персонажам. Circles и Rays для создания круговых и лучевых мазков. 3D Glasses для преобразования в трехмерное изображение, которое можно просматривать с помощью анаглифных очков. Color Sep для разделения выбранных цветов. Saturate и Desaturate для увеличения и уменьшения насыщенности цветов. Kaleido-4, Kaleido-6 и Kaleido-8 для преобразования изображений в калейдоскоп с поворотом фрагментов на 90°, 60° или 45°. Bloom с реализацией дифракционного эффекта свечения. В инструменте Печать (Stamp), применяемом для добавления на рисунок готовых шаблонов изображений, реализована поддержка поворота объекта перед размещением. В инструменте Заливка (Fill) реализована фигурная градиентная заливка, повторяющая контур области. Добавлена настройка, позволяющая автоматически выбрать размер кнопок в панелях, оптимальный для текущего экранного разрешения. Добавлена настройка для выбора способа масштабирования и расположения примеров изображений, не совпадающих по размеру с холстом Tux Paint.
