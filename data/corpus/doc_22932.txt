Бесплатное расширение для организации групповой работы в OpenOffice.org

Компания TeamDrive, работающая над созданием альтернативы Microsoft SharePoint, выпустила бесплатное расширение для организации групповой работы с документами на базе OpenOffice.org. При использовании расширения в OpenOffice.org появляется возможность работы с единым для всех пользователей предприятия хранилищем документов, доступных для одновременной совместной правки, и с поддержкой версионного контроля всех вносимых изменений. В качестве серверной части выступает TeamDrive WebDAV клиент-сервер, который доступен для бесплатной загрузки, но при работе осуществляет показ рекламных объявлений. Программа доступна для платформ: Windows XP, Vista, Mac OSX 10.5.3, Linux KDE, Linux GNOME.
