Выпуск LinHES 8.0, дистрибутива для создания медиа-центров

Представлен выпуск дистрибутива LinHES 8.0 (Linux Home Entertainment System), построенного на пакетной базе Arch Linux и нацеленного на максимальное упрощение процесса развёртывания медиа-центра с использованием системы MythTV (опционально поддерживается XBMC). Размер загрузочного образа 670 Мб. Инсталлятор LinHES предоставляет набор мастеров для первичной настройки параметров медиацентра, задействования различных моделей пультов дистационного управления и установки драйверов для полноценной поддержки имеющихся видео и аудио устройств, в том числе для SDTV/HDTV карт. При наличии доступных по сети уже установленных систем на базе LinHES имеется возможность синхронизации их настроек для использования на новой системе. После установки последующая настройка может производиться удалённо через web-интерфейс. Обновление может производиться как через штатные механизмы обновления пакетов, так и через перезаливку образа системы. Без установки поддерживается работа в Live-режиме. В состав дистрибутива входят средства для автоматизации проведения резервного копирования контента и организации доступа к медиаколлекции по сети через SAMBA или NFS. Новый выпуск примечателен использованием по умолчанию оконного менеджера Enlightenment (ранее использовался Fluxbox, который можно вернуть через правку скрипта set_windowmanager.sh) и переходом к формированию сборок только для архитектуры x86_64 (ранее поставлялась только 32-разрядная сборка). Прекращена поддержка драйверов для старых моделей видеокарт NVIDIA (поддерживаются только 8xxxx series, gt2xx и более новые серии). Переработан процесс загрузки, GRUB 1 заменён на GRUB 2, а для формирования заставки вместо Fbsplash задействован Plymouth. При установке дистрибутива на диск в новом выпуске создаётся 7 разделов, каталоги /home и /data/srv/mysql вынесены в отдельные разделы для упрощения организации обновления системы и предотвращения повреждения системы из-за полного заполнения диска.
