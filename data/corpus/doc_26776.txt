Проект OLPC выпустит обучающий планшет XO-3 в сотрудничестве с Marvell

Проект One Laptop per Child (OLPC) и компания Marvell заключили соглашение о совместной разработке новой модели портативного компьютера для детей - OLPC XO-3, которую решено построить в виде планшета на базе эталонной платформы Marvell Moby. По заявлению разработчиков, XO-3 будет поддерживать кодирования и декодирования 1080p-видео, работу с 3D-графикой, организацию телеконференций, потребляя при этом всего 1 Вт энергии. При проектировании новой модели разработчики учли пожелания детей и их родителей, уже пользующихся ранее выпущенными субноутбуками - пользователей OLPC по всему миру насчитывается около двух миллионов. Размер XO-3 будет близок к листу формата A4. Экран будет построен на основе гибридной технологии Pixel Qi, отличающейся поддержкой мультитач и пониженным потреблением энергии. Кроме того отмечается поддержка многоязычной экранной клавиатуры с активной обратной связью при касании. По сути новый XO-3 будет отождествлен с проектом Mobylize, в рамках которого компания Marvell продвигает прототип обучающего планшета, на котором поддерживается работа Ubuntu Linux и Google Android. Ранее, в числе интересных особенностей OLPC XO-3 планировалось использовать герметичный корпус, лишенный каких бы то ни было отверстий. Для зарядки аккумуляторов планшет рассчитывали помещать в специальную рамку, для передачи энергии через беспроводной индуктивный механизм. Судя по всему эти особенности не будут воплощены в жизнь. Программное обеспечение устройств будет основано на Linux дистрибутиве Fedora, вместо ранее используемого ПО XO Software. В качестве графического интерфейса пользователя по прежнему будет использоваться оболочка Sugar, но пользователи получат возможность включения и стандартного десктоп окружения на базе GNOME. Программное обеспечение позволит преподавателям, учащимся и их семьям создавать собственный контент и делиться им с другими людьми, учиться читать и писать, создавать новые обучающие приложения. В комплект будет включено специальное приложение, которое позволит организовать прямой доступ детей к библиотеки из более двух миллионов бесплатных электронных книг. Для обмена данными будет поддерживаться возможность создания mesh-сетей. Аппаратная начинка XO-3 будет построена на основе Armada 610 SoC с ARMv7-совместимым CPU, работающим на частоте 1 GHz и встроенной 3D-видеоподсистемой, поддерживающей OpenGL ES 2.0. Эталонная платформа Marvell Moby поддерживает 802.11n WiFi, Bluetooth, GPS, и FM-тюнер. Начало производства нового планшета намечено на январь 2012 года, при этом стоимость устройства планируется понизить до $75. Несмотря на анонс партнерства с Marvell в планах OLPC все еще остается выпуск промежуточной модели XO-1.75, оформленной в книжном формате с двумя сенсорными экранами.
