Вышел третий тестовый выпуск openSUSE 11.2

Вышел третий тестовый выпуск openSUSE 11.2 в котором обновлены версии следующих приложений: Linux ядро 2.6.30, Firefox 3.5 RC 2, GNOME 2.27.2, KDE 4.3 beta 2, PulseAudio 0.9.15, Konversation 1.2 Alpha 3, VirtualBox 2.2.4, Wine 1.1.23, OpenOffice.org 3.1.1 Alpha, glibc 2.10.1, Bluez 4.41, Amarok 2.1.1, Базовые новшества openSUSE 11.2: Пакеты дистрибутива собраны GCC 4.4 В качестве файловой системы по умолчанию задействована Ext4 В состав Live CD включен Gimp; Для сжатия данных на LiveCD отныне используется алгоритм сжатия LZMA; По умолчанию LiveCD содержит переводы для немецкого, французского, итальянского, польского и русского языков; В Qt версии конфигуратора YaST появилась поддержка настраиваемого внешнего вида. В YaST добавлены средства работы с кешем пакетов; В Zypp улучшены средства работы с зеркалами репозиториев пакетов; В интерфейсе для настройки параметров дисплея gnome-display-properties добавлена поддержка автоматического сброса настроек в случае проблем. Ведется разработка web-интерфейса к YaST для осуществления удаленного управления дистрибутивом Ожидается появление установочных образов для USB Flash и создание утилиты для их подготовки Улучшение поддержки субноутбуков Релиз openSUSE 11.2 намечен на ноябрь.
