Релиз рабочего стола KDE Plasma 5.18

Доступен релиз пользовательской оболочки KDE Plasma 5.18, построенной с использованием платформы KDE Frameworks 5 и библиотеки Qt 5 с применением OpenGL/OpenGL ES для ускорения отрисовки. Оценить работу новой версии можно через Live-сборку от проекта openSUSE и сборки от проекта KDE Neon User Edition. Пакеты для различных дистрибутивов можно найти на данной странице. Новая версия отнесена к категории выпусков с длительным сроком поддержки (LTS), обновления для которых формируются несколько лет (LTS-выпуски публикуются раз в два года). Ключевые улучшения: Реализована корректная отрисовка GTK-приложений, использующих декорирования окон на стороне клиента для размещения элементов управления в области заголовка окна. Для подобных приложений обеспечена отрисовка теней от окна и добавлена возможность использовать корректные области захвата окна для изменения размера, не требующие вывода толстых рамок (ранее при тонкой рамке было очень трудно захватить край окна для изменения размера, что вынуждало использовать толстые рамки, которые делали окна GTK-приложений чужеродными среди программ KDE). Обработка областей вне окна стала возможна благодаря реализации в оконном менеджере KWin протокола _GTK_FRAME_EXTENTS. Кроме того, для GTK-приложений обеспечено автоматическое наследование настроек Plasma, связанных со шрифтами, пиктограммами, курсорами и другими элементами управления; Интерфейс вставки Emoji теперь может быть вызван из меню приложений (App Launcher → Applications → Utilities) или при помощи комбинации клавиш Meta (Windows) + "."; Представлена новая панель глобального редактирования, упрощающая настройку раскладки элементов на рабочем столе и размещение виджетов, а также предоставляющая доступ к различным настройкам рабочего стола. Новый режим пришёл на смену старой кнопке с инструментами настройки рабочего стола, отображавшейся в верхнем правом углу экрана. Новая панель вызывается через пункт "Customize layout" в контекстном меню, которое показывается при клике правой кнопкой мыши на пустой области рабочего стола; Меню приложений (Kickoff) и интерфейс редактирования виджетов оптимизированы для управления с сенсорных экранов; Для системного лотка (System Tray) реализован новый виджет, позволяющий управлять активацией режима ночной подсветки; В размещённом в системном лотке виджете управления громкостью реализован более компактный интерфейс выбора звукового устройства, применяемого по умолчанию. Кроме того, если программа воспроизводит звук, на кнопке данной программы в панели задач теперь выводится индикатор громкости; В меню приложений реализован круглый значок с аватаром пользователя (ранее значок был квадратным); Добавлена настройка для скрытия часов на экране блокировки входа; Реализована возможность настройки клавиатурных комбинаций для активации и отключения режимов ночной подсветки и блокировки уведомлений; В виджете, отображающем прогноз погоды, реализована наглядная индикация ветреной погоды; Появилась возможность включения прозрачного фона для некоторых виджетов на рабочем столе; В Plasma Network Manager добавлена поддержка технологии защиты беспроводных сетей WPA3; Индикатор времени закрытия на всплывающих уведомлениях реализован в виде убывающей круговой диаграммы, окружающей кнопку закрытия; В уведомления, информирующие о завершении загрузки файла, добавлена перетаскиваемая пиктограмма, позволяющая быстро переместить файл в другое место; Обеспечен вывод уведомлений с предупреждением о низком заряде аккумулятора на подключённом Bluetooth-устройстве; Добавлены настройки уровня детализации отправляемой телеметрии с информацией об системе и частоте обращения пользователя к тем или иным возможностям KDE. Статистика отправляется анонимно и по умолчанию её отправка отключена; В конфигуратор добавлен ползунок для выбора скорости анимации окон (при смещении ползунка вправо окна будут появляться мгновенно, а при смещении влево - с использованием анимации). Улучшен поиск в боковой панели. Добавлена опция для прокрутки к позиции, соответствующей месту клика на полосе прокрутки. Переработан интерфейс настройки режима ночной подсветки. Предложен новый интерфейс настройки стиля оформления приложений; Переделана страница с параметрами системного лотка; В Центре установки приложений и дополнений (Discover) добавлена возможность публикации вложенных комментариев при обсуждении дополнений. Модернизировано оформление заголовка боковой панели и интерфейса с рецензиями. Добавлена поддержка поиска дополнений с основной страницы. Фокус клавиатурного ввода по умолчанию теперь переключается на строку поиска; Проведена работа по устранению визуальных артефактов в приложениях при использовании дробного масштабирования в окружении на базе X11; В KSysGuard обеспечено отображение статистики для GPU NVIDIA (потребление памяти и нагрузка на GPU). При работе в окружении Wayland обеспечена возможность автоматического поворота экрана на устройствах с акселерометрами; Из существенных нововведений, появившихся в KDE Plasma 5.18 по сравнению с прошлым LTS-выпуском 5.12 отмечается полная переработка системы уведомлений, интеграция с браузерами, редизайн системных настроек, улучшение поддержки GTK-приложений (применение цветовых схем, поддержка глобального меню и т.п.), улучшение управления многомониторными конфигурациями, поддержка "порталов" Flatpak для интеграции с рабочим столом и доступа к настройкам, режим ночной подсветки и инструменты для управления устройствами с интерфейсом Thunderbolt.
