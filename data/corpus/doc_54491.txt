Выпуск Wine 6.1

Состоялся экспериментальный выпуск открытой реализации Win32 API - Wine 6.1. С момента выпуска 6.0 было закрыто 37 отчётов об ошибках и внесено 327 изменений. Напомним, что начиная с ветки 2.x проект Wine перешёл на схему нумерации версий, в которой каждый стабильный релиз приводит к увеличению первой цифры в номере версии (5.0.0, 6.0.0), а обновления к стабильным релизам выпускаются с изменением третьей цифры (6.0.1, 6.0.2, 6.0.3). Экспериментальные версии, развиваемые в процессе подготовки следующего значительного релиза, выпускаются с изменением второй цифры (6.1, 6.2, 6.3). Наиболее важные изменения: В компиляторе WIDL (Wine Interface Definition Language) расширена поддержка WinRT IDL (Interface Definition Language). До версии 1.2 обновлён код проекта VKD3D с реализацией Direct3D 12, работающей через трансляцию вызовов в графический API Vulkan. Добавлена поддержка раскладки памяти транслятора Rosetta, используемого в устройствах Apple c чипом M1. Для 32-разрядных систем ARM обеспечена поддержка набора инструкций Thumb-2. Закрыты отчёты об ошибках, связанные с работой игр и приложений The Witcher 3, The Sims 3, Windows Print Preview, Dark Sector, F-Secure BlackLight Engine 2.2.x, Deer Hunter, Rollercoaster Tycoon 3, LabVIEW 2014, ACAT Dashboard, Quest 5.7.x, Skyrim SE, Aldi (Buhl) Steuer 2019/2020, Hearthstone Deck Tracker, SlingPlayer, Star Stable Online, Sniper Elite V2, Batman: Arkham Origins, Studio One 5, Still Life 2, SofTalk 1.56.
