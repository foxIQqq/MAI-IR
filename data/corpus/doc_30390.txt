Вышел релиз свободной web-ориентированной операционной системы eyeOS 2.4 "Japan"

Состоялся релиз web-ориентированной операционной системы eyeOS 2.4, который вышел под кодовым именем "Japan". Несколько дней назад была выпущена корректирующая версия 2.4.1, в которой оперативно были исправлены обнаруженные ошибки. eyeOS - свободная операционная система с реализацией идеи по созданию не зависящего от текущей рабочей машины cloud-десктопа. Пользователь может оказаться в своем eyeOS-окружении из любой точки, где имеется возможность запуска web-браузера, даже с мобильного телефона. Исходные тексты eyeOS распространяются в рамках лицензии AGPLv3. Серверная часть написана на языке PHP5, для хранения данных используется СУБД MySQL. Клиентская часть написана на JavaScript с использованием библиотеки Qooxdoo, без использования Flash- и Java-плагинов. Непосредственно приложения выполняются на стороне сервера, в браузер клиента лишь транслируется интерфейсная оболочка. Для версии 2.4 доступны переводы интерфейса на 32 языках, в том числе русский (неполный). Пакет для установки eyeOS на собственном оборудовании можно загрузить здесь. Также предоставляется возможность попробовать систему в действии без установки. Основные новшества выпуска eyeOS 2.4: Рабочий стол: Добавлен режим Dashboard, в котором вся важная информация отображается в соответствующих блоках на рабочем столе. Интеграция web-ресурсов: Реализована поддержка создания на рабочем столе или в любой папке ярлыков для любой web-ссылки. При запуске этого ярлыка откроется окно на рабочем столе eyeOS с указанным сайтом, web-приложением или любым интернет-ресурсом. Для ярлыка можно задать название, пиктограмму, а также ширину и высоту открываемого окна. Созданный ярлык появится на списке приложений. Общий доступ к файлам: Данная функция позволяет предоставлять доступ к файлам людям, которые не зарегистрированы в eyeOS, при помощи передачи url-адреса файла. Для ограничения доступа можно задать пароль и время в течении которого файл будет доступен. Почтовый клиент: Новый почтовый клиент с богатой функциональностью и удобным интерфейсом, выполненный на базе Roundcube. Мобильная версия: Упрощенный интерфейс для доступа к файлам в eyeOS (без возможности управлять ими) с мобильного телефона. Простая загрузка файлов: Достаточно просто выбрать несколько файлов с локального компьютера (будь то Windows, Mac или Linux) и без установки каких либо сторонних плагинов и расширений для браузеров, перетащить в окно браузера с eyeOS... и файлы загрузятся в eyeOS систему. Эта возможность доступна по умолчанию в eyeOS 2.4 в браузерах поддерживающих HTML5: Firefox, Safari, Opera, Chrome и IE 9. Мгновенный обмен сообщениями: Теперь можно общаться с другими пользователями без установки клиента для обмена мгновенными сообщениями. Из программ в eyeOS доступны текстовый и табличный процессоры с поддержкой форматов Microsoft Office и OpenOffice.org, календарь-планировщик, программы для быстрого обмена сообщениями, почтовый клиент с поддержкой POP3/IMAP, файловый менеджер, приложение Social Bar для организации коллективной работы, обмена файлами и сообщениями с другими пользователями и группами.
