Вышел кластерный дистрибутив Rocks 5.4

Представлен релиз Rocks 5.4, основанного на CentOS дистрибутива для создания вычислительных кластеров, grid-сетей и инфраструктур для обеспечения работы виртуальных окружений на базе Xen. Дистрибутив может быть использован для быстрого развертывания кластеров в локальной сети с загрузкой основного узла с LiveCD/LiveUSB, а для загрузки дополнительных узлов использовать PXE. Размер загрузочного iso-образа 2.5 Гб. Из изменений в новой версии можно отметить: Редизайн инсталлятора Avalanche, с учетом опыта развертывания кластера из тысячи узлов (например, минимизирован трафик к фронтэнду, BitTorrent-инсталлятор переписан с Python на Си ) Поддержка настройки параметров межсетевого экрана и возможность объединения потоков для узлов через управляющий командный интерфейс (команды "rocks * host firewall" и "rocks add host bonded); Добавлен сервис Airboss для предоставления пользователям без root-привилегий возможности управления их виртуальными окружениями; Возможность создания узлов "Login appliance", с которых осуществляется назначение работ, но которые сами не выполняеют задания; Создание единого GIT-репозитория для работы со всеми связанными с проектом исходными текстами. Дополнительно можно отметить проект XScript, в рамках которого разрабатывается сервер приложений, написанный на языке C++ и предназначенный для создания сложных, масштабируемых и высоконагруженных веб-сервисов. Для описания страниц используется основанный на XML язык, для верстки - XSLT.
