Вышел релиз дистрибутива KahelOS 050110

Доступен для загрузки обновленный установочный образ Linux-дистрибутива KahelOS, основанного на кодовой базе Arch Linux и десктоп-окружении GNOME. Дистрибутив проповедует модель непрерывных обновлений (Rolling-release), при которой обновления пакетов выходят постоянно и пользователь в любой момент имеет возможность перейти на последние версии программ, без ожидания их оформления в виде релиза дистрибутива. Размер установочного образа 690 Мб (i686 и x86-64). По сравнению с Arch Linux проект KahelOS отличается упрощенным процессом установки, уменьшенным временем загрузки, отсутствием в комплекте несвободных программ, включением по умолчанию графических 3D-эффектов на рабочем столе и использованием для настройки сети NetworkManager. Из новшеств, добавленных в сборке KahelOS 050110, отмечается: Включение по умолчанию Compiz Fusion (ранее использовался Metacity Compositor); Проведение оптимизации и чистки шрифтов, улучшение хинтинга и рендеринга шрифтов; С целью освобождения дополнительного пространства на диске произведено удаление необязательных пакетов; Выполнена оптимизация пакетов с поддержкой PNG/MNG/TIFF/GIF; Добавлено новое визуальное оформление; Реализована возможность организации из коробки совместного доступа к файлам; В качестве шины ввода по умолчанию задействован iBus; В поставку включены все возможные пакеты с прошивками для Wifi-адаптеров; В состав включен пакет с OpenJDK6; Сформирован новый набор поставляемых по умолчанию приложений: Chromium 5.0-beta Gnome 2.30 Gnome Color Manager 2.30 Linux-ядро 2.6.33 OpenOffice 3.2.0 Pitivi 0.13.4 Simple-Scan 0.9.9 Google Gadgets 0.11.2 Banshee 1.6.0 Tracker 0.8g Tomboy notes Gbrainy Lights Off
