AMD планирует предоставить полностью открытый стек для гетерогенных вычислительных систем (CPU+GPU)

В рамках участия в работе недавно представленной некоммерческая организации HSA Foundation, которая будет заниматься развитием открытых стандартов для гетерогенных вычислительных систем в которых в процессе вычислений кроме CPU используются средства параллельных вычислений GPU, компания AMD планирует полностью открыть свой стек HSA (Heterogenous System Architecture), состоящий из генератора промежуточного кода, патчей к LLVM, ассемблера, библиотек, runtime-компонентов, а также специфичного для GPU AMD модуля ядра для Linux. Исключение будет сделано только для фронтэнда парсера C++, который был разработан сторонней компанией в качестве коммерческого компонента.
