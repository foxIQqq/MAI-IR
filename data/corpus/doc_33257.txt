Релиз видеоконвертера HandBrake 0.9.6

Увидел свет релиз HandBrake 0.9.6, инструмента для многопоточного перекодирования видеофайлов из одного формата в другой. Программа доступна как в варианте, работающем в режиме командной строки, так и в виде GUI-интерфейса. Код проекта написан на языке Си и распространяется в рамках лицензии GPL, бинарные сборки можно загрузить для Fedora, Ubuntu, MacOS X и Windows. Программа может перекодировать видео с BluRay/DVD-дисков, копий директории VIDEO_TS и любых файлов, формат которых поддерживается библиотеками libavformat и libavcodec из состава FFmpeg/LibAV. На выходе могут быть сформированы файлы в таких форматах, как MP4 и MKV, для кодирования видео могут быть применены кодеки MPEG-4, H.264 и Theora, для звука - AAC, MP3, AC-3 или Vorbis. Из дополнительных функций присутствуют: калькулятор битрейта, предпросмотр в процессе кодирования, изменение размера и масштабирование картинки, интегратор субтитров, широкий набор профилей конвертации для заданных типов мобильных устройств. Из новшеств можно отметить: Добавлены кодировщики для видео MPEG-2 и звука FLAC ; Поддержка декодирвоания и проброса звука из Blu-ray в форматах TrueHD, DTS-HD и E-AC3; Поддержка изменения громкости при перекодировании; Поддержка кодирования с переменным битрейтом для обеспечения заданного уровня качества (работает только через интерфейсы CLI и Linux GUI для Lame MP3, Vorbis, Core Audio AAC); Для медиаконтейнеров MP4 и MKV добавлен проброс без перекодирования звука DTS-HD, DTS, MP3, AAC; Улучшены средства для определения в видео средней частоты кадров и выявления дублирующихся кадров; Добавлена поддержка медиаконтейнера WTV; Добавлена поддержка потоков MPEG-1 Program и HD-DVD EVOB; В сборке для Linux добавлена секция с расширенными настройками звука (частота дискретизации, тип сжатия, громкость); Небольшие оптимизации интерфейса; Обновлены сторонние библиотеки, такие как libav*, libass, libx264, libvorbis, libogg, libbluray.
