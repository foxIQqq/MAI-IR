Компания Adobe открыла два связанных с Flash фреймворка

Компания Adobe открыла исходные тексты двух продуктов, нацеленных на работу с технологией Flash: Open Source Media Framework (OSMF) и Text Layout Framework (TLF) под Mozilla Public Licence. Данное решение ориентировано на разработчиков, издателей и медиакомпании. Первый (OSMF) позволяет с лёгкостью оптимизировать работу Media Player для платформы Flash. Он включает стандартные функции для воспроизведения потокового видео и другого динамического содержимого. Кроме того, для интеграции сторонних компонентов доступно API в виде плагина. Второй является расширяемой библиотекой ActionScript (extensible ActionScript library) для поддержки типографических решений для веб-приложений, использующих сложные языки, двунаправленный текст,различные опции для ряда языков. Он основывается на новом текстовом движке Flash Player 10 и AIR 1.5.
