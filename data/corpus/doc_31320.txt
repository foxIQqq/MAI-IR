Второй кандидат в релизы Mandriva 2011.

Началось тестирование второго кандидата в релизы Linux-дистрибутива Mandriva 2011, в котором реализованы такие значительные системные изменения, как переход на пакетный менеджер RPM5 и систему инициализации Systemd. Для загрузки подготовлен образ LiveDVD для архитектур i586 и x86_64. Следующим этапом станет выпуск релиза, который запланирован на 29 августа. Дополнительно отмечается намерение выпустить в конце года LTS-релиз Mandriva 2011, который будет отличаться увеличенным временем поддержки. По сравнению с первым кандидатом в релизы Mandriva 2011 устранено около 300 ошибок и внесены изменения в примерно 700 пакетов. Из изменений можно отметить: В поставку включены дополнения для обеспечения работы гостевых окружений (guest additions) под управлением VirtualBox; Устранены проблемы в конфигураторах pptp, vpn и wi-fi; Обеспечена поддержка новых сетевых карт Atheros и Intel; Возвращены некоторые старые и добавлены некоторые новые драйверы для принтеров Epson, HP и Lexmark; Значительно улучшена поддержка 3G-модемов с интерфейсом USB; Улучшена работа KDM, Dolphin, SimpleWelcome, TimeFrame, RocketBar, Improver, Sphere и Sync; Из состава установочного образа удалены проприетарные драйверы ATI и NVidia; Началась работа по перемещению некоторых пакетов из репозитория PLF в "main", что должно упростить постинсталляционную установку некоторых популярных программ; Добавлена новая тема с изображениями курсоров и новый набор пиктограмм. Одновременно можно отметить, публикацию Евгением Додоновым заметки с описанием процесса формирования варианта Mandriva, работающего без pulseaudio и systemd. Кроме того, Евгений сообщил о публикации на GitHub кода некоторых проектов, в разработке которых он принимал участие во время работы в компании Mandriva. Данные программы написаны для Mandriva, но могут легко перенесены в другие дистрибутивы. msec (Multi-Security Tool) - интерфейс для управления настройками, связанными с безопасностью. Кроме того, Msec может использоваться для формирования отчетов на основе выполненения периодических проверок, например для контроля за правами доступа к файлам; netprofile - инструмент для настройки профилей сетевых соединений. Каждый профиль может содержать свой набор сетевых настроек, правил пакетного фильтра и других связанных с сетью параметров; net_monitor - простая система сетевого мониторинга, позволяющая учитывать трафик, отслеживать изменение сигнала Wifi и накапливать статистику по сетевым портам. Для генерации отчетов используется vnstat; Tomoyo GUI - графический интерфейс для настройки фреймворка TOMOYO, позволяющего привязать политику безопасности процесса к файловому пути, способом похожим с AppArmor. Tomoyo GUI может использоваться для работы как поверх ccs-tools, так и поверх tomoyo-tools. Программа позволяет автоматически генерировать профили безопасности, оценивать особенности поведения приложения, производить операции импорта и экспорта профилей; s2u (Services 2 User) - инструмент для организации вывода системных сообщений в пользовательской десктоп-сессии при помощи DBus. S2u используется различными системными сервисами Mandriva для вывода уведомлений на экран.
