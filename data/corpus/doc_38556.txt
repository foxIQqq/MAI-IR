Выпуск дистрибутива Linux Mint 16

Увидел свет дистрибутив Linux Mint 16, основанный на пакетной базе Ubuntu 13.10, полностью совместимого с ним, но существенно отличающегося подходом к организации интерфейса пользователя и подбором используемых по умолчанию приложений. Разработчики Linux Mint предоставляют десктоп-окружение, соответствующее классическим канонам организации рабочего стола, которое является более привычным для пользователей не принимающих новые методы построения интерфейса Unity и GNOME 3. Для загрузки доступны DVD-сборки на базе оболочек MATE 1.6 (1.2 Гб) и Cinnamon 2.0 (1.3 Гб). Кроме отличий в организации рабочего стола, в дистрибутиве задействован ряд оригинальных приложений, упрощающих настройку и работу в системе начинающих пользователей. Например, в Linux Mint используется переработанная система меню, собственный менеджер установки и обновления приложений, оригинальный интерфейс для настройки системы mintConfig, "мастер" для настройки различных параметров системы для начинающих пользователей в стиле вопрос-ответ, интерфейс для настройки параметров десктопа, средство для выполнения резервного копирования mintBackup и т.п. Ключевые особенности Linux Mint 16: В состав включены новые версии десктоп-окружений MATE 1.6 и Cinnamon 2.0, оформление и организация работы в которых продолжает развитие идей GNOME 2 - пользователю предлагается рабочий стол и панель с меню, областью быстрого запуска, перечнем открытых окон и системным лотком с запущенными апплетами. MATE продолжает развитие кодовой базы GNOME 2.32, основан на GTK2+ и полностью избавлен от пересечений с GNOME 3, что позволяет использовать традиционный рабочий стол GNOME 2 параллельно с рабочим столом GNOME 3. Cinnamon основан на GTK3+ и технологиях GNOME 3. Проект развивает оболочку GNOME Shell и оконный менеджер Mutter в направлении предоставления окружения в стиле GNOME 2 с более современным оформлением и задействованием элементов из GNOME Shell, дополняющих средства классического десктопа. Новая версия Cinnamon примечательна уходом от использования GNOME в качестве зависимости и переходом к поставке полностью обособленного окружения, не пересекающегося с GNOME. Технически Cinnamon по прежнему основывается на компонентах GNOME, но эти компоненты теперь поставляться как периодически синхронизируемый форк GNOME, не связанный внешними зависимостями с GNOME. Кроме того, в Cinnamon 2.0 улучшены режимы закрепления окна с краю или в углу, добавлена поддержка воспроизведения звуков при наступлении типовых событий, реализован новый интерфейс для управления пользователями и группами, расширены возможности файлового менеджера Nemo (форк Nautilus 3.4); До версии 1.4 обновлён используемый в дистрибутиве дисплейный менеджер MDM, представляющий собой форк GDM 2.20. MDM предоставляет графические инструменты настройки параметров входа, возможность использования визуальных тем, средства организации удалённой работы, подключения своих скриптов для обработки событий, режим автоматического входа в систему, возможность выбора языка. В новой версии MDM обеспечена упрощённая поддержка переключения между сеансами пользователей, значительно увеличена производительность, модернизирован интерфейс пользователя, обеспечен выбор по умолчанию последнего активного пользователя, добавлена поддержка Num-Lock. В качестве альтернативного менеджера входа возможно использование LightDM; Подготовлена новая графическая утилита для быстрого форматирования USB-накопителей. В качестве файловых систем поддерживаются NTFS, FAT32 и ext4. Как и интерфейс записи iso-образов, новая утилита полностью интегрирована с файловым менеджером Nemo и может быть вызвана из контекстного меню при клике правой кнопкой мыши на USB-накопителе; Проведена работа по оптимизации производительности различных компонентов дистрибутива. При загрузке и входе больше не осуществляется сканирования наличия разделов Btrfs. В MDM по умолчанию запрещены коммуникации по сети. Менеджер обновлений запускается с определённой задержкой, давая другим компонентам сеанса пользователя загрузиться в первую очередь. Заметно ускорена работа Software Manager. В ISO-образе теперь не используется сжатие, что привело к небольшому увеличению размера, но позволило увеличить скорость запуска сеанса в Live-режиме. Существенно увеличена производительность полноэкранных приложений в Cinnamon 2.0, например, тест "glxgears -fullscreen" без vsync демонстрирует в три раза более высокий FPS; Обновлён менеджер установки приложений. Добавлена поддержка отображения нескольких скриншотов для выбранного приложения. Ускорен запуск программы, сокращено потребление памяти и увеличена производительность поиска программ. В менеджере установки приложений отдельная секция заведена для клиента сервиса доставки игр Steam и связанных с ним дополнений; В новой теме оформления Mint-X улучшена поддержка GTK3+, сокращены отличия во внешнем виде приложений на базе GTK2 и GTK3, реализованы более сглаженные границы окон, добавлены новые пиктограммы и обеспечена поддержка функций прикрепления окон к границам экрана, появившиеся в Cinnamon 2.0; Из системных изменений отмечается реализация более безопасных обновлений ядра, ускорения загрузки, улучшение поддержки работы на системах с EFI, в APT по умолчанию отключена категория рекомендованных пакетов (Recommends), улучшены настройки цветов в терминале, добавлен алиас "ll" для команды "ls -al".
