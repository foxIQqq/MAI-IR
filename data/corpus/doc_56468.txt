Выпуск Chrome OS 97

Опубликован релиз операционной системы Chrome OS 97, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 97. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 97 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 97: Реализована возможность проверки грамматики в offline-режиме, что позволяет проверять правописание в web-приложениях и различных полях ввода при отсутствии подключения к сети. Реализована возможность перемещения области для увеличения содержимого экрана (Magnifier) вслед за движением мыши (Chrome OS Settings > Advanced > Accessibility > Manage accessibility features > Enable fullscreen magnifier). В приложение для просмотра галереи фотографий (Gallery) добавлена возможность воспроизведения звука и открытия нескольких окон для одновременного просмотра или редактирования нескольких мультимедийных файлов. Добавлен API Management Telemetry для отправки данных телеметрии c централизованно управляемых устройств (в корпоративных системах). Например, поддерживается передача сведений о состоянии CPU, памяти, хранилищах и графической подсистеме. Переработан интерфейс с настройками отправки информации (например, отчётов о сбоях). Пользователь теперь может видеть все данные, используемые при отправке отчётов. В API Сrome Policy, позволяющий администраторам управлять устройствами и браузерами пользователей, добавлена возможность управления настройками устройства (Device Settings). Ранее через данный API был предоставлен доступ к настройкам пользователя, параметрам вывода на печать, приложениям и дополнениям. Улучшена интеграция с внешними устройствами на базе платформы Android, например, добавлена поддержка разблокировки экрана со смартфона, быстрого сопряжения устройств, передачи контента между устройствами, организации трансляции мультимедийных данных и доступа к сообщениям.
