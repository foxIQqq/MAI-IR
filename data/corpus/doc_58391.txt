Опубликованы дистрибутивы AV Linux MX 21.2, MXDE-EFL 21.2 и Daphile 22.12

Доступен дистрибутив AV Linux MX 21.2, содержащий подборку приложений для создания/обработки мультимедийного контента. Дистрибутив собран из исходных текстов с использованием инструментария, применяемого для сборки MX Linux, и дополнительных пакетов собственной сборки (Polyphone, Shuriken, Simple Screen Recorder и т.п.). AV Linux может функционировать в Live-режиме и доступен для архитектуры x86_64 (3.9 ГБ). Пользовательское окружение основано на Xfce4. В поставку входят звуковые редакторы Ardour, Harrison Mixbus, система 3D-проектирования Blender, редакторы видео Cinelerra, Openshot, LiVES и инструменты для преобразования мультимедийных форматов файлов. Для коммутации звуковых устройств предлагается JACK Audio Connection Kit (используется JACK1/Qjackctl, а не JACK2/Cadence). Дистрибутив снабжён подробным иллюстрированным руководством (PDF, 72 стр.) В новой версии: Оконный менеджер OpenBox заменён на xfwm, менеджер обоев рабочего стола Nitrogen на xfdesktop, а менеджера входа SLiM на lightDM. Прекращено формирование сборок для 32-разрядных систем x86. Ядро Linux обновлено до версии 6.0 с патчами Liquorix. В состав включена утилита RTCQS для выявления узких мест с производительностью при работе со звуком. Добавлены плагины Auburn Sounds Lens и Socalabs, а также система 3D-моделирования Blender 3.4.0. Предложены специальные правила udev для Ardour и различных устройств. Добавлены новые пиктограммы Evolvere и обновлена тема оформления Diehard. Обновлены версии ACMT Plugin Demos 3.1.2, Ardour 7.2, Audacity 3.2.2, Avidemux 2.8.1, Cinelerra-GG 20221031, Harrison Mixbus 32C 8.1.378 Demo, Kdenlive 22.12.0, Musescore 3.6.2, Reaper 6.71 Demo, Yabridge 5.0.2. Одновременно сформирован выпуск сборки MXDE-EFL 21.2, основанной на наработках MX Linux и поставляемой с рабочим столом на основе окружения Enlightenment. Проект развиваемой разработчиками AV Linux и позиционируется как сборка, экспериментирующая с переводом AV Linux с рабочего стола Xfce на Enlightenment. Сборка содержит базовые оптимизации и настройки AV Linux, но отличается меньшим набором специализированных приложений. Размер live-образа 3.8 ГБ. В новой версии: Ядро Linux обновлено до версии 6.0 с патчами Liquorix. Пользовательское окружение обновлено до Enlightenment 0.25.4. Отключён модуль Procstats, имеющий проблемы со стабильностью. Внесены изменения в тему оформления. Добавлена панель с мультимедийными приложениями Shelf. Перенесены специфичные для дистрибутива AV Linux MX утилиты. Добавлены приложения Desktop Icons и Appfinder. Обновлены версии Blender 3.4.0, Ardour 7.2, Audacity 3.2.2, Avidemux 2.8.1, Cinelerra-GG 20221031, Kdenlive 22.12.0, Reaper 6.71, Yabridge 5.0.2. Дополнительно можно отметить релиз дистрибутива Daphile 22.12, основанного на Gentoo Linux и предназначенного для создания системы для хранения и воспроизведения музыкальной коллекции. Для обеспечения максимального качества звука поддерживается подключения компьютера с Daphile к аналоговым усилителям через цифро-аналоговые преобразователи с интерфейсом USB, среди прочего для создания мультизональных аудиосистем. Дистрибутив также может работать в режиме звукового сервера, сетевого хранилища (NAS, Network-Attached Storage) и точки беспроводного доступа. Поддерживается воспроизведение со внутренних накопителей, сетевых потоковых сервисов и внешних USB-накопителей. Управление производится через специально созданный web-интерфейс. Предлагается три сборки: x86_64 (278 МБ), i486 (279 МБ) и x86_64 с компонентами для работы в режиме реального времени (279 МБ). В новой версии: В CD Ripper добавлен редактор метаданных. Добавлена возможность изменения настроек звукового устройства без перезагрузки. Добавлена поддержка резервного копирования и восстановления настроек дистрибутива. Добавлен экран Now Playing, доступный через вкладку Audio Player или по ссылке http://адрес/nowplaying.html Обновлены версии ядра Linux 5.15.83-rt54, LMS 8.3 и Perl 5.34. Для сборки задействован GCC 11.3.
