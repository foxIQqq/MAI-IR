Выпуск Альт Сервер 10.1

Состоялся релиз дистрибутива Альт Сервер 10.1, построенного на 10 платформе ALT (ветка p10 Aronia). Дистрибутив поставляется в рамках Лицензионного договора, который предоставляет возможность свободного использования физическими лицами, но юридическим лицам допускается только тестирование, а для использования требуется приобрести коммерческую лицензию или заключить лицензионный договор в письменной форме. Установочные образы подготовлены для архитектур x86_64, AArch64 и «Эльбрус». Новые возможности и инструменты: Добавлен сертификат Минцифры России (ca-certificates-digital.gov.ru 1.0). Уделено внимание поддержке Microsoft AD и единым групповым политикам. ОС «Альт Сервер» используется в альтернативном решении компании «Базальт СПО» по реализации доменной инфраструктуры Active Directory под Linux. Обновлён контроллер домена Samba DC 4.16 и инструменты управления групповыми политиками - gpupdate 0.9.12.2, admc 0.11.2, gpui 0.2.17. К другим admx-шаблонам в образе добавлен пакет admx-msi-setup для загрузки на контроллер домена групповых политик Microsoft. Добавлен текстовый режим работы загрузчика GRUB. Стало удобней устанавливать систему на сервер без видеокарты благодаря отдельному пункту в меню загрузчика. При установке системы предоставлена возможность создания разделов с NTFS. Разрешено использование кириллицы в именах или паролях пользователей; Улучшено переключение и расположение индикатора раскладки клавиатуры (greeter). Добавлены virt-manager 3.2 для создания виртуальных машин, сервер удаленного подключения tigervnc 1.10 и система мониторинга Zabbix 6.0. Добавлен инструмент системного администратора Epm (eepm-yum). Вместо пакетов wine-vanilla задействована сборка Wine 7.22 для дистрибутивов «Альт». Добавлен sssd-ldap 2.8. Улучшения и изменения: В дистрибутиве собраны пакеты актуальных версий для системного окружения на базе ядра Linux 5.10. Добавлено ядро 5.15. PHP 7 заменён на PHP 8. В дистрибутив включены пакеты c PostgreSQL 14 в вариантах с патчами 1С и без них. В образ добавлены пакеты с модулями для соответствующих ядер, чтобы не зависеть от сетевого репозитория при установке дополнительных драйверов. При загрузке установщика в режиме Legacy используется grub-pc 2.06. Таким образом, и в Legacy, и в UEFI используется GRUB. Функциональность, аналогичная alt-p10-builder, достигается выбором в установщике опции «Сборщик пакетов» - в образ будет установлен пакет builder-useradd 1.5. Внесены корректирующие изменения с целью ускорить процесс установки в минимальном профиле. В установщик в группу «Серверные приложения» добавлен пункт меню «Поддержка токенов» (isbc-pkcs11 4.9, opensc, pcsc-tools, openssl, pam_pkcs11). Удалены пункты установщика «Mediawiki» и «ALT Domain Server (ldap kerberos)». Настроена работа multipath (0.9) по умолчанию аналогично Серверу Виртуализации. Обновлён брокер подключений и туннель для создания и управления виртуальными рабочими местами OpenUDS 3.5.0. Предусмотрено создание SWAP-раздела по умолчанию. Образ поддерживает возможность создания Btrfs-раздела (btrfs-progs 5.19) на этапе установки. Образ AArch64 содержит модуль ядра xor-neon, что позволяет установить систему на RAID4/5/6 для повышения отказоустойчивости и производительности. В AArch64 для работы в сети интернет выбран web-браузер Chromium 107.0. Jitsi Meet несовместим с AArch64. Для настройки серверов HAProxy установлен haproxy 2.6 и keepalived 2.2. Рабочая среда MATE обновлена до версии 1.26, а LibreOffice обновлён до версии 7.4. Для определения сканеров (Samsung ProXpress M3870FD, Pantum M7100DN, Samsung ProXpress M3870FD, Pantum M7100DN, Oki MB472, Kyocera ECOSYS-M2040dn, Brother MFC L2720DW) установлен sane-airscan.
