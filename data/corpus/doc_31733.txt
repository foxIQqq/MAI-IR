Вышел сентябрьский номер журнала UserAndLINUX

Представлен двенадцатый номер электронного журнала UserAndLINUX (PDF, 9 Мб, 50 стр.). Журнал предназначен для обычных пользователей, для тех, кто использует компьютер для повседневной деятельности и хочет познакомиться с миром СПО. Тема номера – Образование. Среди статей: Интервью с основателями свободного комплекса программ для автоматизации торговли, налогового и бухгалтерского учета iceB – Александром и Андреем Белых; Особенности выбора обучающих программ для детей; Linux для детей: образовательные программы и комплексы; Несколько статей с отзывами про школьные дистрибутивы ALT Linux; Выбор ОС для школьных компьютеров минимальной конфигурации: Тест ALT Linux Desktop 5 KDE против Windows 7 на Celeron-III; Переводим компьютерный класс на Linux за 3 дня.
