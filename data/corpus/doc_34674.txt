Представлены результаты портирования Blender для платформы Android

Александр Кузнецов представил рабочий прототип порта системы 3D-моделирования Blender для платформы Android, запуск которого был продемонстрирован на телефоне Samsung. Работа была выполнена в рамках программы Google Summer of Code 2012. В процессе выполнения задания была проведена адаптация кода Blender Game Engine для работы с OpenGL ES 2.0, выполнено частичное портирование Blender для Android и создано простое средство для оформления пакетов игр на базе движка Blender для платформы Android. Конечной целью разработки является полное портирование всех компонентов Blender на платформу Android и формирование пригодной для конечных пользователей сборки.
