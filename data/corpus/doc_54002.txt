Выпуск дистрибутива OpenIndiana 2020.10, продолжающего развитие OpenSolaris

Опубликован релиз свободного дистрибутива OpenIndiana 2020.10, пришедшего на смену бинарному дистрибутиву OpenSolaris, развитие которого было прекращено компанией Oracle. OpenIndiana предоставляет пользователю рабочее окружение, построенное на базе свежего среза кодовой базы проекта Illumos. Непосредственно разработка технологий OpenSolaris продолжается проектом Illumos, в котором развивается ядро, сетевой стек, файловые системы, драйверы, а также базовый набор пользовательских системных утилит и библиотек. Для загрузки сформировано три вида iso-образов - серверная редакция с консольными приложениями (877 МБ), минимальная сборка (345 МБ) и сборка с графическим окружением MATE (1.6 ГБ). Основные изменения в OpenIndiana 2020.10: Встроена поддержка гипервизора BHyVe, который ранее был перенесён в основной состав кодовой базы illumos-gate из OmniOS. На базе BHyVe реализован новый тип изолированных зон в OpenIndiana. В состав включены приложения для удалённой работы с рабочим столом FreeRDP и Remmina. Добавлена Squeak Virtual Machine, виртуальная машина для запуска программ на языке Smalltalk. Рабочий стол MATE обновлён до выпуска 1.24. Набор компиляторов обновлён до ветки GCC 10. Обновлены пользовательские пакеты: VirtualBox 6.1.16, VLC 3.0.11.1, qt5 5.12.9, nano 5.3, emacs 27.1. Обновлены системные компоненты и библиотеки : mozilla-nspr 4.29, mozilla-nss 3.51.1, sudo 1.9.3p1, SQLite to 3.33. Обновлены инструменты для разработчиков: GCC 10.2.0, Python 3.7.7/3.5.10, Golang 1.14.81.15.3, PHP 7.3.21, Git 2.29.2. Обновлено серверное ПО: PostgreSQL 12, MariaDB 10.3.24/10.1.4, Percona 5.6.49.89.0/5.7.31.34, Apache httpd 2.4.46, Tomcat 8.5.59, Samba 4.12.5, Node.js 14.15.0, BIND 9.16.6, tor 0.4.4.5.
