Корректирующий релиз системы виртуализации VirtualBox 4.2.6

Компания Oracle выпустила корректирующий релиз системы виртуализации VirtualBox 4.2.6, в котором отмечено 41 исправление. Одновременно вышли обновления и для прошлых, ещё поддерживаемых, веток - 3.2.16, 4.0.18 и 4.1.24. Среди изменений: Устранена серия ошибок, приводящих к крахам при различных редких обстоятельствах; Решены проблемы с нарушением целостности конфигурации VM после отмены снапшотов; Налажен процесс определения дисковых образов по UUID; Исправлена порция ошибок, связанных с поддержкой 3D; Устранены зависания при работе с образами QCOW в асинхронном режиме и при открытии образов в новом формате VHD; Добавлены метрики для оценки использования диска и нагрузки на сеть; В VBoxManage налажен процесс преобразования дисков из raw-образа и добавлена опция "storageattach --medium" для монтирования дополнительных образов; Добавлена поддержка импорта файлов OVF, созданных в новых версиях продуктов VMware; Для Linux хоста устранена утечка соединений в таблицах conntrack; В сервис VBoxService vminfo, поставляемый в пакете дополнений для гостевых систем Linux Additions, добавлена поддержка сессий ConsoleKit; Устранён крах гостевых систем c дополнениями Linux Additions в процессе перемонтирования разделов при определённом стечении обстоятельств; В пакеты дополнений для Linux и Solaris добавлена поддержка метрик использования памяти гостевой системой.
