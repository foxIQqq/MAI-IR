Вышел ArtistX 1.0, LiveDVD дистрибутив Linux для создания мультимедиа контента

Вышел релиз LiveDVD-дистрибутива ArtistX 1.0, представляющего собой переносную студию для создания мультимедиа контента. В состав ArtistX включено около 2500 различных мультимедиа приложений, дистрибутив претендует на роль одного из самых полных наборов бесплатных мультимедиа программ. Версия ArtistX 1.0 основана на пакетной базе Ubuntu 10.04, в состав входит большинство мультимедиа пакетов Ubuntu, в качестве пользовательского окружения предлагается использовать GNOME 2.30 или KDE 4.5. Размер iso-образа 3.7 Гб. Из входящих к комплект программ, можно отметить: Программы для работы с 2D графикой: Gimp, Inkscape, Nip2, Krita, Synfig, Rawstudio, Skencil, Hugin; Программы 3D-моделирования: Blender, Wings3D, K3D; Программы для обработки видео: Cinelerra, Openshot, Kino, Openmovieeditor, Kdenlive, Pitivi, Avidemux, Devede; Аудио и видео плееры: Mplayer, Videolan, Xine, Kaffeine, Kmplayer, LastFM; Программы для обработки и создания музыки: PD and externals, Rosegarden, Ardour, TerminatorX, Cecilia/Csound. Диск сформирован при помощи пакета remastersys, позволяющего создать резервную копию уже установленного и настроенного Debian-совместимого дистрибутива, которая будет сформирована в виде загрузочного образа, который может функционировать как в Live-режиме, так и в режиме восстановления системы на диске. Иными словами, используя remastersys можно записать текущую систему со всеми пользовательскими данными на DVD и использовать ее на другом компьютере для демонстрации или в качестве временного рабочего окружения.
