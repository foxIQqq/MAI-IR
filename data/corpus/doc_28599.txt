Firefox празднует свое шестилетие

Проект Mozilla празднует шестилетие с момента выхода первого релиза web-браузера Firefox. Идея построения браузера с интерфейсом на языке XUL возникла в 2002 году, после чего потребовалось около двух лет на подготовку первого стабильного релиза. Для проекта вначале было выбрано имя Phoenix, но из-за пересечения с уже зарегистрированной торговой маркой, браузер был переименован в Firebird, что вызвало недовольство со стороны разработчиков одноименной свободной СУБД, после чего проект было решено вторично переименовать, теперь уже в Firefox. В настоящий момент браузером Firefox пользуется около 400 миллионов пользователей по всему миру, интерфейс браузера локализован на 70 языков, для расширения возможностей разработано около 11 тысяч дополнений, которыми пользуются более 150 миллионов человек. Статистика популярности браузеров (в скобках данные за прошлый год): Доля Firefox по данным сервиса XiTi Monitor летом составляла 30.6% (год назад было 28.4%) против 53.8% (62%) у IE, 6.8% (2.3%) у Google Chrome и 2.4% (2.2%) у Opera; В соответствие с отчетом общемирового рейтинга Net Applications, IE занимает 59.18% (64.64%) рынка, Firefox - 22.83% (24.07%), Chrome - 8.5%, Opera - 3.24%. По данным ресурса StatCounter, Firefox 3 на сегодняшний день отвоевал 31.43% (32.06%) общемирового рынка web-браузеров, Microsoft Internet Explorer занимает 52.94% (57.13%), Chrome - 8.74%, Opera - 1.95%. На европейском рынке Firefox завоевал 38.75% (40.01%), вплотную приблизившись к IE, занимающему 43.23% (46.87%), Chrome - 8.74%, Opera - 4.36%; По последним данным глобальной статистики spylog.ru, Firefox занимает 27.76% (25.04%) рынка, Opera - 25.23% (23.85%), Internet Explorer - 33.33% (42.9%), Chrome - 7.65%; По статистике liveinternet.ru, Firefox занимает 23.5% (20%), Opera - 35.9% (33.7%), Internet Explorer - 31.3% (41.5%), Chrome - 6.6%. На opennet.ru доля пользователей с Firefox составляет 46.15% (49.10%), Opera - 21.27% (26.58%), Internet Explorer - 9.23% (15%), Google Chrome - 19.49% (9.62%).
