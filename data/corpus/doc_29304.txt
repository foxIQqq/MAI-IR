Релиз дистрибутива Clonezilla Live 1.2.6-59

Вышел стабильный релиз Linux дистрибутива Clonezilla Live 1.2.6-59, предназначенного для быстрого клонирования дисков (копируются только используемые блоки). Задачи выполняемые дистрибутивом сходны с проприетарным продуктом Norton Ghost. Дистрибутив основан на Debian GNU/Linux, в своей работе использует код проектов DRBL, Partition Image, ntfsclone, partclone, udpcast. Возможна загрузка с CD, USB Flash и по сети (PXE). Поддерживаются ФС: ext2, ext3, ext4, reiserfs, reiser4, xfs, jfs, FAT, NTFS, UFS, HFS+ (Mac OS X). Поддерживается режим массового клонирования в multicast режиме, например, при тестировании, за 10 минут удалось провести клонирование исходного 5.6 Гб диска на 41 клиентскую машину. Размер iso-образа дистрибутива - 124 Мб. По сравнению с прошлым выпуском отмечены следующие улучшения: Проведена синхронизация с пакетной базой Debian Sid по состоянию на 13 января; Linux-ядро обновлено до версии 2.6.32-30; Обновлены компоненты Live-boot 2.0.14-1.drbl1 и live-config 2.0.14-1.drbl2. Обновлены файлы с переводами; В экспертном режиме добавлена опция "-q1", принудительно указывающая на использование утилиты "dd" при выполнении операции по клонированию дисков (полезно когда требуется скопировать зашифрованный раздел); В состав включен пример по формированию индивидуальной сборки custom-ocs-2, который может быть использован для сохранения и восстановления систем с двойной загрузкой.
