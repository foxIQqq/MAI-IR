Релиз Linux-дистрибутива Mageia 2, в рамках которого развивается форк Mandriva

После почти года разработки увидел свет второй релиз Linux-дистрибутива Mageia, в рамках которого независимым сообществом энтузиастов развивается форк проекта Mandriva. Для загрузки доступны 32- и 64-разрядные DVD-сборки (3.5 Гб), универсальная CD-сборка (692 Мб), минималистичный образ для установки по сети (40 Мб) и набор LiveCD на базе GNOME и KDE с различными наборами локализации. Ключевые улучшения: Проведено обновление версий программ в составе дистрибутива. Для оценки изменений по сравнению с первым релизом Mageia подготовлены таблицы с номерами версий основных приложений и всех пакетов. Среди ключевых десктоп-приложений, использованных в дистрибутиве отмечаются: Десктоп-окружения KDE 4.8.2, GNOME 3.4.1, Xfce 4.8.3, LXDE, Sugar 0.95, а также минималистичное десктоп-окружение на базе Qt - Razor-qt 0.4.1. В репозиториях также доступны для установки оконные менеджеры dwm, scrotwm, i3, pekwm, icewm, awesome, enlightenment (e17), openbox, fvwm2 и Window Maker; Менеджеры фотографий Digikam и Showfoto 2.6; Медиаплееры: VLC 2.0.1 и xine 1.1.19 (+ GUI kaffeine и xine-ui) XBMC 11.0, MythTV 0.25, Miro 4.0.6; Браузеры: Firefox ESR 10.0.4, Chromium 18 и Opera 11.64 ; Офисные пакеты: LibreOffice 3.5, Calligra 2.4.1; Коммуникационные программы: Pidgin 2.10.4, Kopete 4.8.2, Empathy 3.4.1, Ekiga 3.3.2; Проприетарные приложения: Adobe Flash 11.2, Skype, драйверы для видеокарт NVIDIA 295.49 и AMD Catalyst 12.4; Растровый графический редактор GIMP 2.8; Музыкальные плееры: Amarok 2.5, Cantata music player 0.3, kdenlive 0.8.2.1, mplayer (+ GUI gnome-mplayer, kmplayer, mplayer-gui, smplayer); Интеграция системного менеджера systemd. Возможность загрузки с классической системой инициализации сохранена для тех, кто хочет повременить с переходом на systemd; Обновление ядра Linux до версии 3.3.6; Переход на использование звукового сервера PulseAudio 2.0. В качестве причины перехода на PulseAudio 2.0 называется желание задействовать добавленных в ветку ядра 3.3 возможностей по определению аудиоразъёма при подключении наушников, что даст возможность автоматически перенаправить звуковой поток и откорректировать громкость; Переработка инсталлятора. Минимальный размер установки сокращён с 530 Мб до 325 Мб. Значительно расширена поддержка оборудования, в том числе возможность установки с использованием wifi-соединения. Добавлена поддержка установки с виртуальных дисков. Инсталлятор адаптирован для архитектур MIPS и ARM. Улучшена поддержка brtfs, LVM, nilfs2, XFS; Обновлена версия пакетного менеджера RPM до ветки 4.9x; Для создания загрузочных ramdisk-образов (initrd) вместо mkinitrd задействована утилита dracut, развиваемая проектом Fedora Linux; В состав включена система организации удалённого доступа к рабочему столу X2Go: x2goserver 3.1.0.1, x2goclient 3.99.1.1 и pyhoca-gui 0.1.0.10. Кроме того, в состав включены библиотеки и компоненты Nomachine NX 3.5.0.11; Вместо MySQL предлагается использовать MariaDB 5.5.23 (MySQL убран из поставки). В репозиториях также доступны Firebird 2.5.1 и PostgreSQL 8.4.11, 9.0.7 и 9.1.3; Обновление серверного ПО: apache 2.2.22, nginx 1.0.15, postfix 2.8.9, Cyrus-imapd 2.4.13, Dovecot 1.2.17, drupal 7.11, mediawiki 1.16.5, puppet 2.7.11, Samba 3.6.5; Средства виртуализации: VirtualBox 4.1.12, virt-manager 0.9.1; Обновление системных компонентов и средств разработки: glibc 2.14.1, gcc 4.6.2, python 2.7.3/3.2.2, perl 5.14.2, ruby 1.8.7, rails 2.3.14, php 5.3.13, java openjdk-1.6.0.0, sun-java-1.6.0.26. IDE: eclipse 3.7.1, netbeans 7.0.1, anjuta 3.3.90, ninja-ide Python IDE 2.0;
