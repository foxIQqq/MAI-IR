Релиз программы для настройки сетевых соединений Wicd 1.7

Вышел релиз Wicd 1.7, менеджера для настройки проводных и беспроводных соединений, выступающего в роли альтернативы Network Manager. После выпуска версии 1.7 разработчики сообщили, что это будет последняя ветка в серии 1.x, в дальнейшем развитие проекта будет сосредоточено на разработке Wicd 2.0. В новой версии: Реализована новая утилита wicd-cli для управления из командной строки и использования в скриптах; Добавлен вывод информации о состоянии соединения при клике правой кнопкой мыши на иконке Wicd в системном трее; Возможность привязки имен хостов к подсетям для всех DHCP-клиентов; Поддержка версии 0.99 консольной библиотеки urwid; Поддержка использования только цифровых ESSID; В качестве предпочтительного метода запуска GTK GUI представлен wicd-gtk; Для основанных на Debian GNU/Linux систем обеспечена поддержка dhcpcd-bin. Из особенностей Wicd можно отметить: Не зависит от GNOME и GConf, хорошо работает в разных десктоп-окружениях, включая XFCE, Fluxbox, Openbox, Enlightenment. Поддержка всех популярных Linux дистрибутивов, например, Ubuntu, Debian, Arch, Fedora, Slackware и Gentoo Совместимость с wireless-tools, поддержка настройки WEP/WPA/WPA2/LEAP/TTLS/EAP/PEAP; Возможность сворачивания в трей и отображения качества сигнала и интенсивность трафика; Поддержка создания нескольких профилей для разных проводных и беспроводных соединений; Кроме GUI интерфейса имеется возможность полного управления в консольном режиме через программу wicd-curses.
