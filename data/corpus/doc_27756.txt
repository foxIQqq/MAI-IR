Компания Google открыла исходные тексты серверной JavaScript-платформы Acre

Компания Google объявила о переводе в разряд свободных продуктов платформы Acre, предназначенную для разработки web-приложений на языке JavaScript. Особенностью Acre является то, что приложения на JavaScript выполняются на стороне сервера. Код Acre распространяется в рамках лицензии Apache License 2.0. В Google система Acre используется для обеспечения функции разработки работающих на стороне сервера приложений Freebase Apps, расширяющих возможности сервиса Freebase, представляющего собой репозиторий структурированных данных, в котором содержится около 12 миллионов сущностей и около 400 млн. связанных с ними фактов. Код Acre открыт спустя месяц с момента покупки Google поддерживающей данный сервис компании. Из отличительных черт системы можно отметить средства для организации совместного использования кода (разработчик одного приложения может запросить модуль написанный для другого приложения). Открытый вариант Acre полностью совместим с Freebase API и позволяет создать систему выполнения серверных JavaScript-приложений на своих мощностях. Для выполнения JavaScript кода в Acre используется поддерживаемый разработчиками Mozilla JavaScript-интерпретатор Rhino, написанный на языке Java, который совмещается с контейнером сервлетов Jetty.
