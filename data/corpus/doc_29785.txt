Релиз дистрибутива Scientific Linux 6.0

Анонсирован релиз дистрибутива Scientific Linux 6.0, построенного на пакетной базе Red Hat Enterprise Linux 6.0 и дополненного средствами, ориентированными на использование в научных учреждениях. Дистрибутив поставляется в сборках для архитектур i386 и x86_64, для загрузки доступен установочный DVD (3.6 Гб). Отличия от пакетной базы RHEL 6 в первом выпуске Scientific Linux 6 минимальны. В отличии от прошлых веток, число дополнительных программ невелико, тем не менее большинство ранее доступных в Scientific Linux 5 научных приложений можно найти в дополнительных репозиториях, таких как EPEL, elrepo.org и atrpms.net. Также прекращено поддержание репозитория contrib, в котором размещались дополнительные драйверы устройств - аналоги пакетов из contrib можно найти во внешних репозиториях elrepo и atrpms. Минимальное число дополнительных пакетов и их вынос во внешние репозитории разработчики объясняют желанием сократить двойную работу и уменьшить число проблем с зависимостями. Установочный образ Scientific Linux 6 по составу повторяет RHEL 6 и не содержит свежих обновлений, которые следует установить после установки дистрибутива. Из особенностей Scientific Linux 6 (SL6) можно отметить: В SL6 состав включены дополнительные пакеты: Оконный менеджер IceWM 1.2.37; OpenAFS 1.5.1 - открытая реализация распределенной ФС Andrew File System; Пакет визуализации Graphviz; Библиотека Numpy, для организации научных расчетов; Для драйвера iwlwifi включены дополнительные прошивки (firmware для беспроводных карт Intel серий 5150, 3945, 4945, 5000 и, начиная с текущей версии, 6000. Поддержка языка Lua; Поддержка файловой системы XFS для архитектуры x86_64. В программе установки пока нельзя создать XFS раздел, но установить Scientific Linux на ранее созданный XFS раздел можно; FUSE-модули fuse-smb и fuse-sshfs, для монтирования SMB и SSH. Ранее присутствующие в SL5 пакеты, которые можно найти во внешнем репозитории EPEL: Библиотека cfitsio с реализацией поддержки формата FITS (Flexible Image Transport System), широко используемого в астрономических исследованиях, для языков Си и FORTRAN; Технология DKMS (Dynamic Kernel Module Support), позволяющая автоматически перестраивать текущие модули ядра с драйверами устройств после обновления версии ядра. Внедрение DKMS позволит автоматически устанавливать драйверы, которые необходимы для работы определенных аппаратных компонент текущей системы, но отсутствуют в стандартном Linux ядре. Для разработчиков, DKMS предоставляет средства создания драйверов без оглядки на особенности различных версий Linux ядра. Консольный почтовый клиент Alpine; Дополнительные Perl-модули: DBD-XBase, MailTools, Parse-RecDescent, SQL-Statement, TermReadKey, Text-CSV_XS, Text-Template, Tk; Ранее присутствующие в SL5 пакеты, которые можно найти во внешнем репозитории atrpms: Математическая библиотека FFTW (Fastest Fourier Transform); Утилита 915resolution для модификации видео BIOS для графических карт на базе GPU Intel 845G, 855G, 865G, 915G, 915GM и 945G, что позволяет добиться установки дополнительных разрешений экрана; GNU R (язык для статистических расчетов); Дополнительные мультимедиа кодеки (gstreamer-plugins-extras из SL4 и k3b-extras из репозитория livna); Библиотека матричных вычислений SuiteSparse; Ранее присутствующие в SL5 пакеты, которые придется загрузить с сайта разработчика: Научная библиотека SciPy, которую можно использовать для научных, инженерных и математических рассчетов, в также для построения графиков и диаграмм, на основе набора статистических данных; Sun Java/JDK - в составе теперь поставляется только OpenJDK; Набор обучающих приложений для KDE - KDEEDU.
