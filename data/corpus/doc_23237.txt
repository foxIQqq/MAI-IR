Вышел финальный релиз web-браузера Opera 10

После почти года разработки анонсирован выход финального релиза web-браузера Opera 10. Три главных новшества Opera 10: Полностью переработанное оформление пользовательского интерфейса. В работу был вовлечен известный британский дизайнер Джон Хикс (Jon Hicks), ранее создавший логотип Firefox; Поддержка визуального режима переключения табов, при котором мини-изображение содержимого страницы отображается непосредственно на вкладках. Переключение между визуальным и стандартным режимом производится в один клик, имеется возможность изменение размера поля со списком табов. Турбо режим, позволяющий увеличить скорость загрузки данных для медленных соединений за счет использования сжатия при работе через прокси-серверы компании Opera. Другие улучшения: Функциональность для выполнения автообновления, без необходимости переустановки браузера; Поддержка виджетов в сборке для платформы Linux/UNIX. Размещение прозрачных виджетов на десктопе поддерживается только при использовании композитного менеджера и видеорежиме с 32-битным цветом; В Speed Dial появилась новая система настройки содержимого окна, добавлена возможность определения фона и указания более 9 сайтов (режимы 4 x 4 и 5 x 5); Средство для проверки орфографии в web-формах на лету, по мере ввода текста. Словарь для русского языка можно загрузить здесь; Во встроенном почтовом клиенте появилась возможность создания HTML-сообщений, с картинками, шрифтовым оформлением текста и ссылками; Новый движок Presto 2.2, позволивший реализовать: Прохождение 100 из 100 HTML/CSS/JavaScript тестов Acid3 + прохождение дополнительного теста на плавную анимацию; Новая система для обработки регулярных выражений, отличающаяся значительным увеличением производительности. Особенно данная оптимизация должна повлиять на результаты прохождения тестов SunSpider; Улучшение производительности при обработке CSS; При просмотре XML без определения стиля теперь производится автоформатирование вывода; Поддержка возможности загрузки TrueType (TTF), OpenType (OTF) и SVG шрифтов из сети. Управление загрузкой осуществляется через CSS элемент @font-face (свойства font-family и src); Поддержка цветовых моделей RGBA и HSLA в CSS3, поддержка "color: transparent"; Реализация JavaScript-селекторов Selectors API. Поддержка интерфейсов querySelectors и querySelectorsAll, позволяет значительно ускорить доступ к некоторым элементам web-страниц. Добавлена поддержка TLS 1.2 и удалена SSL v2. К сожалению, в состав релиза не вошел код проекта Opera Unite, представляющего собой интегрированный в браузер http-сервер, дающий пользователям возможность без задействования внешних ресурсов и сервисов организовать обмен файлами фотографиями и музыкой, организовать прямой прием текстовых сообщений, поднять локальный чат или просто открыть доступ к своей персональной странице.
