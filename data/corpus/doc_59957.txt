Опубликован офисный пакет ONLYOFFICE 7.5.0

Опубликован выпуск ONLYOFFICE DocumentServer 7.5.0 с реализацией сервера для online-редакторов ONLYOFFICE и организации совместной работы. Редакторы можно использовать для работы с текстовыми документами, таблицами и презентациями. Код проекта распространяется под свободной лицензией AGPLv3. Одновременно сформирован выпуск продукта ONLYOFFICE DesktopEditors 7.5, построенного на единой кодовой базе с online-редакторами. Десктоп-редакторы оформлены в виде приложений для рабочего стола, которые написаны на JavaScript с использованием web-технологий, но объединяют в одном наборе клиентские и серверные компоненты, оформленные для самодостаточного использования на локальной системе пользователя, без обращения к внешнему сервису. Для совместной работы на своих мощностях также можно использовать платформу Nextcloud Hub, в которой обеспечена полная интеграция с ONLYOFFICE. Готовые сборки сформированы для Linux, Windows и macOS. В ONLYOFFICE заявлена полная совместимость с форматами MS Office и OpenDocument. Среди поддерживаемых форматов: DOC, DOCX, ODT, RTF, TXT, PDF, HTML, EPUB, XPS, DjVu, XLS, XLSX, ODS, CSV, PPT, PPTX, ODP. Предусмотрена возможность расширения функциональности редакторов через плагины, например, доступны плагины для создания шаблонов и добавления видео с YouTube. Готовые сборки сформированы для Windows и Linux (deb- и rpm-пакеты). Основные новшества: Добавлен редактор PDF, поддерживающий заполнение PDF-форм, а также добавление аннотаций, комментариев и рисунков. Добавлены дополнительные уровни масштабирования интерфейса: 225% и 275%. Обеспечено сохранение выбранного масштаба между перезапусками. В настройки добавлена возможность просмотра прошлого уровня масштабирования. В меню "File / Open Recent" всех редакторов добавлен список недавно открытых файлов, который ранее показывался только в основном окне. Добавлено предупреждение при попытке закрытия приложения, если открыто несколько файлов. В кнопку вставки символов добавлено меню для быстрого доступа к наиболее популярным символам. Добавлена поддержка изображений в формате SVG. Добавлена настройка для скрытия панели с инструментами для редактирования уравнений. Добавлена возможность создания новых офисных файлов из локальных шаблонов в форматах DOTX и OTT. Добавлены подсказки для изображений в объектах Smart Art. Во встроенном просмотрщике предоставлена возможность открытия файлов, защищённых паролем. В правую панель добавлена поддержка редактирования точек границы AutoShape. Для вставки символов в верхнем и нижнем индексе добавлены клавиатурные комбинации Ctrl + "." и Ctrl + ",". Редактор документов Добавлена поддержка автоматического переноса части слов на другую строку. Изменена логика выделения областей. Добавлена настройка умного выделения абзацев (если при выделении пропущен символ абзаца, но будет добавлен автоматически). Добавлена настройка для удаления элементов управления содержимым (content control) при редактировании. В панели изменена логика работы кнопок "Принять" и "Отклонить" при рецензировании изменений. Редактор таблиц Добавлен режим показа только формул в ячейках. Обеспечена поддержка отслеживания связи между формулами и ячейками. Добавлена функция "SORTBY". Реализованы преднастройки границ при выводе на печать. В сводные таблицы (Pivot Tables) добавлены: опция для показа деталей, возможность установки числового формата для полей и поддержка открытия данных в новой таблице при двойном щелчке мышью на значении. При заполнении ячеек добавлено автозаполнение дней недели. Добавлена возможность перемещения строк и столбцов в режиме Drag-and-drop. Предоставлена возможность фильтрации по дате. Добавлена поддержка вставки изображений в верхние и нижние колонтитулы. В настройках обеспечено запоминание разделителя и кодировки CSV-файлов для использования в будущих сеансах. Добавлена поддержка вставки разрывов страниц для разделения страниц при печати электронной таблицы. Редактор презентаций. Изменён цвет заголовка редактора презентации. Реализована возможность использования морфинга для перехода между слайдами. Предоставлена поддержка назначения имён объектам. Добавлена возможность выбора слайда для начала нумерации. Настройки верхних и нижних колонтитулов разделены на секции, связанные со слайдом и примечаниями (Notes and Handouts). Добавлена возможность задания альтернативных описаний заполнителей слайдов. Предоставлена возможность быстрого перехода на определённый слайд через ввод его номера и нажатия Enter во время предпросмотра презентации. В редакторе форм настройки перенесены в правую панель для унификации с другими редакторами. Устранены уязвимости, позволяющие прочитать содержимое памяти при преобразовании DOC в DOCX, а также вставить внешнее изображение и обратиться к локальным ресурсам при преобразовании HTML в DOCX.
