Выпуск офисного пакета OnlyOffice Desktop 6.4

Доступен выпуск набора OnlyOffice Desktop 6.4, предназначенного для работы с текстовыми документами, таблицами и презентациями. Редакторы оформлены в виде приложений для рабочего стола, которые написаны на JavaScript с использованием web-технологий, но объединяют в одном наборе клиентские и серверные компоненты, оформленные для самодостаточного использования на локальной системе пользователя, без обращения к внешнему сервису. Код проекта распространяется под свободной лицензией AGPLv3. В OnlyOffice заявлена полная совместимость с форматами MS Office и OpenDocument. Среди поддерживаемых форматов: DOC, DOCX, ODT, RTF, TXT, PDF, HTML, EPUB, XPS, DjVu, XLS, XLSX, ODS, CSV, PPT, PPTX, ODP. Предусмотрена возможность расширения функциональности редакторов через плагины, например, доступны плагины для создания шаблонов и добавления видео с YouTube. Готовые сборки сформированы для Windows, macOS и Linux (deb- и rpm-пакеты, в ближайшее время также будут сформированы пакеты в форматах Snap, Flatpak и AppImage). OnlyOffice Desktop включает в себя недавно опубликованные online-редакторы ONLYOFFICE Docs 6.4 и предлагает следующие дополнительные новшества: Добавлена поддержка пакетных операций с комментариями. Например, теперь можно разом удалить или пометить выполненными все просмотренные комментарии. В режиме комментирования реализованы средства для настройки прав доступа пользователя. В редакторе документов добавлена опция для автоматического использования верхнего регистра для первой буквы в предложении. Добавлен новый режим рецензирования - Simple markup. Предоставлена поддержка быстрой конвертации из текста в таблицу и из таблицы в текст. В табличном процессоре реализована возможность добавления, удаления и редактирования правил условного форматирования (правила для привязки стиля оформления ячеек к содержимому). Добавлена поддержка спарклайнов - мини-диаграмм с отображением динамики изменения серии значений, предназначенных для вставки в ячейку. Добавлена поддержка импорта из файлов в форматах txt и csv. Добавлена функция автокоррекции для ссылок, автоматически заменяющая текстовые ссылки и локальные пути на гиперссылки. В табличном процессоре также предоставлена возможность запуска макроса кликом на графическом объекте, добавлена поддержка заморозки от изменения параметров панели, реализована опция для настройки показа нулей в ячейках и добавлена поддержка цепочек комментариев. В редакторе презентаций появилась визуальная история изменений в презентации, а также добавлена поддержка скрытия панели с заметками. Значительно улучшена поддержка стилей диаграмм. Добавлены стили диаграмм для людей с проблемами со зрением (например, появился специальный стиль для дальтоников). Добавлена поддержка протокола WOPI (Web Application Open Platform Interface), используемого для доступа к файлам на серверах Microsoft, Google и Nextcloud. Предоставлена возможность изменения отрисовки символов элементов списков. В управляющих элементах появилась поддержка переключения между элементами при помощи клавиши Tab и комбинации Shift+Tab. Для экранов с высокой плотностью пикселей реализована возможность увеличения масштаба интерфейса до уровней 125% и 175% (помимо ранее доступных 100%, 150% и 200%). В файле конфигурации предоставлена возможность установки темы оформления и включения режима совместного редактирования. Редакторы для мобильных устройств полностью переписаны с использованием фреймворка React. Началась поставка родной версии OnlyOffice Desktop для устройств Apple с ARM-чипом M1, в которых не используется эмулятор Rosetta 2.
