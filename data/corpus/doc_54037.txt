Ноябрьское обновление приложений KDE 20.08.3

В соответствии с внедрённым в прошлом году ежемесячным циклом публикации обновлений представлено ноябрьское сводное обновление приложений (20.08.3), развиваемых проектом KDE. Всего в рамках ноябрьского обновления опубликованы выпуски более 120 программ, библиотек и плагинов. Информацию о наличии Live-сборок с новыми выпусками приложений можно получить на данной странице. Наиболее заметные новшества: Добавлено новое приложение Arkade с коллекцией аркадных игр, разработанных с использованием фреймворка Kirigami, позволяющего создавать универсальные интерфейсы для мобильных и настольных систем. Добавлено новое приложение KGeoTag для работы с геотегами, привязанными к фотографиям. Добавлено новое приложение Neochat c реализацией системы обмена сообщениями, поддерживающей протокол Matrix. Neochat представляет собой форк программы Spectral, переписанный с использованием фреймворка Kirigami. Поддерживается работа в Windows, Android и Linux. Опубликован выпуск растрового графического редактора Krita 4.4.0, в котором появились многопоточная реализация слоёв заливки, поддержка транcформации шаблонов заливки, новые слои заливки, новые кисти и другие изменения, рассмотренные в отдельном обзоре. Доступен выпуск менеджера дисковых разделов KDE Partition Manager 4.2, который также используется в качестве основы интерфейса управления разделами в инсталляторе Calamares. В новой версии улучшена поддержка разделов с неизвестными файловыми системами, работа с точками монтирования и обработка /etc/fstab. Обновлена интегрированная среда разработки на языке R - RKWard 0.7.2. В новой версии скрипты переписаны на Python 3, обеспечено сохранения вертикальной прокрутки при обновлении интерфейса предпросмотра скриптов, в меню добавлен элемент для смены языка, добавлен мастер проверки установки, добавлена поддержка дополнений из редактора Kate. Обновлён музыкальный проигрыватель Elisa. Обновлена версия IRC-клиента Konversation 1.7.7, в которой добавлена поддержка пиктограмм участников в формате SVG и предложен вариант стандартного набора пиктограмм для тёмной темы оформления. В утилите KRename 5.0.1, предназначенной для массового переименования файлов, решена проблема с экранированием спецсимволов в именах файлов и налажена работа на экранах HiDPI. В просмотрщике изображений Gwenview решена проблема с отображением режима просмотра эскизов в отдельном окне при использовании Qt 5.15. В KDEConnect возвращена возможность отправки SMS. В просмотрщике документов Okular устранён крах при выделении текста в аннотациях. Обновлён код каталога приложений apps.kde.org и упрощена навигация с использованием разных языков.
