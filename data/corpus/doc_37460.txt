Новая версия свободной системы 3D-моделирования Blender 2.68

После двух месяцев разработки представлена новая версия свободного пакета 3D-моделирования Blender 2.68. В новой версии: Добавлены дополнительные инструменты моделирования мэшей и улучшены существующие, в том числе добавлен новый инструмент заливки сеткой, в инструменте bridge обеспечена поддержка расстановки перемычек между петлями произвольного размера, добавлена возможность рассечения сетки между двух вершин; Улучшена производительность движка рендеринга Cycles за счёт расширения возможностей по привлечению GPU для вычислений. Скорость рендеринга с использованием СUDA в некоторых тестах возросла на 25%. Добавлена поддержка NVidia Tesla K20, GTX Titan и GTX 780. С привлечением GPU теперь обеспечивается рендеринг волоc; Улучшено качество рендеринга дыма и огня, в том числе устранены квадратичные артефакты и обеспечено реалистичное отображение быстро движущихся источников дыма; Интегрированы новые инструменты для размещения и привязки меток в системе отслеживания движения (motion tracker), которые позволяют уточнять позицию метки во время когда её загораживает другой объект; С целью повышения безопасности при загрузке .blend-файлов по умолчанию отключен автоматический запуск скриптов и драйверов на языке Python (отключенная особенность могла использоваться злоумышленниками для организации выполнения python-кода при открытии файлов из непроверенных источников); Добавлено дополнение Texture Atlas для привязки теней и света ко всей сцене и дополнение Edit Linked Library для быстрого редактирования связанных ресурсов.
