Вышел релиз Midori 0.2.5, легкого web-браузера на базе движка WebKit

Представлен релиз web-браузера Midori 0.2.5, разрабатываемый участниками проекта Xfce. Midori построен на базе движка WebKit и библиотеки GTK+. В браузере имеется поддержка табов, закладок, управления сессиями, имеется панель для обращения к поисковым системам, средства для создания пользовательских меню, возможность использования скриптов и стилей, определенных пользователями, средства для фильтрации рекламы (Adblock). Поддерживаются внешние расширения, написанные на языках Си и Lua. В новой версии добавлены следующие улучшения: Реализована возможность разработки дополнений, написанных на языке Vala; При открытии сайта с корректным SSL-сертификатом адресная строка подсвечивается желтым цветом, а при некорректном сертификате - красным; Для использования браузера в режиме киоска добавлены две новые опции командной строки: --inactivity-reset, определяющая промежуток времени для обрыва сессии при неактивности пользователя в течение заданного таймаута, и --blocked-uris, позволяющая задать список разрешенных для открытия сайтов; Налажена сборка и работа браузера для платформы Windows; Поддержка управляющих жестов мышью расширена возможностью использования правой кнопки мыши; Для любителей полноэкранного режима просмотра подготовлено специально расширение, позволяющее встроить часы в строку статуса; Элементы Speed Dial теперь можно выбирать через нажатие соответствующих цифр на клавиатуре; Интерфейс для настройки сетевых параметров оптимизирован для быстрого выбора режима работы через прокси. Для работы Midori среди обязательных зависимостей присутствуют GTK+ 2.10, WebkitGTK+ 1.1.1, libXML2, libsoup 2.25.2; среди опциональных: Unique 0.9, libidn, sqlite 3.0, docutils, libnotify. Установить новую версию Midori в Ubuntu можно воспользовавшись специальным PPA-репозиторием. Дополнение: спустя неделю вышла версия Midori 0.2.6 в которой устранена проблема с отображением фреймов и добавлена возможность управления областью Speed Dial для всех визуальных тем.
