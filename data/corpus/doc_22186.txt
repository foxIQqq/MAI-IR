Первый кандидат в релизы Firefox 3.5. Конкурс для разработчиков дополнений

Вышел Mozilla Firefox 3.5 Release Candidate 1. Основные новые возможности: Доступны версии браузера на 70 языках. Улучшены настройки контроля за личными данными, включая режим приватного просмотра. Значительно улучшена скорость выполнения JavaScript за счёт нового JavaScript движка TraceMonkey. Возможность навигации с учётом географического положения пользователя. Поддержка нативного JSON и "web worker threads". Улучшения в движке рендеринга Gecko, включая спекулятивный парсинг HTML для более быстрого рендеринга страниц. Поддержка технологий HTML5: тэгов video и audio, скачиваемых шрифтов, новых элементов CSS, селекторов запросов JavaScript, offline хранения данных для HTML5 приложений и SVG трансформаций. Информация для разработчиков. Детальный обзор новшеств, добавленных с первой по третью бета версии Firefox 3.1 можно найти здесь, изменения firefox 3.5 beta 4 представлены здесь. Кроме того, можно отметить новую акцию для разработчиков расширений - "Extend Firefox 3.5", проводимую подразделением Mozilla Labs с целью стимулирования адаптации работы существующих расширений для Firefox 3.5. Опрос среди пользователей показал, что одной из основных причин задержки с обновлением устаревших версий Firefox является отсутствие поддержки новой версии браузера в используемых пользователем расширениях. Акция продлится до 2 октября, среди разработчиков, осуществивших обновление своих расширений или создавших новые расширения для firefox 3.5, будет распределена серия призов, таких как 15- и 13-дюймовые ноутбуки MacBook Pros, лицензионные версии TextMate IDE и ExpanDrive, электронные книги издательства Manning Publication, видеозаписи и книги от InformIT, комплекты атрибутики Firefox. Призы будут вручены лучшим разработчикам на основании решения жюри. Победители будут определены в 4 категориях: лучшие новые расширения, лучшие обновленные расширения, расширения для упрощения совершения покупок и расширения для игр и развлечений.
