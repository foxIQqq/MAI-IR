Проект Android-x86 выпустил сборку Android 6.0 для платформы x86

Разработчики проекта Android-x86, в рамках которого силами независимого сообщества осуществляется портирование платформы Android для архитектуры x86, опубликовали первый стабильный выпуск сборки на базе платформы Android 6.0, в которую включены исправления и дополнения, обеспечивающие бесшовную работу на платформах с архитектурой x86. Для загрузки подготовлены универсальные Live-сборки Android-x86 6.0 для архитектур x86 (532 Мб) и x86_64 (635 Мб), пригодные для использования на типовых ноутбуках, нетбуках и планшетных ПК. Особенности нового выпуска: Поддержка 32- и 64-разрядных ядра и пользовательского окружения; Поддержка аппаратного ускорения OpenGL ES 3.x для чипов Intel, AMD (radeon/radeonsi) и NVIDIA (nouveau). Поставка 3D-драйверов для VMware и QEMU (virgl); Поддержка программной отрисовки OpenGL ES 3.0 для неподдерживаемых GPU или устройств без GPU; Поддержка режима UEFI Secure Boot и установки на диск UEFI; Возможность установки на ФС ext4, ntfs и fat32 при помощи интерактивного текстового инсталлятора; Улучшение процесса обновления с прошлых выпусков android-x86, в том числе обеспечена автоматическая конвертация разделов ext2 и ext3 в ext4; Добавлена функция автоматического монтирования разделов ext4, ntfs, exfat, vfat; Поддержка файловой системы F2FS, оптимизирванной для Flash-накопителей; Поддержка мультитач, звука, Wi-fi, Bluetooth, сенсоров, камеры и Ethernet (настройка по DHCP); Автоматическое монтирование USB-накопителей и SD-карт; Поддержка запуска приложений для иных архитектур (arm, arm64), реализованная через специальную транслирующую прослойку (Settings / Apps Compatibility); Начальная поддержка передачи звука по HDMI; Изменение формы курсора для трекпада (вместо круга теперь выводится нормальный указатель в виде стрелки, как для мыши); Решены проблемы с переходом в спящий и ждущий режимы, вызванные драйверами для беспроводных устройств; Обновление ядра Linux до версии 4.4.20; Обновление Mesa до выпуска 12.0.2; В состав включён компонент Android-x86 Analytics Program, позволяющий по желанию активировать отправку анонимных сведений об используемом оборудовании и работе с приложениями с целью выработки приоритетных направлений в разработке. Одновременно сформировано обновление окружения Remix OS 3.0.204, основанное на Android-x86 6.0 r1. Remix OS предоставляет полностью переработанный интерфейс Android с заменой методов работы, специфичных для мобильных устройств, на решения, традиционные для рабочего стола c полноценной поддержкой управления с использованием мыши и клавиатуры. Поддерживается многооконный режим c возможностью изменения размера окон и кнопками раскрытия и сворачивания окон, реализованы меню приложений, контекстные меню, системный лоток, панель инструментов, традиционный файловый менеджер и боковая панель вывода уведомлений.
