Вышла бета версия интегрированной среды разработки NetBeans IDE 6.7

Вышла бета версия интегрированной среды разработки NetBeans IDE 6.7. Главные новшества: Интеграция поддержки SourceForge-подобной платформы для совместной разработки и хостинга открытых проектов Kenai.com, созданной Sun Microsystems, поддерживающей системы контроля версий SVN, Mercurial и Git. Теперь управлять проектами и создавать проекты Kenai можно через встроенные инструменты NetBeans; Полная интеграция с Bugzilla; Поддержка JavaScript 1.7; Возможность удаленной отладки Ruby On Rails приложений; Поддержка Grails 1.1, реализация режима автодополнения кода для Grails; В модуле разработки для Java ME появилась полная поддержка работы с SVG компонентами в режиме Visual Mobile Designer ; Улучшение интеграции GlassFish Поддержка разработки C/C++ проектов с использованием библиотеки Qt; Система автоматического профилирования кода "Profile Me Now!" с возможностью экспорта отчетов в форматах CSV, HTML и XML ; Для разработчиков на языке PHP добавлена поддержка фреймворка Selenium, в PHP-редакторе появилась возможность автодополнения кода в SQL-вставках, улучшена поддержка PHPUnit;
