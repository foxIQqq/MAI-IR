Проект Debian представил машину времени для архива пакетов

Разработчики Debian GNU/Linux сообщили о введении в строй нового сервиса snapshot.debian.org, позволяющего получить доступ к состоянию архива репозиториев проекта на определенный момент времени в прошлом. В качестве отправной точки может быть указана как дата, так и версия определенной программы. Кроме базовых репозиториев в сервисе представлена информация обо всех обновлениях безопасности, доступны архивы debian-volatile, debian-ports и backports.org. Общий объем проиндексированных данных составляет 6.5 Тб, данные сохранены с марта 2005 года. Для работы с snapshot.debian.org можно задействовать стандартный пакетный менеджер APT для установки пакетов со старыми версиями программ или использовать web-интерфейс для навигации и загрузки исходных текстов. Представленная машина времени в первую очередь ориентирована на разработчиков, занимающихся устранением регрессивных изменений, так как предоставляет удобные средства по фиксации точки возникновения проблемы через постепенный возврат в прошлое для заданного приложения. Для администраторов сервис может оказать помощь в тестировании при планировании обновлений, а для пользователей помочь сформировать окружение из определенных версий программ, необходимых для выполнения специфичного приложения.
