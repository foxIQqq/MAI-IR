Налажено массовое производство Raspberry Pi и выпущен Linux-дистрибутив Raspbian

Разработчики проекта Raspberry Pi сообщили о снятии ограничений на заказ устройств Raspberry Pi "B" на базе SoC Broadcom BCM2835 700 МГц (ARM11 ARM1176JZF-S), поставляемых с 256 Мб ОЗУ, Ethernet и двумя USB-портами по цене 35 долларов. Если раньше один человек мог заказать только одно устройство, то отныне ограничения сняты и каждый может получить столько устройств, сколько ему требуется. Подобное стало возможным благодаря переходу от пробной партии к массовому производству, предусматривающему выпуск до 4000 устройств в день. Спецификация Raspberry Pi "B": Размер: 8.5 на 5.4 см, примерно как кредитная карта. Толщина платы 17 мм. Вес 45 г. SoC-платформа: Broadcom BCM2835 700 МГц (ARM11 ARM1176JZF-S) с графическим процессором, поддерживающим OpenGL ES 2.0 и функциями акселерации декодирования видео (1080p30 H.264); 256 Мб ОЗУ; Питание: 5 В через microUSB-порт; Интерфейсы: USB 2.0, Ethernet, слот для карт SD/MMC/SDIO, 3.5 мм звуковой выход; Видео-выход: композитный выход RCA ("тюльпан") и цифровой порт HDMI. Дополнительно объявлено о выпуске Raspbian, нового специализированного дистрибутива Linux для Raspberry Pi. Дистрибутив основан на пакетной базе Debian Wheezy, специально оптимизированной для Raspberry Pi. Графическое окружении построено на основе LXDE, в качестве web-браузера используется Midori, для работы с мультимедиа контентом задействован медиаплеер omxplayer, в полной мере использующий доступные в Raspberry Pi средства аппаратного ускорения. Необходимость создания отдельного порта Debian для Raspberry Pi обусловлена тем, что штатный порт Debian Wheezy armhf поддерживает только процессоры с архитектурой ARMv7-A, в то время как в Raspberry Pi используется ARMv6 с расширениями "hard float", что позволяет использовать только более медленный порт Debian armel ("soft float"). В отличие от ранее поставляемой для Raspberry Pi сборки Debian Squeeze, Raspbian отличается значительно возросшей производительностью, чего удалось добиться благодаря задействованию VPF-инструкций (Vector Floating Point) для вычислений с плавающей точкой, а также использованию ARMv6-оптимизации для библиотечных функций, таких как memcpy() и memset(). Размер образа для записи на SD-карту составляет 439 Мб. Для установки доступно около 35 тысяч дополнительных пакетов в репозиториях. Armhf-репозитории Debian использовать напрямую нельзя, так как для их работы в Raspberry Pi требуется пересборка c опциями "-march=armv6", "-mfpu=vfp", "-mfloat-abi=hard" (в Debian пакеты собраны с "-march=armv7"). Кроме Raspberry Pi порт Raspbian будет работать и на других устройствах с процессором ARMv6 и VFP-расширенями. Raspbian позиционируется для замены ранее поставляемой сборки Debian GNU/Linux, поддержка которой прекращена. Кроме Raspbian энтузиастами развиваются специализированные сборки Fedora Linux, Arch Linux и других дистрибутивов.
