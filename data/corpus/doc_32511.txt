Компания AMD намерена создать открытый 3D-драйвер для карт Radeon HD 7000

В развиваемом в составе Mesa 3D-драйвере R600g, построенном на базе архитектуры Gallium3D, в настоящее время поддерживаются только карты серии Radeon HD 2000, 3000, 4000, 5000 и 6000. Видеокарты Radeon HD 7000 будут существенно отличаться от ранее выпускаемых серий из-за перехода на новую архитектуру GCN (Graphics Core Next), что не позволяет добавить поддержку данных карт в уже существующий драйвер без его существенной переработки. По словам Джона Бриджмана (John Bridgman), курирующего open source проекты подразделения графических продуктов AMD, компания намерена создать новый открытый 3D-драйвер для карт серии Radeon HD 7000. В качестве основы для драйвера планируется использовать существенно сокращённую копию уже существующего драйвера R600g. Таким образом новый драйвер дополнит уже поддерживаемые драйверы R600g (GPU Radeon HD 2000 - 6000) и R300g (GPU c Radeon 9000 по Radeon X1000 (R500)), а также классические драйверы для GPU R100/R200. В проприетарном драйвере Catalyst для Linux поддержка Radeon HD 7000 появится сразу после поступления новых карт в продажу. Появление работоспособной версии открытого 3D-драйвера для карт Radeon HD 7000 оценивается не раньше лета следующего года.
