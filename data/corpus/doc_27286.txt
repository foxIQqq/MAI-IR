Обновление Java SE 6 Update 21

Вышел двадцать первый корректирующий релиз Java SE Runtime (JRE) 6 и Java SE Development Kit (JDK) 6, в котором исправлено 268 ошибок не связанных с проблемами безопасности. Из добавленных улучшений можно отметить: Включение в состав новой версии виртуальной машины Java HotSpot VM 17, в которой отмечено общее повышение качества работы и добавление таких возможностей, как сжатие указателей на объекты, оптимизация "escape analysis", управление кешированием кода, новый сборщик мусора Concurrent Mark-Sweep; Официальная поддержка продуктов: Oracle Enterprise Linux 4.8/5.4/5.5, Red Hat Enterprise Linux 5.4/5.5, Oracle VM 2.2.0.0.0 и Google Chrome 4.0; В состав включен Java VisualVM 1.2.2, построенный на кодовой базе NetBeans Platform 6.8. В новой версии улучшена производительность HeapWalker и VisualVM-Sampler, в плагине BTrace4VisualVM добавлена поддержка BTrace 1.1, исправлены ошибки в движке профилирования кода. Добавлена защита от возвращения NULL-указателей при определенных обстоятельствах использования режима Drag-and-Drop.
