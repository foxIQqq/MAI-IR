Выпуск дистрибутива Pop!_OS 22.04, развивающего рабочий стол COSMIC

Компания System76, специализирующаяся на производстве ноутбуков, ПК и серверов, поставляемых с Linux, опубликовала выпуск дистрибутива Pop!_OS 22.04. Pop!_OS основан на пакетной базе Ubuntu 22.04 и поставляется с собственным окружением рабочего стола COSMIC. Наработки проекта распространяются под лицензией GPLv3. ISO-образы сформированы для архитектуры x86_64 и ARM64 в вариантах для графических чипов NVIDIA (3.2 ГБ) и Intel/AMD (2.6 ГБ). Сборки для плат Raspberry Pi 4 задерживаются. Дистрибутив в первую очередь ориентирован на людей, использующих компьютер для создания чего-то нового, например, занимающихся разработкой контента, программных продуктов, 3D-моделей, графики, музыки или научной работой. Идея разработки собственной редакции дистрибутива Ubuntu пришла после решения Canonical перевести Ubuntu с Unity на GNOME Shell - разработчики System76 занялись созданием новой темы оформления на базе GNOME, но затем поняли, что готовы предложить пользователям иное окружение рабочего стола, предоставляющего гибкие средства для настройки под текущий рабочий процесс. Дистрибутив поставляется с рабочим столом COSMIC, построенным на базе модифицированной оболочки GNOME Shell и набора оригинальных дополнений к GNOME Shell, собственной темой оформления, своим набором пиктограмм, другими шрифтами (Fira и Roboto Slab) и изменёнными настройками. В отличие от GNOME, в COSMIC продолжает использоваться раздельное представление для навигации по открытым окнам и установленным приложениям. Для манипуляции окнами предоставлен как привычный для новичков традиционный режим управления при помощи мыши, так и мозаичный режим компоновки окон, позволяющий управлять работой только при помощи клавиатуры. В дальнейшем разработчики намерены трансформировать COSMIC в самодостаточный проект, не использующий GNOME Shell и развиваемый на языке Rust. Первый альфа-выпуск нового COSMIC намечен на начало лета. Среди изменений в Pop!_OS 22.04: Осуществлён переход на пакетную базу Ubuntu 22.04 LTS. Ядро Linux обновлено до версии 5.16.19, а Mesa до ветки 22.0. Рабочий стол COSMIC синхронизирован с GNOME 42. В панели "OS Upgrade & Recovery" предоставлена возможность включения автоматического режима установки обновлений. Пользователь может определить в какие дни и в какое время устанавливать обновления в автоматическом режиме. Режим применяется к пакетам в форматах deb, Flatpak и Nix. По умолчанию автоматические обновления отключены и пользователю раз в неделю показывается уведомление о наличии обновлений (в настройках можно установить показ каждый день или раз в месяц). Предложена новая панель поддержки, доступная для вызова в нижней части меню конфигуратора. Панель предоставляет ресурсы для решения типовых проблем, такие как ссылки на статьи по настройке оборудования, чат поддержки и возможность формирования логов для упрощения анализа проблемы. В настройках появилась возможность раздельного назначения обоев рабочего стола для тёмной и светлой тем оформления. В System76 Scheduler обеспечена поддержка повышения производительности через выставление приоритета приложению в активном окне. Улучшен механизм регулирования частоты процессора (cpufreq governor), подгоняющий параметры работы CPU к текущей нагрузке. Улучшены интерфейс и серверная часть каталога приложений Pop!_Shop. Добавлен раздел со списком недавно добавленных и обновлённых программ. Компоновка интерфейса оптимизирована для окон небольшого размера. Повышена надёжность операций с пакетами. Обеспечен показ установленных проприетарных драйверов NVIDIA. Осуществлён переход на использование мультимедийного сервера PipeWire для обработки звука. Улучшена поддержка многомониторных конфигураций и экранов с высокой плотностью пикселей. Обеспечена поддержка экранов для вывода конфиденциальной информации, например, некоторые ноутбуки комплектуются экранами со встроенным режимом конфиденциального просмотра, затрудняющим просмотр со стороны. Для удалённой работы по умолчанию задействован протокол RDP.
