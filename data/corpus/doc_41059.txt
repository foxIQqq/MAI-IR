Язык программирования Go переходит с Mercurial на Git и GitHub

Компания Google объявила о переводе инфраструктуры разработки языка программирования Go на систему управления исходными текстами Git и платформу совместной разработки GitHub. На днях проекту Go исполнилось пять лет, во время основания проекта была выбрана система Subversion, после чего последовал переход на Perforce, а затем и на Mercurial, из-за необходимости интеграции с существующей системой рецензирования кода. За это время мир изменился и Git занял прочную позицию лидера. Большинство участников сообщества Go используют Git и GitHub для своего кода. Проблемы с системой рецензирования остались в прошлом, поэтому теперь ничто не сдерживает миграцию проекта на Git. После миграции на Git в качестве системы рецензирования будет использована система Gerrit, которая уже применяется для кода платформы Android. Миграция на Git начнётся вскоре после релиза 1.4, намеченного на начало декабря. Выпуск 1.5 уже будет сформирован и размещён на GitHub.
