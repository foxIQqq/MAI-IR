Выпуск дистрибутива Mageia 9, форка Mandriva Linux

Спустя два с половиной года с момента прошлого значительного выпуска опубликован релиз Linux-дистрибутива Mageia 9, в рамках которого независимым сообществом энтузиастов развивается форк проекта Mandriva. Для загрузки доступны 32- и 64-разрядные установочные сборки (4.3 ГБ) и набор Live-сборок (3.4-3.8 ГБ) на базе GNOME, KDE и Xfce. Ключевые улучшения: Обновлены версии пакетов, включая ядро Linux 6.4.3, glib 2.36, gcc 12.3.0, rpm 4.18.0, DNF 4.14.0 (опция), Mesa 23.1.3, X.Org 21.1.8, XWayland 22.1.9, Chromium 114.0.5735.198, Firefox ESR 102.13, LibreOffice 7.5.4.2, Python 3.10.11, Perl 5.36, Ruby 3.1.4, Rust 1.70, PHP 8.2. Обновлены версии рабочих столов KDE Plasma 5.27.5, GNOME 44.2, Xfce 4.18.4, Cinnamon 5.6, MATE 1.26.0, Enlightenment 25.4 и LXQt 1.3.0. Обновлено окружение LXDE, которое продолжает поддерживаться проектом Mageia. Заметно сокращён размер минимальной установки (без рекомендованных пакетов). База данных пакетов RPM (rpmdb) переведена с BerkeleyDB на SQLite. В качестве основной причины замены называется применение в rpmdb устаревшей версии Berkeley DB 5.x, которая уже давно не сопровождается, а переходу на более новые выпуски мешает изменение лицензии Berkeley DB 6 на AGPLv3. Кроме того, старая реализация rpmdb на основе BerkeleyDB не обеспечивала необходимую надёжность, так как не использует транзакции и не способна выявлять несоответствия в БД. В установщике обновлена реализация NFS, задействован алгоритм Zstd для сжатия образов, добавлена поддержка использования curl вместо wget для загрузки пакетов (опция "downloader=curl"). В Live-сборке с KDE по умолчанию включён сервис NetworkManager, по аналогии со сборками GNOME и Xfce. В репозиториях RPM-MD (RPM MetaData) предоставлены метаданные AppStream для расширения возможностей поиска и управления приложениями в интерфейсах GNOME Software и Plasma Discover. Для сжатия метаданных Urpmi задействован алгоритм Zstd вместо Xz. Полностью переделано приложение MageiaWelcome, запускаемое при первой загрузке системы. Интерфейс для выполнения типовых настроек после установки теперь имеет линейную форму с последовательным выполнением шагов. Переделан интерфейс приложения Isodumper, применяемого для записи ISO-образов. Расширены возможности интерфейса draklive2, в котором упрощена настройка локализации, часовых поясов, сервисов и межсетевого экрана, добавлена сводная страница с общей информацией. Упрощён выбор отдельных пакетов для установки. В ISO-образах утилита проверки памяти PCMemTest заменена на Memtest86+. Добавлена утилита remove-old-kernels для удаления старых пакетов с ядром Linux. Добавлена утилита Mageia-prime для переключения GPU в гибридных графических системах, в которых сочетаются встроенный GPU Intel и дискретная карта NVIDIA. Предоставлена поддержка звуковых серверов PulseAudio и PipeWire - предустановлены оба сервера, но по умолчанию активирован PulseAudio. В конфигуратор звуковой подсистемы в Mageia Control Center добавлена возможность переключения между звуковыми серверами PulseAudio и PipeWire.
