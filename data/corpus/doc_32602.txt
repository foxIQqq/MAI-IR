Релиз промышленного дистрибутива Oracle Linux 6.2

Компания Oracle представила релиз промышленного дистрибутива Oracle Linux 6.2, созданного на основе пакетной базы Red Hat Enterprise Linux 6.2. Дистрибутив доступен для бесплатной загрузки в сборках для архитектур x86 и x86_64, после регистрации на сайте edelivery.oracle.com. Из отличий дистрибутива Oracle Linux 6.2 от RHEL 6.2 отмечается обновление версий драйверов для сетевых устройств и систем хранения. С общими отличиями линейки Oracle Linux 6.x от RHEL 6.x можно познакомиться здесь. В составе Oracle Linux 6.2 поставляется два варианта пакетов с Linux-ядром 2.6.32: классический вариант от компании Red Hat (kernel-2.6.32-220.el6) и созданный силами Oracle пакет с ядром (kernel-uek-2.6.32-300.3.1.el6uek). По умолчанию в базовую поставку входят оба ядра, но подготовленный в Oracle пакет с ядром загружается по умолчанию. Готовые бинарные сборки ядра от Oracle для совместимых c RHEL 4 и 5 дистрибутивов и src.rpm-пакеты представлены на сайте public-yum.oracle.com, там же можно загрузить и другие присутствующие в дистрибутиве пакеты. Oracle не накладывает никаких ограничений на поставку подготовленного в Oracle ядра в сторонних продуктах. Среди ключевых особенностей Oracle Linux 6 (многие из этих возможностей, несмотря на заявление Oracle, присутствуют и в RHEL 6): поддержка метода удаленного прямого доступа к памяти OFED (OpenFabrics Enterprise Distribution); поддержка кластерной файловой системы OCFS2 1.6; интеграция фреймворка обеспечения целостности данных DIF/DIX (Linux data integrity framework); поддержка tickless-режима работы ядра; поддержка групп контроля за выполнением задач (Task Control Groups - TCG или cgroups); использование подсистемы Performance Counters; автоматический адаптивный тюнинг работы SSD-накопителей; возможность привязки обработчиков ввода/вывода к определенным процессорным ядрам (IO affinity); технология прозрачного распределения нагрузки по обработке входящего сетевого трафика на имеющиеся в системе CPU; поддержка системного вызова fallocate. Дополнение: Релиз CentOS 6.2 намечен на ближайшие дни, уже начался процесс распространения iso-образов по зеркалам.
