Вышел Elive 2.0, Linux-дистрибутив с десктоп-окружением на основе Enlightenment 17

После двух лет разработки вышел релиз Linux дистрибутива Elive 2.0, построенного на пакетной базе Debian и десктоп-технологиях проекта Enlightenment 17. Дистрибутив интересен тем, что при наличии удобного и красивого десктоп окружения в стиле Mac OS X, он способен работать на машинах с 300 Mhz CPU и 128 Мб ОЗУ (заявленный минимум 100 Mhz CPU и 64 Мб ОЗУ). Для настройки системы используется контрольная панель собственной разработки - "Elpanel". Дистрибутив поддерживает установку на жесткий диск, но может работать в качестве полноценной LiveCD-системы, поддерживающей доустановку программ и промежуточное сохранение данных между сессиями на внешнем носителе. Для установки дополнительных приложений можно использовать стандартные репозитории пакетов Debian GNU/Linux. Для управления установкой программ используется Synaptic. Размер iso-образа составляет 700 Мб. В качестве Linux ядра используется версия 2.6.30.9, дополненная дополнительными драйверами и патчами. Например, в дистрибутив интегрирована поддержка файловой системы Reiser4, технологии временного засыпания tuxonice, драйверов для 3G-модемов, Bluetooth-устройств, Wifi-адаптеров (Broadcom 43xx), проприетарных драйверов для видеокарт ATI и Nvidia. Для задействования визуальных эффектов на десктопе поддерживается использование композитного менеджера Compiz. Из поставляемых в комплекте программ можно отметить: AbiWord 2.6.4, Firefox 3.0.6, Gimp 2.4.7, Gnumeric 1.8.3. Для монтирования удаленных SMB-разделов используется утилита LinuxNIT. Для настройки оборудования в рамках проекта разработана серия автоконфигураторов, автоматически настраивающих систему во время загрузки. Инсталлятор может установить систему на дисковые разделы Ext4 и Reiser4, поддерживает режим обновления и миграции установленного Linux дистрибутива на Elive (с сохранением настроек и базы пользователей). Из разработанных в рамках проекта приложений также можно отметить ПО Nurse, предназначенное для восстановления системы после сбоя. Nursе позволяет восстановить целостность конфигурации, провести тестирование оборудования, произвести проверку установленных пакетов, помочь в освобождении свободного места на диске. Дополнение: оказалось, что дистрибутив полу-коммерческий, т.е. без ограничений можно использовать в Live-режиме, возможность установки на диск активируется только после получения инвайта или перечисления пожертвования разработчикам.
