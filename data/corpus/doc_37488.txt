Доступно обновление Firefox OS 1.1

Для пользователей первых телефонов на базе платформы Firefox OS, выпускаемых компанией Geeksphone, началось распространение обновления прошивки c Firefox OS 1.1. Новая версия платформы примечательная устранением многих недоработок и проблем, наблюдаемых в первом релизе. Среди изменений: Налажено корректное отображение интерфейса сторонних приложений на HD-экране; Улучшена работа приложения для электронной почты, при ответе на сообщение обеспечено отображение исходного текста, обеспечена группировка сообщений по нитям; Внесены улучшения в календарь-планировщик, добавлена защита от случайного удаления событий (удалить элемент теперь можно только в режиме редактирования); Переработана карусель запуска приложений, вызываемая при удержании управляющей кнопки. Из проблем отмечается отсутствие фиксированного порядка вывода приложений (новое приложение сдвигает остальные); Обеспечена возможность установки отдельного уровня громкости при прослушивании музыки; Добавлен режим ответа на звонок с использованием наушников без встроенного микрофона (используется микрофон телефона); При подключении телефона по USB теперь предоставляется возможность выбора раздела для монтирования - SD-карта или внутренняя память; Поддержка импорта контактов из SIM, SD-карты, Gmail, Outlook (live.com) и Facebook.
