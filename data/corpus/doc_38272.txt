Представлены официальные сборки GTK+ 3 для Windows

После перехода нескольких известных проектов с GTK+ на Qt из-за недостаточно качественной поддержки Windows и OS X, началось распространение официальных сборок GTK+ 3 для платформы Windows. Версия GTK+ 3 для Windows поставляется в виде набора, в котором кроме GTK+ присутствуют библиотеки и компоненты, необходимые для обеспечения работы GTK-приложений. Например, в состав входят: GLib, ATK, Pango, gdk-pixbuf, zlib, cairo, libpng, Freetype, Fontconfig, gettext, rsvg, croco, pixman и pkg-config. Порт GTK3+ для Windows основан на версии 3.6.4. Ранее для Windows поставлялись только сборки GTK+ 2.24.
