Выпуск Phosh 0.22, GNOME-окружения для смартфонов. Сборки Fedora для мобильных устройств

Состоялся релиз Phosh 0.22.0, экранной оболочки для мобильных устройств, основанной на технологиях GNOME и библиотеке GTK. Окружение изначально развивалось компанией Purism в качестве аналога GNOME Shell для смартфона Librem 5, но затем вошло в число неофициальных проектов GNOME и теперь также используется в postmarketOS, Mobian, некоторых прошивках для устройств Pine64 и редакции Fedora для смартфонов. Phosh использует композитный сервер Phoc, работающий поверх Wayland, а также собственную экранную клавиатуру squeekboard. Наработки проекта распространяются под лицензией GPLv3+. В новом выпуске обновлён визуальный стиль и изменено оформление кнопок. В индикаторе заряда аккумулятора реализована градация изменения состояния с шагом 10%. В уведомлениях, размещаемых на экране блокировки системы, разрешено использование кнопок с действиями. Обновлены конфигуратор phosh-mobile-settings и инструментарий для отладки виртуальной клавиатуры phosh-osk-stub. В то же время Бен Коттон (Ben Cotton), занимающий в компании Red Hat должность Fedora Program Manager, опубликовал предложение о начале формирования полноценных сборок Fedora Linux для мобильных устройств, поставляемых с оболочкой Phosh. Сборки будут формироваться группой Fedora Mobility, до сих пор ограничивавшейся сопровождением набора пакетов 'phosh-desktop' для Fedora. Сборки с Phosh планируют поставлять начиная с выпуска Fedora Linux 38 для архитектур x86_64 и aarch64. Предполагается, что наличие готовых установочных сборок для мобильных устройств позволит расширить область применения дистрибутива, привлечёт к проекту новых пользователей и предоставит готовое решение с полностью открытым интерфейсом для смартфонов, который можно использовать на любых устройствах, поддерживаемых в штатном ядре Linux. Предложение пока не рассмотрено комитетом FESCo (Fedora Engineering Steering Committee), отвечающим за техническую часть разработки дистрибутива Fedora.
