Финальный бета-выпуск Ubuntu 15.10

Представлен финальный бета-выпуск Ubuntu 15.10 "Wily Werewolf". Релиз намечен на 22 октября. Готовые установочные образы для тестового выпуска созданы для Ubuntu Desktop, Ubuntu Server и Ubuntu Core (для них на прошлых этапах тестирования предлагались только ежедневные сборки), а также для Kubuntu, Lubuntu, Ubuntu MATE, Ubuntu Cloud, Ubuntu GNOME и Xubuntu. Основные новшества второго бета-выпуска: Задействовано ядро Linux 4.2 с драйвером AMDGPU для новых GPU Radeon, поддержкой чипов Intel Broxton, возможностью шифрования данных на F2FS и поддержкой энергонезависимой оперативной памяти NV-DIMM; В компонентах графического стека и пользовательского окружения в основном произведены исправления ошибок и выполнена работа по оттачиванию качества. Библиотеки GNOME обновлены до выпуска 3.16. Продолжен перевод системы инициализации на компоненты systemd. В Unity добавлена поддержка показа локальных меню для неактивных окон. Проведена оптимизация удобства использования Dash; В Compiz проведена работа по улучшению интеграции с окружением MATE; Kubuntu: Пользовательское окружение обновлено до Plasma 5.4, а набор приложений до версий из состава KDE Applications 15.08. В состав включены новые выпуски LibreOffice 5.0 и Firefox 41; Ubuntu GNOME: Рабочий стол GNOME Shell и большинство компонентов GNOME обновлены до выпуска 3.16. Вместо Shotwell для работы с коллекцией фотографий задействован GNOME Photos. В поставку по умолчанию добавлено приложение GNOME Music. Bluetooth-стек обновлён до Bluez 5. Представлено вводное руководство, предлагаемое при первом запуске. В GNOME Online Accounts (GOA) добавлена поддержка сервиса syncevolution. Реализован экспериментальный сеанс на базе Wayland, для использования которого следует установить пакет gnome-session-wayland и выбрать при входе "GNOME on wayland" (работает только при использовании открытых видеодрайверов). В интерфейсе для установки приложений Ubuntu Software Centre добавлена поддержка тёмной темы оформления Adwaita Dark; Представлено новое информационное слайд-шоу и инсталляторе Ubiquity; Ubuntu MATE: Завершён переход на выпуск десктоп-окружения MATE 1.10. В Thunderbird добавлен плагин с календарём планировщиком Lightning, включая поддержку интеграции с Google Calendar; Менеджер настройки Bluetooth-соединений Blueman обновлён до выпуска 2.0, использующий стек BlueZ 5.34; В Ubuntu MATE Welcome добавлена возможность быстрой установки Asunder, FileZilla, Iced Tea (Java), Pulse Audio Volume Control, Rapid Photo Downloader и SuperTuxKart. Lubuntu: так как разработка окружения LXQt пока не завершена, переход Lubuntu на LXQt опять откладывается и в Lubuntu 15.10 по-прежнему будет использован LXDE. Система ввода iBus заменена на Fcitx, что решило проблемы с отображением китайских, японских и корейских шрифтов; Xubuntu: Состав добавлен апплет Xfce4 Panel Switch для настройки раскладки панели и создания резервных копий состояния панели. Представлено пять преднастроенных раскладок панели. Добавлена новая тема оформления для оконного менеджера. В состав включены LibreOffice Calc и Writer, которые заменили собой Gnumeric и Abiword. Дополнительно можно отметить изменения, связанные с поставкой Firefox в Ubuntu. В очередном обновлении пакетов произведено отключение и удаление предустанавливаемых дополнений для интеграции с Ubuntu Online Accounts, Unity и web-приложениями Greasemonkey: unity-firefox-extension, webapps-greasemonkey, webaccounts-browser-extension. Удаление выполнено в связи с негативным влиянием на возможность установки поисковых плагинов и будущим переходом Mozilla на обязательную проверку дополнений по цифровой подписи.
