Mozilla Labs работает над созданием игрового движка

Разработчики Mozilla Labs представили проект Paladin, в рамках которого ведется подготовка компонентов и библиотек для создания трехмерных игр на языке JavaScript, работающих внутри браузера. Paladin включает в себя несколько подпроектов: игровой движок Gladius, API для использования джойстика, API для полного контроля над мышью. Для 3D-рендеринга используются наработки проекта CubicVR, а для симулирования физических процессов задействована библиотека ammo.js, представляющая собой версию библиотеки Bullet, преобразованную в JavaScript при помощи компилятора Emscripten. Для демонстрации возможностей игровой подсистемы подготовлен рабочий прототип игры RescueFox.
