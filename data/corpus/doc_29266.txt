Сформирован порт OpenJDK для Mac OS X

Майк Свинглер (Mike Swingler) из компании Apple анонсировал введение в строй порта OpenJDK для платформы Mac OS X. Проект основан на коде порта OpenJDK для BSD-систем и в настоящее время поддерживает все реализованные в нем функции. В настоящее время порт пока находится на начальной стадии развития и некоторые возможности еще предстоит довести до рабочего состояния. В качестве намеченных целей отмечаюся: прохождение полного набора тестов на совместимость с Java SE 7, оптимизация производительности и интеграция тулкита для формирования пользовательского интерфейса на базе Cocoa. Напомним, что ранее реализация Java для Mac OS X поддерживалась силами компании Apple, оригинальный пакет Java SE от компании Oracle не поддерживал платформу Mac OS X. В ноябре компания Apple отказалась от продолжения обособленной разработки Java для своей ОС, приняв решение присоединиться к работе над проектом OpenJDK и открыть все наработки, связанные с обеспечением работы Java на платформе Mac OS X.
