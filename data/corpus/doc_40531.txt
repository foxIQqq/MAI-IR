Новая версия ModemManager 1.4 и Modem Manager GUI 0.0.17

Увидел свет новый выпуск ModemManager 1.4, системы для настройки GSM/CDMA/UMTS/EVDO/LTE-модемов, работающая по аналогии с конфигуратором NetworkManager, а также Modem Manager GUI 0.0.17, фронтэнда для управления Modem Manager в графическом режиме. ModemManager функционирует в виде демона, который берёт на себя непосредственное взаимодействие с модемом или мобильным телефоном, а для внешних приложений предоставляет унифицированный интерфейс, не зависящий от типа поддерживаемых модемом протоколов, используемых каналов связи (2G/3G/4G/CDMA), способов соединения (USB, RS232, Bluetooth) и методов управления (AT, QCDM, QMI, MBIM). ModemManager API поддерживает такие операции, как установка соединения, отправка SMS, чтение и изменение адресной книги, вывод данных о качестве сигнала, чтение параметров GPS-приемника. Для взаимодействия с внешними приложениями и апплетами для настройки соединений используется DBus API. Поддержка новых типов модемов может осуществляться через подключение плагинов. Новшества ModemManager 1.4: Изменена логика настройки свойств IP-соединения (возможность изменения MTU; указание статического IPv6 может сочетаться с получением параметров по DHCP); Добавлена опция для полного отключения модема по питанию; В интерфейс устройств определения местоположения добавлен режим "неуправляемого GPS" ('unmanaged GPS'), позволяющий включить/отключить GPS-модуль в модеме с сохранением предоставления информации о местоположении для приложений. Режим доступен для модемов с GPS TTY, таких как Option/HSO, Cinterion и Huawei; Новый интерфейс Test DBus, предназначенный для проведения внутренних системных проверок; В MBIM добавлена поддержка модемов ZTE и Sequans Communications; В Ericsson MBM добавлена возможность использования поддерживающих AT-команды портов /dev/cdc-wdm; В Huawei улучшено получение информации о времени, добавлена поддержка GPS и поддержка портов /dev/cdc-wdm через драйвер huawei-cdc-ncm; В Cinterion добавлена поддержка GPS. Программа Modem Manager GUI написана с использованием библиотеки GTK3+ и распространяется под лицензией GPL v3. Предоставляется графический интерфейс для отправки/приёма SMS, отправки запросов USSD, отображения различной информации о модеме и SIM-карте (оператор, IMEI, уровень сигнала и др.), cканирования доступных сетей, управления процессом передачи данных и ограничения трафика и времени. Новые функции в Modem Manager GUI 0.0.17: Добавлена возможность работы со стеком приложений oFono и конфигуратором Connman, которые теперь поддерживаются наряду с ModemManager и NetworkManager; Поддержка экспорта записей из адресной книги KDE; Возможность выбора предпочитаемого бэкенда через GUI; Возможность настройки таймаутов через GUI; В диалоге работы с SMS добавлено автодополнение имён из адресной книги; В окне USSD добавлено автодополнение команд; Добавлен раздел помощи в стиле GNOME; Добавлен файл, соответствующий XML-спецификации AppData; Исправления: устранено зацикливание при обработке соединений, обеспечен вывод истории трафика для первого дня месяца, налажена отправка USSD-кодов с первым символом '#', решены проблемы с поддержкой формата 3GPP-позиционирования.
