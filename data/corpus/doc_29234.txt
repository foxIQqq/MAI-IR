Релиз минималистичного Linux-дистрибутива wattOS 3

Вышел релиз минималистичного Linux-дистрибутива wattOS 3, построенного на пакетной базе Ubuntu 10.10 и поставляемого с графическим окружением LXDE. Размер установочного iso-образа составляет 577 Мб, поддерживается как работа в Live-режиме, так и установка на жесткий диск. В новой версии: Добавлена поддержка автоматического входа в десктоп-окружение (Autologin); Вместо openbox теперь используется LXDE, в качестве музыкального плеера вместо Rhythmbox задействован Foobnix; Для редактирования фотографий в комплект включен Fotoxx; В файловый менеджер PCManFM добавлена поддержка работы с сетевыми дисками, корзиной и списком приложений; В состав добавлены менеджер паролей KeePassX и органайзер Osmo; Для управление питанием на портативных устройствах используется xfce4 power manager; Из поставки удален локальный почтовый клиент Claws Mail; Для ведения заметок используется RedNotebook Journal. Из входящих в комплект приложений, можно отметить: foobnix - аудиоплеер wicd - интерфейс для настройки сетевых соединений (ранее поставлялся Network Manager); Brasero - ПО для записи CD/DVD Totem - видеоплеер F-Spot - менеджер для управления коллекций фотографий; Spicebird - система для организации коллективной работы на базе Thunderbird, Lightning и Telepathy; web-браузер SwiftFox на основе Firefox Empathy - программа для мгновенного обмена сообщениями Vinagre - VNC-клиент Transmission - BitTorrent клиент AbiWord - текстовый процессор Gnumeric - электронные таблицы PCMan - файловый менеджер
