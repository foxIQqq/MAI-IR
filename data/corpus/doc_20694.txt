Вышел первый кандидат в релизы Mandriva Linux 2009.1

"Mandriva Linux 2009 Spring RC1 ready for tests!" - вышел первый кандидат в релизы Mandriva Linux 2009.1. По сравнению с прошлым тестовым выпуском обновлены пакеты: KDE 4.2.1, GNOME 2.25.92, Xfce 4.6, X.Org server 1.6, OpenOffice.org 3.0.1, Qt 4.5.0, Linux ядро 2.6.29 RC 6. Начиная с версии RC1 подготовлены гибридные iso-образы LiveCD ("One") сборки Mandriva, которые теперь подходят и для создания загрузочных USB FLash путем простого копирования содержимого ("dd if=mandriva-linux-one-2009.1-rc1-GNOME-int-cdrom-i586.iso of=/dev/sdX bs=8M"). В состав установочного диска включены: Модуль HDT (Hardware Detection Tool) из состава пакет Syslinux , позволяющий на раннем этапе загрузки сформировать полное описание состава текущего аппаратного обеспечения, до начала загрузки ОС. draksnapshot - инструмент для быстрого восстановления всей системы из бэкапа по состоянию на указанный момент времени. Новшества добавленные ранее: Обновление версий: Tcl/Tk 8.6. Python 2.6.1, Linux ядро 2.6.28; В качестве легковесного окружения теперь используется LXDE, вместо icewm; Осуществлен редизайн системы управления безопасностью msec; В инсталлятор, в качестве дополнительной опции, добавлена возможность установки на ext4 раздел. В Diskdrake появилась возможность изменение метки дискового раздела без его переформатирования; Заблокирована возможность перезапуска X-сервера через последовательность клавиш Ctrl+Alt+backspace; В инсталляторе добавлена опциональная возможность использования файловой системы ext4. Но используемая версия загрузчика пока не поддерживает ext4, необходимо подождать обновления GRUB; Произведена интеграция технологии speedboot, призванная значительно сократить время загрузки дистрибутива. Для тестирования необходимо указать слово speedboot в строке ввода параметров ядра при загрузке. Тестовая версия доступна в виде "DVD Free" сборки (без проприетарных драйверов в комплекте) и двух LiveCD сборок с KDE и GNOME. Согласно планам разработки, финальный релиз намечен на 29 апреля (ранее планировалось выпустить релиз 16 апреля).
