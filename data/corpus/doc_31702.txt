В рамках премии Bossie Awards 2011 отмечено лучшее открытое ПО

Объявлены призеры ежегодной премии Best of Open Source Software Awards 2011, присуждаемой изданием InfoWorld лучшему открытому ПО. В качестве критерия отбора программ использовалась оценка полезности приложения как такового для определенной аудитории (корпоративных пользователей, разработчиков, сетевых администраторов). В роли экспертов, выбравших лучшие продукты из всего многообразия открытых решений, выступали редакторы и обозреватели издания InfoWorld. Победители определены в 4 категориях: Лучшие открытые мобильные и десктоп приложения. Категория охватывает офисные пакеты, браузеры, медиаплееры, системы шифрования дисков, платформы виртуализации рабочего стола. Программа для архивирования и сжатия данных 7-Zip; Мобильная платформа Android; Система создания скринкастов CamStudio (только для Windows); Web-браузер Google Chrome; Офисный пакет LibreOffice; Система виртуализации VirtualBox; PDFCreator - виртуальный принтер для печати в PDF-документы (только для Windows); Клиент для мгновенного обмена сообщениями Pidgin; PortableApps.com - ПО для создания переносимых на USB Flash установок Windows программ; Система шифрования дисковых разделов TrueCrypt; Медиаплеер VLC. Лучшие открытые серверные программы для предприятий. Категория затрагивает такие области, как организация внутрикорпоративых поисковых систем, управление web-контентом, системы для ведения блогов, CRM, ERP, управление процессами предприятия и ПО для бизнес-аналитики. Платформа для создания поискового сервера масштаба предприятия на базе Apache Lucene и Apache Solr; Система управления web-контентом Drupal; Система планирования ресурсов предприятия Openbravo ERP; Система бизнес-аналитики Pentaho BI Suite; CRM-система SugarCRM для организации взаимодействия с клиентами на предприятии, обеспечения работы службы продаж, ведения партнерской сети, предоставления сервисов и планирования работы сотрудников; Движок для создания блогов WordPress Лучшие открытые программы для разработки приложений. Категория охватывает серверные JavaScript-платформы, NoSQL БД, мобильные фреймворки, системы распределенной обработки данных, системы контроля версий, средства непрерывной интеграции, web-фреймворки. PHP-фреймворк для разработки web-приложений CakePHP; Компилируемый в JavaScript язык программирования CoffeeScript; Распределенная система управления исходным кодом Git; Платформа для организации распределенных вычислений с использованием парадигмы map/reduce - Apache Hadoop; Инструментарий непрерывной интеграции Hudson и его форк Jenkins; JavaScript-фреймворки для создания мобильных приложений на базе web-технологий - jQuery Mobile и Sencha Touch; Ориентированная на хранение документов высокопроизводительная и высокомасштабируемая NoSQL БД MongoDB; Серверная JavaScript-платформа Node.js; Web-фреймворк на языке Python Web2py. Лучшие открытые программы для дата-центров и облачных систем. Категория охватывает системы виртуализации, связующее ПО и системы интеграции данных. Платформы для поддержания облачной инфраструктуры Eucalyptus и OpenStack; Кластерное хранилище для объектов и файлов GlusterFS; Пакет для интеграции и управления данными Talend Open Studio; Дистрибутив для создания маршрутизаторов и межсетевых экранов Vyatta Linux; Система виртуализации Xen; Модульная SOA-платформа (стек middleware) WSO2.
