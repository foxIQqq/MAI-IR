Открыт код web-браузера Kylo, предназначенного для использования с телевизором

Компания Hillcrest Labs объявила об открытии исходных текстов web-браузера Kylo, предназначенного для навигации по контенту на телеприставках или подключенных к телевизору компьютерах. Kylo основан на браузерном движке Mozilla Gecko, для создания интерфейса использован язык XUL. Код открыт под свободной лицензией MPL 2.0, позволяющей создавать производные продукты и коммерческие версии. По мнению разработчиков Kylo, открытие кода позволит расширить возможности браузера за счёт привлечения новых участников в проект, заинтересованных в добавлении новых функций и охвата новых категорий пользователей. Интерфейс браузера специально оптимизирован для просмотра сайтов с использованием телевизора - все лишние элементы убраны для предоставления максимального пространства для контента. Для улучшения восприятия информации на телевизионных экранах в браузере используются крупные шрифты и увеличенные информационные пиктограммы. Размер элементов выбран для комфортной работы с 32-дюмовым экраном на расстоянии в три метра. Поддерживаются HDTV-телевизоры с разрешением от 720p. При необходимости текст или часть страницы можно увеличить. Также доступна функция панорамирования, позволяющая окинуть одним взглядом всю страницу. Для управления в нижней части экрана оставлена только небольшая навигационная строка. Управление максимально упрощено и адаптировано для применения пульта дистанционного управления. Для работы достаточно пульта, способного передавать команды перемещения курсора (вверх, вниз, вправо, влево и ввод). Ввод данных осуществляется через виртуальную экранную клавиатуру. При отсутствии пульта возможно управление обычной мышью. Имеется интерфейс для управления закладками. В настоящее время бинарные сборки Kylo формируются только для Windows и Mac OS X. Примечательно, что Kylo вошёл в число самых популярных приложений для платформы Mac OS X за 2010 год. Браузер также награждён премией International CES Innovations 2011 Design and Engineering Awards.
