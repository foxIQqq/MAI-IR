Выпуск графического редактора GIMP 2.10.18

Представлен выпуск графического редактора GIMP 2.10.18, в котором продолжено оттачивание функциональности и повышение стабильности ветки 2.10. Выпуск GIMP 2.10.16 был пропущен из-за выявления критической ошибки на стадии после ответвления данной версии. Для установки доступен пакет в формате flatpak (пакет в формате snap пока не обновлён). Кроме исправления ошибок в GIMP 2.10.18 представлены следующие улучшения: По умолчанию предложен сгруппированный режим компоновки панели инструментов. Пользователь может создавать собственные группы и перемещать в них инструменты на своё усмотрение. Например, разные инструменты для трансформации, выделения, заливки и рисования можно скрыть за общие кнопки группы, без вывода каждой кнопки по отдельности. Отключить режим группировки можно в настройках в секции Interface/Toolbox. По умолчанию включено компактное представление кнопок-ползунков (слайдеров), которые обычно используются для задания параметров фильтров и инструментов. Компактный стиль, при котором сокращены верхние и нижние отступы, существенно экономит вертикальное экранное пространство и позволяет вместить в видимую область больше элементов. Для изменения значения параметров можно использовать перемещение после клика левой кнопкой мыши, при этом дополнительное удержание Shift приводит к уменьшению шага изменения, а Ctrl - к увеличению. Улучшен процесс закрепления панелей и диалогов в однооконном интерфейсе. При попытке переместить встраиваемые диалоги в режиме drag&drop теперь не выводится мешающее сообщение с информацией о возможности оставить диалог в текущей позиции. Вместо сообщения для информирования возможности закрепления перемещаемого диалога теперь применяется подсветка всех прикрепляемых областей. Добавлен набор высококонтрастных символьных пиктограмм, который можно выбрать в настройках (по умолчанию оставлены прежние пиктограммы). Добавлен новый режим предпросмотра результатов применения инструментов трансформации, получивший название "Composited Preview". При активации данного режима отрисовка предпросмотра при трансформации осуществляется с учётом позиции изменяемого слоя и корректного режима смешивания. Новый режим также предлагает две дополнительные опции: "Preview linked items" для предпросмотра изменений всех связанных элементов, таких как слои, а не только выбранного элемента, и "Synchronous preview" для отрисовки предпросмотра во время перемещения указателя мыши/стилуса, не дожидаясь пока указатель остановится. Кроме того, реализован автоматический предпросмотр отсечённых частей трансформируемых слоёв (например, при вращении). Добавлен новый инструмент трёхмерной трансформации, позволяющий произвольно изменять перспективу в трехмерной плоскости с вращением слоя по осям X, Y и Z. Доступна возможность ограничения панорамирования и перспективы в привязке к одной из координатных осей. Повышена плавность перемещения указателя кисти за счёт увеличения частоты обновления информации на экране с 20 до 120 FPS. За счёт применения mipmap повышено качество рисования растровыми кистями уменьшенного масштаба. Добавлена опция для отключения привязки к мазкам. Частота работы аэрографа увеличена с 15 до 60 оттисков в секунду. Инструмент искажения (Warp Transform) теперь учитывает настройки указателя. В режиме симметричного рисования появилась опция "калейдоскоп", позволяющая комбинировать вращение и отражение (мазки отражаются по краям симметричных долей). Улучшена панель работы со слоями, в которой объединён интерфейс для слияния слоёв и прикрепления выделенных областей. В нижней части, при наличии выделенной области вместо кнопки объединения слоёв теперь отображается кнопка "якорь". При слиянии можно использовать модификаторы: Shift для слияния группы, Ctrl для слияния всех видимых слоёв и Ctrl + Shift для слияния всех видимых слоёв с прошлыми значениями. Ускорена загрузка кистей в формате ABR (Photoshop), что позволило существенно сократить время запуска при большом числе кистей в данном формате. Улучшена поддержка файлов в формате PSD и ускорена их загрузка за счёт исключения ресурсоёмкой стадии преобразования во внутреннее представление проекта. Крупные PSD-файлы теперь загружаются в полтора-два раза быстрее. Добавлена возможность загрузки PSD-файлов в представлении CMYK(A) через преобразование в профиль sRGB (возможность пока ограничена только файлами с 8-битами на канал). При каждом запуске реализована проверка наличия новой версии GIMP через отправку запросов на сервер проекта. Кроме непосредственно версии GIMP также проверяется наличие более нового установочного набора, на случай, если обновились предлагаемые в комплекте сторонние библиотеки. Информация о версии используется при генерации отчёта о проблеме в случае краха. Автоматическую проверку версии можно отключить в настройках на странице "System Resources" и проверять наличие обновлений вручную через диалог "About". Также можно отключить код проверки версии на этапе сборки при помощи опции "--disable-check-update". Обеспечено автоматизированное тестирование сборки основной ветки GIMP в системе непрерывной интеграции с использованием при сборке Clang и GCC. Для Windows реализовано формирование 32- и 64-разрядных сборок, собранных с crossroad/Mingw-w64. Из планов на будущее отмечается продолжение работы над будущей веткой GIMP 3, в которой будет проведена значительная чистка кодовой базы и выполнен переход на GTK3. Разработчики также изучают возможность улучшения однооконного режима интерфейса и реализации именованных рабочих пространств, оптимизированных для разных задач (общее редактирование, web-дизайн, обработка фотографий, рисование и т.п.). Продолжается разработка проекта Glimpse, развивающего форк графического редактора GIMP (создатели форка считают неприемлемым использование слова gimp из-за отрицательной коннотации). На прошлой неделе началось тестирование бета-версии второго выпуска 0.1.2 (нечётные версии используются для разработки). Релиз ожидается 2 марта. Из изменений отмечается добавление новых тем оформления интерфейса и пиктограмм, избавление фильтров от упоминания слова "gimp" и добавление настройки для выбора языка на платформе Windows.
