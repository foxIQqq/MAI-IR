Компания Oracle выпустила обновление ядра Unbreakable Enterprise Kernel R3U2

Компания Oracle представила второе функциональное обновление для ядра Unbreakable Enterprise Kernel 3. Исходные тексты ядра, включая разбивку на отдельные патчи, доступны в публичном Git-репозитории Oracle. Ядро позиционируется в качестве альтернативы штатному пакету с ядром, поставляемому в Red Hat Enterprise Linux, и предоставляет ряд расширенных возможностей, таких как интеграция DTrace и улучшенная поддержка Btrfs. Новая версия примечательна проведением работы по исправлению ошибок и обновлению драйверов устройств: Поддержка серверов HP Generation 9, включая возможность использования накопителей HP SSD Smart Path с массивами HP Smart Array; Поддержка семейства Ethernet-контроллеров Intel XL710; Обновление паравиртуальных драйверов для запуска Oracle Linux в качестве гостевой системы в Microsoft Hyper-V; Обновлён DTrace-провайдер profile. В новом выпуске добавлена поддержка контрольных вызовов profile-n, дающих возможность через фиксированные промежутки времени сбрасывать статистику с информацией об обработке прерываний на всех активных CPU (по аналогии с tick-n, который позволял сбрасывать данные только по конкретнымм CPU); Обновлены версии драйверов для устройств хранения и сетевых адаптеров компаний Broadcom, Brocade, Cisco, Emulex, HP, Intel, Qlogic.
