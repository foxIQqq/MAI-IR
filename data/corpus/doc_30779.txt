Релиз MySQL Community Server 5.5.13

Представлен корректирующий выпуск СУБД MySQL 5.5.13 в котором исправлено 23 ошибки, из которых одна приводила к краху рабочего процесса при подготовке XA-транзакции без её последующего коммита (InnoDB). Устранены утечки памяти при использовании "LOAD DATA INFILE" и "PROCEDURE ANALYZE()". Дополнительно сообщается о наличии неисправленной проблемы, приводящей к неработоспособности разработанных для MySQL 4.0 старых клиентских библиотек при попытке соединения с MySQL 5.5.12 и 5.5.13.
