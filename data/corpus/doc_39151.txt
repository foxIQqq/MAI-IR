Выпуск SUSE Cloud 3

Компания SUSE представила продукт SUSE Cloud 3, представляющего собой вариант SUSE Linux Enterprise Server, рассчитанный на быстрое развёртывание инфраструктуры для обслуживания приватных облачных сервисов на базе платформы OpenStack. Версия SUSE Cloud 3 примечательная переходом на использование облачной платформы OpenStack 2013.2 "Havana" и обеспечением поддержки развёртывания приватных облачных систем в существующей инфраструктуре, виртуализированной средствами VMware vSphere и VMware vCenter Server. При помощи SUSE Cloud компании могут развернуть локальную облачную инфраструктуру, подобную Amazon EC2, но выполняемую исключительно на собственных серверах, полностью подконтрольных компании. Дистрибутив поддерживает установку сервера Cloud Administration Server для централизованного управления облачной инфраструктурой, а также узлов Cloud Control Nodes, Compute Nodes и Storage Nodes на которых осуществляется управление пользователями, хранение данных и выполнение виртуальных машин. Из компонентов OpenStack в продукте задействованы: инструментарий Nova для управления работой больших групп VPS-серверов, сервис доставки и хранения образов виртуальных машин Glance, система идентификации Keystone, управляющий web-интерфейс Dashboard, сервис для управления жизненным циклом работающих в облаке приложений Orchestration и системы мониторинга Telemetry. В комплект также входят средства интеграции с конструктором образов виртуальных окружений SUSE Studio и системой для управления инфраструктурой Linux-серверов SUSE Manager. Дополнительно подготовлен репозиторий, который можно использовать для установки пакетов с компонентами OpenStack в системах на базе SLES 11 SP3 и openSUSE 12.3/13.1.
