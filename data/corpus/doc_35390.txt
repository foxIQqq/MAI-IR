Увидел свет дистрибутив Linux Mint 14 с десктоп-окружениями MATE и Cinnamon

Анонсирован релиз дистрибутива Linux Mint 14, основанного на кодовой базе Ubuntu 12.10, полностью совместимого с ним, но существенно отличающегося подходом к организации интерфейса пользователя и подбором используемых по умолчанию приложений. Разработчики Linux Mint предоставляют десктоп-окружение, соответствующее классическим канонам организации рабочего стола, которое является более привычным для пользователей не принимающих новые методы построения интерфейса Unity и GNOME 3. Для загрузки доступны DVD-сборки на базе оболочек MATE 1.4 (966 Мб) и Cinnamon 1.6 (884 Мб). Кроме отличий в организации рабочего стола, в дистрибутиве задействован ряд оригинальных приложений, упрощающих настройку и работу в системе начинающих пользователей. Например, в Linux Mint используется переработанная система меню, собственный менеджер установки и обновления приложений, оригинальный интерфейс для настройки системы mintConfig, "мастер" для настройки различных параметров системы для начинающих пользователей в стиле вопрос-ответ, интерфейс для настройки параметров десктопа, средство для выполнения резервного копирования mintBackup и т.п. Ключевые особенности Linux Mint 14: В состав включены новые версии десктоп-окружений MATE 1.4 и Cinnamon 1.6, оформление и организация работы в которых продолжает развитие идей GNOME 2 - пользователю предлагается рабочий стол и панель с меню, областью быстрого запуска, перечнем открытых окон и системным лотком с запущенными апплетами. MATE продолжает развитие кодовой базы GNOME 2.32, основан на GTK2+ и полностью избавлен от пересечений с GNOME 3, что позволяет использовать традиционный рабочий стол GNOME 2 параллельно с рабочим столом GNOME 3. MATE может работать на системах без современных 3D-видеокарт. В новой версии MATE обновлены бэкенды mate-keyring и libmatekeyring, обеспечена возможность передачи файлов через Bluetooth, представлен новый режим для переключения между окнами по Alt+Tab. В файловый менеджер Caja (форк Nautilus) добавлена возможность работы с сервисом Dropbox, добавлена кнопка для сравнения файлов и кнопка для смены режимов просмотра/редактирования пути к файлам. Cinnamon основан на GTK3+ и технологиях GNOME 3. Проект развивает оболочку GNOME Shell и оконный менеджер Mutter в направлении предоставления окружения в стиле GNOME 2 с более современным оформлением и задействованием элементов из GNOME Shell, дополняющих средства классического десктопа. Для работы прошлых выпусков Cinnamon было обязательным использование видеодрайвера с поддержкой OpenGL. Теперь пользователю предлагается новый вид сессии Cinnamon 2D, при котором для формирования вывода применяется программный рендеринг, что позволяет использовать Cinnamon на системах с видеодрайверами без поддержки OpenGL. В новой версии Cinnamon добавлена поддержка создания именованных виртуальных рабочих столов, в состав включен апплет для быстрого показа полного списка окон на всех рабочих столах, новый апплет для вывода уведомлений, возможность смены оформления режима переключения между окнами по Alt-Tab. Произведена интеграция с файловым менеджером Nemo, в рамках которого развивается форк Nautilus 3.4 в котором сохранены такие возможности, как панель инструментов и меню, двухпанельный режим, меню со ссылками для быстрого перехода, компактная форма отображения списка файлов и боковая панель. Подробный обзор Cinnamon 1.6 можно прочитать в ранее опубликованном анонсе. Обновлён используемый в дистрибутиве дисплейный менеджер MDM, представляющий собой форк GDM 2.20, развиваемый в рамках проекта MATE. Как и GDM новый дисплейный менеджер предоставляет графические инструменты настройки параметров входа, возможность использования визуальных тем, средства организации удалённой работы, подключения своих скриптов для обработки событий, режим автоматического входа в систему, возможность выбора языка. В новой версии MDM добавлена поддержка классических тем оформления GDM 2, в поставку включена подборка из 30 подобных визуальных тем. Улучшена организация вывода списка пользователей и ассоциированных с ними картинок. Отныне вместо ввода имени можно сразу выбрать пользователя по его имени или изображению. Улучшен интерфейс переключения между пользователями, поддержка которого интегрирована в хранитель экрана (в режиме блокировки выводится кнопка "Switch user..."); Переработаны некоторые внутренние подсистемы менеджера установки приложений. Вместо процесса aptdaemon, из-за которого наблюдались крахи и зависания, теперь используется собственный APT-клиент. Добавлена полноценная поддержка debconf, благодаря которой отпала необходимость использования Synaptic для установки использующих debconf пакетов с диалогами по настройке, таких как пакеты с шрифтами Microsoft или Wine. Вместо требования ввода пароля в перед началом установки или обновления менеджер установки приложения теперь сразу запускается под пользователем root (пароль вводится один раз). Добавлены настройки режима поиска по мере ввода; Серия системных изменений: Вместо USB-ImageWriter задействован MintStick, отличающийся более удобным пользовательским интерфейсом. Редактор Gedit 2.30 заменён на Gedit 3, который поддерживает больше возможностей и более стабилен; В MintSystem добавлены дополнительные команды: dns-fix для задания собственной привязки имени хоста в OpenDNS, xchat-systray для добавления постоянной иконки Xchat на системный лоток.
