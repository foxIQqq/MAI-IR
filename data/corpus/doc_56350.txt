Выпуск дистрибутива Pop!_OS 21.10, развивающего рабочий стол COSMIC

Компания System76, специализирующаяся на производстве ноутбуков, ПК и серверов, поставляемых с Linux, опубликовала выпуск дистрибутива Pop!_OS 21.10. Pop!_OS основан на пакетной базе Ubuntu 21.10 и поставляется с собственным окружением рабочего стола COSMIC. Наработки проекта распространяются под лицензией GPLv3. ISO-образы сформированы для архитектуры x86_64 и ARM64 в вариантах для графических чипов NVIDIA (2.9 ГБ) и Intel/AMD (2.5 ГБ), а также для плат Raspberry Pi 4 (2.4 ГБ). Дистрибутив в первую очередь ориентирован на людей, использующих компьютер для создания чего-то нового, например, занимающихся разработкой контента, программных продуктов, 3D-моделей, графики, музыки или научной работой. Идея разработки собственной редакции дистрибутива Ubuntu пришла после решения Canonical перевести Ubuntu с Unity на GNOME Shell - разработчики System76 занялись созданием новой темы оформления на базе GNOME, но затем поняли, что готовы предложить пользователям иное окружение рабочего стола, предоставляющего гибкие средства для настройки под текущий рабочий процесс. Дистрибутив поставляется с рабочим столом COSMIC, построенным на базе модифицированной оболочки GNOME Shell и набора оригинальных дополнений к GNOME Shell, собственной темой оформления, своим набором пиктограмм, другими шрифтами (Fira и Roboto Slab) и изменёнными настройками. В отличие от GNOME, в COSMIC продолжает использоваться раздельное представление для навигации по открытым окнам и установленным приложениям. Для манипуляции окнами предоставлен как привычный для новичков традиционный режим управления при помощи мыши, так и мозаичный режим компоновки окон, позволяющий управлять работой только при помощи клавиатуры. После формирования выпуска Pop!_OS 21.10 разработчики намерены трансформировать COSMIC в самодостаточный проект, не использующий GNOME Shell и развиваемый на языке Rust. В новом выпуске: Переделан интерфейс для навигации по установленным приложениям. Вместо полноэкранного режима просмотра список доступных приложений и средства для поиска программ теперь отображаются в небольшом окне, выводимом поверх содержимого рабочего стола. Список программ можно открыть через верхнюю панель, жестом на тачпаде (сдвиг четырьмя пальцами вправо) или горячей клавишей Super + A. Из особенностей нового интерфейса навигации по приложениям отмечается улучшение работы на системах с несколькими мониторами (окно открывается на экране, в котором находится курсор мыши); сортировка в алфавитном порядке; возможность группировки приложений по подкаталогам в режиме drag&drop (разделение напоминает использование вкладок); поддержка фильтрации вывода по маске как уже установленных приложений, так и приложений, доступных для установки; компоновка, более оптимальная для широкоформатных мониторов. Началось формирование экспериментальных сборок для плат Raspberry Pi 4. Расширена поддержка оборудования. Система поставляется с ядром Linux 5.15.5 и свежими проприетарными драйверами NVIDIA. Перед релизом проведено тестирование дистрибутива на широком спектре чипсетов, процессоров и аппаратных компонентов. Упрощён процесс обновления системы. В процессе запуска инсталлятора проверяется наличие уже установленной версии Pop!_OS и в случае обнаружения предлагается опция обновления системы без полной переустановки и с сохранением пользовательских файлов, доступная на стадии до разблокировки зашифрованных разделов. Для повышения надёжности обновления запасной дисковый раздел (restore) теперь обновляется отдельно и до обновления основной операционной системы, что позволяет сохранить работоспособность в случае сбоев во время обновления. Улучшена обработка внесённых пользователем изменений в /etc/fstab. Обеспечено отключение добавленных пользователем PPA-репозиториев. Реализовано распространение обновлений пакетов из собственного репозитория. Введена в строй собственная инфраструктура непрерывной интеграции для тестирования и оценки качества пакетов перед их помещением в репозиторий. Из актуальной кодовой базы GNOME перенесены исправления и улучшения. В том числе в интерфейс настройки Wi-Fi перенесена поддержка сортировки по текущим и прошлым соединениям, а также уровню сигнала. В файловый менеджер перенесена возможность динамического уточнения результатов поиска по мере ввода поискового запроса.
