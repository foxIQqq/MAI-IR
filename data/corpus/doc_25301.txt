Интервью с участниками конференции FOSDEM 2010 (часть 2)

Опубликовано продолжение серии интервью с участниками конференции FOSDEM 2010, которая пройдет 6-7 февраля в Брюсселе (Бельгия). Andrew Tanenbaum (MINIX 3) Benoît Chesneau (CouchDB) Elena Reshetova (инженер по безопасности проекта Maemo) Lindsay Holmwood (разработчик распределенной системы мониторинга Flapjack) David Fetter (PostgreSQL) Philip Tellis (разработчик JavaScript библиотеки для построения диаграмм YUI-Flot) Bernard Li (разработчик Ganglia, системы управления и мониторинга кластером) Andrew Lewman (руководитель анонимной сети Tor); Mark Wielaard (разработчик SystemTap, аналога DTrace для Linux) Isabel Drost (разработчик Apache Hadoop и один из основателей Apache Mahout) Adrian Bowyer (участник проекта RepRap, по созданию 3D-принтера, способный создавать собственные копии) Christoph Pojer (разработчик JavaScript фреймворка MooTools) Ранее анонсированные интервью: Greg Kroah-Hartman (основатель инициативы Linux Driver Project и мантейнер ряда подсистем ядра Linux) Sergey Petrunya (MariaDB) Brooks Davis (член FreeBSD Core Team, участник проектов Ganglia и Sun Grid Engine) David Recordon (один из разработчиков PHP и vBulletin, доклад посвящен использованию открытого ПО для обеспечения работы Facebook) Dylan Schiemann (Dojo Toolkit) David Fifield (участвует в разработке сканера безопасности Nmap) Richard Clayton (эксперт по безопасности компьютерной лаборатории Кэмбриджского университета Wim Remes (доклад о системе обнаружения атак OSSEC)
