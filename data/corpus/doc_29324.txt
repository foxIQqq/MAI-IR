Представлен Linux-дистрибутив Saline OS 1.0

Представлен новый дистрибутив Saline OS 1.0, основанный на пакетной базе Debian 6.0 "Squeeze" и полностью совместимый с ним. По сути Saline является сборкой классического Debian, снабженной графическим окружением Xfce 4.6.2 и поставляемая с достаточно полной подборкой предустановленных программ. Размер установочного iso-образа, который может работать в режиме LiveDVD, составляет 886 Мб. В состав Saline OS включено приложение sgfxi, упрощающее установку проприетарных драйверов; по умолчанию включены дополнительные неофициальные репозитории пакетов, например, бэкпорты, репозитории со свежими версиями WINE и прошивками для беспроводных карт; подготовлен скрипт для быстрой установки мультимедиа кодеков. Из включенных в поставку программ можно отметить: web-браузер Chromium, почтовый клиент Icedove (Thunderbird), клиент для мгновенного обмена сообщениями Pidgin, органайзер Osmo, видеоплеер Parole, музыкальный плеер Rhythmbox, программа для записи CD/DVD XFBurn, Fotoxx, Cheese, Remastersys, Transmission, GIMP и т.д..
