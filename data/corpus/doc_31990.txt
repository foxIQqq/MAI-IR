Релиз независимой сборки мобильной платформы Android - CyanogenMod 7.1

После 6 месяцев разработки увидел свет релиз развиваемой независимым сообществом альтернативной сборки платформы Android - CyanogenMod 7.1. Релиз CyanogenMod 7.1 основан на открытой кодовой базе Android 2.3.7, содержит большинство представленных в данном выпуске новшеств, а также ряд полезных для энтузиастов и разработчиков улучшений. В новой версии сохранены все оригинальные возможности, реализованные в прошлых выпусках CyanogenMod. По сравнению с прошлой версией добавлена стабильная поддержка 24 новых устройств: HTC: Desire S, Incredible S, Incredible 2 LG: Optimus 2X и T-Mobile G2x Motorola: Backflip (Motus), Cliq / Cliq XT, Defy, Droid 2, Droid X; Samsung: Captivate, Fascinate, Mesmerize, Showcase, Vibrant, Galaxy S, Galaxy S2; Sony Ericsson: Xperia X8, Xperia Mini, Xperia Mini Pro, Xperia Neo, Xperia Play, Xperia Ray, Xperia Arc; ZTE V9. В текущем состоянии сборка поддерживает 68 различных моделей устройств. Для каждой модели доступно пошаговое руководство по замене штатной прошивки на CyanogenMod, для упрощения установки подготовлено специальное приложение ROM Manager, входящее в комплект CyanogenMod. Благодаря помощи от компаний Samsung и Sony Ericsson, предоставивших разработчикам CyanogenMod информацию и оборудование, удалось оперативно обеспечить поддержку всего модельного ряда данных производителей. Загрузить прошивку для своей модели телефона и дополнительный пакет для интеграции с сервисами Google можно на данной странице. Отдельно отмечается, что многие устройства не представлены в финальном списке, так как не вышли из стадии бета-тестирования. Например, отдельно можно загрузить прошивки для устройств HTC Pyramid (HTC Sensation), HTC Doubleshot, HTC Shooter, HP Touchpad, LG Optimus 3D, для которых через несколько недель планируется изменить статус на кандидат в релизы. Особенности CyanogenMod: Полноценная подборка музыкальных программ с поддержкой кодека FLAC и возможностью использования эквалайзера; Улучшение поддержки аппаратных устройств, например, обеспечена возможность связывания с другими устройствами по USB и Bluetooth, добавлена поддержка Bluetooth HID и Ad-Hoc wifi, интегрированы дополнительные оптимизирующие патчи для процессоров ARM. Для некоторых устройств имеется поддержка USB host; Наличие стандартных утилит в базовой поставке. Например, в комплект включены BusyBox, rsync, htop, nano, powertop, bash и т.д. Для управления выполнением привилегированных действий используется специальный менеджер доступа, который позволяет разрешить или запретить получения root-прав для определенного приложения; Дополнительный раздел настроек с параметрами тюнинга производительности, пользовательского интерфейса, звуковой подсистемы и методов ввода; Spare Parts - набор сервисных функций для контроля за использованием аккумулятора, слежением за потреблением ресурсов, оценки энергопотребления отдельных приложений, настройки анимационных эффектов и других параметров интерфейса; Поддержка OpenVPN; Дополнительно поставляется пакет с подборкой приложений для работы с сервисами Google: Maps, Market, Gmail, Talk и YouTube; В качестве интерфейса для запуска приложений задействован ADWLauncher; Функция выполнения заданных действий через экранные жесты, которые могут вводиться даже в режиме блокировки экрана. Настройка вывода дополнительной информации в заставке блокировки экрана; Поддержка браузером приватного режима навигации, при котором не сохраняется история посещения и данные не оседают с кэше; Функция Apps2SD для переноса приложений из встроенной памяти на SD-карту и обратно; Поддержка подключения визуальных тем. Возможность изменения оформления статусной области и области уведомлений; Интеграция межсетевого экрана Phone Goggles, позволяющего разрешить совершение телефонных звонков и отправку SMS только указанным приложениям, защитив себя от активности вредоносного ПО. Из добавленных в CyanogenMod 7.1 изменений можно отметить: Поддержка подключения мыши с BlueTooth-интерфейсом; Поддержка режима "Tethering" через Bluetooth, а не только через USB. Режим Tethering позволяет организовать для внешних устройств выход в сеть через телефон, при этом соединение выглядит как виртуальный Ethernet; Улучшение системы вывода уведомлений, возможность очистки списка накопившихся уведомлений экранным жестом; В сканере медиа-контента улучшена поддержка привязки альбомов и музыкантов; Добавлена поддержка опции выхода из спящего режима по кнопке управления громкостью; Возможность отзыва прав, делегированных приложению ранее; Новая версия приложения Superuser; Поддержка изменения яркости через жест скольжения в области статусной строки; В контекстное меню добавлена опция "copy all"; Улучшенный диалог выбора даты/времени; Поддержка настройки вращения экрана на 0/90/180/270 градусов; Улучшение работы DSP Manager; Поддержка запуска программ через использование экранных жестов в фоне; Возможность скрытия кнопки hold во время звонка; Поддержка использования масок (например, "*") при заполнении черного списка телефонных номеров; Опция для уведомления о входящих звонках в фоне, если телефон не находится в режиме блокировки (ранее уведомление всплывало вытесняя текущее приложение из фокуса); Поддержка режима поиска станций для FM-тюнеров; Режим фокусировки камеры через касание к области экрана; Поддержка создания скриншотов через меню по копке Вкл/Выкл; Бэкпортирована поддержка формата WEBP; Дополнительные ARM-оптимизации для Skia/Zlib/libcore; Поддержка вывода календаря на экран в режиме блокировки; Поддержка AVRCP 1.3; Поддержка общесистемного кэша для DNS; Поддержка таймера для создания снимков камерой с определенной задержкой; Для устройств Click и Motorola увеличена скорость записи фотографий, добавлена поддержка учета местоположения в программу для работы с фотографиями; Для устройств Click, Motorola, Crespo/Crespo4G, HTC msm7x30, Buzz ядро Linux обновлено до версии 2.6.35; Для Encore добавлена поддержка режима работы в роли USB-хоста, добавлены новые драйверы GPU.
