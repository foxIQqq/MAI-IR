Вышел августовский номер журнала Linux Games Technologies

Вслед за возрожденным июльским выпуском появился и августовский номер электронного журнала Linux Games Technologies, посвященный разработке игр. В номере: Обзор трех современных коммерческих игровых движков: Shiva 3D, Unigine 3D, id Tech 4/5. Обзор движка для создания текстовых игр Instead 1.2.0; Обзор космического симулятора «Babylon 5: I’ve Found Her», построенного на основе сюжета сериала «Вавилон-5»; Обзорная статья и практический урок по использованию специализированной программы для создания трехмерных моделей растений NgPlants; Руководство по подготовке текстуры камня в Gimp; Создание инсталлятора для игры, работающей в ОС семейства Unix, используя ПО InstallJammer; Продолжение изучения Agile Fish Game Editor, игрового конструктора 2D-игр; Практикум с OpenGL 3.2: Рассматривается использование буферов и шейдеров, а также визуализация Mesh-объектов.
