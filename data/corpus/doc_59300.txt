Новый выпуск клиента Steam c переделанным интерфейсом

Компания Valve представила новый значительный выпуск клиента сервиса доставки игр Steam для Linux. Число игр для Linux, доступных в каталоге Steam, доведено до 11132. По данным сервиса Steam Database, пользователи Linux запускают через Steam 1252 игры. Наиболее заметные улучшения в новой версии Steam: Расширены возможности Steam Desktop и значительно модернизирован интерфейс пользователя, в котором переработаны диалоги, меню, заголовки и нижняя панель, заменены шрифты и изменена цветовая гамма. Переделаны раздел с настройками и менеджер скриншотов. Обновлён интерфейс для вывода уведомлений и индикации их появления (зелёный индикатор теперь загорается только при поступлении действительно важных уведомлений). Добавлена кнопка для просмотра полученных уведомлений в виде списка. Предложены новые настройки для определения того, какие и где уведомления показывать. Полностью переделан всплывающий экран, показываемый во время игры при нажатии Shift+Tab. Добавлена новая панель инструментов, предоставляющая доступ к функциям, которые могут потребоваться во время игры, таким как чат, браузер, просмотр достижений и доступ к обсуждениям. Предоставлена возможность персонализации контента, показываемого на всплывающем экране. Добавлена возможность ведения заметок, которые можно оставлять во время прохождения игры. В заметки можно вставлять изображения и управлять форматированием текста. Заметки сохраняются в привязке к текущей игре и синхронизируются между системами пользователя. Появилась поддержка прикрепления окон из всплывающего экрана для постоянного отображения во время игры поверх другого содержимого. Например, можно закрепить заметку с подсказками, план действий в игре, обсуждение с другими участниками и даже окно web-браузера. Возможно изменение уровня прозрачности закрепляемого окна. Во всплывающий экран добавлена панель Game Overview с обзором событий, произошедших в игре с момента прошлого запуска. Например, в панели показываются достижения, играющие в данный момент друзья, новости и инструкции. Улучшен интерфейс для настройки игровых контроллеров. Задействован новый фреймворк, применяемый для совместного использования кода в реализациях Steam Desktop Client, режима Big Picture и Steam Deck. В сборках для macOS и Linux по умолчанию задействовано аппаратное ускорение графики, которое позволило сравнять производительность интерфейса данных сборок с версией для Windows и добиться более плавной прокрутки, мгновенной анимации и повышения отзывчивости интерфейса.
