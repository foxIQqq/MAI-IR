Корректирующий релиз MySQL Community Server 5.1.49

Вышел релиз комьюнити сборки MySQL 5.1.49, содержащей исправление 26 ошибок, 3 из которых приводили к краху рабочего процесса: при парсинге излишне большого запроса, не влезающего в память; при достаточно редком стечении обстоятельств при выполнении FLUSH LOGS на реплицированном сервере; при передаче в хранимую процедуру при помощи функции mysql_stmt_send_long_data() значения, представленного в некорректном типе. Также исправлена ошибка, из-за которой для некоторых запросов при указании "ORDER BY ... DESC" сортировка выполнялась неверно. Поставляемый в комплекте с MySQL дополнительный InnoDB Plugin обновлен до версии 1.0.10, относящейся к категории пригодных к промышленной эксплуатации.
