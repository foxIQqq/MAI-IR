Выпуск операционной системы Chrome OS 64

Компания Google сформировала релиз операционной системы Chrome OS 64, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 64. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 64 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 64: Упрощено создание скриншотов на устройствах Chromebook с поворачиваемым на 360-градусов экраном. Для создания скриншота при работе в планшетном режиме теперь можно одновременно нажать кнопку питания и уменьшения громкости (как в Android); Добавлены горячие клавиши для переноса окон на внешний монитор: "Search + Alt + стрелка влево" - переместить активное окно на левый ближайший экран, "Search + Alt + стрелка вправо" - на правый, "Search + Alt + стрелка вверх" - на верхний и "Search + Alt + стрелка вниз" - на нижний; Добавлено пять новых геометрических аватаров, предлагаемых по умолчанию (chrome://settings/changePicture); Для Google PixelBook представлен экспериментальный режим разделения экрана (chrome://flags/#enable-tablet-splitview). При нажатии пиктограммы "[]||" в нижнем правом углу экрана показываются все активные окна, которые можно прикрепить к выделенным областям; Модернизировано оформления кнопки отключения уведомлений ("Не беспокоить"), которая приведена в соответствие с концепцией Material Design; Переработан интерфейс выбора обработчиков (Intent Picker) для приложений из Google Play, который теперь открывается в том же окне, перекрывая имеющийся контент; Для приложений из Google Play включена поддержка VPN; Улучшена поддержка доступа к защищённому контенту из Android-приложений; Оптимизирован процесс обновления содержимого контейнера с компонентами платформы Android; Добавлены настройки для калибровки сенсорного экрана; Ускорена работа блокировщика экрана.
