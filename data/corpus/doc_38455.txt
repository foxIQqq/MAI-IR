Развитие CnCNet 5, сетевого стека для игр серии Command and Conquer

CnCNet 5, новая ветка неофициального сетевого стека для игры через интернет в классические игры серии Command and Conquer (Tiberian Dawn, Red Alert, Tiberian Sun, Red Alert 2 и их пакетов дополнений), находится в фазе тестирования довольно долгое время. Ниже представлены некоторые возможности, которые предоставляет текущая тестовая версия CnCNet 5 по сравнению с CnCNet 4: Улучшенный код, обеспечивающий поддержку сети: CnCNet 5 использует модифицированную версию кода Westwood Online вместо сетевого кода от игр по LAN. Это делает игру плавнее и быстрой по сравнению со старым методом использования кода LAN-игр, который использовался в CnCNet 4. Возможность выбора туннельного сервера. Туннельный сервер позволяет игрокам, не имеющим возможности перенаправить порты, использовать CnCNet. Эта возможность была в CnCNet со времени CnCNet 4, но в CnCNet 5 появилась возможность для оптимальной производительности выбрать ближайший туннельный сервер. Поддержка создания туннельного сервера: Жители местности далёкой от существующих туннельных серверов, имеющие достаточно широкие каналы связи, теперь могут создать свой собственный туннельный сервер и предоставить более качественное соединение для игроков, живущих поблизости. Собственный клиент для CnCNet 5, который очень полезен для моддеров. Возможности клиента CnCNet 5: Огромная база данных карт с категориями и возможностью поиска. Режим наблюдателя для RA1 Поддержка CO-OP для RA1 Выбор команды и стартовой позиции для RA1 Предпросмотр карт Новые опции игры такие как "Техника Давит Пехоту" и "Свёртывание MCV", "Исправление эксплоита Формации" и "Исправление Эксплоита Дальности" для RA плюс множество других возможностей. Автоматическая передача карт и вскоре: "мини-моды" IRC сервер CnCNet: Предоставляет возможность слать личные сообщения, позволяет создавать игры защищённые паролем и возможностью выкидывания и бана и многое другое. Будущие возможности CnCNet 5: Турнирные игры и поддержка ладдера совместно с PlayCnC Поддержка Tiberian Sun и Red Alert 2, если кто-то пожелает взять эту задачу Пользователям, заинтересованным в оценке текущей тестовой версии CnCNet 5 с использованием клиента FunkyFr3sh, версии для Command & Conquer (C&C1) и C&C: Red Alert (RA1) можно найти в данной ветке форума проекта. Проект развивается энтузиастами, исходные коды компонентов доступны на github. Игра на платформах отличных от Windows возможна с помощью wine или playonmac.
