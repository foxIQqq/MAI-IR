Вышел релиз Lazarus 0.9.28, среды разработки для FreePascal

Спустя год с момента прошлого релиза вышла новая версия Delphi-подобной свободной среды разработки для FreePascal - Lazarus 0.9.28. Из новшеств можно отметить: Для Linux теперь по умолчанию используется Gtk2 набор виджетов; Улучшена поддержка платформ Windows CE, Windows Vista и Windows 7; Реализован новый класс TFrame, в классе TMonitor появилась поддержка многомониторных конфигураций, в TreeView добавлена возможность редактирования элементов; Новые компоненты: TShellTreeView - для просмотра дерева файлов/директорий, TShellListView - для просмотра списка файлов/директорий, TFilterComboBox - для применения фильтра при выводе файлов; Добавлена возможность экспорта отчетов LazReport в PDF формате
