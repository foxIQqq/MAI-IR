Вышел релиз LiveCD дистрибутива NST (Network Security Toolkit) 2.11.0

После почти года разработки, представлен новый релиз LiveCD дистрибутива NST (Network Security Toolkit) - 2.11.0. Дистрибутив построен на базе Fedora 11 Linux и ориентирован для проведения проверки безопасности сети. В новой версии полностью переработана архитектура сборки дистрибутива, все системные, сетевые и связанные с безопасностью инструменты отныне включены в виде RPM-пакетов, что позволяет осуществить формирование NST LiveCD в виде "спина" проекта Fedora, базирующегося на стандартном Fedora LiveCD. Также дистрибутив теперь поддерживает установку дополнительных пакетов из внешних репозиториев, при помощи утилиты YUM. Другие изменения: Дистрибутив поддерживает возможность записи изменений в файловой системе на отдельный накопитель при загрузке с доступного только на чтение носителя. Добавлена возможность установки на USB Flash, при этом сохраняются все изменения, произведенные в рамках пользовательских сессий. Добавлена возможность установки на жесткий диск, с последующей доустановкой необходимых программ из репозиториев Fedora; В состав включен скрипт "nsttraceroute", позволяющий сформировать на базе вывода от утилиты traceroute карту прохождения маршрута, отражаемую через сервис Google Earth; Добавлены две новые утилиты для перехвата трафика: "driftnet" и "tcpxtract". Первая позволяет выделить из потока изображения в форматах GIF, JPEG и PNG, а вторая - документы, включая PDF и файлы Microsoft Word; Добавлено множество новых утилит для проверки безопасности и обновлены версии включенных ранее программ.
