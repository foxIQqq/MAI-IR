Компания Google наймет двух разработчиков для работы над поддержкой Android в Linux-ядре

Компания Google намерена принять на работу двух дополнительных разработчиков, задачей которых будет обеспечение синхронизации кодовой базы модифицированного ядра платформы Android с основной веткой Linux-ядра, а также курирование связанных с ядром Android вопросов в сообществе. Судя по всему, процесс интеграции разработанных в рамках проекта Android улучшений сдвинулся с места, например, один из разработчиков Android уже представил для рассмотрения в списке рассылки ядра код с реализацией API для управления переводом блочных устройств в спящий режим. Напомню, что в феврале код драйверов для платформы Android был исключен из staging-дерева ядра из-за отсутствие должной поддержки со стороны компании Google, не продолжившей устранение недочётов в рамках слияния кода с ядром Linux. Несколько недель назад, на саммите разработчиков ядра, Крис ДиБона (Chris DiBona), управляющий Open Source проектами Google, обсудил с основными разработчиками Linux-ядра план по постепенному возвращению в основную ветку ядра кода, связанного с платформой Android.
