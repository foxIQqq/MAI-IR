Релиз среды управления контентом предприятия Alfresco Community 3.4

Компания Alfresco выпустила релиз свободной среды управления контентом предприятия Alfresco Community 3.4 ECM (Enterprise Content Management), исходные тексты которой распространяются в рамках лицензии LGPL. Alfresco конкурирует с такими коммерческими системами, как Documentum и Microsoft SharePoint, отличаясь расширяемой модульной структурой, полной поддержкой стандарта OASIS CMIS 1.0 и в несколько раз обгоняющим конкурентов быстродействием. Код системы написан в аспектно-ориентированном стиле на языке Java с использованием компонентов Spring, Hibernate, Lucene и MyFaces. В качестве СУБД поддерживается использование MySQL и PostgreSQL. Ключевые новшества версии 3.4: Alfresco Web Quick Start - содержит набор шаблонов для быстрого создания интерактивных сайтов на базе платформы Alfresco Share, позволяющей организовать совместную работу групп пользователей, подготовку и публикацию web-контента; Инструментарий Office-to-Web позволяет пользователям создать документы в любом офисном пакете, после чего отредактировать их совместно с другими сотрудниками в интерфейсе Alfresco или Google Docs, преобразовать и переназначить в случае необходимости, утвердить у руководства и, в итоге, опубликовать в web. Для передачи документа может использоваться протокол MS SharePoint, сетевой CIFS-диск или API для интеграции с Google Docs; Набор Spring-сервисов для управления web-контентом: на базе Java-фреймворков Spring и Spring Surf подготовлены сервисы для управления контентом, которые могут быть доступны через API OpenCMIS или интегрированы в web-приложения; Интеграция с корпоративными порталами и социальными ресурсами: новый набор DocLib-портлетов (подключаемых компонентов портала) позволяет обеспечить бесшовную интеграцию поддерживаемых в Alfresco репозиториев и контента с порталами, созданными на базе продуктов Liferay, Quickr и Confluence. Распределенная репликация контента: встроенная поддержка репликации контента дает компании возможность создания сети объединенных репозиториев. Новая система позволяет автоматически реплицировать ключевые документы в удаленные офисы, расширяя возможности по совместному использованию информации, сокращая трафик и увеличивая скорость доступа к новой информации; Через 4 недели планируется выпустить корректирующий релиз Alfresco Community 3.4.b, в который будут включены шаблоны для создания типичных категорий корпоративных сайтов (финансы, гос. учреждения, медиа и развлекательная индустрия). Также в пакет будет интегрирован инструмент для автоматического формирования HTML-страниц на основе документов, созданных в Microsoft Word.
