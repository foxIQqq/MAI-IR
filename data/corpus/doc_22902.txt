Вышел релиз Mac OS X 10.5.8. Обзор open source приложений для Mac OS X

Компания Apple выпустила релиз Mac OS X 10.5.8, содержащий исправление накопившихся ошибок и проблем безопасности. Отдельно можно отметить включение в состав новой версии web-браузера Safari 4.0.2. Кроме того, можно отметить публикацию статьи "Open source stars for Mac OS X" с обзором открытого ПО для Mac OS X. В первой части представлен список web-браузеров, email-клиентов, программ для мгновенного обмена сообщениями, планировщиков, офисных программ и редакторов изображений.
