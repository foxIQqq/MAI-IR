Вышел Calculate Linux Desktop 9.3

Вышел релиз десктоп-ориентированного дистрибутива Calculate Linux Desktop 9.3, представляющего собой скомпилированный stage4-образ Gentoo Linux, снабженный программой для быстрой установки. Новые возможности: Добавлена поддержка серверов с включенной репликацией. Что позволяет использовать единый профиль пользователя в нескольких сетях. Поддержка репликации появится в Calculate Directory Server 9.4. Процесс установки системы: На рабочем столе пользователя LiveCD добавлена иконка "Терминал суперпользователя". Добавлена поддержка OpenGL для видеоадаптеров Intel и ATI. Поддержка видеоадаптера ATI реализована через проприетарный драйвер fglrx. Изменения в интерфейсе: В установленной системе по умолчанию включен Composite. При отсутствии курсора, нижняя панель меню убирается, освобождая экран. Меню KDE заменено виджетом "Меню Lancelot". В Konqueror добавлен режим просмотра страниц движком WebKit (меню Вид -> Режим просмотра -> WebKit). В Konqueror добавлено поле ввода для быстрого поиска. Виджет "Корзина" перемещен в правый нижний угол. Прочие улучшения: Добавлен менеджер изображений digiKam. Помимо базовых добавлены виджеты: Translator, Stdin. Клавишей ScrollLock теперь блокируется доступ к рабочему столу. Состав: KDE 4.2.0, Kernel 2.6.27.19, OpenOffice 3.0.1, X.Org 7.4.
