Вышел Calculate Linux 13.19

Доступен релиз отечественного дистрибутива Calculate Linux 13.19, построенного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в шести редакциях: Calculate Directory Server (CDS), Calculate Linux Desktop с рабочим столом KDE (CLD) и XFCE (CLDX), Calculate Media Center (CMC), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS). Все версии дистрибутива распространяются в виде загрузочного livecd образа с возможностью установки на HDD, USB-Flash либо USB-HDD. Это последний релиз в 13.x ветке, на смену которой в скором времени придёт 14-я версия, в которой ожидаются такие улучшения, как полностью переписанные утилиты сборки дистрибутивов, графическая и консольная программы обновления системы, автопроверка обновлений, раздельные профили системы и т.д. Основные изменения: Обновление ПО: ядро Linux 3.14.9, KDE 4.13.2, LibreOffice 4.2.5, Chromium 35. В CLD, CLDX, CDS и CMC добавлена утилита настройки межсетевого экрана Shorewall. В CLD, CLDX и CMC добавлена поддержка файловой системы exFAT; В CLD и CLDX добавлена поддержка PulseAudio. В CLDX включена поддержка GStreamer. На смену аудиопроигрывателю Audacious в CLDX пришёл Clementine, возвращена утилита Blueman для настройки Bluetooth. Исправлена ошибка установки UEFI. Исправлена блокировка экрана в CLDX. Добавлена возможность указания IP-адреса PXE сервера. Исправлено определение наличия видео драйверов в ISO образе. Отменена распаковка Portage во время первой загрузки системы. Возвращена возможность установки системы на Btrfs с LiveCD. Во все дистрибутивы, за исключением CLS и CSS, добавлены утилиты сбора сведений о системе Inxi и Dmidecode. Удалены пакеты gnome-ppp/kppp, DC-клиент, Geany. Улучшена поддержка репозиториев бинарных пакетов: при любом изменении скрипта сборки, зависимые пакеты пересобираются. Состав версий: CDS (Directory Server), i686 - 656 M, x86_64 - 685 M: Bind 9.9.4, ядро Linux 3.14.9, OpenLDAP 2.4.35, Postfix 2.10.3, ProFTPD 1.3.4d, Samba 3.6.23 CLD (KDE desktop), i686 - 2.2 G, x86_64 - 2.4 G: KDE 4.13.2, Amarok 2.8.0, digiKam 4.0, Chromium 35, ядро Linux 3.14.9, LibreOffice 4.2.5 CLDX (XFCE desktop),i686 - 1.6 G, x86_64 - 1.8 G: XFCE 4.10, Gimp 2.8.10, Chromium 35, ядро Linux 3.14.9, LibreOffice 4.2.5 CMC (Media Center), i686 - 941 M, x86_64 - 1.1 G: XBMC 13.0, ядро Linux 3.14.9, Xorg-server 1.15 CLS (Linux Scratch), i686 - 705 M, x86_64 - 728 M: Openbox 3.5.2, Kernel 3.14.9, Xorg-server 1.15 CSS (Scratch Server), i686 - 439 M, x86_64 - 467 M: ядро Linux 3.14.9 Особенности Calculate Linux: 100% совместимость с Gentoo с поддержкой бинарных репозиториев обновлений. Поставка специально разработанных в рамках проекта Calculate утилит для настройки, сборки и установки системы. Готовое клиент-серверное решение. Быстрое развёртывание на предприятии. Полноценная работа в гетерогенных сетях. Удобство администрирования. Rolling-release модель обновлений. Поддерживается интерактивная сборка системы - подготовка ISO-образа системы под свои задачи.
