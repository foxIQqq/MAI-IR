Релиз Bombono DVD 1.2, программы авторинга DVD-дисков

Доступен релиз Bombono DVD 1.2, программы создания видеодисков (DVD-авторинг) для GNU/Linux. По возможностям программа сопоставима с проприетарным пакетами TSUNAMI DVD Author и Sonic DVDit, имеется встроенная возможность просмотра MPEG в режимах монитора и временной шкалы, поддержка импорта макета с DVD-дисков, WYSIWYG-редактор меню c отображением эскизов в процессе работы, поддержка Drag-n-Drop режима, возможность сохранения результатов в директорию, в виде ISO-образа или через прямую запись на DVD. Начиная с версии 1.2 Bombono DVD умеет полностью использовать многоядерные процессоры: задания по перекодированию видео создаются параллельно и/или используется опция "-threads" для ffmpeg. Список изменений: Поддержка многоядерных CPU для перекодирования видео; Расчет оставшегося времени работы; Для меню могут быть использованы стрелки и другие иконки, а не только рамки; можно использовать свои, предварительно добавив их в ~/AppData/Roaming/bombono-dvd/icons (Win7) или ~/.config/bombono-dvd/icons (Linux); Новые рамки для меню, от Dietrich Martin; Множество иконок по умолчанию: галки, стрелки, круги и т.д. Интеграция с веб-браузером для получения фонов меню; Копирование/вставка элементов меню; функция "Скопировать" в меню; Сохранение пропорций при изменении размеров в редакторе (с Shift'ом); Привязка к сетке в редакторе; 3 варианта установить фон для меню: Заполнение, По размеру и Растянуть; Улучшения для DVD с соотношением 16:9; Отчет об ошибках, выявленных перед авторингом; Функция Переименовать в браузерах больше не конфликтует с Drag-n-Drop; SCons теперь не обязателен для работы программы; Поправки для новых версий ffmpeg, gcc, gtk, boost; Исправление ошибок и мелкие улучшения Для программы под Windows появилась платная поддержка, в отличие от Linux-версии; Обновлены старые переводы и добавлены новые переводы (немецкий, китайский упрощенный).
