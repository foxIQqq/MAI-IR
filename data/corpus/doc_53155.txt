В KDE Plasma 5.20 панель задач будет переведена на показ только сгруппированных пиктограмм

Разработчики проекта KDE намерены активировать по умолчанию альтернативную раскладку панели задач, отображаемой внизу экрана и предоставляющей навигацию по открытым окнам и запущенным приложениям. Вместо традиционных кнопок с названием программы планируется перейти к отображению только крупных квадратных пиктограмм (46px), реализованных по аналогии с панелью Windows. Данный вариант опционально поддерживается в панели уже достаточно давно, но теперь его хотят включить по умолчанию, а классическую раскладку перевести в разряд опций. Более того, вместо раздельных кнопок для разных окон планируют включить группировку по приложениям, т.е. все окна одного приложения будут представлены только одной раскрывающейся кнопкой (например, при открытии нескольких окон Firefox в панели будет показана лишь одна кнопка с логотипом Firefox, и только после клика на эту кнопку будут показаны кнопки отдельных окон, т.е. для переключения между окнами вместо одного клика потребуется два и лишнее движение курсора). Подобное поведение можно будет отключить в настройках. Из изменений также отмечается закрепление по умолчанию на панели ярлыков некоторых популярных приложений и возможность вертикального отображения панели. Панель пока оставлена в нижней части, но разработчики намерены обсудить целесообразность переноса панели по умолчанию в левую часть экрана.
