Выпуск библиотеки libamc-2.1.3-20-7c для управления Asterisk из программ на языке C++

Увидела свет стабильная версия библиотеки libamc-2.1.3-20-7c, реализующей C++ класс для удаленного управления АТС на базе Asterisk. Из новшеств можно отметить следующее: Добавлена функция show_dialplan(), которая позволяет получить полный план набора АТС; В архив добавлены бинарные сборки для ОС семейства *BSD; Переработаны установочные скрипты; Для демонстрации работы библиотеки подготовлено приложение vt (VoIPterminal), которое реализует практически все функции доступные в libamc и пригодно для повседневного использования администраторами VoIP-серверов; Устранены ошибки в коде, приводящие к краху приложения. Ключевые особенности libamc: Небольшой размер, высокая производительность; Код оптимизирован для применения в многопоточных приложениях, где достигает максимальной производительности; Проработаны практически все функции Asterisk Manager, включая: управление вызовами, управление очередями, телеконференции(meetme, confbridge), функции администрирования и управление конфгурациями, управление DAHDI-каналами; Круг применения: от простых скриптов, изменяющих пароли пользователей, до построения комплексных SCADA-систем, позволяющих производить с одного терминала мониторинг и управление VoIP-структурой предприятия, состоящей из 1000+ серверов; Лицензия BSD.
