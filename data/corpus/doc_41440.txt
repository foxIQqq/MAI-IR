Собраны средства для разработки GNOME Builder

Кристиан Хергерт (Christian Hergert) сообщил об успешном ходе кампании по сбору средств для разработки новой интегрированной среды разработки GNOME Builder, оптимизированной для создания приложений для GNOME. В настоящее время собрано 35 тыс. долларов из запланированных на первом этапе 30 тысяч, необходимых для создания базовой функциональности. Код проекта распространяется под лицензией GPLv3. Первый выпуск намечен на апрель. Редактор кода построен с использованием GtkSourceView и поддерживает подсветку синтаксиса, автодополнение кода, перекрёстные ссылки, отслеживание изменений, автоматическое форматирование и одновременное редактирование частей кода в разных окнах. Визуальный построитель интерфейсов базируется на Glade и недавно интегрированном в GTK+ отладчике GtkInspector. Для интеграции с Git используется Gitg. В состав также входит система управления проектами на основе autotools, менеджер ресурсов, симулятор на основе OSTree, отладчик и система профилирования. Планируется обеспечить возможность разработки на языках C, Vala, JavaScript и Python. Из планов также отмечаются средства совместной разработки, поддержка отладки GNOME-приложений на внешних мобильных устройствах, поддержка сервисов D-Bus, средства для unit-тестирования, система для поиска примеров использования GNOME API в git.gnome.org, инструменты для создания расширений к GNOME Shell, интеграция с Bugzilla, создание симулятора GNOME.
