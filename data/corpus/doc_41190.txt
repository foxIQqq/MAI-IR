Доступен GNU Ghostscript 9.14.0

Проект GNU представил новый выпуск GNU Ghostscript 9.14.0, дистрибутива Ghostscript, развиваемого компанией Artifex Software набора инструментов для обработки, преобразования и генерации документов в форматах PostScript и PDF. Новый выпуск примечателен переходом с GPLv3 на лицензию AGPLv3, отражая изменение лицензии Ghostscript от Artifex Software. По сравнению с прошлым выпуском GNU Ghostscript 9.06, в новую версию перенесена большая порция новых возможностей и исправлений. Особенностью лицензии AGPLv3 является введение дополнительных ограничений для приложений, обеспечивающих функционирование сетевых сервисов. При использовании AGPL-компонентов для обеспечения работы сервиса, разработчик обязан предоставить пользователю исходный код модифицированных AGPL-компонентов, что не позволяет использовать новый выпуск GNU Ghostscript для создания закрытых web-сервисов.
