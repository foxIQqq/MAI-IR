Релиз тестового комплекта Phoronix Test Suite 2.2

Вышел релиз пакета для измерения производительности системы - Phoronix Test Suite 2.2. Пакет поддерживает работу в Linux, OpenSolaris, системах семейства BSD и Mac OS, распространяется под лицензией GPLv3 и призван обеспечить потребность простых пользователей в удобном инструменте для измерения производительности системы. Phoronix Test Suite 2.2 поддерживает 120 тестовых профилей и 50 различных тестовых наборов. Спецификация проведения каждого теста оформлена в виде XML файла, в качестве тестов используются внешние приложения, например, измеряется FPS в различных играх, оценивается время сборки Linux ядра, скорость кодирования mencoder, время сжатия bzip2 и т.д. Во время проведения тестов осуществляется контроль за изменением состояния оборудования (скорость вращения кулера, температура, напряжение). Итоги тестирования можно опубликовать в каталоге PTS Global, представляющем собой централизованную базу данных с результатами, полученными от различных пользователей. В версии 2.2 можно отметить следующие новшества: Добавлена поддержка автоматического отслеживания влияющих на производительность регрессивных изменений в новых версиях Linux ядра; Добавлена команда для извлечения результатов изменения производительности в случае краха ОС во время тестирования, при этом пользователь имеет возможность продолжить выполнение пакета пропустив проблемный тест; Появился режим статистической проверки корректности результатов - если при повторных проведениях одного и того же теста зафиксированы аномальные расхождения величин, то число попыток запуска теста будет автоматически увеличено; Поддержка проведения графических тестов для нескольких разных экранных разрешений; Проведение работы по улучшению графического интерфейса пользователя; Добавлена поддержка новых версий ОС: Mac OS X 10.6, FreeBSD 8.0, Ubuntu 9.10, Fedora 12, SuSE Linux 11.2 и тестовой сборки OpenSolaris 2010.02; В комплект включены новые тестовые профили для изменения производительности оперативной памяти, дисков и для оценки вычислительной мощности, число тестов расширено новыми операциями по визуализации данных и задействованием возможностей OpenCL.
