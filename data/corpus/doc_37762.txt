Feedbin переведён в разряд открытых проектов

Разработчики агрегатора RSS-лент Feedbin, реализованного в виде платного web-сервиса, объявили об открытии исходных текстов проекта. Отныне каждый желающий может развернуть данную систему для чтения RSS на своём сервере. Feedbin доступен под лицензией MIT. Код написан на языке Ruby с использованием фреймворка Ruby on Rails, СУБД PostgreSQL и NoSQL БД Redis. Вместе с Feedbin также открыт код трёх сопутствующих проектов: refresher - сервис обновления RSS-лент; polyptych - API для извлечения картинок favicons; camo - прокси для кэширования изображений и их отдачи только через HTTPS. В качестве причин открытия кода упоминается желание привлечь заинтересованных энтузиастов к развитию проекта, обеспечение максимальной прозрачности сервиса и нежелание повторить судьбу Google Reader, потерянного для пользователей после закрытия коммерческого проекта.
