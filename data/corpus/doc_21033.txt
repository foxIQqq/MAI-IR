Firefox 3.0 отвоевал 35.05% европейского рынка web-браузеров, опередив конкурентов

По данным ресурса StatCounter, осуществляющего мониторинг использования web-браузеров, Firefox 3.0 по состоянию на 31 марта 2009 года отвоевал 35.05% европейского рынка web-браузеров, обогнав по популярности Microsoft Internet Explorer 7.0, занимающего 34.54% рынка (в связи с начавшейся миграцией пользователей на IE 8.0). По совокупной доле всех версий браузеров IE занимает 50.56% рынка, в то время как Firefox только 37.57%, Opera - 7.41%, Safari - 2.23% и Google Chrome - 1.39%. По альтернативной информации, подготовленной сервисом XiTi Monitor, доля Internet Explorer в Европе составляет 58.1%, Firefox - 32.5%, Google Chrome - 1.5%, Safari - 3%, Opera - 4.1%. При этом приводится интересная статистика по интенсивности обновлений версий, за 7 месяцев на Firefox 3 с Firefox 2 перешло 88%, пользователей, в то время как за два года на IE 7 c IE 6 перешло всего 66%. Что касается общемировых тенденций, то по данным мониторинга общемирового рынка web-браузеров, проводимого в прошлом месяце компанией Net Applications, IE занимает 67.44% рынка, Firefox - 21.77%, Safari - 8.02%, Chrome - 1.15%, а Opera - 0.71%.
