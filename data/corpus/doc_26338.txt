В сервисе цифровой дистрибуции игр Steam ожидается поддержка Linux

По неофициальной информации в июне ожидается анонс реализации поддержки платформы Linux в игровом движке Source и сервисе цифровой дистрибуции игр Steam, через который организована продажа игр компаний Valve, Epic Games, id Software и других известных производителей. В подтверждение данной информации приводится ссылка на мета-файл для загрузки Linux-клиента на сайте Steam, упоминание платформы Linux в бинарных библиотеках Steam, а также появление связанного с Linux кода в поставке с тестовой версией для платформы Mac OS X.
