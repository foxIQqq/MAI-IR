Релиз свободного видеоредактора Avidemux 2.6.0

После трёх лет разработки представлена новая стабильная ветка видеоредактора Avidemux 2.6.0, рассчитанного на решение простых задач по нарезке видео, применении фильтров и кодировании. Поддерживается большое число форматов файлов и кодеков. Выполнение задач может быть автоматизировано при помощи очередей выполнения заданий, написания скриптов и создания проектов. Avidemux поставляется под лицензией GPL и поддерживает работу в Linux, BSD, Mac OS X и Windows. В новой версии полностью переписаны все внутренние компоненты, что привело к улучшению работы в большинстве ситуаций, но не обошлось и без ухудшений в ряде единичных случаев. Основные новшества: Поддержка синхронизации времени в процессе выполнения разных операций, без привязки к фиксированной частоте кадров. Например, можно редактировать файлы .ts не теряя синхронизации между звуком и видео; Поддержка OpenGL-фильтров; Поддержка дополнительных скриптовых языков: JavaScript (Spidermonkey), Python (TinyPy) и QtScript; Возможность задействования аппаратной акселерации, пока поддерживается только VDPAU; Поддержка работы одновременно с несколькими звуковыми дорожками; Все компоненты редактора (muxer/demuxer/filter/...) теперь выступают в роли заменяемых плагинов.
