Утверждено кодовое имя Ubuntu 16.04 - Xenial Xerus

За несколько часов до релиза Ubuntu 15.10 Марк Шаттлворт объявил кодовое имя следующего выпуска. Ubuntu 16.04, который будет отнесён к выпускам с длительным сроком поддержки, получил имя "Xenial Xerus" (гостеприимная земляная белка). Марк отметил, что обитающая в Африке земляная белка является животным, объединяющимся в небольшие социальные группы, которые отлично ладят с особями, обитающими по соседству. Данная особенность как нельзя лучше подходит к релизу Ubuntu 16.04, в качестве ключевых целей для которого названо развитие средств виртуализации и контейнерной изоляции. В данном контексте, прилагательное Xenial подразумевает дружественное взаимодействие между хост-окружениями и гостевыми системами. В Ubuntu 16.04 ожидается более активное продвижение развиваемого Canonical инструментария управления изолированными контейнерами LXD, улучшение поддержки гипервизора KVM, усиление совместимости адаптированного для Ubuntu облачного стека OpenStack c различными системами виртуализации, развитие монолитно обновляемых сборок на базе технологии Snappy, усовершенствование инструментария MAAS (Metal-as-a-Service) для автоматизации развертывания Ubuntu-конфигураций на множестве серверов, продолжение разработки Juju для быстрого развёртывания преднастроенных сервисов. Ожидается, что Ubuntu 16.04 станет последним LTS-релизом по умолчанию поставляемым с традиционным графическим стеком X.Org. В дальнейшем планируется переход на Unity 8 и дисплейный сервер Mir.
