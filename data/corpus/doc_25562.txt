Вышла третья бета-версия Flash Player 10.1 для Linux

Компания Adobe анонсировала третью бета-версию Flash Player 10.1 для платформ Linux, Windows и Mac OS X, в будущем ожидается выпуск сборки для мобильных систем Windows Mobile, Symbian S60, Google Android, MeeGo (Moblin+Maemo) и Palm webOS. Релиз Flash Player 10.1 планируется выпустить в первой половине 2010 года, это будет первый выпуск базирующийся на инициативе Open Screen Project. Основные новшества Flash Player 10.1: Задействование механизмов аппаратной акселерации GPU при выводе данных на экран и при декодировании h.264 видео (акселерация пока работает только в Windows). Для мобильной сборки ожидается появление поддержки мультитач навигации, виртуальной клавиатуры, управления экранными жестами и API для доступа к акселерометру. В десктоп-версии появится режим приватного серфинга без оставления следов в логах, кэше и на локальном диске.
