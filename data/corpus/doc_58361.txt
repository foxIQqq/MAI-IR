Релиз дистрибутива Linux Mint 21.1

Представлен релиз дистрибутива Linux Mint 21.1, продолжающий развитие ветки на основе пакетной базы Ubuntu 22.04 LTS. Дистрибутив полностью совместим с Ubuntu, но существенно отличается подходом к организации интерфейса пользователя и подбором используемых по умолчанию приложений. Разработчики Linux Mint предоставляют десктоп-окружение, соответствующее классическим канонам организации рабочего стола, которое является более привычным для пользователей, не принимающих новые методы построения интерфейса GNOME 3. Для загрузки доступны DVD-сборки на базе оболочек MATE 1.26 (2.1 ГБ), Cinnamon 5.6 (2.1 ГБ) и Xfce 4.16 (2 ГБ). Linux Mint 21 отнесён к выпускам с длительным сроком поддержи (LTS), обновления для которого будут формироваться до 2027 года. Основные изменения в Linux Mint 21.1 (MATE, Cinnamon, Xfce): В состав включён новый выпуск десктоп-окружения Cinnamon 5.6, оформление и организация работы в котором продолжает развитие идей GNOME 2 - пользователю предлагается рабочий стол и панель с меню, областью быстрого запуска, перечнем открытых окон и системным лотком с запущенными апплетами. Cinnamon основан на GTK и технологиях GNOME 3. Проект развивает оболочку GNOME Shell и оконный менеджер Mutter в направлении предоставления окружения в стиле GNOME 2 с более современным оформлением и задействованием элементов из GNOME Shell, дополняющих средства классического десктопа. Редакции с рабочими столами Xfce и MATE поставляются с Xfce 4.16 и MATE 1.26. Основные изменения в Cinnamon 5.6: Добавлен апплет Corner bar, который размещён в правой части панели и заменил собой апплет show-desktop, вместо которого теперь размещён разделитель между кнопкой вызова меню и списком задач. Новый апплет позволяет привязать свои действия к нажатию разных кнопок мыши, например, можно отобразить содержимое рабочего стола без окон, показать десклеты или вызвать интерфейсы переключения между окнами и виртуальными рабочими столами. Размещение в углу экрана позволяет проще позиционировать указатель мыши на апплете. Апплет также даёт возможность быстро помещать файлы на рабочий стол, независимо от того как много окон открыто, для чего достаточно перетащить необходимые файлы в область апплета. В файловом менеджере Nemo в режиме просмотра списка файлов с отображением пиктограмм для выделенных файлов теперь подсвечивается только имя, а пиктограмма остаётся как есть. Пиктограммы, представляющие рабочий стол, теперь повёрнуты вертикально. В файловом менеджере Nemo улучшена реализация строки с файловым путём. Клик на текущем пути теперь приводит к переключению панели в режим ввода местоположения, а дальнейшая навигация по каталогам возвращает исходную панель. Для отображения дат задействован моноширинный шрифт. В контекстное меню, показываемое при клике правой кнопкой мыши на рабочем столе, добавлен пункт для перехода к настройкам экрана. В настройки клавиатурных комбинаций добавлено поле для поиска. Избранные приложения разделены на категории. Предоставлена возможность настройки продолжительности показа уведомлений. В апплет inhibit добавлены клавиатурные комбинации для переключения уведомлений и управления питанием. Списки тем оформления отсортированы для разделения тёмных, светлых и устаревших тем. Возвращён режим размещения окна (Window placement), который был убран в ходе переработки mutter в Cinnamon 5.4. На рабочем столе по умолчанию скрыты пиктограммы "Home", "Computer", "Trash" и "Network" (через настройки их можно вернуть). Пиктограмму "Home" заменила кнопка в панели и секция с избранным в основном меню, а пиктограммы "Computer", "Trash" и "Network" используются редко и быстро доступны через файловый менеджер. Примонтированные накопители, значок установки и файлы, размещённые в каталоге ~/Desktop, как и раньше показываются на рабочем столе. Добавлены дополнительные варианты акцентирующих цветов, применяемых для выделения активных элементов (accent). Прекращено применение акцентирующих цветов в панели и меню. Цвет пиктограмм каталогов заменён на жёлтый. По умолчанию вместо зелёного для выделения задействован голубой цвет. Для возвращения старого оформления (как в Linux Mint 20.2) предложена отдельная тема "Mint-Y-Legacy". В настройках предоставлена возможность выбора произвольных цветов для оформления. Предложено новое оформление указателя мыши и добавлен набор альтернативных указателей. Изменён набор звуковых эффектов по умолчанию. Новые эффекты заимствованы из набора Material Design V2. Добавлены альтернативные темы пиктограмм. Помимо тем Mint-X, Mint-Y и Mint Legacy, также доступны темы Breeze, Papirus, Numix и Yaru. Проведена модернизация менеджера устройств, который теперь запускается под непривилегированным пользователем и не требует ввод пароля. Изменено оформления экрана, показываемого при работе в offline-режиме. Также изменён экран, отображаемый при обнаружении USB- или DVD-накопителя с драйверами. Упрощена установка драйверов для беспроводных адаптеров Broadcom. Обеспечена корректная поддержка Debconf, необходимая при установке драйверов NVIDIA при активном режиме SecureBoot. В Packagekit внесены изменения для удаления пакетов вместе с файлами конфигурации, которые используются в менеджере устройств при удалении драйверов, что решило проблемы с драйверами NVIDIA при переходе с одной ветки на другую. В менеджер обновлений добавлена поддержка пакетов в формате Flatpak и связанных с ними runtime-наборов, которые теперь могут быть обновлены по аналогии с обычными пакетами. В интерфейс менеджера приложений внесены изменения для наглядного разделения Flatpak и системных пакетов. Обеспечено автоматическое добавление новых пакетов из каталога Flathub. Предоставлена возможность выбора версии при наличии искомого приложения в штатном репозитории и в формате Flatpak. Добавлен инструмент для проверки целостности ISO-образов, который можно вызвать через контекстное меню. Для Linux Mint и Ubuntu GPG-файлы и контрольные суммы SHA256 для проверки определяются автоматически, а для других дистрибутивов требуется ручной ввод ссылок или файловых путей. В утилиту для записи ISO-образов добавлена кнопка для инициирования проверки целостности, которая теперь работает и для образов с Windows. Улучшен интерфейс утилит для форматирования USB-накопителей. Продолжено усовершенствование приложений, развиваемых в рамках инициативы X-Apps, нацеленной на унификацию программного окружения в редакциях Linux Mint на базе разных рабочих столов. В приложениях X-Apps применены современные технологии (GTK3 для поддержки HiDPI, gsettings и т.п.), но сохранены традиционные элементы интерфейса, такие как панель инструментов и меню. Среди таких приложений: текстовый редактор Xed, менеджер фотографий Pix, просмотрщик документов Xreader, просмотрщик изображений Xviewer. Предоставлена возможность настройки оформления и размера курсора для экрана входа в систему. Усилена защита Warpinator, утилиты для шифрованного обмена файлами между двумя компьютерами, которая автоматически завершает выполнение после 60 минут неактивности и ограничивает доступ к некоторым настройкам. Расширены возможности менеджера web-приложений (WebApp Manage), в котором появились дополнительные настройки для web-приложений, такие как отображение навигационной панели, изоляция профиля и запуск в режиме приватного просмотра. Переработан код для удаления приложений из главного меню - если для удаления достаточно прав текущего пользователя то пароль администратора теперь не запрашивается. Например, без ввода пароля можно удалить программы в формате Flatpak или ярлыки на локальные приложения. Synaptic и менеджер обновлений переведены на использование pkexec для запоминания введённого пароля, что при выполнении нескольких операций позволяет запрашивать пароль только один раз. В приложении для настройки источников установки пакетов переработан способ обработки ключей для PPA-репозиториев, которые теперь применяются только для конкретного PPA, а не для всех источников пакетов. Тестирование всех проектов Linux Mint переведено с системы непрерывной интеграции Circle на GitHub Actions.
