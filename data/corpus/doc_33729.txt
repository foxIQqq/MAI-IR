Релиз десктоп-окружения Xfce 4.10

После 16 месяцев разработки увидел свет релиз десктоп-окружения Xfce 4.10, нацеленного на предоставление классического рабочего стола при минимальном потреблении системных ресурсов. Xfce состоит из ряда взаимосвязанных компонентов, которые при желании можно использовать в других проектах. Среди таких компонентов: оконный менеджер, панель для запуска приложений, дисплейный менеджер, менеджер управления пользовательскими сессиями и контроля за энергопотреблением, файловый менеджер Thunar, web-браузер Midori и система настройки параметров окружения. В процессе подготовки новой версии основное внимание было уделено увеличению удобства работы пользователя и оттачиванию возможностей рабочего стола. Среди улучшений, добавленных в новой версии: Новый, полностью переписанный, интерфейс для поиска приложений (xfce4-appfinder), совместивший в себе возможности ранее используемых программ xfce4-appfinder и xfrun4. Кроме улучшений, связанных с переработкой интерфейса пользователя, в новом xfce4-appfinder также обеспечена возможность создания собственных действий, привязанных к произвольному префиксу или маске; Плагины (ланчеры) теперь могут располагаться на панели в несколько строк, что особенно полезно при вертикальном размещении панели. Реализован альтернативный вертикальный метод размещения панели, при котором панель отображается сбоку в виде колонки (режим deskbar) с указанным числом столбцов; Добавлен редактор MIME-типов и осуществлён уход от жестко определенного списка предпочитаемых приложений. Отныне пользователь легко может изменить приложение, используемое для обработки тех или иных типов файлов. Расширены возможности интерфейса для настройки параметров десктоп-окружения. В рамках единого интерфейса проведена группировка диалогов настройки, которые теперь разделены на категории с возможностью поиска настроек по имени и описанию. Большинство диалогов настройки теперь встроено в окно менеджера настроек (в Xfce 4.8 такое поведение можно было включить на этапе сборки). Проведена оптимизация отображения диалогов настройки на нетбуках с небольшим вертикальным разрешением, а также на планшетах, управляемых через сенсорные экраны. Улучшена работа конфигуратора подключения нескольких мониторов; В конфигуратор параметров мыши (xfce4-mouse-settings) добавлена поддержка настройки параметров, специфичных для тачпадов Synaptics и планшетов Wacom; Редактор настроек более не сворачивает элементы дерева параметров при редактировании определённого свойства. Большинство свойств теперь доступно для прямого редактирования без открытия лишних окон. Используя редактор настроек также теперь можно отслеживать изменение настроек для выбранного раздела; Cтал возможным запуск приложений и открытие файлов с рабочего стола одним кликом мыши. Для элементов, размещённых на рабочем столе, также могут отображаться эскизы. Добавлена поддержка циклической смены фоновых изображений. В будущем выпуске для формирования привычного десктопа вместо рабочего стола xfdesktop планируется задействовать специальный плагин к Thunar, что позволит существенно расширить возможности рабочего стола; Оконный менеджер xfwm4 может быть настроен для тайлинга окон (плотной компоновки бок о бок) при их перетаскивании к краям экрана (по умолчанию опция отключена, но тайлинг по горячей клавише активен); Обеспечена возможность изменения оформления интерфейса переключения между окнами, вызываемого по Alt-Tab, через смену визуальной темы. Также улучшена навигация между элементами при использовании клавиатуры; Создание плагина-контейнера, который мог бы включать в себя несколько плагинов; Объединение в один плагин плагинов связанных с сессиями (actions, xfce4-session, xfsm-logout и xfswitch); Расширение возможностей файлового менеджера Thunar: Пересмотр подхода к созданию и отображению пользовательских директорий в меню Go; Интеграция поддержки работы с удалёнными ресурсами в боковую панель и древовидную панель, что позволит создавать "закладки" на внешние хранилища; Улучшение интеграции с Tumbler для использования единого кэша миниатюр; Поддержка режима сортировки по расширению файла; Увеличение отзывчивости выполнения операций с файлами (например, копируемые файлы должны появляться в панели более оперативно); Переработана система задания горячих клавиш, добавлена поддержка .Xmodmap; Упрощена установка визуальных тем. Для установки темы достаточно перетащить мышью архив с темой в список стилей или пиктограмм, Xfce при этом автоматически распакует и установит тему в директорию ~/.themes или ~/.icons; Менеджер сессий теперь автоматически блокирует экран при переходе в спящий режим. Добавлена кнопка для быстрой очистки сохранённой сессии (ранее требовалось вручную удалять содержимое ~/.config/sessions); Улучшены средства обеспечения совместимости с GNOME и KDE. Установки совместимости теперь активируют только сервисы, запуск которых производится до других приложений (gnome-keyring и gconf для GNOME и kdeinit для KDE). Активация остальных автоматически запускаемых приложений доступна через диалог "Applications Autostart", но такие приложение выделены курсивом и отключены по умолчанию, чтобы не путать их с программами Xfce. В отличие от прошлых выпусков, необходимые для обеспечения совместимости сервисы могут запускаться независимо друг от друга; Вместо поставляемого в комплекте набора документации вся справочная информация теперь перемещена на wiki-сайт проекта docs.xfce.org, что значительно упростит редактирование документации, её дополнение и сопровождение; В качестве базовых зависимостей пока присутствуют GTK+ 2.20 и GLib 2.24, но для работы движка визуальных тем уже требуется минимум GTK+ 3.2. Полный переход на GTK3+ намечен в следующей версии.
