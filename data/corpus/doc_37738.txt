Релиз распределенной системы управления исходными текстами Git 1.8.4

Увидел свет релиз распределенной системы управления исходными текстами Git 1.8.4. Git является одной из самых эффективных, надёжных и высокопроизводительных систем управления версиями, предоставляющей гибкие средства нелинейной разработки, базирующиеся на ответвлении и слиянии веток. Для обеспечения целостности истории и устойчивости к изменениям задним числом используются криптографические методы, также возможна привязка цифровых подписей разработчиков к тегам и коммитам. Из проектов, разрабатываемых с использованием Git, можно отметить ядро Linux, Android, Libreoffice, Systemd, X.Org, Wayland, Mesa, Gstreamer, Wine, Debian, DragonFly BSD, Perl, Eclipse, GNOME, KDE, Qt, Ruby on Rails, PostgreSQL, VideoLAN, PHP, Xen, Minix. В новой версии внесено более 870 изменений от 100 разработчиков. Среди улучшений: обновление порта Cygwin для поддержки Cygwin 1.7, обновление Git-gui до версии 0.18.0, реализация интерактивного режима для команды "git clean", новая команда "git check-mailmap" для сопоставления имён пользователей с email, в "git diff добавлена поддержка игнорирования добавленных или удалённых пустых строк, многочисленные оптимизации производительности.
