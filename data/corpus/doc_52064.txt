Обновление Oracle Solaris 11.4 SRU16

Опубликовано обновление операционной системы Solaris 11.4 SRU 16 (Support Repository Update), в котором предложена серия очередных исправлений и улучшений для ветки Solaris 11.4. Для установки предложенных в обновлении исправлений достаточно выполнить команду 'pkg update'. В новом выпуске: До версии 3.6.2 обновлён Oracle VM Server для SPARC. В команду add-vsan-dev добавлена поддержка миграции доменов с гостевыми системами. Реализована опция calculate-effective-max-pagesize для управления выставляемыми для домена лимитами на размер страниц; Обновлены Python-модули cherrypy 15.0.0 и six 1.11.0. Платформа Node.js обновлена до версии 8.16.2; Обновлены версии программ с устранением уязвимостей: ImageMagick 6.9.10-68, expat 2.2.9, OpenSSL 1.0.2t, webkitgtk4 2.24.4, python 2.7 2.7.17, rdesktop 1.8.6, D-Bus 1.10.28, GnuPG 2.2.17, sqlalchemy 1.2.19, PHP 7.1.33 и 7.3.11, GNU Tar 1.32; Устранены уязвимости в libgd, libsndfile, sdl, xclients, aspell, libexif, libidn2, ncurses, libtiff, net-snmp, file-roller, libxslt, xml-libxml, libtasn1, proftpd.
