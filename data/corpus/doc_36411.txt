Выпуск Wine 1.5.26

Вышел очередной экспериментальный выпуск открытой реализации Win32 API - Wine 1.5.26. С момента выпуска версии 1.5.25 было закрыто 49 отчётов об ошибках. Из наиболее важных изменений отмечается: Прекращена поддержка платформы SPARC; В драйвер для работы с Mac OS X добавлена поддержка OpenGL, буфера обмена и drag&drop Улучшена работа кэша URL-адресов; Исправлены некоторые ошибки в поддержке бинарных файлов архитектуры ARM; Устранены проблемы при запуске игр: NFS3, Gordian Knot, Entropia Universe, Wing Commander 3 Kilrathi, Priority: Survive, JumpGate: The Reconstruction Initiative, Starcraft 2, Might and Magic Heroes, Railworks 3 2013 Trainsimulator, Bunch of Heroes, Magic: The Gathering Online Wide Beta, Drakensang, Disciples: Sacred Lands, Total Annihilation : Kindoms; Налажена работа приложений: Windows Media Player 9, msvideo, 3DMark 2000/2001SE, Propellerheads reason 4, Houdini 12.x, Delphi, Enterprise Architect 10, AudioSurf; Обновлены переводы; Исправлено множество ошибок.
