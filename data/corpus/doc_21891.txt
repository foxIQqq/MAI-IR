В Ubuntu может появиться поддержка выполнения приложений для платформы Android

На саммите разработчиков Ubuntu, проходящем сегодня в Барселоне, были анонсированы некоторые планы, касающиеся добавления поддержки приложений, разработанных для мобильной платформы Google Android. По предварительным данным возможность выполнения Android-приложений может быть интегрирована уже в следующий релиз Ubuntu 9.10. Приложения будут выполняться в специальной оболочке Android Execution Environment, в которой каждое приложение запускается в отдельном chroot-окружении. Вместо использования оригинальной, не совместимой с X11, графической подсистемы Android, вывод данных планируется организовать через специальных транслятор, отображающий интерфейс программ на обычном X-сервере. В системе также будет добавлена поддержка установки программ в APK (Android Package Files) формате. Для Ubuntu, интеграция поддержки Android интересна в первую очередь в связи с волной заявлений о планах по выпуску на базе Android не только ультрамобильных устройств, но и субноутбуков, что неизбежно приведет к росту числа специализированных приложений. Если данные Android-приложения будут выполняться в Ubuntu Netbook Remix, это существенно повысит конкурентоспособность данной платформы.
