Выпуск открытого медиацентра Kodi 16.0

Увидел свет открытый медиацентр Kodi 16.0, ранее развивавшийся под именем XBMC. Готовые установочные пакеты доступны для OS X (x86, PPC), FreeBSD, Linux (ARM, PPC, x86 и x86-64), Raspberry Pi, Android (ARM, x86), Windows, Apple TV и iOS. Код проекта распространяется под лицензией GPLv2+. Изначально, проект был нацелен на создание открытого мультимедиа плеера для игровой приставки XBOX, но в процессе развития трансформировался в кросс-платформенный медиацентр, работающий на современных программных платформах. Из интересных функций Kodi, можно отметить поддержку широкого спектра форматов мультимедиа и средств аппаратного ускорения декодирования видео; поддержка пультов дистанционного управления; возможность проигрывания файлов по FTP/SFTP, SSH и WebDAV; возможность удалённого управления через web-интерфейс; наличие гибкой системы плагинов, реализованных на языке Python и доступных для установки через специальный каталог дополнений; подготовка плагинов для интеграции с популярными online-сервисами; возможность загрузки метаданных (текстов песен, обложек, рейтинга и т.п.) для имеющегося контента. На базе Kodi развивается около десятка коммерческих телеприставок и несколько открытых ответвлений (Boxee, GeeXboX, 9x9 Player, MediaPortal, Plex). Основные новшества: Реализован журнал действий, в котором пользователю предложен непрерывно обновляемый список наиболее важных операций и возникающих в процессе работы проблем. Например, после сканирования библиотеки контента в журнале будут показаны найденные повреждённые видео, которые не удалось распознать. В списке также отражены факты установки обновлений для дополнений. Добавлена поддержка вызова контекстного меню через длительное удержание клавиши OK/Enter на пульте дистанционного управления. Данная возможность позволяет расширить число доступных управляющих комбинаций на пультах, предоставляющих слишком ограниченный набор кнопок, на которых не предусмотрено отдельной кнопки для меню. Поддержка нелинейного растягивания размера картинки в формате 4:3 на экране с соотношением сторон 16:9 без искажения содержимого в центре экрана. В менеджере дополнений изменён подход к обработке отдельных дополнений: кнопка отката обновления (Roll-back ) в панели с информацией о дополнении заменена на кнопку установки обновления (Update), при нажатии на которую теперь предлагается список доступных для применения версий дополнения. Добавлена возможность отключения системы автоматической установки для избранных дополнений. Панель с информации о дополнении теперь выводится не только в менеджере дополнений, но и доступна для вызова через контекстное меню или горячую клавишу ("I"); Уделено большое внимание развитию интерфейса для работы с музыкальной коллекцией, который ранее заметно отставал по своим возможностям от интерфейса библиотеки видео. Если раньше для добавления музыки требовалось ручное указание директорий с новой музыкой, то теперь Kodi выполняет операцию автоматического сканирования музыкальных файлов для их включения в музыкальную библиотеку, а также предоставляет интерфейс для привязки тегов к новым файлам. В музыкальной библиотеке также добавлена возможность сортировки по дате; Версия для платформы Android при работе на устройствах с очень большим экранным разрешением (UHD или 4K) теперь отрисовывает интерфейс в отдельной поверхности с более низким разрешением (обычно 720p), не зависящей от поверхности для вывода видео, что позволило решить проблемы с недостаточной мощностью CPU для отрисовки интерфейса на экране 4K. Вывод видео при этом возможен в любых поддерживаемых Android разрешениях, в том числе 4K; На платформе Windows обеспечена поддержка DirectX 11 с сохранением возможности использовать DirectX 9 для старых видеокарт; Началась интеграция обработки ADSP (Audio Digital Signal Processing); Алгоритм масштабирования и кэширования изображений улучшен в плане увеличения качества изображений; Добавлена поддержка тега "ARTISTS" в форматах id3v2, ape, xiph; Для платформы Linux реализована поддержка мультитач; Удалена находящаяся в нерабочем состоянии подсистема караоке; Добавлена поддержка платформы iOS9 и возможность сборки компилятором Xcode 7; Для 3D-телевизоров добавлена возможность отображения интерфейса пользователя со стереоскопическим эффектом. Кроме того, сообщается об инициативе по преследованию нарушителей торговой марки Kodi, распространяющих Flash-накопители с предустановленной версией Kodi, поставляемой с дополнениями для просмотра пиратского контента. Подобные накопители позиционируются как средство для бесплатного доступа к платному и лицензионному контенту (например, для доступа к каналам IPTV), чем наносят ущерб репутации проекту Kodi, так как распространяются от его имени и зачастую включают нестабильные и неподдерживаемые пиратские дополнения, которые негативно влияют на стабильность работы медиацентра. В итоге, на форумах появляются всё новые и новые недовольные пользователи, которые уверены, что купленные ими сборки распространяются проектом Kodi, а компании не желают сотрудничать с проектом из-за негативного имиджа, ассоциируемого с потворством пиратству. Для борьбы с подобными злоупотреблениями проект намерен привлечь юристов для преследования лиц, занимающихся продажей сторонних сборок под именем Kodi.
