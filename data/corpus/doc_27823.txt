Релиз Univention Corporate Server 2.4 с интегрированным управлением виртуальными окружениями

Немецкая компания Univention, занимающаяся разработкой промышленных решений для IT-инфраструктуры на базе свободного ПО, включила в состав вышедшей на днях новой версии своего промышленного Linux-дистрибутива Univention Corporate Server (UCS) 2.4 новый инструмент для управления виртуальными окружениями — Univention Virtual Machine Manager (UVMM). Этот программный продукт предоставляет возможность легкого, интуитивно понятного управления виртуальными машинами на базе Xen и KVM. Взаимодействие с пользователем осуществляется через web-интерфейс, интегрированный в общую web-консоль управления сервером, предоставляемую UCS. Работа с хостами (гипервизорами) реализована через стандартный интерфейс управления виртуализацией — libvirt, что позволяет разработчикам при необходимости легко добавить поддержку других систем виртуализации, например, VirtualBox. UVMM распространяется под лицензией AGPLv3. Скриншоты (1, 2). Основные возможности UVMM: Единый интерфейс, объединяющий управление хост-системами на базе как Xen, так и KVM. Централизованное управление гостевыми окружениями: виртуальные машины можно запускать, останавливать, перемещать (migrate) и создавать в один-два щелчка мыши. Единый центр управления образами жестких дисков позволяет создавать образы и подключать их к заданным виртуальным машинам. Возможность on-line мониторинга потребления системных ресурсов (процессоров и памяти). Поддержка live migration — перемещения работающей виртуальной машины на другой хост без ее остановки. Web-интерфейс, не требующий установки специального клиентского ПО на компьютер, с которого осуществляется управление. Быстрое создание виртуальных машин на базе преднастроенных профилей. Возможность использования паравиртуализации для ускорения работы гостевых окружений. Univention Corporate Server, в состав которого входит UVMM, является промышленным дистрибутивом, предназначенным для развертывания и поддержки корпоративной IT-инфраструктуры. UCS основан на Debian GNU/Linux. В его состав входит множество компонентов, обеспечивающих решение таких задач, как Ведение единого каталога учетных записей пользователей (на базе LDAP). Предоставление единой аутентификации и авторизации с использованием возможностей PAM, LDAP, Kerberos и NTLM. Взаимодействие с MS Active Directory. Установка операционных систем на другие сервера или десктопы через PXE. Создание репозитариев для централизованной установки и обновления ПО. Мониторинг состояния сети, серверов и служб (на базе Nagios). Управление DHCP и DNS. Развертывание файл-серверов с поддержкой квот. Поддержка серверов печати. Проксирование интернет-соединений (на базе Squid). Резервное копирование (на базе Bacula). Пересылка почты с фильтрацией от вирусов и спама. Предоставление средств для обеспечения совместной работы сотрудников (groupware). Поддержка сервера терминалов. Обеспечение работы виртуальных окружений на базе Xen и KVM. В выпуске UCS 2.4, помимо добавления рассмотренного выше UVMM, обновлены версии ряда компонентов: ядро Linux (2.6.32), Xen (3.4.3), Samba (3.5.4), OpenLDAP (2.4.23) и других.
