Доступна редакция дистрибутива Linux Mint 15 с десктоп-окружением Xfce

Доступна Xfce-редакция дистрибутива Linux Mint 15. Дистрибутив основан на пакетной базе Xubuntu 13.04 и поставляется с десктоп-окружением Xfce 4.10, дополненном серией оригинальных приложений, упрощающих настройку и работу в системе. В частности, в Linux Mint используется переработанная система меню, собственный менеджер установки и обновления приложений, оригинальный интерфейс для настройки системы mintConfig, средство для выполнения резервного копирования mintBackup и т.п. Размер загрузочного iso-образа составляет 970 Мб. Дистрибутив может работать на системах с 384 Мб ОЗУ, но для комфортной работы рекомендуется 1 Гб оперативной памяти. Ключевые особенности Linux Mint 15 "Xfce": В качестве меню по умолчанию задействован Whisker, предоставляющий быстрый доступ к каталогу приложений, часто используемым и недавно запущенным программам, системным ярлыкам, свежим документам. Обновлён используемый в дистрибутиве дисплейный менеджер MDM 1.2, представляющий собой форк GDM 2.20. В новой версии MDM добавлена поддержка трех новых дополнений для вывода экрана входа в систему: простой GTK greeter, GDM greeter с поддержкой использования тем оформления, созданных для GDM, и HTML greeter, поддерживающий новые анимированные и интерактивные темы оформления, основанные на использовании web-технологий (HTML5, CSS, JavaScript и WebGL). В состав нового выпуска дистрибутива включена большая коллекция готовых тем оформления MDM, в том числе доступна тема "Clouds GL", созданная с использованием WebGL и симулирующая полёт в облаках. Кроме того внесена большая порция изменений под капотом и упрощено использование конфигуратора mdmsetup; В состав включено новое приложение Software Sources (MintSources), созданное специально для Linux Mint и предназначенное для настройки источников установки приложений, выступая заменой ранее используемого компонента software-properties-gtk. Из основного экрана Software Sources пользователь может активировать и выключать опциональные компоненты, включить доступ к репозиториям с бэкпортами и нестабильными версиями пакетов, получить пакеты с исходными текстами приложений, подключить внешние репозитории и PPA, управлять ключами аутентификации; Программа также предоставляет наглядный интерфейс для выбора наиболее быстрого зеркала, отображая для каждого из зеркал результаты выполнения теста для оценки реальной скорости загрузки на текущую машину. В рамках приложения Driver Manager (MintDrivers) подготовлен новый интерфейс для установки имеющихся в наличии проприетарных и альтернативных драйверов для популярных устройств (NVIDIA, AMD, Broadcom, Samsung и т.п.); Расширены возможности менеджера установки приложений: добавлена поддержка автоматической загрузки очередной порции элементов при приближении просмотра к концу страницы со списком; возможность поиска только в отдельных категориях приложений; пакеты из секции Java теперь видны и в категории Программирование; Менеджер установки обновлений теперь не требует прав суперпользователя для обновления кэша; Проведена оптимизация окна с приветствием входа в систему; Внесены другие изменения, свойственные для пакетной базы Ubuntu 13.04, например, осуществлён переход на ядро Linux 3.8.
