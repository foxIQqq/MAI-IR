Вышла первая бета веб-браузера Opera 11

Объявлено о выходе веб-браузера Opera 11 beta 1, который содержит множество нововведений и улучшений по сравнению с 10 версией, а также значительно ускоренный JavaScript движок. Группировка закладок. В Opera 11 появилась возможность группировки закладок (tabs) путём переноса одной закладки на другую. Для того, чтобы увидеть как это работает, смотрите видео. Безопасная адресная строка. Opera 11 последовала примеру Google Chrome и отказалась от привычного отображения протокола доступа к веб сайтам - теперь вместо "http:/"/ и "https://" Opera показывает выразительные картинки. Появилась поддержка расширений. Для написания расширений не нужно изучать достаточно сложный фреймворк XUL, вместо этого разработчики Opera, по примеру Goolge Chrome, реализовали расширения с помощью стандартных web-технологий: HTML5, CSS и JavaScript. Расширения можно загрузить и установить из специально подготовленного каталога. К сожалению, расширения для Opera 11 построены на базе технологии ранее доступных пользовательских JavaScript-виджетов, которые имеют гораздо меньше возможностей, чем, например, уже упомянутые расширения на основе XUL в Mozilla Firefox. Виртуальные жесты мышью. В Opera 11 значительно упрощена работа с жестами, благодаря добавлению удобного и понятного интерфейса. Улучшена производительность. Разработчики Opera в очередной раз представили новую версию собственного JavaScript-движка со значительно возросшей производительностью. Улучшена поддержка новейших веб технологий.В Opera 11 добавлена поддержка серверных событий и веб-сокетов стандарта HTML 5. В тесте html5test.com новый веб-браузер набирает 179 баллов, уступая по этому параметру Google Chrome для Linux, который набирает 231 балл. Улучшен механизм автообновления. Теперь веб-браузер более дотошно следит, чтобы ваши расширения и приложения Opera Unite обновлялись своевременно. Поисковые подсказки от Google. К сожалению, Opera оказалась чуть ли не последним веб-браузером, в котором была реализована возможность отображения на ходу наиболее популярных запросов, в зависимости от того, что вы ищете. Загрузка плагинов по требованию. В Opera 11 появилась возможность загрузки плагинов, таких как Adobe Flash, по требованию. Чтобы использовать эту возможность, её нужно включить в опциях веб браузера в Preferences / Advanced / "Enable plug-ins only on demand"(загружать плагины по требованию). Улучшена работа встроенного почтового клиента. Расширены дополнительные возможности: новая почтовая панель позволяет контролировать порядок отображения учётных записей и писем. Появилась возможность создавать папки для организации почты и новостных списков подписки (RSS). Переработана панель для избранного.
