Релиз Linux-дистрибутива Quirky 8.0, развиваемого автором Puppy Linux

Барри Каулер (Barry Kauler), основатель проекта Puppy Linux, представил релиз Linux-дистрибутива Quirky 8.0. Дистрибутив Quirky развивается параллельно с Puppy Linux и создан для обкатки новых идей и создания окружения для повседневного личного использования. Предлагаемое дистрибутивом графическое окружение базируется на оконном менеджере JWM, файловом менеджере ROX-Filer, собственном наборе GUI-конфигураторов и виджетов. Размер системного образа 347 Мб. Из особенностей дистрибутива можно отметить применение оптимизированной для Flash-накопителей файловой системы F2FS, урезанный графический стек на базе X.Org (без Mesa, вывод через драйвер VESA), отказ от использования начального ram-диска (initrd) и DBus, применение eudev вместо udev. В состав входят такие приложения, как SeaMonkey, Abiword, Gnumeric, Planner, Xine, Osmo, NoteCase, Grisbi, mtPaint, InkscapeLite и Xsane. Новый выпуск примечателен использованием для сборки DEB-пакетов из Ubuntu 16.04 (в прошлом выпуске применялась система сборки T2), что позволило реализовать поддержку установки дополнительных пакетов из штатных репозиториев Ubuntu. Ядро Linux обновлено до версии 4.4.7, в качестве браузера применяется SeaMonkey 2.40. В новом выпуске добавлен BluePup - графический конфигуратор Bluetooth, добавлена поддержка загрузки на системах UEFI, представлен YASSM GUI для управления Samba, в состав включена утилита YouTubeDL для загрузки видео с YouTube.
