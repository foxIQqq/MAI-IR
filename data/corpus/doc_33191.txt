GNOME 3.4 перешел на стадию бета-тестирования

Представлена первая бета-версия GNOME 3.4, которая ознаменовала переход к полной заморозке добавления новых возможностей, связанных с интерфейсом пользователя, и переключение разработки на стадию отладки и исправления ошибок. Релиз GNOME 3.4 запланирован на 28 марта. Вторая бета-версия выйдет 7 марта, кандидат в релизы - 21 марта. Из ожидаемых в GNOME 3.4 новшеств можно отметить: Поддержка глобального меню с управлением через D-Bus (GApplication). Реструктуризация меню некоторых приложений и обеспечение возможности переноса меню текущего активного приложения в верхнюю панель GNOME. Новый подход к размещению меню подразумевает выделение большего места для контента приложения за счёт ухода от использования классической строки меню. Меню может формироваться динамически, в зависимости от текущего состояния программы (активных режимов работы) и типа контента. В настоящее время перенос меню в верхнюю панель выполнен для единичных приложений, таких как Boxes, Contacts, Documents, Iagno, Mines, Tetravex, Web (Epiphany). В скором времени ожидается реализация нового меню для Totem, Empathy, Inkscape, Gimp и Libreoffice; Переработанный вариант web-браузера Epiphany в который интегрированы наработки по реализации нового пользовательского интерфейса, нацеленного на предоставление максимального экранного пространства для контента. Из особенностей нового интерфейса можно отметить: переработанную адресную строку; вынос в глобальное меню не связанных с текущим контентом операций; реализация выпадающего меню в адресной строке; Списки быстрого вызова функций в Gnome-shell, позволяющие при выборе приложения для запуска сразу инициировать выполнение определённых действий с данным приложением. Например, можно не только запустить LibreOffice, но и просмотреть список последних открытых документов, или выбрать между созданием текстового документа, презентации и электронной таблицы; Включение в состав интерфейса Boxes, который сочетает средства для удаленного соединения с рабочими столами с механизмами управления и запуска виртуальных окружений на локальных и внешних компьютерах. Для конечного пользователя Boxes позволяет имея на руках установочный или загрузочный образ системы, с минимальными усилиями подготовить рабочую виртуальную машину, не задумываясь о настройке и используемом гипервизоре (по сути нужно лишь выбрать iso-образ, все остальное будет сконфигурировано автоматически). Находясь дома или в дороге, пользователь также может воспользоваться Boxes для соединения с рабочей машиной, без выполнения сложных настроек на обеих сторонах.Для организации эффективного доступа к графическим приложениям на внешней виртуальной машине поддерживается протокол SPICE. Для запуска локальных виртуальных окружений используется KVM и Qemu; Расширены возможности Documents, приложения для упрощения работы с документами, организации поиска документов и обеспечения их просмотра. Переработано оформление, убрана боковая панель, расширены возможности поиска, реализован режим выбора группы документов. Добавлена поддержка организации коллекций документов. В обзорный режим Gnome Shell добавлен модуль для поиска документов. Среди запланированных на будущее возможностей: поддержка сменных накопителей, предоставление совместного доступа к документам, поддержка печати и портретный режим; Добавление в gnome-control-center диалога для определения параметров масштабирования в Gnome Shell; Расширение средств для настройки графических планшетов Wacom. Добавление функций калибровки планшета, настройки одновременно нескольких планшетов, привязки кнопок; Улучшение панели центра управления для управления аккаунтами пользователей. Добавлена поддержка идентификации через Kerberos; Улучшение панели центра управления для настройки сетевых параметров. Полностью изменено оформление панели, портирована поддержка настройки VPN, реализованы диалоги задания пароля; Реализация эффектов для изменения яркости и контраста элементов рабочего стола в GNOME Shell. Интеграция поддержки фреймворка IBus в меню определения метода ввода (языка и раскладки клавиатуры) в Gnome Shell и блок региональных настроек в центре управления; Расширение возможностей GNOME Shell по подключению дополнений; Элементы интеграции с социальными сетями.
