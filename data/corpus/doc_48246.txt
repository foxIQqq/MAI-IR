Выпуск среды управления данными Kexi 3.1

С небольшим отставанием от офисного пакета Calligra 3.1 состоялся релиз среды для визуального управления данными Kexi 3.1, построенной с использованием технологий KDE Frameworks 5 и Qt 5. Kexi является открытым аналогом таких проприетарных систем как MS Access, FileMaker, Oracle Forms и позволяет быстро создавать формы ввода, обработки и редактирования данных, формировать запросы и генерировать отчёты. Основные новшества: Возобновлена поставка инсталлятора с предварительным вариантом Kexi для платформы Windows. Для упрощения миграции с проприетарного на свободное ПО поставляется компонент для миграции с Microsoft Access, позволяющий организовать редактирование БД из Kexi и MS Access; Большое внимание уделено повышению стабильности и обратной совместимости. Устранено более 200 ошибок. Улучшена интеграция с различными окружениями рабочего стола; В диалоге импорта таблиц (Import Table Assistant) добавлена возможность выбора кодировки символов для исходной БД. Добавлена поддержка групп свойств (Property groups) и возможность задания собственных размеров страниц отчета; Представлен первый официальный релиз набора KEXI Frameworks, в который выделены компоненты Kexi, ориентированные на включение в набор KDE Frameworks 5 для обособленного использования в приложениях на C++ и Qt. В библиотеку KDb, предназначенную для соединения с разными СУБД и работы с базами данных, добавлены новые функции для отладки SQL-выражений и улучшена система кэширования схем БД. В библиотеке KProperty, которая предоставляет инструменты для редактирования свойств и виджет с редактором полей, улучшена поддержка единиц измерения и реализована визуальная группировка свойств. В библиотеке KReport, включающей средства для формирования отчётов в различных форматах (позиционируется как аналог систем отчётов SAP Crystal Reports, FileMaker и MS Access), появилась возможность установки произвольных размеров страниц.
