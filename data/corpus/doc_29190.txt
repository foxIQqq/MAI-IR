Ubuntu и MeeGo удалось установить на коммуникатор HTC HD2

Представлена сборка Ubuntu 10.10 для коммуникатора HTC HD2, изначально поставляемого с Windows Mobile. В качестве интерфейса пользователя задействован специально оптимизированный графический интерфейс на базе EFL (Enlightenment Foundation Libraries), используемый в Ubuntu Netbook Edition для архитектуры ARM. Видеоролик с демонстрацией работы Ubuntu на HTC HD2 можно посмотреть здесь. Использовав Linux-ядро из проекта по портировнию Ubuntu на HTC HD2, один из энтузиастов смог установить свободную мобильную платформу MeeGo 1.1 на коммуникатор HTC HD2. В процессе портирования также были использованы наработки по адаптации работы MeeGo для телефона Nexus One, аппаратная начинка которого достаточно близка к HTC HD2. В настоящее время работает только минимальный набор функций, но уже поддерживается сенсорный экран, WiFi, выход в интернет, осуществление звонков и отправка SMS.
