Вышел Calculate Linux Desktop 9.6

Вышел релиз десктоп-ориентированного дистрибутива Calculate Linux Desktop 9.6, представляющего собой скомпилированный stage4-образ Gentoo Linux, снабженный программной для быстрой установки. Система распространяется на LiveDVD диске (1.6 Гб) с возможностью установки на HDD, либо USB-HDD. Основные изменения: Для загрузки с DVD используется isolinux. Добавлен выбор раскладки клавиатуры при загрузке с DVD. Добавлена графическая утилита настройки сети Wicd. За настройку сетевых интерфейсов теперь отвечает Wicd Manager. Настройка видеокарты при загрузке с DVD осуществляется через видеодрайвера Xorg. Установочный образ отдельным архивом более распространяться не будет. Систему можно будет обновить с версии 9.5 из ISO файла, без записи на DVD диск. Изменен путь хранения учетных записей пользователей на сервере. Не рекомендуется использовать с более старыми версиями системы. При отсутствии связи с сервером, введенный в домен компьютер во время загрузки будет переключен в обычный режим работы. Улучшена совместимость с USB DVD. Состав: KDE 4.2.3, Kernel 2.6.28.10, OpenOffice 3.0.1, X.Org 7.4.
