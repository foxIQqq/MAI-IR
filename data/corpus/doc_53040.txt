Выпуск UDisks 2.9.0 с поддержкой переопределения опций монтирования

Состоялся релиз пакета UDisks 2.9.0, включающего системный фоновый процесс, библиотеки и инструментарий для организации доступа и управления дисками, устройствами хранения и связанными с ними технологиями. UDisks предоставляет D-Bus API для работы с дисковыми разделами, настройки MD RAID, работы с блочными устройствами в файле (loop-монтирование), манипуляций с файловыми системами и т.п. Дополнительно поставляются модули для мониторинга и управления BCache, BTRFS, iSCSI, libStorageManagement, LVM2, LVM Cache и zRAM. Например, UDisks иcпользуется в приложениях GNOME для работы с дисковыми разделами GNOME и различных графических конфигураторах. В новой версии: Реализовано управление опциями монтирования для файловых систем. Через UDisks теперь можно менять предлагаемые по умолчанию опции монтирования для каждого типа файловых систем (для ФС, не определённых в /etc/fstab); Значительно изменён метод обновления свойств объектов D-Bus (свойства объектов обновляются до возвращения вызова метода); Переработан API внутренних модулей. Модули теперь необходимо включать по отдельности через вызов EnableModule(); Добавлена возможность сборки только библиотеки libudisks2 без фонового процесса; Удалён сервис systemd для чистки точек монтирования. Состояние монтирования теперь отслеживается отдельно для непостоянных и постоянных точек монтирования, а чистка выполняется при запуске фонового процесса; Предложен новый модуль интеграции LVM-VDO, позволяющий обойтись без отдельного модуля VDO; Добавлена поддержка разблокировки и блокировки устройств BitLocker.
