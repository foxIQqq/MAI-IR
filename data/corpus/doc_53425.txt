Инициатива по созданию сборок GNOME OS для реального оборудования

На конференции GUADEC 2020 прозвучал доклад, посвящённый развитию проекта "GNOME OS". Изначально вынашиваемые планы по развитию "GNOME OS" как платформы для создания ОС в настоящее время трансформировались в рассмотрение "GNOME OS" как сборки, которую можно использовать для проведения непрерывной интеграции, упрощения тестирования работы приложений в развиваемой для следующего релиза кодовой базе GNOME, оценки хода разработки, проверки совместимости с оборудованием и экспериментов с интерфейсом пользователя. До последнего времени сборки "GNOME OS" были рассчитаны на запуск в виртуальных машинах. Новая инициатива связана с работой по обеспечению использования "GNOME OS" на реальном оборудовании. Разработка новых сборок ведётся для систем x86_64 и ARM (Pinebook Pro, Rock 64, Raspberry Pi 4). По сравнению со сборками для виртуальных машин добавлена возможность загрузки на системах с UEFI, средства управления энергопотреблением, поддержка вывода на печать, Bluetooth, WiFi, звуковых карт, микрофона, сенсорных экранов, графических карт и вебкамер. Добавлены недостающие порталы Flatpak для GTK+. Подготовлены Flatpak-пакеты для разработки приложений (GNOME Builder + SDK). Для формирования системной начинки в GNOME OS задействована система OSTree (системный образ атомарно обновляется из Git-подобного хранилища), по аналогии с проектами Fedora Silverblue и Endless OS. Инициализация производится при помощи Systemd. Графическое окружение базируется на драйверах Mesa, Wayland и XWayland. Для установки дополнительных приложений предлагается использовать Flatpak. В качестве инсталлятора задействован Endless OS installer на базе GNOME Initial Setup. Дополнение: Для тестирования доступны готовые сборки GNOME OS для запуска в виртуальных машинах.
