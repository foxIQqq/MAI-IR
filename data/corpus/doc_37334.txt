Релиз Opera 15 вышел без поддержки Linux, которая ожидается позже

Компания Opera Software представила первый стабильный релиз полностью переработанного варианта браузера Opera, перешедшего с собственного движка Presto на браузерный движок Blink (форк WebKit) и JavaScript-движок V8, развиваемые в рамках проекта Chromium. Одновременно объявлено о переходе на новый сокращённый цикл подготовки новых версий, сформированный по аналогии с Chrome и Firefox. Новые стабильные выпуски отныне будут публиковаться раз в несколько недель. Кроме стабильной ветки параллельно будут формироваться тестовые выпуски Opera Developer (нестабильная экспериментальная версия) и Opera Next (версия в процессе стабилизации). К сожалению по возможностям интерфейса и поддержке платформ выпуск Opera 15 существенно отстаёт от прошлой версии Opera 12, которая пока остаётся на плаву и скоро будет доступно обновление 12.16. Наиболее неприятным изменением является отсутствие поддержки платформы Linux - браузер вышел только в сборках для Windows и Mac OS X. Тем не менее, разработчики подтвердили данные о том, что работа над версией для Linux ведётся, но никаких, даже примерных, сроков выхода сборки для Linux не называется. Из других проблемных моментов можно отметить специфичную систему закладок, прекращение поддержки боковой панели и режима просмотра нескольких вкладок в одном окне, сокращение настроек и возможностей по адаптации интерфейса, пропадание отдельного поля для поиска и встроенных средств блокирования контента. Почтовый клиент и интерфейс для чтения RSS выделены из базовой поставки. Кроме того, прекращена поддержка дополнений, написанных для Opera 11 и 12, для разработки дополнений следует использовать подмножество API проекта Chromium, дополненного такими возможностями, как Speed Dial API. Среди улучшений, появившихся в Opera 15: Переработанная экспресс-панель, которая тесно интегрирована с системой ведения закладок и комбинирует при выводе как ссылки для быстрого запуска, так и разделы системы закладок. Поддерживаются средства группировки, переименования и перемещения содержимого экспресс-панели, а также объединение ссылок по разделам; Адресная строка комбинирует в себе поле для ввода адресов и форму для отправки поисковых запросов; Новая панель Discover, предназначенная для обнаружения нового контента в Сети и слежения за новостными лентами по интересам. При этом браузер сам формирует подборку источников в зависимости от выбранной тематики и производит фильтрацию популярного и востребованного контента; Функция Stash для удобного сравнения нескольких страниц (например, для сравнения цен на разных сайтах); Для ускорения работы на низкоскоростных каналах связи или при необходимости экономить трафик, предусмотрен режим Off-Road, при котором активируется аналог ранее используемой технологии Opera Turbo, переведённой на использование протокола SPDY. При включении режима Off-Road обработка контента производится на стороне серверной инфраструктуры Opera Software с передачей данных клиенту в компактном виде, что позволяет снизить трафик и обеспечить комфортную работу при роуминге или при работе в 2G-сетях;
