Вышел Google Web Toolkit 2.2

Анонсирован выход Google Web Toolkit 2.2 (GWT), набор средств для разработки динамических web-приложений, которые изначально могут быть написаны на языке Java, а затем при помощи специального транслятора преобразованы в представление на базе JavaScript и HTML. Из новшеств можно отметить: Визуальная система проектирования пользовательского интерфейса GWT Designer интегрирована непосредственно в Google Plugin 2.2 для Eclipse. Добавлена поддержка HTML5-специфичных возможностей, таких как элемент Canvas для формирования векторных изображений и теги Audio/Video; Интегрирован вижет CellTable для гибкого формирования интерактивных таблиц с поддержкой сортировки столбцов и ввода данных;
