Представлен Plasma Media Center 1.2, медиацентр от проекта KDE

Проект KDE анонсировал выпуск Plasma Media Center 1.2 (PMC), унифицированного интерфейса для работы с мультимедиа-контентом на стационарных ПК, нетбуках, планшетах, телевизорах и других устройствах, на которых может быть запущено окружение KDE. PMC обладает типичными для медиацентров функциями и позволяет показывать изображения, проигрывать музыку и воспроизводить видео. PMC базируется на технологиях Plasma и KDE, и использует QML для формирования интерфейса, адаптированного для различных классов устройств. Для расширения функциональности предоставляется API для создания плагинов, Поддерживается работа как с содержимыми локальных файловых систем, так и внешних online-сервисов, таких, как YouTube Flickr и Picasa. Управление воспроизведением и навигация по контенту могут осуществляться через клавиатуру/мышь, сенсорный экран или пульты дистанционного управления. Для унифицированного просмотра всего медиаконтента доступна возможность использование средств поиска на десктопе KDE (KDE Desktop Search). Ключевые новшества: Переработан режим работы с музыкальной коллекций ("All Music"). Улучшены средства навигации по музыкальной коллекции, реализованы режимы вывода с группировкой по музыкантам или альбомам, также возможно отображение всей коллекции или элементов, удовлетворяющих заданной маске; Поддержка загрузки обложек альбомов и фотографий музыкантов из сервиса last.fm. Увеличена наглядность определения искомого альбома; При просмотре содержимого локальных дисков на выводимых для каталогов пиктограммах обеспечено отображение миниатюр фотографий, отражающих содержимое каждой директории; Добавлены горячие клавиши для управления воспроизведением. По умолчанию для остановки/проигрывания можно нажать пробел, а для перехода на следующую или предыдущую композицию - клавиши N и Z; Предоставлены средства для управления набором списков воспроизведения. Пользователь может подготовить серию разных плейлистов и переключаться между ними в зависимости от ситуации; Представлен новый набор пиктограмм для элементов управления, более заметный на тёмном фоне. Из планов на следующую версию отмечается: реализация средств для локального поиска файлов; экран для настройки параметров индексации, кастомизации интерфейса и поддержания списка игнорируемого контента (например, можно будет игнорировать мелкие картинки и короткие звуковые файлы); улучшение навигации по галереям в Picasa; вывод детальной информации о роликах Youtube.
