Выпуск мобильной ОС Sailfish 2.2.0

Компания Jolla, основанная бывшими сотрудниками Nokia с целью разработки новых смартфонов, построенных на базе Linux-платформы MeeGo, опубликовала релиз операционной системы Sailfish 2.2.0. Сборки подготовлены для устройств Jolla 1, Jolla C и Sony Xperia X, но поставляются пока только для зарегистрированных участников программы раннего доступа к прошивкам (для всех остальных доступ будет открыт через неделю). Sailfish использует графический стек на базе Wayland и библиотеки Qt5, системное окружение построено на основе Mer (форк MeeGo) и пакетов Mer-дистрибутива Nemo. Пользовательская оболочка, базовые мобильные приложения, QML-компоненты построения графического интерфейса Silica, прослойка для запуска Andrоid-приложений, движок умного ввода текста и система синхронизации данных являются проприетарными, но их код планировалось открыть ещё в прошлом году. В новой версии: В галерее и приложении для работы с камерой появился новый одиночный режим просмотра фотографий вместо разделения экрана на две части. Переработан просмотрщик MMS. Добавлена поддержка изображений в формате TIFF; Добавлена возможность разблокировки с использованием аутентификации по отпечаткам пальцев; В виртуальной клавиатуре появилась новая раскладка для ввода emoji; Стабилизирована поддержка VPN, которая вышла из стадии бета-тестирования. Улучшена автоматическая установка соединения с VPN. Добавлена поддержка дополнительных типов VPN; В диалоге настройки беспроводного соединения добавлена возможность ручной настройки параметров сети; Расширены средства удалённого управления мобильным устройством (MDM, Device Management), добавлен API для отключения передачи данных и SMS для определённых SIM-карт, а также API для выполнения удалённой чистки содержимого SD-карты; Для устройств Xperia X и Jolla C обновлена прослойка Aliendalvik, предоставляющая средства для запуска приложений, написанных для платформы Android; Внесены улучшения в почтовый клиент: добавлен индикатор прогресса выполнения операции, реализована возможность прерывания загрузки вложений, добавлена опция для восстановления случайно отменённого письма; Улучшено качество наведения фокуса для Xperia X. Прекращена поддержка режима непрерывной фокусировки для камеры Xperia X, который не обеспечивал желаемого качества. По умолчанию наведение фокуса теперь производится после касания на объекте или в момент совершения снимка; Добавлена поддержка моделей Xperia X F5122 с двумя SIM-картами. Обновлена инструкция по установке Sailfish X на Sony Xperia X. Для каждой поддерживаемой модели Xperia X (F5121 и F5122) теперь поставляется отдельная сборка; Улучшена работа звуковых уведомлений. PulseAudio обновлён до версии 11.1. Для Xperia X добавлена поддержка USB audio; В уведомлениях обеспечена поддержка отображения индикатора прогресса выполнения операции. Добавлена опция для вывода вибросигнала при поступлении нового уведомления; Представлен Sailfish Secrets, открытый фреймворк для организации хранения данных в зашифрованном виде и предоставления доступа к Crytpo API. Код фреймворка открыт под лицензией BSD; В калькуляторе появилось контекстное меню для сохранения результата в буфере обмена.
