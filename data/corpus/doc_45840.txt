Выпуск программы для управления фотографиями digiKam 5.4

Состоялся релиз программы для управления коллекцией фотографий digiKam 5.4.0. Кроме закрытия более 200 ошибок новый выпуск примечателен полной переработкой кода для работы с видеофайлами, улучшением системы выявления дубликатов и расширением средств для группировки изображений. Для обработки видео и звуковых файлов digiKam переведён с Qt5Multimedia на фреймворк QtAV, который не привязан к GStreamer в Linux и не требует отдельной ручной установки кодеков в Windows и macOS. QtAV использует компоненты пакета FFmpeg, которые едины для всех поддерживаемых платформ. Использование QtAV позволило реализовать полноценные средства построения эскизов для видеофайлов и обеспечить надёжную работу функции быстрого предпросмотра. В результатах поиска дубликатов обеспечено отображение коэффициента похожести найденных изображений с оригиналом, с учётом которого организована сортировка вывода. Предоставлена возможность определения минимального уровня похожести для включения в результаты поиска. Улучшены средства группировки изображений - входящие в одну группу изображения теперь могут быть обработаны все вместе. Если ранее применение операции к группе приводило к обработке только верхнего изображения, идентифицирующего группу, то теперь, например, если привязать тег к верхнему изображению, он будет выставлен для всех изображений в группе. В менеджер пакетных операций (Batch Queue Manager) добавлена возможность обработки только текущей очереди, вместо обработки сразу всех очередей (теперь для обработки всех очередей добавлена отдельная команда "Run all"). Расширены возможности по управлению иерархиями тегов, работе с историей назначения тегов и автодополнением тегов.
