Новая версия MySQL 5.5.7-rc

Компания Oracle представила MySQL Community Server 5.5.7-rc, второй кандидат в релизы новой ветки СУБД MySQL 5.5. Ветка MySQL 5.5 базируется на невыпущенной серии MySQL 5.4 и содержит ряд значительных улучшений, связанных с повышением масштабируемости и производительности. По сравнению с прошлой тестовой версией в MySQL 5.5.7 добавлена поддержка подключаемых модулей аутентификации, позволяющих использовать на стороне сервера и клиента плагины с реализацией новых методов аутентификации. Например, могут быть подготовлены плагины для осуществления аутентификации в LDAP, Kerberos, PAM или Windows-домене. Кроме того, реализована поддержка прокси-режима аутентификации, позволяющего клиенту, аутентифицированному и подключившемуся к серверу под одним пользователем, получить привилегии другого пользователя. Например, для доступа к БД извне создается отдельный пользователь "B", ассоциированный с определенным плагином аутентификации. Если теперь для этого пользователя выполнить GRANT PROXY ON 'A' TO 'B', то при входе пользователем "B" для системы такое подключение будет выглядеть как будто вошел пользователь "A". С другими улучшениями ветки MySQL 5.5 можно познакомиться в тексте анонса прошлой версии. Дополнительно можно отметить, что компания Oracle опубликовала свое видение ситуации, касающейся изменения цен на коммерческую версию MySQL. В заметке указано, что базовый план поддержки за 599$ не пользовался спросом, подразумевал решение только двух инцидентов в год и оказание услуг технической поддержки в рабочее время. Новый план поддержки за 2000$ в год подразумевает круглосуточную поддержку и неограниченное число инцидентов. Ранее за такой сервис компании вынуждены были платить 3000$ вместо $2000, т.е. для крупных клиентов цена стала на 30% меньше.
