Началось тестирование редакции платформы MeeGo для смартфонов

Выпущена первая тестовая версия мобильной платформы MeeGo 1.1, отличающаяся от первого релиза реализацией интерфейса для устройств с небольшим экранным разрешением, поддержкой управления через сенсорный экран и набором улучшений, связанных с работой MeeGo на смартфонах и карманных устройствах. Финальный релиз MeeGo 1.1 ожидается в октябре, кроме карманных устройств в MeeGo также ожидается реализация специализированного интерфейса для планшетных ПК. Улучшения, представленные в первой тестовой версии MeeGo 1.1: MeeGo API, включающее Qt и MeeGo Touch UI Framework (MTF) для быстрой разработки приложений, управляемых с сенсорного экрана; Прототип будущего пользовательского интерфейса для карманных устройств; Набор приложений, ориентированных для использования в карманных устройствах: Строка статуса, содержит в себе часы, индикатор состояния сети, Bluetooth и 3G-соединения, область уведомлений и индикатор заряда батареи; Домашний экран; Хранитель заблокированного экрана; Приложение для запуска программ (Application Launcher); Виртуальная клавиатура; Приложение для осуществления звонков (Dialer) и отправки SMS, web-браузер (на базе QtWebKit), адресная книга, программа для просмотра фотографий; Из аппаратных устройств поддерживается Nokia N900 и карманные устройства на базе CPU Intel Atom (Moorestown); Linux ядро 2.6.35-rc.
