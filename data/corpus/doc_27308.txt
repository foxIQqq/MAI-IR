НауЛинукс Школа 5.5 переведён в состояние опубликованного релиза

Компания ЛИНУКС ИНК представила релиз дистрибутива "НауЛинукс Школа 5.5", основанного на пакетной базе русифицированной сборки Scientific Linux 5.5 - Scientific Linux Cyrillic Edition 5.5 (SLCE 5.5). Дистрибутив НауЛинукс Школа 5.5 представляет собой совмещенное десктопно-серверное решение, подходящее как для обеспечения школьных серверов, так и конечных рабочих мест. Дистрибутив оформлен в виде двух установочных DVD. Дополнительно поставляется CD с набором вспомогательного ПО и документацией. Дистрибутив полностью совместим с пакетной базой RHEL, CentOS и Scientific Linux. Основные изменения по сравнению с предыдущей версией: Обновлены версии офисного пакета OpenOffice.org (до 3.1.1) и веб-браузера Firefox (до 3.6.4) В дистрибутив включены программные пакеты с новыми версиями Samba (samba3x-3.3.8) и PostgreSQL (postgresql84-8.4.4), которые могут быть установлены вместо стандартных samba-3.0.33 и postgresql-8.1.21 В конфигуратор пакета настройки школьного сервера nau-school-server добавлена возможность внесения изменений в список преподаваемых предметов. В состав "НауЛинукс Школа" входит: Типовой комплект популярных пользовательских программ и десктоп-окружений: GNOME, KDE, IceWM, браузеры, почтовые клиенты, клиенты для сервисов мгновенных сообщений и программы для организации видео-конференций, офисный пакет, мультимедийные приложения для просмотра и редактирования аудио и видео данных; Подборка образовательных программ для детей различного возраста (GCompris, Kdeedu) и систем для обучения программированию (Free Pascal Compiler, Lazarus). В комплект также включены пакеты для разработки на языках C/C++, Java, PHP, Perl, Python, Ruby, LISP, интегрированные среды Eclipse, KDevelop, Anjuta, Emacs, платформы Tomcat, Geronimo, Zope и Plone; Набор приложения для развертывания сетевой инфраструктуры и администрирования локальной сети. Имеются средства для создания файлового сервера, службы LDAP каталогов, сервера печати, почтового сервера, web-сервер, сервера баз данных (MySQL, PostgreSQL, FireBird). В состав дистрибутива также включены компоненты для создания вычислительных кластеров, кластеров высокой надежности и кластеров с балансировкой нагрузки; Расширенная система конфигураторов, позволяющих определить структуру школы и набор преподаваемых предметов. Сформированная конфигурация передается затем другим компонентам комплекса "Школьный сервер": школьному веб-порталу и системе администрирования ОфисМастер. Школьный веб-портал на базе Plone CMS, который выступает в роли интерфейса для настройки и управления ресурсами образовательного учреждения, может быть использован для создания сайта школы или интранет-портала; Система администрирования Интернет/Интранет служб и контроля доступа ОфисМастер, которая позволяет манипулировать механизмами доступа в Интернет, вести работу с единой базой пользователей, управлять почтовыми списками рассылки и т.п. Интеграция комплекса с системой удаленного обучения Moodle. На дополнительном CD помещен набор программных пакетов, позволяющий запускать систему "1С:ХроноГраф" при помощи WINE. В состав дистрибутива включен набор средств для сканирования и распознавания текстов gscan2pdf, Cuneiform, Tesseract и GOCR.
