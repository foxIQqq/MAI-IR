Фонд СПО проводит акцию по продвижению свободных PDF-просмотрщиков

Европейское отделение Фонда свободного ПО объявило о проведении с 13 сентября по 27 октября акции, направленной на пропаганду использования свободных программ для просмотра PDF-документов. В настоящее время на многих сайтах государственных учреждений базовые документы распространяются в формате PDF, при этом для их просмотра часто рекомендуется использовать проприетарные продукты (например, предлагается загрузить Adobe Reader). По мнению представителей Фонда СПО такая практика неприемлема и они призывают пользователей, столкнувшихся с подобной рекламой проприетарных программ, не полениться и уведомить организаторов акции о таких сайтах. Форму для отправки уведомления, а также список уже добавленных сайтов можно посмотреть на данной странице. В рамках акции будет проводиться символичное соревнование: в конце октября, после подведения итогов, будут названы и награждены призами три самых активных энтузиаста, обнаружившие наибольшее количество государственных сайтов, рекламирующих проприетарное ПО. В качестве приза будет предоставлена оплаченная поездка на конференцию FSCONS. По заявлению одного из представителей фонда СПО, указания о необходимости загрузки определённых проприетарных программ на государственных сайтах аналогичны таким нелепым ситуациям, как установка дорожного знака "По данному шоссе можно проехать только на Жигулях. Пожалуйста, обратитесь к ближайшему дилеру для заказа бесплатного тестдрайва". Подобная поддержка предоставляет производителям проприетарного ПО необоснованное преимущество, при этом часто ссылки даются в таком виде, что оставляют впечатление необходимости использования только одного возможного приложения для просмотра представленных документов. Дополнительно, всем несогласным с размещением подобной рекламы на государственных сайтах, предлагается подписать специальную петицию. Список свободных PDF-просмотрщиков на любой вкус и для любых операционных систем доступен через сайт pdfreaders.org. Данный сайт может быть рекомендован для формирования ссылки для загрузки программ для чтения PDF-документов.
