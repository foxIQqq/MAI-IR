В дистрибутиве Yellow Dog Linux задействован GPU для оптимизации вычислений

Компания Fixstars анонсировала выход Yellow Dog Enterprise Linux for CUDA, дистрибутива специально адаптированного для задействования в работе мощностей GPU. В дистрибутив интегрирован полный комплекс инструментов для организации выполнения и разработки программ с использованием разработанного компанией NVidia интерфейса CUDA (Compute Unified Device Architecture). Для упрощения разработки, в комплект включена основанная на Eclipse специально созданная интегрированная среда для разработки приложений с использованием CUDA. Что касается стандартных программ, то интеграция поддержки CUDA в дистрибутив позволила увеличить производительность некоторых из них до 9%. Дистрибутив Yellow Dog Linux базируется на пакетной базе CentOS с некоторыми компонентами из Fedora Linux, оптимизирован для процессоров Cell и Power и поддерживает установку на приставки Sony PlayStation 3, рабочие станции Apple G4/G5, YDL PowerStation и IBM System P. Основное пользовательское окружение дистрибутива построено на основе Enlightenment 17. На 26 марта намечен выпуск первых карт GeForce 400, поддерживающих новую CUDA-архитектуру Fermi, которая позволит организовать прямое взаимодействие с операционной системой, выступая в качестве CGPU - гибрида центрального процессора и графического процессора, на котором сможет выполняться специально модифицированная версия Linux.
