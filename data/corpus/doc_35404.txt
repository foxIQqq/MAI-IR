Результаты анализа возможности размещения GPL-приложений в Windows Store

Жан-Батист Кемпф (Jean-Baptiste Kempf), лидер сообщества VideoLAN, опубликовал результаты изучения вопроса совместимости каталога-магазина Windows Store со свободными приложениями, распространяемыми под лицензией GPL. Предлагаемые разработчикам условия размещения программ были проанализировано на предмет наличия требований, ущемляющих свободу по копированию, изменению, распространению и неограниченному использованию приложений. Как известно, примером дружелюбного к свободным лицензиям каталога-магазина является Google Play, через который распространяются приложения для платформы Android. Его противоположностью может выступать Apple App Store, который признан Фондом СПО несовместимым с приложениями под лицензией GPL, так как пользовательское соглашение в Apple Store запрещает дальнейшее распространение и модификацию загруженных из каталога программ. Каталог Windows Store, через который организовано распространение программ для платформ WinRT "Metro" и Windows Phone 8, в целом, оказался пригоден для распространения через него приложений под copyleft-лицензиями, требующими предоставления возможности неограниченного распространения копий приложений. Правила Windows Store явно допускают для свободных приложений использование обратного инжиниринга, декомпиляции, дизасемблирования, копирования, публикации, распространения и отладки. Тем не менее, в условиях размещения есть один момент, требующий от разработчиков явного упоминания возможности установки программы на неограниченное число устройств, так как по умолчанию разрешена установка программы не более чем на 5 устройств пользователя. Кроме того, возможность неограниченного использования и копирования приложений добавлена в качестве исключения только для свободных и открытых лицензий, утверждённых организацией OSI (Open Source Initiative), в остальных случаях обмен загруженными из магазина приложениями с другими пользователями запрещён.
