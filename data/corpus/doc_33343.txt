Релиз свободного инструментария для построения графиков Gnuplot 4.6

После двух лет разработки представлен релиз Gnuplot 4.6, свободного инструментария для построения двумерных и трехмерных научных графиков, поддерживающего широкий спектр форматов вывода и возможность использования скриптов для формирования входящих данных. В новой версии: Новый синтаксис управления потоком, поддерживающий многострочные блоки кода, оформленного через "{...}" ( if () { ... } else { ... }; do for [] { ... }; while () { ... }); Возможность использования определённых пользователем типов линий на графике (задаётся цвет и стиль); Форматы времени могут оперировать точностью от долей секунды до микросекунд; Новые терминальные драйверы: qt, context, epscairo, cairolatex; Средства для статистического расчёта суммарных величин для выводимых на графике данных (команда "stats"); Улучшенная поддержка UTF-8 и многобайтовых кодировок; Значительно улучшена поддержка платформы Windows (GUI, подсказка, терминал native, wxt и инсталлятор).
