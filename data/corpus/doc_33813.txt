Компания Canonical выпустила отдельную сборку Ubuntu Desktop для корпоративных пользователей

Компания Canonical анонсировала первую стабильную версию специализированной сборки для корпоративного сектора и государственных учреждений - Ubuntu Business Desktop Remix, основанную на пакетной базе Ubuntu 12.04 LTS. От стандартного Ubuntu Desktop представленная сборка отличается иной подборкой поставляемых по умолчанию приложений - в состав установочного образа включены программы, наиболее часто используемые на предприятиях и интересные для корпоративных пользователей. Размер доступного для загрузки iso-образа (amd64/i386) составляет 794 Мб. Вместо программ, интересных только пользователям домашних ПК и разработчикам, таких, как игры, музыкальные плееры, приложения для интеграции с социальными сетями, торрент-клиенты и инструменты для разработки программ, в состав сборки включена подборка бизнес-приложений, включая такие компоненты, как Adobe Flash Plugin, VMware View и OpenJDK 6. В состав также входит клиент удалённого управления рабочим столом, совместимый с Microsoft Windows RDP 7.1, и новая версия офисного пакета LibreOffice, примечательная возможностью импорта диаграмм в формате Microsoft Visio. Отдельно отмечается, что в Ubuntu Business Desktop Remix используются только пакеты из стандартных репозиториев, без каких-либо изменений. Все входящие в сборку компоненты можно установить через Software Center и в обычном Ubuntu. В качестве мотива создания отдельной сборки для корпоративных пользователей объясняется желанием обеспечить экономию времени на установку, за счёт минимизации дополнительной работы на удаление и добавление пакетов на рабочих станциях, а также предоставление протестированного и опробованного комплекта, гарантирующего сочетание всех предустановленных компонентов. Наработки Ubuntu Business Desktop Remix также позволяют упростить процесс создания собственных кастомизированных сборок для предприятия, через задействование в качестве шаблона сборочного профиля представленной редакции.
