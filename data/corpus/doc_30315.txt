Обновление проприетарного драйвера NVIDIA: 173.14.30 и 270.41.06

Компанией NVIDIA выпустила пакет проприетарных драйверов для Linux - NVIDIA 270.41.06, имеющий статус пре-релиза. По сравнению с выпуском 270.41.03 в новой версии добавлена поддержка GPU Tesla M2090, Tesla M2075, Tesla C2075, Quadro 7000, устранено зависание в процессе запуска X-сервера, наблюдаемое на платах F104 и GF106. Среди улучшений ветки 270.41.x также можно отметить поддержку Xorg Server 1.10, Linux-ядра 2.6.38 и будущего релиза дистрибутива Ubuntu 11.04. Одновременно выпущена 173.14.30 версия проприетарных драйверов для видеокарт GeForce 5, переведенных в разряд устаревших. Ключевыми улучшениями, добавленными в версии 173.14.30, стала поддержка X-сервера 1.10 и обеспечение совместимости с новыми версиями Linux-ядра. Пока не известно, будет ли выпущено аналогичное обновление для серии драйверов 96.43.xx, в которых обеспечена поддержка карт с GeForce 2 по GeForce 4, и для серии 71.86.xx, в которой поддерживаются карты Riva TNT, TNT2, GeForce и некоторые старые модели GeForce 2.
