Обнародован план неподготовки релиза Python 2.8

Barry Warsaw опубликовал в списке рассылки python-dev план неподготовки новой версии Python 2.8. Версия 2.7, вышедшая в 2010 году, была заявлена как последняя версия в этой серии. Планировалось, что новшества в ветку 2.x больше добавляться не будут, а поддержка (исправление ошибок) продлится до 2015 года. В полном согласии с предыдущими планами дата релиза Python 2.8 назначена на "никогда" (Never). В соответствии с принятым ранее решением Python ветка 2.8 не будет выпущена. Официальный путь модернизации Python 2.7 — Python 3. Ответственным за "не-выпуск" (Un-release Manager) предложен кардинал Бигглз (Cardinal Biggles), герой одного из телевизионных шоу комедийной группы Монти Пайтон (Monty Python), в честь которой назван язык программирования Python.
