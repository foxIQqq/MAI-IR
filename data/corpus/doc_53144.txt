Июньское обновление приложений KDE 20.04.2

В соответствии внедрённым в прошлом году ежемесячным циклом публикации обновлений представлено июньское сводное обновление приложений (20.04.2), развиваемых проектом KDE. Всего в рамках январского обновления опубликованы выпуски более 120 программ, библиотек и плагинов. Информацию о наличии Live-сборок с новыми выпусками приложений можно получить на данной странице. Наиболее заметные новшества: В состав приложений KDE включена программа для резервного копирования Kup, ранее развивавшаяся в виде независимого проекта. Предложен новый выпуск Kup 0.8, в котором переработано поведение при сохранении бэкапов rsync при выборе только одного каталога, добавлены опции для определения файла со списком исключений, добавлена поддержка третьей версии метаданных утилиты Bup (для сохранения версионированных бэкапов при помощи rsync в Kup применяется утилита Bup, которая пока поддерживает работу только с Python 2, но уже развивается порт для Python 3). Обновлён выпуск музыкального проигрывателя Elisa, разработчики которого пытаются воплотить в жизнь рекомендации по визуальному дизайну мультимедийных проигрывателей, разработанных рабочей группой KDE VDG. Из недавних изменений в кодовой базе Elisa отмечается реализация поддержки потокового вещания на устройства в локальной сети при помощи UPnP DLNA. Поддержка DLNA будет включена в следующем значительном выпуске, в версии 20.04.2 представлено только исправление ошибок. В растровом графическом редакторе Krita, развиваемом для художников и иллюстраторов, улучшена поддержка планшетов на базе платформы Android и устройств Chromebook. Android-сборка Krita размещена в Google Play. Версия для планшетов пока представляет собой вариант обычной версии для настольных систем - адаптация интерфейса для управления с сенсорных экранов ещё не проведена, но сборка уже вполне пригодна для начальных экспериментов. Представлен первый бета выпуск универсального KIO-модуля KIO Fuse, позволяющего монтировать каталоги с внешних серверов (SSH, SAMBA, FTP, WebDav) или из архивов (TAR, GZip, BZip2, и т.п.), и работать с ними как с обычными локальными каталогами в Dolphin. В KMail улучшена генерация HTML-разметки при добавлении к письму подписи в формате HTML. В сообщество KDE влился проект KDE Homebrew, занимающийся созданием пакетов KDE-приложений для macOS и их размещением в каталоге Homebrew. Всего подготовлено 106 пакетов, из которых 67 библиотеки, а 39 приложения, среди которых Kate, Dolphin и KDevelop.
