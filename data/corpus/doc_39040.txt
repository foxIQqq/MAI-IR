Выпуск Wine 1.7.12

Вышел очередной экспериментальный выпуск открытой реализации Win32 API - Wine 1.7.12. С момента выпуска версии 1.7.11 было закрыто 89 отчётов об ошибках. Наиболее важные изменения: Начальная поддержка интерфейсов Windows Media Player; Расширение поддержки планировщика задач (Task Scheduler); Серия исправлений, связанных с работой runtime-компонентов C++; Реализация порции DLL для поддержки Windows 8 API; Закрыты отчёты об ошибках, связанные с работой игр и приложений: Capitalism II, Photoshop CS2/CS4, Eudora 7.1.0.9, FlightGear, Dragon Age, µTorrent, Pioneer Avic Feeds v2.1.0, Easyworship 2009, Visual C++ 2005, Visio 2007, Microsoft Visio 2003, Office 2003, Blackberry Device Manager 6.1, Wing Commander Saga, Adventure Game Studio 3.2, MetaTrader 5, Eve Online, World of Warplanes, Dracula 4, Divinity II.
