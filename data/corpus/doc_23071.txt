Вышел Catalyst 9.8, проприетарный Linux драйвер для видеокарт ATI/AMD

Объявлено о выходе новой версии проприетарного видеодрайвера для видеокарт ATI - AMD Catalyst 9.8. Из изменений можно отметить реализацию нескольких новых OpenGL 3.1 расширений, появление поддержки Linux ядер 2.6.29 и 2.6.30, проведение оптимизации, позволившей увеличить производительности некоторых OpenGL операций приблизительно на 20%. В Catalyst Control Center добавлена поддержка API RandR 1.2, функция поворота изображения на дисплее и позиционирования рабочей области для нескольких мониторов. В новой версии также исправлены ошибки, связанные с работой тестового комплекта Unigine Tropics, устранены проблемы с базирующимся на ACPI переключением экрана ноутбука через функции RandR 1.2. К сожалению, опубликованная на официальном сайте AMD ссылка на документ с замечаниями к релизу указывает на несуществующую страницу.
