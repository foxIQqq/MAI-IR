Вышел кандидат в релизы Firefox Mobile 1.1 для платформы Maemo

Разработчики проекта Mozilla представили первый кандидат в релизы мобильной версии web-браузера Firefox 1.1 (Fennec) для платформы Maemo. Для предоставления возможности тестирования пользователям не обладающим мобильными устройствами Nokia N810/N900, подготовлены специальные сборки Firefox Mobile 1.1 для запуска в стандартном десктоп-окружении Windows, Mac и Linux. Основные новшества Firefox Mobile 1.1: Реализация функции автозаполнения web-форм; Изменение оформления стартовой страницы; Поддержка автоматического обновления дополнений; Реализация для смартфона N900 портретного режима просмотра и возможности изменения масштаба через нажатие кнопок управления громкостью; Добавление контекстного меню с функциями по открытию ссылки в новом табе и сохранения изображений; Набор настроек для управления параметрами сайта (очистка сохраненных паролей и т.п.) Возможность записи содержимого страницы в PDF-формате; Новая система обработки прикосновений к экрану, позволяющая более точно выполнять такие операции как клик по ссылке и нажатие кнопок.
