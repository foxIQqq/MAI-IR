Выпуск дистрибутивов Альт Рабочая станция, Альт Сервер и Альт Образование 9.0

Состоялся выпуск трёх новых продуктов версии 9.0 на основе Девятой платформы ALT (p9 Vaccinium): «Альт Рабочая станция 9», «Альт Сервер 9», «Альт Образование 9». При создании дистрибутивов ОС «Альт» версии 9.0 для широкого спектра аппаратных платформ разработчики ОС «Альт» ориентировались на потребности корпоративных заказчиков, учебных заведений и частных лиц. Одна из основных тенденций современного развития вычислительной техники – разнообразие аппаратных платформ, в том числе и в России, где есть собственные наработки в процессорных архитектурах. Рабочие станции и серверы на российских процессорах сегодня востребованы в госсекторе, в сфере образования и здравоохранения, в крупном бизнесе и силовых структурах – во всех организациях, которые реализуют масштабные программы импортозамещения. Отечественные операционные системы доступны одновременно для семи российских и зарубежных аппаратных платформ впервые. Теперь ОС «Альт» работают на следующих процессорах: «Альт Рабочая станция 9» – для x86 (32- и 64-битных), AArch64 (NVIDIA Jetson Nano Developer kit, Raspberry Pi 3 и другие), e2k/e2kv4 («Эльбрус»), mipsel («Таволга Терминал»); «Альт Сервер 9» – для x86 (32 и 64 битных), AArch64 (Huawei Kunpeng, ThunderX и другие), ppc64le (YADRO Power 8 и 9, OpenPower), e2k/e2kv4 («Эльбрус»); «Альт Образование 9» – для x86 (32- и 64-битных), AArch64 (NVIDIA Jetson Nano Developer kit, Raspberry Pi 3 и другие). Для каждой архитектуры сборка осуществляется нативно, без использования кросс-компиляции. В ближайших планах «Базальт СПО» – выпуск дистрибутива «Альт Сервер В 9». Бета-версия продукта уже существует и доступна для тестирования на платформах x86_64, AArch64 («Байкал-М», Huawei Kunpeng), ppc64le (YADRO Power 8/9, OpenPower). Также готовятся к выпуску дистрибутивы «Альт Рабочая станция K» со средой KDE Plasma и Simply Linux для домашних пользователей также для разных аппаратных платформ. Пользователи дистрибутивов, построенных на Восьмой платформе (p8), могут обновить систему из ветки p9 репозитория «Сизиф» (Sisyphus). Для новых корпоративных пользователей предусмотрена возможность получения тестовых версий, а частным пользователям традиционно предлагается бесплатно скачать нужную версию ОС «Альт» с сайта «Базальт СПО» или с нового сайта загрузки getalt.ru. Варианты для процессоров «Эльбрус» доступны по письменному запросу подписавшим NDA с АО МЦСТ юридическим лицам. Помимо расширения спектра аппаратных платформ, для дистрибутивов ОС «Альт» версии 9.0 реализован ряд других существенных усовершенствований: В RPM появилась поддержка rpmlib(FileDigests), что позволит устанавливать сторонние пакеты (Яндекс.Браузер, Google Chrome и другие) без переупаковки, и много других улучшений; Офисный пакет LibreOffice реализован в двух версиях: Still для корпоративных заказчиков и Fresh для экспериментаторов и продвинутых пользователей; Доступен единый пакет samba (для обычных рабочих мест, серверов и для контроллеров домена Active Directory); Реализована поддержка актуальных алгоритмов ГОСТ. Работа над портированием дистрибутивов ALT на новые аппаратные платформы продолжается. В частности, запланирован выпуск версий для систем на «Байкал-М» и Raspberry Pi 4.
