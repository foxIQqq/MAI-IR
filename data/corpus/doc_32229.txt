Доступен шестой кандидат в релизы OpenWRT 10.03.1 (RC6)

Доступен шестой кандидат в релизы OpenWRT 10.03.1, дистрибутива Linux для встраиваемых устройств, таких как маршрутизаторы и точки доступа. OpenWRT поддерживает множество различных платформ и предоставляет удобные средства адаптации образов прошивок под конкретные задачи. Загрузить готовые бинарные образы для различных типов устройств можно здесь. Заметные изменения с момента прошлого релиза (10.03.1 RC5): Добавлена поддержка устройств TL-WR740N; Объединены образы для WZR-HP-G300NH и -G301NH; rdate заменён на ntpd для более надёжной синхронизации времени; Поддержка pptp-конфигураций с текстовыми названиями серверов; Поддержка запрета конкретной беспроводной сети; Переделана генерация MAC-адреса для беспроводных виртуальных точек доступа (VAP); Поддержка привязки Dropbear на конкретный сетевой интерфейс; Для PPPoE реализованы опции AC и Service name, реализован обходной путь для задания порядка аргументов; Реализована конфигурация постоянных (lifetime) параметров для 6to4 В межсетевом экране разрешены отрицания (negations), множественные значения и прочие продвинутые конфигурации; Прекращена поддержка платформ Cobalt и Octeon, которые находятся в состоянии непригодном для релиза; Исправлена конфигурация сети по умолчанию на ASUS WL-330gE; Исправлена поддержка WRT160NL некоторых ревизий, флэш-память которых использует более крупные блоки; Исправления, связанные с работой DMA в драйвере ag71xx ethernet; Исправлена обработка multicast-трафика на платформе RDC; Исправлена компиляция ARM SDK; Улучшена генерация индексов ImageBuilder; ненужные рабочие данные opkg отбрасываются; Исправлено обновление endpoint в 6in4 с учётом изменившегося api HE.net; Исправлена генерация правила mtu_fix, используется таблица mangle вместо filter; Улучшена скорость и совместимость в драйвере ath9k и подсистеме mac80211; Ряд исправлений сетевой конфигурации и базовых файлов; Ряд исправлений в веб сервере uhttpd; Исправлен block-extroot на платформе brcm-2.4; Изменены правила для ICMPv6 по умолчанию; Добавлена конфигурация UCI и модуль для LuCI для скрипта wondershaper; Объявлен устаревшим пакет iptables-utils; Исправлена компиляция openl2tp для дистрибутивов без текущего rpcgen; Исправления freeswitch для Backfire.
