Вышла новая версия программы-органайзера для среды GNOME - GTG 0.1.2

Вышел релиз GTG 0.1.2, новой программы-органайзера для среды GNOME. GTG написан на языке Python и позволяет упростить составление планов и проконтролировать выполнения поставленных перед собой задач (TODO-списки). Готовые установочные пакеты подготовлены для Ubuntu, Debian, OpenSuse, ArchLinux, Gentoo, FreeBSD, дополнительно введен в строй PPA-репозиторий. Небольшой обзор GTG можно найти в заметке "Getting things done with Linux to-do list programs", в которой подчеркивается гибкость модульной структуры приложения, позволившей автору в считанные часы разработать бэкенд для хранения данных в CouchDB с репликацией данных между несколькими машинами, вместо стандартного хранения информации в XML файлах. Из альтернатив упоминаются программы Tasque и KOrganizer.
