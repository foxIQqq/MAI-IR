Доступен для загрузки Ubuntu GamePack 13.04

Доступна для загрузки специализированная сборка Ubuntu GamePack 13.04, предоставляющая средства для запуска более чем 1 420 игр, как оригинальных, разработанных специально для платформы GNU/Linux, так и значительного количества игр для MS Windows. Размер iso-образа 1.6 Гб. В составе сборки поставляется около 340 игр различного жанра. В комплект также включены три системы доставки игр и приложений через интернет: Steam (более 140 игр), Desura (более 320 игр) и Djl (более 120 игр). Для запуска игр, изначально поставляемых для платформы Windows из коробки доступны PlayOnLinux и Wine. Для использования online-игр включена поддержка Java.
