Второй бета-выпуск мобильной платформы /e/ доступен для 49 устройств

Гаэль Дюваль (Gaël Duval), создатель дистрибутива Mandrake Linux, сообщил о доступности второй бета-версии мобильной платформы "/e/" (ранее Eelo), сосредоточенной на обеспечении конфиденциальности пользовательских данных. Второй бета-выпуск примечателен переходом на компоненты Android 8 "Oreo" и существенным расширением числа поддерживаемых устройств, для которых подготовлены сборки платформы. Общее число поддерживаемых устройств доведено до 49. В том числе переход на использование драйверов от Android 8 позволил обеспечить поддержку таких смартфонов, как Xiaomi Redmi Note 5 pro, Xiaomi Mi A1, Xiaomi Mi 6, Xiaomi Pocophone F1, OnePlus 5T и Google Pixel XL. Для поддержания прошивки в актуальном виде предлагается система OTA-обновлений и интерфейс, позволяющий контролировать получение и установку системных обновлений. Начались переговоры с некоторыми производителями мобильных устройств о предустановке /e/ на поступающие в продажу смартфоны. Выпуск первого смартфона с предустановленной платформой /e/ запланирован на начало 2019 года. В новом выпуске представлен собственный интерфейс установки приложений и каталог программ. В каталоге предложено около 60 тысяч открытых и бесплатных Android-приложений. Помимо рейтинга, формируемого на основе оценки качества программы пользователями, в каталоге для каждого приложения также вычисляются и отображаются уровни приватности и энергоэффективности. Данные уровни формируются автоматически на основе анализа на наличие кода для отслеживания перемещений, требуемых приложением полномочий и оценки нагрузки на CPU и сеть в процессе работы. Опционально имеется возможность работы с каталогами F-Droid и Yalp Store. Прошивка развивается как ответвление от платформы Android, что позволяет сохранить поддержку Android-приложений и упростить поддержку оборудования. Для взаимодействия с сервисами Google предустановлен пакет microG, который позволяет обойтись без установки проприетарных компонентов Google. При этом вместо сервисов Google проектом используются независимые аналоги. Например, для определения местоположения по Wi-Fi и базовым станциям (без GPS) задействована прослойка на основе Mozilla Location Service. Вместо поисковой системы Google предлагается метапоисковый сервис на основе форка движка Searx, обеспечивающий анонимность отправляемых запросов (опционально можно использовать поисковые системы Qwant и DuckDuckGo). Ключевой особенностью платформы является существенно переработанный интерфейс пользователя, который включает собственный ланчер BlissLauncher, улучшенную систему уведомлений, новый экран блокировки и иное стилевое оформление. В BlissLauncher задействован специально разработанный для проекта набор автоматически масштабируемых пиктограмм и подборка виджетов (например, виджет для показа прогноза погоды). Проектом развивается собственный менеджер аутентификации, позволяющий использовать для всех сервисов единую учётную запись (user@e.email), которая регистрируется в процессе первой установки. По умолчанию предлагаются такие приложения, как почтовый клиент (форк K9-mail с поддержкой OAuth), программа для отправки мгновенных сообщений (Signal), чат (Telegram), система для ведения заметок, планировщик задач, программа для работы с картами Magic Earth. Проектом также подготовлены серверные компоненты с облачным хранилищем на базе открытой платформы NextCloud, обеспечивающие работу таких сервисов, как электронная почта, файловое хранилище, календарь-планировщик, заметки и задачи. Учётную запись можно использовать для получения доступа к своему окружению через Web или на других устройствах. Серверные компоненты основаны на открытом ПО и доступны для установки на подконтрольных пользователю системах (планируется поставка готовых Docker-образов).
