Началось тестирование Adobe Flash Player 10.1 и AIR 2

Компания Adobe http://labs.adobe.com/ выпустила пре-релиз Flash Player 10.1, доступный для платформ Windows, Linux (только 32-битная сборка) и MacOS X. Релиз Flash Player 10.1 планируется выпустить в первой половине 2010 года, это будет первый выпуск базирующийся на инициативе Open Screen Project. Кроме стандартных настольных систем Flash Player 10.1 будет доступен и для мобильных платформ, таких как Windows Mobile, Symbian S60, Google Android и Palm webOS. Из новшеств можно отметить задействование механизмов аппаратной акселерации GPU при выводе данных на экран и при декодировании h.264 видео. Для мобильной сборки ожидается появление поддержки мультитач-навигации, виртуальной клавиатуры, управления экранными жестами и API для доступа к акселерометру. В десктоп-версии появится режим приватного серфинга без оставления следов в логах, кэше и на локальном диске. Кроме того, выпущена первая бета-версия Adobe Air 2.0, инструментария для создания самодостаточных многоплатформенных десктоп-приложений на базе технологий Adobe Flash, Adobe Flex, HTML и Ajax. SDK и Runtime пакеты Air 2-beta доступны в том числе и для Linux. Официально поддерживаются дистрибутивы Fedora Core 11, Ubuntu 9.04 и openSUSE 11.1. Основные новшества Air 2.0: Поддержка определения подключения к системе съемных накопителей; Расширенные сетевые функции, такие как защищенные сокеты, поддержка UDP и возможность создания принимающих соединения серверных приложений; Поддержка интеграции в приложение нативного кода для заданной архитектуры; Возможность задания приложений-обработчиков вызываемых при попытке открытия файлов определенных типов; Поддержка мультитач навигации и управления через экранные жесты; API для низкоуровневого доступа к получаемым через микрофон данным; Интеграция обновленной версии Webkit с расширенной поддержкой HTML5/CSS3; Механизм глобальной обработки ошибок; Кросс-платформенная система для организации печати; Приведение модели безопасности и поддержки в соответствие с требованиями государственных и промышленных стандартов.
