Исходные тексты игрового движка CryEngine опубликованы на GitHub

Компания Crytek разместила в открытом доступе на GitHub исходные тексты игрового движка CryEngine, которые ранее распространялись только для зарегистрированных разработчиков на платной основе. Теперь код доступен для изучения всеми желающими и поставляется в рамках специального лицензионного соглашения, разрешающего устанавливать и собирать движок, разрабатывать с его помощью игры, вносить исправления в соответствии с документацией. Запрещается распространять движок среди третьих лиц, использовать CryEngine для разработки не игровых приложений (в том числе для симуляции и научного применения), удалять технические методы защиты.
