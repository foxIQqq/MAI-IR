Релиз медиацентра MythTV 32.0

После года разработки состоялся релиз платформы для создания домашнего медиацентра MythTV 32.0, позволяющей превратить настольный ПК в телевизор, видеомагнитофон, музыкальный центр, альбом с фотографиями, станцию для записи и просмотра DVD. Код проекта распространяется в рамках лицензии GPL. Одновременно сформирован выпуск отдельно развивающегося web-интерфейса MythWeb для управления медиацентром через web-браузер. Архитектура MythTV базируется на разделении бэкенда для хранения или захвата видео (IPTV, DVB-карты и т.п.) и фронтэнда для отображения и формирования интерфейса. Фронтэнд может работать одновременно с несколькими бэкендами, которые могут быть запущены как на локальной системе, так и на внешних компьютерах. Функциональность реализуется через плагины. В настоящее время доступно два набора плагинов - официальный и неофициальный. Спектр возможностей, охватываемых плагинами достаточно широк - от интеграции с различными online-сервисами и реализации web-интерфейса для управления системой по сети до средств для работы с web-камерой и организации видеосвязи между ПК. В новой версии в кодовую базу внесено около 1300 улучшений, среди которых: Переписана реализация API Services. Добавлена поддержка библиотеки libzip. Реализована возможность записи с использованием кодека HEVC / H.265. Добавлена экспериментальная поддержка процедурного текстурирования. Добавлена поддержка FreeSync и GSync (Variable Refresh Rate/VRR). Улучшена поддержка графического API Vulkan.
