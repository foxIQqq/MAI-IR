Релиз GNOME 2.30

Официально объявлен релиз десктоп-окружения GNOME 2.30, последний в рамках стабильной ветки 2.x - в сентябре вниманию пользователей будет представлен GNOME 3.0. Основные новшества GNOME 2.30: Для упрощения процесса тестирования GNOME 3.0, в состав включена последняя версия оболочки грядущего рабочего стола GNOME Shell (нужно запустить "gnome-shell --replace"), объединившей в себе такие сущности, как GNOME Panel и оконный менеджер. Оболочка обеспечивает работу таких функций, как запуск приложений, доступ к документам, выполнение апплетов и переключение окон; В файловом менеджере Nautilus добавлена поддержка двухпанельного режима просмотра файлов (активируется через "Split view" в меню View, см. видеодемонстрацию. В каждой панели сохранена возможность использования табов), реорганизован и используется по умолчанию режим браузера; Расширены возможности программы для мгновенного обмена сообщениями, организации VoIP-переговоров и видеочата - Empathy. В чате реализованы частоиспользуемые IRC-команды, такие как /join, /topic и /nick. Добавлена возможность создавать защищенные паролем комнаты. Для отправки файлов между контактами их теперь можно просто перетаскивать мышью. В окно чата добавлена строка для быстрого поиска. Переработан диалог для управления параметрами аккаунта, который теперь реализован в виде отдельного процесса, который можно вызывать из меню настроек Gnome без запуска Empathy. Добавлен виджет для упрощения настройки аккаунта в Facebook. Перерисованы некоторые пиктограммы. Полностью переработан дизайн GUI-интерфейса для выполнения функций по администрированию системы, добавлен новый диалог для создания пользователей, добавлена поддержка создания шифрованных домашних директорий, интегрирована поддержка аутентификации через PolicyKit; Улучшения в просмотрщике документов Evince: улучшены возможности по выводу документов на печать, добавлена вкладка с информацией о лицензии просматриваемого документа, реализована поддержка прикрепленных к PDF файлу аннотаций, добавлен режим инвертирования цветов, реализована поддержка cbt-комиксов, переработан режим презентации; Time Tracker (hamster-applet, аплет для планирования своего времени): добавлена возможность привязки тегов к задачам, расширены средства категоризации, фильтрации элементов по различным критериям. Добавлена поддержка обособленного вызова через меню "Applications/Accessories/Time Tracker"; Удален апплет для индикации состояния клавиатуры (Gnome Keyboard Indicator). При выборе нескольких раскладок статус клавиатуры теперь автоматически отображается в области уведомлений при помощи gnome-settings-daemon. Переработан GUI интерфейс для настройки параметров клавиатуры; Добавлена возможность установки шрифта через двойной клик на файл со шрифтом в nautilus; Улучшения в текстовом редакторе gedit: в диалог записи файлов добавлена опция для выбора типа перевода строки в файле (LF|CR|CR_LF); добавлена возможность проверки орфографии в комментариях и строках внутри открытого файла с исходными текстами на различных языках программирования; улучшена интеграция с Mac OS X; при сборке в Windows теперь работают плагины на Python; В менеджере архивов File Roller добавлена возможность установки недостающих пакетов через PackageKit, которые требуются для открытия заданного архива; В Gnome Terminal реализована поддержка неограниченного скроллинга, улучшена поддержка переменных окружения для задания прокси; В программе для записи CD/DVD Brasero добавлена возможность записи DTS wav-файлов, реализован новый плагин cdda2wav для более точного копирования звуковых дисков, произведена интеграция с PackageKit, добавлена поддержка Tracker; В VNC-клиенте Vinagre добавлен режим ожидания соединения (vncviewer -listen), активирована поддержка сжатия JPEG (кодирование с потерями), реализована возможность организации туннелей VNC поверх SSH; В программе для ведения заметок Tomboy добавлена поддержка автоматической фоновой синхронизации заметок, что позволяет организовать работу с одним и тем же набором заметок на ноутбуке, домашнем и рабочем ПК. Появилась возможность копирования и вставки через буфер обмена текста с HTML-разметкой. В gvfs добавлена поддержка работы с устройствами iPhone и iPod Touch. В web-браузере Epiphany добавлена возможность сохранения вводимых паролей через стандартный механизм gnome-keyring. В настойки добавлена опция для включения/выключения поддержки плагинов, таких как Flash или Java. При получении некорректного или неподтвержденного SSL-сертификата браузер отныне выводит специальное предупреждение. Несколько дополнительных настроек теперь можно изменить через GConf, например, поменять User Agent или поисковую систему по умолчанию. Среди новых расширений к Epiphany отмечено дополнение html5tube, позволяющее на лету конвертировать Flash-ролики с YouTube в HTML5-представление, не требующее для просмотра наличия Flash-плагина. Дополнение tab-key-tab-navigate позволяет использовать комбинацию Ctrl+Tab для изменения текущего таба. Возвращена работа следующих возможностей Epiphany: предварительный просмотр перед печатью; возможность сохранения изображений и других объектов на странице через правый клик мышью; отображение иконок Favicon; вставка из буфера обмена при нажатии правой кнопки мыши; поддержка вывода контекстного меню. Кроме того, GNOME напоминает о продолжении акции по сбору пожертвований для оплаты работы системного администратора, поддерживающего инфраструктуру проекта в режиме полного рабочего дня. Поддержать проект можно купив футболку или кружку с логотипом GNOME в специально созданном интернет-магазине или перечислить пожертвование на странице gnome.org/friends.
