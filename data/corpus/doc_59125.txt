Новый выпуск Simply Linux RV для устройств на базе архитектуры RISC-V

Вышел релиз экспериментальной сборки дистрибутива Simply Linux RV (20230422) для архитектуры riscv64. Дистрибутив представляет собой простую в работе систему с классическим рабочим столом на основе Xfce, предоставляющую полную русификацию интерфейса и большинства приложений. Развивающая дистрибутив компания «Базальт СПО» входит в международное сообщество RISC-V и ведёт работу по поддержке различных плат RISC-V64. Для загрузки подготовлены два образа, сформированные на основе пакетов из репозитория Sisyphus riscv64: образ для плат HiFive Unmatched и HiFive Unleashed, StarFive VisionFive, также подходящий для QEMU. образ для одноплатного компьютера StarFive VisionFive 2 с ядром sf-un. Среди изменений в новом выпуске: Добавлена поддержка одноплатного компьютера StarFive VisionFive 2 с SoC JH7110 на архитектуре RISCV64GC. Обновлены обои рабочего стола, темы интерфейса. В качестве эмулятора терминала по умолчанию задействован xterm. Обновлены версии пакетов: ядро Linux 6.2.11 (основной образ); ядро Linux 5.15.0 (StarFive VisionFive 2); openssl 1.1.1; xorg-server 21.1.8; x11vnc 0.9.16; Firefox 112.0.1; Thunderbird 102.10.0; LibreOffice 7.4.2; XFCE 4.18.3; ca-certificates-digital.gov.ru 1.1; libnss-role 0.5.6. Из планов по дальнейшей поддержке платы VisionFive V2 отмечается ускорение декодирования видео, обеспечение устойчивого воспроизведения 3D-графики и работы других IP-блоков SoC, которым пока требуются проприретарные драйверы, интеграция поддержки VisionFive V2 в основной состав ядра Linux.
