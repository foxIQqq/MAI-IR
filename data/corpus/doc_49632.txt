Выпуск видеоредакторов Shotcut 18.11 и DaVinci Resolve 15.2

Состоялся релиз видеоредактора Shotcut 18.11 (следом выпущено обновление 18.11.18), который развивается автором проекта MLT и использует данный фреймворк для организации редактирования видео. Поддержка форматов видео и звука реализована через FFmpeg. Возможно использование плагинов с реализацией видео и аудио эффектов, совместимых с Frei0r и LADSPA. Из особенностей Shotcut можно отметить возможность многотрекового редактирования с компоновкой видео из фрагментов в различных исходных форматах, без необходимости их предварительного импортирования или перекодирования. Имеются встроенные средства для создания скринкастов, обработки изображения с web-камеры и приёма потокового видео. Для построения интерфейса применяется Qt5. Код написан на C++ и распространяется под лицензией GPLv3. В новом выпуске: Добавлен экран с параметрами для создания нового проекта и списком недавно открытых проектов; В Linux реализована возможность использования VA-API для аппаратного ускорения кодирования видео. В macOS добавлена поддержка API VideoToolbox; В раздел экспорта добавлен расширенный режим, опция для включения аппаратного ускорения кодирования видео и преднастройки для формата Ut Video; Добавлена поддержка наложения 10- и 20-пиксельных сеток при воспроизведении; Добавлен фильтр Spot Remover для удаления пятен; Добавлено меню "View > Scopes > Video Waveform"; Добавлена поддержка отображения видео в квадратной области с качеством 1080p (30 и 60 FPS); Для macOS добавлен SDK и реализована поставка сборки, заверенной цифровой подписью; В фильтрах вращения и масштабирования максимальный масштаб увеличен до 500%; Улучшена поддержка форматов изображений DDS, ICO и WebP; Добавлен фильтр визуализации звуковой волны (Audio Waveform Visualization); Добавлен фильтр, отображающий таймер в форме "MM:SS.SS"; В диалоге выбора цвета Color Grading добавлена возможность изменения цвета колесом мыши. Дополнительно, можно отметить выпуск проприетарной системы цветокоррекции и нелинейного монтажа DaVinci Resolve 15.2, используемой многими известными голливудскими киностудиями в процессе производства фильмов, сериалов, рекламных роликов, телепрограмм и видеоклипов. DaVinci Resolve объединяет в одном приложении средства для монтажа, цветоустановки, наложения звука, финальной обработки и создания конечного продукта. Сборки DaVinci Resolve подготовлены для Linux, Windows и macOS. Для загрузки требуется регистрация. В новой версии: Оптимизирована работа с монтажной линейкой при выполнении операций редактирования и подгонки; Добавлены средства анимации при действиях с клипами, которые упрощают отслеживание влияния выполняемых операций на конечный результат; Добавлена команда Show Duplicate Frames для выявления идентичных фрагментов; В списке клипов реализована индикация использования на текущей монтажной линейке; Добавлена опция Flatten Unused Clips для удаления неиспользуемых клипов со шкалы времени; Увеличена эффективность редактирования видео на нескольких монтажных линейках. Для обработки составных элементов шкалу времени и комбинированные клипы теперь можно загрузить как исходное изображение; В элементах экранного управления, окне Inspector и метаданных на странице Edit обеспечено автоматическое обновление информации о находящемся под курсором клипе, что позволяет обойтись без ручного выбора клипа для изменения параметров; Предложен наглядный интерфейс для настройки горячих клавиш; Добавлены новые плагины ResolveFX: Blanking Fill (автоматически заполняет горизонтальные или боковые полосы дисплея расфокусированным материалом) и Beauty (мягкое и естественное сглаживание текстуры и пятен на поверхностях); Расширено число функций обработки звука и плагинов на странице Fairlight. В том числе добавлен плагин Stereo Fix, автоматизирующий наиболее часто выполняемые операции при назначении каналов, например, позволяющий устранять проблемы с двухканальным монофоническим звуком, смонтированным на стереодорожках. Реализованы группы VCA для объединения нескольких каналов и их одновременного усиления. Добавлены средства для спектрального анализа звука в режиме реального времени.
