Новая версия медиаплеера Exaile 3.3.0

После более чем года разработки вышел релиз медиаплеера Exaile 3.3.0, истоки которого начинались с попытки создания аналога Amarok 1.4, написанного на языке Python с использованием GTK+ и GStreamer. При подготовке нового выпуска Exaile разработчики приняли решение избавиться от нуля в номере версии и выпустили релиз 3.3.0, вместо 0.3.3. Код проекта распространяется под лицензией GPLv2. Exaile может использоваться для управления музыкальной коллекцией, предоставляет удобный интерфейс на основе вкладок (разные списки воспроизведения могут быть открыты в разных вкладках), обеспечивает гибкую систему плагинов и поддерживает такие функции, как автоматическое получение текстов песен и обложек альбомов через разные сетевые сервисы, интеграция с Last.fm, возможность прямой работы с портативными медиа-плеерами, прослушивание потокового сетевого радио. Среди новшеств, добавленных в Exaile 3.3.0: Устранены проблемы в реализации многопоточной работы, которые приводили к подвисаниям в процессе воспроизведения и запуска. Добавлена поддержка загрузки обложек альбомов в асинхронном режиме; Улучшена поддержка импорта носителей в режиме drag&drop Поддержка различных источников при импорте и экспорте композиций и плейлистов, в том числе сетевых ресурсов; При удалении композиции теперь по умолчанию помещаются в корзину без реального удаления файлов; Значительно улучшена поддержка работы на платформе Windows, добавлен инсталлятор для Windows; Переработанный менеджер обложек альбомов; Автоматическая прокрутка содержимого при перетаскивании элементов мышью; Изменение оформления диалога с параметрами композиции; Возможность добавления произвольных меток на шкалу воспроизведения для быстрого перехода по сохранённым позициям; Поддержка восстановления случайно закрытых вкладок; Полностью новая система меню В свойства композиции и плейлисты добавлено поле с комментарием; Полностью переработанный API для работы со списками воспроизведения; По умолчанию включен управляющий DBus-сервис, при помощи которого можно управлять плейлистами и воспроизведением; Поддержка тегов с обложками для файлов MP3/MP4; Новые плагины: A-B Repeat - повтор воспроизведения участка между двумя метками; BPM Counter - ручная установка BPM для композиции; Group Tagger - добавление собственных меток для композиций для их быстрого поиска и фильтрации по тегам; History - ведение истории проигранных композиций с возможностью экспорта в виде списка воспроизведения; Inhibit Suspend - запрет перехода системы в спящий режим в процессе воспроизведения; Last.fm Loved Tracks - пометка любимых композиций в Last.fm; Main Menu Button - основное меню в форме одной кнопки, как в Chrome; Wikipedia - загрузка из википедии данных о текущем музыканте.
