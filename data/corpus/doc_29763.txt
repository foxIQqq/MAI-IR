Вышел Linuх-дистрибутив VortexBox 1.8

Вышел релиз Linuх-дистрибутива VortexBox 1.8, основанного на Fedora Linux и предназначенного для быстрого создания серверов для организации музыкального вещания в сети, сетевых хранилищ музыки или для превращения любого компьютера со звуковой картой в интернет-радио. Размер iso-образа дистрибутива составляет 619 Мб. В новой версии улучшена поддержка риппинга DVD-дисков, позволяющая для создания копии диска просто вставить его в привод и дистрибутив автоматически преобразует записанный на него фильм в формат MKV, предоставив возможность его просмотра по сети. В отличии от прошлой версии функции риппинга теперь научились корректно идентифицировать вставленный DVD и формировать название на основании выделенной информации. Добавлена возможность фильтровать треки по их продолжительности (например, для копирования с DVD только TV-шоу определенной продолжительности). Дополнительно добавлена возможность создания MP4-зеркала для DVD, создавая при этом XML-файлы для упрощения воспроизведения в Windows Media Center. Обновлены версии входящих в комплект программ, например, код сервера SqueezeBox обновлен до версии 7.4.3. В качестве музыкального плеера по умолчанию в дистрибутиве используется программа собственной разработки VortexBox Player, представляющая собой надстройку над mpd (Music Player Daemon). Программа отличается поддержкой аппаратных интернет-плееров SqueezeBox и Sonos, может работать со SqueezeBox-совместимыми серверами, поддерживает вывод звука как на локальную звуковую карту, так и на аудиоустройства, подключённые через USB и S/PDIF порты. Управление работой системы производится через удобный web-интерфейс. Из возможностей можно отметить функцию автоматического копирования и перекодирования данных при вставке в привод аудио-CD, композиции сохраняются локально в форматах FLAC или MP3 и затем становятся доступными для потокового вещания. При необходимости для музыкальных файлов автоматически заполняются ID3-тэги и загружаются изображения обложек альбомов (через обращение к музыкальным онлайн-сервисам). Поддерживается возможность организации работы файлового хранилища музыки, доступного по протоколам SMB и NFS. Дистрибутив может выступать в качестве бэкенд-сервера для XBMC (XBOX Media Center), AppleTV и большого числа аппаратных интернет-плееров. Исходные тексты всех созданных в рамках проекта компонентов распространяются в рамках лицензии GPLv3 и доступны в виде пакетов для Fedora Linux.
