Новые дистрибутивы: Absolute Linux 12.2.2 и Igelle PC/Desktop v0.6.0

Два новых релиза малоизвестных Linux дистрибутивов для маломощных систем: Absolute Linux 12.2.2 - дистрибутив на основе Slackware 12.2, снабженный графическим интерфейсом на базе оконного менеджера IceWM, рабочего стола ROX Desktop и файлового менеджера ROX-Filer. Дистрибутив распространяется на двух CD и включает в себя достаточно полную подборку программ, способных работать на маломощных системах (CPU Pentium 200 и 64 Мб ОЗУ). В комплекте: Firefox, OpenOffice, Pidgin, GIMP, WPClipart, Thunderbird, K3B, Frostwire P2P, Deluge BitTorrent, Linux ядро 2.6.28.7. При установке в качестве файловой системы по умолчанию используется ext4. Igelle PC/Desktop v0.6.0 - самобытный Linux дистрибутив, разработанный с нуля. Не требователен к аппаратному обеспечению. Поставляется в виде LiveCD с возможностью установки на диск. Для управления пакетами используется упрощенный пакетный менеджер ige, сами пакеты представлены в "deb" формате. Для сборки из исходных текстов используется система Melas, позволяющая быстро пересбобрать весь дистрибутив из исходных текстов (контроль зависимостей сведен к жесткому определению порядка сборки). Размер iso-образа 386 Мб. Поддерживаемые платформы: ARM, MIPS и x86. В комплект входят Linux ядро 2.6.27.19, glibc 2.8, GCC 4.3.2, X.Org 1.6, Skype, Flash player, OpenOffice.org, Whaaw media player. Графический интерфейс построен на основе оконного менеджера Metacity, панели BarPanel, менеджера сессий Slim и файлового менеджера PCManFM.
