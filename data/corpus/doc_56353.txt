Выпуск текстового редактора GNU nano 6.0

Состоялся релиз консольного текстового редактора GNU nano 6.0, предлагаемого в качестве редактора по умолчанию во многих пользовательских дистрибутивах, разработчики которых считают vim слишком сложным для освоения. В новом выпуске Добавлена опция "--zero", скрывающая заголовок, строку состояния и область подсказки для высвобождения всего экранного пространства для области редактирования. По отдельности заголовок и строку состояния можно скрыть и вернуть командой M-Z. Предоставлена возможность определения цветов в web-подобном шестнадцатеричном формате "#rgb". Для тех, кто не любит задавать цвета цифрами, предложено 14 текстовых названий цветов: rosy, beet, plum, sea, sky, slate, teal, sage, brown, ocher, sand, tawny, brick и crimson. По умолчанию активирована возможность приостановки редактирования с возвращением в командную строку через горячие клавиши ^T^Z, без необходимости запуска с опцией -z (--suspendable) или включения настройки 'set suspendable'. Счётчик слов, показываемый командой M-D, теперь зависит от параметра "--wordbounds", при установке которого подсчёт слов начинает соответствовать утилите 'wc', а иначе обрабатывая символы пунктуации как пробелы. Обеспечено применение жёсткой обрезки (hard-wrapping) по границе строки при вставке из буфера обмена. В состав включён файл с описанием синтаксиса YAML.
