Сервис установки программ Ninite стал доступен для Linux

Ninite, популярный сервис быстрой установки программ для Windows, реализовал в тестовом режиме поддержку платформы Linux. Суть сервиса сводится к предоставлению возможности выбора через web-интерфейс набора интересующих приложений и нажатии кнопки для формирования специального инсталлятора, после запуска которого автоматически загрузится и установится выбранный комплект программ. Текущая тестовая Linux-версия Ninite поддерживает только Debian-совместимые дистрибутивы поддерживающие установку deb-пакетов. Установка программ производится классическим для Linux способом - установочный скрипт подключает необходимые репозитории и из них устанавливает программы. Из присутствующих в каталоге приложений можно отметить последние версии Chrome, Thunderbird, Pidgin, Skype, OpenOffile.org, Gimp, Inskape, Picasa, VLC, DropBox, Gnome-Do, AIR, Wine и набор мультимедиа кодеков.
