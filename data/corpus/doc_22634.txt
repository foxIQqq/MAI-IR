Релиз Firefox 3.5.1. Firefox празднует 1 млрд. загрузок

Вышел первый корректирующий релиз Firefox 3.5.1, в котором устранена критическая уязвимость и исправлено около десяти ошибок, влияющих на стабильность работы. Решена проблема с излишне долгой загрузкой браузера на некоторых системах с Windows, исправлена и ошибка из разряда "нарочно не придумаешь" - при открытии сайта www.hp.com на платформе Sparc наблюдался крах браузера. Исправленная уязвимость позволяла выполнить код злоумышленника при открытии специально оформленной страницы. Проблема была вызвана ошибкой в Just-in-Time (JIT) компиляторе нового JavaScript движка. Опасность уязвимости усугубляется наличием в открытом доступе готового эксплоита. Кроме того, разработчики сообщили о начале акции по празднованию дня, когда будет преодолен рубеж в один миллиард загрузок браузера. По предварительной оценке это событие должно произойти в августе. В настоящий момент счетчик загрузок насчитывает 973 564 286. В заключение еще несколько событий, связанных с Firefox: Mozilla проводит конкурс дизайнеров "People and robots" в котором требуется отразить как технологии делают жизнь лучше. Главный приз $1000. Опубликована заметка о прогрессе развития проекта Test Pilot, в рамках которого ведется работа над инфраструктурой для оценки предпочтений пользователей Firefox, с целью формирования портрета типичного пользователя; Вышла версия пакета Google Gears для создания offline web-приложений, с поддержкой новых возможностей Firefox 3.5; Вышел релиз Jetpack 0.3, технологии по упрощению создания расширений для Firefox Подведены итоги конкурса по усовершенствованию интерфейса пользователя Firefox, связанного с работой табов и оптимизацией просмотра нескольких сайтов в одном браузере.
