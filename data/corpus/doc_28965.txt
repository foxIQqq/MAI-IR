Обновление Java SE 6 Update 23

Вышел двадцать третий корректирующий релиз Java SE Runtime (JRE) 6 и Java SE Development Kit (JDK) 6, в котором отмечено 213 исправлений, направленных на увеличение стабильности и производительности. Исправленные ошибки не связаны с проблемами безопасности, последним корректирующим релизом, направленным на устранение узявимостей, был выпуск Java SE 6 Update 22. В состав Java SE 6 Update 23 включена виртуальная машина Java Hotspot VM 19.0, в которую были внесены связанные с повышением стабильности и надежности изменения. В поставку также входит профайлер Java VisualVM 1.3.1, построенный на кодовой базе платформы NetBeans 6.9.1.
