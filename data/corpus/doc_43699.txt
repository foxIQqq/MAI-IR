Bq Mobile готовит первый планшет на базе Ubuntu Linux

Испанская компания Bq Mobile, уже выпускающая смартфоны BQ Aquaris на базе платформы Ubuntu Touch, подтвердила информацию о подготовке к выпуску первого планшета, официально поставляемого с Ubuntu. Планшет Bq Aquaris M10 Ubuntu Edition впервые будет продемонстрирован на выставке Mobile World Congress 2016, которая состоится 22-25 февраля в Барселоне. Предполагается, что аппаратная начинка планшета будет идентична недавно анонсированному устройству Aquaris X10, который укомплектован 10-дюймовым сенсорным экраном (1080p), 64-разрядным 4-ядерным процессором MediaTek MTK 8163 (ARM Cortex-A53, Mali GPU), 2 Гб ОЗУ, 16 Гб MMC Flash и аккумулятором ёмкостью 7280 mAh. Поступление устройства в продажу намечено на апрель, информация о стоимости не разглашается (BQ Aquaris M10 на базе Android стоит 239 евро). Отличительной особенностью нового планшета Bq Aquaris M10 Ubuntu Edition станет интеграция фреймворка Snappy Ubuntu Personal и реализация переключения в десктоп-режим. Устройство можно будет подсоединить к монитору/телевизору через порт HDMI, подключить клавиатуру и мышь, и получить полноценную рабочую станцию. Рабочее окружение при работе в режиме настольной системы построено на основе штатной оболочки Unity 8 и обычных настольных приложений, используемых в сборке Ubuntu для десктоп-систем, в том числе приложений, рассчитанных на работу с X-сервером, таких как GIMP и LibreOffice. Запуск X11-приложений организован при помощи прослойки XMir, при этом непосредственно X11-приложения выполняются в отдельном Snappy-контейнере. Контейнер построен при помощи пакета Libertine, позволяющего создавать изолированные окружения, предназначенные для установки и выполнения программ из deb-пакетов. Устанавливаемые в окружении Libertine программы с графическим интерфейсом на базе X11 интегрируются в интерфейс на базе Unity 8 через специальную прослойку Puritin и доступны через область "Puritin Scope". Поставляемая в мобильных устройствах сборка Ubuntu Touch по умолчанию поставляется с графическим стеком на базе Mir, Qt 5 и Unity 8, и основана на наработках проекта Ubuntu Personal, в котором применяется технология Snappy с возможностью атомарного обновления образа всей системы. Для организации установки дополнительных программ в один клик, без манипуляций с подключением репозиториев, задействован формат пакетов Click. Для поддержки оборудования в Ubuntu Touch используются компоненты платформы Android (CyanogenMod), которые подключаются при помощи библиотеки libhybris c изоляцией в контейнере LXC.
