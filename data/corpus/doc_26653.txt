Увидел свет свободный LDAP-сервер Mandriva Directory Server 2.4.0

Спустя два года с момента выпуска 2.3.0 компания Mandriva представила релиз сервера директорий Mandriva Directory Server 2.4.0, свободно распространяемого в рамках лицензии GPL. Для управления данными в LDAP-каталоге поставляется специальный web-интерфейс MMC (Mandriva Management Console), поддерживающий управление пользователями и группами, настройку аккаунтов в SAMBA, управлением совместным доступом, настройку инфраструктуры печати на базе CUPS, доставки почты (Postfix) и конфигурирование прокси (Squid) и служб DNS/DHCP. Для организации управления LDAP, SAMBA и SQUID представлен специальный API для разработки дополнений на языке Python. В новой версии добавлен модуль для аудита прав доступа, позволяющий администратору отследить и отразить в отчетах все действия, выполненные через web-интерфейс MMC, включая изменения связанные с LDAP, модификацией файлов и перезапуском сервисов. Кроме того добавлен модуль для определения политики формирования паролей в LDAP, позволяющий задать такие обязательные к выполнению параметры как длина пароля, время действия, уровень сложности. Из разработанных силами сообщества модулей представлены модули для массового импорта и обновления параметров пользователей из CSV-файлов, а также для определения сетевых и дисковых квот. Дополнительно отмечаются функция по управлению публичными OpenSSH-ключами пользователей в LDAP, поддержка LDAP TLS и возможность хранения данных PowerDNS в LDAP. Начиная с версии 2.4 пакет разделен на две составляющие: "MMC-CORE" - фреймворк, обеспечивающий работу управляющей web-консоли, также используемый в проекте Mandriva Pulse 2, и "MDS" - содержащий в себе все связанные с сервером директорий модули.
