Apple на пути к удалению Java из состава Mac OS X

Пакет с Java для платформы Mac OS X 10.6, развиваемый компанией Apple, объявлен устаревшим. Иными словами компания Apple больше не будет поддерживать Java для Mac OS X на должном уровне и не исключает возможности его удаления из поставки будущих версий Mac OS X. Ситуацию усугубляет тот факт, что оригинальный пакет Java SE от компании Oracle недоступен для платформы MacOS X. Более жесткие меры предприняты по отношению к Flash-плагину, который уже перестал предустанавливаться Apple на представленной несколько дней назад новой модели ноутбука MacBook Air (ранее Flash входил в поставку Mac OS X).
