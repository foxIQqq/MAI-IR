Релиз дистрибутива Tails 4.0

Представлен релиз специализированного дистрибутива Tails 4.0 (The Amnesic Incognito Live System), основанного на пакетной базе Debian и предназначенного для обеспечения анонимного выхода в сеть. Анонимный выход в Tails обеспечивается системой Tor. Все соединения, кроме трафика через сеть Tor, по умолчанию блокируются пакетным фильтром. Для хранения пользовательских данных в режиме сохранения пользовательских данных между запусками применяется шифрование. Для загрузки подготовлен iso-образ (1 ГБ), способный работать в Live-режиме. Основные изменения: Осуществлён переход на пакетную базу Debian 10 "Buster". Применены накопившиеся исправления проблем с безопасностью; Менеджер паролей KeePassX заменён на более активно развиваемый сообществом форк KeePassXC; До версии 1.3.2 обновлено приложение OnionShare, позволяющее безопасно и анонимно передавать и получать файлы, а также организовать работу публичного сервиса для обмена файлами. Переход на ветку OnionShare 2.x пока отложен; Tor Browser обновлён до версии 9.0 в которой при изменении размера окна обеспечено отображение серой рамки (letterboxing) вокруг содержимого веб-страниц. Данная рамка не позволяет сайтам идентифицировать браузер по размеру окна. Содержимое значка Onion перенесено из панели в меню "(i)" вначале адресной строки и в кнопку создания новой идентичности на панели; Инструмент очистки метаданных MAT обновлён до выпуска 0.8.0 (ранее поставлялась версия 0.6.1). MAT больше не поддерживает собственный графический интерфейс, а поставляется только в форме утилиты командной строки и дополнения к файловому менеджеру Nautilus. Для очистки метаданных в Nautilus теперь достаточно вызвать для файла контекстное меню и выбрать "Удалить метаданные"; Задействовано свежее ядро Linux 5.3.2. Улучшена поддержка оборудования (добавлены новые драйверы для Wi-Fi и графических карт). Добавлена поддержка устройств с интерфейсом Thunderbolt; Обновлены версии большинства программ, включая: Electrum 3.3.8; Enigmail 2.0.12; Gnupg 2.2.12; Audacity 2.2.2.2; GIMP 2.10.8; Inkscape 0.92.4; LibreOffice 6.1.5; git 2.20.1; Tor 0.4.1.6. Из базовой поставки удалён Scribus (можно установить из репозитория, используя интерфейс установки дополнительного ПО; Улучшен интерфейс первичной настройки после первого входа в систему (Tails Greeter). Упрощена начальная настройка для пользователей, не владеющих английским языком. В диалоге выбора языка проведена чистка языков, оставлены только языки с достаточным объёмом переводов. Упрощён выбор раскладки клавиатуры. Устранены проблемы с открытием доступных страниц помощи на языках, отличных от английского. Налажена настройка форматов. Обеспечено игнорирование дополнительных настроек после нажатие кнопок "Отмена" или "Назад"; Проведена оптимизация производительности и потребления памяти. Время загрузки сокращено на 20%, а потребность в оперативной памяти уменьшена примерно на 250 МБ. Размер загрузочного образа сокращён на 46 МБ; Переработана экранная клавиатура, которая стала проще в использовании; Добавлена возможность отображение пароля постоянного хранилища при его создании. Добавлена поддержка подключения к сети через iPhone, подключённый через USB-порт (USB Tethering); В документацию добавлены новые руководства по безопасному удалению всех данных с устройства, включая USB-накопители и SSD-накопители, а также по созданию резервных копий постоянного хранилища; Home launcher удалён с рабочего стола. В Pidgin удалены предлагаемые по умолчанию учётные записи; Решена проблема с открытием разделов с данными Tails с других USB-накопителей.
