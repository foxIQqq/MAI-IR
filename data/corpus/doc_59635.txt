Выпуск Bazzite 1.0, редакции Fedora Silverblue для любителей компьютерных игр

Представлен первый выпуск проекта Bazzite, развивающего специализированную сборку Fedora Linux 38, оптимизированную для запуска компьютерных игр. Сборка основана на технологиях Fedora Silverblue, использует для формирования базовой системы OSTree-репозитории ublue-os без разделения на отдельные пакеты, а обновление производит в атомарном режиме через замену системного образа. Установочный образ сформирован для архитектуры x86_64 (685 MB). Особенности сборки: Предустановка проприетарных драйверов NVIDIA. Поддержка аппаратного ускорения декодирования видео H264. Полная поддержка runtime AMD ROCM OpenCL/HIP. Полная поддержка DisplayLink. Включение в состав разработанных компанией Valve тем оформления для KDE, применяемых в SteamOS. Включение в базовую поставку прослоек LatencyFleX, vkBasalt, MangoHud и OBS VkCapture. Поддержка Wallpaper Engine. Включение в базовую поставку Distrobox, инструментария для вложенного запуска других дистрибутивов. Для Distrobox подготовлены образы на базе Arch Linux со Steam и Lutris. Включение сервиса duperemove для сокращения дискового пространства за счёт дедупликации повторяющихся окружений для Wine. Включение в состав сервиса автоматической оптимизации System76 Scheduler, динамически настраивающего параметры планировщика задач CFS (Completely Fair Scheduler) и меняющего приоритеты выполнения процессов для снижения задержек и обеспечения максимальной производительности процесса, с которым в текущий момент работает пользователь. Задействование по умолчанию предложенного компанией Google алгоритма управления сетевой перегрузкой BBR для TCP. Включение в базовую поставку приложения Input Remapper для настройки поведения устройств ввода. Добавление мастера первого запуска, позволяющего установить дополнительные приложения и изменить настройки. Например, можно установить CoreCtrl для изменения настроек системы в привязке к приложениям и GreenWithEnvy для разгона видеокарт NVIDIA. В качестве опции предоставлена возможность использования пакетного менеджера Nix. Включение в базовую поставку Waydroid, окружения для запуска Android-приложений. Включение в состав драйвера xpadneo для беспроводных контроллеров Xbox One. Включение в состав драйверов i2c-piix4 и i2c-nct6775 для управления встроенной в материнские платы подсветкой, используя пакет OpenRGB. Включение в состав драйвера GCAdapter_OC для разгона до 1000hz контролллеров Nintendo Gamecube. Поддержка клавиатур Wooting. Поставка окружения KDE и включение по умолчанию протокола Wayland для систем с GPU Intel и AMD (для NVIDIA используется X11). Возможность загрузки в интерфейс Big Picture клиента Steam, например, при использовании на телеприставках. Возможность выбора на этапе установки образа с окружением на базе GNOME. Дополнительно развивается вариант Bazzite для игровых приставок Steam Deck, в котором предустановлены Steam и Lutris, по умолчанию используется интерфейс Game Mode, композитный менеджер Gamescope и модифицированный пакет Mesa. Компоненты сборки, такие как драйверы, прошивки, утилиты управления кулером и специализированные пакеты, портированы из SteamOS (перенесены из репозитория evlaV). Перенесены исправления для использования ФС Btrfs на SD-карте. По умолчанию задействован сервер SDGyroDSU. В качестве опций доступны пакеты Decky Loader, EmuDeck и ProtonUp-Qt. В интерфейс Gamemode добавлены инструмнты для обновления системы, пакетов flatpak и образов Distrobox. По умолчанию задействованы ZRAM, amd-pstate и планировщик ввода/вывода BFQ. В поставку включены утилиты для управления напряжением Steam Deck, изменения частоты экрана и ограничения уровня заряда аккумулятора.
