Компания Oracle официально приняла обязательства по отношению к MySQL

Компания Oracle опубликовала пресс-релиз в котором дала официальные обещания, касающиеся политики дальнейшего развития проекта MySQL. В представленном списке обязательств учтены все пожелания, озвученные представителями Еврокомиссии и представителями независимого сообщества разработчиков MySQL. В ответ Еврокомиссия опубликовала заявление в котором одобрила шаг Oracle и теперь с оптимизмом рассматривает возможность положительного рассмотрения сделки. Напомню, что сегодня состоится заседание Еврокомиссии на котором будет принято решение, касающиеся одобрения или запрещения сделки по покупке Oracle компании Sun Microsystems. Решение может быть вынесено уже сегодня или в течение 2-х недель. Oracle обещает: Производить улучшения MySQL в будущем, включая развитие ветки MySQL 6, открывая код под лицензией GPL. Более того, Oracle обязуется не выпускать коммерческую версию MySQL Enterprise Edition без одновременного выпуска свободной редакции MySQL Community Edition. Также дано обещание не накладывать каких либо дополнительных ограничений на распространение исходных текстов; Усилить активность разработки и исследований, связанных с развитием MySQL. Oracle обещает обеспечить надлежащее финансирование продолжения разработки MySQL, превышающее текущие денежные вливания в проект; Сохранить доступность API для разработки и подключения внешних хранилищ, разрабатываемых и поддерживаемых сторонними организациями и сообществами разработчиков; Снять текущие ограничения, обязывающие распространять внешние хранилища под лицензией GPL или приобретать коммерческую лицензию. Разрабатывать внешние расширения к MySQL отныне можно будет без дополнительных условий. Но текущее MySQL OEM соглашение, по которому работают клиенты купившие у Sun коммерческую лицензию, будет активно до истечения своего срока действия в декабре 2014 года; Не навязывать клиентам обязательный сервис поддержки, пользователи будут вправе отказаться от покупки подписки на техническую поддержку при приобретении коммерческой лицензии на MySQL Enterprise Edition; Учредить независимый надзорный совет пользователей MySQL - Customer Advisory Board для решения важных для пользователей MySQL стратегических вопросов, контроля и определения ключевых приоритетов в развитии; Учредить надзорный совет представителей разработчиков дополнительных хранилищ для MySQL - Storage Engine Vendor Advisory Board, для решения вопросов связанных с разработкой внешних дополнений; Продолжить поддержание в актуальном состоянии руководства по MySQL, поддерживать высокое качество документации и ее свободное распространение; Сохранить текущие условия при предоставлении сервиса технической поддержки. Дополнение: Майкл "Монти" Видениус, создатель MySQL, опубликовал заметку с анализом и критикой обязательств Oracle. Основные претензии - все обещания ограничены сроком в 5 лет и это только пресс-релиз, который не имеет юридической силы. Дополнение 2: Майкл "Монти" Видениус опубликовал FAQ с ответами на часто задаваемые вопросы по его видению проблемы.
