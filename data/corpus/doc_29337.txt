Deltacloud Aggregator переименован в проект Aeolus

Компания Red Hat представила новый проект Aeolus, под началом которого планируется развивать базирующиеся на Deltacloud API инструменты для управления инфраструктурой виртуализации, ранее выпущенные под лицензией GPL. Флагманским приложением нового проекта выступает Aeolus Conductor, построенный на кодовой базе Deltacloud Aggregator и предназначенный для организации единого интерфейса для управления различными cloud-окружениями, работы с образами виртуальных окружений, их распространения и контроля за их использованием в cloud-инфраструктуре. Готовые бинарные пакеты с Aeolus Conductor подготовлены для Fedora 13, Fedora 14 и RHEL6. Из других проектов, вошедших в состав Aeolus, отмечены: Oz - система автоматизации установки виртуальных машин; Image Factory - поддерживающий QMF-интерфейс демон для организации сборки образов виртуальных окружений; Image Warehouse - демон для перемещения образов виртуальных машин из cloud-окружения в другое, в соответствии с заданными администратором правилами; Audrey - набор утилит для конфигурирования уже работающих cloud-систем. Приложения из состава Aeolus используют Deltacloud API, единый программный интерфейс для унификации доступа и управления «облачной» инфраструктурой. С помощью Deltacloud API пользователи имеют возможность работать как с публичными сервисами, такими, как Amazon EC2, так и собственными, развернутыми с помощью технологии Red Hat Enterprise Virtualisation (RHEL-V).
