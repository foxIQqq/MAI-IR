Анонсирован выход релиза Scientific Linux 5.3. CentOS 5.3 намечен на выходные

Сегодня ночью был официально анонсирован выход дистрибутива Scientific Linux 5.3, построенного на пакетной базе Red Hat Enterprise Linux 5.3 и доступного в сборках для архитектур i386 и x86_64. С новшествами заимствованными из RHEL можно познакомиться в в анонсе RHEL 5.3. Ниже представлен список улучшений, добавленных разработчиками Scientific Linux: Улучшена поддержка беспроводных карт Intel, добавлен драйвер iwlwifi с прошивками для серий 3945, 4945 и 5000. Расширена поддержка беспроводных карт Atheros G (включен драйвер madwifi). Добавлен пакет NdisWrapper, позволяющий использовать в Linux NDIS драйверы из Windows; В состав дистрибутива включена научная библиотека SciPy, которую можно использовать для научных, инженерных и математических рассчетов, в также для построения графиков и диаграмм, на основе набора статистических данных. Также в состав была включена библиотека Numpy, для организации научных расчетов; Вместо почтового клиента Pine используется Alpine; В поставку включен код AUFS, альтернативной реализации UnionFS; CFITSIO - библиотека C и FORTRAN подпрограмм для работы с FITS (Flexible Image Transfer System) данными; Оконный менеджер IceWM 1.2.37; OpenJDK заменен на пакет Sun Java/JDK 1.6.0p12; OpenAFS - открытая реализация распределенной ФС Andrew File System; GNU R - язык для статистических расчетов. Поддержка файловой системы XFS. В программе установки пока нельзя создать XFS раздел, но установить Scientific Linux на ранее созданный XFS раздел уже можно; yum обновлен до версии 1.1.16; В состав включен набор обучающих приложений для KDE - KDEEDU; Добавлены дополнительные мультимедиа кодеки (gstreamer-plugins-extras из SL4 и k3b-extras из репозитория livna); FUSE модули fuse-smb и fuse-sshfs, библиотека cfitsio, оболочка для управления пакетами Yumex, фреймвок DKMS (DynamicKernel Module Support), пакет визуализации Graphviz и т.д. По сообщению разработчиков другого, базирующегося на Red Hat Enterprise Linux дистрибутива, релиз CentOS 5.3 находится на стадии распространения по зеркалам и будет публично доступен через один или два дня.
