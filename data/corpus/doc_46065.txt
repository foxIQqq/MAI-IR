Вышел Ubuntu 16.04.2 LTS c обновлением графического стека и ядра Linux

Доступно обновление дистрибутива Ubuntu 16.04.2 LTS, в которое включены изменения, связанные с улучшением поддержки оборудования, обновлением ядра Linux и графического стека, исправлением ошибок в инсталляторе и загрузчике. В состав также включены актуальные обновления для нескольких сотен пакетов, связанные с устранением уязвимостей и проблем, влияющих на стабильность. Одновременно представлены аналогичные обновления Kubuntu 16.04.2 LTS, Xubuntu 16.04.2 LTS, Mythbuntu 16.04.2 LTS, Ubuntu GNOME 16.04.2 LTS, Lubuntu 16.04.2 LTS, Ubuntu Kylin 16.04.2 LTS, Ubuntu MATE 16.04.2 LTS и Ubuntu Studio 16.04.2 LTS. В состав выпуска включены некоторые улучшения, бэкпортированные из выпуска Ubuntu 16.10. Например, предложено обновление пакетов с ядром Linux 4.8 (в Ubuntu 16.04.1 использовалось ядро 4.4). Графический стек обновлён до Mesa 12.0.3 и X.org Server 1.18.4, протестированных в осеннем выпуске Ubuntu 16.10. Включены свежие версии видеодрайверов для чипов Intel, AMD и NVIDIA. Обновлены версии пакетов OpenStack, snapd, lxc, lxcd, samba, ceph, postgresql-9.5, network-manager (1.2.2), freeipmi, systemd-networkd (231) и python (3.5.2). Обновления ядра и графического стека доступны для всех архитектур, за исключением 32-разрядных систем powerpc. В сборках для рабочего стола новые ядро и графический стек предложены по умолчанию. Для серверных систем новое ядро добавлено в качестве опции в инсталляторе, что привело к увеличению размера iso-образа, который теперь не умещается в 700MB CD. Использовать новые сборки имеет смысл только для новых установок, системы, установленные ранее, могут получить все присутствующие в Ubuntu 16.04.2 изменения через штатную систему установки обновлений. Поддержка выпуска обновлений и исправлений проблем безопасности для серверной и десктоп редакций Ubuntu 16.04 LTS продлится до апреля 2021 года. При этом, начиная с Ubuntu 16.04 для поставки новых версий ядра и графического стека применяется новая rolling-модель поддержки обновлений, в соответствии с которой бэкпортированные ядра и драйверы будут поддерживаться только до выхода следующего корректирующего обновления LTS-ветки Ubuntu. Например, предложенное в текущем выпуске ядро Linux 4.8 будет поддерживаться до выхода Ubuntu 16.04.3, в котором будет предложено ядро из состава Ubuntu 17.04. Изначально поставляемое базовое ядро 4.4 будет поддерживаться в течение всего цикла сопровождения до 2021 года. Для перевода уже существующих установок на новые версии ядра и графического стека следует выполнить команду: sudo apt-get install --install-recommends xserver-xorg-hwe-16.04 Одновременно объявлено о заморозке пакетной базы от добавления новых возможностей для весеннего релиза Ubuntu 17.04, функциональность которого можно считать сформированной. После заморозки вся работа будет сосредоточена на выявлении и исправлении ошибок.
