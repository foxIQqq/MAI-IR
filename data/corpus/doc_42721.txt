Ubuntu 16.04 останется на Unity 7. Unity 8, Mir и Snappy лягут в основу Ubuntu Personal

Оливер Райс (Oliver Ries), занимающий пост директора по развитию продуктов в компании Canonical, рассказал о планах по формированию следующего LTS-релиза Ubuntu 16.04 и серии параллельно развиваемых продуктов. Выпуск Ubuntu 15.10 по умолчанию продолжит поставку окружения Unity 7 на базе классического графического стека X.Org, соответственно и LTS-релиз Ubuntu 16.04 также не перейдёт по умолчанию на Mir и Unity 8. При этом Unity 8 будет доступен пользователям Ubuntu 16.04 в форме альтернативного графического сеанса. Для продвижения новых технологий, которые в настоящее время развиваются в рамках платформ Ubuntu Touch и Snappy Ubuntu Core, будет предложен новый продукт Ubuntu Personal, в котором технология атомарного обновления образа всей системы, без разбиения на отдельные deb-пакеты, будет совмещена с новым графическим окружением на базе Mir, Qt5 и Unity 8. Предвестником появления Ubuntu Personal являются сборки Ubuntu Desktop Next, которые можно рассматривать как ознакомительные предварительные выпуски Ubuntu Personal. Релиз Ubuntu Personal 1.0 ожидается в апреле 2016 года и будет позиционирован для предустановки на различные потребительские устройства со встроенным экраном. Возможность запуска приложений X.org в Ubuntu Personal будет сохранена и реализована через прослойку XMir. Snappy Ubuntu Core продолжит развиваться как система для оснащения различных автономных аппаратных устройств. Ubuntu останется на основе классической системы deb-пакетов и продолжит являться основным звеном для формирования всех остальных инициатив проекта. Из ожидаемых в Ubuntu 16.04 изменений отмечается расширение возможностей интерфейса Dash, увеличение удобства использования и более качественная поддержка экранов сверхвысокого разрешения, а также работа по оттачиванию существующих возможностей и стабилизации дистрибутива.
