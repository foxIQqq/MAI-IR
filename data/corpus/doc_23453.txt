Komodo 5.2.0 - кроссплатформенная многоязыковая IDE

Вышла версия 5.2.0 интегрированной среды разработки Activestate Komodo, построенной на основе технологий Mozilla (XUL-интерпретатор из состава Firefox 3). Komodo поддерживает языки PHP, Python, Ruby, Perl, Tcl, JavaScript, CSS, HTML. В версии 5.2.0 произошли следующие изменения: Увеличение производительности отладчика регулярных выражений Rx Toolkit и поиска Добавлена поддержка php 5.3 (автодополнение, подсветка синтаксиса и т.д.) Возможность изменять цвет фона вторичного языка (например, изменить цвет фона javascript внутри HTML документа) Поддержка сторонних утилит для поиска отличий в файлах (diff) Возможность изменения прав доступа для сетевых файлов Komodo существует в двух вариантах: платный Komodo IDE и бесплатный Komodo Edit, основанный на репозитории с открытым исходным кодом Open Komodo. Из отличий Komodo Edit от Komodo IDE можно отметить отсутствие: встроенного отладчика, интерактивного shell, HTTP-инспектора, режима просмотра DOM-иерархии для HTML документов, режима древовидного отображения структуры кода, системы наглядного построения регулярных выражений, интеграции с системами контроля версий. Версию 5.2.0 для различных платформ (Windows, Linux, MacOS) можно загрузить здесь. В директории remotedebugging находятся дополнения для удаленной отладки приложений.
