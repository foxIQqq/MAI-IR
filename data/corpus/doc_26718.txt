Компания Microsoft открыла код двух инструментов для работы с форматом PST

Спустя несколько месяцев после открытия спецификаций с полным описанием формата PST (Personal Storage Table), используемого для хранения данных Outlook (включая архив переписки, список контактов и события из календаря-планировщика), компания Microsoft открыла под лицензией Apache исходные тексты нескольких инструментов: "PST Data Structure View Tool" - утилита для просмотра внутреннего содержимого PST-файлов; "PST File Format SDK" - кроссплатформенная С++ библиотека для разбора структур PST.
