Вышел релиз легковесного Linux дистрибутива Puppy Linux 4.3

Вышел релиз легковесного Linux дистрибутива Puppy Linux 4.3. Установочный LiveCD образ занимает всего 105 Мб, но при этом в дистрибутиве представлен полный спектр современных возможностей, функционирующих при достаточно скромных требованиях к аппаратному обеспечению. Графическое окружение пользователя базируется на оконном менеджере JWM, собственном наборе GUI-конфигураторов (Puppy Control Panel) и виджетов (Pwidgets - часы, календарь, RSS, состояние соединения и т.п.). В новой версии представлены следующие улучшения: Переход на новый пакетный менеджер PPM (Puppy Package Manager), позволяющий использовать в Puppy пакеты программ, созданные для других дистрибутивов (Debian, Ubuntu, Arch, Slackware и т.д.); Представлена новая система сборки загрузочных образов Puppy - Woof, позволяющая пользователю легко сформировать LiveCD с заданным набором пакетов и настроек. При этом допускается построение образа на базе всех поддерживаемых в PPM пакетов, включая пакеты из других дистрибутивов, например, см. сборку Dpup на базе пакетов из Debian GNU/Linux 5.0; Переход на использование Linux ядра 2.6.30.5 с применением дополнительных патчей для поддержки Aufs2. В дистрибутив интегрирована полноценная поддержка файловой системы ext4. Обновлены сопутствующие программы GParted, e2fsprogs, disktype, GRUB и guess_fstype; Улучшена поддержка выхода в сеть через диалап. В состав дистрибутива включены драйверы для работы с win- и soft-модемами, построенными на базе чипсетов Agere, ESS, Lucent, Conexant, Smartlink, Pctel и Intel, настройка драйверов для которых в Puppy производится автоматически. Кроме того, проведена большая работа по обеспечению выхода в сеть через 3G адаптеры; В состав включен Pstreamvid, специально разработанный для Puppy Linux GUI-интерфейс для просмотра потокового видео и Internet TV; В состав включено несколько новых виджетов: CPU Scaling Ondemand для управления частотой процессора; Psync для синхронизации времени; Mscw для выбора звуковой карты для вывода звука; pCD для проигрывания аудио-CD; Screenshot для создания скринштов; Pmirrorget для загрузки сайтов целиком; Gtkhash для генерации хэш-функций; Bcrypt GUI для шифрования файлов; BootFlash для упрощения установки на USB Flash; Xdelta GUI - rag-and-drop интерфейс к утилите Xdelta. Обновлены виджеты Pburn, Pfind и Pbackup; Для упрощения создания визуальных тем подготовлено специальное приложение JWM Theme Maker; Представлена простая система для создания динамических web-приложений QUISP, http-сервер Hiawatha и SQLiteManager, простой фронт-энд для работы с SQLite базами; В состав включен пакет Linux-dvb-apps с программами для настройки DVB-карт; Добавлен текстовый редактор NicoEdit, с подсветкой синтаксиса для языков Vala и Genie, на которых создано большинство программ для Puppy; Добавлен PuppyBrowser - web-браузер на базе библиотеки 'gtkmozembed' (SeaMonkey), который используется для локального просмотра файлов помощи в HTML формате и для просмотра страниц CUPS, PPLOG и QUISP. Также добавлено дополнение к SeaMonkey - FullerScreen, предназначенное для создания презентаций; В качестве программы для просмотра изображений теперь используется Viewnior; Улучшена работа GUI-интерфейса для настройки сети (Network Wizard).
