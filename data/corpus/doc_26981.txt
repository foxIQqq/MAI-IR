OpenVBX - открытая web-ориентированная система телефонии для бизнеса

Представлен новый проект OpenVBX, в рамках которого разработан web-интерфейс для организации корпоративного телефонного сервиса, напоминающего по своим возможностям Google Voice. Система поддерживает простой интерфейс для перенаправления звонков, имеет систему голосового меню и голосовой почты (с отправкой записи на email), позволяет вести архив записей переговоров и дает возможность управлять приемом и отправкой SMS. Кроме того, в OpenVBX реализована поддержка виртуальных телефонных номеров и возможна организация линий для проведения конференций. Код системы написан на языке PHP, для хранения данных используется MySQL, внешний вид легко изменяется при помощи шаблонов. Для разработчиков предусмотрен API для написания собственных дополнений и апплетов. В настоящий момент уже разработано несколько плагинов, позволяющих интегрировать OpenVBX c системами helpdesk и CRM, расширить средства аутентификации, организовать перенаправление по caller ID и изменить алгоритм помещения звонков в очередь. По умолчанию система OpenVBX может функционировать только используя для организации работы с телефонией Cloud-сервисы Twilio, не требуя от администратора каких-либо навыков в развертывании инфраструктуры VoIP-телефонии. Для желающих интегрировать систему в текущую инфраструктуру телефонии предприятия, без использования сервисов Twilio, предусмотрена возможность обособленной работы с задействованием сторонних бэкендов.
