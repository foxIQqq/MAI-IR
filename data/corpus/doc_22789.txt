Выпущен релиз Bugzilla 3.4

Выпущен релиз Bugzilla 3.4, системы для ведения базы данных ошибок, контроля за их исправлением и общего координирования процесса разработки. В новой версии: Значительно упрощен процесс создания записи для новой проблемы. По умолчанию все вторичные поля скрыты, оставлены только основные блоки; Представлена новая базовая страница; Реализована возможность скрытия email-адресов от незарегистрированных посетителей; Поддержка асинхронной рассылки почтовых уведомлений; Дата и время в системе теперь отображается с учетом часового пояса текущего зарегистрированного пользователя; URL страницы результатов поиска упрощен для повышения удобства обмена ссылками; Улучшена система создания пользовательских полей, добавлен новый тип полей - Bug ID; Добавлено поле для ссылок на описание ошибок в других системах отслеживания ошибок, например, в Launchpad; При установке добавлены новые зависимости, теперь требуется наличие Perl модулей Digest::SHA, DateTime, DateTime::TimeZone, URI.
