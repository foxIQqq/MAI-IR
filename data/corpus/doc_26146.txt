Вышел релиз платформы для VoIP телефонии FreeSWITCH 1.0.6

Команда разработчиков открытой платформы для VoIP телефонии FreeSWITCH объявила об официальном выпуске версии 1.0.6 и готовности новой версии популярной открытой исходной библиотеки soft-switch. FreeSWITCH 1.0.6 унаследовал приемущества предыдущих версий и приобрел множество новых возможностей и улучшений. Усовершенствования коснулись почти всех областей проекта, например: Добавлены новые аудиокодеки SILK (Skype), BroadVoice (Broadcom) и G.729; Реализован предложенный компанией Broadsoft метод организации совместного использования одной линии связи (Shared Call Appearance, SCA) в нескольких VoIP системах; Добавлено новое диалплан-приложение valet_park для паркинга звонков и их последующего возобновления с другого телефона; Добавлено несколько новых команд, которые можно использовать для отладки работы системы: "uuid_audio", "uuid_debug_audio", "uuid_simplify", "simplifies"; В пакет интегрированы дополнения ESL (Event Socket Library) к языку TCL; В клиент для управления FreeSWITCH из командной строки fs_cli добавлена поддержка автодополнения ввода по нажатию клавиши "Tab"; В Linux/Unix появилась возможность использования fail2ban, приложения для противостояния атакам по подбору регистрационных параметров в SIP (brute-force); Добавлены новые модули: mod_skinny (поддержка протокола Cisco SCCP), mod_directory, mod_tts_commandline, mod_shell_stream, mod_h323.
