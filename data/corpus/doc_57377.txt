Релиз дистрибутива Manjaro Linux 21.3

Состоялся релиз дистрибутива Manjaro Linux 21.3, построенного на основе Arch Linux и ориентированного на начинающих пользователей. Дистрибутив примечателен наличием упрощённого и дружественного пользователю процесса установки, поддержкой автоматического определения оборудования и установки необходимых для его работы драйверов. Manjaro поставляется в виде live-сборок с графическими окружениями KDE (3.5 ГБ), GNOME (3.3 ГБ) и Xfce (3.2 ГБ). При участии сообщества дополнительно развиваются сборки с Budgie, Cinnamon, Deepin, LXDE, LXQt, MATE и i3. Для управления репозиториями в Manjaro используется собственный инструментарий BoxIt, спроектированный по образу Git. Репозиторий поддерживается по принципу непрерывного включения обновлений (rolling), но новые версии проходят дополнительную стадию стабилизации. Кроме собственного репозитория, имеется поддержка использования репозитория AUR (Arch User Repository). Дистрибутив снабжён графическим инсталлятором и графическим интерфейсом для настройки системы. Основные новшества: Инсталлятор обновлён до свежего выпуска фреймворка Calamares. В модуле инсталлятора для редактирования дисковых разделов расширена поддержка шифрованных разделов LUKS. В модуле начального создания пользователей реализован список запрещённых имён учётных записей и имён хостов, которые пересекаются с именами, используемыми в процессе установки. Редакция на базе GNOME обновлена до выпуска GNOME 42, в котором реализованы глобальные настройки тёмного стиля оформления интерфейса. Для приложений предусмотрена возможность раздельного включения светлого или тёмного оформления, независимо от общего системного стиля. Многие приложения переведены на использование GTK 4 и библиотеку libadwaita, которая предлагает готовые виджеты и объекты для построения приложений, соответствующие рекомендациям GNOME HIG (Human Interface Guidelines) и способные адаптивно подстраиваться под экраны любого размера. Оптимизирована отрисовка приложений, работающих в полноэкранном режиме. Редакция на основе KDE обновлена до KDE Plasma 5.24. В режиме редактирования предоставлена возможность произвольного перемещения панели мышью, удерживая за любую область, а не только за специальную кнопку. Реализован новый обзорный эффект (Overview) для просмотра содержимого виртуальных рабочих столов. В интерфейсе поиска программ (KRunner) предложена встроенная подсказка по доступным операциям поиска, показываемая при клике на значке вопроса или вводе команды "?". Основная редакция продолжает поставляться с пользовательским окружением Xfce 4.16. Как и в прошлом выпуске в дистрибутиве продолжает использоваться ветка ядра Linux 5.15.
