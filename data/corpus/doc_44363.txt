Chrome обогнал Internet Explorer по данным сервиса Net Applications

По данным общемирового рейтинга Net Applications, браузер Chrome занял 41.71% рынка, что позволило ему вырваться на первое место, опередив Internet Explorer и Edge (41.33%), Firefox (10.06%), Safari (4.47%) и Opera (2.01%). Год назад доля Chrome составляла 25.68%, а Internet Explorer - 55.83%, Firefox - 11.70%, Safari - 5.12% и Opera - 1.19%. Следует отметить, что Net Applications до сих пор оставался единственным рейтингом, в котором Chrome не занимал первое место. Интересно, что в других рейтингах доля Internet Explorer существенно ниже. Например, по данным рейтинга StatCounter, Chrome обогнал Internet Explorer ещё в 2012 году, а в настоящее время доля Chrome составляет 56.74%, Firefox - 14.24%, Internet Explorer - 12.14%, Safari - 9.47%. По данным отечественного рейтинга OpenStat, Internet Explorer занимает 4.99% рынка, Safari - 8.49%, Opera - 9.27%, Firefox - 9.93%, YandexBrowser - 12.44%, Chrome - 44.04%. По статистике liveinternet.ru Internet Explorer занимает 3.2% рынка, Safari - 13.2%, Opera - 4.4%, Firefox - 5%, YandexBrowser - 9.4%, Chrome - 44.9%, Android Browser - 10.5%.
