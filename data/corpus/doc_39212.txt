Стартовал конкурс разработчиков приложений Ubuntu App Showdown

Компания Canonical объявила о проведении конкурса разработчиков приложений Ubuntu App Showdown, направленного на стимулирование создание приложений для платформы Ubuntu с использованием нового Ubuntu SDK. Работы принимаются в 4 категориях: QML: новые приложения, написанные с использованием QML или комбинации QML и JavaScript/C++; HTML5: новые приложения, написанные с использованием web-технологий (HTML5/CSS/JavaScript) или при помощи Apache Cordova; Портированные приложения: программы, портированные с других платформ. При написании кода могут использоваться любые технологии; Приложения для Китая: программы, учитывающие специфику использования в Китае. На подготовку работ отводится шесть недель, после чего жюри определит победителей. При выборе победителей будут учтены востребованность разработки, универсальность интерфейса (мобильный и десктоп режимы), качество кода, функциональность и дизайн. Победители в первых трёх категориях получат планшеты Nexus 7, за два лучших приложения для Китая будут вручены смартфоны Meizu MX3. В связи с проведением конкурса, Джоно Бэкон (Jono Bacon), менеджер по взаимодействию с комьюнити компании Canonical, опубликовал список приложений, которые ему бы хотелось видеть среди конкурсных работ: Почтовый клиент в стиле Discourse; Клиент для работы с социальными сетями с поддержкой вкладок, функциями поиска и просмотром публикаций в Twitter по шкале времени; Клиент для интеграции с Google+; Приложение для участников локальных сообществ Ubuntu, работающее с контентом loco.ubuntu.com; Программа для записи звука с возможностью настройки чувствительности микрофона.
