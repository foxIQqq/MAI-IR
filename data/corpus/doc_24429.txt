Вышел релиз поискового движка Apache Lucene 3.0.0

Вышел релиз одного из самых мощных открытых поисковых движков - Apache Lucene 3.0.0, некоторые модули которой в частности задействованы даже в поисковой технологии Microsoft Kumo. В версии Lucene Java 3.0.0 не добавлено заметных новшеств, но произведена глобальная чистка API и кода, удалены компоненты объявленные ранее устаревшими, для работы Lucene отныне требуется как минимум Java 5 (поддержка совместимости с Java 1.4 прекращена).
