Доступен Solaris 11.4 SRU33

Компания Oracle опубликовала обновление операционной системы Solaris 11.4 SRU 33 (Support Repository Update), в котором предложена серия очередных исправлений и улучшений для ветки Solaris 11.4. Для установки предложенных в обновлении исправлений достаточно выполнить команду 'pkg update'. В новом выпуске: Добавлена поддержка 18 ТБ жёстких дисков Western Digital (WDC Paris-C SAS). В состав включены SpiderMonkey 78, Valgrind, buildbot-worker, Graphene, jQuery, jQuery tablesorter, libhandy, libpsl, python toml. Для Python 3.9 добавлены пакеты с модулем numpy. До версии 3.38 обновлены компоненты рабочего стола GNOME: evince, evolution-data-server, totem, gcr, seahorse, gnome-screenshot, gnome-clocks. До версии 3.36 обновлён gnome-keyring. Библиотека GTK обновлена до версии 3.24.23. Обновлены многие пакеты, в том числе GNU Tar 1.33 Node.js 14.16.0, Samba 4.13.4, Subversion 1.14.1, cups-filters 1.28.7, gawk 5.1.0, hplip 3.20.11, meson 0.56.1, nss 60.1, puppet 5.5.21, rsync 3.2.3, vala 0.50.2, xterm 366. Обновлены версии программ с устранением уязвимостей: Bison has 3.7.3, ImageMagick 6.9.12-3, Jinja2 2.11.3, OpenSSH 8.4p1, OpenSSL 1.1.1j, PyYAML 5.4.1, curl 7.74.0, git 2.30.2, glib 2.66.8, libSDL 2.0.14, libtiff 4.2.0, mutt 2.0.5. Устранены проблемы с безопасностью в пакетах: core/ipc, gnome/libraries, kernel/syscall, library/cairo, python-mod/pil, utility/linker, utility/screen, utility/smb, utility/sudo, utility/svn, x11/xclients.
