Выпущен стабильный релиз дистрибутива КСоНоМи 10.2

Анонсирован новый выпуск основанной на Ubuntu Linux 10.04 отечественной сборки КСоНоМи 10.2, с подборкой программ для людей творческих и инженерных профессий. Размер iso-образа (LiveDVD) - 2.3 Гб. Особенности системы: Интеграция "медиатеки", подборки ссылок для быстрого доступа к популярными русскоязычными Интернет ресурсами и информационными сервисами; Четыре базовых варианта формирования рабочего стола, включая нтерфейс планшета или медиацентра: Liquid (на основе GNOME 2.30.1, CompizFusion0.86 и панели AWN) KDE 4.4.2 UNE - упрощённый вариант Liquid UNE Liquid UNE - окружение удобное для использования в основе медиацентра, а также с сенсорными мониторами или для управления планшетным портативным компьютером. Подборка мультимедиа кодеков и библиотек для поддержки технологии аппаратного ускорения вывода видео VA API (AMD/ATI) и VDPAU (NVIDIA). В состав также включена подборка популярных проприетарных драйверов и программ, таких как Adobe Flash, NVIDIA 256.44, AMD/ATI fglrx 8.783 (Catalyst 10.10); Базовые приложения для работы с графикой: Blender, Inkscape, GIMP, Scribus, OpenOffice.Org Draw, FontForge, Qcad, PDF Editor, XSANE + YAGF + CuneiForm; Базовые приложения для разработчиков: PHP5, MySQL 5.1, Qt4 Creator, Eclipse, Anjunta, Bluefish, GLADE, FreePascal, MySQL Query Browser, MySQL Admin, gitg, KBugBuster, Lokalize, VirtualBox OSE; Программы для работы со звуком: Audacity, Linux MultiMedia Studio, SoundKonverter; Программы для работы с видео: OpenShot, AviDemux, XvidCap; Программы для записи CD/DVD: k3b, Brasero, DVD Styler, ISOmaster, DVDisaster, ManDVD; Интернет и сети: FireFox 3.6, Konqueror, Thunderbird, Evolution, Kmail, QutIM, Pidgin, Kopete, QuteCom, Deluge, KTorrent, Transmission, EiskaltDC, gFTP, NetworkManger, VPNpptp; Мультимедиа: VLC, XINE, MPlayer, Totem, XvidCap, Amarok, Rhythmbox; Офисные программы: OpenOffice.org 3.2, Kaddressbook, Okular, KOrganizer, Ktimetracker, Kontact, GNUCash, KMyMoney, Словари.
