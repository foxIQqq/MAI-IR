Вышел Finnix 107, Live-дистрибутив для системных администраторов

Вышел релиз Live-дистрибутива Finnix 107, основанный на срезе пакетной базы Debian testing. Дистрибутив поддерживает только работу в консоли, но содержит хорошую подборку утилит для нужд администратора. Размер iso-образа - 117 Мб, при этом без учёта сжатия в комплект входит около 395 Мб утилит. Например, в состав включены LVM2, cryptsetup, cdpr, iftop, irssi, OTP calculator, VLAN tools и т.д. Кроме сборки x86/x86_64 дистрибутив доступен в сборке для архитектуры PowerPC. Из изменений отмечается синхронизация пакетной базы с Debian testing по состоянию на 23 декабря, переход на ядро Linux 3.6, проведение работы по ускорению загрузки и завершения работы, включение в состав утилит arping, bridge-utils, chntpw, cmospwd, ifenslave-2.6, sshfs, testdisk, udftools и zerofree. Iso-образ сформирован в гибридном виде и подходит для использования на USB Flash, в загрузочное меню добавлен вызов iPXE, обеспечена поддержка опции findiso для загрузки из файла (findiso=/path/to/finnix-107.iso в Grub уже установленной системы). Полностью переработана утилита hwsubmit для отправки отчётов об оборудовании пользователя.
