Выпуск ReactOS 0.3.16

Представлен релиз ReactOS 0.3.16, операционной системы с открытым исходным кодом, нацеленной на обеспечение совместимости с программами и драйверами Microsoft Windows семейства NT (XP/2003/Vista). Серия 0.3.x находится на "альфа"-стадии разработки. Для тестирования выпуска подготовлен демонстрационный LiveCD (55 Мб), а также образы для запуска под управлением различных систем виртуализации VirtualBox, QEMU и VMware. В новом выпуске внесено 416 изменений, в том числе устранено 263 ошибок, добавлено 106 улучшений и 11 новых возможностей. Из наиболее существенных улучшений можно отметить завершение переработки реализации режима CSRSS и начало переработки shell32.dll. Из новшеств выделяется начальная поддержка тем оформления (отключена по умолчанию, но для тестирования доступна демонстрационная тема Lautus) и создание сетевого драйвера для чипов RTL8139, который позволяет из коробки запускать ReactOS в новых версиях QEMU. Разработчики также напоминают про проведение краудфандинг-кампании по финансированию разработки Thorium Core, коммерческого дистрибутива операционной системы ReactOS. В настоящее время собрано 23 тысячи долларов из необходимых 120 тысяч. Успешное завершение кампании позволит проекту трудоустроить несколько разработчиков в режиме полного рабочего дня и довести ReactOS до состояния, пригодного для ежедневного использования. Thorium Core позволит организовать работу окружения для выполнения Windows-приложений, отличающегося низким потреблением ресурсов и отсутствием необходимости приобретения лицензий на отдельные копии Windows. Модульный дизайн Thorium Core позволит использовать только компоненты ОС, необходимые для выполнения конкретного приложения, в том числе с возможностью формирования виртуализированного контейнера для изолированного запуска Windows-программы или создания минимального окружения для выполнения на встраиваемых системах.
