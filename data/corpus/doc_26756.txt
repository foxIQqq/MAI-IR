Релиз офисного пакета KOffice 2.2

После 6 месяцев разработки увидел свет релиз свободного офисного пакета KOffice 2.2. Одним из ключевых улучшений KOffice 2.2 является интеграция в пакет среды для управления данными Kexi (открытая альтернатива MS Access), которая ранее входила в состав KOffice 1.x. Работа над переработанной версией Kexi велась около трех лет, за которые интерфейс программы был полностью пересмотрен. В целом текущее состояние Kexi отмечено как стабильное, программа готова для повсеместного использования. Кроме того, в отличие от прошлых выпусков, большой вклад в разработку версии 2.2 оказали сторонние компании и организации. Компания Nokia в рамках портирования пакета для мобильной платформы Mаemo провела значительную работу по общей стабилизации кода и улучшению поддержки форматов MS Office. Организация NLnet поддержала работу, связанную с поддержкой ODF-метаданных в RDF и реализацией механизма отслеживания вносимых пользователями изменений. Компания KO GmbH, в кооперации с NLnet, провела мероприятие по улучшению совместимости при поддержке ODF в различных офисных пакетах. В состав KOffice 2.2 входят базовые приложения: KWord - текстовый процессор; KSpread - табличный процессор; KPresenter - система для подготовки презентаций; KPlato - система управления проектами; Karbon - векторный графический редактор, в качестве основного формата используется SVG; Krita - растровый графический редактор; Kexi - среда для управления данными; Kivio - ПО для построение диаграмм, блок-схем, карт сети; Плагины: Chart - построение графиков и диаграмм; Formula - редактор формул; Picture - вставка изображений. Основные улучшения KOffice 2.2: Улучшена поддержка форматов MS Office 2007 и добавлены дополнительные фильтры импорта для формата MS OOXML (docx, pptx); В KWord налажена поддержка проверки орфографии, расширено число инструментов для форматирования документа. Значительно улучшены средства для отслеживания изменений в редактируемых документах, добавлен диалог для принятия или отмены изменений. Добавлена поддержка мета-данных на базе RDF для документов в формате OpenDocument 1.2. В табличном процессоре KSpread добавлены новые функции, расширены функции по добавлению в ячейки форматированного текста; В KPresenter добавлена поддержка переменных, добавлен новый фреймворк для создания анимации, улучшены опции по настройке cinerama; В Krita отмечен ряд значительных улучшений: добавлена поддержка форматов PPM, XCF и JPEG2000; добавлена возможность импорта GIF-изображений; добавлены новые фильтры; значительно улучшены элементы, связанные с выбором и использованием кистей; реализован инструмент для смешивания цветов в цифровом представлении; организована запись действий, произведенных инструментами для рисования; В Karbon добавлена поддержка редактирования областей фильтров в SVG-файлах, осуществлен переход на более активное использование типовых общих библиотек и подсистем KOffice; В KPlato добавлен новый планировщик проектов и улучшены средства по генерации отчетов с использованием KoReport (совместно используется с Kexi); Добавлено два новых плагина, с поддержкой интеграции в документы вставок с видео и векторными изображениями в форматах WMF и EMF; Во всех приложениях интегрирована поддержка нового фреймворка для применения к элементам (shape) трансформирующих эффектов, таких как размывание. В настоящий момент поддерживается 11 базовых эффектов, в будущем их число будет расширено. Увеличена совместимость пакета с платформами отличными от Unix, такими как Windows и Mac OS X. Также проведена большая работа по оптимизации кода, чистке API и увеличению производительности.
