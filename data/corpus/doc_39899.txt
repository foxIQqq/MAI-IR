Выпуск дистрибутива Linux Mint 17

Увидел свет дистрибутив Linux Mint 17, основанный на пакетной базе Ubuntu 14.04 LTS, полностью совместимого с ним, но существенно отличающегося подходом к организации интерфейса пользователя и подбором используемых по умолчанию приложений. Разработчики Linux Mint предоставляют десктоп-окружение, соответствующее классическим канонам организации рабочего стола, которое является более привычным для пользователей не принимающих новые методы построения интерфейса Unity и GNOME 3. Для загрузки доступны DVD-сборки на базе оболочек MATE 1.8 (1.2 Гб) и Cinnamon 2.2 (1.2 Гб). Linux Mint 17 примечателен переходом к новой схемe формирования выпусков, для которых теперь будут использоваться только релизы Ubuntu с длительным сроком поддержки (LTS) с периодическим бэкпортированием новых версий программ. Выпуск Linux Mint 17 будет поддерживаться до 2019 года и следующие два года будет выступать основой для промежуточных выпусков 17.x. В ближайшие два года, до выпуска Ubuntu 16.04 LTS, раз в полгода будут выходить промежуточные версии 17.1, 17.2, 17.3, для которых будет производиться бэкпортирование новых версий программ, в том числе новых выпусков рабочих столов Cinnamon и MATE. После выхода Ubuntu 16.04 LTS в 2016 году будет представлен выпуск Linux Mint 18.0 на который будет переключена работа по бэкпортирвоанию, а ветка Linux Mint 17.x будет переведена в режим сопровождения, предусматривающий только выпуск обновлений с устранением уязвимостей и серьёзных проблем. Кроме отличий в организации рабочего стола, в дистрибутиве задействован ряд оригинальных приложений, упрощающих настройку и работу в системе начинающих пользователей. Например, в Linux Mint используется переработанная система меню, собственный менеджер установки и обновления приложений, оригинальный интерфейс для настройки системы mintConfig, "мастер" для настройки различных параметров системы для начинающих пользователей в стиле вопрос-ответ, интерфейс для настройки параметров десктопа, средство для выполнения резервного копирования mintBackup и т.п. Ключевые особенности Linux Mint 17: В состав включены новые версии десктоп-окружений MATE 1.8 и Cinnamon 2.2, оформление и организация работы в которых продолжает развитие идей GNOME 2 - пользователю предлагается рабочий стол и панель с меню, областью быстрого запуска, перечнем открытых окон и системным лотком с запущенными апплетами. Cinnamon основан на GTK3+ и технологиях GNOME 3. Проект развивает оболочку GNOME Shell и оконный менеджер Mutter в направлении предоставления окружения в стиле GNOME 2 с более современным оформлением и задействованием элементов из GNOME Shell, дополняющих средства классического десктопа. В новой версии Cinnamon обновлён интерфейс системного конфигуратора, переработаны модули для настройки даты/времени, управления питанием, хранителем экрана, локализацией, раскладкой клавиатуры. Добавлен конфигуратор для настройки действий при попадании курсора в каждый из углов экрана. Реализована возможность привязки к апплетам ролей для автоматического скрытия дублирующих пиктограмм системного лотка. В апплет управления звуком добавлена поддержка управления медиапроигрывателями, поддерживающими стандарт MPRIS. Добавлена поддержка экранов сверхвысокого разрешения (HiDPI/Retina). Расширены возможности файлового менеджера Nemo (форк Nautilus 3.4). MATE продолжает развитие кодовой базы GNOME 2.32, основан на GTK2+ и полностью избавлен от пересечений с GNOME 3, что позволяет использовать традиционный рабочий стол GNOME 2 параллельно с рабочим столом GNOME 3. В новом выпуске добавлена поддержка тайлинга. Добавлена утилита mpaste для быстрого обмена данными между компьютерами с использованием сервиса paste.mate-desktop.org. В файловом менеджере Caja добавлена настройка для вывода размерности в единицах МЭК, а не Си (например, мебибайт вместо мегабайта). В диалоге хранителя экрана обеспечен вывод текущей даты и времени. В апплет с реализацией липких заметок добавлена поддержка операции отката изменений (undo). В апплет управления громкостью добавлена возможность быстрого отключения звука через клик средней кнопкой мыши; Значительно улучшен менеджер установки обновлений. Программа стала заметно быстрее, не требует перезапуска с правами root, не заставляет ждать установки сетевого соединения и не выставляет блокировку на кэш APT при запуске сеанса. Улучшен интерфейс пользователя, немного изменены пиктограммы и значительно ускорена загрузка списка изменений. Добавлен новый столбец "тип", позволяющий наглядно отделить записи с бэкпортами, обычными обновлениями и обновлениями для репозитория Romeo. Обновления с устранением проблем безопасности теперь могут выводиться вне зависимости от выбранного уровня безопасности. В частности, добавлены опции для выбора показывать ли обновления безопасности всегда или только при их явном выборе. Расширены средства вывода истории операций, если раньше выводились только обновления, применённые через менеджер обновлений, то теперь отображается список всех обновлений, в том числе установленных при помощи apt-get, aptitude, gdebi и dpkg. Добавлен новый раздел для обзора всех доступных ядер Linux. В секции выводятся сведения какое ядро установлено, какие новые ядра доступны для установки и какие в них представлены исправления и регрессивные изменения. В Driver Manager, интерфейс для установки проприетарных и альтернативных драйверов (NVIDIA, AMD, Broadcom, Samsung и т.п.), добавлена возможность установки драйверов в offline-режиме. Если сетевое соединение отсутствует, программа предложит подключить установочный носитель и использовать его как временный репозиторий для установки драйвера. Например, указанная возможность заметно упрощает установку драйверов для неподдерживаемых из коробки беспроводных карт. Представлен новый интерфейс для настройки параметров языка, который пришёл на смену старому конфигуратору и модулю языковой настройки Cinnamon. Новый интерфейс позволил унифицировать конфигурацию в Cinnamon, MATE и Xfce редакциях Linux Mint и LMDE. Инструмент даёт вохможность выбрать язык, изменить общесистемные настройки и установить дополнительные пакеты поддержки языков. Проведена оптимизация интерфейса Software Sources, предназначенного для настройки источников установки приложений и выбора наиболее быстрого зеркала. При выборе репозиториев "backport" и "romeo" добавлен дополнительный диалог с предупреждением и пояснением когда эти репозитории не следует использовать. Переработано оформление экрана с приветствием входа в систему. Новая реализация больше не использует WebKit, потребляет меньше ресурсов и запускается значительно быстрее. Обновлён дисплейный менеджер MDM (форк GDM 2.20), используемый для организации входа в систему и предоставляющий графические инструменты настройки параметров входа, возможность использования визуальных тем, средства организации удалённой работы, подключения своих скриптов для обработки событий, режим автоматического входа в систему, возможность выбора языка. В новом выпуске заметно улучшена поддержка многомониторных конфигураций (появилась возможность выбора первичного монитора и унифицирован процесс установки обоев рабочего стола), добавлена поддержка экранов сверхвысокого разрешения (HiDPI), обеспечена возможность ввода имени пользователя, не присутствующего в списке, сокращена нагрузка на CPU при работе анимационных эффектов, добавлено определение сеанса и языка пользователя, реализована новая команда "mdm-recovery" для перехода в режим восстановления.
