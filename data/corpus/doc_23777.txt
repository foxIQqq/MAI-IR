Red Hat и Microsoft сертифицировали продукты виртуализации друг у друга

В рамках объявленной в феврале инициативы по расширению взаимодействия и взаимной поддержке технологий виртуализации, компании Red Hat и Microsoft провели взаимную сертификацию поддерживаемых продуктов виртуализации. Так, Red Hat сертифицировал выполнение продуктов Windows Server 2003, 2008 и Windows Server 2008 R2 в качестве гостевых систем на платформе RHEL 5.4, а компания Microsoft сертифицировала выполнение Red Hat Enterprise Linux 5.2-5.4 в Hyper-V окружении на Windows Server. Отдельно отмечается, что RHEL сертифицирован для выполнения в режиме аппаратной эмуляции, а не при использовании открытых летом паравиртуальных драйверов, которые планируется включить в состав Linux ядра 2.6.32.
