Демонстрация Linux версии web-браузера Chromium

Компания Google объявила о выходе бета версии web-браузера Google Chrome 2, построенного на базе новой экспериментальной ветке открытого проекта Chromium. По сравнению с первым стабильным релизом Chrome производительность новой сборки значительно возросла, скорость прохождения тестов V8 benchmark увеличилась на 25%, а Sunspider - на 35%. Из новшеств появилась поддержка автозаполнения форм, полноэкранного просмотра, автоматического скроллинга (при нажатой средней кнопке мышью, перемещение манипулятора приводит к смещению страницы в любом направлении), возможность перетаскивания табов в новое окно, поддержка профилей настроек пользователя, и выполнение пользовательских Greasemonkey-подобных скриптов в контексте любых просматриваемых страниц. Для демонстрации текущего состояния Linux варианта web-браузера Chromium, энтузиасты подготовили на базе текущего репозитория исходных текстов ежедневно обновляемую пре-альфа сборку, оформленную в виде бинарного пакета для Ubuntu Linux. Несмотря на раннюю стадию развития Linux сборка отображает страницы так же качественно и работает так же быстро как и версия для Windows. Многие элементы интерфейса еще не реализованы: нет поддержки строки табов и закладок, отсутствует меню настроек. Тем не менее в браузере присутствует базовая навигация, возможность просмотра истории, режим приватного просмотра, менеджер загрузок, работает создание нового таба. Офицально Google планирует выпустить версии web-браузера Chrome для платформ Mac OS X и Linux в течение первой половины 2009 года. Графический интерфейс Linux версии основан на использовании библиотек Gtk+, Glib, Pango и Cairo, а также оригинальной системе рендеринга графики Skia (используется в Windows сборке Chrome), портированной в Linux. Для установки в Ubuntu необходимо подключить дополнительный репозиторий, в файл /etc/apt/sources.list.d/chromium-daily.list занести строку: для ubuntu 8.04 hardy deb http://ppa.launchpad.net/chromium-daily/ppa/ubuntu hardy main #Chromium daily builds для ubuntu 8.10 intrepid deb http://ppa.launchpad.net/chromium-daily/ppa/ubuntu intrepid main #Chromium daily builds для ubuntu 9.04alpha jaunty deb http://ppa.launchpad.net/chromium-daily/ppa/ubuntu jaunty main #Chromium daily builds Импортировать ключ: # apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 5A9BF3BB4E5E17B5 Обновить список пакетов: # aptitude update И установить пакет chromium-browser: # aptitude install chromium-browser
