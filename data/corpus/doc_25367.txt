Вышел Kerio Connect 7, сервер для организации групповой работы

Анонсирован выход сервера для организации групповой работы Kerio Connect 7, позиционируемого в роли полноценной замены Microsoft Exchange. Продукт доступен для платформ Windows, Mac OS X и Linux (Red Hat, CentOS, SUSE, Debian и Ubuntu), но распространяется только на платной основе (загрузить можно только 30-дневную тестовую версию). Kerio Connect объединяет в себе почтовый сервер, систему фильтрации спама (на базе SpamAssassin), средства для организации коллективной работы, календарь-планировщик, единую адресную книгу (с поддержкой CardDAV), подсистему для архивирования почты и автоматического резервного копирования, модуль для интеграции с MS Outlook, web-ориентированный интерфейс администратора и пользователя.
