Корпорация PGP выпустила новое средство для защиты данных в Linux

Корпорация PGP анонсировала доступность продукта PGP Desktop для платформ Linux и MacOS X. PGP Desktop представляет собой систему для организации шифрования данных на рабочих станциях. Из Linux дистрибутивов поддерживаются Ubuntu Linux и Red Hat Linux. Продукт коммерческий и распространяется по цене от 99$ за одно рабочее место. Продукт включает в себя гибридный криптографический оптимизиатор (Hybrid Cryptographic Optimizer, HCO) позволяющий ускорить выполнение операций по полному шифрованию дисков, USB-брелоков и SSD-накопителей. Управление шифрованием на рабочих станциях в локальной сети и применение новых политик доступа производится централизованно через web-интерфейс администратора.
