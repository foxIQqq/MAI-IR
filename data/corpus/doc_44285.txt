Вышел Turnkey Linux 14.1, набор мини-дистрибутивов для быстрого развертывания приложений

Доступен релиз набора Turnkey Linux 14.1, в рамках которого сформирована коллекция из 106 минималистичных сборок Debian, пригодных для использования в системах виртуализации и облачных окружениях. Средний размер каждой сборки - 250 Мб. Сразу после установки пользователь имеет возможность получить полностью работоспособные из коробки рабочие окружения с LAMP (Linux, Apache, MySQL, PHP/Python/Perl), Ruby on Rails, Joomla, MediaWiki, Wordpress, Drupal, Apache Tomcat, LAPP, Django, MySQL, PostgreSQL, Node.js, Jenkins, Typo3, Plone, SugarCRM, punBB, OS Commerce, ownCloud, MongoDB, OpenLDAP, gitlab, CouchDB и т.д. Управление программным обеспечением производится через специально подготовленный web-интерфейс (для настройки используется Webmin и Adminer). Сборки снабжены системой автоматического резервного копирования и средством для автоматической установки обновлений. В новом выпуске обновлена начинка предлагаемых для установки сборок, в которых задействованы свежие обновления Debian 8.4. Представлена новая сборка Mediaserver, позволяющая развернуть сервер для хранения мультимедийной коллекции и организации потокового вещания. Для доступа к файлам может применяться SMB, SFTP, NFS, WebDAV и rsync. Для просмотра контента можно использовать DLNA или встроенный Youtube-подобный web-интерфейс.
