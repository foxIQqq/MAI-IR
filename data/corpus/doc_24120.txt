Релиз дистрибутива Scientific Linux 5.4

Анонсирован релиз дистрибутива Scientific Linux 5.4, построенного на пакетной базе Red Hat Enterprise Linux 5.4 и дополненного средствами, ориентированными на использование в научных учреждениях. Дистрибутив доступен в сборках для архитектур i386 и x86_64. В отличие от RHEL 5.4 можно отметить следующие дополнения, внесенные разработчиками Scientific Linux 5.4: В состав дистрибутива для драйвера iwlwifi включены дополнительные прошивки (firmware для беспроводных карт Intel серий 5150, 3945, 4945 и 5000. Также добавлены драйверы для беспроводных карт Ralink; Добавлена библиотека cfitsio с реализацией поддержки формата FITS (Flexible Image Transport System), широко используемого в астрономических исследованиях, для языков Си и FORTRAN; В состав дистрибутива включена научная библиотека SciPy, которую можно использовать для научных, инженерных и математических рассчетов, в также для построения графиков и диаграмм, на основе набора статистических данных. Кроме того, состав были включены: пакет визуализации Graphviz; библиотека Numpy, для организации научных расчетов; математическая библиотека FFTW (Fastest Fourier Transform); Интеграция технологии DKMS (Dynamic Kernel Module Support), позволяющей автоматически перестраивать текущие модули ядра с драйверами устройств после обновления версии ядра. Внедрение DKMS позволит автоматически устанавливать драйверы, которые необходимы для работы определенных аппаратных компонент текущей системы, но отсутствуют в стандартном Linux ядре. Для разработчиков, DKMS предоставляет средства создания драйверов без оглядки на особенности различных версий Linux ядра. OpenJDK заменен на пакет Sun Java/JDK 1.6.0p16; Добавлен оконный менеджер IceWM 1.2.37; Добавлена поддержка языков Lua и GNU R (язык для статистических расчетов); Поддержка файловой системы XFS для архитектуры x86_64. В программе установки пока нельзя создать XFS раздел, но установить Scientific Linux на ранее созданный XFS раздел можно; В состав включен набор обучающих приложений для KDE - KDEEDU; В поставку включен код AUFS, альтернативной реализации UnionFS; Вместо консольного почтового клиента Pine используется Alpine; Добавлены дополнительные мультимедиа кодеки (gstreamer-plugins-extras из SL4 и k3b-extras из репозитория livna); Добавлен пакет NdisWrapper, позволяющий использовать в Linux NDIS драйверы из Windows; OpenAFS 1.4.11 - открытая реализация распределенной ФС Andrew File System; Yum обновлен до версии 1.1.16. Добавлена оболочка для управления пакетами Yumex; Отдельно отмечается, что ряд дополнительных возможностей Scientific Linux теперь поставляется в составе RHEL, что избавляет разработчиков Scientific Linux от поддержки таких дополнений, как драйверы для беспроводных карт Atheros (драйвер madwifi), драйвер r1000 для RealTek Ethernet контроллеров, подсистема FUSE (но, в SL оставлены модули fuse-smb и fuse-sshfs).
