Проект по созданию ультрабука для Linux-разработчиков Sputnik перешёл на бета-стадию

Начатый open-source энтузиастами из компании Dell проект Sputnik понемногу продвигается вперед. Компания Dell открыла программу бета-тестирования, в рамках которой участники со скидкой смогут приобрести предварительную версию Sputnik (Dell XPS13 + Ubuntu 12.04). Число заявок ограничено. Цель проекта - выпуск в продажу ультрабука для разработчиков с Ubuntu Linux на борту. Продукт рассчитан прежде всего на разработчиков открытых web-приложений и программного обеспечения для мобильных и облачных систем. Состав продукта достаточно прост - ультрабук Dell (13.3-дюймовый экран, CPU Intel Core i5-2467M (1.60 GHz), видеоподсистемой Intel HD 3000, 4 Гб ОЗУ и 128 Гб SSD-накопителем. Вес 1.36 кг) + базовая минимальная ОС Ubuntu 12.04 + драйверы для оборудования + репозиторий с набором "профилей разработчика", с помощью которых можно быстро сформировать среду для разработки на той или иной платформе. В первую очередь создатели проекта работают над инструментарием для управления профилями и непосредственно профилями для Android, Ruby и JavaScript. В дальнейшем к работе над новыми профилями должно подключиться сообщество проекта.
