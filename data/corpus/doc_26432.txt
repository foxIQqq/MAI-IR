Компания Novell обогнала Red Hat по числу сертифицированных приложений

Компания Novell сообщила о преодолении отметки в 5 тыс. сертифицированных для SUSE Linux приложений от независимых поставщиков программного обеспечения (ISV). По заявлению Novell число сертифицированных продуктов для SUSE примерно в два раза выше, чем достижения на данном поприще основного конкурента - Red Hat Enterprise Linux. Среди факторов, способствующих поддержке ISV-поставщиков отмечается создание системы по подготовке виртуальных серверов SUSE Studio, развитие программы SUSE Appliance Program, нацеленной на подготовку полностью сконфигурированных виртуальных серверов, интеграции в SUSE Linux Enterprise Server средств для развертывания cloud-окружений, инициативу по улучшению совместимости с продуктами Microsoft, развитие проекта Mono, формирование группы инженеров для оказания помощи в портировании продуктов независимых вендоров.
