Выпуск Chrome OS 100

Доступен релиз операционной системы Chrome OS 100, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 100. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 100 доступна для большинства актуальных моделей Chromebook. Исходные тексты распространяются под свободной лицензией Apache 2.0. Кроме того, продолжается тестирование Chrome OS Flex, редакции для Chrome OS использования на обычных компьютерах. Энтузиастами также формируются неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Основные изменения в Chrome OS 100: Предложена новая реализация панели приложений (Launcher), в которой модернизировано оформление и расширены возможности поиска. Панель со списком приложений теперь выводится сбоку экрана, оставляя больше места для открытых окон. Предоставлена возможность группировки приложений в произвольной форме. Переработано представление результатов поиска ответов на произвольные вопросы - кроме предпросмотра результатов обращения к поисковой системе, теперь выводятся и информационные блоки, позволяющие сразу получить нужные сведения не переходя в браузер. Помимо поиска приложений и файлов из Launcher также можно осуществлять поиск горячих клавиш и охватывать поиском открытые в браузере вкладки и окна. В приложение для работы с камерой добавлены инструменты для создания анимированных GIF-изображений. При включении переключателя "GIF" в режиме съёмки видео автоматически будет записан и преобразован в формат GIF 5-секундный ролик. Данный ролик можно сразу отправить на email, передать в другое приложение или отправить на Android-смартфон при помощи сервиса Nearby Share. Функция голосового ввода текста расширена возможностью редактирования содержимого. В процессе редактирования распознаются такие голосовые команды, как "удалить" для удаления последней буквы, "перейти к следующему/предыдущему символу" для изменения позиции курсора, "отменить" для отмены изменения и "выделить всё" для выделения текста. В дальнейшем число голосовых команд будет расширено. Для включения голсового ввода можно использовать клавиатурную комбинацию "Search + d" или настройки в секции "Settings > Accessibility > Keyboard and text input". Расширено число устройств, на которых можно установить окружение Chrome OS Flex, позволяющее использовать Chrome OS на обычных компьютерах, например, для продления жизненного цикла старых ПК и ноутбуков, снижения издержек (например не нужно платить за ОС и дополнительное ПО, такое как антивирусы) или повышения безопасности инфраструктуры. С момента первого анонса работа с Chrome OS Flex подтверждена для более ста устройств. Предоставлена возможность назначения своих пиктограмм и названий для сайтов, предлагаемых для использования в управляемых сеансах с ограниченным набором доступных сайтов (Managed Session). В консоль Google Admin добавлен новый отчёт со сводкой устройств, на которые требуется обратить внимание, например, из-за проблем с производительностью. Для передачи расширенной информации о состоянии устройства при включении централизованного управления предложен новый API Chrome Management Telemetry.
