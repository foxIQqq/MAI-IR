Вышла вторая альфа-версия Mandriva Linux 2010.1

Доступна для тестирования вторая альфа версия дистрибутива Mandriva Linux 2010.1, релиз которого запланирован на 3 июня. Тестовая версия доступна в DVD-сборке "Free" (без проприетарных драйверов в комплекте) для архитектур x86-32 и x86-64. Из особенностей Mandriva Linux 2010 Spring Alpha 2 можно отметить: В состав первой альфа версии включены приложения: KDE 4.4-RC3, GNOME 2.29.6, Firefox 3.6, OpenOffice.org 3.2 beta 3, Qt 4.6.1, Amarok 2.2.2, KTorrent 4.0beta1, Digikam 1.1, X.org 7.5, Mesa 7.7, Linux ядро 2.6.33 rc6, Glibc 2.11, ntfs-3g 2010.01.16, снапшот свободного Flash плеера Gnash 0.9.7, Spamassassin 3.3.0; Для организации начального этапа загрузки вместо mkinitrd теперь используется dracut (образ ram-диска с возможностью динамического обновления содержимого); В состав дистрибутива добавлен пакет USB/IP для организации доступа к USB устройствам на других машинах по сети; При сборке пакетов на этапе связывания задействованы опции "-Wl,-O1", что позволяет уменьшить время запуска приложений связанных с большим числом библиотек; Продолжение интеграции в десктоп-окружение KDE и пользовательские приложения системы семантических связей Nepomuk. Осуществлен переход Nepomuk на использование бэкенда для хранения данных Virtuoso, который позволяет ускорить работу с тегами и поддерживает индексацию хранимых данных; Значительно улучшена организация работы гостевых аккаунтов, позволяющих организовать одноразовый доступ к машине случайных людей, работа которых будет производиться в изолированной временной домашней директории, очищаемой при выходе гостя из системы. Например, для гостевых аккаунтов теперь используется специальный ограниченный shell, закрыт доступ к файлам других пользователей, заблокирован запуск административных системных приложений; Включение в комплект тестовой версии web-браузера Chromium; В качестве базового свободного драйвера для видеокарт NVidia вместо драйвера nv теперь используется Nouveau, в котором реализована поддержка переключения видеорежимов на уровне ядра, 2D и 3D акселерация, поддержка RandR 1.2 для упрощения управления несколькими мониторами; Демон для ведения системных логов Sysklogd заменен на более продвинутую систему rsyslog, поддерживающую возможность хранения логов в СУБД, отправку email уведомлений при обнаружении в логах заданных масок, перенаправление логов на другой сервер с использованием TCP; В поставку интегрирован пакет мониторинга Munin 1.4 с плагинами для отслеживания работы MySQL, PostgreSQL, Asterisk и т.п. Некоторые планы: Поддержка интерфейса KDE для нетбуков; Упрощение процесса шифрования содержимого домашних директорий; Улучшение работы утилит для управления установкой программного обеспечения, проведения обновлений и поиска нужных программ; Интеграция в дистрибутив наработок GNOME 3 и доступность для тестирования GNOME Shell, GNOME Color Manager, Gnome Activity Journal; Наращивание возможностей cloud-сервисов для пользователей дистрибутива и интеграция web-приложений на десктоп;
