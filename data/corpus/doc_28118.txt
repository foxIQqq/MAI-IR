Ubuntu Server Edition сертифицирован для серверов Dell PowerEdge

Компании Dell и Canonical объявили о завершении двухгодичной работы по сертификации дистрибутива Ubuntu Server Edition на серверах линейки Dell PowerEdge. Сертификация гарантирует, что у пользователей Dell PowerEdge при промышленной эксплуатации не возникнет проблем, при использовании на данном оборудовании Ubuntu. Для организации локального и удаленного управления серверами для Ubuntu адаптирован пакет Dell OpenManage 6.3, который позволяет отслеживать состояние системы, отображать данные с температурных датчиков и различных сенсоров, управлять сетевой подсистемой и RAID-контроллером, изменять параметры BIOS, обновлять прошивки, контролировать потребление питания, оценивать состояние памяти и слотов расширения. Сообщается, что несмотря на то, что Dell не предустанавливает Ubuntu на свои сервера и официально не оказывает услуги поддержки для данной системы, полноценный сервис коммерческой поддержки может быть оказан через заключение контракта с компанией Canonical.
