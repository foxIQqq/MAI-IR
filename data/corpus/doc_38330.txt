Вышел GeeXBoX 3.1, минималистичный Linux дистрибутив для создания медиацентра

Представлен новый выпуск специализированного Linux-дистрибутива GeeXBoX 3.1, предназначенного для создания медиацентров и домашних кинотеатров. Дистрибутив развивается с 2002 года и базируется на собственном сборочном инструментарии. Новая версия GeeXBoX доступна в сборках для платформ i386, x86_64, Cubox v1/v2/Pro, RaspberryPi, Utilite и CubieBoard 1. Размер загрузочного iso-образа 160 Мб. Силами разработчиков дистрибутива развивается несколько известных мультимедийных проектов, таких как медиацентр Enna и фреймворк для создания оптимизированных минидистрибутивов OpenBricks. Загрузившись с USB-накопителя, SD-карты или по сети, GeeXboX предлагает готовый к использованию медиацентр, позволяющий просматривать и прослушивать любые типы медиаконтента (звук, видео, фотографии) из разнообразных источников (CD, DVD, Bluray, HDD, USB, Samba, NFS, UPnP, DLNA, HTTP, FTP, SSH, SHOUTcast). В GeeXBoX из коробки доступны средства акселерации выполнения графических операций для оборудования, поддерживающего OpenGL и OpenGL ES, а также ускорения декодирования видео через задействование VDPAU, VA-API и Crystal HD. Ключевые новшества GeeXBoX 3.1 Интеграция фронтэнда для медиацентра XBMC 12.2; Поддержка записи и просмотр видео с DVB-адаптеров для цифрового телевидения с использованием vdr и tvheadend Поддержка Full HD видео на платформах Utilite, Cubox и RaspberryPi; Прекращение поддержки платформ PandaBoard, ux500 и nvidia Tegra2; Обновление системных компонентов: ядро Linux 3.10.9, Systemd 206, BusyBox 1.21.1, Connman Network Manager 1.9, X.org Server 1.14.1, Mesa 9.2.2, OPKG Package Manager 0.18, новые версии видеодрайверов Intel, NVIDIA и AMD.
