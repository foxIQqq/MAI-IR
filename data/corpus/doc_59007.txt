Выпуск дистрибутива Ubuntu 23.04

Опубликован релиз дистрибутива Ubuntu 23.04 "Lunar Lobster", который отнесён к промежуточным выпускам, обновления для которых формируются в течение 9 месяцев (поддержка будет осуществляться до января 2024 года). Установочные образы созданы для Ubuntu, Ubuntu Server, Lubuntu, Kubuntu, Ubuntu Mate, Ubuntu Budgie, Ubuntu Studio, Xubuntu, UbuntuKylin (редакция для Китая), Ubuntu Unity, Edubuntu и Ubuntu Cinnamon. Основные изменения: Рабочий стол обновлён до выпуска GNOME 44, в котором продолжен перевод приложений на использование GTK 4 и библиотеки libadwaita (на GTK4 среди прочего переведена пользовательская оболочка GNOME Shell и композитный менеджер Mutter). В диалог выбора файлов добавлен режим отображения содержимого в форме сетки значков. Внесены многочисленные изменения в конфигуратор. В меню быстрого изменения настроек добавлена секция для управления Bluetooth. В панели Ubuntu Dock на пиктограммах приложений обеспечен показ метки со счётчиком непросмотренных уведомлений, сгенерированных приложением. В число официальных редакций Ubuntu включена сборка Ubuntu Cinnamon, предлагающая пользовательское окружение Cinnamon, построенное в классическом стиле GNOME 2. Возвращена официальная сборка Edubuntu, предоставляющая подборку образовательных программ для детей разного возраста. Добавлена новая минималистичная сборка Netboot, размером 143 МБ. Сборку можно использовать для записи на CD/USB или для динамической загрузки через UEFI HTTP. Сборка предоставляет текстовое меню, при помощи которого можно выбрать интересующую редакцию Ubuntu, установочный образ для которой будет загружен в оперативную память. Для установки Ubuntu Desktop по умолчанию задействован новый инсталлятор, реализованный в форме надстройки над низкоуровневым инсталлятором curtin, который уже применяется в инсталляторе Subiquity, используемом по умолчанию в Ubuntu Server. Новый инсталлятор для Ubuntu Desktop написан на языке Dart и использует для построения пользовательского интерфейса фреймворк Flutter. Оформление нового инсталлятора спроектировано с учётом современного стиля рабочего стола Ubuntu и рассчитано на предоставление единого процесса установки для всей линейки продуктов Ubuntu. Старый инсталлятор доступен в форме опции на случай возникновения непредвиденных проблем. В разряд стабильных переведён snap-пакет с клиентом Steam, который предоставляет готовое окружение для запуска игр, позволяющее не смешивать необходимые для игр зависимости с основной системой и получить преднастроенное актуальное окружение, не требующее дополнительной настройки. В пакет включены свежие версии Proton, Wine и актуальные версии зависимостей, необходимые для запуска игр (пользователю не требуется выполнение ручных операций, установка набора 32-разрядных библиотек и подключение PPA-репозиториев с дополнительными драйверами Mesa). Игры запускаются без доступа к системному окружению, что создаёт дополнительный бастион защиты на случай компрометации игр и игровых сервисов. Усовершенствована работа с обновлениями пакетов в формате snap. Если раньше пользователю выводилось уведомление о наличии обновления snap-пакетов, но для установки требовалось запускать Ubuntu Software, совершать манипуляции в командной строке или ждать пока обновление установится автоматически, то теперь обновления загружаются в фоновом режиме и применяются сразу после закрытия связанного с ними приложения (при желании установку обновлений можно приостановить). В Ubuntu Server задействована новая редакция инсталлятора Subiquity, позволившая загружать серверные сборки в live-режиме и быстро устанавливать Ubuntu Desktop для пользователей серверов. В системе Netplan, применяемой для хранения настроек сетевых интерфейсов, добавлена новая команда "netplan status" для показа текущего состояния сети. Изменено поведение сопоставления физических сетевых интерфейсов при помощи параметра "match.macaddress", который сверяется со значением PermanentMACAddress, а не MACAddress. Добавлена поддержка аутентификации с использованием Azure Active Directory (Azure AD), что позволяет подключаться к Ubuntu пользователям Microsoft 365 (M365), используя те же параметры входа, что применяются в M365 и Azure. Официальные редакции Ubuntu прекратили поддержку Flatpak в базовой поставке и по умолчанию исключили из базового окружения deb-пакет flatpak и пакеты для обеспечения работы с форматом Flatpak в Центре установки приложений. Пользователи ранее установленных систем, в которых использовались пакеты Flatpak, сохранят возможность использования данного формата после обновления до Ubuntu 23.04. Пользователи не применявшие Flatpak после обновления по умолчанию будут иметь только доступ к Snap Store и штатным репозиториям дистрибутива, при желании использовать формат Flatpak следует отдельно установить пакет для его поддержки из репозитория (deb-пакет flatpak) и при необходимости активировать поддержку каталога Flathub. Ядро Linux обновлено до выпуска 6.2. Обновлены версии Mesa 22.3.6, systemd 252.5, PulseAudio 16.1, Ruby 3.1, PostgreSQL 15, QEMU 7.2.0, Samba 4.17, CUPS 2.4.2, Firefox 111, LibreOffice 7.5.2, Thunderbird 102.9, VLC 3.0.18, BlueZ 5.66, NetworkManager 1.42, Pipewire 0.3.65, Poppler 22.12, xdg-desktop-portal 1.16, cloud-init 23.1, Docker 20.10.21, Containerd 1.6.12, runc 1.1.4, Dnsmasq 2.89, libvirt 9.0.0, Open vSwitch 3.1.0. Сформированы пакеты с LibreOffice для архитектуры RISC-V. Включены AppArmor-профили для защиты rsyslog и isc-kea. Расширены возможности сервиса debuginfod.ubuntu.com, позволяющего при отладке поставляемых в дистрибутиве программ обойтись без установки отдельных пакетов с отладочной информацией из репозитория debuginfo. При помощи нового сервиса пользователи могут динамически загружать отладочные символы с внешнего сервера непосредственно во время отладки. В новой версии обеспечена индексация и обработка исходных текстов пакетов, что позволяет обойтись без отдельной установки пакетов с исходными текстами через "apt-get source" (исходные тексты прозрачно будут загружены отладчиком). Добавлена поддержка отладочных данных для пакетов из PPA-репозиториев (пока индексируются только ESM PPA (Expanded Security Maintenance)). В Kubuntu предложены рабочий стол KDE Plasma 5.27, библиотеки KDE Frameworks 5.104 и набор приложений KDE Gear 22.12. Обновлены версии Krita, Kdevelop, Yakuake и многих других приложений. В Ubuntu Studio по умолчанию задействован мультимедийный сервер PipeWire. Обновлены версии программ: RaySession 0.13.0, Carla 2.5.4, lsp-plugins 1.2.5, Audacity 3.2.4, Ardour 7.3.0, Patchance 1.0.0, Krita 5.1.5, Darktable 4.2.1, digiKam 8.0.0 Beta, OBS Studio 29.0.2, Blender 3.4.1, KDEnlive 22.12.3, Freeshow 0.7.2, OpenLP 3.0.2, Q Light Controller Plus 4.12.6, KDEnlive 22.12.3, GIMP 2.10.34, Ardour 7.3.0, Scribus 1.5.8, Inkscape 1.2.2, MyPaint v2.0.1. В Ubuntu MATE задействован выпуск MATE Desktop 1.26.1, а панель MATE Panel обновлена до ветки 1.27 и включает дополнительные патчи. В Ubuntu Budgie задействован выпуск рабочего стола Budgie 10.7. Полностью переработана система выполнения действий через подведение указателя мыши к углам и краям экрана. Добавлена новая система управления мозаичной компоновкой через перемещение окна к краю экрана. В Lubuntu по умолчанию предложено пользовательское окружение LXQt 1.2. Инсталлятор обновлён до Calamares 3.3 Alpha 2. Для Firefox вместо deb-пакета задействован snap. В Xubuntu рабочий стол Xfce обновлён до выпуска 4.18. В состав включён мультимедийный сервер Pipewire. Обновлены версии Catfish 4.16.4, Exo 4.18.0, Gigolo 0.5.2, Mousepad 0.5.10, Ristretto 0.12.4, Thunar File Manager 4.18.4, Xfce Panel 4.18.2, Xfce Settings 4.18.2 Xfce Task Manager 1.5.5, Atril 1.26.0, Engrampa 1.26.0. Добавлена урезанная сборка Xubuntu Minimal, которая занимает 1.8 ГБ вместо 3 ГБ. Новая сборка будет полезна тем, кто предпочитает иной набор приложений, чем в базовой поставке - набор устанавливаемых приложений пользователь может выбрать и загрузить из репозитория во время установки дистрибутива.
