Проект OpenVZ Web Panel обновился до версии 2.0

Вышла вторая версия веб-среды для управления виртуальными машинами на основе OpenVZ - OpenVZ Web Panel. Данный проект занял 3-е место в номинации «Индивидуальный проект» конкурса «Лучший свободный проект России — 2010», проводимый журналом «Linux Format». Из добавленных в новой версии улучшений можно отметить: Возможность удаленного выполнения операций в оболочке при помощи специального API на базе формата JSON; Поддержка биллинговой системы WHMCS; Функции для управления IP-адресами; Автоматическая синхронизация состояния виртуальных серверов; Выполнение миграции виртуальных окружений с одного физического сервера на другой; Возможность формирования виртуальных серверов на основе шаблонов; Добавлены файлы с переводами на украинский язык; Поддержка платформы Android; Возможность просмотра более 15 присоединенных физических серверов; Поддержка аутентификации через LDAP; Возможность использования SSL-шифрования при связи с серверами.
