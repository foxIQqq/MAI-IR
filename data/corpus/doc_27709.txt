Вышел дистрибутив Lunar Linux 1.6.5

После двух лет разработки анонсирован выпуск дистрибутива Lunar Linux 1.6.5, отличающегося использованием специфичного пакетного менеджера lin, выполняющего сборку программы из исходных текстов для каждого устанавливаемого пакета. Пакеты выполнены в виде модулей, представленных в иерархическом каталоге Moonbase, напоминающей систему портов FreeBSD. Инсталляционный образ Lunar 1.6.5 занимает 354 Мб и содержит базовое окружение, пригодное только для сборки дополнительных пакетов из Moonbase. Сборка программ не представляет сложности, например, для установки пользовательского окружения на базе KDE4 достаточно выполнить "lin kde4", GNOME - "lin gnome2", Xfce - "lin xfce4". Улучшения, добавленные в релизе Lunar Linux 1.6.5: Загрузочный iso-образ представлен в гибридном виде, пригодном для загрузки с USB Flash; Осуществлен переход на Linux-ядро 2.6.35.3 и системную библиотеку glibc 2.11.2; Добавлена поддержка файловой системы EXT4; В состав LiveCD добавлены пакеты с OpenSSH, консольным оконным менеджером screen и коллекцией прошивок для беспроводных карт; Добавлены новые модули: yasm, gperf, libusb-compat, usbutils, xz, gmp, mpfr, cron-common, logrotate, rsyslog, iwlwifi-3945-ucode, iwlwifi-4965-ucode, iwlwifi-5000-ucode, screen, slang и mc Для ведения логов по умолчанию задействован Rsyslog.
