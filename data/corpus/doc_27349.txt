Проект VideoLAN представил библиотеку для работы с форматом Blu-ray

Представлена новая библиотека libbluray, развиваемая под эгидой проекта VideoLAN. Библиотека создана разработчиками ресурса Doom9, нацеленного на подготовку инструментария для резервного копирования DVD-дисков, и предназначена для интеграции в мультимедиа приложения и видеоплееры функций по проигрыванию дисков в формате Blu-Ray. Библиотека соблюдает требования DRM (Digital rights management) и не реализует методов для обхода коммерческих методов защиты, поэтому для проигрывания защищенных с использованием технологий AACS и BD+ дисков требуется подключение дополнительных проприетарных компонентов. Среди особенностей библиотеки отмечается: Переносимость: в настоящее время поддерживаются платформы GNU/Linux, Windows и MacOS X, при этом число зависимостей, необходимых для работы libbluray, минимально; Код библиотеки распространяется под свободной лицензией LGPL, не мешающей интеграции библиотеки с закрытыми проектами; Кроме базовых функций библиотека поддерживает такие возможности, как навигацию, парсинг списка воспроизведения, работу с меню и поддержку базы данных в формате BD-J; Код libbluray можно использовать не опасаясь на появление претензий, связанных с нарушением DRM; Библиотека находится на пути становления, проект предоставляет заинтересованным энтузиастам возможность для экспериментов и разработки улучшений.
