Вышла вторая альфа версия Mandriva Linux 2010

Вышла вторая альфа версия Mandriva Linux 2010, доступная в виде 32- и 64-рязрядной DVD Free сборки без проприетарных драйверов в комплекте. Дата выхода релиза Mandriva Linux 2010 перенесена на две недели, относительно первоначально запланированного времени, и ожидается 3 ноября. Во второй альфа версии осуществлено обновление следующих программ: KDE 4.3 RC 3, GNOME 2.27.5, Xfce 4.6.1, Linux ядро 2.6.31-rc4, Xorg 1.6.3 с последней версией видеодрайверов Intel в комплекте, KOffice 2.0.1, Amarok 2.1.1, Digikam 1.0 beta 3, php 5.3.0, apache 2.2.12. В состав репозитория пакетов включена программа для ведения заметок GNote (аналог TomBoy на языке C++). Добавлен GUI-интерфейс tomoyo-gui для настройки нового фремворка для задания политик безопасности Tomoyo, заменившего собой AppArmor в Mandriva. Основные направления развития Mandriva 2010: Продолжение работы по уменьшению времени загрузки; Переработка загрузочного меню Grub и начального меню инсталлятора (режим защиты от сбоев (failsafe), загрузка в консольном режиме (init3), восстановление после сбоя, проверка целостности инсталляционного диска); Переход на версию набора компиляторов GCC 4.4 для пересборки пакетов; Задействование режима бесшовной графической загрузки через использование Plymouth, для карт не поддерживающих переключение видеорежимов на уровне ядра по прежнему будет использован Splashy; Переход на использование, появившихся в Linux ядре 2.6.30, модулей безопасности TOMOYO Linux, позволяющих привязать политику безопасности процесса к файловому пути, способом похожим с AppArmor. Для управления будет создан специальный GUI интерфейс, который будет интегрирован в интерфейс msec (Mandriva Security Center); Поддержка создания гостевых аккаунтов на лету, при возникновении необходимости предоставления доступа к машине случайному пользователю; Создания набора пакетов с окружением Moblin для субноутбуков, который будет автоматически предлагаться к установке на слабых машинах и при низких экранных разрешениях; В апплет mdkonline будет добавлена возможность информирования об окончании времени поддержки дистрибутива и выпуска обновлений, связанных с безопасностью; Автоматическое определение наличия у пользователя локальных urpmi репозиториев; Linux ядро для построения управляющего Xen окружения (Dom0) будет обновлено с версии 2.6.18 до 2.6.27; Улучшение поддержки гибридных iso-сборок, подходящих как для Flash, так и для CD/DVD; Мультимедиа центр, для управления коллекцией фотографий, музыки и поддержания видеотеки, Elisa переименован в проект Moovida; KDE 3 больше не будет входить в состав дистрибутива как полное десктоп окружение, некоторые пакеты на базе KDE 3 останутся, но в будущем поддержка будет полностью прекращена, как только для KDE 4 будут портированы все самые интересные приложения; Основные программные компоненты будущей версии: Linux ядро 2.6.31, X.org 7.5 (xorg-server 1.7.0 или 1.6.2+ в случае задержки выхода), KDE 4.3.x, GNOME 2.28, GCC 4.4.x, Glibc 2.10.1 или 2.10.2+, Go-OO сборка OpenOffice.org 3.1. Перевод процесса сборки дистрибутива на использование нового инструментария bcd, вместо ранее используемого mkcd, что должно положительно сказаться на качестве дистрибутива из-за значительного упрощения и усиления процесса тестирования. Splashy заменён на Plymouth, что позволило организовать графическую загрузку без мерцания и лишних переключений видеорежимов; Полностью переписан инструмент Netprofile, предназначенный для настройки профилей сетевых соединений; Tomoyo-gui - первая,написанная сотрудниками Mandriva, графическая оболочка для системы безопасности Tomoyo, появившейся в Linux ядре 2.6.30 и позволяющая привязать политику безопасности процесса к файловому пути, способом похожим с AppArmor. В состав репозиторий contrib включен Python 3, но из соображений совместимости по умолчанию по прежнему присутствует Python 2.6
