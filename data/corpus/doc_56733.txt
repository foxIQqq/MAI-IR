Выпуск операционной системы Solaris 11.4 SRU42

Компания Oracle опубликовала обновление операционной системы Solaris 11.4 SRU 42 (Support Repository Update), в котором предложена серия очередных исправлений и улучшений для ветки Solaris 11.4. Для установки предложенных в обновлении исправлений достаточно выполнить команду 'pkg update'. В новом выпуске: Добавлены пакеты с новой веткой библиотеки OpenSSL 3.0. В одном из следующих выпусков OpenSSL 3.0 будет задействован по умолчанию и предложен для миграции с OpenSSL 1.0.2 и 1.1.1. Добавлены пакеты с системой управления конфигурацией Ansible 2.10. Реализованы новые команды "ldm console -e" для указания escape-символа и "ldm unbind -a" для выполнения операции для всех доменов. Добавлена поддержка live-миграции гостевых систем в логических виртуальных окружениях (LDoms) между серверами на базе процессоров SPARC M7, T7, S7, M8 и T8 (cross-CPU migration). В mdb добавлена возможность переключения на отладку дочерних процессов, порождённых при помощи системных вызовов fork и spawn, не прекращая отладку родительского процесса. В стандартную Си-библиотеку libc добавлены функции freezero и freezeroall, обнуляющие содержимое освобождаемой памяти. Прекращена установка исполняемого бита на объектные файлы и разделяемые библиотеки. В команду "split -b" добавлена поддержка дополнительных размерностей (g - гигабайт, t - терабайт) и нецелых значений ('.5t'). В состав включены пакеты Zipp, typing-extensions (для Python), importlib-metadata, Sphinx, Alabaster и Docutils. В coreadm для хранения core-файлов задействован каталог /var/cores/. Добавлена поддержка локали C.UTF-8. Добавлены команды "zfs get -I state" и "zpool status/import -s". В команды plimit, pmadvise и pmap добавлены опции "-h" и "--scale". В библиотеку libkmip добавлена поддержка протокола KMIP 1.4 (Key Management Interoperability Protocol). Обновлены версии программ с устранением уязвимостей: Apache httpd 2.4.52, Java 7u331/8u321, ModSecurity 2.9.5, MySQL 5.7.36, NSS 3.70, Samba 4.13.14, Django 2.2.25/3.2.10, fetchmail 6.4.22, libexif 0.6.24, ncurses 6.3, webkitgtk 2.34.1, g11n/im-ibus, kernel/streams, library/gd2, library/polkit, utility/imagemagick, utility/junit, utility/mailman, utility/php, utility/pip, utility/vim и x11/xorg-server. Обновлены многие пакеты, в том числе GNOME 41, HPLIP 3.21.8, gtk 3.24.30, meson 0.59.2, mutt 2.1.3, nano 5.9.
