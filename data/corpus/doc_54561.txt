Выпуск видеоредакторов Flowblade 2.8 и Shotcut 21.01.29

Состоялся релиз многотрековой системы нелинейного видеомонтажа Flowblade 2.8, позволяющей компоновать фильмы и видеоролики из набора отдельных видео, звуковых файлов и изображений. Редактор предоставляет средства для обрезки клипов с точностью до отдельных кадров, их обработки при помощи фильтров и многоуровневой компоновки изображений для встраивания в видео. Имеется возможность произвольного определения порядка применения инструментов и корректировки поведения шкалы времени. Код проекта написан на языке Python и распространяется под лицензией GPLv3. Сборки подготовлены в формате deb. Для организации редактирования видео применяется фреймворк MLT. Для обработки различных форматов видео, звука и изображений применяется библиотека FFmpeg. Интерфейс построен с использованием PyGTK. Для математических вычислений задействована библиотека NumPy. Для обработки изображений применяется PIL. Возможно использование плагинов с реализацией видео эффектов из коллекции Frei0r, а также звуковых плагинов LADSPA и фильтров изображений G'MIC. В новом выпуске: Добавлено подменю "View->Panel Placement" для выбора позиции размещения панели на экранах с разрешением 1680x1050 и выше. Предложены две новые тёмные темы оформления - Flowblade Neutral с нейтральным фоном и Flowblade Gray с очень лёгким смещением оттенка в сторону синего цвета. Ранее предлагаемая по умолчанию тема оформления оставлена под именем Flowblade Blue. Предоставлена возможность изменения порядка следования элементов в средней панели, включаемая через меню "View->Middlebar layout->Free Bar" и "View->Middlebar layout->Configure Free Bar" Добавлена поддержка изменения горячих клавиш. Добавлен виджет Tool Dock для выбора инструментов редактирования по шкале времени, который можно использовать вместо выпадающего меню средней панели. Обновлена панель с фильтрами. Панели редактирования фильтров теперь располагаются в стековой форме. Панель Filter Selection перенесена в правую часть шкалы времени и позволяет пользователю добавлять фильтры через двойной клик на элемент фильтра или перетаскивая фильтры на клип мышью. Для средней панели предложен дополнительный набор цветных пиктограмм. Обновлены переводы на русский и украинский языки. Дополнительно можно отметить обновление видеоредактора Shotcut 21.01.29, который развивается автором проекта MLT и использует данный фреймворк для организации редактирования видео. Поддержка форматов видео и звука реализована через FFmpeg. Возможно использование плагинов с реализацией видео и аудио эффектов, совместимых с Frei0r и LADSPA. Из особенностей Shotcut можно отметить возможность многотрекового редактирования с компоновкой видео из фрагментов в различных исходных форматах, без необходимости их предварительного импортирования или перекодирования. Имеются встроенные средства для создания скринкастов, обработки изображения с web-камеры и приёма потокового видео. Для построения интерфейса применяется Qt5. Код написан на C++ и распространяется под лицензией GPLv3. Выпуск Shotcut 21.01.29 примечателен добавлением поддержки кодирования и декодирования видео в формате AV1. В новой версии также проведена оптимизация производительности многопоточных операций смешивания видеодорожек и увеличена скорость работы многих фильтров. Завершён перевод интерфейсов для работы со шкалой времени, фильтрами и ключевыми кадрами на использование API Qt Quick Controls 2. Обновлены версии MLT 6.24.0 и Qt 5.15.2. Добавлен расширенный режим настройки преобразования видео перед редактированием, в котором можно настроить параметры HDR, цветового пространства BT.709, устранения чересстрочности и переопределения частоты кадров. В состав включён новый фильтр подавления шума "Quantization".
