Вышла первая тестовая сборка веб-браузера Opera для Linux без Qt

В качестве подарка на новый год команда разработчиков компании Opera выпустила первую тестовую сборку веб-браузера Opera для Linux/FreeBSD без использования библиотеки Qt. Вместо Qt пользовательский интерфейс Opera формируется напрямую с помощью библиотеки Xlib. В настоящий момент интеграция с KDE отсутствует, но уже в рабочем состоянии находится интеграция со средой рабочего стола Gnome. Данная тестовая версия Opera впервые для Unix'ов поставляется с JavaScript движком Carakan, который стал работать в некоторых тестах на порядок быстрее предыдущего движка, а также браузер стал поддерживать HTML5 тэг Video. Бинарные сборки доступны только в виде tar пакетов для платформ x86 и x86-64 для ОС Linux и FreeBSD.
