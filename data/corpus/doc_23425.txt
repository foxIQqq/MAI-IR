Компания id Software возвращается к поддержке Linux в новых игровых движках

Компания id Software, ранее планировавшая прекратить поддержку платформы Linux в игровом движке пятого поколения, изменила свою позицию - движок "Tech 5 Engine" (на его базе будут основаны игры Doom 4 и Rage) скорее всего будет портирован в Linux. Данную информацию обнародовал в своем блоге Тимоти Бессет (Timothee Besset), работающий в id Software и занимающийся портированием продуктов для платформ Linux и Mac OS X. Не последнюю роль, повышающую интерес к Linux, послужил тот факт, что за несколько недель появления Linux поддержки в online-игре Quake Live, число Linux пользователей данного сервиса уже составляет ориентировочно 5%. О целесообразности поддержки в играх Linux говорит и заявление студии Koonsolo Games, разработавшей игру Mystic Mine, о неожиданно высокой активности покупателей с ОС Linux. Доля продаж Linux версии игры занимает 34% продаж, в то время как Mac OS X - 42%, а Windows - 23%. Если рассмотреть коммерческую эффективность платформы с точки зрения процента проданных копий от числа посетителей сайта, то Linux лидирует с показателем 0.43%, для Mac OS X - 0.39%, Windows - 0.19%.
