Доступна первая тестовая версия Thunderbird 3.3

Вышла первая альфа-версия почтового клиента Thunderbird 3.3. Новая версия построена на базе нового браузерного движка Mozilla - Gecko 2.0, используемого в Firefox 4. Из других улучшений можно отметить: Появление нового менеджера для управления дополнениями (Addon Manager) и реализация нового API для написания расширений; Реализовано отображение размера приложений к письму, рядом с самими приложениями; Налажено корректное отображение пользовательского интерфейса на платформах Windows Vista/Windows 7; Начато формирование универсальных 32/64-разрядных сборок для MacOS X; Внесение более 190 исправлений, направленных на повышение стабильности и производительности.
