Тестовые версии Flash Player 10.1-rc2 и Opera 10.52-rc2

Компания Adobe выпустила второй кандидат в релизы Flash Player 10.1 для платформ Linux, Windows и Mac OS X. Одновременно началось тестирование бета-версии Flash Player 10.1 для мобильной платформы Android. В будущем ожидаются сборки для платформ Windows Mobile, Symbian S60, MeeGo (Moblin+Maemo) и Palm webOS. Список основных новшеств можно посмотреть в тексте анонса прошлой тестовой версии. Кроме того, можно отметить выход очередной тестовой версии web-браузера Opera 10.52 для Unix-подобных операционных систем. Отмечается, что для платформы Linux скоро будет выпущена первая бета-версия, а для Mac OS X - финальный релиз. В версии для Linux еще не реализована поддержка печати, наблюдаются проблемы с настройкой шрифтов и обработкой MIME-типов. Главные улучшения Opera 10.5: вместо использования библиотеки Qt, пользовательский интерфейс формируется с помощью низкоуровневой библиотеки Xlib; интегрирован новый JavaScript-движок Carakan, отличающийся значительным приростом производительности; добавлена поддержка HTML5 тэга Video. Более полный список новшеств можно посмотреть в анонсе первой тестовой версии.
