Обновление антивирусного Live-дистрибутива Virtual Antivirus Recheck

Выпущена новая версия основанного на OpenSuse 11.2 дистрибутива ViAvRe AntiVirus Recheck 04.10, предназначенного для поиска вирусов и программ-шпионов. В составе дистрибутива поставляется 5 антивирусных пакетов: Avg, Avast, BitDefender, CureIt и Fprot. Дистрибутив подготовлен в двух вариантах, с десктоп-окружением KDE 4.4.2 для систем с 768 Мб ОЗУ и с LXDE для компьютеров с 256 Мб ОЗУ. По сравнению с прошлой версией из поставки удалены VirtualBox, k3b и McAfee. Из идущих в комплекте приложений можно отметить: Linux-ядро 2.6.31 Firefox 3.5.9 c Flash-плагином pppoe Средство для разметки дисковых разделов Gparted Midnight Commander Программа для установки на жесткий диск rdesktop (управление терминальными сессиями windows) x11vnc tightvnc wine 1.1.43 unrar samba-клиент
