Выпуск Chrome OS 112

Доступен релиз операционной системы Chrome OS 112, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 112. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Исходные тексты распространяются под свободной лицензией Apache 2.0. Сборка Chrome OS 112 доступна для большинства актуальных моделей Chromebook. Для использования на обычных компьютерах предлагается редакция Chrome OS Flex. Основные изменения в Chrome OS 112: Обновлено меню с быстрыми настройками, в котором увеличен размер кнопок и осуществлена группировка похожих функций для упрощения навигации. Добавлена отдельная панель для уведомлений, индикатор которой показывается слева от даты. Для управления включением нового меню предложен параметр "chrome://flags#qs-revamp". Предоставлена возможность восстановления забытого пароля, основанная на использовании online-процесса восстановления доступа к учётной записи в Google. Для работы восстановления необходимо явное включение данной функции в настройках (Security / Sign-in / Local data recovery). В приложение Screencast, которое позволяет записывать и просматривать видеоролики, отражающие содержимое экрана, добавлена возможность создания транскрипций речи на языках, отличных от английского. В настройки быстрого сопряжения устройств (Fast Pair) добавлен раздел для просмотра и удаления сохранённых устройств, с которыми ранее была установлена связь. В интерфейс захвата экрана (Screen Capture) добавлен режим отображения информации о совершённых во время записи кликах мышью и нажатых комбинациях клавиш.
