Доступна LiveCD сборка CentOS 5.3

Объявлено о выходе LiveCD сборки Linux дистрибутива CentOS 5.3, доступной в настоящий момент только для платформы i386. Из приложений, включенных в комплект, можно отметить: Программы для рабочей станции: Офисный пакет openoffice.org 2.3.0 Web-браузер firefox 3.0.6 Почтовый клиент thunderbird 2.0.0.18 Программа для мгновенного обмена сообщениями pidgin 2.5.5 Чат xchat 2.6.6 Графический редактор gimp 2.2.13 Программы для диагностики и восстановления системы после сбоя: Программа для тестирования памяти memtest86+-1.65 Полный набор утитит для работы с LVM и RAID дисковыми разделами; Сканер портов Nmap и GUI интерфейс NMapFE traceroute samba-3.0.33 + поддержка cifs в Linux ядре System Log Viewer - программа для аудита системных логов; GUI-интерфейс для проверки и управления аппаратной конфигурацией. С целью уменьшения размера iso-образа из поставки были исключены программы: тектовый редактор emacs, программа записи CD/DVD дисков k3b и издательский пакет scribus. Подробный обзор новшеств базового выпуска CentOS 5.3 на русском языке можно прочитать на данной странице.
