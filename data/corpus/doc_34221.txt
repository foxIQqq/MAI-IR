Доступна для тестирования вторая альфа-версия Ubuntu 12.10

Представлена вторая альфа-версия Ubuntu 12.10 "Quantal Quetzal". Загрузочные iso-образы с тестовой версией Ubuntu 12.10 можно получить на данной странице. Напомним, что CD и DVD образы десктоп-редакции Ubuntu 12.10 объединены в единый образ размером 725 Мб (к релизу будет 800 Мб), пригодный для установки на USB Flash и DVD. Релиз запланирован на 18 октября, после выхода еще одной альфа и двух бета версий. По сравнению с первым альфа-выпуском, в Ubuntu 12.10-alpha2 отмечены следующие улучшения: Осуществлён переход на предварительный выпуск ядра Linux 3.5-rc4. Улучшена поддержка драйвера brcmsmac для беспроводных карт на базе чипов BCM4313, BCM43224 и BCM43225; Продолжен перевод дистрибутива на использование по умолчанию Python 3.0 в десктоп-сборке 12.10 (на время тестирования Python 2 пока остаётся в поставке до момента завершения портирования всех приложений); Обновлён стек приложений для поддержки файловых систем, в том числе обновлены пакеты e2fsprogs 1.42.4 с поддержкой создания ext4 разделов, превышающих 16 Тб; mdadm 3.2.5 с поддержкой форматов DDF (Common raid Disk Data Format) и Intel Matrix RAID; autofs 2012-06-01 c переименованием autofs5 в autofs; btrfs-tools 2012-03-28 с базовой поддержкой утилиты проверки и восстановления целостности ФС btrfsck; Десктоп сборки для платформы ARM преобразованы из формы образов с предустановленной системой в традиционные Live-образы с поддержкой установки на USB-накопители и оптимизацией установки на SD-карты; Обновление пользовательских приложений и графического стека: до последних бета-версий обновлены Firefox и Thunderbird, в состав дистрибутива включена тестовая версия GTK+ 3.5, упрощающая переход на использование некоторых компонентов GNOME 3.5, обновлены пакеты с X.Org и обновлён драйвер для видеокарт AMD/ATI; Продолжена работа по сокращению числа установочных образов для серверной редакции, в которой будет прекращено формирование CD-образов для архитектуры i386, а в качестве первичной платформы будет рассматривается X86_64 (AMD64). Для установки на i386 будет продолжена поддержка mini-образа и образа для установки по сети. Установочный образ серверной версии по прежнему будет вмещаться на CD (703MB), но в будущих тестовых выпусках ситуация может измениться; Изменения в серверной редакции: Tomcat7 заменил собой Tomcat6 в репозитории Main; Подготовлены установочные образы для кластерной платформы Calxeda Highbank (многоядерный сервер на базе архитектуры ARM, состоящий из нескольких десятков плат, каждая из которых включает несколько SoC); Доступны пакеты с фреймворком OpsCode Chef 0.10.x, использование которого позволяет осуществить установку определенных пакетов после инсталляции базовой ОС и внести изменения в конфигурацию; Упрощена поддержка определения политик безопасности AppArmor для контейнеров LXC; Обновлены многие серверные пакеты, в том числе Ceph 0.47.2 и mysql-5.5.25; Облачный стек Openstack синхронизирован с тестовым репозиторием, на базе которого развивается релиз Folsom; Обновлены компоненты Kubuntu: KDE SC 4.9 Beta 2; Вместо Kopete задействован клиент для мгновенного обмена сообщениями из состава Kde-telepathy; В качестве дисплейного менеджера вместо KDM задействован Lightdm; Вместо LibreOffice в поставку Kubuntu теперь входит офисный пакет Calligra (финальное решение пока не принято, LibreOffice может быть возвращён к релизу); Добавлена программа для сканирования изображений skanlite и интерфейс для работы с web-камерой Kamoso; Для установки языковых пакетов задейстован kcm; Для конфигурирования планшетов Wacom добавлен конфигуратор Kde-config-tablet; Xubuntu и Ubuntu Studio перешли на Xfce 4.10 (ранее поставлялся Xfce 4.8), обновлены входящие в состав приложения, в том числе GIMP 2.8 и gThumb 3.0.1.
