Вышел набор JavaScript-виджетов jQuery UI 1.8

Представлен релиз пакета jQuery UI 1.8, представляющего собой набор виджетов с реализацией готовых элементов пользовательского интерфейса, анимации и визуальных эффектов для использования совместно с JavaScript-библиотекой jQuery 1.4. В версии jQuery UI 1.8 добавлено: Виджет для создания настраиваемых кнопок; Виджет для организации автодополнения текста; Утилита для позиционирования элементов относительно других элементов на экране; Новый оптимизированный плагин для работы с мышью; Новый визуальный эффект; На 71% уменьшен размер кода базовой части jQuery UI.
