Релиз платформы для управления системами RHQ 4.6.0

Доступна новая версия открытой системы централизованного управления, конфигурирования, инвентаризации и мониторинга за работой программного обеспечения на машинах в сети предприятия - RHQ 4.6. Код RHQ написан на языке Java и распространяется в рамках лицензии GPL. Для выполнения RHQ 4.6.0 теперь рекомендуется использовать Java7, но поддержка работы на системах с Java6 сохранена. В качестве СУБД поддерживается использование PostgreSQL 8.2.4+ и Oracle. Платформа построена по модульному принципу и позволяет организовать удаленное конфигурирование и управление практически всеми аспектами работы операционной системы и пользовательских приложений. В частности, RHQ даёт возможность контролировать работу различных дистрибутивов Linux и систем с ОС Windows, web-серверов Apache, контейнеров Apache Tomcat, сервера приложений JBoss, СУБД MySQL, PostgreSQL и других популярных открытых проектов. Служба отправки уведомлений базируется на плагинах и расширяется в зависимости от нужд администратора, например, можно организовать запуск произвольных скриптов при возникновении событий и легко создать модуль для интеграции RHQ с корпоративными системами обработки жалоб клиентов. В состав также входит набор инструментов для упрощения развёртывания программных пакетов на группе обслуживаемых компьютеров. Управление может производиться как через web-интерфейс, так и через специальное клиентское приложение, работающее в режиме командной строки. Среди новшеств, добавленных в версии RHQ 4.6.0: Осуществлён переход на использование сервера приложений JBoss AS 7.1, вместо JBoss Messaging задействован HornetQ; Для построения интерфейса задействован новый выпуск фреймворка GWT 2.5 (Google Web Toolkit); Выполнен частичный перевод интерфейса на русский язык; Реализована новая система установки; Расширены возможности REST API и удалена поддержка web-сервисов; В состав включён плагин для управления и мониторинга БД Cassandra; Переработан плагин для управления JBoss AS 7;
