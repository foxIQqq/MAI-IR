Компания Oracle перевела JRockit Java VM в разряд бесплатных продуктов

Компания Oracle объявила о начале бесплатного распространения виртуальной машины JRockit, ранее доступной только в виде коммерческого продукта. Виртуальная машина JRockit разработана компанией BEA и отличается рекордной производительностью, благодаря продвинутым механизмам сборки мусора, и поддержкой функций горячей замены, позволяющих вносить изменения без остановки сервера. Продукт распространяется в бинарном виде под лицензией Binary Code Licence (BCL) и бесплатен для применения в процессе разработки или для внутреннего использования на компьютерах общего назначения. Поставка JRockit в составе коммерческих проектов по прежнему требует покупки специальной лицензии. В будущем, на базе JRockit и Hotspot VM будет создана новая виртуальная машина Java, код которой будет поставляться под лицензией GPL в составе OpenJDK.
