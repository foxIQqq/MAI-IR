Вышел Live-дистрибутив Parted Magic 5.3

Вышел Parted Magic 5.3, компактный LiveCD/LiveUSB дистрибутив (125 Мб) для восстановления системы после сбоя. Parted Magic построен на базе Linux From Scratch (LFS) и графического окружения Xfce. В минимальном режиме дистрибутив может работать на машинах с 48 Мб ОЗУ. В выпуске Parted Magic 5.3 проведена работа по оптимизации потребления ОЗУ в процессе работы дистрибутива. Для обеспечения возможности записи поверх основы, доступной только для чтения, осуществлена миграция с реализованной на уровне ядра поддержки Unionfs в пользу работающей на уровне пользователя Unionfs-fuse, базовый образ системы теперь оформлен в виде единого файла в формате squashfs. Кроме того, в новой версии пересмотрен способ организации загрузки с USB-накопителей: для создания загрузочных Flash теперь рекомендуется использовать Unetbootin. Из обновленных в представленном релизе версий программ можно упомянуть: truecrypt 7.0, g4l 34a, busybox 1.17.1, freetype 2.4.2, ntfs-3g 2010.8.8, udev 161, unetbootin 471, linux-ядро 2.6.34.5, lxterminal 0.1.8. Из базовых программ можно отметить: web-браузер Chromium, GParted, TestDisk, PhotoRec, Partition Image, Xarchive, Xfburn, ISO Master, Leafpad, Grsync. Из утилит командной строки присутствуют dd, ddrescue , cfdisk, fdisk, tcpdump. Имеется поддержка восстановления файловых систем ext2, ext3, ext4, btrfs, FAT16, FAT32, HFS, HFS+, JFS, NTFS, ReiserFS, Reiser4, XFS.
