Выпуск программы для хранения заметок OutWiker 1.8

Вышла новая версия OutWiker - открытой кроссплатформенной программы для хранения заметок в виде дерева. К особенностям программы можно отнести хранение заметок в виде каталогов на диске, возможность прикрепления файлов к заметкам, работа с тегами, а также возможность добавления новых функций с помощью плагинов. Код программы написан на языке Python и распространяется под лицензией GPLv3. Основные изменения: Возможность настройки горячих клавиш. Возможность ссылаться на страницы, используя их уникальные идентификаторы. Возможность перехода по ссылкам на якоря, расположенные на других страницах (page://bla-bla-bla/#anchor). Возможность использовать относительные пути в ссылках на страницы (../../Страница 1/Страница 2) Возможность установки стиля страницы разом для всех страниц ветки дерева (или всего дерева). Возможность поиска и замены по странице. Кнопки Вперед / Назад для возврата к предыдущим открытым страницам. Возможность настройки цветов редактора. абзаца). Новый тег викисинтаксиса для цитирования: [>...{]. Кнопка и пункт меню для вставки текущей даты и времени. Команды (:crdate:) и (:eddate:) для вставки даты создания и редактирования страницы соответственно. Диалоги для вставки команд (:attachlist:), (:childlist:) и (:include:). В команде (:childlist:) добавлена возможность сортировки дочерних страниц по дате создания и дате последнего изменения. На странице глобального поиска добавлена возможность сортировки найденных страниц по дате создания. Новый параметр командной строки "-r" или "--readonly", позволяющий открывать дерево заметок в режиме "Только для чтения". Всплывающие подсказки над иконками в окне свойств страниц. Новые стили оформления страниц. Кнопка и пункт меню для открытия папки с прикрепленными файлами в системном файловом менеджере. Сохранение последнего используемого стиля страницы. Сохранение положения курсора на каждой странице перед закрытием. В панели с прикрепленными файлами показываются иконки файлов. Изменен алгоритм расстановки переносов строк. Открытие заметок и поиск страниц осуществляется в отдельном потоке. Для каждой страницы сохраняется дата создания. Теперь все HTML-теги, создаваемые википарсером, будут строчными буквами. Новые иконки. Настройки перенесены из ~/.outwiker идеологически более верные места (зависит от операционной системы). Ускорение работы.
