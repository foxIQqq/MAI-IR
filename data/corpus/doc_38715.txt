Представлен интерфейс для двойной загрузки Ubuntu Touch и Android на одном устройстве

Компания Canonical анонсировала предварительную версию системы для организации совместного использования окружений Ubuntu Touch и Android на одном устройстве. После установки системы двойной загрузки, пользователь может быстро переключаться между окружениями Ubuntu Touch и Android. Для переключения между платформами подготовлены специальные приложения для обеих платформ: приложение для Ubuntu Touch, позволяющее загружать Android, и приложение для Android, позволяющее загружать Ubuntu Touch. Кроме штатной прошивки Android (поддерживается 4.2 и более новые выпуски) возможно использование альтернативных сборок, таких как CyanogenMod и AOSP. Система двойной загрузки пока ориентирована на разработчиков и не рассчитана на конечных пользователей. В частности, система позволяет разработчику установить Ubuntu Touch на свой основной телефон и переключаться в данное окружение при рабочей необходимости, оставив Android для повседневного использования. При использовании двойной загрузки загрузчик окружения Ubuntu Touch устанавливается в системный раздел "/recovery", предназначенный для восстановления основной прошивки (по сути при обычной загрузке стартует Android, а при загрузке в режиме "recovery" - Ubuntu Touch). Системный образ Ubuntu Touch размещается в файле /data/system.img, а пользовательские и системные данные в директориях /data/user-data и /data/system-data внутри раздела "/data" платформы Android. Обновление и установка окружения Ubuntu Touch производится из приложения для Android. Разработка протестирована на смартфоне Nexus 4, но теоретически должны поддерживаться и другие устройства линейки Nexus.
