Вышла первая бета версия GNOME 2.28.0

Вышла первая бета версия GNOME 2.28, релиз намечен на 23 сентября. Переход на этап бета тестирования ознаменует переход на стадию усиленного тестирования и устранения ошибок, код переведен на стадию заморозки UI, т.е. все улучшения в пользовательском интерфейсе теперь могут быть внесены только после специального утверждения. GNOME 2.28 будет последним релизом ветки серии 2.x, весной 2010 года будет выпущен GNOME 3.0, в котором ожидается переход на GTK+ 3.0, значительное изменение программных/бинарных интерфейсов (API/ABI) и полная переработка некоторых подсистем. Некоторые из ожидаемых в GNOME 2.28 новшеств: Переработка визуального оформления: Новые темы для виджетов: темная, плоская и компактная (для небольших экранов). Улучшение иконок для панели GNOME. Поддержка определения цветовых схем в темах. Добавление пиктограмм размером 256x256; Улучшение программы для просмотра документов Evince: Интеграция функций обработки мета-данных с базовыми фреймворками GNOME. Поддержка средств для распознавания текста (OCR). Улучшение поддержка формата PostScript, например, улучшение функций поиска текста; В Evolution планируется уход от использования устаревшей библиотеки Bonobo, переход на WebKit вместо GtkHtml; Мультимедиа возможности: поддержка web-камер, разработка интерфейса для конвертирования медиафайлов, интеграция поддержки аa2dp/upnp, возможность записи звука из интернет-радиостанций, средство для тестирования конфигурации звуковой подсистемы (например, surround); Панель GNOME: разработка новой библиотеки для апплетов, создание прослойки для обеспечения совместимости с устаревшим механизмом Bonobo-апллетов, уход от использования Bonobo в панели. Создание новой модели для управления апплетами, например, активации и дезактивации вместо добавления и удаления. Добавление режима для редактирования содержимого панели. Улучшение интеграции с многопользовательскими настройками (например, возможность быстрой смены пользователя в диалоге выхода). Упрощение кода, связанного с пользовательскими сессиями и GDM (предлагается вызывать методы GNOME Session через DBus). Улучшение внешнего вида панели. Оконный менеджер Metacity: улучшение поддержки EWMH/ICCCM, улучшение "Drag and Drop" взаимодействия приложений, добавление поддержки SELinux меток для окон; GConf: миграция на GSettings и dconf, новую платформу для управления настройками.
