Началось альфа-тестирование дистрибутива Nexenta Core Platform 3.0

Началось альфа-тестирование дистрибутива Nexenta Core Platform 3.0, построенного на базе программного окружения Ubuntu и ядра OpenSolaris (сборка 124). Дистрибутив сочетает возможность использования пакетного менеджера APT и обилия пакетов программ для Ubuntu с такими возможностями Solaris, как файловая система ZFS, инфраструктура динамической трассировки DTrace и система виртуализации IP стека Crossbow. Nexenta Core Platform предоставляет собой только минимальный костяк для построения дистрибутивов, например, для пользователей на его основе создан StormOS.
