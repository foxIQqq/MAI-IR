Выпуск файлового менеджера vifm 0.8

Состоялся выпуск vifm 0.8.0, консольного файлового менеджера, предоставляющего vi-подобные клавиатурные комбинации, позволяющие полностью управлять работой программы с клавиатуры с использованием команд, привычных для пользователей vi. В новой версии: Поддержка произвольных представлений, позволяющих виртуально объединить в одной панели файлы из разных директорий, которые становятся доступны для поиска как будто находятся в одной директории; Поддержка подсветки, специфичной для различных файлов; Добавлена панель для отображения активных фоновых операций; Добавлены аббревиатуры для режимов командной стройки; Добавлена директория с глобальными настройками и цветовыми схемами; Поддержка подсветки совпавших при поиске элементов; Опциональное отображение прогресса выполнения файловых операций в разрезе отдельных файлов; Возможность указания масок полных путей ({{globs}} и //regexp//); Добавлена опция, упрощающая использование vifm как интерфейса для выбора файла или директории; Цветовые схемы теперь размещаются в файлах с расширением ".vifm"; Поддержка спецификации XDG; Возможность использования консольных приложений (например, w3mimgdisplay) для предпросмотра графики.
