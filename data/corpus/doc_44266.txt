Доступно ПО для управления проведением презентаций OpenSlides 2.0

Представлен релиз свободной системы организации мероприятий и презентаций OpenSlides 2.0, позволяющей управлять выводимой через проектор информацией при проведении встреч и конференций разного масштаба. OpenSlides предоставляет средства для автоматизации работы с типовыми элементами, такими как списки участников, повестка дня, доклады, рассмотрение предложений, проведение голосований и выборов. ПО написано на языке Python и распространяется под лицензией MIT. Управление производится через web-интерфейс через обычный браузер. Новый выпуск примечателен полной переработкой интерфейса, который преобразован в форму одностраничного динамически обновляемого web-приложения, построенного с использованием фреймворка AngularJS и технологии WebSockets. Изменена организация управления проектором, работа которого теперь контролируется через постоянно видимую боковую панель, отображающую текущее состояния видимого аудиторией экрана. Для интеграции с внешними приложениями добавлен REST API. Оценить возможности системы можно в online-демонстрации.
