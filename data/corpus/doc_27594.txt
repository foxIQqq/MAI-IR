Релиз системы мониторинга Zenoss 3.0.1

Вышел релиз системы сетевого мониторинга Zenoss Core 3.0.1, распространяемой под лицензией GPL и позиционируемой в качестве открытой альтернативы продуктам IBM Tivoli, HP OpenView, BMC Patrol. Программа написана на языке Python с использованием Zope, в качестве СУБД используется MySQL, в web-интерфейсе присутствуют элементы Ajax. Поддерживается подключение модулей мониторинга от системы Nagios. В ветке Zenoss 3.0 полностью переработан интерфейс пользователя, осуществлен переход на Python 2.6 и Zope 2.12, добавлено более 50 новых расширяющих функциональность дополнений, представлено новое REST API для интеграции со сторонними продуктами.
