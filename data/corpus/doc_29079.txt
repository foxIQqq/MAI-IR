Релиз дистрибутива Zorin OS 4

Артем Зорин представил релиз Linux-дистрибутива Zorin OS 4, основанного на пакетной базе Ubuntu 10.10 и оптимизированного для использования неискушенными начинающими пользователями, привыкшими работать в Windows. Пользовательский интерфейс дистрибутива приближен к Windows 7, в состав включена подборка программ, близких к программам к которым привыкли пользователи Windows. Размер загрузочного iso-образа составляет 1.2 Гб (поддерживается работа в роли LiveDVD). Особенностью дистрибутива является приложение Look Changer, позволяющее изменить оформление рабочего стола, придав ему вид свойственный таким ОС, как Windows 7, Windows Vista, Windows XP, Windows 2000 и Mac OS X. Для выбора web-браузера представлен интерфейс Internet Browser Manager, который дает возможность использования Firefox, Chrome, Opera и Midori. В качестве файлового менеджера задействован Nautilus-Elementary, работающий вкупе с GNOME-расширением Gloobus Preview, реализующим функции сходные с Apple Quicklook. Дополнительно подготовлена легковесная редакция дистрибутива, пользовательское окружение в которой основано на LXDE и может быть стилизовано под Windows 2000 и MacOS X.
