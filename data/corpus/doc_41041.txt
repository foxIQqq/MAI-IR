Выпуск Firefox OS 2.0 для разработчиков

Проект Mozilla выпустил предварительную версию мобильной платформы Firefox OS 2.0, нацеленную на первичное ознакомление разработчиков приложений. Выпуск основан на движке Gecko, используемом в Firefox 32. Тестовая версия Firefox OS 2.0 доступна в форме прошивки для смартфонов Flame, Geeksphone Keon и Geeksphone Peak, а также в виде симулятора, реализованного в форме дополнения к настольной версии Firefox. Основные новшеcтва: Поддержка NFC (Near Field Communication): представлен API WebNFC, предоставляющий средства для обмена данными и возможность чтения NFC-меток. Например, можно поднести друг к другу два любых телефона, поддерживающих формат NDEF, и по NFC перенести записи адресной книги, фотографии, ссылки и видео; Реализация аппаратного ускорения кодирования и декодирования видео H264, что открывает более широкие возможности по использовнию WebRTC; Функция поиска потерянного устройства (Find My Device), позволяющая посмотреть местоположение аппарата на карте через сайт find.firefox.com, инициировать проигрывание звука, вывести на экран сообщение, удалённо заблокировать устройство или удалить персональные данные. Управление функцией осуществляется через учётную запись в сервисе Firefox Accounts при включении в настройках опции Find My Device; Реализована концепции сменяющих друг друга страниц, позволяющих перейти от экрана одного приложения к другому без необходимости вызова менеджера задач или интерфейса для запуска приложений. Для переключения между открытыми приложениями теперь можно использовать жест скольжения от левого до правого края экрана и наоборот (перелистывание); Новая реализация домашнего экрана, в котором отражается набор более крупных пиктограмм с вертикальной прокруткой; Возможность создавать собственные рингтоны из имеющихся композиций в музыкальной коллекции или музыки, загруженной из Web; В приложении для работы с камерой представлено несколько режимов установки фокуса: фокус по касанию, отслеживание лиц и непрерывный автофокус; Переработано оформление почтового клиента, календаря-планировщика и часов; В менеджере сетевых соединений обеспечена одновременная поддержка IPv4 и IPv6; Новый инструмент Developer HUD, предоставляющий разработчикам приложений средства для изменения производительности, оценки потребления памяти и изучения иных характеристик; Поддержка потокового вещания с использованием протокола RTSP; Новые API: Resource Statistics API, Resource Timing API, fastSeek API, Network Information API, Feature Detection API, MSISDN Verification API, CameraControl API.
