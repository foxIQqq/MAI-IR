Опубликован Wa-tunnel для туннелирования трафика через мессенджер WhatsApp

Опубликован инструментарий Wa-tunnel, позволяющий пробрасывать TCP-трафик через другой хост, используя туннель, работающий поверх мессенджера WhatsApp. Подобные манипуляции могут оказаться полезными при необходимости получения доступа к внешней сети из окружений, в которых доступен только мессенджер, или для экономии трафика при подключении к сетям или провайдерам, предоставляющим безлимитные опции для трафика мессенджеров (например, неограниченный доступ к WhatsApp предоставляется в бортовых сетях самолётов некоторых авиакомпаний). Код написан на языке JavaScript с использованием Node.js и распространяется под лицензией MIT. Для взаимодействия с API WhatsApp используется библиотека Baileys. Для организации туннеля требуются две учётные записи в WhatsApp - одна используется на стороне клиента, а другая на стороне сервера. Возможен проброс на сервер как отдельных локальных портов, так и организация работы прокси. Связывание с учётной записью в WhatsApp осуществляется через сканирование QR-кода мобильным приложением WhatsApp.
