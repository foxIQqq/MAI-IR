Релиз платформы для организации групповой работы Kolab 3.1

Анонсирован релиз Kolab 3.1, платформы для организации совместной коллективной работы, базирующейся на технологиях KDE. Основу платформы составляет серверное ПО Kolab Server, позволяющее организовать работу серверной инфраструктуры, реализующей средства групповой работы, единой адресной книги, календаря-планировщика и почтового сервиса. В качестве клиентских приложений могут выступать как стандартные программы из состава KDE PIM (Kontact, KMail, KOrganizer и KAddressbook), так и сторонние системы, в том числе доступен набор коннекторов для Microsoft Outlook (Bynari Connector, плагин Toltec Connector 2 и MAPI-хранилище KONSEC Konnektor). В качестве базового web-клиента в составе платформы поставляется Roundcube Webmail, в полной мере поддерживающий вторую и третью версии формата хранилища Kolab. Для интеграции с Thunderbird и SeaMonkey проектом развивается специальный плагин SyncKolab. Модуль интеграции также доступен для GNOME Evolution. Для организации работы с мобильными системами предоставляется ряд решений по синхронизации данных, таких как CalDAV/CardDAV и ActiveSync-совместимые решения Kolab ActiveSync Server и Z-Push. Для организации работы в Kolab Server задействованы стандартные открытые компоненты, такие как сервер директорий 389, Apache httpd, Postfix, Cyrus IMAP, SASL, OpenSSL. Для управления системой подготовлен специальный административный web-интерфейс. Поддерживается ведение единой адресной книги, интегрируемой с почтовым сервисом с поддержкой POP3 и IMAP4. Для шифрования почты предлагается S/MIME-шифрование на стороне клиента. Бинарные сборки Kolab 3.1 подготовлены для RHEL, SUSE, Debian, Fedora, Ubuntu и openSUSE. Особенности новой версии: В состав включён модуль Chwala с реализацией облачного файлового хранилища, которое может быть интегрировано с web-клиентом Kolab или использовано в виде самодостаточного сервиса. Для упрощения навигации файлы автоматически группируются по виртуальным коллекциям (изображения, музыка/видео, документы). В интерфейс добавлены возможности для просмотра и редактирования контента из облачного хранилища, например, можно воспроизводить видео, прослушивать музыку, просматривать документы DOC, ODF и PDF, редактировать текстовые файлы и исходные тексты программ при помощи редактора Ace. При работе с хранилищем из интерфейса Kolab предоставляются не только функции доступа к файлам, но и возможность прикрепления к электронным письмам файлов из облачного хранилища (или сохранения в облачном хранилище содержимого приложений), без их промежуточного сохранения на локальной системе. Возможны локальное монтирование хранилища и синхронизация данных с использованием протокола WebDAV. Добавлен модуль iRony для организации подсоединения мобильных клиентов к серверу с использованием протоколов CalDAV и CardDAV. Поддержка CalDAV/CardDAV основана на коде SabreDAV и добавлена в дополнение к ранее задействованному протоколу ActiveSync на базе стека Syncroton. Использование открытых стандартов CalDAV и CardDAV позволяет обеспечить независимость от отдельных вендоров и даёт возможность обеспечить более полную интеграцию с приложениями для OS X, Android и GNOME. В web-интерфейс администратора добавлена возможность создания совместно используемых папок, для создания которых ранее необходимо было запускать специальную утилиту из командной строки.
