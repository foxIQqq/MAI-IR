Доступен менеджер фотографий Shotwell 0.32

После четырёх с половиной лет разработки опубликован первый выпуск новой стабильной ветки программы для управления коллекцией фотографий Shotwell 0.32.0, которая предоставляет удобные возможности каталогизации и навигации по коллекции, поддерживает группировку по времени и тегам, предоставляет инструменты для импорта и конвертации новых фотографий, поддерживает выполнение типовых операций по обработке изображений (вращение, устранение эффекта красных глаз, корректировка экспозиции, оптимизация цветности и т.п.), содержит средства для публикации в социальных сетях, таких как Google Photos, Flickr и MediaGoblin. Программа написана на языке Vala, использует библиотеки GNOME и распространяется под лицензией LGPLv2.1. В новом выпуске: Добавлена поддержка форматов изображений JPEG XL, WEBP и AVIF (AV1 Image Format), а также форматов файлов HEIF (HEVC), AVIF, MXF и CR3 (raw-формат Canon). Включено по умолчанию распознавание лиц на фотографиях и установка меток для привязки к лицам. Подобные метки можно использовать для группировки, сортировки и поиска людей на других фотографиях. Предусмотрена возможность сборки Shotwell без распознавания лиц для сокращения размера зависимостей (OpenCV). Интерфейс просмотра фотографий и инструменты для их обработки адаптированы для работы на экранах с высокой плотностью пикселей (HiDPI). Добавлена поддержка профилей и интерфейс создания/редактирования профилей. При импорте файлов из каталогов реализована обработка файла .nomedia, позволяющего выборочно отключать сканирование содержимого. Добавлен профиль haarcascade для использования одноимённого алгоритма распознавания объектов на фотографиях. Улучшена обработка изображений с метаданными GPS. Добавлена поддержка экспорта метаданных GPS. Улучшено управление масштабом и прокрутка при помощи тачпада. Предоставлена возможность указания иерархических тегов, состоящих из нескольких уровней (например "группа/тег"). Для отправки фотографий и установки обоев рабочего стола из изолированных окружений (например, при установке пакета в формате flatpak) задействована библиотека libportal. Добавлена возможность использования нескольких учётных записей для каждого внешнего фотосервиса (пока работает только для Piwigo). Для хранения параметров подключения к внешним сервисам задействована библиотека libsecret. Переработана реализация OAuth1. Реализована новая панель для настройки плагинов. Ускорена навигация по каталогам с очень большим числом файлов. Ускорено чтение raw-изображении. Улучшена поддержка Flickr, Google Photos и Piwigo. Налажена загрузка фотографий в Google Photos в пакетном режиме. Удалён код публикации в Facebook (был неработоспособен). Проведена реорганизация исходных текстов. Улучшен диалог для редактирования прошлых поисковых запросов. Добавлена опция командной строки -p/--show-metadata для отображения метаданных изображения. Размер прикрепляемого комментария увеличен до 4 КБ.
