На большинстве виртуальных машин в Azure установлен Linux

Скотт Гатри (Scott Guthrie), вице-президент Microsoft, отвечающий за облачные сервисы и корпоративные продукты компании, сообщил, что в настоящее время более половины виртуальных машин в облачном сервисе Azure работают под управлением Linux, а ранее доминировавшие в Azure окружения с Windows Server остались в меньшинстве и их доля со временем неуклонно снижается. Примечательно, что три года назад доля Linux в Azure составляла 25%, а в прошлом году преодолела рубеж в 40%. Более того, Linux применяется не только в пользовательских окружениях, но и в служебных сервисах Azure. Например, SDN-сервисы Azure работают под управлением Linux. Кроме Azure поддержка Linux также появляется и в других проектах Microsoft, в том числе выпуски SQL Server теперь одновременно формируются для Linux. Скотт Гатри также считает, что в настоящее время Microsoft является крупнейшим сподвижником открытого ПО в мире, так как компания развивает более 200 открытых проектов, а более 20 тысяч сотрудников Microsoft зарегистрированы на GitHub (не говоря о том, что Microsoft теперь владеет GitHub).
