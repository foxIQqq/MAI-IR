Релиз GNU Wget 1.14

После года разработки доступна новая версия программы для автоматизации загрузки контента с использованием протоколов HTTP и FTP - GNU Wget 1.14. По сравнению с прошлой версией добавлена поддержка RFC 2617 (Digest Access Authentication), поддержка файлов WARC, новая опция "--report-speed", возможность указания аргументов для опций "--accept-regex" и "--reject-regex". Улучшена поддержка защищённых соединений при использовании GNU TLS, в том числе устранена утечка памяти при использовании GNU TLS.
