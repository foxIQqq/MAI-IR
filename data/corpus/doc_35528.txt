Новая версия среды разработки Code::Blocks 12.11

После двух лет разработки доступен новый выпуск Code::Blocks, интегрированной среды для разработки проектов на языках C и C++, примечательной наличием гибких механизмов для расширения функциональности через плагины. Код проекта написан на языке С++ с использованием тулкита wxWidgets и распространяется в рамках лицензии GPLv3. Бинарные сборки подготовлены для Linux (Debian, Fedora, RHEL/CentOS, SUSE), Mac OS X и Windows. В новой версии проведена работа по увеличению стабильности. Реализован новый отладочный фреймворк, позволяющий создавать плагины с новыми реализациями отладчиков. Улучшена работа плагина СС с реализацией системы автодополнения кода и навигации по классам. Улучшена поддержка макросов, внесены разнообразные небольшие улучшения в пользовательский интерфейс, добавлена поддержка GCC 4.7. Добавлено 11 новых плагинов, среди которых: отдельный плагин Abbreviatons для автодополнения кода, плагин Cscope для интеграции с Cscope для обеспечения поиска кода, DoxyBlocks для поддержки работы с документацией doxygen, EditorConfig для настройки редактора (использование табуляции длина табуляции окончания строк и тд) на уровне проекта, EditorTweaks для тонкой настройки редактора кода, файловый менеджер, SmartIndent для автоматического расставления отступов, SpellChecker для проверки правописания в комментариях и строках.
