Увидел свет GeeXBoX 3.0, минималистичный Linux дистрибутив для создания медиацентра

После года разработки увидел свет релиз специализированного Linux-дистрибутива GeeXBoX 3.0, предназначенного для создания медиацентров и домашних кинотеатров. Дистрибутив развивается с 2002 года и базируется на собственном сборочном инструментарии. Загрузившись с USB-накопителя, SD-карты или по сети, GeeXboX предлагает готовый к использованию медиацентр, позволяющий просматривать и прослушивать любые типы медиаконтента (звук, видео, фотографии) из разнообразных источников (CD, DVD, Bluray, HDD, USB, Samba, NFS, UPnP, DLNA, HTTP, FTP, SSH, SHOUTcast). В GeeXBoX из коробки доступны средства акселерации выполнения графических операций для оборудования, поддерживающего OpenGL и OpenGL ES, а также ускорения декодирования видео через задействование VDPAU, VA-API и Crystal HD. Силами разработчиков дистрибутива развивается несколько известных мультимедийных проектов, таких как медиацентр Enna и фреймворк для создания оптимизированных минидистрибутивов OpenBricks. Новая версия GeeXBoX доступна в сборках для платформ x86, ARM TI OMAP4 Pandaboard, ARM nVidia Tegra2 Harmony, ARM Cubox (armada5xx) и ARM Snowball ux500. Размер загрузочного iso-образа 136 Мб. Ключевые новшества GeeXBoX 3.0: Интеграция фронтэнда для медиацентра XBMC 11; Поддержка записи и просмотра видео с DVB-адаптеров для цифрового телевидения; Улучшение поддержки дистанционного управления; Поддержка Full HD видео на платформах PandaBoard и Cubox; Улучшение системы установки на жесткие диски и SSD-накопители; Поддержка загрузки на системах с (U)EFI; Обновление версий программ: ядро Linux 3.4.4, Systemd v188, BusyBox v1.21, Connman Network Manager v1.6, Xorg Server v1.11.4, Mesa v8.0.3, OPKG Package Manager v0.18, ALSA v1.0.24.1, свежие версии открытых и проприетарных видеодрайверов.
