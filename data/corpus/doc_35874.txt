Релиз Wine 1.5.22

Вышел очередной экспериментальный релиз открытой реализации Win32 API - Wine 1.5.22. С момента выпуска версии 1.5.21 было закрыто 50 отчётов об ошибках. Основные изменения в новой версии: Обновлён HTML (Gecko) движок; Начата реализация драйвера графики для Mac OS X; Добавлена поддержка архитектуры ARM64; Исправлены проблемы с RTL-текстом из Uniscribe; Произведено множество улучшений в поддержке библиотек DirectX и C/C++; Обновлены переводы; Исправлено множество ошибок.
