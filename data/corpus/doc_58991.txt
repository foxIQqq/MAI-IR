Выпуск программы для управления фотографиями digiKam 8.0

После почти трёх лет с момента формирования прошлой ветки состоялся релиз программы для управления коллекцией фотографий digiKam 8.0.0, развиваемой в рамках проекта KDE. Программа предоставляет исчерпывающий набор инструментов для импорта, управления, редактирования и публикации фотографий, а также изображений с цифровых камер в формате raw. Код написан на языке C++ с использованием Qt и библиотек KDE, и распространяется под лицензией GPLv2. Установочные пакеты подготовлены для Linux (AppImage, FlatPak), Windows и macOS. Основные изменения: Проведена значительная работа по повышению удобства работы с интерфейсом - в панель инструментов добавлено новое меню "гамбургер", которое появляется при скрытии основного меню или при переходе в полноэкранный режим. Добавлены настройки для управления видимостью меток в режиме навигации с использованием пиктограмм (Icon-View). Запущен новый сайт docs.digikam.org с online-документацией, которая переведена с использования формата DocBook на Sphinx/ReStructuredText. Изменение позволило повысить наглядность информации (например, добавлены скриншоты и скринкасты), упростить сопровождение и улучшить работу над переводами (среди прочего предложен перевод на украинский язык, перевода на русский пока нет). Расширена поддержка форматов файлов. Реализована возможность экспорта, версионированного хранения и пакетной обработки файлов в форматах JPEG-XL, WEBP и AVIF. Добавлена поддержка расширенных редакций формата JPEG-2000. Добавлена поддержка RAW-изображений, предоставляемых камерами Phase One/Leaf IIQ-S v2, Canon CR3, Canon CRM, Canon EOS R3, R7 и R10, Fujifilm X-H2S, X-T30 II, Olympus System OM-1, Leica M11, Sony A7-IV (ILCE-7M4) и DJI Mavic 3. Добавлен новый инструмент для оптического распознавания текста на отсканированных изображениях. Для распознавания используется OCR-движок Tesseract. Улучшена работа с текстовой информацией и добавлена поддержка проверки правописания в свойствах, примечаниях и редактируемых текстовых данных. Добавлены настройки языка для виджета редактирования текста, позволяющие организовать автоматический перевод введённых текстовых данных при помощи online-сервисов и сохранение результата на другом языке. Улучшена обработка метаданных. Инструментарий ExifTool обновлён до версии 12.59. Значительно повышена производительность использования пакета ExifTool, который в будущем планируют использовать вместо библиотеки Exiv2. Добавлена опция для записи метаданных в файлы, используя бэкенд на базе ExifTool, который по умолчанию используется для форматов DNG и RAW, не поддерживаемых в библиотеке Exiv2. На страницу с расширенными настройками метаданных добавлены опции для сохранения и загрузки профилей конфигурации, определяющих информацию, записываемую в область с метаданными изображения. Например, доступен профиль для работы с DarkTable. Улучшен интерфейс для редактирования метаданных, который оптимизирован для экранов с небольшим разрешением. В менеджер выполнения работ в пакетном режиме (Batch Queue Manager) добавлено два инструмента: первый для выставления рейтинга и меток, а второй для обработки заголовков и примечаний на разных языках, проверки правописания и добавления переводов. Предложен движок для расширенного переименования файлов, который можно использовать при импорте, в режиме навигации (Icon-View) и в менеджере пакетных операций (Batch-Queue-Manager). Возможно использование в именах увеличивающихся счётчиков ([c] или [c,step]), случайных последовательностей ([r]) и уникальных идентификаторов ({unique:n,c}). В поисковый движок добавлены возможности для поиска с учётом параметров распознавания лиц. Добавлен режим работы БД SQLite WAL (Write-Ahead-Logging) для повышения надёжности хранения. Добавлен новый движок для автоматической классификации изображений, использующих машинное обучение. Движок может использоваться для сортировки по уровню качества изображений в зависимости от содержимого, дизайна, расположения объектов и других параметров. Улучшена интеграция с фреймворком для обработки изображений G'MIC-Qt, который можно использовать для применения фильтров в режиме редактирования изображений. Кодовая база адаптирована для сборки с Qt6 (по умолчанию для официальных сборок продолжает использоваться Qt5).
