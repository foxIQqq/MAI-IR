Издание Washington Times открыло исходные тексты нескольких проектов

Издание Washington Times открыло под лицензией Apache 2.0 исходные тексты нескольких модулей для web-фреймворка Django, написанного на языке Python: django-projectmgr - модуль для управления репозиторием исходных текстов и поддержки контроля за исправлением ошибок. Позволяет организовывать привязанные к исходным текстам обсуждения проблем, ошибок, новых возможностей, задач. django-supertagging - интерфейс к сервису семантической разметки Open Calais. django-massmedia - приложение для управления мультимедиа контентом. Позволяет создать галереи с представлением различных типов информации с возможностью редактирования содержимого в интерактивном режиме (используется fckeditor). django-clickpass - интерфейс к OpenID сервису clickpass.com, позволяющему создавать аккаунты на базе учетных данных в Google, Yahoo!, Facebook, Hotmail и AIM. Представленные модули были разработаны для расширения функций web-инфраструктуры газеты, при построении которой использовано только открытое ПО: серверная редакция Ubuntu Linux, СУБД PostgreSQL, http-сервер Apache, web-акселератор на основе lighttpd, система кеширования на базе memcached, web-приложения написаны на языке Python с использованием фреймворка django. Для публикации своих open source разработок издание Washington Times открыло специальный сайт opensource.washingtontimes.com.
