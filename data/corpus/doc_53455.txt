Выпуск дистрибутивов Альт Рабочая станция, Альт Сервер и Альт Образование 9.1

Доступно обновление трёх основных вариантов ОС Альт версии 9.1 на основе Девятой платформы ALT (p9 Vaccinium): «Альт Рабочая станция 9», «Альт Сервер 9», «Альт Образование 9». Наиболее значимым изменением является дальнейший рост списка поддерживаемых аппаратных платформ. ОС «Альт» доступны для восьми российских и зарубежных аппаратных платформ: 32-/64-битные процессоры x86 и ARM, процессоры Эльбрус (v3 и v4), а также Power8/9 и 32-битный MIPS. Поставляются сборки для отечественных систем на процессорах «Эльбрус», «Байкал-М» (впервые), «Байкал-Т», «Elvees». Оригинальные отечественные операционные системы стали доступны одновременно для восьми российских и зарубежных аппаратных платформ. Теперь они работают на следующих процессорах: «Альт Рабочая станция 9» – x86 (32-/64-битных), ARM64 (NVIDIA Jetson Nano Developer kit, Raspberry Pi 3/4 и другие), ARM32 («Салют-ЭЛ24ПМ2»), e2k/e2kv4 («Эльбрус»), mipsel («Таволга Терминал»); «Альт Сервер 9» – для x86 (32-/64-битных), ARM64 (Huawei Kunpeng, ThunderX и другие), ppc64le (YADRO Power 8/9, OpenPower), e2k/e2kv4 («Эльбрус»); «Альт Образование 9» – для x86 (32-/64-битных), ARM64 (NVIDIA Jetson Nano Developer kit, Raspberry Pi 3/4 и другие), e2kv4 («Эльбрус», в т.ч. в многоместных конфигурациях). Для каждой архитектуры сборка осуществляется нативно, без использования кросс-компиляции. Новое в версии ОС «Альт 9.1»: впервые доступен образ ОС «Альт Рабочая станция» для платы mini-ITX на отечественном процессоре «Байкал-М» (ARM64); впервые дистрибутив «Альт Рабочая станция» выпущен на платформе ARM32, он работает на компьютерах с платами Elvees MCom-02 (Салют-ЭЛ24ПМ2); впервые представлены образы для популярного одноплатного компьютера Raspberry Pi 4 (ARM64) дистрибутивов «Альт Рабочая станция» и «Альт Образование»; в число поддерживаемых платформ включен Huawei Kunpeng Desktop (ARM64); впервые представлена новая разработка для поддержки групповых политик Active Directory; для большинства платформ осуществлен переход на версию ядра Linux 5.4; в дистрибутив сервера 64-битной платформы x86 включена популярная свободная платформа для организации видеоконференций Jitsi Meet; при установке в полях ввода паролей можно показать пароль, чтобы избежать последствий неожиданной раскладки. Также в новом выпуске «Альт Образование» дополнительно сделаны следующие изменения: добавлен пакет deploy, служащий для разворачивания системных служб с помощью Ansible (в настоящее время поддерживается развёртывание PostgreSQL), а также пакеты afce, libva-intel-media-driver и grub-customizer; в LiveCD исправлен профиль установки по умолчанию и запуск программ с повышением привилегий. «Альт Сервер виртуализации», доступный для x86_64, ARM64 и ppc64le, планируется обновить до версии 9.1 в начале осени 2020 года. Дистрибутивы «Альт» для всех платформ, кроме ВК «Эльбрус», доступны для свободного скачивания. В соответствии с лицензионным соглашением физические лица могут использовать дистрибутивы бесплатно для личных целей. Юридическим лицам для полноценного использования необходимо приобретение лицензии. Более подробная информация о лицензировании и приобретении программных доступна по запросу на sales@basealt.ru. По вопросам приобретения дистрибутивов «Альт» для отечественных компьютеров «Эльбрус» предлагается обращаться в АО МЦСТ: sales@mcst.ru. Разработчикам предлагается принять участие в совершенствовании репозитория «Сизиф» и его стабильных веток; также есть возможность воспользоваться для своих целей инфраструктурой разработки, сборки и поддержания жизненного цикла, с помощью которой развиваются данные продукты. Эти технологии и инструменты созданы и совершенствуются специалистами из ALT Linux Team.
