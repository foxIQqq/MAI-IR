Новый выпуск легковесного дистрибутива antiX 21

Опубликован релиз легковесного Live-дистрибутива AntiX 21, оптимизированного для установки на устаревшее оборудование. Выпуск основан на пакетной базе Debian 11, но поставляется без системного менеджера systemd и с eudev вместо udev. Для инициализации на выбор могут использоваться runit или sysvinit. Пользовательское окружение по умолчанию сформировано при помощи оконного менеджера IceWM. Для работы с файлами доступны zzzFM и ROX-Filer. Дистрибутив доступен в четырёх редакциях: Full, Base, Core и Net: antiX-full (antiX-21_x64-full.iso 1.4GB): 4 оконных менеджера - IceWM (по умолчанию), fluxbox, jwm и herbstluftwm плюс полный пакет LibreOffice. Сборки x86_64 поставляются с 2 ядрами Linux - 4.9 и 5.10. antiX-base (antiX-21_x64-base.iso 774MB – для того, чтоб разместиться на 800MB CD): 4 менеджера окон - IceWM (по умолчанию), fluxbox, jwm и herbstluftwm. antiX-core (antiX-21_x64-core.iso 437MB) - без X, cli-installer без шифрования, но должен поддерживать большинство беспроводных сетей. antiX-net (antiX- 21-net_x64-net.iso 176MB) - без X, cli-installer без шифрования. Достаточно интернет-подключения, для подготовленных пользователей. Новый выпуск включает ядра Linux ​​4.9.0-279 с fbcondecor splash и ​​5.10.57 (только x64 full), LibreOffice 7.0.4-4, Firefox-esr 78.14.0esr-1 в antiX-full, Seamonkey 2.53.9.1 в antiX-base, Claws-mail 3.17.8-1, CUPS для печати, ​​XMMS для прослушивания музыки, Celluloid и mpv для воспроизведения видео, SMTube для воспроизведение видео с YouTube без использования браузера, Streamlight-antix для потоковой передача видео с очень низким использованием оперативной памяти, Qpdfview для чтения PDF-файлов. Из других новшеств выпуска можно отметить: Файловый менеджер SpaceFM заменён на zzzFM; Менеджер входа Slim заменён на slimski; mps-youtube заменён на ytfzf; Задействован сервисный менеджер runit (только для версии с runit); Добавлена программа установки приложений App Select; Добавлен интерфейс настройки пиктограмм в панели IceWM. Приложения собственной разработки доступные в репозиториях: 1-to-1-voice-antix - голосовой чат между двумя ПК с поддержкой шифрования 1-to-1-aid-antix - удалённый помощник ssh-pipeline - удаленный доступ через зашифрованное соединение ssh
