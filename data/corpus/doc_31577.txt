Релиз менеджера фотографий Shotwell 0.11

Вышел релиз программы для управления коллекцией фотографий Shotwell 0.11. Готовые бинарные сборки доступны для Linux-дистрибутивов Ubuntu и Fedora. Базовые возможности Shotwell: Инструменты для импорта и конвертации новых фотографий; Удобные возможности каталогизации и навигации по имеющейся коллекции, поддержка группировки по времени и тегам; Поддержка выполнения типовых операций по обработке изображений (вращение, устранение эффекта красных глаз, корректировка экспозиции, оптимизация цветности и т.п.); Средства для публикации в социальных сетях, таких как Facebook, Flickr и Picasa. Из добавленных в новой версии улучшений, можно отметить: Теги теперь можно оформлять в иерархическом представлении; Загруженная с фотоаппарата пара файлов RAW + JPEG теперь воспринимается как одна фотография; Возможность выбора модуля для вывода RAW-изображений, созданного силами Shotwell или предоставляемого камерой; Для хранения параметров конфигурации теперь используется GSettings, вместо GConf; Настройка "Скрыть уже импортированные фотографии" остается видимой между сеансами импорта данных; Добавлены новые сохраняемые опции поиска; JPEG-миниатюры для RAW-изображений больше не сохраняются в домашней директории и создаются при необходимости налету; Реализована поддержка изображений Windows Bitmap (.bmp).
