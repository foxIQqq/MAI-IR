Релиз PhoneGap 1.0, фреймворка для разработки приложений для мобильных платформ

Представлен первый стабильный релиз PhoneGap 1.0, фреймворка для разработки универсальных мобильных приложений, работающих на различных мобильных платформах и написанных с использованием web-технологий (HTML5, CSS3, JavaScript). Написанное с использованием PhoneGap приложение может быть собрано для использования на платформах Android, Symbian, Apple iOS, Blackberry, webOS, Bada и Windows Phone 7. Процесс установки и внешний вид таких приложений ничем не отличается от программ, созданных с использованием родного для каждой мобильной платформы инструментария. Через специальную прослойку поддерживается использование специфичных для каждой платформы функций, а также доступ к API для взаимодействия с оборудованием, телефонным стеком, адресной книгой, GPS, звуковой подсистемой, камерой и другими специфичными компонентами. Через создание плагинов поддерживается расширение функциональности и API PhoneGap. Из особенностей нового выпуска отмечается общая стабилизация API и интерфейса для написания плагинов, обеспечение совместимости с W3C DAP API, реализация API для работы с адресной книгой (Contacts API) и подготовка средств для организации удаленной отладки. Код PhoneGap распространяется под модифицированной лицензией BSD или лицензией MIT.
