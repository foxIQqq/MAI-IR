Релиз MySQL 5.0.87 и вторая альфа-версиия PostgreSQL 8.5

Представлена вторая альфа версия PostgreSQL 8.5 в которой представлены следующие новшества: Для упрощения разграничения доступа реализованы конструкции "DEFAULT PRIVILEGES" и "GRANT ALL ON"; Через оператор "DO" теперь можно выполнять код в SQL-выражении на процедурном языке без создания отдельной функции; Возможность назначать триггеры для столбцов; Возможность использования в функциях именованных параметров. Новшества реализованные в первой альфа версии PostgreSQL 8.5: Возможность задания флага DEFERRABLE для выражений с признаком UNIQUE; Поддержка конструкции IF EXISTS в DROP COLUMN/CONSTRAINT; Возможность вывода результатов работы EXPLAIN в XML и JSON форматах; Новый шестнадцатиричный формат ввода и вывода для типа данных BYTEA Поддержка многопоточности в утилите pgbench, что позволяет задействовать все CPU в системе и сгенерировать более реалистичную тестовую нагрузку; Улучшение в системе сборки и уточнение документации; В будущих альфа-версиях PostgreSQL 8.5 планируется добавить поддержку режима "горячего резерва", с возможностью выполнения select запросов на запасном сервере. Кроме того можно отметить выход корректирующего релиза Community-редакции СУБД MySQL 5.0.87, в котором исправлено 19 ошибок, из которых 6 приводили к краху серверного процесса.
