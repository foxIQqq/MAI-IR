Релиз программы для обмена сообщениями Empathy 2.30.0

Вышел релиз программы для мгновенного обмена сообщениями, организации VoIP-переговоров и видеочата - Empathy 2.30.0, немного опередив выход родительского проекта GNOME 2.30, релиз которого выйдет ближе к вечеру. Основные улучшения: Переработан диалог для управления параметрами аккаунта, который теперь реализован в виде отдельного процесса, который можно вызывать из меню настроек Gnome без запуска Empathy. Добавлен виджет для упрощения настройки аккаунта в Facebook. Перерисованы некоторые пиктограммы. Изменена логика работы мастера первичной настройки, в котором теперь можно сразу указать расширенные параметры XMPP-подключения; В чате реализованы частоиспользуемые IRC-команды, такие как /join, /topic и /nick. Добавлена возможность создавать защищенные паролем комнаты. Улучшено информирование об ошибках, например, более наглядно отображаются проблемы с подключением, при выводе которых одновременно предлагается произвести реконнект или изменить настройки. При проблемах с VoIP звонками ошибки отображаются в отдельном всплывающем окне; Для отправки файлов между контактами их теперь можно просто перетаскивать мышью; В окно чата добавлена строка для быстрого поиска; На ссылки в строке статуса теперь можно кликать; Исправлено 310 ошибок. Кроме того, разработчики отметили, что в состав версии Empathy 2.30 не успели добавить несколько интересных улучшений. В частности добавлены две новые группы контактов "Любимые" (Favourite Contacts) и "Частоиспользуемые" (Frequent Contacts), которые всегда находятся вверху списка, при этом первая группа заполняется пользователем, а вторая автоматически. Также реализован новый сервис telepathy-logger, который позволяет организовать доступ к истории переговоров и статистике использования контактов из других приложений, построенных на базе фреймворка Telepathy.
