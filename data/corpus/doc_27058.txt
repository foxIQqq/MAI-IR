Упрощение тестирования экспериментальных версий Ubuntu при помощи TestDrive

В рамках проекта TestDrive разработан графический интерфейс для упрощения тестирования экспериментальных сборок Ubuntu и отправки уведомлений при обнаружении проблем. Проект рассчитан на начинающих пользователей, мнение которых хочется учесть при тестировании. TestDrive автоматизирует загрузку тестовых образов и их запуск в виртуальной машине. Для установки интерфейса необходимо выполнить: sudo add-apt-repository ppa:testdrive/ppa sudo add-apt-repository ppa:testdrive-gtk/ppa sudo apt-get install testdrive testdrive-gtk А затем запустить приложение TestDrive GTK Front-end из меню "Приложения / Системные утилиты".
