Сервис Windows Live переходит на открытую платформу WordPress

Компания Microsoft приняла решение по использованию для ведения блогов по умолчанию в сервисе Windows Live (бывший MSN) открытой системы управления контентом WordPress, вместо ранее используемой системы для ведения блогов Windows Live Space. Подобный шаг укрепит позиции WordPress в роли наиболее популярной платформы для ведения блогов - после завершения миграции Windows Live, аудитория данной CMS возрастет примерно на 30 миллионов пользователей. В настоящее время под управлением WordPress уже работает около 26 миллионов блогов, что составляет примерно 8.5% от всех сайтов сети. По заявлению директора сервиса Windows Live, процесс миграции всех существующих блогов с технологии Windows Live Space на новый движок займет около шести месяцев. Миграция не будет носить принудительный характер, пользователям предоставлена опция для преобразования блога в новый формат, что дает возможность отложить миграцию на будущее (на миграцию отводится 6 месяцев). Пользователю также предоставлена возможность загрузить все данные страницы на локальный диск. При попытке создания нового блога отныне по умолчанию осуществляется переброс на регистрационную форму движка WordPress. В дальнейшем, в рамках партнерства Microsoft и WordPress, планируется интеграция поддержки блогов сервиса WordPress.com в Windows Live Messenger, что позволит организовать отправку уведомлений друзьям о новых публикациях в блогах.
