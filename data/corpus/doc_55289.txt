В Wayland-драйвере для Wine появилась поддержка Vulkan и многомониторных конфигураций

Компания Collabora представила улучшенный вариант Wayland-драйвера, позволяющего запускать через Wine приложения, использующие GDI и OpenGL/DirectX, напрямую в окружении на базе Wayland, без применения прослойки XWayland и избавившись от привязки Wine к протоколу X11. Продолжается обсуждение с разработчиками Wine вопроса включения поддержки Wayland в ветку Wine Staging с последующим переносом в основной состав Wine. В новой версии: Добавлена поддержка графического API Vulkan, в том числе для управления окнами (изменения размера, полноэкранный вывод и т.п.). API Vulkan может использоваться как напрямую, так и для реализации возможностей Direct3D через применение WineD3D или DXVK. Добавлена поддержка многомониторных конфигураций. Поддерживается динамическое добавление и удаление мониторов, изменение разрешения и режима для каждого монитора, масштабирование для HiDPI. Обеспечена работа с экранами с высокой плотностью пикселей (HiDPI). Добавлена поддержка ограничения границ курсора (clipping) и относительного перемещения курсора. Реализована обработка раскладок клавиатур в Wayland.
