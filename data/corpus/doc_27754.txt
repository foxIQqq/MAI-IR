Motorola поглотила компанию-производителя открытого web-фреймворка Cappuccino

Компания Motorola купила компанию 280 North, занимающуюся развитием открытого (LGPL) web-фреймворка Cappuccino. По неофициальной информации сумма сделки составила 20 миллионов долларов. Motorola планирует использовать наработки 280 North для создания новых web-технологий и приложений для устройств на базе платформы Android. Фреймворк Cappucinio позволяет разрабатывать десктоп-подобные web-приложения, работающие внутри браузера. Для разработки приложений используется Cappucinio API, похожий на API GNUstep и Cocoa, и язык Objective-J, похожий на Objective-C, но построенный на базисе JavaScript и поддерживающий наследование, динамический обмен сообщениями и функции, позволяющие осуществлять более полный контроль над апплетами, запущенными в браузере. Итоговое приложение транслируется в представление на HTML, CSS и JavaScript.
