Наглядное представление взаимодейтсвия разработчиков в сервисе Github

Подготовлено несколько наглядных карт, демонстрирующих связь разработчиков различных проектов в репозиториях сервиса Github. Кроме общей карты, построены частные визуальные отчеты, привязанные к используемым в Github языкам программирования (PHP, Python, Perl, Javascript, Ruby). При просмотре карт сразу бросается в глаза разобщенность программирующих на PHP и единство разработчиков на языке Ruby. Карты построены с использованием открытого инструментария Gephi, при желании любой желающий может сгенерировать подобные отчеты для любого набора данных.
