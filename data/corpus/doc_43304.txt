Выпуск мобильной платформы Firefox OS 2.5

Проект Mozilla объявил о доступности мобильной платформы Firefox OS 2.5 и специальной редакции Firefox OS 2.5 Developer Preview, поставляемой в виде приложения для платформы Android и позволяющей оценить работу приложений Firefox OS без запуска эмулятора и без перепрошивки имеющихся устройств. Также доступны и готовые прошивки с Firefox OS 2.5, которые сформированы для 12 моделей смартфонов, включая Nexus 4, Nexus 5, Moto G и Xperia T2/Z2/Z3. Основные новшества Firefox OS 2.5: Система дополнений, реализованная по аналогии с дополнениями для web-браузера. Дополнения позволяют изменить поведение и расширить возможности отдельных программ или всех приложений, включая системные программы. Дополнения распространяются наряду с обычными программами через каталог Firefox Marketplace, для управления дополнениями предлагается отдельный раздел в настройках. Режим защиты конфиденциальности (Private Browsing), включающий средства для противодействия отслеживанию перемещений пользователя при посещении сайтов; Возможность "Pin the Web", позволяющая манипулировать сайтами как web-приложениями, например, закрепить определённые страницы на домашнем экране; Поддержка смены домашних экранов, дающая возможность создавать альтернативные реализации пользовательской оболочки; Поддержка Web Speech API для создания приложений с голосовым управлением; Оптимизация производительности и сокращения времени запуска; Новая архитектура оболочки Gaia (NGA, New Gaia Architecture), являющаяся первым шагом на пути разделения приложений на фронтэнды и бэкенды; При переводе устройства в режим разработчика, появилась возможность просмотра исходных текстов web-приложений непосредственно на мобильном устройстве; Presentation API, позволяющий создавать приложения, выводящие информацию на внешние экраны; Расшиение Camera API, Data Store API и Telephony API; Сборка Firefox OS 2.5 Developer Preview, оформленная в виде приложения для платформы Android и реализующая альтернативный домашний экран для имеющихся устройств на базе Android, позволяющий разместить бок о бок приложения для Firefox OS и Android. Альтернативный домашний экран основан на коде штатной пользовательской оболочки Gaia, предлагаемой в Firefox OS.
