В Индии развивают собственную сборку Debian GNU/Linux для замены Windows в госучреждениях

Индийский Центр разработки расширенных компьютерных технологий, созданный по инициативе правительства Индии, c 2007 года развивает собственный Linux-дистрибутив BOSS (Bharat Operating System Solutions), представляющий собой адаптированную для местного применения сборку Debian GNU/Linux. Представленный в этом году выпуск BOSS 6.1, основанный на пакетной базе Debian 8 "Jessie" и оснащённый рабочим столом на основе GNOME 3.14, одобрен для внедрения в государственных учреждениях Индии в рамках программы по замене Windows на собственную подконтрольную программную платформу. На новую ОС планируется перевести все официальные государственные компьютерные системы, в настоящее время оснащённые Windows. Первым к миграции приступило Министерство внутренних дел Индии. Дистрибутив примечателен проведением работы по добавлению в GNOME и используемые приложения поддержки двадцати языков, находящихся в обиходе жителей Индии. Поддержка используемых в Индии языков в том числе доступна в экранной клавиатуре, системе голосовой навигации и синтезаторе речи (языки доступны для людей с ослабленным зрением). В состав входит подборка программ, нацеленных на упрощение миграции госучреждений с Windows на GNU/Linux, такие как конвертеры документов и запускаемый из Windows инсталлятор. В том числе подготовлены специализированные сборки для армии, школ и колледжей. В качестве офисного пакета предлагается Libreoffice, в качестве браузера - Firefox, в качестве медиаплеера - VLC. При разработке повышенные требования уделяются обеспечению безопасности и надёжности - дистрибутив успешно прошёл тесты, разработанные Организацией оборонных исследований и разработок Индии для утверждения программных систем, используемых для военных нужд. Одним из основных мотивов внедрения дистрибутива является желание повысить защищённость IT-инфраструктуры и обеспечить независимость от сторонних поставщиков (в опубликованных Эдвардом Сноуденом документах, упоминалось желание спецслужб США контролировать IT-системы в Индии, попытки осуществления атак и существование программы по сбору информации о гражданах Индии). Продукт сертифицирован организацией Linux Foundation на предмет соответствия требованиям спецификации LSB. Для загрузки доступна сборка для архитектуры i386, размером 1.8 Гб.
