Релиз платформы для организации групповой работы Kolab 3.3

Представлен релиз Kolab 3.3, платформы для организации совместной коллективной работы, базирующейся на технологиях KDE. Основу платформы составляет серверное ПО Kolab Server, позволяющее организовать работу серверной инфраструктуры, реализующей средства групповой работы, единой адресной книги, календаря-планировщика и почтового сервиса. В качестве клиентских приложений могут выступать как стандартные программы из состава KDE PIM (Kontact, KMail, KOrganizer и KAddressbook), так и сторонние системы, в том числе доступен набор коннекторов для Microsoft Outlook (Bynari Connector, плагин Toltec Connector 2 и MAPI-хранилище KONSEC Konnektor). В качестве базового web-клиента в составе платформы поставляется Roundcube Webmail, в полной мере поддерживающий вторую и третью версии формата хранилища Kolab. Для интеграции с Thunderbird и SeaMonkey проектом развивается специальный плагин SyncKolab. Модуль интеграции также доступен для GNOME Evolution. Для организации работы с мобильными системами предоставляется ряд решений по синхронизации данных, таких как CalDAV/CardDAV и ActiveSync-совместимые решения Kolab ActiveSync Server и Z-Push. Для организации работы в Kolab Server задействованы стандартные открытые компоненты, такие как сервер директорий 389, Apache httpd, Postfix, Cyrus IMAP, SASL, OpenSSL. Для управления системой подготовлен специальный административный web-интерфейс. Поддерживается ведение единой адресной книги, интегрируемой с почтовым сервисом с поддержкой POP3 и IMAP4. Для шифрования почты предлагается S/MIME-шифрование на стороне клиента. Бинарные сборки Kolab 3.3 подготовлены для RHEL, SUSE, Debian, Fedora, Ubuntu и openSUSE. Особенности новой версии: Поддержка установки тегов для писем электронной почты, наглядного выделения помеченных писем в общем списке и поиска по тегам. В будущих выпусках поддержку тегов планируется добавить во все остальные модули, что позволит использовать одни и те же теги для почты, адресной книги, задач и событий; Добавлена возможность работы с заметками прямо из web-клиента, создания нескольких разделов заметок и их совместного использования c другими людьми. Заметки автоматически синхронизируются с Kolab Desktop Client и мобильными устройствами; Добавлен интерфейс для управления совместным использованием ресурсов, например, можно распланировать доступ к машине, проектору и комнате переговоров. Сводный режим навигации по содержимому папок, позволяющий искать и подписываться на доступные для совместного использования календари, адресные книги, списки задач и т.п. Режим быстрого обзора предстоящих событий из всех доступных разделов календаря-планировщика; Возможность поиска одновременно в разных папках; Календарь дней рождения, автоматически показывающий информацию о грядущих днях рождения людей, занесённых в адресную книгу; Средства для уведомления коллег о недоступности в офисе. Пользователь имеет возможность определить период, в который его не будет на месте, и написать сообщение, которое будет выдаваться при попытке связаться с сотрудником; Возможность делегирования другому сотруднику части календаря, задач или отдельных папок электронной почты, например, в связи с большой занятостью из-за выполнения первоочерёдного задания; Интерфейс администратора Kolab Webadmin улучшен для выполнения типовых задач администратора через web-интерфейс, без выполнения операций в командной строке. В том числе в Kolab Webadmin теперь можно создавать новые совместно используемые папки, редактировать ACL, управлять содержимым каталога LDAP.
