Корректирующий релиз GNOME 3.8.2

Доступен корректирующий выпуск GNOME 3.8.2, в который включены исправления ошибок, обновлена документация, улучшены переводы и внесены незначительные улучшения, направленные на увеличение стабильности. Демонстрационную Live-сборку на основе GNOME 3.8, можно загрузить с данной страницы. Следующий значительный релиз GNOME 3.10 запланирован на 25 сентября. В осеннем выпуске ожидается появление опциональной возможности запуска оболочки GNOME Shell в качестве композитного сервера Wayland, для чего будет обеспечена поддержка базовых интерфейсов, идентичных с эталонным композитным сервером Weston. Среди других планируемых улучшений: средства для автоматического обновления дополнений к GNOME Shell; интеграция поддержки Flickr в приложение для просмотра фотографий; новые базовые приложения для работы с картами и для навигации по коллекции видео; обновлённые программы для навигации по музыкальной коллекции и ведения заметок; подготовка инструментов для работы с Git-репозиториями, которые лягут в основу будущей интегрированной среды разработки GNOME IDE; реализация единого системного статусного меню; интеграция c пакетом Zimbra для организации совместной работы.
