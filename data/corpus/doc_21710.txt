Вышел релиз LiveCD дистрибутива SystemRescueCd 1.2.0

Вышел релиз SystemRescueCd 1.2.0, специализированного LiveCD дистрибутива на основе Gentoo Linux, предназначенного для восстановления системы после сбоя. В новой версии представлено несколько значительных изменений: произведен переход на Linux ядро 2.6.29.3 и использование XFce4 в качестве графического окружения пользователя вместо JWM. При этом удалось сохранить размер установочного iso-образа (234 Мб). Обновлены версии программ, входящих в комплект дистрибутива: Xorg-server-1.5.3; testdisk 6.11; ntfs3g 2009.4.4; GNU tar 1.22; GParted 0.4.5 с улучшенной поддержкой dmraid и crypt-luks; fsarchiver 0.5.4 - ПО для клонирования и архивирования файловых систем с поддержкой ext4, btrfs и ntfs, а также с возможностью шифрования архивов и сжатия методом lzma. В состав SystemRescueCd 1.2.0 входит около 400 пакетов программ, самые полезные из которых: Утилиты для восстановления после сбоев: sfdisk - сохранение и восстановление таблицы разделов; fsarchiver - разработанная авторами SystemRescueCd утилита для резервного копирования содержимого файловых систем; partimage - резервное копирование образов дисковых разделов, включая возможность работы с архивом по сети; TestDisk - пакет для восстановления потерянной таблицы разделов, удаленных файлов, загрузочных секторов. Поддерживаются ФС reiserfs, ntfs, fat32, ext2/3 и т.д. PhotoRec - восстановление файлов из сбойных разделов по сигнатурам, поддерживается большое число форматов изображений, документов и мультимедиа форматов. Утилиты для выполнения системных операций: dvd+rw-format, mkudffs, cdrecord, mkisofs, dvd+rwtools - средства для записи CD/DVD; gparted - программа для управления дисковыми разделами, поддерживает изменение размера разделов без потери данных; GNU-Parted - консольный редактор дисковых разделов; Наборы утилит для обслуживания файловых систем: reiserfsprogs, Reiser4progs, xfsprogs (XFS), jfsprogs (JFS), e2fsprogs (ext2/ext3/ext4), ntfsprogs, ntfs-3g-AR (NTFS), dosfstools (FAT/VFAT); Набор архиваторов и программ для сжатия данных: tar, gzip, bzip2, zip, unzip, rar, unrar, unace, Dar (Disk Archiver); Текстовые редакторы: nano, vim, qemacs, elvis, joe, Leafpad; Файловый менеджер Midnight Commander; Сетевые утилиты: lftp, ssh (клиент и сервер), ppp, lynx, nslookup, nmap, netcat. Поддержка LUFS (Linux Userland File System), NFS, Samba "mount -t smbfs //ip/share /mnt/path"; Утилиты, связанные с безопасностью: скрипт для поиска руткитов Chkrootkit, антивирусный пакет ClamAv, утилиты для полной очистки удаленных файлов shred и wipe, программа для вычисления контрольных сумм содержимого директорий - md5deep; Языки программирования: Perl и Python.
