Выпуск Simply Linux и Альт Сервер Виртуализации на 10-й Платформе ALT

Доступен выпуск ОС Альт Сервер Виртуализации 10.0 и Simply Linux (Симпли Линукс) 10.0 на основе Десятой платформы ALT (p10 Aronia). Альт Сервер Виртуализации 10.0, разработанный для использования на серверах и реализации функций виртуализации в корпоративной инфраструктуре, доступен для всех поддерживаемых архитектур: x86_64, AArch64, ppc64le. Изменения в новой версии: Системное окружение на базе ядра Linux 5.10.85-std-def-kernel-alt1, Glibc 2.32, OpenSSL1.1.1, а также поддержка нового оборудования. В p10 по-умолчанию задействована единая унифицированная иерархия cgroup (cgroup v2). Механизм ядра cgroups широко используется такими важными и популярными инструментами, как Docker, Kubernetes, LXC и CoreOS. В состав репозитория p10 включен pve-backup для создания сервера, позволяющего организовать управление резервными копиями виртуальных машин в PVE. Docker 20.10.11, Podman 3.4.3, LXC 4.0.10/LXD 4.17. Обновлены официальные образы контейнеров: docker и linuxcontainers. Обновлены образы для установки в облачных окружениях. ZFS 2.1 (может использоваться для организации хранилища в PVE). Системы виртуализации: PVE 7.0, OpenNebula 5.10. Клиентская часть FreeIPA 4.9.7. QEMU 6.1.0. Менеджер виртуальных машин libvirt 7.9.0. Open vSwitch 2.16.1. Новые версии файловых систем Ceph 15.2.15 (octopus), GlusterFS 8.4. В системе управления контейнерами Kubernetes 1.22.4 осуществлен переход на использование cri-o. Simply Linux 10.0 подготовлен для архитектур x86_64, AArch64 (включая поддержку «Байкал-М»), AArch64 для RPi4, i586, e2k v3/v4/v5 (от 4С до 8СВ) и riscv64 (впервые). Дистрибутив представляет собой простую в работе систему с классическим рабочим столом на основе Xfce, предоставляющую полную русификацию интерфейса и большинства приложений. Изменения в новой версии Simply Linux (на не-x86/arm версии ПО могут отличаться): Системное окружение на базе ядра Linux 5.10.85-std-def-kernel-alt1, Glibc 2.32, набора компиляторов GCC10, systemd 249.7. Графические средства обновления ядра реализуются утилитой alterator-update-kernel 1.4. Xorg 1.20.13. Wine 6.14 для i586 и x86_64. Графическая оболочка Xfce 4.16 (изменение интерфейса в связи с переходом на GTK+3 (3.22), улучшение функционала настроек дисплея и новый дизайн). Также присутствует MATE 1.24. Файловый менеджер Thunar 4.16. Программа управления настройками сети NetworkManager 1.32. Центр управления системой Alterator 5.4. Браузер Chromium 96.0. В riscv64 - Epiphany 41.3 и в e2k - Mozilla Firefox ESR 52.9. Почтовый клиент Thunderbird 91.3 - улучшена работа с вложениями, есть обновления по безопасности. В riscv64 почтовый клиент Claws Mail 3.18. Клиент обмена мгновенными сообщениями Pidgin 2.14.3 (есть во всех архитектурах, кроме riscv64). Офисные приложения LibreOffice 7.1.8. Редактор растровой графики GIMP 2.10 с обновлённым переводом на русский. Редактор векторной графики Inkscape 1.1 (присутствует во всех архитектурах, кроме riscv64). Добавлен экспорт в форматы JPG, TIFF, optimized PNG и WebP, появился менеджер расширений. Аудиопроигрыватель Audacious 4.1 с выбором интерфейса Qt (возможна настройка горячих клавиш) или GTK. Видеопроигрыватель VLC 3.0.16. В e2k и для riscv64 — Celluloid 0.21. Клиент удалённого рабочего стола Remmina 1.4. Установочные образы Альт Сервер Виртуализации и Simply Linux доступны для загрузки (яндекс-зеркало). Продукты распространяются в рамках лицензионного договора. Свободно использовать загруженную версию могут физические лица, в том числе – индивидуальные предприниматели. Коммерческие и государственные организации могут скачивать и тестировать дистрибутив. Для постоянной работы с «Альт Сервер Виртуализации» в корпоративной инфраструктуре юридическим лицам необходимо приобретать лицензии или заключать лицензионные договоры в письменной форме. Пользователи дистрибутивов, построенных на Девятой платформе (p9), могут обновить систему из ветки p10 репозитория «Сизиф» (Sisyphus). Для новых корпоративных пользователей предусмотрена возможность получения тестовых версий, а частным пользователям традиционно предлагается бесплатно загрузить нужную версию ОС «Альт» с сайта «Базальт СПО» или с нового сайта загрузки getalt.ru. Варианты для процессоров «Эльбрус» доступны юридическим лицам, подписавшим NDA с АО МЦСТ, по письменному запросу. Срок поддержки в части обновлений по безопасности (если иное не предусмотрено условиями поставки) - до 31 декабря 2024 года. Разработчикам предлагается принять участие в совершенствовании репозитория «Сизиф»; также есть возможность воспользоваться для своих целей инфраструктурой разработки, сборки и поддержания жизненного цикла, с помощью которой развиваются ОС Альт. Эти технологии и инструменты созданы и совершенствуются специалистами из ALT Linux Team.
