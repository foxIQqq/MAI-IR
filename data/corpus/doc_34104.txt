Доступен первый стабильный релиз Skype 4.0 для Linux

Представлен релиз Skype 4.0 для Linux, первый стабильный выпуск после многолетнего бета-тестирования. Представленный выпуск также является первым релизом клиента Skype для Linux после поглощения бизнеса Skype компанией Microsoft. Skype для Linux не только перепрыгнул с версии 2.2 до 4.0, но и лишился статуса бета-версии и во многом догнал версию для Windows по своим возможностям. Из новшеств отмечается: Множественные улучшения внешнего вида. Представлен новый вид окна чата, в котором пользователю стало проще отслеживать все свои чаты в рамках одного унифицированного окна. Старый режим при желании можно вернуть в настройках; Новое оформление диалога осуществления вызова (Call View); Новые смайлики и пиктограммы для определения статуса; Улучшено качество как аудио звонков, так и видео; Добавлена поддержка многих web-камер; Увеличение стабильности программы, решение множества проблем, ранее приводившим к крахам и зависаниям на платформе Linux; Улучшения в синхронизации чата; Увеличение скорости загрузки истории общения в чате; Поддержка возможности сохранения и просмотра номеров телефонов в профиле пользователя Skype. Среди возможностей, поддерживаемых в Skype 4.0 для Linux, отмечается поддержка организации конференц-связи, перенаправление звонков, отправка SMS на мобильные телефоны, организация совместного доступа к рабочему столу, осуществление звонков на стационарные номера, выполнение видеовызовов, отправка файлов. Сборки подготовлены для Ubuntu 10.04 (32-битная версия в отдельных пакетах для 32-бит и 64-бит систем), Debian 6 (32-битная версия в отдельных пакетах для 32-бит и 64-бит систем), Fedora 16 (32-битная версия в универсальном пакете для 32-бит и 64-бит систем) и openSUSE 12.1 (32-битная версия в универсальном пакете для 32-бит и 64-бит систем). В качестве системных требований приводятся следующие версии программ: Qt 4.6.0, D-Bus 1.0.0, libasound2 1.0.18, а также опционально BlueZ 4.0.0 и PulseAudio 1.0.
