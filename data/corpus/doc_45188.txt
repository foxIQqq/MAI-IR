В ноутбуках Lenovo выявлена проблема, не позволяющая установить Linux

Сегодня несколько изданий опубликовали заявление о преднамеренном блокировании установки сторонних операционных систем в ноутбуках Lenovo с меткой "Microsoft Signature PC", которое было осуществлено в рамках специального соглашения с компанией Microsoft. Сведения о проблеме были озвучены по мотивам обсуждений в ряде форумов и комментария в разделе отзывов на Best Buy пользователя, представившегося экспертом по продукции Lenovo. В итоге информация не подтвердилась и представители компании Lenovo опровергли данные сведения, указав, что Lenovo не блокирует сторонние ОС и стремится обеспечить сертификацию Linux и средства для установки Linux на широком спектре своих продуктов. Возникшую проблему Lenovo списывает на использование дистрибутивов без необходимых драйверов. С сообщением, что блокировка не является преднамеренной, также выступил Мэтью Гаррет (Matthew Garrett), известный разработчик ядра Linux и один из директоров Фонда Свободного ПО. Проблема выражается в том, что установка Linux на различных ноутбуках Lenovo, таких как Yoga 900 ISK2 и Ideapad 710S, невозможна из-за работы SSD-накопителя только в проприетарном RAID-режиме, недоступном для Linux. Данный RAID-режим не отключается в BIOS и для SSD невозможно включить типовые режимы AHCI или NVMe, которые предусмотрены в платформе Intel. RAID-режим реализован компанией Intel, которая пока не предоставила патчей с его поддержкой для ядра Linux. Мэтью Гаррет предполагает, что использование RAID-режима по умолчанию связано с необходимостью установки оптимальных режимов энергосбережения, которые выставляются только драйвером Intel для RAID-режима. Удаление из BIOS функции смены режима может быть связано с желанием недопустить установку неоптимальных настроек, при которых сокращается время автономной работы. По мнению Гаррета, в возникшей проблеме больше виновата компания Intel, которая до сих пор не опубликовала информацию о настройке оптимальных режимов энергосбережения и не предоставила для Linux драйвер с поддержкой режима RAID для SSD.
