Является ли Oracle Enterprise Linux форком Red Hat Enterprise Linux ?

Руководитель Linux направления в Oracle попытался развеять представление о том, что Oracle Unbreakable Linux является точным форком Red Hat Enterprise Linux и паразитирует на разработках Red Hat. Никто не отрицает, что Unbreakable Linux основан на RHEL, но по утверждению представителей Oracle это не форк. Компания вносит множество изменений, которые возвращаются в родительские проекты (upstream). При этом полная бинарная совместимость с RHEL не подразумевает идентичность версий программ в пакетах. Разработчики Oracle активно участвуют в разработке Linux ядра, развивают Btrfs и кластерную файловую систему OCFS (Oracle Cluster File System), развивают системы виртуализации на базе Xen. В состав дистрибутива Unbreakable Linux входит кластерное ПО Oracle Clusterware, которое раньше поставлялось только для клиентов Real Application Clusters (RAC). Red Hat в системах виртуализации и кластерных технологиях придерживается другой позиции, отдавая предпочтение KVM и файловой системе GFS.
