Вторая тестовая версия Ubuntu для автомобильных информационно-развлекательных систем

Компания Canonical анонсировала выход второй бета-версии продукта Ubuntu IVI Remix, ориентированного для использования на автомобильных информационно-развлекательных системах (IVI). В качестве графической оболочки задействован Unity 2D на базе Qt. Для загрузки доступно два установочных образа: для архитектуры x86 с оптимизациями для процессоров Intel Atom (308 Мб, можно тестировать в VirtualBox) и для эталонной платформы Freescale iMX53 Sabre ARD Rev B, базирующейся на архитектуре ARM A8 (177 Мб). Пакеты со специфичными для IVI-систем наработками можно загрузить из специального PPA-репозитория. В отличие от первого августовского выпуска в новой версии произведена синхронизация с актуальной пакетной базой Ubuntu и осуществлён переход на использование штатного ядра Ubuntu. Использование синхронизированной пакетной базы с десктоп и серверной редакцией Ubuntu позволяет упростить разработку, давая возможность использования одних и тех же библиотек и инструментов для создания приложений для настольных и IVI-систем. Кроме того по сравнению с первым выпуском примерно в два раза уменьшено время загрузки и снижены требования к расходу постоянной памяти (для работы достаточно 508 Мб). Поставка готовых продуктов на базе Ubuntu IVI Remix ожидается в первой половине 2012 года. Особенностью Ubuntu IVI Remix является возможность использования широкого спектра приложений из репозиториев Ubuntu и поддержка сервиса Ubuntu One, в рамках которого предоставляется доступ к внешнему хранилищу и возможность синхронизации настроек и данных между несколькими системами (например, пользователь сможет прослушать в автомобиле музыку из коллекции на домашнем ПК). Дистрибутив полностью соответствует техническим требованиям альянса GENIVI, курирующего разработку и внедрение открытой автомобильной информационно-развлекательной платформы (IVI), и содержит описанные в базовых спецификациях GENIVI компоненты, такие как Layer Manager и DLT (Diagnostic Log and Trace).
