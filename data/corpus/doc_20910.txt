Вышел Calculate Directory Server 9.4

Calculate Directory Server - это свободная операционная система на базе Gentoo Linux, представляющая собой LDAP-сервер для централизованного хранения настроек приложений, профилей и настроек пользователей, прав доступа в едином сетевом хранилище. Возможности: Поддерживает работу как с Windows, так и с Linux клиентами. Имеет встроенные средства для настройки сервисов LDAP, Mail, Samba, FTP, Jabber. Распространяется как в виде установочного образа, так и в виде LiveCD, что позволяет оценить все возможности системы без инсталляции на ПК. Основные изменения в версии 9.4: Добавлена утилита cl-replication, включающая репликацию Unix и Samba веток между LDAP серверами. Добавлена поддержка переноса настроек рабочего стола пользователей Calculate Linux Desktop между реплицируемыми серверами. В утилитах cl-setup и cl-update добавлена поддержка настройки POP3 сервера. В утилитах cl-setup, cl-update и cl-rebuild добавлен ключ '--allow' - вызов диалога ввода разрешенных сетей, при настройке сервисов Samba и Mail. Изменились пути к профилям Linux и Windows пользователей Samba сервера. Состав: Bind 9.4.3-P1, Kernel 2.6.27.20, OpenLDAP 2.4.11, Postfix 2.5.5, ProFTPD 1.3.2, Samba 3.0.33.
