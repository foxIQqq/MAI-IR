Выпуск дистрибутива Альт Рабочая станция К 9.1

Доступен выпуск дистрибутива Альт Рабочая станция К 9.1, оснащённого графическим окружением на основе KDE Plasma и предназначенного для корпоративных рабочих мест и персонального использования. ОС внесена в Единый реестр российских программ и баз данных. Сборки подготовлены для архитектуры x86_64 в форме установочного (4,3 Гб) и Live-образа (3,1 Гб). Продукт поставляется в рамках Лицензионного договора, который предоставляет возможность свободного использования физическими лицами, но юридическим лицам допускается только тестирование, а для использования требуется приобрести коммерческую лицензию или заключить лицензионный договор в письменной форме (причины). Дистрибутив оснащён графическим интерфейсом для настройки системы, включая аутентификацию (в том числе через Active Directory и LDAP/Kerberos), установку и синхронизацию времени, управление пользователями, группами, просмотр системных журналов и добавление принтеров. В поставку включены проприетарные драйверы NVIDIA вместо свободного nouveau. Из новшеств по сравнению с восьмой версией отмечаются: существенно расширенная поддержка оборудования, в т.ч. NVMe на Intel RST и недавно вышедших видеоускорителей NVIDIA; возможен ОЕМ-режим установки с первоначальной настройкой системы при первом запуске; улучшение интеграции в гетерогенную корпоративную ИТ-инфраструктуру за счет расширения поддержки групповых политик Microsoft для пользователей и машин под управлением Linux; модули групповых политик, системных ограничений пользователей, дисковых квот, ограничения доступа к консолям/использования скриптовых языков/ использования макросов в приложениях, выключения системы в заданное время, сжатого файла подкачки ZRAM/ZSWAP, выбора алгоритмов шифрования в настройке клиента и сервера OpenVPN; веб-браузер по умолчанию -- chromium-gost вместо firefox-esr; возможность подписать файл электронной подписью прямо в офисном пакете; исключен раздел образовательных программ; Skanlite заменён на XSane, а KDE Telepathy -- на набор программ схожей функциональности; возможность использования заранее подготовленных подтомов BTRFS при установке; показ списка запланированных операций при разбивке дисков при установке; для EFI загрузчик GRUB вместо rEFInd при установке системы; редактор по умолчанию для текстового режима -- mcedit; Adobe Flash Player исключён из дистрибутива; работа NVIDIA Optimus через PRIME Render Offload (через Bumblebee более не поддерживается); возможность запускать программы с заданным ограничением потребления ресурсов; центр приложений с поддержкой Flatpak и дополнений Plasma; добавлены утилита настройки для загрузчика Grub, KDE Connect -- программа для связи компьютера и Android-смартфона, графическая утилита для запуска программ из-под другого пользователя с заданным приоритетом; автоматическая настройка сетевых принтеров, имеющих универсальный драйвер; поддержка актуальных алгоритмов ГОСТ, в т.ч. возможность задания хешей паролей пользователей в соответствии с ГОСТ и возможность создания защищённых VPN-туннелей c контролем целостности заголовков IP-пакетов в соответствии с ГОСТ; улучшены переводы приложений; улучшена работа устройств с сенсорным вводом; сделано отображение графического запроса пароля LUKS при загрузке системы; сохраняется UUID при форматировании SWAP-раздела при установке. Версии программного обеспечения: графическая среда KDE SC: Plasma 5.18, Applications 19.12, Frameworks 5.70; ядро Linux 5.10; драйверы NVIDIA 460, 390, 340; Mesa 20.1; xorg-server 1.20; LibreOffice 6.4; среда запуска win32-приложений WINE 5.20; Qt 5.12.
