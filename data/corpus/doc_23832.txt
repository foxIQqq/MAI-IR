Открытая реализация нейронной сети для распознавания изображений

В новой версии свободного Java фреймворка для организации нейронных сетей Neuroph v2.3 появился готовый механизм для распознавания изображений. Для конечного пользователя подготовлен специальный GUI-интерфейс, предназначенный для проведения обучения нейронной сети и налаживания необходимых для распознавания изображений связей. Иными словами, вначале пользователь вручную обучает систему указывая на наличие определенного элемента на изображении, потом система начинает выявлять данный элемент на новых изображениях самостоятельно. Для интеграции подобной возможности в сторонние приложения подготовлен специальный API. Например, данная функция уже интегрирована в библиотеку DotA AutoScript, распространяемую в исходных текстах. На сайте Neuroph доступна online-демонстрация возможностей пакета и обзор используемых технологий
