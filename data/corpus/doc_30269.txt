Вышел Linux-дистрибутив Amnesic Incognito Live System 0.7

Представлен новый релиз Linux-дистрибутива Amnesic Incognito Live System 0.7, основанного на пакетной базе Debian Squeeze и предназначенного для обеспечения анонимного выхода в сеть. Дистрибутив выполнен в виде Live-системы, позволяющей из коробки обеспечить максимальный уровень анонимности и безопасности. В состав входит типичный набор сетевых приложений, анонимный выход обеспечивается системой Tor 0.2.1.30. Размер загрузочного iso-образа 678 Мб. Из особенностей новой версии отмечается: Защита от анализа остаточных данных в памяти компьютера, оставшихся после использования дистрибутива. В процессе завершения работы или при вытаскивании носителя, теперь используется более надежный метод очистки памяти; Расширено число поставляемых драйверов: добавлены дополнительные драйверы принтеров, сканеров и МФУ, обеспечена поддержка 3G-модемов, добавлены прошивки для беспроводных карт Atheros и Broadcom; В комплекте с Iceweasel (Firefox) теперь поставляется дополнение HTTPS Everywhere, принуждающее использовать HTTPS при любой возможности. Конфигурация браузера изменена с целью максимального сокращения отправляемой служебной информации, которая может отрицательно повлиять на анонимность. Обеспечена возможность работы с FTP. Через FoxyProxy обеспечена поддержка i2p; В комплект добавлен набор инструментов для выполнения шифрования данных: TrueCrypt, gnome-disk-utility (Palimpsest), плагины Seahorse; Настройка межсетевого экрана по умолчанию блокируют все входящие и исходящие пакеты (кроме трафика через Tor).
