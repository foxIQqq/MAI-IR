Обновление VirtualBox 5.0.22 и третья бета-версия VirtualBox 5.1

Компания Oracle выпустила корректирующий релиз системы виртуализации VirtualBox 5.0.22, в котором отмечено 29 исправлений, направленных на увеличение стабильности и устранение регрессивных изменений. Наиболее заметные изменения: Решены проблемы при работе хост-системы в Red Hat Enterprise Linux 6.8; Добавлена поддержка ядра Linux 4.7 для хостов и гостевых систем; В дополнениях для гостевых систем на базе Linux решены проблемы с некоторыми 32-разрядными системами; устранена проблема со смещением указателя мыши; исправлены ошибки, приводившие к краху X.Org после перезапуска; устранено зависание настройки X11 при загрузке модуля для совместного доступа к директориям; убрано предупреждение при отсутствии сервиса Drag&Drop на стороне хоста; Устранена появившаяся в VirtualBox 5.0.18 проблема из-за которой перестали работать старые выпуски X.Org при использовании ядра Linux, новее 3.11; Устранены потенциальные крахи гостевых систем с Linux на хостах с процессорами AMD; В дополнения для гостевых систем на базе Solaris добавлена поддержка сервера X.org 1.18; Решены проблемы с работой на некоторых хост-системах с процессором Intel Atom; Устранены проблемы, возникающие при запуске 32-разрядных гостевых систем в 64-разрядном хост-окружении с процессорами Intel Sandy Bridge и Ivy Bridge; Проведена оптимизация производительности новой реализации I/O-APIC; Устранено потенциальное зависание 32-разрядных гостевых систем с EFI на системах с процессороами Intel; Увеличено максимальное разрешения захвата видео с экрана; Обеспечено корректное информирование гостевой системы через ACPI при изменении состояния заряда аккумулятора. Дополнительно можно отметить выход третьей бета-версии новой ветки VirtualBox 5.1. По сравнению со второй бета-версией отмечаются только исправления ошибок, оптимизации производительности и корректировки начальных значений ОЗУ, видеопамяти и размера жесткого диска для некоторых типов гостевых систем. В установщике для Linux обеспечено удаление старого модуля ядра перед установкой нового. Со списком улучшений VirtualBox 5.1 можно познакомиться в тексте анонса первого бета-выпуска
