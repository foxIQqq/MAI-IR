Новая версия программного комплекса для организации видеонаблюдения - AVReg 5.5

Представлена версия AVReg 5.5, программного обеспечения для организации системы видеонаблюдения на базе Linux. AvReg бесплатен при использовании до 4 аудио и видео каналов (аналоговых видеокамер, сетевых мультемедийных потоков), часть компонентов системы, например, веб-интерфейс, распространяются под лицензией GPLv2. Готовые установочные пакеты сформированы для 32-разрядных сборок Debian 5 Lenny, Debian 6 Squeeze, Ubuntu 9.10 Karmic и Ubuntu 10.04 Lucid. Из новшеств можно отметить: Добавлена поддержка HTTP/1.1 с возможностью постоянных соединений (persistent connection) для http-клиента; Реализован захват аудио-потоков по протоколу в форматах MP4/aac (Axis) и PCM/ADPCM (Axis/Planet). Добавлена возможность захвата с устаревших video4linux1 устройств (старенькие USB-камеры) и захват с устройств video4linux2, чьи драйвера не поддерживают потоковый режим захвата. Добавлен сплошной режим записи, который ранее можно было реализовать только отключив детектор движения. Выборочный режим записи предполагает включение/выключение по команде, а не только детектором движения. Для выборочного режима записи реализована возможность записи всех кадров внутри сессии движения, без "выкусывания" детектором "спокойных" кадров. Реализована начальная возможность записи звука в форматы: wav, mp2, ogg/flac, mov, m4a. Запись ведётся в сплошном режиме, отдельно от видео, с разбивкой на файлы по размеру файла и(или) длительности. httpd сервер демона avregd поддерживает базовые возможности протокола http/1.1 (включая persistent connection) и обрабатывает следующие URL: jpeg (image/jpeg), mjpeg (multipart/x-mixed-replace) и remote control (пока только вкл./выкл. принудительной записи). Добавлена новая пользовательская группа: "viewers" - только просмотр, так как предполагается что пользователи группы "operators" будут иметь доступ к вкл./выкл. принудительной записи. Добавлена базовая поддержка просмотра браузерами Opera и Google Chrome. Из полностью свободных близких по функциональности программ, можно отметить ZoneMinder.
