В день семилетия проекта Марк Шаттлворт обозначил цели развития Ubuntu 12.04

Семь лет назад, 20 октября 2004 года, был выпущен первый релиз Linux-дистрибутива Ubuntu 4.10 "Warty Warthog". Проект был основан Марком Шаттлвортом, южноафриканским миллионером, принимавшим участие в разработке Debian GNU/Linux и горящим идеей о создания доступного для конечных пользователей настольного дистрибутива, обладающего предсказуемым фиксированным циклом выхода релизов. К работе было привлечено несколько разработчиков из проекта Debian, многие из которых до сих пор принимают участие в развитии обоих проектов. Для желающих оценить прогресс развития настольных Linux-систем за последние 7 лет предлагается загрузить Ubuntu 4.10 и оценить работу в виртуальной машине или загрузившись с LiveCD. В день семилетия Марк Шаттлворт опубликовал в своём блоге цели развития будущего выпуска Ubuntu 12.04 LTS. Так как выпуск отнесён к категории версий с длительным временем поддержки (3 года для десктоп-редакции и 5 лет для серверной), ответвление пакетной базы будет произведено не из Debian Unstable, а из Debian Testing, на основе которого формируется релиз Debian 7 Wheezy". По сравнению с не LTS выпусками релиз 12.04 будет консервативным и не будет содержать каких-то прорывных новшеств. Основные усилия будут направлены на доработку и стабилизацию текущих достижений и возможностей. LTS-выпуск позиционируется как подходящий для использования в серверных системах, требующих особой надёжности (mission-critical), а также для развёртывания больших парков настольных систем. Подготовка Ubuntu 12.04 LTS будет отличаться выделением большего времени на выявление и исправление ошибок, а также непрерывным отслеживанием параметров дистрибутива, относящихся к производительности и стабильности. Одним из ключевых направлений развития Ubuntu 12.04 является подготовка лучшего на рынке решения для обеспечения работы облачных инфраструктур на базе OpenStack и создания гостевых систем для различных провайдеров облачных окружений, таких как Rackspace и Amazon. Большое внимание будет уделено упрощению выполнения операций по развёртыванию и управлению сервисами в публичных, гибридных или приватных облачных инфраструктурах. Появившийся в Ubuntu 11.10 фреймворк Juju позволяет выполнять операции по развёртыванию облачных сервисов, используя привычную семантику пакетного менеджера APT. Постоянно растущая коллекция типовых сервисов, поддерживаемых в Juju, позволяет сразу использовать готовые решения для развёртывания различных программ в облачной инфраструктуре, в обычном серверном парке, не использующем виртуализацию, или на рабочей машине разработчика, использующего LXC-контейнеры для тестирования решений перед их развёртыванием в облаке. Большое внимание будет уделено развитию серверной сборки Ubuntu 12.04, которая развивается в лучших традициях Debian, но имеет ряд преимуществ. Например, используемое в серверной сборке ядро Linux позволяет использовать дополнительные механизмы обеспечения безопасности, такие как AppArmor, ограничение использования отладочного вызова PTRACE и защита от атак, связанных с установкой символических и жёстких ссылок. Большое значение для предприятий также имеет проводимая Canonical программа сертификации оборудования, услуги поддержки и предоставление гарантий, при сохранении свойственной Debian точности и качеству. Кроме того, Ubuntu 12.04 станет первым LTS-выпуском с поддержкой серверных сборок для систем на базе архитектуры ARM. Canonical связывает большие надежды с серверными системами на базе архитектуры ARM, которые могут стать заделом для нового поколения оборудования для датацентров, нацеленного на высокую энергоэффективность. По сравнению с прошлым LTS-выпуском десктоп-редакция дистрибутива претерпела множество заметных изменений: произведена миграция на технологии GNOME 3, внедрена пользовательская оболочка Unity, интегрирован собственный набор шрифтов. В Ubuntu 12.04 ожидается улучшение средств для обеспечения управления и обслуживания большого парка рабочих станций, подготовка инструментов для прозрачного обновления систем на базе Ubuntu 10.04 LTS, улучшение поддержки многомониторных конфигураций. Общие же принципы организации десктоп-окружения, которые можно наблюдать в Ubuntu 11.10, не изменятся. Работа будет сосредоточена в основном вокруг повышения юзабилити, оттачивания интерфейса, увеличения производительности и повышения предсказуемости. Кроме того, намечена реализация некоторых улучшений для продвинутых пользователей.
