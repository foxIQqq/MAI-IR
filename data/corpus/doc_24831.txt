KahelOS - новый дистрибутив на базе Arch Linux

Представлен новый Linux дистрибутив KahelOS, основанный на разработках проекта Arch Linux с десктоп-окружением GNOME и подборкой пользовательских приложений на базе GTK+. Размер установочного образа 700 Мб (i686 и x86-64). Дистрибутив проповедует модель непрерывных обновлений (Rolling-release), при которой обновления пакетов выходят постоянно и пользователь в любой момент имеет возможность перейти на последние версии программ, без ожидания их оформления в виде релиза дистрибутива. В дистрибутиве использовано ядро 2.6.32, X.Org 7.5, GNOME 2.28.2, в качестве web-браузера по умолчанию используется Firefox, для мгновенного обмена сообщениями и для осуществления VoIP звонков задействован Empathy, в качестве офисного пакета в поставку включен OpenOffice.org, для управления фотографиями по умолчанию устанавливается F-Spot, а для просмотра мультимедиа - Beagle. Для управления пакетами в KahelOS используется пакетный менеджер Pacman с напоминающей Synaptics графической оболочкой gtkpacman. Дополнительно поддерживается система установки программ из исходных текстов Yaourt, напоминающая инфраструктуру портов FreeBSD. KahelOS является первым дистрибутивом в состав которого по умолчанию интегрирован Gnome-Color-Manager. Из отличительных черт дистрибутива называется: Упрощенный процесс установки; Очень быстрая загрузка; Интеграция системы управления видеорежимами на уровне ядра (KMS); Использование только свободных программ (100% GNU), в комплект не включены плагины gstreamer-bad/ugly/ffmpeg и adobe, предпочтение отдается открытым стандартам (theora/ogg); Задействование по умолчанию графических 3D эффектов на рабочем столе (вместо Compiz используется Metacity Compositor); Для настройки сети и создания VPN задействован NetworkManager.
