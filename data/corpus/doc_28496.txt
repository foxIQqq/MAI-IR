Вышла KDE-редакция Linux-дистрибутива Salix 13.1

Представлен первый официальный релиз дополнительной редакции Linux-дистрибутива Salix 13.1.2, базирующийся на десктоп-окружении KDE 4.4.3. Дистрибутив на 100% совместим со Slackware Linux 13.1 и исповедует подход "одно приложение на одну задачу". Salix развивается группой разработчиков, покинувших проект Zenwalk, в результате конфликта с другими разработчиками и создателем и лидером проекта Жаном-Филиппом Гийемином (Jean-Philippe Guillemin), и остаивают политику максимальной похожести на Slackware. Размер загрузочного iso-образа составляет 700 Мб. В состав дистрибутива включены такие программы, как web-браузер Konqueror, почтовый клиент KMail, torrent-клиент Ktorrent, RSS-агрегатор Akregator, программа для мгновенного обмена сообщениями Kopete, программа для просмотра фотографий Gwenview, система настройки сетевых подключений Wicd, GUI-интерфейс для управления пакетами Gslapt. Из мультимедийных приложений отмечается наличие медиаплеера Kaffeine, программы для управления музыкальной коллекцией Clementine, системы записи CD/DVD K3b. Для упрощения установки медиа-кодеков в состав дистрибутива включено специализированное приложение. Для выполнения офисных задач предлагается использовать Koffice, KOrganizer, KAddressBook и Okular. Дистрибутив поддерживает три режима установки: Полный - установка всех доступных приложений. Основной - установка только базового состава KDE, web-браузера и системных конфигураторов; Базовый - только минимальный набор консольных утилит, подходит для создания серверов.
