Проект GNOME ввёл в строй каталог дополнений для GNOME Shell

Проект GNOME анонсировал начало тестирования нового ресурса extensions.gnome.org, на котором ведётся работа по формированию каталога дополнений к оболочке GNOME Shell. Используя представленные в каталоге дополнения пользователи могут изменить и дополнить базовое пользовательское окружение GNOME 3. Установка дополнений производится непосредственно с сайта в один клик, но для этого требуется установка специального плагина для web-браузера. Дополнения разрабатываются на языке JavaScript с использованием CSS-стилей, что решает проблему с переносимостью между разными платформами. По задумке разработчиков, новый ресурс поможет пользователям обустроить рабочее окружение в зависимости от личных предпочтений. Например, привыкшие к классическому GNOME 2 пользователи, которым не по душе кардинальные изменения GNOME 3, смогут через установку нескольких дополнений вернуть окружению более привычный вид. В настоящее время в каталоге представлено около 20 дополнений, среди которых меню приложений в стиле GNOME 2, альтернативная строка статуса, индикатор заряда батареи, апплет управления монитором, апплет для мониторинга нагрузки, диалог завершения работы, модуль интеграции с Gajim, индикатор рабочих мест, апплет для управления внешними накопителями, Dock-панель, а также различные варианты меню и панелей. Поддержка непосредственной установки дополнений с сайта extensions.gnome.org доступна через браузерный плагин "GNOME Shell Integration", появившийся начиная с версии GNOME 3.2. Для пользователей более старых выпусков GNOME или других систем отображается только ссылка на код дополнения. Из-за наличия нескольких ошибок, использование плагина с браузерами на основе WebKit затруднено, поэтому разработчики советуют пока использовать плагин только в Firefox. В версии GNOME 3.4 проблемы будут решены и спектр поддерживаемых браузеров будет расширен.
