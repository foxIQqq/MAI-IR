Представлен релиз системы для сборки Linux окружений - rBuilder 5

Компания rPath представила новую версию продукта rBuilder 5.0 и начала тестирование его online-аналога, представляющего собой online-сервис для формирования силами сообщества требуемых Linux окружений и их открытой публикации в сети. Начиная с данного выпуска доступна бесплатная упрощенная версия rBuilder, поддерживающая подготовку окружений на базе ОС rPath Linux, CentOS и Ubuntu, которые могут быть автоматически развернуты в Cloud-инфраструктуре Amazon EC2. Бесплатная версия имеет ограничение на одновременное поддержание работоспособности до 20 таких окружений (настройка и обновление производятся централизовано). Коммерческая версия отличается поддержкой SUSE Linux, возможностью автоматического развертывания окружений на локальных виртуальных серверах и поддержкой создания реплицированных серверов. Основное назначение rBuilder - формирование максимально урезанных программных окружений (Jeos - Just Enough Operating System), предназначенных для работы одного или нескольких приложений на виртуальных серверах или cloud-окружениях. При этом, производитель ПО может сразу создать обособленный и самодостаточный системный образ, дополненный удобным web-интерфейсом для выполнения типовых операций по его обслуживанию, не требующий квалифицированного персонала для его развертывания. В rBuilder 5 можно отметить следующие улучшения: Управляющая консоль, через которую пользователя могут размещать, запускать и останавливать созданные рабочие окружения. Размещение поддерживается для платформ VMware ESX, Citrix Xen, Microsoft Hyper-V, Amazon EC2, BlueLock, Globus, GoGrid и Rackspace. Открытый API позволяет интегрировать возможности управляющей консоли в уже используемые на предприятии инструменты для управления. Новые возможности Web-GUI интерфейса, который отныне решает многие из задач требующих в прошлой версии написания скриптов или работы с утилитами из командной строки. Например, поддерживается определение конфигурации и управление отдельным пакетом. Расширенный API для интеграции с другими продуктами.
