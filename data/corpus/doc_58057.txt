Релиз десктоп-окружения Trinity R14.0.13, продолжающего развитие KDE 3.5

Опубликован релиз десктоп-окружения Trinity R14.0.13, продолжающего развитие кодовой базы KDE 3.5.x и Qt 3. Бинарные пакеты в ближайшее время будут подготовлены для Ubuntu, Debian, RHEL/CentOS, Fedora, openSUSE и других дистрибутивов. Из особенностей Trinity можно отметить собственные средства для управления параметрами экрана, основанная на udev прослойка для работы с оборудованием, новый интерфейс для настройки оборудования, переход на композитный менеджер Compton-TDE (форк Compton с расширениями TDE), улучшенный конфигуратор сети и механизмы аутентификации пользователей. Окружение Trinity может быть установлено и использовано одновременно с более актуальными выпусками KDE, в том числе предоставлена возможность использования в Trinity уже установленных в системе KDE-приложений. Также присутствуют средства для корректного отображения интерфейса GTK-программ без нарушения единого стиля оформления. Среди изменений: Добавлен новый обработчик tdeio-slave "appinfo:/" (tdeio-appinfo), выводящий сведения о файлах конфигурации, каталогах с данными, руководствах пользователя и временных файлах, связанных с указанным приложением. Добавлен набор twin-style-machbunt со стилем декорирования окон, напоминающим тему оформления KDE из SUSE 9.1/9.2. В приложениях Konsole, Kate, KWrite, TDevelop и различных программах, использующих компонент редактирования на базе Kate, обеспечена поддержка изменения размера шрифта через вращения колеса мыши, удерживая клавишу Ctrl. В текстовом редакторе Kate реализована подсветка синтаксиса для файлов с разметкой Markdown. Улучшен интерфейс для настройки обоев рабочего стола. В браузере/файловом менеджере Konqueror в контекстном меню Action появилась возможность выбора режима размещения текущего изображения в качестве обоев рабочего стола. В панели задач реализована возможность применения операций из меню Move Task Button и интерфейса drag&drop для перемещения сгруппированных кнопок. В разделе настройки подстановок ввода (Input Actions) предложено новое действие для вставки задержки между операциями, добавлены кнопки для перемещения строки вверх или вниз, налажен интерфейс для создания и редактирования действий. Добавлен новый tdeio-slave обработчик для протокола SFTP, основанный на использовании libssh. Добавлена поддержка FFmpeg 5.0, Jasper 3.x и Poppler >= 22.04. Улучшена поддержка Python3. Добавлены man-руководства для приложений abakus, amarok, arts, k3b, k9copy, kile, koffice, krecipes, ktorrent, libksquirrel, rosegarden, tellico, tdeaddons, tdeartwork, tdebase, tdebindings, tdegraphics, tdemultimedia, tdenetwork, tdesdk и tdeutils. В документации улучшено оформление вызовов API. Устранены уязвимости в tdeio-slave модуле для FISH (CVE-2020-12755) и KMail (атака EFAIL). Решены проблемы с открытием файлов через URL media:/ и system:/media/ из приложений, не входящих в состав TDE. Обеспечена совместимость с OpenSSL 3.0. Улучшена поддержка Gentoo. Добавлена поддержка Ubuntu 22.10, Fedora 36/37, openSUSE 15.4, сборок Arch Linux для архитектур arm64 и armhf. Прекращена поддержка Ubuntu 20.10.
