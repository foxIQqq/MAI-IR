Компания Novell подвела итоги конкурса по разработке сборок на базе SUSE Studio

Компания Novell анонсировала победителей конкурса по созданию наиболее инновационного и интересного самодостаточного программного окружения, подготовленного с использованием сервиса SUSE Studio. Победители, которым вручен приз, размером 10 тыс. долларов: Лучший коммерческий проект: Illumination Software Creation Station - окружение на базе openSUSE для визуального создания десктоп и web-приложений для людей без навыков программирования; Лучший комьюнити-проект: Hypergrid to Go - образ виртуальной машины с преднастроенной конфигурацией для формирования многопользовательских 3D-миров, наподобие Second Life, и их связывания с OpenSim-серверами; Второе место заняли: EasySpooler Appliance - образ виртуальной машины для создания корпоративного сервера печати; BrowserBox - сборка для тестирования разработок в различных web-браузерах. В комплект включено 22 версии 13 различных браузеров. Отдельно отмечен проект FreeSWITCHBOX, позволяющий быстро создавать VoIP-системы на базе ПО FreeSWITCH. SUSE Studio предоставляет разработчикам набор инструментов для формирования сконфигурированной начинки виртуальных серверов и подготовки установочных образов (Software Appliance). За год существования online-сервиса SUSE Studio в нем было зарегистрировано более 80 тыс. пользователей и сформировано более 400 тысяч виртуальных окружений, которые были загружены более 3 миллионов раз. Созданные окружения можно опубликовать в каталоге SUSE Gallery. Пользователю достаточно загрузить образ системы из каталога и получить полностью работающее и сконфигурированное решение, требующее минимальных усилий для внедрения.
