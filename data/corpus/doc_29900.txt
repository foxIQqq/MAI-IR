Российская компания PingWin Software вошла в состав Linux Foundation

Некоммерческая организация Linux Foundation, курирующая широкий спектр работ, связанных с развитием Linux, анонсировала вступление в свои ряды компании PingWin Software. В составе Linux Foundation компания PingWin Software намерена принять участие в развитии программы Open Compliance Program, развивающей инструменты для отслеживания возможных лицензионных конфликтов и разрабатывающей спецификации для обмена информацией о лицензиях. Кроме того, PingWin Software намерена использовать возможности Linux Foundation для развития более тесного сотрудничества, обсуждения бизнес-стратегии и технических вопросов с представителями Linux-сообщества по всему миру. Компания PingWin Software (ООО «ПингВин Софтвер») была основана в 2009 году как дочернее предприятие группы АйТи, нацеленное на оказание сервисных услуг по развертыванию, модернизации и поддержке корпоративных информационных систем на основе свободного программного обеспечения (СПО). Дмитрий Комиссаров, руководитель компании, дополнительно входит в состав совета директоров компании Mandriva и курирует развитие отечественного Linux-дистрибутива Rosa. PingWin Software является первой представленной в Linux Foundation российской компанией (в состав LF уже входит компания Parallels, которая несмотря на ведение разработки в России, зарегистрирована в Швейцарии). По словам Дмитрия Комиссарова участие в работе Linux Foundation дает возможность взглянуть изнутри на последние тенденции развития Linux и открытых технологий, а также позволяет напрямую сотрудничать с лидерами, представляющими разные части экосистемы Linux. Участие в программе Open Compliance Program и других инициативах Linux Foundation позволит PingWin Software получить важные знания, которые можно будет применить при организации работы компании и при взаимодействии с клиентами. Аманда Макферсон (Amanda McPherson), вице-президент Linux Foundation, подчеркнула, что подключение к организации PingWin Software является важным шагом на пути к адаптации и продвижения инициативы Open Compliance Program в разных странах, с учетом особенностей их законодательства.
