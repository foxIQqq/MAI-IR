Выпуск рабочего стола Jade 0.6

Доступен выпуск рабочего стола Jade 0.6, написанного с использованием языка Python и web-технологий. Проектом реализуется принципиально иная концепция рабочего стола, предлагающая собственное представление об организации взаимодействия с пользователем и напоминающая работу в браузере. На базе рабочего стола Jade тем же автором развивается дистрибутив Manjaro WebDad, который примечателен системой голосовых уведомлений и обеспечением интеграции со смартфоном (требуется установка мобильного приложения KDE Connect). Пакеты c Jade предложены для Manjaro Linux и Sparky Linux. Код проекта распространяется под лицензией GPLv2. В основе рабочего стола лежит фреймворк Jade Application Kit, позволяющий создавать приложения и компоненты для рабочего стола с использованием HTML5, JavaScript и CSS. Фреймворк написан на Python с использованием GTK3 и браузерного движка Webkit2 (Webkit2Gtk). Из JavaScript-библиотек применяются Materialise, Jquery и Masonry. В текущем виде Jade можно рассматривать скорее как экспериментальный проект. Предлагается базовый набор функций, таких как поиск программ, обзор установленных приложений, выключение системы, просмотр недавно используемых файлов, отображение текущего времени и свободного дискового пространства, интерфейс для настройки параметров системы. Рабочий стол предоставляет гибридное окружение, которое позволяет запускать как обычные настольные приложения (например, LibreOffice), так и web-приложения.
