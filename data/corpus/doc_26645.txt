Доступен релиз системы виртуализации VirtualBox 3.2.0

Компания Oracle выпустила релиз системы виртуализации VirtualBox 3.2.0. Примечательно, что это первый релиз, распространяемый целиком под брендом Oracle, изменилось даже название, официально продукт теперь именуется "Oracle VM VirtualBox". Кроме исправления около 30 ошибок, из которых 7 связаны с поддержкой 3D-акселерации, по сравнению с прошлым стабильным выпуском добавлены следующие новшества: Экспериментальная поддержка MacOS X в качестве гостевой ОС; Динамическое уменьшение или увеличение доступного гостевой ОС размера оперативной памяти (пока работает только на 64-разрядных хост-системах); Page Fusion - технология дедупликации страниц памяти - объединение областей памяти, используемых разными гостевыми окружениями, с одинаковым содержимым. В настоящий момент поддерживаются только Windows-гости на 64-разрядных хост-системах; Горячее подключение и выключение процессоров для Linux-гостей и только подключения для некоторых Windows-гостей; Гипервизор теперь поддерживает большие страницы памяти на 64-битных платформах с поддержкой VT-x/AMD-V для увеличения производительности; В гипервизор добавлена поддержка прямого выполнения инструкций гостевой ОС на процессорах Intel с технологией VT-x (для этого должны быть включена опция nested paging) - это должно ускорить запуск гостевых ОС, использующих для этого защищённый режим; Возможность удалять снэпшоты без остановки работы гостевой ОС; Поддержка настройки многомониторных конфигураций Windows-гостей через GUI-конфигуратор; Улучшена эмуляция USB-планшетов и клавиатур, которые теперь работают без установки дополнений для гостевой ОС (Guest Additions); Добавлена эмуляция SAS-контроллера LsiLogic; Реализовано ускорение видео в RDP-сессии; Добавлена подсистема конфигурации NAT через API и утилиту VBoxManage; Возможность настройки параметров кэширования ввода/вывода на стороне хоста; Дополнения для гостей: добавлена возможность запуска гостевых приложений со стороны хост-системы; Улучшена поддержка стандарта виртуализации жёстких дисков OVF - теперь неподдерживаемые поля описания формата сохраняются.
