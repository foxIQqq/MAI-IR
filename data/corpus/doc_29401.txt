Вышел десктоп-ориентированный дистрибутив Sabayon Linux 5.5

Вышел релиз дистрибутива Sabayon Linux 5.5, сочетающегося в себе удобство для конечного пользователя с гибкостью Gentoo. Дистрибутив доступен в сборках с GNOME 2.32 (iso 1.5 Гб) и KDE 4.5.5 (iso 2 Гб). Sabayon Linux оформлен в виде Live-дистрибутива, снабженного возможностью легкой установки на жесткий диск. В комплекте поставляется полный набор всех необходимых мультимедиа кодеков и проприетарных драйверов для видеокарт, беспроводных адаптеров и софт-модемов. Пакетная база дистрибутива синхронизирована с коллекцией портажей Gentoo и полностью совместима с Portage 2.2 (EAPI2), пользователь имеет возможность установки программ как из исходных текстов, так и используя готовые бинарные пакеты, собранные для платформ x86 и x86-64. Для управления пакетами подготовлен GUI интерфейс "Sulfur" и консольная программа "equo". Пользователи KDE-редакции имеют возможность перейти на KDE 4.6 через установку штатного обновления. Особенности выпуска Sabayon Linux 5.5: С момента прошлого релиза обновлено более 1000 пакетов, исправлено более 100 ошибок; Дистрибутив поставляется с Linux-ядром 2.6.37, оптимизированным для выполнения десктоп-задач. Дополнительно поставляются пакеты с Linux-ядром, оптимизированным для использования на сервере или для развертывания изолированных контейнеров на базе OpenVZ или Vserver; Быстрая загрузка и легковесная комплектация приложений по умолчанию (для GNOME заявлено требование к 512 Мб ОЗУ, для KDE - 768 Мб); Официальная поддержка файловой системы Btrfs. В качестве файловой системы по умолчанию используется Ext4, из коробки поддерживается шифрование ФС; Обновленный графический стек: mesa-7.10 + xorg-server-1.9 + KMS-модули ядра 2.6.37. В комплекте поставляется набор проприетарных и открытых (Gallium3d) драйверов для карт NVIDIA и ATI/AMD; В GNOME-редакции предусмотрена возможность быстрой активации оболочки GNOME Shell, которая разрабатывается в рамках проекта GNOME 3.0; Включение 3D-эффектов на десктопе одним кликом мыши, поддерживаются Compiz, Compiz Fusion и KWin; В состав входит новая версия пакетного менеджера Entropy Framework 1.0_alpha9, обеспечена полная поддержка PackageKit; В качестве офисного пакета поставляется OpenOffice 3.2; Упрощенная система трансформации дистрибутива в медиацентр на базе XBMC; В поставку входит демо-версия популярной 2D-игры World of Goo; Возможность оценки дистрибутива не выходя из Windows, поддерживается запуск Sabayon DVD в эмуляторе QEMU; Инсталлятор базируется на исходных текстах новой ветки Anaconda, используемой для установки Fedora Linux. В инсталляторе присутствует возможность шифрования дисковых разделов, разбивки программных RAID, работы с LVM, установки языковых пакетов; Обеспечена поддержка установки, не выходя из Windows;
