Релиз звукового редактора Audacity 2.2.0

После двух с половиной лет разработки доступен релиз свободного редактора звука Audacity 2.2, предоставляющего средства для редактирования звуковых файлов (Ogg Vorbis, FLAC, MP3 и WAV), записи и оцифровки звука, изменения параметров звукового файла, наложения треков и применения эффектов (например, подавление шума, изменение темпа и тона). Код Audacity распространяется под лицензией GPL, бинарные сборки доступны для Linux, Windows и macOS. Основные новшества: Предложены четыре базовые темы оформления (классическая, лёгкая, тёмная и контрастная), которые можно выборать через меню Edit/ Preferences/ Interface. Также расширены средства для изменения параметров оформления с учётом предпочтений пользователей; Добавлена возможность воспроизведения файлов MIDI и Allegro. Для работы в Linux требуется установка TiMidity++, freepats и pmidi в качестве дополнительных зависимостей; Переработана структура меню. Использованы более короткие и чёткие наименования, упрощена работа с меню (без потери функциональности), наиболее востребованные возможности вынесены на первый уровень, чётко структурированы подменю. Добавлены отдельные элементы меню для экспорта в MP3 и WAV. Добавлены две расширенные секции Ext-Bar и Ext-Command, скрытые по умолчанию (View / Extra Menus on/off), в которые вынесены редко используемые или интересные узкой категории пользователей команды; Изменено поведение при записи звука - отныне по умолчанию при повторном нажатии кнопки записи, осуществляется добавление в конец текущего трека, а не создание нового трека (для создания нового трека необходимо удерживать клавишу Shift); Многие элементы интерфейса и диалоги снабжены кнопкой "?" для вызова справки; Добавлены четыре дополнительные опции для определения позиции при выделении части трека; Появилась возможность изменения цвета спектрограммы.
