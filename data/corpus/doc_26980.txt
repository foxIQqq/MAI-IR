Анонсирован проект PGXN, аналог CPAN для PostgreSQL

В рамках проекта PGXN (PostgreSQL Extension Network) создан напоминающий CPAN архив для распространения дополнений к СУБД PostgreSQL. В настоящее время проект находится на начальной стадии развития, но в будущем планируется реализация инфраструктуры для загрузки и распространения дополнений, подготовка централизованного индекса и API для доступа к мета-данным, создание системы поиска дополнений и набора утилит для загрузки, тестирования и установки дополнений из командной строки. Кроме того, можно отметить выпуск обновленного варианта пакета с PostgreSQL 8.4.4 для CentOS 5.5, в котором устранена досадная оплошность, связанная с включением в первой сборке пакета отладочных опций, приводящих к ощутимому замедлению работы.
