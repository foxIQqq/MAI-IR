Корректирующий выпуск Python 3.4.2

Доступен корректирующий выпуск языка программирования Python 3.4.2, в котором представлено 139 исправлений. Из изменений отмечается начало поставки установщика для OS X в форме пакета с цифровой подписью, совместимого с OS X Gatekeeper. В сборках для Windows обновлены поставляемые в комплекте компоненты OpenSSL, которые ранее содержали уязвимость CVE-2014-0224. Устранены потенциальные проблемы с безопасностью в CGIHTTPServer, связанные с неверным экранированием путей при проверке наличия CGI-скриптов.
