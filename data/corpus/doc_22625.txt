Знакомство с универсальным SCSI-драйвером в Linux

В статье представлены некоторые команды SCSI и методы их выполнения с помощью имеющегося в Linux SCSI API. Даны начальные сведения о клиент-серверной модели SCSI и командах SCSI для работы с устройствами хранения данных. Также рассказывается об API универсального (generic) SCSI-драйвера Linux и приводится для изучения пример системы, выполняющей с помощью этого драйвера SCSI-команду inquiry.
