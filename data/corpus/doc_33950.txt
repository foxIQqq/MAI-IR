Для файлового менеджера KDE представлена поддержка облачного сервиса Ubuntu One

Для файлового менеджера Dolphin подготовлен плагин c реализацией поддержки облачного сервиса Ubuntu One. Плагин добавляет сходную с Nautilus функциональность, например, позволяет организовать синхронизацию содержимого директорий и публикацию ссылок в хранилище Ubuntu One. Из нереализованных возможностей Nautilus указывается на отсутствие отображения в общем списке, какие из файлов и директорий синхронизированы в Ubuntu One. В качестве причины создания плагина отмечается наличие проблем в ранее представленной и не развивающейся реализации KDE-клиента для Ubuntu One. Для решения этих проблем решено было реализовать плагин к Dolphin на базе стандартного клиента Ubuntu One для GNOME. Для установки плагина достаточно распаковать предоставляемый tar-архив в директорию "/home/пользователь/.kde/share/kde4/services/ServiceMenus/".
