Вышел дистрибутив Calculate Linux 14.12

Доступен релиз отечественного дистрибутива Calculate Linux 14.12, построенного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в шести редакциях, собранных для архитектур i686 и x86_64: Calculate Directory Server (CDS), Calculate Linux Desktop с рабочим столом KDE (CLD) и XFCE (CLDX), Calculate Media Center (CMC), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS). Все версии дистрибутива распространяются в виде загрузочного Live-образа с возможностью установки на жесткий диcк или USB-накопитель. Для обновления с выпуска Calculate Linux 14 следует выполнить cl-update, загрузить новый образ в директорию /var/calculate/linux и запустить cl-install. Основные изменения: Оптимизирована утилита обновления системы: Ускорена работа проверки обновлений в фоновом режиме. Обновление более не выполняется отдельными этапами для утилит, библиотек perl/python и остальных пакетов. Если в настраиваемом файле был изменён только заголовок, изменения будут автоматически приняты; Добавлена поддержка чистки устаревших версий загруженных пакетов программ. Добавлена поддержка установки в качестве гостевой системы KVM на Virtio-диски. В утилите смены профиля cl-update-profile добавлен выбор способа обновления состава пакетов системы, добавлено обновление настроек системы при смене профиля. В Calculate Console обновлены иконки и упорядочено размещение служб. В CLD и CLDX включены обои нескольких последних версий. Добавлена утилита настройки автовхода и шифрования директории пользователя. Улучшена работа тачпада. Обновлена база часовых поясов. Изменения в составе ПО Из состава CLDX исключён RSS-агрегатор новостей Liferea в пользу плагина RSS в Claws-Mail. Просмотрщик PDF-документов Evince в CLDX заменён на QpdfView. Просмотрщик изображений Eog заменён на Nomacs. В CLDX добавлена утилита сведений о системе и пакет тестирования производительности HardInfo. Исключён из образа CLDX менеджер фотографий Shotwell. В образ CLS включены библиотеки FFmpeg. Для сокращения размера образа, из состава CLS удален пакет обоев. Утилиты управления частотой процессора cpufrequtils заменёны на cpupower. Исправления ошибок Исправлена настройка разрешения framebuffer. Исправлено появление двух иконок обновления в трее в CLD. Исправлена смена темы иконок в CLDX. Исправлена проблема с исчезновением разделов в главном меню при установке программ. Исправлено получение профиля пользователя в доменах с репликацией. Проверка орфографии на русском, теперь корректно воспринимает слова с буквой "ё". Исправлена тема plymouth, некорректно работающая на некоторых моделях Nvidia при использовании драйвера Nouveau.
