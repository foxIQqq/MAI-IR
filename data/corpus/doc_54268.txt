Пятнадцатое обновление прошивки Ubuntu Touch

Проект UBports, который взял в свои руки разработку мобильной платформы Ubuntu Touch, после того как от неё отстранилась компания Canonical, опубликовал обновление прошивки OTA-15 (over-the-air). Проектом также развивается экспериментальный порт рабочего стола Unity 8, который переименован в Lomiri. Обновление Ubuntu Touch OTA-15 сформировано для смартфонов OnePlus One, Fairphone 2, Nexus 4, Nexus 5, Nexus 7 2013, Meizu MX4/PRO 5, VollaPhone, Bq Aquaris E5/E4.5/M10, Sony Xperia X/XZ, OnePlus 3/3T, Xiaomi Redmi 4X, Huawei Nexus 6P и Sony Xperia Z4 Tablet, а также по сравнению с прошлым выпуском началось формирование стабильных сборок для устройств Google Pixel 3a, OnePlus Two, F(x)tec Pro1/Pro1 X, Xiaomi Redmi Note 7 и Samsung Galaxy Note 4. Отдельно, без метки "OTA-15", будут подготовлены обновления для устройств Pine64 PinePhone и PineTab. Выпуск сформирован на основе Ubuntu 16.04 (сборка OTA-3 была основана на Ubuntu 15.04, а начиная с OTA-4 осуществлён переход на Ubuntu 16.04). Ожидавшийся в этом выпуске переход с Qt 5.9 на 5.12 отложен до OTA-16, после чего начнётся работа по обновлению до компонентов Ubuntu 20.04. Начиная с выпуска OTA-16 также будет прекращена поддержка устаревшего web-движка Oxide (на базе QtQuick WebView, не обновлялся с 2017 года), на смену которому уже давно пришёл движок на базе QtWebEngine, на который переведены все базовые приложения Ubuntu Touch. В новой версии: Продолжена работа по улучшению поддержки устройств (портирование драйверов и исправление ошибок), поставляемых с платформой Android 9. Настройки ядра изменены для повышения качества звука. Решены проблемы с настройкой телефонного стека oFono, связанной с автонастройкой APN для передачи данных через сотового оператора. Налажена отправка кодов USSD, используемых для проверки состояния тарифа и обращения к сервисам оператора связи. В браузере Morph полностью переработан интерфейс переключения вкладок, в котором стало более удобно убирать вкладки. Добавлена возможность смены вкладок через скользящее движение снизу вверх по экрану. Решены проблемы с предпросмотром вкладок. Изменён интерфейс со специфичными для доменов настойками, в которых вверх вынесены недавно используемые домены. Добавлена поддержка обращения из JavaScript к буферу обмена Ubuntu Touch. Реализован обработчик ошибок, связанных с MMS, и добавлено уведомеление о сбоях при приёме MMS. Через Bluetooth-гарнитуру налажена возможность совершения звонков и работа функции повторного звонка по последнему набранному номеру. В сборке для устройств на базе архитектуры arm64 решена проблема с отображением в списке пропущенных вызовов цифровых номеров вместо имён из адресной книги. Улучшена тёмная тема оформления.
