Первый релиз мобильной версии Firefox

После двух лет разработки увидел свет первый релиз web-браузера Firefox для мобильных устройств (проект Fennec) на базе платформы Maemo (планшетные ПК Nokia N900 и N8x0). Параллельно ведется разработка версии Firefox для платформ Windows Mobile, Android и Symbian, но код для них еще не стабилизирован и находится на разных стадиях тестирования. Для оценки работы мобильной версии Firefox на стационарных десктоп-системах подготовлены специальные сборки для Linux, MacOS X и Windows. Для мобильной версии Firefox в настоящий момент уже создано около 60 дополнений, среди которых можно найти: Adblock Plus для блокирования рекламы, NoScript для выборочного запрещения выполнения JavaScript и Flash роликов, Weave Sync для синхронизации настроек, закладок, паролей и истории работы с настольной версией браузера. Так как в мобильной версии Firefox по умолчанию отключен плагин Flash, потому что он создает излишнюю нагрузку на систему, для просмотра роликов на YouTube рекомендуется установить дополнение YouTube Enabler. Основные особенности мобильной версии Firefox: Инновационный интерфейс пользователя, не похожий на другие браузеры для мобильных устройств. Вместо привычной строки управления и списка табов, в Fennec реализован режим одновременного просмотра эскизов содержимого табов в боковой панели, динамически отображаемой в правой части экрана, переход на который осуществляется путем скроллинга влево за границы активного окна. Также через скроллинг осуществляется отображение боковой панели навигации. Просмотр страниц производится в полноэкранном режиме, все дополнительные элементы, включая строку ввода адреса, отображаются адаптивно. Прокрутка производится в iPhone-подобном инертном виде, когда незначительное касание приводит к прокрутке с постепенным замедлением. Отличная производительность (задействован высокопроизводительный движок TraceMonkey) и низкое потребление памяти, в сочетании с поддержкой всех web-стандартов на уровне полнофункциональной версии Firefox 3.6 (CSS2.1, CSS3, HTML5, тег video, SVG-фильтры, нативный JSON, MathML, APNG, межсайтовый XMLHttpRequest, Web Worker Threads). Дополнительно проведены оптимизации, нацеленные на увеличение скорости работы на архитектуре ARM; Интеллектуальная строка ввода адреса с возможностью автодополнения URL по ключевым словам или формирования запроса к поисковым системам. Поддержка закладок и тегов (назначение закладкам ключевых слов); Встроенная поддержка блокирования всплывающих окон; Поддержка проверки орфографии в web-формах; Система оповещения о наличии обновлений, устанавливаемых в автоматизированном режиме после согласия пользователя; Гибкие возможность навигации по документу, автоскрытие управляющих панелей, удобный инструмент для изменения масштаба; Менеджер хранения пользовательских паролей, система блокировки всплывающих окон, кнопка быстрой очистки всех приватных данных (история посещений, cookie и т.п.); Возможность инициирования телефонного звонка при нажатии на телефонный номер на web-странице, поддержка API для определения геопозиции; Менеджер загрузок с поддержкой докачки после прерывания сессии и панель настройки параметров браузера. Поддержка плагинов и дополнений. Например, можно поставить Adobe Flash или около 60 дополнений, адаптированных для мобильной версии Firefox.
