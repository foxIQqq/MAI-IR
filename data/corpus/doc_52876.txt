Обновление дистрибутива Elementary OS 5.1.4

Представлен выпуск дистрибутива Elementary OS 5.1.4, позиционируемого в качестве быстрой, открытой и уважающей конфиденциальность альтернативы Windows и macOS. Основное внимание в проекте уделяется качественному дизайну, нацеленному на создание простой в использовании системы, потребляющей минимальные ресурсы и обеспечивающей высокую скорость запуска. Пользователям предлагается собственное окружение рабочего стола Pantheon. Для загрузки подготовлены загрузочные iso-образы (1.48 Гб), доступные для архитектуры amd64 (при загрузке с сайта, для бесплатной загрузки в поле с суммой пожертвования необходимо ввести 0). При разработке оригинальных компонентов Еlementary OS используется GTK3, язык Vala и собственный фреймворк Granite. В качестве основы дистрибутива используются наработки проекта Ubuntu. На уровне пакетов и поддержки репозиториев Еlementary OS 5.1.x совместим с Ubuntu 18.04. Графическое окружение основано на собственной оболочке Pantheon, которая объединяет собой такие компоненты, как оконный менеджер Gala (на базе LibMutter), верхнюю панель WingPanel, ланчер Slingshot, панель управления Switchboard, нижнюю панель задач Plank (переписанный на Vala аналог панели Docky) и менеджер сессий Pantheon Greeter (на основе LightDM). В состав окружения входит набор тесно интегрированных в единое окружение приложений, необходимых для решения задач пользователей. Среди приложений большую часть составляют собственные разработки проекта, такие как эмулятор терминала Pantheon Terminal, файловый менеджер Pantheon Files, текстовый редактор Code и музыкальный проигрыватель Music (Noise). Проектом также развиваются менеджер фотографий Pantheon Photos (ответвление от Shotwell) и почтовый клиент Pantheon Mail (ответвление от Geary). Ключевые новшества: Инструменты родительского контроля переименованы из "Parental Controls" в "Screen Time & Limits" и расширены возможностью определения правил, связанных с ограничением времени активности экрана, доступом в интернет и использованием приложений. Подобные правила теперь можно задавать для собственной учётной записи, например, для самоорганизации, чтобы слишком долго не засиживаться перед компьютером. Меню приложений оптимизировано для повышения удобства при управлении с сенсорных экранов, а также для снижения задержек и обеспечения плавности при навигации при помощи трекпадов. Приближен к классическому меню режим просмотра категорий приложений, который теперь вместо сетки представлен в виде прокручиваемого списка. Улучшено управление при помощи клавиатуры и повышена производительность. Полностью переработана система поиска настроек, которая приближена к реализации поиска в меню приложений, может использоваться для поиска отдельных настроек и показывает путь к каждому найденному параметру. В настройках рабочего стола наглядно иллюстрирован предоставленный на выбор размер пиктограмм. Решена проблема с дублированием обоев рабочего стола. Расширены настройки, доступные для поиска (размер текста, анимация окон, прозрачность панели). В настройках экрана обеспечено корректное центрирование дисплеев, для которых применён режим поворота экрана. В настройках учётной записи добавлены более точные пояснения причин доступности той или иной настройки только администратору. Запрос подтверждения прав администратора теперь производится непосредственно при выборе привилегированной операции, например, при активации или отключении учётных записей. В центре установки приложений (AppCenter) проведена работа по повышению производительности - проверка наличия обновлений теперь производится не чаще, чем раз в день, при загрузке и входе в систему, а также при каждом запуске AppCenter пользователем. Модернизирован интерфейс управления дополнениями, установленные дополнения теперь показываются только если для них есть обновление. При выборе дополнения на странице с информацией о приложении обеспечен переход на страницу с информацией о дополнении. Упрощена навигация при помощи клавиатуры - фокус ввода теперь устанавливается на поисковой строке и можно сразу использовать клавиши управления курсором для перемещения по результатам поиска. В видеопроигрывателе обеспечено запоминание последнего воспроизводимого видео и последней позиции. В оконном менеджере Gala устранён крах при изменении виртуального рабочего стола и наличии некоторых видов открытых окон. В программе для просмотра фотографий добавлено меню "Open In", упрощающее её использование для предпросмотра перед запуском другого просмотрщика. Обновлена библиотека Granite, в которой реализован новый метод для совместного доступа к настройкам приложений.
