Релиз дистрибутива Scientific Linux 6.1

Анонсирован релиз дистрибутива Scientific Linux 6.1, построенного на пакетной базе Red Hat Enterprise Linux 6.1 и дополненного средствами, ориентированными на использование в научных учреждениях. Дистрибутив поставляется для архитектур i386 и x86_64, для загрузки доступно несколько сборок: установочный DVD (3.5 Гб), полный комплект из двух DVD (4.2 Гб + 400 Мб), LiveCD (GNOME, 699 Мб), LiveMiniCD (IceWM, 389 Мб), LiveDVD (GNOME/KDE/IceWM, 2.2 Гб) и сокращенный образ для установки по сети (163 Мб). Кроме изменений добавленных в RHEL 6.1, из специфичных отличий новой версии от Scientific Linux 6.0 можно отметить реализацию полностью переработанной графической темы "Edge Of Space" и добавление пакета abattis-cantarell-fonts, с набором шрифтов "Cantarell", которые используются в GNOME 3.0 и в новой графической теме Scientific Linux. Из других изменений можно отметить обновление версий пакетов для построения Live-образов (livecd-tools и liveusb-creator), удаление упоминания репозиториев "fastbugs" и "testing" в списке базовых репозиториев (включение данных репозиториев теперь производится через пакет yum-conf-sl-other, так как иначе некоторые пользователи выбирали их в процессе установки, что приводило к проблемам установки обновлений из инсталлятора). Из особенностей работы инсталлятора упоминается появление экрана настройки сети перед стадией выбора типа установки при использовании DVD-образа, данный шаг не является обязательным и может быть отменен пользователем. По сравнению с RHEL 6.x в Scientific Linux 6.x добавлены следующие дополнения: Оконный менеджер IceWM 1.2.37; OpenAFS 1.5.1 - открытая реализация распределенной ФС Andrew File System; Набор развиваемых в рамках проекта Fedora инструментов для создания Live-дисков и производных сборок. Для создания собственного Live-диска можно использовать пакет livecd-tools, для записи образа на USB-накопитель нужно использовать liveusb-creator; Пакет yum-autoupdate для автоматизации ежедневного обновления системы; Поддержка дополнительных репозиториев: adobe, atrpms, elrepo, epel и rpmforge; Добавлено несколько скриптов для изменения принятых по умолчанию настроек: SL_desktop_tweaks (изменение оформления меню GNOME и KDE), SL_enable_serialconsole (включение входа через последовательный порт), SL_no_colorls (выключение цветного вывода для команды ls), SL_password_for_singleuser (включение необходимости ввода root-пароля для входа в однопользовательский режим (single mode)); Пакет визуализации Graphviz; Библиотека Numpy, для организации научных расчетов; Для драйвера iwlwifi включены дополнительные прошивки (firmware для беспроводных карт Intel серий 5150, 3945, 4945, 5000 и, начиная с текущей версии, 6000. Поддержка языка Lua; Поддержка файловой системы XFS для архитектуры x86_64. В программе установки пока нельзя создать XFS раздел, но установить Scientific Linux на ранее созданный XFS раздел можно; FUSE-модули fuse-smb и fuse-sshfs, для монтирования SMB и SSH; Выпущенные одновременно с установочными дисками Live-сборки поддерживают функцию установки на жесткий диск, могут работать без модификации загрузчика с USB-накопителей и поддерживают возможность монтирования через NFS для организации работы бездисковых клиентов. Состав Live-дисков сформирован при добавлении программ из внешних репозиториев EPEL, elrepo.org и atrpms.net, включая пакеты с расширенной поддержкой файловых систем (ntfs, reiserfs), организации безопасных сетевых соединений (openvpn, vpnc, pptp) и восстановления после сбоя (testdisk, dd_rescue, ddrescue, gparted). Начиная с данного выпуска пользователь для работы с LiveCD переименован с "sluser" на "liveuser". Базовые программы: Linux-ядро 2.6.32-131.6.1.el6 xorg-server 1.7.7 icewm 1.2.37 gnome 2.28.6 firefox 3.6.18 thunderbird 3.1.11 openoffice.org 3.2.1 (только на LiveDVD) kde 4.3.4 (только на LiveDVD) Программы, добавленные по сравнением со стандартной поставкой Scientific Linux (по сравнению с прошлым выпуском из поставки удалены пакеты progsreiserfs и testdisk): ntfs-3g (rpmforge) fuse-sshfs (rpmforge) ntfsprogs (rpmforge) dd_rescue (rpmforge) ddrescue (rpmforge) iperf (rpmforge) flash-plugin (rpmforge) rxvt-unicode (rpmforge) gstreamer-ffmpeg (rpmforge) gparted (epel) NetworkManager-openvpn (epel) NetworkManager-vpnc (epel) NetworkManager-pptp (epel) vpnc-consoleuser (epel) kmod-reiserfs (elrepo) kmod-ndiswrapper (elrepo) reiserfs-utils (elrepo) Отдельно стоит напомнить, что большинство ранее доступных в Scientific Linux 5 научных приложений теперь не включены в базовые репозитории Scientific Linux, но их можно найти в дополнительных репозиториях, таких как EPEL, elrepo.org и atrpms.net. Также прекращена поддержка репозитория contrib, в котором размещались дополнительные драйверы устройств - аналоги пакетов из contrib можно найти во внешних репозиториях elrepo и atrpms. Ранее присутствующие в SL5 пакеты, которые можно найти во внешнем репозитории EPEL: Библиотека cfitsio с реализацией поддержки формата FITS (Flexible Image Transport System), широко используемого в астрономических исследованиях, для языков Си и FORTRAN; Технология DKMS (Dynamic Kernel Module Support), позволяющая автоматически перестраивать текущие модули ядра с драйверами устройств после обновления версии ядра. Внедрение DKMS позволит автоматически устанавливать драйверы, которые необходимы для работы определенных аппаратных компонент текущей системы, но отсутствуют в стандартном Linux ядре. Для разработчиков, DKMS предоставляет средства создания драйверов без оглядки на особенности различных версий Linux ядра. Консольный почтовый клиент Alpine; Дополнительные Perl-модули: DBD-XBase, MailTools, Parse-RecDescent, SQL-Statement, TermReadKey, Text-CSV_XS, Text-Template, Tk; Ранее присутствующие в SL5 пакеты, которые можно найти во внешнем репозитории atrpms: Математическая библиотека FFTW (Fastest Fourier Transform); Утилита 915resolution для модификации видео BIOS для графических карт на базе GPU Intel 845G, 855G, 865G, 915G, 915GM и 945G, что позволяет добиться установки дополнительных разрешений экрана; GNU R (язык для статистических расчетов); Дополнительные мультимедиа кодеки (gstreamer-plugins-extras из SL4 и k3b-extras из репозитория livna); Библиотека матричных вычислений SuiteSparse; Ранее присутствующие в SL5 пакеты, которые придется загрузить с сайта разработчика: Научная библиотека SciPy, которую можно использовать для научных, инженерных и математических рассчетов, в также для построения графиков и диаграмм, на основе набора статистических данных; Sun Java/JDK - в составе теперь поставляется только OpenJDK; Набор обучающих приложений для KDE - KDEEDU.
