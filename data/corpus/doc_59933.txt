Выпуск программы для работы с эфемеридами EPM

Состоялся выпуск программы EphEPM 1.1, предназначенной для расчёта барицентрических координат тел Солнечной системы и физической либрации Луны. Код программы написан на языке С++ и распространяется (GitFlic, GitHub) под лицензией GPLv3. Для использования программы необходимы файлы эфемерид (доступны для загрузки на сайте Лаборатории эфемеридной астрономии ИПА РАН). Поддерживаются версии эфемерид с 2015 по 2021 г. (как "обычной", так и "долгой" версий) в формате SPICE (расширения файлов .bsp и .bpc). Графический интерфейс пользователя программы реализован с помощью библиотеки GTK4 (GTKMM-4.0, необходима версия выше 4.10, для использования более ранних версий библиотеки необходимо загрузить EphEPM 1.0.2). Программа может рассчитывать барицентрические координаты тел Солнечной системы с различными параметрами: временные шкалы: UTC, TT, TDB, с учётом часовых поясов; системы координат: экваториальная, эклиптическая; варианты расчёта: координаты, скорость в разложении по осям; экватор и равноденствие: средние (J2000), средние на дату (IAU2000), истинные на дату (IAU2000); единицы измерения для координат: астрономические единицы, километры, метры; единицы измерения для скорости: ае/день, км/день, км/с, м/с; единицы измерения для либрации: радианы, градусы. Также программа может строить схематический график орбит с учётом описанных параметров (масштабы размеров тел и расстояний соблюдены) и с указанием положения тел на заданный момент времени.
