Доступен серверный дистрибутив NethServer 7.7

Представлен релиз дистрибутива NethServer 7.7, предлагающего модульное решение для быстрого развёртывания серверов в небольших офисах или на предприятиях среднего размера. Дистрибутив основан на пакетной базе CentOS 7.7 и предоставляет web-интерфейс для управления доступными серверными компонентами. Размер установочного образа 1.1 Гб. Для ознакомления с возможностями интерфейса предоставляется online-демонстрация. Наработки проекта распространяются под свободными лицензиями. Пользователю предлагаются готовые модули для организации работы почтового сервера (Postfix, Dovecot, Amavis, ClamAV + web-клиент Roundcube), системы совместной работы (SOGo), межсетевого экрана (Shorewall), web-сервера (LAMP), файлового сервера и контроллера домена Active Directory (Samba), фильтрующего прокси (Squid, ClamAV и SquidGuard), VPN-сервера (OpenVPN, L2TP), облачного хранилища (ownCloud), систем обнаружения и предотвращения вторжений. Установка и введение в строй нужного сервиса производится в один клик и не требует знания особенностей настройки каждого серверного компонента. Типовые работы по администрированию можно выполнять через web-интерфейс. Основные новшества: Новый интерфейс пользователя, построенный на базе Cockpit и предлагающий более современное оформление, перешёл на стадию бета-тестирования и включён в поставку по умолчанию. Ранее установленные системы могут протестировать интерфейс установив приложение Server Manager в Software Center. Интерфейс предоставляет инструменты для управления учётными записями, DNS, DHCP, FQDN, установки времени, создания резервных копий, настройки сети, применения TLS-шифрования, управления системой, установки приложений, управления хранилищами и сертификатами SSL; Новый интерфейс для настройки VPN, позволяющий оценить трафик для каждого туннеля, отследить историю подключений каждого пользователя и быстро отправить параметры подключения по email. Через интерфейс также можно определить собственные маршруты, изменить протокол UDP/TCP, активировать или отключить учётную запись; В интерфейсе для управления межсетевым экраном полностью переработаны разделы определения правил блокировки соединений, перенаправления портов, урезания трафика и привязки правил к локальному или внешнему интерфейсу; Новый интерфейс со статистикой блокировки попыток подбора паролей к сервисам в изолированных окружениях (используется Fail2Ban); Новая панель со статистикой работы через Web Proxy; Новая панель файлового сервера, наглядно показывающая состояние совместных каталогов; Новый метод управления резервным копированием и восстановлением, поддерживающий создания нескольких резервных копий; Новая панель для интеграции с NextCloud; Модуль развёртывания инфраструктуры для организации совместной работы на базе платформы WebTop (Web Desktop) обновлён до версии 5.7.3; Новая панель отслеживания состояния источников бесперебойного питания; Дополнительные системные отчёты, включающие выборку данных из логов, СУБД и сервисов; Новая панель со статистикой работы http-сервера Apache; Облачное хранилище Nextcloud обновлено до версии 16.0.5, а платформа для обмена сообщениями между разработчикам Mattermost до выпуска 5.15.
