Доступна бета-версия web-браузера Google Chrome 6

Анонсирована первая бета-версия web-браузера Google Chrome 6, сборки которой доступны для платформ Windows, MacOS X и Linux. Ключевые улучшения: Продолжена работа по увеличению производительности JavaScript-движка V8. По сравнению с выпуском Chrome 5 новая версия проходит тесты SunSpider и V8 benchmark на 15% быстрее. Дополнительно на 64% улучшено прохождение тестов Mozilla Dromeao DOM Core Tests. Реализована новая система интеллектуального автозаполнения форм Auto-fill, которая поддерживает комплексное запоминание и восстановление содержимого сложных форм c учетом типа используемых данных, таких как email, адрес, ФИО, номер кредитной карты и т.п. Например, начиная заполнять анкету на сайте, при начале ввода пользователем своих инициалов браузер сразу определит данные, наиболее релевантные для остальных элементов формы, т.е. заполнит, адрес, телефон, email. Персональные данные запоминаются для последующего использования только после явного согласия пользователя. Расширены возможности по синхронизации данных между машинами, кроме закладок, настроек и визуального оформления синхронизация теперь доступна и для базы автозаполнения полей (номера кредитных карт не синхронизируются). Информация сохраняется в Google-аккаунте пользователя. Проведена оптимизация панели управления, меню сведено в одну унифицированную кнопку.
