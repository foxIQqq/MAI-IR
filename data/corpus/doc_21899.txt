Представлены Qt Mobility, Qt Service Framework, репозиторий Qt для S60 и спецификации N900

Подразделение Qt Software компании Nokia представило новый экспериментальный проект Qt Mobility, цель которого расширить область применения тулкита Qt через разработку нового API для создания приложений, способных работать на широком спектре мобильных платформ, как поддерживаемых Nokia (S40, S60, Maemo), так и от других производителей. Qt Mobility базируется на Qt 4.5 и будет состоять из нескольких семейств API: для управления сервисами (Service Framework), румингом/выходом в сеть (Bearer Management API), контактами (Contacts API), обменом сообщениями (Messaging) и т.д. Первым из компонентов Qt Mobility опубликован Qt Service Framework, определяющий унифицированный способ межпроцессного взаимодействия и органиации доступа к сервисам. Qt Service Framework представляет собой абстрактную прослойку, работающую поверх текущего IPC механизма базовой операционной системы, т.е. COM для Windows, DBUS для Linux и LIW/SAPI для Symbian S60. Дополнительно, спустя несколько недель после открытия публичного Git репозитория библиотеки Qt, доступен репозиторий исходных текстов порта Qt для платформы S60. Кроме того, стали известны новые подробности, касающиеся будущей модели планшетного ПК Nokia N900, сочетающего в себе функции телефона, о котором уже была опубликована новость на прошлой неделе. Спецификация устройства: Размер: 59.7 x 111 x 18.2 мм, что немного толще, но заметно уже и короче Nokia N810. Вес: 180 гр. Батарея 1320 mAh; 3.5-дюймовый сенсорный экран (800×480). Выдвижная клавиатура (заявлено о поддержке русского языка); Процессор OMAP3430 500/600 Mhz; Поддержка GSM частот 850, 900, 1800 и 1900. WCDMA 900, 1700/2100, 2100; Встроенный GPS, акселерометр, WiFi, HSPA, microUSB разъем, Video-выход (!); 5-мегапиксельная камера с LED вспышкой и автофокусом; 256 Мб ОЗУ (+ 768 Мб виртуальной памяти). 32 Гб Flash; Программное обеспечение будет основано на Linux платформе Maemo 5.0, из особенностей которой приводится: Многозадачный интерфейс с поддержкой одновременного размещения нескольких окон на экране; Live Dashboard - позволяет просматривать все открытые задачи и непрочитанные сообщения в едином представлении; Браузер на основе Firefox 3 с поддержкой Flash 9.4; Поддержка автоматического обновления программ; Адресная книга с возможностью установки статусов и синхронизацией с Ovi и Google Talk; Система мгновенного обмена сообщениями, одновременно поддерживающая работу с SMS; Возможность записи видео с разрешением 800×480 в формате AVC/H.264; Поддержка установки тэгов и геотэгов для снятых на встроенную камеру фотографий; Расширение числа виджетов, например, будет доступен виджет для Jaiku/Twitter.
