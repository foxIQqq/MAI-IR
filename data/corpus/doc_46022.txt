Обновление программы для учёта расходных материалов Severcart 0.8.1

Подготовлен релиз Severcart 0.8.1, программы для управления жизненным циклом расходных материалов для офисной оргтехники, в том числе картриджей для лазерных и струйных принтеров различных производителей. Предоставляемый программой интерфейс позволяет планировать закупку расходных материалов, отслеживать оставшиеся ресурсы, вести историю перезаправок, генерировать отчётность, печатать акты передачи на заправку и формировать договоры поставки. Программа написана на Python 3 с использованием фреймворка Django и СУБД PostgreSQL. Код программы распространяется под лицензией GPLv2. Ключевые изменения: В разделе «Обслуживание / Мониторинг» добавлена функция мониторинга используемых наименований расходных материалов, которая реализована в виде отдельных сенсоров, отслеживающих количество пригодных к использованию расходных материалов на складе. Пользователь самостоятельно определяет наименование и пороговый уровень, при котором сенсор примет зелёный или красный цвет. Также, на его усмотрение, есть возможность отображение сенсора на главной странице. Улучшены возможности по работе с комментариями для каждого расходного материала. Теперь каждый объект может хранить перечень текстовых записей, отображая их в хронологическом порядке, с отображением имени пользователя, оставившего комментарий. Существенно доработан отчёт «Справочники / Отчёты / Пользователи». При формировании отчёта приложение предоставляет более подробную информацию по картриджам, используемым пользователем. Для каждого пользователя формируется отдельная таблица с отображением количества оприходованных расходных материалов за определённый период, а также конкретная информация о том, когда и какие картриджи были израсходованы. Предоставлена возможность выгрузки отчёта в отдельный CSV-файл, для последующей работы в табличном процессоре. В раздел «Установленные» добавлена кнопка «Ошибочная передача». Она позволяет программным способом вернуть переданный в пользование объект расходного материала обратно на склад, обновив перечень пригодных к использованию. При этом не происходит искажение статистики при формировании отчётов. Улучшена локализация для русского и английского языков.
