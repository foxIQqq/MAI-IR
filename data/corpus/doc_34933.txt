Компания Red Hat выпустила платформы MRG 2.2 и Network Satellite 5.5

Компания Red Hat представила релиз платформы MRG 2.2 (Messaging, Real Time and Grid) для обеспечения работы задач с гарантированным временем выполнения, организации обмена сообщениями между бизнес приложениями и создания среды распределенных вычислений. Базовые компоненты, составляющие инфраструктуру MRG: "MRG Messaging" - для организации быстрого и надежного обмена сообщениями между бизнес приложениями или web-сервисами. Решение построено на основе открытого протокола AMQP. В новой версии увеличена производительность Java-кода, добавлена поддержка SSL с аутентификацией клиентов по PEM-сертификатам при доступе к инструментам управления, в брокер Apache qpid добавлена опция для защиты от DoS-атак; "MRG Realtime" - выполнение критических к задержкам выполнения задач за счет использования специального Linux ядра с Realtime расширениями, пригодного для установки в дистрибутивах Red Hat Enterprise Linux. В новой версии добавлена поддержка режима High Performance Networking (HPN) для организации прямого доступа к памяти по сети (RocE - Remote Direct Memory Access over Converged Ethernet), ядро Linux обновлено до версии 3.2, добавлена поддержка работы в роли бездискового клиента с загрузкой по сети; "MRG Grid" - создание грид-среды для распределенного решения разноплановых задач. Разработка базируется на открытом проекте Condor. Поддерживается создание сети на основе локальных и удаленных машин, виртуальных и cloud-окружений, на основе задействовании времени простоя процессора на обычных рабочих станциях. В новой версии добавлена поддержка Apache Deltacloud, обеспечена возможность аутентификации через LDAP, представлен новый Condor Resource Agent; Средства для централизованного и унифицированного управления конфигурацией набора машин; High Performance Computing (HPC) - организация высокопроизводительных вычислений, с полной поддержкой разработки параллельных алгоритмов на Java, JMS, .NET, C++, Ruby и Python. Одновременно подготовлен релиз платформы для управления инфраструктурой Linux-серверов - Red Hat Network Satellite 5.5, которая основана на свободном проекте Spacewalk и развивается синхронно с ним. Продукт позволяет организовать инвентаризацию оборудования и программного обеспечения, автоматизировать установку и обновление ПО на группе машин, централизовано управлять файлами конфигурации, осуществлять мониторинг, управлять работой виртуальных машин, автоматизировать установку типовых конфигураций системы, организовать сетевую загрузку и обновление группы машин. Новая версия примечательна обеспечением полной поддержки IPv6 во всех компонентах платформы, а также интеграцией поддержки фреймворка OpenSCAP с поддержкой протокола SCAP (Security Content Automation Protocol) и набором инструментов для автоматической проверки системы на соответствие требованиям безопасности.
