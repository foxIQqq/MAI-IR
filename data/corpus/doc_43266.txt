В Ubuntu 16.04 на смену Ubuntu Software Center придёт менеджер установки приложений GNOME

Разработчики Ubuntu приняли решение отказаться от поставки в Ubuntu 16.04 собственного интерфейса установки приложений Ubuntu Software Center и задействовать вместо него штатный установщик приложений GNOME. Поддержка специфичных для Ubuntu Software Center возможностей, таких как рейтинг, обзоры и платные приложения, будет реализована через плагины. Напомним, что ранее от Ubuntu Software Center уже отказался дистрибутив Ubuntu MATE, мотивировав свой шаг недовольством пользователей в отношении стабильности и нагромождённости Ubuntu Software Center. Кроме того, из базовой поставки будут исключены программа для записи CD/DVD Brasero и клиент для мгновенного обмена сообщениями Empathy. Подобный шаг объясняется потерей актуальности дисков CD/DVD и классических систем обмена сообщениями, которые вытеснили Flash-накопители и мобильные/web-ориентированные чат-сервисы. Из добавленных в поставку новых приложений отмечен GNOME Calendar.
