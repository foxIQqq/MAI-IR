Вышел релиз Linux-дистрибутива Pardus Linux 2009.2

Вышел релиз Pardus 2009.2, турецкого десктоп-ориентированного Linux дистрибутива, интересного тем, что не является ответвлением от других проектов и реализует несколько интересных решений, делающих его привлекательным для пользователей других стран. Например, скрипты инициализации написаны на языке Python, присутствует менеджер истории изменений системы, разработан собственный пакетный менеджер PiSi, своя инфраструктура управления конфигурацией (подробнее см. прошлый обзор). В новой версии обновлены следующие базовые программы: Linux kernel 2.6.31.13 Xorg 1.6.5 KDE 4.4.4 Qt 4.6.2 Firefox 3.6.3 Gimp 2.6.8 OpenOffice.org 3.2.1 Python 2.6.5 В новой версии также убрана зависимость GUI-утилит дистрибутива от библиотек KDE, добавлена утилита plasma-runner-pisi для поиска и установки пакетов из интерфейса KDE. Увеличена производительность пакетного менеджера PiSi и GUI-интерфейса Package Manager, добавлены функции группировки пакетов для обновления. Для загрузки доступны несколько вариантов дистрибутива: Многоязычный установочный дистрибутив (914 Мб) Двуязычный (английский и турецкий) установочный дистрибутив (680 Мб) LiveCD (698 Мб)
