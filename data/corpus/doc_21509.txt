Информация об ожидаемых новшествах релиза FreeBSD 7.2

Релиз операционной системы FreeBSD 7.2 ожидается в начале мая, но уже сейчас можно ознакомится со списком нововведений. Некоторые интересные изменения ожидаемые в релизе: Увеличение адресного пространства памяти доступной ядру до 6 Гб, и kmem map до 3.6 Гб; Изменения в подсистеме jail (возможность назначения множества IP4/IP6 адресов; поддержка протокола SCTP; привязка jail к выделенному процессору с помощью cpuset; возможность запуска jail с указанием использовать для нее отдельную таблицу маршрутизации (FIB); Поддержка Superpages (страницы памяти увеличенного размера с возможностью группировки в Superpages нескольких обычных страниц памяти); Обновленный DRM (Direct Rendering Manager); glabel поддерживает использование ufsid меток для уникальной идентификации файловых систем; gjournal поддерживает журналирование корневой файловой системы; Портирован btpand из NetBSD. Демон поддерживает профили Bluetooth Network Access Point (NAP), Group Ad-hoc Network (GN) и Personal Area Network User (PANU); Официально поддерживаемая в портах версия KDE обновлена с 3.5.10 (x11/kde3) до 4.2.2 (x11/kde4). Добавлена новая версия GNOME 2.26.
