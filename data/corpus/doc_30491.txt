Проект NETSHe выпустил SDK и корректирующий релиз 0.6.1

Проект NETSHe, в рамках которого развивается прошивка и веб-интерфейс для маршрутизаторов на базе OpenWRT, представил NETSHe SDK - базирующуюся на OpenWRT систему сборки дистрибутивов/прошивок для сетевых и встраиваемых устройств. В SDK включены код сборки трех версий OpenWRT (Kamikaze 7.09, Kamikaze 8.09.2 и Backfire 10.03), пакетная база, код сборки пакетов, добавленных STAsoft.net/NETSHe Lab (или модифицированных в рамках проекта), примеры конфигурационных файлов. Дополнительно упоминается выпуск корректирующего релиза NETSHe 0.6.1, в котором представлено исправление нескольких ошибок, в том числе связанных с работой ntpdate и устройствами без часов реального времени. Проведена работа по уменьшению установочного образа для предотвращения ошибок загрузчика на некоторых платах производства Ubiquiti. Основным улучшением ранее выпущенного релиза 0.6.0, является реализация полнофункционального хотспот-контроллера. Для упрощения взаимодействия с заинтересованными разработчиками, проект открыл доступ к SVN-репозиторию (в том числе, к ветке для разработчиков).
