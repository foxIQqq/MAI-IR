Google снял ограничения на распространение четырех инструментариев на базе Eclipse

Компания Google открыла возможность безвозмездного использования четырех продуктов для разработчиков на базе платформы Eclipse, созданных компанией Instantiations, купленной Google в августе. Отныне развитие данных продуктов будет проходить в рамках проекта Google Web Toolkit. Для загрузки представлены четыре пакета: GWT Designer - инструментарий разработчика на базе Eclipse, позволяющий упростить создание web-приложений, активно использующих технику Ajax, на языке Java. Преобразование в представление на базе JavaScript и HTML осуществляется при помощи специального транслятора из комплекта Google Web Toolkit (GWT); CodePro AnalytiX - инструментарий для автоматизированного анализа качества и безопасности кода; WindowBuilder Pro - графический интерфейс для оформления элементов пользовательского интерфейса для фреймворков Swing, SWT, GWT, RCP и XWT; WindowTester Pro - пакет для тестирования взаимодействия GUI-интерфейса в клиентских интерактивных Java-приложениях на основе фреймворков SWT и Swing.
