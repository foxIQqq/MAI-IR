Для Fedora 11 опубликован драйвер для Intel GMA500 (Poulsbo)

Для Fedora 11 опубликован драйвер с работающей поддержкой 3D / compiz для интегрированных видеокарт Intel GMA500 (Poulsbo), поставляемых в устройствах на базе процессора Atom. К сожалению драйвер доступен только в бинарном виде - Intel пока не открыла связанные с Intel GMA500 (Poulsbo) исходные тексты. Пакет для Fedora создан на базе официально поддерживаемого Intel проприетарного драйвера для Ubuntu Netbook Remix, развитие которого решено не продолжать для релиза Ubuntu 9.10.
