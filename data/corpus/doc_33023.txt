RIM откроет код нового Blackberry Native SDK, в котором сделана ставка на Qt

Компания Research in Motion, ранее отдававшая предпочтение проприетарной модели распространения ПО, заявила о решении открыть код инструментария для портирования и разработки мобильных приложения на C/C++ и скриптовых языках (Native SDK) для готовящегося релиза платформы Blackberry 10. Напомним, что WebWorks, базовый фреймворк разработки приложений для Blackberry, базируется на использовании технологий HTML5. В состав SDK входит интегрированная среда разработки на базе Eclipse, компилятор, сборочный инструментарий, отладчик, модули поддержки скриптовых языков, фреймворк для создания игр, мультимедиа-компоненты, набор интерфейсных библиотек для доступа к API платформы Blackberry, симулятор устройств, документация и набор примеров приложений. Отдельно отмечается, что ключевым звеном нового Native SDK является библиотека Qt, которую предлагается использовать для создания пользовательского интерфейса. Среди других открытых проектов, задействованных в SDK, отмечается система симулирования физических процессов Bullet Physics, JavaScript-движок Spidermonkey и язык Lua.
