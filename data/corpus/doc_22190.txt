Компания Red Hat выпустила бета-версию новой платформы виртуализации

Компания Red Hat анонсировала бета-версию новой промышленной виртуализационной платформы Red Hat Enterprise Virtualization, основанной на технологии KVM (Kernel Virtual Machine). Новая инфраструктура виртуализации на базе KVM будет поставляться в составе Red Hat Enterprise Linux начиная с версии 5.4. Одновременно представители Red Hat опубликовали обзор новшеств Fedora 11, связанных с поддержкой виртуализации: улучшенный интерфейс для управления виртуальными окружениями virt-manager; поддержка sVirt, позволяющего использовать SELinux Mandatory Access Control (MAC) политики для гостевых виртуальных окружений; расширение возможностей KVM. Например, в virt-manager появился мастер для быстрого создания виртуальных окружений, улучшена поддержка миграции окружений с одного хоста на другой, добавлен интерфейс для организации эксклюзивного использования физических устройств и ресурсов в виртуальных окружениях, на экране статистики теперь отображаются данные по работе дисков и системы ввода/вывода, улучшены средства аутентификации при подключении к виртуальному окружению через VNC клиент.
