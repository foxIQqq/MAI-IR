KDE SC 4.10 и Amarok 2.7 доступны в сборках для Windows

После отсутствия выпусков на протяжении более чем года представлен пакет с подборкой приложений из состава KDE SC 4.10.2, собранных для установки в ОС Windows. В комплект включены все поставляемые в составе KDE приложения. Для обеспечения работы с мультимедиа информацией задействован бэкенд Phonon-VLC. Кроме обновлений версий компонентов, новый выпуск примечателен устранением специфичных для платформы Windows ошибок в приложениях (например, в kate, kompare и okular), а также частичной поддержкой семантического движка Nepomuk. Для установки программ подготовлен специальный инсталлятор KDEWin Installer, который кроме стандартных KDE-приложений из состава наборов kdebase, kdeedu, kdegames, kdegraphics, kdemultimedia, kdenetwork, kdesdk, kdetoys и kdeutils предоставляет возможность установки последних версий amarok, konversation, kile, quassel и digiKam с набором KIPI-плагинов. Для упрощения установки сходных по своим функциям приложений, подготовлен ряд метапакетов, среди которых: графика, инструменты для разработчиков, игры. Дополнительно можно отметить выход после длительного перерыва Windows-версии музыкального проигрывателя Amarok 2.7. Сборка работоспособна в Windows XP, Vista, Windows 7 и Windows 8. По сравнению с выпуском для Linux в Windows-версии недоступен плагин для интеграции с Nepomuk, нет поддержки воспроизведения звуковых CD, не поддерживается перекодирование форматов и не работают возможности, связанные с Gstreamer (в качестве бэкенда задействован VLC).
