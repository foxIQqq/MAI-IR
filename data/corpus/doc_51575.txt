Бета-выпуск Ubuntu 19.10

Представлен бета-выпуск дистрибутива Ubuntu 19.10 "Eoan Ermine", который ознаменовал переход к первой стадии заморозки пакетной базы и смещение вектора разработки от развития новых возможностей к тестированию и исправлению ошибок. Готовые тестовые образы созданы для Ubuntu Desktop, Ubuntu Server, Lubuntu, Kubuntu, Ubuntu Mate, Ubuntu Budgie, Ubuntu Studio, Xubuntu и UbuntuKylin (редакция для Китая). Релиз Ubuntu 19.10 запланирован на 17 октября. Основные новшества: Рабочий стол GNOME обновлён до выпуска 3.34 c поддержкой группировки пиктограмм приложений в папки и новой панелью выбора обоев рабочего стола. Вместо ранее предлагаемой темы оформления с тёмными заголовками по умолчанию задействована светлая тема оформления, близкая к штатному внешнему виду GNOME. В качестве опции предложена полностью тёмная тема, в которой использован тёмный фон внутри окон; Ядро Linux обновлено до выпуска 5.3. Для сжатия ядра Linux и начального загрузочного образа initramf задействован алгоритм LZ4, что позволит сократить время загрузки за счёт более быстрой распаковки данных; Инструментарий обновлён до glibc 2.30, GCC 8.3 (опционально GCC 9), OpenJDK 11, rustc 1.37, Python 3.7.3, ruby 2.5.5, php 7.2.15, perl 5.28.1, go 1.10.4; Офисный пакет LibreOffice обновлён до выпуска 6.3; Улучшена поддержка кросс-компиляции - инструментарий для архитектур POWER и AArch64 теперь поддерживает кросс-компиляцию для платформ ARM, S390X и RISCV64; Для систем с GPU Intel обеспечен режим загрузки без мерцания при переключении видеорежимов; В состав установочных iso-образов по согласованию с компанией NVIDIA включены пакеты с проприетарными драйверами NVIDIA. Для систем с графическими чипами NVIDIA по умолчанию продолжают предлагаться свободные драйверы "Nouveau", а проприетарные драйверы доступны в качестве опции для быстрой установки после завершения инсталляции; Прекращена поставка deb-пакетов с браузером Chromium, вместо которых теперь предлагаются только самодостаточные образы в формате snap; В репозитории прекращено распространение пакетов для 32-разрядной архитектуры x86. Для запуска 32-разрядных приложений в 64-разрядном окружении будет обеспечена сборка и поставка отдельного набора 32-разрядных пакетов, включающих компоненты, необходимые для продолжения работы устаревших программ, остающихся только в 32-разрядном виде или требующих 32-разрядных библиотек; В инсталлятор добавлена экспериментальная возможность установки на корневой раздел с ZFS; В Kubuntu предложен рабочий стол KDE Plasma 5.16, набор приложений KDE Applications 19.04.3 и фреймворк Qt 5.12.4. Обновлены версии latte-dock 0.9.2, Elisa 0.4.2, Kdenlive 19.08.1, Yakuake 19.08.1, Krita 4.2.6, Kdevelop 5.4.2, Ktorrent. Продолжается тестирование сеанса на базе Wayland (после установки пакета plasma-workspace-wayland появляется опциональный пункт "Plasma (Wayland)" на экране входа); В Xubuntu предложен новый выпуск рабочего стола Xfce 4.14. Вместо Light Locker для блокировки экрана задействован Xfce Screensaver, обеспечивающий интеграцию с Xfce Power Manager и улучшенную поддержку спящего и ждущего режимов; В Ubuntu Budgie добавлены новые апплеты Window Previews (замена менеджера задач (Alt+Tab)), QuickChar (просмотр таблиц символов), FuzzyClock, Workspace Stopwatch (секундомер) и Budgie Brightness Controller (управление яркостью экрана). Улучшена интеграция с GNOME 3.34. В Ubuntu MATE проведена работа по устранению недоработок и повышению качества интерфейса. Рабочий стол MATE обновлён до выпуска 1.22.2. Добавлен новый индикатор для вывода уведомлений, поддерживающий функцию "не беспокоить". Вместо Thunderbird по умолчанию задействован почтовый клиент Evolution, а вместо VLC - Celluloid (бывший GNOME MPV). Из базовой поставки удалены Qt4 и программа записи CD/DVD Brasero. В установочный образ включены проприетарные драйверы NVIDIA и комплект локализации для русского языка; В Ubuntu Studio добавлены пакет для организации потокового видеовещания OBS Studio и менеджер сеансов Raysession для управления программами обработки звука. В Ubuntu Studio Controls добавлено несколько прослоек для PulseAudio, реализован индикатор запуска Jack и добавлена возможность выбора бэкенда для Jack (Firewire, ALSA или Dummy). Обновлены версии компонентов: Blender 2.80, KDEnlive 19.08, Krita 4.2.6, GIMP 2.10.8, qJackCtl 0.5.0, Ardour 5.12.0, Scribus 1.4.8, Darktable 2.6.0, Pitivi 0.999, Inkscape 0.92.4, Carla 2.0.0, Ubuntu Studio Controls 1.11.3, В Lubuntu отмечено только исправление ошибок.
