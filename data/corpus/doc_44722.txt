Выпуск видеоредактора Pitivi 0.96

Доступен релиз свободной системы нелинейного видеомонтажа Pitivi 0.96, предоставляющей такие возможности, как поддержку неограниченного числа слоёв, сохранение полной истории операций с возможностью отката, отображение эскизов на шкале времени, поддержку типовых операций обработки видео и звука. Редактор написан на языке Python с использованием библиотеки GTK+ (PyGTK), GES (GStreamer Editing Services) и может работать со всеми форматами звука и видео, поддерживаемыми в GStreamer, включая формат MXF (Material eXchange Format). Код распространяется под лицензией LGPL. Основные новшества: Реализован режим прокси-редактирования (Proxy editing), позволяющем упростить работу с форматами, не подходящими для редактирования видео, за счёт использования в процессе монтажа варианта исходного материала в другом формате. При итоговом сохранении результата использованные при редактировании подменённые материалы будут заменены на оригинальные изображения, звук и видео. Вариант для подмены генерируется в форме набора специальных proxy-файлов, заменяющих оригинальные файлы в процессе работы. Прокси-редактирование удобно использовать для покадрового редактирования данных в форматах (например, VP8 и H.264), в которых производится разделение потока на ключевые и промежуточные кадры и не поддерживается быстрое и точное перемещение внутри клипа. Форматы, которые можно использовать в режиме прокси: QuickTime, Ogg, Matroska, WebM, H.264, MJPEG, VP8, Theora, raw, Opus, MP3, Vorbis, FLAC. Для создания прокси-файлов используются пакеты MJPEG или ProRes. В будущем выпуске Pitivi планируется расширить режим прокси-редактирования возможностью подмены исходных материалов высокого качества на варианты более низкого качества для ускорения редактирования и снижения нагрузки на систему. Переработан инструмент трансформации графических клипов (Transformation box), который теперь не ограничен системами с OpenGL и использует виджет GtkSink; Обеспечен автоматический расчёт настроек для проекта, который позволяет обойтись без открытия диалога с параметрами при первом сохранении проекта. При создании проекта теперь используются параметры по умолчанию, а при импорте первого видео производится перенос его свойств в настройки проекта. Возможность ручного изменения настроек сохранена и может быть вызвана в любое время, например, для изменения размера предпросмотра во время редактирования; Представлен новый интерфейс для настройки и рендеринга проекта - виджеты в диалогах настройки и рендеринга избавлены от вкладок и теперь предлагают все параметры на одном экране. Для компактного представления информации, которая должна умещаться на одном экране, применены виджеты ComboBox и MenuButton, которые позволили вынести элементы выбора в выпадающие меню; Добавлено окно для управления горячими клавишами; Модернизировано поведение шкалы времени. Добавлена опция отключающая смену позиции на шкале при клике левой кнопкой мыши (левая кнопка также используется для выделения, перемещения и редактировния клипов, что часто приводит к непреднамеренным сменам позиции на шкале). При включении опции для смены позиции предлагается использовать правую кнопку мыши. Отменено удаление операций undo/redo; Для создания универсальных пакетов с Pitivi вместо AppImage задействована система Flatpak; В качестве минимальных зависимостей заявлены GStreamer 1.8.2, GTK+ 3.20 и gst-transcoder 1.8.
