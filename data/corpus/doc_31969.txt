Hewlett-Packard создаёт облачный сервис на базе Ubuntu

Компания Canonical сообщила, что для построения облачного сервиса HP Public Cloud будет использовано решение на базе серверной редакции Ubuntu Linux. Ubuntu выбран как наиболее оптимальное решение для развертывания облачной инфраструктуры на базе платформы OpenStack. В итоге HP планирует создать похожий на Amazon EC2 сервис по предоставлению виртуальных серверов (IaaS - инфраструктура как сервис) для компаний любых размеров. В качестве ОС для гостевых систем также будет рекомендовано использовать Ubuntu. HP Public Cloud станет первым крупным облачным проектом, выполненным на основе Ubuntu и относительно новой платформы OpenStack. Напомним, что начиная с выпуска 11.10 дистрибутив Ubuntu одним из первых перешел на использование OpenStack в качестве основной cloud-платформы и больше не будет базироваться на Eucalyptus.
