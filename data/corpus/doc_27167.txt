Новая версия менеджера фотографий Shotwell 0.6

Вышел релиз программы для управления коллекцией фотографий Shotwell 0.6.1 (выпуск 0.6.0 был пропущен), которая будет включена в базовую поставку дистрибутива Ubuntu 10.10, заменив собой основанный на Mono проект F-Spot. Программа сочетает в себе инструменты для импорта и конвертации новых фотографий; удобные возможности каталогизации и навигации по имеющейся коллекции, поддержка группировки по времени и тегам; поддержку выполнения типовых операций по обработке изображений (вращение, устранение эффекта красных глаз, корректировка экспозиции, оптимизация цветности и т.п.); средства для публикации в социальных сетях, таких как Facebook, Flickr и Picasa. Основные улучшения версии 0.6: Базовая поддержка RAW-форматов, таких как CR2 и DNG; Полная поддержка средств для работы с PNG-изображениями; Поддержка масштабирования фотографий; Возможность открытия фотографий во внешних программах, например, в редакторе Gimp; Функции автоматического импорта и экспорта тегов и названий фотографий через мета-данные в форматах XMP и IPTC; Поддержка корзины для нежелательных фотографий; Новый диалог для настройки параметров программы; Из планов на будущее отмечаются такие возможности, как автоматическое отслеживание появление новых фотографий в файловой системе, добавление видеоэффектов и фоновой музыки при показе слайдшоу, инструменты для автоматической группировки фотографий по данным о местоположении и размещении на диске, базовая поддержка показа видео, механизмы для синхронизации фотографий между несколькими компьютерами. Пользователи Ubuntu могут установить новую версию, воспользовавшись специально подготовленным PPA-репозиторем. Готовые бинарные сборки также доступны для Fedora Linux и Windows.
