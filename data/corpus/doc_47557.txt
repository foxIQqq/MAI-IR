Доступен полностью свободный вариант ядра Linux-libre 4.14

Латиноамериканский Фонд свободного ПО опубликовал полностью свободный вариант ядра 4.14 - Linux-libre 4.14-gnu, очищенный от элементов прошивок и драйверов, содержащих несвободные компоненты или участки кода, область применения которых ограничена производителем. Кроме того, Linux-libre отключает функции ядра по загрузке несвободных компонентов, не входящих в поставку ядра, и удаляет упоминание об использовании несвободных компонентов из документации. Для очистки ядра от несвободных частей, в рамках проекта Linux-libre создан универсальный shell-скрипт, который содержит тысячи шаблонов для определения наличия бинарных вставок и исключения ложных срабатываний. Также доступны для загрузки готовые патчи, созданные на основе использования вышеупомянутого скрипта. Ядро Linux-libre рекомендовано для использования в дистрибутивах, соответствующих критериям Фонда СПО по построению полностью свободных дистрибутивов GNU/Linux. Например, ядро Linux-libre используется в таких дистрибутивах, как Dragora Linux, Trisquel, Dyne:Bolic, gNewSense, Parabola, Musix и Kongoni. В новом выпуске прекращена очистка набора прошивок, поставляемых в каталоге "firmware" (в связи с удалением из ядра данных прошивок). При этом привязки к прошивкам по-прежнему раскиданы по драйверам и компонентам ядра, которые не могут функционировать без их использования. Для предоставления прошивок для ядра проектом Linux-libre запущен собственный репозиторий linux-libre-firmware (Git), в котором собраны только свободные прошивки. Код адаптирован для задействования нового интерфейса загрузки прошивок request_firmware_into_buf. В новой версии также обновлён код для вырезания блобов из микрокода, а также драйверов xrx200, amdgpu, irda-usb, netronome nfp, brcmfmac, iwlwifi, rt5514 и звукового драйвера для чипов Skylake.
