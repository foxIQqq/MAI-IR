Microsoft откажется от поддержки Linux и Unix версии поисковых продуктов FAST

Компания Microsoft заявила о постепенном прекращении поддержки платформ Linux и Unix для продуктов, основанных на изначально многоплатформенной платформе FAST Enterprise Search Platform (ESP), предназначенных для организации локальных поисковых систем для web-ресурсов предприятий. Microsoft завладела продуктами линейки FAST после поглощения в 2008 году компании Fast Search & Transfer, код FAST основан изначально на многоплатрформенных технологиях и написан на языках C++, Java и Python. В будущем ESP будет позиционироваться как продукт только для Windows. Тем временем энтузиасты находят способы обхода отсутствия поддержки свободных технологий в проприетарных продуктах Microsoft, например, на днях была создана специальная промежуточная прослойка для обеспечения поддержки медиаконтейнера Ogg, видеокодека Theora и аудиокодека Vorbis для платформы Silverlight. Разработка поддерживает проигрывание видео Theora, заданного через HTML5 тег "video".
