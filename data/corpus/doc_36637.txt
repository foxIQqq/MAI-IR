Организация Linux Foundation представила проект OpenDaylight

Организация Linux Foundation представила новый открытый проект OpenDaylight, в рамках которого ряд ведущих производителей отрасли решили объединить свои усилия для создания единой открытой платформы для организации работы программно-конфигурируемых сетей (SDN, Software-Defined Networking). В числе компаний, подключившихся к разработке и передавших проекту некоторые из своих технологий, отмечены IBM, Juniper Networks, Cisco, Red Hat, VMware, Citrix, Ericsson, Microsoft, NEC, Big Switch Networks и Brocade. В итоге совместной работы планируется подготовить открытый фреймворк, на базе которого заинтересованные компании и отдельные разработчики смогут создавать готовые продукты и сервисы для SDN-сетей, в которых управление сетью отделено от уровня передачи данных и конфигурируется программно. Например, наиболее востребованным применением SDN являются облачные системы, в которых виртуализация сетевого уровня позволяет динамически предоставлять клиентам желаемые виртуальные сетевые ресурсы. Используя уже готовый унифицированный фреймворк, поддерживаемый всеми производителями оборудования, появится возможность существенно упростить и ускорить создание конечных приложений и сервисов для SDN, основанных на уже сформированном едином высокоуровневом стеке, не привязанном к решениям отдельного вендора. Первый выпуск OpenDaylight планируется подготовить в третьем квартале 2013 года. Выпуск будет включать реализацию контроллера для SDN-сети, систему для создания виртуальных сетей, набор плагинов для поддержки различных протоколов и набор улучшений для виртуальных коммутаторов. Таким образом OpenDaylight будет выступать в роли полнофункциональной SDN-платформы, пригодной для прямого развёртывания в различных сетевых окружениях, без необходимости использования каких-либо дополнительных компонентов. Все расширения и дополнительные возможности будут поставляться в форме плагинов. В дополнение к модульному управляющему фреймворку, OpenDaylight будет поддерживать существующие стандарты и протоколы SDN, в том числе I2RS, VxLAN и PCEP, сетевые сервисы для систем виртуализации, API для использования в конечных приложениях и элементы для поддержки интерфейсов физических устройств и виртуальных комутаторов. Построенная на основе OpenDaylight инфраструктура позволит управлять конфигурацией всех сетевых компонентов не настраивая каждое сетевое устройство, маршрутизировать потоки трафика, выполнять балансировку нагрузки, легко создавать собственные сетевые политики и устанавливать приоритеты для разных видов трафика. Фреймворк будет поддерживать любые операционные системы, будет поставляться под свободной лицензией Eclipse Public License (EPL) и будет развиваться в соответствии с открытыми спецификациями и стандартами, такими как OpenFlow. Процесс разработки, приёма новшеств и управления проектом будет соответствовать модели, используемой в полностью открытых проектах, развиваемых сообществом и не зависимых от отдельных производителей. Гарантом открытости и независимости проекта будет выступать организация Linux Foundation.
