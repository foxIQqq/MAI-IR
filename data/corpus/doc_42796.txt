Обновление Firefox 40.0.2

Доступно корректирующее обновление web-браузера Firefox 40.0.2, в котором включена поддержка API, позволяющего отобразить для пользователей Windows 10 диалог с настройками для выбора браузера по умолчанию. В состав 32-разрядных сборок для Windows добавлен mozalloc.lib, который забыли включить, что привело к проблемам при использовании SDK при сборке дополнений. Устранена проблема с крахом при запуске на системах с определённым сочетанием драйверов и оборудования DisplayLink.
