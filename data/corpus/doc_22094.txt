Объявлены победители конкурса Trophees du Libre 2009

Стали известны победители конкурса "Les Trophees du Libre", проводимого французской организацией-популяризатором свободного ПО CETRIL. Цель соревнования - поддержка и широкая огласка перспективных молодых свободных проектов. За прошлые годы существования конкурса в финал попал только один проекта из эксСССР (векторный редактор sK1), в нынешнем году ситуация изменилась - в финале оказались сразу четыре отечественных проекта, из которых два признаны победителями в своих категориях (MathGL и UniConvertor), а два - заняли третье и пятое места (Inquisitor и GSQL). Победители определены в 7 номинациях, дополнительно присуждены 3 специальные награды. Медиа. UniConvertor - украинский проект, представляет собой универсальный конвертер векторной графики. Поддерживает форматы CorelDraw, Adobe Illustrator, WMF, SVG и др. Несколько дней назад выпущен новый релиз UniConvertor 1.1.4, в котором появилась поддержка импорта и экспорта в формате PLT, подготовлена для загрузки бинарная win32 сборка; Coherence - ПО для распределения видео/аудио контента и изображений по различным носителям; Kinovea - программа для анализа видео и декомпозиции перемещений объектов в кадре; SMILE - среда для подготовки слайдшоу в Linux. Инструменты профессионала. GOsa - написанный на PHP инструмент для управления учетными записями в LDAP каталоге; Support Incident Tracker - интерфейс для обеспечения работы службы поддержки (обработки звонков и запросов по электронной почте). Написан на PHP, в качестве СУБД используется MySQL; Projectivity 3.0 - web-платформа для организации совместной работы над проектами и ведения базы знаний; Tine 2.0 - ПО комбинирующее в себе функции средства для организации групповой работы, CRM и ERP; GSQL - разработанное в России GNOME-приложение для управления СУБД. В настоящий момент реализована поддержка MySQL 5 и Oracle. Из возможностей присутствует редактор SQL запросов, режим редактирования PL/SQL для Oracle, средство для древовидной навигации по структурам БД, возможность экспорта результатов запроса в CSV или XML форматах и т.д. В будущем планируется реализовать поддержку СУБД PostgreSQL, SQLite и Firebird. Безопасность KSplice - пакет для обновления ядра Linux на лету, без необходимости перезагрузки и остановки работы. Enigform (расширение к Firefox и apache модуль mod_openpgp) - организация нового метода HTTP-аутентификации и проверки валидности http-запросов через задействование цифровых подписей. Русский проект Inquisitor - открытая платформа для диагностики, всестороннего тестирования, мониторинга и измерения производительности аппаратного обеспечения. Администрирование Linea21 - Web-приложение для организации работы муниципалитетов во Франции; Tellmatic -ПО для организации персонализированной почтовой рассылки; Наука MathGL - отечественная библиотека для построения научных графиков; Gmsh - пакет для визуализации 3D структур; GRAPHITE - платформа для разработки в области 3D моделирования и синтеза изображений. Хобби Pychess - шахматы целиком написанные на языке Python; Neverball - аркадная 3D игра; Djl - менеджер видеоигр для Linux; Обучение Vallect - программа для поддержания домашней библиотеки; OSCAR - ПО для управления компьютерным классом; Emustru - система online-обучения английскому языку. Специальные награды: Награда "истинный дух open source" присуждена системе Ksplice; Награда "выбор жюри" присуждена проекту Coherence; Награда "инновационный проект" получила платформа GRAPHITE.
