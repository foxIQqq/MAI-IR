Релиз дистрибутива Ubuntu 22.04 LTS

Состоялся релиз дистрибутива Ubuntu 22.04 "Jammy Jellyfish", который отнесён к категории выпусков с длительным сроком поддержки (LTS), обновления для которых формируются в течение 5 лет, в данном случае - до апреля 2027 года. Установочные и загрузочные образы созданы для Ubuntu, Ubuntu Server, Lubuntu, Kubuntu, Ubuntu Mate, Ubuntu Budgie, Ubuntu Studio, Xubuntu и UbuntuKylin (редакция для Китая). Основные изменения: Рабочий стол обновлён до выпуска GNOME 42, в котором добавлены общие для всего окружения настройки тёмного стиля оформления интерфейса и проведена оптимизация производительности GNOME Shell. При нажатии кнопки PrintScreen предоставлена возможность создания скринкаста или скриншота выделенной части экрана или отдельного окна. Для сохранения целостного оформления и стабильности работы пользовательского окружения в Ubuntu 22.04 оставлены версии некоторых приложений из ветки GNOME 41 (в основном речь о приложениях, переведённых в GNOME 42 на GTK 4 и libadwaita). Для большинства конфигураций по умолчанию задействован сеанс рабочего стола на базе протокола Wayland, но при входе в систему предоставлена возможность отката на использование X-сервера. Использование X-сервера также оставлено по умолчанию для систем с проприетарными драйверами NVIDIA. Предложено 10 вариантов цветового оформления в тёмном и светлом стилях. Пиктограммы на рабочем столе по умолчанию перемещены в правый нижний угол экрана (данное поведение можно изменить в настройках внешнего вида). В теме оформления Yaru для всех элементов кнопок, ползунков, виджетов и переключателей вместо баклажанового цвета задействован оранжевый цвет. Аналогичная замена выполнена в наборе пиктограмм. Цвет кнопки закрытия активного окна изменён с оранжевого на серый, а цвет ручек ползунков с светло-серого на белый. Добавлены новые настройки для управления внешним видом и поведением Dock-панели. Улучшена интеграция с панелью файлового менеджера и виджетов устройств. Обеспечена поддержка экранов для вывода конфиденциальной информации, например, некоторые ноутбуки комплектуются экранами со встроенным режимом конфиденциального просмотра, затрудняющим просмотр со стороны. Предоставлена возможность использования протокола RDP для организации совместного доступа к рабочему столу (поддержка VNC сохранена в качестве опции, включаемой в конфигураторе). Браузер Firefox теперь поставляется только в формате Snap. Deb-пакеты firefox и firefox-locale замены на заглушки, устанавливающие Snap-пакет с Firefox. Для пользователей deb-пакета предоставлен прозрачный процесс перехода на snap через публикацию обновления, которое установит snap-пакет и перенесёт текущие настройки из домашнего каталога пользователя. С целью повышения безопасности по умолчанию отключена утилита os-prober, находящая загрузочные разделы других ОС и добавляющая их в загрузочное меню. Для загрузки альтернативных ОС рекомендуется использовать загрузчик UEFI. Для возвращения автоматического определения сторонних ОС в /etc/default/grub можно изменить настройку GRUB_DISABLE_OS_PROBER и выполнить команду "sudo update-grub". Отключён доступ к NFS-разделам с использованием протокола UDP (ядро собрано с опцией CONFIG_NFS_DISABLE_UDP_SUPPORT=y). В сборки для архитектуры ARM64 в состав набора linux-restricted-modules добавлены проприетарные драйверы NVIDIA (ранее поставлялись только для систем x86_64). Для установки и настройки драйверов NVIDIA можно использовать штатную утилиту ubuntu-drivers. В качестве основного задействовано ядро Linux 5.15, но в Ubuntu Desktop на некоторых проверенных устройствах (linux-oem-22.04) будет предоставлено ядро 5.17. Дополнительно для архитектур x86_64 и ARM64 предложен для тестирования бета-выпуск пакета с ядром, включающего патчи PREEMPT_RT и ориентированного на применение в системах, работающих в режиме реального времени. Системный менеджер systemd обновлён до версии 249. Для раннего реагирования на нехватку памяти в Ubuntu Desktop по умолчанию задействован механизм systemd-oomd, который базируется на подсистеме ядра PSI (Pressure Stall Information), позволяющей в пространстве пользователя проанализировать информацию о времени ожидания получения различных ресурсов (CPU, память, ввод/вывод) для точной оценки уровня загруженности системы и характера замедления работы. Для проверки состояния OOMD можно использовать утилиту oomctl. Обновлены версии инструментов для разработчиков: GCC 11.2, LLVM 14, glibc 2.35, Python 3.10.4, Ruby 3.0, PHP 8.1.2, Perl 5.34, Go 1.18, Rust 1.58, OpenJDK 18 (также поставляется и OpenJDK 11), PostgreSQL 14, MySQL 8.0.28. Обновлены версии LibreOffice 7.3, Firefox 99, Thunderbird 91, Mesa 22, BlueZ 5.63, CUPS 2.4, NetworkManager 1.36, Poppler 22.02, Chrony 4.2, PulseAudio 16, xdg-desktop-portal 1.14, samba 4.15.5, Apache httpd 2.4.52, containerd 1.5.9, runc 1.1.0, QEMU 6.2, libvirt 8.0.0, virt-manager 4.0, openvswitch 2.17, LXD 5.0. Осуществлён переход на новые значительные ветки OpenLDAP 2.5, BIND 9.18 и OpenSSL 3.0. В состав репозитория main для Ubuntu Server включены пакеты wireguard и glusterfs. В состав включён стек протоколов маршрутизации FRRouting (BGP4, MP-BGP, OSPFv2, OSPFv3, RIPv1, RIPv2, RIPng, PIM-SM/MSDP, LDP, IS-IS), который заменил собой ранее применявшийся пакет Quagga (FRRouting является ответвлением от Quagga, поэтому совместимость не пострадала). По умолчанию задействован пакетный фильтр nftables. Для сохранения обратной совместимости доступен пакет iptables-nft, предоставляющий утилиты с тем же синтаксисом командной строки, как и в iptables, но транслирующий полученные правила в байткод nf_tables. В OpenSSH по умолчанию отключена поддержка цифровых подписей на базе RSA-ключей с хэшем SHA-1 ("ssh-rsa"). В утилиту scp добавлена опция "-s" для работы по протоколу SFTP. В сборках Ubuntu Server для систем IBM POWER (ppc64el) прекращена поддержка процессоров Power8, сборки теперь собираются для CPU Power9 ("--with-cpu=power9"). Обеспечено формирование работающих в live-режиме установочных сборок для архитектуры RISC-V. Ubuntu 22.04 стал первым LTS-выпуском с официальными сборками для плат Raspberry Pi. Добавлена поддержка светодиодной матрицы Pimoroni Unicorn HAT и сенсорных экранов DSI. Для плат Raspberry Pi Compute добавлена утилита rpiboot. Для микроконтроллеров с поддержкой MicroPython, таких как Raspberry Pi Pico, добавлена утилита rshell (пакет pyboard-rshell). Для предварительной настройки загрузочного образа добавлена утилита imager (пакет rpi-imager). В Kubuntu предложен рабочий стол KDE Plasma 5.24.3 и набор приложений KDE Gear 21.12.3. В Xubuntu продолжена поставка рабочего стола Xfce 4.16. Набор тем оформления Greybird обновлён до версии 3.23.1 с поддержкой GTK 4 и libhandy, что улучшило унификацию оформления приложений GNOME и GTK4 с общим стилем Xubuntu. Обновлён набор elementary-xfce 0.16, в котором предложено много новых пиктограмм. Задействован текстовый редактор Mousepad 0.5.8 с поддержкой сохранения сеансов и плагинов. В просмотрщике изображений Ristretto 0.12.2 улучшена работа с миниатюрами. В Ubuntu MATE рабочий стол MATE обновлён до корректирующего выпуска 1.26.1. Стилевое оформление переведено на вариант темы Yaru (применяется в Ubuntu Desktop), адаптированный для работы в MATE. В основной состав включены новые приложения GNOME Clocks, Maps и Weather. Обновлён набор индикаторов для панели. За счёт удаления проприетарных драйверов NVIDIA (теперь загружаются отдельно), устранения дубликатов пиктограмм и удаления старых тем оформления размер установочного образа уменьшен до 2.8 ГБ (до чистки был 4.1 ГБ). В Ubuntu Budgie задействован новый выпуск рабочего стола Budgie 10.6. Обновлены апплеты. В Ubuntu Studio обновлены версии Blender 3.0.1, KDEnlive 21.12.3, Krita 5.0.2, Gimp 2.10.24, Ardour 6.9, Scribus 1.5.7, Darktable 3.6.0, Inkscape 1.1.2, Carla 2.4.2, Studio Controls 2.3.0, OBS Studio 27.2.3, MyPaint 2.0.1. В сборках Lubuntu продолжает поставляться графическое окружение LXQt 0.17. Дополнительно можно отметить выпуски двух неофициальных редакций Ubuntu 22.04 - Ubuntu Cinnamon Remix 22.04 (iso-образы) с рабочим столом Cinnamon и Ubuntu Unity 22.04 (iso-образы) с рабочим столом Unity7.
