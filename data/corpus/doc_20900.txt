Обзор изменений, ожидаемых в SUSE Linux Enterprise 11

"Novell kicks out SUSE Linux Enterprise 11" - обзор изменений, ожидаемых в SUSE Linux Enterprise 11. Например, появится коммерческая поддержка инфраструктуры для выполнения .Net приложений, основанная на Mono 2.4. Начнется распространение сборки High Availability Extension, нацеленной на создание высоконадежных кластеров, базируясь на кластерной файловой системе OCFS2 (Oracle Cluster File System 2), системе репликации блочных устройств DRDB 8, менеджере логических томов cLVM2, менеджере ресурсов Pacemaker и пакете для переключения нагрузки на запасной сервер OpenAIS. Инфраструктура виртуализации будет основана на Xen 3.3, в состав дистрибутива также войдет новая система для развертывания и управления виртуальными машинами.
