Компания Canonical проводит конкурс разработчиков приложений Ubuntu App Showdown

Компания Canonical объявила о проведении Ubuntu App Showdown - конкурса по разработке приложений для Ubuntu Linux. Для разработки рекомендуется использовать пакет Quickly, предназначенный для быстрого создания приложений с использованием языка Python, библиотеки GTK+ и инструментария, позволяющего автоматизировать многие рутинные операции при работе с такими системами, как Launchpad, Bazaar и PPA (Personal Package Archives). Тем не менее, жестких ограничений на технологии разработки не налагается, поэтому можно использовать любые языки и тулкиты. Работы на конкурс будут приниматься с 18 июня по 9 июля, после чего жюри определит победителей. На конкурс должны быть представлены только новые приложения, на разработку которых отводится три недели. Победитель получит ноутбук System76 Gazelle Professional и смартфон Nokia N9, занявший второе место - ноутбук System76 Lemur Ultra и смартфон Nokia N9, занявший третье место - смартфон Nokia N9. Дополнительно будет проведено общественное голосование пользователей через Ubuntu Software Center, по результатам которого ещё трём участникам будут подарены смартфоны Nokia N9.
