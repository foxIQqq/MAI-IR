Наиболее важные события 2013 года

Небольшая итоговая подборка наиболее важных и заметных событий уходящего года: Развитие компанией Canonical собственного дисплейного сервера Mir, что заметно подстегнуло разработчиков конкурирующих проектов и привело к ускорению развития и продвижения Wayland, а также проведению работы по поддержке Wayland в KDE, GNOME, Enlightenment. Использование Wayland в мобильной платформе Sailfish и решение по переходу на Wayland мобильной платформы Tizen. Создание нового десктоп-окружения Hawaii, базирующегося на Wayland. Расцвет инициативы по продвижению Linux в качестве первичной игровой платформы. Релиз клиента Steam для Linux. Выпуск компанией Valve тестовой версии дистрибутива SteamOS и начало тестирования прототипов игровых консолей на его основе. Положительное влияние инициативы Valve на качество и производительность графических драйверов для Linux. Общее число доступных в каталоге Steam игр для Linux достигло 283. Выход набора лицензий Creative Commons 4.0. Заметное увеличение интенсивности разработки FreeBSD. Подготовка релиза FreeBSD 10.0. Использование кода FreeBSD в PlayStation 4. Продвижение средств виртуализации уровня отдельных приложений и минималистичных операционных систем для систем виртуализации: CoreOS, Lmctfy, FusedOS, OSv. Новый гипервизор Jailhouse. Развитие и продвижение в дистрибутивы инструментария Docker и контейнеров LXC. Реализация в GNOME 3.8 классического рабочего стола в виде набора дополнений к GNOME Shell. Поддержка Wayland в GNOME 3.10. Выпуск окружения Cinnamon 2.0, ставшего полностью независимым от GNOME. Выбор JavaScript в качестве основного языка разработки приложений для GNOME. Раскрытие инициатив АНБ по организации прослушивания и внедрения бэкдоров, что подстегнуло сообщество к аудиту кода и средств шифрования и перевернуло представления о приватности. Волна перехода дистрибутивов на systemd и начало тестирования Red Hat Enterprise Linux 7.0; Поступление в продажу первых устройств на базе Firefox OS. Мобильные инициативы Ubuntu и попытка создания смартфона Ubuntu Edge. Развитие мобильной платформы Tizen, анонс первого планшета и телевизора на базе Tizen. Мобильные инициативы проекта KDE, в том числе OpenHardware-платформа Improv. Выпуск смартфона на ОС Sailfish, основанной на Mer, Qt и Wayland. Открытая платформа Ara для создания модульных смартфонов. Развитие компонентов KDE 5 и переход к раздельному выпуску компонентов окружения. Начало тестирования KDE Plasma 2. Создание компанией Google форка движка WebKit - Blink, переход на Blink Qt и Opera. Продвижение решений для написания полноценных приложений на JavaScript. Стабильный выпуск языка программирования Dart. Интеграция в Firefox 22 поддержки Asm.js, низкоуровневого высокопроизводительного подмножества JavaScript. Интеграция в Chrome 31 Portable Native Client для запуска платформонезависимых C/C++ программ в браузере. Открытие кода Duetto, системы для запуска в web-браузере проектов на языке C++ Инициатива по объединению десктоп-окружений Razor-qt и LXDE. Угроза стандартизации средств DRM для HTML5-контента. Релиз Debian 7.0 "Wheezy". Выпуск DNS-сервера BIND 10 и обновления BIND 10 1.1.0; Выпуск LibreOffice 4.0 и Apache OpenOffice 4.0; Релизы MySQL 5.6, VoltDB 3.0, PostgreSQL 9.3. Уход многих проектов с MySQL на MariaDB, в том числе Google Cloud SQL, RHEL 7, Wikimedia, Arch Linux, Fedora и openSUSE. Первый стабильный выпуск кластерной СУБД MariaDB Galera. Универсальное решение для использования UEFI Secure Boot в любых дистрибутивах, добавление поддержки UEFI Secure Boot в дистрибутивы. Китайское правительство утвердило Ubuntu в качестве стандартной операционной системы. Завершение миграции администрации Мюнхена на Linux. Дополнения: Компания Oracle объявила о прекращении поддержки JDK 6 и сосредоточении ресурсов на JDK 7. Компания Red Hat объявила о поддержке пользователей JDK 6. Вышли интегрированные среды программирования JetBrains IDEA 13, Oracle Netbeans 7.4 и Eclipse 4.3 с полной поддержкой Java 7.0. Компания Oracle анонсировала новую версию платформы Java EE 7 с сервером приложений GlassFish 4 и заявила о прекращении коммерческой поддержки пользователей сервера приложений GlassFish. Компания RedHat выпустила язык Ceylon, претендующий на роль замены Java. Компания Cisco открыла исходные тексты видеокодека Openh264. Выпуск аудиокодеков Opus 1.1 и FLAC 1.3.0. Стабилизация и продвижение кодека VP9, его интеграция в Chrome и Firefox. Разработка свободного видеокодека Daala с технологиями, опережающими VP9 и H.265/HEVC.
