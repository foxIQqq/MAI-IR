Компания Palm выпустила новую версию основанной на Linux платформы webOS 1.2

Представлен релиз основанной на Linux мобильной операционной системы webOS 1.2, используемой на смартфонах Palm Pre. Пакеты с исходными текстами различных модифицированных open source компонентов webOS 1.2 в ближайшее время можно будет загрузить с сайта opensource.palm.com. Для разработчиков приложений одновременно выпущен Mojo SDK v1.2. В webOS 1.2 представлены следующие улучшения: Расширены средства синхронизации данных (добавлена поддержка LinkedIn); В email-клиенте добавлена возможность расширенного поиска через фильтрацию писем по заголовкам и отправителю; В web-браузере появилась возможность загрузки и сохранения файлов. Кроме того, теперь можно сохранить локально изображение с web-страницы; Добавлена возможность прямой загрузки музыки из сервиса Amazon MP3 Store через WAN и WiFi соединения; Появилась возможность инициирования звонка из заметки в календаре-планировщике; При продолжении прослушивании прерванного подкаста, проигрывание теперь начинается с последней прослушанной позиции; Добавлена возможность выделения для помещения в буфер обмена текста, изображений, URL и email адресов с web-страниц и из сообщений электронной почты (копирование через Shift-Tap). В меню редактирования появилась дополнительная опция "Select All"; В менеджере установки приложений App Catalog добавлена поддержка покупки программ; Наряду с Jpeg изображениями платформа теперь поддерживает и PNG формат; Решены проблемы с сопряжением Bluetooth устройств; В web-браузере добавлена возможность бэкапа и восстановления закладок и истории посещений. При выборе формы, элементы теперь автоматически увеличиваются на экране для улучшения восприятия. Не решены проблемы синхронизации с iTunes 9.01, даже несмотря на то, что WebOS представляется на USB-шине как iPod. WebOS является многозадачной ОС и отличается тесной интеграцией с интернет-технологиями и широким использованием web-приложений. Весь пользовательский интерфейс WebOS оформлен с использованием CSS, HTML5 и JavaScript. Вместо окон используется парадигма сменяющих друг друга карт. Обмен данными между приложениями построен на основе шины, манипулирующей данными в формате JSON. Для доступа web-приложений к системному API используется D-Bus Bridge. Из стандартных программ присутствует основанный на Webkit web-браузер, близкий по возможностям к браузерам настольных систем, приложения для чтения электронной почты, календарь-планировщик, калькулятор, заметки, клиент для мгновенного обмена сообщений, мультимедиа плеер (видео - MPEG-4, H.263, H.264; аудио - MP3, AAC, AAC+, AMR, QCELP, WAV; изображения - GIF, Animated GIF, JPEG, PNG, BMP). Для синхронизации данных (календарь, почта, адресная книга, фотографии и т.п.) с web-сервисами (Facebook, Gmail и т.п.) используется технология Synergy. Поддерживается навигация через жесты и одновременные касания к экрану (мультитач режим).
