Вышел дистрибутив для создания межсетевых экранов IPCop 2.0

В канун десятилетия проекта, после трех лет разработки представлен релиз IPCop 2.0, одного из старейших Linux-дистрибутивов для создания межсетевых экранов для небольших организаций и домашних сетей. Управления работой производится через web-интерфейс. В 2009 году IPCop стал лауреатом премии Best of Open Source Software Awards в номинации "Лучшее открытое сетевое ПО". Размер iso-образа 59 Мб. Особенности новой версии: Новый инсталлятор, позволяющий установить дистрибутив на USB Flash или жесткий диск и сразу настроить сетевые интерфейсы; Осуществлен переход на ядро Linux 2.6.32 (в прошлой версии использовалось ядро 2.4.36); Переработанный web-интерфейс: полностью переделан раздел настройки межсетевого экрана; добавлена страница-планировщик для настройки выполнения периодических заданий; в раздел просмотра статистики добавлены страницы для отображения системной информации, расхода трафика, отчет IPTables и переработанная страница с отображением текущих соединений; Добавлена поддержка OpenVPN (включая интеграцию настройки в web-интерфейс); Вместо dhcpd в качестве DHCP-сервера задействован dnsmasq. Упрощен интерфейс для настройки DHCP-сервера; Переработан интерфейс для настройки сервера точного времени, для работы которого используется ntpd; Добавлена поддержка нового оборудования, включая платформы Cobalt, SPARC и PPC; Доступ ко всем страницам web-интерфейса теперь требует обязательной аутентификации по паролю; Изменение номеров сетевых портов по умолчанию: для web-интерфейса теперь используется порт 8443 вместо 445, а для SSH 8022 вместо 222; Доступен перевод интерфейса на русский язык; Поддержка системы обнаружения атак Snort вынесена из базовой поставки и доступна в качестве дополнения.
