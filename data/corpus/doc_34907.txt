Исходные тексты игрового движка Torque 3D размещены на GitHub

Следуя ранее данному обещанию, компания GarageGames открыла исходные тексты игрового движка Torque 3D. Код движка и набор из четырёх шаблонов для создания типовых игр размещены на GitHub под свободой лицензией MIT. Дополнительно опубликован набор документации с описанием архитектуры Torque 3D, техники создания игр и руководством по написанию скриптов для определения игровой логики. GitHub выбран для размещения кода с целью упрощения приема изменений в проект и стимулирования вовлечения в разработку новых участников. В GitHub размещена основная master-ветка, содержащая последний стабильный выпуск движка (в данный момент код аналогичен по своему составу коммерческой версии Torque 3D 1.2), а также development-ветка, в которой будет вестись основная работа по дальнейшему развитию движка. В процессе подготовки релизов улучшения и исправления из development-ветки будут переноситься в ветку master. Для координации приёма изменений в открытый репозиторий и принятия решений по развитию проекта сформирован специальный комитет, который будет принимать решения о предоставлении доступа к репозиторию на запись и по приёму pull-запросов c патчами. В настоящее время в комитет вошли четыре основных разработчика движка Torque 3D, но в дальнейшем не исключается возможность вхождения в управляющий комитет и представителей сообщества. Если перевод Torque 3D в разряд открытых проектов окажется успешным и вокруг него удастся сформировать активное сообщество разработчиков, то компания GarageGames не исключает перевод в разряд открытых и других своих продуктов. Движок Torque 3D дебютировал в игре Tribes 2, выпущенной в 2001 году, но его развитие продолжается до сих пор и на различных версиях Torque 3D выпущено несколько десятков популярных коммерческих игр. В настоящее время Torque 3D позиционируется в роли платформы для создания игр, включающей такие подсистемы, как редактор ресурсов, физический движок, система рендеринга, средства для определения игровой логики и инструменты публикации игр. Для упрощения создания игр сформированы базовые шаблоны для разработки шутеров от первого лица, стратегий реального времени, RPG и гоночных симуляторов. Движком поддерживается использование 3D-моделей в форматах DTS и DIF. Сборка возможна для платформ Windows и Mac OS X.
