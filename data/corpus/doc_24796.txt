Проект VideoLAN анонсировал создание свободного редактора видео

Проект VideoLAN, в рамках которого разрабатывается популярный видеопроигрыватель VLC, объявил на проходящей в эти дни в Париже конференции "VideoLAN dev'days 2009" о создании нового редактора видео, получившего название VLMC (VideoLAN Movie Creator). По замыслу разработчиков, VLMC, обладая дружелюбным и простым пользовательским интерфейсом, будет поддерживать спектр возможностей, достаточный для создания фильмов полупрофессионального качества. Первый предварительный релиз для платформ Linux, Windows и MacOS X ожидается в ближайшее время. В настоящий момент пользователи могут попытаться собрать VLMC из исходных текстов, распространяемых через Git-репозиторий проекта (git clone git://github.com/VLMC/vlmc.git). В качестве лицензии используется GPLv2, интерфейс редактора построен с использованием библиотеки Qt 4.5. Видеоролик с демонстрацией работы VLMC можно посмотреть здесь.
