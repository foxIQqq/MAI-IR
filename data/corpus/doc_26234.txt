Эксперименты с Firefox: запуск на Palm Pre и управление через Wiimote

Энтузиастам удалось запустить web-браузер Firefox на смартфоне Palm Pre, работающем под управлением платформы webOS, обычные приложения для которой оформляются с использованием CSS, HTML5 и JavaScript. Для организации выполнения Firefox был использован инструментарий Plug-in Development Kit (PDK), позволяющий дополнить платформу webOS средствами для создания плагинов на языках языках C и C++. Так как в webOS был запущен браузер с интерфейсом для настольных систем, он мало подходит для повседневного использования на мобильном телефоне, но в будущем разработчики планируют попытаться адаптировать для смартфонов Palm специально оптимизированный для небольших сенсорных экранов браузер Firefox Mobile (Fennec). Дополнительно можно отметить проект, нацеленный на интеграцию в Firefox драйвера для организации удаленного управление браузером при помощи дистанционного пульта Wiimote от игровой приставки Nintendo Wii. Для разработчиков проект интересен демонстрацией, как можно интегрировать в Firefox дополнение, написанное на языке C++, а не XUL.
