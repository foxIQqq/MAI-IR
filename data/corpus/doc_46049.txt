Выпуск проприетарного драйвера NVIDIA 378.13

Компания NVIDIA представила первый выпуск новой стабильной ветки проприетарного драйвера NVIDIA 378.13. Драйвер доступен для Linux (ARM, x86, x86_64), FreeBSD (x86, x86_64) и Solaris (x86_64). Основные новшества: Добавлена инфраструктура для загрузки драйвером EGL дополнительных внешних библиотек, обеспечивающих поддержку новых клиентских оконных систем, помимо существующей библиотеки для Wayland (libnvidia-egl-wayland.so); Добавлена поддержка GPU: Quadro GP100, Quadro P4000, P3000, P2000, P1000, P600, P400, Quadro M1200 и M2200; Реализовано расширение ARB_parallel_shader_compile, позволяющее реализовать многопоточную компиляцию шейдеров GLSL; В X11-драйвер добавлена опция "AllowHMD", позволяющая пропустить обнаружение подключения шлемов виртуальной реальности; В nvidia-settings добавлена поддержка просмотра настроенных экранов PRIME; Добавлена поддержка новых расширений для API Vulkan: VK_KHR_display, VK_KHR_display_swapchain; Добавлена возможность использования screen_info.ext_lfb_base на ядрах с поддержкой данного параметра, что позволяет корректно обрабатывать консоли на базе фреймбуфера для UEFI в конфигурациях с физическими адресами памяти выше 4GB; Добавлена поддержка X.Org xserver ABI 23 (xorg-server 1.19); Включены по умолчанию (кроме Xinerama) многопоточные оптимизации для OpenGL, которые отключаются автоматически при фиксации снижения производительности, что позволяет сохранить уровень производительности в приложениях для которых данные оптимизации бесполезны и повысить скорость работы приложений, которым они необходимы.
