Вышла первая бета версия Javascript библиотеки Yahoo! User Interface 3

Вышла первая бета версия javascript библиотеки Yahoo! User Interface (YUI) для создания графического интерфейса веб-приложений. При подготовке этой версии ключевое внимание было уделено трем основным компонентам этой библиотеки - YUI, Node и Event. Также появились компоненты из ветки 2.x: DataSource, ImageLoader, History и StyleSheet В планах команды разработчиков - выпуск 3-ей версии в третьем квартале и версии 3.1 - в четвертом. Стоит также отметить, что развитие версии 2.x продолжается (несмотря на призывы разработчиков к переходу на версию 3.x). В планах - версия 2.8.0, наиболее значимым изменением в которой будет создание унифицированного интерфейса для хранения данных в браузере с использованием технологий HTML 5 и Flash.
