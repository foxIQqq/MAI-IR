Релиз Linux-дистрибутива Zenwalk 7.2

Вышел релиз пользовательского дистрибутива Zenwalk Linux 7.2, совместимого с пакетной базой Slackware и использующего графическое окружение на основе Xfce 4.10. Дистрибутив отличается современной начинкой, дружественностью к пользователю, высокой скоростью работы, рациональностью подхода к выбору приложений (одно приложение для одной задачи), самодостаточностью (нет необходимости что-то доустанавливать) и использованием для управления пакетами системы Netpkg. Размер установочного iso-образа составляет 679 Мб. В новой версии проведена работа по достижению 100% совместимости со Slackware, произведено обновление пакетов. В состав входят: ядро Linux 3.4.8 c включенным по умолчанию планировщиком задач BFS, Xfce 4.10, GTK 2.24.10/3.4.4, Libreoffice 3.6.2, Firefox 15.0.1, Thunderbird 15.0.1, Gimp 2.8.2, X Server 1.12.3, GCC 4.7.1, glibc 2.15.
