Вышел седьмой тестовый выпуск openSUSE 11.2

Вышел седьмой тестовый выпуск openSUSE 11.2 (Milestone 7). Релиз openSUSE 11.2 намечен на ноябрь. В новой тестовой версии представлены следующие изменения: Начальная версия нового визуального оформления инсталлятора YaST2; Пакеты сформированы с использование RPM версии 4.7.1 При установке c DVD, десктоп-окружение KDE отныне выбрано по умолчанию; Начальная версия новой визуальной темы для KDE; В zypper добавлена опция "--download-only", позволяющая осуществить только загрузку компонент, без выполнения действи. Опцию можно использовать совместно с командами install, update, patch, dup и verify; Обновление версий программ: GNOME 2.28 beta 2 KDE 4.3.1 Linux ядро 2.6.31-rc9 Samba 3.4.0 Evolution 2.27.91 OpenOffice.org 3.1.1 RC 1 Pidgin 2.6.1 Mono 2.4.2.3 Wine 1.1.28 Alsa 1.0.21 bluez 4.50 PulseAudio 0.9.16-test7 Apache 2.2.13 Базовые новшества openSUSE 11.2: В качестве десктоп-окружения по умолчанию будет использоваться KDE 4.3; Время поддержки дистрибутива (выпуска обновлений) будет сокращено с 24 до 18 месяцев; Пакеты дистрибутива собраны GCC 4.4.1 Образ LiveCD теперь может быть использован и на USB Flash; В состав LiveCD добавлен Midnight Commander, а в KDE-вариант LivCD программа YaKuake; Для YaST создан новый центр управления конфигурацией (YaST Control Center) с интерфейсом на базе Qt4; В качестве файловой системы по умолчанию задействована Ext4 В состав Live CD включен Gimp; Для сжатия данных на LiveCD отныне используется алгоритм сжатия LZMA; По умолчанию LiveCD содержит переводы для немецкого, французского, итальянского, польского и русского языков; В Qt версии конфигуратора YaST появилась поддержка настраиваемого внешнего вида. В YaST добавлены средства работы с кешем пакетов; В Zypp улучшены средства работы с зеркалами репозиториев пакетов; В интерфейсе для настройки параметров дисплея gnome-display-properties добавлена поддержка автоматического сброса настроек в случае проблем. Ведется разработка web-интерфейса к YaST для осуществления удаленного управления дистрибутивом Появление установочных образов для USB Flash и утилита для их подготовки Улучшение поддержки субноутбуков; В состав LiveCD с KDE добавлены новые программы: Choqok, KNode, Kompare, KSystemlog, Okteta; В состав GNOME LiveCD добавлен Bittorrent-клиент Transmission;
