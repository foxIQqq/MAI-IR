Компания Google выпустила релиз Android SDK 1.1. Archos выпустит Android-планшет

Вышел Android SDK 1.1, новая версия инструментария для разработки приложений для телефонов на основе открытой платформы Google Android. Одновременно выпущено обновление прошивки (RC33) для телефонов T-Mobile G1, содержащее исправления ошибок и незначительные улучшения, например, появилась возможность сохранения картинок из MMS сообщений, режим голосового ввода ключевых слов для поиска, возможность пометки спама в Android Market и проверки наличия обновлений для дополнительных приложений. Вопреки ожиданиям, обновление прошивки не вобрало в себя все новшества доступные в экспериментальной ветке "cupcake", такие как запись видео со встроенной камеры, помещение в буфер обмена статических областей текста (сейчас можно только выделять области в полях ввода), Bluetooth профили A2DP и AVRCP (стерео звук), обновление браузера (перевод на ноябрьскую сборку WebKit и JavaScript движок SquirrelFish, возможность поиска в содержимом текущего окна), добавление фреймворка для экранных клавиатур, переход на Linux ядро 2.6.27. Что касается Android SDK 1.1, то в нем представлены незначительные изменения API не влияющие на совместимость с прошлыми выпусками, например, добавлен класс для прямой работы с данными Google Maps, улучшены средства тестирования. Кроме того, улучшена работа эмулятора телефона, добавлена поддержка локализации для немецкого языка. В заключение можно отметить заявление компании Archos о подготовке мультимедиа планшета на базе платформы Android. Устройство, олщиной 1 сантиметр, будет снабжено 5-дюймовым сенсорным экраном, жестким диском размером до 500 Гб, будет поддерживать работу в 3.5G HSDPA/HSUPA сетях. Время автономной работы составляет 7 часов в режиме просмотра видео. Из программ будет присутствовать полный набор мультимедиа приложений, web-браузер с Flash плагином, набор игр. В продажу устройство поступит в третьем квартале текущего года.
