Вышла вторая бета-версия Red Hat Enterprise Linux 6.0

Компания Red Hat выпустила вторую тестовую версию дистрибутива RHEL 6.0, релиз которого планируется выпустить в конце года. Во второй бета-версии обновлена программа установки, проведена работа по устранению обнаруженных в beta1 недоработок, обновлено входящее в поставку Linux-ядро и включены новые версии программ. Начиная со второй бета-версии дистрибутив реорганизован следующим образом: Серверный вариант: Red Hat Enterprise Linux Server Кластерное хранилище (Clustered Storage); Файловая система большого размера (Large File System); Высокая доступность (High Availability); Балансировка нагрузки (Load Balance). Десктоп-вариант Red Hat Enterprise Linux Workstation Large File System Подробный обзор ожидаемых в RHEL 6 новшеств можно прочитать в анонсе первой бета-версии, кроме того можно упомянуть обзор планов по улучшению входящего в комплект Linux-ядра. Загрузить RHEL 6.0 beta2 можно здесь.
