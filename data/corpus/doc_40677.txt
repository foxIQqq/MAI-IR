Компания Oracle выпустила пакеты с OpenStack для Oracle Linux

Компания Oracle выпустила новый продукт - Oracle OpenStack for Oracle Linux 1.0, который предоставляет средства для развёртывания облачных инфраструктур и выступает конкурентом Red Hat Enterprise Linux OpenStack Platform. Поддерживается два набора пакетов: для развёртывания управляющих и вычислительных узлов. Пакеты с OpenStack для Oracle Linux, а также все связанные с ними обновления, доступны для установки без ограничений через публичный репозиторий. Также подготовлен демонстрационный образ для экспериментов с облачными системами в VirtualBox. Кроме того, заявлено о сотрудничестве Oracle и Canonical по предоставлению поддержки клиентам, использующим Ubuntu и Oracle Linux в гостевых системах, выполняемых в облачных инфраструктурах на основе OpenStack-решений Oracle и Canonical. В рамках сотрудничества Canonical обеспечит поддержку гостевых систем с Ubuntu в Oracle Linux OpenStack и проведёт тестирование работы Oracle Linux в Ubuntu OpenStack. В свою очередь Oracle обеспечит поддержку Oracle Linux в Ubuntu OpenStack и проведёт тестирование Ubuntu в Oracle Linux OpenStack.
