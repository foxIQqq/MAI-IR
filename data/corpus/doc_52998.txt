Доступен Solaris 11.4 SRU21

Опубликовано обновление операционной системы Solaris 11.4 SRU 21 (Support Repository Update), в котором предложена серия очередных исправлений и улучшений для ветки Solaris 11.4. Для установки предложенных в обновлении исправлений достаточно выполнить команду 'pkg update'. В новом выпуске: Добавлен новый пакет driver/network/ethernet/mlxne с драйвером для поддержки Ethernet-адаптеров Mellanox ConnectX-4 и ConnectX-5 100Gb; Обновлены компоненты подсистемы печати: вместо старого сups -1.4.5 добавлены cups 2.3.1, cups-filters 1.25.11, ghostscript 9.27, hplip 3.19.5. gutenprint 5.2.13; В состав включены fribidi, libsass, sassc, libdazzle, mallard-ducktype, cidfmap и dbus-python (для Python 3.7); Обновлены компоненты GNOME (до ветки 3.32 и частично до 3.34) и графический стек (Xorg Server 1.20.6, GTK 3.24.11); Предложены новые версии программ, среди которых: Doxygen 1.8.15 GNU emacs 26.3 OpenLDAP 2.4.49 OpenSSH 8.1 OpenSSL 1.1.1 PHP 7 7.4.3 SCons 3.1.1 9 Samba 4.11.6 Swig 4.0.1 cairo 1.16.0 clutter 1.26.2 dconf 0.34.0 flac 1.3.3 fltk 1.3.5 gawk 5.0.1 glib 2.62.2 gstreamer 1.16.2 harfbuzz 2.6.4 hg 5.1.1 libevent 2.1.11 libvdpau 1.3 libvpx 1.8.2 libwebp 1.0.3 mutt 1.13.4 ninja 1.9.0 pango 1.43.0 pcre 8.43 pidgin 2.13.0 rdesktop 1.9.0 rrdtool 1.7.0 rsync 3.1.3 sqlite 3.31.1 vala 0.46.1 vinagre 3.22.0 xattr 0.9.7 Обновлены версии с устранением уязвимостей: Ant 1.10.7 BIND 9.11.13 Django 1.11 1.11.29 Django 2.2 2.2.11 Imagemagick 6.9.10-91 MySQL 5.6 5.6.47 MySQL 5.7 5.7.29 Node.js 10.19.0 Node.js 8.15.0 OpenSSL 1.0.2u PyYAML 5.2 Qt library 5.12.7 aspell 0.60.8 cups 2.3.0 gettext 0.20.1 libarchive 3.4.2 libdwarf 20191104 libssh2 1.9.0 pcre2 10.34 squid to 4.10 tigervnc 1.10.1 waitress 1.4.3 webkitgtk 2.26.4 gnome/applications gnome/multimedia gnome/pango library/gd2 library/libexif library/icu library/libxml library/qt library/webkitgtk service/proftpd utility/gnu-compiler utility/python utility/qemu-img
