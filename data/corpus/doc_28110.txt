Релиз открытой CRM-системы vtiger CRM 5.2.0

Представлен релиз vtiger CRM 5.2.0, многоплатформенной системы для организации взаимодействия с клиентами на предприятии, обеспечения работы службы продаж, ведения партнерской сети, организации работы службы маркетинга, оценки эффективности маркетиговых акций, поддержания инвентаризационной базы и корпоративной базы знаний, а также для координирования и планирования работы сотрудников. Код vtiger CRM написан на языке PHP и распространяется в полном объеме в рамках лицензии MPL. Система снабжена гибкими механизмами для подключения дополнений, расширения функциональности и адаптации системы под нужды конкретного предприятия. Vtiger CRM является самодостаточной системой, все необходимые для работы компоненты входят в состав единого установочного образа, который доступен для Windows, RedHat, Debian, SuSe, Fedora и Mandriva. Такие составляющие как web-сервер Apache, СУБД MySQL и почтовый сервер, также автоматически настраиваются программой установки и не требуют привлечения дополнительных специалистов. Из дополнительных возможностей отмечается наличие модуля для создания корпоративного портала для организации взаимодействия с клиентами и системы для поддержания инфраструктуры электронной почты, с наличием плагинов для поддержки Microsoft Outlook и Microsoft Word, а также поставка специального дополнения к Thunderbird. Из компаний использующих vtiger CRM отмечаются такие гиганты, как Nokia, Aegon USA, биржа NYSE Euronext, крупнейший инвестиционный банк Германии - Postbank. В новой версии исправлено около 350 ошибок и представлено более 50 улучшений, из которых можно отметить: Модуль для организации управления проектами; Модуль для управления активами предприятия; Поддержка отправки SMS-уведомлений клиентам; Модуль для обеспечения работы с системой с мобильных устройств, таких как IPhone и Android. Создание специального API для разработки мобильных приложений; Возможность добавления комментариев ко всем аккаунтам, контактам и объектам других модулей; Поддержка импорта и экспорта данных календаря-планировщика в формате iCal; Расширение возможностей генератора отчетов, реализация новых операторов для формирования выборки данных для отчета, добавление функции создания групп, улучшение качества создания отчетов в формате PDF.
