В Web-браузере Opera 24 обеспечена поддержка платформы Linux

Спустя год после перехода на браузерный движок Chromium и прекращения формирования выпуска для Linux, компания Opera Software объявила о возобновлении поддержи платформы Linux и опубликовала тестовый выпуск Opera 24, доступный не только для Windows и OS X, но и для Linux. В настоящее время сформированы только 64-разрядные сборки для Ubuntu Linux, рассчитанные на работу в окружениях Unity и GNOME Shell. Работа в других системах не гарантируется и будет реализована по мере развития продукта. Ключевые отличия Opera 24 от ранее доступного для Linux выпуска Opera 12.16: Переход с собственного движка Presto на браузерный движок Blink (форк WebKit) и JavaScript-движок V8, развиваемые в рамках проекта Chromium. Для запуска плагинов, таких как Adobe Flash, предлагается поддержка PPAPI вместо NPAPI. Новый API для разработки дополнений, основанный на API Chrome и позволяющий использовать дополнения, созданные для Chrome; Полностью переработанный пользовательский интерфейс. Проведение чистки расширенной функциональности, удалены такие компоненты, как боковая панель, классическая система закладок, почтовый клиент и интерфейс для чтения RSS. Интерфейс реализован с использованием графического стека Aura; Функция Stash, выступающая в качестве аналога системы закладок. После пометки сайта через "сердечко" рядом с адресной строкой, миниатюра сайта добавляется в специальный список избранного и становится доступна для поиска на странице при открытии новой вкладки; Новая панель Discover, предназначенная для обнаружения нового контента в Сети и слежения за новостными лентами по интересам. При этом браузер сам формирует подборку источников в зависимости от выбранной тематики и производит фильтрацию популярного и востребованного контента; Переработанная экспресс-панель, которая тесно интегрирована с системой ведения закладок и комбинирует при выводе как ссылки для быстрого запуска, так и разделы системы закладок. Поддерживаются средства группировки, переименования и перемещения содержимого экспресс-панели, а также объединение ссылок по разделам; Для ускорения работы на низкоскоростных каналах связи или при необходимости экономить трафик, предусмотрена новая реализация режима Opera Turbo, переведённая на использование протокола SPDY. При включении режима Turbo обработка контента производится на стороне серверной инфраструктуры Opera Software с передачей данных клиенту в компактном виде, что позволяет снизить трафик и обеспечить комфортную работу при роуминге или при работе в 2G-сетях; Режим Tab Peek для быстрого предпросмотра вкладки при наведении курсора, без явного перехода на вкладку; Комбинированная адресная строка, в которой объединены поле для ввода адресов и форма для отправки поисковых запросов. Дополнение: Доступен неофициально переработанный пакет для Arch Linux и инструкция по ручной установке в различных дистрибутивах.
