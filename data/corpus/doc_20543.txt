Для Ubuntu создан архив пакетов с немодифицированным ядром Linux

Для Ubuntu создан архив пакетов с немодифицированным ядром Linux, собранным на базе исходных текстов представленных на сайте kernel.org ("ванильное" ядро), без добавления дополнительных патчей. Пакеты представлены для всех вновь выпускаемых стабильных и предварительных ("-rc") релизов. Сборка организована для пользователей желающих принять участие в тестировании ядра, дает возможность без лишних усилий познакомиться с представленными в последних релизах новшествами и может быть использованы для проверки на наличие регрессивных изменений в базовом пакете с ядром из дистрибутива Ubuntu, связанных с внесением специфичных для Ubuntu патчей. В архиве в настоящий момент представлены deb-пакеты с ядрами 2.6.16, 2.6.24.x, 2.6.27.x, 2.6.28.x, 2.6.29-rc*. Для представленных ядер отсутствуют сборки проприетарных модулей и специфичных Ubuntu драйверов, требующих модификации ядра.
