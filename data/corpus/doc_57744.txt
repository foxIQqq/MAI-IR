Новый выпуск дистрибутива Raspberry Pi OS

Разработчики проекта Raspberry Pi опубликовали осеннее обновление дистрибутива Raspberry Pi OS 2022-09-06 (Raspbian), основанного на пакетной базе Debian. Для загрузки подготовлены три сборки - сокращённая (338 МБ) для серверных систем, с базовым рабочим столом (891 МБ) и полная с дополнительным набором приложений (2.7 ГБ). Дистрибутив поставляется с пользовательским окружением PIXEL (ответвление от LXDE). Для установки из репозиториев доступно около 35 тысяч пакетов. В новом выпуске: В меню приложений реализована возможность поиска по именам установленных программ, что упрощает навигацию с использованием клавиатуры - пользователь может вызвать меню нажатием клавиши Windows, после чего сразу начать набор маски для поиска и после получения списка соответствующих запросу приложений выбрать нужное курсорными клавишами. На панели реализованы раздельные индикаторы для управления громкостью и чувствительностью микрофона (ранее предлагался общий индикатор). При клике на индикаторах правой кнопкой мыши показываются списки доступных устройств ввода и вывода звука. Предложен новый программный интерфейс для управления камерой - Picamera2, представляющий собой высокоуровневую обвязку над библиотекой libcamera на языке Python. Предложены новые клавиатурные комбинации: Ctrl-Alt-B для открытия меню Bluetooth и Ctrl-Alt-W для открытия меню Wi-Fi. Обеспечена совместимость с сетевым конфигуратором NetworkManager, который теперь можно использовать в качестве опции для настройки беспроводного соединения вместо штатно применяемого фонового процесса dhcpcd. По умолчанию пока остаётся dhcpcd, но в будущем планируется переход на NetworkManager, в котором предлагается множество дополнительных полезных возможностей, таких как поддержка VPN, возможность создания беспроводной точки доступа и подключение к беспроводным сетям со скрытым SSID. Перейти на NetworkManager можно в разделе расширенных настроек конфигуратора raspi-config.
