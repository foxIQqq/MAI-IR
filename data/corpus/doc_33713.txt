Бывший глава Sun Microsystems считает, что Android не нуждался в лицензии на Java API

На проходящее в эти дни судебное разбирательство, касающееся обвинения компании Google в нарушении связанной с Java интеллектуальной собственности в платформе Android, в качестве свидетеля был вызван Джонатан Шварц (Jonathan Schwartz), бывший руководитель компании Sun Microsystems, который выступил с неожиданным заявлением. Анализируя положение вещей в момент зарождения платформы Android юристы Google попросили Джонатана Шварца высказаться по поводу утверждений Oracle о том, что Java API в то время являлся проприетарной собственностью Sun. На что Шварц ответил, что он не согласен с такой формулировкой, так как в то время платформа Java уже была переведена в разряд открытых проектов и воспринималась Sun именно как открытый продукт. По словам Шварца, Java API позиционировался как открытый и компания Sun была заинтересована в привлечении к его развитию как можно большего числа людей. Ограничения касались только имени Java, но так как Google не использовала в названии своей виртуальной машины слово Java, она вольна была использовать элементы Java API, не нарушая при этом интеллектуальной собственности Sun.
