Представлен проект Qt for Python

Разработчики Qt представили проект Qt for Python, в рамках которого будет развиваться набор модулей для создания графических приложений на языке Python с использованием Qt5. Модуль PySide2 сохранит своё прежнее имя, Qt for Python лишь проект для разработки PySide2 и других возможностей, связанных с поддержкой языка Python. Смена имени пакета призвана предоставить цельное решение для использования Qt в Python-приложениях, включающие сопутствующие сервисы, такие как оказание коммерческой технической поддержки. На текущем этапе содержимое Qt for Python полностью аналогично PySide2. Первый экспериментальный выпуск (Technology Preview) Qt for Python планируется опубликовать вскоре после Qt 5.11.
