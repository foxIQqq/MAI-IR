Обновление Java SE 6 Update 38 и Java SE 7 Update 10

Доступны корректирующие выпуски Java SE 6 Update 38 с исправлением 33 ошибок и Java SE 7 Update 10 с устранением 55 ошибок. В представленных выпусках представлены только не связанные с безопасностью исправления, устранения уязвимостей были представлены в версиях Java SE 6 Update 37 и Java SE 7 Update 9. JDK 7 Update 10 доступен в сборках для Linux x86, Linux ARM, Solaris, Windows и Mac OS X. Отдельно стоит напомнить, что выпуск обновлений для ветки Java SE 6 продлится до февраля 2013 года, после чего обновления будут доступны только в рамках платного расширенного сервиса поддержки. Из изменений, появившихся в Java SE 7 Update 10, можно отметить появление двух дополнительных сертифицированных конфигураций (Mac OS X 10.8 и Windows 8) и реализацию нескольких улучшений, касающихся безопасности: возможность блокирования через Java Control Panel запуска в браузере произвольного Java-приложения; поддержка выбора одного из четырёх предлагаемых уровней безопасности для неподписанных апплетов, приложений Java Web Start или JavaFX, запускаемых в браузере; новые диалоги с предупреждениями об использовании небезопасной (устаревшей) версии JRE и с предложением выполнить обновление.
