Обновление VirtualPBX 4846, продукта для развертывания голосового сервиса на базе Asterisk

Подготовлена новая сборка продукта VirtualPBX, предназначенного для быстрого развертывания публичного Voice2.0-сервиса на базе ПО Asterisk. С обзором возможностей VirtualPBX можно познакомиться в тексте первого анонса. Список основных изменений: Добавлена поддержка TTS (text-to-speech) для русского языка на основе GoogleTranslate; Добавлена возможность создания предварительно настроенных шаблонов для популярных SIP провайдеров; Добавлена возможность записи исходящих пользовательских звонков; В chatroom интегрирован IceCast для трансляции конференции в интернет; Добавлена возможность записи конференции в chatroom; В vmware имидже обновлен CentOS до 5.6, доустановлен fail2ban, cache-dns; Работа с GoogleCalendar: добавлена возможность чтения событий/даты/времени через телефон ( используя TTS ), возможность "Подтверждения/Отмены" событий в google-календаре через телефон; При просмотре CDR добавлена возможность получить полную DTMF-историю для каждого звонка; В CDR добавлена информацию о том кто завершил входящий звонок (USER/PBX/IVR); При просмотре голосовых сообщений добавлены новые фильтры для более быстрого поиска сообщений; Добавлен параметр настройки InterDigit Timeout для всего IVR; Добавлена настраиваемая пауза после воспроизведения приветствия автоответчика; Добавления скрипт для установки дополнительных файлов Music On Hold; Изменен WEB-интерфейс администратора системы; Лимиты максимальной длительности входящего звонка для группы пользователей; Возможность задать свой путь к звуковым файлам для группы пользователей. Загрузочный образ для VMWare можно загрузить здесь, документация: администратора, пользователя.
