Выпуск Foliate 1.0, программы для чтения книг в формате EPUB

Доступен первый выпуск программы для чтения электронных книг Foliate. Код написан на JavaScript и поставляется под лицензией GPLv3. Для формирования интерфейса применяется библиотека GJS (обвязка над GTK), а для обработки формата EPUB - Epub.js. Основные функции: Просмотр файлов epub; Двухстраничный просмотр; Настройка шрифта; Выбор цвета фона; Навигация с отметками глав; Закладки и аннотации; Поиск по книге; Быстрый поиск по словарю; Поддержка жестов.
