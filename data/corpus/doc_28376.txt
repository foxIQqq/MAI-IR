Релиз Bordeaux 2.0.10 для OpenIndiana и CrossOver 9.2 для Linux и MacOS X

Представлены две новые версии конкурирующих между собой проприетарных пакетов для обеспечения работы Windows-приложений в Unix-подобных ОС: Вышла версия Bordeaux 2.0.10 для Solaris-дистрибутива OpenIndiana, в рамках которого создано полностью свободное и развиваемое независимым сообществом ответвление от кодовой базы OpenSolaris. Bordeaux является коммерческим продуктом, бинарные сборки распространяются по цене 25$, в которую входит 6-месячная поддержка. Но проект не бесполезен для сообщества, все примененные к Wine патчи, скрипты, а также компоненты для упрощения установки и загрузки приложений, доступны в исходных текстах под открытыми лицензиями. Bordeaux основан на Wine 1.2.1 и использует Winetricks, Cabextract, Firefox 3.6.8 , Unzip, Wget и другие открытые компоненты, позволяя выполнять следующие приложения: Microsoft Office 97, 2000, 2003, 2007 Microsoft Office Visio 2003 Microsoft Office Project 2003 Adobe Photoshop 6, 7, CS, CS2 Adobe Image Ready 3, 7 Microsoft Internet Explorer 6, 7 Apple Safari 5.0 Игры на базе Steam и Steam Google SketchUp 7.1 Apple QuickTime 6.5.2 Player IrfanView 4.27 Компания CodeWeavers выпустила финальный релиз Crossover 9.2 для платформ Linux и MacOS X - коммерческой версии Wine, поддерживающей из коробки большое количество Windows-приложений, в особенности приложений из набора Microsoft Office. Компания CodeWeavers спонсирует разработку Wine и возвращает в проект около 98% всех новшеств, реализованных для коммерческих продуктов компании. Ключевой особенностью версии Crossover 9.2 версии является поддержка игры Civilization 5.
