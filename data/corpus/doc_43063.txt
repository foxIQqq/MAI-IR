Первый выпуск Raspbian на базе Debian 8.0 "Jessie". Ubuntu Snappy для Raspberry Pi

Представлен выпуск дистрибутива Raspbian 2015-09-24, официально рекомендуемого для использования на Raspberry Pi и специально оптимизированного для данной платформы (сборка для ARMv6 с расширениями "hard float"). Размер образа для записи на SD-карту составляет 1.3 Гб. Для установки из репозиториев доступно около 35 тысяч пакетов. Основные новшества: Переход на пакетную базу Debian Jessie (ранее использовался Debian Wheezy); По умолчанию загрузка теперь осуществляется в графический интерфейс пользователя (при желании загрузку в командную строку можно вернуть в конфигураторе образа); Модернизировано оформление пользовательской оболочки LXDE, которая переведена на использование GTK3+; Представлен новый графический конфигуратор, аналог ранее предлагаемого консольного конфигуратора raspi-config, поддержка которого сохранена; Новый диалог настройки раскладки клавиатуры; В базовую поставку включён офисный пакет LibreOffice и почтовый клиент Claws Mail; Добавлены два окружения для разработки приложений на языке Java - Greenfoot (для обучения программирования на Java) и BlueJ, а также среда для создания игр Pygame Zero; Добавлены новые утилиты: редактор меню Alacarte, Lxkeymap, scrot, tree и pip; Обновлены версии браузера Epiphany, визуальной системы программирования Scratch и среды программирования Sonic Pi; Возможность управления GPIO без использования sudo; Новый плагин панели инструментов для извлечения примонтированных USB-накопителей; К клавише Print screen привязан запуск утилиты для создания скриншота; Реализовано автоматическое распознание типовых клавиатур и загрузка драйвера для них. Дополнительно можно отметить выход новой сборки (202 Мб) Ubuntu Snappy 15.04 для Raspberry Pi. Сборка монолитная, сформированная с использованием модели атомарного обновления образа всей системы, без разделения на отдельные deb-пакеты. В новой версии задействовано ядро Linux 4.2, добавлена поддержка по умолчанию I2C, SPI и GPIO, включено содержимое пакета linux-firmware с подборкой прошивок для разнообразных дополнительных устройств.
