Компания Oracle выпустила обновление ядра Unbreakable Enterprise Kernel R3U4

Компания Oracle выпустила четвёртое функциональное обновление для ядра Unbreakable Enterprise Kernel 3 (UEKR3U4). Исходные тексты ядра, включая разбивку на отдельные патчи, доступны в публичном Git-репозитории Oracle. Ядро позиционируется в качестве альтернативы штатному пакету с ядром, поставляемому в Red Hat Enterprise Linux, и предоставляет ряд расширенных возможностей, таких как интеграция DTrace и улучшенная поддержка Btrfs. Пакеты с ядром подготовлены для Oracle Linux 6 и Oracle Linux 7. Наиболее заметные улучшения: Обеспечение возможности выборочной автоматической загрузки модулей DTrace. Модули, которые следует загрузить при старте системы, следует указать в файле /etc/dtrace-modules. Поставка драйвера для SAS/SATA контроллера PMC-Sierra PM8001, используемого в новых 8- и 16-портовых SAS/SATA адаптерах Adaptec Series 7H; Включение драйвера i40evf для сетевых адаптеров Intel XL710 X710 Virtual Function; Добавление драйвера intel_pstate для управление параметрами энергопотребления (P-state) на системах с процессорами Intel Sandy Bridge; Внесение порции оптимизаций, направленных на увеличение производительности виртуальных машин в окружении Microsoft Hyper-V; Включение предварительной реализации клиента для работы NFS поверх RDMA (Remote Data Memory Access); Обновление версий драйверов для устройств хранения и сетевых карт на чипах Broadcom, Emulex, HP, Intel, LSI, Mellanox, PMCS, QLogic и Tigon.
