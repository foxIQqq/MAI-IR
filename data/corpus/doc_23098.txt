Видеозаписи с конференции LVEE-2009

Опубликованы 28 видеозаписей докладов с конференции Linux Vacation Eastern Europe, прошедшей в республике Беларусь в июле. Некоторые из докладов: "Использование XEN вместе с DRBD"; "Настройка и использование SELinux"; "Интеграция IP ATC Asterisk cо Skype"; "Система обнаружения атак LIDS"; "Новые продукты ZyXEL на базе Linux"; "Корпоративная CMS/CMF Typo3"; "Использование IP-АТС Asterisk на платформе Blackfin"; "Опыт применения и проблемы использования приложений с открытым исходным кодом в медицинских научных исследованиях"; "Использование платформы TERM для автоматизации измерительного оборудования"; "Модернизация программного обеспечения ADSL-роутера"; "Использование свободного ПО в задачах компьютерной лингвистики"; "SynCE для синхронизации данных с КПК Windows Mobile"; "Neo Freerunner, Gentoo, Openmoko"; "Система управления распределенным набором виртуальных машин";
