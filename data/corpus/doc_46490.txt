Выпуск операционной системы Chrome OS 58

Компания Google представила релиз операционной системы Chrome OS 58, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 58. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 58 доступна для всех актуальных моделей Chromebook, за исключением Acer Chromebase 24, Acer Chromebook R11, AOpen Chromebase Mini, AOpen Chromebox Mini, Google Chromebook Pixel (2015), ASUS Chromebook Flip C100PA, Dell Chromebook 11 (3120), Lenovo ThinkPad 11e Chromebook и Samsung Chromebook Plus. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 58: Модернизированы правила определения часового пояса на устройствах, подключающихся к сотовой сети; Обновлено оформление интерфейса для первого знакомства пользователя с устройством; Анимация при загрузке приведена к виду, более соответствующему концепции Material Design; В файловый менеджер добавлена возможность доступа к изображениям, видео и звуковым файлам от установленных приложений для Android (доступ открыт в режиме только для чтения); На устройствах с сенсорными экранами с поддержкой стилуса (Samsung Chromebook Plus), появилась возможность использования стилуса для рисования, для написания заметок, для создания снимков экрана, в качестве лазерной указки и для увеличения области на экране; В приложение Camera добавлена возможность переключения между подключенными к устройству несколькими камерами; По умолчанию предложены новые обои для рабочего стола; Возможность настройки производителями устройств ссылки для обращения в службу поддержки; Устранена уязвимость CVE-2017-0561 в прошивке Wi-Fi чипов Broadcom, позволяющая организовать удалённое выполнение кода через отправку по беспроводной сети специально оформленных управляющих кадров.
