Началось публичное тестирование технологии синхронизации компьютеров BitTorrent Sync

Компания BitTorrent, Inc, отвечающая за развитие P2P-протокола BitTorrent и клиентского ПО μTorrent, объявила о начале открытого тестирования приложения BitTorrent Sync, предназначенного для синхронизации данных между компьютерами с использованием P2P-коммуникаций без наличия централизованных серверных систем. Например, BitTorrent Sync можно использовать для удалённого сохранения бэкапов, передачи больших наборов данных между пользователями и компьютерами, организации совместной работы над общим набором данных. Поддерживается как полностью децентрализованная работа с задействованием распределённой хэш-таблицы, так и использованием трекеров, которые получают доступ только к связанным с данными хэшам SHA1, а также прямое соединение клиентов по IP-адресам и c автоматическим обнаружением устройств в пределах локальной сети. Имеется возможность поднять собственный трекер на одной из синхронизированных машин. Использование трекера увеличивает скорость и надёжность синхронизации. Синхронизированные данные размещаются только на собственных компьютерах и не оседают на промежуточных и неподконтрольных системах. Участвующие в синхронизации системы должны быть доступны в сети. В процессе передачи данные шифруются с использованием алгоритма блочного шифрования AES 256. Протокол оптимизирован для передачи между системами очень больших файлов, которые могут эффективно передаваться без дополнительного сжатия. По сравнению с первой тестовой версией реализован режим однонаправленной синхронизации (одна их сторон доступна только в режиме чтения), возможность использования одноразовых ключей доступа, задание фильтров для выборочной синхронизации данных, улучшен web-интерфейс для Linux. Программа доступна для Linux, Windows и Mac OS X.
