Chrome занял первое место по недельным показателям рейтинга браузеров StatCounter

По данным рейтинга StatCounter, осуществляющего мониторинг общемировой статистики использования web-браузеров, Chrome вырвался на первое место, заняв 32.76% рынка браузеров по средним показателям за прошлую неделю. Internet Explorer вытеснен на второе место c показателем 31.94%. Напомним, что в начале года Chrome занимал 26% рынка, а Internet Explorer - 41%. Доля Firefox сохраняется на отметке 25%. Статистика других источников: По статистике Wikimedia за апрель доля Internet Explorer составляет 25.36%, Chrome - 24.99%, Firefox - 21.77%, Safari - 5.82%, Opera - 3.71%; Согласно апрельскому отчету общемирового рейтинга Net Applications, IE занимает 54.09% рынка, Firefox - 20.20%, Chrome - 18.85%, Safari - 4.81%, Opera - 1.63%; По данным глобальной статистики spylog.ru, Chrome занимает 26.30% рынка, Internet Explorer - 23.31%, Firefox - 20.72%, Opera - 19.91% (+ Opera mini 3.86%), Safari - 3.61%; По статистике liveinternet.ru, Chrome занимает 20.7%, Opera - 16.4%, Opera Mini - 13.3%, Internet Explorer - 11.7%, Firefox - 13.7%; На opennet.ru доля пользователей Firefox составляет 37.37%, Google Chrome - 37.23%, Opera - 15.30%, Internet Explorer - 5.70%.
