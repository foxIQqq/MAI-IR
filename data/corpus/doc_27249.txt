Субноутбук OLPC XO-1.75 будет поставляться с поддержкой мультитач-режима

Представители проекта One Laptop Per Child сообщили, что следующая модель обучающего субноутбука XO-1.75, оформленного в книжном формате, будет снабжена двумя 8.9-дюймовыми сенсорными экранами, поддерживающими технологию мультитач. Программное обеспечение устройства, несмотря на переход на архитектуру ARM, будет по прежнему основано на Linux-дистрибутиве Fedora и обучающей оболочке Sugar. Для продвинутых пользователей будет предоставлена возможность запуска стандартного десктоп-окружения GNOME. Программное обеспечения планируется модифицировать для обеспечения управления через сенсорный экран и для распознавания одновременных касаний к экрану. В частности, поддержка мультитач будет добавлена в экранную клавиатуру fvkbd, что позволит значительно повысить удобство ввода текста с экрана. Видеоролик с демонстрацией прототипа мультитач-интерфейса оболочки Sugar в сочетании с fvkbd можно посмотреть здесь (OGV, 800 Кб). В продажу модель XO-1.75, основанная на ARM-совместимой платформе, поступит в начале следующего года. Следом за XO-1.75 в 2012 году ожидается выход планшета XO-3, который так же будет снабжен поддержкой режима мультитач. Дополнительно сообщается о выпуске новой версии дистрибутива OLPC 10.1.1, основанного на пакетной базе Fedora. В новой версии обеспечена поддержка акселерации при просмотре видео, добавлена возможность использования GSM- и 3G-модемов через GUI-интерфейс Modem Configuration, добавлена кнопка для принудительного выключения беспроводной карты, в GNOME-окружение добавлен индикатор заряда батареи. Для первой модели XO-1 проект OLPC официально не планирует адаптировать новую версию платформы, решив ограничиться только моделью XO-1.5, но группа энтузиастов намерена выпустить неофициальную сборку OLPC 10.1.1 для устаревших устройств XO-1.
