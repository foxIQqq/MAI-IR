Выпуск Ubuntu 16.04.4 LTS c обновлением графического стека и ядра Linux

Опубликовано обновление дистрибутива Ubuntu 16.04.4 LTS, в которое включены изменения, связанные с улучшением поддержки оборудования, обновлением ядра Linux и графического стека, исправлением ошибок в инсталляторе и загрузчике. В состав также включены актуальные обновления для нескольких сотен пакетов, связанные с устранением уязвимостей и проблем, влияющих на стабильность. Одновременно представлены аналогичные обновления Kubuntu 16.04.4 LTS, Xubuntu 16.04.4 LTS, Mythbuntu 16.04.4 LTS, Ubuntu GNOME 16.04.4 LTS, Lubuntu 16.04.4 LTS, Ubuntu Kylin 16.04.4 LTS, Ubuntu MATE 16.04.4 LTS и Ubuntu Studio 16.04.4 LTS. В состав выпуска включены некоторые улучшения, бэкпортированные из выпуска Ubuntu 17.10: Предложено обновление пакетов с ядром 4.13 (в Ubuntu 16.04.3 использовалось ядро 4.10, а в основном 16.04 - 4.4). Обновлены компоненты графического стека, включая X Server 1.19.5, libdrm 2.4.83, Compiz и Mesa 17.2.8, которые были протестированы в осеннем выпуске Ubuntu 17.10. Добавлены свежие версии видеодрайверов для чипов Intel, AMD и NVIDIA. Обновлены версии пакетов modemmanager, skiboot, libseccomp, PHP (7.0.25), snapd, walinuxagent, curtin, ceph, cloud-init, lxd (2.0.11), maas (2.3.0), juju-core, OpenStack, intel-microcode, pollinate, postgresql-9.5 и lxcfs. В редактор разделов parted добавлена поддержка устройств NVMe. В debian-installer в сборки ARM64 добавлен системный раздел EFI (EFI System Partition) для корректной загрузки с USB-носителей при использовании EFI. Перенесена большая порция исправлений, связанных с окружением Unity. В PulseAudio добавлена поддержка вывода звука через USB на док-станцию Dell TB16. В systemd-networkd добавлена поддержка настроек AgeingTImeSec, Priority, DefaultPVID, ActiveSlave и PrimarySlave. Обновления ядра и графического стека доступны для всех архитектур, за исключением 32-разрядных систем powerpc. В сборках для рабочего стола новые ядро и графический стек предложены по умолчанию. Для серверных систем новое ядро добавлено в качестве опции в инсталляторе. Использовать новые сборки имеет смысл только для новых установок, системы, установленные ранее, могут получить все присутствующие в Ubuntu 16.04.4 изменения через штатную систему установки обновлений. Поддержка выпуска обновлений и исправлений проблем безопасности для серверной и десктоп редакций Ubuntu 16.04 LTS продлится до апреля 2021 года. При этом, для поставки новых версий ядра и графического стека применяется rolling-модель поддержки обновлений, в соответствии с которой бэкпортированные ядра и драйверы будут поддерживаться только до выхода следующего корректирующего обновления LTS-ветки Ubuntu. Например, предложенное в текущем выпуске ядро Linux 4.13 будет поддерживаться до выхода Ubuntu 16.04.5, в котором будет предложено ядро из состава Ubuntu 18.04. Изначально поставляемое базовое ядро 4.4 будет поддерживаться в течение всего цикла сопровождения до 2021 года. Для перевода уже существующих установок на новые версии ядра и графического стека следует выполнить команду: sudo apt-get install --install-recommends linux-generic-hwe-16.04 xserver-xorg-hwe-16.04
