Обновление Solaris 11.4 SRU 19

Опубликовано обновление операционной системы Solaris 11.4 SRU 19 (Support Repository Update), в котором предложена серия очередных исправлений и улучшений для ветки Solaris 11.4. Для установки предложенных в обновлении исправлений достаточно выполнить команду 'pkg update'. В новом выпуске: До версии 20.1 обновлён Oracle Explorer, инструментарий для построения детального профиля конфигурации и состояния системы; В состав включён сборочный инструментарий meson; Для Python 3.7 добавлены пакеты с idna и Django 2.2; В SNMP реализована поддержка аутентификации с использованием хэшей SHA2; В auditd исправлена ошибка в функции gethname() модуля audit_syslog, приводившая к аварийному завершению работы с генерацией дампа памяти; В состав включены Python-модули colorama, wcwidth, pyperclip, Automat, constantl,y PyHamcrest, hyperlink, incremental python; Обновлены версии cmd2 0.9.13, pyparsing 2.4.0, WebOb 1.8.5, Cython 0.29.14, idna 2.8, mysqlclient 1.4.5; Обновлены версии с устранением уязвимостей: git 2.19.3, libarchive 3.4.1, python 3.5.9/3.7.5, Twisted 18.9.0, Django 1.11.27/2.2.9, ruby 2.5.7.2.6.5, psutil 5.6.7, Pillow 7.0.0, PHP 7.3.14, Firefox 68.5.0esr, Thunderbird 68.5.0, proftpd.
