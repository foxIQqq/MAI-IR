Консорциум W3C представил JavaScript API для телефонии

Консорциум W3C, курирующий разработку web-стандартов, опубликовал черновой вариант спецификации Web Telephony API, определяющей JavaScript API для управления телефонными вызовами. Спецификация разработана в процессе развития Firefox OS и продвигается при участии компаний Mozilla, Telefónica и Intel. Web Telephony API позволяет создавать приложения, взаимодействующие с подсистемой телефонии на мобильных устройствах, например, программы для приёма и осуществления звонков, поддерживающие работу с различными телефонными сервисами.
