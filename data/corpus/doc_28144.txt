Релиз Wine 1.3.4

Вышел очередной экспериментальный релиз реализации Win32 API для Linux - Wine 1.3.4, с момента релиза 1.3.3 было закрыто 50 отчётов об ошибках. Основные изменения в новой версии: Библиотека Winelib теперь поддерживает платформу ARM; Добавлена поддержка окон программ, зеркально отображённых справа налево; Новое приложение taskkill.exe для завершения работы приложений Windows из командной строки; Появилась начальная реализация апплета настройки интернета контрольной панели; Реализован AcceptEx; Усилены проверки безопасности при установлении зашифрованных соединений SSL; Обновлены переводы; Исправлено множество ошибок.
