Релиз менеджера фотографий Shotwell 0.9

Вышел релиз программы для управления коллекцией фотографий Shotwell 0.9, которая сочетает в себе: инструменты для импорта и конвертации новых фотографий; удобные возможности каталогизации и навигации по имеющейся коллекции, поддержка группировки по времени и тегам; поддержку выполнения типовых операций по обработке изображений (вращение, устранение эффекта красных глаз, корректировка экспозиции, оптимизация цветности и т.п.); средства для публикации в социальных сетях, таких как Facebook, Flickr и Picasa. Готовые бинарные сборки доступны для Linux-дистрибутивов Ubuntu и Fedora. Наиболее важные новшества Shotwell 0.9: Новая панель поиска фотографий, поддерживающая фильтрацию по типу, рейтингу и тексту (заголовок, имя файла, теги); Добавлена поддержка формата TIFF; Подготовлен расширяемый фреймворк для создания плагинов, например, для организации публикации в web или для создания визуальных преобразований для режима слайдшоу; Поддержка печати сразу нескольких фотографий, включая режим размещения нескольких изображений на одной странице; Возможность настройки шаблона пользовательских директорий в процессе импорта фотографий; Поддержка редактирования заголовков сразу нескольких фотографий; Улучшено отображение RAW-фотографий без параметров экспозиции.
