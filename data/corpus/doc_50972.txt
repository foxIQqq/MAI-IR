Релиз Chrome OS 75

Компания Google представила релиз операционной системы Chrome OS 75, основанной на ядре Linux, системном менеджере upstart, сборочном инструментарии ebuild/portage, открытых компонентах и web-браузере Chrome 75. Пользовательское окружение Chrome OS ограничивается web-браузером, а вместо стандартных программ задействованы web-приложения, тем не менее, Chrome OS включает в себя полноценный многооконный интерфейс, рабочий стол и панель задач. Сборка Chrome OS 75 доступна для большинства актуальных моделей Chromebook. Энтузиастами сформированы неофициальные сборки для обычных компьютеров с процессорами x86, x86_64 и ARM. Исходные тексты распространяются под свободной лицензией Apache 2.0. Основные изменения в Chrome OS 75: В окружение для запуска Linux-приложений добавлена возможность использования приложениями VPN-соединений, установленных через существующие VPN-соединения Android или Chrome OS (весь трафик из Linux-окружения может быть завёрнут в существующий VPN); Для Linux-окружения также реализована возможность обращения к Android-устройствам, подключенным через USB-порт (в основном окружении Chrome OS пользователь должен установить опцию совместного использования USB-порта с Linux-окружением); Добавлена поддержка печати с PIN-кодом (при отправке на печать пользователь задаёт PIN-код, а затем подтверждает вывод на печать через ввод данного PIN на клавишной панели принтера). Подобное подтверждение позволяет гарантировать, что важный документ будет распечатан именно на необходимом принтере, а не отправлен по ошибке на другое устройство. Функция доступна только при работе системы в управляемом режиме и при поддержке принтером IPPS и IPP-атрибута "job-password"; В файловом менеджере добавлена поддержка сторонних провайдеров документов (произвольные внешние хранилища), поддерживающих API DocumentsProvider. Пользователь может установить приложения на базе данного API и получить в боковой панели доступ к файлам через выбранного провайдера документов; Добавлена возможность отображения контента, снабжённого средствами защиты авторских прав (DRM), на вторичном внешнем мониторе; В средства родительского контроля добавлена возможность предоставления детям дополнительного бонусного времени работы за компьютером; Для детских учётных записей реализован адаптированный для детей голосовой помощник Google Assistant; Добавлена защита от атак MDS (Microarchitectural Data Sampling) на процессоры Intel.
