Вышел отечественный дистрибутив Calculate Linux 11.6

К 4-х летию проекта выпущен релиз дистрибутива Calculate Linux 11.6, созданного на основе Gentoo Linux, поддерживающего непрерывный цикл выпуска обновлений и оптимизированного для быстрого развёртывания в корпоративной среде. Дистрибутив доступен в 6 редакциях: Calculate Linux Desktop с рабочим столом KDE (CLD), GNOME (CLDG) и XFCE (CLDX), Calculate Directory Server (CDS), Calculate Linux Scratch (CLS) и Calculate Scratch Server (CSS). Особенности Calculate Linux: Готовое клиент-серверное решение. Быстрое развертывание на предприятии. Полноценная работа в гетерогенных сетях. Rolling-release модель обновлений. Включает специально разработанные утилиты Calculate для настройки, сборки и установки системы. Поддерживается интерактивная сборка системы - подготовка ISO-образа системы под свои задачи. Удобство администрирования. 100% совместимость с Gentoo с поддержкой бинарных репозиториев обновлений. Основные изменения: Установка Добавлена поддержка выбора менеджера сети между OpenRC и NetworkManager. В консольном установщике добавлена поддержка настройки сети. Загрузка В initramfs интегрирован Udev для лучшей поддержки устройств и ускорения загрузки livecd. По умолчанию используется загрузчик Grub 2. Добавлено распознавание установленных на диске операционных систем Графика Добавлено автоопределение разрешения экрана в splash. Добавлено меню выбора видеодрайвера. Дополнительно поддерживается драйверы nouveau и ATI. По умолчанию используются свободные видеодрайверы с поддержкой KMS; Проприетарный драйвер Nvidia более не является предустановленным драйвером и устанавливается при выборе пользователем. Распространение Использование XZ формата сжатия livecd образа. Переход на гибридные образы с поддержкой записи на CD/DVD и USB Flash. Для Calculate Scratch Server создан репозиторий бинарных пакетов. Прочее Добавлена интеграция с веб-сервисом pastebin.calculate-linux.ru; В Calculate Directory Server добавлена поддержка хранения истории почтовой переписки. В Calculate Directory Server включен пакет Shorewall для настройки межсетевого экрана. Добавлена поддержка португальского и румынского языков.
