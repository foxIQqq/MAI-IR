Доступен OpenIndiana 2016.04, продолжающий развитие OpenSolaris

Представлен выпуск проекта OpenIndiana 2016.04, в рамках которого независимым сообществом развивается полностью свободный дистрибутив, пришедший на смену бинарному дистрибутиву OpenSolaris, развитие которого было прекращено компанией Oracle. Для загрузки сформировано два вида iso-образов - редакция с консольными приложениями (649 Мб) и сборка с графическим окружением GNOME (1.3 Гб). Сообщается, что 2016.04 является последним выпуском OpenIndiana доступным для 32-разрядных систем (поддержка запуска 32-программ на 64-разрядных системах будет сохранена). OpenIndiana предоставляет пользователю рабочее окружение, построенное на базе свежего среза кодовой базы проекта Illumos. Непосредственно разработка технологий OpenSolaris продолжается проектом Illumos, в котором развивается ядро, сетевой стек, файловые системы, драйверы, а также базовый набор пользовательских системных утилит и библиотек. Основные изменения в OpenIndiana 2016.04: Представлен новый тип изолированных окружений nlipkg (nlipkg zone), которые ведут себя по аналогии со старыми окружениями ipkg zone, но без проверки соответствия компонентов дочернего образа родительскому окружению. Для преобразования зон ipkg в зоны nlipkg достаточно установить system/zones/brand/nlipkg, изменить обработчик на nlipkg и удалить /var/pkg/linked внутри зоны. Вместо закрытого инструмента sysidtool, который используется для установки начальных системных параметров при первой загрузке системы и при конфигурации зон, задействована открытая утилита sysding; Начата публикация файлов userland-incorporation, определяющих состав версий установленных программ и закрепляющих зависимости для выявления несовместимых пакетов. Для установки версий, не определённых в userland-incorporation, следует использовать facet-ы; В текстовом инсталляторе реализована возможность настройки параметров сетевого соединения при помощи утилиты sysding; Добавлена возможность обновления неподдерживаемой ветки "/dev" до актуальной ветки OpenIndiana "/hipster"; Обновлены версии пользовательских программ и библиотек, в том числе Firefox 45.0.2, Thunderbird 38.6.0, GTK2+ 2.24.29 (по умолчанию), ffmpeg 2.8.6, dbus 1.10.8, libotr 4.1.1, openttd 1.5.3, lynx 2.8.8, mutt 1.6.0. В состав включены vlc 2.2.1, GTK3+ 3.14.15, libusb 1.0, fuse 2.7.6, sshfs, geany 1.26, hexchat 2.12.0. Обновлены компоненты графического стека; Удалён сервис postrun, неподдерживаемые панели, EsounD (заменён на pulseaudio) и Openproj; Обновлены средства для разработчиков: GCC 4.9 (GCC 4.7 и Clang 3.3 удалены), Automake 1.15, Erlang 17.5, Ruby 2.3, OpenSSL 1.0.2g, PHP 5.6.20/5.5.34/5.4.45, Perl 5.22.1, SCons 2.4.1, SDL2 2.0.4, OpenMPI 1.10.1, MPICH 3.2, libjpeg-turbo 1.4.2. Компоненты языка Python остались на версии 2.6, к которой привязан illumos-gate; Обновлено серверное ПО: Apache httpd 2.4.20, Nginx 1.8.1, PostgreSQL 9.5.2, Percona server 5.7.10.3, Bind 9.10.3-P4, Samba 4.2.11, Squid 3.5.15, Tomcat 8.0.32, ISC DHCPD 4.3.3-P1, KVM 20160303. Добавлен инструмент резервного копирования Borg 1.0.0. Вместо Sun SSH может быть установлен OpenSSH 7.2p2; Обновлены консольные программы bash 4.3.42, zsh 5.2, rsync 3.1.2, gawk 4.1.3, wget 1.17.1, less 481, gzip 1.7, GNU grep 2.24, tuntap 43816b1.
